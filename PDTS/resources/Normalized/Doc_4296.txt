 فعدتها بوضع الحمل اي انفصال كله حتي ثاني توامين ولو بعد الوفاه لقوله تعالي  :  { واولات الاحمال اجلهن ان يضعن حملهن } فهو مقيد لقوله تعالي  :  { والذين يتوفون منكم ويذرون ازواجا يتربصن بانفسهن اربعه اشهر وعشرا } ولقوله صلي الله عليه وسلم لسبيعه الاسلميه وقد وضعت بعد موت زوجها بنصف شهر  :  { قد حللت فانكحي من شئت } متفق عليه .
