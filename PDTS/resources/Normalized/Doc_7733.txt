وقد عرفت وجه التخصيص قريبا { وسباع البر } بالفتح ضد البحر { وانعامه } جمع نعم بالتحريك  ،  وقد يسكن عينه  ،  وهي  :  الابل  ،  والبقر  ،  والغنم او خاص بالابل ويجمع علي اناعيم كما نقل عن القاموس { لان العلم } المقرون بالعمل  ،  والاخلاص { حياه القلوب من } موت { الجهل  ،  ومصابيح الابصار } يعني نور الابصار وضياءها { من الظلم } لان كل خفي ينكشف بالعلم .
