ولقد ضرب عمر رضي الله عنه من اراد ادخال ادني شبهه علي الناس بسؤاله عن نحو قوله تعالي  :  { فاقبل بعضهم علي بعض يتساءلون } مع قوله تعالي  :  { فلا انساب بينهم يومئذ ولا يتساءلون } وعن قوله تعالي  :  { اليوم نختم علي افواههم وتكلمنا ايديهم وتشهد ارجلهم } مع قوله تعالي  :  { يوم تشهد عليهم السنتهم وايديهم وارجلهم } قوله تعالي  :  { هذا يوم لا ينطقون } ونفاه من المدينه لانه خشي من فتح هذا الباب ان يتطرق الناس الي اعتقاد نوع نقص في القران المنزه المكرم .
