ثم قال تبارك وتعالي  :  { فبما رحمه من الله لنت لهم ولو كنت فظا غليظ القلب لانفضوا من حولك } اي لتركوك { فاعف عنهم } اي فتجاوز عنهم { واستغفر لهم وشاورهم في الامر فاذا عزمت فتوكل علي الله ان الله يحب المتوكلين } فذكر لنبيه صلي الله عليه وسلم ص 305 لينه لهم وصبره عليهم لضعفهم وقله صبرهم علي الغلظه لو كانت منه عليهم في كل ما خالفوا عنه مما افترض عليهم من طاعه نبيهم صلي الله عليه وسلم . ثم قال تبارك وتعالي  :  { فاعف عنهم } اي تجاوز عنهم { واستغفر لهم } ذنوبهم من قارف من اهل الايمان منهم { وشاورهم في الامر } اي لتريهم انك تسمع منهم وتستعين بهم وان كنت غنيا عنهم تالفا لهم بذلك علي دينهم { فاذا عزمت } اي علي امر جاءك مني وامر من دينك في جهاد عدوك لا يصلحك ولا يصلحهم الا ذلك فامض علي ما امرت به علي خلاف من خالفك  ،  وموافقه من وافقك  ،  { فتوكل علي الله } اي ارض به من العباد { ان الله يحب المتوكلين ان ينصركم الله فلا غالب لكم وان يخذلكم فمن ذا الذي ينصركم من بعده } اي لئلا تترك امري للناس وارفض امر الناس الي امري  ،  { وعلي الله } لا علي الناس { فليتوكل المؤمنون } ص 305 وذكر قوله { وشاورهم في الامر } وفسره وقد جاء عن ابن عباس انه قال نزلت في ابي بكر وعمر امر بمشاورتهما . 3 / 304
