وهو الذي يعرف بدليل الخطاب مثل قوله عليه الصلاه والسلام { في سائمه الغنم الزكاه } فان قوما فهموا منه ان لا زكاه في غير السائمه  ،  او نشات مما يعرض لتلك الالفاظ من النسخ اي جوازه  ،  وكونه ينقسم الي اقسام احدها نسخ الكتاب بالكتاب كحكم { والذين يتوفون منكم ويذرون ازواجا وصيه لازواجهم متاعا الي الحول غير اخراج } بحكم { والذين يتوفون منكم ويذرون ازواجا يتربصن بانفسهن اربعه اشهر وعشرا } لتاخرها نزولا وان تقدمت تلاوه  ،  وثانيها نسخ السنه بالسنه كحديث { كنت نهيتكم عن زياره القبور فزوروها } .
