الضَّابِطِ ) أَيْ : قَوْلِ الْمُصَنِّفِ إذَا اتَّفَقَا إلَخْ أَيْ : عَلَى مَنْعِهِ ( قَوْلُهُ : إذْ لَمْ يَتَوَارَدَا ) أَيْ : الِادِّعَاءَانِ ( قَوْلُهُ مَعَ أَنَّهُمَا اتَّفَقَا إلَخْ ) أَيْ : فَيَشْمَلُهُ الضَّابِطُ ، وَلَيْسَ مِنْ أَفْرَادِهِ ( قَوْلُهُ : فَيَحْلِفُ كُلٌّ إلَخْ ) تَفْرِيعٌ عَلَى قَوْلِهِ فَلَا تَحَالُفَ ( قَوْلُهُ : اُدُّعِيَ عَلَيْهِ ) بِبِنَاءِ الْمَفْعُولِ ( قَوْلُهُ : عَلَى الْأَصْلِ ) أَيْ أَصَالَةِ النَّفْيِ .
