أَلَا تَرَى إلَى قَوْله تَعَالَى : { ثُمَّ اسْتَوَى عَلَى الْعَرْشِ } مَعَ قَوْلِهِ : { وَنَحْنُ أَقْرَبُ إلَيْهِ مِنْ حَبْلِ الْوَرِيدِ } { وَهُوَ مَعَكُمْ أَيْنَمَا كُنْتُمْ } وَمَعَ خَبَرِ : { لَوْ أَدْلَيْتُمْ حَبْلًا لَوَقَعَ عَلَى اللَّهِ } فَأَحَدُ تِلْكَ النُّصُوصِ يَجِبُ تَأْوِيلُهُ ؛ إذْ لَا يُمَكَّنُ أَحَدٌ أَنْ يَقُولَ بِظَوَاهِرِ تِلْكَ النُّصُوصِ جَمِيعِهَا ، وَإِذَا وَجَبَ تَأْوِيلُ بَعْضِهَا وَجَبَ تَأْوِيلُ كُلِّهَا .
