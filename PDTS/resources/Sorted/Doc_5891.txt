( وَيُسَنُّ لِمُرِيدِهِ ) أَيْ : الْإِحْرَامِ ( أَنْ يَغْتَسِلَ ذَكَرًا كَانَ أَوْ أُنْثَى وَلَوْ حَائِضًا وَنُفَسَاءَ ) لِأَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { أَمَرَ أَسْمَاءَ بِنْتَ عُمَيْسٍ وَهِيَ نُفَسَاءُ أَنْ تَغْتَسِلَ } رَوَاهُ مُسْلِمٌ { وَأَمَرَ عَائِشَةَ أَنْ تَغْتَسِلَ لِإِهْلَالِ الْحَجِّ وَهِيَ حَائِضٌ } ( فَإِنْ رَجَتَا ) أَيْ : الْحَائِضُ وَالنُّفَسَاءُ ( الطُّهْرَ قَبْلَ الْخُرُوجِ مِنْ الْمِيقَاتِ اُسْتُحِبَّ ) لَهُمَا ( تَأْخِيرُ ) الْغُسْلِ ( حَتَّى تَطْهُرَا ) لِيَكُونَ أَكْمَلَ لَهُمَا .
