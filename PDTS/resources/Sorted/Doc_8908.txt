( وَإِنْ ضَمِنَ الْمُؤَجَّلَ حَالًا صَحَّ ) الضَّمَانُ وَلَمْ يَصِرْ حَالًا ( وَلَمْ يَلْزَمْهُ ) أَيْ : الضَّامِنَ ( قَبْلَ أَجَلِهِ ) ؛ لِأَنَّ الضَّامِنَ فَرْعُ الْمَضْمُونِ عَنْهُ فَلَا يَسْتَحِقُّ مُطَالَبَتُهُ دُونَ أَصْلِهِ وَالْفَرْقُ بَيْنَهُمَا وَبَيْنَ الَّتِي قَبْلَهَا : أَنَّ الْحَالَ ثَابِتٌ مُسْتَحِقٌّ الْقَضَاءَ فِي جَمِيعِ الزَّمَانِ فَإِذَا ضَمِنَهُ مُؤَجَّلًا فَقَدْ الْتَزَمَ بَعْضَ مَا يَجِبُ عَلَى الْمَضْمُونِ عَنْهُ فَصَحَّ ، كَمَا لَوْ كَانَ الدَّيْنُ عَشَرَةً فَضَمِنَ خَمْسَةً وَأَمَّا الْمُؤَجَّلُ فَلَا يُسْتَحَقُّ قَضَاؤُهُ إلَّا عِنْدَ أَجَلِهِ فَإِذَا ضَمِنَهُ حَالًا الْتَزَمَ مَا لَمْ يَجِبْ كَمَا لَوْ كَانَ الدَّيْنُ عَشَرَةً فَضَمِنَ عِشْرِينَ .( 10 / 332 )
