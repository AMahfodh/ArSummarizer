( قَوْلُهُ : لَا عَلَى مَنْ فِي الصَّلَاةِ ) وَيُسَنُّ لَهُ الرَّدُّ بِالْإِشَارَةِ وَلَا يَجِبُ الرَّدُّ بَعْدَ السَّلَامِ حَجَرٌ ( قَوْلُهُ : وَالثَّانِي خُرُوجُهُ ) وَإِنْ الْمُغْتَسِلُ فِي غَيْرِ الْحَمَّامِ لَا يَجِبُ الرَّدُّ عَلَيْهِ وَسَيُعْلَمُ مِنْ قَوْلِهِ الْآتِي وَمُغْتَسِلٍ ( قَوْلُهُ : وَهُوَ الظَّاهِرُ ) وَحَيْثُ لَا يَجِبُ عَلَيْهِ يُسَنُّ لَهُ بِاللَّفْظِ وَكَذَلِكَ الْآكِلُ لَا يَجِبُ عَلَيْهِ حَيْثُ كَانَتْ اللُّقْمَةُ فِي فَمِهِ ؛ لِأَنَّهُ لَا يُسَنُّ السَّلَامُ عَلَيْهِ وَيُسَنُّ لَهُ الرَّدُّ حَجَرٌ قَوْلُهُ : وَمُقِيمٍ ) وَيُسَنُّ لَهُ الرَّدُّ بِالْإِشَارَةِ وَبَعْدَهُ بِاللَّفْظِ حَجَرٌ ( قَوْلُهُ : وَمُلَبٍّ ) وَيُسَنُّ لَهُ الرَّدُّ وَلَا يَجِبُ كَمَا نَصَّ عَلَيْهِ وَيُحْمَلُ قَوْلُ الرَّوْضَةِ وَأَصْلِهَا فِي الْحَجِّ عَنْ النَّصِّ لَوْ سَلَّمَ عَلَيْهِ رَدَّ أَيْ : نَدْبًا وَمَرَّ أَنَّهُ يَجِبُ الرَّدُّ عَلَى مُسْتَمِعِ الْخُطْبَةِ وَإِنْ كُرِهَ عَلَيْهِ حَجَرٌ( 18 / 426 )
