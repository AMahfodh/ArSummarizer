( وَإِنْ أَقَرَّ الْمُرْتَهِنُ بِالْإِذْنِ ) فِي الْوَطْءِ ( وَأَنْكَرَ ) الْمُرْتَهِنُ ( كَوْنَ الْوَلَدِ مِنْ الْوَطْءِ الْمَأْذُونِ فِيهِ ) وَقَالَ : هُوَ مِنْ وَطْءٍ آخَرَ ( أَوْ قَالَ ) هُوَ أَيْ : الْوَلَدُ ( مِنْ زَوْجٍ أَوْ زِنًا فَقَوْلُ الرَّاهِنِ بِغَيْرِ يَمِينٍ ) ؛ لِأَنَّا لَمْ نُلْحِقْهُ بِهِ بِدَعْوَاهُ ، بَلْ بِالشَّرْعِ ( إنْ اعْتَرَفَ الْمُرْتَهِنُ بِالْإِذْنِ فِي الْوَطْءِ ، وَ ) اعْتَرَفَ ( بِالْوَطْءِ وَ ) اعْتَرَفَ ( بِالْوِلَادَةِ ، وَ ) اعْتَرَفَ ( بِمُضِيِّ مُدَّةٍ بَعْدَ الْوَطْءِ يُمْكِنُ أَنْ تَلِدَهُ فِيهَا ) فَإِنْ عَاشَ اُعْتُبِرَ مُضِيُّ سِتَّةِ أَشْهُرٍ مِنْ وَطْئِهِ ؛ لِأَنَّهَا أَقَلُّ مُدَّةِ الْحِمْلِ .( 10 / 134 )
