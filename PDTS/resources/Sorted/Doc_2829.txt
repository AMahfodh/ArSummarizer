( فَوَائِدُ ) : قَالَ ابْنُ عَقِيلٍ : ( كُرِهَ تَسْمِيَةُ مَنْ لَمْ يَحُجَّ صَرُورَةً ) لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ : { لَا صَرُورَةَ فِي الْإِسْلَامِ } وَ ( لِأَنَّهُ اسْمٌ جَاهِلِيٌّ ، وَ ) يُكْرَهُ ( قَوْلُ : حَجَّةِ الْوَدَاعِ ، لِأَنَّهُ اسْمٌ عَلَى أَنْ لَا يَعُودَ وَقَوْلُ : شَوْطٌ ، بَلْ ) يُقَالُ : ( طَوْفَةٌ وَطَوْفَتَانِ .
