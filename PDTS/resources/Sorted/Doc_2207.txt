وَهَذَا أَوْلَى مِمَّا قِيلَ ؛ لِأَنَّهُ أَرَادَ مُسَارَعَةَ بَيَانِ الْحَقِّ ( { أَمَا } ) بِفَتْحِ الْهَمْزَةِ وَتَخْفِيفِ الْمِيمِ حَرْفُ تَنْبِيهٍ وَأَكْثَرُ مَا يَقَعُ بَعْدَهُ الْقَسَمُ ( { وَاَللَّهِ إنِّي لَأَخْشَاكُمْ } ) أَكْثَرُكُمْ خَشْيَةً ( { لِلَّهِ تَعَالَى } ) وَالْخَشْيَةُ خَوْفٌ مَعَ هَيْبَةٍ وَإِجْلَالٍ وَمُتَابَعَةٍ لِلْعِلْمِ وَكُلَّمَا ازْدَادَ الْعِلْمُ إلَى ذَاتِهِ( 1 / 305 )
