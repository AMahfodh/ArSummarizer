( وَيَلْزَمُ ) بِإِخْرَاجٍ ( عَنْ ) مَالِ ( صَغِيرٍ وَمَجْنُونٍ وَلِيَّهُمَا ) ، فَيُخْرِجُ ( مِنْ مَالِهِمَا ) لِأَنَّهَا حَقٌّ وَاجِبٌ عَلَيْهِمَا ، فَوَجَبَ عَلَى الْوَلِيِّ أَدَاؤُهَا عَنْهُمَا ( بِنِيَّةٍ مِنْهُ ) ، أَيْ : الْوَلِيِّ ( كَنَفَقَةِ قَرِيبٍ ) لَهُمَا ، ( وَ ) نَفَقَةِ ( زَوْجَةٍ وَأَرْشِ جِنَايَةٍ لَهُمَا ) .( 5 / 235 )
