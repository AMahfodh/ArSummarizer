( قَوْلُهُ : إذْ هُوَ ) أَيْ تَعَمُّدُ قَتْلِ الصَّيْدِ فِي الْحَرَمِ ( قَوْلُهُ : أَوْ تَابَ ) عَطْفٌ عَلَى قَوْلِهِ قَبْلُ إنْ لَمْ يَفْسُقْ سم ( قَوْلُهُ : إذْ الظَّاهِرُ أَنَّهُ لَا يُشْتَرَطُ هُنَا اسْتِبْرَاءٌ إلَخْ ) أَيْ فَيَحْكُمَانِ بِهِ حَالًا وَلَا يَتَوَقَّفُ عَلَى اسْتِبْرَاءٍ ع ش ( قَوْلُهُ : كَانَ مِثْلِيًّا ) أَيْ ؛ لِأَنَّ مَعَهُمَا زِيَادَةَ عِلْمٍ بِمَعْرِفَةِ دَقِيقِ الشَّبَهِ ( وَقَوْلُهُ : تَخَيَّرَ ) أَيْ كَمَا فِي اخْتِلَافِ الْمُفْتِيَيْنِ نِهَايَةٌ وَمُغْنِي أَيْ الْمُجْتَهِدِينَ أَمَّا غَيْرُهُمَا فَيَنْبَغِي أَنَّ مَنْ غَلَبَ عَلَى ظَنِّهِ صِدْقُهُ فِي إصَابَةِ الْمَنْقُولِ أَخَذَ بِقَوْلِهِ ، وَإِلَّا لَمْ يَأْخُذْ بِقَوْلِ وَاحِدٍ مِنْهُمَا لِلتَّعَارُضِ ع ش ( قَوْلُهُ : وَنَحْوُهُ إلَخْ ) أَيْ كَالْفَوَاخِتِ وَالْيَمَامِ وَالْقُمْرِيِّ وَكُلِّ ذِي طَوْقٍ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : عَبَّ ) أَيْ شَرِبَ الْمَاءَ بِلَا مَصٍّ ( وَهَدَرَ ) أَيْ رَجَعَ صَوْتُهُ وَغَرَّدَ مُغْنِي عِبَارَةُ بَاعَشَنٍ أَيْ شَرِبَ الْمَاءَ جَرْعًا بِلَا مَصٍّ وَلَا تَنَفُّسٍ بِخِلَافِ غَيْرِ الْحَمَامِ فَيَشْرَبُهُ قَطْرَةً بَعْدَ قَطْرَةٍ جَرْعًا بَعْدَ جَرْعٍ وَهَدَرَ أَيْ رَجَعَ صَوْتُهُ وَبَعْضُهُمْ اقْتَصَرَ عَلَى الْعَبِّ وَهُوَ كَافٍ .
