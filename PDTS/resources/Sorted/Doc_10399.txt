( قُطِعَتْ يَدُهُ عَمْدًا فَارْتَدَّ وَالْعِيَاذُ بِاَللَّهِ وَمَاتَ مِنْهُ أَوْ لَحِقَ ) فَحَكَمَ بِهِ ( فَجَاءَ مُسْلِمًا فَمَاتَ مِنْهُ ضَمِنَ الْقَاطِعُ نِصْفَ الدِّيَةِ فِي مَالِهِ لِوَارِثِهِ ) فِي الْمَسْأَلَتَيْنِ لِأَنَّ السِّرَايَةَ حَلَّتْ مَحَلًّا غَيْرَ مَعْصُومٍ فَأُهْدِرَتْ ، قَيَّدَ بِالْعَمْدِ لِأَنَّهُ فِي الْخَطَأِ عَلَى الْعَاقِلَةِ ( وَ ) قَيَّدْنَا بِالْحُكْمِ بِلَحَاقِهِ لِأَنَّهُ ( إنْ ) عَادَ قَبْلَهُ أَوْ ( أَسْلَمَ هَاهُنَا ) وَلَمْ يَلْحَقْ ( فَمَاتَ مِنْهُ ) بِالسِّرَايَةِ ( ضَمِنَ ) الدِّيَةَ ( كُلَّهَا ) لِكَوْنِهِ مَعْصُومًا وَقْتَ السِّرَايَةِ أَيْضًا ارْتَدَّ الْقَاطِعُ فَقُتِلَ أَوْ مَاتَ ثُمَّ سَرَى إلَى النَّفْسِ فَهَدَرٌ لَوْ عَمَدَ الْفَوَاتَ مَحَلَّ الْقَوَدِ ، وَلَوْ خَطَأً فَالدِّيَةُ عَلَى الْعَاقِلَةِ فِي ثَلَاثِ سِنِينَ مِنْ يَوْمِ الْقَضَاءِ عَلَيْهِمْ خَانِيَّةٌ .
