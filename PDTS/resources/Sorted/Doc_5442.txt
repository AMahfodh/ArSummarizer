( قَوْلُهُ : لَمْ يُحَدَّ ) أَيْ وَيَجِبُ عَلَيْهِ التَّقَايُؤُ ( قَوْلُهُ : إنْ ادَّعَاهُ ) أَيْ الْجَهْلَ ( قَوْلُهُ : حَيْثُ بَيَّنَهُ ) ظَاهِرُهُ وَإِنْ لَمْ يَثْبُتْ ذَلِكَ وَلَا وُجِدَتْ قَرِينَةٌ تَدُلُّ عَلَيْهِ ( قَوْلُهُ : آخَرِ إنَائِهَا ) أَيْ أَسْفَلِهِ ( قَوْلُهُ وَلَمْ يَبْقَ إلَّا أَثَرُهَا ) أَيْ وَالْحَالُ لَمْ يَبْقَ إلَخْ ( قَوْلُهُ : وَمَا فِيهِ بَعْضُهَا ) الظَّاهِرُ أَنَّ الْمَاءَ مِثَالٌ فَمِثْلُهُ سَائِرُ الْمَائِعَاتِ( 26 / 219 )
