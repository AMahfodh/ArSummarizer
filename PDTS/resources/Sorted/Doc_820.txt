وَإِنْ ( اتَّهَمَهَا ) أَيْ زَوْجَتَهُ ( بِسَرِقَةٍ ، فَقَالَ ) لَهَا ( إنْ لَمْ تَصْدُقِينِي ) فِي أَمْرِ هَذِهِ السَّرِقَةِ ( فَأَنْتِ طَالِقٌ ، فَقَالَتْ ) لَهُ قَوْلَيْنِ : أَحَدُهُمَا ، ( سَرَقْت ) وَالْآخَرُ ( مَا سَرَقْت لَمْ تَطْلُقْ ) لِأَنَّهَا صَادِقَةٌ فِي أَحَدِ الْقَوْلَيْنِ .( 14 / 15 )
