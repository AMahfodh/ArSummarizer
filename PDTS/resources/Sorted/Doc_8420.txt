مِنْهُ مُدَّانِ مَثَلًا مِنْ الْأَقِطِ كَمَا قِيلَ بِمِثْلِهِ فِي زَكَاةِ الْفِطْرِ إذَا كَانُوا يَقْتَاتُونَ اللَّبَنَ أَنَّ الْوَاجِبَ مِنْ اللَّبَنِ مَا يَتَحَصَّلُ مِنْهُ صَاعٌ مِنْ الْأَقِطِ ( قَوْلُهُ وَيَكُونُ ) أَيْ الْمَاءُ ( قَوْلُهُ : لَا تَمْلِيكًا ) وَلَعَلَّ الْفَرْقَ بَيْنَهُ وَبَيْنَ الْمَأْكُولِ تَفَاهَتُهُ ( قَوْلُهُ : أَنَّهُ تَمْلِيكٌ ) أَيْ الْمَاءُ ( قَوْلُهُ : وَهُوَ الْمُعْتَمَدُ ) وَعَلَيْهِ فَيَنْبَغِي أَنْ يُمَلِّكَهَا مَا يَكْفِيهَا غَالِبًا ( قَوْلُهُ : فَتَكْفِي عَنْ الْأُدُمِ ) أَيْ إنْ اُعْتِيدَ الِاكْتِفَاءُ بِهَا عَنْ الْأُدُمِ ( قَوْلُهُ : نَعَمْ يُتَّجَهُ كَمَا بَحَثَهُ الْأَذْرَعِيُّ الرُّجُوعُ فِيهِ لِلْعُرْفِ ) .
