الْإِذْنِ ( قَوْلُهُ : وَإِنْ رَضِيَ بِاسْتِدَامَتِهَا ) أَيْ أَوْ قَالَ أَبْرَأْتُك عَنْ ضَمَانِ مَا يَتْلَفُ فِيهَا ( قَوْلُهُ : فَهَلْ يَكُونُ كَالرِّضَا ) أَشَارَ إلَى تَصْحِيحِهِ قَوْلُهُ : أَوْجُهُهُمَا الْأَوَّلُ إلَخْ ) وَجَزَمَ بِهِ صَاحِبُ الْأَنْوَارِ وَغَيْرُهُ وَرَجَّحَهُ الْبُلْقِينِيُّ وَغَيْرُهُ ( قَوْلُهُ : وَلَمْ تَزِدْ مَشَقَّتُهُ ) بِأَنْ تَكُونَ مَسَافَتُهُ كَمَسَافَةِ أَرْضِ الْمَالِكِ أَوْ أَقَلَّ وَقَوْلُهُ : عَلَى الْأَوْجَهِ أَشَارَ إلَى تَصْحِيحِهِ .( 11 / 262 )
