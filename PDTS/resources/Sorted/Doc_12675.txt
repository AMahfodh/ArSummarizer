وَكَذَا لَوْ كَانَ مَعَهُ أَرْبَعُ نِسْوَةٍ وَقَالَ لِأَمَتِهِ مَا يَأْتِي فَلَا يَكُونُ نِكَاحًا لِأَنَّهُ حِينَئِذٍ لَا يَحِلُّ لَهُ نِكَاحُهَا لِأَنَّهَا خَامِسَةٌ وَقَوْلُهُمْ : لَوْ كَانَتْ حُرَّةً لِدَفْعِ اعْتِبَارِ عَدَمِ الطَّوْلِ وَخَوْفِ الْعَنَتِ الْمُعْتَبَرِ فِي نِكَاحِ الْأَمَةِ مَعَ مَا تَقَدَّمَ ( أَعْتَقْتُكِ وَجَعَلْتُ عِتْقَك صَدَاقَك أَوْ ) قَالَ ( جَعَلْتُ عِتْقَ أَمَتِي صَدَاقَهَا أَوْ ) قَالَ ( صَدَاقُ أَمَتِي عِتْقُهَا أَوْ ) قَالَ : ( قَدْ أَعْتَقْتُهَا وَجَعَلْتُ عِتْقَهَا صَدَاقَهَا أَوْ ) أَعْتَقْتُهَا عَلَى أَنَّ عِتْقَهَا صَدَاقُهَا أَوْ قَالَ ( أَعْتَقْتُكِ عَلَى أَنْ أَتَزَوَّجَكِ وَعِتْقك صَدَاقُكِ ) أَوْ قَالَ أَعْتَقْتُكِ عَلَى أَنْ أَتَزَوَّجَكِ وَعِتْقِي صَدَاقُكِ ( صَحَّ ) الْعِتْقُ وَالنِّكَاحُ فِي هَذِهِ الصُّوَرِ كُلِّهَا وَإِنْ لَمْ يَقُلْ : وَتَزَوَّجْتُكِ أَوْ وَتَزَوَّجْتُهَا ، لِأَنَّ قَوْلَهُ : وَجَعَلْتُ عِتْقَهَا صَدَاقَهَا وَنَحْوَهُ يَتَضَمَّنُ ذَلِكَ وَالْأَصْلُ فِي ذَلِكَ مَا رَوَى أَنَسٌ { أَنَّ النَّبِيَّ أَعْتَقَ صَفِيَّةَ وَجَعَلَ عِتْقَهَا صَدَاقَهَا } رَوَاهُ أَحْمَدُ وَالنَّسَائِيُّ وَأَبُو دَاوُد وَالتِّرْمِذِيُّ وَصَحَّحَهُ .
