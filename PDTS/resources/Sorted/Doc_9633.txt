( وَلَوْ سَامَتْ ) الْمَاشِيَةُ ( بِنَفْسِهَا ) فَلَا زَكَاةَ بِنَاءً عَلَى الْأَصَحِّ أَنَّهُ يُشْتَرَطُ قَصْدُ السَّوْمِ ( أَوْ اعْتَلَفَتْ السَّائِمَةُ ) بِنَفْسِهَا الْقَدْرَ الْمُؤَثِّرَ فَلَا زَكَاةَ أَيْضًا لِحُصُولِ الْمُؤْنَةِ ، وَقَصْدُ الْعَلَفِ غَيْرُ شَرْطٍ لِرُجُوعِهِ إلَى الْأَصْلِ ، وَهُوَ عَدَمُ الْوُجُوبِ ( أَوْ كَانَتْ عَوَامِلَ ) لِلْمَالِكِ وَلَوْ فِي مُحَرَّمٍ أَوْ بِأُجْرَةٍ أَوْ لِغَاصِبٍ ( فِي حَرْثٍ وَنَضْحٍ ) ، وَهُوَ مَحَلُّ الْمَاءِ الْمُعَدِّ لِلشُّرْبِ ( وَنَحْوِهِ ) كَحَمْلٍ ( فَلَا زَكَاةَ فِي الْأَصَحِّ ) ؛ لِأَنَّهَا مُعَدَّةٌ لِاسْتِعْمَالٍ مُبَاحٍ فَأَشْبَهَتْ ثِيَابَ الْبَدَنِ وَصَحَّ { لَيْسَ فِي الْبَقَرِ الْعَوَامِلِ شَيْءٌ } ، وَفِي رِوَايَةٍ { لَيْسَ عَلَى الْعَوَامِلِ شَيْءٌ } .
