( قَوْلُهُ : أَوْ أَصْفَرُ ) قَوِيٌّ أَوْ رَقِيقٌ وَيُقْبَلُ مَا رَقَّ لِحَرٍّ لَا لِعَيْبٍ ا ه حَجّ ( قَوْلُهُ : لِأَنَّ الْأَوَّلَ ) أَيْ الْجَبَلِيَّ ( قَوْلُهُ : وَفِيهِ بُعْدٌ ) أَيْ فَلَوْ اتَّفَقَ وُجُودُ ذَلِكَ فِي بَلَدٍ اُشْتُرِطَ وَإِلَّا فَلَا ( قَوْلُهُ : إذْ كُلُّ شَيْءٍ يُحْفَظُ بِهِ ) أَيْ مِنْ خَوَاصِّهِ أَنَّهُ إذَا طُرِحَ فِيهِ شَيْءٌ وَتُرِكَ الْمَطْرُوحُ فِيهِ بِحَالِهِ لَا يَتَغَيَّرُ .( 13 / 276 )
