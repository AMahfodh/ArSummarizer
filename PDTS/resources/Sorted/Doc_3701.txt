بِحَذْفٍ ( قَوْلُهُ : فِي الْأُولَى ) أَيْ الْمُعَيَّنَةِ ابْتِدَاءً ( قَوْلُهُ : سَبَقَهُ ) أَيْ الرَّافِعِيُّ وَقَوْلُهُ : إلَيْهِ أَيْ الْبَحْثِ ( قَوْلُهُ فِي رَدِّهِ ) أَيْ الْمَاوَرْدِيِّ ( قَوْلُهُ : بَلْ هِيَ ) أَيْ الْأُولَى أَوْلَى أَيْ بِالِامْتِنَاعِ ( قَوْلُهُ مِنْ نَذْرِ الْمُجَازَاةِ ) أَيْ نَذْرِ التَّبَرُّرِ الْمُعَلَّقِ كَإِنْ شُفِيَ مَرِيضِي فَلِلَّهِ عَلَيَّ أَنْ أُضَحِّيَ بِهَذِهِ الشَّاةِ أَوْ بِشَاةٍ ا ه .
