( وَمَنْ قُلِعَتْ سِنُّهُ فَاقْتَصَّ مِنْ قَالِعِهَا ثُمَّ نَبَتَتْ ) أَيْ نَبَتَ مَكَانَهَا أُخْرَى ( فَعَلَيْهِ دِيَةُ سِنِّ الْمُقْتَصِّ مِنْهُ ) لِأَنَّهُ تَبَيَّنَ أَنَّهُ اسْتَوْفَى بِغَيْرِ حَقٍّ لِأَنَّ الْمُوجِبَ فَسَادُ الْمَنْبَتِ وَلَمْ يَفْسُدْ حَيْثُ نَبَتَ مَكَانَهَا أُخْرَى فَانْعَدَمَتْ الْجِنَايَةُ ( وَيُسْتَأْنَى فِي اقْتِصَاصِ السِّنِّ ، وَ ) اقْتِصَاصِ ( الْمُوضِحَةِ حَوْلًا ) الِاسْتِنَانُ الِانْتِظَارُ كَمَا فِي الْمُغْرِبِ .( 9 / 22 )
