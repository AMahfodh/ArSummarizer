( وَكَذَا ) تَقَعُ الْمُقَاصَّةُ بِهِ ( إنْ كَانَ لَهُ ) أَيْ لِلْمُشْتَرِي ( عَلَى الْوَكِيلِ دَيْنٌ ) عِنْدَ الطَّرَفَيْنِ لِكَوْنِهِ يَمْلِكُ الْإِبْرَاءَ عَنْهُ عِنْدَهُمَا ( خِلَافًا لِأَبِي يُوسُفَ ) ؛ لِأَنَّ عِنْدَهُ لَا يَجُوزُ الْإِبْرَاءُ وَلَا تَقَعُ الْمُقَاصَّةُ ( وَ ) لَكِنَّهُ ( يَضْمَنُهُ الْوَكِيلُ لِلْمُوَكِّلِ ) فِي فَصْلِ الْمُقَاصَّةِ عِنْدَهُمَا كَمَا يَضْمَنُهُ فِي فَصْلِ الْإِبْرَاءِ .
