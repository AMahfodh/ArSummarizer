( وَلَهُ ) أَيْ : الذَّكَرِ الْمُعَصِّبِ ( مِثْلَا مَا لِلْأُنْثَى ) مِنْ الْمُعَصَّبَاتِ بِهِ ( وَلَا يُعَصِّبُ ) ابْنُ ابْنٍ فَأَكْثَرُ وَإِنْ نَزَلَ ( ذَاتَ فَرْضٍ أَعْلَى مِنْهُ ) ؛ لِأَنَّ فِيهِ إضْرَارًا بِذَاتِ الْفَرْضِ بَلْ لَهُ مَا فَضُلَ ( وَلَا ) يُعَصِّبُ ( مَنْ هِيَ أَنْزَلُ مِنْهُ ) بَلْ يَحْجُبُهَا لِئَلَّا تُشَارِكَهُ ، وَالْأَبْعَدُ لَا يُشَارِكُ الْأَقْرَبَ .
