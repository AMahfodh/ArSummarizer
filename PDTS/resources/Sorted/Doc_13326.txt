( أَوْ ) جَمَعَ مَعَ بَيْعٍ ( صَرْفًا ) بِعِوَضٍ وَاحِدٍ بِأَنْ بَاعَهُ عَبْدًا وَصَارَفَهُ مِائَةَ دِرْهَمٍ بِمِائَةِ دِينَارٍ قَالَ الشَّيْخُ التَّقِيُّ فِي شَرْحِ الْمُحَرَّرِ وَلَا بُدَّ أَنْ يَكُونَ الثَّمَنُ مِنْ غَيْرِ جِنْسٍ مَا مَعَ الْمَبِيعِ مِثْلَ أَنْ يَبِيعَهُ ثَوْبًا وَدَرَاهِمَ بِذَهَبٍ فَإِنْ كَانَ مِنْ جِنْسِهِ فَهِيَ مَسْأَلَةُ مُدِّ عَجْوَةٍ ( أَوْ ) جَمَعَ مَعَ بَيْعٍ ( خُلْعًا ) بِعِوَضٍ وَاحِدٍ بِأَنْ قَالَتْ ابْتَعْتُ مِنْكَ عَبْدَكَ وَاخْتَلَعْتُ نَفْسِي بِمِائَةِ دِرْهَمٍ ( أَوْ ) جَمَعَ مَعَ بَيْعٍ ( نِكَاحًا بِعِوَضٍ وَاحِدٍ ) كَبِعْتُكَ عَبْدِي وَزَوَّجْتُكَ أَمَتِي بِأَلْفٍ ( صَحَّ ) الْبَيْعُ وَمَا مَعَهُ ( فِيهِنَّ ) أَيْ فِي الْمَسَائِلِ الْمَذْكُورَةِ ؛ لِأَنَّ اخْتِلَافَ الْعَقْدَيْنِ لَا يَمْنَعُ الصِّحَّةَ ( وَيُقَسِّطُ الثَّمَنَ عَلَى قِيمَتِهِمَا ) أَيْ قِيمَةِ الْمَبِيعِ وَقِيمَةِ الْمَنْفَعَةِ وَهِيَ أُجْرَةُ الْمِثْلِ فِي الْإِجَارَةِ أَوْ قِيمَةُ الْمَبِيعِ وَالْمَصْرُوفِ فِي الصَّرْفِ ( وَمَهْرُ مِثْلٍ فِي خُلْعٍ وَنِكَاحٍ كَقِيمَةٍ ) فَيُوَزَّعُ الْعِوَضُ فِيهِمَا عَلَى قِيمَةِ الْمَبِيعِ وَمَهْرُ الْمِثْلِ وَمَتَى اُعْتُبِرَ قَبْضٌ لِأَحَدِهِمَا لَمْ يَبْطُلْ الْآخَرُ بِتَأَخُّرِهِ .( 9 / 4 )
