ع ش ( قَوْلُهُ : وَمِنْ ثَمَّ ) أَيْ : مِنْ أَجْلِ الْفَرْقِ ( قَوْلُهُ : لَمْ يَتَوَقَّفْ ) أَيْ : الْحَلِفُ ( قَوْلُهُ : مَعَهُ ) أَيْ : الْوَارِثِ ( قَوْلُهُ : وَسَكَتُوا ) أَيْ : الْغُرَمَاءُ ( قَوْلُهُ : فَإِنْ سَكَتَ ) أَيْ : الْوَارِثُ ، وَمِثْلُهُ الْغُرَمَاءُ فِيمَا يَظْهَرُ ، بَلْ يُمْكِنُ إرْجَاعُهُ لَهُمَا بِتَأْوِيلِ الْجَمِيعِ مَثَلًا ( قَوْلُهُ : فَيَقْضِي عَلَيْهِمَا بِلَا يَمِينٍ ) اعْتَمَدَ شَيْخُنَا الشِّهَابُ الرَّمْلِيُّ مَا صَحَّحَهُ الْبُلْقِينِيُّ أَنَّهُ لَا بُدَّ مِنْ الْيَمِينِ .( 43 / 244 )
