فَصْلٌ إذَا جَاءَ إلَى الْحَاكِمِ خَصْمَانِ سُنَّ أَنْ يُجْلِسَهُمَا بَيْنَ يَدَيْهِ ( إذَا جَاءَ إلَى الْحَاكِمِ خَصْمَانِ سُنَّ أَنْ يُجْلِسَهُمَا بَيْنَ يَدَيْهِ ) لِمَا رَوَى أَبُو دَاوُد أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { قَضَى أَنْ يَجْلِسَ الْخَصْمَانِ بَيْنَ يَدَيْ الْحَاكِمِ } وَلِقَوْلِ عُمَرَ : وَلَكِنْ أَجْلِسُ مَعَ خَصْمِي مَجْلِسًا بَيْنَ يَدَيْ زَيْدٍ وَقَالَ عَلِيٌّ حِينَ خَاصَمَ الْيَهُودِيَّ دِرْعَهُ إلَى شُرَيْحٍ لَوْ أَنَّ خَصْمِي مُسْلِمٌ لَجَلَسْتُ مَعَهُ بَيْنَ يَدِكَ .( 22 / 197 )
