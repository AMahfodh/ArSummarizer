( وَيَحْرُمُ أَنْ يُجْرِيَ ) شَخْصٌ ( فِي أَرْضِ غَيْرِهِ أَوْ ) فِي ( سَطْحِهِ ) أَيْ الْغَيْرِ ( مَاءً ) وَلَوْ تَضَرَّرَ بِتَرْكِهِ ( بِلَا إذْنِهِ ) أَيْ رَبِّ الْأَرْضِ أَوْ السَّطْحِ لِتَضَرُّرِهِ أَوْ تَضَرُّرِ أَرْضِهِ وَكَزَرْعِهَا ( وَيَصِحُّ صُلْحُهُ عَلَى ذَلِكَ ) أَيْ إجْرَاءِ مَائِهِ فِي أَرْضِ غَيْرِهِ أَوْ سَطْحِهِ ( بِعِوَضٍ ) ؛ لِأَنَّهُ إمَّا بَيْعٌ أَوْ إجَارَةٌ ( فَ ) إنْ صَالَحَهُ عَلَى إجْرَاءِ مَائِهِ فِي أَرْضِهِ أَوْ سَطْحِهِ ( مَعَ بَقَاءِ مِلْكِهِ ) أَيْ رَبِّ الْمَحَلِّ الَّذِي يَجْرِي فِيهِ الْمَاءُ بِأَنْ تَصَالَحَا عَلَى إجْرَائِهِ فِيهِ وَمِلْكُهُ بِحَالِهِ فَهُوَ ( إجَارَةٌ ) ؛ لِأَنَّ الْمَعْقُودَ عَلَيْهِ الْمَنْفَعَةُ .
