( وَ ) يُسَنُّ أَيْضًا ( أَخْذُهَا مِسْكًا ، إنْ لَمْ تَكُنْ مُحْرِمَةً فَتَجْعَلُهُ فِي فَرْجِهَا فِي قُطْنَةٍ أَوْ غَيْرِهَا ) كَخِرْقَةٍ ( بَعْدَ غُسْلِهَا لَيَقْطَعَ الرَّائِحَةَ ) أَيْ : رَائِحَةَ الْحَيْضِ أَوْ النِّفَاسِ ، { لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ لِأَسْمَاءِ لَمَّا سَأَلَتْهُ عَنْ غُسْلِ الْحَيْضِ : ثُمَّ تَأْخُذُ فِرْصَةً مُمَسَّكَةً فَتَطْهُرُ بِهَا } رَوَاهُ مُسْلِمٌ مِنْ حَدِيثِ عَائِشَةَ وَالْفِرْصَةُ الْقِطْعَةُ مِنْ كُلِّ شَيْءٍ ( فَإِنْ لَمْ تَجِدْ ) مِسْكًا ( فَطِيبًا ) لِقِيَامِهِ مَقَامَ الْمِسْكِ فِي( 1 / 425 )
