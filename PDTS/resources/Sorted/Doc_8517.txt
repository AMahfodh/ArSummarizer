( وَ ) نُدِبَ ( رَمَلُ ) رَجُلٍ ( مُحْرِمٍ ) بِحَجٍّ أَوْ عُمْرَةٍ ( مِنْ كَالتَّنْعِيمِ ) وَالْجِعْرَانَةِ ( أَوْ ) رَمَلٍ ( بِ ) طَوَافِ ( الْإِفَاضَةِ لِمُرَاهِقٍ ) وَنَحْوِهِ مِمَّنْ لَمْ يَطُفْ الْقُدُومَ لِضِيقِ الْوَقْتِ عَنْ فِعْلِهِ لِخَشْيَةِ فَوَاتِ وُقُوفِ عَرَفَةَ أَوْ لِنِسْيَانِهِ ، فَإِنْ كَانَ غَيْرَ مُرَاهِقٍ وَطَافَ الْقُدُومَ وَرَمَلَ فِيهِ أَوْ تَرَكَهُ وَلَوْ عَمْدًا فَلَا يَرْمُلُ بِالْإِفَاضَةِ ( لَا ) يُنْدَبُ الرَّمَلُ فِي طَوَافِ ( تَطَوُّعٍ وَ ) لَا فِي طَوَافِ ( وَدَاعٍ ) وَالظَّاهِرُ كَرَاهَتُهُ فِيهِمَا عَطْفُ خَاصٍّ عَلَى عَامٍّ .( 4 / 323 )
