( قَوْلُهُ : إذْ لَا تَبَرُّعَ بَعْدَ الْمَوْتِ ) أَقُولُ هَذَا لَا يَمْنَعُ احْتِمَالَ التَّبَرُّعِ عَنْ الْأَبِ ؛ لِأَنَّ الْإِقْرَارَ إخْبَارٌ عَنْ سَابِقٍ فَيُحْتَمَلُ أَنَّ الْأَبَ قَبْلَ مَوْتِهِ تَبَرَّعَ عَلَيْهِ بِأَلْفٍ مُعَيَّنٍ أَوْ شَائِعٍ عَلَى سَبِيلِ الْوَصِيَّةِ ( قَوْلُهُ : وَلَا يَتَعَلَّقُ بِالتَّرِكَةِ ) بِخِلَافِ مَا يَأْتِي فِي قَوْلِهِ : وَظَاهِرٌ إلَخْ وَالْفَارِقُ بَيْنَهُمَا تَقْدِيمُ قَوْلِهِ : لَهُ عَلَيَّ أَلْفٌ عَلَى قَوْلِهِ : فِي مِيرَاثِ أَبِي هُنَا فَقَدْ تَمَّتْ صِيغَةُ الْإِقْرَارِ فَيُلْغِي مَا عَقَّبَهُ بِهِ مِنْ قَوْلِهِ : فِي مِيرَاثِ أَبِي وَتَأَخُّرَهُ عَنْهُ فِيمَا يَأْتِي فَقَدْ قَيَّدَ الْإِقْرَارَ قَبْلَ تَمَامِهِ بِإِضَافَتِهِ إلَى الْمِيرَاثِ فَتَقَيَّدَ بِهِ ( قَوْلُهُ لَزِمَهُ أَلْفٌ فِي التَّرِكَةِ ) الَّذِي فِي عِبَارَةِ غَيْرِهِ لَزِمَهُ الْأَلْفُ وَلَمْ يَذْكُرُوا أَنَّهَا فِي التَّرِكَةِ وَهُوَ مَوْضِعٌ تَأَمَّلْ بِرّ ( قَوْلُهُ وَأَنَّ الشَّافِعِيَّ إنَّمَا إلَخْ ) هَذَا شُرُوعٌ فِي الْجَوَابِ عَنْ مَسْأَلَةِ الرَّهْنِ وَكَذَا قَوْلُهُ : أَيْضًا إلَخْ مُتَعَلِّقٌ بِهَا ( قَوْلُهُ لِكُلِّ مَا يُمْكِنُ إلَخْ ) فَيَشْمَلُ ذَلِكَ عِمَامَتَهُ وَسَتْرَ عَوْرَتِهِ وَغَيْرَ ذَلِكَ مِمَّا يَبْعُدُ كَوْنُهُ رَهْنًا بِرّ ( قَوْلُهُ : فَلَا أَثَرَ لِذَلِكَ هُنَا ) أَيْ لِأَنَّ الْمَيِّتَ خَرِبَتْ ذِمَّتُهُ فَلَا يُحْتَاجُ فِي ثُبُوتِ الدَّيْنِ عَلَى الْمُورَثِ إلَى تَصْرِيحِ الْمُقِرِّ بِأَنَّ الدَّيْنَ( 11 / 161 )
