( وَإِذَا أَدَّى ) الْمُكَاتَبُ ( الْعِوَضَ ) الَّذِي كُوتِبَ عَلَيْهِ جَمِيعَهُ ( وَعَتَقَ فَبَانَ الْعِوَضُ مَعِيبًا فَلَهُ ) أَيْ السَّيِّدِ ( أَرْشُهُ ) إنْ أَمْسَكَهُ ( أَوْ عَوَّضَهُ إنْ رَدَّهُ وَلَمْ يَبْطُلْ عِتْقُهُ ) لِأَنَّهُ إتْلَافٌ ، فَإِذَا وَقَعَ لَمْ يَرْتَفِعْ وَكَالْخُلْعِ ، وَإِطْلَاقُ عَقْدِ الْكِتَابَةِ يَقْتَضِي سَلَامَةَ الْعِوَضِ فِيهَا وَقَدْ تَعَذَّرَ رَدُّ الْمُكَاتَبِ رَقِيقًا فَوَجَبَ أَرْشُ الْعَيْبِ أَوْ عِوَضُ الْمَعِيبِ جَبْرًا لِمَا اقْتَضَاهُ إطْلَاقُ الْعَقْدِ .( 16 / 257 )
