تَنْبِيهٌ : لَوْ حَذَفَ الْمُصَنِّفُ قَوْلَهُ : بِالرُّخْصِ لَكَانَ أَوْلَى لِيَشْمَلَ مَا قَدَّرْته ، ( وَكَذَا لَوْ تَلِفَ بَعْضُهُ ) أَيْ مَالُ الْقِرَاضِ ( بِآفَةٍ ) سَمَاوِيَّةٍ كَحَرْقٍ وَغَرَقٍ ( أَوْ غَصْبٍ أَوْ سَرِقَةٍ ) وَتَعَذَّرَ أَخْذُهُ أَوْ أَخْذُ بَدَلِهِ ( بَعْدَ تَصَرُّفِ الْعَامِلِ ) فِيهِ بِالْبَيْعِ أَوْ الشِّرَاءِ مَحْسُوبٌ مِنْ الرِّبْحِ ( فِي الْأَصَحِّ ) قِيَاسًا عَلَى مَا مَرَّ ، وَالثَّانِي : لَا ؛ لِأَنَّهُ نَقْصٌ لَا تَعَلُّقَ لَهُ بِتَصَرُّفِ الْعَامِلِ وَتِجَارَتِهِ ، بِخِلَافِ الْحَاصِلِ بِالرُّخْصِ ، وَلَيْسَ نَاشِئًا مِنْ نَفْسِ الْمَالِ بِخِلَافِ الْمَرَضِ ، وَالْعَيْبِ ، ( وَإِنْ تَلِفَ ) بِمَا ذُكِرَ ( قَبْلَ تَصَرُّفِهِ ) فِيهِ بِبَيْعٍ أَوْ شِرَاءٍ ( فَمِنْ ) أَيٍّ ، فَيُحْسَبُ مَا تَلِفَ مِنْ ( رَأْسِ الْمَالِ ) لَا مِنْ الرِّبْحِ ( فِي الْأَصَحِّ ) ؛ لِأَنَّ الْعَقْدَ لَمْ يَتَأَكَّدْ بِالْعَمَلِ ، وَالثَّانِي : مِنْ الرِّبْحِ ؛ لِأَنَّهُ بِقَبْضِ الْعَامِلِ صَارَ مَالَ قِرَاضٍ .
