( قَوْلُهُ : فَالْقِيَاسُ بِنَاؤُهُ ) أَيْ بِأَنْ قُلْنَا لِلْمُشْتَرِي أَوْ مَوْقُوفٌ ( قَوْلُهُ : انْفَسَخَ ) وَيَضْمَنُهُ الْمُشْتَرِي بِالْبَدَلِ الشَّرْعِيِّ وَهُوَ الْمِثْلُ فِي الْمِثْلِيِّ وَالْقِيمَةُ فِي الْمُتَقَوِّمِ ( قَوْلُهُ : فَإِنْ قُلْنَا يَنْفَسِخُ ) بِأَنْ قُلْنَا الْمِلْكُ فِيهِ لِلْبَائِعِ ( قَوْلُهُ : أَوْ لَا يَنْفَسِخُ ) بِأَنْ كَانَ لِلْمُشْتَرِي أَوْ مَوْقُوفًا ( قَوْلُهُ : فَلَا أَثَرَ لِحُدُوثِهِ ) فَيَمْتَنِعُ الرَّدُّ ( قَوْلُهُ : وَالْأَوْجَهُ أَنَّ لَهُ حُكْمَ مَا قَبْلَ الْقَبْضِ ) فَيَثْبُتُ ، بِهِ الْخِيَارُ وَيُمْكِنُ شُمُولُ قَوْلِ الْمُصَنِّفِ قَبْلَ الْقَبْضِ لَهُ بِأَنْ يُرَادَ بِقَبْلِ الْقَبْضِ مَا قَبْلَ تَمَامِ الْقَبْضِ ( قَوْلُهُ : كَسَائِرِ الْعُيُوبِ ) أَيْ وَلَهُ الْأَرْشُ ، وَمَفْهُومُ قَوْلِهِ نَقَصَتْ أَنَّهَا لَوْ لَمْ تَنْقُصْ كَانَ لَهُ الرَّدُّ وَهُوَ ظَاهِرٌ ( قَوْلُهُ : لِلْمُشْتَرِي أَرْشُ الْمَرَضِ مِنْ الثَّمَنِ ) أَيْ فَيَكُونُ جُزْءًا مِنْهُ نِسْبَتُهُ إلَيْهِ كَنِسْبَةِ مَا نَقَصَ الْمَرَضُ مِنْ الْقِيمَةِ عَلَى مَا يَأْتِي فَفِي قَوْلِهِ وَهُوَ مَا بَيْنَ قِيمَتِهِ صَحِيحًا وَمَرِيضًا مُسَامَحَةً .( 12 / 150 )
