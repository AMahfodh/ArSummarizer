( وَإِذَا وُلِدَ السَّقْطُ لِأَكْثَرَ مِنْ أَرْبَعَةِ أَشْهُرٍ ) أَيْ : لِأَرْبَعَةِ أَشْهُرٍ فَأَكْثَرَ ( غُسِّلَ وَصُلِّيَ عَلَيْهِ ) نَصَّ عَلَيْهِ فِي رِوَايَةِ حَرْبٍ وَصَالِحٍ لِقَوْلِهِ: صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { وَالسَّقْطُ يُصَلَّى عَلَيْهِ وَيُدْعَى لِوَالِدَيْهِ بِالْمَغْفِرَةِ وَالرَّحْمَةِ } رَوَاهُ أَحْمَدُ وَأَبُو دَاوُد وَرَوَاهُ النَّسَائِيُّ وَالتِّرْمِذِيُّ وَصَحَّحَهُ وَلَفْظُهُمَا { : وَالطِّفْلُ يُصَلَّى عَلَيْهِ } وَاحْتَجَّ بِهِ أَحْمَدُ وَلِأَنَّهُ نَسَمَةٌ نُفِخَ فِيهَا الرُّوحُ .
