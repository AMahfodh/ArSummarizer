وَلَوْ تَزَوَّجَ كُلٌّ مِنْهُمَا أُمَّ الْآخَرِ فَهُمَا الْقَائِلَتَانِ : مُرْحَبًا بِابْنَيْنَا وَزَوْجَيْنَا وَوَلَدُ كُلِّ وَاحِدٍ مِنْهُمَا عَمُّ وَلَدِ الْآخَرِ ( وَأَوْلَى وَلَدِ كُلِّ أَبٍ أَقْرَبُهُمْ إلَيْهِ ) فَابْنُ عَمٍّ أَوْلَى مِنْ ابْنِ ابْنِ عَمٍّ ( حَتَّى فِي أُخْتٍ لِأَبٍ ) فَقَطْ أَوْ مَعَ أُمٍّ ( وَابْنِ أَخٍ ) وَلَوْ لِأَبَوَيْنِ ( مَعَ بِنْتٍ ) فَالْأُخْتُ هُنَا عَصَبَةٌ يَسْقُطُ بِهَا ابْنُ الْأَخِ ؛ لِأَنَّ الْعُصُوبَةَ جَعَلَتْهَا فِي مَعْنَى الْأَخِ لِأَبٍ ( فَإِنْ اسْتَوَوْا ) دَرَجَةً ( فَمَنْ لِأَبَوَيْنِ ) أَوْلَى مِمَّنْ لِأَبٍ حَتَّى فِي أُخْتٍ لِأَبٍ وَبِنْتٍ ، ؛ لِأَنَّ الْعُصُوبَةَ جَعَلَتْهَا فِي مَعْنَى الْأَخِ لِأَبَوَيْنِ( 7 / 471 )
