( وَمَنْ قَالَ : أَنَا أُؤَدِّي ) مَا عَلَيْهِ ، ( أَوْ ) أَنَا ( أُحْضِرُ ) مَا عَلَيْهِ ( أَوْ أَضْمَنُ ) مَا عَلَيْهِ ؛ ( لَمْ يَصِرْ ضَامِنًا ) بِذَلِكَ ؛ لِأَنَّهُ وَعْدٌ لَا يَجِبُ الْوَفَاءُ بِهِ ، بَلْ يُسَنُّ ، ( وَقَالَ الشَّيْخُ ) تَقِيُّ الدِّينِ : ( قِيَاسُ الْمَذْهَبِ يَصِحُّ ) الضَّمَانُ ( بِكُلِّ لَفْظٍ فُهِمَ مِنْهُ الضَّمَانُ عُرْفًا ) ؛ كَقَوْلِهِ : ( زَوِّجْهُ وَأَنَا أُؤَدِّي الصَّدَاقَ ، أَوْ ) قَوْلُهُ : ( بِعْهُ وَأَنَا أُعْطِيكَ الثَّمَنَ أَوْ ) قَوْلُهُ : ( اُتْرُكْهُ أَوْ لَا تُطَالِبْهُ وَأَنَا أُعْطِيكَ ) مَا عَلَيْهِ وَنَحْوُ ذَلِكَ مِمَّا يُؤَدِّي هَذَا الْمَعْنَى ؛ لِأَنَّ الشَّارِعَ لَمْ يَحُدَّ ذَلِكَ بِحَدٍّ ، فَرُجِعَ فِيهِ إلَى الْعُرْفِ ؛ كَالْحِرْزِ وَالْقَبْضِ .( 8 / 431 )
