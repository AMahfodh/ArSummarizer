( قَوْلُهُ : وَأَخَّرَ هَدْيَهُ ) أَيْ نَدْبًا ، وَقَوْلُهُ بَعْدُ : أَجْزَأَهُ أَيْ مَعَ الْكَرَاهَةِ ( قَوْلُهُ : الْجَابِرِ النُّسُكِيِّ ) أَيْ ، وَهُوَ رُجُوعُهُ لِلْعُمْرَةِ أَوْ الْحَجِّ وَالْجَابِرُ الْمَالِيُّ ، وَهُوَ الْهَدْيُ ( قَوْلُهُ : إنْ رَكِبَ كَثِيرًا ) أَيْ فِي غَيْرِ الْمَنَاسِكِ وَسَوَاءٌ كَانَ مُخْتَارًا فِي رُكُوبِهِ أَوْ مُضْطَرًّا ( قَوْلُهُ : فِي نَفْسِهِ ) أَيْ ، وَلَيْسَ الْمُرَادُ بِكَثِيرًا أَكْثَرَ الْمَسَافَةِ فَقَطْ لِاقْتِضَائِهِ أَنَّ النِّصْفَ مِنْ حَيِّزِ الْيَسِيرِ ، وَلَيْسَ كَذَلِكَ ( قَوْلُهُ : فَيُهْدِي فَقَطْ ) أَيْ ، وَلَا يَمْشِي مَا رَكِبَهُ ( قَوْلُهُ : أَيْ أَنَّ الْكَثْرَةَ وَالْقِلَّةَ ) يَعْنِي فِي النَّفْسِ مَنْظُورٌ فِيهَا لِاعْتِبَارِ الْمَسَافَةِ سُهُولَةً وَصُعُوبَةً ( قَوْلُهُ : وَمِسَاحَةً ) أَيْ أَوْ مِسَاحَةً( 7 / 108 )
