( إلَّا أَنْ ) بِفَتْحٍ فَسُكُونٍ حَرْفُ مَصْدَرٍ صِلَتُهُ ( يَكُونَ عِنْدَهُ ) أَيْ : الْمَدِينِ ( مُعَشَّرٌ ) بِضَمِّ الْمِيمِ وَفَتْحِ الْعَيْنِ وَالشَّيْنِ مُثَقَّلًا أَيْ : مَا يُزَكَّى بِالْعُشْرِ أَوْ نِصْفِهِ مِنْ حَبٍّ وَثَمَرٍ ( زُكِّيَ ) بِضَمٍّ فَكَسْرٍ مُثَقَّلًا أَيْ : أُخْرِجَتْ زَكَاتُهُ وَأَوْلَى إنْ لَمْ تَجِبْ الزَّكَاةُ فِيهِ لِنَقْصِهِ عَنْ النِّصَابِ وَمِثْلُهُ الْمَاشِيَةُ فَيُحْمَلُ مَا ذُكِرَ فِي الدَّيْنِ وَيُزَكِّي الْعَيْنَ ( أَوْ مَعْدِنٌ ) أَيْ : مَا خَرَجَ مِنْهُ فَيُقَابِلُ بِهِ الدَّيْنَ وَيُزَكِّي الْعَيْنَ .
