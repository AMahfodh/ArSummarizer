( وَ ) لَهُ ( مَنْعُهَا مِنْ ) تَنَاوُلِ ( مُمْرِضٍ وَمُنْتِنِ ) كَثُومٍ وَكُرَّاثٍ دَفْعًا لِلضَّرَرِ ( وَمِنْ خُرُوجٍ ) مِنْ مَسْكَنِهَا لِزِيَارَةٍ أَوْ غَيْرِهَا وَلَوْ لِزِيَارَةِ أَبَوَيْهَا ؛ لِأَنَّهَا مُلْزَمَةٌ بِمُلَازَمَتِهِ نَعَمْ إنْ أَعْسَرَ وَأُمْهِلَ أَوْ رَضِيَتْ بِالْمُقَامِ مَعَهُ فَلَهَا الْخُرُوجُ نَهَارًا لِتَحْصِيلِ الْمُؤْنَةِ بِكَسْبٍ أَوْ تِجَارَةٍ أَوْ سُؤَالٍ وَإِنْ قَدَرَتْ عَلَى الْإِنْفَاقِ مِنْ مَالِ نَفْسِهَا أَوْ كَانَتْ تَكْتَسِبُ بِمَا لَا يُحْوِجُهَا إلَى الْخُرُوجِ كَغَزْلٍ وَخِيَاطَةٍ ؛ لِأَنَّهُ إذَا لَمْ يُوَفِّهَا حَقَّهَا لَا يَمْلِكُ الْحَجْرَ عَلَيْهَا ( وَ ) مِنْ ( دُخُولِ الْمَسْكَنِ أُصُولُهَا ) بِالرَّفْعِ بِالْفَاعِلِيَّةِ لِدُخُولٍ وَالْمُرَادُ أَنَّ لَهُ مَنْعَ أُصُولِهَا مَعَ الْكَرَاهَةِ وَكَذَا مَنْعُ غَيْرِهِمْ بِطَرِيقِ الْأَوْلَى مِنْ دُخُولِهِمْ مَسْكَنَهَا عَلَيْهَا وَإِنْ كَانَ مُؤَدَّى كَلَامِهِ كَأَصْلِهِ أَنَّ لَهُ مَنْعَهَا مِنْ دُخُولِهِمْ عَلَيْهَا لِفَسَادِ هَذَا ظَاهِرًا وَلَهُ مَنْعُهَا مِنْ إدْخَالِ قُمَاشٍ وَأَثَاثٍ لَهَا إلَى مَكَانِهِ .( 17 / 149 )
