( وَذُو زَيْتٍ ) أَيْ أَصْنَافٌ وَيَعْلَمُ مِنْهَا أَنَّهَا رِبَوِيَّةٌ ( كَفُجْلٍ ) أَيْ بِزْرِ الْفُجْلِ الْأَحْمَرِ ؛ لِأَنَّهُ الَّذِي يَخْرُجُ مِنْهُ الزَّيْتُ وَدَخَلَ بِالْكَافِ سَلْجَمٌ وَجُلْجُلَانُ وَقُرْطُمٌ وَزَيْتُونٌ وَبِزْرُ الْكَتَّانِ أَوْلَى بِالدُّخُولِ مِنْ السَّلْجَمِ عَلَى التَّحْقِيقِ ( وَالزُّيُوتُ أَصْنَافٌ ) أَيْ أَجْنَاسٌ كَأُصُولِهَا ( كَالْعُسُولِ ) الْمُخْتَلِفَةِ مِنْ قَصَبٍ وَنَحْلٍ وَرُطَبٍ وَعِنَبٍ فَإِنَّهَا أَصْنَافٌ يَجُوزُ التَّفَاضُلُ بَيْنَهَا مُنَاجَزَةً ( لَا الْخُلُولِ ) فَلَيْسَتْ بِأَصْنَافٍ بَلْ كُلُّهَا صِنْفٌ وَاحِدٌ ؛ لِأَنَّ الْمُبْتَغَى مِنْهَا شَيْءٌ وَاحِدٌ وَهُوَ الْحُمُوضَةُ
