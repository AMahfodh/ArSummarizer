( وَزَادَ ) الْمَشْهُودُ بِعَدَمِهِ فِي يَمِينِهِ ( وَإِنْ وَجَدَ ) مَالًا ( لِيَقْضِيَنَّ ) بِهِ مَا عَلَيْهِ ( وَأُنْظِرَ ) بِضَمِّ الْهَمْزِ وَكَسْرِ الظَّاءِ الْمُعْجَمَةِ أَيْ أُمْهِلَ وَلَا يُطْلَبُ بِمَا عَلَيْهِ إلَى يُسْرِهِ بِهِ لِقَوْلِهِ تَعَالَى { وَإِنْ كَانَ ذُو عُسْرَةٍ فَنَظِرَةٌ إلَى مَيْسَرَةٍ } ، وَلَا يُلَازِمُهُ رَبُّ الدَّيْنِ ، وَفَائِدَةُ هَذِهِ الزِّيَادَةِ عَدَمُ تَحْلِيفِهِ إنْ( 12 / 192 )
