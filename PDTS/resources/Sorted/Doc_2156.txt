( وَ ) قَالَ ( فِي الْمُغْنِي : لَوْ اسْتَأْجَرَهُ لِعَمَلٍ ، فَكَانَ ) ، الْأَجِيرُ ( يَقْرَأُ الْقُرْآنَ حَالَ عَمَلِهِ ، فَإِنْ ضَرَّ الْمُكْرِيَ ) بِقِرَاءَةِ الْقُرْآنِ ؛ ( رَجَعَ ) الْمُكْرِي ( عَلَيْهِ ) - أَيْ الْأَجِيرِ - ( بِقِيمَةِ مَا فَوَّتَ عَلَيْهِ ) مِنْ الْعَمَلِ بِسَبَبِ اشْتِغَالِهِ عَنْهُ بِالْقِرَاءَةِ .( 10 / 290 )
