( وَ ) نُدِبَ ( تَحِيَّةُ الْمَسْجِدِ ) بِرَكْعَتَيْنِ قَبْلَ الْجُلُوسِ بِهِ ( لِدَاخِلٍ ) فِيهِ ( يُرِيدُ - الْجُلُوسَ بِهِ ) أَيْ : بِالْمَسْجِدِ - لَا الْمُرُورَ فِيهِ - وَلَا تَفُوتُ بِالْجُلُوسِ ( فِي وَقْتِ جَوَازٍ ) : لَا وَقْتِ نَهْيٍ ( وَتَأَدَّتْ ) التَّحِيَّةُ ( بِفَرْضٍ ) ، فَيَسْقُطُ طَلَبُهَا بِصَلَاتِهِ .
