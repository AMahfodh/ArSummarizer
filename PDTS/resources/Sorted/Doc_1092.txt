أَوْ انْقَضَتْ عِدَّتُهَا وَهُمَا ) أَيْ الْعَاقِدَانِ ( يَعْلَمَانِ ذَلِكَ ) أَيْ أَنَّهَا بِنْتُهُ وَأَنَّهُ وَلِيُّهَا وَأَنَّ عِدَّتَهَا انْقَضَتْ وَالشَّرْطُ الْحَاضِرُ أَشَارَ إلَيْهِ بِقَوْلِ ( أَوْ ) زَوَّجْتُكَهَا ( إنْ شِئْتَ فَقَالَ : شِئْتُ وَقَبِلْتُ وَنَحْوِهِ ) فَيَصِحُّ النِّكَاحُ ؛ لِأَنَّهُ لَيْسَ بِتَعْلِيقِ حَقِيقَةً بَلْ تَوْكِيدٌ وَتَقْوِيَةُ النَّوْعِ .( 8 / 436 )
