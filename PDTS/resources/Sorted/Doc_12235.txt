( وَتَثْبُتُ ) أَيْ تَسْتَقِرُّ كَفَّارَةُ الظِّهَارِ ( فِي ذِمَّتِهِ ) أَيْ الْمُظَاهِرِ ( بِالْعَوْدِ ) لِقَوْلِهِ تَعَالَى { ثُمَّ يَعُودُونَ } ( وَهُوَ الْوَطْءُ ) نَصًّا لَا الْعَزْمُ عَلَيْهِ فَلَا تَسْتَقِرُّ بِذَلِكَ إلَّا أَنَّهَا شَرْطٌ لِحِلِّ الْوَطْءِ فَيُؤْمَرُ بِهَا مَنْ أَرَادَهُ لِيَسْتَحِلَّهُ بِهَا كَمَا يُؤْمَرُ بِعَقْدِ النِّكَاحِ مَنْ أَرَادَ حِلَّ الْمَرْأَةِ أَوْ لَوْ كَانَ الْوَطْءُ مِنْ مَجْنُونٍ بِأَنْ ظَاهَرَ ثُمَّ جُنَّ وَكَذَا لَوْ بَانَتْ مِنْهُ ثُمَّ زَنَا بِهَا لَا إنْ كَانَ الْوَطْءُ ( مِنْ مُكْرَهٍ ) ؛ لِأَنَّهُ مَعْذُورٌ بِالْإِكْرَاهِ وَوَجْهُ الْقَوْلِ بِأَنَّ الْعَوْدَ هُوَ الْوَطْءُ لِأَنَّهُ فِعْلٌ ضِدُّ قَوْلِ الْمُظَاهِرِ إذْ الْمُظَاهِرُ حَرَّمَ الْوَطْءَ عَلَى نَفْسِهِ وَمَنَعَهَا مِنْهُ فَالْعَوْدُ فِعْلُهُ وَأَمَّا الْإِمْسَاكُ عَنْ الْوَطْءِ فَلَيْسَ بِعَوْدٍ وَلِقَوْلِهِ تَعَالَى : { ثُمَّ يَعُودُونَ لِمَا قَالُوا } وَثُمَّ لِلتَّرَاخِي وَالْإِمْسَاكُ غَيْرُ مُتَرَاخٍ وَلِأَنَّ الظِّهَارَ يَمِينٌ يَقْتَضِي تَرْكَ الْوَطْءِ فَلَا تَجِبُ كَفَّارَتُهُ إلَّا بِهِ كَالْإِيلَاءِ .
