( وَ ) إلَّا ( رَبْطَ سِنٍّ ) تَخَلْخَلَ ، أَوْ سَقَطَ بِخَيْطِ ذَهَبٍ ، أَوْ فِضَّةٍ ( مُطْلَقًا ) عَنْ التَّقْيِيدِ بِأَحَدِ النَّقْدَيْنِ ، أَوْ بِوَزْنٍ مَخْصُوصٍ ، رَاجِعٌ لِلْمُصْحَفِ وَمَا بَعْدَهُ ( وَ ) إلَّا ( خَاتَمَ الْفِضَّةِ ) فَيَجُوزُ لُبْسُهُ لِلذَّكَرِ الْبَالِغِ إنْ قَصَدَ بِهِ الِاقْتِدَاءَ بِالنَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ وَكَانَ وَاحِدًا وَوَزْنُهُ دِرْهَمَيْنِ شَرْعِيَّيْنِ ، أَوْ أَقَلَّ ، وَإِلَّا حَرُمَ ، وَإِنْ اسْتَوْفَى الشُّرُوطَ نُدِبَ كَلُبْسِهِ بِالْيُسْرَى وَجَعْلِ فَصِّهِ لِلْكَفِّ ( لَا ) يَجُوزُ لِلذَّكَرِ الْبَالِغِ ( مَا ) أَيْ خَاتَمٌ ( بَعْضُهُ ) وَأَوْلَى كُلُّهُ ( ذَهَبٌ وَلَوْ قَلَّ ) الذَّهَبُ عَنْ الْفِضَّةِ وَأَشَارَ بِ وَلَوْ إلَى الْقَوْلِ بِجَوَازِ مَا قَلَّ ذَهَبُهُ بِأَنْ كَانَ ثُلُثَهُ فَأَقَلَّ وَاعْتَمَدَ الْمُتَأَخِّرُونَ كَرَاهَتَهُ حِينَئِذٍ .( 1 / 93 )
