( وَلَوْ شَهِدَ اثْنَانِ بِأَنَّ عَمْرَا غَاصِبُ أَوْ سَارِقُ شَيْءٍ فَجْرَا ) أَيْ : بِأَنَّهُ غَصَبَ أَوْ سَرَقَ الشَّيْءَ الْفُلَانِيَّ مِنْ فُلَانٍ وَقْتَ الْفَجْرِ ، ( وَ ) شَهِدَ ( آخَرَانِ ) بِأَنَّهُ غَصَبَهُ أَوْ سَرَقَهُ مِنْهُ ( فِي عَشِيٍّ وَقَعَا تَعَارُضٌ ) بَيْنَ الشَّهَادَتَيْنِ لِعَدَمِ الْمُرَجِّحِ ( فَلْيَتَسَاقَطَا مَعًا ) ، وَاحْتِمَالُ اسْتِرْدَادِهِ ثُمَّ غَصْبِهِ أَوْ سَرِقَتِهِ ثَانِيًا بَعِيدٌ ، ( وَ ) لَوْ شَهِدَ ( شَاهِدٌ كَذَا ) أَيْ : بِأَنَّهُ غَصَبَهُ أَوْ سَرَقَهُ مِنْهُ وَقْتَ الْفَجْرِ ( وَ ) شَهِدَ ( شَاهِدٌ ) آخَرُ ( كَذَا ) أَيْ : بِأَنَّهُ غَصَبَهُ أَوْ سَرَقَهُ مِنْهُ عَشِيَّةً ( يَحْلِفُ ) الْمُدَّعِي ( مَعْ فَرْدٍ ) مِنْهُمَا مُوَافِقٌ لِدَعْوَاهُ ، ( وَغُرْمًا أَخَذَا ) بِالشَّاهِدِ وَالْيَمِينِ ؛ لِأَنَّ الْوَاحِدَ لَيْسَ بِحُجَّةٍ حَتَّى يَقَعَ بِهِ التَّعَارُضُ .
