( وَيُيَمَّمُ ) وُجُوبًا ( رَبُّ مَاءٍ مَاتَ ) بَدَلَ غُسْلِهِ ( لِعَطَشِ رَفِيقِهِ ) ، كَمَا لَوْ كَانَ حَيًّا ، ( وَيَغْرَمُ ) رَفِيقُهُ ( ثَمَنَهُ ) - أَيْ : قِيمَةَ الْمَاءِ - ( مَكَانَهُ وَقْتَ إتْلَافِهِ ) لِوَرَثَةِ الْمَيِّتِ لِانْتِقَالِهِ إلَيْهِمْ ، وَإِنَّمَا يَغْرَمُ قِيمَتَهُ ( مَعَ أَنَّهُ مِثْلِيٌّ ) دَفْعًا لِلضَّرَرِ عَنْ الْوَرَثَةِ ، إذْ الْمَاءُ لَا قِيمَةَ لَهُ فِي الْحَضَرِ غَالِبًا ، ( وَمُقْتَضَاهُ ) أَنَّ ( كُلَّ مِثْلِيٍّ أُتْلِفَ حَالَ غَلَائِهِ ) يُضْمَنُ بِقِيمَتِهِ مَكَانَهُ وَقْتَ التَّلَفِ ، مَعَ أَنَّ الْأَمْرَ لَيْسَ كَذَلِكَ بَلْ الْوَاجِبُ الْمِثْلُ .( 1 / 458 )
