( قَوْلُهُ : بِمَنْ ظَهَرَ صَلَاحُهُ إلَخْ ) ، وَلَوْ فُرِضَ تَأَذِّيهِمْ بِهِ احْتَمَلَ الْكَرَاهَةَ أَيْضًا سم أَيْ كَمَا هُوَ الظَّاهِرُ مِنْ التَّعْلِيلِ ( قَوْلُهُ : وَقَضِيَّتُهَا ) أَيْ الْعِلَّةِ ( أَنَّ مَحَلَّهُ ) أَيْ عَدَمُ الْكَرَاهَةِ وَ ( قَوْلُهُ : فِي تَخَطِّي إلَخْ ) خَبَرُ أَنَّ ( قَوْلُهُ : وَأَنَّهُ لَا فَرْقَ إلَخْ ) اعْتَمَدَهُ ع ش وَالْبُجَيْرِمِيُّ( 9 / 371 )
