( فَرْعٌ ، وَإِنْ اسْتَرْسَلَ ) الْجَارِحُ ( الْمُعَلَّمُ بِنَفْسِهِ فَأَكَلَ مِنْ الصَّيْدِ لَمْ يَخْرُجْ عَنْ كَوْنِهِ مُعَلَّمًا ) إذْ لَا يُعْتَبَرُ الْإِمْسَاكُ إلَّا إذَا أَرْسَلَهُ صَاحِبُهُ ( وَلَا يَحِلُّ ) لِمَفْهُومِ خَبَرِ إذَا أَرْسَلْت كَلْبَكَ الْمُعَلَّمَ فَكُلْ ( وَلَوْ زَادَ عَدْوُهُ بِإِغْرَاءٍ حَدَثَ ) بَعْدَ اسْتِرْسَالِهِ بِنَفْسِهِ فَإِنَّهُ لَا يَحِلُّ تَغْلِيبًا لِلتَّحْرِيمِ( 7 / 106 )
