( قَالَ لِمَوْطُوءَةٍ وَهِيَ ) حَالَ كَوْنِهَا مِمَّنْ تَحِيضُ ( أَنْتِ طَالِقٌ ثَلَاثًا ) أَوْ ثِنْتَيْنِ ( لِلسُّنَّةِ وَقَعَ عِنْدَ كُلِّ طُهْرٍ طَلْقَةٌ ) وَتَقَعُ أُولَاهَا فِي طُهْرٍ لَا وَطْءَ فِيهِ ، فَلَوْ كَانَتْ غَيْرَ مَوْطُوءَةٍ أَوْ لَا تَحِيضُ تَقَعُ وَاحِدَةٌ لِلْحَالِ ثُمَّ كُلَّمَا نَكَحَهَا أَوْ مَضَى شَهْرٌ تَقَعُ ( وَإِنْ نَوَى أَنْ تَقَعَ الثَّلَاثُ السَّاعَةَ أَوْ ) أَنْ تَقَعُ عِنْدَ رَأْسِ ( كُلِّ شَهْرٍ وَاحِدَةٌ صَحَّتْ نِيَّتُهُ ) لِأَنَّهُ مُحْتَمَلُ كَلَامِهِ
