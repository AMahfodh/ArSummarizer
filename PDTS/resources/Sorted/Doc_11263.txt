( قَوْلُهُ وَتُشْتَرَطُ النِّيَّةُ هُنَا ) أَيْ فِيمَا لَوْ عَيَّنَهَا عَمَّا فِي الذِّمَّةِ بِخِلَافِ مَا لَوْ عَيَّنَهَا فِي نَذْرِهِ ابْتِدَاءً ( قَوْلُهُ : فَاحْتَاجَ لَهَا ) أَيْ النِّيَّةِ ( قَوْلُهُ : كَالْمَمْلُوكَةِ وَمَصْرِفُهُ ) أَيْ الْأَرْشِ فَظَاهِرُهُ أَنَّهُ يَصْرِفُهُ دَرَاهِمَ وَلَا يُشْتَرَى بِهِ لَحْمٌ وَلَا شِقْصٌ ( قَوْلُهُ : فَكَإِتْلَافِهِ ) فَتَلْزَمُ الْقِيمَةَ الْفُضُولِيُّ بِتَمَامِهَا وَيَدْفَعُهَا لِلنَّاذِرِ فَيَشْتَرِي بِهَا بَدَلَهَا وَيَذْبَحُهَا فِي وَقْتِ التَّضْحِيَةِ ، وَإِنَّمَا لَمْ يَكْتَفِ بِتَفْرِيقِ الْفُضُولِيِّ مَعَ أَنَّهَا خَرَجَتْ عَنْ مِلْكِ النَّاذِرِ بِالنَّذْرِ لِأَنَّهُ فَوَّتَ تَفْرِقَةَ الْمَالِكِ الَّتِي هِيَ حَقُّهُ ( قَوْلُهُ : وَبَحَثَ الزَّرْكَشِيُّ إلَخْ ) ضَعِيفٌ ، وَقَوْلُهُ إسْلَامُهُ : أَيْ الْوَكِيلِ ( قَوْلُهُ : لَا يَمْنَعُهُ مِنْ وُقُوعِهِ ) أَيْ حَيْثُ وَلَّى الْمَالِكُ تَفْرِقَتَهُ وَإِلَّا فَكَإِتْلَافِهِ كَمَا مَرَّ( 27 / 215 )
