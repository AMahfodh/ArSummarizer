وَقَوْلُ سم : وَإِلَّا انْفَسَخَ : أَيْ فَيَسْتَرِدُّ الْمُشْتَرِي الثَّمَنَ وَيَغْرَمُ لِلْبَائِعِ بَدَلَ الْمَبِيعِ مِنْ قِيمَةٍ أَوْ مِثْلٍ ( قَوْلُهُ : لَا وَكِيلُهُ ) أَيْ وَلَا وَلِيُّهُ مِنْ أَبٍ أَوْ جَدٍّ أَوْ وَصِيٍّ أَوْ قَيِّمٍ فَلَا يَكُونُ إتْلَافُهُمْ قَبْضًا ( قَوْلُهُ : وَإِنْ بَاشَرَ ) صِلَةُ وَكِيلِهِ ( قَوْلُهُ : وَلَمْ يَكُنْ لِعَارِضٍ ) كَالصِّيَالِ أَوْ اسْتِحْقَاقِ الْمُشْتَرِي الْقِصَاصَ ( قَوْلُهُ : أَوْ لِرِدَّتِهِ ) وَاسْتُشْكِلَ بِأَنَّهُ غَيْرُ مَضْمُونٍ .
