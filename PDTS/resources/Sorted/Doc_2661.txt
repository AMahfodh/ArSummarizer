( وَ ) لَهُ ( تَرْكُهُ غَنِيمَةً ) لِلْغَانِمِينَ ؛ لِأَنَّ الْحَقَّ لَهُ ، فَإِنْ شَاءَ اسْتَوْفَاهُ ، وَإِنْ شَاءَ تَرَكَهُ ( فَإِنْ أَخَذَهُ ) قَبْلَ الْقِسْمَةِ ( أَخَذَهُ مَجَّانًا ) لِمَا تَقَدَّمَ ( وَإِنْ أَبَى أَخْذَهُ ) قُسِّمَ ؛ لِأَنَّ رَبَّهُ لَمْ يَمْلِكْهُ ، وَإِنَّمَا هُوَ أَحَقُّ بِهِ فَإِذَا تَرَكَهُ سَقَطَ مِنْ التَّقْدِيمِ .( 8 / 111 )
