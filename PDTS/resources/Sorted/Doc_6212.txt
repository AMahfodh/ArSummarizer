( وَ ) إنْ فَسَخَ ( بَعْدَهُ ) أَيْ بَعْدَ الدُّخُولِ ( أَوْ بَعْدَ خَلْوَةٍ ) فَ ( لَهَا الْمُسَمَّى ) لِأَنَّهُ نِكَاحٌ صَحِيحٌ وُجِدَ بِأَرْكَانِهِ وَشُرُوطِهِ ، فَتَرَتَّبَ عَلَيْهِ أَحْكَامُ الصِّحَّةِ وَلِأَنَّ الْمَهْرَ يَجِبُ بِالْعَقْدِ وَيَسْتَقِرُّ بِالْخَلْوَةِ فَلَا يَسْقُطُ بِحَادِثٍ بَعْدَهُ ، وَكَمَا لَوْ طَرَأَ الْعَيْبُ ( وَيَرْجِعُ ) الزَّوْجُ ( بِهِ ) أَيْ بِالْمَهْرِ عَلَى مَنْ غَرَّهُ مِنْ امْرَأَةٍ عَاقِلَةٍ ( وَوَلِيٍّ وَوَكِيلٍ ) رَوَاهُ مَالِكٌ عَنْ عُمَرَ ، وَكَمَا لَوْ غَرَّ بِحُرِّيَّةِ أَمَةٍ .
