( وَمَنْ أَحْرَمَ بِحَجٍّ ) وَحَجَّ ( ثُمَّ أَحْرَمَ يَوْمَ النَّحْرِ بِآخَرَ ، فَإِنْ ) كَانَ قَدْ ( حَلَقَ لِلْأَوَّلِ ) لَزِمَهُ الْآخَرُ فِي الْعَامِ الْقَابِلِ ( بِلَا دَمٍ ) لِانْتِهَاءِ الْأَوَّلِ ( وَإِلَّا ) يَحْلِقُ لِلْأَوَّلِ ( فَمَعَ دَمٍ قَصَرَ ) عَبَّرَ بِهِ لِيَعُمَّ الْمَرْأَةَ ( أَوْ لَا ) لِجِنَايَتِهِ عَلَى إحْرَامِهِ بِالتَّقْصِيرِ أَوْ التَّأْخِيرِ
