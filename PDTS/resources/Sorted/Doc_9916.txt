فَإِذَا أَخَذَ الْبَعْضَ لَمْ يَنْدَفِعْ الضَّرَرُ ( فَإِنْ طَلَبَ ) الشَّفِيعُ ( بَعْضَهُ ) أَيْ : الْمَبِيعِ ( مَعَ بَقَاءِ الْكُلِّ ) أَيْ : كُلِّ الْمَبِيعِ ( سَقَطَتْ ) شُفْعَتُهُ لِمَا تَقَدَّمَ ، وَلِأَنَّ حَقَّ الْأَخْذِ إذَا سَقَطَ بِالتَّرْكِ فِي الْبَعْضِ سَقَطَ فِي الْكُلِّ كَعَفْوِهِ عَنْ بَعْضِ قَوَدٍ يَسْتَحِقُّهُ ، ( وَإِنْ تَلِفَ بَعْضُهُ ) أَيْ الْمَبِيعِ ، كَانْهِدَامِ بَيْتٍ مِنْ دَارٍ بِيعَ بَعْضُهَا بِأَمْرٍ سَمَاوِيٍّ كَمَطَرٍ أَوْ بِفِعْلِ آدَمِيٍّ مُشْتَرٍ أَوْ غَيْرَهُ ( أَخَذَ ) الشَّفِيعُ ( بَاقِيَهُ ) أَيْ : الْمَبِيعِ إنْ شَاءَ ( بِحِصَّتِهِ ) أَيْ : الْمَبِيعِ بَعْدَ مَا تَلِفَ ( مِنْ ثَمَنِهِ ) أَيْ : ثَمَنِ جَمِيعِ الشِّقْصِ .
