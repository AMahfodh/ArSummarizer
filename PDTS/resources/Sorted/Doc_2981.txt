وَهِيَ مُشْتَمِلَةٌ عَلَى تَوْجِيهِ صَرَاحَتِهِ فِي الْفَسْخِ وَأَنَّهُ مَعَ كَوْنِهِ صَرِيحًا فِيهِ يَكُونُ كِنَايَةً فِي الطَّلَاقِ ( قَوْلُهُ : اسْتِقْلَالِيٌّ ) احْتَرَزَ بِهِ عَنْ تَعْلِيقِ الطَّلَاقِ فَإِنَّهُ يَصِحُّ مَعَ كَوْنِهِ اخْتِيَارًا لَكِنَّهُ ضِمْنِيٌّ كَمَا يَأْتِي ( قَوْلُهُ : اسْتِدَامَةٌ ) أَيْ لِلنِّكَاحِ ( قَوْلُهُ وَهُوَ لَا يَقْبَلُ ) أَيْ الْمَنَاطُ وَكَانَ الْأَوْلَى وَهِيَ ( قَوْلُهُ وَلَوْ( 21 / 99 )
