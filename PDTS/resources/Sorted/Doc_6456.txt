( قَوْلُهُ : فَإِنَّهُ يَمْتَنِعُ مُمَاكَسَتُهُ ) شَامِلٌ لِلْمُمَاكَسَةِ عِنْدَ الْعَقْدِ وَلِمُمَاكَسَتِهِ عِنْدَ الْأَخْذِ بِأَنْ عَقَدَ عَلَى الْأَوْصَافِ أَخْذًا مِنْ كَلَامِ الْبَغَوِيّ الْآتِي ؛ لِأَنَّهُ إذَا لَمْ يَصِحَّ الْتِزَامُهُ زِيَادَةً عَلَى الدِّينَارِ فَلَا تُؤْخَذُ مِنْهُ زِيَادَةٌ إذَا عُقِدَ عَقْدٌ عَلَى الْأَوْصَافِ فَكِلَا الْمُمَاكَسَتَيْنِ مُمْتَنِعَةٌ فَلْيُتَأَمَّلْ سم ( قَوْلُهُ : وَيُمْكِنُ أَنْ يَكُونَ ذَلِكَ ) إشَارَةٌ إلَى قَوْلِهِ نَفَعَهُ النَّدَمُ ( قَوْلُهُ : فَلَا يُغَيِّرُ ) بَقِيَ مَا لَوْ حَصَلَ النَّدَمُ فِي أَثْنَاءِ الْعَامِ( 18 / 497 )
