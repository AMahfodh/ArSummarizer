وَلَا يُبْعِدُ الْكَرَاهَةَ تَغْلِيبًا لِلْمُعَظَّمِ وَيُحْتَمَلُ أَنْ يُنْظَرَ فِيهِ لِقَصْدِ الْمُسْتَعْمِلِ عَلَى مَا مَرَّ ( قَوْلُهُ لِنَفْسِهِ ) وَغَيْرِهِ تَبَرُّعًا قِيَاسًا عَلَى مَا مَرَّ فِي التَّمِيمَةِ ا ه حَجّ ( قَوْلُهُ : وَإِلَّا فَالْمَكْتُوبُ لَهُ ) وَبَقِيَ الْإِطْلَاقُ وَيَنْبَغِي عَدَمُ الْكَرَاهَةِ لِمَا مَرَّ مِنْ أَنَّ الْأَصْلَ الْإِبَاحَةُ ( قَوْلُهُ : قَالَ فِي الْمُهِمَّاتِ ) أَيْ الْإِسْنَوِيُّ ( قَوْلُهُ : وَفِي حِفْظِي أَنَّهَا كَانَتْ تُقْرَأُ مِنْ أَسْفَلَ إلَخْ ) قَالَ ابْنُ حَجَرٍ : وَلَمْ يَصِحَّ فِي كَيْفِيَّةِ وَضْعِ ذَلِكَ شَيْءٌ ( قَوْلُهُ : نَعَمْ يُمْكِنُ حَمْلُ إلَخْ ) وَيُمْكِنُ أَنْ يَبْقَى عَلَى ظَاهِرِهِ ، وَيُقَالُ الْوَاحِدُ بِالشَّخْصِ( 1 / 405 )
