( وَلَوْ ) ( صَالَحَهُ عَنْ بَيْتٍ ) ادَّعَى عَلَيْهِ بِهِ ، ( وَأَقَرَّ لَهُ بِهِ عَلَى بَعْضِهِ ) ؛ أَيْ : الْبَيْتِ ( أَوْ ) عَلَى ( سُكْنَاهُ ) ؛ أَيْ : سُكْنَى الْمُدَّعَى عَلَيْهِ الْبَيْتَ ( مُدَّةً ) مَعْلُومَةً ؛ كَسَنَةِ كَذَا أَوْ مَجْهُولَةً كَمَا عَاشَ ، ( أَوْ عَلَى بِنَاءِ غُرْفَةٍ لَهُ ) ؛ أَيْ : الْمُدَّعَى عَلَيْهِ ( فَوْقَهُ ) أَيْ : الْبَيْتِ ، لَمْ يَصِحَّ الصُّلْحُ ؛ لِأَنَّهُ صَالَحَهُ عَنْ مِلْكِهِ عَلَى مِلْكِهِ أَوْ عَلَى مَنْفَعَةِ مِلْكِهِ .( 9 / 34 )
