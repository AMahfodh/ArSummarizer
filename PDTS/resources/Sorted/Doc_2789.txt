( وَ ) يَجُوزُ ( مِنْ خَارِجِ بَلَدِهِ دُونَ مَسَافَةِ الْقَصْرِ ) ؛ ؛ لِأَنَّ مَا دُونَهَا فِي حُكْمِ الْحَاضِرِ ، وَ ( لَا ) يَجُوزُ أَنْ يُسْتَنَابَ عَنْهُ مِمَّا ( فَوْقَهَا ) أَيْ : فَوْقَ مَسَافَةِ الْقَصْرِ لِمَا تَقَدَّمَ ( وَلَا يُجْزِئُهُ ) حَجُّ مَنْ اُسْتُنِيبَ عَنْهُ مِمَّا فَوْقَ الْمَسَافَةِ لِعَدَمِ إتْيَانِهِ بِالْوَاجِبِ .( 6 / 337 )
