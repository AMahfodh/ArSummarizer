وَذَكَرَهَا عَلَى هَذَا التَّرْتِيبِ فَقَالَ ( وَإِنْ ) ( وَجَدَ ) أَحَدُ الْمُتَقَاسِمَيْنِ فِي حِصَّتِهِ ( عَيْبًا ) قَدِيمًا لَمْ يَظْهَرْ لَهُ عِنْدَ الْقِسْمَةِ ( بِالْأَكْثَرِ ) مِنْ حِصَّتِهِ بِأَنْ زَادَ عَلَى نِصْفِهَا ( فَلَهُ رَدُّهَا ) أَيْ الْقِسْمَةِ أَيْ إبْطَالُهَا وَتَكُونُ الشَّرِكَةُ كَمَا كَانَتْ قَبْلَ الْقِسْمَةِ وَسَوَاءٌ كَانَ الْمَقْسُومُ عَقَارًا أَوْ حَيَوَانًا أَوْ عُرُوضًا أَيْ وَلَهُ التَّمَسُّكُ بِالْحِصَّةِ وَلَا يَرْجِعُ بِشَيْءٍ ؛ لِأَنَّ خِيرَتَهُ تَنْفِي ضَرَرَهُ وَلَيْسَ لَهُ التَّمَسُّكُ بِهَا وَيَرْجِعُ بِأَرْشِ الْعَيْبِ لِقَوْلِهِ وَحَرُمَ التَّمَسُّكُ بِأَقَلَّ اسْتَحَقَّ أَكْثَرَهُ أَيْ أَوْ تَعَيَّبَ أَيْ حَرُمَ التَّمَسُّكُ بِأَقَلَّ وَالرُّجُوعُ .
