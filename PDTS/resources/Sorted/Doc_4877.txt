( وَيَحْرُمُ ) عَلَى مَنْ عِنْدَهُ شَهَادَةٌ بِحَقِّ آدَمِيٍّ لَا يَعْلَمُهَا ( كَتْمُهَا ) لِلْآيَةِ ( فَيُقِيمُهَا ) أَيْ الشَّهَادَةَ ( بِطَلَبِهِ ) أَيْ الْمَشْهُودِ لَهُ ، ( وَلَوْ لَمْ يَطْلُبْهَا حَاكِمٌ ) مِنْهُ لِمَا تَقَدَّمَ ، ( وَلَا يَقْدَحُ ) أَدَاءُ الشَّهَادَةِ بِلَا طَلَبِ حَاكِمٍ وَبِلَا طَلَبِ مَشْهُودٍ لَهُ لَمْ يَعْلَمْ بِهِ ( فِيهِ كَشَهَادَةِ حِسْبَةً ) بِحَقٍّ لَلَهُ تَعَالَى مِنْ غَيْرِ تَقَدُّمِ دَعْوَى( 12 / 350 )
