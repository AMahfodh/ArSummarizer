( قَوْلُهُ : السَّفِيهُ ) ، وَأَمَّا الرَّشِيدُ فَكَالْأَجْنَبِيِّ ( قَوْلُهُ : وَإِنْ لَمْ يَمْرَضْ إلَخْ ) الْمُنَاسِبُ وَإِنْ مَرِضَ السَّيِّدُ وَقَرُبَ الْأَجَلُ كَمَا هُوَ ظَاهِرٌ بَلْ وَيَدُلُّ عَلَيْهِ كَلَامُ بَعْضِهِمْ ( قَوْلُهُ : وَقَالَ بَعْضٌ ) ، أَيْ : وَاعْتَمَدَهُ بَعْضُهُمْ ( قَوْلُهُ : التَّعْلِيلُ ) وَهُوَ إحْرَازُ النَّفْسِ ، وَالْمَالِ أَيْ فَالْمُبَعَّضُ لَا يُنْتَزَعُ مَالُهُ .( 16 / 393 )
