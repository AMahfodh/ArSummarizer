( وَلِكُلٍّ ) مِنْ الْمُتَآجِرَيْنِ فِيمَا إذَا اسْتَأْجَرَهُ كُلَّ يَوْمٍ أَوْ شَهْرٍ بِعِوَضٍ مَعْلُومٍ ( الْفَسْخُ أَوَّلُ كُلِّ يَوْمٍ ) إذَا قَالَ : كُلُّ يَوْمٍ بِكَذَا ( أَوْ ) أَوَّلَ كُلِّ ( شَهْرٍ ) إذَا قَالَ : كُلُّ شَهْرٍ بِكَذَا ( فِي الْحَالِ ) أَيْ : فَوْرًا ؛ لِأَنَّ تَمَهُّلَهُ دَلِيلُ رِضَاهُ بِلُزُومِ الْإِجَارَةِ فِيهِ .
