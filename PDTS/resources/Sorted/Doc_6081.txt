الشَّرِيكِ إذَا كَانَ لَا يَنْقَسِمُ وَإِلَّا قَسَمَ ( أَوْ لِخَشْيَةِ انْتِقَالِ الْعِمَارَةِ ) عَنْهُ فَيَصِيرُ مُنْفَرِدًا فَتَقِلُّ قِيمَتُهُ فَيُبَاعُ ( أَوْ ) لِخَشْيَةِ ( الْخَرَابِ عَلَيْهِ ، وَلَا مَالَ لَهُ ) : أَيْ لِلْمَحْجُورِ عَلَيْهِ يَعْمُرُ بِهِ ( أَوْ لَهُ مَالٌ وَالْبَيْعُ أَوْلَى ) : مِنْ التَّعْمِيرِ ( فَيَسْتَبْدِلُ ) أَيْ فَيُبَاعُ فِي جَمِيعِ مَا تَقَدَّمَ وَيُسْتَبْدَلُ ( لَهُ خِلَافُهُ ) : إلَّا أَنْ يَبِيعَ لِحَاجَةِ النَّفَقَةِ أَوْ الدَّيْنِ أَوْ بَيْعِ شَرِيكِهِ فَلَا يَلْزَمُ اسْتِبْدَالٌ .
