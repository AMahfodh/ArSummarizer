وَلَمَّا ذَكَرَ أَنَّ الْقَوَدَ مُتَعَيِّنٌ رَتَّبَ عَلَيْهِ قَوْلَهُ ( وَلَا دِيَةَ لِعَافٍ ) أَيْ لِوَلِيِّ عَافٍ عَنْ الْقَاتِلِ ( مُطْلِقٍ ) فِي عَفْوِهِ بِكَسْرِ اللَّامِ اسْمُ فَاعِلٍ بِأَنْ لَمْ يُصَرِّحْ حَالَ الْعَفْوِ بِدِيَةٍ وَلَا غَيْرِهَا ( إلَّا أَنْ تَظْهَرَ ) بِقَرَائِنِ الْأَحْوَالِ ( إرَادَتُهَا ) وَيَقُولُ بِالْحَضْرَةِ إنَّمَا عَفَوْتُ عَلَى الدِّيَةِ ( فَيَحْلِفُ ) أَيْ فَيُصَدَّقُ بِيَمِينٍ ( وَيَبْقَى عَلَى حَقِّهِ ) فِي الْقَتْلِ ( إنْ امْتَنَعَ ) الْقَاتِلُ مِنْ إعْطَاءِ الدِّيَةِ فَإِنْ لَمْ يَقُلْ ذَلِكَ بِالْحَضْرَةِ بَلْ بَعْدَ طُولٍ فَلَا شَيْءَ لَهُ وَبَطَلَ حَقُّهُ لِمُنَافَاةِ الطُّولِ الْإِرَادَةَ الْمَذْكُورَةَ ( كَعَفْوِهِ ) أَيْ الْوَلِيِّ ( عَنْ الْعَبْدِ ) الَّذِي قَتَلَ عَبْدًا مِثْلَهُ ، أَوْ حُرًّا وَقَالَ إنَّمَا عَفَوْت لِآخُذَهُ ، أَوْ آخُذَ قِيمَتَهُ ، أَوْ آخُذَ قِيمَةَ الْمَقْتُولِ ، أَوْ دِيَةَ الْحُرِّ فَلَا شَيْءَ لَهُ إلَّا أَنْ تَظْهَرَ إرَادَةُ ذَلِكَ فَيَحْلِفُ وَيُخَيِّرُ سَيِّدُ الْعَبْدِ الْجَانِي بَيْنَ دَفْعِهِ ، أَوْ دَفْعِ قِيمَتِهِ ، أَوْ قِيمَةِ الْمَقْتُولِ ، أَوْ دِيَةِ الْحُرِّ وَيَدْفَعُهَا حَالَّةً كَمَا فِي الْمُدَوَّنَةِ وَقِيلَ مُنَجَّمَةً ، وَالْخِلَافُ فِي الْعَمْدِ وَأَمَّا فِي الْخَطَإِ فَتُنَجَّمُ قَطْعًا كَمَا يَأْتِي ( وَاسْتَحَقَّ وَلِيُّ ) الْمَقْتُولِ لِمَقْتُولٍ قَتَلَ قَاتِلَهُ أَجْنَبِيٌّ ( دَمَ مَنْ ) أَيْ دَمَ الْأَجْنَبِيِّ الَّذِي ( قَتَلَ الْقَاتِلَ ) فَلَوْ قَتَلَ زَيْدٌ عَمْرًا فَقَتَلَ أَجْنَبِيٌّ زَيْدًا فَوَلِيُّ عَمْرٍو يَسْتَحِقُّ دَمَ الْأَجْنَبِيِّ الْقَاتِلِ( 18 / 42 )
