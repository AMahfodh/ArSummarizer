فَزَوَّجَهَا غَيْرَ كُفْءٍ جَازَ ) كَمَا لَوْ قَالَتْ زَوِّجْنِي مَنْ شِئْت كُفُؤًا أَوْ غَيْرَهُ ( وَإِذَا أَذِنَتْ لَهُ ) أَيْ لِوَلِيِّهَا ( مُطْلَقًا ) أَيْ مِنْ غَيْرِ تَعْيِينِ زَوْجٍ ( فَلَهُ التَّوْكِيلُ مُطْلَقًا ) كَذَلِكَ ( فَإِنْ عَيَّنَتْهُ ) فِي إذْنِهَا لَهُ ( وَجَبَ تَعْيِينُهُ لِلْوَكِيلِ ) فِي التَّوْكِيلِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يُعَيِّنْ فِي التَّوْكِيلِ ( لَمْ يَصِحَّ ) النِّكَاحُ ( وَلَوْ زَوَّجَ ) هَا ( الْمُعَيَّنُ ) ؛ لِأَنَّ التَّفْوِيضَ الْمُطْلَقَ مَعَ أَنَّ الْمَطْلُوبَ مُعَيَّنٌ فَاسِدٌ وَهَذَا .
