( فَصْلٌ ) ( وَصَايَا الذِّمِّيِّ ) عَلَى أَرْبَعَةِ أَوْجُهٍ لِأَنَّهَا ( إمَّا بِمَعْصِيَةٍ عِنْدَنَا وَعِنْدَهُمْ كَمَا لِلْمُغَنِّيَاتِ ، وَالنَّائِحَاتِ فَتَصِحُّ ) لَوْ كَانَتْ ( لِقَوْمٍ مُعَيَّنِينَ تَمْلِيكًا مِنْ الثُّلُثِ ) فَإِنَّهُمْ لَمَّا تَعَيَّنُوا جَازَ تَمْلِيكُهُمْ ( وَإِلَّا ) أَيْ ، وَإِنْ لَمْ يَكُونُوا مُعَيَّنِينَ ( فَلَا ) أَيْ لَا تَصِحُّ أَصْلًا أَمَّا تَمْلِيكًا فَلِأَنَّ التَّمْلِيكَ لِلْمَجْهُولِ لَا يَصِحُّ وَأَمَّا قُرْبَةً فَلِأَنَّهَا مَعْصِيَةٌ عِنْدَ الْكُلِّ فَكَيْفَ تَصِحُّ قُرْبَةً ( وَأَمَّا بِمَعْصِيَةٍ عِنْدَهُمْ وَقُرْبَةٍ عِنْدَنَا كَجَعْلِ دَارِهِ مَسْجِدًا أَوْ الْإِسْرَاجِ فِي الْمَسْجِدِ فَلَا تَصِحُّ اتِّفَاقًا ) اعْتِبَارًا لِاعْتِقَادِهِمْ لِأَنَّا نَعْمَلُ مَعَهُمْ بِدِيَانَتِهِمْ ( إلَّا أَنْ تَكُونَ لِقَوْمٍ بِأَعْيُنِهِمْ ) فَحِينَئِذٍ تَصِحُّ تَمْلِيكًا مِنْهُمْ وَذِكْرُ الْجِهَةِ مَشُورَةٌ ( وَإِمَّا بِقُرْبَةٍ عِنْدَنَا وَعِنْدَهُمْ كَجَعْلِ ثُلُثِهِ لِلْفُقَرَاءِ أَوْ عِتْقِ الرَّقَبَةِ أَوْ الْإِسْرَاجِ فِي بَيْتِ الْمَقْدِسِ فَتَصِحُّ اتِّفَاقًا ) لِأَنَّ الدِّيَانَةَ مُتَّفِقَةٌ مِنْ الْكُلِّ ( وَإِمَّا بِقُرْبَةٍ عِنْدَهُمْ وَمَعْصِيَةٍ عِنْدَنَا كَجَعْلِ دَارِهِ بِيعَةً ) لِلْيَهُودِ ( أَوْ كَنِيسَةً ) لِلنَّصَارَى ( أَوْ بَيْتَ نَارٍ ) لِلْمَجُوسِ ( فَتَصِحُّ مُطْلَقًا ) أَيْ سَوَاءٌ عَيَّنَ قَوْمًا أَوْ لَا ( وَعِنْدَهُمَا لَا ) أَيْ لَا تَصِحُّ إلَّا أَنْ يُوصَى ( لِمُعَيَّنِينَ ) لَهُمَا أَنَّهُ وَصِيَّةٌ بِالْمَعْصِيَةِ وَفِي تَنْفِيذِهَا تَقْرِيرٌ لِلْمَعْصِيَةِ ، وَالسَّبِيلُ فِي الْمَعْصِيَةِ رَدُّهَا لَا تَنْفِيذُهَا وَلَهُ أَنَّ الْمُعْتَبَرَ دِيَانَتُهُمْ فِي حَقِّهِمْ لِأَنَّا أُمِرْنَا بِأَنْ نَتْرُكَهُمْ وَمَا يَدِينُونَ وَهِيَ قُرْبَةٌ عِنْدَهُمْ فَتَصِحُّ .
