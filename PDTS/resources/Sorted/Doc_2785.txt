ا ه كُرْدِيٌّ عَلَى بَافَضْلٍ ( قَوْلُهُ : لَا يَجُوزُ إخْرَاجُ الْمِثْلِ حَيًّا ) أَيْ وَلَا أَكْلُ شَيْءٍ مِنْهُ نِهَايَةٌ وَمُغْنِي قَوْلُ الْمَتْنِ ( وَبَيْنَ أَنْ يُقَوَّمَ الْمِثْلُ ) أَيْ بِالنَّقْدِ الْغَالِبِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : وَإِنْ كَانَ أَحَدُهُمَا ) أَيْ أَوْ كِلَاهُمَا أَخْذًا مِمَّا مَرَّ فِي شَرْحِ يَحْكُمُ بِمِثْلِهِ عَدْلَانِ ( قَوْلُهُ :( 16 / 121 )
