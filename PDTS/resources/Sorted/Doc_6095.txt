( وَ ) الْفَرْعُ ( لِمُعَاهَدٍ ) بِعَقْدِ جِزْيَةٍ ، أَوْ أَمَانٍ نُقِضَ عَهْدُهُ ، وَالْتَحَقَ بِدَارِ الْحَرْبِ ، وَتَرَكَ فَرْعَهُ لَا يُغْتَالُ ، وَلَا يُسْتَرَقُّ بَلْ ( بِجِزْيَةٍ أَقَرْ ) إنْ قَبِلَهَا ( أَوْ أُلْحِقَ الْمَأْمَنَ ) إنَّ لَمْ يَقْبَلْهَا بِأَنْ نَمْنَعَهُ مِنْ الْمُسْلِمِينَ ، وَأَهْلِ عَهْدِهِمْ حَتَّى يَلْحَقَ بِدَارِ الْحَرْبِ ، وَمَحَلِّ الْإِقْرَارِ ، وَالْإِلْحَاقِ ( بَعْدَ أَنْ كَبِرْ ) بِكَسْرِ الْبَاءِ أَيْ : بَلَغَ كَمَا عَبَّرَ بِهِ الْحَاوِي( 18 / 159 )
