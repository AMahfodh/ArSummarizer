( وَ ) يَتَّجِهُ أَيْضًا ( أَنَّهُ لَيْسَ لِبَائِعٍ وَمُؤَجِّرٍ لِزَامُهُ ) ؛ أَيْ : إلْزَامُ مُشْتَرٍ أَوْ مُسْتَأْجِرٍ ( بِ ) دَفْعِ ( بَقِيَّةِ ثَمَنٍ وَأُجْرَةٍ ) فِي مُدَّةِ الِاشْتِرَاطِ ، ( وَإِنْ لَزِمَ عَقْدٌ ) مِنْ بَيْعٍ أَوْ إجَارَةٍ ( بِتَصَرُّفٍ ؛ لِأَنَّهُ ) ؛ أَيْ : هَذَا الْعَقْدَ ( يُشْبِهُ تَعْلِيقَ فَسْخٍ ) عَلَى شَرْطٍ ، ( وَيَأْتِي ) .
