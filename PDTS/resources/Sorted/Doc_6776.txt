( وَلَا ) تَصِحُّ إجَارَةٌ ( فِي امْرَأَةٍ ذَاتِ زَوْجٍ بِلَا إذْنِهِ ) لِتَفْوِيتِ حَقِّ الزَّوْجِ فِي الِاسْتِمْتَاعِ ؛ لِاشْتِغَالِهَا عَنْهُ بِمَا اُسْتُؤْجِرَتْ لَهُ ( وَلَا يُقْبَلُ قَوْلُهَا ) بِلَا بَيِّنَةٍ بَعْدَ أَنْ أَجَّرَتْ نَفْسَهَا ( إنَّهَا مُتَزَوِّجَةٌ ) فِي بُطْلَانِ الْإِجَارَةِ ( أَوْ ) أَيْ : وَلَا يُقْبَلُ قَوْلُ مَنْ تَزَوَّجَتْ ثُمَّ ادَّعَتْ أَنَّهَا ( مُؤَجَّرَةٌ قَبْلَ نِكَاحٍ ) فِي حَقِّ زَوْجٍ بِلَا بَيِّنَةٍ ؛ لِأَنَّهَا مُتَّهَمَةٌ فِي الصُّورَتَيْنِ وَالْأَصْلُ عَدَمُ مَا تَدَّعِيهِ .( 6 / 202 )
