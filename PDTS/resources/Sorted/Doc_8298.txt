( وَلَا سَهْوَ ) أَيْ لَا سُجُودَ لِسَهْوِهِ عَنْ الْحَرَكَةِ لِلرُّكُوعِ وَقَالَ الْإِمَامُ عَبْدُ الرَّحْمَنِ بْنُ الْقَاسِمِ لَا يَعْتَدُّ بِهِ وَيَخِرُّ سَاجِدًا فَلَا يَعْتَدُّ بِهِ وَيَخِرُّ سَاجِدًا وَيَسْجُدُ بَعْدَ السَّلَامِ ، إلَّا إذَا ذَكَرَهَا قَبْلَ طُمَأْنِينَتِهِ فِي رُكُوعِهِ ( بِخِلَافِ تَكْرِيرِهَا ) أَيْ سَجْدَةِ التِّلَاوَةِ سَهْوًا فَإِنَّهُ يَسْجُدُ بَعْدَ سَلَامِهِ فَإِنْ كَرَّرَهَا عَمْدًا بَطَلَتْ صَلَاتُهُ ( أَوْ سُجُودٍ ) لِلتِّلَاوَةِ ( قَبْلَ ) قِرَاءَةِ مَحَلِّ ( هَا ) أَيْ السَّجْدَةِ لِظَنِّهِ أَنَّ الَّذِي قَرَأَهُ مَحَلُّهَا ( سَهْوًا ) فَيَسْجُدُ بَعْدَ السَّلَامِ سَوَاءٌ قَرَأَهَا بَعْدَ ذَلِكَ وَسَجَدَ لَهَا أَمْ لَا .
