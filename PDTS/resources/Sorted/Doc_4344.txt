وَالْحَاصِلُ مِنْ كُلٍّ مِنْهُمَا أَصْلُ الْمَسْأَلَةِ ( أَوْ تَمَاثَلَا اُكْتُفِيَ بِأَحَدِهِمَا ) وَهُوَ أَصْلُ الْمَسْأَلَةِ هَذَا إنْ كَانَتْ الْوَرَثَةُ غَيْرَ عَصَبَاتٍ ( فَإِنْ كَانَ الْوَرَثَةُ عَصَبَاتٍ فَمَسْأَلَتُهُمْ مِنْ عَدَدِ رُءُوسِهِمْ ) ذُكُورًا كَانُوا كَثَلَاثَةِ بَنِينَ أَوْ إنَاثًا كَثَلَاثِ نِسْوَةٍ أَعْتَقْنَ عَبْدًا بِالسَّوِيَّةِ أَوْ ذُكُورًا وَإِنَاثًا ( وَيُقَدَّرُ ) فِيهِ ( كُلُّ ذَكَرٍ أُنْثَيَيْنِ ) فَأَصْلُ مَسْأَلَةِ ابْنٍ وَبِنْتٍ ثَلَاثَةٌ : لِلِابْنِ :( 13 / 303 )
