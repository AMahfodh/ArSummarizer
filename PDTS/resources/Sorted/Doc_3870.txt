( وَ ) حَرُمَ ( حَمْلُ رَأْسٍ ) مِنْ عَدُوٍّ وَمِنْ بَلَدِ قَتْلِهِ ( لِبَلَدٍ ) آخَرَ ( أَوْ ) لِ ( وَالٍ ) أَيْ أَمِيرِ جَيْشٍ فِي بَلَدِ الْقِتَالِ وَيَجُوزُ حَمْلُهَا فِي بَلَدِ الْقِتَالِ لِغَيْرِ وَالٍ وَاسْتُظْهِرَ جَوَازُ حَمْلِهَا لِبَلَدٍ آخَرَ لِمَصْلَحَةٍ شَرْعِيَّةٍ كَاطْمِئْنَانِ الْقُلُوبِ بِالْجَزْمِ بِمَوْتِهِ ، وَقَدْ حُمِلَ رَأْسُ كَعْبِ بْنِ الْأَشْرَفِ مِنْ خَيْبَرَ إلَى الْمَدِينَةِ .( 6 / 17 )
