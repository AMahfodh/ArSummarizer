ع ش ( قَوْلُهُ : وَهُوَ مُتَّجَهٌ إلَخْ ) وِفَاقًا لِلنِّهَايَةِ ، وَالْمُغْنِي ( قَوْلُهُ : وَهُوَ ) أَيْ : مَا قَالَهُ الْأَذْرَعِيُّ إلَخْ وَكَذَا ضَمِيرُ وَيُؤَيِّدُهُ ( قَوْلُهُ : لِحِيَازَةِ فَضِيلَتِهِ ) إلَى قَوْلِهِ : وَفَارَقَ فِي الْمُغْنِي وَإِلَى الْفَرْعِ فِي النِّهَايَةِ بِمُخَالَفَةٍ يَسِيرَةٍ أُنَبِّهُ عَلَيْهَا ( قَوْلُهُ : وَأُخِذَ مِنْهُ ) أَيْ : مِنْ التَّعْلِيلِ ( قَوْلُهُ : إذَا كَانَ التَّأْخِيرُ أَفْضَلَ ) أَيْ : لِنَحْوِ إبْرَادٍ نِهَايَةٌ وَمُغْنِي اُنْظُرْ هَلْ يُسَنُّ الْإِبْرَادُ فِي حَقِّ الْمَرْأَةِ مَعَ أَنَّ صَلَاتَهَا فِي بَيْتِهَا أَفْضَلُ ؟ رَشِيدِيٌّ ( قَوْلُهُ : وَفَارَقَ ) أَيْ عَدَمُ الْمَنْعِ مِنْ تَعْجِيلِ الْمَكْتُوبَةِ ع ش وَسَمِّ ( قَوْلُ الْمَتْنِ : وَسُنَنٍ رَاتِبَةٍ ) الْمُرَادُ بِالرَّاتِبَةِ مَا لَهُ وَقْتٌ مُعَيَّنٌ سَوَاءٌ تَوَابِعُ الْفَرَائِضِ وَغَيْرُهَا وَقَدْ ذَكَرَ( 35 / 497 )
