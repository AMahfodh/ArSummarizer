وَأَقَرَّهُ الشِّهَابُ الرَّمْلِيُّ ( قَوْلُهُ : أَيْ مِنْ ابْتِدَائِهِ ) مُجَرَّدُ تَصْوِيرٍ وَالْمُرَادُ الْأَقَلُّ مِنْ الْأُجْرَةِ مُدَّةَ الِاسْتِخْدَامِ ( قَوْلُهُ : أَحَدُ ذَيْنِك ) أَيْ مُدَّةَ الِاسْتِخْدَامِ وَالْحَبْسِ ( قَوْلُهُ : مُطْلَقًا ) أَيْ أَقَلَّ أَوْ أَكْثَرَ ( قَوْلُهُ لِانْتِفَاءِ الضَّرَرِ عَلَى الزَّوْجَةِ إلَخْ ) أَيْ لِلُزُومِ السَّيِّدِ أَقَلُّ الْأَمْرَيْنِ مِنْ الْأُجْرَةِ وَالنَّفَقَةِ وَالْمَهْرِ ( قَوْلُهُ وَإِلَّا كَالْأَتُونِيّ ) عِبَارَةُ الْمِصْبَاحِ : وَالْأَتُونِيّ وَازِنُ رَسُولٍ .
