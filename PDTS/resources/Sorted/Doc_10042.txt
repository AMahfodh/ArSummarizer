( وَهَلْ تُفْسَخُ ) الْإِجَارَةُ ( إنْ اعْتَمَرَ ) أَجِيرُ الْحَجِّ ( عَنْ نَفْسِهِ ) مِنْ الْمِيقَاتِ وَحَجَّ عَنْ الْمَيِّتِ ( فِي ) الْعَامِ ( الْمُعَيَّنِ ) سَوَاءٌ أَحْرَمَ بِهِ مِنْ مَكَّةَ أَوْ الْمِيقَاتِ ؛ لِأَنَّهُ بِاعْتِمَارِهِ عَنْ نَفْسِهِ أَوْ لَا عَلِمَ أَنَّ سَفَرَهُ لَيْسَ لِلْمَيِّتِ ( أَوْ ) تُفْسَخُ فِي كُلِّ حَالٍ ( إلَّا أَنْ يَرْجِعَ ) الْأَجِيرُ ( لِلْمِيقَاتِ فَيُحْرِمُ ) مِنْهُ بِالْحَجِّ ( عَنْ الْمَيِّتِ فَيُجْزِيهِ ) ؛ لِأَنَّهُ لَمْ يَنْقُصْ حِينَئِذٍ فِي الْجَوَابِ ( تَأْوِيلَانِ ) مَحَلُّهُمَا فِي اعْتِمَارِهِ عَنْ نَفْسِهِ فِي عَامٍ مُعَيَّنٌ لَا يُمْكِنُهُ فِيهِ الرُّجُوعُ لِبَلَدِهِ وَالْعَوْدُ مِنْهُ بِحَيْثُ يُدْرِكُ الْحَجَّ فِي عَامِهِ ، وَيُمْكِنُهُ الرُّجُوعُ لِلْمِيقَاتِ فَقَطْ .
