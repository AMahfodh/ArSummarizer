( قَوْلُهُ : يَخْرُجُ بِقِيَاسِهِ ) أَيْ الْقِيَاسِ الْمُسْتَنِدِ إلَيْهِ ( قَوْلُهُ : فَيَكُونُ الْوَصْفُ عِلَّةً ) فِيهِ أَنَّهُ يَلْزَمُ الدَّوْرُ ؛ لِأَنَّ الْقِيَاسَ مُتَوَقِّفٌ عَلَى الْعِلَّةِ ، وَقَدْ تَوَقَّفَتْ عَلَيْهِ ( قَوْلُهُ : وَلَيْسَ كَذَلِكَ ) لِجَوَازِ أَنْ يَثْبُتَ بِقِيَاسٍ آخَرَ ( قَوْلُهُ : فَكَمَا فِي الْمُعْجِزَةِ ) أَيْ قِيَاسًا عَلَى الْمُعْجِزَةِ فَهُوَ تَنْظِيرٌ ( قَوْلُهُ : ، وَهُنَا مِنْ الْخَصْمِ ) وَيُمْكِنُ أَنْ يَنْتَفِيَ الْعَجْزُ عَنْ خَصْمٍ آخَرَ .( 5 / 224 )
