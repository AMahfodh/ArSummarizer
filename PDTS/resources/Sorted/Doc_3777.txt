( وَ ) وَيُسَنُّ ( أَنْ يَقُولَ : مُطِرْنَا بِفَضْلِ اللَّهِ وَرَحْمَتِهِ وَيَحْرُمُ ) قَوْلُ مُطِرْنَا ( بِنَوْءِ كَذَا ) لِخَبَرِ زَيْدِ بْنِ خَالِدٍ وَهُوَ فِي الصَّحِيحَيْنِ وَلِمُسْلِمٍ عَنْ أَبِي هُرَيْرَةَ مَرْفُوعًا { أَلَمْ تَرَوْا إلَى مَاذَا قَالَ رَبُّكُمْ ؟ قَالَ : مَا أَنْعَمْت عَلَى عِبَادِي مِنْ نِعْمَةٍ إلَّا أَصْبَحَ فَرِيقٌ مِنْهُمْ بِهَا كَافِرِينَ ، يُنَزِّلُ اللَّهُ الْغَيْثَ فَيَقُولُونَ : كَوْكَبُ كَذَا وَكَذَا } .
