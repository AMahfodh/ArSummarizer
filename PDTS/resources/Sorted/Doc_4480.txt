( قَوْلُهُ : وَإِلَّا تُعْلَمْ حَيَاتُهُ ) أَيْ بِأَنْ لَمْ يَسْتَهِلَّ وَلَمْ يَبْكِ نِهَايَةٌ وَمُغْنِي قَوْلُ الْمَتْنِ ( كَاخْتِلَاجٍ ) أَيْ أَوْ تَحَرُّكٍ نِهَايَةٌ وَمُغْنِي أَيْ وَلَوْ دُونَ أَرْبَعَةِ أَشْهُرٍ إنْ فُرِضَ ع ش ( قَوْلُهُ : اخْتِيَارِيٍّ ) بِمَاذَا يَتَمَيَّزُ عَنْ الِاضْطِرَارِيِّ بَصْرِيٌّ ( قَوْلُهُ : لِاحْتِمَالِ الْحَيَاةِ ) إلَى قَوْلِهِ وَمِنْ ثَمَّ فِي النِّهَايَةِ وَالْمُغْنِي ( قَوْلُهُ : عَلَيْهَا ) أَيْ الْحَيَاةِ أَيْ الدَّالَّةِ عَلَيْهَا .
