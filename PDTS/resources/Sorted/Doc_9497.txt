الْعِتْقُ ) فَلَا يَكُونُ لِزَيْدٍ فِي الْمِثَالِ شَيْءٌ ( أَوْ ) اجْتَمَعَ تَبَرُّعَاتٌ ( مُنْجَزَةٌ ) كَأَنْ أَعْتَقَ وَتَصَدَّقَ وَوَقَفَ ( قُدِّمَ الْأَوَّلُ ) مِنْهَا ( فَالْأَوَّلُ حَتَّى يَتِمَّ الثُّلُثُ ) وَيَتَوَقَّفُ مَا بَقِيَ عَنْ إجَازَةِ الْوَارِثِ ( فَإِنْ وُجِدَتْ دُفْعَةً ) بِضَمِّ الدَّالِ ( وَاتَّحَدَ الْجِنْسُ كَعِتْقِ عَبِيدٍ أَوْ إبْرَاءِ جَمْعٍ ) كَأَنْ قَالَ أَعْتَقْتُكُمْ أَوْ أَبْرَأْتُكُمْ ( أُقْرِعَ فِي الْعِتْقِ ) حَذَرًا مِنْ التَّشْقِيصِ فِي الْجَمِيعِ ( وَقُسِّطَ فِي غَيْرِهِ ) بِالْقِيمَةِ كَمَا تَقَدَّمَ ( وَإِنْ اخْتَلَفَ ) الْجِنْسُ ( وَتَصَرَّفَ وُكَلَاءُ فَإِنْ لَمْ يَكُنْ فِيهَا عِتْقٌ ) كَأَنْ تَصَدَّقَ وَاحِدٌ ، وَوَقَفَ آخَرُ ، وَأَبْرَأَ آخَرُ دُفْعَةً .
