( قَوْلُهُ وَذَلِكَ ) أَيْ : التَّعْلِيلُ الْمَذْكُورُ ( قَوْلُهُ بِمَا ذَكَرَ إلَخْ ) أَيْ بِعَدَمِ التَّجَاوُزِ عَنْ نِصْفِ السِّكَّةِ ( قَوْلُهُ وَقَوْلُهُ إلَخْ ) أَيْ : الْغَزِّيِّ ( قَوْلُهُ فَإِنَّهُ لَا يَلْزَمُ مِنْ مُجَاوَزَتِهِ إلَخْ ) أَيْ : وَلَا مِنْ عَدَمِهَا عَدَمُ الْإِبْطَالِ ( قَوْلُهُ لِمَالِ الْجَارِ ) كَأَنْ يُصِيبَ مَاؤُهُ جِدَارَ الْغَيْرِ بِحَيْثُ يَعِيبُهُ أَوْ يُتْلِفُهُ ا ه سم .( 21 / 10 )
