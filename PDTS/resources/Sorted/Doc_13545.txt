( وَيَتَّجِهُ ) أَنَّهُ لَا يُقْطَعُ الْمَسْرُوقُ مِنْهُ بِسَرِقَتِهِ مِنْ مَالِ سَارِقٍ أَوْ غَاصِبٍ مِنْ حِرْزٍ غَيْرِ الَّذِي بِهِ الْمَالُ الْمَسْرُوقُ أَوْ الْمَغْصُوبُ [ ( عَلَى تَوَهُّمٍ أَنَّهُ ) ؛ أَيْ : الْمَالَ الْمَسْرُوقَ أَوْ الْمَغْصُوبَ فِيهِ ] لِلشُّبْهَةِ ، وَهُوَ مُتَّجِهٌ ( أَوْ سَرَقَ مِنْ مَالِ مَنْ لَهُ عَلَيْهِ دَيْنٌ ) قُطِعَ ؛ لِأَنَّهُ لَا شُبْهَةَ لَهُ فِي الْمَالِ وَلَا فِي الْحِرْزِ ( لَا ) إنْ سَرَقَ مِنْ مَالِ مَدِينِهِ ( بِقَدْرِهِ ) ؛ أَيْ : الدَّيْنِ ( لِعَجْزِهِ ) عَنْ اسْتِخْلَاصِهِ بِحَاكِمٍ ؛ لِإِبَاحَةِ بَعْضِ الْعُلَمَاءِ الْأَخْذَ إذَنْ كَالْوَطْءِ فِي نِكَاحِ مُخْتَلَفٍ فِيهِ ، فَإِنْ سَرَقَ أَكْثَرَ مِنْ دَيْنِهِ ، وَبَلَغَ الزَّائِدُ نِصَابًا قُطِعَ ( أَوْ ) سَرَقَ ( عَيْنًا قُطِعَ بِهَا ) ؛ أَيْ : بِسَرِقَتِهَا ( فِي سَرِقَةٍ أُخْرَى ) مُتَقَدِّمَةٍ مِنْ حِرْزِهَا الْأَوَّلِ أَوْ غَيْرِهِ ؛ قُطِعَ ؛ لِأَنَّهُ لَمْ يَنْزَجِرْ بِالْقَطْعِ الْأَوَّلِ ؛ أَشْبَهَ مَا لَوْ سَرَقَ غَيْرَهَا ؛ بِخِلَافِ حَدِّ قَذْفٍ فَلَا يُعَادُ بِإِعَادَةِ الْقَذْفِ ؛ لِأَنَّ الْغَرَضَ إظْهَارُ كَذِبِهِ ، وَقَدْ ظَهَرَ( 18 / 280 )
