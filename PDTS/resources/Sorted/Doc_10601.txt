( وَ ) بِخِلَافِ ( خَمْرِ الْمُسْلِمِ وَخِنْزِيرِهِ ) بِأَنْ أَسْلَمَ وَهُمَا فِي يَدِهِ ( إذَا أَتْلَفَهُمَا ) مُسْلِمٌ أَوْ ذِمِّيٌّ فَلَا ضَمَانَ ( وَضَمِنَ ) الْمُتْلِفُ الْمُسْلِمُ قِيمَتهَا ؛ لِأَنَّ الْخَمْرَ فِي حَقِّنَا قِيَمِيٌّ حُكْمًا ( لَوْ كَانَا لِذِمِّيِّ ) وَالْمُتْلِفُ غَيْرُ الْإِمَامِ أَوْ مَأْمُورُهُ يَرَى ذَلِكَ عُقُوبَةً فَلَا يَضْمَنُ وَلَا الزِّقَّ خِلَافَ مُحَمَّدٍ مُجْتَبَى وَلَا ضَمَانَ فِي مَيْتَةٍ وَدَمٍ أَصْلًا ( بِخِلَافِ مَا لَوْ ) ( اشْتَرَاهَا ) أَيْ الْخَمْرَ ( مِنْهُ ) أَيْ الذِّمِّيِّ ( وَشَرِبَهَا فَلَا ضَمَانَ وَلَا ثَمَنَ ) ؛ لِأَنَّ فِعْلَهُ بِتَسْلِيطِ بَائِعِهِ ، بِخِلَافِ غَصْبِهَا مُجْتَبَى ، وَفِيهِ أَتْلَفَ ذِمِّيٌّ خَمْرَ ذِمِّيٍّ ثُمَّ أَسْلَمَا أَوْ أَحَدُهُمَا لَا شَيْءَ عَلَيْهِ إلَّا فِي رِوَايَةٍ عَلَيْهِ قِيمَةُ الْخَمْرِ
