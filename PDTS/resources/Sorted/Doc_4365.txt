رَحِمَهُ اللَّهُ ( قَوْلُهُ : فَالْحَقُّ قَوْلُ الْأَكْثَرِينَ ) أَيْ : وَلَا يَلْزَمُ الْوِلَاءُ لِمَا مَرَّ أَنَّهُ لَيْسَ مِنْ جِنْسِ الزَّمَنِ ( قَوْلُهُ : وَجْهَيْنِ ) أَوْجَهُهُمَا عَدَمُ الْوُجُوبِ شَرْحُ الرَّوْضِ ( قَوْلُهُ : وَيُوَافِقُ مَا نَقَلَاهُ إلَخْ ) إنْ أَرَادَ مَا نَقَلَاهُ عَلَى عُمُومِهِ فَرَّقَ بَيْنَ مَا هُوَ مِنْ الْجِنْسِ وَمَا لَيْسَ مِنْهُ وَإِنْ أُرِيدَ مَا نَقَلَاهُ بِاعْتِبَارِ التَّوَسُّطِ فَالْأَمْرُ ظَاهِرٌ( 7 / 219 )
