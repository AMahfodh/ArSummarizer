وَلَمَّا كَانَ كَلَامُهُ يُوهِمُ أَنَّ كُلَّ مَنْ طَلَّقَ يُمَتِّعُ وَكَانَ لَهُمْ مَسَائِلُ لَا مُتْعَةَ فِيهَا رَفَعَ ذَلِكَ التَّوَهُّمَ بِالتَّنْصِيصِ عَلَى بَعْضِ ذَلِكَ فَقَالَ : ( وَاَلَّتِي ) أَيْ الْمُطَلَّقَةُ الَّتِي ( لَمْ يَدْخُلْ بِهَا وَ ) الْحَالُ أَنَّهُ كَانَ ( قَدْ فَرَضَ لَهَا ) صَدَاقًا ( فَ ) إنَّهُ ( لَا مُتْعَةَ لَا ) ؛ لِأَنَّهَا قَدْ أَخَذَتْ نِصْفَ الصَّدَاقِ مَعَ بَقَاءِ سِلْعَتِهَا ، وَمَفْهُومُهُ أَنَّهَا إذَا لَمْ يَفْرِضْ لَهَا فَإِنَّ لَهَا الْمُتْعَةَ ، وَهُوَ كَذَلِكَ كَمَا قَدَّمْنَا ( وَ ) كَذَلِكَ ( لَا ) مُتْعَةَ ( لِلْمُخْتَلِعَةِ ) ؛ لِأَنَّهَا قَدْ دَفَعَتْ شَيْئًا مِنْ مَالِهَا لِأَجْلِ فِرَاقِهَا مِنْ زَوْجِهَا كَرَاهِيَةً فِيهِ ، فَلَا أَلَمَ عِنْدَهَا الْمَطْلُوبُ رَفْعُهُ بِإِعْطَاءِ الْمُتْعَةِ ، وَبَقِيَّةُ الْمَسَائِلِ الَّتِي لَا مُتْعَةَ فِيهَا مَذْكُورَةٌ فِي الْأَصْلِ .
