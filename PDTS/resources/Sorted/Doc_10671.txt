( فَإِنْ أُسِرَ ) الْمُسْتَأْمَنُ ( أَوْ ظَهَرَ عَلَيْهِمْ ) أَيْ أَهْلِ الْحَرْبِ ( فَقُتِلَ سَقَطَ دَيْنٌ ) كَانَ ( لَهُ عَلَى مَعْصُومٍ ) مُسْلِمٍ أَوْ ذِمِّيٍّ ؛ لِأَنَّ إثْبَاتَ الْيَدِ عَلَيْهِ بِوَاسِطَةِ الْمُطَالَبَةِ وَقَدْ سَقَطَتْ وَيَدُ مَنْ عَلَيْهِ أَسْبَقُ مِنْ يَدِ الْعَامَّةِ فَيَخْتَصُّ بِهِ فَيَسْقُطُ ( وَأُفِيءَ ) أَيْ صَارَ فَيْئًا ( وَدِيعَةً لَهُ عِنْدَهُ ) أَيْ مَعْصُومٍ ؛ لِأَنَّهَا فِي يَدِهِ تَقْدِيرًا لِأَنَّ يَدَ الْمُودَعِ كَيَدِهِ فَيَصِيرُ فَيْئًا تَبَعًا لِنَفْسِهِ وَعَنْ أَبِي يُوسُفَ أَنَّ الْوَدِيعَةَ تَصِيرُ لِلْمُودَعِ ؛ لِأَنَّ يَدَهُ بِهَا أَسْبَقُ فَهُوَ بِهَا أَحَقُّ ( وَأَخَذَ الْمُرْتَهِنُ رَهْنَهُ بِدَيْنِهِ عِنْدَ أَبِي يُوسُفَ وَيُبَاعُ وَيُوفَى بِثَمَنِهِ الدَّيْنُ وَالْفَاضِلُ لِبَيْتِ الْمَالِ عِنْدَ مُحَمَّدٍ ) ذَكَرَهُ الزَّيْلَعِيُّ .( 3 / 380 )
