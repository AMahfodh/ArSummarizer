وَبِالْجُمْلَةِ فَمَا فِي الرَّوْضَةِ وَأَصْلِهَا أَوْجَهُ مِمَّا فِي الْكِتَابِ وَإِنْ قَالَ الشَّارِحُ : إنَّ الْقَبْضَ مُتَضَمِّنٌ لِلْإِقْبَاضِ ( وَلَوْ عَلَّقَ ) طَلَاقَهَا ( بِإِعْطَاءِ ) نَحْوِ ( عَبْدٍ ) كَثَوْبٍ ( وَوَصَفَهُ بِصِفَةِ سَلَمٍ ) وَهِيَ الَّتِي يَصِحُّ بِهَا ثُبُوتُهُ فِي الذِّمَّةِ أَوْ وَصَفَهُ بِصِفَةٍ دُونَ صِفَةِ السَّلَمِ بِأَنْ لَمْ يَسْتَوْفِهَا ( فَأَعْطَتْهُ ) عَبْدًا ( لَا بِالصِّفَةِ ) الَّتِي وَصَفَهَا ( لَمْ تَطْلُقْ ) لِعَدَمِ وُجُودِ الصِّفَةِ ( أَوْ ) أَعْطَتْهُ عَبْدًا ( بِهَا( 13 / 228 )
