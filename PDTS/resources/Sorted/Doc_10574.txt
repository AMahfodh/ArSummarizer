( قَوْلُهُ : بِيَدِ شَخْصٍ يَدَّعِيهِ ) أَيْ : وَلَمْ يُقِمْ بَيِّنَةً ؛ لِأَنَّهُ لَوْ أَقَامَ لَكَانَ هُوَ الْمُرَجَّحَ عَلَى غَيْرِهِ ( قَوْلُهُ : فَإِنَّ الدَّارَ تَبْقَى فِي يَدِ الَّتِي هِيَ فِي يَدِهِ ) أَيْ مِلْكًا ( قَوْلُهُ : الشَّارِحَانِ ) بَهْرَامُ وَالْبِسَاطِيُّ ( قَوْلُهُ : يَتَكَرَّرُ إلَخْ ) الْمُنَاسِبُ يُنَافِي ، وَقَوْلُهُ : وَقَوْلُ الْبِسَاطِيِّ أَيْ زِيَادَةً عَلَى مَا تَقَدَّمَ لَهُ مَعَ بَهْرَامَ ذَكَرَهُ جَوَابًا عَنْ التَّكْرَارِ أَيْ فَقَدْ سَلَّمَ التَّكْرَارَ وَأَجَابَ بِأَنَّهُ ذَكَرَهُ لِيُرَتِّبَ عَلَيْهِ خَبَرَ قَوْلِهِ ، وَقَوْلُهُ لِعَدَمِ ظَاهِرُهُ أَنَّهُ عِلَّةٌ لِلْبُعْدِ مَعَ أَنَّهُ لَيْسَ عِلَّةً لَهُ بَلْ إنَّمَا يُفِيدُ عَدَمَ التَّكْرَارِ مِنْ أَصْلِهِ ؛ لِأَنَّهُ إنَّمَا يُفِيدُ الْمُنَافَاةَ فَلَا يُعْقَلُ تَكْرَارٌ .( 22 / 219 )
