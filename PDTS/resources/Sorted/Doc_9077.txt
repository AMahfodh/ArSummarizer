( وَ ) رُدَّ الْمَالُ الْمُخَالَعُ بِهِ ( بَ ) تَبَيُّنِ ( كَوْنِهَا ) أَيْ الزَّوْجَةِ الْمُخَالَعَةِ ( بَائِنًا ) مِنْ مُخَالِعِهَا وَقْتَ خُلْعِهِ ؛ لِأَنَّهُ لَمْ يُصَادِفْ مَحَلًّا ( لَا ) يُرَدُّ الْمَالُ الْمُخَالَعُ بِهِ إنْ تَبَيَّنَ بَعْدَ الْخُلْعِ أَنَّهَا كَانَتْ مُطَلَّقَةً طَلْقَةً ( رَجْعِيَّةً ) لَمْ تَنْقَضِ عِدَّتُهَا ؛ لِأَنَّهَا زَوْجَةٌ مَمْلُوكَةُ الْعِصْمَةِ فَيَلْحَقُهَا الطَّلَاقُ ( أَوْ لِكَوْنِهِ ) أَيْ النِّكَاحِ فَاسِدًا مَعًا عَلَى فَسَادِهِ ( يُفْسَخُ ) بِضَمِّ التَّحْتِيَّةِ ( بِلَا طَلَاقٍ ) كَنِكَاحِ خَامِسَةٍ وَمَحْرَمٍ مِنْ نَسَبٍ ، أَوْ رَضَاعٍ ، أَوْ صِهْرٍ فَيُرَدُّ الْمَالُ الْمُخَالَعُ بِهِ لِعَدَمِ مُصَادَفَةِ خُلْعِهِ مَحَلًّا .
