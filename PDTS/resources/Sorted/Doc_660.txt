( وَلِمَنْ لَزِمَتْ غَيْرَهُ فِطْرَتُهُ ) كَزَوْجَةٍ وَوَلَدٍ مُعْسِرٍ ( طَلَبُهُ بِإِخْرَاجِهَا ) ، أَيْ : الْفِطْرَةِ عَنْهُ كَالنَّفَقَةِ ، لِأَنَّهَا تَابِعَةٌ لَهَا ، ( وَ ) لَهُ ( أَنْ يُخْرِجَهَا ) ، أَيْ : الْفِطْرَةَ ( حُرٌّ ) مُكَلَّفٌ وَجَبَتْ فِطْرَتُهُ عَلَى غَيْرِهِ ( عَنْ نَفْسِهِ ) مِمَّا اقْتَرَضَهُ أَوْ اتَّهَبَهُ .
