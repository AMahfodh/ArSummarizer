قَالَ الْأَثْرَمُ : قُلْت لِأَبِي عَبْدِ اللَّهِ فَحَدِيثُ عُمَرَ بْنِ أَبِي سَلَمَةَ حِينَ { زَوَّجَ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أُمَّهُ أُمَّ سَلَمَةَ أَلَيْسَ كَانَ صَغِيرًا ؟ قَالَ : وَمَنْ يَقُول كَانَ صَغِيرًا ؟ أَلَيْسَ فِيهِ بَيَانٌ } وَلِأَنَّهُ عَدْلٌ مِنْ عَصَبَتِهَا فَثَبَتَتْ لَهُ وِلَايَةُ تَزْوِيجِهَا كَأَخِيهَا فَأَخٌ لِأَبَوَيْنِ ( فَ ) أَخٌ ( لِأَبٍ ) لِأَنَّ وِلَايَةَ النِّكَاحِ حَقٌّ يُسْتَفَادُ بِالتَّعْصِيبِ فَقُدِّمَ فِيهِ الْأَخُ لِأَبَوَيْنِ كَالْمِيرَاثِ وَكَاسْتِحْقَاقِ الْمِيرَاثِ بِالْوَلَاءِ ( فَابْنُ أَخٍ لِأَبَوَيْنِ فَ ) ابْنُ أَخٍ ( لِأَبٍ وَإِنْ سَفَلَا ) أَيْ ابْنُ الْأَخِ لِأَبَوَيْنِ أَوْ لِأَبٍ .
