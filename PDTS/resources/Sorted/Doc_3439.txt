( وَإِنْ أَمَرَهُ بِهِ ) - أَيْ : بِالْخِتَانِ - ( وَلِيُّ أَمْرٍ فِي حَرٍّ أَوْ بَرْدٍ أَوْ مَرَضٍ يَخَافُ مِنْهُ مَوْتٌ ) فَتَلِفَ بِسَبَبِهِ ضَمِنَهُ لِأَنَّهُ لَيْسَ لَهُ ( أَوْ ) أَمَرَهُ وَلِيُّ أَمْرٍ بِهِ ( وَزَعَمَ الْأَطِبَّاءُ أَنَّهُ يَتْلَفُ ، أَوْ ظَنَّ تَلَفَهُ ) فَتَلِفَ : ( ضَمِنَهُ ) وَلِيُّ الْأَمْرِ ، لِأَنَّهُ لَيْسَ لَهُ .( 1 / 203 )
