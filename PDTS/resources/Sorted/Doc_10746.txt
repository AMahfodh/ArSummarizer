فَاتِّجَاهُ الْمُصَنِّفُ مَحْمُولٌ عَلَى هَذَا ، وَهُوَ حَسَنٌ ( وَلَوْ ) كَانَ الْمُخْبِرُ عَدْلًا ( ظَاهِرًا ) أَيْ : مَسْتُورَ الْحَالِ ، ( أَوْ ) كَانَ ( أُنْثَى أَوْ قِنًّا أَوْ أَعْمَى ) لِأَنَّ لِلْأَعْمَى طَرِيقًا إلَى الْعِلْمِ بِذَلِكَ بِالْخَبَرِ مِنْ عَدْلٍ ، أَوْ الْحِسِّ بِحَاسَّةِ غَيْرِ الْبَصَرِ ( بِنَجَاسَةِ شَيْءٍ ) مُتَعَلِّقٌ بِأَخْبَرَ ( وَلَوْ ) كَانَ ذَلِكَ الشَّيْءُ ( مُبْهَمًا كَأَحَدِ هَذَيْنِ الثَّوْبَيْنِ ) أَوْ الْإِنَاءَيْنِ ( وَعَيَّنَ ) الْمُخْبِرُ ( السَّبَبَ ) أَيْ : سَبَبَ مَا أَخْبَرَ بِهِ مِنْ نَجَاسَةِ الْمَاءِ وَنَحْوِهِ ، وَالْمُخْبِرُ ( مُخَالِفٌ ) لِمَذْهَبِ مَنْ أَخْبَرَهُ أَوْ فَقِيهٌ مُوَافِقٌ كَمَا نُقِلَ مِنْ إمْلَاءِ التَّقِيِّ الْفَتُوحِيِّ ( قَبِلَ لُزُومًا ) لِأَنَّهُ خَبَرٌ دِينِيٌّ ، كَالْقِبْلَةِ وَهِلَالِ رَمَضَانِ .
