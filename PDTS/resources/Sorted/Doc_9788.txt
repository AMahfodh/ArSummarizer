( وَإِنْ كَانَ ) الْمُؤَجِّرُ ( قَبَضَ الْأُجْرَةَ ) مِنْ الْمُسْتَأْجِرِ ، ثُمَّ فَسَخَ ( رَجَعَ عَلَيْهِ ) أَيْ عَلَى الْمُؤَجِّرِ ( مُسْتَأْجِرٌ بِالْقِسْطِ مِنْ الْمُسَمَّى مِنْ الْأَجْرِ فِي الْمُسْتَقْبَلِ ) الْبَاقِي مِنْ مُدَّةِ الْإِجَارَةِ ( وَ ) رَجَعَ عَلَيْهِ أَيْضًا ( بِمَا زَادَ مِنْ أُجْرَةِ الْمِثْلِ فِي الْمَاضِي إنْ كَانَ هُوَ الْمَغْبُونَ وَإِنْ كَانَ ) الْمَغْبُونُ هُوَ ( الْمُؤَجِّرَ ، فَ ) إنَّهُ يَرْجِعُ ( بِمَا نَقَصَ عَنْ أُجْرَةِ الْمِثْلِ فِي الْمَاضِي ) لِمَا تَقَدَّمَ ( وَالْغَبْنُ مُحَرَّمٌ ) لِأَنَّهُ تَغْرِيرٌ وَغِشٌّ ( وَالْعَقْدُ صَحِيحٌ فِيهِنَّ ) أَيْ فِي الصُّوَرِ الثَّلَاثِ لِمَا تَقَدَّمَ فِي تَلَقِّي الرُّكْبَانِ .( 9 / 90 )
