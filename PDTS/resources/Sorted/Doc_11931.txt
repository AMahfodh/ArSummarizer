( وَفِي اخْتِلَاعِ أَمَةٍ ) ، وَلَوْ سَفِيهَةً ( وَ ) قَدْ ( أَذِنَا سَيِّدُهَا ) فِيهِ ( يَكُونُ ) مُسَمَّاهُ حَيْثُ عَيْنُهُ مِنْ أَعْيَانِ مَالِهِ ( مِمَّا عَيَّنَا وَ ) يَكُونُ مِنْ ( كَسْبِ هَذِهِ ) الْأَمَةِ ( وَمَا تَتَّجِرُ فِيهِ ) لِسَيِّدِهَا ( مُسَمَّى سَيِّدٍ ) حَيْثُ ( يُقَدَّرُ ) أَيْ : يَقْدِرُهُ ( دَيْنًا وَمَهْرُ الْمِثْلِ لِلْإِطْلَاقِ ) أَيْ : حَيْثُ يُطْلَقُ فَقَوْلُهُ : مُسَمًّى سَيِّدِ إلَى آخَرِهِ مَعْطُوفٌ عَلَى اسْمِ يَكُونُ وَمَا قَبْلَهُ مَعْطُوفٌ عَلَى خَبَرِهَا ، فَإِنْ لَمْ تَكُنْ مُكْتَسِبَةً ، وَلَا مَأْذُونًا لَهَا فِي التِّجَارَةِ بَقِيَ ذَلِكَ فِي ذِمَّتِهَا إلَى عِتْقِهَا وَيَسَارِهَا ( وَمَا تَزِدْ ) ه الْأَمَةُ عَلَى الْمُقَدَّرِ فِي التَّقْدِيرِ وَعَلَى مَهْرِ الْمِثْلِ فِي الْإِطْلَاقِ ( تَغْرَمْهُ بِالْإِعْتَاقِ ) أَيْ : بِسَبَبِهِ يَعْنِي بَعْدَهُ وَسَيَأْتِي حُكْمُ مَا لَوْ جَرَى الْخُلْعُ بِرَقَبَتِهَا .
