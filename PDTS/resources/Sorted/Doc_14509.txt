السِّنُّ أَخْضَرَ وَنَحْوُهُ كَمَا لَوْ عَادَ مَائِلًا عَنْ مَحَلِّهِ ( فَ ) عَلَى جَانٍ ( حُكُومَةٌ ) لِحُدُوثِ النَّقْصِ بِفِعْلِهِ ؛ فَضَمِنَهُ ، وَيَأْتِي ( ثُمَّ إنْ كَانَ ) الْمَجْنِيُّ عَلَيْهِ ( أَخَذَ دِيَةَ ) مَا أَذْهَبَهُ قَبْلَ أَنْ يَعُودَ ثُمَّ عَادَ ( رَدَّهَا ) إلَى مَنْ أَخَذَهَا مِنْهُ ، ( أَوْ ) كَانَ الْمَجْنِيُّ عَلَيْهِ ( اقْتَصَّ ) مِنْ جَانٍ نَظِيرَ مَا أَذْهَبَهُ مِنْهُ ، ثُمَّ عَادَ ( فَلِجَانٍ الدِّيَةُ ) ؛ لِتَبَيُّنِ أَنَّهُ اسْتَوْفَى ذَلِكَ بِلَا حَقٍّ ، وَلَا قِصَاصٍ ؛ لِلشُّبْهَةِ ( وَيَرُدُّهَا ) أَيْ : وَيَرُدُّ الْجَانِي مَا أَخَذَهُ دِيَةً عَمَّا اقْتَصَّ مِنْهُ ( وَإِنْ عَادَ ) مَا أَخَذَ الْجَانِي دِيَتَهُ بِسَبَبِ عَوْدِ مَا جُنِيَ عَلَيْهِ كَمَا قُلْنَا فِي الْمَجْنِيِّ عَلَيْهِ ؛ لِأَنَّا تَبَيَّنَّا أَنَّهُ قَدْ أَخَذَ ذَلِكَ بِغَيْرِ حَقٍّ ، فَكَانَ عَلَيْهِ رَدُّهُ ( وَمَنْ قَلَعَ سِنَّهُ أَوْ ظُفُرَهُ تَعَدِّيًا ، أَوْ قَطَعَ طَرَفَهُ كَمَارِنٍ وَأُذُنٍ فَرَدَّهُ فَالْتَحَمَ ؛ فَلَهُ ) أَيْ : الْمَجْنِيِّ عَلَيْهِ ( أَرْشُ نَقْصِهِ ) أَيْ : حُكُومَةٌ ؛ لِأَنَّهَا أَرْشُ كُلِّ نَقْصٍ بِجِنَايَةٍ لَا مُقَدَّرَ فِيهَا ( وَإِنْ قَلَعَهُ ) أَيْ : مَا قُطِعَ ، ثُمَّ الْتَحَمَ ( قَالِعٌ بَعْدَ ذَلِكَ ؛ فَعَلَيْهِ دِيَةٌ ، لَا الْقِصَاصُ ) ؛ لِأَنَّهُ لَا يُقَادُ بِهِ الصَّحِيحُ بِأَصْلِ الْخِلْقَةِ ؛ لِنَقْصِهِ بِالْقَطْعِ الْأَوَّلِ .
