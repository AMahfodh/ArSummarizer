( وَمَا اسْتَدَانَ ) شَرِيكٌ ( بِدُونِ إذْنِ ) شَرِيكِهِ بِاقْتِرَاضٍ أَوْ شِرَاءِ بِضَاعَةٍ ضَمَّهَا إلَى مَالِ الشَّرِكَةِ أَوْ بِثَمَنٍ نَسِيئَةً لَيْسَ عِنْدَهُ مِنْ جِنْسِهِ غَيْرُ النَّقْدَيْنِ ( فَعَلَيْهِ ) أَيْ : الْمُسْتَدِينِ وَحْدَهُ الْمُطَالَبَةُ بِمَا اسْتَدَانَهُ ( وَرِبْحُهُ لَهُ ) ؛ لِأَنَّهُ لَمْ يَقَعْ لِلشَّرِكَةِ ( وَإِنْ أَخَّرَ ) أَحَدُهُمَا ( حَقَّهُ مِنْ دَيْنٍ ) ( جَازَ ) لِصِحَّةِ انْفِرَادِهِ بِإِسْقَاطِ حَقِّهِ مِنْ الطَّلَبِ بِهِ ، كَالْإِبْرَاءِ بِخِلَافِ حَقِّ شَرِيكِهِ ( وَلَهُ ) أَيْ : الَّذِي أَخَّرَ حَقَّهُ مِنْ الدَّيْنِ ( مُشَارَكَةُ شَرِيكِهِ ) الَّذِي لَمْ يُؤَخِّرْ ( فِيمَا يَقْبِضُهُ مِنْ الدَّيْنِ مِمَّا لَمْ يُؤَخِّرْ ) لِاشْتِرَاكِهِ بَيْنَهُمَا ( وَإِنْ تَقَاسَمَا دَيْنًا فِي ذِمَّةِ ) شَخْصٍ ( أَوْ أَكْثَرَ ) ( لَمْ يَصِحَّ ) نَصًّا ؛ لِأَنَّ الذِّمَمَ لَا تَتَكَافَأُ وَلَا تَتَعَادَلُ ، وَالْقِسْمَةُ تَقْتَضِيهِمَا ؛ لِأَنَّهَا بِغَيْرِ تَعْدِيلٍ بِمَنْزِلَةِ الْبَيْعِ وَبَيْعُ الدَّيْنِ غَيْرُ جَائِزٍ .
