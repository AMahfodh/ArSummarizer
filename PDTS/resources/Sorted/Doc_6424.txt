( وَ ) كَأَثَرِ دَمٍ فِي ( مَوْضِعِ ) كَ ( حِجَامَةٍ ) وَفَصَادَةٍ ، وَنَعَتَ الْمَوْضِعِ بِجُمْلَةِ ( مُسِحَ ) بِضَمٍّ فَكَسْرٍ أَيْ الْمَوْضِعُ مِنْ عَيْنِ الدَّمِ فَيُعْفَى عَنْهُ حَتَّى يَبْرَأَ ( فَإِذَا بَرِئَ ) الْمَوْضِعُ ( غَسَلَ ) الْمُكَلَّفُ الْمَوْضِعَ اسْتِنَانًا أَوْ وُجُوبًا إنْ ذَكَرَ وَقَدَرَ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَغْسِلْهُ بَعْدَهُ وَصَلَّى ( أَعَادَ ) الْمُكَلَّفُ الصَّلَاةَ الَّتِي صَلَّاهَا قَبْلَ الْغُسْلِ وَبَعْدَ الْبُرْءِ ( فِي الْوَقْتِ ) الظُّهْرَيْنِ( 1 / 113 )
