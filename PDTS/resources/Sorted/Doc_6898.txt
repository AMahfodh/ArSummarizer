سَقَطَ مِنْ نُسَخِ الشَّارِحِ مِنْ الْكَتَبَةِ ( قَوْلُهُ : لَمْ يُعْتَدَّ بِهِ ) لَعَلَّ الْمُرَادَ لَمْ يُعْتَدَّ بِطَلَبِهِ فَلَا يَجِبُ عَلَى الْإِمَامِ إجَابَتُهُ فِي ذَلِكَ الطَّلَبِ قَوْلُهُ : أَوْ إلَى دُونَ مَسَافَةِ الْقَصْرِ ) أَيْ مِنْ أَحَدِهِمَا ( قَوْلُ الْمَتْنِ بَلْ مَعَ زَوْجٍ ) أَيْ كَأَنْ كَانَتْ أَمَةً أَوْ حُرَّةً قَبْلَ الدُّخُولِ أَوْ طَرَأَ التَّزْوِيجُ بَعْدَ الزِّنَى فَلَا يُقَالُ إنَّ مَنْ لَهَا زَوْجٌ مُحْصَنَةٌ ( قَوْلُهُ : إلَّا مَعَ مَحْرَمٍ أَوْ سَيِّدٍ ) أَيْ أَوْ نَحْوِهِمَا( 25 / 498 )
