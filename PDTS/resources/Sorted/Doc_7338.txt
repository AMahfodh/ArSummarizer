( لَا إنْ ادَّعَى ) مُدَّعٍ ( عَلَى غَائِبٍ فَضَمِنَ ) ضَامِنٌ ذَلِكَ الْغَائِبَ فِيمَا ادَّعَى بِهِ عَلَيْهِ ( ثُمَّ أَنْكَرَ ) الْغَائِبُ عِنْدَ حُضُورِهِ ( أَوْ قَالَ ) شَخْصٌ ( لِمُدَّعٍ عَلَى مُنْكِرٍ إنْ لَمْ آتِك بِهِ لِغَدٍ فَأَنَا ضَامِنٌ وَلَمْ يَأْتِ بِهِ ) فَلَا يَلْزَمُهُ الضَّمَانُ ؛ لِأَنَّهُ وَعْدٌ وَهُوَ لَا يُقْضَى بِهِ ( إنْ لَمْ يَثْبُتْ حَقُّهُ بِبَيِّنَةٍ ) فِي الْمَسْأَلَتَيْنِ فَإِنْ ثَبَتَ بِهَا لَزِمَ وَفَرْضُ الْمَسْأَلَةِ أَنَّهُ لَمْ يَأْتِ بِهِ فَإِنْ أَتَى بِهِ لَمْ يَلْزَمْ الضَّامِنَ شَيْءٌ مَعَ الثُّبُوتِ بِالْبَيِّنَةِ
