( وَإِنْ تَعَلَّقَ ) الدَّيْنُ ( بِذِمَّتِهِ ) ؛ أَيْ : الْعَبْدِ ؛ ( كَإِقْرَارِهِ بِمَالٍ ) ؛ أَيْ : بِأَنْ أَقَرَّ بِهِ مَأْذُونٌ ، وَلَمْ يُصَدِّقْهُ سَيِّدُهُ ، فَمَلَكَهُ رَبُّ الدَّيْنِ ؛ سَقَطَ الدَّيْنُ ( عَلَى مَا يَأْتِي فِي الْإِقْرَارِ ) مُفَصَّلًا ، ( أَوْ غَرَّ ) الْعَبْدَ ( فِي نِكَاحٍ بِأَمَةٍ ) ؛ بِأَنْ تَزَوَّجَ امْرَأَةً عَلَى أَنَّهَا حُرَّةٌ ، فَبَانَتْ أَمَةً بَعْدَ الدُّخُولِ ؛ تَعَلَّقَ الْمَهْرُ بِذِمَّتِهِ ؛ لِاسْتِقْرَارِهِ بِالدُّخُولِ ، وَوَلَدُهُ حُرٌّ ؛ لِأَنَّهَا وَطِئَهَا مُعْتَقِدٌ الْحُرِّيَّةَ يَفْدِيه الْعَبْدُ بِقِيمَتِهِ يَوْمَ الْوِلَادَةِ ؛ لِتَعَلُّقِ الْفِدَاءِ بِذِمَّتِهِ ، وَلِلْعَبْدِ الرُّجُوعُ عَلَى مَنْ غَرَّهُ ، فَلَوْ كَانَ الْغَارُّ سَيِّدَ الْأَمَةِ فَمِلْكُ الْعَبْدِ بِعِوَضٍ أَوَّلًا ؛ سَقَطَ الْمَهْرُ وَالْفِدَاءُ ، أَوْ كَانَ الْغَارُّ مَنْ اسْتَدَانَ مِنْهُ الْعَبْدُ الْمَهْرَ وَالْفِدَاءَ ، وَأَدَّاهُ ، ( فَمَلَكَهُ ) رَبُّ الدَّيْنِ ( بِعِوَضٍ ) ؛ أَيْ : بِشِرَاءٍ ( أَوْ ) مَلَكَهُ ( لَا ) بِعِوَضٍ ؛ كَهِبَةٍ ؛ سَقَطَ الدَّيْنُ ؛ لِأَنَّ السَّيِّدَ لَا يَثْبُتُ لَهُ الدَّيْنُ بِذِمَّةِ عَبْدِهِ ، ( أَوْ ) مَلَكَ رَبُّ الدَّيْنِ ( مَنْ تَعَلَّقَ ) دِينُهُ ( بِرَقَبَتِهِ ) ، وَكَانَ مِلْكُهُ لَهُ ( بِلَا عِوَضٍ ) ؛ بِأَنْ وَرِثَهُ أَوْ وُهِبَ لَهُ ؛ ( سَقَطَ ) الدَّيْنُ ؛ لِأَنَّهُ لَا يَدَ لِلرَّقَبَةِ ، يَتَحَوَّلُ الدَّيْنُ إلَيْهِ .( 9 / 292 )
