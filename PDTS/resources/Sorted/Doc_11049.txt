( فَرْعٌ ) لَوْ ( سَاقَى شَرِيكَهُ الْمُنَاصِفَ ) لَهُ فِي الشَّرِكَةِ ( عَلَى الثُّلُثَيْنِ ) فَأَكْثَرَ ( صَحَّ ) وَقَدْ شَرَطَ لَهُ فِي مَسْأَلَةِ الثُّلُثَيْنِ ثُلُثَ ثَمَرَتِهِ فَكَأَنَّهُ سَاقَاهُ عَلَى نِصْفِهِ بِالثُّلُثِ ( أَوْ ) سَاقَاهُ عَلَى ( النِّصْفِ فَمَا دُونَهُ فَلَا ) يَصِحُّ لِخُلُوِّ الْمُسَاقَاةِ عَنْ الْعِوَضِ بَلْ شَرَطَ عَلَيْهِ فِي مَسْأَلَةِ مَا دُونَ النِّصْفِ أَنْ يَتْرُكَ بَعْضَ ثَمَرَتِهِ أَيْضًا ( وَلَا أُجْرَةَ لَهُ ) إذَا عَمِلَ ؛ لِأَنَّهُ لَمْ يَطْمَعْ فِي شَيْءٍ ( أَوْ ) سَاقَاهُ ( عَلَى الْكُلِّ بَطَلَ وَ ) لَكِنْ ( لَهُ ) عَلَيْهِ ( الْأُجْرَةُ ) لِأَنَّهُ عَمِلَ طَامِعًا ، وَقَيَّدَهُ الْغَزَالِيُّ - كَإِمَامِهِ - تَفَقُّهًا بِمَا إذَا لَمْ يَعْلَمْ الْفَسَادَ ، وَالظَّاهِرُ صِحَّةُ مُسَاقَاةِ أَحَدِ الشَّرِيكَيْنِ عَلَى نَصِيبِهِ أَجَنِيبًا .
