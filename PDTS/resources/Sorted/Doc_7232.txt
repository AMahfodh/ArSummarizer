( قَوْلُهُ : وَمَرَّ ) أَيْ قُبَيْلَ قَوْلِ الْمَتْنِ وَلَا يَضُرُّ تَفَاوُتُ كِبَرٍ إلَخْ ( قَوْلُهُ : ثُمَّ شُلَّ ) بِبِنَاءِ الْمَفْعُولِ ( قَوْلُهُ : وَقَدْ يَشْكُلُ ) أَيْ مَا مَرَّ ( قَوْلُهُ : بِمَا يَأْتِي ) أَيْ فِي آخِرِ الْفَصْلِ ( قَوْلُهُ : ذَاتَ الْكَفَّيْنِ ) أَيْ أَنْفُسِهِمَا ( قَوْلُهُ : وَقَدْ زَالَ ) أَيْ الْمَانِعُ ، وَلَوْ أَنَّثَ كَانَ أَنْسَبَ ( قَوْلُهُ : هُنَا ) أَيْ فِي مَسْأَلَةِ جِنَايَةِ السَّلِيمِ عَلَى يَدٍ شَلَّاءَ ( قَوْلُهُ : بَعْدَهَا ) أَيْ الْجِنَايَةِ .
