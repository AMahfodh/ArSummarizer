( وَلَوْ مَاتَ مَا ) أَيْ حَيَوَانٌ ( اُسْتُثْنِيَ مِنْهُ ) شَيْءٌ ( مُعَيَّنٌ ) مِنْ جِلْدٍ وَسَاقِطٍ أَوْ أَرْطَالٍ ( ضَمِنَ الْمُشْتَرِي ) لِلْبَائِعِ مِنْ الْمُعَيَّنِ ( جِلْدًا وَسَاقِطًا ) ؛ لِأَنَّهُ لَا يُجْبَرُ عَلَى الذَّبْحِ فِيهِمَا إذْ لَهُ دَفْعُ مِثْلِهِمَا فَكَأَنَّهُمَا فِي ذِمَّتِهِ ( لَا لَحْمًا ) وَهُوَ مَا عَبَّرَ عَنْهُ قَبْلُ بِالْأَرْطَالِ فَلَا يَضْمَنُهُ كَاسْتِثْنَاءِ الْجُزْءِ لِتَفْرِيطِ الْبَائِعِ فِي طَلَبِهِ بِالذَّبْحِ وَجَبْرِهِ عَلَيْهِ .
