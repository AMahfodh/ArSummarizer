( وَالْحَلَالُ مِنْهَا ) أَرْبَعَةُ أَنْوَاعٍ : الْأَوَّلُ ( نَبِيذُ التَّمْرِ وَالزَّبِيبِ إنْ طُبِخَ أَدْنَى طَبْخَةً ) يَحِلُّ شُرْبُهُ ( وَإِنْ اشْتَدَّ ) وَهَذَا ( إذَا شَرِبَ ) مِنْهُ ( بِلَا لَهْوٍ وَطَرَبٍ ) فَلَوْ شَرِبَ لِلَّهْوِ فَقَلِيلُهُ وَكَثِيرُهُ حَرَامٍ ( وَمَا لَمْ يُسْكِرْ ) فَلَوْ شَرِبَ مَا يَغْلِبُ عَلَى ظَنِّهِ أَنَّهُ مُسْكِرٌ فَيَحْرُمُ ، لِأَنَّ السُّكْرَ حَرَامٌ فِي كُلِّ شَرَابٍ .
