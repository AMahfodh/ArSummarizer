قَالَ الْبُهُوتِيُّ : قُلْت : وَكَذَا كِتَابَتُهُ ( وَإِنْ نَجَّزَ ) طَلْقَةً فَقَالَ : أَنْتِ طَالِقٌ ؛ طَلْقَةً إلَّا أَنْ تَشَائِي أَوْ يَشَاءَ زَيْدٌ ثَلَاثًا ( أَوْ عَلَّقَ طَلْقَةً ) فَقَالَ : إنْ قُمْتِ فَأَنْتِ طَالِقٌ طَلْقَةً إنْ ( تَشَأْ هِيَ ؛ أَوْ يَشَأْ زَيْدٌ ثَلَاثًا ، أَوْ ) نَجَّزَ أَوْ عَلَّقَ ( ثَلَاثًا ) بِأَنْ قَالَ : أَنْتِ طَالِقٌ ثَلَاثًا ، أَوْ إنْ قُمْتِ فَأَنْتِ طَالِقٌ ثَلَاثًا ( إلَّا أَنْ تَشَاءَ ) وَاحِدَةً ( أَوْ ) إلَّا أَنْ ( يَشَاءَ ) زَيْدٌ ( وَاحِدَةً ، فَشَاءَتْ ) هِيَ ( أَوْ شَاءَ ) زَيْدٌ ( ثَلَاثًا فِي ) الْمَسْأَلَةِ ( الْأُولَى ، وَقَعَتْ ) الثَّلَاثُ ؛ لِوُجُودِ شَرْطِهَا ( كَوَاحِدَةٍ ) أَيْ : كَمَا يَقَعُ طَلْقَةٌ وَاحِدَةٌ إنْ شَاءَتْ هِيَ أَوْ زَيْدٌ ( فِي ) الْمَسْأَلَةِ ( الثَّانِيَةِ ) لِأَنَّهُ مُقْتَضَى صِيغَتِهِ .
