قَالَ : وَيَتَّجِهُ وُجُوبُ الْحَلِفِ إذَا كَانَتْ الْوَدِيعَةُ رَقِيقًا وَالظَّالِمُ يُرِيدُ قَتْلَهُ أَوْ الْفُجُورَ بِهِ : قَالَ وَأَطْلَقَ الْغَزَالِيُّ فِي وَسِيطِهِ أَنَّهُ يَجِبُ عَلَيْهِ الْحَلِفُ كَاذِبًا ؛ لِأَنَّ الْكَذِبَ لَيْسَ مُحَرَّمًا لَعَيْنِهِ ( وَيُكَفِّرُ ) عَنْ يَمِينِهِ ؛ لِأَنَّهُ كَاذِبٌ فِيهَا ( وَإِنْ حَلَّفَهُ بِالطَّلَاقِ ) أَوْ الْعِتْقِ ( مُكْرَهًا ) عَلَيْهِ أَوْ عَلَى اعْتِرَافِهِ فَحَلَفَ ( حَنِثَ ) لِأَنَّهُ فَدَى الْوَدِيعَةَ بِزَوْجَتِهِ أَوْ رَقِيقِهِ وَإِنْ اعْتَرَفَ بِهَا وَسَلَّمَهَا ضَمِنَ ؛ لِأَنَّهُ فَدَى زَوْجَتَهُ أَوْ رَقِيقَهُ بِهَا ( وَإِنْ أَعْلَمَ اللُّصُوصَ بِمَكَانِهَا ) فَضَاعَتْ بِذَلِكَ كَمَا صَرَّحَ بِهِ الْأَصْلُ ( ضَمِنَ ) لِمُنَافَاةِ ذَلِكَ الْحِفْظِ ( لَا ) إنْ أَعْلَمَهُمْ ( بِأَنَّهَا عِنْدَهُ ) مِنْ غَيْرِ تَعْيِينِ مَكَانِهَا فَلَا يَضْمَنُ بِذَلِكَ .
