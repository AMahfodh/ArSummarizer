( قَوْلُهُ فِي الْقَاعِدَةِ ) أَيْ قَوْلُ الْمُصَنِّفِ وَمَنْ لَزِمَهُ إلَخْ وَ ( قَوْلُهُ أَنَّ الْوُجُوبَ ) أَيْ الْفِطْرَةَ نَفْسُ الْعَبْدِ ( وَقَوْلُهُ ؛ لِأَنَّهَا ) أَيْ نَفَقَةَ زَوْجَةِ الْأَبِ سم ( قَوْلُهُ فِيهِمَا ) أَيْ فِي الْعِلَّتَيْنِ ( قَوْلُهُ وَمِمَّنْ تَجِبُ ) إلَى قَوْلِهِ وَمَنْ آجَرَ فِي النِّهَايَةِ وَإِلَى قَوْلِهِ وَهَلْ الْحُرَّةُ فِي الْمُغْنِي إلَّا قَوْلَهُ قِنٌّ شُرِطَ إلَى مَنْ آجَرَ ( قَوْلُهُ أَيْضًا ) أَيْ مِثْلُ مَا ذُكِرَ فِي الْمَتْنِ ( قَوْلُهُ مُطْلَقًا إلَخْ ) أَيْ سَوَاءٌ كَانَ مُسْلِمًا أَوْ كَافِرًا كُرْدِيٌّ وَيُحْتَمَلُ أَنَّ الْمُرَادَ لَا عَلَى نَفْسِهِ وَلَا عَلَى غَيْرِهِ ( قَوْلُهُ وَالْمَسْجِدِ ) أَيْ سَوَاءٌ كَانَ الْعَبْدُ مِلْكًا لَهُ أَمْ وَقْفًا عَلَيْهِ مُغْنِي وَإِيعَابٌ وَأَسْنَى ( قَوْلُهُ وَمَنْ عَلَى مَيَاسِيرِ الْمُسْلِمِينَ إلَخْ ) أَيْ الْحُرُّ الْفَقِيرُ عَنْ الْكَسْبِ مُغْنِي وَكُرْدِيٌّ .
