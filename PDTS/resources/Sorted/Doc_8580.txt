( وَيَنْفَكُّ ) الرَّهْنُ ( بِفَسْخِ الْمُرْتَهِنِ ) وَحْدَهُ أَوْ مَعَ الرَّاهِنِ ( وَبِالْبَرَاءَةِ مِنْ الدَّيْنِ ) بِقَضَاءٍ أَوْ إبْرَاءٍ أَوْ حَوَالَةٍ أَوْ غَيْرِهَا ( فَإِنْ بَقِيَ شَيْءٌ مِنْهُ لَمْ يَنْفَكَّ شَيْءٌ مِنْ الرَّهْنِ ) أَيْ الْمَرْهُونِ لِأَنَّهُ وَثِيقَةٌ لِجَمِيعِ أَجْزَاءِ الدَّيْنِ ( وَلَوْ رَهَنَ نِصْفَ عَبْدٍ بِدَيْنٍ وَنِصْفَهُ بِآخَرَ فَبَرِئَ مِنْ أَحَدِهِمَا انْفَكَّ قِسْطُهُ ) لِتَعَدُّدِ الْعَقْدِ ( وَلَوْ رَهَنَاهُ ) بِدَيْنٍ ( فَبَرِئَ أَحَدُهُمَا ) مِمَّا عَلَيْهِ ( انْفَكَّ نَصِيبُهُ ) لِتَعَدُّدِ مَنْ عَلَيْهِ الدَّيْنُ ، وَلَوْ رَهَنَهُ عِنْدَ اثْنَيْنِ فَبَرِئَ مِنْ دَيْنِ أَحَدِهِمَا انْفَكَّ قِسْطُهُ لِتَعَدُّدِ مُسْتَحِقِّ الدَّيْنِ .
