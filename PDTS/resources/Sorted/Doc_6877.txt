أُجِيبَ بِأَنَّهُ فَرَّعَهُ عَلَى الْجَدِيدِ ( فَلَوْ رَضِيَ ) الْمَالِكُ بِالْوَضْعِ ( بِلَا عِوَضٍ ) وَقُلْنَا بِعَدَمِ الْإِجْبَارِ ( فَهُوَ إعَارَةٌ ) لِصِدْقِ حَدِّهَا عَلَيْهِ فَيَسْتَفِيدُ بِهَا الْمُسْتَعِيرُ الْوَضْعَ مَرَّةً وَاحِدَةً حَتَّى لَوْ رَفَعَ جُذُوعَهُ أَوْ سَقَطَتْ بِنَفْسِهَا ، أَوْ سَقَطَ الْجِدَارُ فَبَنَاهُ صَاحِبُهُ بِتِلْكَ الْآلَةِ لَمْ يَكُنْ لَهُ الْوَضْعُ ثَانِيًا فِي الْأَصَحِّ ؛ لِأَنَّ الْإِذْنَ إنَّمَا يَتَنَاوَلُ مَرَّةً فَقَطْ ( وَلَهُ ) أَيْ لِلْمَالِكِ ( الرُّجُوعُ قَبْلَ الْبِنَاءِ عَلَيْهِ ) قَطْعًا ( وَكَذَا بَعْدَهُ فِي الْأَصَحِّ ) كَسَائِرِ الْعَوَارِيِّ .
