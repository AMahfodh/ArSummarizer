وَيُعْتَبَرُ بَعْدَ ذَلِكَ بِهَا غَيْرُهَا ) أَيْ مِنْ مُعَاصِرِيهَا وَمَنْ بَعْدَهُمْ ( قَوْلُهُ : صُدِّقَتْ فِي ذَلِكَ ) وَمَعْلُومٌ أَنَّ الْكَلَامَ حَيْثُ لَمْ تَقُمْ عَلَيْهَا بَيِّنَةٌ بِخِلَافِ مَا قَالَتْهُ ( قَوْلُهُ : وَانْقِطَاعُهُ ) أَيْ وَذَلِكَ لَا يُعْلَمْ إلَّا مِنْهَا وَهُوَ الْمَقْصُودُ بِالدَّعْوَى وَالسِّنِّ وَقَعَ تَبَعًا فَقُبِلَ قَوْلُهَا فِيهِ .( 23 / 309 )
