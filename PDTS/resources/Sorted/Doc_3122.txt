( فَرْعٌ مَتَى قَالَ ) لَهُ ( بِعْ الْعَبْدَ بِمِائَةٍ فَبَاعَهُ بِمِائَةٍ وَثَوْبٍ أَوْ ) وَ ( دِينَارٍ صَحَّ ) ؛ لِأَنَّهُ حَصَّلَ غَرَضَهُ ، وَزَادَ خَيْرًا وَقَوْله وَثَوْبٍ أَوْلَى مِنْ قَوْلِهِ أَصْلُهُ وَثَوْبٍ يُسَاوِي مِائَةً ( وَلَوْ قَالَ بِعْ بِأَلْفِ دِرْهَمٍ فَبَاعَ بِأَلْفِ دِينَارٍ لَمْ يَصِحَّ ) إذْ الْمَأْتِيُّ بِهِ لَيْسَ مَأْمُورًا بِهِ وَلَا مُشْتَمِلًا عَلَيْهِ( 10 / 317 )
