وَقَالَ - صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ - { لَا رَضَاعَ بَعْدَ فِصَالٍ } يَعْنِي الْعَامَيْنِ ( وَكَذَا لَوْ خَالَعَتْهُ ) الزَّوْجَةُ ( عَلَى كَفَالَتِهِ ) أَيْ الْوَلَدِ مُدَّةً مُعَيَّنَةً ( أَوْ ) خَالَعَتْهُ عَلَى ( نَفَقَتِهِ مُدَّةً مُعَيَّنَةً كَعَشْرِ سِنِينَ وَنَحْوِهَا ) صَحَّ وَلَوْ لَمْ يَصِفْ النَّفَقَةَ ، فَلَا يُشْتَرَطُ ذِكْرُ الطَّعَامِ وَجِنْسِهِ كَمَا يَأْتِي ( وَالْأَوْلَى أَنْ يَذْكُرَ مُدَّةَ الرَّضَاعِ ) مِنْ تِلْكَ الْمُدَّةِ .
