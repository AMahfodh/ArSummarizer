( وَ ) إنْ كَانَ الْعَيْبُ ( مِنْ جِنْسِهَا ) كَسَوَادِ دَرَاهِمَ وَوُضُوحِ دَنَانِيرَ ( يُخَيَّرُ ) مَنْ صَارَتْ إلَيْهِ ( بَيْنَ فَسْخِ ) الْعَقْدِ لِلْعَيْبِ ( وَإِمْسَاكٍ بِلَا أَرْشٍ إنْ تَعَاقَدَا عَلَى مِثْلَيْنِ ) ، كَدِينَارٍ بِدِينَارٍ ؛ لِأَنَّ أَخْذَهُ يُفْضِي إلَى التَّفَاضُلِ ، أَوْ مَسْأَلَةِ مُدِّ عَجْوَةٍ وَدِرْهَمٍ ( وَإِلَّا ) يَكُنْ الْعَقْدُ عَلَى مِثْلَيْنِ ( فَلَهُ ) أَيْ مَنْ صَارَتْ لَهُ الْمَعِيبَةُ ( أَخْذُهُ ) أَيْ الْأَرْشِ بِمَجْلِسِ الْعَقْدِ لَا مِنْ جِنْسِ السَّلِيمِ فِي صَرْفٍ ؛ لِأَنَّ أَكْثَرَ مَا فِيهِ حُصُولُ زِيَادَةٍ مِنْ أَحَدِ الطَّرَفَيْنِ وَلَا تُمْنَعُ فِي الْجِنْسَيْنِ ، وَلَا يُمْنَعُ فِي الْجِنْسَيْنِ( 5 / 74 )
