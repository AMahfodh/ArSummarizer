( وَلَا يَسْقُطُ ) الْمَهْرُ ( بَعْدَهُ ) أَيْ : الدُّخُولِ بِوَطْءٍ أَوْ خَلْوَةٍ وَنَحْوِهِمَا مِمَّا يُقَرِّرُهُ لِتَقَرُّرِهِ ( وَلَا يَرْجِعُ الزَّوْجُ عَلَيْهَا ) أَيْ : الزَّوْجَةِ ( بِخِلَافِ أَجْنَبِيٍّ ) وَإِلَيْهِ الْإِشَارَةُ بِقَوْلِهِ : ( وَإِنْ أَفْسَدَهُ ) أَيْ : النِّكَاحَ ( غَيْرُهَا ) أَيْ : الزَّوْجَةِ ( لَزِمَهُ ) أَيْ .
