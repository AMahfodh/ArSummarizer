قَوْلُهُ ( مَنْ أَحْيَاهُ بِإِذْنِ الْإِمَامِ مَلَكَهُ ، وَإِنْ أَحْيَاهُ بِغَيْرِ إذْنِ الْإِمَامِ لَمْ يَمْلِكْهُ عِنْدَ أَبِي حَنِيفَةَ ، وَقَالَ أَبُو يُوسُفَ وَمُحَمَّدٌ يَمْلِكُهُ ) لِقَوْلِهِ عَلَيْهِ السَّلَامُ { مَنْ أَحْيَا أَرْضًا مَيْتَةً فَهِيَ لَهُ } وَلِأَبِي حَنِيفَةَ قَوْلُهُ عَلَيْهِ السَّلَامُ { لَيْسَ لِلْمَرْءِ إلَّا مَا طَابَتْ بِهِ نَفْسُ إمَامِهِ } وَلِأَنَّهُ حَقٌّ لِلْمُسْلِمِينَ فَلَيْسَ لِأَحَدٍ أَنْ يَخْتَصَّ بِهِ بِدُونِ إذْنِ الْإِمَامِ كَمَالِ بَيْتِ الْمَالِ ثُمَّ عِنْدَ أَبِي حَنِيفَةَ إذَا لَمْ يَمْلِكْهَا( 3 / 424 )
