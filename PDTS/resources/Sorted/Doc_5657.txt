( وَلَوْ ) ( قَطَعَ فَسَرَى ) الْقَطْعُ لِلنَّفْسِ ( فَلِلْوَلِيِّ حَزُّ رَقَبَتِهِ ) تَسْهِيلًا عَلَيْهِ ( وَلَهُ الْقَطْعُ ) طَلَبًا لِلْمُمَاثَلَةِ مَا لَمْ يَقُلْ : قَصْدِي الْعَفْوُ عَنْهُ بَعْدَهُ ( ثُمَّ الْحَزُّ ) لِلرَّقَبَةِ ( وَإِنْ شَاءَ انْتَظَرَ ) بَعْدَ الْقَطْعِ ( السِّرَايَةَ ) لِتَكْمُلَ الْمُمَاثَلَةُ وَلَيْسَ لِلْجَانِي فِي الْأُولَى طَلَبُ الْإِمْهَالِ بِقَدْرِ حَيَاةِ الْمَجْنِيِّ عَلَيْهِ ، وَمِنْ ثَمَّ جَازَ أَنْ يُوَالِيَ عَلَيْهِ قَطْعَ أَطْرَافٍ فَرَّقَهَا وَلَا فِي الثَّانِيَةِ طَلَبُ الْقَتْلِ أَوْ الْعَفْوِ
