( وَ ) الْحَالُ ( الرَّابِعُ إذَا لَمْ يَكُنْ لِوَلَدِهَا أَبٌ لِكَوْنِهِ وَلَدَ زِنًا أَوْ ) لِكَوْنِهَا ( ادَّعَتْهُ ) أَيْ : ادَّعَتْ أَنَّهُ وَلَدُهَا ( وَأُلْحِقَ ) بِالْبِنَاءِ لِلْمَجْهُولِ ( بِهَا أَوْ ) لِكَوْنِهِ ( مَنْفِيًّا بِلِعَانٍ فَإِنَّهُ يَنْقَطِعُ تَعْصِيبُهُ ) أَيْ : الْوَلَدِ ( مِمَّنْ نَفَاهُ ) بِلِعَانِهِ ( وَنَحْوِهِ ) كَجَحْدِ زَوْجِ الْمُقِرَّةِ بِهِ ( فَلَا يَرِثُهُ ) مَنْ نَفَاهُ وَلَا مَنْ جَحَدَهُ ( وَلَا ) يَرِثُهُ ( أَحَدٌ مِنْ عَصَبَتِهِ ) ؛ لِأَنَّهُ لَمْ يُنْسَبْ إلَيْهِ وَلَا إلَى الزَّانِي ( وَلَوْ ) كَانَ التَّعْصِيبُ ( بِأُخُوَّةٍ مِنْ أَبٍ إذَا وَلَدَتْ تَوْأَمَيْنِ ) مِنْ زِنًا أَوْ نُفِيَا بِلِعَانِ .
