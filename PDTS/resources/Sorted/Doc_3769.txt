( وَيَصِحُّ شَرْطُهُ ) ؛ أَيْ : الْخِيَارِ ( لَهُمَا ) ؛ أَيْ : الْمُتَعَاقِدَيْنِ عَلَى السَّوَاءِ ، ( وَ ) يَصِحُّ شَرْطُهُ ( مُتَفَاوِتًا ) ؛ كَأَنْ يَشْرِطَاهُ لِأَحَدِهِمَا مُدَّةً وَلِلْآخَرِ دُونَهَا ، ( وَ ) يَصِحُّ شَرْطُهُ ( لِأَحَدِهِمَا ) دُونَ الْآخَرِ ؛ لِأَنَّ ذَلِكَ حَقُّهُمَا ، وَإِنَّمَا جُوِّزَ ؛ رِفْقًا بِهِمَا ، فَكَيْفَمَا تَرَاضَيَا بِهِ جَازَ .( 7 / 468 )
