( وَإِنْ ادَّعَاهُ ) أَيْ : الْوَلَدَ ( مُشْتَرٍ لِنَفْسِهِ ) وَقَدْ بِيعَتْ قَبْلَ اسْتِبْرَاءٍ ، وَوَلَدَتْهُ لِفَوْقِ سِتَّةِ أَشْهُرٍ وَدُونَ أَرْبَعِ سِنِينَ مِنْ بَيْعٍ ، وَالْمُشْتَرِي مُقِرٌّ بِوَطْئِهَا ؛ أُرِيَ الْقَافَةَ ( أَوْ ) ادَّعَى ( كُلٌّ مِنْهُمَا ) أَيْ : الْبَائِعِ وَالْمُشْتَرِي فِي الصُّورَةِ الْمَذْكُورَةِ ( أَنَّهُ ) أَيْ : الْوَلَدَ ( لِلْآخَرِ وَالْمُشْتَرِي مُقِرٌّ بِوَطْئِهَا ، أُرِي ) الْوَلَدُ ( الْقَافَةَ ) لِأَنَّ نَظَرَهَا طَرِيقٌ شَرْعِيٌّ إلَى مَعْرِفَةِ النَّسَبِ عِنْدَ الِاحْتِمَالِ كَمَا تَقَدَّمَ فِي اللَّقِيطِ .
