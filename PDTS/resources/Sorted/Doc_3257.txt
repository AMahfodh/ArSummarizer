( فَيُصَلِّي ) الْإِمَامُ بِمَنْ حَضَرَهُ رَكْعَتَيْنِ كَالْعِيدِ وَتَقَدَّمَ ( ثُمَّ يَخْطُبُ خُطْبَةً وَاحِدَةً ) عَلَى الْمِنْبَرِ وَالنَّاسُ جُلُوسٌ عِنْدَهُ ، لِأَنَّهُ لَمْ يُنْقَلْ غَيْرُهُ عَنْهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ( يَفْتَتِحُهَا ) أَيْ الْخُطْبَةَ ( بِالتَّكْبِيرِ ) تَسْعًا نَسَقًا ( كَخُطْبَةِ الْعِيدِ ) لِقَوْلِ ابْنِ عَبَّاسٍ { صَنَعَ الرَّسُولُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فِي الِاسْتِسْقَاءِ كَمَا صَنَعَ فِي الْعِيدِ } .
