( وَمَتَى لَمْ يَثْبُتْ نَسَبُهُ ) أَيْ : الْمُقَرِّ بِهِ مِنْ مَيِّتٍ بِأَنْ أَقَرَّ بِهِ بَعْضُ الْوَرَثَةِ وَلَمْ يَشْهَدْ بِنَسَبِهِ عَدْلَانِ ( أَخَذَ ) الْمُقَرُّ بِهِ ( الْفَاضِلَ بِيَدِ الْمُقِرِّ ) عَنْ نَصِيبِهِ عَلَى مُقْتَضَى إقْرَارِهِ ( إنْ فَضَلَ ) بِيَدِهِ ( شَيْءٌ ) عَنْ نَصِيبِهِ ( أَوْ ) أَخَذَ مَا فِي يَدِهِ ( كُلَّهُ إنْ سَقَطَ ) الْمُقَرُّ ( بِهِ ) أَيْ : الْمُقَرِّ بِهِ لِإِقْرَارِهِ أَنَّهُ لَهُ فَلَزِمَهُ دَفْعُهُ إلَيْهِ( 8 / 85 )
