وَمَا فِي الْأَصْلِ أَظْهَرُ لِأَنَّهُ إذَا كَانَ التَّدْبِيرُ مِنْ مُورِثِهِ يَصْدُقُ عَلَى الْوَارِثِ أَنَّهُ لَا مَالَ لَهُ ( قَوْلُهُ : وَمَا أَوْصَى بِهِ ) أَيْ لِغَيْرِهِ ( قَوْلُهُ : فَالْمُتَّجَهُ إطْلَاقُهُمْ ) أَيْ وَهُوَ الْحِنْثُ بِالدَّيْنِ وَلَوْ عَلَى مَيِّتٍ مُعْسِرٍ ( قَوْلُهُ : بِدَلِيلِ عَدَمِ صِحَّةِ الِاعْتِيَاضِ ) قَضِيَّةُ هَذَا أَنَّ الْكَلَامَ فِي نُجُومِ الْكِتَابَةِ وَأَنَّهُ يَحْنَثُ بِغَيْرِهَا مِمَّا لَهُ عَلَى مُكَاتَبِهِ مِنْ الدَّيْنِ قَطْعًا ( قَوْلُهُ : مَرْدُودٌ ) أَيْ خِلَافًا لحج ( قَوْلُهُ : أَنَّهُ لَا أَثَرَ لِتَعْجِيزِهِ ) أَيْ فَلَا حِنْثَ لِأَنَّهُ لَمْ يَكُنْ مَالَهُ حَالَ الْحَلِفِ .
