قَالَ فِي الْإِنْصَافِ : قُلْت : الَّذِي يَنْبَغِي أَنْ يُجْزَمَ بِهِ أَنَّهُ مُرَادُ الْأَصْحَابِ ، وَكَلَامُهُمْ يَدُلُّ عَلَيْهِ ( أَوْ رَوَّعَهُ بِأَنْ شَهَرَهُ ) ؛ أَيْ : السَّيْفَ وَنَحْوَهُ ( فِي وَجْهِهِ ) فَمَاتَ خَوْفًا ( أَوْ دَلَّاهُ مِنْ شَاهِقٍ فَمَاتَ أَوْ ذَهَبَ عَقْلُهُ ) خَوْفًا ( أَوْ حَفَرَ بِئْرًا مُحَرَّمًا حَفْرُهُ ) كَفِي طَرِيقٍ ضَيِّقٍ ( أَوْ وَضَعَ حَجَرًا ، أَوْ قِشْرَ بِطِّيخٍ ، أَوْ صَبَّ مَاءً بِفِنَائِهِ ) ؛ أَيْ : مَا اتَّسَعَ أَمَامَ دَارِهِ ( أَوْ بِطَرِيقٍ ، وَيَتَّجِهُ لَا ) إنْ كَانَ صَبُّ الْمَاءِ بِالْأَفْنِيَةِ وَالطُّرُقِ ( لِنَفْعٍ عَامٍّ ) كَتَسْكِينِ الْغُبَارِ ، فَإِنْ فَعَلَ ذَلِكَ قَاصِدًا بِهِ الثَّوَابَ ( وَلَمْ يُسْرِفْ ) فَلَا شَيْءَ عَلَيْهِ ؛ لِأَنَّهُ مُحْسِنٌ ، وَهُوَ مُتَّجِهٌ ( أَوْ( 17 / 333 )
