( وَحَرِيمُ أَرْضٍ ) أُحْيِيَتْ ( لِزَرْعٍ ) قَدْرَ ( مَا يَحْتَاجهُ ) زَارِعُهَا ( لِسَقْيِهَا وَرَبْطِ دَوَابِّهَا وَطَرْحِ سَبَخِهَا وَنَحْوِ ذَلِكَ ) كَمَصْرِفِ مَائِهَا عِنْدَ الِاسْتِغْنَاءِ عَنْهُ ؛ لِأَنَّ ذَلِكَ كُلَّهُ مِنْ مَرَافِقِهَا ( وَحَرِيمُ دَارٍ مِنْ مَوَاتٍ حَوْلَهَا مَطْرَحُ تُرَابٍ وَكُنَاسَةٍ وَثَلْجٍ وَمَاءٍ وَمِيزَابٍ وَمَمَرٍّ إلَى بَابِهَا ) ؛ لِأَنَّ هَذَا كُلَّهُ يَرْتَفِقُ بِهِ سَاكِنُهَا .( 14 / 18 )
