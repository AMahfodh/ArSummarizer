( وَلَا يَضْمَنُ غَاصِبُ عَبْدٍ مَا أَكَلَ ) الْغَاصِبُ ( مِنْ أَجْرِهِ ) الَّذِي آجَرَ الْعَبْدَ نَفْسَهُ بِهِ لِعَدَمِ تَقَوُّمُهُ عِنْدَ أَبِي حَنِيفَةَ ( كَمَا ) لَا يَضْمَنُ اتِّفَاقًا ( لَوْ آجَرَهُ الْغَاصِبُ ) ؛ لِأَنَّ الْأَجْرَ لَهُ لَا لِمَالِكِهِ ( وَجَازَ لِلْعَبْدِ قَبْضُهَا ) لَوْ آجَرَ نَفْسَهُ لَا لَوْ آجَرَهُ الْمَوْلَى إلَّا بِوَكَالَةٍ ؛ لِأَنَّهُ الْعَاقِدُ عِنَايَةٌ ( فَلَوْ وَجَدَهَا مَوْلَاهُ ) قَائِمَةً ( فِي يَدِهِ أَخَذَهَا ) لِبَقَاءِ مِلْكِهِ كَمَسْرُوقٍ بَعْدَ الْقَطْعِ .
