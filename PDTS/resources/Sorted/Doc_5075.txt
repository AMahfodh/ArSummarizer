( وَلَوْ قَطَعَ حُرٌّ أَنْفَ عَبْدٍ قِيمَتُهُ أَلْفٌ فَانْدَمَلَ ) الْجَرْحُ ( ثُمَّ عَتَقَ ) بِهِ الْعَبْدُ بِنَوْعٍ مِنْ أَنْوَاعِ الْعِتْقِ ( أَوْ ) قَطَعَ أَنْفِهِ ثُمَّ ( عَتَقَ ثُمَّ انْدَمَلَ ) فَقِيمَتُهُ بِكَمَالِهَا لِلسَّيِّدِ ، أَوْ قَطَعَ أَنْفِهِ ( وَمَاتَ مِنْ سِرَايَةِ الْجُرْحِ ) ( فَقِيمَتُهُ ) بِكَمَالِهَا ( لِلسَّيِّدِ ) ؛ لِأَنَّهُ حِينَ الْجِنَايَةِ كَانَ رَقِيقًا لَهُ ، وَالْجِنَايَةُ يُرَاعَى فِيهَا حَالُ وُجُودِهَا .( 17 / 253 )
