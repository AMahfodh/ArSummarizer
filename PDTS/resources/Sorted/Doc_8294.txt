( تَنْبِيهٌ ) شَمَلَ قَوْلُهُمْ مُجْتَهِدًا الْمُجْتَهِدَ الْمُطْلَقَ وَمُجْتَهِدَ الْمَذْهَبِ وَمُجْتَهِدَ الْفُتْيَا قَوْلُهُ : لِخَبَرِ النَّسَائِيّ { الْأَئِمَّةُ مِنْ قُرَيْشٍ } ) وَلِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { قَدِّمُوا قُرَيْشًا وَلَا تَقَدَّمُوهَا } ، وَقَدْ انْعَقَدَ الْإِجْمَاعُ عَلَى ذَلِكَ ( قَوْلُهُ : فَإِنْ فُقِدَ قُرَشِيٌّ إلَخْ ) ، قَالَ الْإِمَامُ لَوْ عُقِدَتْ لِغَيْرِ قُرَشِيٍّ لِلْعَدَمِ ثُمَّ نَشَأَ قُرَشِيٌّ بِالشُّرُوطِ فَإِنْ عَسُرَ خَلْعُ الْأَوَّلِ أُقِرَّ ، وَإِلَّا فَالْوَجْهُ عِنْدِي تَسْلِيمُ الْأَمْرِ لِلْقُرَشِيِّ ( قَوْلُهُ : ثُمَّ إلَى جُرْهُمَ ) هُمْ الَّذِينَ رَبَّوْا إسْمَاعِيلَ ، وَتَزَوَّجَ مِنْهُمْ( 19 / 350 )
