( وَيُدْفَنُ ) مَيِّتٌ ( فِي مُسَبَّلَةٍ وَلَوْ بِقَوْلِ بَعْضِ الْوَرَثَةِ ) لِأَنَّهُ أَقَلُّ ضَرَرًا وَلَا مِنَّةَ فِيهِ ، بِخِلَافِ مَا لَوْ طَلَبَ بَعْضُهُمْ أَنْ يُكَفَّنَ مِنْ أَكْفَانِ الْمُسْلِمِينَ ( وَيُقَدَّمُ فِيهَا ) أَيْ الْمُسْبَلَةِ عِنْدَ ضِيقٍ ( بِسَبْقٍ ) لِأَنَّهُ سَبْقٌ إلَى مُبَاحٍ ( ثُمَّ ) مَعَ تَسَاوٍ فِي سَبْقٍ يُقَدَّمُ ب ( قُرْعَةٍ ) لِأَنَّهَا لِتَمْيِيزِ مَا أُبْهِمَ ( وَيَحْرُمُ الْحَفْرُ فِيهَا ) أَيْ الْمُسْبَلَةِ ( قَبْلَ الْحَاجَةِ ) إلَيْهِ ، ذَكَرَهُ ابْنُ الْجَوْزِيِّ .
