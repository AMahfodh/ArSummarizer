قَوْلُ الْمَتْنِ ( مَخِيطًا ) أَيْ وَلَا مَا فِي مَعْنَاهُ مِمَّا يَحْرُمُ عَلَى الْمُحْرِمِ لُبْسُهُ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : وَلَا تُشَدُّ عَلَيْهِ أَكْفَانُهُ ) إنْ كَانَ الْمُرَادُ : لَا يُنْدَبُ فَمُحْتَمَلٌ أَوْ : لَا يَجُوزُ فَمَحَلُّ تَأَمُّلٍ إذَا كَانَ بِنَحْوِ خَيْطٍ أَوْ فِي مَحَلِّ التِّكَّةِ فَلْيُتَأَمَّلْ بَصْرِيٌّ وَفِي سم نَحْوُهُ وَصَنِيعُ النِّهَايَةِ وَالْمُغْنِي ظَاهِرٌ فِي الْأَوَّلِ كَمَا مَرَّ قَوْلُ الْمَتْنِ ( وَلَا يَسْتُرُ رَأْسَهُ إلَخْ ) أَيْ يَحْرُمُ ذَلِكَ نِهَايَةٌ وَمُغْنِي أَيْ فَلَوْ خَالَفُوا وَفَعَلُوا وَجَبَ الْكَشْفُ مَا لَمْ يُدْفَنْ الْمَيِّتُ مِنْهُمَا ع ش أَيْ الْمُحْرِمِ وَالْمُحْرِمَةِ ( قَوْلُهُ : قُبَيْلَ الْفَصْلِ ) مُتَعَلِّقٌ بِقَوْلِهِ مَرَّ .( 11 / 28 )
