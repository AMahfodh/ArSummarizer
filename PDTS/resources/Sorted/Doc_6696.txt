نَعَمْ لَوْ رَأَى الْمُتَيَمِّمُ الْمَاءَ فِي أَثْنَاءِ عَدَدٍ نَوَاهُ لَيْسَ لَهُ زِيَادَةٌ عَلَيْهِ كَمَا عُلِمَ مِمَّا مَرَّ فِي بَابِهِ ( وَإِلَّا ) أَيْ ، وَإِنْ لَمْ يُغَيِّرْ النِّيَّةَ قَبْلَهُمَا ( فَتَبْطُلُ ) صَلَاتُهُ بِذَلِكَ لِعَدَمِ شُمُولِ نِيَّتِهِ لِمَا أَحْدَثَهُ ( فَلَوْ ) ( نَوَى رَكْعَتَيْنِ ) مَثَلًا ( ثُمَّ قَامَ إلَى ) رَكْعَةٍ ( ثَالِثَةٍ سَهْوًا ) ثُمَّ تَذَكَّرَ ( فَالْأَصَحُّ أَنَّهُ يَقْعُدُ ) حَتْمًا ( ثُمَّ يَقُومُ لِلزِّيَادَةِ إنْ شَاءَ ) هَا ثُمَّ( 5 / 369 )
