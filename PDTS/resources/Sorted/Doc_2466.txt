( وَ ) إنْ ( امْرَأَةٌ ) حُرَّةٌ ( لَمْ تَرَ حَيْضًا أَصْلَا ) ، وَإِنْ رَأَتْ نِفَاسًا كَمَا صَحَّحَهُ النَّوَوِيُّ ( أَوْ نَسِيَتْ ) عَادَتَهَا قَدْرًا وَوَقْتًا ، أَوْ أَحَدَهُمَا ( أَوْ بَلَغَتْ لِأَعْلَى مُدَّةِ يَأْسِ نِسْوَةِ الْعَشِيرِ لَهَا ) أَيْ عَشِيرَتِهَا مِنْ الْأَبَوَيْنِ فَإِنَّهَا تَعْتَدُّ لِفُرْقَةِ الْحَيَاةِ ( ثَلَاثَةً ) أَيْ بِثَلَاثَةٍ ( مِنْ الشُّهُورِ ) .
