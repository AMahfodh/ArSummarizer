( قَوْلُهُ وَإِنْ لَمْ يَذْكُرْ ذَلِكَ ) أَيْ : مَا يُشْعِرُ بِالتَّرَدُّدِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ قَصْدَهُ لِلصَّوْمِ إلَخْ ) عَطْفٌ عَلَى اسْمِ إنَّ وَخَبَرِهِ ( قَوْلُهُ بِذِكْرِ ذَلِكَ ) أَيْ : فَإِنْ لَمْ يَكُنْ مِنْهُ فَتَطَوُّعٌ كُرْدِيٌّ وَالْأَوْلَى أَيْ : مَا يُشْعِرُ بِالتَّرَدُّدِ ( قَوْلُهُ وَعَلَيْهِ إلَخْ ) أَيْ : التَّفْصِيلِ الْمَذْكُورِ ( قَوْلُهُ وَلَا يُنَافِي ) إلَى الْمَتْنِ فِي النِّهَايَةِ ( قَوْلُهُ هَذَا ) أَيْ : مَا ذُكِرَ فِي الْمَتْنِ مِنْ الِاسْتِثْنَاءِ ( قَوْلُهُ مَا يَأْتِي ) أَيْ : فِي فَصْلِ شُرُوطِ الصَّوْمِ مِنْ حَيْثُ الْفَاعِلُ ( قَوْلُهُ مِنْ هَؤُلَاءِ ) أَيْ : السَّابِقَةِ فِي الْمَتْنِ .
