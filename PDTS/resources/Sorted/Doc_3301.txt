( قَوْلُهُ وَالْوَصِيَّةُ ) مُبْتَدَأٌ خَبَرُهُ تَخْتَصُّ بِفُقَرَائِهِمْ قَوْلُهُ وَإِلَّا فَالْحَاكِمُ ) يَنْبَغِي أَخْذًا مِمَّا تَقَدَّمَ أَوْ الْوَارِثُ ، ثُمَّ رَأَيْت قَوْلَهُ الْآتِيَ آنِفًا فَإِنْ دَفَعَ الْوَصِيُّ إلَخْ وَهُوَ دَالٌّ عَلَى ذَلِكَ ( قَوْلُهُ الْأَفْضَلَ ) وَصْفٌ لِلتَّعْمِيمِ ( قَوْلُهُ وَمَرَّ ) أَيْ فِي بَحْثِ الْقَبُولِ أَنَّهُمْ أَيْ الْفُقَرَاءَ ( قَوْلُهُ وَقَدْ يُفَرَّقُ ) أَيْ عَلَى الْأَوَّلِ( 28 / 49 )
