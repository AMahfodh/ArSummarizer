( وَإِنْ كَانَتْ ) الْبَائِنُ الْحَامِلُ ( مُرْضِعَةً ) وَلَدًا لِزَوْجِهَا ( فَلَهَا نَفَقَةُ ) أَيْ أُجْرَةُ ( الرَّضَاعِ أَيْضًا ) أَيْ كَمَا لَهَا نَفَقَةُ الْحَمْلِ لِقَوْلِهِ تَعَالَى { فَإِنْ أَرْضَعْنَ لَكُمْ فَآتُوهُنَّ أُجُورَهُنَّ } وَالْبَائِنُ لَا يَجِبُ عَلَيْهَا الْإِرْضَاعُ أَبُو الْحَسَنِ وَأُجْرَةُ الرَّضَاعِ نَقْدٌ لَا طَعَامٌ ، وَيُشْتَرَطُ أَنْ لَا يَضُرَّ إرْضَاعُهَا الْوَلَدَ ، وَإِلَّا فَأُجْرَتُهُ لِمَنْ تُرْضِعُهُ وَلَا حَقَّ فِيهَا لِأُمِّهِ( 9 / 251 )
