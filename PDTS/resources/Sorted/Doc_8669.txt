وَكَذَا لَوْ جَعَلَ لَهُ جُزْءًا مِنْ نَفْسِ الْمَالِ الْمَدْفُوعِ ( لَهُ ) أَيْ لِلْعَامِلِ ( أَوْ لِعَبْدِهِ ) أَيْ عَبْدِ الْعَامِلِ إذْ الْمَشْرُوطُ لِلْعَبْدِ لِسَيِّدِهِ ( أَوْ ) شَرَطَ الْجُزْءَ لِلْعَامِلِ ، وَ ( لِأَجْنَبِيٍّ مَعَ عَمَلٍ مِنْهُ ) أَيْ مِنْ الْأَجْنَبِيِّ بِأَنْ يَقُولَ : اعْمَلْ فِي هَذَا الْمَالِ بِثُلُثِ الرِّبْحِ لَكَ وَلِزَيْدٍ ، عَلَى أَنْ يَعْمَلَ مَعَكَ ؛ لِأَنَّهُ فِي قُوَّةِ قَوْلِهِ : اعْمَلَا فِي هَذَا الْمَالِ بِالثُّلُثِ ( وَيُسَمَّى أَيْضًا ) دَفْعُ الْمَالِ عَلَى الْوَجْهِ الْمَذْكُورِ ( قِرَاضًا ) وَتَقَدَّمَ ( وَمُعَامَلَةً ) مِنْ الْعَمَلِ .( 11 / 304 )
