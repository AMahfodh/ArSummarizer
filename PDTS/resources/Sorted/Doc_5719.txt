( وَإِنْ أَنْكَرَهُ ) أَيْ الْعَيْبَ ( الْوَكِيلُ ) وَلَمْ يَعْتَرِفْ بِأَنَّ الْمَبِيعَ كَانَ مَعِيبًا ( فَتَوَجَّهَتْ الْيَمِينُ عَلَيْهِ فَنَكَلَ ) عَنْ الْيَمِينِ ( فَرَدَّهُ ) الْمُشْتَرِي ( عَلَيْهِ بِنُكُولِهِ لَمْ يَمْلِكْ ) الْوَكِيلُ ( رَدَّهُ عَلَى مُوَكِّلِهِ ) لِأَنَّهُ غَيْرُ مُعْتَرِفٍ بِعَيْبِهِ وَهَذَا كُلُّهُ إذَا قُلْنَا : إنَّ الْقَوْلَ قَوْلُ الْبَائِعِ وَالْمَذْهَبُ : أَنَّ الْقَوْلَ قَوْلُ الْمُشْتَرِي فَيَحْلِفُ وَيَرُدُّهُ عَلَى الْمُوَكِّلِ ، كَمَا يُعْلَمُ مِمَّا ذَكَرَهُ بِقَوْلِهِ .( 9 / 137 )
