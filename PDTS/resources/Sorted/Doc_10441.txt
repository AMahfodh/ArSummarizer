( وَ ) يَقَعُ ( بِثَلَاثِ أَنْصَافِ طَلْقَتَيْنِ ) ثَلَاثٌ ؛ لِأَنَّ نِصْفَ الطَّلْقَتَيْنِ طَلْقَةٌ ، وَإِذَا جَمَعَ بَيْنَ ثَلَاثَةِ أَنْصَافٍ يَكُونُ ثَلَاثَ تَطْلِيقَاتٍ ضَرُورَةً ( وَ ) يَقَعُ بِثَلَاثَةِ أَنْصَافِ ( طَلْقَةٍ طَلْقَتَانِ ) ؛ لِأَنَّ ثَلَاثَةَ أَنْصَافِ طَلْقَةٍ تَكُونُ طَلْقَةً وَنِصْفًا فَيَتَكَامَلُ النِّصْفُ فَيَحْصُلُ طَلْقَتَانِ ( وَقِيلَ ) يَقَعُ ( ثَلَاثٌ ) ؛ لِأَنَّ كُلَّ نِصْفٍ يَتَكَامَلُ فَيَحْصُلُ ثَلَاثٌ ( وَوَاحِدَةً ) بِالنَّصْبِ أَيْ يَقَعُ بِقَوْلِهِ أَنْتِ طَالِقٌ وَاحِدَةً ( فِي ثِنْتَيْنِ وَاحِدَةٌ إنْ لَمْ يَنْوِ ) لِكَوْنِهِ صَرِيحًا ( أَوْ نَوَى الضَّرْبَ ) ؛ لِأَنَّهُ لَا يَزِيدُ شَيْئًا فِي الْمَضْرُوبِ ( وَإِنْ نَوَى وَاحِدَةً وَثِنْتَيْنِ فَثَلَاثٌ ) ؛ لِأَنَّهُ مُحْتَمَلُ اللَّفْظِ هَذَا الَّذِي ذَكَرْنَاهُ كَانَ فِي الْمَوْطُوءَةِ
