( وَلَوْ صَالَحَ الْوَرَثَةُ مَنْ وُصِيَّ لَهُ ) مِنْ قِبَلِ مُوَرِّثِهِمْ ( بِخِدْمَةِ ) رَقِيقٍ مِنْ التَّرِكَةِ ( أَوْ ) ب ( سُكْنَى ) دَارٍ مُعَيَّنَةٍ ( أَوْ ) ب ( حَمْلِ أَمَةٍ ) مُعَيَّنَةٍ ( بِدَرَاهِمَ ) مَثَلًا ( مُسَمَّاةٍ جَازَ ) ذَلِكَ صُلْحًا ؛ لِأَنَّهُ إسْقَاطُ حَقٍّ ، فَصَحَّ فِي الْمَجْهُولِ لِلْحَاجَةِ ( لَا بَيْعًا ) لِعَدَمِ الْعِلْمِ بِالْمَبِيعِ( 5 / 327 )
