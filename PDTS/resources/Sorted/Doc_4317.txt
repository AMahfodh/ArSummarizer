وَإِنْ قَالَ مُسْتَعِيرٌ : أَنَا أَدْفَعُ قِيمَةَ الْأَرْضِ لِتَصِيرَ لِي ؛ لَمْ يَلْزَمْ الْمُعِيرَ ؛ لِأَنَّ الْغِرَاسَ وَالْبِنَاءَ تَابِعٌ لِلْأَرْضِ ، وَلِذَلِكَ يَتْبَعُهَا الْغِرَاسُ وَالْبِنَاءُ فِي الْبَيْعِ وَلَا تَتْبَعُهُمَا فِيهِ ، ( أَوْ ) - أَيْ : وَلِمُعِيرٍ - ( قَلْعُهُ ) - أَيْ : الْغِرَاسِ وَالْبِنَاءِ - ( جَبْرًا ، وَيَضْمَنُ ) الْمُعِيرُ ( نَقْصَهُ ) ؛ لِأَنَّ فِي ذَلِكَ دَفْعًا( 11 / 149 )
