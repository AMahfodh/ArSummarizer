( وَ ) شَرْطُ صِحَّتِهَا أَيْضًا ( الْجَمَاعَةُ وَأَقَلُّهَا ثَلَاثَةُ رِجَالٍ سِوَى الْإِمَامِ فَإِنْ نَفَرُوا ) أَيْ تَفَرَّقَ الْجَمَاعَةُ ( قَبْلَ سُجُودِهِ ) أَيْ الْإِمَامِ ( بَطَلَتْ ) الْجُمُعَةُ لِانْتِفَاءِ شَرْطِهَا وَلَزِمَ الْبَدْءُ بِالظُّهْرِ ( وَإِنْ بَقِيَ ثَلَاثَةٌ أَوْ نَفَرُوا بَعْدَ سُجُودِهِ أَتَمَّهَا ) ؛ لِأَنَّ الْجَمَاعَةَ شَرْطُ الِانْعِقَادِ ، وَقَدْ انْعَقَدَتْ فَلَا يُشْتَرَطُ دَوَامُهَا ؛ لِأَنَّهَا لَيْسَتْ شَرْطًا لَهُ .
