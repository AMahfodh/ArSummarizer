( وَ ) حَنِثَ ( بِجَعْلِ ثَوْبٍ قَبَاءً ) بِالْمَدِّ ثَوْبٌ مُفَرَّجٌ ( أَوْ عِمَامَةً فِي ) حَلِفِهِ ( لَا أَلْبَسُهُ ) ، وَلَبِسَهُ عَلَى هَذِهِ الْحَالَةِ أَوْ وَضَعَهُ عَلَى كَتِفِهِ أَوْ اتَّزَرَ بِهِ ( لَا ) يَحْنَثُ بِجَعْلِهِ قَبَاءً أَوْ عِمَامَةً ( إنْ كَرِهَهُ لِضِيقِهِ ) أَوْ لِسُوءِ صَنْعَتِهِ أَيْ إنْ كَانَ الْحَامِلُ لَهُ عَلَى الْحَلِفِ ذَلِكَ ( وَلَا وَضَعَهُ ) عَطْفٌ عَلَى جَعَلَهُ الْمُقَدَّرُ بَعْدَ لَا مِنْ قَوْلِهِ لَا إنْ كَرِهَهُ عَلَى كَرِهَهُ لِفَسَادِ الْمَعْنَى ( عَلَى فَرْجِهِ ) لَيْلًا أَوْ نَهَارًا مِنْ غَيْرِ لَفٍّ وَلَا إدَارَةٍ .
