وَشَبَّهَ فِي الْجَبْرِ فَقَالَ ( كَفَضْلِ ) بِفَتْحِ الْفَاءِ وَسُكُونِ الضَّادِ الْمُعْجَمَةِ أَيْ زَائِدُ مَاءِ ( بِئْرِ ) سَقْيِ ( مَاشِيَةٍ ) حُفِرَتْ ( بِصَحْرَاءَ ) لَا اخْتِصَاصَ لِأَحَدٍ بِهَا فَيَجِبُ عَلَى حَافِرِهَا دَفْعُهُ لِوَارِدِهَا ( هَدَرًا ) بِفَتْحِ الْهَاءِ وَالدَّالِ الْمُهْمَلَةِ ، أَيْ بِلَا عِوَضٍ ؛ لِأَنَّهُ لَيْسَ مَالِكًا لَهُ فَلَا يَبِيعُهُ وَلَا يَهَبُهُ وَلَا يُورَثُ إذَا مَاتَ ( إنْ لَمْ يُبَيِّنْ ) حِينَ حَفْرِهِ أَنَّهُ قَصَدَ ( الْمِلْكِيَّةَ ) لِلْبِئْرِ وَمَا فِيهَا مِنْ الْمَاءِ ، فَإِنْ بَيَّنَهَا فَلَهُ مَنْعُهُ وَبَيْعُهُ وَهِبَتُهُ وَتُورَثُ عَنْهُ إنْ مَاتَ ، وَمِنْ الْبَيَانِ أَنْ يُشْهِدَ حِينَ حَفْرِهَا أَنَّهَا لِنَفْسِهِ خَاصَّةً .
