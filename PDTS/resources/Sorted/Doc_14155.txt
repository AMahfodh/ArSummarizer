وَعُطِفَ عَلَى بِلَا شَوْبِ بَعْضُ مُحْتَرَزِهِ عَلَى عَادَتِهِ فَقَالَ ( لَا ) يُجْزِئُ عِتْقُ رَقِيقٍ ( مُشْتَرًى ) بِفَتْحِ الرَّاءِ بِشَرْطِ كَوْنِ شِرَائِهِ ( لِلْعِتْقِ ) عَنْ ظِهَارِهِ لِشَوْبِ الْعِوَضِ لِتَقْدِيرِ تَرْكِ الْبَائِعِ بَعْضَ ثَمَنِهِ فِي نَظِيرِ رِضَا الْمُشْتَرَى بِشَرْطِ عِتْقِهِ وَنَعْتُ رَقَبَةٍ بِ ( مُحَرَّرَةٍ ) بِضَمِّ الْمِيمِ وَفَتْحِ الْحَاءِ وَالرَّاءِ أَيْ مُعْتَقَةٍ ( لَهُ ) أَيْ الظِّهَارِ وَعُطِفَ عَلَيْهِ بَعْضُ مُحْتَرَزِهِ بِقَوْلِهِ : ( لَا ) يُجْزِئُ عِتْقُ ( مَنْ ) أَيْ رَقِيقٍ أَوْ الرَّقِيقِ الَّذِي ( يَعْتِقُ عَلَيْهِ ) أَيْ الْمُظَاهِرِ بِمُجَرَّدِ مِلْكِهِ لِقَرَابَتِهِ كَأَصْلِهِ وَفَرْعِهِ وَحَاشِيَتِهِ الْقَرِيبَةِ ، أَوْ تَعْلِيقِ عِتْقِهِ عَلَى شِرَائِهِ نَحْوَ إنْ اشْتَرَيْته فَهُوَ حُرٌّ لِأَنَّ عِتْقَهُ لِلْقَرَابَةِ أَوْ التَّعْلِيقِ لَا لِلظِّهَارِ ، ( وَ ) إنْ قَالَ الْمُظَاهِرُ ( إنْ اشْتَرَيْته ) أَيْ هَذَا الرَّقِيقَ الْمُعَيَّنَ ( فَهُوَ حُرٌّ عَنْ ظِهَارِي ) ثُمَّ اشْتَرَاهُ وَأَعْتَقَهُ عَنْ ظِهَارِهِ فَفِي إجْزَاءِ عِتْقِهِ عَنْهُ وَعَدَمِهِ ( تَأْوِيلَانِ ) الْبُنَانِيُّ مَوْضُوعُ الْمَسْأَلَةِ عِنْدَ الْأَئِمَّةِ مَنْ لَا سَبَبَ فِيهِ لِلْعِتْقِ إلَّا التَّعْلِيقُ الْمَذْكُورُ ، وَعِبَارَةُ الْمُدَوَّنَةِ قَالَ مَالِكٌ رَضِيَ اللَّهُ تَعَالَى عَنْهُ وَلَا يُجْزِيهِ أَنْ يُعْتِقَ عَبْدًا ، قَالَ إنْ اشْتَرَيْته فَهُوَ حُرٌّ ، فَإِنْ اشْتَرَاهُ وَهُوَ مُظَاهِرٌ فَلَا يُجْزِيهِ ا ه .
