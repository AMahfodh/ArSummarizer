( وَإِذَا تَطَيَّبَ نَاسِيًا أَوْ عَامِدًا لَزِمَهُ إزَالَتُهُ بِمَا أَمْكَنَ مِنْ الْمَاءِ وَغَيْرِهِ مِنْ الْمَائِعَاتِ ) ؛ لِأَنَّ الْقَصْدَ الْإِزَالَةُ ( فَإِنْ لَمْ يَجِدْ ) مَائِعًا يُزِيلُ بِهِ الطِّيبَ ( ف ) إنَّهُ يُزِيلُهُ ( بِمَا أَمْكَنَهُ مِنْ الْجَامِدَاتِ كَحَكِّهِ بِخِرْقَةٍ وَتُرَابٍ وَوَرَقِ شَجَرٍ وَنَحْوِهِ ) كَحَجَرٍ وَخَشَبٍ ؛ لِأَنَّ الْوَاجِبَ إزَالَتُهُ حَسَبَ الْإِمْكَانِ وَقَدْ فَعَلَ .
