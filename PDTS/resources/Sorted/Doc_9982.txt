( وَهِيَ ) أَيْ : اللُّقَطَةُ ( ثَلَاثَةُ أَقْسَامٍ ) بِالِاسْتِقْرَاءِ : الْأَوَّلُ ( مَا لَا تَتْبَعُهُ هِمَّةُ أَوْسَاطِ النَّاسِ ) أَيْ : لَا يَهْتَمُّونَ فِي طَلَبِهِ ( كَسَوْطٍ وَشِسْعٍ ) بِتَقْدِيمِ الْمُعْجَمَةِ أَحَدُ سُيُورِ النَّعْلِ الَّذِي يَدْخُلُ بَيْنَ الْأُصْبُعَيْنِ ( وَرَغِيفٍ ) وَثَمَرَةٍ وَكُلِّ مَا لَا خَطَرَ لَهُ ( فَيُمْلَكُ بِأَخْذِهِ ) وَيُبَاحُ الِانْتِفَاعُ بِهِ نَصًّا لِحَدِيثِ جَابِرٍ { رَخَّصَ النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فِي الْعِصِيِّ وَالسَّوْطِ وَالْحَبْلِ يَلْتَقِطُهُ الرَّجُلُ يَنْتَفِعُ بِهِ } رَوَاهُ أَبُو دَاوُد ( وَلَا يَلْزَمُهُ تَعْرِيفُهُ ) لِأَنَّهُ مِنْ قَبِيلِ الْمُبَاحَاتِ ( وَلَا ) يَلْزَمُهُ ( بَدَلُهُ إنْ وَجَدَ رَبَّهُ ) الَّذِي سَقَطَ مِنْهُ لِمِلْكِ مُلْتَقِطِهِ لَهُ بِأَخْذِهِ .
