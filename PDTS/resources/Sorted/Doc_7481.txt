( وَيُمْهَلُ مُولٍ ) طُلِبَتْ فِيئَتُهُ بَعْدَ الْمُدَّةِ ( لِصَلَاةِ فَرْضٍ وَتَغَدٍّ وَهَضْمِ طَعَامٍ وَنَوْمٍ عَنْ نُعَاسٍ وَتَحَلُّلٍ مِنْ إحْرَامٍ وَنَحْوَهُ ) كَفِطْرٍ مِنْ صَوْمٍ وَاجِبٍ وَدُخُولِ خَلَاءٍ وَرُجُوعٍ إلَى بَيْتِهِ ( بِقَدْرِهِ ) أَيْ بِقَدْرِ الْحَاجَةِ فَقَطْ لِأَنَّهُ الْعَادَةُ ( وَ ) يُمْهَلُ مُولٍ ( مُظَاهِرٍ لِطَلَبِ رَقَبَةٍ ) يَعْتِقُهَا عَنْ ظِهَارِهِ ( ثَلَاثَةَ أَيَّامٍ ) ؛ لِأَنَّهُ يَسِيرٌ و ( لَا ) يُمْهَلُ مُظَاهِرٌ ( لِصَوْمٍ ) عَنْ كَفَّارَتِهِ بَلْ يُطَلِّقُ الْحَاكِمُ عَلَيْهِ لِأَنَّ زَمَنَ الصَّوْمِ كَثِيرٌ .( 10 / 20 )
