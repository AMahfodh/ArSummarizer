( قَالَ فِي التَّرْغِيبِ : إنْ خَرَجَتْ لِلْعَزَايَا وَالزِّيَارَاتِ وَلَمْ تُكْثِرْ فَهِيَ مُخَدَّرَةً ) فَلَا يُحْضِرُهَا الْقَاضِي بَلْ تُوَكِّلُ ( وَمَرِيضٌ وَنَحْوُهُ ) مِنْ ذَوِي الْأَعْذَارِ ( كَمُخَدَّرَةٍ ) فِي أَنَّهُ يُؤْمَرُ بِالتَّوْكِيلِ وَلَا يَحْضُرُ لِلْحَرَجِ وَالْمَشَقَّةِ ( وَإِنْ اسْتَعْدَى عِنْدَهُ عَلَى غَائِبٍ فِي غَيْرِ عَمَلِهِ لَمْ يُعْدِ عَلَيْهِ ) لِأَنَّهُ لَيْسَ مَوْلَى عَلَيْهِ .( 22 / 192 )
