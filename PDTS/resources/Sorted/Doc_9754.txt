[ فَرْعٌ ] لَوْ ادَّعَتْ الْحَيْضَ وَلَكِنْ فِي زَمَنِ الْيَأْسِ فَالظَّاهِرُ تَصْدِيقُهَا لِقَوْلِهِمْ إنَّهَا لَوْ حَاضَتْ رَجَعَتْ الْعِدَّةُ مِنْ الْأَشْهُرِ إلَى الْأَقْرَاءِ بِرّ ا ه ( قَوْلُهُ : لِعُمُومِ الْآيَةِ ) أَيْ قَوْله تَعَالَى { وَلَا يَحِلُّ لَهُنَّ } الْآيَةَ ( قَوْلُهُ : ذَكَرَيْنِ ) أَيْ أَوْ رَجُلٍ وَامْرَأَتَيْنِ ( قَوْلُهُ : وَلَا تُصَدَّقُ فِيهِ ) أَيْ الْحَيْضِ ، وَقَوْلُهُ فِي تَعْلِيقٍ : أَيْ تَعْلِيقِ طَلَاقِ غَيْرِهَا عَلَى حَيْضِهَا ( قَوْلُهُ : فَيُصَدَّقُ الزَّوْجُ ) وَالْقِيَاسُ أَنَّهُ يَحْلِفُ عَلَى نَفْيِ الْعِلْمِ إنْ ادَّعَتْ عِلْمَهُ بِهِ لَا عَلَى الْبَتِّ بِنَاءً عَلَى الْقَاعِدَةِ فِيمَنْ حَلَفَ عَلَى نَفْيِ فِعْلِ غَيْرِهِ .( 22 / 384 )
