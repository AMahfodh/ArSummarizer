( فَإِنْ تَرَكَ ) الْمُكَاتَبُ ( وَلَدًا مِنْ حُرَّةٍ ) أَيْ مُعْتَقَةٍ ( وَتَرَكَ دَيْنًا يَفِي بِبَدَلِهَا فَجَنَى الْوَلَدُ فَقُضِيَ بِهِ ) بِمَا جَنَى ( عَلَى عَاقِلَةِ أُمِّهِ ) ضَرُورَةَ أَنَّ الْأَبَ لَمْ يَعْتِقْ بَعْدُ ( لَمْ يَكُنْ ذَلِكَ ) الْقَضَاءُ ( تَعْجِيزًا لِأَبِيهِ ) لِعَدَمِ الْمُنَافَاةِ وَلَا رُجُوعَ ، قَيَّدَ بِالدَّيْنِ لِأَنَّ فِي الْعَيْنِ لَا يَتَأَتَّى الْقَضَاءُ بِالْإِلْحَاقِ بِالْأُمِّ لِإِمْكَانِ الْوَفَاءِ فِي الْحَالِ .
