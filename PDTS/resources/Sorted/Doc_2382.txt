( كَحَلْقِ ) رَأْسٍ ، ( وَقَصِّ ) أَظْفَارٍ بِمِقَصٍّ ، ( وَتَقْلِيمِ ) هَا - وَالْوَاوُ بِمَعْنَى أَوْ - ( وَنَتْفِ إبِطٍ وَاكْتِحَالٍ وَإِدْهَانٍ فِي بَدَنٍ وَشَعْرٍ غِبًّا ) يَفْعَلُهُ ( يَوْمًا وَ ) يَتْرُكُهُ ( يَوْمًا ) لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { نَهَى عَنْ التَّرَجُّلِ إلَّا غِبًّا ، وَنَهَى أَنْ يَتَمَشَّطَ أَحَدُهُمْ كُلَّ يَوْمٍ } .
