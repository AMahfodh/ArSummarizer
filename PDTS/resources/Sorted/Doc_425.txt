( وَلِأُمِّ وَلَدٍ فِي الْمَوْتِ ) : أَيْ مَوْتِ سَيِّدِهَا ، ( وَ ) فِي تَنْجِيزِ ( الْعِتْقِ ) لَهَا مِنْ سَيِّدِهَا وَهُوَ حَيٌّ ( السُّكْنَى ) مُدَّةَ اسْتِبْرَائِهَا بِحَيْضَةٍ أَوْ وَضْعٍ ( وَزِيدَ ) لَهَا ( فِي الْعِتْقِ نَفَقَةُ الْحَمْلِ ) إنْ كَانَتْ حَامِلًا ، بِخِلَافِ الْمَوْتِ لِأَنَّ الْوَلَدَ وَارِثٌ .
