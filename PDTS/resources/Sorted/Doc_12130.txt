( وَإِنْ لَمْ يَقْدِرْ ) عَلَى إزَالَةِ الْمُنْكَرِ ( لَمْ يَحْضُرْ ) وَحَرُمَتْ الْإِجَابَةُ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { مَنْ كَانَ يُؤْمِنُ بِاَللَّهِ وَالْيَوْمِ الْآخِرِ فَلَا يَجْلِسْ عَلَى مَائِدَةٍ يُدَارُ عَلَيْهَا الْخَمْرُ } رَوَاهُ أَحْمَدُ مِنْ حَدِيثِ عُمَرَ وَالتِّرْمِذِيِّ مِنْ حَدِيثِ جَابِرٍ ( فَإِنْ لَمْ يَعْلَمْ ) بِالْمُنْكَرِ ( حَتَّى حَضَرَ وَشَاهَدَهُ أَزَالَهُ وَجَلَسَ ) بَعْدَ ذَلِكَ إجَابَةً لِمَنْ دَعَاهُ ( فَإِنْ لَمْ يَقْدِرْ ) عَلَى إزَالَتِهِ ( انْصَرَفَ ) لِمَا تَقَدَّمَ وَرَفَعَ { نَافِعٌ قَالَ : كُنْتُ أَسِيرُ مَعَ عَبْدِ اللَّهِ بْنِ عُمَرَ فَسَمِعَ زَمَّارَةَ رَاعٍ فَوَضَعَ إصْبَعَيْهِ فِي أُذُنَيْهِ ثُمَّ عَدَلَ عَنْ الطَّرِيقِ فَلَمْ يَزَلْ يَقُولُ : يَا نَافِعُ أَتَسْمَعُ ؟ حَتَّى قُلْتُ : لَا فَأَخْرَجَ إصْبَعَيْهِ مِنْ أُذُنَيْهِ ثُمَّ رَجَعَ إلَى الطَّرِيقِ ثُمَّ قَالَ هَكَذَا رَأَيْتُ رَسُولَ اللَّه صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ صَنَعَ } رَوَاهُ أَبُو دَاوُد وَالْخَلَّالُ .
