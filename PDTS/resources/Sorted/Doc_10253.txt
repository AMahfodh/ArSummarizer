الْأَنْسَبُ لِمَا قَبْلَهُ وَالْحَقُّ هُنَا ( قَوْلُهُ لِبَقِيَّةِ الْوَرَثَةِ إلَخْ ) فِيهِ تَأَمُّلٌ ( قَوْلُهُ أَنَّ بَقِيَّةَ إلَخْ ) خَبَرُ قَوْلِهِ وَقَضِيَّتُهُ إلَخْ ( قَوْلُهُ فَأَوْلَى إلَخْ ) فِيهِ تَأَمُّلٌ ( قَوْلُهُ وَلِلْمُوصَى بِهِ ) إلَى قَوْلِهِ وَيَظْهَرُ فِي النِّهَايَةِ وَالْمُغْنِي إلَّا قَوْلُهُ فَتَصِحُّ إلَى الْمَتْنِ ( قَوْلُهُ لِغَيْرِ مَنْ هُوَ إلَخْ ) تَصِحُّ بِهِ لِمَنْ هُوَ عَلَيْهِ وَالْعَفْوُ عَنْهُ فِي الْمَرَضِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ لَا يُبْطِلُهَا إلَخْ ) أَيْ أَمَّا الَّتِي يُبْطِلُهَا التَّأْخِيرُ فَلَا يُتَصَوَّرُ الْوَصِيَّةُ بِهَا ؛ لِأَنَّ اشْتِغَالَهُ بِالْوَصِيَّةِ يُفَوِّتُ الشُّفْعَةَ فَلَمْ يَبْقَ شَيْءٌ يُوصَى بِهِ ا ه ع ش ( قَوْلُهُ فَتَصِحُّ إلَخْ ) هَذَا التَّفْرِيعُ فِيهِ نَظَرٌ( 27 / 370 )
