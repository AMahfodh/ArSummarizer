( وَ ) تَجِبُ ( عَلَى وَارِثٍ ) حَمْلٌ ( مِنْ زَوْجٍ ) أَوْ سَيِّدٍ أَوْ وَاطِئٍ بِشُبْهَةِ ( مَيِّتٍ ) لِلْقَرَابَةِ ، ( وَ ) تَجِبُ نَفَقَةُ حَامِلٍ ( مِنْ مَالٍ حَمْلٍ مُوسِرٍ ) بِأَنْ كَانَ قَدْ وَصَّى لَهُ بِشَيْءٍ وَقَبِلَهُ لَهُ أَبُوهُ ( فَتَسْقُطُ عَنْ أَبِيهِ ) وَعَنْ وَارِثَةٍ ؛ لِأَنَّهُ صَارَ مُوسِرًا .
