( وَإِنْ أَحْضَرَهُ ) أَيْ : أَحْضَرَ الْكَفِيلُ الْمَكْفُولَ ، بِهِ ( وَامْتَنَعَ ) الْمَكْفُولُ لَهُ ( مِنْ تَسَلُّمِهِ ) بِلَا ضَرَرٍ ( بَرِئَ ) الْكَفِيلُ ( وَلَوْ لَمْ يُشْهِدْ عَلَى امْتِنَاعِهِ ) أَيْ : الْمَكْفُولِ لَهُ ( مِنْ تَسَلُّمِهِ ) وَقَالَ الْقَاضِي : يَرْفَعُهُ إلَى الْحَاكِمِ فَيُسَلِّمُهُ إلَيْهِ فَإِنْ لَمْ يَجِدْهُ أَشْهَدَ .( 10 / 356 )
