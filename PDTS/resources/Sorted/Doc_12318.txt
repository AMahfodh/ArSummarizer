قَوْلُهُ : فِي الِاسْتِحْبَابِ ) ؛ لِأَنَّهُ أَقْرَبُ إلَى التَّثْلِيثِ الَّذِي رَاعَاهُ الْمُعْتِقُ وَلِفِعْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ( قَوْلُهُ الْوُجُوبِ ) لَعَلَّهُ ؛ لِأَنَّ وَرَقَةَ الْعِتْقِ رُبَّمَا خَرَجَتْ عَلَى ثَلَاثَةٍ فَيَنْحَصِرُ الْعِتْقُ فِيهِمْ وَهُوَ أَقْرَبُ لِغَرَضِ مُعْتِقِ الثُّلُثِ فَتَأَمَّلْهُ ( قَوْلُهُ مِنْهُمْ ) أَيْ الْمُقْتَسِمِينَ ( قَوْلُهُ لَا بِظُهُورِ طَائِرٍ ) أَيْ لَا يَجُوزُ كَمَا فِي شَرْحِ الْعِرَاقِيِّ وُقُوفًا مَعَ الْوَارِدِ كَمَا فِي شَرْحِ الْإِرْشَادِ لِحَجَرٍ وَهُوَ كَالصَّرِيحِ فِي عَدَمِ حُصُولِ الْمِلْكِ بِذَلِكَ ( قَوْلُهُ وَكَأَنْ يَجُوزَ إلَخْ ) هَذَا بَحْثٌ لِلشَّيْخَيْنِ فِي قَوْل الْجُوَيْنِيِّ أَنَّهُ يَتَوَقَّف إلَخْ كَمَا يُفِيدُهُ شَرْحُ الرَّوْضِ مَعَ بَيَانِ حُكْمِ مَا تَرَكَهُ ( قَوْلُهُ : أَخَذَهُ مِنْ اللَّذَيْنِ قَبْلَهُ ) أَيْ ثُمَّ يُخْرِجُ بِاسْمِ الْأَخِيرِينَ شَرْحُ الرَّوْضِ ( قَوْلُهُ : وَيُمْكِنُ أَنْ يَبْدَأَ إلَخْ ) هَذَا مُقَابِلُ قَوْلِهِ وَيُمْكِنُ الِاحْتِرَازُ عَنْ التَّفْرِيقِ بِأَنْ لَا يَبْدَأَ إلَخْ( 20 / 343 )
