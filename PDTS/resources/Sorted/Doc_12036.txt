( وَأَمَّا فِي الْجَهْلِ ) أَيْ مِنْ أَيِّ شَيْءٍ ( مَاتَ ) وَعَدَمِ الْبَيِّنَةِ لِكُلٍّ مِنْهُمَا ( فَيُعْتَقُ سَالِمٌ لِأَنَّ الْأَصْلَ دَوَامَ الْمَرَضِ وَعَدَمَ الْبُرْءِ وَإِنْ أَتْلَفَ ثَوْبًا ) وَنَحْوَهُ مِنْ الْمُتَقَوِّمَاتِ تَعَدِّيًا أَوْ نَحْوَهُ ( فَشَهِدَتْ بَيِّنَةٌ أَنَّ قِيمَتَهُ عِشْرُونَ وَ ) شَهِدَتْ ( بَيِّنَةٌ ) أُخْرَى ( أَنَّ قِيمَتَهُ ثَلَاثُونَ لَزِمَهُ مَا اتَّفَقَا عَلَيْهِ وَهُوَ عِشْرُونَ ) دُون مَا تَعَارَضَتَا فِيهِ لِتَسَاقُطِهِمَا فِيهِ ( وَكَذَا لَوْ كَانَ بِكُلِّ ، قِيمَةٍ شَاهِدٌ ) ثَبَتَ مَا اتَّفَقَا عَلَيْهِ ( وَلَهُ ) أَيْ الْمُدَّعِي ( أَنْ يَحْلِفَ مَعَ الْآخَرِ ) الشَّاهِدِ بِالْعَشَرَةِ الزَّائِدَةِ ( عَلَى الْعَشَرَةِ كَمَا يَأْتِي آخِرُ الْبَابِ بَعْدَهُ ) كَمَا لَوْ لَمْ يَكُنْ غَيْرُهُ لِأَنَّ الشَّاهِدَ مَعَ الْيَمِينِ نِصَابٌ لَا يُعَارِضُهُ شَهَادَةُ الْوَاحِدِ قَالَ ابْنُ نَصْرِ اللَّهِ : ( لَوْ اخْتَلَفَتْ بَيِّنَتَانِ فِي قِيمَةِ عَيْنٍ( 22 / 450 )
