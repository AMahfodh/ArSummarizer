( قَوْلُهُ : فِي أَيِّ وَقْتٍ إلَخْ ) فِيهِ أَنَّهُ يَلْزَمُ أَنْ يَكُونَ الْوَقْتُ ظَرْفًا لِلْوَقْتِ وَلَا يَصِحُّ فَيُجَابُ بِأَنَّ الظَّرْفِيَّةَ غَيْرُ مُرَادَةٍ ، وَالْكَلَامُ مَبْنِيٌّ عَلَى التَّسَامُحِ وَكَأَنَّهُ قَالَ وَقْتَ الْإِحْرَامِ لِلْعُمْرَةِ أَيَّ وَقْتٍ كَانَ ( قَوْلُهُ : أَنْ يَتَحَلَّلَا ) أَيْ : بِفِعْلِ عُمْرَةٍ ( قَوْلُهُ يُحْمَلُ عَلَى الْمُحْرِمِ بِالْحَجِّ ) فِيهِ أَنَّ الْمُحْرِمَ بِالْحَجِّ لَا يَتَقَيَّدُ حَالُهُ بِأَيَّامِهَا قَوْلُهُ : كَمَا هُوَ الْمُتَبَادِرُ إلَخْ ) أَيْ : أَنَّ الْمُتَبَادِرَ مِنْ لَفْظِ التَّحَلُّلِ - أَفْرَدَ ، أَوْ ثَنَّى - رَمْيُ جَمْرَةِ الْعَقَبَةِ وَطَوَافُ الْإِفَاضَةِ ( قَوْلُهُ : وَيَكُونُ خَارِجَ إلَخْ ) وَانْظُرْ لَوْ دَخَلَ فِي الْحَرَمِ قَبْلَ الْغُرُوبِ وَلَمْ يَعْمَلْ عَمَلًا إلَّا بَعْدَ الْغُرُوبِ ، وَالظَّاهِرُ عَلَى بَحْثِهِ أَنَّ دُخُولَهُ لَغْوٌ وَيُؤْمَرُ بِالْعَوْدِ إلَى الْحِلِّ لِيَدْخُلَ مِنْهُ بَعْدَ الْغُرُوبِ وَلَمْ أَرَهُ مَنْصُوصًا قَالَهُ الْحَطَّابُ .( 7 / 306 )
