اعْتَقَدُوا تَحْرِيمَهَا ( وَيَحْرُمُ عَلَيْنَا إطْعَامُهُمْ ) أَيْ : الْيَهُودِ ( شَحْمًا ) مُحَرَّمًا عَلَيْهِمْ ( مِنْ ذَبِيحَتِنَا لِبَقَاءِ تَحْرِيمِهِ ) عَلَيْهِمْ نَصًّا لِثُبُوتِ تَحْرِيمِهِ عَلَيْهِمْ بِنَصٍّ كِتَابِنَا فَإِطْعَامُهُمْ مِنْهُ حَمْلٌ لَهُمْ عَلَى الْمَعْصِيَةِ كَإِطْعَامٍ مُسْلِمٍ مَا يَحْرُمُ عَلَيْهِ ( وَتَحِلُّ ذَبِيحَتُنَا لَهُمْ مَعَ اعْتِقَادِهِمْ تَحْرِيمَهَا ) لِقَوْلِهِ تَعَالَى : { وَطَعَامُكُمْ حِلٌّ لَهُمْ }( 11 / 397 )
