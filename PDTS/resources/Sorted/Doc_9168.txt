( وَ ) يُمْنَعُونَ ( مِنْ بِنَاءِ مَا اسْتُهْدِمَ ) مِنْ نَحْوِ كَنِيسَةٍ وَبِيعَةٍ ( أَوْ هُدِمَ ظُلْمًا مِنْهَا وَلَوْ ) كَانَ مَا اسْتُهْدِمَ أَوْ هُدِمَ ظُلْمًا مِنْهَا ( كُلُّهَا ) لِأَنَّهُ بَعْدَ الْهَدْمِ كَأَنَّهُ لَمْ يَكُنْ ( كَ ) مَا يُمْنَعُونَ مِنْ ( زِيَادَتِهَا ) أَيْ الْكَنَائِسِ وَنَحْوِهَا لِأَنَّهُ إحْدَاثٌ فِيهَا لِمَا لَمْ يَكُنْ ، فَيَدْخُلُ فِي حَدِيثِ عُمَرَ مَرْفُوعًا { لَا تُبْنَى كَنِيسَةٌ فِي الْإِسْلَامِ وَلَا يُجَدَّدُ مَا خَرِبَ مِنْهَا } وَ ( لَا ) يُمْنَعُونَ ( رَمَّ شُعْثِهَا ) أَيْ الْكَنَائِسَ وَنَحْوِهَا ؛ لِأَنَّهُمْ مَلَكُوا اسْتِدَامَتَهَا فَمَلَكُوا رَمَّ شَعْثِهَا( 4 / 310 )
