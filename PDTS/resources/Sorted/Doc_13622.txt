غُرْمِ ( الْأَرْشِ ) لِأَنَّ جَمِيعَهَا مَضْمُونٌ عَلَيْهِ فَكَذَا بَعْضُهَا ( إنْ نَقَصَتْ ) وَفِي نُسْخَةٍ تَعَيَّبَتْ ( بَعْدَ التَّمَلُّكِ ) فَلَوْ أَرَادَ الْمَالِكُ بَدَلَهَا ، وَالْمُلْتَقِطُ رَدَّهَا مَعَ الْأَرْشِ أُجِيبَ الْمُلْتَقِطُ ؛ لِأَنَّ الْعَيْنَ النَّاقِصَةَ مَعَ الْأَرْشِ كَالتَّامَّةِ وَيَتَعَيَّنُ رَدُّهَا ( بِالزَّوَائِدِ الْمُتَّصِلَةِ ) وَإِنْ حَدَثَتْ بَعْدَ التَّمَلُّكِ تَبَعًا لِلْأَصْلِ بَلْ لَوْ حَدَثَتْ قَبْلَهُ ثُمَّ انْفَصَلَتْ رَدَّهَا كَنَظِيرِهِ مِنْ الرَّدِّ بِالْعَيْبِ وَغَيْرِهِ فَلَوْ الْتَقَطَ حَائِلًا فَحَمَلَتْ قَبْلَ تَمَلُّكِهَا ، ثُمَّ وَلَدَتْ رَدَّ الْوَلَدَ مَعَ الْأُمِّ ( لَا ) بِالزَّوَائِدِ ( الْمُنْفَصِلَةِ ) الْحَادِثَةِ بَعْدَ التَّمَلُّكِ لِحُدُوثِهَا عَلَى مِلْكِ الْمُلْتَقِطِ وَتَقَدَّمَ فِي الرَّدِّ بِالْعَيْبِ أَنَّ الْحَمْلَ الْحَادِثَ بَعْدَ الشِّرَاءِ كَالْمُنْفَصِلِ فَيَكُونُ الْحَادِثُ هُنَا بَعْدَ التَّمَلُّكِ لِلْمُلْتَقِطِ ( وَإِنْ جَاءَ الْمَالِكُ وَقَدْ بِيعَتْ ) أَيْ اللُّقَطَةُ ( فَلَهُ الْفَسْخُ ) لِلْبَيْعِ ( فِي زَمَنِ الْخِيَارِ ) لِاسْتِحْقَاقِهِ الرُّجُوعَ لِعَيْنِ مَالِهِ مَعَ بَقَائِهِ وَقِيلَ لَيْسَ لَهُ الْفَسْخُ ؛ لِأَنَّ خِيَارَ الْعَقْدِ إنَّمَا يَسْتَحِقُّهُ الْعَاقِدُ دُونَ غَيْرِهِ وَهَذَا جَزَمَ بِهِ ابْنُ الْقَطَّانِ ، وَالْأَوَّلُ صَحَّحَهُ صَاحِبُ الِانْتِصَارِ وَالتَّرْجِيحُ مِنْ زِيَادَةِ الْمُصَنِّفِ قَالَ فِي الْأَصْلِ بَعْدَ( 13 / 131 )
