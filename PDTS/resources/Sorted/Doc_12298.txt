لَكِنْ فِي الْجَوْهَرَةِ أَقَلُّ مُدَّةِ حَمْلِ الشَّاةِ أَرْبَعَةُ أَشْهُرٍ وَأَقَلُّهَا لِبَقِيَّةِ الدَّوَابِّ سِتَّةُ أَشْهُرٍ ( وَ ) صَحَّ ( لَهُ إنْ بَيَّنَ ) الْمُقِرُّ ( سَبَبًا صَالِحًا ) يُتَصَوَّرُ لِلْمَحَلِّ ( كَالْإِرْثِ وَالْوَصِيَّةِ ) كَقَوْلِهِ : مَاتَ أَبُوهُ فَوَرِثَهُ أَوْ أَوْصَى لَهُ بِهِ فُلَانٌ يَجُوزُ ، وَإِلَّا - فَلَا كَمَا يَأْتِي ( فَإِنْ وَلَدَتْهُ حَيًّا لِأَقَلَّ مِنْ نِصْفِ حَوْلٍ ) مُذْ أَقَرَّ ( فَلَهُ مَا أَقَرَّ ، وَإِنْ وَلَدَتْ حَيَّيْنِ فَلَهُمَا ) نِصْفَيْنِ ، وَلَوْ أَحَدُهُمَا ذَكَرًا وَالْآخَرُ أُنْثَى فَكَذَلِكَ فِي الْوَصِيَّةِ بِخِلَافِ الْمِيرَاثِ ( وَإِنْ وَلَدَتْ مَيِّتًا فَ ) يُرَدُّ ( لِوَرَثَةِ ) ذَلِكَ ( الْمُوصِي وَالْمُورِثِ ) لِعَدَمِ أَهْلِيَّةِ الْجَنِينِ ( وَإِنْ فَسَّرَهُ بِ ) مَا لَا يُتَصَوَّرُ كَهِبَةٍ أَوْ ( بَيْعٍ أَوْ إقْرَاضٍ أَوْ أَبْهَمَ الْإِقْرَارَ ) وَلَمْ يُبَيِّنْ سَبَبًا ( لَغَا ) وَحَمَلَ مُحَمَّدٌ الْمُبْهَمَ عَلَى السَّبَبِ الصَّالِحِ ، وَبِهِ قَالَتْ الثَّلَاثَةُ
