( أَوْ ) أَيْ : وَيَحْرُمُ ( أَنْ يُقِيمَهُ ) أَيْ الْحَدَّ ( إمَامٌ أَوْ نَائِبُهُ بِعِلْمِهِ ) أَيْ : بِلَا بَيِّنَةٍ ؛ لِقَوْلِهِ تَعَالَى : { فَاسْتَشْهِدُوا عَلَيْهِنَّ أَرْبَعَةً مِنْكُمْ } وَلِقَوْلِهِ تَعَالَى : { فَإِذْ لَمْ يَأْتُوا بِالشُّهَدَاءِ فَأُولَئِكَ عِنْدَ اللَّهِ هُمْ الْكَاذِبُونَ } وَلِأَنَّهُ لَا يَجُوزُ لَهُ التَّكَلُّمُ بِهِ فَالْعَمَلُ أَوْلَى حَتَّى لَوْ رَمَاهُ بِمَا عَلِمَهُ مِنْهُ لَكَانَ قَاذِفًا بِحَدٍّ لِلْقَذْفِ( 11 / 79 )
