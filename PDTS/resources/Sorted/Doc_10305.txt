فَصْلٌ ( وَإِنْ ) ( تَجَاذَبَ حُرَّانِ ) بَصِيرَانِ أَوْ ضَرِيرَانِ أَوْ أَحَدُهُمَا ( مُكَلَّفَانِ نَحْوَ حَبْلٍ ) كَثَوْبٍ ( فَانْقَطَعَ ) مَا تَجَاذَبَاهُ ( فَسَقَطَا فَمَاتَا ) ( فَعَلَى عَاقِلَةِ كُلٍّ ) مِنْهُمَا ( دِيَةُ الْآخَرِ ) هَذَا الْمَذْهَبُ ، جَزَمَ بِهِ الْخِرَقِيِّ وَ الْمُحَرَّرُ وَ الْمُغْنِي وَ الشَّرْحُ وَالزَّرْكَشِيُّ وَالنَّظْمُ وَالْوَجِيزُ وَ الْمُنَوِّرُ وَ مُنْتَخَبُ الْآدَمِيِّ وَغَيْرُهُمْ ؛ لِتَسَبُّبِ كُلٍّ مِنْهُمَا فِي قَتْلِ الْآخَرِ ( وَقِيلَ بَلْ ) يَجِبُ ( نِصْفُهَا ) أَيْ الدِّيَةِ عَلَى عَاقِلَةِ كُلِّ وَاحِدٍ مِنْهُمَا ، قَالَ فِي الْإِنْصَافِ : وَهُوَ تَخْرِيجٌ لِبَعْضِهِمْ ( لِأَنَّهُ هَلَكَ بِفِعْلِ نَفْسِهِ وَ ) فِعْلِ صَاحِبِهِ ، ( فَيُهْدَرُ فِعْلُ نَفْسِهِ ) وَيَبْقَى فِعْلُ صَاحِبِهِ .
