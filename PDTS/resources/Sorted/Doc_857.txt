قَالَهُ فِي الْمُبْدِعِ ( وَبَرِئَ كَفِيلٌ ، وَبَطَلَ رَهْنٌ ) إنْ كَانَ ؛ لِأَنَّ الْحَوَالَةَ كَالتَّسْلِيمِ ؛ لِفَوَاتِ الْمَحِلِّ ( وَكَذَا لَوْ أَقَرَّ ) رَبُّ دَيْنٍ ( بِهِ ) ؛ أَيْ : الدَّيْنِ ( لِلْغَيْرِ ) فَيَبْرَأُ ضَامِنٌ وَكَفِيلٌ ؛ لِأَنَّهُ إنَّمَا ضَمِنَهُ لَهُ ، لَا لِلْغَيْرِ .( 8 / 435 )
