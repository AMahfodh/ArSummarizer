( أَوْ بَلَغَ صَغِيرٌ ) ذَكَرٌ أَوْ أُنْثَى ( فِي أَثْنَائِهِ ) أَيْ : يَوْمٍ مِنْ رَمَضَانَ وَهُوَ مُفْطِرٌ لَزِمَهُ إمْسَاكُ بَقِيَّةِ الْيَوْمِ لِتَكْلِيفِهِ وَالْقَضَاءُ ( مَا لَمْ يَبْلُغْ ) الصَّغِيرُ ( صَائِمًا بِسِنٍّ أَوْ احْتِلَامٍ ، وَقَدْ نَوَى ) الصَّوْمَ ( مِنْ اللَّيْلِ فَيُتِمُّ ) صَوْمَهُ ( وَيُجْزِئُ ) عَنْهُ ، فَلَا قَضَاءَ عَلَيْهِ ( كَنَذْرِ إتْمَامِ نَفْلٍ ) بِخِلَافِ صَلَاةٍ وَحَجٍّ بَلَغَ فِيهِمَا غَيْرَ مَا يَأْتِي فِي الْحَجِّ( 3 / 311 )
