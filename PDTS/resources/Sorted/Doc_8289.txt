وَالْأَوَّلُ أَظْهَرُ بَلْ مُتَعَيِّنٌ ( قَوْلُهُ : وَحَرَّمَ جَمْعٌ إلَخْ ) عِبَارَةُ النِّهَايَةِ وَشَمِلَ ذَلِكَ مَا لَوْ بَلَغُوا اثْنَيْ عَشَرَ أَلْفًا وَأَمَّا خَبَرُ { لَنْ يُغْلَبَ اثْنَا عَشَرَ أَلْفًا مِنْ قِلَّةٍ } فَالْمُرَادُ أَنَّ الْغَالِبَ إلَخْ ( قَوْلُهُ : الِانْصِرَافُ مُطْلَقًا ) أَيْ : زَادُوا عَلَى الْمِثْلَيْنِ أَمْ لَا ( قَوْلُهُ : وَبِهِ ) أَيْ : بِذَلِكَ الْخَبَرِ ( قَوْلُهُ خُصَّتْ الْآيَةُ ) أَيْ : مَفْهُومُهَا ( قَوْلُهُ : أَيْ : مُنْتَقِلًا ) إلَى قَوْلِهِ وَأَمَّا جَعْلُهُ فِي الْمُغْنِي ( قَوْلُهُ لَيَكْمُنَ ) أَيْ : يَخْتَفِيَ فِي مَوْضِعٍ فَيَهْجُمَ .
