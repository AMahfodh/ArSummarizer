فَصْلٌ ) فِي بَيَانِ أَخْذِ غَيْرِ الْمُسْلَمِ فِيهِ ( قَوْلُهُ : وَوَقْتَ أَدَائِهِ ) أَيْ وَمَا يَتَعَلَّقُ بِوَقْتِ أَدَائِهِ وَمَكَانِهِ لِأَنَّهُ لَمْ يَذْكُرْ هُنَا نَفْسَ الزَّمَانِ الَّذِي يَجِبُ التَّسْلِيمُ فِيهِ وَلَا الْمَكَانَ بَلْ عِلْمًا مِمَّا مَرَّ ( قَوْلُهُ : لَا يَصِحُّ ) أَيْ لَا يَجُوزُ لِأَنَّ عَدَمَ الْجَوَازِ لَازِمٌ لِعَدَمِ الصِّحَّةِ ( قَوْلُهُ : نِيَابَةً عَنْ الْفَاعِلِ ) وَيَجُوزُ نَصْبُهُ بِبِنَاءِ الْفِعْلِ لِلْفَاعِلِ وَجَعْلِ الْفَاعِلِ ضَمِيرًا يَعُودُ عَلَى الْمُسَلَّمِ ( قَوْلُهُ : عَلَى مَا نَقَلَهُ الرِّيمِيُّ ) نِسْبَةً إلَى رَيْمَةَ بِالْفَتْحِ مِخْلَافٌ بِالْيَمَنِ وَحِصْنٌ بِالْيَمَنِ قَامُوسٌ ( قَوْلُهُ : اللَّهُمَّ إلَّا أَنْ يَعْلَمَ ) أَيْ فَلَا يُتَوَجَّهُ النَّظَرُ ، وَإِنْ فُرِضَ اخْتِلَافٌ فَلَعَلَّهُ لِجَوَازِ أَنَّ تَأْثِيرَ الْمَطَرِ النَّازِلِ عَلَى الزَّرْعِ يُخَالِفُ تَأْثِيرَ مَا اجْتَمَعَ فِي الْوَادِي مِنْهُ ثُمَّ سُقِيَ بِهِ الزَّرْعُ لِتَكَيُّفِ الْمُجْتَمِعِ فِي الْوَادِي بِصِفَةِ أَرْضِهِ فَتَحْصُلُ لَهُ حَالَةٌ تُخَالِفُ مَا نَزَلَ مِنْ السَّمَاءِ عَلَى الزَّرْعِ بِلَا مُخَالَطَةٍ لِشَيْءٍ ( قَوْلُهُ : وَزَعَمَ بَعْضُهُمْ إلَخْ ) مُعْتَمَدٌ ( قَوْلُهُ : وَذَلِكَ ) أَيْ عَدَمُ الصِّحَّةِ .
