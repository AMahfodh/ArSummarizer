( وَلَوْ قَالَ وَاحِدٌ مِنْ ) شُهُودٍ ( ثَلَاثَةٍ فَأَكْثَرَ عَمَدْنَا قَتْلَهُ وَ ) قَالَ ( آخَرُ ) مِنْهُمْ ( أَخْطَأْنَا فَلَا قَوَدَ ) عَلَى وَاحِدٍ مِنْهُمْ لِتَمَامِ النِّصَابِ بِدُونِهِ ( وَعَلَى مَنْ قَالَ ) مِنْهُمْ ( عَمَدْنَا حِصَّتُهُ مِنْ الدِّيَةِ الْمُغَلَّظَةِ ) مُؤَاخَذَةً لَهُ بِإِقْرَارِهِ ( وَ ) عَلَى ( الْآخَرِ ) حِصَّتُهُ ( مِنْ ) الدِّيَةِ ( الْمُخَفَّفَةِ ) ؛ لِأَنَّهُ مُقْتَضَى إقْرَارِهِ( 10 / 296 )
