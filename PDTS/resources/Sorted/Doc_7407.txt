( وَ ) مُطَالَبَةُ ( الْوَكِيلِ ) ؛ لِأَنَّهُ صَارَ مُسَلِّمًا لِلْمَبِيعِ قَبْلَ أَخْذِ عِوَضِهِ ( وَهَلْ يُطَالَبُ الْوَكِيلُ بِقِيمَةِ الْعَيْنِ ) ؛ لِأَنَّهُ فَوَّتَهَا عَلَيْهِ ( أَمْ بِالثَّمَنِ ) ؛ لِأَنَّ حَقَّهُ انْتَقَلَ مِنْهَا إلَيْهِ بِالْبَيْعِ ( وَجْهَانِ ) أَصَحُّهُمَا الْأَوَّلُ كَمَا جَزَمَ بِهِ أَصْلُهُ بَعْدُ وَسَيَأْتِي قُبَيْلَ فَصْلٍ وَكَّلَهُ فِي قَضَاءِ دَيْنِهِ ( فَإِنْ قُلْنَا ) يُطَالِبُهُ ( بِالْقِيمَةِ فَأَخَذَهَا ) مِنْهُ ( طَالَبَ الْوَكِيلُ الْمُشْتَرِيَ بِالثَّمَنِ وَ ) إذَا أَخَذَهُ مِنْهُ ( دَفَعَهُ لِلْمُوَكِّلِ وَاسْتَرَدَّ )( 10 / 337 )
