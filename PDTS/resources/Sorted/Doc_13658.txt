قَالَ عُبَيْدٌ : يَعْنِي : عَنْ طِيبِ نَفْسٍ بِهِ كَمَا تَطِيبُ النَّفْسُ بِالْهِبَةِ ، وَقِيلَ : نِحْلَةٌ مِنْ اللَّهِ لِلنِّسَاءِ ، ؛ وَلِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ تَزَوَّجَ ، وَزَوَّجَ بَنَاتِهِ عَلَى صَدَاقَاتٍ ، وَلَمْ يَتْرُكْهُ فِي النِّكَاحِ ( وَتُسْتَحَبُّ تَسْمِيَتُهُ ) أَيْ : الصَّدَاقِ ( فِيهِ ) أَيْ : النِّكَاحِ لِقَوْلِهِ تَعَالَى : { وَأُحِلَّ لَكُمْ مَا وَرَاءَ ذَلِكُمْ أَنْ تَبْتَغُوا بِأَمْوَالِكُمْ مُحْصِنِينَ غَيْرَ مُسَافِحِينَ } ، وَلِمَا تَقَدَّمَ مِنْ فِعْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ؛ وَلِأَنَّ تَسْمِيَتَهُ أَقْطَعُ لِلنِّزَاعِ وَلَيْسَتْ شَرْطًا لِقَوْلِهِ تَعَالَى : { لَا جُنَاحَ عَلَيْكُمْ إنْ طَلَّقْتُمْ النِّسَاءَ مَا لَمْ تَمَسُّوهُنَّ أَوْ تَفْرِضُوا لَهُنَّ فَرِيضَةً } وَرُوِيَ أَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { زَوَّجَ رَجُلًا امْرَأَةً ، وَلَمْ يُسَمِّ لَهَا مَهْرًا } ( وَ ) يُسْتَحَبُّ ( تَخْفِيفُهُ ) أَيْ : الصَّدَاقِ لِحَدِيثِ عَائِشَةَ مَرْفُوعًا { أَعْظَمُ النِّسَاءِ بَرَكَةً أَيْسَرُهُنَّ مُؤْنَةً } رَوَاهُ أَبُو حَفْصٍ ، ، وَعَنْ أَبِي هُرَيْرَةَ { أَنَّ رَجُلًا تَزَوَّجَ امْرَأَةً مِنْ الْأَنْصَارِ فَقَالَ النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ عَلَى كَمْ تَزَوَّجْتَهَا ؟ فَقَالَ : عَلَى أَرْبَعِ أَوَاقٍ .
