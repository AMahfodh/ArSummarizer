( قَوْلُهُ وَلِأَنَّهُ إلَخْ ) عَطْفٌ عَلَى قَوْلِهِ بِذَلِكَ يُعْلَمُ إلَخْ بِحَسَبِ الْمَعْنَى ( قَوْلُهُ لَا يَمْنَعُهُ ) أَيْ : الْجَارَ الثَّانِيَ فِي الْحَدِيثِ وَكَذَا ضَمِيرُ أَنْ يَضَعَ إلَخْ ( قَوْلُهُ وَإِنْ تَضَرَّرَ ) أَيْ : الْجَارُ الْأَوَّلُ ( قَوْلُهُ فَإِنْ جُعِلَ إلَخْ ) أَيْ : كَمَا هُوَ الْمُتَبَادِرُ وَجَرَى عَلَيْهِ رِوَايَةُ أَبُو هُرَيْرَةَ رَضِيَ اللَّهُ عَنْهُ ( قَوْلُهُ لِلْأَوَّلِ ) أَيْ : لِلْجَارِ الْأَوَّلِ فِي الْحَدِيثِ ( قَوْلُهُ ذَيْنِك الْخَبَرَيْنِ ) أَيْ : الْحَسَنِ وَالصَّحِيحِ وَأَمَّا قَوْلُهُ وَفِي رِوَايَةٍ إلَخْ فَدَاخِلٌ فِي الصَّحِيحِ ( قَوْلُهُ : لِأَنَّهُ صَرِيحٌ ) أَيْ : فِي الْقَدِيمِ ( قَوْلُهُ عَدَمُ صِحَّةٍ هَذَا ) أَيْ مَا رَوَاهُ أَحْمَدُ وَأَبُو يَعْلَى ( قَوْلُهُ فَذَاكَ إلَخْ ) أَيْ : الْخَبَرُ الْمُتَّفَقُ عَلَيْهِ ( قَوْلُهُ مَا يَلْزَمُهُ ) أَيْ : الْقَدِيمَ أَيْ : حَمْلُ الْخَبَرِ الْمُتَّفَقِ عَلَيْهِ عَلَى الْقَدِيمِ( 21 / 50 )
