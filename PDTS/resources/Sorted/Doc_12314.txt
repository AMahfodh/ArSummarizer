( وَ ) الصَّاعُ : ( أَرْبَعَةُ ) أَرْطَالٍ ( وَخَمْسَةُ أَسْبَاعِ رَطْلٍ وَثُلُثُ سُبْعِ رَطْلٍ مِصْرِيٍّ ) وَمَا وَافَقَهُ ، أَيْ : أَرْبَعَةُ أَرْطَالِ وَتِسْعُ أَوَاقٍ وَسُبْعُ أُوقِيَّةٍ مِصْرِيَّةٍ ، ( وَ ) ذَلِكَ ( رَطْلٌ وَسُبْعُ ) رَطْلٍ ( دِمَشْقِيٍّ ) وَمَا وَافَقَهُ ، ( وَ ) ذَلِكَ ( إحْدَى عَشْرَةَ أُوقِيَّةً وَثَلَاثَةَ أَسْبَاعِ ) أُوقِيَّةٍ ( حَلَبِيَّةٍ ) وَمَا وَافَقَهَا وَذَلِكَ ( عَشْرُ أَوَاقٍ وَسُبْعَانِ ) مِنْ أُوقِيَّةٍ ( قُدْسِيَّةٍ ) وَمَا وَافَقَهَا قَالَ الْمُنَقِّحُ : ( وَهَذَا يَنْفَعُكَ هُنَا وَفِي الْفِطْرَةِ ) ، أَيْ : فِطْرَةِ الصَّوْمِ ، ( وَ ) فِي ( الْفِدْيَةِ ) فِي الْحَجِّ وَالْعُمْرَةِ ، ( وَ ) فِي ( الْكَفَّارَةِ ) ، أَيْ : كَفَّارَةِ ظِهَارٍ وَيَمِينٍ ( وَغَيْرِهِمَا ، وَ ) فِي ( غَيْرِهَا ) ، كَنَذْرِ الصَّدَقَةِ بِمُدٍّ أَوْ صَاعٍ .( 1 / 417 )
