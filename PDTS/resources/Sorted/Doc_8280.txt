( وَإِنْ اخْتَلَفَا فِي رَدِّهِ ) فَقَالَ الْغَاصِبُ : رَدَدْتُهُ وَأَنْكَرَهُ الْمَالِكُ فَقَوْلُ الْمَالِكِ ؛ لِأَنَّ الْأَصْلَ مَعَهُ ( أَوْ ) اخْتَلَفَا فِي عَيْبٍ ( فِيهِ بَعْدَ تَلَفِهِ ) بِأَنْ قَالَ الْغَاصِبُ : كَانَ الْعَبْدُ أَعْمَى مَثَلًا وَأَنْكَرَهُ الْمَالِكُ ( فَقَوْلُ الْمَالِكِ ) بِيَمِينِهِ ؛ لِأَنَّ الْأَصْلَ السَّلَامَةُ ( لَكِنْ لَوْ شَاهَدَتْ الْبَيِّنَةُ الْعَبْدَ مَعِيبًا عِنْدَ الْغَاصِبِ فَقَالَ الْمَالِكُ حَدَثَ ) الْعَيْبُ ( عِنْدَ الْغَاصِبِ وَقَالَ الْغَاصِبُ : بَلْ كَانَ ) الْعَيْبُ ( فِيهِ قَبْلَ غَصْبِهِ فَقَوْلُ الْغَاصِبِ ) بِيَمِينِهِ ؛ لِأَنَّهُ غَارِمٌ وَالظَّاهِرُ أَنَّ صِفَةَ الْعَبْدِ لَمْ تَتَغَيَّرْ .
