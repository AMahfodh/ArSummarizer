( وَمَتَى نَوَى مَدْيُونٌ وَفَاءً ) عَمَّا عَلَيْهِ ( بِدَفْعٍ بَرِئَ ) مِنْهُ ( وَإِلَّا ) يَنْوِ وَفَاءً ( فَتَبَرُّعٌ ) لِحَدِيثِ { وَإِنَّمَا لِكُلِّ امْرِئٍ مَا نَوَى } وَمَا ذَكَرُوهُ فِي الْأُصُولِ : أَنَّ رَدَّ الْأَمَانَةِ وَقَضَاءَ الدَّيْنِ وَاجِبٌ لَا يَقِفُ عَلَى النِّيَّةِ أَيْ نِيَّةِ التَّقَرُّبِ ( وَتَكْفِي نِيَّةُ حَاكِمٍ وَفَّاهُ قَهْرًا مِنْ ) مَالِ ( مَدْيُونٍ ) لِامْتِنَاعِهِ أَوْ مَعَ غَيْبَتِهِ لِقِيَامِهِ مَقَامَهُ وَمَنْ عَلَيْهِ دَيْنٌ لَا يَعْلَمُ بِهِ رَبُّهُ وَجَبَ عَلَيْهِ إعْلَامُهُ .( 5 / 147 )
