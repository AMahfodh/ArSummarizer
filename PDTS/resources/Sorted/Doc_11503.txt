إلَخْ ) أَيْ فِي الْفِقْهِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ هِيَ مَا كَثُرَ إلَخْ ) الْأَوْلَى هُنَا وَفِيمَا يَأْتِي تَذْكِيرُ الضَّمِيرِ ( قَوْلُهُ هِيَ مَا قَلَّ لَفْظُهَا إلَخْ ) بَقِيَ قِسْمٌ آخَرُ مَوْجُودٌ قَطْعًا وَهُوَ مَا قَلَّ لَفْظُهُ وَمَعْنَاهُ فَكَانَ الْوَجْهُ أَنْ يَقُولَ مَا قَلَّ لَفْظُهُ سَوَاءٌ كَثُرَ مَعْنَاهُ أَوْ لَا سم وَعِ ش ( قَوْلُهُ وَالْإِيجَازُ ) مُبْتَدَأٌ وَقَوْلُهُ غَيْرُ الِاخْتِصَارِ خَبَرُهُ ( قَوْلُهُ لِكَوْنِهِ إلَخْ ) عِلَّةً مُتَوَسِّطَةً بَيْنَ طَرَفَيْ الْمُدَّعِي ( قَوْلُهُ وَهُوَ ) أَيْ طُولُ الْكَلَامِ الْإِطْنَابُ جُمْلَةٌ مُعْتَرِضَةٌ ( قَوْلُهُ : لِأَنَّهُ ) أَيْ الِاخْتِصَارَ ( قَوْلُهُ وَيَشْهَدُ لَهُ ) أَيْ لِتَفْسِيرِ الِاخْتِصَارِ بِذَلِكَ ( قَوْلُهُ إذْ لَيْسَ فِي الْآيَةِ إلَخْ ) فِيهِ إشَارَةٌ إلَى أَنَّ هَذَا الْقَائِلَ يَجْعَلُ الِاخْتِصَارَ حَذْفَ عَرَضِ الْكَلَامِ ، وَإِنَّ عَرَضَهُ هُوَ تَكْرِيرُهُ سم .
