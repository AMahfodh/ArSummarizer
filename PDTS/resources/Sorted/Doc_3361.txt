ثُمَّ ذَكَرَ مَفْهُومَ : بَائِنَتَهُ بِقَوْلِهِ : ( وَكَمُرْتَجِعٍ ) لِمُطَلَّقَتِهِ الرَّجْعِيَّةِ ، ( وَإِنْ لَمْ يَمَسَّ ) : أَيْ يَطَأْهَا بَعْدَ ارْتِجَاعِهِ ( طَلَّقَ أَوْ مَاتَ ) ، فَإِنَّهَا تَسْتَأْنِفُ عِدَّةَ طَلَاقٍ أَوْ وَفَاةٍ لِأَنَّ ارْتِجَاعَهَا يَهْدِمُ الْعِدَّةَ الْأُولَى ، وَمَثَّلَ لِطُرُوِّ الِاسْتِبْرَاءِ عَلَى الْعِدَّةِ مِنْ طَلَاقٍ بِقَوْلِهِ : ( وَكَمُعْتَدَّةِ طَلَاقٍ وُطِئَتْ ) الشَّرْح ( فَاسِدًا( 6 / 106 )
