( وَلَا ) يَحْنَثُ ( بِتَسَحُّرٍ ) بِضَمِّ الْحَاءِ أَيْ أَكْلٍ آخِرَ اللَّيْلِ ( فِي ) حَلِفِهِ ( لَا أَتَعَشَّى ) مَا لَمْ يَقْصِدْ التَّضْيِيقَ بِتَرْكِ الْأَكْلِ فِي لَيْلَتِهِ فَيَحْنَثُ بِهِ ( وَ ) لَا يَحْنَثُ بِ ( ذَوَاقٍ ) لِطَعَامٍ أَوْ مَاءٍ بِلِسَانِهِ وَ ( لَمْ يَصِلْ ) الْمَذُوقُ ( جَوْفَهُ ) أَيْ الذَّائِقِ فِي حَلِفِهِ لَا آكُلُ أَوْ لَا أَشْرَبُ كَذَا ، وَمَفْهُومُهُ حِنْثُهُ إنْ وَصَلَ جَوْفَهُ وَهُوَ كَذَلِكَ وَقَدْ دَخَلَ فِي قَوْلِهِ وَبِالْبَعْضِ .
