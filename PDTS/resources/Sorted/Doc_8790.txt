( وَلَهُ دَفْعُ مُسْلِمٍ عَنْ ذِمِّيٍّ وَوَالِدٍ عَنْ وَلَدِهِ وَسَيِّدٍ عَنْ عَبْدِهِ ) ؛ لِأَنَّهُمْ مَعْصُومُونَ مَظْلُومُونَ ( وَ ) دَفْعُ ( مَالِكٍ عَنْ إتْلَافِ مِلْكِهِ ) بِإِحْرَاقٍ أَوْ تَغْرِيقٍ أَوْ نَحْوِهِ ؛ لِأَنَّهُ مُحَرَّمٌ ( فَلَوْ كَانَ ) الْمِلْكُ الَّذِي أَرَادَ مَالِكُهُ إتْلَافَهُ ( حَيَوَانًا وَجَبَ دَفْعُهُ ) عَنْهُ لِحُرْمَتِهِ ( وَيَضْمَنُ جَرَّةً ) سَقَطَتْ عَلَيْهِ مِنْ عُلُوٍّ ( لَمْ تَنْدَفِعْ عَنْهُ إلَّا بِكَسْرِهَا ) وَكَسَرَهَا إذْ لَا قَصْدَ لَهَا بِخِلَافِ الْآدَمِيِّ ، وَالْبَهِيمَةِ نَعَمْ إنْ كَانَتْ مَوْضُوعَةً بِمَحَلِّ عُدْوَانٍ كَأَنْ وُضِعَتْ بِرَوْشَنٍ أَوْ عَلَى مُعْتَدِلٍ لَكِنَّهَا مَائِلَةٌ( 20 / 203 )
