( وَ ) جَازَ بَيْعُ ( حِنْطَةٍ ) بِكَسْرِ الْحَاءِ الْمُهْمَلَةِ وَسُكُونِ النُّونِ وَإِهْمَالِ الطَّاءِ أَيْ قَمْحٍ مَثَلًا بَعْدَ يُبْسِهَا ( فِي سُنْبُلٍ ) بِضَمِّ السِّينِ الْمُهْمَلَةِ وَالْمُوَحَّدَةِ جَمْعُ سُنْبُلَةٍ كَذَلِكَ قَائِمَةٌ بِأَرْضِهَا قَبْلَ حَصْدِهَا أَوْ بَعْدَهُ قَتًّا أَوْ مَنْفُوشًا قَبْلَ دَرْسِهَا ( وَ ) فِي ( تِبْنٍ ) بَعْدَ دَرْسِهَا وَقَبْلَ تَذْرِيَتِهَا إنْ ) كَانَ الْبَيْعُ ( بِكَيْلٍ ) فِيهِمَا كَكُلِّ إرْدَبٍّ بِكَذَا ، أَوْ لَمْ يَتَأَخَّرْ تَمَامُ حَصْدِهِ وَدَرْسِهِ وَتَذْرِيَتِهِ أَكْثَرَ مِنْ نِصْفِ شَهْرٍ .( 9 / 405 )
