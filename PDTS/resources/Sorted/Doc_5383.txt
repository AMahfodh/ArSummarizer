( قَوْلُهُ كُلَّهُ ) إلَى قَوْلِهِ وَحَذَفَهُ فِي النِّهَايَةِ وَالْمُغْنِي قَوْلُ الْمَتْنِ ( لِغُرَمَائِهِ ) وَلَوْ بَاعَهُ لِأَجْنَبِيٍّ بِإِذْنِ الْغُرَمَاءِ لَمْ يَصِحَّ نِهَايَةٌ وَمُغْنِي قَوْلُ الْمَتْنِ ( بِدَيْنِهِمْ ) أَوْ بِعَيْنٍ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ بِدَيْنِهِ ) أَيْ : أَوْ بَعْضِهِ ( قَوْلُهُ بِالْأَوْلَى ) مَحَلُّ تَأَمُّلٍ ( قَوْلُهُ لِبَقَاءِ الْحَجْرِ عَلَيْهِ ) عِبَارَةُ النِّهَايَةِ وَالْمُغْنِي ؛ لِأَنَّ الْحَجْرَ يَثْبُتُ عَلَى الْعُمُومِ وَمِنْ الْجَائِزِ أَنْ يَكُونَ لَهُ غَرِيمٌ آخَرُ ا ه .
