وَلَمَّا فَرَغَ مِنْ بَيَانِ وَقْتِ الْمَغْرِبِ شَرَعَ فِي بَيَانِ وَقْتِ الْعِشَاءِ بِقَوْلِهِ : ( وَوَقْتُ صَلَاةِ الْعَتَمَةِ ) الْمُخْتَارُ : وَلَمَّا كَانَتْ تَسْمِيَتُهَا بِالْعَتَمَةِ غَرِيبَةً قَالَ : ( وَهِيَ صَلَاةُ الْعِشَاءِ ) بِكَسْرِ الْعَيْنِ وَالْمَدِّ ( وَهَذَا الِاسْمُ أَوْلَى بِهَا ) وَأَفْضَلُ مِنْ لَفْظِ الْعَتَمَةِ لِأَنَّهُ الْمُصَرَّحُ بِهِ فِي الْقُرْآنِ وَالسُّنَّةِ وَخَبَرِ وَقْتِ ( غَيْبُوبَةِ الشَّفَقِ ) وَفَسَّرَهُ بِقَوْلِهِ : ( وَالشَّفَقُ ) هُوَ ( الْحُمْرَةُ الْبَاقِيَةُ فِي ) جِهَةِ ( الْمَغْرِبِ مِنْ بَقَايَا شُعَاعِ الشَّمْسِ ) مِنْ ضَوْئِهَا كَالْقُضْبَانِ ( فَإِذَا لَمْ يَبْقَ فِي ) نَاحِيَةِ ( الْمَغْرِبِ صُفْرَةٌ وَلَا حُمْرَةٌ فَقَدْ وَجَبَ ) أَيْ دَخَلَ ( الْوَقْتُ ) الْمَذْكُورُ لِلْعِشَاءِ .
