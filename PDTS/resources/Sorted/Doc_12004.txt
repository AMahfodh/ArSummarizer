( فَرْعٌ وَيُقَدَّمُ ) وُجُوبًا ( السَّابِقُ ) لِمَجْلِسِ الْحُكْمِ إنْ جَاءُوا مُتَرَتِّبِينَ وَعُرِفَ السَّابِقُ ( وَالْعِبْرَةُ بِالْمُدَّعِي ) أَيْ بِسَبْقِهِ لَا بِسَبْقِ الْمُدَّعَى عَلَيْهِ ( فَإِنْ جُهِلَ ) السَّابِقُ ( أَوْ اسْتَوَوْا ) فِي مَجِيئِهِمْ ( أَقْرَعَ ) بَيْنَهُمْ وَقَدَّمَ مَنْ خَرَجَتْ قُرْعَتُهُ ( فَإِنْ كَثُرُوا ) وَعَسِرَ الْإِقْرَاعُ ( كَتَبَ الرِّقَاعَ ) أَيْ كَتَبَ فِيهَا أَسْمَاءَهُمْ وَصُبَّتْ بَيْنَ يَدَيْ الْقَاضِي لِيَأْخُذَهَا وَاحِدَةً وَاحِدَةً ( وَيَدَّعِي مَنْ خَرَجَ اسْمُهُ ) فِي كُلِّ مَرَّةٍ وَيُسْتَحَبُّ أَنْ يُرَتِّبَ ثِقَةً يَكْتُبُ أَسْمَاءَهُمْ يَوْمَ قَضَائِهِ لِيَعْرِفَ تَرْتِيبَهُمْ وَلَوْ قَدَّمَ الْأَسْبَقُ غَيْرَهُ عَلَى نَفْسِهِ جَازَ ذِكْرُ ذَلِكَ فِي الْأَصْلِ ( وَلَا يُقَدَّمُ سَابِقٌ وَقَارِعٌ ) أَيْ مَنْ خَرَجَتْ قُرْعَتُهُ ( إلَّا بِدَعْوَى ) وَاحِدَةٍ ، وَإِنْ اتَّحَدَ الْمُدَّعَى عَلَيْهِ دَفْعًا لِلضَّرَرِ عَنْ الْبَاقِينَ فَإِنْ كَانَ لَهُ دَعْوَى أُخْرَى انْتَظَرَ فَرَاغَهُمْ أَوْ حَضَرَ فِي مَجْلِسٍ آخَرَ
