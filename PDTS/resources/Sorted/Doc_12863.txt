( فَصْلٌ الْقَوْلُ قَوْلُ مَنْ يُنْكِرُ الْوَطْءَ ) مِنْ الزَّوْجَيْنِ بِيَمِينِهِ وَإِنْ وَافَقَ عَلَى جَرَيَانِ خَلْوَةٍ ؛ لِأَنَّ الْأَصْلَ عَدَمُهُ فَلَوْ ادَّعَى وَطْأَهَا بِتَمْكِينِهَا وَطَلَبَ تَسْلِيمَهَا إلَيْهِ فَأَنْكَرَتْهُ وَامْتَنَعَتْ لِتَسْلِيمِ الْمَهْرِ صُدِّقَتْ أَوْ ادَّعَتْ جِمَاعَهَا قَبْلَ الطَّلَاقِ وَطَلَبَتْ جَمِيعَ الْمَهْرِ فَأَنْكَرَهُ صُدِّقَ ( إلَّا ثَلَاثَةً ) أَحَدُهَا ( الْعِنِّينُ فِي ) دَعْوَى ( الْإِصَابَةِ ) بِأَنْ ادَّعَاهَا وَأَنْكَرَتْهُ فَالْقَوْلُ قَوْلُهُ بِيَمِينِهِ كَمَا مَرَّ ( وَلَوْ كَانَ مَقْطُوعَ الذَّكَرِ إنْ بَقِيَ ) مِنْهُ ( مَا يُمْكِنُ بِهِ الْوَطْءُ ) سَوَاءٌ ادَّعَى ذَلِكَ قَبْلَ الْمُدَّةِ أَمْ بَعْدَهَا ( فَإِنْ اخْتَلَفَا فِي إمْكَانِ الْوَطْءِ بِهِ ) أَيْ بِالْمَقْطُوعِ ( صُدِّقَتْ بِيَمِينِهَا ) لِزَوَالِ أَصْلِ السَّلَامَةِ وَهَذَا مَا نَقَلَهُ الْأَصْلُ عَنْ قَوْلِ الْأَكْثَرِينَ ثُمَّ قَالَ وَقَالَ صَاحِبُ الشَّامِلِ يَنْبَغِي أَنْ يُؤْخَذَ بِقَوْلِ أَهْلِ الْخِبْرَةِ كَمَا لَوْ ادَّعَتْ جَبَّهُ وَأَنْكَرَ قَالَ الْمُتَوَلِّي وَهُوَ الصَّحِيحُ ( فَإِنْ شَهِدَ أَرْبَعٌ ) مِنْ النِّسْوَةِ بِبَكَارَتِهَا ( صُدِّقَتْ بِلَا يَمِينٍ ) لِدَلَالَةِ الْبَكَارَةِ عَلَى صِدْقِهَا .
