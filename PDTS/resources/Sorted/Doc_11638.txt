وَقَالَ الشَّيْخُ تَقِيُّ الدِّينِ : يَدُ الْوَقْفِ ثَابِتَةٌ عَلَى الْمُتَّصِلِ بِهِ مَا لَمْ تَأْتِ حُجَّةٌ تَدْفَعُ مُوجِبَهَا ، كَمَعْرِفَةِ كَوْنِ الْغَارِسِ غَرَسَهَا لَهُ بِحُكْمِ إجَارَةٍ أَوْ إعَارَةٍ أَوْ غَصْبٍ ، وَيَدُ الْمُسْتَأْجِرِ عَلَى الْمَنْفَعَةِ فَلَيْسَ لَهُ دَعْوَى الْبِنَاءِ بِلَا حُجَّةٍ وَيَدُ أَهْلِ عَرْصَةٍ مُشْتَرَكَةٍ ثَابِتَةٍ عَلَى مَا فِيهَا بِحُكْمِ الِاشْتِرَاكِ إلَّا مَعَ بَيِّنَةٍ بِاخْتِصَاصِهِ بِبِنَاءٍ وَنَحْوِهِ ( وَيُنْفَقُ عَلَى ) مَوْقُوفٍ ( ذِي رُوحٍ ) كَرَقِيقٍ وَخَيْلٍ ( وَمِمَّا عَيَّنَ وَاقِفٌ ) أَنْ يُنْفَقَ مِنْهُ عَلَيْهِ رُجُوعًا لِشَرْطِهِ ( فَإِنْ لَمْ يُعَيِّنْ ) وَاقِفُهُ مَحَلًّا لِنَفَقَتِهِ ( فَ ) نَفَقَتُهُ ( مِنْ غَلَّتِهِ ) لِأَنَّ بَقَاءَهُ لَا يَكُونُ بِدُونِ الْإِنْفَاقِ عَلَيْهِ فَهُوَ مِنْ ضَرُورَتِهِ ( فَإِنْ لَمْ تَكُنْ ) لَهُ غَلَّةٌ لِضَعْفِهِ وَنَحْوِهِ ( ف ) نَفَقَتُهُ ( عَلَى مَوْقُوفٍ عَلَيْهِ مُعَيَّنٍ ) لِأَنَّهُ مِلْكُهُ .
