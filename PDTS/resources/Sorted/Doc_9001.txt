وَبِهَا نَعْلَمُ أَنَّهَا الْمُرَادَةُ لِلشَّارِحِ بِقَوْلِهِ أَوَّلًا فَإِنْ بَعَثَهُمْ إلَخْ ( قَوْلُهُ : حَقُّ تَمَلُّكِهِ ) أَيْ لَا نَفْسَ الْمِلْكِ فَلَا يُورَثُ الْمَالُ عَنْهُ بِمُجَرَّدِ ذَلِكَ بَلْ الْأَمْرُ مُفَوَّضٌ لِرَأْيِ الْوَارِثِ إنْ شَاءَ تَمَلَّكَ وَإِنْ شَاءَ أَعْرَضَ ( قَوْلُهُ : وَمَرَضُهُ ) أَيْ الْمُقَاتِلِ ( قَوْلُهُ وَالْإِغْمَاءُ ) وَيَنْبَغِي أَنَّ مَحَلَّهُ فِي الْإِغْمَاءِ إذَا لَمْ يَكُنْ نَاشِئًا عَنْ الْقِتَالِ وَإِلَّا فَهُوَ مِنْ الْمَرَضِ ( قَوْلُهُ : وَلَمْ يَسْتَحِقَّ السَّهْمَ ) أَيْ وَلَا الرَّضْخَ أَيْضًا ا ه حَجّ ( قَوْلُهُ : وَهُوَ الْمُعْتَمَدُ ) قَالَ سم عَلَى حَجّ وَهَلْ يُعْطَى السَّلَبُ أَمْ لَا ؟ فِيهِ نَظَرٌ .
