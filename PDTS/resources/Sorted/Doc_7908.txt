( بِخِلَافِ الْعَيْنِ ) الذَّهَبُ وَالْفِضَّةُ ( فَيُسْقِطُهَا ) الدَّيْنُ ( وَلَوْ ) كَانَ الدَّيْنُ ( مُؤَجَّلًا أَوْ ) كَانَ ( مَهْرًا ) عَلَيْهِ لِامْرَأَتِهِ أَوْ مُؤَخَّرًا ( أَوْ ) مُقَدَّمًا كَانَ ( نَفَقَةً كَزَوْجَةٍ ) أَوْ أَبٍ أَوْ ابْنٍ ( تَجَمَّدَتْ ) عَلَيْهِ ( أَوْ ) كَانَ ( دَيْنَ زَكَاةٍ ) انْكَسَرَتْ عَلَيْهِ ، ( لَا ) دَيْنَ ( كَفَّارَةٍ ) لِيَمِينٍ أَوْ غَيْرِهِ كَظِهَارٍ وَصَوْمٍ ، ( وَ ) لَا دَيْنَ ( هَدَى ) وَجَبَ عَلَيْهِ فِي حَجٍّ أَوْ عُمْرَةٍ فَلَا يَسْقُطَانِ زَكَاةُ الْعَيْنِ .
