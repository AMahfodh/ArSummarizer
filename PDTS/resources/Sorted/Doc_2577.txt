ع ش ( قَوْلُهُ : بِأَنَّ حَاكِمَا إلَخْ ) مُتَعَلِّقٌ بِأَقْرٍ ( قَوْلُهُ : حَكَمَ بِهِ ) أَيْ بِصِحَّةِ الْوَقْفِ ( قَوْلُهُ : وَيَجُوزُ نَقْضُ الْوَقْفِ إلَخْ ) عِبَارَةُ النِّهَايَةِ وَنَقْضُ الْوَقْفِ إلَخْ ( قَوْلُهُ : فِي حَقِّ غَيْرِهِ ) أَيْ فِي حَقِّ مَنْ يَتَلَقَّى مِنْهُ كَمَا يَأْتِي ( قَوْلُهُ : وَخَالَفَهُ التَّاجُ الْفَزَارِيّ إلَخْ ) وَهُوَ الْأَوْجَهُ .
