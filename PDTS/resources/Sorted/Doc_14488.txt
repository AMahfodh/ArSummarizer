فَصْلٌ وَلِلْقَذْفِ صَرِيحٌ وَكِنَايَةٌ ( وَصَرِيحُهُ يَا مَنْيُوكَةَ إنْ لَمْ يُفَسِّرْهُ ) قَاذِفٌ ( بِفِعْلِ زَوْجٍ أَوْ سَيِّدٍ ) فَإِنْ فَسَّرَهُ بِذَلِكَ فَلَيْسَ قَذْفًا ( وَيَتَّجِهُ ) أَنَّهُ يُقْبَلُ تَفْسِيرُهُ ذَلِكَ بِفِعْلِ زَوْجٍ أَوْ سَيِّدٍ ( وَلَوْ تَرَاخَى ) وَهُوَ مُتَّجِهٌ ( يَا مَنْيُوكَةَ يَا زَانِي يَا عَاهِرُ ، أَوْ قَدْ زَنَيْتِ أَوْ زَنَى فَرْجُكِ وَنَحْوُهُ ) كَ رَأَيْتُكِ تَزْنِي ، وَأَصْلُ الْعِهْرِ إتْيَانُ الرَّجُلِ الْمَرْأَةَ لَيْلًا لِلْفُجُورِ بِهَا ، ثُمَّ غَلَبَ عَلَى الزِّنَا ، سَوَاءٌ جَاءَهَا أَوْ جَاءَتْهُ ؛ لَيْلًا أَوْ نَهَارًا ، ( أَوْ ) قَالَ لَهُ ( يَا مَفْعُوجٍ ) بِالْفَاءِ وَالْجِيمِ نَصًّا ، لِاسْتِعْمَالِ النَّاسِ لَهُ بِمَعْنَى الْوَطْءِ فِي الدُّبُرِ ، وَأَصْلُهُ الضَّرْبُ ( أَوْ ) قَالَ لَهُ ( يَا لُوطِيُّ ) لِأَنَّهُ فِي الْعُرْفِ مَنْ يَأْتِي الذُّكُورَ ؛ لِأَنَّهُ عَمَلُ قَوْمِ لُوطٍ ( فَإِنْ قَالَ أَرَدْتُ ) بِقَوْلِي : يَا زَانِي وَنَحْوُهُ ( زَانِي الْعَيْنِ ) وَنَحْوَهُ ، ( أَوْ ) أَرَدْتُ بِقَوْلِي يَا عَاهِرُ ( عَاهِرَ الْيَدِ ) ( أَوْ ) قَالَ أَرَدْتُ بِقَوْلِي يَا لُوطِيُّ ( أَنَّكَ مِنْ قَوْمِ لُوطٍ ، أَوْ إنَّكَ تَعْمَلُ عَمَلَهُمْ مِنْ غَيْرِ إتْيَانِ الذُّكُورِ لَمْ يُقْبَلْ مِنْهُ ) ذَلِكَ ؛ لِأَنَّهُ خِلَافُ الظَّاهِرِ وَلَا دَلِيلَ عَلَيْهِ ( وَحُدَّ ) لِإِتْيَانِهِ بِصَرِيحِ الْقَذْفِ .( 18 / 168 )
