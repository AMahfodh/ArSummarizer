وَلَا يَتَقَرَّرُ الْمَهْرُ كَامِلًا ( إنْ تَحَمَّلَتْ بِمَائِهِ ) أَيْ مَنِيِّ زَوْجِهَا مِنْ غَيْرِ خَلْوَةٍ بِهَا ؛ لِأَنَّهُ لَا اسْتِمْتَاعَ مِنْهُ بِهَا ( وَيَثْبُتُ بِهِ ) أَيْ : بِتَحَمُّلِ الْمَرْأَةِ مَاءَ الرَّجُلِ ( عِدَّةٌ ) فَعَلَيْهَا أَنْ تَعْتَدَّ مِنْهُ لِاحْتِمَالِ الْحَمْلِ ( خِلَافًا لَهُ ) أَيْ : لِصَاحِبِ الْإِقْنَاعِ بِقَوْلِهِ : ( فِي ) كِتَابِ ( الْعِدَدِ ) وَلَا بِتَحَمُّلِهَا مَاءَ الرَّجُلِ ، أَيْ : وَلَا تَجِبُ عَلَيْهَا الْعِدَّةُ بِتَحَمُّلِهَا مَاءَ الرَّجُلِ .
