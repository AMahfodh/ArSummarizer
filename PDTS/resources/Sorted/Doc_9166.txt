( قَوْلُهُ وَنَازَعَ ) إلَى التَّنْبِيهِ فِي النِّهَايَةِ إلَّا قَوْلَهُ وَإِنْ لَمْ يَصْلُحْ عَمَلُهُ وَقَوْلُهُ وَلَا نَظَرَ إلَى نَعَمْ ( قَوْلُهُ وَنَازَعَ إلَخْ ) عِبَارَةُ النِّهَايَةِ وَالْمُغْنِي وَأَنَّ إلَخْ بِزِيَادَةِ أَنْ الْوَصْلِيَّةِ ( قَوْلُهُ عَلَيْهِمَا ) أَيْ : الْقَوْلَيْنِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ دُونَ غَيْرِهِمَا ) أَيْ : كَقَتْلِهِ بِمِثْلِ مَا قَتَلَ بِهِ ( قَوْلُهُ جُرْحًا فِيهِ قَوَدٌ ) أَيْ : أَمَّا غَيْرُهُ كَجَائِفَةٍ فَوَاجِبُهُ الْمَالُ ا ه مُغْنِي ( قَوْلُهُ أَوْ قَتَلَ عَقِبَهُ ) عِبَارَةُ الْمُغْنِي قَوْلُهُ فَانْدَمَلَ يُوهِمُ أَنَّ الِانْدِمَالَ قَيْدٌ لِمَحَلِّ الْخِلَافِ وَلَيْسَ مُرَادًا فَلَوْ قَطَعَ يَدَهُ ثُمَّ قَتَلَهُ( 39 / 117 )
