( وَإِنْ أَخَذَهُ ) أَيْ أَخَذَ السَّاعِي الْفَرْضَ ( مِنْ الْآخَرِ ) رَبِّ الثُّلُثَيْنِ ( رَجَعَ ) عَلَى شَرِيكِهِ ( بِقِيمَةِ ثُلُثِهِ ) أَيْ الْمُخْرَجِ لِأَنَّ لَهُ ثُلُثَ الْمَالِ ( فَإِنْ اخْتَلَفَا فِي ) قَدْرِ ( قِيمَةِ الْمَأْخُوذِ ) ( فَ ) الْقَوْلُ ( قَوْلُ الْمَرْجُوعِ عَلَيْهِ ) لِأَنَّهُ غَارِمٌ ( مَعَ يَمِينِهِ ) لِاحْتِمَالِ صِدْقِ شَرِيكِهِ ( إذَا اُحْتُمِلَ صِدْقُهُ ) فِيمَا ذَكَرَهُ قِيمَةً ، وَإِلَّا رُدَّ ، لِتَكْذِيبِ الْحِسِّ لَهُ ( وَ ) مَحَلِّهِ : إذَا ( عُدِمَتْ الْبَيِّنَةُ ) لِأَنَّهَا تَرْفَعُ النِّزَاعَ ، فَيَجِبُ الْعَمَلُ بِمَا تَقُولُهُ .( 5 / 109 )
