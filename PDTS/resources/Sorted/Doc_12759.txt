كُلِّ جِهَةٍ تَحْجُبُ الْبُعْدَى مِنْهَا ) كَأُمِّ أُمٍّ وَأُمِّ أُمِّ أُمٍّ وَأُمِّ أَبٍ وَأُمِّ أُمِّ أَبٍ ، ( وَالْقُرْبَى مِنْ جِهَةِ الْأُمِّ ) كَأُمِّ أُمٍّ ( تَحْجُبُ الْبُعْدَى مِنْ جِهَةِ الْأَبِ كَأُمِّ أُمِّ أَبٍ ، وَالْقُرْبَى مِنْ جِهَةِ الْأَبِ ) كَأُمِّ أَبٍ ( لَا تَحْجُبُ الْبُعْدَى مِنْ جِهَةِ الْأُمِّ ) كَأُمِّ أُمِّ أُمٍّ ( فِي الْأَظْهَرِ ) بَلْ يَشْتَرِكَانِ فِي السُّدُسِ ، وَالثَّانِي تَحْجُبُهَا كَالْقُرْبَى مِنْ جِهَةِ الْأُمِّ وَفُرِّقَ الْأَوَّلُ بِقُوَّةِ قَرَابَةِ الْأُمِّ يَحْجُبُهَا الْجَدَّاتُ ( وَالْأُخْتُ مِنْ الْجِهَاتِ كَالْأَخِ ) فِيمَا يَحْجُبُ فِيهِ فَيَحْجُبُ الْأُخْتَ لِأَبَوَيْنِ الْأَبُ وَالِابْنُ وَابْنُ الِابْنِ وَلِأَبٍ هَؤُلَاءِ وَأَخٌ لِأَبَوَيْنِ ، وَلِأُمِّ أَبٍ وَجَدٍّ وَوَلَدِ وَوَلَدِ ابْنٍ ( وَالْأَخَوَاتُ الْخُلَّصُ لِأَبٍ يَحْجُبُهُمْ أَيْضًا أُخْتَانِ لِأَبَوَيْنِ ) فَإِنْ كَانَ مَعَهُنَّ أَخٌ عَصَّبَهُنَّ كَمَا سَيَأْتِي ( وَالْمُعْتِقَةُ كَالْمُعْتِقِ ) يَحْجُبُهَا عَصَبَةُ النَّسَبِ ( وَكُلُّ عَصَبَةٍ ) مِمَّنْ يَحْجُبُ ( يَحْجُبُهُ أَصْحَابُ فُرُوضٍ مُسْتَغْرِقَةٍ ) لِلْمَالِ كَزَوْجٍ وَأُمٍّ وَجَدٍّ وَعَمٍّ لَا شَيْءَ لِلْعَمِّ
