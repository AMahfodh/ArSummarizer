( وَجَوَابُهُ ) أَيْ التَّخَلُّفِ عَلَى الْقَوْلِ بِأَنَّهُ قَادِحٌ ( مَنْعُ وُجُودِ الْعِلَّةِ ) فِيمَا اُعْتُرِضَ بِهِ ( أَوْ ) مَنْعُ ( انْتِفَاءِ الْحُكْمِ ) عَنْ ذَلِكَ ( إنْ لَمْ يَكُنْ انْتِفَاؤُهُ مَذْهَبَ الْمُسْتَدِلِّ ) ، وَإِلَّا فَلَا يَتَأَتَّى الْجَوَابُ بِمَنْعِهِ ( وَعِنْدَ مَنْ يَرَى الْمَوَانِعَ ) أَيْ يَعْتَبِرُهَا بِالنَّفْيِ فِي قَدْحِ التَّخَلُّفِ حَتَّى إذَا وُجِدَتْ أَوْ وَاحِدٌ مِنْهَا لَا يَقْدَحُ عِنْدَهُ ( بَيَانُهَا ) فَيَحْصُلُ الْجَوَابُ عَلَى رَأْيِهِ بِبَيَانِهَا أَوْ بَيَانِ وَاحِدٍ مِنْهَا .
