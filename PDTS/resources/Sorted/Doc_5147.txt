( قَوْلُهُ : وَإِنْ مَاتَ ) أَيْ : الْأَصْلُ ( قَوْلُهُ : فَتَجِبُ شَاةٌ ) هَلْ الْمُرَادُ شَاةٌ كَبِيرَةٌ ( قَوْلُهُ : أَوْ مَعَ آخِرِهِ ) قَالَ فِي شَرْحِ الرَّوْضِ : إنَّ ذَلِكَ قَضِيَّةُ كَلَامِهِ كَأَصْلِهِ وَأَنَّهُ ظَاهِرٌ ( قَوْلُهُ : فِي الْحَوْلِ إلَخْ ) وَظَاهِرٌ أَنَّهُ إنْ وَقَعَ الْمَوْتُ قَبْلَ آخِرِ الْحَوْلِ أَوْ مَعَ آخِرِهِ فَلَا زَكَاةَ لِذَلِكَ الْحَوْلِ أَوْ عَقِبَهُ وَجَبَ إخْرَاجُهَا مِنْ التَّرِكَةِ( 12 / 58 )
