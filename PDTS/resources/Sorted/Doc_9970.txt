( وَمَنْ لَهُ عَلَى آخَرَ حَقٌّ فَخَبَّأَ ) صَاحِبُ الْحَقِّ ( قَوْمًا ثُمَّ سَأَلَهُ ) أَيْ سَأَلَهُ الْآخَرُ ( عَنْهُ ) أَيْ عَنْ الْحَقِّ الَّذِي عَلَيْهِ ( فَأَقَرَّ بِهِ ) أَيْ بِذَلِكَ الْحَقِّ ( وَهُمْ ) أَيْ الْقَوْمُ ( يَرَوْنَهُ ) أَيْ الْمُقِرَّ ( وَيَسْمَعُونَهُ ) أَيْ يَسْمَعُونَ إقْرَارَهُ ( وَهُوَ ) أَيْ الْمُقِرُّ ( لَا يَرَاهُمْ صَحَّتْ شَهَادَتُهُمْ عَلَيْهِ ) بِذَلِكَ الْإِقْرَارِ ؛ لِأَنَّ الْإِقْرَارَ مُوجِبٌ بِنَفْسِهِ ، وَقَدْ عَلِمُوهُ وَالْعِلْمُ هُوَ الرُّكْنُ فِي إطْلَاقِ أَدَاءِ الشَّهَادَةِ قَالَ اللَّهُ تَعَالَى { إلَّا مَنْ شَهِدَ بِالْحَقِّ وَهُمْ يَعْلَمُونَ } وَقَالَ صَلَّى اللَّهُ تَعَالَى عَلَيْهِ وَسَلَّمَ { إذَا عَلِمْتَ مِثْلَ الشَّمْسِ فَاشْهَدْ وَإِلَّا فَدَعْ } .
