( وَ ) إنْ قَدِمَ ( نَهَارًا وَهُوَ ) أَيْ : النَّاذِرُ ( صَائِمٌ وَقَدْ بَيَّتَ النِّيَّةَ لِخَبَرٍ سَمِعَهُ صَحَّ ) صَوْمُهُ ( وَأَجْزَأَهُ ) لِوَفَائِهِ بِنَذْرِهِ ( وَإِلَّا ) يَكُنْ بَيَّتَ النِّيَّةَ لِخَبَرٍ سَمِعَهُ ( أَوْ كَانَ مُفْطِرًا أَوْ وَافَقَ قُدُومَهُ يَوْمًا مِنْ رَمَضَانَ أَوْ ) وَافَقَ قُدُومَهُ ( يَوْمَ عِيدٍ أَوْ ) وَافَقَ قُدُومَهُ يَوْمَ ( حَيْضِ ) نَاذِرَةٍ ( قَضَى وَكَفَّرَ ) لِأَنَّهُ نَذْرٌ مُنْعَقِدٌ لَمْ يَفِ بِهِ كَسَائِرِ النُّذُورِ ( وَإِنْ وَافَقَ قُدُومَهُ ) أَيْ : فُلَانٍ ( وَهُوَ ) أَيْ : النَّاذِرُ ( صَائِمٌ عَنْ نَذْرٍ مُعَيَّنٍ أَتَمَّهُ ) أَيْ : لِوُجُوبِهِ ( وَلَا يُسْتَحَبُّ قَضَاؤُهُ وَيَقْضِي نَذْرَ الْقُدُومِ كَصَائِمٍ فِي قَضَاءِ رَمَضَانَ أَوْ ) فِي ( كَفَّارَةٍ أَوْ نَذْرٍ مُطْلَقٍ ) فَيُتِمُّهُ وَيَقْضِي نَذْرَ الْقُدُومِ ( وَإِنْ وَافَقَ يَوْمَ نَذْرِهِ ) أَيْ : يَوْمَ قُدُومِ فُلَانٍ ( وَهُوَ ) أَيْ : النَّاذِرُ ( مَجْنُونٌ فَلَا قَضَاءَ ) عَلَيْهِ ( وَلَا كَفَّارَةَ ) لِخُرُوجِهِ عَنْ أَهْلِيَّةِ التَّكْلِيفِ فِيهِ كَمَنْ نَذَرَ صَوْمَ شَهْرٍ بِعَيْنِهِ وَجُنَّهُ ( وَنَذَرَ اعْتِكَافَهُ ) فِيمَا تَقَدَّمَ ( كَ ) نَذْرِ ( صَوْمِهِ ) عَلَى مَا تَقَدَّمَ تَفْصِيلُهُ( 12 / 44 )
