( وَكَذَا لَوْ قَالَ أَخَذْته مِنْهُ ثُمَّ فَسَّرَ ) ه ( الْوَدِيعَةِ ) قُبِلَ قَوْلُهُ بِيَمِينِهِ ( فَإِنْ تَلِفَ ) وَلَوْ بِدَعْوَاهُ ( لَمْ يَضْمَنْ ) كَمَا فِي سَائِرِ الْوَدَائِعِ ؛ وَلِأَنَّ لَفْظَهُ لَيْسَ فِيهِ مَا يَدُلُّ عَلَى ضَمَانٍ وَلَا عَلَى دَيْنِيَّةٍ ( لَا إنْ قَالَ ) لَهُ أَلْفٌ ( فِي ذِمَّتِي أَوْ دَيْنٌ عَلَيَّ ) فَلَا يُقْبَلُ تَفْسِيرُهُ الْوَدِيعَةِ بَلْ الْقَوْلُ قَوْلُ الْمُقَرِّ لَهُ بِيَمِينِهِ إذْ الْعَيْنُ لَا تُوصَفُ بِكَوْنِهَا دَيْنًا أَوْ فِي الذِّمَّةِ ( وَإِنْ قَالَ ) لَهُ عِنْدِي أَلْفٌ ( وَدِيعَةً دَيْنًا أَوْ مُضَارَبَةً دَيْنًا لَزِمَهُ ) الْأَلْفُ ( مَضْمُونًا ) عَلَيْهِ فَلَا يُصَدَّقُ فِي دَعْوَى الرَّدِّ وَالتَّلَفِ ؛ لِأَنَّ كَوْنَهُ دَيْنًا عِبَارَةٌ عَنْ كَوْنِهِ مَضْمُونًا ( فَإِنْ قَالَ ) فِي تَفْسِيرِهِ ( مُنْفَصِلًا أَرَدْت أَنَّهُ أَوْدَعَنِي ) أَوْ قَارَضَنِي ( بِشَرْطِ الضَّمَانِ ) إنْ تَلِفَ ( لَمْ يُقْبَلْ ) قَوْلُهُ ؛ لِأَنَّ شَرْطَ( 11 / 39 )
