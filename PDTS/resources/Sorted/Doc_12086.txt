وَلَمَّا تَكَلَّمَ عَلَى مَا إذَا كَانَ الرَّاجِعُ لِلْيَدِ الْأُولَى هُوَ مَا خَرَجَ مِنْهَا أَوْ مِثْلُهُ شَرَعَ فِيمَا إذَا عَادَ إلَيْهَا بَعْضُهُ بِقَوْلِهِ ( وَإِنْ اشْتَرَى ) الْبَائِعُ لِثَوْبَيْنِ مَثَلًا لِأَجَلٍ ( أَحَدَ ثَوْبَيْهِ ) ، وَلَوْ قَالَ : وَإِنْ اشْتَرَى بَعْضَ مَا بَاعَ كَانَ أَشْمَلَ ( لِأَبْعَدَ ) مِنْ الْأَجَلِ ( مُطْلَقًا ) أَيْ بِمِثْلِ الثَّمَنِ أَوْ أَقَلَّ أَوْ أَكْثَرَ ( أَوْ ) اشْتَرَى أَحَدَهُمَا بِثَمَنٍ ( أَقَلَّ ) مِنْ الثَّمَنِ الْأَوَّلِ ( نَقْدًا ) يُرِيدُ أَوْ لِدُونِ الْأَجَلِ ( امْتَنَعَ ) فِي الصُّوَرِ الْخَمْسِ لِمَا فِي الْمُسَاوِي وَالْأَكْثَرِ مِنْ سَلَفٍ جَرَّ نَفْعًا وَلَمَّا فِي الْأَقَلِّ نَقْدًا أَوْ لِدُونِ الْأَجَلِ أَوْ لِأَبْعَدَ مِنْ بَيْعٍ وَسَلَفٍ ( لَا بِمِثْلِهِ ) نَقْدًا أَوْ لِدُونِ الْأَجَلِ ( أَوْ ) بِثَمَنٍ ( أَكْثَرَ ) كَذَلِكَ فَيَجُوزُ فِي الْأَرْبَعِ صُوَرٍ كَصُوَرِ الْأَجَلِ الثَّلَاثَةِ فَالْجَوَازُ فِي سَبْعٍ .
