فَيَحِلُّ لَهُ وَطْؤُهَا بَعْدَ تَطَهُّرِهَا ( فَلَوْ أَنْكَرَتْهُ ) أَيْ الْحَيْضَ بِأَنْ قَالَتْ : لَمْ أَحِضْ لِتَمْنَعَهُ مِنْ وَطْئِهَا لِعَدَمِ الِاسْتِبْرَاءِ ( فَقَالَ : أَخْبَرَتْنِي بِهِ ) أَيْ بِأَنَّهَا حَاضَتْ وَقَدْ مَضَى مَا يُمْكِنُ حَيْضُهَا فِيهِ ( صُدِّقَ ) ؛ لِأَنَّهُ الظَّاهِرُ ( وَإِنْ ادَّعَتْ ) أَمَةٌ ( مَوْرُوثَةٌ تَحْرِيمَهَا عَلَى وَارِثٍ بِوَطْءِ مُورِثِهِ ) كَأَبِيهِ أَوْ ابْنِهِ صُدِّقَتْ وَلَعَلَّهُ مَا لَمْ تَكُنْ مَكَّنَتْهُ قَبْلُ ( أَوْ ) ادَّعَتْ أَمَةٌ ( مُشْتَرَاةٌ أَنَّ لَهَا زَوْجًا صُدِّقَتْ ) فِيهِ ؛ لِأَنَّهُ لَا يُعْرَفُ إلَّا مِنْ جِهَتِهَا .( 10 / 180 )
