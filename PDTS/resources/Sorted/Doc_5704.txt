( وَ ) إنْ قَالَ ( أَنْتِ طَالِقٌ إلَّا أَنْ يَشَاءَ زَيْدٌ فَمَاتَ ) زَيْدٌ ( أَوْ جُنَّ طَلُقَتْ فِي الْحَالِ ) لِأَنَّهُ أَوْقَعَ لِلطَّلَاقِ وَعَلَّقَ رَفْعَهُ بِشَرْطٍ وَلَمْ يُوجَدْ ، وَكَذَا لَوْ أَبَى الْمَشِيئَة ( وَإِنْ خَرِسَ ) زَيْدٌ ( فَشَاءَ بِالْإِشَارَةِ وَفُهِمَتْ ) إشَارَتُهُ ( فَكَنُطْقِهِ ) لِدَلَالَتِهَا عَلَى مَا فِي نَفْسِهِ قُلْتُ وَكَذَا يَنْبَغِي كِتَابَتُهُ ( إنْ لَمْ يُقَيِّدْ فِي التَّعْلِيقِ وَالنُّطْقِ ) فَتَتَقَيَّدُ بِهِ .( 18 / 290 )
