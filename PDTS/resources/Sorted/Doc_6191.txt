( وَيَجِبُ لَهُ ) ؛ أَيْ : الْمُفْلِسِ ( وَلِعِيَالِهِ ) مِنْ زَوْجَةٍ وَوَلَدٍ وَنَحْوِهِ ( أَدْنَى نَفَقَةً مِثْلَهُمْ مِنْ مَأْكَلٍ وَمَشْرَبٍ وَكِسْوَةٍ ) بَيَانٌ لَمَا يُنْفَقُ عَلَى مِثْلِهِمْ ، ( وَإِنَّمَا لَزِمَتْهُ ) ؛ أَيْ : الْمُفْلِسَ ( نَفَقَةُ قَرِيبٍ بِشَرْطِ ) الْآتِي فِي النَّفَقَاتِ ( لِيَسَارِهِ ) حَالًا ( بِالنِّسْبَةِ لَمَا فِي يَدِهِ ) مِنْ مَالِهِ ، فَإِذَا وُزِّعَ عَلَى الْغُرَمَاءِ ، وَلَمْ يَبْقَ لَهُ إلَّا مَا يَكْفِيهِ سَقَطَتْ .( 9 / 169 )
