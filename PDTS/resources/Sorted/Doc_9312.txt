( فَإِنْ شَرَطَ ) الْمُبْتَاعُ وَنَحْوُهُ ( قَطْعَهُ أَوْ تَضَرَّرَ الْأَصْلُ ) بِبَقَائِهِ ( أُجْبِرَ ) الْبَائِعُ وَنَحْوُهُ ( عَلَى الْقَطْعِ ) ؛ عَمَلًا بِالشَّرْطِ فِي الْأَوْلَى ، وَإِزَالَةً لِلضَّرَرِ فِي الثَّانِيَةِ ( هَذَا ) أَيْ كَوْنُ الثَّمَرِ لِلْمُعْطِي مَحَلَّهُ ( إنْ لَمْ يَشْتَرِطْهُ آخِذُ الْأَصْلِ ) وَهُوَ الْمُبْتَاعُ وَنَحْوُهُ ، لِمَا تَقَدَّمَ مِنْ حَدِيثِ ابْنِ عُمَرَ فَإِنْ اشْتَرَطَهُ كَانَ لَهُ وَمَا عَدَا الْبَيْعِ مِنْ الْمَذْكُورَاتِ مَقِيسٌ عَلَيْهِ ( بِخِلَافِ وَقْفٍ وَوَصِيَّةٍ فَإِنَّ الثَّمَرَةَ تَدْخُلُ فِيهِمَا ) إذَا أُبْقِيَتْ إلَى يَوْمِ الْمَوْتِ وَإِنْ تَشَقَّقَتْ وَظَهَرَتْ ( كَفَسْخٍ بِعَيْبٍ وَمُقَابَلَةٍ فِي بَيْعٍ وَرُجُوعِ أَبٍ فِي هِبَةٍ ) أَيْ لِوَلَدِهِ .
