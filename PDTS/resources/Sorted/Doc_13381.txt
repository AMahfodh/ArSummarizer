وَيُجَابُ عَنْهُ بِأَنَّ الْمُصَنِّفَ لَمْ يَقُلْ بَطَلَ الرَّهْنُ وَإِنَّمَا قَالَ : قُوِّمَ الْمَجْنِيُّ وَهُوَ شَامِلٌ لِلْقِصَاصِ وَالْمَالُ عَلَى مَا فَصَّلَهُ بَعْدُ ( قَوْلُهُ : بِدَلِيلِ أَنَّهُ ) أَيْ الْمَرْهُونَ ( قَوْلُهُ : حَقُّهُ ) أَيْ الْمَجْنِيِّ عَلَيْهِ ( قَوْلُهُ : الْمُتَوَثِّقِ ) أَيْ الْمُرْتَهِنِ ( قَوْلُهُ : وَقَضِيَّةُ التَّوْجِيهِ الْأَوَّلِ ) هُوَ قَوْلُهُ : سَقَطَ حَقُّهُ ( قَوْلُهُ : الْعَبْدُ ) أَيْ الْمَرْهُونُ وَقَوْلُهُ : أَنْ لَا يُقَدَّمَ أَيْ الْمَجْنِيُّ عَلَيْهِ ( قَوْلُهُ : كَمَا لَوْ كَانَ الْعَبْدُ مَغْصُوبًا ) أَيْ سَوَاءٌ تَقَدَّمَ الْغَصْبُ عَلَى الرَّهْنِ بِأَنْ رَهَنَهُ لِمَنْ يَقْدِرُ عَلَى انْتِزَاعِهِ وَقَبْضِهِ بِنَفْسِهِ أَوْ نَائِبِهِ مِنْ الْغَاصِبِ ثُمَّ اسْتَوْلَى عَلَيْهِ الْغَاصِبُ بَعْدُ أَوْ تَأَخَّرَ الْغَصْبُ عَنْ الرَّهْنِ ( قَوْلُهُ : وَيُرَدُّ بِأَنَّ الْمُعَوَّلَ إلَخْ ) التَّعْوِيلُ عَلَى مَا ذَكَرَ لَا يَصْلُحُ رَدًّا عَلَى الْمُعْتَرِضِ ، بَلْ إنَّمَا يَتِمُّ الرَّدُّ عَلَيْهِ لَوْ مَنَعَ أَنَّ مُقْتَضَى التَّعْلِيلِ مَا ذَكَرَ فَالْأَوْلَى أَنْ يُقَالَ : هُوَ وَإِنْ كَانَ قَضِيَّتُهُ ذَلِكَ لَكِنَّ الْحُكْمَ إذَا كَانَ مُعَلَّلًا بِعِلَّتَيْنِ يَبْقَى مَا( 14 / 77 )
