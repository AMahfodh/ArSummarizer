وَ ( لَا ) يَسْقُطُ خِيَارٌ ( بِرُكُوبِ دَابَّةٍ ) مَبِيعَةٍ ( بِطَرِيقِ رَدِّهَا ) ؛ لِأَنَّهُ لَا يَدُلُّ عَلَى الرِّضَا بِالنَّقْصِ أَوْ التَّغَيُّرِ ( وَإِنْ أَسْقَطَ ) مُشْتَرٍ ( حَقَّهُ مِنْ الرَّدِّ ) بِنَقْصِ صِفَةٍ شُرِطَتْ ، أَوْ تَغَيَّرَ بَعْدَ رُؤْيَتِهِ ( فَلَا أَرْشَ ) لَهُ ؛ لِأَنَّ الصِّفَةَ لَا يُعْتَاضُ عَنْهَا وَكَالْمُسْلَمِ فِيهِ( 4 / 360 )
