( وَ ) يُسَنُّ ( اغْتِسَالٌ بِصَاعٍ ) ، وَهُوَ : أَرْبَعَةُ أَمْدَادٍ ، ( وَزِنَتُهُ : سِتُّمِائَةِ ) دِرْهَمٍ ( وَخَمْسَةٌ وَثَمَانُونَ ) دِرْهَمًا ( وَخَمْسَةُ أَسْبَاعِ دِرْهَمٍ ) إسْلَامِيٍّ ، ( وَهِيَ ) بِالْمَثَاقِيلِ : ( أَرْبَعُمِائَةِ ) مِثْقَالٍ ( وَثَمَانُونَ مِثْقَالًا ، وَ ) بِالْأَرْطَالِ : ( خَمْسَةٌ أَرْطَالٍ وَثُلُثُ ) رَطْلٍ ( عِرَاقِيَّةٍ ) ، لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ لِكَعْبٍ { أَطْعِمْ سِتَّةَ مَسَاكِينَ فَرَقًا مِنْ طَعَامٍ } قَالَ أَبُو عُبَيْدٍ : لَا اخْتِلَافَ بَيْنَ النَّاسِ أَعْلَمُهُ أَنَّ الْفَرَقَ ثَلَاثَةُ آصُعٍ ، وَالْفَرَقُ : بِفَتْحِ الرَّاءِ : سِتَّةَ عَشَرَ رَطْلًا بِالْعِرَاقِيِّ ، وَيُعْتَبَرُ ( بِبُرٍّ رَزِينٍ ) - أَيْ : جَيِّدٍ - وَهِيَ مَا يُسَاوِي الْعَدَسَ فِي زِنَتِهِ .
