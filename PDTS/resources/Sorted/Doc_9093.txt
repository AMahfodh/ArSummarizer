وَإِنَّمَا جَازَ تَغْطِيَةُ قَدَمَيْهَا بِكُلِّ شَيْءٍ ؛ لِأَنَّهُمَا عَوْرَةٌ فِي الصَّلَاةِ ( وَفِيهِ ) أَيْ : لُبْسِ الْقُفَّازَيْنِ أَوْ أَحَدِهِمَا ( الْفِدْيَةُ كَالنِّقَابِ قَالَ الْقَاضِي وَمِثْلُهُمَا لَوْ لَفَّتْ عَلَى يَدَيْهَا خِرْقَةً أَوْ خِرَقًا وَشَدَّتْهَا عَلَى حِنَّاءٍ أَوْ لَا كَشَدِّهِ ) أَيْ : الرَّجُلِ ( عَلَى جَسَدِهِ شَيْئًا ) وَذَكَرَهُ فِي الْفُصُولِ عَنْ أَحْمَدَ وَجَزَمَ بِمَعْنَاهُ فِي الْمُنْتَهَى وَشَرْحِهِ ( وَظَاهِرُ كَلَامِ الْأَكْثَرِ : لَا يَحْرُمُ وَإِنْ لَفَّتْهَا بِلَا شَدٍّ فَلَا بَأْسَ ) ؛ لِأَنَّ الْمُحَرَّمَ اللُّبْسُ لَا التَّغْطِيَةُ كَيَدَيْ الرَّجُلِ وَلَا بَأْسَ أَنْ تَطُوفَ مُنْتَقِبَةً إنْ لَمْ تَكُنْ مُحْرِمَةً فَعَلَتْهُ عَائِشَةُ .( 7 / 48 )
