وَيُؤَيِّدُهُ مَا يَأْتِي عَنْ الرَّوْضَةِ ( فَلَوْ وَلَدَتْهُ لِمَا بَيْنَهُمَا ) أَيْ دُونَ السِّتَّةِ وَفَوْقَ الْأَرْبَعَةِ مِنْ الْوَطْءِ ( وَلَمْ يَسْتَبْرِئْهَا بِحَيْضَةٍ ) بَعْدَ وَطْئِهِ أَوْ اسْتَبْرَأَهَا بِهَا وَكَانَ بَيْنَ الْوِلَادَةِ وَالِاسْتِبْرَاءِ أَقَلُّ مِنْ سِتَّةِ أَشْهُرٍ ( حُرِّمَ النَّفْيُ ) لِلْوَلَدِ لِأَنَّهُ لَاحِقٌ بِفِرَاشِهِ ، وَلَا عِبْرَةَ بِرِيبَةٍ يَجِدُهَا فِي نَفْسِهِ ، وَفِي خَبَرِ أَبِي دَاوُد وَالنَّسَائِيُّ وَغَيْرِهِمَا { أَيُّمَا رَجُلٍ جَحَدَ وَلَدَهُ وَهُوَ يَنْظُرُ إلَيْهِ احْتَجَبَ اللَّهُ مِنْهُ يَوْمَ الْقِيَامَةِ وَفَضَحَهُ عَلَى رُءُوسِ الْخَلَائِقِ } ( وَإِنْ وَلَدَتْهُ فَوْقَ سِتَّةِ أَشْهُرٍ مِنْ الِاسْتِبْرَاءِ ) بِحَيْضَةٍ أَيْ مِنْ( 23 / 227 )
