الْحُكْمُ ( الثَّانِي الْأَمَانَةُ ) لِأَنَّ الْوَدِيعَ يَحْفَظُهَا لِلْمَالِكِ فَيَدُهُ كَيَدِهِ وَلَوْ ضَمِنَ لَرَغِبَ النَّاسُ عَنْ قَبُولِ الْوَدَائِعِ ( وَ ) إنَّمَا ( يَضْمَنُ بِالتَّقْصِيرِ وَلَهُ ) أَيْ لِلتَّقْصِيرِ ( أَسْبَابٌ ) ثَمَانِيَةٌ ( أَحَدُهَا : إيدَاعُهَا ) بِغَيْرِ إذْنِ مَالِكِهَا ( بِلَا عُذْرٍ ) عِنْدَ غَيْرِهِ ( وَلَوْ عِنْدَ الْقَاضِي ) ؛ لِأَنَّ الْمَالِكَ لَمْ يَرْضَ بِيَدِ غَيْرِهِ وَأَمَانَتِهِ وَلَا عُذْرَ وَاسْتَثْنَى السُّبْكِيُّ وَغَيْرُهُ مَا لَوْ طَالَتْ غَيْبَةُ الْمَالِكِ فَأَوْدَعَهَا الْوَدِيعُ الْقَاضِيَ ( وَلَهُ الِاسْتِعَانَةُ فِي حِفْظِهَا وَعَلَفِهَا ) وَسَقْيِهَا وَلَوْ بِأَجْنَبِيٍّ ( وَنَظَرُهُ ) بَاقٍ ( عَلَيْهَا كَالْعَادَةِ ) لِجَرَيَانِ الْعَادَةِ بِذَلِكَ ( فَإِنْ كَانَتْ بِمَخْزَنِهِ فَخَرَجَ لِحَاجَتِهِ وَاسْتَحْفَظَ ) عَلَيْهَا ( ثِقَةً يَخْتَصُّ بِهِ وَهُوَ يُلَاحِظُ ) بِهَا فِي عَوْدَاتِهِ ( فَلَا بَأْسَ ) بِهِ ( وَإِنْ قَطَعَ نَظَرُهُ عَنْهَا وَلَمْ يُلَاحِظْهَا فَفِي تَضْمِينِهِ تَرَدُّدٌ ) عَنْ الْإِمَامِ وَصَرَّحَ الْفُورَانِيُّ بِالْمَنْعِ وَقَالَ إنَّهُ الَّذِي يُشْعِرُ بِهِ فَحْوَى كَلَامِ الْأَئِمَّةِ ( وَإِنْ كَانَتْ فِي غَيْرِ مَسْكَنِهِ وَلَمْ يُلَاحِظْ ) هَا ( ضَمِنَ ) لِتَقْصِيرِهِ أَمَّا إذَا اُسْتُحْفِظَ غَيْرَ ثِقَةٍ أَوْ مَنْ لَا يَخْتَصُّ بِهِ فَعَلَيْهِ الضَّمَانُ .
