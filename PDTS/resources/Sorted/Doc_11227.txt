( وَحَرُمَ ) ( إخْرَاجُ نَحْوِ دَكَّةٍ ) كَدُكَّانٍ قَالَ فِي الْقَامُوسِ : وَالدَّكَّةُ - بِالْفَتْحِ - وَالدُّكَّانُ - بِالضَّمِّ - بِنَاءٌ يَصْلُحُ أَعْلَاهُ لِلْمَقْعَدِ ، وَفِي مَوْضِعٍ آخَرَ كَرُمَّانِ الْحَانُوتِ ( بِطَرِيقٍ نَافِذٍ - وَلَوْ ) كَانَ الطَّرِيقُ ( وَاسِعًا ) - سَوَاءٌ ضَرَّ بِالْمَارَّةِ أَوْ لَا ؛ لِأَنَّهُ إنْ لَمْ يَضُرَّ حَالًا فَقَدْ يَضُرُّ مَآلًا - ( وَ ) لَوْ ( أَذِنَ فِيهِ إمَامٌ ) - لِأَنَّهُ لَيْسَ لَهُ أَنْ يَأْذَنَ فِيمَا لَيْسَ لَهُ فِيهِ مَصْلَحَةٌ ؛ لَا سِيَّمَا مَعَ احْتِمَالِ أَنْ يَضُرَّ ؛ ( فَيَضْمَنُ ) مُخْرِجُ دُكَّانٍ أَوْ دَكَّةٍ ( مَا تَلِفَ بِهِ ) ؛ لِتَعَدِّيهِ ؛ كَمَا يَحْرُمُ ( حَفْرُ بِئْرٍ بِطَرِيقٍ ضَيِّقٍ ) ، وَيَضْمَنُ مَا تَلِفَ بِهِ .( 9 / 79 )
