( كِتَابُ الْهِبَةِ ) ( سُئِلَ ) رَحِمَهُ اللَّهُ عَمَّنْ قَالَ لِغَيْرِهِ وَهَبْتُك كَذَا فَلَمْ تَقْبَلْ فَقَالَ لَهُ بَلْ قَبِلْت هَلْ الْقَوْلُ قَوْلُهُ أَمْ قَوْلُ الْمُتَّهَبِ ؟ ( فَأَجَابَ ) بِأَنَّ الْقَوْلَ قَوْلُ الْمُتَّهَبِ بِيَمِينِهِ لِأَنَّهُ أَعْرَفُ بِذَلِكَ وَلِأَنَّ الْوَاهِبَ قَدْ عَقَّبَ إقْرَارَهُ بِمَا يَرْفَعُهُ .( 3 / 453 )
