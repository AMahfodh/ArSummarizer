( وَ ) حَيَوَانٌ ( مُلْتَوٍ عُنُقُهُ كَمَعْجُوزٍ عَنْهُ ) لِلْعَجْزِ عَنْ الذَّبْحِ فِي مَحَلِّهِ كَالْمُتَرَدِّيَةِ فِي بِئْرٍ ( وَمَا أَصَابَهُ سَبَبُ مَوْتٍ ) مِنْ حَيَوَانٍ مَأْكُولٍ ( مِنْ مُنْخَنِقَةٍ ) الَّتِي تَخْنُقُ حَلْقَهَا ( وَمَوْقُوذَةٍ ) ؛ أَيْ : مَضْرُوبَةٍ حَتَّى تُشْرِفَ عَلَى الْمَوْتِ ( وَمُتَرَدِّيَةٍ ) أَيْ : وَاقِعَةٍ مِنْ عُلُوٍّ كَجَبَلٍ وَحَائِطٍ وَسَاقِطَةٍ فِي نَحْوِ بِئْرٍ ( وَنَطِيحَةٍ ) ؛ بِأَنْ نَطَحَتْهَا نَحْوُ بَقَرَةٍ ( وَأَكِيلَةِ سَبُعٍ ) أَيْ : حَيَوَانٍ مُفْتَرِسٍ ؛ بِأَنْ أَكَلَ بَعْضَهَا نَحْوُ ذِئْبٍ أَوْ نَمِرٍ ( وَمَرِيضَةٍ وَمَا صِيدَ بِشَبَكَةٍ أَوْ شَرَكٍ أَوْ أُحْبُولَةٍ أَوْ فَخٍّ ) فَأَصَابَهُ شَيْءٌ مِنْ ذَلِكَ ، وَلَمْ يَصِلْ إلَى حَدٍّ( 18 / 497 )
