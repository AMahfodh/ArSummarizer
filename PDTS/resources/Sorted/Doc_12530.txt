فَصْلٌ ( فَإِنْ لَمْ يَسْتَطِعْ صَوْمًا لِكِبَرٍ أَوْ مَرَضٍ ، وَلَوْ رُجِيَ بُرْؤُهُ ) اعْتِبَارًا بِوَقْتِ الْوُجُوبِ ( أَوْ يَخَافُ زِيَادَتَهُ ) أَيْ : الْمَرَضِ ( أَوْ تَطَاوُلَهُ ) بِصَوْمِهِ ( أَوْ ) لَمْ يَسْتَطِعْ صَوْمًا ( لِشَبَقٍ ) لَا يَصْبِرُ فِيهِ عَنْ جِمَاعِ الزَّوْجَةِ إذَا لَمْ يَقْدِرْ عَلَى غَيْرِهَا ، أَوْ لِضَعْفٍ عَنْ مَعِيشَةٍ يَحْتَاجُهَا ( أَطْعَمَ سِتِّينَ مِسْكِينًا ) إجْمَاعًا ؛ لِقَوْلِهِ تَعَالَى { فَمَنْ لَمْ يَسْتَطِعْ فَإِطْعَامُ سِتِّينَ مِسْكِينًا } { وَلَمَّا أَمَرَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَوْسَ بْنَ الصَّامِتِ بِالصَّوْمِ قَالَتْ امْرَأَتُهُ : يَا رَسُولَ اللَّهِ إنَّهُ شَيْخٌ كَبِيرٌ مَا بِهِ مِنْ صِيَامٍ قَالَ : فَلْيُطْعِمْ سِتِّينَ مِسْكِينًا } { وَلَمَّا أَمَرَ سَلَمَةَ بْنَ صَخْرٍ بِالصِّيَامِ قَالَ : وَهَلْ أَصَبْت مَا أَصَبْت إلَّا مِنْ الصِّيَامِ ؟ قَالَ : فَأَطْعِمْ } ، فَنَقَلَهُ إلَيْهِ لَمَّا أَخْبَرَهُ أَنَّهُ بِهِ مِنْ الشَّبَقِ وَالشَّهْوَةِ مَا يَمْنَعُهُ مِنْ الصَّوْمِ .
