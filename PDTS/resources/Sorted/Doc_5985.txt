وَمَنْ بِيَدِهِ نَحْوُ عَبْدٍ فَقَالَ ( أَخَذْتُهُ مِنْ زَيْدٍ ) فَطَلَبَهُ زَيْدٌ ( لَزِمَ رَدُّهُ لَهُ لِاعْتِرَافِهِ ) لَهُ ( بِالْيَدِ ، وَإِنْ قَالَ مَلَكْتُهُ ) عَلَى يَدِ زَيْدٍ ( أَوْ قَالَ قَبَضْته ) عَلَى يَدِ زَيْدٍ أَوْ قَالَ وَصَلَ إلَيَّ عَلَى ( يَدِهِ ) ؛ أَيْ : زَيْدٍ ( لَمْ يُعْتَبَرْ لِزَيْدٍ قَوْلٌ ) مِنْ تَصْدِيقٍ أَوْ ضِدِّهِ ، لِأَنَّهُ لَمْ يَعْتَرِفْ لَهُ بِيَدٍ ، بَلْ كَانَ سَفِيرًا .( 20 / 282 )
