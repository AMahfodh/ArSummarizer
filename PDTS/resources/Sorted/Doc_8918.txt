النَّوْعُ ( السَّادِسُ نَذْرُ تَقَرُّبٍ كَصَلَاةٍ وَصَوْمٍ وَاعْتِكَافٍ وَصَدَقَةٍ ) مِمَّا لَا يَضُرُّهُ وَلَا عِيَالَهُ وَلَا غَرِيمَهُ ( وَحَجٍّ وَعُمْرَةٍ ) وَزِيَارَةِ أَخٍ فِي اللَّهِ تَعَالَى وَعِيَادَةِ مَرِيضٍ وَشُهُودِ جِنَازَةٍ ( بِقَصْدِ التَّقَرُّبِ مُطْلَقًا ) أَيْ : غَيْرِ مُعَلَّقٍ ( بِشَرْطٍ أَوْ عُلِّقَ بِشَرْطِ ) وُجُودِ ( نِعْمَةٍ ) يَرْجُوهَا ( أَوْ دَفْعِ نِقْمَةٍ ) يَخَافُهَا ( كَ ) قَوْلِ ( إنْ شَفَى اللَّهُ مَرِيضِي أَوْ سَلِمَ مَالِي ) لَأَتَصَدَّقَنَّ بِكَذَا ( أَوْ حَلَفَ بِقَصْدِ التَّقَرُّبِ كَ ) قَوْلِهِ ( وَاَللَّهِ لَئِنْ سَلِمَ مَالِي لَأَتَصَدَّقَنَّ بِكَذَا ، فَوُجِدَ شَرْطُهُ لَزِمَهُ ) الْوَفَاءُ بِنَذْرِهِ نَصًّا .
