( قَوْلُهُ بِأَنْ لَا يَكُونَ بِنَصِيبٍ أَصْلًا إلَخْ ) وَإِنَّمَا نَصَّ عَلَى الْمُتَوَهَّمِ ؛ لِأَنَّهُ رُبَّمَا يُتَوَهَّمُ عَدَمُ الْجَوَازِ حَيْثُ اشْتَرَطَ لِلْغُلَامِ ؛ لِأَنَّهُ اشْتَرَطَ عَلَيْهِ قَدْرٌ زَائِدٌ ( قَوْلُهُ أَنْ لَا يَقْصِدَ بِذَلِكَ تَعْلِيمَهُ ) أَيْ بَلْ قَصَدَ بِهِ إعَانَةَ الْعَامِلِ ( قَوْلُهُ وَكَأَنَّ الْمُؤَلِّفَ لَمْ يَعْتَبِرْهُ ) أَقُولُ وَبَعْضُ الشُّرَّاحِ اعْتَبَرَهُ .( 19 / 303 )
