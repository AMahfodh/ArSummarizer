( وَنَزْعِ لَامَتِهِ حَتَّى يُقَاتِلَ ) نَقَلَ ابْنُ شَاسٍ : { حَرُمَ عَلَيْهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ إذَا لَبِسَ لَأْمَتَهُ أَنْ يَخْلَعَهَا حَتَّى يُقَاتِلَ بِهَا أَوْ يَحْكُمَ اللَّهُ بَيْنَهُ وَبَيْنَ مُحَارِبِهِ } ( وَالْمَنِّ لِيَسْتَكْثِرَ ) ابْنُ الْعَرَبِيِّ : { حَرُمَ عَلَيْهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَنْ يَمُنَّ بِشَيْءٍ عَلَى أَحَدٍ لِقَوْلِهِ تَعَالَى : { وَلَا تَمْنُنْ تَسْتَكْثِرُ } أَوْ تَسْتَكْثِرُ حِينَ عَمَلِهِ } .
