( وَإِنْ قَالَ ) الشَّاهِدُ بَعْدَ أَدَاءِ شَهَادَتِهِ ( لَهُ ) أَيْ لِلْقَاضِي ( تَوَقَّفْ ) عَنْ الْحُكْمِ لَزِمَهُ التَّوَقُّفُ ( ثُمَّ ) إنْ قَالَ لَهُ بَعْدُ ( اقْضِ فَلْيَقْضِ ) ؛ لِأَنَّهُ لَمْ يَتَحَقَّقْ رُجُوعُهُ وَلَا بَطَلَتْ أَهْلِيَّتُهُ وَإِنْ عَرَضَ شَكٌّ فَقَدْ زَالَ قَالَ الْبُلْقِينِيُّ : وَيَنْبَغِي أَنْ يَسْأَلَهُ عَنْ سَبَبِ التَّوَقُّفِ هَلْ هُوَ لِشَكٍّ طَرَأَ أَمْ لِأَمْرٍ ظَهَرَ لَهُ فَإِنْ قَالَ لِشَكٍّ طَرَأَ قَالَ بَيِّنْهُ فَإِنْ ظَهَرَ مَا لَا يُؤَثِّرُ عِنْدَ الْحَاكِمِ لَمْ يَمْنَعْهُ مِنْ الْحُكْمِ ( وَلَنْ يُعِيدَا ) لِقَضَائِهِ تِلْكَ الشَّهَادَةَ ؛ لِأَنَّهَا صَدَرَتْ مِنْ أَهْلٍ جَازِمٍ ، وَالتَّوَقُّفُ الطَّارِئُ قَدْ زَالَ( 20 / 208 )
