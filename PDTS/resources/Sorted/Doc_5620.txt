( وَلَوْ ) ( بَانَ إمَامُهُ ) بَعْدَ الصَّلَاةِ عَلَى خِلَافِ ظَنِّهِ ( امْرَأَةً ) أَوْ خُنْثَى أَوْ مَجْنُونًا ( أَوْ كَافِرًا مُعْلِنًا ) كُفْرَهُ كَذِمِّيٍّ ( قِيلَ أَوْ ) بَانَ كَافِرًا ( مُخْفِيًا ) كُفْرَهُ كَزِنْدِيقٍ ( وَجَبَتْ الْإِعَادَةُ ) ؛ لِأَنَّهُ مُقَصِّرٌ بِتَرْكِ الْبَحْثِ إذْ أَمَارَةُ الْمُبْطِلِ مِنْ أُنُوثَةٍ أَوْ كُفْرٍ ظَاهِرَةٌ لَا تَخْفَى ، وَالْخُنْثَى يَنْتَشِرُ أَمْرُهُ غَالِبًا ، بِخِلَافِ الْمَخْفِيِّ فَإِنَّهُ لَا يَطَّلِعُ عَلَيْهِ فَلَا تَجِبُ الْإِعَادَةُ فِيهِ .
