عَنْ الطَّمَعِ صَدُوقَ اللَّهْجَةِ لَا يَهْزِلُ وَلَا يَمْجِنُ ) أَيْ يَمْزَحُ لِأَنَّ ذَلِكَ يُخِلُّ بِهَيْئَتِهِ ( ذَا رَأْيٍ وَمَشُورَةٍ ) لِمَا تَقَدَّمَ عَنْ عَلِيٍّ ( لِكَلَامِهِ لَيِّنٌ ذَا قُرْبٍ وَهَيْبَةٍ إذَا أَوْعَدَ وَوَفَاءٍ إذَا وَعَدَ ) يُقَالُ : وَعَدَ فِي الْخَيْرِ وَأَوْعَدَ فِي ضِدِّهِ هَذَا هُوَ الْأَصْلُ وَقَدْ يُسْتَعْمَلُ كُلٌّ مِنْهُمَا بِمَعْنَى الْآخَرِ ( وَلَا يَكُونُ ) الْقَاضِي ( جَبَّارًا وَلَا عَسُوفًا ) لِأَنَّهُ لَا يَحْصُلُ الْمَقْصُودُ بِتَوْلِيَتِهِ مِنْ وُصُولِ الْحَقِّ لِمُسْتَحِقِّهِ ( وَلَهُ أَنْ يَنْتَهِرَ الْخَصْمَ إذَا الْتَوَى ) لِأَنَّ الْحَاجَةَ دَاعِيَةٌ إلَى ذَلِكَ لِإِقَامَةِ الْعَدْلِ ( وَ ) أَنْ ( يَصِيحَ عَلَيْهِ ) أَيْ عَلَى الْخَصْمِ عِنْدَ الْتِوَائِهِ ( وَإِنْ اسْتَحَقَّ التَّعْزِيرَ عَزَّرَهُ بِمَا يَرَى مِنْ أَدَبٍ ) لَا يَزِيدُ عَلَى عَشَرَةِ أَسْوَاطٍ ( أَوْ حَبْسٍ ) .( 22 / 109 )
