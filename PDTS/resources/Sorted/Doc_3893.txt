( وَإِنْ دُعِيَ فَاسِقٌ لِتَحَمُّلِهَا ) ؛ أَيْ : الشَّهَادَةِ ( فَلَهُ الْحُضُورُ مَعَ عَدَمِ غَيْرِهِ ) إذْ التَّحَمُّلُ يُعْتَبَرُ لَهُ الْعَدَالَةُ ، فَلَوْ لَمْ يُؤَدِّ حَتَّى صَارَ عَدْلًا ، قُبِلَتْ ( وَلَا يَحْرُمُ أَدَاؤُهُ ) ؛ أَيْ : الْفَاسِقِ الشَّهَادَةَ ( وَلَوْ لَمْ يَكُنْ فِسْقُهُ ظَاهِرًا ) لِأَنَّهُ لَا يَمْنَعُ صِدْقَهُ ، وَلِهَذَا لَا يَضْمَنُ مَنْ بَانَ فِسْقُهُ .( 20 / 47 )
