الزَّوْجَ ( قَبْلَ دُخُولٍ نِصْفُهُ ) أَيْ : الْمَهْرِ ؛ لِأَنَّهُ لَا فِعْلَ لَهُمَا فِي الْفَسْخِ ؛ أَشْبَهَ مَا لَوْ طَلَّقَهَا ، ( وَ ) لَزِمَهُ ( بَعْدَهُ ) أَيْ : الدُّخُولِ ( كُلُّهُ ) أَيْ : الْمَهْرِ ؛ لِتَقَرُّرِهِ ( وَيَرْجِعُ ) زَوْجٌ بِمَا لَزِمَهُ مِنْ مَهْرٍ أَوْ نِصْفِهِ ( فِيهِمَا ) أَيْ : فِيمَا إذَا أُفْسِدَا لِغَيْرِ النِّكَاحِ قَبْلَ دُخُولٍ وَبَعْدَهُ ( عَلَى مُفْسِدٍ ) لِنِكَاحِهِ ، نَصَّ عَلَيْهِ فِي رِوَايَةِ ابْنِ قَاسِمٍ ؛ لِأَنَّ ( الْمَرْأَةَ تَسْتَحِقُّ الْمَهْرَ كُلَّهُ ) عَلَى زَوْجِهَا ، فَيَرْجِعُ بِمَا لَزِمَهُ كَنِصْفِ الْمَهْرِ فِي غَيْرِ الْمَدْخُولِ بِهَا .
