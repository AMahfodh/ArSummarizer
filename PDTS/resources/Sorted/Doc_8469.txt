( وَإِنْ وَصَّى لَهُ بِثُلُثِ عَبْدٍ فَاسْتُحِقَّ ثُلُثَاهُ فَلَهُ ) أَيْ : الْمُوصَى لَهُ ( ثُلُثُهُ الْبَاقِي إنْ خَرَجَ مِنْ الثُّلُثِ ) لِأَنَّهُ مُوصَى بِهِ خَرَجَ مِنْ الثُّلُثِ فَاسْتَحَقَّهُ كَمَا لَوْ كَانَ مُعَيَّنًا ( وَإِلَّا ) بِأَنْ لَمْ يَخْرُجْ مِنْ الثُّلُثُ فَلَمْ يَكُنْ لَهُ مَالٌ غَيْرُهُ ( فَلَهُ تِسْعَةٌ ) أَيْ : الْعَبْدِ ( إنْ لَمْ تُجِزْ الْوَرَثَةُ وَمِثْلُهُ لَوْ وَصَّى بِثُلُثِ صُبْرَةٍ مِنْ مَكِيلٍ أَوْ مَوْزُونٍ فَتَلِفَ ) ثُلُثَاهَا ( أَوْ اُسْتُحِقَّ ثُلُثَاهَا ) فَلِلْمُوصَى لَهُ الثُّلُثُ الْبَاقِي إنْ خَرَجَ مِنْ الثُّلُثِ وَإِلَّا فَالتُّسُعُ .( 15 / 279 )
