( وَإِنْ تَغَيَّرَ ) الْمَغْصُوبُ بِنَقْصٍ فِي ذَاتِهِ بِأَمْرٍ سَمَاوِيٍّ حَالَةَ كَوْنِهِ ( فِي يَدَيْهِ ) أَيْ يَدَيْ الْغَاصِبِ ( فَرَبُّهُ مُخَيَّرٌ بَيْنَ أَخْذِهِ بِنَقْصِهِ ) مِنْ غَيْرِ أَرْشِ الْعَيْبِ ( أَوْ تَضْمِينِهِ ) أَيْ الْغَاصِبِ ( الْقِيمَةَ ) يَوْمَ الْغَصْبِ ، ظَاهِرُهُ وَإِنْ تَغَيَّرَ سُوقُهُ وَهُوَ الْمَشْهُورُ ( وَلَوْ كَانَ النَّقْصُ ) فِي الْمَغْصُوبِ ( بِتَعَدِّيهِ ) أَيْ الْغَاصِبِ ( خُيِّرَ ) الْمَغْصُوبُ مِنْهُ ( أَيْضًا فِي أَخْذِهِ وَأَخْذِ ) أَيْ مَعَ أَخْذِ ( مَا نَقَصَهُ ) ك : أَوْ تَضْمِينِهِ الْقِيمَةَ يَوْمَ التَّعَدِّي .
