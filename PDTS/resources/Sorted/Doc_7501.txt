( وَالسِّقْطُ ) بِتَثْلِيثِ السِّينِ مِنْ السُّقُوطِ ( إنْ ) عُلِمَتْ حَيَاتُهُ بِأَنْ ( اسْتَهَلَّ ) أَيْ صَاحَ ( أَوْ بَكَى ) وَهُوَ مُشْتَقٌّ مِنْ الْبُكَاءِ ، وَهُوَ بِالْقَصْرِ الدَّمْعُ ، وَبِالْمَدِّ رَفْعُ الصَّوْتِ ، فَإِذَا مَاتَ بَعْدَ ذَلِكَ فَحُكْمُهُ ( كَكَبِيرٍ ) فَيُغَسَّلُ وَيُكَفَّنُ وَيُصَلَّى عَلَيْهِ وَيُدْفَنُ لِتَيَقُّنِ مَوْتِهِ بَعْدَ حَيَاتِهِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَسْتَهِلَّ أَوْ لَمْ يَبْكِ ( فَإِنْ ظَهَرَتْ أَمَارَةُ الْحَيَاةِ كَاخْتِلَاجٍ ) أَوْ تَحَرُّكٍ ( صُلِّيَ عَلَيْهِ فِي الْأَظْهَرِ ) لِاحْتِمَالِ الْحَيَاةِ بِهَذِهِ الْقَرِينَةِ الدَّالَّةِ عَلَيْهَا وَلِلِاحْتِيَاطِ .
