سم ( قَوْلُهُ : فَهُوَ ) أَيْ الْمُسْتَعِيرُ الَّذِي يَضْمَنُهُ خِلَافًا لِلْمُغْنِي ( قَوْلُهُ وَبِهَذَا ) أَيْ التَّعْلِيلِ الْمَذْكُورِ ( قَوْلُهُ : قِيَاسُ الْإِسْنَوِيِّ إلَخْ ) وَافَقَهُ الْمُغْنِي كَمَا مَرَّ فِي مَبْحَثِ تَلَفِ الْأُضْحِيَّةَ الْمَنْذُورَةِ ( قَوْلُهُ : لِهَذَا ) أَيْ مُسْتَعِيرٌ لِأُضْحِيَّةٍ مِنْ نَاذِرِهَا ( قَوْلُهُ : مِنْ نَحْوِ مُسْتَأْجِرٍ ) أَيْ كَالْمُوصَى لَهُ بِالْمَنْفَعَةِ .
