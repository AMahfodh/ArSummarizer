( قَوْلُهُ : مَا عُلِمَ مِنْهُ ) أَيْ قَوْلًا ( قَوْلُهُ : بِغَيْرِ حَجْرٍ فِي الْحَيَاةِ ) أَفْهَمَ أَنَّهُ لَوْ كَانَ سَبَبُ تَعَلُّقِهِ بِالتَّرِكَةِ الْحَجْرَ عَلَيْهِ فِي الْحَيَاةِ تَقْدِيمَ مُؤْنَةِ التَّجْهِيزِ عَلَى الدَّيْنِ الْمُتَعَلِّقِ بِهَا وَسَنَذْكُرُ عِبَارَةَ صَاحِبِ الْإِرْشَادِ ( قَوْلُهُ : وَلَوْ مِنْ غَيْرِ الْجِنْسِ ) أَيْ كَالشَّاةِ الْوَاجِبَةِ فِي الْإِبِلِ ، وَكَالْوَاجِبِ فِي مَالِ التِّجَارَةِ إنْ كَانَتْ مِنْ غَيْرِ جِنْسِ الْمَالِ ( قَوْلُهُ : فَتُؤَخَّرُ ) أَيْ عَنْ مُؤَنِ التَّجْهِيزِ ( قَوْلُهُ : أَنَّا نَخْتَارُ الْأَوَّلَ ) هُوَ قَوْلُهُ : أَنَّ تَعَلُّقَهَا تَعَلُّقُ شَرِكَةٍ ( قَوْلُهُ : بِالِاعْتِبَارِ الْمَذْكُورِ ) أَيْ وَهُوَ كَوْنُهَا لَيْسَتْ شَرِكَةً حَقِيقِيَّةً ( قَوْلُهُ : كَمَا فِي قَوْله تَعَالَى { الْحَجُّ أَشْهُرٌ مَعْلُومَاتٌ } ) أَيْ مِنْ حَيْثُ إطْلَاقُ الْأَشْهُرِ مَعَ كَوْنِ زَمَنِ الْحَجِّ شَهْرَيْنِ وَعَشْرَ لَيَالٍ ( قَوَّاهُ فَإِنْ كَانَ الْمُتَعَلِّقُ إلَخْ ) مُحْتَرَزُ قَوْلِهِ إذَا تَعَلَّقَ أَرْشُ الْجِنَايَةِ إلَخْ ( قَوْلُهُ : وَلِلْوَارِثِ التَّصَرُّفُ ) أَيْ وَيَبْقَى الْقَرْضُ فِي ذِمَّةِ الرَّقِيقِ إلَى أَنْ يَعْتِقَ ، وَمُسْتَحِقُّ الْقِصَاصِ يُمَكَّنُ مِنْ الِاقْتِصَاصِ مِنْ الْعَبْدِ مَتَى شَاءَ قَبْلَ الْبَيْعِ أَوْ بَعْدَهُ ، وَإِذَا اقْتَصَّ مِنْهُ رَجَعَ الْمُشْتَرِي عَلَى الْبَائِعِ بِمَا دَفَعَهُ إنْ كَانَ جَاهِلًا بِتَعَلُّقِ الْقِصَاصِ بِرَقَبَتِهِ وَاسْتَمَرَّ جَهْلُهُ إلَى أَنْ قُتِلَ ، فَإِنْ كَانَ عَالِمًا أَوْ جَاهِلًا وَلَمْ يُفْسَخْ عِنْدَ الْعِلْمِ فَلَا رُجُوعَ وَيَلْزَمُهُ تَجْهِيزُهُ .
