( هَذَا وَحِينَ صَارَ الْجَمْعُ مَعَ اللَّامِ كَالْمُفْرَدِ كَانَ تَقْسِيمُهُ ) أَيْ الْجَمْعِ ( مِثْلَهُ ) أَيْ الْمُفْرَدِ ( إلَّا أَنَّ كَوْنَهُ ) أَيْ الْجَمْعِ ( مَجَازًا عَنْ الْجِنْسِ يَبْعُدُ بَلْ ) هُوَ ( حَقِيقَةٌ لِكُلٍّ ) مِنْ الِاسْتِغْرَاقِ وَالْجِنْسِ ( لِلْفَهْمِ ) أَيْ فَهْمِ الْجِنْسِ مِنْهُ ( كَمَا ذَكَرْنَا فِي نَحْوِ : { الْأَئِمَّةُ مِنْ( 2 / 46 )
