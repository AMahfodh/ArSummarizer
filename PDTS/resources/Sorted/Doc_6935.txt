بِخِلَافِهِ ؛ يَحْنَثُ فِي ) حَلِفِ ( طَلَاقٍ وَعِتْقٍ ) لِأَنَّ كُلًّا مِنْهُمَا مُعَلَّقٌ بِشَرْطٍ ، وَقَدْ وُجِدَ ، وَلِأَنَّهُ تَعَلَّقَ بِهِ حَقُّ آدَمِيٍّ كَالْإِتْلَافِ ( فَقَطْ ) أَيْ : دُونَ الْيَمِينِ الْمُكَفَّرَةِ ؛ فَلَا يَحْنَثُ فِيهَا نَصًّا ؛ لِأَنَّهُ مَحْضُ حَقِّ اللَّهِ تَعَالَى فَيَدْخُلُ فِي حَدِيثِ { عُفِيَ لِأُمَّتِي عَنْ الْخَطَإِ وَالنِّسْيَانِ } ( وَ ) إنْ حَلَفَ عَنْ شَيْءٍ ( لَيَفْعَلَنَّهُ ) كَلَيَقُومَنَّ ( فَتَرَكَهُ مُكْرَهًا ) عَلَى تَرْكِهِ ؛ لَمْ يَحْنَثْ ؛ لِأَنَّ التَّرْكَ لَا يُضَافُ إلَيْهِ .
