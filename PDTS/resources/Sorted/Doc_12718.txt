( وَلَوْ قَطَعَ ) شَخْصٌ ( وَلَهُ أُصْبُعٌ أُصْبُعَيْنِ مِنْ آخَرَ قُطِعَ ) الْأُصْبُعُ الْمَوْجُودُ ( وَأُخِذَ أَرْشُ الْمَفْقُودِ فَإِنْ قَطَعَ كَامِلُ ) أَصَابِعِهِ ( نَاقِصَةَ أُصْبُعٍ ) مِنْ آخَرَ ( فَلَهُ قَطْعُ مِثْلِ أَصَابِعِهِ ) الْمَقْطُوعَةِ ( مَعَ ) أَخْذِ ( حُكُومَةِ كُلِّ الْكَفِّ ، أَوْ دِيَةِ الْمَوْجُودِ ) مِنْ الْأَصَابِعِ ( وَحُكُومَةِ خُمُسِ الْكَفِّ ) دُونَ حُكُومَةِ مَنَابِتِ الْأَرْبَعِ ؛ لِأَنَّهَا مِنْ جِنْسِ الدِّيَةِ دُونَ الْقِصَاصِ فَدَخَلَتْ فِيهَا دُونَهُ ؛ وَلِأَنَّ الدِّيَةَ بَدَلٌ حُكْمِيٌّ فَجَازَ أَنْ يُجْعَلَ بَدَلًا عَنْ الْكُلِّ ، وَالْقِصَاصُ اسْتِيفَاءُ الْمِثْلِ حِسًّا فَلَا يُمْكِنُ أَنْ يُجْعَلَ الْأَصَابِعَ وَحْدَهَا فِي مُقَابَلَةِ الْأَصَابِعِ وَمَنَابِتِهَا مَعَ التَّفَاوُتِ الْمَحْسُوسِ بَيْنَ الْجُمْلَتَيْنِ وَمَا ذَكَرَهُ يَجْرِي فِيمَا لَوْ كَانَتْ يَدُ الْجَانِي زَائِدَةً أُصْبُعَ وَيَدُ الْمَجْنِيِّ عَلَيْهِ مُعْتَدِلَةً ذَكَرَهُ الْأَصْلُ ( وَتُقْطَعُ فَاقِدَةُ الْأَصَابِعِ بِمِثْلِهَا ) لِلْمُسَاوَاةِ ( وَبِكَامِلَةٍ مَعَ دِيَةِ الْأَصَابِعِ ) هَذَا مَعْلُومٌ مِنْ أَوَّلِ الْفَصْلِ ( وَإِنْ قَطَعَ( 18 / 401 )
