( وَعَنْ تَنَصُّفِ النِّعْمَةِ ) الَّتِي لِلْحُرِّ فِي حَقِّ الرَّقِيقِ ( تَنَصَّفَ حَدُّهُ ) كَمَا نَصَّ عَلَيْهِ النَّصُّ الْقُرْآنِيُّ السَّابِقُ وَهَذَا فِيمَا يُمْكِنُ تَنْصِيفُهُ ، وَأَمَّا مَا لَا يُمْكِنُ تَنْصِيفُهُ فَيَتَكَامَلُ كَالْقَطْعِ فِي السَّرِقَةِ فَإِنَّ الْحُرَّ وَالْعَبْدَ فِيهِ سَوَاءٌ ( وَإِنَّمَا نَقَصَتْ دِيَتُهُ إذَا سَاوَتْ قِيمَتُهُ دِيَةَ الْحُرِّ ) كَمَا هُوَ قَوْلُ أَبِي حَنِيفَةَ وَمُحَمَّدٍ ( لِأَنَّهُ ) أَيْ الْمُؤَدَّى ( ضَمَانُ النَّفْسِ وَهُوَ ) أَيْ ضَمَانُ النَّفْسِ وَاجِبٌ ( بِخَطَرِهَا ) أَيْ بِسَبَبِ شَرَفِهَا ( وَهُوَ ) أَيْ خَطَرُهَا ( بِالْمَالِكِيَّةِ لِلْمَالِ وَلِمِلْكِ النِّكَاحِ وَهَذَا ) أَيْ مِلْكُ النِّكَاحِ ( مُنْتَفٍ فِي الْمَرْأَةِ ) الْحُرَّةِ إذْ هِيَ مَمْلُوكَةٌ فِيهِ لَا مَالِكَةٌ ( فَتَنَصَّفَتْ دِيَتُهَا ) عَنْ دِيَةِ الْحُرِّ الذَّكَرِ ( وَثَابِتٌ لِلْعَبْدِ مَعَ نَقْصٍ فِي ) مَالِكِيَّةِ ( الْمَالِ لِتَحَقُّقِهِ ) أَيْ مِلْكِهِ الْمَالَ ( يَدًا ) أَيْ تَصَرُّفًا ( فَقَطْ ) أَيْ لَا رَقَبَةً فَلَزِمَ بِوَاسِطَةِ نُقْصَانِ مِلْكِ الْيَدِ نُقْصَانُ شَيْءٍ مِنْ قِيمَتِهِ .
