( وَلَوْ ) ( أَوْضَحَ ) وَاحِدٌ ( فَهَشَّمَ آخَرُ ) فِي مَحَلِّ الْإِيضَاحِ وَلَوْ مُتَرَاخِيًا أَوْ عَكْسُهُ ( وَنَقَّلَ ثَالِثٌ وَأَمَّ رَابِعٌ ) وَالْمَجْنِيُّ عَلَيْهِ كَامِلٌ ( فَعَلَى كُلٍّ مِنْ الثَّلَاثِ خَمْسَةٌ ) إنْ لَمْ تُوجِبْ الْمُوضِحَةُ قَوَدًا أَوْ عَفَا عَنْهُ عَلَى الْأَرْشِ ( وَ ) عَلَى ( الرَّابِعِ تَمَامُ الثُّلُثِ ) وَهُوَ عُشْرٌ وَنِصْفُهُ وَثُلُثُهُ ، وَلَوْ دَمَغَ خَامِسٌ فَإِنْ ذَفَّفَ لَزِمَهُ دِيَةُ النَّفْسِ ، إنْ قُلْنَا بِأَنَّهَا مُذَفِّفَةٌ وَهُوَ رَأْيٌ ضَعِيفٌ وَإِلَّا فَفِيهَا حُكُومَةٌ كَمَا جَزَمَ بِهِ فِي الْعُبَابِ
