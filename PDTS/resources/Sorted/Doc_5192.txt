( وَإِنْ لَمْ يَتَّسِعْ الْوَقْتُ ) أَيْ وَقْتُ الصُّبْحِ الضَّرُورِيُّ ( إلَّا لِرَكْعَتَيْنِ ) : أَيْ لِمِقْدَارِ مَا يَسَعُهُمَا - وَلَمْ يَكُنْ صَلَّى الْوَتْرَ وَعَلَيْهِ الصُّبْحُ - ( تَرَكَ الْوَتْرَ ) وَأَدْرَكَ الصُّبْحَ ( لَا ) إنْ اتَّسَعَ الْوَقْتُ ( لِثَلَاثٍ ) : أَيْ لِقَدْرِ مَا يَسَعُ ثَلَاثَ رَكَعَاتٍ أَوْ أَرْبَعًا ؛ فَلَا يَتْرُكُهُ بَلْ يُصَلِّيهِ وَلَوْ بِالْفَاتِحَةِ فَقَطْ ، ثُمَّ يُصَلِّي الصُّبْحَ وَيُؤَخِّرُ الْفَجْرَ لِحِلِّ النَّافِلَةِ وَسَقَطَ عَنْهُ الشَّفْعُ .
