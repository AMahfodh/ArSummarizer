( قَوْلُهُ : كَجَنِينٍ ) فَإِذَا أَعْتَقَ الزَّوْجُ الْجَنِينَ الْمُخَالَعَ بِهِ شَرْعًا صَارَ حُرًّا بِبَطْنِ أُمِّهِ ( قَوْلُهُ : وَغَيْرِ مَوْصُوفٍ ) وَيَدْخُلُ فِيهِ اللُّؤْلُؤُ ( قَوْلُهُ : وَلَهُ الْوَسَطُ ) رَاجِعٌ لِقَوْلِهِ وَغَيْرِ مَوْصُوفٍ كَمَا أَفَادَهُ مُحَشِّي تت ( قَوْلُهُ : وَإِذَا انْفَشَّ الْحَمْلُ ) أَيْ أَوْ نَزَلَ مَيِّتًا وَكَذَا إذَا كَانَتْ الْأَمَةُ فِي مِلْكِ الْغَيْرِ أَيْ وَالْجَنِينُ لَمْ يَكُنْ مِلْكًا لَهَا .( 12 / 46 )
