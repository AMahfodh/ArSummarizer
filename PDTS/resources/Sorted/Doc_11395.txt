( قَوْلُهُ : إذْ لَا عُذْرَ لَهُمْ فِي تَرْكِهِ ) أَيْ وَإِنْ تَوَقَّفَ الْفَكُّ عَلَى بَذْلِ مَالٍ وَجَبَ عَلَى التَّرْتِيبِ الَّذِي قَدَّمْنَاهُ ( قَوْلُهُ : تَوَلَّى بَعْدَ عَقْدِهَا ) أَيْ الْجَائِزُ ( قَوْلُهُ : إنْ كَانَتْ فَاسِدَةً ) اُنْظُرْ مَا مَعْنَى النَّقْضِ مَعَ فَرْضِ فَسَادِهَا ، وَلَعَلَّ الْمُرَادَ بِهِ إعْلَامُهُمْ بِفَسَادِ الْهُدْنَةِ وَتَبْلِيغُهُمْ الْمَأْمَنَ ( قَوْلُهُ : بِخِلَافِ أَذَى أَهْلِ الْحَرْبِ ) أَيْ وَإِنْ قَدَرْنَا عَلَى دَفْعِهِمْ ( قَوْلُهُ أَوْ قَتْلِ مُسْلِمٍ ) أَيْ ثُمَّ إنْ لَمْ يُنْكِرْ غَيْرُ الْقَاتِلِ مَثَلًا عَلَيْهِ بَعْدَ عِلْمِهِ انْتَقَضَ عَهْدُهُ أَيْضًا كَمَا يَأْتِي ( قَوْلُهُ : أَوْ إيوَاءِ عَيْنٍ لِلْكُفَّارِ ) أَيِّ إيوَاءِ شَخْصٍ يَتَجَسَّسُ عَلَى عَوْرَاتِ الْمُسْلِمِينَ لِيَنْقُلَ الْأَخْبَارَ ( قَوْلُهُ : وَلَمْ يُنْكِرْ الْبَاقُونَ ) ظَاهِرُهُ وَإِنْ قَلُّوا جِدًّا( 27 / 89 )
