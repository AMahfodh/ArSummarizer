( قَوْلُهُ ثُمَّ رَجَعَ وَخَفَّفَهُ ) أَيْ : تَخْفِيفًا وَصَلَ إلَى دَرَجَةِ الرَّاجِحِيَّةِ بِدَلِيلِ قَوْلِهِ أَحْسَنُ ( فَإِنْ قُلْت ) حَيْثُ رَجَعَ وَخَفَّفَهُ كَانَ الْوَاجِبُ عَدَمَ ذِكْرِ الرِّوَايَةِ الْأُولَى ؛ لِأَنَّ رُجُوعَهُ بِمَثَابَةِ النَّسْخِ قُلْت : الْمَرْجُوعُ عَنْهُ لَمَّا كَانَ غَيْرَ خَارِجٍ عَنْ قَوَاعِدِهِ لَمْ يَلِغْ أَصْلًا كَمَا ذَكَرُوا .( 4 / 276 )
