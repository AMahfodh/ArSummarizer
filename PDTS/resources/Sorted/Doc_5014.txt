( وَإِنْ لَمْ يُحَلِّلَاهُمَا مِنْ نَذْرٍ ) غَيْرِ مُعَيَّنٍ ( خَالَفَا ) ، وَشَرَعَا ( فِيهِ ) بِغَيْرِ إذْنٍ ، ( صَحَّ وَأَجْزَأَ ) عَلَى الصَّحِيحِ مِنْ الْمَذْهَبِ ، ( مَعَ أَنَّهُ ) ، أَيْ : اعْتِكَافَهُمَا بِغَيْرِ إذْنٍ ، ( حَرَامٌ ، فَلَعَلَّ الْحُرْمَةَ عَارِضَةٌ ) لِحَقِّ الزَّوْجِ وَالسَّيِّدِ ، ( وَلِمُكَاتَبٍ ، لَا نَحْوُ أُمِّ وَلَدٍ ) كَمُدَبَّرٍ ، ( اعْتِكَافٌ بِلَا إذْنِ( 5 / 458 )
