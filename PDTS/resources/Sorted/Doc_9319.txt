( وَلَا يُخْرَقُ الْكَفَنُ ) لِأَنَّهُ إفْسَادٌ لَهُ وَتَقْبِيحٌ ، مَعَ الْأَمْرِ بِتَحْسِينِهِ قَالَ أَبُو الْوَفَاءِ ( وَلَوْ خِيفَ نَبْشُهُ ) قَالَ فِي الْمُبْدِعِ وَغَيْرِهِ وَهُوَ ظَاهِرُ كَلَامِ غَيْرِهِ وَجَوَّزَهُ أَبُو الْمَعَالِي إنْ خِيفَ نَبْشُهُ ( وَكَرِهَهُ ) أَيْ : تَخْرِيقُ الْكَفَنِ الْإِمَامُ ( أَحْمَدُ ) لِمَا تَقَدَّمَ ( وَإِنْ كُفِّنَ فِي قَمِيصٍ ) كَقَمِيصِ الْحَيِّ ( بِكُمَّيْنِ وَدَخَارِيصَ ، وَ ) فِي ( إزَارٍ وَلِفَافَةٍ جَازَ مِنْ غَيْرِ كَرَاهَةٍ وَظَاهِرُهُ : وَلَوْ لَمْ تَتَعَذَّرْ اللَّفَائِفُ وَيُجْعَلُ الْمِئْزَرُ مِمَّا يَلِي جَسَدَهُ ) لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { أَلْبَسَ عَبْدَ اللَّهِ بْنَ أُبَيٍّ قَمِيصَهُ( 4 / 341 )
