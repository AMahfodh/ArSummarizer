( وَ ) لَهُ ( مَنْعُ أَبَوَيْهَا الدُّخُولَ عَلَيْهَا ) مَعَ الْكَرَاهَةِ ( وَمَنْ لَا تُخْدَمُ ) فِي بَيْتِ أَبِيهَا لَا تُخْدَمُ بَلْ ( يُوَصِّلُ ) الزَّوْجُ ( مُؤْنَتَهَا إلَيْهَا ) مِنْ طَعَامٍ وَغَيْرِهِ ( وَلَهُ إخْرَاجُ خَادِمِهَا ) الَّذِي صَحِبَتْهُ ، وَلَوْ بِشِرَائِهَا لَهُ ( فَإِنْ مَرِضَ ) وَلَوْ مَرَضًا لَا يَدُومُ أَوْ زَمِنَتْ ( وَاحْتَاجَتْ خَادِمًا فَأَكْثَرَ لَزِمَهُ ) وَلَوْ كَانَتْ أَمَةً لِضَرُورَتِهَا .
