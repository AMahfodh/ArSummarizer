( وَإِنْ أَقَامَ الْخَارِجُ بَيِّنَةً أَنَّهَا مِلْكُهُ ، وَأَقَامَ الْآخَرُ ) ؛ أَيْ : الدَّاخِلُ ( بَيِّنَةً أَنَّهُ ) ، أَيْ : الْخَارِجَ ( بَاعَهَا لَهُ ) ، أَيْ : الدَّاخِلِ ( أَوْ وَقَفَهَا عَلَيْهِ ) أَيْ : الدَّاخِلِ ( أَوْ أَعْتَقَهَا ) ، أَيْ : الرَّقَبَةَ ( قُدِّمَتْ ) الْبَيِّنَةُ ( الثَّانِيَةُ ) لِشَهَادَتِهَا بِأَمْرٍ حَدَثَ عَلَى الْمِلْكِ ، خَفِيَ عَلَى الْأُولَى ، وَالْبَيْعُ أَوْ الْوَقْفُ أَوْ الْعِتْقُ مِنْهُ ( وَلَمْ تَرْفَعْ بَيِّنَةُ الْخَارِجِ يَدَهُ ) ، أَيْ الْمُدَّعَى عَلَيْهِ ( كَقَوْلِهِ أَبْرَأَنِي مِنْ الدَّيْنِ ) وَيُقِيمُ بِهِ بَيِّنَةً ، وَتُقَدَّمُ .( 19 / 497 )
