( أَوْ خَرَجَتْ لِضَرُورَةٍ ) : أَيْ وَكَذَا تَرْجِعُ لِمَسْكَنِهَا لِتَعْتَدَّ فِيهِ إذَا خَرَجَتْ مَعَ زَوْجِهَا أَوْ غَيْرِهِ لِحَجَّةِ الْفَرِيضَةِ فَطَلَّقَهَا ، أَوْ مَاتَ زَوْجُهَا ( فِي كَالثَّلَاثَةِ الْأَيَّامِ ) أَدْخَلَتْ الْكَافُ رَابِعًا لَا أَزْيَدَ : فَلَا تَرْجِعُ كَمَا لَوْ تَلَبَّسَتْ بِالْإِحْرَامِ ، ( وَ ) رَجَعَتْ إنْ خَرَجَتْ ( لِتَطَوُّعٍ ) مِنْ الْحَجِّ ( أَوْ غَيْرِهِ كَرِبَاطٍ ، وَلَوْ وَصَلَّتْ ) ذَلِكَ الْمَحَلَّ ( أَوْ أَقَامَتْ ) بِهِ وَلَوْ ( عَامًا ) عَلَى مَا رَجَّحَهُ بَعْضُهُمْ ، وَمَحَلُّ رُجُوعِهَا فِيمَا تَقَدَّمَ إنَّمَا هُوَ ( مَعَ ثِقَةٍ ) مِنْ النَّاسِ لَا مُجَرَّدَةً ، ( وَأَمْنِ طَرِيقٍ ) لَا إنْ كَانَتْ مَخُوفَةً ( إنْ أَدْرَكَتْ شَيْئًا مِنْ الْعِدَّةِ ) فِي مَنْزِلِهَا .
