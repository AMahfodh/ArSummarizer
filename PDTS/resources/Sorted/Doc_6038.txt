( وَ ) تُجْعَلُ ( مِنْ خَمْسَةٍ لَوْ ) كَانَ فِيهَا ( ثُلُثٌ وَنِصْفٌ ) كَأُخْتٍ لِأَبٍ وَأُمٍّ أَوْ أُخْتَيْنِ لِأُمٍّ وَكَأُخْتٍ لِأَبٍ وَأُمٍّ أَوْ أُمٍّ ( أَوْ سُدُسَانِ وَنِصْفٌ ) كَبِنْتٍ وَبِنْتِ ابْنٍ وَأُمٍّ ( أَوْ ثُلُثَانِ وَسُدُسٌ ) كَبِنْتَيْنِ وَأُمٍّ فَالْمَسْأَلَةُ فِي هَذِهِ الصُّوَرِ الثَّلَاثِ أَيْضًا مِنْ سِتَّةٍ وَالسِّهَامُ الَّتِي أُخِذَتْ مِنْهَا خَمْسَةٌ ، فَفِي الصُّورَةِ الْأُولَى لِلْأُخْتِ مِنْ الْأَبَوَيْنِ ثَلَاثَةُ أَسْهُمٍ وَلِلْأُخْتَيْنِ لِأُمٍّ سَهْمَانِ وَقِسْ عَلَيْهَا سَائِرَهَا .( 9 / 400 )
