( قَوْلُهُ : وَلَوْ مُحْتَرَمَةً ) أَيْ بِأَنْ كَانَتْ لِذِمِّيٍّ أَوْ لِمُسْلِمٍ عَصَرَهَا بِقَصْدِ الْخَلِّيَّةِ أَوْ بِلَا قَصْدٍ ( قَوْلُهُ : كَمَا مَرَّ ) أَيْ فِي قَوْلِهِ هِيَ لُغَةً : أَخْذُ الشَّيْءِ ( قَوْلُهُ : بِخِلَافِ جِلْدٍ دُبِغَ ) أَيْ فَإِنَّهُ يُقْطَعُ بِهِ ؛ لِأَنَّ لَهُ قِيمَةً وَقْتَ الْإِخْرَاجِ ( قَوْلُهُ : وَلَمْ يَقْصِدْ بِإِخْرَاجِهِ إرَاقَتَهَا ) أَيْ وَيُصَدَّقُ فِي ذَلِكَ ( قَوْلُهُ : أَوْ دَخَلَ بِقَصْدِ إفْسَادِهِ ) لَوْ دَخَلَ بِقَصْدِ سَرِقَتِهِ وَإِفْسَادِهِ فَلَا يَبْعُدُ عَدَمُ الْقَطْعِ لِلشُّبْهَةِ ا ه سم عَلَى حَجّ .( 26 / 61 )
