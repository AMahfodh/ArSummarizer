نَعَمْ لَوْ رَضِيَ الْمُشْتَرِي بِذِمَّةِ الشَّفِيعِ تَعَيَّنَ عَلَيْهِ الْأَخْذُ حَالًّا وَإِلَّا سَقَطَ حَقُّهُ ا ه سم عَلَى حَجّ ( قَوْلُهُ : بَلْ الْأَخْذُ بِهَا ) أَيْ بِأَنْ يَقُولَ أَخَذْت بِالشُّفْعَةِ ( قَوْلُهُ : وَإِنَّمَا لَمْ يَكُنْ تَصَرُّفُ ) أَيْ بِالْبَيْعِ وَالْإِعْتَاقِ إلَخْ ( قَوْلُهُ : فَلَا بُدَّ مِنْ أَنْ يَرْجِعَ ) أَيْ بِلَفْظٍ يَدُلُّ عَلَيْهِ ( قَوْلُهُ : عَنْ تَصَرُّفِهِ ) أَيْ وَهُوَ الْهِبَةُ ( قَوْلُهُ : لِتَعَدِّي الْمُشْتَرِي ) أَيْ لِأَنَّ كُلَّ جُزْءٍ مُشْتَرَكٌ بَيْنَهُ وَبَيْنَ الشَّرِيكِ الْقَدِيمِ ، وَقَدْ فَعَلَ بِلَا إذْنٍ مِنْهُ ( قَوْلُهُ : بَعْدَ الْقِسْمَةِ ) وَيَنْبَغِي أَنَّ مِثْلَ الْقِسْمَةِ مَا لَوْ اسْتَأْذَنَ الشَّفِيعَ فَأَذِنَ لِظَنِّهِ أَنَّهُ اتَّهَبَهُ أَوْ اسْتَعَارَ مِنْ الْبَائِعِ نَصِيبَهُ أَوْ اسْتَأْجَرَ مِنْهُ ( قَوْلُهُ : لَمْ يَقْلَعْ مَجَّانًا ) أَيْ بَلْ يُخَيَّرُ الشَّفِيعُ بَعْدَ الْأَخْذِ بَيْنَ التَّمَلُّكِ بِالْقِيمَةِ وَالْقَلْعِ مَعَ أَرْشِ النَّقْصِ( 17 / 12 )
