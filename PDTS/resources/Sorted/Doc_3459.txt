( وَ ) فِي ( لِسَانِ ) لِنَاطِقٍ ، ( وَلَوْ لَاكِنٍ وَأَرَتَّ ) بِالْمُثَنَّاةِ ( وَأَلْثَغَ ) بِالْمُثَلَّثَةِ ( وَطِفْلٍ دِيَةٌ ) ، لِحَدِيثِ عَمْرِو بْنِ حَزْمٍ { وَفِي اللِّسَانِ الدِّيَةُ } رَوَاهُ مَنْ ذُكِرَ قَبْلُ وَأَبُو دَاوُد ( وَقِيلَ شَرْطُ الطِّفْلِ ظُهُورُ أَثَرِ نُطْقٍ بِتَحْرِيكِهِ لِبُكَاءٍ وَمَصٍّ ) فَإِنْ لَمْ يَظْهَرْ فَحُكُومَةٌ ( وَلِأَخْرَسَ حُكُومَةٌ ) فَإِنْ ذَهَبَ ذَوْقُهُ وَجَبَتْ الدِّيَةُ
