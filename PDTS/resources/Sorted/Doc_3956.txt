( قَوْلُهُ : لَيْسَ لَهُ النَّوْمُ لَيْلًا فِي ثَوْبٍ اسْتَأْجَرَهُ لِلُّبْسِ ) ظَاهِرُ كَلَامِ الْمُصَنِّفِ أَنَّهُ لَوْ مَشَى طُولَ اللَّيْلِ لِحَاجَةٍ وَلَمْ يَنَمْ لَمْ يَجِبْ نَزْعُهُ وَلَعَلَّ الظَّاهِرَ خِلَافُهُ فَإِنَّ اللَّيْلَ مَظِنَّةُ النَّوْمِ ( قَوْلُهُ : نَعَمْ لَا يَلْزَمُهُ نَزْعُ الْإِزَارِ ) أَشَارَ إلَى تَصْحِيحِهِ وَكَذَا قَوْلُهُ : وَقَالَ الْأَذْرَعِيُّ الظَّاهِرُ إلَخْ ( قَوْلُهُ : أَوْجَهُهُمَا الْأَوَّلُ ) هُوَ الصَّحِيحُ .( 12 / 215 )
