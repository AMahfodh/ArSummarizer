وَذَكَرَ مُحْتَرَزَ تَعَيُّنٍ بِقَوْلِهِ ( وَلَا ) تَجُوزُ الْإِجَارَةُ عَلَى ( مُتَعَيِّنٍ ) أَيْ مَطْلُوبٍ مِنْ كُلِّ شَخْصٍ بِعَيْنِهِ ، وَلَا تَصِحُّ فِيهِ النِّيَابَةُ ، وَلَوْ غَيْرَ فَرْضٍ ( كَرَكْعَتَيْ الْفَجْرِ بِخِلَافِ الْكِفَايَةِ ) كَغُسْلِ الْمَيِّتِ أَوْ حَمْلِهِ فَيَصِحُّ الِاسْتِئْجَارُ عَلَيْهِ مَا لَمْ يَتَعَيَّنُ بِخِلَافِ صَلَاةِ الْجِنَازَةِ فَلَا يَجُوزُ الِاسْتِئْجَارُ عَلَيْهَا مُطْلَقًا ( وَعُيِّنَ ) فِي عَقْدِ الْإِجَارَةِ وُجُوبًا ( مُتَعَلِّمٌ ) لِقِرَاءَةٍ أَوْ صَنْعَةٍ لِاخْتِلَافِ حَالِهِ ذَكَاءً وَبَلَادَةً ( وَرَضِيعٌ ) لِاخْتِلَافِ حَالِهِ بِكَثْرَةِ الرَّضَاعِ وَقِلَّتِهِ ( وَ ) عُيِّنَ ( دَارٌ وَحَانُوتٌ ) وَحَمَّامٌ وَخَانٌ وَنَحْوُهَا إذْ لَا يَصِحُّ أَنْ يَكُونَ الْعَقَارُ فِي الذِّمَّةِ
