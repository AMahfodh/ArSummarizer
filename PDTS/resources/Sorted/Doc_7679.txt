( وَإِنْ دَفَعَ ) إنْسَانٌ ( ثَوْبَهُ إلَى قَصَّارٍ أَوْ خَيَّاطٍ وَنَحْوِهِمَا ) كَصَبَّاغٍ ( لِيَعْمَلَهُ ) أَيْ لِيَقْصُرَهُ أَوْ يَخِيطَهُ أَوْ يَصْبُغَهُ وَنَحْوِهِ ( وَلَوْ لَمْ تَكُنْ لَهُ ) أَيْ لِلْقَصَّارِ وَنَحْوِهِ عَادَةٌ ( بِأَخْذِ أُجْرَةٍ وَلَمْ يَعْقِدَا ) أَيْ الْقَصَّارُ وَالْخَيَّاطُ عَقْدَ إجَارَةٍ صَحَّ ، وَلَهُ أُجْرَةُ مِثْلِهِ حَيْثُ كَانَا مُنْتَصِبَيْنِ لِذَلِكَ وَإِلَّا لَمْ يَسْتَحِقَّا أَجْرًا إلَّا بِشَرْطٍ أَوْ عَقْدٍ أَوْ تَعْرِيضٍ ؛ لِأَنَّهُ لَمْ يُوجَدْ عُرْفٌ يَقُومُ مَقَامَ الْعَقْدِ ، فَهُوَ كَمَا لَوْ عَمِلَ بِغَيْرِ إذْنِ مَالِكِهِ .( 11 / 492 )
