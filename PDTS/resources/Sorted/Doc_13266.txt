( فَإِنْ ادَّعَى ) بِأَنْ ذَكَرَ أَنَّهُ يَدَّعِي ( مُعَامَلَةً ) أَوْ إتْلَافَ مَالٍ أَوْ عَيْنًا أَخَذَهَا بِغَصْبٍ أَوْ نَحْوِهِ ( أَحْضَرَهُ ) وَفَصَلَ خُصُومَتَهُ مِنْهُ ( كَغَيْرِهِ وَكَذَا ) لَوْ ادَّعَى عَلَيْهِ ( رِشْوَةً ) بِتَثْلِيثِ الرَّاءِ ( أَوْ حُكْمًا بِعَبْدَيْنِ مَثَلًا ) أَيْ بِشَهَادَةِ عَبْدَيْنِ أَوْ غَيْرِهِمَا مِمَّنْ لَا تُقْبَلُ شَهَادَتُهُ ( وَإِنْ لَمْ يَتَعَرَّضْ لِلْأَخْذِ ) أَيْ لِأَخْذِ الْمَالِ الْمَحْكُومِ بِهِ ( مِنْهُ فَإِنْ أَقَامَ عَلَى الْمَعْزُولِ ) بَعْدَ الدَّعْوَى عَلَيْهِ بَيِّنَةً أَوْ أَقَرَّ الْمَعْزُولُ ( حَكَمَ عَلَيْهِ وَإِلَّا صُدِّقَ بِيَمِينِهِ ) كَسَائِرِ الْأُمَنَاءِ إذَا ادَّعَى عَلَيْهِمْ خِيَانَةً وَلِعُمُومِ خَبَرِ { الْبَيِّنَةُ عَلَى الْمُدَّعِي وَالْيَمِينُ عَلَى مَنْ أَنْكَرَ } وَقِيلَ بِلَا يَمِينٍ ؛ لِأَنَّهُ أَمِينُ الشَّرْعِ فَيُصَانُ مَنْصِبُهُ عَنْ التَّحْلِيفِ وَالِابْتِذَالِ بِالْمُنَازَعَاتِ ، وَهَذَا صَحَّحَهُ الرَّافِعِيُّ قَالَ الزَّرْكَشِيُّ كَغَيْرِهِ ، وَقَدْ اخْتَلَفَ تَصْحِيحُ النَّوَوِيِّ فِيهِ وَالصَّوَابُ الثَّانِي فَإِنَّهُ الْمَنْصُوصُ كَمَا نَقَلَهُ الْقَاضِي شُرَيْحٌ الرُّويَانِيُّ وَغَيْرُهُ قَالَ ، وَهَذَا فِيمَنْ عُزِلَ مَعَ بَقَاءِ أَهْلِيَّتِهِ أَمَّا مَنْ ظَهَرَ فِسْقُهُ وَشَاعَ جَوْرُهُ وَخِيَانَتُهُ فَالظَّاهِرُ أَنَّهُ يَحْلِفُ قَطْعًا
