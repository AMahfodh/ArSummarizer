( وَ ) إنْ سَلَّمَ ( قَبْلَهُ عَمْدًا بِلَا عُذْرٍ تَبْطُلُ ) ؛ لِأَنَّهُ تَرَكَ فَرْضَ الْمُتَابَعَةِ مُتَعَمِّدًا وَ ( لَا ) تَبْطُلُ إنْ سَلَّمَ قَبْلَ إمَامِهِ ( سَهْوًا ، فَيُعِيدُهُ ) أَيْ السَّلَامَ ( بَعْدَهُ ) أَيْ بَعْدَ سَلَامِ إمَامِهِ ؛ لِأَنَّهُ لَا يَخْرُجُ مِنْ صَلَاتِهِ قَبْلَ إمَامِهِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يُعِدْهُ بَعْدَهُ ( بَطَلَتْ ) صَلَاتُهُ ؛ لِأَنَّهُ تَرَكَ فَرْضَ الْمُتَابَعَةِ أَيْضًا ( وَالْأَوْلَى : أَنْ يُسَلِّمَ الْمَأْمُومُ عَقِبَ فَرَاغِ الْإِمَامِ مِنْ التَّسْلِيمَتَيْنِ فَإِنْ سَلَّمَ ) الْمَأْمُومُ ( الْأُولَى بَعْدَ سَلَامِ الْإِمَامِ الْأُولَى ) وَقَبْلَ سَلَامِهِ الثَّانِيَةَ .
