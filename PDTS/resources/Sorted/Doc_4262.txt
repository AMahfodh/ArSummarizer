مِنْ قَوْلِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ : { عُفِيَ لِأُمَّتِي عَمَّا حَدَّثَتْ بِهِ أَنْفُسَهَا مَا لَمْ تَتَكَلَّمْ أَوْ تَعْمَلْ بِهِ } إنَّمَا يَدُلُّ عَلَى مُؤَاخَذَتِهِمْ بِمَا نَوَوْهُ عِنْدَ الْعَمَلِ بِهِ ، وَهَذَا لَمْ يَنْوِ طَلَاقًا يُؤَاخَذُ بِهِ ( أَوْ قَرَأَ مَا كَتَبَهُ ، وَقَالَ : لَا أَقْصِدُ إلَّا الْقِرَاءَةَ قُبِلَ ) مِنْهُ ذَلِكَ ( حُكْمًا ) كَلَفْظِ الطَّلَاقِ إذَا قَصَدَ بِهِ الْحِكَايَةَ .( 16 / 47 )
