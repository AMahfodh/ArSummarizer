( قَوْلُهُ : ؛ لِأَنَّ الْقَصْدَ هُنَا مَنْعُ نُفُوذِ الْمَاءِ ) أَيْ وَمِنْ لَازِمِهِ مَنْعُ الرُّؤْيَةِ ( قَوْلُهُ : فَلَا يَكْفِي نَجِسٌ إلَى قَوْلِهِ ، وَالْمُتَنَجِّسُ كَالنَّجِسِ ) أَيْ لَا يَكْفِي الْمَسْحُ عَلَيْهِمَا كَمَا هُوَ صَرِيحُ كَلَامِهِ بَعْدُ فَلَيْسَتْ الطَّهَارَةُ شَرْطًا لِلُّبْسِ وَإِنْ اقْتَضَى جَعْلَ قَوْلِ الْمُصَنِّفِ طَاهِرًا حَالًا مِنْ ضَمِيرِ يَلْبَسُ خِلَافَ ذَلِكَ( 2 / 145 )
