( قَوْلُهُ : مُطْلَقًا ) أَيْ سَوَاءٌ قَصَدَ بِهَا الْقَتْلَ أَمْ لَا كُرْدِيٌّ ( قَوْلُهُ : وَفَارَقَ هَذَا ) أَيْ الْمُكْرَهُ عَلَى صُعُودِ الشَّجَرَةِ حَيْثُ ضَمِنَ وَقَوْلُهُ الْمُكْرَهَ إلَخْ أَيْ حَيْثُ لَمْ يَضْمَنْ ( قَوْلُهُ : لَا تَجُوزُ إلَخْ ) مِنْ التَّجْوِيزِ ( قَوْلُهُ : مُطْلَقًا ) أَيْ أَزَلَقَتْ غَالِبًا أَمْ لَا .( 36 / 297 )
