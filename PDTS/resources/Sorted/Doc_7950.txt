( وَيَتَّجِهُ صِحَّتُهُ ) أَيْ : صِحَّةُ هَذَا الْقَوْلِ ( لِمُوَافَقَتِهِ الْقَوَاعِدَ ) وَالنَّفْسُ تَمِيلُ إلَيْهِ ، لَكِنَّ الْمَذْهَبَ الْأَوَّلُ ( فَإِنْ كَانَ أَحَدُهُمَا مُنْكَبًّا فَنِصْفُ دِيَتِهِ ) عَلَى عَاقِلَةِ الْمُسْتَلْقِي ( مُغَلَّظَةٍ ) لِأَنَّ قَتْلَ الْمُنْكَبِّ شِبْهُ الْعَمْدِ ، ( وَنِصْفُ دِيَةِ الْمُسْتَلْقِي ) عَلَى عَاقِلَةِ الْمُنْكَبِّ ( مُخَفَّفَةٍ ) لِأَنَّ قَتْلَهُ شِبْهُ الْخَطَأِ ، قَالَهُ فِي الرِّعَايَةِ ( وَإِنْ اصْطَدَمَا وَلَوْ ) كَانَا ضَرِيرَيْنِ ( أَوْ ) كَانَ أَحَدُهُمَا ضَرِيرًا ، ( فَمَاتَا فَ ) هُمَا ( كَمُتَجَاذِبَيْنِ ) عَلَى عَاقِلَةِ كُلٍّ مِنْهُمَا دِيَةُ الْآخَرِ .
