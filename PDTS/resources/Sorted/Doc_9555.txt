( قَوْلُهُ : نَعَمْ فِي مَوْتِ الرَّاكِبِ يَقُومُ وَارِثُهُ إلَخْ ) بِخِلَافِ الرَّامِي ( قَوْلُهُ : لِأَنَّهُ حِينَئِذٍ مُسَابَقَةٌ بِلَا مَالٍ ) يُتَأَمَّلُ فِي الْأَوَّلِ ( قَوْلُهُ : وَشَرْطُ الْمَالِ مِنْ جِهَتِهِ لَغْوٌ ) فَعِنْدَهُ لَا يُشْتَرَطُ إمْكَانُ سَبْقِ كُلِّ وَاحِدٍ ( قَوْلُهُ : وَمِنْهُ يُؤْخَذُ أَنَّ الْكَلَامَ فِي بَغْلٍ أَحَدُ أَبَوَيْهِ حِمَارٌ ) فِيهِ تَصْرِيحٌ بِأَنَّهُ قَدْ لَا يَكُونُ أَحَدُ أَبَوَيْهِ حِمَارًا ( قَوْلُهُ : كَمَا بَحَثَهُ الْبُلْقِينِيُّ ) تَقَدَّمَ فِي الْهَامِشِ عَنْ الشَّارِحِ فِي غَيْرِ هَذَا الشَّرْحِ خِلَافُهُ ( قَوْلُهُ وَإِطْلَاقُهُ التَّصَرُّفَ إلَخْ ) تَقَدَّمَ هَذَا فِي شَرْحِ قَوْلِهِ وَمَحَلُّ أَخْذِ عِوَضٍ عَلَيْهِمَا .( 41 / 337 )
