وَلَدَيْ أُمٍّ وَأُخْتٍ كَمَلَتْ ) أَيْ لِأَبَوَيْنِ ( وَزَوْجِهَا وَأُمِّهَا ، ثُمَّ نَكَحْ لِأُخْتِهَا ) أَيْ نَكَحَ الزَّوْجُ أُخْتَهَا الْكَامِلَةَ ( فَقُبِضَتْ ) أَيْ مَاتَتْ ( عَمَّنْ سَرَحْ ) بِالْمُهْمَلَاتِ أَيْ مَضَى فَقَطْ وَهُوَ الزَّوْجُ وَالْأُمُّ وَوَلَدَاهَا فَصَارَ الْمَذْكُورُ إلَى هُنَا ثَلَاثَةَ أَحْوَالٍ ( يُفْرَضُ ) فِيهَا بِالنَّظَرِ إلَى الْحِسَابِ أَنَّ الْمَيِّتَ الثَّانِيَ ( لَمْ يَكُنْ ) وَتُقْسَمُ تَرِكَةُ الْأَوَّلِ عَلَى الْمَوْجُودِينَ .
