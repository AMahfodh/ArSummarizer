( وَإِنْ هَادَنَهُمْ مُطْلَقًا ) بِأَنْ لَمْ يُقَيِّدْ بِمُدَّةٍ لَمْ يَصِحَّ ؛ لِأَنَّ الْإِطْلَاقَ يَقْتَضِي التَّأْيِيدَ وَذَلِكَ يُفْضِي إلَى تَرْكِ الْجِهَادِ بِالْكُلِّيَّةِ وَهُوَ غَيْرُ جَائِزٍ ( أَوْ ) هَادَنَهُمْ ( مُعَلِّقًا بِمَشِيئَةٍ كَمَا شِئْنَا أَوْ شِئْتُمْ أَوْ شَاءَ فُلَانٌ أَوْ مَا أَقَرَّكُمْ اللَّهُ ) عَلَيْهِ ( لَمْ يَصِحَّ ) كَالْإِجَارَةِ وَلِجَهَالَةِ الْمُدَّةِ .( 8 / 256 )
