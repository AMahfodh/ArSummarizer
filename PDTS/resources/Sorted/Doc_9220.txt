( وَلَا ) يُجْزِئُ ( أَخْرَسُ أَصَمُّ وَلَوْ فُهِمَتْ إشَارَتُهُ ) لِأَنَّهُ نَاقِصٌ بِفَقْدِ حَاسَّتَيْنِ تَنْقُصُ بِنَقْصِهِمَا قِيمَتُهُ نَقْصًا كَثِيرًا ، وَكَذَا أَخْرَسُ لَا تُفْهِمُ إشَارَتُهُ ( وَمَجْنُونٌ مُطْبَقٌ ) لِأَنَّهُ يُمْنَعُ مِنْ الْعَمَلِ بِالْكُلِّيَّةِ ( وَغَائِبٌ لَمْ تَتَبَيَّنْ حَيَاتُهُ ) لِأَنَّ وُجُودَهُ غَيْرُ مُحَقَّقٍ ، فَلَا يَبْرَأُ بِالشَّكِّ ( فَإِذَا ) أَعْتَقَهُ ، ثُمَّ ( تَبَيَّنَتْ ) حَيَاتُهُ ( أَجْزَأَ ) قَوْلًا وَاحِدًا ( وَلَا مُوصًى بِخِدْمَتِهِ أَبَدًا ) لِنَقْصِهِ ( وَأُمُّ وَلَدٍ ) لِأَنَّ عِتْقَهَا مُسْتَحَقٌّ بِسَبَبٍ آخَرَ ( وَلَا جَنِينٌ ) وَلَوْ وُلِدَ بَعْدَ عِتْقِهِ حَيًّا ؛ لِأَنَّهُ لَمْ تَثْبُتْ لَهُ أَحْكَامُ الدُّنْيَا بَعْدُ .
