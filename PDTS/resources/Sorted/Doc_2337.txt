وَالسَّبُعُ : كُلُّ مُخْتَطِفٍ مُنْتَهِبٍ جَارِحٍ قَاتِلٍ عَادَةً ( أَوْ طَيْرٍ ) بَيَانٌ لِذِي مِخْلَبٍ ( وَلَا ) ( الْحَشَرَاتُ ) هِيَ صِغَارُ دَوَابِّ الْأَرْضِ وَاحِدُهَا حَشَرَةٌ ( وَالْحُمُرُ الْأَهْلِيَّةُ ) بِخِلَافِ الْوَحْشِيَّةِ فَإِنَّهَا وَلَبَنَهَا حَلَالٌ ( وَالْبَغْلُ ) الَّذِي أُمُّهُ حِمَارَةٌ ، فَلَوْ أُمُّهُ بَقَرَةٌ أُكِلَ اتِّفَاقًا وَلَوْ فَرَسًا فَكَأُمِّهِ ( وَالْخَيْلُ ) وَعِنْدَهُمَا ، وَالشَّافِعِيُّ تَحِلُّ .
