وَعَنْ جَابِرٍ { لَمَّا نَزَلَتْ { قُلْ هُوَ الْقَادِرُ عَلَى أَنْ يَبْعَثَ عَلَيْكُمْ عَذَابًا مِنْ فَوْقِكُمْ } قَالَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَعُوذُ بِوَجْهِك ، فَلَمَّا نَزَلَتْ { أَوْ مِنْ تَحْتِ أَرْجُلِكُمْ } قَالَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَعُوذُ بِوَجْهِك ؛ فَلَمَّا نَزَلَتْ { أَوْ يَلْبِسَكُمْ شِيَعًا وَيُذِيقَ بَعْضَكُمْ بَأْسَ بَعْضٍ } قَالَ هَذِهِ أَهْوَنُ أَوْ أَيْسَرُ } رَوَاهُ الْبُخَارِيُّ .
