( وَإِنْ كُفِّنَ بِثَوْبٍ غُصِبَ ) وَطَلَبَهُ رَبُّهُ لَمْ يُنْبَشْ وَغُرْمُ ذَلِكَ مِنْ تَرِكَتِهِ ، لِإِمْكَانِ دَفْعِ الضَّرَرِ مَعَ عَدَمِ هَتْكِ حُرْمَتِهِ ( أَوْ بَلَعَ مَالَ غَيْرِهِ بِغَيْرِ إذْنِهِ وَتَبْقَى مَالِيَّتُهُ ، كَخَاتَمٍ ، طَلَبَهُ رَبُّهُ لَمْ يُنْبَشْ وَغُرْمُ ذَلِكَ مِنْ تَرِكَتِهِ ) صَوْنًا لِحُرْمَتِهِ مَعَ عَدَمِ الضَّرَرِ ( كَمَنْ غَصَبَ عَبْدًا فَأَبِقَ ؛ تَجِبُ قِيمَتُهُ ) عَلَى الْغَاصِبِ ( لِأَجْلِ الْحَيْلُولَةِ ) أَيْ : حَيْلُولَتِهِ بَيْنَ الْمَالِ وَرَبِّهِ ( فَإِنْ تَعَذَّرَ الْغُرْمُ ) أَيْ : غُرْمُ الْكَفَنِ الْمَغْصُوبِ أَوْ الْمَالِ الَّذِي بَلَعَهُ الْمَيِّتُ ( لِعَدَمِ تَرِكَةٍ وَنَحْوِهِ نُبِشَ ) الْقَبْرُ ( وَأُخِذَ الْكَفَنُ ) الْغَصْبُ فَدُفِعَ لِرَبِّهِ .
