مُغْنِي ( قَوْلُهُ : وَبِهِ يُرَدُّ إلَخْ ) أَيْ : بِمَا مَرَّ ، وَيُحْتَمَلُ بِقَوْلِهِ ؛ لِأَنَّ الْغَرَضَ إلَخْ ( قَوْلُهُ : أَوْ صَغَارٌ ) بِفَتْحِ الصَّادِ أَيْ : ذُلٌّ وَهَذَا أَوْلَى لِقَوْلِهِ تَعَالَى { وَهُمْ صَاغِرُونَ } نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : وَفِي نَعَمِ بَقِيَّةِ إلَخْ ) الْأَنْسَبُ وَفَيْءٌ فِي نَعَمِ بَقِيَّةِ الْفَيْءِ ( قَوْلُهُ : كَكَافِ الزَّكَاةِ ) وَصَادِ الصَّدَقَةِ وَجِيمِ( 29 / 127 )
