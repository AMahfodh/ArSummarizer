ثُمَّ أَشَارَ إلَى الْقِسْمِ الثَّانِي مِنْ زِيَادَةِ الْأَفْعَالِ بِقَوْلِهِ : ( وَعَمَلٌ مُتَوَالٍ مُسْتَكْثِرٌ فِي الْعَادَةِ مِنْ غَيْرِ جِنْسِ الصَّلَاةِ كَمَشْيٍ ، وَفَتْحِ بَابٍ وَنَحْوِهِ ) كَلَفِّ عِمَامَةٍ وَخِيَاطَةٍ وَكِتَابَةٍ ( يُبْطِلهَا ) أَيْ الصَّلَاةُ ( عَمْدُهُ وَسَهْوُهُ وَجَهْلُهُ ) لِقَطْعِهِ الْمُوَالَاةَ بَيْنَ الْأَرْكَانِ ( إنْ لَمْ تَكُنْ ضَرُورَةً ) كَخَوْفٍ وَهَرَبٍ مِنْ عَدْوٍ أَوْ سَيْلٍ وَنَحْوِهِ ، فَلَا يُبْطِلُ الصَّلَاةَ لِأَنَّ الضَّرُورَاتِ تُبِيحُ الْمَحْظُورَاتِ ( وَتَقَدَّمَ ) فِي الْبَابِ قَبْلَهُ .( 3 / 164 )
