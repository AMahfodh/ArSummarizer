الْأَنْصِبَاءِ ) مُتَعَلِّقٌ بِ تُعَدَّلُ ، هَذَا ( إنْ اسْتَوَتْ ) تِلْكَ الْأَنْصِبَاءُ ، كَمَا إذَا كَانَتْ بَيْنَ ثَلَاثَةٍ أَثْلَاثًا فَتُجْعَلُ ثَلَاثَةَ أَجْزَاءٍ ، ثُمَّ يُؤْخَذُ ثَلَاثُ رِقَاعٍ ( وَيَكْتُبُ ) مَثَلًا هُنَا وَفِيمَا يَأْتِي مِنْ بَقِيَّةِ الْأَنْوَاعِ ( فِي كُلِّ رُقْعَةٍ ) إمَّا ( اسْمَ شَرِيكٍ ) مِنْ الشُّرَكَاءِ ( أَوْ جُزْءًا ) مِنْ الْأَجْزَاءِ ( مُمَيَّزًا ) عَنْ الْبَقِيَّةِ ( بِحَدٍّ أَوْ جِهَةٍ ) أَوْ غَيْرِ ذَلِكَ ( وَتُدْرَجُ ) الرُّقَعُ ( فِي بَنَادِقَ ) مِنْ نَحْوِ شَمْعٍ أَوْ طِينٍ ( مُسْتَوِيَةٍ ) وَزْنًا وَشَكْلًا لِئَلَّا تَسْبِقَ الْيَدُ لِإِخْرَاجِ الْكَبِيرَةِ وَفِيهِ تَرْجِيحٌ لِصَاحِبِهَا .
