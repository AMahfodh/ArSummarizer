( قَوْلُهُ : وَبِهِ ) أَيْ بِقَوْلِهِ مِنْ غَيْرِ حَاجَةٍ وَلَا ضَرُورَةٍ ( فَارَقَ ) أَيْ : الْوُجُوبُ هُنَا ( وُجُوبَ أُجْرَةِ تَعْلِيمِهِ ) أَيْ : لِمَا لَيْسَ بِوَاجِبٍ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ وَمُؤَنِ إلَخْ ) عُطِفَ عَلَى أُجْرَةِ تَعْلِيمِهِ وَ ( قَوْلُهُ : فِي مَالِ إلَخْ ) مُتَعَلِّقٌ بِوُجُوبِ إلَخْ .
