( وَمَنْ كَانَ لَهُ عَرَضُ قِنْيَةٍ يُبَاع لَوْ أَفْلَسَ ) أَيْ : حُجِرَ عَلَيْهِ لِفَلْسٍ ، كَعَقَارٍ وَأَثَاثٍ لَا يَحْتَاجُهُ وَكَانَ ثَمَنُهُ ( يَفِي بِمَا عَلَيْهِ مِنْ الدَّيْنِ ) وَمَعَهُ مَالٌ زَكَوِيٌّ ( جَعَلَ ) الدَّيْنَ ( فِي مُقَابَلَةِ مَا مَعَهُ ) مِنْ الْمَالِ الزَّكَوِيِّ ( فَلَا يُزَكِّيه ) لِئَلَّا يُخِلَّ بِالْمُوَاسَاةِ وَلِأَنَّ عَرَضَ الْقِنْيَةِ كَمَلْبُوسِهِ فِي أَنَّهُ لَا زَكَاةَ فِيهِ فَكَذَا فِيمَا يَمْنَعهَا ( وَكَذَا مَنْ بِيَدِهِ أَلْفٌ وَلَهُ عَلَى مَلِيءٍ ) دَيْنُ ( أَلْف ، وَعَلَيْهِ ) دَيْنُ ( أَلْفٍ ) فَيَجْعَل الْأَلْفَ الَّذِي بِيَدِهِ فِي مُقَابَلَةِ مَا عَلَيْهِ فَلَا يُزَكِّيهِ وَأَمَّا الدَّيْنُ فَيُزَكِّيهِ إذَا قَبَضَهُ .( 5 / 43 )
