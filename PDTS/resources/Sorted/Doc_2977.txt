( وَيَتَّجِهُ مِثْلُهُ ) أَيْ : مِثْلُ مَاءِ الْبِئْرِ الَّذِي فِي الْمَقْبَرَةِ ( مَا سُخِّنَ بِنَجَاسَةٍ ) ، إذْ لَا يُؤْمَنُ تَخَلُّلُ دُخَانِهَا الْمَاءَ ، ( أَوْ غَصْبٍ ) لِلْأَمْرِ بِاتِّقَاءِ الشُّبُهَاتِ اسْتِبْرَاءً لِلْعِرْضِ وَالدِّينِ ، وَهُوَ مُتَّجِهٌ ، ( وَكَرِهَ ) الْإِمَامُ ( أَحْمَدُ بَقْلَ مَقْبَرَةٍ ) وَشَوْكَهَا إذَا لَمْ يَتَكَرَّرْ نَبْشُهَا وَإِلَّا فَنَجِسٌ .( 1 / 46 )
