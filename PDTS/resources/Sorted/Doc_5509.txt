( وَإِنْ قُطِعَ ) بَعْدَ الْجِنَايَةِ ( عُضْوُ قَاطِعٍ ) لِعُضْوِ غَيْرِهِ عَمْدًا ( بِسَمَاوِيٍّ ) مُرْتَبِطٍ بِقَطْعٍ بِمَعْنَى سَقَطَ ( أَوْ ) قُطِعَ بِسَبَبِ ( سَرِقَةٍ أَوْ ) قُطِعَ ( بِقِصَاصٍ لِغَيْرِهِ ) : أَيْ لِغَيْرِ الْمَجْنِيِّ عَلَيْهِ أَوَّلًا ( فَلَا شَيْءَ لِلْمَجْنِيِّ عَلَيْهِ ) : لَا قِصَاصَ وَلَا دِيَةَ ؛ لِأَنَّهُ إنَّمَا تَعَلَّقَ حَقُّهُ بِالْعُضْوِ الْمُمَاثِلِ وَقَدْ ذَهَبَ ، وَكَذَا لَوْ مَاتَ الْقَاطِعُ بِخِلَافِ مَقْطُوعِ الْعُضْوِ قَبْلَ الْجِنَايَةِ فَعَلَيْهِ الدِّيَةُ .
