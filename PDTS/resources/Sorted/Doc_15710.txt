طَائِفٌ بِكُلِّ بَدَنِهِ وَيَسْتَقْبِلُهُ بِوَجْهِهِ ( أَوْ ) يُحَاذِي ( بَعْضَهُ ) أَيْ الْحَجَرِ ( بِكُلِّ بَدَنِهِ ) لِأَنَّ مَا لَزِمَ اسْتِقْبَالُهُ لَزِمَ بِجَمِيعِ الْبَدَنِ كَالْقِبْلَةِ ( وَيَسْتَلِمُهُ ) أَيْ يَمْسَحُ الْحَجَرَ ( بِيَدِهِ الْيُمْنَى ) وَالِاسْتِلَامُ مِنْ السَّلَامِ وَهُوَ التَّحِيَّةُ وَأَهْلُ الْيَمَنِ يُسَمُّونَ الْحَجَرَ الْأَسْوَدَ الْمُحَيَّا لِأَنَّ النَّاسَ يُحَيُّونَهُ بِالِاسْتِلَامِ وَرَوَى التِّرْمِذِيُّ مَرْفُوعًا { أَنَّهُ نَزَلَ مِنْ الْجَنَّةِ أَشَدَّ بَيَاضًا مِنْ اللَّبَنِ فَسَوَّدَتْهُ خَطَايَا بَنِي آدَمَ } وَقَالَ : حَسَنٌ صَحِيحٌ ( وَيُقَبِّلُهُ ) بِلَا صَوْتٍ يَظْهَرُ لِلْقُبْلَةِ لِحَدِيثِ عُمَرَ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ اسْتَقَلَّ الْحَجَرَ وَوَضَعَ شَفَتَيْهِ عَلَيْهِ يَبْكِي طَوِيلًا ثُمَّ الْتَفَتَ فَإِذَا هُوَ بِعُمَرَ بْنِ الْخَطَّابِ يَبْكِي فَقَالَ : يَا عُمَرُ هَهُنَا تُسْكَبُ الْعَبَرَاتُ } رَوَاهُ ابْنُ مَاجَهْ ( وَيَسْجُدُ عَلَيْهِ ) فَعَلَهُ ابْنُ عُمَرَ وَابْنُ عَبَّاسٍ وَإِذَا شَقَّ ) لِنَحْوِ زِحَامٍ اسْتِلَامُهُ وَتَقْبِيلُهُ ( لَمْ يُزَاحِمْ وَاسْتَلَمَهُ بِيَدِهِ وَقَبَّلَهَا ) رُوِيَ عَنْ ابْنِ عُمَرَ وَجَابِرٍ وَأَبِي هُرَيْرَةَ وَأَبِي سَعِيدٍ وَابْنِ عَبَّاسٍ لِمَا رَوَى ابْنُ عَبَّاسٍ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ اسْتَلَمَهُ وَقَبَّلَ يَدَهُ } رَوَاهُ مُسْلِمٌ وَإِذَا شَقَّ ) اسْتِلَامُهُ بِيَدِهِ ( فَ ) إنَّهُ يَسْتَلِمُهُ ( بِشَيْءٍ وَيُقَبِّلُهُ ) أَيْ مَا اسْتَلَمَهُ بِهِ رُوِيَ عَنْ ابْنِ عَبَّاسٍ مَوْقُوفًا وَإِذَا شَقَّ ) عَلَيْهِ اسْتِلَامُهُ أَيْضًا بِشَيْءٍ ( أَشَارَ إلَيْهِ ) أَيْ الْحَجَرِ ( بِيَدِهِ الْيُمْنَى أَوْ بِشَيْءٍ ) لِحَدِيثِ الْبُخَارِيِّ عَنْ ابْنِ عَبَّاسٍ قَالَ : { طَافَ النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ عَلَى بَعِيرٍ ، فَلَمَّا أَتَى الْحَجَرَ أَشَارَ إلَيْهِ بِشَيْءٍ فِي يَدِهِ وَكَبَّرَ } ( وَلَا يُقَبِّلُهُ ) أَيْ مَا أَشَارَ بِهِ إلَيْهِ ( وَاسْتَقْبَلَهُ ) أَيْ الْحَجَرَ ، إذَا شَرَعَ فِي الطَّوَافِ ( بِوَجْهِهِ وَقَالَ : بِسْمِ اللَّهِ وَاَللَّهُ أَكْبَرُ اللَّهُمَّ إيمَانًا بِكَ ، وَتَصْدِيقًا( 4 / 19 )
