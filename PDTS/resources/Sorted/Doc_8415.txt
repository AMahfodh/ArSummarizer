( فَرْعٌ ) لَوْ ( نَكَحَهَا عَلَى أَنَّ النِّكَاحَ يَنْتَهِي بِالْوَطْءِ بَطَلَ ) ؛ لِأَنَّهُ ضَرْبٌ مِنْ نِكَاحِ الْمُتْعَةِ وَعَلَيْهِ حُمِلَ خَبَرُ { لَعَنَ اللَّهُ الْمُحَلِّلَ وَالْمُحَلَّلَ لَهُ } رَوَاهُ التِّرْمِذِيُّ وَقَالَ حَسَنٌ صَحِيحٌ ( وَكَذَا إنْ شَرَطَ طَلَاقَهَا ) قَبْلَ الْوَطْءِ أَوْ بَعْدَهُ ؛ لِأَنَّهُ شَرْطٌ يَمْنَعُ دَوَامَ النِّكَاحِ فَأَشْبَهَ التَّأْقِيتَ وَتَعْبِيرُهُ بِمَا قَالَهُ أَعَمُّ مِنْ قَوْلِ أَصْلِهِ فَإِنْ شَرَطَ أَنَّهُ إذَا وَطِئَهَا طَلَّقَهَا قَالَ الزَّرْكَشِيُّ وَلَوْ تَزَوَّجَهَا عَلَى أَنْ يُحِلَّهَا لِلْأَوَّلِ فَفِي الِاسْتِذْكَارِ لِلدَّارِمِيِّ فِيهِ وَجْهَانِ وَجَزَمَ الْمَاوَرْدِيُّ بِالصِّحَّةِ ؛ لِأَنَّهُ لَمْ يَشْتَرِطْ الْفُرْقَةَ بَلْ شَرَطَ مُقْتَضَى الْعَقْدِ .
