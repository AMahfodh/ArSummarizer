( وَيَثْبُتُ لِوَكِيلِ ) وَلِيٍّ ( مَا لَهُ ) أَيْ الْوَلِيِّ ( مِنْ إجْبَارٍ وَغَيْرِهِ ) لِأَنَّهُ نَائِبُهُ وَكَذَا سُلْطَانٌ وَحَاكِمٌ يَأْذَنُ لِغَيْرِهِ فِي التَّزْوِيجِ ( لَكِنْ لَا بُدَّ مِنْ إذْنِ غَيْرِ مُجْبَرَةٍ لِوَكِيلِ ) وَلِيِّهَا لِأَنَّهُ نَائِبٌ عَنْ غَيْرِ مُجْبِرٍ فَيَثْبُتُ لَهُ مَا يَثْبُتُ لِمَنْ يَنُوبُ عَنْهُ ( فَلَا يَكْفِي إذْنُهَا لِوَلِيِّهَا بِتَزْوِيجٍ أَوْ تَوْكِيلٍ فِيهِ ) أَيْ التَّزْوِيجِ ( بِلَا مُرَاجَعَةِ وَكِيلٍ أَيْ اسْتِئْذَانٍ لَهَا ) أَيْ لِغَيْرِ الْمُجْبَرَةِ فِي التَّزْوِيجِ ( وَإِذْنِهَا لَهُ ) أَيْ الْوَكِيلِ ( فِيهِ ) أَيْ التَّزْوِيجِ ( بَعْدَ تَوْكِيلِهِ ) لِأَنَّ الَّذِي يُعْتَبَرُ إذْنُهَا فِيهِ لِلْوَكِيلِ هُوَ غَيْرُ مَا يُوَكَّلُ فِيهِ الْمُوَكَّلُ فَهُوَ كَالْمُوَكِّلِ فِي ذَلِكَ وَلَا أَثَرَ لِإِذْنِهَا فِيهِ قَبْلَ أَنْ يُوَكِّلَهُ الْوَلِيُّ لِأَنَّهُ أَجْنَبِيٌّ إذَنْ ، وَأَمَّا بَعْدَهُ فَكَوَلِيٍّ ( فَلَوْ وَكَّلَ وَلِيُّ ) غَيْرِ مُجْبَرَةٍ فِي تَزْوِيجِهَا ( ثُمَّ أَذِنَتْ لِوَكِيلِهِ ) أَيْ وَكِيلِ وَلِيِّهَا فِي تَزْوِيجِهَا فَزَوَّجَهَا ( صَحَّ ) النِّكَاحُ ( وَلَوْ لَمْ تَأْذَنْ لِلْوَلِيِّ ) فِي التَّوْكِيلِ أَوْ التَّزْوِيجِ لِقِيَامِ وَكِيلِهِ مَقَامَهُ( 8 / 355 )
