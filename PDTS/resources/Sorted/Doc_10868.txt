وَلَا يَحِلُّ لِلْمُشْتَرِي ذَلِكَ فَالْأَحْسَنُ حَذْفُ وَلَوْ ، وَشَبَّهَ فِي وُجُوبِ الِاسْتِبْرَاءِ فَقَالَ ( كَ ) الْأَمَةِ ( الْمَوْطُوءَةِ ) مِنْ سَيِّدِهَا الْبَالِغِ الْحُرِّ ( إنْ بِيعَتْ ) أَيْ أَرَادَ سَيِّدُهَا بَيْعَهَا فَيَجِبُ عَلَيْهِ اسْتِبْرَاؤُهَا مِنْ مَائِهِ بِحَيْضَةٍ ( أَوْ زُوِّجَتْ ) بِضَمِّ الزَّايِ وَكَسْرِ الْوَاوِ مُشَدَّدَةً أَيْ أَرَادَ سَيِّدُهَا تَزْوِيجَهَا فَيَجِبُ عَلَيْهِ اسْتِبْرَاؤُهَا مِنْ مَائِهِ بِحَيْضَةٍ وَمَفْهُومُ الْمَوْطُوءَةِ أَنَّ غَيْرَهَا لَا يَجِبُ اسْتِبْرَاؤُهَا لِبَيْعِهَا وَلَوْ زَنَتْ وَلَا لِتَزْوِيجِهَا إلَّا أَنْ تَزْنِيَ فَفِيهِ تَفْصِيلٌ ، ( وَقُبِلَ ) بِضَمِّ الْقَافِ وَكَسْرِ الْمُوَحَّدَةِ بِلَا يَمِينٍ ( قَوْلُ سَيِّدِهَا ) أَيْ الْأَمَةِ لِزَوْجِهَا عِنْدَ إرَادَةِ تَزْوِيجِهَا لَهُ أَنَّهُ اسْتَبْرَأَهَا ؛ لِأَنَّهُ أَمْرٌ لَا يُعْلَمُ إلَّا مِنْهُ فَيَعْتَمِدُ عَلَيْهِ الزَّوْجُ وَيَطَؤُهَا بِدُونِ اسْتِبْرَاءٍ( 9 / 134 )
