( وَ ) تَصِحُّ الْوَصِيَّةُ ( بِمُصْحَفٍ لِيُقْرَأَ فِيهِ ) لِأَنَّهَا قُرْبَةٌ وَيُوضَعُ بِجَامِعٍ أَوْ مَوْضِعٍ حَرِيزٍ لِيَحْفَظَهُ ( وَ ) تَصِحُّ الْوَصِيَّةُ ( لِفَرَسٍ حَبِيسٍ ) لِأَنَّهُ جِهَةُ قُرْبَةٍ ( وَيَتَّجِهُ وَنَحْوِهِ ) كَبَعِيرٍ وَرِبَاطٍ وَ ( يُنْفِقُ ) الْمُوصَى بِهِ لِلْفَرَسِ الْحَبِيسِ وَنَحْوَهُ ( عَلَيْهِ ) لِأَنَّهُ مِنْ أَنْوَاعِ الْبِرِّ ( فَإِنْ مَاتَ ) الْفَرَسُ الْحَبِيسُ وَالْبَعِيرُ ( وَيَتَّجِهُ أَوْ خَرِبَ ) الرِّبَاطُ وَهُمَا مُتَّجِهَانِ ( رَدَّ مُوصَى بِهِ ) إنْ لَمْ يَكُنْ أُنْفِقَ مِنْهُ شَيْءٌ ( أَوْ ) رَدَّ ( بَاقِيَهُ لِلْوَرَثَةِ ) لِبُطْلَانِ مَحَلِّ الْوَصِيَّةِ كَمَا لَوْ وَصَّى لِإِنْسَانٍ بِشَيْءٍ ، فَرَدَّهُ ، وَلَا يُصْرَفُ فِي فَرَسٍ حَبِيسٍ آخَرَ نَصًّا ، وَإِنْ شَرَدَ الْفَرَسُ الْمُوصَى لَهُ أَوْ سُرِقَ أَوْ غُصِبَ ؛ انْتَظَرَ عَوْدَهُ ؛ لِأَنَّهُ مُمْكِنٌ ، وَإِنْ أَيِسَ مِنْ عَوْدِهِ رَدَّ الْمُوصَى بِهِ إلَى الْوَرَثَةِ ، إذْ لَا مَصْرِفَ لَهُ .
