( لَا ) يَصِحُّ تَوْكِيلُ ( وَلِيٍّ ) لِامْرَأَةٍ ( إلَّا كَهُوَ ) أَيْ إلَّا مِثْلَهُ فِي الذُّكُورَةِ وَالْبُلُوغِ وَالْحُرِّيَّةِ وَالْإِسْلَامِ وَعَدَمِ الْإِحْرَامِ وَالْعَتَهِ ( وَعَلَيْهِ ) أَيْ عَلَى الْوَلِيِّ ، وَلَوْ أَبًا غَيْرَ مُجْبِرٍ وُجُوبًا ( الْإِجَابَةُ لِكُفْءٍ ) رَضِيَتْ بِهِ ( وَ ) لَوْ دَعَتْ لِكُفْءٍ وَدَعَا وَلِيُّهَا لِكُفْءٍ غَيْرِهِ كَانَ ( كُفْؤُهَا أَوْلَى ) أَيْ أَوْجَبَ أَيْ فَيَتَعَيَّنُ كُفْؤُهَا ( فَيَأْمُرُهُ الْحَاكِمُ ) بِتَزْوِيجِهَا فِي الْمَسْأَلَتَيْنِ بَعْدَ أَنْ يَسْأَلَهُ عَنْ وَجْهِ امْتِنَاعِهِ وَلَمْ يَظْهَرْ لَهُ وَجْهٌ صَحِيحٌ ( ثُمَّ ) إنْ امْتَنَعَ ( زَوَّجَ ) الْحَاكِمُ أَوْ وَكَّلَ مَنْ يَعْقِدُ عَلَيْهَا ، وَلَوْ أَجْنَبِيًّا مِنْهَا وَلَا يَنْتَقِلُ الْحَقُّ لِلْأَبْعَدِ ؛ لِأَنَّ الْوَلِيَّ يَصِيرُ عَاضِلًا بِرَدِّهِ أَوَّلَ كُفْءٍ بِخِلَافِ الْمُجْبِرِ .
