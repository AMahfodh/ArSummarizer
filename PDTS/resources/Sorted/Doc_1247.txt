و ( لَا ) تَثْبُتُ لَهُ الْوِلَايَةُ عَلَى وَلَدِهِ ( الْحُرِّ ، ثُمَّ ) تَثْبُتُ الْوِلَايَةُ عَلَى صَغِيرٍ وَمَجْنُونٍ ( لِوَصِيِّ الْأَبِ ) الْعَدْلِ ( وَلَوْ ) كَانَ ( بِجَعْلٍ ) مِنْ الْمُوصِي أَوْ الْحَكَمِ ( وَثَمَّ مُتَبَرِّعٌ ) بِالْوِلَايَةِ - لِأَنَّهُ نَائِبُ الْأَبِ - أَشْبَهَ وَكِيلَهُ فِي الْحَيَاةِ .( 9 / 210 )
