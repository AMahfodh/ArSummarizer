وَكَذَا الْإِقْلَاعُ ( وَلَا يُشْتَرَطُ لِصِحَّتِهَا ) أَيْ التَّوْبَةِ ( مِنْ قَذْفٍ وَغِيبَةٍ وَنَحْوِهِمَا ) كَنَمِيمَةٍ وَشَتْمٍ ( إعْلَامُهُ ) أَيْ الْمَقْذُوفِ وَالْمُغْتَابِ وَنَحْوِهِمَا ( وَالتَّحَلُّلُ مِنْهُ ) قَالَ أَحْمَدُ : إذَا قَذَفَهُ ثُمَّ تَابَ لَا يَنْبَغِي أَنْ يَقُولَ لَهُ : قَدْ قَذَفْتُكَ ، بَلْ يَسْتَغْفِرُ اللَّهَ ؛ لِأَنَّ فِيهِ إيذَاءً صَرِيحًا ، وَإِذَا اسْتَحَلَّهُ يَأْتِي بِلَفْظٍ مُبْهَمٍ لِصِحَّةِ الْبَرَاءَةِ مِنْ الْمَجْهُولِ ، ( وَمَنْ أَخَذَ بِالرُّخَصِ ) أَيْ تَتَبَّعَهَا مِنْ الْمَذَاهِبِ فَعَمِلَ بِهَا ( فَسَقَ ) نَصًّا ، وَذَكَرَهُ ابْنُ عَبْدِ الْبَرِّ( 12 / 385 )
