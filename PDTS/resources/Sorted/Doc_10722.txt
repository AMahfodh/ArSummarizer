لِخَاطِبَيْنِ وَخَاطِبٍ وَاحِدٍ ( وَإِنْ وَكَّلَتْهُ ) أَيْ الْمَرْأَةُ رَجُلًا أَوْ امْرَأَةً عَلَى تَزْوِيجِهَا ( مِمَّنْ ) أَيْ رَجُلٍ أَوْ الرَّجُلِ الَّذِي ( أَحَبَّ ) ه الْوَكِيلُ أَوْ وَكَالَةً مُفَوَّضَةً وَأَحَبَّ الْوَكِيلُ رَجُلًا ( عَيَّنَ ) بِفَتَحَاتٍ مُثَقَّلًا الْوَكِيلُ الرَّجُلَ الَّذِي أَحَبَّهُ لِمُوَكِّلَتِهِ لِاخْتِلَافِ أَغْرَاضِ النِّسَاءِ فِي أَعْيَانِ وَصِفَاتِ الرِّجَالِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يُعَيِّنْهُ وَعَقَدَ لَهَا عَلَيْهِ ( فَلَهَا ) أَيْ الْمُوَكِّلَةِ ( الْإِجَازَةُ ) أَيْ الْإِمْضَاءُ لِعَقْدِ وَكِيلِهَا إنْ قَرُبَ مَا بَيْنَ عَقْدِهِ وَعِلْمِهَا بِهِ ، بَلْ ( وَلَوْ بَعُدَ ) بِضَمِّ الْعَيْنِ مَا بَيْنَهُمَا ، وَظَاهِرُهُ وَلَوْ طَالَ الزَّمَنُ جِدًّا ، وَظَاهِرُهُ أَيْضًا وَلَوْ عَلِمَ الزَّوْجُ بِعَدَمِ تَعْيِينِهِ وَأَنَّهُ لَازِمٌ ، وَسَوَاءٌ زَوَّجَهَا الْوَكِيلُ لِغَيْرِهِ أَوْ لِنَفْسِهِ .
