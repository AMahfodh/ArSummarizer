( فَائِدَةٌ ) أَفَادَ بَعْضُ الشُّيُوخِ أَنَّ الْمُسَافِرِينَ لِلْحَجِّ فَكُلُّ مَنْ سَبَقَ إلَى الْمَاءِ فَهُوَ أَحَقُّ بِهِ وَكَانَ ابْنُ عَرَفَةَ سَنَةَ حَجَّ يَسْبِقُ وَيَأْخُذُ مَا يَكْفِيهِ مِنْ الْمَاءِ وَيَقُولُ كُلُّ مَنْ نَازَعَنِي قَاتَلْته ؛ لِأَنَّى أَحَقُّ قَالَ الْبُرْزُلِيُّ مَعْنَاهُ مَا لَمْ يَخْشَ عَلَى مَنْ بَعْدَهُ ضَرَرًا ( قَوْلُهُ فَلَيْسَ لَهُ أَنْ يَمْنَعَ ذَلِكَ ) أَيْ : وَلَوْ لَمْ يَكُنْ مُضْطَرًّا وَلَا صَاحِبَ زَرْعٍ ( قَوْلُهُ فَلَهُ حِينَئِذٍ أَنْ يَمْنَعَ النَّاسَ عَنْهَا ) ؛ لِأَنَّ ذَلِكَ صَارَ إحْيَاءً أَيْ : لِأَنَّ قَوْلَ الْمُصَنِّفِ وَلَا حَفَرَ بِئْرَ مَاشِيَةِ لَا يَكُونُ إحْيَاءٌ مُقَيَّدٌ بِأَنْ لَا يَكُونَ بَيْنَ الْمِلْكِيَّةِ وَإِلَّا كَانَ إحْيَاءً وَبَيَانُ الْمِلْكِيَّةِ بِأَنْ يَشْهَدَ عِنْدَ الْحَفْرِ أَنَّهُ يَحْفِرَهَا لِيَمْلِكَ مَاءَهَا .( 20 / 373 )
