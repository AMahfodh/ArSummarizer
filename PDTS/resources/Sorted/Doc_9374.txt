فَصْلٌ وَيَحْرُمُ رِبَا النَّسِيئَةِ مِنْ النَّسَاءِ بِالْمَدِّ وَهُوَ التَّأْخِيرُ ( بَيْنَ مَا ) أَيْ مَبِيعَيْنِ ( اتَّفَقَا فِي عِلَّةِ رِبَا الْفَضْلِ ) وَهِيَ الْكَيْلُ وَالْوَزْنُ وَإِنْ اخْتَلَفَ الْجِنْسُ وَأَمَّا الْجِنْسُ فَشَرْطٌ لِتَحْرِيمِ رِبَا الْفَضْلِ كَمَا أَنَّ الزِّنَا عِلَّةُ الْحَدِّ ، وَالْإِحْصَانُ شَرْطٌ لِلرَّجْمِ ( كَ ) بَيْعِ ( مُدَبَّرٍ بِمِثْلِهِ ) أَيْ مُدَبَّرٍ ( أَوْ ) ب ( شَعِيرٍ وك ) بَيْعِ دِرْهَمٍ مِنْ قَزٍّ ب ) رَطْلٍ مِنْ خُبْرٍ فَيُشْتَرَطُ ) لِذَلِكَ ( حُلُولٌ وَقَبْضٌ بِالْمَجْلِسِ ) مُطْلَقًا وَتَمَاثُلٌ إنْ اتَّحَدَ الْجِنْسُ وَتَقَدَّمَ وَلِأَنَّهُمَا مَالَانِ مِنْ أَمْوَالِ الرِّبَا عِلَّتُهُمَا مُتَّفِقَةٌ ، فَحَرُمَ التَّفَرُّقُ فِيهَا قَبْلَ الْقَبْضِ كَالصَّرْفِ .
