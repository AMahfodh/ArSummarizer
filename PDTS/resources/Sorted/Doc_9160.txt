مَضْمُونَةٌ عَلَى الْغَاصِبِ مَا دَامَ فِي يَدِهِ ( قَوْلُهُ : اسْتَقَرَّ عَلَيْهِ ) أَيْ الْغَاصِبُ ( قَوْلُهُ : وَيُقَوَّمُ قَبْلَ الْبُرْءِ ) أَيْ فَيُعْتَبَرُ قِيمَتُهُ سَلِيمًا ذَا أُصْبُعٍ زَائِدَةٍ وَمَجْرُوحًا سَائِلَ الدَّمُ وَيَجِبُ التَّفَاوُتُ بَيْنَهُمَا ( قَوْلُهُ : قِصَاصًا أَوْ حَدًّا ) أَيْ بِجِنَايَةٍ وَقَعَتْ مِنْهُ بَعْدَ الْغَصْبِ ، بِخِلَافِ مَا لَوْ قُطِعَتْ بِجِنَايَةٍ فِي يَدِ الْمَالِكِ فَإِنَّهَا غَيْرُ مَضْمُونَةٍ لِأَنَّ الْمُسْتَنِدَ إلَى سَبَبٍ سَابِقٍ عَلَى الْغَصْبِ كَالْمُتَقَدِّمِ عَلَيْهِ ( قَوْلُهُ : مَعَ رُبُعِ الدِّيَةِ ) أَيْ الْمُقَابِلِ لِجُزْئِهِ الْحُرِّ ( قَوْلُهُ : وَنِصْفِ الْأَرْشِ ) وَهُوَ نِصْفُ مَا نَقَصَ مِنْ قِيمَتِهِ( 16 / 318 )
