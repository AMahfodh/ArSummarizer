( فَصْلٌ : وَالصَّرْفُ بَيْعُ نَقْدٍ بِنَقْدِ ) مِنْ جِنْسِهِ أَوْ غَيْرِهِ - مَأْخُوذٌ مِنْ الصَّرِيفِ وَهُوَ تَصْوِيتُ النَّقْدِ بِالْمِيزَانِ - ( وَيَبْطُلُ ) صَرْفٌ كَبُطْلَانِ ( سَلَمٍ بِتَفَرُّقٍ ) بِبَدَنٍ ( يُبْطِلُ خِيَارَ الْمَجْلِسِ ) قَبْلَ تَقَابُضٍ ، ( وَ ) يَبْطُلُ أَيْضًا ( بِمَوْتِ ) أَحَدِ الْمُتَصَارِفَيْنِ ( قَبْلَ تَقَابُضٍ ) مِنْ الْجَانِبَيْنِ فِي صَرْفٍ ، لِمَا تَقَدَّمَ مِنْ قَوْلِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ : { يَدًا بِيَدٍ } .
