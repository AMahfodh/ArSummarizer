فَصْل وَيُبَاحُ لِمَنْ أَرَادَ خِطْبَةَ امْرَأَةٍ بِكَسْرِ الْخَاءِ ( وَغَلَبَ عَلَى ظَنِّهِ إجَابَتُهُ نَظَرُ مَا يَظْهَرُ ) مِنْهَا ( غَالِبًا كَوَجْهٍ وَرَقَبَةٍ وَيَدٍ وَقَدَمٍ ) لِحَدِيثِ { إذَا خَطَبَ أَحَدُكُمْ الْمَرْأَةَ فَقَدَرَ أَنْ يَرَى مِنْهَا بَعْضَ مَا يَدْعُوهُ إلَى نِكَاحِهَا فَلْيَفْعَلْ } رَوَاهُ أَحْمَدُ وَأَبُو دَاوُد ، وَقَالَ أَيْ النَّبِيُّ : { إذَا أَلْقَى اللَّهُ عَزَّ وَجَلَّ فِي قَلْبِ امْرِئٍ خِطْبَةَ امْرَأَةٍ فَلَا بَأْسَ أَنْ يَنْظُرَ إلَيْهَا } رَوَاهُ أَحْمَدُ وَابْنُ مَاجَهْ مِنْ حَدِيثِ مُحَمَّدِ بْنِ سَلَمَةَ .
