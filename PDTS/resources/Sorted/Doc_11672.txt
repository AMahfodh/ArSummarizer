وَعَنْ أَبِي هُرَيْرَةَ مَرْفُوعًا { أَمَا يَخْشَى الَّذِي يَرْفَعُ رَأْسَهُ قَبْلَ الْإِمَامِ أَنْ يُحَوِّلَ اللَّهُ رَأْسَهُ رَأْسَ حِمَارٍ ، أَوْ يَجْعَلَ صُورَتَهُ صُورَةَ حِمَارٍ } مُتَّفَقٌ عَلَيْهِ ، وَلَا تَبْطُلُ إنْ عَادَ لِلْمُتَابَعَةِ ( وَعَلَيْهِ ) أَيْ : الَّذِي فَعَلَ ذَلِكَ عَمْدًا ( وَعَلَى جَاهِلٍ وَنَاسٍ ) فَعَلَ ذَلِكَ وَ ( ذَكَرَ أَنْ يَرْجِعَ لِ ) يَفْعَلَ مَا سَبَقَ بِهِ إمَامَهُ لِ ( يَأْتِيَ بِهِ ) أَيْ : بِمَا فَعَلَهُ قَبْلَ الْإِمَامِ ( مَعَهُ ) أَيْ : مَعَ إمَامِهِ أَيْ : عَقِبَهُ ، لِيَكُونَ مُؤْتَمًّا بِهِ ( فَإِنْ أَبَى ) الرُّجُوعَ ( عَالِمًا ) وُجُوبَهُ ( عَمْدًا ) أَيْ : غَيْرَ سَاهٍ ( حَتَّى أَدْرَكَهُ ) إمَامُهُ ( فِيهِ ) أَيْ : فِيمَا سَبَقَهُ بِهِ ( بَطَلَتْ ) صَلَاتُهُ ، لِتَرْكِهِ الْمُتَابَعَةَ الْوَاجِبَةَ بِلَا عُذْرٍ .
