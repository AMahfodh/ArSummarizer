لِحَدِيثِ أَبِي دَاوُد { أَنَّ رَجُلًا سَأَلَ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فَقَالَ : إنِّي نَذَرْتُ أَنْ أَذْبَحَ بِالْأَبْوَاءِ ، فَقَالَ : أَبِهَا صَنَمٌ ؟ قَالَ : لَا ، قَالَ : أَوْفِ بِنَذْرِكَ } وَلِأَنَّهُ قَصَدَ نَفْعَ أَهْلِ ذَلِكَ الْمَوْضِعِ ، فَكَانَ عَلَيْهِ إيصَالُهُ إلَيْهِمْ ( فَإِنْ كَانَ بِهِ ) أَيْ : ذَلِكَ الْمَوْضِعِ ( نَحْوُ صَنَمٍ ) كَبُيُوتِ نَارٍ أَوْ كَنِيسَةٍ أَوْ أَمْرِ كُفْرٍ ( فَ ) هُوَ ( نَذْرُ مَعْصِيَةٍ ) يَحْرُمُ الْوَفَاءُ بِهِ ، لِحَدِيثِ { وَمَنْ نَذَرَ أَنْ يَعْصِيَ اللَّهَ فَلَا يَعْصِهِ } .
