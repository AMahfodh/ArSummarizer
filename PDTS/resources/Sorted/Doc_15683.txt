( الثَّالِثُ الشَّهَادَةُ فَيَثْبُتُ الْمَالُ بِشَاهِدٍ وَامْرَأَتَيْنِ أَوْ ) شَاهِدٍ ( وَيَمِينٍ وَلَا يَثْبُتُ الْقَطْعُ إلَّا بِشَاهِدَيْنِ ) فَلَا يَثْبُتُ بِمَا ذُكِرَ كَسَائِرِ الْعُقُوبَاتِ وَهَذَا كَمَا يَثْبُتُ بِهِ الْغَصْبُ الْمُعَلَّقُ عَلَيْهِ الطَّلَاقُ أَوْ الْعِتْقُ دُونَهُمَا وَيُخَالِفُ مَا لَوْ شَهِدَ بِالْقَتْلِ الْعَمْدِ رَجُلٌ وَامْرَأَتَانِ حَيْثُ لَا يَجِبُ الْقِصَاصُ وَلَا الدِّيَةُ ؛ لِأَنَّ الدِّيَةَ بَدَلٌ عَنْ الْقَتْلِ ، وَالْغُرْمُ هُنَا لَيْسَ بَدَلًا عَنْ الْقَطْعِ وَوَصَفَ الشَّاهِدَيْنِ بِقَوْلِهِ ( يُبَيِّنَانِ السَّارِقَ ، وَالْمَسْرُوقَ مِنْهُ ) وَقَدْرَ الْمَسْرُوقِ كَمَا يُؤْخَذُ مِنْ كَلَامِ الْأَصْلِ ( وَالْحِرْزَ ) بِتَعْيِينٍ أَوْ وَصْفٍ كَنَظِيرِهِ فِيمَا مَرَّ فِي الْمُقِرِّ بِالسَّرِقَةِ ( وَ ) يُشْتَرَطُ ( أَنْ يَقُولَ ) الشَّاهِدُ ( لَا أَعْلَمُ لَهُ فِيهِ شُبْهَةً ) وَقِيَاسُهُ اشْتِرَاطُ ذَلِكَ فِي الْإِقْرَارِ بِالسَّرِقَةِ ( وَإِنْ شَهِدَ ) لَهُ ( وَاحِدٌ بِثَوْبٍ أَبْيَضَ ) أَيْ بِسَرِقَتِهِ ( وَآخَرُ بِأَسْوَدَ فَلَهُ أَنْ يَحْلِفَ مَعَ أَحَدِهِمَا وَلَهُ ) مَعَ ذَلِكَ ( أَنْ يَدَّعِيَ ) الثَّوْبَ ( الْآخَرَ وَيَحْلِفَ مَعَ شَاهِدِهِ وَاسْتَحَقَّهُمَا ) أَيْ الثَّوْبَيْنِ ؛ لِأَنَّ ذَلِكَ مِمَّا يَثْبُتُ بِالشَّاهِدِ ، وَالْيَمِينِ كَمَا مَرَّ ( وَلَا قَطْعَ ) لِاخْتِلَافِ الشَّاهِدَيْنِ ( أَوْ ) شَهِدَ ( اثْنَانِ ) بِسَرِقَةٍ ( وَاثْنَانِ ) بِسَرِقَةٍ ( فَإِنْ لَمْ يَتَوَارَدَا عَلَى عَيْنٍ وَاحِدَةٍ ) كَالْمِثَالِ السَّابِقِ وَكَمَا لَوْ شَهِدَ اثْنَانِ بِسَرِقَةِ كِيسٍ غُدْوَةً وَاثْنَانِ بِسَرِقَةِ كِيسٍ عَشِيَّةً ( ثَبَتَ الْقَطْعُ ، وَالْمَالَانِ ) لِتَمَامِ الْحُجَّتَيْنِ ( وَإِنْ تَوَارَدَا عَلَى عَيْنٍ وَاحِدَةٍ ) وَاخْتَلَفَ الْوَقْتُ كَأَنْ شَهِدَ اثْنَانِ بِسَرِقَةِ كَذَا غُدْوَةً وَاثْنَانِ بِسَرِقَتِهِ عَشِيَّةً ( تَعَارَضَتَا ) فَيَتَسَاقَطَانِ وَفِي صُورَةِ شَهَادَةِ الْوَاحِدِ ، وَالْوَاحِدِ لَا يُقَالُ تَعَارَضَتَا ؛ لِأَنَّ الْحُجَّةَ لَا تَتِمُّ ( وَإِنْ شَهِدَ أَحَدُهُمَا بِكِيسٍ ، وَالْآخَرُ بِكِيسَيْنِ ثَبَتَ الْكِيسُ وَقُطِعَ بِهِ ) السَّارِقُ ( إنْ بَلَغَ نِصَابًا وَإِنْ شَهِدَ( 20 / 124 )
