( قَوْلُهُ إذْ مَنْ قُبِلَ فِي شَيْءٍ ) أَيْ إذْ مَنْ قُبِلَ قَوْلُهُ فِي شَيْءٍ إلَخْ ( قَوْلُهُ : نَعَمْ تُقْبَلُ إلَخْ ) اسْتِدْرَاكٌ عَلَى قَوْلِهِ وَإِنَّمَا صُدِّقَتْ إلَخْ ( قَوْلُهُ : فَالْأَوْلَى التَّعْلِيلُ ) أَيْ بَدَلُ قَوْلِهِ لِتَغْلِيظِهَا عَلَى نَفْسِهَا ( قَوْلُهُ : وَيُقْبَلُ ) هُوَ عَطْفٌ عَلَى قَوْلِهِ نَعَمْ تُقْبَلُ هِيَ إلَخْ ( قَوْلُهُ : فَقَالَتْ ) أَيْ الرَّجْعِيَّةُ ( قَوْلُهُ و قَيَّدَهُ الْقَفَّالُ إلَخْ ) مُعْتَمَدٌ ( قَوْلُهُ : وَأَخَذَ مِنْهُ ) لَعَلَّ هَذَا الْأَخْذَ مُتَعَيِّنٌ لِأَنَّا وَإِنْ تَحَقَّقْنَا بَقَاءَ الْعِدَّةِ فِي الْبَائِنِ لَا تَنْتَقِلُ لِعِدَّةِ الْوَفَاةِ ( قَوْلُهُ : وَلَوْ مَاتَتْ ) أَيْ الرَّجْعِيَّةُ ( قَوْلُهُ : وَالْوَارِثُ ) أَيْ حَيْثُ ادَّعَاهُ فِي زَمَنٍ يُمْكِنُ فِيهِ ذَلِكَ ( قَوْلُهُ : فِيمَا عَدَاهَا ) أَيْ مِنْ الْحَمْلِ وَالْأَقْرَاءِ ( قَوْلُهُ : بِتَصْدِيقِهِ ) أَيْ الْوَارِثِ ( قَوْلُهُ : وَصَغِيرَةٌ ) أَيْ لَمْ تَبْلُغْ زَمَنًا يُمْكِنُ فِيهِ الْحَمْلُ ، أَمَّا مَنْ بَلَغَتْ وَلَمْ تَحِضْ فَهِيَ كَالصَّغِيرَةِ وَلَيْسَتْ صَغِيرَةً كَمَا يَأْتِي فِي كَلَامِ الشَّارِحِ ( قَوْلُهُ وَحَذَفَهَا ) أَيْ الصَّغِيرَةَ ( قَوْلُهُ : لِأَنَّهَا مُؤْتَمَنَةٌ عَلَى مَا فِي رَحِمِهَا ) تَعْلِيلٌ لِتَصْدِيقِهَا بِالنِّسْبَةِ لِانْقِضَاءِ الْعِدَّةِ ، وَلَمْ يُعَلِّلْ عَدَمَ قَبُولِ قَوْلِهَا فِي النَّسَبِ وَالِاسْتِيلَادِ مَعَ أَنَّ الْعِلَّةَ جَارِيَةٌ فِيهِمَا فَكَانَ الْقِيَاسُ الْقَبُولَ ، إلَّا أَنْ يُقَالَ : لَمَّا كَانَ النَّسَبُ وَالْوِلَادَةُ مُتَعَلِّقَيْنِ بِالْغَيْرِ وَأَمْكَنَتْ إقَامَةُ الْبَيِّنَةِ عَلَى الْوِلَادَةِ لَمْ يُقْبَلْ قَوْلُهَا فِيهِمَا ، بِخِلَافِ انْقِضَاءِ الْعِدَّةِ لِتَعَلُّقِهَا بِهَا فَصُدِّقَتْ فِيهَا ( قَوْلُهُ : وَلَا بُدَّ مِنْ انْفِصَالِ جَمِيعِ الْحَمْلِ ) هَلْ يَكْفِي فِي صِحَّةِ الرَّجْعَةِ بَقَاءُ الشَّعْرِ وَحْدَهُ ، أَوْ يُفَرَّقُ بَيْنَ كَوْنِهِ بَقِيَ مُعَلَّقًا بِهِ فَتَصِحُّ الرَّجْعَةُ وَكَوْنِهِ انْفَصَلَ عَنْهُ فَلَا تَصِحُّ ؟ فِيهِ نَظَرٌ ، وَالْأَقْرَبُ الْأَوَّلُ لِأَنَّهُ صُدِّقَ عَلَيْهِ أَنَّهُ لَمْ يَنْفَصِلْ بِتَمَامِهِ لِشَغْلِ الرَّحِمِ( 23 / 2 )
