سم ( قَوْلُهُ : وَمِنْ ثَمَّ ) أَيْ مِنْ أَجْلِ أَنَّهُ لَوْ اطَّرَدَ عُرْفُهُمْ إلَخْ ( قَوْلُهُ : بَحَثَ الْأَذْرَعِيُّ إلَخْ ) أَقَرَّهُ النِّهَايَةُ وَالْمُغْنِي ( قَوْلُهُ : وَقَوْلُ ابْنِ الصَّبَّاغِ ) مُبْتَدَأٌ وَخَبَرُهُ قَوْلُهُ : يَحْمِلُ إلَخْ ( قَوْلُهُ : بِأَنَّهُ إلَخْ ) أَيْ قَوْلُ ابْنِ الصَّبَّاغِ ( قَوْلُهُ : وَإِنَّمَا لَمْ يَصِحَّ ) إلَى الْمَتْنِ فِي النِّهَايَةِ ( قَوْلُهُ : وَإِنَّمَا لَمْ يَصِحَّ إلَخْ ) رَاجِعٌ إلَى قَوْلِ الْمَتْنِ وَفِي الْبَلَدِ نَقْدٌ غَالِبٌ تَعَيَّنَ .
