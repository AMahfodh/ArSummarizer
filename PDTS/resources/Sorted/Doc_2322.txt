لَكِنْ ( بِعِلَاجٍ ) فِيهِ ( كَرُطَبٍ يَتَجَفَّفُ ) أَيْ يُمْكِنُ تَجْفِيفُهُ وَلَبَنٍ يَصِيرُ أَقِطًا ( فَإِنْ كَانَتْ الْغِبْطَةُ فِي بَيْعِهِ بِيعَ ) جَمِيعُهُ بِإِذْنِ الْحَاكِمِ إنْ وَجَدَهُ ، وَإِلَّا اسْتِقْلَالًا كَمَا يُؤْخَذُ مِمَّا مَرَّ ( أَوْ ) كَانَتْ الْغِبْطَةُ ( فِي تَجْفِيفِهِ وَتَبَرَّعَ بِهِ الْوَاجِدُ ) لَهُ أَوْ غَيْرُهُ ( جَفَّفَهُ ) ؛( 10 / 275 )
