( وَهُوَ ) أَيْ الصَّدَاقُ ( الْعِوَضُ فِي النِّكَاحِ ) سَوَاءٌ سُمِّيَ فِي الْعَقْدِ أَوْ فُرِضَ بَعْدَهُ بِتَرَاضِيهِمَا أَوْ الْحَاكِمِ ( وَنَحْوَهُ ) أَيْ : نَحْوَ النِّكَاحِ كَوَطْءِ الشُّبْهَةِ وَالزِّنَا بِأَمَةٍ أَوْ مُكْرَهَةٍ ( وَيُسَنُّ تَخْفِيفُهُ ) أَيْ : الصَّدَاقُ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { أَعْظَمُ النِّكَاحِ بَرَكَةً أَيْسَرُهُ مُؤْنَةً } رَوَاهُ أَحْمَدُ وَفِيهِ ضَعْفٌ .
