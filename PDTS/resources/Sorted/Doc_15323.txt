( قَوْلُهُ : وَإِنْ نَزَعَ رِجْلًا ) قَالَ بْن يَصِحُّ فَرْضُهُ فِيمَنْ كَانَ عَلَى طَهَارَةٍ وَأَرَادَ نَزْعَهُمَا لِيَغْسِلَ رِجْلَيْهِ وَيَصِحَّ فَرْضُهُ فِيمَنْ كَانَ عَلَى غَيْرِ طَهَارَةٍ وَأَرَادَ نَزْعَهُمَا لِيَتَوَضَّأَ وَيَغْسِلَ رِجْلَيْهِ ا ه ( قَوْلُهُ : فَلَمْ يَقْدِرْ عَلَيْهِ ) أَيْ لَا بِنَفْسِهِ وَلَا بِغَيْرِهِ كَمَا قَالَ شَيْخُنَا ( قَوْلُهُ : وَضَاقَ الْوَقْتُ الَّذِي هُوَ فِيهِ مِنْ اخْتِيَارِيٍّ أَوْ ضَرُورِيٍّ ) هَذَا هُوَ الْأَظْهَرُ كَمَا فِي عبق وَشُبْ وَفِي ح قَصُرَ الْوَقْتُ عَلَى الِاخْتِيَارِيِّ ( قَوْلُهُ : إعْطَاءً لِسَائِرِ الْأَعْضَاءِ ) أَيْ أَعْضَاءِ الْوُضُوءِ وَقَوْلُهُ : حُكْمُ مَا تَحْتَ الْخُفِّ أَيْ وَهِيَ الَّتِي تَعَذَّرَ نَزْعُهَا فَلَمَّا تَعَذَّرَ نَزْعُهَا صَارَتْ مُتَعَذِّرَةَ الْغُسْلِ وَحَيْثُ صَارَتْ مُتَعَذِّرَةَ الْغُسْلِ صَارَتْ الْأَعْضَاءُ كُلُّهَا كَأَنَّهَا مُتَعَذِّرَةَ الْغُسْلِ فَلِذَا قِيلَ إنَّهُ يَتَيَمَّمُ ( قَوْلُهُ : وَتَعَذُّرِ بَعْضِ الْأَعْضَاءِ ) أَيْ وَهِيَ الرِّجْلُ الَّتِي تَعَذَّرَ نَزْعُ خُفَّيْهَا وَهَذَا تَوْضِيحٌ لِمَا قَبْلَهُ ( قَوْلُهُ : فَيَجْمَعُ بَيْنَ مَسْحٍ وَغُسْلٍ ) اُنْظُرْ لَوْ قُلْنَا بِالْقَوْلِ الثَّانِي وَاحْتَاجَ لِطَهَارَةٍ أُخْرَى قَبْلَ نَقْضِ الطَّهَارَةِ الْأُولَى فَهَلْ يَلْبَسُ الْمَنْزُوعَةَ وَيَمْسَحُ عَلَيْهَا أَوْ كَيْفَ الْحَالُ وَالظَّاهِرُ الْأَوَّلُ ( قَوْلُهُ : مَا تَحْتَ الْحَائِلِ ) أَيْ وَهُوَ الْخُفُّ الَّذِي تَعَذَّرَ نَزْعُهُ وَالْجَبِيرَةُ ( قَوْلُهُ : مَسَحَ كَالْجَبِيرَةِ ) أَيْ مَسَحَ عَلَى مَا عَسُرَ نَزْعُهُ وَيَغْسِلُ الرِّجْلَ الْأُخْرَى الَّتِي نَزَعَ خُفَّهَا فَيَجْمَعُ بَيْنَ الْغُسْلِ وَالْمَسْحِ كَالْجَبِيرَةِ ( قَوْلُهُ : وَالْأَظْهَرُ اعْتِبَارُ الْقِيمَةِ بِحَالِ الْخُفِّ ) أَيْ فَإِنْ كَانَتْ قِيمَتُهُ فِي ذَاتِهِ قَلِيلَةً مُزِّقَ وَلَوْ كَانَتْ كَثِيرَةً بِالنِّسْبَةِ لِلَّابِسِ وَإِنْ كَانَتْ قِيمَتُهُ فِي ذَاتِهِ كَثِيرَةً فَلَا يُمَزَّقُ وَإِنْ كَانَتْ قَلِيلَةً بِالنِّسْبَةِ لِلَّابِسِ وَقِيلَ إنَّ قِيمَةَ الْخُفِّ تُعْتَبَرُ بِالنِّسْبَةِ لِحَالِ اللَّابِسِ( 2 / 28 )
