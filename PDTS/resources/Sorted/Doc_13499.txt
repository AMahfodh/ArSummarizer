( وَلَيْسَ لَهُ ) أَيْ : الزَّوْجِ ( بُدَاءَةٌ ) فِي قَسْمٍ ( وَلَا سَفَرٍ بِإِحْدَاهُنَّ ) طَالَ السَّفَرُ أَوْ قَصُرَ ( بِلَا قُرْعَةٍ ) ؛ لِأَنَّهُ تَفْضِيلٌ لَهَا ، وَالتَّسْوِيَةُ وَاجِبَةٌ { ، وَكَانَ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ إذَا أَرَادَ السَّفَرَ أَقْرَعَ بَيْنَ نِسَائِهِ فَمَنْ خَرَجَتْ لَهَا الْقُرْعَةُ خَرَجَ بِهَا مَعَهُ } مُتَّفَقٌ عَلَيْهِ ، وَإِذَا سَافَرَ بِهَا بِقُرْعَةٍ إلَى مَحِلٍّ ثُمَّ بَدَا لَهُ غَيْرُهُ وَلَوْ أَبْعَدَ مِنْهُ فَلَهُ أَنْ يَصْحَبَهَا مَعَهُ ( إلَّا بِرِضَاهُنَّ ، وَرِضَاهُ ) فَإِذَا رَضِيَ الزَّوْجَاتُ ، وَالزَّوْجُ بِالْبُدَاءَةِ بِإِحْدَاهُنَّ أَوْ السَّفَرِ بِهَا جَازَ ؛ لِأَنَّ الْحَقَّ لَا يَخْرُجُ عَنْهُمْ ( وَيَقْضِي ) زَوْجٌ لِبَقِيَّةِ زَوْجَاتِهِ ( مَعَ قُرْعَةٍ ) فِي سَفَرٍ بِإِحْدَاهُنَّ ( أَوْ ) مَعَ ( رِضَاهُنَّ ) بِسَفَرٍ بِمُعَيَّنَةٍ مِنْهُنَّ ( مَا تَعَقَّبَهُ سَفَرٌ ) أَيْ : مَا أَقَامَهُ فِي الْبَلَدِ الَّذِي سَافَرَ إلَيْهِ ( أَوْ تَخَلَّلَهُ ) سَفَرٌ ( مِنْ إقَامَةٍ ) أَيْ : مُدَّةَ إقَامَتِهِ فِي أَثْنَاءِ سَفَرِهِ لِتَسَاكُنِهِمَا إذَنْ لَا زَمَنِ مَسِيرِهِ ، وَحِلِّهِ وَتَرْحَالِهِ ؛ لِأَنَّهُ لَا يُسَمَّى سَكَنًا .
