( وَإِنْ ذَرَعَهُ الْقَيْءُ وَخَرَجَ ) وَلَمْ يَعُدْ ( لَا يُفْطِرُ مُطْلَقًا ) مَلَأَ أَوْ لَا ( فَإِنْ عَادَ ) بِلَا صُنْعِهِ ( وَ ) لَوْ ( هُوَ مِلْءُ الْفَمِ مَعَ تَذَكُّرِهِ لِلصَّوْمِ لَا يَفْسُدُ ) خِلَافًا لِلثَّانِي ( وَإِنْ أَعَادَهُ ) أَوْ قَدْرُ حِمَّصَةٍ مِنْهُ فَأَكْثَرَ حَدَّادِيٌّ ( أَفْطَرَ إجْمَاعًا ) وَلَا كَفَّارَةَ ( إنْ مَلَأَ الْفَمَ وَإِلَّا لَا ) هُوَ الْمُخْتَارُ ( وَإِنْ اسْتِقَاءٌ ) أَيْ طَلَبَ الْقَيْءَ ( عَامِدًا ) أَيْ مُتَذَكِّرًا لِصَوْمٍ ( إنْ كَانَ مِلْءَ الْفَمِ فَسَدَ بِالْإِجْمَاعِ ) مُطْلَقًا ( وَإِنْ أَقَلَّ لَا ) عِنْدَ الثَّانِي وَهُوَ الصَّحِيحُ ، لَكِنَّ ظَاهِرَ الرِّوَايَةِ كَقَوْلِ مُحَمَّدٍ إنَّهُ يَفْسُدُ كَمَا فِي الْفَتْحِ عَنْ الْكَافِي ( فَإِنْ عَادَ بِنَفْسِهِ لَمْ يُفْطِرْ وَإِنْ أَعَادَهُ فَفِيهِ رِوَايَتَانِ ) أَصَحُّهُمَا لَا يُفْسِدُ مُحِيطٌ ( وَهَذَا ) كُلُّهُ ( فِي قَيْءِ طَعَامٍ أَوْ مَاءٍ أَوْ مِرَّةٍ ) أَوْ دَمٍ ( فَإِنْ كَانَ بَلْغَمًا فَغَيْرُ مُفْسِدٍ ) مُطْلَقًا خِلَافًا لِلثَّانِي وَاسْتَحْسَنَهُ الْكَمَالُ وَغَيْرُهُ .
