( وَإِنْ حَلَفَ ) إنْسَانٌ ( بِ ) اسْمِ ( اللَّهِ ) أَوْ بِصِفَةٍ مِنْ صِفَاتِهِ النَّفْسِيَّةِ أَوْ الْمَعْنَوِيَّةِ ( لَيَفْعَلَنَّ مَعْصِيَةً ) مِنْ مَعَاصِي اللَّهِ تَعَالَى كَشُرْبِ الْخَمْرِ أَوْ قَتْلِ نَفْسٍ أَوْ سَبِّ مَنْ لَا يَجُوزُ سَبُّهُ ( فَلْيُكَفِّرْ عَنْ يَمِينِهِ ) الَّذِي حَلَفَهُ ( وَلَا يَفْعَلْ ذَلِكَ ) الْمَحْلُوفَ عَلَيْهِ ( وَإِنْ تَجَرَّأَ ) أَيْ اقْتَحَمَ ( فَفَعَلَهُ ) أَيْ الْمَحْلُوفَ عَلَيْهِ مَعَ عِلْمِهِ بِأَنَّهُ مَعْصِيَةٌ ، وَلَمْ يُبَالِ بِعُقُوبَةِ عَاقِبَتِهِ ( فَهُوَ آثِمٌ ) لِفِعْلِهِ الْمَعْصِيَةَ ( وَلَا كَفَّارَةَ عَلَيْهِ لِيَمِينِهِ ) ؛ لِأَنَّهُ بَرَّ فِي يَمِينِهِ .
