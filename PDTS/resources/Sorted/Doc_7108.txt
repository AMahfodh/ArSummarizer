وَرُوِيَ عَنْ ابْنِ عُمَرَ وَابْنِ عَبَّاسٍ وَعَائِشَةَ لِعُمُومِ الْأَحَادِيثِ وَكَحَرَمِ الْمَدِينَةِ وَلِأَنَّهَا أَمَانَةٌ فَلَمْ يَخْتَلِفْ حُكْمُهَا بِالْحِلِّ وَالْحَرَمِ كَالْوَدِيعَةِ ، وَحَدِيثِ { لَا تَحِلُّ سَاقِطَتُهَا إلَّا لِمُنْشِدٍ } يَحْتَمِلُ أَنْ يُرَادَ بِهِ إلَّا لِمَنْ عَرَّفَهَا عَامًا وَتَخْصِيصُهَا بِذَلِكَ لِتَأَكُّدِهَا كَحَدِيثِ { ضَالَّةُ الْمُسْلِمِ حَرْقُ النَّارِ } ( أَوْ لَمْ يَخْتَرْ ) الْمُلْتَقِطُ تَمَلُّكَهَا هُوَ مَعْنَى قَوْلِهِ : دَخَلَتْ فِي مِلْكِهِ حُكْمًا وَتَقَدَّمَ ( أَوْ أَخَّرَهُ ) أَيْ التَّعْرِيفَ ( لِعُذْرٍ ) ثُمَّ عَرَّفَهَا فَيَمْلِكُهَا وَتَقَدَّمَ مَا فِيهِ( 7 / 76 )
