مَسْأَلَةٌ حَرْفُ النَّفْيِ قَدْ يَدْخُلُ عَلَى الْمَاهِيَّةِ وَالْمُرَادُ نَفْيُ الْأَصْلِ ، كَقَوْلِهِ : { لَا يَسْمَعُونَ فِيهَا لَغْوًا وَلَا تَأْثِيمًا } وَقَوْلُهُ : { فَالْيَوْمَ لَا يَخْرُجُونَ مِنْهَا } وَقَدْ يُرَادُ بِهِ نَفْيُ الْكَمَالِ مَعَ بَقَاءِ الْأَصْلِ ، كَقَوْلِهِ تَعَالَى : { إنَّهُمْ لَا أَيْمَانَ لَهُمْ لَعَلَّهُمْ يَنْتَهُونَ } ثُمَّ قَالَ : { أَلَا تُقَاتِلُونَ قَوْمًا نَكَثُوا أَيْمَانَهُمْ } فَنَفَاهَا أَوَّلًا ، ثُمَّ أَثْبَتَهَا ثَانِيًا ، فَدَلَّ عَلَى أَنَّهُ لَمْ يُرِدْ نَفْيَ الْأَصْلِ ، بَلْ نَفْيَ الْكَمَالِ .
