( أَقَرَّتْ الْحُرَّةُ الْمُكَلَّفَةُ بِدَيْنٍ ) لِآخَرَ ( فَكَذَّبَهَا زَوْجُهَا صَحَّ ) إقْرَارُهَا ( فِي حَقِّهِ أَيْضًا ) عِنْدَ أَبِي حَنِيفَةَ ( فَتُحْبَسُ ) الْمُقِرَّةُ ( وَتُلَازِمُ ) وَإِنْ تَضَرَّرَ الزَّوْجُ وَهَذِهِ إحْدَى الْمَسَائِلِ السِّتِّ الْخَارِجَةِ مِنْ قَاعِدَةِ الْإِقْرَارُ حُجَّةٌ قَاصِرَةٌ عَلَى الْمُقِرِّ وَلَا يَتَعَدَّى إلَى غَيْرِهِ وَهِيَ فِي الْأَشْبَاهِ ، وَيَنْبَغِي أَنْ يَخْرُجَ أَيْضًا مَنْ كَانَ فِي إجَارَةِ غَيْرِهِ فَأَقَرَّ لِآخَرَ بِدَيْنٍ فَإِنَّ لَهُ حَبْسَهُ وَإِنْ تَضَرَّرَ الْمُسْتَأْجِرُ وَهِيَ وَاقِعَةُ الْفَتْوَى وَلَمْ نَرَهَا صَرِيحَةً ( وَعِنْدَهُمَا لَا ) تُصَدَّقُ فِي حَقِّ الزَّوْجِ فَلَا تُحْبَسُ وَلَا تُلَازِمُ دُرَرٌ .
