( وَتُصَلَّى الْجُمُعَةُ فِي ) حَالِ ( الْخَوْفِ حَضَرًا ) لَا سَفَرًا ( بِشَرْطِ كَوْنِ كُلِّ طَائِفَةٍ أَرْبَعِينَ ) رَجُلًا ( فَأَكْثَرَ ) مِنْ أَهْلِ وُجُوبِهَا لِاشْتِرَاطِ الْعَدَدِ وَالِاسْتِيطَانِ ( فَيُصَلِّي بِكُلِّ طَائِفَةٍ رَكْعَةً بَعْدَ حُضُورِهَا الْخُطْبَةَ ) - يَعْنِي خُطْبَتَيْ الْجُمُعَةِ يَعْنِي أَنَّهُ يُشْتَرَطُ أَنْ يُحْرِمَ بِمَنْ حَضَرَتْ الْخُطْبَةَ لِاشْتِرَاطِ الْمُوَالَاةِ بَيْنَ الْخُطْبَتَيْنِ وَالْمُوَالَاةِ بَيْنَ الْخُطْبَتَيْنِ وَالصَّلَاةِ ( فَإِنْ أَحْرَمَ بِ ) الطَّائِفَةِ ( الَّتِي لَمْ تَحْضُرْهَا لَمْ تَصِحَّ ) الْجُمُعَةُ ( حَتَّى يَخْطُبَ لَهَا ) كَغَيْرِ حَالَةِ الْخَوْفِ ( وَتَقْضِي كُلُّ طَائِفَةٍ رَكْعَةً بِلَا جَهْرٍ ) بِالْقِرَاءَةِ ، كَالْمَسْبُوقِ إذَا فَاتَهُ مِنْ الْجُمُعَةِ رَكْعَةٌ .
