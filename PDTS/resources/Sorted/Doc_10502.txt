( وَإِنْ أَسْلَمَ ) عَبْدٌ ( وَعَتَقَ ثُمَّ أَسْلَمْنَ أَوْ أَسْلَمْنَ ثُمَّ عَتَقَ ثُمَّ أَسْلَمَ اخْتَارَ ) مِنْهُنَّ ( أَرْبَعًا بِشَرْطِهِ ) وَهُوَ عَدَمُ الطَّوْلِ وَخَوْفُ الْعَنَتِ وَقْتَ اجْتِمَاعِ إسْلَامِهِ بِإِسْلَامِهِنَّ ؛ لِأَنَّهُ حُرٌّ إذْ ذَاكَ يَجُوزُ لَهُ ابْتِدَاءُ نِكَاحِهِنَّ فَجَازَ لَهُ إبْقَاؤُهُ ( وَلَوْ كَانَ تَحْتَهُ ) أَيْ الْعَبْدِ ( حَرَائِرُ فَأَسْلَمْنَ مَعَهُ لَمْ يَكُنْ لَهُنَّ خِيَارُ الْفَسْخِ ) لِرِضَاهُنَّ بِهِ عَبْدًا كَافِرًا فَعَبْدٌ مُسْلِمٌ أَوْلَى ( وَلَوْ أَسْلَمَتْ مَنْ تَزَوَّجَتْ بِاثْنَيْنِ فِي عَقْدٍ لَمْ يَكُنْ لَهَا أَنْ تَخْتَارَ أَحَدَهُمَا وَلَوْ أَسْلَمُوا مَعًا ) ؛ لِأَنَّ ذَلِكَ لَيْسَ سَائِغًا عِنْدَ أَحَدٍ مِنْ أَهْلِ الْأَدْيَانِ ، وَلِأَنَّ الْمَرْأَةَ لَيْسَ لَهَا اخْتِيَارُ النِّكَاحِ وَفَسْخُهُ بِخِلَافِ الرَّجُلِ .( 8 / 478 )
