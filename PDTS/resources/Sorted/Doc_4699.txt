قَالَهُ أَصْحَابُنَا ( وَ ) يَكُونُ ( وَلَاؤُهُ لِسَيِّدِهِ ) قَالَ فِي الشَّرْحِ ( وَإِنْ عَجَزَ ) الْمُكَاتَبُ عَنْ أَدَاءِ مَالِ الْكِتَابَةِ كُلِّهِ أَوْ بَعْضِهِ ؛ فَهُوَ ( رَقِيقٌ لِصَاحِبِ الرَّقَبَةِ ) عَمَلًا بِالْوَصِيَّةِ ( وَبَطَلَتْ وَصِيَّةُ صَاحِبِ الْمَالِ فِيمَا بَقِيَ ) لِفَوَاتِ مَحَلِّهَا ( وَمَا كَانَ قَبَضَهُ ) الْمُوصَى لَهُ بِالْمَالِ ( فَ ) هُوَ ( لَهُ ) وَلَا يَرْجِعُ بِهِ عَلَيْهِ .( 13 / 279 )
