( قَوْلُهُ : بَعْدَ زَمَنِ التَّعْمِيرِ ) أَيْ بَعْدَ مُضِيِّ زَمَنِ التَّعْمِيرِ مِنْ وِلَادَتِهِ ( قَوْلُهُ : أَوْ إرْثِ شُرَكَائِهِ فِيهِ ) أَيْ فِي ذَلِكَ الْمُوَرِّثِ ( قَوْلُهُ : وَتَنْقُصُ الْأُمُّ ) أَيْ وَيَحْصُلُ لِلزَّوْجِ زِيَادَةٌ ( قَوْلُهُ : وَأُعْطِيَ الْوَارِثُ ) عَطْفٌ عَلَى قَوْلِ الْمُصَنِّفِ قُدِّرَ حَيًّا وَمَيِّتًا ( قَوْلُهُ : وَمَا اخْتَلَفَ فِيهِ حَالُهُ ) أَيْ بِحَيَاةِ الْمَفْقُودِ وَمَوْتِهِ ، وَهُوَ مَا زَادَ عَلَى أَقَلِّ النَّصِيبَيْنِ ( قَوْلُهُ : وَإِنْ لَمْ يَثْبُتْ ذَلِكَ ) أَيْ بِبَيِّنَةٍ وَاسْتَمَرَّ الْمَالُ مَوْقُوفًا ( قَوْلُهُ :( 20 / 156 )
