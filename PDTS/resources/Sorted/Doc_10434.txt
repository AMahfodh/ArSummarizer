وَيَسْقُطُ الْبَعِيدُ مِنْ بَنِي الْإِخْوَةِ بِالْقَرِيبِ مِنْهُمْ كَمَا سَبَقَ ( فَأَعْمَامٌ ) لِأَبَوَيْنِ فَأَعْمَامٌ لِأَبٍ ( فَأَبْنَاؤُهُمْ كَذَلِكَ ) ؛ أَيْ : يُقَدَّمُ ابْنُ الْعَمِّ لِأَبَوَيْنِ عَلَى ابْنِ الْعَمِّ لِأَبٍ ( فَأَعْمَامُ أَبٍ فَأَبْنَاؤُهُمْ كَذَلِكَ ) يُقَدَّمُ مَنْ لِأَبَوَيْنِ عَلَى مَنْ لِأَبٍ ( فَ ) أَعْمَامُ ( جَدٍّ فَأَبْنَاؤُهُمْ كَذَلِكَ ) يُقَدَّمُ مَنْ لِأَبَوَيْنِ عَلَى مَنْ لِأَبٍ ، ثُمَّ أَعْمَامُ أَبِي الْجَدِّ ، ثُمَّ أَبْنَاؤُهُمْ كَذَلِكَ أَبَدًا ( فَلَا يَرِثُ بَنُو أَبٍ أَعْلَى مَعَ بَنِي ) أَبٍ ( أَقْرَبَ مِنْهُ ، وَإِنْ نَزَلَتْ دَرَجَتُهُمْ ) لِمَا رَوَى ابْنُ عَبَّاسٍ : أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ { : أَلْحِقُوا الْفَرَائِضَ بِأَهْلِهَا ، فَمَا بَقِيَ فَلِأَوْلَى رَجُلٍ ذَكَرٍ } .
