( أَوْ ) نَحْوَ : ( إنْ لَمْ أَفْعَلْ ) أَنَا أَوْ : إنْ لَمْ تَفْعَلِي يَا هِنْدُ أَوْ : إنْ لَمْ تَفْعَلْ يَا زَيْدُ ( كَذَا ) - كَلُبْسِ ثَوْبٍ - ( فَعَلَيَّ صَوْمُ كَذَا ) كَشَهْرٍ وَالصَّوْمُ قُرْبَةٌ ، ( أَوْ : فَأَنْتَ ) يَا عَبْدِي ( حُرٌّ ) وَتَحْرِيرُ الرَّقَبَةِ مِنْ الْقُرَبِ ، ( أَوْ : فَأَنْتِ ) يَا زَوْجَتِي ( طَالِقٌ ) وَالطَّلَاقُ حَلُّ عِصْمَةِ النِّكَاحِ ، وَهَذَا فِي صِيغَةِ الْحِنْثِ لِأَنَّهُ قَدْ تَعَلَّقَ بِهِ الْحِنْثُ وَلَا يَبَرُّ إلَّا بِفِعْلِ مَدْخُولِ النَّفْيِ .
