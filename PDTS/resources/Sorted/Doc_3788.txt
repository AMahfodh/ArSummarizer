وَالظَّاهِرُ أَنَّ الْآبِقَ وَنَحْوَهُ مِمَّنْ لَمْ يَبْلُغْ كَالْبَالِغِ وَإِنْ لَمْ يَلْحَقْهُ الْإِثْمُ وَخَرَجَ بِالْعَاصِي بِسَفَرِهِ الْعَاصِي فِيهِ وَهُوَ مَنْ يَقْصِدُ سَفَرًا مُبَاحًا فَتَعْرِضُ لَهُ فِيهِ مَعْصِيَةٌ فَيَرْتَكِبُهَا فَلَهُ التَّرَخُّصُ ؛ لِأَنَّ سَبَبَ تَرَخُّصِهِ مُبَاحٌ قَبْلَهَا وَبَعْدَهَا ( فَلَوْ ) ( أَنْشَأَ ) سَفَرًا ( مُبَاحًا ثُمَّ جَعَلَهُ مَعْصِيَةً فَلَا تَرَخُّصَ ) لَهُ ( فِي الْأَصَحِّ ) مِنْ حِينِ جَعْلِهِ كَمَا لَوْ أَنْشَأَهُ بِهَذِهِ النِّيَّةِ .
