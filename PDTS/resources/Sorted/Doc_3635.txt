( وَلَا يُكْرَهُ ) غُسْلٌ فِي الْمَسْجِدِ ( وَلَا وُضُوءٌ ) فِيهِ ( مَا لَمْ يُؤْذِ ) الْمَسْجِدَ أَوْ مَنْ بِهِ ( بِهِمَا ) أَيْ بِمَاءِ الْغُسْلِ وَالْوُضُوءِ ( وَتُكْرَهُ إرَاقَةُ مَاءَيْهِمَا بِهِ ) أَيْ الْمَسْجِدِ ( وَبِمَا يُدَاسُ ) تَنْزِيهًا لِلْمَاءِ ( وَمُصَلَّى الْعِيدِ ، لَا ) مُصَلَّى ( الْجَنَائِزِ مَسْجِدٌ ) لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { وَلْيَعْتَزِلْنَ الْحُيَّضُ الْمُصَلَّى } .
