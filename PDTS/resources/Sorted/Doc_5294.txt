( وَ ) لَا يُجْزِئُ صِنْفٌ مِنْ الْخَمْسَةِ ( مُخْتَلِطٌ بِكَثِيرٍ مِمَّا لَا يُجْزِئُ ) كَقَمْحٍ اخْتَلَطَ بِكَثِيرِ زُوَانٍ أَوْ عَدَسٍ أَوْ نَحْوِهِ ؛ لِأَنَّهُ لَا يَعْلَمُ قَدْرَ مُجْزِئٍ مِنْهُ ( وَيُزَادُ ) عَلَى صَاعٍ ( إنْ قَلَّ ) خَلِيطٌ لَا يُجْزِئُ ( بِقَدْرِهِ ) أَيْ : الْخَلِيطِ بِحَيْثُ يَكُونُ الْمُصَفَّى صَاعًا ؛ لِأَنَّهُ لَيْسَ عَيْبًا لِقِلَّةِ مَشَقَّةِ تَنْقِيَتِهِ ، وَلَا يُجْزِئُ إخْرَاجُ قِيمَةِ الصَّاعِ نَصًّا .( 3 / 184 )
