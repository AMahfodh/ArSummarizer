الرِّضَا تَرْكُ الِاعْتِرَاضِ عَلَى الْفَاعِلِ وَإِنْ لَمْ يَكُنْ مَعَهُ مَحَبَّةٌ ط ( قَوْلُهُ لِأَنَّ الْبَاءَ لِلْإِلْصَاقِ ) أَيْ هُوَ الْمَعْنَى الْحَقِيقِيُّ لَهَا فَيَلْتَصِقُ وُقُوعُ الطَّلَاقِ بِأَحَدِ هَذِهِ الْأَرْبَعَةِ وَهِيَ غَيْبٌ لَا يُطَّلَعُ عَلَيْهَا فَلَا تَطْلُقُ بِالشَّكِّ ط ( قَوْلُهُ وَإِنْ أَضَافَهُ ) أَيْ بِالْبَاءِ ( قَوْلُهُ أَيْ الْمَذْكُورَ ) جَوَابٌ عَنْ الْمُصَنِّفِ حَيْثُ أَفْرَدَ الضَّمِيرَ وَمَرْجِعُهُ مُتَعَدِّدٌ ط ( قَوْلُهُ فَيَقْتَصِرُ عَلَى الْمَجْلِسِ ) أَيْ مَجْلِسِ عِلْمِهِ ، فَإِنْ شَاءَ فِيهِ طَلُقَتْ وَإِلَّا خَرَجَ الْأَمْرُ مِنْ يَدِهِ ( قَوْلُهُ كَمَا مَرَّ ) أَيْ فِي فَصْلِ الْمَشِيئَةِ ح( 11 / 385 )
