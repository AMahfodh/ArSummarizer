ثُمَّ أَخَذَ يَتَكَلَّمُ عَلَى الْمُعَيَّنَةِ فَقَالَ : ( وَشُرِطَ فِي ) إجَارَةِ عَيْنٍ ( مُعَيَّنَةٍ خَمْسَةُ ) شُرُوطٍ : أَحَدُهَا : ( صِحَّةُ بَيْعٍ ) ؛ أَيْ : كَوْنُهَا يَصِحُّ بَيْعُهَا ؛ كَالْأَرْضِ وَالدَّارِ وَالْعَبْدِ وَالْبَهِيمَةِ وَالثَّوْبِ وَالْخَيْمَةِ وَالْخَيْلِ وَالْجَمَلِ وَالسَّيْفِ وَالرُّمْحِ وَالْفَرَسِ وَاللِّجَامِ وَالسَّرِيرِ وَالْإِنَاءِ وَأَشْبَاهِ ذَلِكَ ؛ فَلَا تَصِحُّ إجَارَةُ كَلْبٍ وَخِنْزِيرٍ لِحِرَاسَةٍ وَلَا لِصَيْدٍ وَلَا لِغَيْرِ ذَلِكَ ، ( سِوَى وَقْفٍ ) - أَيْ مَوْقُوفٍ - ( وَ ) سِوَى ( أُمِّ وَلَدٍ وَحُرٍّ وَحُرَّةٍ ) ؛ فَتَصِحُّ إجَارَتُهَا ؛ لِأَنَّ مَنَافِعَهَا مَمْلُوكَةٌ ، وَمَنَافِعُ الْحُرِّ تُضْمَنُ بِالْغَصْبِ ؛ فَجَازَتْ إجَارَتُهَا كَمَنَافِعِ الْقِنِّ .
