( وَلَا تَعْيِينَ ) لِهَدْيٍ وَلَا أُضْحِيَّةٍ ( بِنِيَّةِ ) ذَلِكَ حَالَ ( شِرَاءٍ ) لِأَنَّ التَّعْيِينَ إزَالَةُ مِلْكٍ عَلَى وَجْهِ الْقُرْبَةِ ، فَلَمْ يُؤَثِّرْ فِيهِ مُجَرَّدُ النِّيَّةِ كَالْعِتْقِ وَالْوَقْفِ ( أَوْ ) أَيْ : وَلَا تَعْيِينَ ( بِسَوْقٍ ) مَعَ نِيَّةِ هَدْيٍ أَوْ أُضْحِيَّةٍ مِنْ غَيْرِ تَقْلِيدٍ أَوْ إشْعَارٍ ، لِأَنَّهُ لَا يَخْتَصُّ بِالْهَدْيِ ( كَإِخْرَاجِهِ مَالًا لِصَدَقَةٍ بِهِ ) فَلَا يَلْزَمُهُ التَّصَدُّقُ بِهِ لِلْخَبَرِ ( وَمَا تَعَيَّنَ ) مِنْ هَدْيٍ أَوْ أُضْحِيَّةٍ ( جَازَ نَفْلٌ فِيهِ وَشِرَاءُ خَيْرٍ مِنْهُ ) لِحُصُولِ الْمَقْصُودِ بِهِ مِنْ نَفْعِ الْفُقَرَاءِ بِالزِّيَادَةِ ، وَلِأَنَّهُ يَجُوزُ إبْدَالُهَا بِخَيْرٍ مِنْهَا ، وَالْإِبْدَالُ نَوْعٌ مِنْ الْبَيْعِ .
