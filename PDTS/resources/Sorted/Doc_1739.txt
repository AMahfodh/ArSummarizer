( وَ ) لِلسَّيِّدِ أَنْ ( يَقِفَ ) رَقِيقًا عَلَّقَ عِتْقَهُ بِصِفَةٍ قَبْلهَا ( وَ ) لَهُ أَنْ ( يَنْقُلَ مِلْكَ مِنْ عَلَّقَ عِتْقَهُ ) بِصِفَةٍ ( قَبْلَهَا ) ثُمَّ إنْ وُجِدَتْ وَهُوَ فِي مِلْكِ غَيْرِ الْمُعَلَّقِ لَمْ يَعْتِقْ ؛ لِحَدِيثِ : { لَا طَلَاقَ وَلَا عَتَاقَ وَلَا بَيْعَ فِيمَا لَا يَمْلِكُ ابْنُ آدَمَ } .
