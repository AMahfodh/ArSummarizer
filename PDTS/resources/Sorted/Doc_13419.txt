الشَّرْطُ ( الثَّانِي - انْتِفَاءُ الشُّبْهَةِ ) لِحَدِيثِ : { ادْرَءُوا الْحُدُودَ بِالشُّبُهَاتِ مَا اسْتَطَعْتُمْ } ( فَلَا يُحَدُّ مَنْ وَطِئَ زَوْجَتَهُ ) أَوْ سُرِّيَّتَهُ ( فِي حَيْضٍ أَوْ نِفَاسٍ أَوْ دُبُرٍ ) لِأَنَّهُ وَطْءٌ صَادَفَ مِلْكًا ( أَوْ ) وَطِئَ ( أَمَتَهُ الْمُحَرَّمَةَ أَبَدًا بِرَضَاعٍ أَوْ غَيْرِهِ ) كَمَوْطُوءَةِ أَبِيهِ أَوْ ابْنِهِ أَوْ أُمِّ زَوْجَتِهِ ، أَوْ وَطِئَ أَمَتَهُ ( الْمُزَوَّجَةَ ) أَوْ ( أَمَتَهُ ) الْمُعْتَدَّةَ أَوْ ( أَمَتَهُ ) ( الْمُرْتَدَّةَ أَوْ ) أَمَتَهُ ( الْمَجُوسِيَّةَ ) ( أَوْ ) وَطِئَ أَمَةً ( لَهُ فِيهَا ) شِرْكٌ ( أَوْ ) لِوَلَدِهِ ( فِيهَا ) شِرْكٌ ( أَوْ لِمُكَاتَبِهِ ) فِيهَا شِرْكٌ ( أَوْ لِبَيْتِ الْمَالِ فِيهَا شِرْكٌ ، وَهُوَ ) ؛ أَيْ : الْوَاطِئُ ( مُسْلِمٌ حُرٌّ ) فَلَا حَدَّ لِشُبْهَةِ مِلْكِ الْوَاطِئِ أَوْ لِوَلَدِهِ ؛ لِتَمَكُّنِ الشُّبْهَةِ فِي مِلْكِ وَلَدِهِ ؛ لِحَدِيثِ : { أَنْتَ وَمَالُكَ لِأَبِيكَ } وَلِشُبْهَةِ مِلْكِ مُكَاتَبٍ الْوَاطِئَ ، وَكَذَا إنْ كَانَ لِبَيْتِ الْمَالِ فِيهَا شِرْكٌ ؛ لِأَنَّ لِكُلِّ مُسْلِمٍ فِيهِ حَقًّا .
