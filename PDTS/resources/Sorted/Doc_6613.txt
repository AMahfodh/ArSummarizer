( وَلَا ) بَيْعُ ( خَمْرٍ وَلَوْ كَانَا ) أَيْ : الْمُتَبَايِعَانِ ( ذِمِّيَّيْنِ ) لِحَدِيثِ جَابِرٍ سَمِعْتُ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يَقُولُ { إنَّ اللَّهَ وَرَسُولَهُ حَرَّمَ بَيْعَ الْخَمْرِ وَالْمَيْتَةِ وَالْخِنْزِيرِ وَالْأَصْنَامِ } مُتَّفَقٌ عَلَيْهِ ( لَا ) بَيْعُ وَلَوْ مُبَاحَ الِاقْتِنَاءِ ( كَلْبِ صَيْدٍ لِحَدِيثِ أَبِي سَعِيدٍ الْأَنْصَارِيِّ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ نَهَى عَنْ ثَمَنِ الْكَلْبِ } ) مُتَّفَقٌ عَلَيْهِ ( وَمَنْ قَتَلَهُ ) أَيْ : الْكَلْبَ ( وَهُوَ مُعَلَّمُ ) الصَّيْدِ .
