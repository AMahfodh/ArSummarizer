( وَ ) أَيُّهُمَا أَدَّى ( لَمْ يَرْجِعْ ) عَلَى الْآخَرِ ؛ لِأَنَّهُ مُتَبَرِّعٌ فِي حَقِّ الْآخَرِ ( وَقَبُولُ الْغَائِبِ لَغْوٌ ) فَلَا يُؤْخَذُ بِشَيْءٍ لِنَفَاذِ الْعَقْدِ عَلَى الْحَاضِرِ ( فَإِنْ حَرَّرَهُ ) أَيْ أَعْتَقَ الْمَوْلَى الْغَائِبَ ( سَقَطَ عَنْ الْحَاضِرِ حِصَّتُهُ ) مِنْ الْبَدَلِ ؛ لِأَنَّ الْغَائِبَ دَخَلَ فِي الْعَقْدِ مَقْصُودًا فَكَانَ الْبَدَلُ مُنْقَسِمًا عَلَيْهِمَا وَإِنْ لَمْ يَكُنْ مُطَالَبًا بِهِ بِخِلَافِ الْوَلَدِ الْمَوْلُودِ فِي الْكِتَابَةِ حَيْثُ لَا يَسْقُطُ عَنْ الْأَمَةِ شَيْءٌ مِنْ الْبَدَلِ بِعِتْقِهِ ؛ لِأَنَّهُ لَمْ يَدْخُلْ مَقْصُودًا وَلَمْ يَكُنْ يَوْمَ الْعَقْدِ مَوْجُودًا وَإِنَّمَا دَخَلَ فِي الْكِتَابَةِ تَبَعًا وَكَذَا وَلَدُهَا الْمُشْتَرَى ( وَإِنْ حَرَّرَ ) الْمَوْلَى الْمُكَاتَبَ ( الْحَاضِرَ أَوْ مَاتَ ) الْحَاضِرُ ( سَقَطَ ) حِصَّةُ الْحَاضِرِ ( وَأَدَّى الْغَائِبُ حِصَّتَهُ حَالًّا ،( 5 / 123 )
