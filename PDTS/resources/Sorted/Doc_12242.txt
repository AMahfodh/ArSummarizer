{ الْأُولَى } أَخْرَجَهُ الشَّيْخَانِ مِنْ رِوَايَةِ يُونُسَ بْنِ يَزِيدَ عَنْ الزُّهْرِيِّ عَنْ سَعِيدِ بْنِ الْمُسَيِّبِ عَنْ أَبِي هُرَيْرَةَ لَفْظُ الْبُخَارِيِّ { لَا يَزَالُ قَلْبُ الْكَبِيرِ شَابًّا فِي اثْنَتَيْنِ فِي حُبِّ الدُّنْيَا وَطُولِ الْأَمَلِ } وَلَفْظُ مُسْلِمٍ { قَلْبُ الشَّيْخِ شَابٌّ عَلَى حُبِّ اثْنَتَيْنِ طُولُ الْحَيَاةِ وَحُبُّ الْمَالِ } وَأَخْرَجَهُ مُسْلِمٌ أَيْضًا مِنْ رِوَايَةِ سُفْيَانَ بْنِ عُيَيْنَةَ عَنْ أَبِي الزِّنَادِ عَنْ الْأَعْرَجِ عَنْ أَبِي هُرَيْرَةَ بِلَفْظِ { قَلْبُ الشَّيْخِ شَابٌّ عَلَى حُبِّ اثْنَتَيْنِ حُبُّ الْعَيْشِ وَالْمَالِ } ، وَاتَّفَقَ عَلَيْهِ الشَّيْخَانِ مِنْ رِوَايَةِ هِشَامٍ الدَّسْتُوَائِيِّ عَنْ قَتَادَةَ عَنْ أَنَسٍ بِلَفْظِ { يَكْبُرُ ابْنُ آدَمَ وَيَكْبُرُ مَعَهُ اثْنَتَانِ حُبُّ الْمَالِ وَطُولُ الْعُمْرِ } لَفْظُ الْبُخَارِيِّ ، وَلَمْ يَسُقْ مُسْلِمٌ لَفْظَهُ وَأَخْرَجَهُ مُسْلِمٌ مِنْ رِوَايَةِ أَبِي عَوَانَةَ عَنْ قَتَادَةَ عَنْ أَنَسٍ بِلَفْظِ { يَهْرَمُ ابْنُ آدَمَ وَتَشِبُّ مِنْهُ اثْنَتَانِ الْحِرْصُ عَلَى الْمَالِ وَالْحِرْصُ عَلَى الْعُمْرِ } .
