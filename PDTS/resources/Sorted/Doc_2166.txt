( وَإِنْ قَالَ ) لِامْرَأَتِهِ ( أَنْتِ طَالِقُ لَا شَيْءَ ) طَلُقَتْ ( أَوْ ) أَنْتِ طَالِقُ ( طَلْقَةً لَا تَقَعُ عَلَيْكِ أَوْ لَا يَنْقُصُ بِهَا عَدَدُ الطَّلَاقِ طَلُقَتْ ) لِأَنَّ ذَلِكَ رَفْعٌ لِجَمِيعِ مَا أَوْقَعَهُ ، فَلَمْ يَصِحَّ كَاسْتِثْنَاءِ الْجَمِيعِ وَإِنْ كَانَ ذَلِكَ خَبَرًا فَهُوَ كَذِبٌ لِأَنَّ الشَّيْءَ إذَا أَوْقَعَهُ وَقَعَ .( 18 / 78 )
