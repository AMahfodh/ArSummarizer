وَنُقِلَ فِي تَصْحِيحِ الْفُرُوعِ عَنْ أَكْثَرَ الْأَصْحَابِ خِلَافُهُ لِاسْتِقْلَالِهِ بِمَنَافِعِهِ وَكَسْبِهِ وَ ( لَا ) يُقِيمُهُ سَيِّدٌ عَلَى أَمَةٍ ( مُزَوَّجَةٍ ) لِقَوْلِ ابْنِ عُمَرَ إذَا كَانَتْ الْأَمَةُ مُزَوَّجَةً رُفِعَتْ إلَى السُّلْطَانِ ، فَإِنْ لَمْ يَكُنْ لَهَا زَوْجٌ جَلَدَهَا سَيِّدُهَا نِصْفَ مَا عَلَى الْمُحْصَنِ وَلَا يُعْرَفُ لَهُ مُخَالِفٌ مِنْ الصَّحَابَةِ ، وَلِأَنَّ مَنْفَعَتَهَا مَمْلُوكَةٌ لِغَيْرِهِ مِلْكًا غَيْرَ مُقَيَّدٍ بِوَقْتٍ أَشْبَهَتْ الْمُشْتَرَكَةَ ( وَمَا ثَبَتَ ) مِمَّا يُوجِبُ الْحَدَّ عَلَى رَقِيقٍ ( يَعْلَمُهُ ) أَيْ : السَّيِّدُ بِرُؤْيَةٍ أَوْ غَيْرِهَا ( أَوْ إقْرَارِ ) رَقِيقٍ ( كَ ) الثَّالِثِ ( بِبَيِّنَةٍ ) لِأَنَّهُ يَجْرِي مَجْرَى التَّأْدِيبِ بِخِلَافِ الْحَاكِمِ فَإِنَّهُ مُتَّهَمٌ ، وَلِلسَّيِّدِ سَمَاعُ الْبَيِّنَةِ عَلَى رَقِيقِهِ إذَا عَلِمَ شُرُوطَهَا ( وَلَيْسَ لَهُ ) أَيْ السَّيِّدِ ( قَتْلٌ فِي رِدَّةٍ .
