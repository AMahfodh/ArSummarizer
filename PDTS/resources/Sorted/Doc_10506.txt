قَطَعَ بِهِ أَكْثَرُ الْأَصْحَابِ ، وَإِنْ شَرَطَهُ الْوَكِيلُ ( لِنَفْسِهِ دُونَ مُوَكِّلِهِ ) ؛ لَمْ يَصِحَّ الشَّرْطُ ؛ كَمَا لَوْ شَرَطَهُ أَحَدُ الْمُتَعَاقِدَيْنِ لِأَجْنَبِيٍّ دُونَهُ ، ( أَوْ ) شَرَطَ الْوَكِيلُ ( لِأَجْنَبِيٍّ ؛ لَمْ يَصِحَّ ) الشَّرْطُ ، وَظَاهِرُهُ ، وَلَوْ لَمْ يَقُلْ دُونِي ؛ لِأَنَّ الْوَكِيلَ لَيْسَ لَهُ أَنْ يُوَكِّلَ فِي مِثْلِ ذَلِكَ ، ( وَ ) يَصِحُّ شَرْطُ الْخِيَارِ لِمُتَعَاقِدَيْنِ - وَلَوْ كَانَا وَكِيلَيْنِ - لِأَنَّ النَّظَرَ فِي تَحْصِيلِ الْحَظِّ مُفَوَّضٌ إلَى الْوَكِيلِ ( وَإِنْ لَمْ يُؤْمَرَا ) ؛ أَيْ : يَأْمُرْ الْمُوَكِّلَانِ الْوَكِيلَيْنِ ( بِهِ ) ؛ أَيْ : شَرْطِ الْخِيَارِ ؛ لِمَا مَرَّ : أَنَّ شَرْطَ الْحَظِّ مُفَوَّضٌ إلَى الْوَكِيلِ .( 7 / 469 )
