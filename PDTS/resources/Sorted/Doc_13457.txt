( قَوْلُهُ : وَثَانِيهَا الِاخْتِيَارُ ) مُفَادُهُ أَنَّ هَذَا لَمْ يَذْكُرْهُ الْمُصَنِّفُ ؛ لِأَنَّهُ لَمْ يَكُنْ مُتَعَمَّدًا وَفِي بَعْضِ الشُّرُوحِ أَنَّ قَيْدَ التَّعَمُّدِ مُحْرَزٌ لِلِاخْتِيَارِ فَقَالَ : فَلَا كَفَّارَةَ عَلَى نَاسٍ وَلَا عَلَى غَيْرِ مُخْتَارٍ كَمَنْ فَعَلَ شَيْئًا مِنْ مُوجِبَاتِهَا مُكْرَهًا ، أَوْ غَلَبَةً لِعَدَمِ وَصْفِ هَذِهِ الثَّلَاثَةِ بِالتَّعَمُّدِ حَقِيقَةً إلَّا مَنْ اسْتَاكَ بِجَوْزَاءَ نَهَارًا عَمْدًا وَابْتَلَعَهَا غَلَبَةً عَلَيْهِ الْكَفَّارَةُ ( قَوْلُهُ : الِانْتِهَاكُ لِلْحُرْمَةِ ) أَيْ : عَدَمُ الْمُبَالَاةِ بِهَا ( قَوْلُهُ : كَحَدِيثِ عَهْدٍ بِالْإِسْلَامِ ) أَيْ : قَرِيبِ اتِّصَافٍ بِالْإِسْلَامِ ، أَوْ قَرِيبِ عِلْمٍ بِالْإِسْلَامِ مِنْ حَيْثُ الِاتِّصَافُ بِهِ ( قَوْلُهُ : لَا يَدْخُلُ بَابُ إلَخْ ) أَيْ : لَا يَصِحُّ دُخُولُهُ وَقَوْلُهُ : أَوْ يَدْخُلُهُ أَيْ : يَصِحُّ دُخُولُهُ ( قَوْلُهُ : وَلَكِنْ لِرَمَضَانَ حُرْمَةٌ ) أَيْ : فَلَوْ قِسْنَا لَكَانَ قِيَاسًا مَعَ الْفَارِقِ ( قَوْلُهُ : هَذَا مَعْمُولُ تَعَمَّدَ ) فِيهِ إشَارَةٌ إلَى أَنَّ ، أَوْ رَفَعَ مَعْطُوفٌ عَلَى جِمَاعًا فَهُوَ يُقْرَأُ اسْمًا وَلَا يَرُدُّ عَلَيْهِ أَنَّ الرَّافِعَ مُتَعَمِّدٌ لِأَنَّ الرَّفْعَ قَدْ يَكُونُ سَهْوًا ا ه .
