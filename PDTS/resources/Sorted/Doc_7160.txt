سم وَمَيْلُ الْقَلْبِ إلَى عَدَمِ الْعَوْدِ ؛ لِأَنَّ الْأَرْضَ هِيَ الْأَصْلُ الْمَقْصُودُ فِي الْمَسْجِدِيَّةِ ( قَوْلُهُ : ؛ لِأَنَّهَا تَلْزَمُهُ ) أَيْ الْأُجْرَةَ تَلْزَمُ الْوَاقِفُ ( قَوْلُهُ : وَلِلْمُسْتَحِقِّ ) أَيْ مُسْتَحَقِّ الْأُجْرَةِ وَهُوَ مَالِكُ الْأَرْضِ ( قَوْلُهُ : مُطَالَبَتُهُ ) أَيْ الْوَاقِفِ ( قَوْلُهُ : بِالتَّفْرِيغِ ) أَيْ تَفْرِيغِ الْأَرْضِ عَمَّا فِيهَا مِنْ الْبِنَاءِ وَالْغِرَاسِ ( قَوْلُهُ : وَفَارَقَ ) أَيْ نَحْوَ الْبِنَاءِ أَيْ ضَرَرِهِ فِي الْأَرْضِ ( قَوْلُهُ : جِنَايَةَ الْقِنِّ إلَخْ ) أَيْ حَيْثُ يَلْزَمُهُ أَيْ الْوَاقِفِ أَرْشُهَا .
