فَصْلٌ : ، وَيَحْرُمُ فِرَارُ مُسْلِمٍ مِنْ كَافِرَيْنِ وَيَحْرُمُ فِرَارُ ( جَمَاعَةٍ مِنْ مِثْلَيْهِمْ ) لِقَوْلِهِ تَعَالَى { فَإِنْ يَكُنْ مِنْكُمْ مِائَةٌ صَابِرَةٌ يَغْلِبُوا مِائَتَيْنِ } قَالَ ابْنُ عَبَّاسٍ مَنْ فَرَّ مِنْ اثْنَيْنِ فَقَدْ فَرَّ ، وَمَنْ فَرَّ مِنْ ثَلَاثَةٍ فَمَا فَرَّ ( وَيَلْزَمُهُمْ ) أَيْ : الْمُسْلِمِينَ ( الثَّبَاتُ وَإِنْ ظَنُّوا التَّلَفَ ) لِقَوْلِهِ تَعَالَى { إذَا لَقِيتُمْ الَّذِينَ كَفَرُوا زَحْفًا فَلَا تُوَلُّوهُمْ الْأَدْبَارَ } ؛ وَلِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ عَدَّ الْفِرَارَ مِنْ الْكَبَائِرِ ( إلَّا مُتَحَرِّفِينَ لِقِتَالٍ ) لِقَوْلِهِ تَعَالَى { وَمَنْ يُوَلِّهِمْ يَوْمَئِذٍ دُبُرَهُ إلَّا مُتَحَرِّفًا لِقِتَالٍ أَوْ مُتَحَيِّزًا إلَى فِئَةٍ فَقَدْ بَاءَ بِغَضَبٍ مِنْ اللَّهِ } .
