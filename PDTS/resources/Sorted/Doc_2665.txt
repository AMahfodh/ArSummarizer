( قَوْلُهُ : إذْ قَوْلُهُ : إلَخْ ) بَيَانٌ لِوَجْهِ دَلَالَةِ الْآيَةِ عَلَى الْمَتْنِ ( قَوْلُهُ : فَانْدَفَعَ إلَخْ ) أَيْ بِقَوْلِهِ إذْ قَوْلُهُ : إلَخْ ( قَوْلُهُ : يُمْكِنُ حَمْلُهَا ) أَيْ الْآيَةِ ( قَوْلُهُ : يُرِيدَ أَنْ يَسْتَعْفِفَ ) الْجُمْلَةُ حَالٌ مِنْ النَّاكِحِ ( قَوْلُهُ وَحَمَلُوا ) أَيْ الْكَثِيرُونَ وَقَوْلُهُ أَصَحُّ خَبَرٍ قَوْلُهُ( 29 / 176 )
