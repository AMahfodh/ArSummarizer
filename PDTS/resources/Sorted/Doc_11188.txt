( وَإِنْ اسْتَتَمَّ قَائِمًا وَلَمْ يَقْرَأْ ) أَيْ لَمْ يَشْرَعْ فِي الْقِرَاءَةِ ( فَعَدَمُ رُجُوعِهِ أَوْلَى ) مِنْ رُجُوعِهِ لِمَا تَقَدَّمَ مِنْ حَدِيثِ الْمُغِيرَةِ وَإِنَّمَا جَازَ رُجُوعُهُ ؛ لِأَنَّهُ لَمْ يَلْتَبِسْ بِرُكْنٍ مَقْصُودٍ ؛ لِأَنَّ الْقِيَامَ لَيْسَ بِمَقْصُودٍ فِي نَفْسِهِ وَلِهَذَا جَازَ تَرْكُهُ ، عِنْد الْعَجْزِ ، بِخِلَافِ غَيْرِهِ مِنْ الْأَرْكَانِ ( وَيُتَابِعُهُ ) أَيْ الْإِمَامَ إذَا قَامَ سَهْوًا عَنْ التَّشَهُّدِ ( الْمَأْمُومُ ) وَيَسْقُطُ عَنْهُ التَّشَهُّدُ فِي الْجُلُوسِ إذَنْ كَمَا تَقَدَّمَ ( وَلَوْ عَلِمَ ) الْمَأْمُومُ ( تَرْكَهُ ) أَيْ تَرْكَ الْإِمَامُ التَّشَهُّدَ ( قَبْلَ قِيَامِهِ ) أَيْ الْمَأْمُومِ أَوْ الْإِمَامِ ( وَلَا يَتَشَهَّدُ ) الْمَأْمُومُ بَعْدَ قِيَامِ إمَامِهِ سَهْوًا لِحَدِيثِ { إنَّمَا جُعِلَ الْإِمَامُ لِيُؤْتَمَّ بِهِ ، فَلَا تَخْتَلِفُوا عَلَيْهِ } .( 3 / 202 )
