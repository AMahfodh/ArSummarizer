وَيُفَارِقُ الْعِتْقَ فَإِنَّهُ يَخْرُجُ الْمَعْتُوقُ عَنْ الْمَالِيَّةِ وَامْتِنَاعُ التَّصَرُّفِ فِي الرَّقَبَةِ لَا يَمْنَعُ الْمِلْكَ كَأُمِّ الْوَلَدِ ( فَيَنْظُرُ فِيهِ ) أَيْ الْوَقْفِ ( هُوَ ) أَيْ الْمَوْقُوفُ عَلَيْهِ إنْ كَانَ مُكَلَّفًا رَشِيدًا ( أَوْ وَلِيُّهُ ) إنْ كَانَ مَحْجُورًا عَلَيْهِ كَالطَّلْقِ ( وَيَتَمَلَّكُ ) مَوْقُوفٌ عَلَيْهِ مُعَيَّنٌ أَرْضًا غُصِبَتْ وَزُرِعَتْ ( زَرْعَ غَاصِبٍ ) بِنَفَقَتِهِ ، وَهِيَ مِثْلُ بَذْرِهِ وَعِوَضُ لَوَاحِقِهِ كَمَالِكِ الْأَرْضِ الطَّلْقِ( 7 / 150 )
