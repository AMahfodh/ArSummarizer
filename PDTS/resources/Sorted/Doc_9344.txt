( وَلَوْ ) تَصَرَّفَ تَصَرُّفًا مَالِيًّا مُفَوِّتًا فِي الْحَيَاةِ بِالْإِنْشَاءِ مُبْتَدَأً كَأَنْ ( بَاعَ ) أَوْ اشْتَرَى بِالْعَيْنِ ( أَوْ وَهَبَ أَوْ أَعْتَقَ ) أَوْ أَجَّرَ أَوْ وَقَفَ أَوْ كَاتَبَ ( فَفِي قَوْلٍ يُوقِفُ تَصَرُّفَهُ ) الْمَذْكُورَ ( فَإِنْ فَضَلَ ذَلِكَ عَنْ الدَّيْنِ ) لِارْتِفَاعِ الْقِيمَةِ أَوْ إبْرَاءِ الْغُرَمَاءِ أَوْ بَعْضِهِمْ ( نَفَذَ ) أَيْ بَانَ أَنَّهُ كَانَ نَافِذًا ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَفْضُلْ ( لَغَا ) أَيْ بَانَ أَنَّهُ كَانَ لَاغِيًا ( وَالْأَظْهَرُ بُطْلَانُهُ ) فِي الْحَالِّ لِتَعَلُّقِ حَقِّهِمْ بِهِ كَالْمَرْهُونِ ، وَلِأَنَّهُ مَحْجُورٌ عَلَيْهِ بِحُكْمِ الْحَاكِمِ فَلَا يَصِحُّ تَصَرُّفُهُ عَلَى مُرَاغَمَةِ مَقْصُودِ الْحَجْرِ كَالسَّفِيهِ .
