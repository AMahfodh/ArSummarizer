( فَصْلٌ : وَإِنْ دَفَعَهَا ) - أَيْ : الْوَدِيعَةَ - ( إلَى مَنْ يَحْفَظُ مَالَهُ ) - أَيْ : الْمُسْتَوْدَعِ - ( عَادَةً ) أَوْ دَفَعَهَا إلَى مَنْ يَحْفَظُ مَالَ رَبِّهَا ( كَزَوْجَتِهِ وَعَبْدِهِ وَخَادِمِهِ ) وَنَحْوهمْ كَخَازِنِهِ ، وَتَلِفَتْ ؛ لَمْ يَضْمَنْ ؛ لِأَنَّهُ قَدْ وَجَبَ عَلَيْهِ حِفْظُهَا فَلَهُ تَوَلِّيهِ بِنَفْسِهِ وَبِمَنْ يَقُومُ مَقَامَهُ ؛ كَمَا لَوْ دَفَعَ الْمَاشِيَةَ إلَى الرَّاعِي ، وَالْبَهِيمَةَ إلَى غُلَامِهِ لِيَسْقِيَهَا ، وَلِقِيَامِهِمْ مَقَامَ الْمَالِكِ فِي الرَّدِّ ، أَوْ دَفَعَهَا ( لِعُذْرٍ ) كَمَنْ حَضَرَهُ الْمَوْتُ أَوْ أَرَادَ سَفَرًا وَلَيْسَ السَّفَرُ أَحْفَظَ لَهَا ( إلَى أَجْنَبِيٍّ ثِقَةٍ ) ؛ لَمْ يَضْمَنْ .
