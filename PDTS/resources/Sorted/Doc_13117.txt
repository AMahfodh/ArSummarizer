( الرُّكْنُ الثَّانِي الْعَمَلُ وَيُشْتَرَطُ كَوْنُهُ تِجَارَةً ) وَتَوَابِعَ لَهَا كَنَشْرِ الثِّيَابِ وَطَيِّهَا كَمَا سَيَأْتِي ( غَيْرَ مُضَيَّقَةٍ ) بِالتَّعْيِينِ ( وَلَا مُؤَقَّتَةٍ ) بِوَقْتٍ ( فَإِنْ قَارَضَهُ عَلَى أَنْ يَشْتَرِيَ الْحِنْطَةَ وَيَطْحَنَهَا ، أَوْ ) أَنْ ( يَشْتَرِيَ النَّخْلَ لِثَمَرَتِهِ أَوْ شَبَكَةً لِيَصْطَادَ بِهَا ) وَالْفَوَائِدُ ( وَالصَّيْدُ بَيْنَهُمَا لَمْ يَصِحَّ ) لِلِاسْتِغْنَاءِ عَنْ جَهَالَةِ الْعِوَضِ بِالِاسْتِئْجَارِ فَإِنَّهَا أَعْمَالٌ مَضْبُوطَةٌ وَلِأَنَّ مَا حَصَلَ مِنْ ثَمَرَةِ النَّخْلِ لَيْسَ بِتَصَرُّفِ الْعَامِلِ وَإِنَّمَا هُوَ مِنْ عَيْنِ الْمَالِ وَلَوْ عَبَّرَ بَدَلَ الصَّيْدُ بِالْفَوَائِدِ كَانَ أَوْلَى وَالصَّيْدُ لِلصَّائِدِ وَعَلَيْهِ أُجْرَةُ الشَّبَكَةِ كَمَا صَرَّحَ بِهِ الْأَصْلُ ( فَلَوْ طَحَنَ الْحِنْطَةَ بِلَا شَرْطٍ لَمْ يَنْفَسِخْ الْقِرَاضُ ) كَمَا لَوْ زَادَ عَبْدٌ الْقِرَاضَ بِكِبَرٍ أَوْ سِمَنٍ ، أَوْ تَعَلُّمِ صَنْعَةٍ ( لَكِنَّ عَلَيْهِ الضَّمَانَ ) لِتَعَدِّيهِ فَيَغْرَمُ نَقْصَ الدَّقِيقِ إنْ نَقَصَ ( فَإِنْ بَاعَهُ لَمْ يَضْمَنْ ثَمَنَهُ ) لِأَنَّهُ لَمْ يَتَعَدَّ فِيهِ ( وَلَمْ يَسْتَحِقَّ أُجْرَةَ الطَّحْنِ ) وَلَوْ اسْتَأْجَرَ عَلَيْهِ لَزِمَتْهُ الْأُجْرَةُ ( وَالرِّبْحُ بَيْنُهُمَا ) كَمَا شَرَطَا .
