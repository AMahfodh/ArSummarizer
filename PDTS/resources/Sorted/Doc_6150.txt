: مَعَ الْمُدَّعِي ( قَوْلُهُ : بِإِتْلَافِهِ ) أَيْ : الْغَائِبِ ( قَوْلُهُ : قَالَ ) أَيْ : - أَبُو زُرْعَةَ ( قَوْلُهُ : ذَلِكَ ) أَيْ : سَمَاعُ الدَّعْوَى ، وَالْبَيِّنَةِ بِأَنَّ لَهُ تَحْتَ يَدِهِ ، وَدِيعَةً ( قَوْلُهُ : فَيَضْبِطُهَا ) أَيْ : الْوَدِيعَةَ ، وَيَحْتَمِلُ الْبَيِّنَةَ بِإِقَامَتِهَا أَيْ : الْبَيِّنَةِ ( قَوْلُهُ : وَإِشْهَادِهِ ) أَيْ : الْقَاضِي ( قَوْلُهُ : بِثُبُوتِ ذَلِكَ ) أَيْ : الْوَدِيعَةِ ( قَوْلُهُ : بِإِقَامَتِهَا إلَخْ .
