وَيَجِبُ تَعْرِيفُهَا أَبَدًا حَالَ كَوْنِهِ ( ضَامِنًا ) لَهَا إذَا جَاءَ رَبُّهَا ( فِيهِمَا ) أَيْ فِي التَّصَدُّقِ بِوَجْهَيْهِ وَالتَّمَلُّكِ ( كَنِيَّةِ أَخْذِهَا ) أَيْ كَمَا يَضْمَنُ إذَا أَخَذَهَا بِنِيَّةِ التَّمَلُّكِ ( قَبْلَهَا ) أَيْ قَبْلَ الْتِقَاطِهَا وَلَوْ قَالَ كَنِيَّةِ تَمَلُّكِهَا قَبْلَهُ كَانَ أَوْضَحَ يَعْنِي أَنَّ الْمُلْتَقِطَ إذَا رَأَى اللُّقَطَةَ فَنَوَى أَخْذَهَا تَمَلُّكًا ، ثُمَّ أَخَذَهَا فَإِنَّهُ يَضْمَنُهَا لِرَبِّهَا وَلَوْ تَلِفَتْ بِسَمَاوِيٍّ ؛ لِأَنَّهُ بِتِلْكَ النِّيَّةِ مَعَ وَضْعِ يَدِهِ عَلَيْهَا صَارَ كَالْغَاصِبِ فَيَضْمَنُ كَمَا إذَا نَوَى التَّمَلُّكَ قَبْلَ السَّنَةِ بَعْدَ وَضْعِ يَدِهِ عَلَيْهَا ( وَ ) كَمَا يَضْمَنُ فِي ( رَدِّهَا ) لِمَوْضِعِهَا ، أَوْ غَيْرِهِ ( بَعْدَ أَخْذِهَا لِلْحِفْظِ ) أَيْ لِلتَّعْرِيفِ ( إلَّا ) أَنْ يَرُدَّهَا لِمَوْضِعِهَا ( بِقُرْبٍ ) مِنْ أَخْذِهَا فَضَاعَتْ ( فَتَأْوِيلَانِ ) فِي الضَّمَانِ وَعَدَمِهِ فَإِنْ أَخَذَهَا لِغَيْرِ الْحِفْظِ وَرَدَّهَا بِقُرْبٍ فَلَا ضَمَانَ قَطْعًا وَعَنْ بُعْدٍ ضَمِنَ أَخَذَهَا لِلْحِفْظِ أَمْ لَا .
