( وَهِيَ ) أَيْ الْفِطْرَةُ ( عَلَى الْمُعْسِرِ ) وَقْتَ وُجُوبِهَا ( لَيْسَتْ تَسْتَقِرّ ) سَوَاءٌ كَانَتْ لِنَفْسِهِ أَمْ لِغَيْرِهِ مِمَّنْ تَلْزَمُهُ مُؤْنَتُهُ كَمَا فَصَّلَهُ مِنْ زِيَادَتِهِ بِقَوْلِهِ : ( لِلنَّفْسِ ، وَالْعِرْسِ وَكُلِّ مَنْ ذَكَرَ ) بِخِلَافِ نَفَقَةِ الزَّوْجَةِ ؛ لِأَنَّهَا عِوَضٌ ، وَالْفِطْرَةُ عِبَادَةٌ مَشْرُوطَةٌ بِالْيَسَارِ فَلَوْ أَيْسَرَ بَعْدُ وَلَوْ بِلَحْظَةٍ لَا يَلْزَمُهُ الْأَدَاءُ( 7 / 7 )
