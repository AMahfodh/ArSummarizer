( { وَتَرَكَ زِينَةَ الدُّنْيَا وَآثَرَ مَا يَبْقَى } ) ، وَهُوَ الْآخِرَةُ ( { عَلَى مَا يَفْنَى } ) ، وَهُوَ الدُّنْيَا ( { ، وَلَمْ يَعُدَّ } ) أَيْ لَمْ يَحْسِبْ ( { غَدًا مِنْ أَيَّامِهِ } ) كِنَايَةً عَنْ قِصَرِ الْأَمَلِ مِنْ أَيَّامِهِ ( { ، وَعَدَّ نَفْسَهُ مِنْ الْمَوْتَى } ) لِقُوَّةِ خَوْفٍ مِنْ مَوْلَاهُ ، وَقُوَّةِ رَغْبَتِهِ فِي تَشْيِيدِ أُخْرَاهُ .
