( وَنِصَاب عَلْس ) بِفَتْحِ الْعَيْنِ الْمُهْمَلَةِ وَسُكُونِ اللَّامِ وَفَتْحِهَا ( وَهُوَ نَوْعٌ مِنْ الْحِنْطَةِ ، وَ ) نِصَابُ ( أُرْزٍ يُدَّخَرَانِ ) أَيْ الْعَلْسُ وَالْأُرْزُ ( فِي قِشْرَيْهِمَا عَادَةً لِحِفْظِهِمَا ) لِأَنَّهُمَا إذَا خَرَجَا مِنْ قِشْرِهِمَا لَا يَبْقَيَانِ بَقَاءَهُمَا فِي الْقِشْرِ ( عَشَرَةَ أَوْسُقٍ ، إذَا كَانَ ) الْعَلْسُ أَوْ الْأُرْزُ ( بِبَلَدٍ قَدْ خَبِرَهُ ) أَيْ امْتَحَنَهُ وَجَرَّبَهُ ( أَهْلُهُ وَعَرَفُوا أَنَّهُ يَخْرُج مِنْهُ مُصَفَّى النِّصْف ) عَمَلًا بِالْعَادَةِ ( لِأَنَّهُ يَخْتَلِفُ فِي الْخِفَّةِ وَالثِّقَلِ فَيَرْجِعُ إلَى أَهْلِ الْخِبْرَةِ ) بِذَلِكَ .
