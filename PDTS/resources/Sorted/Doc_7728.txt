( وَإِنْ كَانَ ) التَّفْضِيلُ أَوْ التَّخْصِيصُ ( عَلَى أَنَّ بَعْضَهُمْ ) أَيْ : لِأَجْلِ أَنَّ الْمُفَضَّلَ أَوْ الْمُخَصَّصَ ( لَهُ عِيَالٌ أَوْ بِهِ حَاجَةٌ ) كَمَسْكَنَةٍ أَوْ عَمًى ، وَنَحْوُهُ ( أَوْ خَصَّ ) أَوْ فَضَّلَ ( الْمُشْتَغِلِينَ بِالْعِلْمِ أَوْ ذَا الدِّينِ دُونَ الْفُسَّاقِ أَوْ ) خَصَّ أَوْ فَضَّلَ ( الْمَرِيضَ أَوْ ) خَصَّ ( أَوْ فَضَّلَ مَنْ لَهُ فَضِيلَةٌ ) مَا مِنْ الْفَضَائِلِ ( مِنْ أَجْلِ فَضِيلَتِهِ فَلَا بَأْسَ ) بِذَلِكَ نَصَّ عَلَيْهِ ؛ لِأَنَّهُ لِغَرَضٍ مَقْصُودٍ شَرْعًا .( 14 / 461 )
