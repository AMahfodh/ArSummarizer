( لَا ) يُقْطَعُ بِسَرِقَةِ ( مِلْكِهِ ) أَوْ السَّارِقِ ( مِنْ مُرْتَهِنٍ ) لَهُ مُتَوَثِّقٌ بِهِ فِي دَيْنِهِ ( وَلَا ) يُقْطَعُ بِسَرِقَةِ مِلْكِهِ مِنْ ( مُسْتَأْجِرٍ ) بِكَسْرِ الْجِيمِ أَوْ مُسْتَعِيرٍ لَهُ أَوْ مُودِعٍ عِنْدَهُ ، وَشَبَّهَ فِي عَدَمِ الْقَطْعِ فَقَالَ ( كَمِلْكِهِ ) أَيْ السَّارِقِ النِّصَابَ بِإِرْثٍ أَوْ هِبَةٍ أَوْ شِرَاءٍ ( وَقَبْلَ ) خُرُوجِهِ أَيْ النِّصَابِ مِنْ حِرْزِهِ فَلَا يُقْطَعُ ، وَمَفْهُومُ قَبْلَ خُرُوجِهِ أَنَّهُ إنْ مَلَكَهُ بَعْدَ خُرُوجِهِ يُقْطَعُ وَهُوَ كَذَا .
