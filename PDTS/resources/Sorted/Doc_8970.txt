قَالَهُ فِي الْإِقْنَاعِ ( أَوْ ) أَنْتِ طَالِقٌ وَطَالِقٌ وَطَالِقٌ ( إلَّا طَالِقًا أَوْ ) أَنْتِ طَالِقٌ ( ثِنْتَيْنِ وَطَلْقَةً إلَّا طَلْقَةً أَوْ ثِنْتَيْنِ وَنِصْفَ إلَّا طَلْقَةً أَوْ ثِنْتَيْنِ وَثِنْتَيْنِ إلَّا ثِنْتَيْنِ ) يَقَعُ ثَلَاثٌ لِمَا تَقَدَّمَ ( أَوْ ) أَنْتِ طَالِقٌ ثِنْتَيْنِ وَثِنْتَيْنِ ( إلَّا وَاحِدَةً يَقَعُ ثَلَاثُ ) طَلْقَاتٍ لِبَقَائِهَا بَعْدَ الِاسْتِثْنَاءِ ( كَعَطْفِهِ بِالْفَاءِ أَوْ بِثُمَّ ) بِأَنْ قَالَ أَنْتِ طَالِقٌ ثِنْتَيْنِ فَثِنْتَيْنِ إلَّا ثِنْتَيْنِ أَوْ إلَّا وَاحِدَةً أَوْ أَنْتِ طَالِقٌ ثِنْتَيْنِ ثُمَّ ثِنْتَيْنِ إلَّا ثِنْتَيْنِ أَوْ إلَّا وَاحِدَةً ، وَإِنْ قَالَ : أَنْتِ طَالِقٌ وَاحِدَةً وَوَاحِدَةً وَوَاحِدَةً إلَّا وَاحِدَةً وَوَاحِدَةً وَوَاحِدَةً .
