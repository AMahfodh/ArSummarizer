( وَيَحْرُمُ الِاحْتِكَارُ فِي قُوتِ الْآدَمِيِّ فَقَطْ ) لِحَدِيثِ أَبِي أُمَامَةَ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ نَهَى أَنْ يُحْتَكَرَ الطَّعَامُ } رَوَاهُ الْأَثْرَمُ وَعَنْهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { الْجَالِبُ مَرْزُوقٌ وَالْمُحْتَكِرُ مَلْعُونٌ } ( وَهُوَ ) أَيْ : الِاحْتِكَارُ فِي الْقُوتِ ( أَنْ يَشْتَرِيَهُ لِلتِّجَارَةِ وَيَحْبِسَهُ لِيَقِلَّ فَيَغْلُوَ ) وَهُوَ بِالْحَرَمَيْنِ أَشَدُّ تَحْرِيمًا .
