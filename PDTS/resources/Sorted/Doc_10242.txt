وَفِي حَجّ مَا يَقْتَضِي أَنَّهُ مَتَى ضَمِنَ بِلَا إذْنٍ بَعْدَ الْإِذْنِ فِي الْأَدَاءِ لَا رُجُوعَ لَهُ سَوَاءٌ قَصَدَ الْأَدَاءَ عَنْ الضَّمَانِ أَوْ بِسَبَبِ الْإِذْنِ أَوْ لَمْ يَقْصِدْ شَيْئًا ( قَوْلُهُ : وَأَدَّى ) أَيْ فَلَا رُجُوعَ لَهُ ( قَوْلُهُ : عَنْ جِهَةِ الْإِذْنِ ) أَيْ بِأَنْ أَدَّى عَنْ جِهَةِ الضَّمَانِ أَوْ أَطْلَقَ فَلْيُتَأَمَّلْ وَلَوْ اخْتَلَفَا فِي النِّيَّةِ وَعَدَمِهَا صُدِّقَ الدَّافِعُ فَإِنَّ النِّيَّةَ لَا تُعْلَمُ إلَّا مِنْ جِهَتِهِ ( قَوْلُهُ : وَلَمْ يَنْهَهُ عَنْهُ ) أَيْ الْأَدَاءِ ( قَوْلُهُ : أَمَّا لَوْ نَهَاهُ ) أَيْ عَنْ الْأَدَاءِ ( قَوْلُهُ : فَلَا تَأْثِيرَ لَهُ ) أَيْ النَّهْيِ فَيَرْجِعُ بِمَا أَدَّى ( قَوْلُهُ : وَانْفَصَلَ عَنْ( 15 / 206 )
