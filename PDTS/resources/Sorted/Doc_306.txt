( الْمَادَّةُ 409 ) الْآجِرُ هُوَ الَّذِي أُعْطِيَ الْمَأْجُورَ بِالْإِجَارَةِ وَيُقَالُ لَهُ أَيْضًا الْمُكَارِي بِضَمِّ الْمِيمِ وَمُؤَجِّرٌ بِكَسْرِ الْجِيمِ أَمَّا قَوْلُ الْبَعْضِ مُؤَجَّرًا فَخَطَأٌ وَقَبِيحٌ ( زَيْلَعِيٌّ ) أَمَّا خَطَؤُهُ ؛ فَلِأَنَّ ( آجَرَ ) مِنْ بَابِ أَفْعَلَ لَا فَاعَلَ وَأَمَّا قُبْحُهُ فَلِأَنَّهُ يُسْتَعْمَلُ فِي مَوْضِعِ الْقُبْحِ ( شِبْلِيٌّ ) .
