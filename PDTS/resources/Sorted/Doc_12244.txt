( وَمَا يَحْتَاجُ إلَيْهِ ) مِنْ نَفَقَةٍ وَكِسْوَةٍ وَسُكْنَى وَدَوَاءٍ وَمَهْرٍ إذَا زَوَّجَهُ السُّلْطَانُ ( فِي بَيْتِ الْمَالِ ) إنْ بَرْهَنَ عَلَى الْتِقَاطِهِ ( وَإِنْ كَانَ لَهُ مَالٌ ) أَوْ قَرَابَةٌ ( فَفِي مَالِهِ ) أَوْ عَلَى قَرَابَتِهِ ( وَارِثُهُ ) وَلَوْ دِيَةً ( فِي بَيْتِ الْمَالِ كَجِنَايَتِهِ ) لِأَنَّ الْغُرْمَ بِالْغُنْمِ ( وَلَيْسَ لِأَحَدٍ أَخْذُهُ مِنْهُ قَهْرًا ) وَهَلْ لِلْإِمَامِ الْأَعْظَمِ أَخْذُهُ بِالْوِلَايَةِ الْعَامَّةِ فِي الْفَتْحِ لَا ، وَأَقَرَّهُ الْمُصَنِّفُ تَبَعًا لِلْبَحْرِ وَحُرِّرَ فِي النَّهْرِ ، نَعَمْ لَكِنْ لَا يَنْبَغِي أَخْذُهُ إلَّا بِمُوجِبٍ ( فَلَوْ أَخَذَهُ أَحَدٌ وَخَاصَمَهُ الْأَوَّلُ رُدَّ إلَيْهِ ) إلَّا إذَا دَفَعَهُ بِاخْتِيَارِهِ لِأَنَّهُ أَبْطَلَ حَقَّهُ ( وَ ) هَذَا إذَا اتَّحَدَ الْمُلْتَقِطُ ، فَلَوْ تَعَدَّدَ وَتَرَجَّحَ أَحَدُهُمَا كَمَا ( لَوْ وَجَدَهُ مُسْلِمٌ وَكَافِرٌ فَتَنَازَعَا قُضِيَ بِهِ لِلْمُسْلِمِ ) لِأَنَّهُ أَنْفَعُ لِلَّقِيطِ خَانِيَّةٌ ، وَلَوْ اسْتَوَيَا فَالرَّأْيُ لِلْقَاضِي بَحْرٌ بَحْثًا .
