( وَإِنْ جَنَى ) مُدَبَّرٌ ( بِيعَ ) أَيْ : جَازَ بَيْعُهُ ( فِي الْجِنَايَةِ وَإِنْ فَدَى ) أَيْ : فَدَاهُ سَيِّدُهُ بِأَقَلِّ الْأَمْرَيْنِ مِنْ أَرْشِ الْجِنَايَةِ وَقِيمَتِهِ ( بَقِيَ تَدْبِيرُهُ ) بِحَالِهِ كَأَنَّهُ لَمْ يَجْنِ ( وَإِنْ بِيعَ بَعْضُهُ ) أَيْ : الْمُدَبَّرُ فِي جِنَايَةٍ ( فَبَاقِيهِ ) الَّذِي لَمْ يُبَعْ ( مُدَبَّرٌ ) بِحَالِهِ ( وَإِنْ مَاتَ ) سَيِّدٌ مُدَبَّرٌ ( قَبْلَ بَيْعِهِ ) وَفِدَائِهِ أَيْ : الْمُدَبَّرِ ( عَتَقَ إنْ وَفَّى ثُلُثُهُ ) أَيْ : مَالِ السَّيِّدِ ( بِهَا ) أَيْ : الْجِنَايَةِ( 8 / 218 )
