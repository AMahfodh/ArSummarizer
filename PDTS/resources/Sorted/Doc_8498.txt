( وَهُوَ ) أَيْ الشَّرِيكُ الَّذِي بَاشَرَ الْعِتْقَ ( مُوسِرٌ بِقِيمَةِ بَاقِيهِ يَوْمَ ) أَيْ حِينَ ( عِتْقِهِ عَلَى مَا ذُكِرَ فِي زَكَاةِ فِطْرٍ عَتَقَ ) الْعَبْدِ ( كُلِّهِ ) أَوْ الْأَمَةِ كُلِّهَا ( وَعَلَيْهِ ) أَيْ الشَّرِيكِ الْمُبَاشِرِ لِلْعِتْقِ ( قِيمَةُ بَاقِيهِ لِشَرِيكِهِ ) لِمَا رَوَى ابْنُ عُمَرَ أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ { مَنْ أَعْتَقَ شِرْكًا لَهُ فِي عَبْدٍ وَكَانَ لَهُ مَالٌ يَبْلُغُ ثَمَنَ الْعَبْدِ ، قُوِّمَ عَلَيْهِ الْعَبْدُ قِيمَةَ عَدْلٍ ، فَأَعْطَى شُرَكَاءَهُ حِصَّتَهُمْ وَعَتَقَ عَلَيْهِ الْعَبْدُ وَإِلَّا فَقَدْ عَتَقَ عَلَيْهِ مَا عَتَقَ } مُتَّفَقٌ عَلَيْهِ .
