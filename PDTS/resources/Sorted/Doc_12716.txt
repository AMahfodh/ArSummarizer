( وَهُوَ ) أَيْ : يَوْمُ الشَّكِّ ( يَوْمُ الثَّلَاثِينَ مِنْ شَعْبَانَ إنْ لَمْ يَكُنْ فِي السَّمَاءِ ) فِي مَطْلِعِ الْهِلَالِ ( عِلَّةٌ ) مِنْ غَيْمٍ أَوْ قَتَرٍ وَنَحْوِهِمَا ، ( وَلَمْ يُرَ الْهِلَالُ أَوْ شَهِدَ بِهِ مَنْ رُدَّتْ شَهَادَتُهُ ) لِفِسْقٍ وَنَحْوِهِ ( إلَّا أَنْ يُوَافِقَ ) يَوْمَ الشَّكِّ ( عَادَةٌ ) كَمَنْ عَادَتُهُ يَصُومُ يَوْمَ الْخَمِيسِ وَالِاثْنَيْنِ فَوَافَقَ يَوْمَ الشَّكِّ أَحَدُهُمَا فَلَا كَرَاهَةَ ، أَوْ عَادَتُهُ يَصُومُ يَوْمًا وَيُفْطِرُ آخَرَ فَوَافَقَ صَوْمُهُ ذَلِكَ فَلَا كَرَاهَةَ ( أَوْ يَصِلُهُ ) أَيْ : يَوْمَ الشَّكِّ ( بِصِيَامٍ قَبْلَهُ ) لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { لَا تَقَدَّمُوا رَمَضَانَ بِصَوْمِ يَوْمٍ أَوْ يَوْمَيْنِ إلَّا رَجُلًا كَانَ يَصُومُ صَوْمًا فَلْيَصُمْهُ } مُتَّفَقٌ عَلَيْهِ مِنْ حَدِيثِ أَبِي هُرَيْرَةَ ( أَوْ يَصُومُهُ ) أَيْ : يَوْمَ الشَّكِّ ( عَنْ قَضَاءٍ أَوْ نَذْرٍ ) أَوْ كَفَّارَةٍ فَلَا كَرَاهَةَ ؛ لِأَنَّ صَوْمَهُ وَاجِبٌ إذَنْ .( 6 / 142 )
