( وَ ) إنْ قَالَ لِمَنْ لَهَا سُنَّةٌ وَبِدْعَةٌ أَنْتِ طَالِقٌ ( لِلْبِدْعَةِ ) فَقَطْ وَهِيَ ( فِي حَيْضٍ أَوْ فِي طُهْرٍ وَطِئَ فِيهِ ؛ يَقَعُ ) الطَّلَاقُ عَلَيْهِ ( فِي الْحَالِ ) لِأَنَّهُ وَصَفَ الطَّلْقَةَ بِصِفَتِهَا ( وَإِنْ ) كَانَتْ فِي طُهْرٍ ( لَمْ يَطَأْ ) هَا ( فِيهِ ) فَيَقَعُ الطَّلَاقُ ( إذَا حَاضَتْ أَوْ وَطِئَهَا ) لِوُجُودِ شَرْطِهِ ( وَيَنْزِعُ فِي( 16 / 27 )
