( قَوْلُهُ : وَلَهُمْ مَا يَدَّعُونَ ) أَيْ يَطْلُبُونَ ( قَوْلُهُ : وَأَلِفُهَا لِلتَّأْنِيثِ ) أَيْ لِأَنَّهَا بِوَزْنِ فَعْلَى ( قَوْلُهُ : إخْبَارٌ عَنْ وُجُوبِ حَقٍّ إلَخْ ) لَمْ يُقَيَّدْ الْحَقُّ بِكَوْنِهِ لَهُ لِيَشْمَلَ مَا لَوْ ادَّعَى الْوَلِيُّ بِمَالِ مُوَلِّيهِ أَوْ الْوَكِيلُ لِمُوَكِّلِهِ أَوْ النَّاظِرُ لِلْوَقْفِ ( قَوْلُهُ : عِنْدَ حَاكِمٍ ) أَوْ مَا فِي مَعْنَاهُ وَهُوَ الْمُحَكَّمُ وَالسَّيِّدُ كَمَا يَأْتِي وَمَا يَلْحَقُ بِهِمَا كَذِي الشَّوْكَةِ إذَا تَصَدَّى لِفَصْلِ الْأُمُورِ بَيْنَ أَهْلِ مَحَلَّتِهِ كَمَا تَقَدَّمَ لَهُ أَيْضًا وَيَأْتِي فِي قَوْلِهِ وَمَرَّ أَنَّهُ يَجِبُ الْأَدَاءُ عِنْدَ نَحْوِ أَمِيرٍ إلَخْ( 28 / 433 )
