( وَإِنْ أَزَالَ ) غَاصِبٌ أَوْ غَيْرُهُ ( اسْمَهُ ) - أَيْ : الْمَغْصُوبِ - بِعَمَلِهِ فِيهِ ( كَنَسْجِ غَزْلٍ ) فَصَارَ يُسَمَّى ثَوْبًا ، ( وَطَحْنِ حَبٍّ ) غَصَبَهُ فَصَارَ يُسَمَّى دَقِيقًا ، ( أَوْ طَبْخِهِ ) - أَيْ : الْحَبَّ - فَصَارَ يُسَمَّى طَبِيخًا ، ( وَنَجْرِ خَشَبٍ ) بَابًا أَوْ رُفُوفًا وَنَحْوَهَا ( وَضَرْبِ نَحْوِ حَدِيدٍ ) مَسَامِيرَ أَوْ سَيْفًا وَنَحْوَهُ ، ( وَ ) ضَرْبِ ( فِضَّةٍ ) دَرَاهِمَ أَوْ حُلِيًّا ، ( وَجَعْلِ طِينٍ ) غَصَبَهُ ( لَبِنًا ) أَوْ آجُرًّا ، ( أَوْ فُخَّارًا ) كَجِرَارٍ وَنَحْوِهَا ؛ ( رَدَّهُ ) الْغَاصِبُ وُجُوبًا مَعْمُولًا ؛ لِقِيَامِ عَيْنِ الْمَغْصُوبِ فِيهِ ، ( وَ ) رَدَّ ( أَرْشَهُ إنْ نَقَصَهُ ) ؛ لِحُصُولِ نَقْصِهِ بِفِعْلِهِ ، وَسَوَاءٌ نَقَصَتْ عَيْنُهُ أَوْ قِيمَتُهُ أَوْ هُمَا ، ( وَلَا شَيْءَ لَهُ ) - أَيْ : الْغَاصِبِ - ( لِعَمَلِهِ فِيهِ ) ، وَلَوْ زَادَ بِهِ ؛ لِتَبَرُّعِهِ بِهِ ؛ كَمَا لَوْ غَلَى زَيْتًا ، فَزَادَتْ قِيمَتُهُ ، بِخِلَافِ مَا لَوْ غَصَبَ ثَوْبًا فَصَبَغَهُ فَإِنَّهُ يَصِيرُ شَرِيكًا فِي زِيَادَةِ الثَّوْبِ ، وَالْفَرْقُ بَيْنَهُمَا أَنَّ الصَّبْغَ عَيْنُ مَالٍ لَا يَزُولُ مِلْكُ مَالِكِهِ عَنْهُ بِجَعْلِهِ مَعَ مِلْكِ غَيْرِهِ .
