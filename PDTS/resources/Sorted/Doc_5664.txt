الشَّرْطُ ( الْخَامِسُ كَفَاءَةُ زَوْجٍ عَلَى رِوَايَةٍ ) وَهِيَ الْمَذْهَبُ عِنْدَ أَكْثَرِ الْمُتَقَدِّمِينَ ( فَتَكُونُ ) الْكَفَاءَةُ ( حَقًّا لَلَهُ تَعَالَى وَلَهَا ) أَيْ الزَّوْجَةِ ( وَلِأَوْلِيَائِهَا كُلِّهِمْ فَ ) عَلَى هَذِهِ الرِّوَايَةِ ( لَوْ رَضِيَتْ ) امْرَأَةٌ ( مَعَ أَوْلِيَائِهَا بِ ) تَزْوِيجِ ( غَيْرِ كُفْءٍ لَمْ يَصِحَّ ) النِّكَاحُ ، لِفَوَاتِ شَرْطِهِ ( وَلَوْ زَالَتْ ) الْكَفَاءَةُ ( بَعْدَ عَقْدٍ فَلَهَا فَقَطْ ) دُونَ أَوْلِيَائِهَا ( الْفَسْخُ ) كَعِتْقِهَا تَحْتَ عَبْدٍ .
