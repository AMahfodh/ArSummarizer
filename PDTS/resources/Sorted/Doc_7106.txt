( وَإِنْ شَاءَ ) الْحَالِفُ ( كَفَّرَ قَبْلَ الْحِنْثِ فَتَكُونُ ) الْكَفَّارَةُ ( مُحَلِّلَةً لِلْيَمِينِ وَإِنْ شَاءَ ) كَفَّرَ ( بَعْدَهُ ) أَيْ الْحِنْثِ ( فَتَكُونُ مُكَفِّرَةً ) وَمِمَّنْ رُوِيَ عَنْهُ تَقْدِيمُ الْكَفَّارَةِ قَبْلَ الْحِنْثِ : عُمَرُ وَابْنُهُ وَابْنُ عَبَّاسٍ وَسَلْمَانُ وَعَنْ عَبْدِ الرَّحْمَنِ بْنِ سَمُرَةَ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ : يَا عَبْدَ الرَّحْمَنِ إذَا حَلَفْتَ عَلَى يَمِينٍ فَرَأَيْتَ غَيْرَهَا خَيْرًا مِنْهَا فَكَفِّرْ عَنْ يَمِينِكَ ثُمَّ ائْتِ الَّذِي هُوَ خَيْرٌ } رَوَاهُ أَبُو دَاوُد وَالنَّسَائِيُّ وَرِجَالُهُ ثِقَاتٌ .
