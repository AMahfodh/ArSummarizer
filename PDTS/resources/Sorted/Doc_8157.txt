( مُشْتَرٍ ) لِعَبْدٍ أَوْ أَمَةٍ ( قَالَ أَعْتَقَ الْبَائِعُ ) الْعَبْدَ ( أَوْ دَبَّرَ أَوْ اسْتَوْلَدَ ) الْأَمَةَ ( أَوْ هُوَ حُرُّ الْأَصْلِ وَأَنْكَرَ الْبَائِعُ حَلَفَ ) لِعَجْزِ الْمُشْتَرِي عَنْ الْإِثْبَاتِ ( فَإِنْ حَلَفَ قَضَى عَلَى الْمُشْتَرِي بِمَا قَالَهُ ) مِنْ الْعِتْقِ وَنَحْوِهِ لِإِقْرَارِهِ بِذَلِكَ ( وَرَجَعَ بِالْعَيْبِ إنْ عَلِمَ بِهِ ) ؛ لِأَنَّ الْمُبْطِلَ لِلرُّجُوعِ إزَالَتُهُ عَنْ مِلْكِهِ إلَى غَيْرِهِ بِإِنْشَائِهِ أَوْ إقْرَارِهِ وَلَمْ يُوجَدْ ( حَتَّى لَوْ قَالَ بَاعَهُ وَهُوَ مِلْكُ فُلَانٍ وَصَدَّقَهُ ) فُلَانٌ ( وَأَخَذَهُ لَا ) يَرْجِعُ بِالنُّقْصَانِ لِإِزَالَتِهِ بِإِقْرَارِهِ كَأَنَّهُ وَهَبَهُ .
