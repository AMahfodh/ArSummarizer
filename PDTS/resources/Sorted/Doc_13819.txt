( وَضَمِنَ ) بِفَتْحٍ فَكَسْرٍ شَخْصٌ ( بَائِعٌ ) شَيْئًا ( مَكِيلًا ) كَحَبٍّ وَغَايَةُ ضَمَانِهِ ( لِيَقْبِضَهُ ) أَيْ الْمَكِيلَ مُبْتَاعُهُ ( بِكَيْلٍ ) الظَّاهِرُ أَنَّ الْبَاءَ سَبَبِيَّةٌ أَوْ بِمَعْنَى بَعْدُ صِلَةُ قَبْضٍ ، فَهُوَ كَقَوْلِ ابْنِ الْحَاجِبِ ، وَالْقَبْضُ فِي الْمَكِيلِ بِكَيْلٍ وَشَبَّهَ فِي الضَّمَانِ فَقَالَ : ( كَ ) شَيْءٍ ( مَوْزُونٍ ) فَيَضْمَنُهُ بَائِعُهُ فِي حَالِ وَزْنِهِ ( وَ ) شَيْءٍ ( مَعْدُودٍ ) فَيَضْمَنُهُ بَائِعُهُ فِي حَالِ عَدِّهِ ( وَالْأُجْرَةُ ) لِلْكَيْلِ أَوْ الْوَزْنِ أَوْ الْعَدِّ الَّذِي يَحْصُلُ بِهِ التَّوْفِيَةُ لِلْمُشْتَرِي ( عَلَيْهِ ) أَيْ الْبَائِعِ لِوُجُوبِ التَّوْفِيَةِ عَلَيْهِ وَلَا تَحْصُلُ إلَّا بِذَلِكَ ، وَأُجْرَةُ كَيْلِ الثَّمَنِ أَوْ وَزْنِهِ أَوْ عَدِّهِ عَلَى الْمُشْتَرِي لِأَنَّهُ بَائِعُهُ إلَّا لِشَرْطٍ أَوْ عُرْفٍ ، بِخِلَافِ ذَلِكَ فِي الْمَسْأَلَتَيْنِ ( بِخِلَافِ الْإِقَالَةِ ) أَيْ تَرْكِ الْمَبِيعِ لِبَائِعِهِ بِثَمَنِهِ ( وَالتَّوْلِيَةِ ) أَيْ تَرْكِ الْمَبِيعِ بِثَمَنِهِ لِغَيْرِ بَائِعِهِ ( وَالشَّرِكَةِ ) أَيْ تَرْكِ بَعْضِ الْمَبِيعِ بِحِصَّتِهِ مِنْ ثَمَنِهِ لِغَيْرِ بَائِعِهِ فَالْأُجْرَةُ عَلَى الْمُقَالِ وَالْمُوَلِّي وَالْمُشْرِكُ بِالْكَسْرِ فِيهِمَا فَعَلَ مَعْرُوفًا ، فَلَا يَغْرَمُ ، وَلِذَا كَانَ السَّائِلُ الْمُقِيلَ : وَالْمُوَلِّي وَالْمُشْرِكُ بِالْكَسْرِ لَكَانَتْ الْأُجْرَةُ عَلَيْهِ .
