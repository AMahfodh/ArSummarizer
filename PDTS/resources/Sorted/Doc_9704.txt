( قَوْلُهُ : ، وَقَالَ ابْنُ الْوَكِيلِ : إلَخْ ) وَإِلَّا وُجِّهَ الْأَوَّلُ كَمَا قَالَهُ فِي شَرْحِ الرَّوْضِ فَافْتَرَقَ حِينَئِذٍ ( قَوْلُهُ : فَلَوْ قَطَعَ نِصْفَ لِسَانِهِ إلَخْ ) لَوْ قَطَعَ نِصْفَ لِسَانِهِ فَذَهَبَ نِصْفُ كَلَامِهِ فَاقْتُصَّ مِنْهُ فَذَهَبَ رُبْعُ الْكَلَامِ كَانَ لِلْمَجْنِيِّ عَلَيْهِ رُبْعُ الدِّيَةِ وَلَوْ ذَهَبَ ثَلَاثَةُ أَرْبَاعِهِ فَلَا شَيْءَ فِي ذَلِكَ بِرّ ؛ لِأَنَّ سِرَايَةَ الْقِصَاصِ مُهْدَرَةٌ شَرْحُ رَوْضٍ ( قَوْلُهُ : تَجِبُ الدِّيَةُ ) فَقَدْ وَجَبَ الْأَكْثَرُ ( قَوْلُهُ : وَإِنْ فَاتَ مِنْ الْمَنْفَعَةِ دُونَ الْخُمْسَيْنِ فَقَدْ وَجَبَ الْأَكْثَرُ ) وَقِيَاسُ ذَلِكَ أَنَّهُ إذَا فَاتَ مِنْ الْمَنْفَعَةِ أَكْثَرُ مِنْ الْخُمْسَيْنِ وَجَبَ الْأَكْثَرُ فَلْيُرَاجَعْ( 17 / 435 )
