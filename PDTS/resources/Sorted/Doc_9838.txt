( فَصْلٌ وَلَا تَصِحُّ الْوَصِيَّةُ لِكَنِيسَةٍ وَلَا لِحُصْرِهَا وَقَنَادِيلِهَا وَنَحْوِهِ ) ( وَلَا ) لِ ( بَيْتِ نَارٍ ، وَ ) لَا لِ ( بِيَعَةٍ وَصَوْمَعَةٍ ، وَ ) لَا ( دَيْرٍ وَلَا لِإِصْلَاحِهَا وَشُعَلِهَا وَخِدْمَتِهَا وَلَا لِعِمَارَتِهَا ) وَلَوْ مِنْ ذِمِّيٍّ لِأَنَّ ذَلِكَ إعَانَةٌ عَلَى مَعْصِيَةٍ ( وَلَا لِكُتُبِ التَّوْرَاةِ وَالْإِنْجِيلِ وَالزَّبُورِ وَالصُّحُفِ وَلَوْ ) كَانَتْ الْوَصِيَّةُ ( مِنْ ذِمِّيٍّ لِأَنَّهَا كُتُبٌ مَنْسُوخَةٌ وَالِاشْتِغَالُ بِهَا غَيْرُ جَائِزٍ ) لِمَا فِيهَا مِنْ التَّغْيِيرِ وَالتَّبْدِيلِ ( وَإِنْ وَصَّى بِبِنَاءِ بَيْتٍ يَسْكُنُهُ الْمُجْتَازُونَ ) أَيْ : الْمَارُّونَ ( مِنْ أَهْلِ الذِّمَّةِ وَأَهْلِ الْحَرْبِ صَحَّ ) لِأَنَّ بِنَاءَ مَسَاكِنِهِمْ لَيْسَ بِمَعْصِيَةٍ .( 15 / 231 )
