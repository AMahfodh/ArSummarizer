فِي مَوَانِعِ وِلَايَةِ النِّكَاحِ ( قَوْلُهُ : وَكَالْمُكَاتَبِ بِالْإِذْنِ ) أَيْ مِنْ سَيِّدِهِ ( قَوْلُهُ : وَإِنْ تَقَطَّعَ الْجُنُونُ ) لَيْسَ الْمُرَادُ أَنَّهُ لَا وِلَايَةَ لَهُ حَتَّى فِي زَمَنِ الْإِفَاقَةِ بَلْ مَعْنَاهُ أَنَّ الْأَبْعَدَ يُزَوِّجُ فِي زَمَنِ الْجُنُونِ وَلَا يَجِبُ انْتِظَارُ الْإِفَاقَةِ وَأَمَّا هُوَ فِي زَمَنِ إفَاقَتِهِ فَيَصِحُّ تَزْوِيجُهُ وَلِهَذَا عَبَّرَ فِي الرَّوْضِ بِقَوْلِهِ وَذِي جُنُونٍ فِي حَالَتِهِ وَلَوْ تَقَطَّعَ انْتَهَى وَعَبَّرَ الشَّارِحُ بِقَوْلِهِ الْآتِي فَيُزَوِّجُ الْأَبْعَدُ زَمَنَهُ فَقَطْ انْتَهَى ( قَوْلُهُ : أَيْ مِنْ حَيْثُ عَدَمُ انْتِظَارِهِ إلَخْ ) عَلَى هَذَا يُسَاوِي هَذَا الْقِسْمُ مَا تَقَدَّمَ أَوَّلًا إلَّا أَنْ يَلْتَزِمَ هُنَا صِحَّةَ تَزْوِيجِ الْأَبْعَدِ زَمَنَ الْإِفَاقَةِ أَيْضًا وَفِيهِ نَظَرٌ ( قَوْلُهُ : وَبَحْثُ الْأَذْرَعِيِّ إلَخْ ) كَذَا شَرْحُ م ر قَوْلُهُ : لِبَقَاءِ أَهْلِيَّتِهِ ) أَيْ الْغَائِبِ ( قَوْلُهُ : مُطْلَقًا ) أَيْ حُجِرَ عَلَيْهِ أَوْ لَا ( قَوْلُهُ : فَيَلِي كَمَا بَحَثَهُ إلَخْ ) اعْتَمَدَهُ م ر ( قَوْلُهُ : وَعَلَيْهِ ) أَيْ عَلَى الْخِلَافِ .( 29 / 497 )
