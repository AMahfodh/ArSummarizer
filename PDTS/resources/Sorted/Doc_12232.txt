( وَإِنْ اجْتَمَعَ قَوْمٌ بِمَحَلٍّ فَقَتَلَ ) بَعْضٌ بَعْضًا ( أَوْ جَرَحَ بَعْضٌ مِنْهُمْ ) بَعْضًا ( وَجُهِلَ الْحَالُ ) أَيْ حَالُ الْقَاتِلِينَ وَالْمَقْتُولِينَ ( فَعَلَى عَاقِلَةِ الْمَجْرُوحِينَ دِيَةُ الْقَتْلَى ) مِنْهُمْ ( يَسْقُطُ مِنْهَا ) أَيْ الدِّيَةِ ( أَرْشَ الْجِرَاحِ ) نَصَّ عَلَيْهِ لِرِوَايَتِهِ بِإِسْنَادِهِ إلَى الشَّعْبِيِّ قَالَ : أَشْهَدُ عَلَى عَلِيٍّ أَنَّهُ قَضَى بِهِ وَظَاهِرُهُ أَنَّهُ لَا شَيْءَ مِنْ الدِّيَةِ عَلَى مَنْ لَيْسَ بِهِ جُرْحٌ قَالَ فِي تَصْحِيحِ الْفُرُوعِ وَهُوَ ظَاهِرُ كَلَامِ جَمَاعَةٍ مِنْ الْأَصْحَابِ ( وَمَنْ ادَّعَى عَلَى آخَرَ أَنَّهُ قَتَلَ مُورِثَهُ فَقَالَ : إنَّمَا قَتَلَهُ زَيْدٌ فَصَدَقَ زَيْدٌ ) بِأَنَّ أَقَرَّ أَنَّهُ قَتَلَهُ ( أَخَذَ ) زَيْدٌ ( بِهِ ) نَصَّ عَلَيْهِ فِي رِوَايَةِ مُهَنَّا وَقَالَ : قُلْتُ أَلَيْسَ قَدْ ادَّعَى عَلَى الْأَوَّلِ قَالَ إنَّمَا هَذَا بِالظَّنِّ فَأَعَدْتُ عَلَيْهِ فَقَالَ يُؤْخَذُ الَّذِي أَقَرَّ أَنَّهُ قَتَلَهُ .( 10 / 329 )
