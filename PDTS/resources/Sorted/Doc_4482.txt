( وَيُجْزِئُ فِيهِمَا ) أَيْ الْهَدْيِ وَالْأُضْحِيَّةِ ( جَمَّاءُ ) لَمْ يُخْلَقْ لَهَا قَرْنٌ ( وَبَتْرَاءُ ) لَا أُذُنَ لَهَا خِلْقَةً ، أَوْ مَقْطُوعًا ( وَصَمْعَاءُ ) بِصَادٍ وَعَيْنٍ مُهْمَلَتَيْنِ صَغِيرَةُ الْأُذُنِ ( وَخَصِيٌّ ) مَا قُطِعَتْ خُصْيَتَاهُ أَوْ سُلَّتَا ( وَمَرْضُوضُ الْخُصْيَتَيْنِ ) لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { ضَحَّى بِكَبْشَيْنِ مَوْجُوءَيْنِ } وَالْوَجْءُ رَضُّ الْخُصْيَتَيْنِ وَلِأَنَّ الْخِصَاءَ أَذْهَبَ عُضْوًا غَيْرَ مُسْتَطَابٍ يَطِيبُ اللَّحْمُ بِذَهَابِهِ وَيَسْمَنُ .
