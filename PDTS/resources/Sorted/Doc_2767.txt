( أَوْ ) أَيْ : وَلَا يَصِحُّ وَقْفُ ( مَا لَا يَصِحُّ بَيْعُهُ كَأُمِّ وَلَدٍ وَكَلْبٍ ) وَلَوْ لِنَحْوِ صَيْدٍ ( وَمَرْهُونٍ ) ؛ لِأَنَّهُ لَا يَصِحُّ بَيْعُهَا وَالْوَقْفُ تَصَرُّفٌ بِإِزَالَةِ الْمِلْكِ ( أَوْ لَا يُنْتَفَعُ بِهِ مَعَ بَقَائِهِ كَمَطْعُومٍ ) وَمَشْرُوبِ غَيْرِ مَاءٍ ( وَمَشْمُومٍ ) لَا يُنْتَفَعُ بِهِ مَعَ بَقَاءِ عَيْنِهِ ، بِخِلَافِ نَدٍّ وَصَنْدَلٍ وَقِطَعِ كَافُورٍ .
