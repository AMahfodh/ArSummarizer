وَقَالَ ابْنُ كِنَانَةَ مَقْتَلٌ ( وَنَثْرِ ) بِفَتْحِ النُّونِ وَسُكُونِ الْمُثَلَّثَةِ أَيْ : خُرُوجِ ( دِمَاغٍ ) أَيْ : مُخٍّ حَوَتْهُ الْجُمْجُمَةُ فَشَدْخُ الرَّأْسِ دُونَ نَثْرِ دِمَاغٍ لَيْسَ مَقْتَلًا قَالَهُ عَبْدُ الْحَقِّ ، وَلَا خَرْقُ خَرِيطَةٍ أَيْ : جِلْدَةٍ سَاتِرَةٍ لِلدِّمَاغِ وَلَا رَضُّ الْأُنْثَيَيْنِ ، وَكَسْرُ عَظْمِ الصَّدْرِ وَغَيْرِهَا مِنْ بَاقِي الْمَتَالِفِ ( أَوْ ) نَثْرِ ( حَشْوَةٍ ) بِضَمِّ الْحَاءِ الْمُهْمَلَةِ وَكَسْرِهَا وَسُكُونِ الشِّينِ الْمُعْجَمَةِ أَيْ : مَا حَوَاهُ( 5 / 152 )
