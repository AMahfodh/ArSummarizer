( سُئِلَ ) فِيمَا إذَا كَانَ لِزَيْدٍ دَيْنٌ مَعْلُومٌ مِنْ الدَّرَاهِمِ بِذِمَّةِ عَمْرٍو فَدَفَعَ لِزَيْدٍ قَدْرًا مَعْلُومًا مِنْ الْحِنْطَةِ وَقَالَ لَأُحَاسِبُك بِهِ مِنْ دَيْنِك بِسِعْرِ الْبَلْدَةِ وَالسِّعْرُ مَعْلُومٌ بَيْنَهُمَا وَلَمْ يَذْكُرَا ثَمَنًا فَأَخَذَهُ وَقَبِلَهُ كَمَا ذُكِرَ فَهَلْ يَكُونُ ذَلِكَ بَيْعًا بِالدَّيْنِ بِالسِّعْرِ يَوْمَ الْأَخْذِ ؟ ( الْجَوَابُ ) : نَعَمْ .( 3 / 281 )
