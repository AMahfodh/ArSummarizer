( وَإِنْ تَبَرَّعَ ) وَارِثٌ ( بِعِتْقِهِ ) عَنْ الْمَيِّتِ فِي وَاجِبٍ عَلَيْهِ ( وَلَا تَرِكَةَ ) لِلْمَيِّتِ ( أَجْزَأَ ) الْعِتْقُ عَنْهُ ( كَ ) مَا لَوْ تَبَرَّعَ عَنْهُ بِإِطْعَامٍ أَوْ كِسْوَةٍ فِي كَفَّارَةِ يَمِينٍ ، وَإِنْ تَبَرَّعَ بِهِمَا ( أَجْنَبِيٌّ ) بِعِتْقٍ أَجْزَأَ كَقَضَائِهِ عَنْهُ دَيْنًا .( 14 / 128 )
