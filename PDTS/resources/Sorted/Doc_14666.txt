( وَإِنْ ) ( وَقَعَ ) النِّكَاحُ ( بِقُلَّةِ خَلٍّ ) مُعَيَّنَةٍ حَاضِرَةٍ ( فَإِذَا هِيَ خَمْرٌ ) ( فَمِثْلُهُ ) أَيْ فَلِلزَّوْجَةِ مِثْلُ الْخَلِّ وَالنِّكَاحُ ثَابِتٌ بِخِلَافِ الْبَيْعِ فَيُفْسَخُ ، ثُمَّ ذَكَرَ أَرْبَعَ مَسَائِلَ كَالْمُسْتَثْنَاةِ مِنْ قَوْلِهِ كَالثَّمَنِ لِعَدَمِ صِحَّةِ كَوْنِ شَيْءٍ مِنْهَا ثَمَنًا فَقَالَ ( وَجَازَ ) النِّكَاحُ ( بِشَوْرَةٍ ) مَعْرُوفَةٍ عِنْدَهُمْ وَهِيَ بِالْفَتْحِ مَتَاعُ الْبَيْتِ وَبِالضَّمِّ الْجَمَالُ ( أَوْ ) عَلَى ( عَدَدٍ ) مَعْلُومٍ كَعَشَرَةٍ ( مِنْ كَإِبِلٍ أَوْ رَقِيقٍ ) ، وَلَوْ فِي الذِّمَّةِ غَيْرَ مَوْصُوفٍ لَا عَدَدٍ مِنْ شَجَرٍ إلَّا إنْ عَيَّنَ ( أَوْ ) عَلَى ( صَدَاقِ مِثْلٍ ) أَيْ مِثْلِهَا ( وَلَهَا ) فِي الْمَسَائِلِ الْأَرْبَعِ ( الْوَسَطُ ) مِنْ شَوْرَةِ مِثْلِهَا فِي حَضَرٍ لِحَضَرِيَّةٍ وَبَدْوٍ لِبَدْوِيَّةٍ وَالْوَسَطُ مِنْ كَإِبِلٍ وَرَقِيقٍ مِنْ السِّنِّ الَّذِي يَتَنَاكَحُ بِهِ النَّاسُ وَالْوَسَطُ مِنْ صَدَاقِ مِثْلٍ يُرْغَبُ بِهِ فِي مِثْلِهَا بِاعْتِبَارِ الْأَوْصَافِ الَّتِي تُعْتَبَرُ فِي صَدَاقِ الْمِثْلِ مِنْ جَمَالٍ وَحَسَبٍ وَنَسَبٍ وَيُعْتَبَرُ الْوَسَطُ مِنْ ذَلِكَ ( حَالًّا ) لَا مُؤَجَّلًا ( وَفِي شَرْطِ ذِكْرِ جِنْسٍ ) أَيْ صِنْفِ ( الرَّقِيقِ ) إذَا تَزَوَّجَهَا عَلَى عَدَدٍ مَعْلُومٍ مِنْهُ تَقْلِيلًا لِلْغَرَرِ كَبَرْبَرِيٍّ أَوْ حَبَشِيٍّ أَوْ زِنْجِيٍّ أَوْ رُومِيٍّ وَعَدَمُ اشْتِرَاطِهِ لَهَا أَغْلَبَ الصِّنْفَيْنِ بِالْبَلَدِ مِنْ السُّودِ وَالْحُمْرِ فَإِنْ اسْتَوَيَا أُعْطِيت النِّصْفَ الْوَسَطَ مِنْ كُلٍّ فَإِنْ كَانَتْ الْأَصْنَافُ ثَلَاثَةً أُعْطِيت مِنْ وَسَطِ كُلِّ صِنْفٍ ثُلُثَهُ وَهَكَذَا .
