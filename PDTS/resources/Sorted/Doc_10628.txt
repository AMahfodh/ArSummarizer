( وَيَحْرُمُ إيقَاعُ ) طَلْقَاتٍ ( ثَلَاثٍ ، وَلَوْ بِكَلِمَاتٍ ) وَلَوْ ( فِي طُهْرٍ لَمْ يُصِبْهَا ) زَوْجُهَا ( فِيهِ أَوْ ) أَيْ : وَيَحْرُمُ إيقَاعُ ثَلَاثٍ فِي ( أَطْهَارٍ مِنْهُ لَا ) يَحْرُمُ إيقَاعُ ذَلِكَ ( بَعْدَ رَجْعَةٍ أَوْ ) بَعْدَ ( عَقْدٍ ) رُوِيَ ذَلِكَ عَنْ عُمَرَ وَعَلِيٍّ وَابْنِ مَسْعُودٍ وَابْنِ عَبَّاسٍ وَابْنِ عُمَرَ ؛ لِقَوْلِهِ تَعَالَى { يَا أَيُّهَا النَّبِيُّ إذَا طَلَّقْتُمْ النِّسَاءَ فَطَلِّقُوهُنَّ لِعِدَّتِهِنَّ } : إلَى قَوْله تَعَالَى : { وَمَنْ يَتَّقِ اللَّهَ يَجْعَلْ لَهُ مَخْرَجًا } { وَمَنْ يَتَّقِ اللَّهَ يَجْعَلْ لَهُ مِنْ أَمْرِهِ يُسْرًا } وَمَنْ جَمَعَ الثَّلَاثَ لَمْ يَبْقَ لَهُ أَمْرٌ يَحْدُثُ ، وَلَمْ يَجْعَلْ لَهُ مَخْرَجًا ، وَلَا مِنْ أَمْرِهِ يُسْرًا .
