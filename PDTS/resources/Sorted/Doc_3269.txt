مُغْنِي ( قَوْلُهُ : وَكَخَشَبَةٍ إلَخْ ) عَطْفٌ عَلَى كَثَوْبٍ إلَخْ ( قَوْلُهُ : وَذَلِكَ ) أَيْ عَدَمُ صِحَّةِ مَا ذُكِرَ ( قَوْلُهُ : لِتَوَقُّفِهِ ) أَيْ التَّسْلِيمِ ( عَلَى مَا ) أَيْ كُسِرَ أَوْ قُطِعَ ( يُنْقِصُ مَالِيَّتَهُ ) أَيْ مَالِيَّةَ الْمَبِيعِ أَوْ الْبَاقِي نَقْصًا لَا يُمْكِنُ تَدَارُكُهُ ( قَوْلُهُ : وَقَدْ نُهِينَا عَنْ إضَاعَةِ الْمَالِ ) أَيْ فَهِيَ حَرَامٌ .
