( وَتَلَاعَنَا ) بِقَوْلِهِ ( زَنَيْتِ وَهَذَا الْحَمْلُ مِنْهُ ) لِلْقَذْفِ الصَّرِيحِ ( وَلَمْ يَنْفِ ) الْحَاكِمُ ( الْحَمْلَ ) لِعَدَمِ الْحُكْمِ عَلَيْهِ قَبْلَ وِلَادَتِهِ ، وَنَفْيُهُ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ وَلَدَ هِلَالٍ لِعِلْمِهِ بِالْوَحْيِ ( نَفْيُ الْوَلَدِ ) الْحَيِّ ( عِنْدَ التَّهْنِئَةِ ) وَمُدَّتُهَا سَبْعَةُ أَيَّامٍ عَادَةً ( وَ ) عِنْدَ ( ابْتِيَاعِ آلَةِ الْوِلَادِ صَحَّ وَبَعْدَهُ لَا ) لِإِقْرَارِهِ بِهِ دَلَالَةً ، وَلَوْ غَائِبًا فَحَالَةُ عِلْمِهِ كَحَالَةِ وِلَادَتِهَا ( وَلَاعَنَ فِيهِمَا ) فِيمَا إذَا صَحَّ أَوَّلًا لِوُجُودِ الْقَذْفِ ، فَقَدْ تَحَقَّقَ اللِّعَانُ بِنَفْيِ الْوَلَدِ وَلَمْ يَنْتَفِ النَّسَبُ ، فَقَوْلُهُ فِيمَا مَرَّ وَنَفَى نَسَبَهُ لَيْسَ عَلَى إطْلَاقِهِ .
