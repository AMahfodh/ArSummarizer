( وَإِنْ تَلِفَ جُزْءُ نِصَابٍ ) بِلَا تَفْرِيطٍ بَعْدَ الْحَوْلِ وَأَوْلَى جَمِيعُهُ ( وَ ) الْحَالُ أَنَّهُ ( لَمْ يُمْكِنْ الْأَدَاءُ ) مِنْهُ إمَّا لِعَدَمِ مُسْتَحِقٍّ أَوْ لِعَدَمِ الْوُصُولِ إلَيْهِ أَوْ لِغَيْبَةِ الْمَالِ ( سَقَطَتْ ) الزَّكَاةُ فَإِنْ أَمْكَنَ الْأَدَاءُ وَفَرَّطَ ضَمِنَ ، وَأَمَّا مَا تَلِفَ قَبْلَ الْحَوْلِ فَيُعْتَبَرُ الْبَاقِي بِلَا تَفْصِيلٍ وَمِنْهُ مَا قَبْلَ هَذِهِ ( كَعَزْلِهَا ) بَعْدَ الْحَوْلِ لِمُسْتَحِقِّهِمَا فَضَاعَتْ أَوْ تَلِفَتْ بِلَا تَفْرِيطٍ وَلَا إمْكَانِ أَدَاءً سَقَطَتْ فَإِنْ وَجَدَهَا لَزِمَهُ إخْرَاجُهَا ، وَأَمَّا لَوْ عَزَلَهَا قَبْلَ الْحَوْلِ ( فَضَاعَتْ ) ضَمِنَ أَيْ يَعْتَبِرُ مَا بَقِيَ ( لَا إنْ ضَاعَ أَصْلُهَا ) بَعْدَ الْحَوْلِ فَلَا تَسْقُطُ وَيُعْطِيهَا لِمُسْتَحِقِّهَا فَرَّطَ أَمْ لَا
