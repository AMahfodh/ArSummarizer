( وَإِنْ أَوْلَدَ مُرْتَدٌّ أَمَتَهُ وَأَسْلَمَ صَارَتْ أُمَّ وَلَدٍ ) لَهُ ( وَإِلَّا فَلَا ) فَالِاسْتِيلَادُ قَبْلَ إسْلَامِهِ مَوْقُوفٌ كَمِلْكِهِ ( وَلَا تُبَاعُ مُسْتَوْلَدَةٌ كَافِرَةٌ أَسْلَمَتْ ) وَلَا مَنْ اسْتَوْلِدْهَا بَعْدَ إسْلَامِهَا كَمَا صَرَّحَ بِهِ الْأَصْلُ ( بَلْ تُجْعَلُ عِنْدَ امْرَأَةٍ ثِقَةٍ ) لِيُحَالَ بَيْنَهُمَا ( وَقَدْ ذَكَرَ ) ذَلِكَ ( فِي ) كِتَابِ ( الْبَيْعِ وَنَفَقَتُهَا عَلَيْهِ وَكَسْبُهَا لَهُ ) فَإِنْ أَسْلَمَ رُفِعَتْ الْحَيْلُولَةُ ، وَإِنْ مَاتَ عَتَقَتْ ( وَيُزَوِّجُهَا الْحَاكِمُ ) لَا السَّيِّدُ لِانْقِطَاعِ الْمُوَالَاةِ ( بِإِذْنِهَا إنْ طَلَبَتْ ) ذَلِكَ ( أَوْ بِإِذْنِ السَّيِّدِ إنْ طَلَبَ ) هُوَ ، وَإِنْ كَرِهَتْ هِيَ ( وَالْمَهْرُ لِلسَّيِّدِ ) وَقِيلَ لَا يُزَوِّجُهَا الْحَاكِمُ أَيْضًا وَالتَّرْجِيحُ مِنْ زِيَادَتِهِ وَكَذَا قَوْلُهُ بِإِذْنِهَا وَلَا حَاجَةَ إلَيْهِ مَعَ قَوْلِهِ إنْ طَلَبَتْ مَعَ أَنَّ كَلَامَهُ كَأَصْلِهِ وَيُوهِمُ أَنَّهُ لَا حَاجَةَ مَعَ ذَلِكَ إلَى إذْنِ السَّيِّدِ وَالْأَوْجَهُ خِلَافُهُ ؛ لِأَنَّهُ لَا يُجْبَرُ عَلَى تَزْوِيجِ أَمَتِهِ فَتَحَرَّرَ أَنَّهَا إنْ طَلَبَتْ مِنْ الْحَاكِمِ تَزْوِيجَهَا فَلَا بُدَّ مِنْ إذْنِ السَّيِّدِ ، وَإِنْ أَذِنَ لَهُ السَّيِّدُ فِيهِ فَلَا حَاجَةَ إلَى طَلَبِهَا .
