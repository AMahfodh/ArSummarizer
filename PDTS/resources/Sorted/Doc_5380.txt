وَفَرْضُ عَيْنٍ عَلَى الْوَاحِدِ لِقَوْلِهِ تَعَالَى { وَإِذَا حُيِّيتُمْ بِتَحِيَّةٍ فَحَيُّوا بِأَحْسَنَ مِنْهَا أَوْ رُدُّوهَا } وَلِمَا نَذْكُرُهُ مِنْ الْأَخْبَارِ النَّبَوِيَّةِ ( لَيْسَ ) رَدُّكَ السَّلَامَ ( نَدْبًا ) أَيْ مَنْدُوبًا بَلْ وَاجِبٌ خِلَافًا لِظَاهِرِ كَلَامِ جَمَاعَةٍ مِنْ الْأَصْحَابِ ، رَحِمَهُمْ الْمَلِكُ الْوَهَّابُ ( بِأَوْطَدَ ) أَيْ بِأَثْبَتَ وَأَشْهَرَ ، يُقَالُ وَطَدَ الشَّيْءَ يَطِدُهُ وَطْدًا فَهُوَ وَطِيدٌ وَمَوْطُودٌ أَثْبَتَهُ وَثَقَّلَهُ كَوَطَّدَهُ فَتَوَطَّدَ ، وَوَطَدَ الشَّيْءُ دَامَ وَثَبُتَ( 1 / 420 )
