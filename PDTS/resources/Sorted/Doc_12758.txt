( وَإِنْ رَهَنَ ذِمِّيٌّ عِنْدَ مُسْلِمٍ خَمْرًا ) وَلَوْ شَرَطَ جَعْلَهُ ( بِيَدِ ذِمِّيٍّ لَمْ يَصِحَّ ) الرَّهْنُ ؛ لِأَنَّهُ لَمْ يَصِحَّ بَيْعُهَا ( فَإِنْ بَاعَهَا ) أَيْ : الْخَمْرَ ( الْوَكِيلُ ) صُورَةً أَيْ : الذِّمِّيُّ الَّتِي هِيَ عِنْدَهُ أَوْ بَاعَهَا رَبُّهَا ( حَلَّ ) لِرَبِّ دَيْنٍ أَخْذُ دَيْنِهِ مِنْ ثَمَنِهَا ؛ لِأَنَّهُ يُقَرُّ عَلَيْهِ لَوْ أَسْلَمَ ( فَيَقْبِضُهُ ) أَيْ : الدَّيْنَ مِنْ ثَمَنِ خَمْرٍ بَاعَهَا ذِمِّيٌّ وَإِنْ لَمْ يَكُنْ رَهْنٌ لِقَوْلِ عُمَرَ فِي أَهْلِ الذِّمَّةِ مَعَهُمْ الْخُمُورُ وَلُّوهُمْ بَيْعَهَا وَخُذُوا مِنْ أَثْمَانِهَا ( أَوْ يُبْرِئُ ) رَبُّ الدَّيْنِ ( مِنْهُ ) وَعُلِمَ مِمَّا سَبَقَ أَنَّهُ لَا يُشْتَرَطُ كَوْنُ رَهْنٍ مِنْ مَدِينٍ وَلَا بِإِذْنِهِ ؛ لِأَنَّهُ إذَا جَازَ أَنْ يَقْضِيَ عَنْهُ دَيْنَهُ بِلَا إذْنِهِ فَأَوْلَى أَنْ يَرْهَنَ عَنْهُ قَالَ الشَّيْخُ تَقِيُّ الدِّينِ : يَجُوزُ أَنْ يَرْهَنَ الْإِنْسَانُ مَالَ نَفْسِهِ عَلَى دَيْنِ غَيْرِهِ كَمَا يَجُوزُ أَنْ يَضْمَنَهُ وَأَوْلَى .( 5 / 184 )
