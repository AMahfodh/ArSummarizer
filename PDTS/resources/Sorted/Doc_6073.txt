( وَلَا ) يَعْقِلُ ( صَغِيرٌ أَوْ مَجْنُونٌ ) ؛ لِأَنَّهُمَا لَيْسَا مِنْ أَهْلِ النُّصْرَةِ وَالْمُعَاضَدَةِ ( أَوْ امْرَأَةٌ ) وَلَوْ مُعْتَقَةً ( أَوْ خُنْثَى مُشْكِلٌ ) ؛ لِاحْتِمَالِ أَنْ تَكُونَ امْرَأَةً ( أَوْ قِنٌّ ) ؛ لِأَنَّهُ لَا مَالَ لَهُ ( أَوْ مُبَايِنٌ لِدِينِ جَانٍ ) ؛ لِأَنَّ حَمْلَهَا لِلنُّصْرَةِ ، وَلَا نُصْرَةَ لِمُخَالِفٍ فِي دِينِهِ ( وَإِخْوَةٌ لِأُمٍّ أَوْ ذُو رَحِمٍ ) ؛ لِأَنَّهُمْ لَيْسُوا مِنْ أَهْلِ النُّصْرَةِ .( 18 / 11 )
