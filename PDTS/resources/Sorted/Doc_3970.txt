( وَهُوَ ) أَيْ الرَّطْلُ الْعِرَاقِيُّ ( سُبْعُ الْقُدْسِيِّ وَثُمُنُ سُبْعِهِ ) ؛ لِأَنَّ سُبْعَ الْقُدْسِيِّ مِائَةٌ وَأَرْبَعَةَ عَشَرَ دِرْهَمًا وَسُبْعَا دِرْهَمٍ ( وَسُبْعُ الْحَلَبِيِّ وَرُبْعُ سُبُعِهِ ) ؛ لِأَنَّ سُبْعَهُ مِائَةٌ وَدِرْهَمَانِ وَسِتَّةُ أَسْبَاعِ دِرْهَمٍ ( وَسُبْعُ الدِّمَشْقِيِّ وَنِصْفُ سُبْعِهِ ) ؛ لِأَنَّ سُبْعَهُ خَمْسَةٌ وَثَمَانُونَ دِرْهَمًا وَخَمْسَةُ أَسْبَاعِ دِرْهَمٍ ( وَسِتَّةُ أَسْبَاعِ الْمِصْرِيِّ وَرُبْعُ سُبْعِهِ ) لِأَنَّ سُبْعَهُ عِشْرُونَ دِرْهَمًا وَأَرْبَعَةُ أَسْبَاعِ دِرْهَمٍ .
