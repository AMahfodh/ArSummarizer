( وَ ) يَصِحُّ بَيْعُ النَّحْلِ فِي كُوَّارَتِهِ ( بِدُونِهَا إذَا شُوهِدَ دَاخِلًا إلَيْهَا ) أَيْ : إلَى كُوَّارَتِهِ هَذَا قَوْلُ الْأَكْثَرِ وَاقْتَصَرَ عَلَيْهِ فِي الْمُنْتَهَى وَغَيْرِهِ وَقَوْلُهُ : ( فَيُشْتَرَطُ مَعْرِفَتُهُ بِفَتْحِ رَأْسِهَا ) أَيْ : الْكُوَّارَةِ ( وَمُشَاهَدَتِهِ ) أَيْ : النَّحْلِ : يَقْتَضِي أَنَّهُ لَا يُشْتَرَطُ مُشَاهَدَتُهُ دَاخِلًا إلَيْهَا ، بَلْ يَكْفِي رُؤْيَتُهُ فِيهَا ، وَهُوَ قَوْلُ أَبِي الْخَطَّابِ قَالَ ( وَخَفَاءُ بَعْضِهِ لَا يَمْنَعُ الصِّحَّةَ ) أَيْ صِحَّةَ الْبَيْعِ ( كَالصُّبْرَةِ ) لَا يَمْنَعُ صِحَّةَ بَيْعِهَا اسْتِتَارُ بَعْضِهَا بِبَعْضٍ فَفِي كَلَامِهِ نَظَرٌ ظَاهِرٌ ( وَلَا يَصِحُّ بَيْعُهَا ) أَيْ : الْكُوَّارَةِ ( بِمَا فِيهَا مِنْ عَسَلٍ وَنَحْلٍ ) لِلْجَهَالَةِ ( وَلَا ) يَصِحُّ ( بَيْعُ مَا كَانَ مَسْتُورًا ) مِنْ النَّحْلِ ( بِأَقْرَاصِهِ ) لِلْجَهَالَةِ .( 8 / 414 )
