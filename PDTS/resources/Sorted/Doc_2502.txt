وَهَذِهِ فِي كُلٍّ مِنْ فَصْلَيْ الشِّتَاءِ وَالصَّيْفِ ( وَيَزِيدُ فِي الشِّتَاءِ ) عَلَى ذَلِكَ فِي الْمَحَلِّ الْبَارِدِ ( جُبَّةً ) مَحْشُوَّةً أَوْ نَحْوَهَا فَأَكْثَرَ بِحَسَبِ الْحَاجَةِ ( وَجِنْسُهَا ) أَيْ الْكِسْوَةِ ( قُطْنٌ ) لِأَنَّهُ لِبَاسُ أَهْلِ الدِّينِ وَمَا زَادَ عَلَيْهِ تَرَفُّهٌ وَرُعُونَةٌ فَعَلَى مُوسِرٍ لَيِّنُهُ وَمُعْسِرٍ خَشِنُهُ ، وَمُتَوَسِّطٍ مُتَوَسِّطُهُ ( فَإِنْ جَرَتْ( 35 / 390 )
