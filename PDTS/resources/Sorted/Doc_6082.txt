وَشَبَّهَ فِي اللُّزُومِ فَقَالَ ( كَ ) كِرَاءِ ( وَجِيبَةٍ ) بِفَتْحِ الْوَاوِ ، أَيْ مُدَّةٍ مُعَيَّنَةٍ مُصَوَّرَةٍ ( بِشَهْرِ كَذَا ) أَيْ بِتَسْمِيَةِ الشَّهْرِ أَوْ السَّنَةِ كَرَمَضَانَ وَسَنَةِ كَذَا كَسَنَةِ سَبْعَةٍ وَثَمَانِينَ بِتَقْدِيمِ السِّينِ ( أَوْ ) بِ ( هَذَا الشَّهْرِ ) أَوْ هَذِهِ السَّنَةِ ( أَوْ ) بِقَوْلِهِ أَكْتَرِيهَا ( أَشْهُرًا ) بِفَتْحٍ فَسُكُونٍ فَضَمٍّ جَمْعُ شَهْرٍ أَوْ سَنَتَيْنِ ( أَوْ ) بِقَوْلِهِ أَكْتَرِيهَا ( إلَى كَذَا ) أَيْ كَتَمَامِ سَنَةِ سَبْعَةٍ وَثَمَانِينَ .
