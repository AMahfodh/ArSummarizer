( وَإِنْ كَانَ ) مَنْ عَلَيْهِ الْحَقَّ ( مِمَّنْ لَا يُقْبَلُ قَوْلُهُ فِي الرَّدِّ ، أَوْ ) كَانَ مِمَّنْ هُوَ ( مُخْتَلِفٌ فِي قَبُولِ قَوْلِهِ ) فِي الرَّدِّ ( كَالْغَاصِبِ وَالْمُسْتَعِيرِ ) وَالْمُقْتَرِضِ ( وَالْمُرْتَهِنِ ) وَالْوَصِيِّ وَالْوَكِيلِ بِجُعْلٍ ، وَكُلُّ مَنْ قَبَضَ الْعَيْنَ لِحَظِّ نَفْسِهِ ( لَمْ يَلْزَمْهُ تَسْلِيمُ مَا قَبْلَهُ ) مِنْ دَيْنٍ أَوْ عَيْنٍ ( إلَّا بِالْإِشْهَادِ ) عَلَى الْقَابِضِ بِالْقَبْضِ لِحَدِيثِ { لَا ضَرَرَ وَلَا ضِرَارَ } .
