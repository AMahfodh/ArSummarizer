وَالثَّانِي : لَا يَرِثْنَ لِإِدْلَائِهِنَّ بِجَدٍّ فَأَشْبَهْنَ أُمَّ أَبِي الْأُمِّ ( وَضَابِطُهُ ) أَيْ إرْثِ الْجَدَّاتِ الْوَارِثَاتِ ، هُوَ ( كُلُّ جَدَّةٍ أَدْلَتْ ) أَيْ وَصَلَتْ ( بِمَحْضِ إنَاثٍ ) كَأُمِّ أُمِّ الْأُمِّ ( أَوْ ذُكُورٍ ) كَأُمِّ أَبِي الْأَبِ ( أَوْ إنَاثٍ إلَى ذُكُورٍ ) كَأُمِّ أُمِّ الْأَبِ ( تَرِثُ ، وَمَنْ أَدْلَتْ بِذَكَرٍ بَيْنَ أُنْثَيَيْنِ ) كَأُمِّ أَبِي الْأُمِّ ( فَلَا ) تَرِثُ كَمَا لَا يَرِثُ ذَلِكَ الذَّكَرُ ، وَحَكَى ابْنُ الْمُنْذِرِ فِيهِ الْإِجْمَاعَ .
