( فَعِدَّتُهَا بِوَضْعِ الْحَمْلِ ) أَيْ انْفِصَالِ كُلِّهِ حَتَّى ثَانِي تَوْأَمَيْنِ وَلَوْ بَعْدَ الْوَفَاةِ لِقَوْلِهِ تَعَالَى : { وَأُولَاتُ الْأَحْمَالِ أَجَلُهُنَّ أَنْ يَضَعْنَ حَمْلَهُنَّ } فَهُوَ مُقَيَّدٌ لِقَوْلِهِ تَعَالَى : { وَاَلَّذِينَ يُتَوَفَّوْنَ مِنْكُمْ وَيَذَرُونَ أَزْوَاجًا يَتَرَبَّصْنَ بِأَنْفُسِهِنَّ أَرْبَعَةَ أَشْهُرٍ وَعَشْرًا } وَلِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ لِسُبَيْعَةَ الْأَسْلَمِيَّةِ وَقَدْ وَضَعَتْ بَعْدَ مَوْتِ زَوْجِهَا بِنِصْفِ شَهْرٍ : { قَدْ حَلَلْت فَانْكِحِي مَنْ شِئْت } مُتَّفَقٌ عَلَيْهِ .
