( وَيُسْتَحَبُّ لَهُ أَنْ لَا يَنْفَرِدَ عَنْ أَبَوَيْهِ ) ؛ لِأَنَّهُ أَبْلَغُ فِي بِرِّهِمَا وَصِلَتِهِمَا ( وَإِنْ اسْتَوَى اثْنَانِ فَأَكْثَرَ فِيهَا ) كَأَخَّيْنِ فَأَكْثَرَ أَوْ أُخْتَيْنِ فَأَكْثَرَ ( أُقْرِعَ ) بَيْنَهُمَا أَوْ بَيْنَهُمْ ؛ لِأَنَّهُ لَا مُرَجِّحَ غَيْرُهَا ( مَا لَمْ يَبْلُغْ مَحْضُونٌ سَبْعًا ) أَيْ يَتِمَّ لَهُ سَبْعُ سِنِينَ ( وَلَوْ أُنْثَى فَيُخَيَّرُ ) بَيْنَهُمَا أَوْ بَيْنَهُمْ ؛ لِأَنَّهُ لَا يُمْكِنُ الْجَمْعُ وَلَا مَزِيَّةَ لِلْبَعْضِ ( وَالْأَحَقُّ مِنْ عَصَبَةِ ) مَحْضُونٍ .
