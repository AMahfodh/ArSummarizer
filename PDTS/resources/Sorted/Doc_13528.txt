( قَوْلُهُ : يَكُونُ حُرَّ الْأَصْلِ ) فِيهِ أَنَّ الْحُرِّيَّةَ الْأَصْلِيَّةَ لَا تُمَيَّزُ بِالْقُرْعَةِ وَهَذِهِ حُرِّيَّةٌ أَصْلِيَّةٌ فَلِمَ مُيِّزَتْ بِهَا قَالَهُ الْفَقِيهُ أَحْمَدُ بْنُ مُوسَى عُجَيْلٌ ا ه نَاشِرِيٌّ وَإِنَّمَا لَمْ تُمَيَّزْ بِالْقُرْعَةِ لِاحْتِمَالِ خُرُوجِ سَهْمِ الرِّقِّ عَلَى حُرِّ الْأَصْلِ ا ه وَقَدْ يُدْفَعُ هَذَا بِأَنَّ الْحُرِّيَّةَ فِيمَا لَوْ اشْتَبَهَ حُرٌّ بِرَقِيقٍ ثَابِتَةٌ فِي الْوَاقِعِ قَطْعًا فَلَمْ تُمَيَّزْ بِالْقُرْعَةِ لِاحْتِمَالِ خُرُوجِ سَهْمِ الرِّقِّ لِلْحُرِّ قَطْعًا وَهُنَا الْحُرِّيَّةُ إنَّمَا هِيَ بِإِقْرَارِهِ وَيُحْتَمَلُ كَذِبُهُ فَلَمْ تُسَاوِ الْحُرِّيَّةَ الْأَصْلِيَّةَ فِيمَا ذَكَرَ تَأَمَّلْ ( قَوْلُهُ : عَتَقَ إلَخْ ) الْأَوْلَى التَّعْبِيرُ بِالْحُرِّيَّةِ كَمَا مَرَّ ( قَوْلُهُ : انْحِصَارِ الْعِتْقِ فِيهِ ) أَيْ بِخُرُوجِ الْقُرْعَةِ لَهُ ( قَوْلُهُ : لِأَنَّهُ إشْكَالٌ إلَخْ ) أَيْ لِأَنَّ الْقُرْعَةَ لَا تَتَعَدَّى إلَى الْإِرْثِ ( قَوْلُهُ : وَلَك أَنْ تَقُولَ إلَخْ ) يُجَابُ عَنْهُ بِأَنَّ هَذَا الِاحْتِمَالَ بَعِيدٌ جِدًّا إذْ الْفَرْضُ أَنَّهُ لَا وَارِثَ أَوْ قَالَ : لَا أَعْلَمُ أَوْ نَفَى النَّسَبَ أَوْ أَنَّ الْقَائِفَ بَعُدَ أَوْ تَحَيَّرَ وَمَعَ ذَلِكَ يَبْعُدُ الْإِلْحَاقُ فَلَمْ يَعْتَبِرُوا احْتِمَالَ وُجُودِهِ م ر( 11 / 216 )
