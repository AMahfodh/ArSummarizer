( وَإِنْ ظَنَّ وَلِيُّ دَمٍ أَنَّهُ اقْتَصَّ فِي النَّفْسِ فَلَمْ يَكُنْ اسْتَوْفَى ، وَدَاوَاهُ ) أَيْ : الْجَانِيَ ( أَهْلُهُ حَتَّى بَرِئَ فَإِنْ شَاءَ الْوَلِيُّ دَفَعَ إلَيْهِ دِيَةَ فِعْلِهِ ) الَّذِي فَعَلَهُ بِهِ ( وَقَتَلَهُ وَإِلَّا ) يَدْفَعْ إلَيْهِ دِيَةَ فِعْلِهِ ( تَرَكَهُ ) فَلَا يَتَعَرَّضُ لَهُ قَالَ فِي الْفُرُوعِ : هَذَا رَأْيُ عُمَرَ وَعَلِيٍّ وَيَعْلَى بْنِ أُمَيَّةَ ذَكَرَهُ أَحْمَدُ .( 17 / 297 )
