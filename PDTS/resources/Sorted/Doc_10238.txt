( أَنْ نَتَمَلَّكَهَا ) أَيْ الشُّحُومَ الْمُحَرَّمَةَ عَلَيْهِمْ ( مِنْهُمْ بِمَا يَنْقُلُ الْمِلْكَ ) مِنْ بَيْعٍ أَوْ نَحْوِهِ لِمَا رَوَى عَبْدُ اللَّهِ بْنُ الْمُغَفَّلِ قَالَ { أَصَبْتُ مِنْ الشُّحُومِ يَوْمَ خَيْبَرَ فَالْتَزَمْتُهُ فَقُلْتُ : لَا أُعْطِي الْيَوْمَ أَحَدًا شَيْئًا فَالْتَفَتُّ فَإِذَا رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ مُتَبَسِّمًا } رَوَاهُ مُسْلِمٌ وَلِأَنَّهَا ذَكَاةٌ أَبَاحَتْ اللَّحْمَ فَأَبَاحَتْ الشَّحْمَ كَذَكَاةِ الْمُسْلِمِ وَكَذَبْحِ حَنَفِيِّ حَيَوَانًا فَتَبَيَّنَ حَامِلًا وَكَذَبْحِ مَالِكِيِّ فَرَسًا مُسَمِّيًا عَلَيْهَا ( وَالْأَوْلَى تَرْكُهَا ) أَيْ الشُّحُومِ الْمُحَرَّمَةِ عَلَيْهِمْ خُرُوجًا مِنْ خِلَافِ مَنْ حَرَّمَهُ كَأَبِي الْحَسَنِ التَّمِيمِيِّ وَالْقَاضِي ( وَلَا يَحِلُّ لِمُسْلِمٍ ) وَلَا لِغَيْرِهِ ( أَنْ يُطْعِمَهُمْ ) أَيْ الْيَهُودَ ( شَحْمًا مِنْ ذَبْحِنَا( 21 / 265 )
