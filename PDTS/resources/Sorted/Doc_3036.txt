( قَوْلُهُ : يَجِبُ فِيهِ مَا يَجِبُ فِي الْكَبِيرِ ) أَيْ وَإِنْ لَمْ يَظْهَرْ فِيهِ تَخْطِيطٌ وَلَا غَيْرُهُ حَيْثُ عُلِمَ أَنَّهُ آدَمِيٌّ ( قَوْلُهُ : وَالِاسْتِثْنَاءُ مِعْيَارُ الْعُمُومِ ) أَيْ دَلِيلُ الْعُمُومِ ( قَوْلُهُ : بِمَنْ عُلِمَتْ ) أَيْ مَعَ بِمَنْ عُلِمَتْ حَيَاتُهُ ( قَوْلُهُ : كَاخْتِلَاجٍ ) أَيْ وَلَوْ دُونَ أَرْبَعَةِ أَشْهُرٍ إنْ فُرِضَ .( 8 / 235 )
