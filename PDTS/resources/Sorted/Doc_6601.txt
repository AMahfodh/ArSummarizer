( وَ ) إنْ قَلَعَ أُذُنَهُ مِنْ مَحَلِّهَا وَرَدَّهَا الْمَجْنِيُّ عَلَيْهِ إلَيْهِ فَثَبَتَتْ فَ ( فِي ) أَخْذِ عَقْلِ ( الْأُذُنِ ) وَعَدَمِ رَدِّهِ إنْ كَانَ أَخَذَهُ ( إنْ ثَبَتَتْ ) الْأُذُنُ فِي مَحَلِّهَا بَعْدَ قَلْعِهَا وَرَدِّهَا إلَيْهِ فِي الْبَيَانِ وَهُوَ مَذْهَبُ الْإِمَامِ مَالِكٍ رَضِيَ اللَّهُ تَعَالَى عَنْهُ ، وَعَدَمِ لُزُومِهِ لِلْجَانِي وَإِنْ كَانَ أُخِذَ مِنْهُ رُدَّ لَهُ وَهُوَ رَأْيُ عَبْدِ الْحَقِّ أَنَّهُ مَذْهَبُ الْمُدَوَّنَةِ ( تَأْوِيلَانِ ) الْأَوَّلُ لِابْنِ رُشْدٍ وَالثَّانِي لِعَبْدِ الْحَقِّ( 19 / 214 )
