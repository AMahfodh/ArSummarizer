( وَ ) شَرَائِطُ ( إحْصَانِ الرَّجْمِ ) سَبْعَةٌ ( الْحُرِّيَّةُ وَالتَّكْلِيفُ ) عَقْلٌ وَبُلُوغٌ ( وَالْإِسْلَامُ وَالْوَطْءُ ) وَكَوْنُهُ ( بِنِكَاحٍ صَحِيحٍ ) حَالَ الدُّخُولِ ( وَ ) كَوْنُهُمَا ( بِصِفَةِ الْإِحْصَانِ ) الْمَذْكُورَةِ وَقْتَ الْوَطْءِ ، فَإِحْصَانُ كُلٍّ مِنْهُمَا شَرْطٌ لِصَيْرُورَةِ الْآخَرِ مُحْصَنًا ، فَلَوْ نَكَحَ أَمَةً أَوْ الْحَرَّةُ عَبْدًا فَلَا إحْصَانَ إلَّا أَنْ يَطَأَهَا بَعْدَ الْعِتْقِ فَيَحْصُلُ الْإِحْصَانُ بِهِ لَا بِمَا قَبْلَهُ ، حَتَّى لَوْ زَنَى ذِمِّيٌّ بِمُسْلِمَةٍ ثُمَّ أَسْلَمَ لَا يُرْجَمُ بَلْ يُجْلَدُ .
