مَنْزِلَتُهُمْ مِنْهُ ) بِلَا سَبْقٍ كَأَوْلَادِهِ وَكَإِخْوَتِهِ الْمُتَفَرِّقِينَ الَّذِينَ لَا وَاسِطَةَ بَيْنَهُ وَبَيْنَهُمْ ( فَنَصِيبُهُ لَهُمْ ) كَإِرْثِهِمْ مِنْهُ ، لَكِنْ هُنَا ( ذَكَرَ كَأُنْثَى ) لِأَنَّهُمْ يَرِثُونَ بِالرَّحِمِ ، فَاسْتَوَى ذَكَرُهُمْ وَأُنْثَاهُمْ كَوَلَدِ الْأُمِّ ( فَبِنْتُ أُخْتٍ وَابْنٌ وَبِنْتٌ ) ( لِ ) أُخْتٍ ( أُخْرَى ، لِ ) بِنْتِ الْأُخْتِ ( الْأُولَى النِّصْفُ ) لِأَنَّهُ إرْثُ أُمِّهَا فَرْضًا وَرَدًّا ( وَ لِ ) بِنْتِ الْأُخْتِ ( الْأُخْرَى وَأَخِيهَا النِّصْفُ ) لِأَنَّهُ إرْثُ أُمِّهَا حَيْثُ اسْتَوَتْ الْأُخْتَانِ فِي كَوْنِهِمَا لِأَبَوَيْنِ أَوْ لِأَبٍ أَوْ لِأُمٍّ ( بِالسَّوِيَّةِ ) بَيْنَ الْأُخْتِ وَأَخِيهَا ، فَتَصِحُّ مِنْ أَرْبَعَةٍ .
