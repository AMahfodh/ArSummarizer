و ( لَا ) يَصِحُّ ( تَبَرُّعُ مَأْذُونٍ لَهُ بِدَرَاهِمَ وَكِسْوَةٍ وَنَحْوِهِمَا ) كَكِتَابٍ ؛ لِأَنَّهُ لَيْسَ مِنْ التِّجَارَةِ وَلَا يَحْتَاجُ إلَيْهِ فَلَمْ يَتَنَاوَلْ الْإِذْنُ ( وَلَهُ ) أَيْ الرَّقِيقِ الْمَأْذُونِ لَهُ ( هَدِيَّةُ مَأْكُولٍ وَإِعَارَةٌ دَابَّةٍ وَعَمَلُ دَعْوَةٍ وَنَحْوَهُ ) كَصَدَقَةٍ بِيَسِيرٍ ( بِلَا إسْرَافٍ ) فِي الْكُلِّ ؛ لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ كَانَ يُجِيبُ دَعْوَةَ الْمَمْلُوكِ وَعَنْ أَبِي سَعِيدٍ مَوْلَى أَبِي أَسِيد أَنَّهُ تَزَوَّجَ فَحَضَرَ دَعَوْتَهُ جَمَاعَةٌ مِنْ الصَّحَابَةِ مِنْهُمْ ابْنُ مَسْعُودٍ وَأَبُو حُذَيْفَةَ فَأَمَّهُمْ وَهُوَ يَوْمَئِذٍ عَبْدٌ رَوَاهُ صَالِحٌ فِي مَسَائِلِهِ ، وَلِجَرَيَانِ عَادَةِ التُّجَّارِ بِهِ فِيمَا بَيْنَهُمْ فَيَدْخُلُ فِي عُمُومِ الْإِذْنِ ( وَلِ ) رَقِيقٍ ( غَيْرِ مَأْذُونٍ لَهُ ) فِي تِجَارَةٍ ( أَنْ يَتَصَدَّقَ مِنْ قُوتِهِ بِمَا لَا يَضُرُّ بِهِ كَرَغِيفٍ وَنَحْوِهِ ) كَفَلْسٍ وَبَيْضَةٍ لِجَرَيَانِ الْعَادَةِ بِالْمُسَامَحَةِ فِيهِ( 5 / 455 )
