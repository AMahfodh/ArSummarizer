الْعُقُودِ بِالْأَلْفَاظِ ( قَوْلُهُ : وَهُوَ مَا اقْتَضَاهُ كَلَامُ الرَّوْضَةِ ) مُعْتَمَدٌ ( قَوْلُهُ : إنْ كَانَ بِالثَّمَنِ إلَخْ ) ضَعِيفٌ ( قَوْلُهُ : وَقَضِيَّةُ الْعِلَّةِ ) وَهِيَ قَوْلُهُ لِأَنَّ كُلًّا مِنْهَا عَقْدٌ إلَخْ ( قَوْلُهُ : فَإِنَّهَا صَحِيحَةٌ ) أَيْ وَلَوْ بِأَكْثَرَ مِنْ الْأُجْرَةِ الْأُولَى أَوْ بِغَيْرِ جِنْسِهَا أَوْ صِفَتِهَا ( قَوْلُهُ : فَلَمْ يُؤَثِّرْ فِيهِ عَدَمُ قَبْضِهَا ) قَضِيَّتُهُ أَنَّ مِثْلَ الْمَبِيعِ الصَّدَاقُ وَعِوَضُ الْخُلْعِ وَغَيْرُهُمَا مِنْ كُلِّ مَا مُلِكَ بِعَقْدٍ مِنْ الْأَعْيَانِ وَهُوَ ظَاهِرٌ .
