( قَوْلُهُ : عَلَيْهِ ) أَيْ مُنِعَ بَيْعُ اللَّحْمِ بِالْحَيَوَانِ ( قَوْلُهُ : أَنَّهُ لَا فَرْقَ ) لَعَلَّ الْمُرَادَ بَيْنَ مُرْسَلِهِ وَمُرْسَلِ غَيْرِهِ ا ه سم ( قَوْلُهُ : وَبِأَنَّ أَبَا بَكْرٍ قَالَ ) مَقُولُهُ لَا يَصْلُحُ هَذَا ( وَقَوْلُهُ : وَقَدْ نُحِرَتْ إلَخْ ) جُمْلَةٌ مُعْتَرِضَةٌ ا ه كُرْدِيٌّ( 17 / 37 )
