( وَ ) إنْ حَدَثَ مَانِعٌ مِنْ كُفْرٍ وَفِسْقٍ وَغَيْرِهِمَا ( بَعْدَهُ ) أَيْ : الْحُكْمِ وَقَبْلَ اسْتِيفَاءِ مَحْكُومٍ بِهِ ( يُسْتَوْفَى مَالٌ ) حُكِمَ بِهِ ( لِأَحَدٍ مُطْلَقًا ) أَيْ لِلَّهِ أَوْ لِآدَمِيٍّ كَحَدِّ قَذْفٍ ( وَلَا قَوَدَ ) ؛ لِأَنَّهُ إتْلَافُ مَا لَا يُمْكِنُ تَلَافِيَهُ ، ( وَتُقْبَلُ شَهَادَةُ الشَّخْصِ عَلَى فِعْلِ نَفْسِهِ كَحَاكِمٍ عَلَى حُكْمِهِ بَعْدَ عَزْلِهِ وَقَاسِمٍ وَمُرْضِعَةٍ عَلَى قِسْمَتِهِ وَإِرْضَاعِهَا وَلَوْ بِأُجْرَةٍ ) ؛ لِأَنَّ كُلًّا مِنْهُمْ يَشْهَدُ لِغَيْرِهِ فَتُقْبَلُ كَمَا لَوْ شَهِدَ عَلَى فِعْلِ غَيْرِهِ ، وَلِحَدِيثِ عُقْبَةَ بْنِ الْحَارِثِ فِي الرَّضَاعِ وَقِيسَ عَلَيْهِ الْبَاقِي .( 12 / 395 )
