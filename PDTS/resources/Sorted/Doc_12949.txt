( وَمَنْ خَلَّفَ أُمًّا مُزَوَّجَةً ) بِغَيْرِ أَبِيهِ ( وَ ) خَلَّفَ ( وَرَثَةً لَا تَحْجُبُ وَلَدَهَا ) أَيْ : الْأُمُّ بِأَنْ لَمْ يُخَلِّفْ وَلَدًا وَلَا وَلَدَ ابْنٍ وَلَا أَبًا وَلَا جَدًّا ( لَمْ تُوطَأْ ) الْأُمُّ ( حَتَّى تُسْتَبْرَأَ لِيُعْلَمَ أَحَامِلٌ ) هِيَ حِينَ مَوْتِ وَلَدِهَا فَيَرِثُ مِنْهُ حَمْلُهَا ( أَوْ لَا ؟ ) وَكَذَا حُرَّةٌ تَحْتَ عَبْدٍ وَطِئَهَا وَلَهُ أَخٌ حُرٌّ فَمَاتَ أَخُوهُ الْحُرُّ فَيُمْنَعُ أَخُوهُ مِنْ وَطْءِ زَوْجَتِهِ حَتَّى يَتَبَيَّنَ أَهِيَ حَامِلٌ أَمْ لَا لِيَرِثَ الْحَمْلُ مِنْ عَمِّهِ ( فَإِنْ وُطِئَتْ ) مَنْ وَجَبَ اسْتِبْرَاؤُهَا لِذَلِكَ ( وَلَمْ تُسْتَبْرَأْ فَأَتَتْ بِهِ ) أَيْ : الْوَلَدِ ( بَعْدَ نِصْفِ سَنَةٍ مِنْ وَطْئِهِ لَمْ يَرِثْهُ ) أَيْ : الْمَيِّتَ لِاحْتِمَالِ حُدُوثِهِ بَعْدَ مَوْتِهِ أَوْ إنْ أَتَتْ بِهِ لِدُونِ نِصْفِ سَنَةٍ مِنْ مَوْتِهِ وَرِثَهُ وَكَذَا إنْ كَفَّ عَنْ وَطْئِهَا وَأَتَتْ بِهِ لِأَرْبَعِ سِنِينَ فَأَقَلُّ ، ؛ لِأَنَّ الظَّاهِرَ أَنَّهَا كَانَتْ حَامِلًا بِهِ حَالَ الْمَوْتِ( 8 / 31 )
