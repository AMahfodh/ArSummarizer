( قَوْلُهُ نَعَمْ يَظْهَرُ حَمْلُهُ ) أَيْ الْغَصْبِ ( قَوْلُهُ : عَلَى مَا إذَا ادَّعَى وُقُوعَهُ فِي خَلْوَةٍ ) أَيْ فِي مَحَلٍّ لَيْسَ فِيهِ أَحَدٌ ( قَوْلُهُ وَإِلَّا طُولِبَ بِبَيِّنَةٍ عَلَيْهِ ) مُعْتَمَدٌ ( قَوْلُهُ أَنَّهُ لَا يَعْلَمُهُ ) أَيْ فَلَا يُكَلَّفُ الْحَلِفُ أَنَّهَا مَا تَلِفَتْ .( 19 / 487 )
