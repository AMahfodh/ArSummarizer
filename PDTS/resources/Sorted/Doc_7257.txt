( وَمَتَى رَجَعَ بَائِعٌ ) فَاعْتَرَفَ بِعِتْقِهِ الْمَشْهُودِ بِهِ عَلَيْهِ مَعَ رَدِّ الشَّهَادَةِ ( رَدَّ ) الْبَائِعُ ( مَا أَخَذَ ) مِنْهُمَا عَلَى أَنَّهُ ثَمَنٌ وُجُوبًا ؛ لِاعْتِرَافِهِ بِأَنَّ قَبْضَهُ بِغَيْرِ حَقٍّ ( وَاخْتَصَّ بِإِرْثِهِ ) بِالْوَلَاءِ ؛ لِأَنَّهُ لَا مُنَازِعَ لَهُ فِيهِ حَيْثُ بَقِيَ الشَّاهِدَانِ عَلَى شَهَادَتِهِمَا ( وَيُوقَفُ ) إرْثُهُ ( إنْ رَجَعَ الْكُلُّ ) ؛ أَيْ : الشَّاهِدَانِ عَنْ شَهَادَتِهِمَا بِعِتْقِهِ ، وَرَجَعَ الْبَائِعُ عَنْ إنْكَارِهِ الْعِتْقَ بَعْدَ بَيْعِهِ ( حَتَّى يَصْطَلِحُوا ) عَلَيْهِ ؛ لِأَنَّهُ لَا مُرَجِّحَ لِأَحَدِهِمْ .
