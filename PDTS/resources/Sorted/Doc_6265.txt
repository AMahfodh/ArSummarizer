( وَنَافِي الْإِسْلَامِ ) كُلِّهِ أَوْ بَعْضِهِ كَنَافِي بِعْثَةِ مُحَمَّدٍ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ( مُخْطِئٌ آثِمٌ كَافِرٌ ) ؛ لِأَنَّهُ لَمْ يُصَادِفْ الْحَقَّ ( وَقَالَ الْجَاحِظُ وَالْعَنْبَرِيُّ لَا يَأْثَمُ الْمُجْتَهِدُ ) فِي الْعَقْلِيَّاتِ الْمُخْطِئُ فِيهَا لِلِاجْتِهَادِ ( قِيلَ مُطْلَقًا ، وَقِيلَ إنْ كَانَ مُسْلِمًا ) فَهُوَ عِنْدَهُمَا مُخْطِئٌ غَيْرُ آثِمٍ ( وَقِيلَ زَادَ الْعَنْبَرِيُّ ) عَلَى نَفْيِ الْإِثْمِ ( كُلٌّ ) مِنْ الْمُجْتَهِدَيْنِ فِيهَا ( مُصِيبٌ ) وَقَدْ حُكِيَ الْإِجْمَاعُ عَلَى خِلَافِ قَوْلِهِمَا قَبْلَ ظُهُورِهِمَا .
