وَلَوْ بِإِذْنِهِ حَيْثُ لَمْ يُعَيِّنْ لَهُ الدَّوَاءَ أَخْذًا مِمَّا بَعْدَهَا ( قَوْلُهُ : فَإِنْ كَانَ بِمُوحٍ ) بِمُوَحَّدَةٍ ، وَآخِرُهُ حَاءٌ مُهْمَلَةٌ أَيْ قَاتِلٌ سَرِيعًا ( قَوْلُهُ : لَوْ كَحَلَ ) هُوَ بِالتَّخْفِيفِ قَوْلُهُ : خِيَاطَةُ جُرْحِهِ ) أَيْ فَإِنْ أَذِنَ لَهُ فِي خِيَاطَتِهِ عَلَى وَجْهٍ مَخْصُوصٍ فَهَدَرٌ وَإِلَّا فَعَلَى عَاقِلَتِهِ ( قَوْلُهُ : فَالْقَوَدُ ) أَيْ عَلَى مَنْ خَاطَ حَيْثُ عَلِمَ بِحَالِ الْجُرْحِ وَتَعَمَّدَ ( قَوْلُهُ : فَنِصْفٌ دِيَةٍ ) أَيْ عَلَى مَنْ خَاطَ ( قَوْلُهُ : وَإِنْ خَاطَهُ ) أَيْ بِنَفْسِهِ أَوْ مَأْذُونِهِ( 24 / 311 )
