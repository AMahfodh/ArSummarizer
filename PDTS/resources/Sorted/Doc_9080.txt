قَالَ الشَّيْخُ تَقِيُّ الدِّينِ : وَيُحْكَمُ بِإِسْلَامِهِ وَأَنَّهُ يَسْرِي كَالْعِتْقِ أَيْ لَوْ كَانَتْ كَافِرَةً ( وَيَصِحُّ قَوْلُهُ ) أَيْ السَّيِّدِ ( لِأَمَتِهِ : يَدُك أُمُّ وَلَدِي ) فَهُوَ كَقَوْلِهِ لَهَا : أَنْتِ أُمُّ وَلَدِي لِأَنَّ إقْرَارَهُ بِأَنَّ جُزْءًا مِنْهَا مُسْتَوْلِدٌ يُلْزِمُهُ الْإِقْرَارَ بِاسْتِيلَادِهَا كَقَوْلِهِ يَدُك حُرَّةٌ ( أَوْ ) أَيْ وَكَذَا قَوْلُهُ ( لِابْنِهَا ) أَيْ ابْنِ أَمَتِهِ ( يَدُك ابْنِي ) فَهُوَ إقْرَارٌ بِأَنَّهُ ابْنُهُ ، كَقَوْلِهِ : أَنْتَ ابْنِي وَإِنْ لَمْ يَقُلْ وَلَدْتِيهِ فِي مِلْكِي لَمْ تَصِرْ أُمَّ وَلَدٍ لَهُ إلَّا أَنْ تَدُلَّ قَرِينَةٌ عَلَى وِلَادَتِهَا لَهُ فِي مِلْكِهِ .( 8 / 301 )
