( قَوْلُهُ : كَشَرْطِ عَقْدٍ ) أَيْ فِي عَقْدٍ ( قَوْلُهُ : فَأُلْغِيَتْ وَحْدَهَا ) يُتَأَمَّلُ مَعْنَى إلْغَاءِ شَرْطِ الْخِيَارِ لِلْمَضْمُونِ لَهُ فَإِنَّهُ صَاحِبُ الْحَقِّ وَمُتَمَكِّنٌ مِنْ الْإِبْرَاءِ مَتَى شَاءَ فَاشْتِرَاطُ الْخِيَارِ لَهُ تَصْرِيحٌ بِمُقْتَضَى الْعَقْدِ ، وَيُمْكِنُ أَنْ يُجَابَ بِأَنَّ مَعْنَى إلْغَائِهَا أَنَّهُ لَا يَتَرَتَّبُ عَلَيْهَا شَيْءٌ يَزِيدُ عَلَى مُقْتَضَى الْعَقْدِ ( قَوْلُهُ : وَمَا عُورِضَ بِهِ ) أَيْ قَوْلُهُ كَمَا قَالَ وَالزَّرْكَشِيُّ إلَخْ ( قَوْلُهُ : فِي دَعْوَى الصِّحَّةِ ) أَيْ مِنْ قَوْلِهِ صَحَّتْ الْكَفَالَةُ ( قَوْلُهُ : مِنْ أَرْضٍ ) أَيْ الدَّيْنِ( 15 / 168 )
