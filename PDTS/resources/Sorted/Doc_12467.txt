( وَإِنْ خَلَا ) الزَّوْجُ ( بِهَا وَهِيَ مُطَاوِعَةٌ وَلَوْ لَمْ يَمَسَّهَا ) مَعَ عِلْمِهِ بِهَا ( وَلَوْ ) كَانَتْ الْخَلْوَةُ ( فِي نِكَاحٍ فَاسِدٍ فَعَلَيْهَا الْعِدَّةُ سَوَاءٌ كَانَ بِهِمَا ) أَيْ الزَّوْجَيْنِ مَانِعٌ ( أَوْ ) كَانَ ( بِأَحَدِهِمَا مَانِعٌ مِنْ الْوَطْءِ ) حِسِّيٍّ أَوْ شَرْعِيٍّ ( كَإِحْرَامٍ وَصِيَامٍ وَحَيْضٍ وَنِفَاسٍ وَمَرَضٍ وَجَبٍّ وَعُنَّةٍ وَرَتْقٍ وَظِهَارٍ وَإِيلَاءٍ وَاعْتِكَافٍ أَوْ لَمْ يَكُنْ ) لِمَا رَوَى الْأَثْرَمُ عَنْ زُرَارَةَ بْنِ أَوْفَى قَالَ : قَضَى الْخُلَفَاءُ الرَّاشِدُونَ أَنَّ مَنْ أَغْلَقَ بَابًا أَوْ أَرْخَى سِتْرًا ، فَقَدْ وَجَبَ الْمَهْرُ وَوَجَبَتْ الْعِدَّةُ وَهَذِهِ قَضِيَّةٌ اشْتَهَرَتْ وَلَمْ تُنْكَرْ ، فَكَانَتْ كَالْإِجْمَاعِ وَضَعَّفَ أَحْمَدُ مَا رُوِيَ خِلَافُهُ وَلِأَنَّهُ عَقْدٌ عَلَى الْمَنَافِعِ فَالتَّمْكِينُ مِنْهُ يَجْرِي مَجْرَى الِاسْتِيفَاءِ فِي الْأَحْكَامِ كَعَقْدِ الْإِجَارَةِ ، وَالْآيَةُ مَخْصُوصَةٌ بِمَا ذَكَرْنَاهُ ، وَالْحُكْمُ مُعَلَّقٌ عَلَى الْخَلْوَةِ الَّتِي هِيَ مَظِنَّةُ الْإِصَابَةِ دُونَ حَقِيقَتِهَا فَلَمْ تُؤَثِّرْ ، .( 19 / 131 )
