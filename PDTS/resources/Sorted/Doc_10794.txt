وَهُوَ مُتَّجِهٌ ؛ ( كَرَهْنٍ ) فِيمَا سَبَقَ تَفْصِيلُهُ ، وَإِنْ مَاتَ فَكَفَنُهُ وَمُؤْنَةُ تَجْهِيزِهِ ؛ كَذَلِكَ ( وَإِنْ انْهَدَمَتْ مَرْهُونَةٌ ، فَعَمَرَهَا مُرْتَهِنٌ بِلَا إذْنِ ) رَاهِنٍ ؛ ( لَمْ يَرْجِعْ ) الْمُرْتَهِنُ بِمَا أَنْفَقَهُ فِي عِمَارَتِهَا ؛ لِأَنَّهُ لَيْسَ بِوَاجِبٍ عَلَى الرَّاهِنِ ، بِخِلَافِ نَفَقَةِ الْحَيَوَانِ - ( وَلَوْ نَوَاهُ ) ؛ أَيْ : نَوَى الْمُرْتَهِنُ الرُّجُوعَ - ( لَكِنْ لَهُ ) ؛ أَيْ : الْمُرْتَهِنِ - إذَا عَمَرَ - ( أَخْذُ أَعْيَانِ آلَتِهِ ) الَّتِي عَمَرَ بِهَا ؛ لِأَنَّهَا عَيْنُ مَالِهِ مَا لَمْ تَخْرُجْ عَنْ مِلْكِهِ ( فَقَطْ ) ؛ أَيْ : دُونَ ثَمَنِ مَاءٍ وَرَمَادٍ وَطِينٍ وَجِصٍّ وَنَوْرَةٍ وَأُجْرَةِ مُعَمِّرِينَ ، وَكَذَا مُسْتَأْجِرٌ وَوَدِيعٌ .( 8 / 410 )
