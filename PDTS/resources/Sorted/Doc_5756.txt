( فَرْعٌ ) لَوْ ( سَبَقَ أَحَدُهُمَا ) عِبَارَةُ الْأَصْلِ وَسَطَ ( الْمَيْدَانِ وَالثَّانِي آخِرَهُ فَالسَّابِقُ الثَّانِي ) ؛ لِأَنَّ الْعِبْرَةَ بِآخِرِهِ ( وَإِنْ عَثَرَ أَحَدُهُمَا ) أَيْ أَحَدُ الْمَرْكُوبَيْنِ ( أَوْ وَقَفَ ) بَعْدَ مَا جَرَى ( لِمَرَضٍ ) أَوْ نَحْوِهِ ( فَسُبِقَ فَلَا سَبْقٌ أَوْ ) وَقَفَ ( بِلَا عِلَّةٍ فَمَسْبُوقٌ لَا ) إنْ وَقَفَ ( قَبْلَ أَنْ يَجْرِيَ ) فَلَيْسَ مَسْبُوقًا سَوَاءٌ أَوَقَفَ لِمَرَضٍ أَمْ غَيْرِهِ( 21 / 280 )
