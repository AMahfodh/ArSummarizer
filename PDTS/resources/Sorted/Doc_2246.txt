وَالْمُخْتَارُ ) لِلَّخْمِيِّ مِنْ الْخِلَافِ ( أَنَّ ) الرَّجُلَ ( الْمُكْرَهَ ) بِفَتْحِ الرَّاءِ عَلَى وَطْءِ مَنْ لَا يَحِلُّ لَهُ وَطْؤُهَا بِخَوْفِ مُؤْلِمٍ مِنْ قَتْلٍ أَوْ ضَرْبٍ ، وَخَبَرُ أَنَّ الْمُكْرَهَ ( كَذَلِكَ ) أَيْ الْمَذْكُورَ فِيمَا تَقَدَّمَ فِي عَدَمِ حَدِّهِ ( وَالْأَكْثَرُ ) مِنْ أَهْلِ الْمَذْهَبِ ( عَلَى خِلَافِهِ ) أَيْ كَوْنِ الْمُكْرَهِ كَذَلِكَ وَهُوَ حَدُّهُ .
