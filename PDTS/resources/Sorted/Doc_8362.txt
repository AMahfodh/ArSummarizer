( وَيَضْمَنُ ) مُتَعَدٍّ ( مَا فِي وَثِيقَةٍ ) مِنْ نَحْوِ دَيْنٍ ( أَتْلَفَهَا إنْ تَعَذَّرَ ثُبُوتُ الْحَقِّ بِدُونِهَا ) وَكَذَا لَوْ تَلِفَتْ بِتَعَدِّيهِ ( وَهِيَ كَالْكَفَالَةِ تَقْضِي إحْضَارَ الْمَكْفُولِ أَوْ ضَمَانَ مَا عَلَيْهِ وَيَتَّجِهُ وَعَلَى قِيَاسِهِ ) ؛ أَيْ : ضَمَانِ مَا فِي الْوَثِيقَةِ لَوْ أَتْلَفَ مُتَعَدٍّ ( حُجَّةٌ ) فِيهَا ( وَظِيفَةٌ ) لِغَيْرِهِ ؛ فَتَعَذَّرَ اسْتِيفَاءُ الْوَظِيفَةِ بِدُونِ إحْضَارِ الْحُجَّةِ ؛ فَيَضْمَنُهَا مُتْلِفُهَا ؛ لِتَعَدِّيهِ ، فَإِنْ كَانَتْ عِنْدَهُ أَمَانَةً ، وَتَلِفَتْ بِلَا تَعَدٍّ وَلَا تَفْرِيطٍ ؛ لَمْ يَضْمَنْ ، وَهُوَ مُتَّجِهٌ .( 18 / 255 )
