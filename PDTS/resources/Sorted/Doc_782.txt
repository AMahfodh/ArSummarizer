( وَإِذَا تَبَايَعَ ذِمِّيَّانِ خَمْرًا بِثَمَنٍ بِذِمَّةٍ ، وَأُقْبِضَ ) - بِالْبِنَاءِ لِلْمَجْهُولِ - ( الْخَمْرُ ثُمَّ مَاتَ بَائِعُهُ ) ؛ أَيْ : الْخَمْرِ ، ( وَأَسْلَمَ وَارِثُهُ ؛ جَازَ لَهُ ) ؛ أَيْ : الْوَارِثِ ( أَخْذُ الثَّمَنِ نَصًّا ) ؛ لِمَا تَقَدَّمَ مِنْ حَدِيثِ عُمَرَ : { وَلُّوهُمْ بَيْعَهَا } .
