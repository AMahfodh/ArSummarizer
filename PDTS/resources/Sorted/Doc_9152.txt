( وَ ) سُنَّ أَيْضًا ( أَنْ يَجْلِسَ ) مُؤَذِّنٌ ( بَعْدَ أَذَانِ مَغْرِبٍ جِلْسَةً خَفِيفَةً ) بِقَدْرِ رَكْعَتَيْنِ ، ( ثُمَّ يُقِيمَ ) - أَيْ : الصَّلَاةَ - ( بِمَوْضِعِ أَذَانٍ وَ ) سُنَّ ( أَنْ تُؤَخَّرَ إقَامَةٌ ) عَنْ أَذَانٍ ( بِقَدْرِ فَرَاغٍ ) مِنْ قَضَاءِ ( حَاجَةٍ وَوُضُوءٍ وَصَلَاةِ رَكْعَتَيْنِ وَفَرَاغِ ) آكِلٍ مِنْ ( أَكْلٍ وَنَحْوِهِ ) كَشَارِبٍ مِنْ شُرْبٍ ، لِحَدِيثِ أُبَيِّ بْنِ كَعْبٍ { يَا بِلَالُ اجْعَلْ بَيْنَ أَذَانِكَ وَإِقَامَتِكَ نَفَسًا يَفْرُغُ الْآكِلُ مِنْ طَعَامِهِ ، وَالشَّارِبُ مِنْ شُرْبِهِ ، وَالْمُقْتَضِي إذَا دَخَلَ لِقَضَاءِ حَاجَتِهِ } رَوَاهُ أَبُو دَاوُد وَالتِّرْمِذِيُّ .
