( وَمَنْ اسْتَحَقَّ ) أَيْ تَجَدَّدَ لَهُ دَيْنٌ ( عَلَى غَرِيمِهِ مِثْلُ مَالِهِ عَلَيْهِ ) مِنْ دَيْنٍ جِنْسًا وَ ( قَدْرًا وَصِفَةً حَالَّيْنِ ) بِأَنْ اقْتَرَضَ زَيْدُ مِنْ عَمْرٍو دِينَارًا مِصْرِيًّا مَثَلًا ثُمَّ اشْتَرَى عَمْرٌو مِنْ زَيْدٍ شَيْئًا بِدِينَارٍ مِصْرِيٍّ حَالٍّ ( أَوْ مُؤَجَّلَيْنِ أَجَلًا وَاحِدًا ) كَثَمَنَيْنِ اتَّحَدَ أَجَلُهُمَا ( تَسَاقَطَا ) إنْ اسْتَوَيَا ( أَوْ ) سَقَطَ مِنْ الْأَكْثَرِ ( بِقَدْرِ الْأَقَلِّ ) إنْ تَفَاوَتَا قَدْرًا بِدُونِ تَرَاضٍ ؛ لِأَنَّهُ لَا فَائِدَةَ فِي أَخْذِ الدَّيْنِ مِنْ أَحَدِهِمَا ثُمَّ رَدَّهُ إلَيْهِ وَظَاهِرُهُ وَلَوْ لَمْ يَسْتَقِرّ وَصَرَّحُوا بِهِ فِي مَوَاضِعَ مِنْهَا مَا إذَا بَاعَ عَبْدَهُ لِزَوْجَتِهِ الْحُرَّةِ قَبْلَ الدُّخُولِ بِثَمَنٍ مِنْ جِنْسِ مَا سُمِّيَ لَهَا وَ ( لَا ) يَتَسَاقَطَانِ ( إذَا كَانَا ) أَيْ الدَّيْنَانِ دَيْنَ سَلَمٍ ( أَوْ ) كَانَ ( أَحَدُهُمَا دَيْنَ سَلَمٍ ) وَلَوْ تَرَاضَيَا ؛ لِأَنَّهُ تَصَرُّفٌ فِي دَيْنِ سَلَمٍ قَبْلَ قَبْضِهِ( 5 / 145 )
