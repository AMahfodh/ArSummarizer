( وَفُسِخَ ) بِضَمٍّ فَكَسْرٍ نِكَاحٌ ( مُوصًى ) بِضَمِّ الْمِيمِ وَفَتْحِ الصَّادِ الْمُهْمَلَةِ بِكَتْمِهِ مِنْ الزَّوْجِ وَالزَّوْجَةِ وَالْوَلِيِّ وَالشُّهُودِ وَسَائِرِ الْحَاضِرِينَ عَنْ كُلِّ أَحَدٍ أَبَدًا ، بَلْ ( وَإِنْ ) أَوْصَى الزَّوْجُ ( بِكَتْمِ شُهُودٍ ) فَقَطْ عَنْ كُلِّ أَحَدٍ أَوْ ( مِنْ امْرَأَةٍ ) لِلزَّوْجِ ( أَوْ ) مِنْ أَهْلِ ( مَنْزِلٍ ) فَقَطْ أَبَدًا ( أَوْ ) فِي ( أَيَّامٍ ) ثَلَاثَةٍ فَقَطْ .
