( فَإِنْ ضَاقَ نَيْلُهُ ) أَيْ الْحَاصِلِ مِنْهُ عَنْ اثْنَيْنِ تَسَابَقَا إلَيْهِ وَمِثْلُهُ فِي هَذَا الْبَاطِنُ الْآتِي ( قُدِّمَ السَّابِقُ ) مِنْهُمَا إلَيْهِ لِسَبْقِهِ وَإِنَّمَا يُقَدَّرُ ( بِقَدْرِ حَاجَتِهِ ) عُرْفًا فَيَأْخُذُ مَا تَقْتَضِيهِ عَادَةُ أَمْثَالِهِ وَيَبْطُلُ حَقُّهُ بِانْصِرَافِهِ وَإِنْ لَمْ يَأْخُذْ شَيْئًا ( فَإِنْ طَلَبَ زِيَادَةً ) عَلَى حَاجَتِهِ ( فَالْأَصَحُّ إزْعَاجُهُ ) لِشِدَّةِ الْحَاجَةِ إلَى الْمَعَادِنِ وَبِهِ فَارَقَ مَا مَرَّ فِي نَحْوِ مَقَاعِدِ الْأَسْوَاقِ ، وَمَحَلُّ الْخِلَافِ إنْ لَمْ يَضُرَّ الْغَيْرَ وَإِلَّا أُزْعِجَ جَزْمًا ( فَلَوْ جَاءَا ) إلَيْهِ ( مَعًا ) أَوْ جُهِلَ السَّابِقُ ( أُقْرِعَ ) بَيْنَهُمَا وَإِنْ كَانَ أَحَدُهُمَا غَنِيًّا ( فِي الْأَصَحِّ ) إذْ لَا مُرَجِّحَ وَإِنْ وَسِعَهُمَا اجْتَمَعَا ، وَلَيْسَ لِأَحَدِهِمَا أَنْ يَأْخُذَ أَكْثَرَ مِنْ الْآخَرِ إلَّا بِرِضَاهُ كَذَا فِي الْجَوَاهِرِ وَحُمِلَ عَلَى أَخْذِ الْأَكْثَرِ مِنْ الْبُقْعَةِ لَا النَّيْلِ فَلَهُ أَخْذُ الْأَكْثَرِ مِنْهُ
