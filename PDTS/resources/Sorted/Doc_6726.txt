( وَ ) قَوْلُهُ لِزَوْجَتِهِ ( أَنْتِ ) عَلَيَّ حَرَامٌ ( أَوْ الْحِلُّ ) عَلَيَّ حَرَامٌ ( أَوْ مَا أَحَلَّ اللَّهُ عَلَيَّ حَرَامٌ ظِهَارٌ وَلَوْ نَوَى ) بِهِ ( طَلَاقًا ) ؛ لِأَنَّهُ صَرِيحٌ فِي تَحْرِيمِهَا ( كَنِيَّتِهِ ) أَيْ : الطَّلَاقِ ( بِ ) قَوْلِهِ ( أَنْتِ عَلَيَّ كَظَهْرِ أُمِّي ) أَوْ أُخْتِي وَنَحْوَهُ ، وَقَوْلُهُ عَلَيَّ الْحَرَامُ أَوْ يَلْزَمُنِي الْحَرَامُ أَوْ الْحَرَامُ لَازِمٌ لِي مَعَ نِيَّةٍ أَوْ قَرِينَةٍ كَأَنْتِ عَلَيَّ حَرَامٌ قَدَّمَهُ ابْنُ رَزِينٍ وَصَوَّبَهُ فِي الْإِنْصَافِ .
