( قَوْلُهُ : لَكِنْ فِيهِ ) أَيْ : قَوْلِ الشَّيْخِ ( قَوْلُهُ : وَلَا تَفْرِيطُ أَبِي شُكَيْلٍ ) أَيْ : إنْ لَمْ يُرِدْ مِنْ قَوْلِهِ الْمَارِّ مَا ذَكَرَهُ الشَّارِحُ ( قَوْلُهُ : عَنْ الشَّيْخِ ) أَيْ أَبِي حَامِدٍ ( قَوْلُهُ : لِمَنْ قَدْ يَنْتَجِعُونَ إلَخْ ) نَعْتٌ ثَالِثٌ لِلْمَحَلِّ ( قَوْلُهُ : كَمَا يَأْتِي ) أَيْ : قُبَيْلَ قَوْلِ الْمَتْنِ وَلَوْ عَدِمَ إلَخْ ( قَوْلُهُ : وَهَذِهِ الْمَقَالَةُ ) أَيْ : مَا نَقَلَهُ الزَّرْكَشِيُّ عَنْ الشَّيْخِ وَابْنِ الصَّبَّاغِ ( قَوْلُهُ : يَنْقُلُ إلَيْهِمْ إلَخْ ) أَيْ : أَنْ يَنْقُلَ بَعْضَ الْمُعَدِّينَ إلَى بَعْضِهِمْ وَقَوْلُهُ : فِيهَا تَقْيِيدٌ إلَخْ خَبَرٌ وَمُبْتَدَأٌ ، وَالْجُمْلَةُ خَبَرٌ وَهَذِهِ الْمَقَالَةُ ( قَوْلُهُ : وَمَعَ ذَلِكَ )( 29 / 113 )
