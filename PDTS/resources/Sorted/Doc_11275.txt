يُخْرِجُ الْوَطْءَ وَالْعِتْقَ عَنْ كَوْنِهِمَا إجَازَةً ، وَقَدْ يُقَالُ إنَّهُ أَشَارَ إلَى أَنَّ مَا قُطِعَ فِيهِ بِأَنَّهُ فَسْخٌ مِنْ الْبَائِعِ قُطِعَ فِيهِ بِأَنَّهُ إجَازَةٌ مِنْ الْمُشْتَرِي وَمَا جَرَى فِيهِ الْخِلَافُ إذَا وَقَعَ مِنْ الْبَائِعِ جَرَى فِي مِثْلِهِ الْخِلَافُ إذَا وَقَعَ مِنْ الْمُشْتَرِي ( قَوْلُهُ : أَوْ كَانَتْ ) أَيْ التَّصَرُّفَاتُ ( قَوْلُهُ : مَعَهُ ) أَيْ الْبَائِعِ ( قَوْلُهُ : وَيُفَارِقُ ) أَيْ تَصَرُّفُ الْمُشْتَرِي ( قَوْلُهُ : مَا مَرَّ فِي الْبَائِعِ ) أَيْ حَيْثُ نَفَذَ وَالْخِيَارُ لَهُمَا وَإِنْ لَمْ يَأْذَنْ الْمُشْتَرِي ( قَوْلُهُ : وَبَانَ صِحَّتُهَا ) مِنْ الْمُشْتَرِي لَوْ قُلْنَا بِهِ ( قَوْلُهُ : مُسْقِطَةٌ لِفَسْخِهِ ) أَيْ الْبَائِعِ ( قَوْلُهُ : أَيْ إذَا لَمْ يَكُنْ ) خَبَرٌ قَوْلُهُ وَقَوْلُ الشَّارِحِ إلَخْ .( 12 / 118 )
