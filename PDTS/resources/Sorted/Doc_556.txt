( أَوْ ) أَيْ وَلَا تَنْفَسِخُ الْإِجَارَةُ بِظُهُورِ ( فِسْقِ مُسْتَأْجِرِ ) دَارٍ مَثَلًا فَلَا تَنْفَسِخُ الْإِجَارَةُ وَيَنْهَى عَنْ فِسْقِهِ ، فَإِنْ انْتَهَى عَنْهُ أَقَرَّ فِيهَا ( وَ ) إلَّا ( آجَرَ الْحَاكِمُ ) الدَّارَ مَثَلًا عَلَيْهِ لِغَيْرِهِ بَعْدَ إخْرَاجِهِ مِنْهَا ( إنْ لَمْ يَكُفَّ ) بِفَتْحٍ فَضَمٍّ مُثَقَّلًا عَنْ فِسْقِهِ .
