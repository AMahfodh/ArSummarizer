( وَيَجِبُ ) عَلَى الْمُسْتَعِيرِ ( رَدُّ ) الْعَارِيَّة ( بِطَلَبِ مَالِكٍ ) لَهُ بِالرَّدِّ ، وَلَوْ لَمْ يَنْقَضِ غَرَضُهُ مِنْهَا ، أَوْ يَمْضِي الْوَقْتُ ؛ لِأَنَّ الْإِذْنَ هُوَ الْمُسَلِّطُ لِحَبْسِ الْعَيْنِ - وَقَدْ انْقَطَعَ بِالطَّلَبِ - وَيَجِبُ الرَّدُّ أَيْضًا ( بِانْقِضَاءِ غَرَضٍ ) مِنْ الْعَيْنِ الْمُعَارَةِ ؛ لِأَنَّ الِانْتِفَاعَ هُوَ الْمُوجِبُ لِلْحَبْسِ - وَقَدْ زَالَ - ( أَوْ انْتِهَاءِ مُدَّةٍ ) إنْ كَانَتْ الْعَارِيَّةُ مُؤَقَّتَةً ؛ لِانْتِهَائِهَا ، ( أَوْ مَوْتِ أَحَدِهِمَا ) - أَيْ : الْمُعِيرِ وَالْمُسْتَعِيرِ - لِبُطْلَانِ الْعَارِيَّةِ بِذَلِكَ ؛ لِأَنَّهَا عَقْدٌ جَائِزٌ مِنْ الطَّرَفَيْنِ ، ( فَإِنْ أَخَّرَ ) الْمُسْتَعِيرُ الرَّدَّ فِيمَا ذَكَرَ فَتَلِفَتْ الْعَارِيَّةُ ؛ ( ضَمِنَ ) قِيمَتَهَا ( مَعَ أُجْرَةِ مِثْلِهَا ) لِمُدَّةِ تَأْخِيرِهِ ، ( وَعَلَيْهِ ) - أَيْ : الْمُسْتَعِيرِ ( مُؤْنَةُ رَدِّ ) الْعَارِيَّةِ إلَى مَالِكِهَا .
