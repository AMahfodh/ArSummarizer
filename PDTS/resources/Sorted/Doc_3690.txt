( وَلَا ) تُسْمَعُ دَعْوَى ( حِسْبَةٍ بِحَقِّ اللَّهِ تَعَالَى كَعِبَادَةٍ ) مِنْ صَلَاةٍ وَزَكَاةٍ وَحَجٍّ وَنَحْوِهَا ( وَحَدِّ ) زِنًا أَوْ شُرْبٍ ( وَكَفَّارَةٍ وَنَذْرٍ وَنَحْوِهِ ) كَجَزَاءِ صَيْدٍ قَتَلَهُ مُحْرِمًا أَوْ فِي الْحَرَمِ ، ( وَتُسْمَعُ ) بِلَا دَعْوَى ( بَيِّنَةٌ بِذَلِكَ وَبِعِتْقٍ وَلَوْ أَنْكَرَ مَعْتُوقٌ ) لِعِتْقِ الْمَشْهُودِ بِهِ لِحَقِّ اللَّهِ تَعَالَى ، وَكَذَا تُسْمَعُ بِطَلَاقٍ( 12 / 154 )
