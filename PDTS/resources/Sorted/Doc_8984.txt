( وَ ) يَجِبُ ( عَلَى رَبِّ أَصْلٍ فِعْلُ مَا يَحْفَظُهُ ) - أَيْ : الْأَصْلَ - ( كَسَدِّ حَائِطٍ ) وَتَحْصِيلِ سِيَاجٍ - وَهُوَ الشَّوْكُ - يُجْعَلُ عَلَى الْجُدُرِ لَيُحْفَظَ مِنْ الدُّخُولِ ، ( وَإِجْرَاءِ نَهْرٍ وَحَفْرِ بِئْرٍ وَثَمَنِ دُولَابٍ وَمَا يُدِيرُهُ ) - أَيْ : الدُّولَابَ - مِنْ آلَةٍ وَبَهَائِمَ ، ( وَشِرَاءِ مَاءٍ وَشِرَاءِ مَا يُلَقَّحُ بِهِ ) مِنْ طَلْعِ فُحَّالٍ وَيُسَمَّى الْكُثْرُ - بِضَمِّ الْكَافِ وَسُكُونِ الْمُثَلَّثَةِ وَفَتْحِهَا - ( وَتَحْصِيلِ زِبْلٍ وَسِبَاخٍ ) ؛ لِأَنَّ هَذَا كُلَّهُ لَيْسَ مِنْ الْعَمَلِ ، فَهُوَ عَلَى رَبِّ الْمَالِ .( 10 / 220 )
