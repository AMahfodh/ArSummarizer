( قَوْلُهُ : أَوْ يَكْفِي إلَخْ ) عَطْفٌ عَلَى قَوْلِهِ بِثُبُوتِ إلَخْ أَوْ عَلَى قَوْلِهِ الْعِبْرَةَ فِي الْبُلُوغِ إلَخْ بِاعْتِبَارِ الْمَعْنَى أَيْ وَيَتَرَدَّدُ النَّظَرُ فِي أَنَّهُ هَلْ يُشْتَرَطُ فِي الْبُلُوغِ ثُبُوتُ مَا ذُكِرَ بِالْبَيِّنَةِ أَوْ يَكْفِي إخْبَارٌ إلَخْ ( قَوْلُهُ : بِذَلِكَ ) مُتَعَلِّقٌ بِالْإِخْبَارِ ، وَقَوْلُهُ : كُلَّهُ أَيْ بِأَنَّ الْمَرْئِيَّ حَيْضٌ وَأَنَّهُ فِي زَمَنِ إلَخْ وَأَنَّهُ انْقَطَعَ إلَخْ ( قَوْلُهُ : الْأَوَّلُ ) أَيْ : اشْتِرَاطُ ثُبُوتِ تِلْكَ الثَّلَاثَةِ ( قَوْلُهُ : أَنَّ مَنْ صَدَّقَهَا ) أَيْ : ذَاتَ الدَّمِ ( قَوْلُهُ : فِي حَقِّهِ ) أَيْ : مَنْ صَدَّقَهَا( 35 / 4 )
