( فَرْعٌ ) لَوْ ( سَرَقَ ) مِنْ الْغَنِيمَةِ ( غَانِمٌ أَوْ وَلَدُهُ أَوْ وَالِدُهُ أَوْ عَبْدُهُ ) أَوْ سَيِّدُهُ ( قَدْرَ نَصِيبِهِ ) مِنْهَا ( رَدَّهُ ) إلَيْهَا ( فَإِنْ تَلِفَ فَبَدَلُهُ ) يَرُدُّهُ إلَيْهَا ( وَلَمْ يُقْطَعْ ) حُرًّا كَانَ أَوْ عَبْدًا ؛ لِأَنَّ لَهُ حَقًّا فِيهَا ( وَكَذَا ) لَوْ سَرَقَ ( أَكْثَرَ ) مِنْ نَصِيبِهِ يَرُدُّهُ فَإِنْ تَلِفَ فَبَدَلُهُ وَلَا يُقْطَعُ وَيَنْبَغِي أَنْ يُسْقِطَ عَنْهُ قَدْرَ نَصِيبِهِ إنْ كَانَ الْغَانِمُونَ مَحْصُورِينَ كَمَا فِي الْمَهْرِ الْآتِي بَيَانُهُ فِي الْفَرْعِ الْآتِي ( وَلَوْ بَعْدَ إفْرَازِ الْخُمْسِ ) سَوَاءٌ أَسَرَقَ مِنْهُ أَمْ مِنْ الْأَخْمَاسِ الْأَرْبَعَةِ وَإِنْ بَلَغَ مَا سَرَقَهُ مِنْهَا نِصَابًا لِذَلِكَ .
