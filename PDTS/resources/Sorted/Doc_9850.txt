( قَوْلُهُ : لَكِنْ يَلْزَمُهُ ) أَيْ : السَّاعِيَ رِعَايَةُ مَصْلَحَةِ إلَخْ وَيُسَنُّ لِلْمَالِكِ إذَا كَانَ دَافِعًا اخْتِيَارَ الْأَنْفَعِ لَهُمْ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : أَخْذًا ) أَيْ : لِلْأَغْبَطِ لِلْجُبْرَانِ لِئَلَّا يُنَافِيَ مَا قَبْلَهُ ، وَيُمْكِنُ إرَادَتُهُ بِأَنْ فَوَّضَ الْمَالِكُ الْخِيرَةَ بَيْنَ أَخْذِ الشَّاتَيْنِ وَأَخْذِ الْعِشْرِينَ إلَيْهِ فَلَا تَنَافِيَ ، أَوْ الْمُرَادُ بِالْأَخْذِ طَلَبُهُ ، وَإِنْ لَمْ يَلْزَمْ الْمَالِكَ مُوَافَقَتُهُ شَوْبَرِيٌّ وَتَقَدَّمَ الْجَوَابُ الْأَخِيرُ عَنْ الْمُغْنِي وَالنِّهَايَةِ ( قَوْلُهُ : هَذَا مَا بَحَثَهُ الزَّرْكَشِيُّ ) أَيْ : وَأَقَرَّهُ الْأَسْنَى ( قَوْلُهُ : مُطْلَقًا ) أَيْ : وَافَقَهُ السَّاعِي أَوْ لَا ( قَوْلُهُ : وَمَحَلُّ الْخِلَافِ ) إلَى قَوْلِ( 12 / 6 )
