( وَإِنْ قَالَ لِامْرَأَتِهِ الَّتِي فِي حِبَالِهِ لَمْ تَزْنِي ) وَلَكِنْ لَيْسَ هَذَا الْوَلَدُ مِنِّي ( أَوْ ) قَالَ لَهَا ( لَمْ أَقْذِفْكِ وَلَكِنْ لَيْسَ هَذَا الْوَلَدُ مِنِّي فَهُوَ وَلَدُهُ فِي الْحُكْمِ ) لِأَنَّ الْوَلَدَ لِلْفِرَاشِ ، وَهِيَ فِرَاشُهُ ( وَلَا حَدَّ عَلَيْهِ ) لِأَنَّهُ لَمْ يَقْذِفْهَا بِالزِّنَا ( وَإِنْ قَالَ ) أَيْ لَيْسَ هَذَا الْوَلَدُ مِنِّي لِامْرَأَتِهِ ( بَعْد أَنْ أَبَانَهَا أَوْ قَالَهُ لِسُرِّيَّتِهِ فَشَهِدَتْ بِبَيِّنَةٍ ، وَتَكْفِي أَنَّهَا امْرَأَةٌ مُرْضِيَةٌ أَنَّهُ وُلِدَ عَلَى فِرَاشِهِ لَحِقَهُ : نَسَبُهُ ) إذْ الْوَلَدُ لِلْفِرَاشِ ( وَإِنْ قَالَ ) عَنْ وَلَد بِيَدِهَا ( مَا بِتَجْتَنِبُ وَإِنَّمَا التقطتيه أَوْ استعرتيه فَقَالَتْ : بَلْ هُوَ وَلَدِي مِنْك لَمْ يُقْبَلْ قَوْلُهَا ) عَلَيْهِ لِأَنَّ الْوِلَادَةَ يُمْكِنُ إقَامَةُ الْبَيِّنَةِ عَلَيْهَا ، وَالْأَصْلُ عَدَمُهُ ( وَلَا يَلْحَقهُ نَسَبُهُ إلَّا بِبَيِّنَةٍ ، وَتَكْفِي امْرَأَةٌ مُرْضِيَةٍ تَشْهَدُ بِوِلَادَتِهَا لَهُ ، فَإِذَا ثَبَتَتْ وِلَادَتُهَا ) لَهُ ( لَحِقَهُ نَسَبُهُ ) لِأَنَّهَا فِرَاشُهُ ، وَالْوَلَدُ لِلْفِرَاشِ ( وَكَذَلِكَ لَا تُقْبَلُ دَعْوَاهَا الْوِلَادَةُ ، فَإِذَا عَلَّقَ طَلَاقَهَا بِهَا ) لِإِمْكَانِ إقَامَةِ الْبَيِّنَةِ بِهَا وَتَقَدَّمَ أَنَّهَا تُقْبَلُ إذَا أَقَرَّ بِالْحَمْلِ عِنْدَ الْقَاضِي وَأَصْحَابِهِ وَجَزَمَ بِهِ فِي الْمُنْتَهَى فِي فَصْلِ تَعْلِيقِهِ بِالْحَمْلِ وَالْوِلَادَةِ ( وَلَا ) تُقْبَلُ ( دَعْوَى الْأَمَةِ لَهَا ) أَيْ لِلْوِلَادَةِ ( لِتَصِيرَ أُمَّ وَلَدٍ ) لِأَنَّهَا خِلَافُ الْأَصْلِ ( وَيُقْبَلُ قَوْلُهَا فِيهِ ) أَيْ فِي أَنَّهَا وَلَدَتْ ( لِتَنْقَضِي عِدَّتُهَا بِهِ ) لِأَنَّهَا أَمِينَةٌ عَلَى نَفْسِهَا فِي ذَلِكَ .( 19 / 85 )
