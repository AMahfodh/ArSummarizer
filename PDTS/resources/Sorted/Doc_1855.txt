( وَلَوْ أَوْصَى بِعِتْقِ شِقْصَيْنِ مِنْ عَبْدَيْنِ ) مُشْتَرَكَيْنِ ( وَتَكَمَّلَ عِتْقُهُمَا وَاتَّسَعَ الثُّلُثُ لَهُمَا كَمُلَا ) عَتَقَا ( وَإِنْ اتَّسَعَ لِتَكْمِيلِ وَاحِدٍ فَقَطْ أَقْرَعَ بَيْنَهُمَا فَيَعْتِقُ مَنْ قَرَعَ وَ ) يَعْتِقُ ( نَصِيبُهُ مِنْ الثَّانِي ) فَقَطْ ، وَقِيلَ يَعْتِقُ مِنْ كُلِّ وَاحِدٍ ثَلَاثَةُ أَرْبَاعِهِ ، وَالتَّصْرِيحُ بِالتَّرْجِيحِ مِنْ زِيَادَتِهِ .( 23 / 395 )
