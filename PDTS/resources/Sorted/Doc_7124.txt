( قَوْلُهُ : أَنْ يَكُونَ أَلْحَنَ ) أَيْ أَقْدَرَ قَوْلُهُ : بَلْ وَالْقَتْلُ ) وَمِثْلُهُ مَنْ عَرَفَتْ وُقُوعَ الطَّلَاقِ عَلَى زَوْجِهَا وَلَمْ يُمْكِنْهَا الْخَلَاصُ مِنْهُ ( قَوْلُهُ إنْ قَدَرَتْ عَلَيْهِ ) أَيْ وَلَوْ بِسُمٍّ إنْ تَعَيَّنَ طَرِيقًا ( قَوْلُهُ : فَكَالْأَوَّلِ ) أَيْ كَالْمُخَالِفِ لِلنَّصِّ الَّذِي يَنْقُضُهُ الْحَاكِمُ وَغَيْرُهُ ( قَوْلُهُ : نَعَمْ لَوْ قَضَى قَاضٍ ) كَانَ الْأَوْلَى لَهُ أَنْ يَقُولَ : وَمِمَّا يُنْقَضُ فِيهِ الْحُكْمُ لِمُخَالَفَتِهِ مَا مَرَّ مَا لَوْ حَكَمَ بِصِحَّةِ نِكَاحٍ إلَخْ .( 28 / 151 )
