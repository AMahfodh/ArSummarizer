( وَإِنْ أَقَرَّ ) الْمُدَّعَى عَلَيْهِ بِعَيْنٍ ( بِهَا لِغَائِبٍ ) عَنْ الْبَلَدِ ( أَوْ غَيْرِ مُكَلَّفٍ ) مِنْ صَغِيرٍ أَوْ مَجْنُونٍ ( وَلِلْمُدَّعِي بَيِّنَةٌ ) شَهِدَتْ بِأَنَّهَا مِلْكُهُ ( فَهِيَ ) ؛ أَيْ : الْعَيْنُ لَهُ ؛ لِتَرَجُّحِ جَانِبِهِ بِالْبَيِّنَةِ ( بِلَا يَمِينٍ ) اكْتِفَاءً بِالْبَيِّنَةِ ؛ لِخَبَرِ : { الْبَيِّنَةُ عَلَى الْمُدَّعِي وَالْيَمِينُ عَلَى مَنْ أَنْكَرَ } .
