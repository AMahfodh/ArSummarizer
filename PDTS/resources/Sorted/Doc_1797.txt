( وَلَهُ ) أَيْ : الْعَبْدِ ( إعْلَامُ سَيِّدِهِ الْعَدْلِ بِهَا إنْ أَمِنَهُ ) عَلَيْهَا ( وَإِلَّا ) يَأْمَنْ سَيِّدَهُ عَلَيْهَا ( لَزِمَ ) الْعَبْدَ ( سَتْرُهَا عَنْهُ ) أَيْ : عَنْ سَيِّدِهِ ؛ لِأَنَّهُ يَلْزَمُهُ حِفْظُهَا ، وَذَلِكَ وَسِيلَةٌ إلَيْهِ ، وَيُسَلِّمُهَا لِلْحَاكِمِ لِيُعَرِّفَهَا ثُمَّ يَدْفَعَهَا إلَى سَيِّدِهِ بِشَرْطِ الضَّمَانِ .( 14 / 232 )
