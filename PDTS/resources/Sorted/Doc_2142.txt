فَلِلْأُمِّ السُّدُسُ وَلِلْبِنْتِ النِّصْفُ وَمَا بَقِيَ لَهُمَا بِالتَّعْصِيبِ ) ؛ لِأَنَّهُمَا أُخْتَانِ مَعَ بِنْتٍ فَتَصِحُّ مِنْ سِتَّةٍ لِلْكُبْرَى اثْنَانِ وَلِلصُّغْرَى أَرْبَعَةٌ ( فَلَوْ مَاتَتْ الصُّغْرَى بَعْدَهَا ) أَيْ : بَعْدَ الْوُسْطَى ( فَأُمُّ أُمِّهَا أُخْتٌ لِأَبٍ فَلَهَا الثُّلُثَانِ ) النِّصْفُ ؛ لِأَنَّهَا أُخْتٌ لِأَبٍ وَالسُّدُسُ ؛ لِأَنَّهَا جَدَّةٌ ( وَمَا بَقِيَ ) فَهُوَ ( لِلْعَمِّ ) تَعْصِيبًا .
