( وَأَدْنَى مَا يَكْفِي ) مِنْ الْوَطْء حَتَّى تَحِلَّ لِمُطَلِّقِهَا ثَلَاثًا ( تَغْيِيبُ الْحَشَفَةِ ) فِي الْقُبُلِ مَعَ الِانْتِشَارِ ( وَإِنْ لَمْ يُنْزِلْ ) لِأَنَّ أَحْكَامَ الْوَطْءِ تَتَعَلَّقُ بِهِ ( فَإِنْ كَانَ ) الزَّوْجُ الثَّانِي ( مَجْبُوبًا قَدْ بَقِيَ مِنْ ذَكَرِهِ قَدْرُ الْحَشَفَةِ فَأَكْثَرُ فَأَوْلَجَهُ ) مَعَ الِانْتِشَارِ فِي قُبُلِهَا ( أَحَلَّهَا ) لِمُطَلِّقِهَا ثَلَاثًا ، لِأَنَّ ذَلِكَ بِمَنْزِلَةِ الْحَشَفَةِ مِنْ غَيْرِهِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَبْقَ مِنْ ذَكَرِهِ قَدْرُ الْحَشَفَةِ ، بَلْ دُونِهِ ( فَلَا ) يَحِلُّهَا إيلَاجُهُ لِأَنَّهُ بِمَنْزِلَةِ إيلَاجِ بَعْضِ الْحَشَفَةِ وَلَا تَتَعَلَّقُ بِهِ أَحْكَامُ الْوَطْءِ ( فَلَا يَحِلُّهَا ) أَيْ الْمُطَلَّقَةَ ثَلَاثًا وَطْءُ السَّيِّدِ إنْ كَانَتْ أَمَةً لِأَنَّهُ لَيْسَ بِزَوْجِ .
