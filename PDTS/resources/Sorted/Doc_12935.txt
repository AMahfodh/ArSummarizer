( قَوْلُهُ أَوْ التَّنْبِيهُ ) مُقَابِلٌ لِلتَّنْبِيهِ وَالْإِصْلَاحِ بِالْأَلْفَاظِ ( قَوْلُهُ أَوْ بِالْكِتَابَةِ ) مَعْطُوفٌ عَلَى بِكِتَابِهِ أَيْ أَوْ التَّنْبِيهِ عَلَى ذَلِكَ بِالْكِتَابَةِ فِي حَوَاشِي كِتَابِهِ ( قَوْلُهُ وَالتَّغْيِيرِ ) عَطْفُ تَفْسِيرٍ ( قَوْلُهُ بِالْكِتَابَةِ ) الْبَاءُ لِلتَّصْوِيرِ أَوْ أَرَادَ بِالْكِتَابَةِ الْمَكْتُوبَ ( قَوْلُهُ يَعْنِي أَلْفَاظَهُ ) أَيْ دَالَّ أَلْفَاظِهِ وَهِيَ النُّقُوشُ ؛ لِأَنَّ الَّذِي يُكْشَطُ هُوَ النُّقُوشُ ( قَوْلُهُ أَوْ يُزَادُ فِيهَا ) يُسْتَثْنَى مِنْهُ مَا فَاتَهُ مِنْ الْمَسَائِلِ مِمَّا بُيِّضَ لَهُ وَلَمْ يُكْمِلْهُ ، وَهُوَ قَوْلُهُ ، وَإِنْ ادَّعَتْ اسْتَكْرَاهَا عَلَى غَيْرِ لَائِقٍ بِلَا تَعَلُّقٍ حَدَثَ لَهُ وَمِنْ الْأَبْوَابِ مِمَّا بُيِّضَ لَهُ كَذَلِكَ وَلَمْ يَذْكُرْ ، وَهُوَ بَابُ الْمُقَاصَّةِ فَإِنَّ الْأَوَّلَ كَمَّلَهُ الْأَقْفَهْسِيُّ وَالثَّانِي أَلَّفَهُ بَهْرَامُ ( قَوْلُهُ وَرُبَّمَا ظَنَّ إلَخْ ) الْوَاوُ لِلتَّعْلِيلِ أَيْ ؛ لِأَنَّهُ رُبَّمَا ظَنَّ إلَخْ ( قَوْلُهُ فَمِنْ بَابِ تَوَاضُعِهِ ) أَيْ فَمِنْ بَابٍ هُوَ تَوَاضُعُهُ فَالْإِضَافَةُ لِلْبَيَانِ ( قَوْلُهُ مَعَ أَنَّ ) لِلتَّعْلِيلِ أَيْ ؛ لِأَنَّ( 1 / 238 )
