( وَإِنْ عَلِمَتْ بِالْعَيْبِ ) الَّذِي تَمْلِكُ بِهِ الْفَسْخَ ( بَعْدَ الْعَقْدِ أَوْ حَدَثَ بِهِ ) أَيْ : بِالزَّوْجِ الْعَيْبُ بَعْدَ الْعَقْدِ ( لَمْ يَمْلِكْ الْوَلِيُّ إجْبَارَهَا عَلَى الْفَسْخِ لِأَنَّ حَقَّهُ فِي النِّكَاحِ لَا فِي دَوَامِهِ ) لِأَنَّهَا لَوْ دَعَتْ وَلِيَّهَا أَنْ يُزَوِّجهَا بِعَبْدٍ لَمْ يَلْزَمهُ إجَابَتُهَا وَلَوْ عَتَقَتْ تَحْتَ عَبْدٍ لَمْ يَمْلِكْ إجْبَارَهَا عَلَى الْفَسْخِ .( 17 / 216 )
