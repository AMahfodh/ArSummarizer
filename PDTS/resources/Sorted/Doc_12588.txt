( وَاسْتَبَدَّ ) أَيْ اسْتَقَلَّ شَرِيكٌ ( آخِذُ قِرَاضٍ ) مِنْ أَجْنَبِيٍّ يَتَّجِرُ لَهُ بِهِ وَلَوْ بِإِذْنِ شَرِيكِهِ بِرِبْحِهِ وَخُسْرِهِ ؛ لِأَنَّ الْمُقَارَضَةَ لَيْسَتْ مِنْ التِّجَارَةِ وَإِنَّمَا هُوَ أَجْرُ نَفْسِهِ بِجُزْءٍ مِنْ الرِّبْحِ وَيَجُوزُ إنْ كَانَ لَا يَشْغَلُهُ عَنْ الْعَمَلِ فِي مَالِ الشِّرْكَةِ أَوْ أَذِنَ لَهُ شَرِيكُهُ فِيهِ ( و ) اسْتَبَدَّ شَرِيكٌ ( مُسْتَعِيرٌ دَابَّةً بِلَا إذْنٍ ) مِنْ شَرِيكِهِ ( وَإِنْ لِلشِّرْكَةِ ) الْوَاوُ لِلْحَالِ ، وَإِنْ زَائِدَةٌ فَالْأَوْلَى حَذْفُهُمَا أَيْ لِيَحْمِلَ عَلَيْهَا أَمْتِعَةَ الشِّرْكَةِ فَيَخْتَصُّ بِالرِّبْحِ وَهُوَ الْأُجْرَةُ فَيُحَاسِبُ بِهَا شَرِيكَهُ وَبِالْخُسْرِ وَهِيَ ضَمَانُهَا إنْ تَلِفَتْ بِتَفْرِيطٍ ، فَإِنْ أَذِنَ شَرِيكُهُ فَبَيْنَهُمَا ( و ) اسْتَبَدَّ ( مُتَّجِرٌ ) مِنْهُمَا بِغَيْرِ إذْنِ الْآخَرِ ( بِوَدِيعَةٍ ) أُودِعَتْ عِنْدَهُمَا أَوْ عِنْدَ أَحَدِهِمَا ( بِالرِّبْحِ وَالْخُسْرِ ) ( إلَّا أَنْ يَعْلَمَ شَرِيكُهُ بِتَعَدِّيهِ ) بِالتَّجْرِ ( فِي الْوَدِيعَةِ ) الَّتِي عِنْدَهُمَا أَوْ عِنْدَ غَيْرِ الْمُتَّجِرِ بِهَا( 13 / 489 )
