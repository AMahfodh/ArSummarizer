( قَوْلُهُ : بَلْ يَبْطُلُ النِّكَاحَانِ ) لَعَلَّهُ إذَا لَمْ يَكُنْ هُنَاكَ وَلِيٌّ مُجْبِرٌ وَإِلَّا فَلَهُمَا تَحْلِيفُهُ وَيَتَرَتَّبُ عَلَيْهِ حُكْمُهُ لِأَنَّ إقْرَارَهُ مَقْبُولٌ وَلَوْ بَعْدَ حَلِفِهِمَا فَرَاجِعْهُ ( قَوْلُهُ : وَهُوَ الْمَذْهَبُ ) وَصَرَّحَ بِهِ الْجُرْجَانِيُّ وَاقْتَضَاهُ كَلَامُ غَيْرِهِ شَرْحُ م ر ( قَوْلُهُ : فُسِخَا أَيْضًا ) عِبَارَةُ م ر وَيَنْفَسِخُ النِّكَاحُ قَوْلُهُ : وَأَفْرَدَهُ لِأَنَّ إلَخْ ) أَوْ عَلَى التَّأْوِيلِ الْمَذْكُورِ ( قَوْلُهُ : وَيُغَرِّمُهَا إلَخْ ) أَيْ فِي الْحَالَيْنِ ( قَوْلُهُ : مَا لَمْ يَمُتْ الْأَوَّلُ ) وَتَعْتَدَّ مِنْ الْأَوَّلِ عِدَّةَ الْوَفَاةِ إنْ لَمْ يَطَأْهَا وَإِلَّا اعْتَدَّتْ بِأَكْثَرِ الْأَمْرَيْنِ مِنْهَا وَمِنْ ثَلَاثَةِ أَقْرَاءٍ عِدَّةَ الْوَطْءِ مَا لَمْ تَكُنْ حَامِلًا شَرْحُ رَوْضٍ ( قَوْلُهُ : وَإِلَّا صَارَتْ إلَخْ ) قَالَ فِي شَرْحِ الرَّوْضِ وَالْقِيَاسُ أَنَّهَا تَرْجِعُ عَلَى الثَّانِي بِمَا غَرِمَتْ لَهُ لِأَنَّهَا إنَّمَا غَرِمَتْهُ لِلْحَيْلُولَةِ ا ه .
