ابْنُ عَرَفَةَ مُتَعَلِّقُ الْجِنَايَةِ غَيْرُ نَفْسٍ إنْ أَبَانَتْ بَعْضَ الْجِسْمِ فَقُطِعَ ، وَإِلَّا فَإِنْ أَزَالَتْ اتِّصَالَ عَظْمٍ لَمْ يَبِنْ فَكَسْرٌ ، وَإِلَّا فَإِنْ أَثَّرَتْ فِي الْجِسْمِ فَجُرْحٌ وَإِلَّا فَإِتْلَافُ مَنْفَعَةٍ أَيْ الْقِصَاصُ بِهِ ( كَ ) الْقِصَاصِ بِقَتْلِ ( النَّفْسِ فِي ) شَرْطِ ( الْفِعْلِ ) وَهُوَ كَوْنُهُ عَمْدًا عُدْوَانًا ( وَ ) شَرْطُ ( الْفَاعِلِ ) وَهُوَ كَوْنُهُ مُكَلَّفًا غَيْرَ حَرْبِيٍّ وَلَا زَائِدَ حُرِّيَّةٍ أَوْ إسْلَامٍ ( وَ ) شَرْطُ ( الْمَفْعُولِ ) وَهُوَ كَوْنُهُ مَعْصُومًا مِنْ الرَّمْيِ لِلْإِصَابَةِ .
