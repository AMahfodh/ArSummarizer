( وَمَنْ أُخْبِرَتْ ) بِالْبِنَاءِ لِلْمَفْعُولِ ( بِمَوْتِ زَوْجِهَا فَاعْتَدَّتْ ) لِلْوَفَاةِ ( ثُمَّ تَزَوَّجَتْ ) ثُمَّ وَلَدَتْ ( لَحِقَ بِثَانٍ مَا وَلَدَتْهُ لِنِصْفِ سَنَةٍ فَأَكْثَرَ ) مُنْذُ تَزَوَّجَتْهُ نَصًّا ؛ لِأَنَّهَا فِرَاشُهُ وَأَمَّا مَا وَلَدَتْهُ لِدُونِ نِصْفِ سَنَةٍ وَعَاشَ فَيَلْحَقُ بِالْأَوَّلِ ؛ لِأَنَّهُ لَيْسَ مِنْ الثَّانِي يَقِينًا وَكَذَا لَوْ مَاتَ زَوْجُهَا عِنْدَهَا أَوْ فُسِخَ نِكَاحُ غَائِبٍ .( 10 / 100 )
