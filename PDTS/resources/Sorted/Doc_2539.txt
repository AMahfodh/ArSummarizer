( فَرْعٌ : قَوْلُهُ ) لَهُ عَلَيَّ ( دِرْهَمٌ فِي دِينَارٍ كَأَلْفٍ ) أَيْ كَقَوْلِهِ لَهُ عَلَيَّ أَلْفٌ ( فِي ) هَذَا ( الْعَبْدِ ) فَيَأْتِي فِيهِ مَا مَرَّ ( إلَّا أَنْ يُرِيدَهُمَا مَعًا ) فَيَلْزَمَانِهِ ( فَإِنْ لَمْ يَنْوِ شَيْئًا فَدِرْهَمٌ ) فَقَطْ يَلْزَمُهُ وَظَاهِرٌ أَنَّ هَذَيْنِ الْحُكْمَيْنِ يَأْتِيَانِ فِي لَهُ عَلَيَّ أَلْفٌ فِي هَذَا الْعَبْدِ أَيْضًا .
