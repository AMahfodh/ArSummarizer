( بَابُ كَيْفِيَّةِ الْقِصَاصِ ) ( قَوْلُهُ : فَكَلَامُهُ عَلَى الْغَالِبِ ) هَذَا التَّفْرِيعُ فِيهِ حَزَازَةٌ بَعْدَ تَفْسِيرِهِ الْمَتْنَ بِمَا ذَكَرْنَا ، وَعِبَارَةُ التُّحْفَةِ عَقِبَ الْمَتْنِ نَصُّهَا : عَبَّرَ بِهِ لِلْغَالِبِ ، وَالْمُرَادُ لَا تُؤْخَذُ لِيَشْمَلَ الْمَعَانِيَ أَيْضًا ( قَوْلُهُ : فَفِي الْمَأْخُوذِ بَدَلًا دِيَتُهُ ) لَعَلَّهُ إذَا قَالَ لَهُ خُذْهَا قِصَاصًا أَخْذًا مِمَّا يَأْتِي فَلْيُرَاجَعْ ( قَوْلُهُ : فِي الْأَوَّلِ ) أَيْ عُضْوُ الْمَجْنِيِّ عَلَيْهِ ( قَوْلُهُ : دُونَهُ ) قَيْدٌ فِي الْأَصْلِيِّ وَالزَّائِدِ بِقَرِينَةِ مَا بَعْدَهُ ( قَوْلُهُ : وَمَحَلُّ ذَلِكَ ) يَعْنِي : مَا فِي الْمَتْنِ ، وَعِبَارَةُ التُّحْفَةِ : وَمَحَلُّ عَدَمِ ضَرَرِ ذَلِكَ( 24 / 365 )
