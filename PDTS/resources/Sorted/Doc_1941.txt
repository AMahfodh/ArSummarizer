( الثَّالِثَةُ ) مِنْ الْمُعْتَدَّاتِ ( ذَاتُ الْأَقْرَاءِ الْمُفَارَقَةِ فِي الْحَيَاةِ ) بَعْدَ دُخُولٍ أَوْ خَلْوَةٍ ( وَلَوْ بِطَلْقَةٍ ثَالِثَةٍ ) إجْمَاعًا قَالَهُ فِي الْفُرُوعِ ( فَتَعْتَدُّ حُرَّةٌ وَمُبَعَّضَةٌ ) مُسْلِمَةً كَانَتْ أَوْ كَافِرَةً ( بِثَلَاثَةِ قُرُوءٍ ) لِقَوْلِهِ تَعَالَى : { وَالْمُطَلَّقَاتُ يَتَرَبَّصْنَ بِأَنْفُسِهِنَّ ثَلَاثَةَ قُرُوءٍ } ( وَهِيَ ) أَيْ الْقُرُوءُ ( الْحَيْضُ ) .
