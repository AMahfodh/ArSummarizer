( وَ ) إنْ كَانَ الْمُدَّعَى عَلَيْهِ بِبَلَدٍ لَهُ قَاضٍ وَالْمُدَّعِي بِهِ بِبَلَدٍ آخَرَ لَهُ قَاضٍ آخَرُ فَ ( هَلْ يُدَّعَى ) بِضَمِّ التَّحْتِيَّةِ وَفَتْحِ الدَّالِ وَالْعَيْنِ مُثَقَّلًا ، أَيْ تُقَامُ الدَّعْوَى وَيَتَحَاكَمُ ( حَيْثُ ) يَكُونُ الشَّخْصُ ( الْمُدَّعَى عَلَيْهِ وَبِهِ ) أَيْ الِادِّعَاءِ بِمَحَلِّ الْمُدَّعَى عَلَيْهِ ( عُمِلَ ) بِضَمٍّ فَكَسْرٍ ، أَيْ قَضَى مُطَرِّفٌ بِهِ جَرَى الْحُكْمُ بِالْمَدِينَةِ وَحَكَمَ بِهِ ابْنُ بَشِيرٍ بِالْأَنْدَلُسِ ( أَوْ ) يُدْعَى حَيْثُ يَكُونُ ( الْمُدَّعِي ) فِيهِ وَهُوَ قَوْلُ الْإِمَامِ مَالِكٍ وَعَبْدِ الْمَلِكِ رَضِيَ اللَّهُ تَعَالَى عَنْهُمَا ( وَأُقِيمَ ) بِضَمِّ الْهَمْزِ وَفَتْحِ الْمِيمِ ، أَيْ فَهِمَهُ فَضْلٌ ( مِنْهَا ) أَيْ الْمُدَوَّنَةِ .
