( وَإِنْ أَخَّرَ طَوَافَ الزِّيَارَةِ ) وَنَصَّهُ ( أَوْ الْقُدُومِ فَطَافَهُ عِنْدَ الْخُرُوجِ كَفَاهُ ) ذَلِكَ الطَّوَافُ ( عَنْهُمَا ) ؛ لِأَنَّ الْمَأْمُورَ بِهِ أَنْ يَكُونَ آخِرُ عَهْدِهِ بِالْبَيْتِ الطَّوَافَ وَقَدْ فَعَلَ ؛ وَلِأَنَّ مَا شُرِعَ مِثْلُ تَحِيَّةِ الْمَسْجِدِ يُجْزِئ عَنْهُ الْوَاجِبُ مِنْ جِنْسِهِ كَإِجْزَاءِ الْمَكْتُوبَةِ عَنْ تَحِيَّةِ الْمَسْجِدِ وَكَإِجْزَاءِ الْمَكْتُوبَةِ أَيْضًا عَنْ رَكْعَتَيْ الطَّوَافِ وَعَنْ رَكْعَتَيْ الْإِحْرَامِ فَإِنْ نَوَى بِطَوَافِهِ الْوَدَاعَ لَمْ يُجْزِئهُ عَنْ طَوَافِ الزِّيَارَةِ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { وَإِنَّمَا لِكُلِّ امْرِئٍ مَا نَوَى } .( 7 / 310 )
