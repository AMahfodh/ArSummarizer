( وَإِنْ كَانَ لَهُ ثَلَاثُ نِسْوَةٍ كِبَارٍ وَوَاحِدَةٌ صَغِيرَةٌ فَأَرْضَعَتْ كُلُّ وَاحِدَةٍ مِنْ الثَّلَاثِ ) الْكِبَارِ ( الصَّغِيرَةَ أَرْبَعَ رَضَعَاتٍ ثُمَّ حَلَبْنَ فِي إنَاءٍ وَسَقَيْنَهُ لِلصُّغْرَى حُرِّمَ الْكِبَارُ ) لِأَنَّهُنَّ مِنْ أُمَّهَاتِ نِسَائِهِ ( وَإِنْ لَمْ يَكُنْ دَخَلَ بِهِنَّ فَنِكَاحُ الصَّغِيرَةِ ثَابِتٌ وَعَلَيْهِ لِكُلِّ وَاحِدَةٍ ثُلُثُ صَدَاقِهَا يَرْجِعُ بِهِ عَلَى ضَرَّاتِهَا ) لِتَسَبُّبِهِنَّ فِي إقْرَارِ ذَلِكَ عَلَيْهِ وَيَسْقُطُ السُّدُسُ الْبَاقِي مِنْ النِّصْفِ فِي نَظِيرِ فِعْلِهَا ، لِأَنَّ كُلَّ وَاحِدَةٍ مِنْهُمَا ثَالِثَةٌ لِضَرَّتَيْهَا مُشَارَكَةٌ لَهُمَا ( لِأَنَّ إفْسَادَ نِكَاحِهَا حَصَلَ بِفِعْلِهَا وَفِعْلِهِمَا وَإِنْ كَانَ قَدْ دَخَلَ بِإِحْدَى الْكِبَارِ حُرِّمَتْ الصَّغِيرَةُ أَيْضًا ) لِأَنَّهَا رَبِيبَةُ زَوْجَةٍ دُخِلَ بِهَا ( وَلَهَا ) أَيْ الصَّغِيرَةِ ( نِصْفُ صَدَاقِهَا يُرْجَعُ بِهِ عَلَيْهِنَّ أَثْلَاثًا ) لِأَنَّهُنَّ تَسَبَّبْنَ فِي فَسَادِ نِكَاحِهَا ( وَلِلَّتِي دَخَلَ بِهَا الْمَهْرُ كَامِلًا ) لِاسْتِقْرَارِهِ بِالدُّخُولِ .( 19 / 284 )
