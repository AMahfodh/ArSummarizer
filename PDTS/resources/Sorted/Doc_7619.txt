وَ ( لَا ) تُضْمَنُ ( دَابَّةٌ ) غُصِبَتْ وَ ( عَلَيْهَا مَالِكُهَا الْكَبِيرُ وَمَتَاعُهُ ) ؛ لِأَنَّهَا فِي يَدِ مَالِكِهَا ( وَإِنْ اسْتَعْمَلَهُ ) أَيْ : الْحُرُّ ( كُرْهًا ) فِي خِدْمَةٍ أَوْ خِيَاطَةٍ أَوْ غَيْرِهِمَا فَعَلَيْهِ أُجْرَتُهُ ؛ لِاسْتِيفَائِهِ مَنَافِعَهُ الْمُتَقَوِّمَةَ فَيَضْمَنُهَا كَمَنَافِعِ الْعَبْدِ ( أَوْ حَبَسَهُ ) أَيْ : الْحُرُّ ( مُدَّةً ) لَهَا أُجْرَةٌ ( فَعَلَيْهِ أُجْرَتُهُ ) مُدَّةَ حَبْسِهِ ، ؛ لِأَنَّهُ فَوَّتَ مَنْفَعَتَهُ زَمَنَ الْحَبْسِ ، وَهِيَ مَالٌ يَجُوزُ أَخْذُ الْعِوَضِ عَنْهُ كَمَنَافِعِ الْعَبْدِ .( 6 / 340 )
