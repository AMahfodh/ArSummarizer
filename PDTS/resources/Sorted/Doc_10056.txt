( وَإِنْ سَأَلَ غُرَمَاءُ مَنْ لَهُ مَالٌ لَا يَفِي بِدَيْنِهِ ) الْحَالِّ ( أَوْ ) ( سَأَلَ بَعْضُهُمْ الْحَاكِمَ الْحَجْرَ عَلَيْهِ ) ؛ أَيْ : الْمَدِين ؛ ( لَزِمَهُ ) ؛ أَيْ : الْحَاكِمَ ( إجَابَتُهُمْ ) ؛ أَيْ : السَّائِلِينَ ، وَحَجَر عَلَيْهِ ؛ لِحَدِيثِ كَعْبِ بْنِ مَالِكٍ : { أَنَّ رَسُولَ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ حَجَر عَلَى مُعَاذٍ ، وَبَاعَ مَالَهُ } رَوَاهُ الْخَلَّالُ فَإِنْ لَمْ يَسْأَلْهُ أَحَدٌ مِنْهُمْ ؛ لَمْ يَحْجُرْ عَلَيْهِ ، ( لَا إنْ سَأَلَهُ الْمُفْلِسُ ) أَنْ يَحْجُرَ عَلَيْهِ ، فَلَا تَلْزَمُ الْحَاكِمَ إجَابَتُهُ ؛ لِأَنَّ الْحَجْرَ عَلَيْهِ حَقٌّ لِغُرَمَائِهِ ، لَا لَهُ .( 9 / 133 )
