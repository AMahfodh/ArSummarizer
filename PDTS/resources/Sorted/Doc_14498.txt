( وَ ) يُجْزِئُ أَنْ يُسْتَنَابَ عَنْهُ ( مِنْ خَارِجِ بَلَدِهِ إلَى دُونِ مَسَافَةِ الْقَصْرِ ) لِأَنَّهُ فِي حُكْمِ الْحَاضِرِ ( وَيَسْقُطُ ) حَجٌّ عَمَّنْ وَجَبَ عَلَيْهِ وَمَاتَ قَبْلَهُ ( بِحَجِّ أَجْنَبِيٍّ عَنْهُ ) بِدُونِ مَالٍ وَدُونِ إذْنِ وَارِثٍ لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ شَبَّهَهُ بِالدَّيْنِ وَكَذَا عُمْرَةٌ وَ ( لَا ) يَسْقُطُ حَجٌّ ( عَنْ ) مَعْضُوبٍ ( حَيٍّ بِلَا إذْنِهِ ) وَلَوْ مَعْذُورًا كَدَفْعِ زَكَاةِ مَالِ حَيٍّ عَنْهُ بِلَا إذْنِهِ بِخِلَافِ الدَّيْنِ لِأَنَّهُ لَيْسَ بِعِبَادَةٍ ( وَيَقَعُ ) حَجُّ مَنْ حَجَّ عَنْ حَيٍّ بِلَا إذْنِهِ ( عَنْ نَفْسِهِ ) أَيْ الْحَاجِّ ( وَلَوْ ) كَانَ الْحَجُّ ( نَفْلًا ) عَنْ مَحْجُوجٍ عَنْهُ بِلَا إذْنٍ ، لَكِنَّ قِيَاسَ مَا سَبَقَ آخِرَ الْجَنَائِزِ : يَصِحُّ جَعْلُ ثَوَابِهِ لِحَيٍّ وَمَيِّتٍ ( وَ ) مَنْ وَجَبَ عَلَيْهِ نُسُكٌ وَمَاتَ قَبْلَهُ ، وَ ( ضَاقَ مَالُهُ ) عَنْ أَدَائِهِ مِنْ بَلَدِهِ اُسْتُنِيبَ بِهِ مِنْ حَيْثُ بَلَغَ ( أَوْ لَزِمَهُ دَيْنٌ ) وَعَلَيْهِ حَجٌّ وَضَاقَ مَالُهُ عَنْهُمَا ( أَخَذَ ) مِنْ مَالِهِ ( لِحَجٍّ بِحِصَّتِهِ ) كَسَائِرِ الدُّيُونِ ( وَحَجَّ بِهِ ) أَيْ بِمَا أَخَذَ لِلْحَجِّ ( مِنْ حَيْثُ بَلَغَ ) لِحَدِيثِ : { إذَا أَمَرْتُكُمْ بِأَمْرٍ فَأْتُوا مِنْهُ مَا اسْتَطَعْتُمْ } ( وَإِنْ مَاتَ ) مَنْ وَجَبَ عَلَيْهِ حَجٌّ( 3 / 423 )
