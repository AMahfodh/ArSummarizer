( وَيَتَّجِهُ فِ ) يمَا إذَا قَالَ لِزَوْجَتِهِ : أَنْتِ طَالِقٌ ( لَا ضَرَبْت زَيْدًا أَوْ عَمْرًا ، أَنَّهُ لَا حِنْثَ بِضَرْبِ أَحَدِهِمَا ) أَيْ : زَيْدٍ وَعَمْرٍو ؛ لِمَا تَقَدَّمَ مِنْ أَنَّهَا لَا تَطْلُق بِقَوْلِهِ أَنْتِ طَالِقٌ لَا قُمْتُ وَقَعَدْتُ إلَّا بِوُجُودِهِمَا ( بِلَا نِيَّةٍ أَوْ سَبَبٍ ) أَمَّا إنْ نَوَى بِقَوْلِهِ ضَرْبِ أَحَدِهِمَا ؛ فَتَطْلُق بِضَرْبِهِ ، أَوْ كَانَ ثَمَّ سَبَبٌ يَقْتَضِي ضَرْبَ أَحَدِهِمَا ؛ فَتَطْلُق بِحُصُولِهِ ( وَ ) يَتَّجِهُ ( أَنَّهُ يَحْنَثُ ) قَائِلُ ذَلِكَ بِضَرْبِ أَحَدِهِمَا ( إنْ أَعَادَ الْعَامِلَ ) بِأَنْ قَالَ : أَنْتِ طَالِقٌ لَا ضَرَبْتُ زَيْدًا وَلَا عَمْرًا ؛ لِأَنَّ لَا هُنَا بِمَعْنَى إنْ ، فَكَأَنَّهُ قَالَ : أَنْتِ طَالِقٌ إنْ ضَرَبْتُ زَيْدًا أَوْ( 16 / 188 )
