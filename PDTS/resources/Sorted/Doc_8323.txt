وَهَذَا صَالِحٌ لِلْكُلِّ وَالْفَرْقُ بَيْنَهُمَا بِحَسَبِ الْمُقَامِ ( أَوْ ) مُتَلَازِمَيْنِ ( لَفْظًا ) فَيُطْلَقُ اسْمُ أَحَدِهِمَا بِخُصُوصِهِ عَلَى الْآخَرِ مُشَاكَلَةً كَقَوْلِهِ تَعَالَى { وَجَزَاءُ سَيِّئَةٍ سَيِّئَةٌ مِثْلُهَا } فَأَطْلَقَ السَّيِّئَةَ عَلَى الْجَزَاءِ مَعَ أَنَّهُ حَسَنٌ لِوُقُوعِهِ فِي صُحْبَتِهَا وَقَدْ يُقَالُ : إنَّمَا سُمِّيَ جَزَاؤُهَا سَيِّئَةً لِأَنَّهُ يَسُوءُ مَنْ يَنْزِلُ بِهِ وَحِينَئِذٍ فَهُوَ لَيْسَ مِثَالًا لِمَا نَحْنُ فِيهِ بَلْ مِنْ مِثْلِهِ قَوْلُهُ قَالُوا اقْتَرِحْ شَيْئًا نُجِدْ لَك طَبْخَهُ قُلْت اُطْبُخُوا لِي جُبَّةً وَقَمِيصًا أَيْ خِيطُوا فَذَكَرَهَا بِلَفْظِ الطَّبْخِ لِوُقُوعِهَا فِي صُحْبَةِ طَبْخِ الطَّعَامِ وَنَحْوِهِ .( 2 / 415 )
