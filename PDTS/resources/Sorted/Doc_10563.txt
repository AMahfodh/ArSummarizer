( وَإِذَا وَقَعَ عِيدٌ يَوْمَ جُمُعَةٍ فَصَلَّوْا الْعِيدَ وَالظُّهْرَ جَازَ ) ذَلِكَ ( وَسَقَطَتْ الْجُمُعَةُ عَمَّنْ حَضَرَ الْعِيدَ ) مَعَ الْإِمَامِ لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { صَلَّى الْعِيدَ وَقَالَ مَنْ شَاءَ أَنْ يُجَمِّعَ فَلْيُجَمِّعْ } رَوَاهُ أَحْمَدُ مِنْ حَدِيثِ زَيْدِ بْنِ أَرْقَمَ ، وَحِينَئِذٍ فَتَسْقُطُ الْجُمُعَةُ ( إسْقَاطَ حُضُورٍ ، لَا ) إسْقَاطَ ( وُجُوبٍ ) فَيَكُونُ حُكْمُهُ ( كَمَرِيضٍ وَنَحْوِهِ ) مِمَّنْ لَهُ عُذْرٌ أَوْ شُغْلٌ يُبِيحُ تَرْكَ الْجُمُعَةِ ، وَ ( لَا ) يَسْقُطُ عَنْهُ وُجُوبُهَا فَيَكُونُ ( كَمُسَافِرٍ وَعَبْدٍ ) لِأَنَّ الْإِسْقَاطَ لِلتَّخْفِيفِ فَتَنْعَقِدُ بِهِ الْجُمُعَةُ وَيَصِحُّ أَنْ يَؤُمَّ فِيهَا ( وَالْأَفْضَلُ : حُضُورُهُ ) خُرُوجًا مِنْ الْخِلَافِ ( إلَّا الْإِمَامَ فَلَا يَسْقُطُ عَنْهُ ) حُضُورُ الْجُمُعَةِ .
