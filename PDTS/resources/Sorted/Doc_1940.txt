( قَوْلُهُ : وَلَوْ لَمْ يَعُدْ ) غَايَةً ( قَوْلُهُ : وَفِيمَا غَنِمَ فِي غِيبَتِهِ مُطْلَقًا ) أَيْ قَرُبَ أَوْ بَعُدَ ( قَوْلُهُ : يَحْرُمُ انْصِرَافُ مِائَةِ بَطَلٍ ) أَيْ مِنَّا ( قَوْلُهُ : عَنْ مِائَتَيْنِ وَوَاحِدٍ ضُعَفَاءَ ) أَيْ مِنْ الْكُفَّارِ ( قَوْلُهُ : وَيَجُوزُ انْصِرَافُ مِائَةٍ ) أَيْ لِأَنَّهُمْ لَا( 26 / 420 )
