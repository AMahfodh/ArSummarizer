( فَرْعٌ ) لَوْ ( اسْتَلْحَقَ أَحَدَ وَلَدَيْ أَمَتَيْهِ ، وَهُمَا مُزَوَّجَتَانِ ) أَوْ مُسْتَفْرَشَتَانِ لَهُ ( لَغَا ) الِاسْتِلْحَاقُ لِلُحُوقِ وَلَدِ كُلِّ أَمَةٍ بِزَوْجِهَا فِي الْأُولَى وَلُحُوقِهِمَا بِهِ فِي الثَّانِيَةِ بِالِاسْتِفْرَاشِ لَا بِالِاسْتِلْحَاقِ كَمَا مَرَّ نَظِيرُهُ ( أَوْ ) كَانَتْ ( إحْدَاهُمَا ) غَيْرَ مُزَوَّجَةٍ أَوْ مُسْتَفْرَشَةٍ دُونَ الْأُخْرَى ( أَوْ هُمَا غَيْرُ مُزَوَّجَتَيْنِ وَلَا مُسْتَفْرَشَتَيْنِ ) لَهُ ( لَزِمَهُ التَّعْيِينُ ) كَمَا لَوْ أَقَرَّ بِطَلَاقِ إحْدَى زَوْجَتَيْهِ فَإِذَا عَيَّنَ أَحَدَهُمَا رُتِّبَ عَلَيْهِ حُكْمُهُ مِنْ نَسَبٍ وَغَيْرِهِ وَلَوْ ادَّعَتْ الْأُخْرَى أَنَّهَا الْمُسْتَوْلَدَةُ ، وَوَلَدُهَا الْمُسْتَلْحَقُ أَوْ بَلَغَ الْوَلَدُ وَادَّعَاهُ صُدِّقَ السَّيِّدُ بِيَمِينِهِ فَإِنْ نَكَلَ حَلَفَ الْمُدَّعِي وَقُضِيَ بِمُقْتَضَى يَمِينِهِ صَرَّحَ بِذَلِكَ الْأَصْلُ .
