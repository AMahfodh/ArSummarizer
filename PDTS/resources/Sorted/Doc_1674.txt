( وَحِصَّةُ رَبِّهِ بِرِبْحِهِ نِصَابٌ ) فَأَكْثَرُ ، وَالْوَاوُ لِلْحَالِ : لَا أَقَلَّ وَإِنْ نَابَهُ هُوَ نِصَابٌ بَلْ يَسْتَقْبِلُ حِينَئِذٍ بِهِ ( أَوْ ) حِصَّةُ رَبِّهِ بِرِبْحِهِ ( أَقَلُّ ) مِنْ نِصَابٍ ، ( وَ ) لَكِنْ ( عِنْدَهُ ) أَيْ رَبِّهِ ( مَا يُكَمِّلُهُ ) فَيُزَكِّي الْعَامِلُ وَإِنْ أَقَلَّ لِأَنَّ زَكَاتَهُ تَابِعَةُ لِزَكَاةِ رَبِّهِ .
