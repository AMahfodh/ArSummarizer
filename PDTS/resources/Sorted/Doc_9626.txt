( وَمَنْ أَرْضَعَتْ بِلَبَنِ حَمْلٍ مِنْ زِنًى ) طِفْلًا ( أَوْ ) أَرْضَعَتْ بِلَبَنِ حَمْلٍ ( نَفْيٍ بِلِعَانٍ طِفْلَةً ) فِي الْحَوْلَيْنِ ( صَارَتْ بِنْتَهَا ) فَقَطْ ، فَتَثْبُتُ الْأُمُومَةُ وَفُرُوعُهَا مِنْ الْجُدُودَةِ لَهَا وَالْخُؤُولَةِ دُونَ الْأُبُوَّةِ وَفُرُوعِهَا ؛ لِأَنَّهُ تَابِعٌ لِلنَّسَبِ ( وَحَرُمَتْ ) الطِّفْلَةُ ( عَلَى وَاطِئٍ تَحْرِيمَ مُصَاهَرَةٍ ) لِأَنَّهَا بِنْتُ مَوْطُوءَتِهِ ( وَتَحِلُّ لِابْنِ وَاطِئٍ وَأَبِيهِ ) لِأَنَّهَا أَجْنَبِيَّةٌ مِنْهَا ( وَلَا تَثْبُتُ حُرْمَةُ الرَّضَاعِ فِي حَقِّ وَاطِئٍ ) بِزِنًا أَوْ مُلَاعِنٍ ( مِنْ حَيْثُ الْمَحْرَمِيَّةِ ) لِحَدِيثِ : { يَحْرُمُ مِنْ الرَّضَاعِ مَا يَحْرُمُ مِنْ النَّسَبِ } ، وَلَا نَسَبَ هُنَا .( 17 / 4 )
