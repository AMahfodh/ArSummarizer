وَظَاهِرٌ أَنَّ قَوْلَهُ : كَأَصْلِهِ بَعْدَ دَفْعِ الْمَهْرِ مِثَالٌ ، فَقَبْلَ دَفْعِهِ كَذَلِكَ ( وَحَيْثُ كَانَ الْعَبْدُ ) الَّذِي أَذِنَ لَهُ سَيِّدُهُ فِي نِكَاحِ أَمَةٍ ( مَهْرَهَا ) ، ثُمَّ طَلَّقَهَا قَبْلَ الْوَطْءِ ( بَقِيَ ) أَيْ : الْعَبْدُ كُلُّهُ ( لِمَالِكِ الْعِرْسِ ) أَيْ : الزَّوْجَةِ ( وَإِنْ أَعْتَقَهَا أَوْ بَاعَهَا ) مَالِكُهَا الْمَالِكُ لِلْعَبْدِ ( مِنْ قَبْلِ أَنْ طَلَّقَهَا ) أَمَّا النِّصْفُ الْمُسْتَقِرُّ قَبْلَ طَلَاقِهَا فَظَاهِرٌ ، وَأَمَّا النِّصْفُ الَّذِي يَرْجِعُ بِطَلَاقِهَا ، فَلِأَنَّ مَالِكَهَا الْمُزَوِّجَ لَهَا هُوَ الْمَالِكُ لِلْعَبْدِ عِنْدَ الطَّلَاقِ فَرَجَعَ إلَيْهِ وَلَا يَرْجِعُ لِمَالِكِهِ الْأَوَّلِ ، كَمَا لَا يَرْجِعُ لِلْبَائِعِ وَالْمُعْتِقُ فِيمَا مَرَّ وَقَوْلُهُ : أَوْ بَاعَهَا مِنْ زِيَادَتِهِ ( لَوْ مَالِكُ الْعِرْسِ لِهَذَا ) بِزِيَادَةِ اللَّازِمِ لِتَقْوِيَةِ الْعَامِلِ وَهُوَ ( أَعْتَقَا ، أَوْ بَاعَ ) أَيْ : فَلَوْ أَعْتَقَ مَالِكُ الزَّوْجَةِ هَذَا الْعَبْدَ ، أَوْ بَاعَهُ ( ثُمَّ انْفَسَخَتْ ) عُقْدَةُ نِكَاحِهَا بِسَبَبِهَا (( 15 / 137 )
