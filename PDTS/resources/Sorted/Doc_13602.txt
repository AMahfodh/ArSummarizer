( قَوْلُهُ : إقْلِيمُهَا ) سَيَأْتِي رَدُّهُ وَإِقْلِيمُهَا مِنْ إسْكَنْدَرِيَّةَ إلَى أُسْوَانَ وَهَذَا كُلُّهُ حَيْثُ لَمْ يَنْوِ وَاحِدًا مِمَّا ذُكِرَ بِعَيْنِهِ فَإِنْ نَوَى وَاحِدًا بِعَيْنِهِ عُمِلَ بِهِ وَكَذَا لَوْ جَرَى عُرْفُ الْحَالِفِ بِإِطْلَاقِ مِصْرَ عَلَى خُصُوصِ الْبَلَدِ الْمُعَيَّنَةِ كَمَا عِنْدَ رِيفِ مِصْرَ ( قَوْلُهُ : وَحَيْثُ أُطْلِقَ مِصْرُ ) الْمُنَاسِبُ أَنْ يُؤَخِّرَ ذَلِكَ وَيَقُولَ وَلَكِنَّ الْعُرْفَ جَرَى بِإِطْلَاقِ مِصْرَ عَلَى الْقَاهِرَةِ فَلَا يُعَوَّلُ عَلَى مَا قَالَهُ الْمُصَنِّفُ لِأَنَّ الْأَيْمَانَ مَبْنَاهَا الْعُرْفُ ( قَوْلُهُ : وَالظَّاهِرُ أَنَّ الْمُرَادَ بِعَمَلِهَا الْقَضَائِيُّ ) أَيْ الَّذِي يَحْكُمُ فِيهِ قَاضِي الْعَسْكَرِ الَّذِي بِمِصْرَ وَأَمَّا الصَّعِيدُ وَالْبُحَيْرَةُ وَنَحْوُ ذَلِكَ فَلَيْسَ مِنْ عَمَلِهَا الْقَضَائِيِّ لِأَنَّ قُضَاةَ تِلْكَ الْمَوَاضِعِ مِنْ إسْتَانْبُولَ وَالْحَقُّ أَنَّ الْمُرَادَ بِالْعَمَلِ الْعَمَلُ السُّلْطَانِيُّ لِأَنَّهُ مَتَى أُطْلِقَ لَا يَنْصَرِفُ إلَّا إلَيْهِ فَإِنْ نَوَى الْعَمَلَ انْصَرَفَ لِلسُّلْطَانِيِّ مَا لَمْ يَجْرِ عُرْفٌ بِخِلَافِهِ فَإِذَا جَرَى عُرْفٌ بِخِلَافِهِ عُمِلَ عَلَيْهِ وَكَذَا يُعْمَلُ بِالْعُرْفِ إذَا لَمْ يَنْوِ ( قَوْلُهُ : وَلَهُ الْمُوَاعَدَةُ ) إنَّمَا جَازَتْ هُنَا وَمُنِعَتْ فِي الْعِدَّةِ لِأَنَّهَا مِنْ الْخِطْبَةِ وَالْمُوَاعَدَةُ لَيْسَتْ مِنْ التَّزَوُّجِ الْمَحْلُوفِ عَلَيْهِ قَالَهُ تت .( 12 / 202 )
