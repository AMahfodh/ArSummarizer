( فَإِنْ تَقَدَّمَ عَلَيْهِ ) أَيْ عَلَى مَا لَفَظَهُ الْبَحْرَ ( مِلْكٌ ) لِأَحَدٍ ( فَإِنْ كَانَ ) مَنْ تَقَدَّمَ لَهُ مَلَكَ ( حَرْبِيًّا فَكَذَلِكَ ) : أَيْ فَهُوَ لِوَاجِدِهِ لَكِنَّهُ يُخَمَّسُ لِأَنَّهُ مِنْ الرِّكَازِ ، فَالتَّشْبِيهُ لَيْسَ بِتَامٍّ بِدَلِيلِ مَا بَعْده ، وَمُرَادُهُ بِالْحَرْبِيِّ الْمُتَحَقِّقِ حِرَابَتُهُ وَإِلَّا فَمَا بَعْدَهُ يُغْنِي عَنْهُ أَيْ قَوْلِهِ ؛ ( وَ ) إنْ كَانَ مَنْ تَقَدَّمَ مَلَكَهُ ( جَاهِلِيًّا ) أَيْ غَيْرَ مُسْلِمٍ وَذِمِّيٍّ ( وَلَوْ بِشَكٍّ ) فِي جَاهِلِيَّتِهِ وَغَيْرِهَا ( فَرِكَازٌ ) يُخَمَّسُ وَالْبَاقِي لِوَاجِدِهِ .
