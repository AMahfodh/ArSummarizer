وَ ( لَا ) يُجْزِئُ ( مَرِيضٌ مَأْيُوسٍ ) مِنْهُ ؛ لِعَدَمِ تَمَكُّنِهِ مِنْ الْعَمَلِ ( وَلَا مَغْصُوبٌ مِنْهُ ) عَلَى الصَّحِيحِ مِنْ الْمَذْهَبِ ، لِعَدَمِ تَمَكُّنِهِ مِنْ مَنَافِعِهِ ( وَيَتَّجِهُ ) عَدَمُ إجْزَاءِ عِتْقِ رَقِيقٍ مَغْصُوبٍ مِنْهُ ( مَا لَمْ يُخَلِّصْهُ ) مُعْتِقُهُ ( بَعْدَ ) ذَلِكَ مِنْ الْغَاصِبِ ؛ فَإِنْ خَلَّصَهُ فَلَا مَانِعَ مِنْ إجْزَائِهِ ؛ لِأَنَّهُ إذَا تَخَلَّصَ مِمَّنْ هُوَ بِيَدِهِ يَتَمَكَّنُ مِنْ مَنَافِعِهِ ، وَهُوَ مُتَّجِهٌ .( 16 / 338 )
