( وَلَا ) يَنْقُضُ ( مَسُّهُ ) أَيْ : الذَّكَرِ ( بِغَيْرِ يَدٍ ) كَالذِّرَاعِ ( غَيْرَ مَا تَقَدَّمَ ) مِنْ مَسِّ الذَّكَرِ بِفَرْجِ غَيْرِهِ فَإِنَّهُ يَنْقُضُ ( وَلَا ) يَنْقُضُ ( مَسُّ ) ذَكَرٍ ( زَائِدٍ ) ؛ لِأَنَّهُ لَيْسَ فَرْجًا ( فَإِنْ لَمَسَ ) رَجُلٌ أَوْ امْرَأَةٌ أَوْ خُنْثَى ( قُبُلَ خُنْثَى مُشْكِلٍ وَذَكَرَهُ ، وَلَوْ كَانَ هُوَ ) أَيْ : الْخُنْثَى ( اللَّامِسَ ) لِقُبُلِ نَفْسِهِ وَذَكَرِهِ ( نُقِضَ ) الْوُضُوءُ ؛ لِأَنَّ لَمْسَ الْفَرْجِ مُتَيَقَّنٌ ؛ لِأَنَّ الْخُنْثَى إنْ كَانَ ذَكَرًا فَقَدْ لَمَسَ ذَكَرَهُ ، وَإِنْ كَانَ أُنْثَى فَقَدْ لَمَسَ فَرْجَهَا .
