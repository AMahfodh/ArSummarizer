( أَوْ غَسَلَ صَحِيحًا وَتَيَمَّمَ لِجُرْحٍ ) ثُمَّ لَبِسَ حَائِلًا ، جَازَ لَهُ الْمَسْحُ عَلَيْهِ ؛ لِأَنَّهُ تَقَدَّمَهُ طَهَارَةٌ كَامِلَةٌ بِالنِّسْبَةِ إلَيْهِ ( فَلَا يَمْسَحُ عَلَى خُفٍّ ) وَلَا جُرْمُوقٍ وَلَا جَوْرَبٍ وَلَا عِمَامَةٍ وَلَا خِمَارٍ وَلَا جَبِيرَةٍ ( لَبِسَهُ عَلَى طَهَارَةٍ تَيَمَّمَ ) ؛ لِأَنَّهُ لَا يَرْفَعُ حَدَثًا ( وَلَوْ غَسَلَ رِجْلًا ثُمَّ أَدْخَلَهَا الْخُفَّ ) قَبْلَ غَسْلِ الْأُخْرَى ( خَلَعَ ) الْخُفَّ ( ثُمَّ لَبِسَ بَعْدَ غَسْلِ الْأُخْرَى ) لِتَكْمُلَ الطَّهَارَةُ .
