328 - 328 - ( 7 ) - حَدِيثُ ابْنِ عُمَرَ : { كَانَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يَرْفَعُ يَدَيْهِ حَذْوَ مَنْكِبَيْهِ إذَا افْتَتَحَ الصَّلَاةَ } مُتَّفَقٌ عَلَيْهِ بِزِيَادَةِ : وَإِذَا كَبَّرَ لِلرُّكُوعِ ، وَإِذَا رَفَعَ رَأْسَهُ مِنْ الرُّكُوعِ رَفَعَهُمَا كَذَلِكَ ، فَقَالَ : { سَمِعَ اللَّهُ لِمَنْ حَمِدَهُ } ، زَادَ الْبَيْهَقِيُّ { : فَمَا زَالَتْ تِلْكَ صَلَاتَهُ حَتَّى لَقِيَ اللَّهَ } .
