( وَقَالَ الشَّيْخُ ) تَقِيُّ الدِّينِ : يَحْرُمُ الذَّبْحُ وَالتَّضْحِيَةُ عِنْدَ الْقَبْرِ وَ ( لَوْ نَذَرَهُ ) نَاذِرٌ ( لَمْ يَفِ بِهِ ) ، لِحَدِيثِ : { مَنْ نَذَرَ أَنْ يُطِيعَ اللَّهَ فَلْيُطِعْهُ ، وَمَنْ نَذَرَ أَنْ يَعْصِيَ اللَّهَ فَلَا يَعْصِهِ } ( وَلَوْ شَرَطَهُ ) ، أَيْ : الذَّبْحَ ( وَاقِفٌ ) حِينَ وَقَفَ عَقَارَهُ ، فَقَالَ : يُؤْخَذُ مِنْ غَلَّتِهِ كُلَّ سَنَةٍ أَوْ شَهْرٍ نَحْوُ شَاةٍ فَتُذْبَحُ عِنْدَ الْقَبْرِ وَتُفَرَّقُ عَلَى الْفُقَرَاءِ ؛ ( فَشَرْطُهُ فَاسِدٌ ) ، لِأَنَّهُ غَيْرُ مَشْرُوعٍ .( 4 / 498 )
