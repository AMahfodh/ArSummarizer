وَأَشَارَ إلَى بَيَانِ مَنْ يَصْلُحُ لِلْإِمَامَةِ بِقَوْلِهِ : ( وَيَؤُمُّ النَّاسَ أَفْضَلُهُمْ وَأَفْقَهُهُمْ ) لِخَبَرِ : { أَئِمَّتُكُمْ شُفَعَاؤُكُمْ } وَخَبَرِ : { وَلْيَؤُمَّكُمْ أَكْبَرُكُمْ } وَقَالَ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ : { إنْ سِرَّكُمْ أَنْ تُقْبَلَ مِنْكُمْ صَلَاتُكُمْ فَلْيَؤُمَّكُمْ خِيَارُكُمْ فَإِنَّهُ وَفْدٌ بَيْنَكُمْ وَبَيْنَ رَبِّكُمْ فَلَا يَؤُمَّكُمْ إلَّا الذُّكُورُ } .
