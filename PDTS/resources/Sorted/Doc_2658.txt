وَالْأَوْلَى أَنْ يُقَالَ بِأَنَّهُ يُطَالِبُ وَكِيلُهُ الْحُكْمَ ( قَوْلُهُ : وَلَا يَتَوَقَّفُ ) أَيْ : الْحُكْمُ ( قَوْلُهُ : فِيمَا مَرَّ ) أَيْ : فِي الْمَتْنِ ( قَوْلُهُ : وَلَوْ ادَّعَى قَيِّمُ صَبِيٍّ ) إلَى قَوْلِهِ : وَبِهِ صَرَّحَ الْقَاضِي فِي الْمُغْنِي ، وَقَوْلُهُ : دَيْنًا لَهُ أَفْرَدَ الضَّمِيرَ لِكَوْنِ الْعَطْفِ بِأَوْ ( قَوْلُهُ : لَمْ يُؤَخَّرْ الِاسْتِيفَاءُ إلَخْ .
