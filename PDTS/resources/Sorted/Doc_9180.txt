( فَيَبْسُطُ ) مَنْ يُكَفِّنُ الرَّجُلَ الْمَيِّتَ ( بَعْضَ اللَّفَائِفِ ) الثَّلَاثِ ( فَوْقَ بَعْضٍ ) لِيُوضَعَ عَلَيْهَا مَرَّةً وَاحِدَةً وَلَا يُحْتَاجُ إلَى حَمْلِهِ ، وَوَضْعِهِ عَلَى وَاحِدَةٍ بَعْدَ وَاحِدَةٍ ( وَيُجَمِّرُهَا بِالْعُودِ ) أَوْ نَحْوِهِ أَوْصَى بِهِ أَبُو سَعِيدٍ وَابْنُ عُمَرَ وَابْنُ عَبَّاسٍ وَلِأَنَّ هَذَا عَادَةُ الْحَيِّ ( بَعْدَ رَشِّهَا بِمَاءِ وَرْدٍ أَوْ غَيْرِهِ لِيَتَعَلَّقَ بِهِ ) رَائِحَةُ الْبَخُورِ ، وَإِنْ لَمْ يَكُنْ الْمَيِّتُ مُحْرِمًا ( ثُمَّ يُوضَعُ ) الْمَيِّتُ ( عَلَيْهَا ) أَيْ : اللَّفَائِفِ ( مُسْتَلْقِيًا ) لِأَنَّهُ أَمْكَنُ لِإِدْرَاجِهِ فِيهَا وَالْأَوْلَى أَنْ يُسْتَرَ بِثَوْبٍ فِي حَالِ حَمْلِهِ ، وَ أَنْ يُوضَعَ مُتَوَجِّهًا .
