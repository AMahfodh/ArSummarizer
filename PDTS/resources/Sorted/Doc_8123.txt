( وَ ) نَوْمٌ ( تَحْتَ سَمَاءٍ مُتَجَرِّدًا ) مِنْ ثِيَابٍ ، وَالْمُرَادُ : مَسْتُورُ الْعَوْرَةِ ، وَعَلَى سَطْحٍ لَيْسَ عَلَيْهِ تَحْجِيرٌ ، ( وَ ) نَوْمٌ ( وَحْدَهُ ) ، لِحَدِيثِ ابْنِ عُمَرَ مَرْفُوعًا { نَهَى عَنْ الْوَحْدَةِ } ( كَ ) مَا يُكْرَهُ شُرُوعُهُ فِي ( سَفَرٍ ) وَحْدَهُ لِحَدِيثِ { الْوَاحِدُ شَيْطَانٌ } ( وَ ) نَوْمٌ ( بَيْنَ أَيْقَاظٍ ) ، لِإِيذَائِهِمْ وَإِخْلَالِهِ بِالْمُرُوءَةِ ، ( وَ ) كَذَا ( نَوْمٌ وَجُلُوسٌ بَيْنَ شَمْسٍ ، وَظِلٍّ ) لِنَهْيِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ رَوَاهُ أَحْمَدُ .
