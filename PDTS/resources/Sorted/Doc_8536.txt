( وَإِذَا دَخَلَ وَقْتُ صَلَاةٍ ) مَكْتُوبَةٍ ( بِقَدْرِ ) تَكْبِيرَةِ ( إحْرَامٍ ) كَمَا لَوْ زَالَتْ الشَّمْسُ ( ثُمَّ ) بَعْدَ مُضِيِّ قَدْرِ تَكْبِيرَةٍ فَأَكْثَرَ ( طَرَأَ مَانِعٌ ) مِنْ الصَّلَاةِ ( بِجُنُونٍ وَحَيْضٍ وَنِفَاسٍ وَرِدَّةٍ ) ، ثُمَّ زَالَ الْمَانِعُ : ( قُضِيَتْ ) تِلْكَ الصَّلَاةُ الَّتِي أَدْرَكَ وَقْتَهَا ، لِوُجُوبِهَا بِدُخُولِهِ عَلَى مُكَلَّفٍ لَا مَانِعَ بِهِ وُجُوبًا مُسْتَقِرًّا ، فَإِذَا قَامَ بِهِ مَانِعٌ بَعْدَ ذَلِكَ لَمْ يُسْقِطْهَا فَوَجَبَ قَضَاؤُهَا عِنْدَ زَوَالِهِ ( فَقَطْ ) ، فَلَا يَلْزَمُهُ قَضَاءُ مَا بَعْدَهَا ، وَلَوْ جَمَعَ إلَيْهَا( 2 / 290 )
