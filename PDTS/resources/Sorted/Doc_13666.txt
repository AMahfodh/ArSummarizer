( فَصْلٌ ) لَوْ ( عَاقَدَ الْإِمَامُ عِلْجًا ) وَهُوَ الْكَافِرُ الْغَلِيظُ الشَّدِيدُ سُمِّيَ بِهِ لِدَفْعِهِ عَنْ نَفْسِهِ بِقُوَّتِهِ وَمِنْهُ سُمِّيَ الْعِلَاجُ عِلَاجًا لِدَفْعِهِ الدَّاءَ ( لِيَدُلَّ عَلَى قَلْعَةٍ ) بِإِسْكَانِ اللَّامِ أَشْهُرُ مِنْ فَتْحِهَا ( وَلَوْ كَانَ الْإِمَامُ ) نَازِلًا ( تَحْتَهَا وَهُوَ لَا يَدْرِي ) بِهَا ( بِجَارِيَةٍ مُعَيَّنَةٍ أَوْ مُبْهَمَةٍ مِنْهَا لَا مِنْ غَيْرِهَا اسْتَحَقَّهَا ) وَفَاءً بِالشَّرْطِ وَصَحَّ ذَلِكَ مَعَ إبْهَامِهَا وَعَدَمِ مِلْكِهَا وَالْقُدْرَةِ عَلَى تَسْلِمِيهَا لِلْحَاجَةِ إلَيْهِ رَقِيقَةً كَانَتْ أَوْ حُرَّةً ؛ لِأَنَّهَا تَرِقُّ بِالْأَسْرِ هَذَا ( إنْ فُتِحَتْ ) أَيْ الْقَلْعَةُ ( بِدَلَالَتِهِ وَلَوْ فِي وَقْتٍ آخَرَ ) كَأَنْ تَرَكْنَاهَا ثُمَّ عُدْنَا إلَيْهَا ( وَلَوْ لَمْ يَظْفَرْ ) مِنْهَا ( بِغَيْرِهَا ) أَيْ بِغَيْرِ الْجَارِيَةِ بِخِلَافِ مَا لَوْ عَاقَدَهُ بِجَارِيَةٍ مِنْ غَيْرِهَا فَيُعْتَبَرُ فِي الصِّحَّةِ مَا يُعْتَبَرُ فِيهَا فِي سَائِرِ الْجَعَالَاتِ وَخَرَجَ بِالْعِلْجِ مَا لَوْ عَاقَدَ مُسْلِمًا بِمَا ذُكِرَ ؛ لِأَنَّ فِيهِ أَنْوَاعَ غَرَرٍ فَلَا يَحْتَمِلُ مَعَهُ وَاحْتَمَلَتْ مَعَ الْكَافِرِ ؛ لِأَنَّهُ أَعْرَفُ بِأَحْوَالِ قِلَعِهِمْ وَطُرُقِهِمْ غَالِبًا وَلِأَنَّ الْمُسْلِمَ يَتَعَيَّنُ عَلَيْهِ فَرْضُ الْجِهَادِ وَالدَّلَالَةُ نَوْعٌ مِنْهُ فَلَا يَجُوزُ أَخْذُ الْعِوَضِ عَلَيْهِ .
