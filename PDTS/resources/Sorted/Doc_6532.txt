وَالثَّالِثُ أَنَّهُ مِنْ خُمُسِ الْخُمُسِ سَهْمَ الْمَصَالِحِ ( قُلْت : إنَّمَا ) ( يُرْضَخُ لِذِمِّيٍّ ) وَمَا أُلْحِقَ بِهِ مِنْ الْكُفَّارِ ( حَضَرَ بِلَا أُجْرَةٍ ) وَلَوْ بِجَعَالَةٍ وَإِلَّا فَلَا شَيْءَ لَهُ غَيْرُهَا جَزْمًا وَإِنْ زَادَتْ عَلَى سَهْمِ رَاجِلٍ ( وَ ) كَانَ حُضُورُهُ ( بِإِذْنِ الْإِمَامِ ) أَوْ الْأَمِيرِ وَبِلَا إكْرَاهٍ مِنْهُ ( عَلَى الصَّحِيحِ ، وَاَللَّهُ أَعْلَمُ ) وَإِلَّا فَإِنْ أَكْرَهَهُ الْإِمَامُ أَوْ نَائِبُهُ الْأَمِيرُ عَلَى الْحُضُورِ فَلَهُ أُجْرَةُ مِثْلِهِ فَمَا يَظْهَرُ وَلَا أَثَرَ لِإِذْنِ الْآحَادِ .
