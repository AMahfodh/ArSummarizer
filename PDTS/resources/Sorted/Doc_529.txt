قَوْلُ الْمَتْنِ ( جُنُبٌ ) أَيْ أَوْ نَحْوُهُ كَحَائِضٍ وَنُفَسَاءَ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : وَهُوَ مَعَ أَهْلِهِ ) الْجُمْلَةُ حَالٌ مِنْ ضَمِيرِ سَمَاعِهِ الْفَاعِلِ فِي الْمَعْنَى ( قَوْلُهُ : إلَيْهَا ) أَيْ الدَّعْوَةِ وَالْجَارُّ مُتَعَلِّقٌ بِالْخُرُوجِ ( قَوْلُهُ : كَمَا مَرَّ ) أَيْ فِي الْغُسْلِ .( 11 / 224 )
