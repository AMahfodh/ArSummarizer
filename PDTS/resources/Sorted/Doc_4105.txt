( وَإِنْ قَالَ ) مَنْ لَهُ ثَلَاثُ زَوْجَاتٍ مُشِيرًا إلَيْهِنَّ ( هَذِهِ أَوْ هَذِهِ ) طَالِقٌ ( ، وَهَذِهِ ) طَالِقٌ ( وَقَعَ ) الطَّلَاقُ ( بِالثَّلَاثَةِ ) لِإِيقَاعِهِ بِهَا ( وَ ) وَقَعَ ( بِإِحْدَى الْأَوَّلَيْنِ ) ؛ لِأَنَّ أَوْ لِأَحَدِ الشَّيْئَيْنِ ( كَ ) مَا لَوْ قَالَ ( هَذِهِ أَوْ هَذِهِ ) طَالِقٌ ( بَلْ هَذِهِ ) طَالِقٌ فَيَقَعُ بِالثَّالِثَةِ ، وَإِحْدَى الْأَوَّلِيَّيْنِ .
