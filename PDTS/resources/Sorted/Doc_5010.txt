( قَوْلُهُ : تَقَاوَيَاهُ ) أَيْ تَزَايَدَا فِيهِ حَتَّى يَقِفَ عَلَى حَدٍّ يَلْتَزِمُهُ أَحَدُهُمَا بِهِ وَالتَّقَاوِي مَأْخُوذٌ مِنْ الْقُوَّةِ ؛ لِأَنَّ كُلَّ وَاحِدٍ مِنْ الشَّرِيكَيْنِ يُظْهِرُ قُوَّتَهُ ( قَوْلُهُ : وَلَا يُقَوَّمُ عَلَى مَنْ دَبَّرَ ) أَيْ نَصِيبَ شَرِيكِهِ لِيُكْمِلَ عَلَيْهِ ( قَوْلُهُ : مَعْنَاهُ ) أَيْ التَّقَاوِي ( قَوْلُهُ : أَتَسَلَّمُهُ لِصَاحِبِك ) أَيْ الْمُتَمَسِّكِ بِالرُّقْيَةِ ( قَوْلُهُ : حَتَّى يَقِفَ عَلَى حَدٍّ ) أَيْ يَلْتَزِمَهُ أَحَدُهُمَا بِهِ .
