وَلَوْ أُمَّ وَلَدٍ ( وَمُكَاتَبَةٌ ) وَلَوْ حُكْمًا كَمُعْتَقَةِ بَعْضٍ ( عَتَقَتْ تَحْتَ حُرٍّ أَوْ عَبْدٍ وَلَوْ كَانَ النِّكَاحُ بِرِضَاهَا ) دَفْعًا لِزِيَادَةِ الْمِلْكِ عَلَيْهَا بِطَلْقَةٍ ثَالِثَةٍ ، فَإِنْ اخْتَارَتْ نَفْسَهَا فَلَا مَهْرَ لَهَا أَوْ زَوْجَهَا فَالْمَهْرُ لِسَيِّدِهَا ، وَلَوْ صَغِيرَةً تُؤَخَّرُ لِبُلُوغِهَا ، وَلَيْسَ لَهَا خِيَارُ بُلُوغٍ فِي الْأَصَحِّ ( أَوْ ) ( كَانَتْ ) الْأَمَةُ ( عِنْدَ النِّكَاحِ حُرَّةً ثُمَّ صَارَتْ أَمَةً ) بِأَنْ ارْتَدَّا وَلَحِقَا بِدَارِ الْحَرْبِ ثُمَّ سُبِيَا مَعًا فَأُعْتِقَتْ خُيِّرَتْ عِنْدَ الثَّانِي خِلَافًا لِلثَّالِثِ مَبْسُوطٌ ( وَالْجَهْلُ بِهَذَا الْخِيَارِ ) خِيَارِ الْعِتْقِ ( عُذْرٌ ) فَلَوْ لَمْ تَعْلَمْ بِهِ حَتَّى ارْتَدَّا وَلَحِقَا فَعَلِمَتْ فَفَسَخَتْ صَحَّ إلَّا إذَا قَضَى بِاللِّحَاقِ وَلَيْسَ هَذَا حُكْمًا بَلْ فَتْوَى كَافِيٌّ ( وَلَا يَتَوَقَّفُ عَلَى الْقَضَاءِ ) وَلَا يَبْطُلُ بِسُكُوتٍ وَلَا يَثْبُتُ لِغُلَامٍ وَيَقْتَصِرُ عَلَى مَجْلِسٍ كَخِيَارِ مُخَيَّرَةٍ ، بِخِلَافِ خِيَارِ الْبُلُوغِ فِي الْكُلِّ خَانِيَّةٌ .
