( وَإِنْ قَالَ ) الْمُوصِي : اعْتِقُوا ( أَرْبَعَةَ ) أَعْبُدٍ ( بِكَذَا ) كَخَمْسِمِائَةٍ ( جَازَ الْفَضْلُ بَيْنَهُمْ ) بِأَنْ يُشْتَرَى وَاحِدٌ بِمِائَةٍ وَآخَرُ بِمِائَتَيْنِ وَعِشْرِينَ ، وَآخَرُ بِثَمَانِينَ ؛ لِأَنَّ لَفْظَهُ يَحْتَمِلُ ذَلِكَ ( مَا لَمْ يُسَمِّ لِكُلٍّ ) مِنْ الْأَرْبَعَةِ ( ثَمَنًا مَعْلُومًا ) فَيَتَعَيَّنُ عَلَى مَا قَالَ .( 13 / 180 )
