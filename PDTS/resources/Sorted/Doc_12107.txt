( فَلَوْ ) كَانَ ( جَارِحًا ) كَبَازٍ ( فَقَتَلَ حَمَامَ الْحَرَمِ فَلَا شَيْءَ عَلَيْهِ ) لِفِعْلِهِ مَا وَجَبَ عَلَيْهِ ( فَلَوْ بَاعَهُ رَدَّ الْمَبِيعَ إنْ بَقِيَ وَإِلَّا فَعَلَيْهِ الْجَزَاءُ ) لِأَنَّ حُرْمَةَ الْحَرَمِ وَالْإِحْرَامِ تَمْنَعُ بَيْعَ الصَّيْدِ ( وَلَوْ أَخَذَ حَلَالٌ صَيْدًا فَأَحْرَمَ ضَمِنَ مُرْسِلُهُ ) مِنْ يَدِهِ الْحُكْمِيَّةِ اتِّفَاقًا ، وَمِنْ الْحَقِيقِيَّةِ عِنْدَهُ خِلَافًا لَهُمَا ، وَقَوْلُهُمَا اسْتِحْسَانٌ كَمَا فِي الْبُرْهَانِ ( وَلَوْ أَخَذَهُ مُحْرِمٌ لَا ) يَضْمَنُ مُرْسِلُهُ اتِّفَاقًا لِأَنَّ الْمُحْرِمَ لَمْ يَمْلِكْهُ ، وَحِينَئِذٍ فَلَا يَأْخُذُهُ مِمَّنْ أَخَذَهُ ( وَالصَّيْدُ لَا يَمْلِكُهُ الْمُحْرِمُ بِسَبَبٍ اخْتِيَارِيٍّ ) كَشِرَاءٍ وَهِبَةٍ ( بَلْ ) بِسَبَبٍ ( جَبْرِيٍّ ) وَالسَّبَبُ الْجَبْرِيُّ فِي إحْدَى عَشْرَ مَسْأَلَةً مَبْسُوطَةً فِي الْأَشْبَاهِ فَلِذَا قَالَ تَبَعًا لِلْبَحْرِ عَنْ الْمُحِيطِ ( كَالْإِرْثِ ) وَجَعَلَهُ فِي الْأَشْبَاهِ بِالِاتِّفَاقِ ، لَكِنْ فِي النَّهْرِ عَنْ السِّرَاجِ أَنَّهُ لَا يَمْلِكُهُ بِالْمِيرَاثِ وَهُوَ الظَّاهِرُ
