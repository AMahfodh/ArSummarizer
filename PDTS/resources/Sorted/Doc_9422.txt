وَلِلشَّقِيقِ الثُّلُثَانِ كَمَا أَشَارَ لَهُ بِقَوْلِهِ ( ثُمَّ ) إذَا أَخَذَ الشَّقِيقُ نَصِيبَهُ ( رَجَعَ ) عَلَى الَّذِي لِلْأَبِ فَيَأْخُذُ مَا صَارَ لَهُ بِالْمُعَادَّةِ ؛ لِأَنَّهُ يَحْجُبُ الَّذِي لِلْأَبِ وَشَبَّهَ فِي الرُّجُوعِ بَعْدَ الْمُقَاسَمَةِ لِلْجَدِّ قَوْلَهُ ( كَالشَّقِيقَةِ ) تَعُدُّ عَلَى الْجَدِّ الْإِخْوَةَ لِلْأَبِ ثُمَّ تَرْجِعُ عَلَيْهِمْ ( بِمَالِهَا ) وَهُوَ النِّصْفُ لِلْوَاحِدَةِ وَالثُّلُثَانِ لِلْأَكْثَرِ ( لَوْ لَمْ يَكُنْ جَدٌّ ) وَمَا فَضَلَ بَعْدَ ذَلِكَ فَهُوَ لِلْأَخِ وَالْإِخْوَةِ لِلْأَبِ فَمَعْنَى كَلَامِهِ أَنْ يَعُدَّ الشَّقِيقُ ذَكَرًا أَوْ أُنْثَى عَلَى الْجَدِّ جِنْسَ الْإِخْوَةِ لِلْأَبِ ثُمَّ بَعْدَ عَدِّهِمْ يَسْقُطُ الَّذِي لِلْأَبِ بِالشَّقِيقِ وَسَوَاءٌ كَانَ مَعَهُمْ ذُو سَهْمٍ كَأُمٍّ أَوْ زَوْجَةٍ أَمْ لَا .
