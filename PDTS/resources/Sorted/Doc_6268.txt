( وَ ) أَمَّا ( إنْ تَلِفَتْ ) اللُّقَطَةُ ( أَوْ نَقَصَتْ قَبْلَهُ ) أَيْ : الْحَوْلِ بِيَدِ مُلْتَقِطٍ ( وَلَمْ يُفَرِّطْ لَمْ يَضْمَنْهَا ) ؛ لِأَنَّهَا أَمَانَةٌ بِيَدِهِ كَالْوَدِيعَةِ ، ( وَ ) إنْ تَلِفَتْ أَوْ نَقَصَتْ ( بَعْدَهُ ) أَيْ : الْحَوْلِ ( يَضْمَنُهَا ) مُلْتَقِطٌ ( مُطْلَقًا ) أَيْ فَرَّطَ أَوْ لَا ، لِدُخُولِهَا فِي مِلْكِهِ فَتَلَفُهَا مِنْ مَالِهِ ، وَمِلْكُ الْمُلْتَقِطِ لَهَا مُرَاعًى يَزُولُ بِمَجِيءِ صَاحِبِهَا وَيَضْمَنُ لَهُ بَدَلَهَا إنْ تَعَذَّرَ رَدُّهَا .
