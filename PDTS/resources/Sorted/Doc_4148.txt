( وَ ) إنْ وَصَّى لِحُرٍّ ( بِأَبِيهِ ) الرَّقِيقِ ( فَمَاتَ ) مُوصًى لَهُ بَعْدَ مَوْتِ مُوصٍ وَ ( قَبْلَ قَبُولِهِ ) الْوَصِيَّةَ ( فَقَبِلَ ابْنُهُ ) أَيْ ابْنُ الْمُوصَى لَهُ الْوَصِيَّةَ بِجَدِّهِ ( عَتَقَ الْمُوصَى بِهِ حِينَئِذٍ ) أَيْ حِينَ قَبُولِ الْوَصِيَّةِ لِمِلْكِ ابْنِ ابْنِهِ لَهُ إذَنْ ( وَلَمْ يَرِثْ ) الْعَتِيقُ مِنْ ابْنِهِ الْمَيِّتِ ، لِحُدُوثِ حُرِّيَّتِهِ بَعْدَ أَنْ صَارَ الْمِيرَاثُ لِغَيْرِهِ .
