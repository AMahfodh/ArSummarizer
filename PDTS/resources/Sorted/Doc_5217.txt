( وَإِنْ تَخَلَّفَ ) مَأْمُومٌ ( عَنْهُ ) أَيْ : عَنْ إمَامِهِ ( بِرُكْنٍ ) فَعَلَى أَيْ : رُكْنٍ كَانَ ( فَأَكْثَرَ بِلَا عُذْرٍ ) مِنْ نَوْمٍ وَسَهْوٍ وَزِحَامٍ وَعَجَلَةٍ ، ( فَكَسَبْقٍ ، فَتَبْطُلُ ) الصَّلَاةُ بِهِ ( لِعَامِدٍ ، وَتَصِحُّ لِجَاهِلٍ وَنَاسٍ ) ، وَقَوْلُهُ : ( وَتَبْطُلُ رَكْعَةُ ) جَاهِلٍ وَنَاسٍ ( بِرُكُوعِ ) إمَامِهِ ، ( لِتَخَلُّفِهِ ) عَنْهُ بِمَا هُوَ مُعْظَمُ الرَّكْعَةِ ، فِيهِ نَظَرٌ .
