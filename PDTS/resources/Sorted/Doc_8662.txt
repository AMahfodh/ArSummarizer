( وَكَذَا ) لَا تَبْطُلُ الصَّلَاةُ إنْ انْكَشَفَ مِنْ الْعَوْرَةِ شَيْءٌ ( كَثِيرٌ فِي زَمَنٍ قَصِيرٍ ، فَلَوْ أَطَارَتْ الرِّيحُ سُتْرَتَهُ وَنَحْوُهُ ) أَيْ : نَحْوُ الرِّيحِ ( عَنْ عَوْرَتِهِ ، فَبَدَا ) أَيْ : ظَهَرَ ( مِنْهَا مَا لَمْ يُعْفَ عَنْهُ لَوْ ) طَالَ زَمَنُهُ لِفُحْشِهِ ( وَلَوْ ) كَانَ الَّذِي بَدَا ( كُلَّهَا ) أَيْ : كُلَّ الْعَوْرَةِ ( فَأَعَادَهَا سَرِيعًا بِلَا عَمَلٍ كَثِيرٍ لَمْ تَبْطُلْ ) صَلَاتُهُ ، لِقِصَرِ مُدَّتِهِ أَشْبَهَ الْيَسِيرَ فِي الزَّمَنِ الطَّوِيلِ فَإِنْ احْتَاجَ فِي أَخْذِ سُتْرَتِهِ لِعَمَلٍ كَثِيرٍ بَطَلَتْ صَلَاتُهُ .( 2 / 259 )
