الْأَمْرِ } لَكِنْ نَصَّ الشَّافِعِيُّ عَلَى عَدَمِ وُجُوبِهَا عَلَيْهِ حَكَاهُ الْبَيْهَقِيُّ فِي الْمَعْرِفَةِ عِنْدَ اسْتِئْذَانِ الْبِكْرِ ( وَتَغْيِيرُ مُنْكَرٍ رَآهُ ) قَالَ الْغَزَالِيُّ وَلَمْ يَعْلَمْ أَوْ يَظُنَّ أَنَّ فَاعِلَهُ يَزِيدُ فِيهِ عِنَادًا ( مُطْلَقًا ) عَنْ التَّقْيِيدِ بِعَدَمِ الْخَوْفِ ( وَمُصَابَرَةُ الْعَدُوِّ وَإِنْ كَثُرَ ) وَلَوْ زَادَ عَلَى الضِّعْفِ وَلَوْ مَعَ الْخَوْفِ ؛ لِأَنَّهُ مَوْعُودٌ بِالْعِصْمَةِ وَالنَّصْرِ ( وَقَضَاءُ دَيْنِ مُسْلِمٍ مَاتَ مُعْسِرًا ) لِخَبَرِ الصَّحِيحَيْنِ { أَنَا أَوْلَى بِالْمُؤْمِنِينَ مِنْ أَنْفُسِهِمْ فَمَنْ تُوُفِّيَ مِنْهُمْ فَتَرَك دَيْنًا فَعَلَيَّ قَضَاؤُهُ } وَقَيَّدَهُ الْإِمَامُ بِمَا إذَا اتَّسَعَ الْمَالُ .
