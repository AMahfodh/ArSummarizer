( قَوْلُهُ : وَحُكْمُهُ ) أَيْ : الْعُذْرِ أَوْ صَاحِبِهِ ( قَوْلُهُ : الْوُضُوءُ ) أَيْ : مَعَ الْقُدْرَةِ عَلَيْهِ وَإِلَّا فَالتَّيَمُّمُ ( قَوْلُهُ : لَا غَسْلُ ثَوْبِهِ ) أَيْ : إنْ لَمْ يُفِدْ كَمَا يَأْتِي مَتْنًا ( قَوْلُهُ : وَنَحْوِهِ ) كَالْبَدَنِ وَالْمَكَانِ ط ( قَوْلُهُ : اللَّامُ لِلْوَقْتِ ) أَيْ : فَالْمَعْنَى لِوَقْتِ كُلِّ صَلَاةٍ ، بِقَرِينَةِ قَوْلِهِ بَعْدَهُ فَإِذَا خَرَجَ الْوَقْتُ بَطَلَ ، فَلَا يَجِبُ لِكُلِّ صَلَاةٍ خِلَافًا لِلشَّافِعِيِّ أَخْذًا مِنْ حَدِيثِ { تَوَضَّئِي لِكُلِّ صَلَاةٍ } قَالَ فِي الْإِمْدَادِ : وَفِي شَرْحِ مُخْتَصَرِ الطَّحَاوِيِّ رَوَى أَبُو حَنِيفَةَ عَنْ هِشَامِ بْنِ عُرْوَةَ عَنْ أَبِيهِ عَنْ عَائِشَةَ رَضِيَ اللَّهُ عَنْهَا { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ لِفَاطِمَةَ بِنْتِ أَبِي حُبَيْشٍ : تَوَضَّئِي لِوَقْتِ كُلِّ صَلَاةٍ } وَلَا شَكَّ أَنَّهُ مُحْكَمٌ ؛ لِأَنَّهُ لَا يَحْتَمِلُ غَيْرَهُ بِخِلَافِ حَدِيثِ لِكُلِّ صَلَاةٍ فَإِنَّ لَفْظَ الصَّلَاةِ شَاعَ اسْتِعْمَالُهُ فِي لِسَانِ الشَّرْعِ وَالْعُرْفِ فِي وَقْتِهَا فَوَجَبَ حَمْلُهُ عَلَى الْمُحْكَمِ وَتَمَامُهُ فِيهِ .
