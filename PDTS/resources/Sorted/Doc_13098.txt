( قَوْلُهُ : وَلَا بَأْسَ بِهِ ) أَيْ بِرِبَا الْفَضْلِ فِي مُخْتَلِفِ الْجِنْسِ فَيَجُوزُ بَيْعُ ذَهَبٍ بِفِضَّةٍ مُتَفَاضِلًا إذَا كَانَ يَدًا بِيَدٍ وَبَيْعُ قَمْحٍ بِأُرْزٍ أَوْ فُولٍ مُتَفَاضِلًا إذَا كَانَ يَدًا بِيَدٍ ( قَوْلُهُ : مُطْلَقًا ) أَيْ اتَّحَدَ الْجِنْسُ أَوْ اخْتَلَفَ فَلَا يَجُوزُ بَيْعُ ذَهَبٍ بِذَهَبٍ قَدْرُهُ لِأَجَلٍ وَلَا فِضَّةٍ بِفِضَّةٍ قَدْرُهَا لِأَجَلٍ وَلَا بَيْعُ ذَهَبٍ بِفِضَّةٍ قَدْرُهَا أَوْ أَكْثَرَ مِنْهَا أَوْ أَقَلَّ لِأَجَلٍ ( قَوْلُهُ : وَكَذَا فِي الطَّعَامِ ) أَيْ مُطْلَقًا اتَّحَدَ الْجِنْسُ أَوْ اخْتَلَفَ فَلَا يَجُوزُ بَيْعُ إرْدَبِّ قَمْحٍ بِمِثْلِهِ أَوْ بِإِرْدَبِّ فُولٍ لِأَجَلٍ ( قَوْلُهُ : وَلَوْ غَيْرَ رِبَوِيٍّ ) أَيْ كَخَوْخٍ وَتُفَّاحٍ فَلَا يَجُوزُ بَيْعُ قِنْطَارٍ مِنْ أَحَدِهِمَا بِقِنْطَارٍ مِنْ الْآخَرِ لِأَجَلٍ ( قَوْلُهُ : فَكُلُّ مَا يَدْخُلُهُ رِبَا الْفَضْلِ ) أَيْ وَهُوَ النَّقْدُ وَالطَّعَامُ الرِّبَوِيُّ ( قَوْلُهُ : دُونَ عَكْسٍ ) أَيْ وَلَيْسَ كُلُّ مَا يَدْخُلُهُ رِبَا النَّسَاءِ يَدْخُلُهُ رِبَا الْفَضْلِ أَيْ لِأَنَّ الطَّعَامَ غَيْرُ الرِّبَوِيِّ يَدْخُلُهُ رِبَا النَّسَاءِ وَلَا يَدْخُلُهُ( 11 / 72 )
