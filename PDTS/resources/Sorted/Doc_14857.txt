وَمَا قَالَهُ هُوَ قَضِيَّةُ نَصِّ الشَّافِعِيِّ فِي الْأُمِّ فِي آخِرِ سِيَرِ الْوَاقِدِيِّ وَيُؤْخَذُ مِنْ تَوْجِيهِهِ أَنَّ ذَلِكَ فِي أَمَةٍ مُعَيَّنَةٍ وَهُوَ ظَاهِرٌ ، أَمَّا إذَا أَسْلَمَتْ قَبْلَ الْعَقْدِ فَلَا شَيْءَ لَهُ إذَا عَلِمَ بِذَلِكَ وَبِأَنَّهَا قَدْ فَاتَتْهُ ؛ لِأَنَّهُ عَمِلَ مُتَبَرِّعًا ذَكَرَهُ الْبُلْقِينِيُّ وَكَلَامُ غَيْرِهِ يَقْتَضِيهِ ( قُلْتُ ) مَحَلُّ مَا ذُكِرَ فِي الْمَيْتَةِ ( إذَا تَمُوتُ بَعْدَ الظَّفَرِ ) وَقَبْلَ التَّسْلِيمِ ( فَإِنْ تَمُتْ قَبْلُ ) أَيْ : قَبْلَ الظَّفَرِ ( فَلَا ) شَيْءَ لَهُ ( فِي الْأَظْهَرِ ) فِي أَصْلِ الرَّوْضَةِ لِعَدَمِ الْقُدْرَةِ عَلَيْهَا ، وَقَضِيَّتُهُ أَنَّ الْحُكْمَ كَذَلِكَ بَعْدَ الظَّفَرِ وَقَبْلَ الْقُدْرَةِ عَلَيْهَا وَلَيْسَ كَذَلِكَ ، وَمُقَابِلُ الْأَظْهَرِ اسْتِحْقَاقُهُ وَرَجَّحَهُ الْبُلْقِينِيُّ وَقَالَ إنَّهُ الْمَنْصُوصُ فِي الْأُمِّ وَالْمُخْتَصَرِ وَلَيْسَ لَهُ نَصٌّ يُخَالِفُهُ وَلَوْ هَرَبَتْ فَهِيَ كَمَا لَوْ مَاتَتْ ( أَمَّا الَّتِي قَدْ أَسْلَمَتْ فَالْمَذْهَبُ ) قَائِلٌ ( بِأَنَّ أَجْرَ الْمِثْلِ ) لِلْعَمَلِ ( عَنْهَا يَجِبُ ) لَا قِيمَتُهَا كَمَا فِي الْحَاوِي وَتَبِعَ النَّاظِمُ فِي هَذَا الْمِنْهَاجَ كَأَصْلِهِ وَالْمَنْصُوصُ فِي الْأُمِّ مَا نَقَلَهُ فِي الرَّوْضَةِ كَأَصْلِهَا عَنْ الْجُمْهُورِ أَنَّ الْوَاجِبَ قِيمَتُهَا كَمَا فِي الْحَاوِي وَمَا تَقَرَّرَ مَحَلُّهُ فِي حِصْنٍ فُتِحَ عَنْوَةً لَا صُلْحًا كَمَا أَفَادَهُ قَوْلُهُ ( لَكِنْ زَعِيمُ الْحِصْنِ ) أَيْ : سَيِّدُ أَهْلِهِ ( إنْ نُؤْمِنْهُ وَأَهْلَهُ بِالصُّلْحِ وَهِيَ ) أَيْ : الْأُنْثَى ( مِنْهُ ) أَيْ : مِنْ أَهْلِ الزَّعِيمِ ( وَمَا رَضِي هَذَا وَلَا ذَا ) أَيْ : الزَّعِيمُ( 18 / 453 )
