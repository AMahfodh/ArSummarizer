وَفِي الْفُرُوعِ : وَيَتَوَجَّهُ أَنْ يَجِبَ لَهُ أُجْرَةُ مِثْلِهِ فَقَطْ لَا النَّفَقَةُ ، كَقَائِدِ الْأَعْمَى وَلَا دَلِيلَ يَخُصُّ وُجُوبَ النَّفَقَةِ ( وَمَنْ أَيِسَتْ مِنْهُ ) أَيْ الْمَحْرَمِ ( اسْتَنَابَتْ ) مَنْ يَفْعَلُ النُّسُكَ عَنْهَا ، كَكَبِيرٍ عَاجِزٍ فَإِنْ تَزَوَّجَتْ بَعْدُ فَحُكْمُهَا كَالْمَعْضُوبِ وَالْمُرَادُ أَيِسَتْ بَعْدَ أَنْ وَجَدَتْ الْمَحْرَمَ وَفَرَّطَتْ بِالتَّأْخِيرِ حَتَّى فُقِدَ ؛ لِمَا قَدَّمَاهُ مِنْ نَصِّ الْإِمَامِ ( وَإِنْ حَجَّتْ امْرَأَةٌ بِدُونِهِ ) أَيْ الْمَحْرَمِ ( حَرُمَ ) سَفَرُهَا بِدُونِهِ ( وَأَجْزَأَهَا ) حَجُّهَا كَمَنْ حَجَّ وَتَرَكَ حَقًّا يَلْزَمُهُ مِنْ نَحْوِ( 3 / 429 )
