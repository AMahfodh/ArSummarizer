( وَحُكْمُ الْعَامِلِ ) فِي مُسَاقَاةٍ وَمُزَارَعَةٍ ( حُكْمُ الْمُضَارِبِ فِيمَا يُقْبَلُ قَوْلُهُ وَ ) فِي ( مَا يُرَدُّ ) قَوْلُهُ فِيهِ ؛ لِأَنَّ رَبَّ الْمَالِ ائْتَمَنَهُ بِدَفْعِ مَالِهِ ( فَإِنْ اتَّهَمَ ) رَبُّ الْمَالِ الْعَامِلَ بِخِيَانَةٍ ( حَلَفَ ) الْعَامِلُ ، لِاحْتِمَالِ صِدْقِ الْمُدَّعِي ( وَإِنْ ثَبَتَتْ خِيَانَتُهُ ) قَبْلَ تَمَامِ الْعَمَلِ بِإِقْرَارِهِ أَوْ بِبَيِّنَةٍ أَوْ نُكُولِهِ ( ضُمَّ إلَيْهِ مَنْ يُشَارِكُهُ كَالْوَصِيِّ إذَا ثَبَتَتْ خِيَانَتُهُ ) تَحْصِيلًا لِلْغَرَضَيْنِ .
