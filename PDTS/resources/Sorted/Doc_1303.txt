( وَإِنْ بَنَاهُ ) أَيْ : الْحَائِطَ ( مَائِلًا إلَى مِلْكِ غَيْرِهِ بِإِذْنِهِ أَوْ ) بَنَاهُ مَائِلًا ( إلَى مِلْكِ نَفْسِهِ ) لَمْ يَضْمَنْ لِعَدَمِ تَعَدِّيهِ ( أَوْ مَالَ ) الْحَائِطُ ( إلَيْهِ ) أَيْ : إلَى مِلْكِ رَبِّهِ ( بَعْدَ الْبِنَاءِ لَمْ يَضْمَنْ ) رَبُّهُ مَا تَلِفَ بِهِ .( 13 / 233 )
