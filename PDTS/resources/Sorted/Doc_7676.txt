و ( قَوْلُهُ : لِذَلِكَ ) أَيْ لِمَا بَعْدَ النَّوْمِ ( قَوْلُهُ : لِلْأَمْرِ ) إلَى قَوْلِهِ عَلَى أَنَّ الْقَصْدَ فِي الْمُغْنِي ( قَوْلُهُ : وَبِهِ إلَخْ ) أَيْ بِالْوِتْرِ بَعْدَ النَّوْمِ ( قَوْلُهُ : فَمَا وَقَعَ لَهُمَا إلَخْ ) أَيْ فِي غَيْرِ الْمِنْهَاجِ ( قَوْلُهُ : مِنْ صِدْقِهِ عَلَيْهِ ) أَيْ صِدْقِ التَّهَجُّدِ عَلَى الْوِتْرِ وَيَحْتَمِلُ الْعَكْسَ ( قَوْلُهُ أَوَّلًا ) أَيْ قَبْلَ النَّسْخِ ( قَوْلُهُ : وَأَنَّ الَّذِي اُخْتُلِفَ إلَخْ ) عِبَارَةُ الرَّوْضِ فِي بَابِ النِّكَاحِ وَنُسِخَ وُجُوبُ التَّهَجُّدِ عَلَيْهِ لَا الْوِتْرُ انْتَهَى .
