( وَإِنْ اشْتَرَى ) الْمُبْتَاعُ ( أَصْلِهَا ) أَيْ الثَّمَرَةِ ( فَقَطْ ) أَيْ دُونَ الثَّمَرَةِ لِعَدَمِ وُجُودِهَا فِيهِ حِينَ الشِّرَاءِ ثُمَّ أَثْمَرَ وَقَامَ الشَّفِيعُ ( أُخِذَتْ ) بِضَمٍّ فَكَسْرٍ الثَّمَرَةُ مَعَ أَصْلِهَا بِالشُّفْعَةِ إنْ لَمْ تُؤَبَّرْ ، بَلْ ( وَإِنْ أُبِّرَتْ ) بِضَمٍّ فَكَسْرٍ مُثَقَّلًا الثَّمَرَةُ قَبْلَ قِيَامِ الشَّفِيعِ مَا لَمْ تَيْبَسْ أَوْ تُجَذَّ ( وَرَجَعَ ) الْمُشْتَرِي عَلَى الشَّفِيعِ ( بِالْمُؤْنَةِ ) لِلثَّمَرَةِ مِنْ تَأْبِيرٍ وَسَقْيٍ وَنَحْوِهِمَا وَالْقَوْلُ لَهُ فِي قَدْرِ الْمُؤْنَةِ بِيَمِينِهِ مَا لَمْ يَظْهَرْ كَذِبُهُ .
