( وَإِنْ خَرَقَهَا ) أَيْ : خَرَقَ السَّفِينَةَ إنْسَانٌ ( عَمْدًا فَغَرِقَتْ بِمَنْ فِيهَا ) مِنْ الْأَنْفُسِ وَالْأَمْوَالِ ( وَهُوَ ) أَيْ : خَرْقُهُ إيَّاهَا ( مِمَّا يُغْرِقُهَا غَالِبًا أَوْ يُهْلِكُ مَنْ فِيهَا ) غَالِبًا ( لِكَوْنِهِمْ فِي اللُّجَّةِ أَوْ لِعَدَمِ مَعْرِفَتِهِمْ بِالسِّبَاحَةِ ) وَإِنْ لَمْ يَكُونُوا فِي اللُّجَّةِ ( فَعَلَيْهِ ) أَيْ : الْخَارِقِ لَهَا ( الْقِصَاصُ إنْ قُتِلَ ) بِسَبَبِ ذَلِكَ ( مَنْ يَجِبُ الْقِصَاصُ بِقَتْلِهِ ) ؛ لِأَنَّهُ أَهْلَكَهُ بِفِعْلِهِ .
