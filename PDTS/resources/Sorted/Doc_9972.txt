وَالْأُمِّ ) تَكْمِلَةُ الثُّلُثَيْنِ كَمَا فِي الْبِنْتِ وَبَنَاتِ الِابْنِ ( وَهُوَ ) أَيْ السُّدُسُ ( فَرْضُ الْأَبِ مَعَ الْوَلَدِ ) ذَكَرًا كَانَ أَوْ غَيْرَهُ ( أَوْ ) مَعَ ( وَلَدِ الِابْنِ ) وَإِنْ سَفَلَ ( وَ ) هُوَ أَيْضًا ( فَرْضُ الْجَدِّ ) لِلْأَبِ ( عِنْدَ عَدَمِ الْأَبِ ) الْمُتَوَسِّطِ بَيْنَهُ وَبَيْنَ الْمَيِّتِ إذَا كَانَ لِلْمَيِّتِ وَلَدٌ أَوْ وَلَدُ ابْنٍ لِقَوْلِهِ تَعَالَى : { وَلِأَبَوَيْهِ لِكُلِّ وَاحِدٍ مِنْهُمَا السُّدُسُ } الْآيَةَ وَوَلَدُ الِابْنِ كَالْوَلَدِ كَمَا مَرَّ وَالْجَدُّ كَالْأَبِ ( وَهُوَ ) أَيْضًا ( لِلْوَاحِدِ مِنْ وَلَدِ الْأُمِّ ) ذَكَرًا كَانَ أَوْ أُنْثَى أَوْ خُنْثَى لِقَوْلِهِ تَعَالَى : { وَلَهُ أَخٌ أَوْ أُخْتٌ } الْآيَةَ .
