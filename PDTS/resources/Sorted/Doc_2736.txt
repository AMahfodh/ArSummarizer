( وَطَحْنِ حَبٍّ ) مِنْ بُرٍّ أَوْ نَحْوِهِ ( أَوْ ) كَانَ ( حَبًّا فَصَارَ زَرْعًا أَوْ عَكْسَهُ ) بِأَنْ اشْتَرَى زَرْعًا فَحَصَدَهُ وَصَارَ حَبًّا ( أَوْ ) كَانَ ( نَوًى ) فَغَرَسَهُ ( فَنَبَتَ شَجَرًا ، أَوْ ) كَانَ ( بَيْضًا فَصَارَ فِرَاخًا ) وَنَحْوَ ذَلِكَ فَيُمْنَعُ الرُّجُوعُ وَيَكُونُ رَبُّهَا أُسْوَةَ الْغُرَمَاءِ ، لِأَنَّهُ لَمْ يَجِدْ مَتَاعَهُ بِعَيْنِهِ .
