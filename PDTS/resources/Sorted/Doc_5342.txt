( وَلَيْسَ لَهُ وَطْءُ جَارِيَةٍ لِمُكَاتَبِهِ وَلَا ) وَطْءُ ( مُكَاتَبَتِهِ ) أَيْ مُكَاتَبِهِ لِأَنَّ مِلْكَهُمَا لِلْمُكَاتَبِ بِدَلِيلِ صِحَّةِ تَصَرُّفِهِ فِيهِمَا ( فَإِنْ فَعَلَ ) بِأَنْ وَطِئَ جَارِيَةَ مُكَاتَبِهِ أَوْ مُكَاتَبَتِهِ ( أَثِمَ وَعُزِّرَ وَلَا حَدَّ ) لِشُبْهَةِ الْمِلْكِ لِأَنَّهُ مَالِك الْمَالِكِ ، فَهُوَ مَالِكٌ بِوَاسِطَةٍ ( وَعَلَيْهِ ) بِوَطْئِهِ لَهَا ( مَهْرُهَا لِسَيِّدِهَا ) الَّذِي هُوَ الْمُكَاتَبُ لِأَنَّهُ عِوَضُ مَنْفَعَتِهَا وَهِيَ لَهُ ، فَكَذَا عِوَضُهُمَا .( 16 / 311 )
