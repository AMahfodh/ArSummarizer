وَأَمَّا حَدِيثُ { لَا يُجْزِئُ وَلَدٌ وَالِدَهُ إلَّا أَنْ يَجِدَهُ مَمْلُوكًا فَيَشْتَرِيَهُ فَيُعْتِقَهُ } رَوَاهُ مُسْلِمٌ : فَقَوْلُهُ : ( فَيَشْتَرِيَهُ فَيُعْتِقَهُ ) يُحْتَمَلُ أَنَّهُ أَرَادَ بِقَوْلِهِ : ( فَيَشْتَرِيَهُ فَيُعْتِقَهُ ) بِشِرَائِهِ كَمَا يُقَالُ ضَرَبَهُ فَقَتَلَهُ ، وَالضَّرْبُ هُوَ الْقَتْلُ وَسَوَاءٌ مَلَكَهُ بِشِرَاءٍ أَوْ هِبَةٍ أَوْ إرْثٍ أَوْ غَنِيمَةٍ أَوْ غَيْرِهِمَا لِعُمُومِ الْخَبَرِ وَلَا يُعْتَقُ ابْنُ عَمِّهِ وَنَحْوِهِ بِمِلْكِهِ ؛ لِأَنَّهُ لَيْسَ بِمَحْرَمٍ .
