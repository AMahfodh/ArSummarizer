( أَوْ ادَّعَى ) وَاطِئُ امْرَأَةٍ ( أَنَّهَا زَوْجَتُهُ وَأَنْكَرَتْ ) زَوْجِيَّتَهُ ؛ فَلَا يُحَدُّ لِأَنَّ دَعْوَاهُ ذَلِكَ شُبْهَةٌ تَدْرَأُ الْحَدَّ عَنْهُ ، لِاحْتِمَالِ صِدْقِهِ وَلِلتِّرْمِذِيِّ مِنْ حَدِيثِ عَائِشَةَ مَرْفُوعًا : { ادْرَءُوا الْحُدُودَ عَنْ الْمُسْلِمِينَ مَا اسْتَطَعْتُمْ ، فَإِنْ كَانَ لَهُ مَخْرَجٌ فَخَلُّوا سَبِيلَهُ ، فَإِنَّ الْإِمَامَ أَنْ يُخْطِئَ فِي الْعَفْوِ خَيْرٌ مِنْ أَنْ يُخْطِئَ فِي الْعُقُوبَةِ } وَلِلدَّارَقُطْنِيِّ عَنْ ابْنِ مَسْعُودٍ وَمُعَاذِ بْنِ جَبَلٍ وَعُقْبَةَ بْنِ عَامِرٍ : إذَا اشْتَبَهَ عَلَيْكَ الْحَدُّ فَادْرَأْ مَا اسْتَطَعْتَ ( فَإِنْ أَقَرَّتْ هِيَ ) أَرْبَعَ مَرَّاتٍ ( أَنَّهُ زَنَى ) بِهَا مُطَاوِعَةً عَالِمَةً بِتَحْرِيمِهِ ( حُدَّتْ ) وَحْدَهَا وَلَا مَهْرَ نَصًّا مُؤَاخَذَةً لَهَا بِإِقْرَارِهَا .( 18 / 128 )
