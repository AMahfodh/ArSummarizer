وَالتَّبْقِيَةِ بِالْأُجْرَةِ كَمَا يُعْلَمُ مِنْ قَوْلِهِ وَلِبِنَاءِ الْمُشْتَرِي ( قَوْلُهُ : حُكْمُ بِنَاءِ الْمُسْتَعِيرِ وَغِرَاسِهِ ) أَيْ مِنْ التَّخْيِيرِ بَيْنَ الْأُمُورِ الثَّلَاثَةِ الْمُتَقَدِّمَةِ ( قَوْلُهُ : وَلِلشَّفِيعِ تَأْخِيرُ الْأَخْذِ ) أَيْ وَلَوْ كَانَ يُرِيدُ السَّفَرَ وَتَكُونُ غَيْبَتُهُ عُذْرًا أَوْ يُوَكِّلُ مَنْ يَتَمَلَّكُ لَهُ بَعْدَ الْحَصَادِ ( قَوْلُهُ : لَا تُسْتَحَقُّ ) أَيْ بِأَنْ حَدَثَتْ بَعْدَ الْعَقْدِ وَتَأَبَّرَتْ قَبْلَ الْأَخْذِ كَمَا تَقَدَّمَ ( قَوْلُهُ : الْمَنْعُ ) أَيْ مَنْعُ التَّأْخِيرِ ( قَوْلُهُ : وَيُمْكِنُ حَمْلُ الْجَوَازِ ) أَيْ لِلتَّأْخِيرِ ( قَوْلُهُ : تَنْقُصُ بِهَا ) أَيْ بِالْأَرْضِ ، وَقَوْلُهُ مَعَ بَقَائِهِ : أَيْ الثَّمَرِ .( 17 / 13 )
