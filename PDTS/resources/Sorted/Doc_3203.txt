( قَوْلُهُ : أَوْ شِبْهَ عَمْدٍ ) يَنْبَغِي أَوْ عَمْدًا وَيَسْتَحِقُّ الْمُقْسِمُ نِصْفَ الدِّيَةِ فِيهِ ( قَوْلُهُ : عَلَى مَا عَيَّنَهُ ) أَيْ مِنْ عَمْدٍ أَوْ خَطَأٍ أَوْ شِبْهِ عَمْدٍ ( قَوْلُهُ : حَلَفَ كُلٌّ خَمْسِينَ إلَخْ ) هَذَا إنْ لَمْ يَنْفِ كُلٌّ مَا أَثْبَتَهُ الْآخَرُ وَإِلَّا بَطَلَ اللَّوْثُ ( قَوْلُهُ : وَحِصَّتُهُ مِنْهُ ) أَيْ النِّصْفِ( 25 / 324 )
