( وَإِنْ اخْتَلَفَا فِي مَكَانِ الْمُحَلِّلِ ) بِأَنْ رَضِيَ أَحَدُهُمَا بِعُدُولِهِ عَنْ الْوَسَطِ وَلَمْ يَرْضَ الْآخَرُ أَوْ رَضِيَا بِتَرْكِ تَوَسُّطِهِ وَقَالَ أَحَدُهُمَا : يَكُونُ عَنْ الْيَمِينِ وَقَالَ الْآخَرُ : عَنْ الْيَسَارِ ( لَزِمَ تَوَسُّطُهُ ) فَعُلِمَ بِذَلِكَ مَا صَرَّحَ بِهِ الْأَصْلُ أَنَّهُ يَنْبَغِي أَنْ يُجْرِيَ فَرَسَهُ بَيْنَ فَرَسَيْهِمَا فَإِنْ لَمْ يَتَوَسَّطْهُمَا وَأَجْرَاهُ بِجَنْبِ أَحَدِهِمَا جَازَ إنْ تَرَاضَيَا بِهِ ( فَإِنْ تَنَازَعَ الْمُتَسَابِقَانِ فِي الْيَمِينِ ) وَالْيَسَارِ ( أَقْرَعَ ) بَيْنَهُمَا( 21 / 344 )
