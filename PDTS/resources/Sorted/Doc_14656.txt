( قَوْلُهُ : وَهَلْ إنْ صَامَ الْعِيدَ ) هَذَا ضَعِيفٌ ( قَوْلُهُ : أَوْ يُفْطِرُهُنَّ ) ظَاهِرُهُ أَنَّهُ مَطْلُوبٌ بِالْفِطْرِ وَلَيْسَ كَذَلِكَ بَلْ مَأْمُورٌ بِصَوْمِهِمَا عَلَى طَرِيقِ النَّدْبِ فِيمَا يَظْهَرُ ثُمَّ عَلَى الْقَوْلِ الْأَوَّلِ وَهُوَ صَوْمُ الْجَمِيعِ يَقْضِي مَا لَا يَصِحُّ صَوْمُهُ وَهُوَ يَوْمُ الْعِيدِ الْأَوَّلُ فَقَطْ عَلَى الرَّاجِحِ ( قَوْلُهُ : جَاهِلًا إلَخْ ) الْفَرْقُ بَيْنَهُ وَبَيْنَ قَوْلِهِ أَوْ غَافِلًا أَنَّ الْأَوَّلَ لَيْسَ عِنْدَهُ غَفْلَةٌ عَنْ الْعَدِّ بَلْ عَدَّ إلَّا أَنَّهُ جَهِلَ بِأَنْ اعْتَقَدَ أَنَّهُ فِي أَوَّلِ شَوَّالٍ وَأَمَّا الثَّانِي فَهُوَ عَالِمٌ بِأَنَّ الَّذِي شَرَعَ فِيهِ الْقَعْدَةُ إلَّا أَنَّهُ غَفَلَ عَنْ كَوْنِ الْعِيدِ يَأْتِي فِي الصِّيَامِ ( قَوْلُهُ : وَأَنَّهُ قَضَاهَا مُتَّصِلَةً ) قَدْ عَلِمْت أَنَّ الرَّاجِحَ أَنَّهُ لَا يَقْضِي إلَّا الْأَوَّلَ فَقَطْ ( قَوْلُهُ : بَلْ يَبْنِي قَضَاءَهُنَّ ) أَيْ بَلْ يَبْنِي فِي حَالِ كَوْنِهِ قَدْ قَضَاهُنَّ مُتَّصِلًا ( قَوْلُهُ : لَا جَهْلُ حُكْمِهِ ) الْحُكْمُ هُوَ كَوْنُ الْعِيدِ يَقْطَعُ التَّتَابُعَ ( قَوْلُهُ : وَالْمُرَادُ بِأَيَّامِ التَّشْرِيقِ إلَخْ ) إشَارَةٌ إلَى أَنَّهُ تَفْسِيرٌ مُرَادٌ وَإِلَّا فَأَيَّامُ التَّشْرِيقِ اشْتَمَلَ الرَّابِعَ ( قَوْلُهُ : وَجَهْلُ رَمَضَانَ كَالْعِيدِ إِلَخْ ) هَلْ الْمُرَادُ أَنَّهُ جَهْلُ ذَاتِ الشَّهْرِ كَمَا لَوْ اعْتَقَدَ أَنَّ شَعْبَانَ رَجَبٌ أَوْ جَهْلُ الْحُكْمِ ( قَوْلُهُ : عَلَى الْأَرْجَحِ ) وَمُقَابِلُهُ أَنَّ جَهْلَ رَمَضَانَ لَيْسَ كَالْعِيدِ فَلَا يُجْزِئُهُ لِأَنَّهُ تَفْرِيقٌ كَثِيرٌ .( 13 / 129 )
