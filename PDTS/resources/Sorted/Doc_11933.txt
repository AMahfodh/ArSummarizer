( قَوْلُهُ : سَبَبًا فِي الْإِسْقَاطِ ) أَيْ فَلَوْ جُعِلَ حَوْلُهَا مِنْ وَقْتِ الشِّرَاءِ الَّذِي هُوَ سَبَبٌ لِلنَّمَاءِ مُسْقِطًا لِمَا مَضَى مِنْ حَوْلِ النَّقْدِ لَزِمَ مَا ذُكِرَ ( قَوْلُهُ : أَمَّا لَوْ اشْتَرَاهُ بِنَقْدٍ فِي الذِّمَّةِ ثُمَّ نَقَدَهُ ) أَيْ بَعْدَ مُفَارَقَةِ الْمَجْلِسِ ا ه سم عَلَى حَجّ نَقْلًا عَنْ شَرْحِ الْإِرْشَادِ وَإِنْ نَافَاهُ التَّعْلِيلُ بِقَوْلِهِ إذْ صَرْفُهُ إلَى هَذِهِ إلَخْ لَكِنَّهُ لَمَّا كَانَ الْمَجْلِسُ مِنْ حَرِيمِ الْعَقْدِ نَزَلَ الْوَاقِعُ فِيهِ مَنْزِلَةَ الْوَاقِعِ فِي الْعَقْدِ فَكَأَنَّهُ عَيَّنَهُ ( قَوْلُهُ : وَلَوْ بَاعَ الْعَرْضَ ) أَيْ بَعْدَ حَوَلَانِ الْحَوْلِ ( قَوْلُهُ : زَكَّى الْقِيمَةَ ) أَيْ مَا بَاعَ بِهِ فَقَطْ ؛ لِأَنَّهُ فَوَّتَ الزِّيَادَةَ بِاخْتِيَارِهِ فَضَمِنَهَا وَيُصَدَّقُ فِي قَدْرِ مَا فَوَّتَهُ ( قَوْلُهُ : وَثَمَرَهُ ) وَمِنْهُ هُنَا صُوفٌ وَغُصْنُ شَجَرٍ وَوَرَقَةٌ وَنَحْوُهَا ا ه حَجّ( 9 / 109 )
