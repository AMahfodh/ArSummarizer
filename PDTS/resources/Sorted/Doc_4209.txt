( قَوْلُهُ ذَلِكَ ) يُحْتَمَلُ أَنَّهُ فَاعِلُ قَوْلِهِ فِيهِ فَالْإِشَارَةُ إلَى لَفْظِ الْجَوَادِ وَقَوْلِهِ بِأَنِّي جَوَادٌ مَاجِدٌ بَدَلٌ مِنْهُ ، وَيُحْتَمَلُ أَنَّ الْمَجْمُوعَ هُوَ الْفَاعِلُ وَلَفْظُ ذَلِكَ مِنْ الْحَدِيثِ وَهُوَ الْأَقْرَبُ فَلْيُرَاجَعْ ( قَوْلُهُ وَلَا فَرْقَ إلَخْ ) جَوَابُ سُؤَالٍ غَنِيٌّ عَنْ الْبَيَانِ ( قَوْلُهُ وَبِالْإِجْمَاعِ ) عَطْفٌ عَلَى قَوْلِهِ بِمُسْنَدٍ ( قَوْلُهُ الْمُسْتَلْزِمِ إلَخْ ) فِيهِ نَظَرٌ سم أَيْ لِجَوَازِ أَنْ( 1 / 54 )
