( أَوْ ) عَلَّقَهُ ( بِمَا ) أَيْ بِشَيْءٍ ( لَا يُشْبِهُ الْبُلُوغَ إلَيْهِ ) عَادَةً ، بِأَنْ زَادَ أَمَدَهُ عَلَى مُدَّةِ التَّعْمِيرِ ، ( كَ : بَعْدَ ثَمَانِينَ سَنَةً ) أَنْتِ طَالِقٌ ( أَوْ ) قَالَ : ( إذَا مِتّ ) أَنَا ( أَوْ مِتّ ) أَنْتِ ( أَوْ إنْ ) مِتّ أَوْ مِتّ ( أَوْ مُتِّي ) مِتّ أَوْ مِتّ أَنْتِ فَأَنْتِ طَالِقٌ ، فَلَا شَيْءَ عَلَيْهِ إذْ لَا طَلَاقَ بَعْدَ مَوْتٍ ، بِخِلَافِ يَوْمَ مَوْتِي أَوْ قَبْلَهُ كَمَا تَقَدَّمَ ، ( أَوْ قَالَ ) لِخَلِيَّةٍ مِنْ الْحَمْلِ تَحْقِيقًا لِصِغَرٍ أَوْ إيَاسٍ أَوْ فِي طُهْرٍ لَمْ يَمَسَّ فِيهِ : ( إنْ وَلَدْت ) وَلَدًا ( أَوْ إنْ حَمَلْت ) فَأَنْتِ طَالِقٍ فَلَا شَيْءَ عَلَيْهِ لِتَحَقُّقِ عَدَمِ حَمْلِهَا ، وَقَدْ عَلَّقَ الطَّلَاقَ عَلَى وُجُودِهِ ( إلَّا أَنْ يَطَأَهَا وَلَوْ مَرَّةً ، وَهِيَ مُمْكِنَةُ الْحَمْلِ ) بَعْدَ يَمِينِهِ بَلْ ، ( وَإِنْ ) وَطِئَهَا ( قَبْلَ يَمِينِهِ ) وَلَمْ تَحِضْ بَعْدَهُ ( فَيُنَجَّزُ ) الطَّلَاقُ عَلَيْهِ لِلشَّكِّ .
