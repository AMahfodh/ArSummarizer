( وَ ) إنْ قَالَا مَحَلُّهُ ( إلَى أَوَّلِهِ ) ؛ أَيْ : شَهْرِ كَذَا ، ( أَوْ ) إلَى ( آخِرِهِ ؛ يَحِلُّ بِأَوَّلِ جُزْءٍ مِنْهَا ) ؛ أَيْ : مِنْ أَوَّلِهِ أَوْ آخِرِهِ ؛ كَتَعْلِيقِ الطَّلَاقِ ، ( وَلَا يَصِحُّ ) إنْ قَالَا : ( يُؤَدِّيه فِيهِ ) ؛ أَيْ فِي شَهْرِ كَذَا ؛ لِجَعْلِهِ كُلِّهِ ظَرْفًا ، فَيُحْتَمَلُ أَوَّلُهُ وَآخِرُهُ ؛ فَهُوَ مَجْهُولٌ .
