( قَوْلُهُ : أَوْ عَلَى مَا تَقَعُ عَلَيْهِ إلَخْ ) أَيْ بِأَنْ يَكُونَ فِي الْمَجْلِسِ سِلَعٌ كَمَقَاطِعِ قُمَاشٍ فَيَشْتَرِي مَقْطَعًا بِدِينَارٍ ، وَقَالَ الْبَائِعُ لِلْمُشْتَرِي : بِشَرْطِ أَنْ يَكُونَ الْمَقْطَعُ الَّذِي تَأْخُذُهُ هُوَ الَّذِي تَقَعُ عَلَيْهِ الْحَصَاةُ فَيَأْخُذُ حَصْوَةً وَيَرْمِيهَا فَكُلُّ مَا جَاءَتْ عَلَيْهِ كَانَ هُوَ الْمَبِيعَ وَالْفَرْضُ أَنَّهُ لَيْسَ هُنَاكَ قَصْدٌ لِمَقْطَعٍ مُعَيَّنٍ ( قَوْلُهُ : إنْ كَانَ ) أَيْ ذَلِكَ الْقَصْدُ ( قَوْلُهُ : بِأَنْ يَقُولَ ) أَيْ الْبَائِعُ لِلْمُشْتَرِي ( قَوْلُهُ : فَمَا خَرَجَ ) أَيْ مِنْ أَجْزَاءِ تِلْكَ الْحَصَاةِ الَّتِي تَكَسَّرَتْ وَقَوْلُهُ : فَمَا خَرَجَ أَيْ وُجِدَ ( قَوْلُهُ : كَانَ لَك ) أَيْ أَيُّهَا الْبَائِعُ ( قَوْلُهُ : لِلْحَدِيثِ ) أَيْ وَهُوَ مَا فِي مُسْلِمٍ مِنْ { نَهْيِ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ( 11 / 220 )
