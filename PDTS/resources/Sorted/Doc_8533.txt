( هُوَ ) لُغَةً مَا يُلْقَطُ فَعِيلٌ بِمَعْنَى مَفْعُولٍ ثُمَّ غُلِّبَ عَلَى الْوَلَدِ الْمَنْبُوذِ بِاعْتِبَارِ الْمَآلِ وَشَرْعًا ( اسْمٌ لِحَيٍّ مَوْلُودٍ طَرَحَهُ أَهْلُهُ خَوْفًا مِنْ الْعَيْلَةِ أَوْ فِرَارًا مِنْ تُهْمَةِ الرِّيبَةِ ) مُضَيِّعُهُ آثِمٌ مُحْرِزُهُ غَانِمٌ ( الْتِقَاطُهُ فَرْضُ كِفَايَةٍ إنْ غَلَبَ عَلَى ظَنِّهِ هَلَاكُهُ لَوْ لَمْ يَرْفَعْهُ ) وَلَوْ لَمْ يَعْلَمْ بِهِ غَيْرُهُ فَفَرْضُ عَيْنٍ ، وَمِثْلُهُ رُؤْيَةُ أَعْمَى يَقَعُ فِي بِئْرٍ شُمُنِّيٌّ ( وَإِلَّا فَمَنْدُوبٌ ) لِمَا فِيهِ مِنْ الشَّفَقَةِ وَالْإِحْيَاءِ و ( هُوَ حُرٌّ ) مُسْلِمٌ تَبَعًا لِلدَّارِ ( إلَّا بِحُجَّةِ رِقِّهِ ) عَلَى خَصْمٍ وَهُوَ الْمُلْتَقِطُ لِسَبْقِ يَدِهِ
