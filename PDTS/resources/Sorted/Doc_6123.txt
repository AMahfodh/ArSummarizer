كَمَالٌ ( قَوْلُهُ : لِأَنَّ فَرْضَ الْوَقْتِ فِي هَذِهِ الْحَالَةِ غَيْرُ الظُّهْرِ ) أَيْ وَهُوَ الْعَصْرُ ( قَوْلُهُ : يَجُوزُ مُطْلَقًا ) أَيْ قَبْلَ خُرُوجِ الْوَقْتِ وَبَعْدَهُ ا ه قَوْلُهُ : وَتَلْغُو نِيَّةُ التَّعْيِينِ ) فِي نُسْخَةٍ أُخْرَى التَّغْيِيرُ ( قَوْلُهُ : وَمِنْهُمْ مَنْ أَجَازَهُ ) وَفِي فَتَاوَى الْعَتَّابِيِّ الْأَصَحُّ أَنَّهُ يُجْزِيهِ ا ه فَتْحٌ ( قَوْلُهُ : وَتَعْيِينُ قَضَاءِ مَا شَرَعَ فِيهِ ) أَيْ وَشَرْطُ تَعْيِينٍ وَفِي نُسْخَةٍ وَيُعَيِّنُ .( 1 / 481 )
