الثَّانِي : وَهُوَ أَشْهَرُهُمَا : مَا لَهُ آلَةُ الرِّجَالِ وَالنِّسَاءِ ( إنْ لَمْ يَخْتَلِفْ إرْثُهُ ) بِذُكُورَتِهِ وَأُنُوثَتِهِ ( كَوَلَدِ أُمٍّ وَمُعْتِقٍ فَذَاكَ ) ظَاهِرٌ فَيُدْفَعُ إلَيْهِ نَصِيبُهُ ( وَإِلَّا ) بِأَنْ اخْتَلَفَ إرْثُهُ بِهِمَا ( فَيُعْمَلُ بِالْيَقِينِ فِي حَقِّهِ ) أَيْ : الْخُنْثَى ( وَ ) فِي ( حَقِّ غَيْرِهِ وَيُوقَفُ الْمَشْكُوكُ فِيهِ حَتَّى يَتَبَيَّنَ ) حَالُهُ وَلَوْ بِإِخْبَارِهِ ، وَلَا دَلَالَةَ عَلَى اتِّضَاحِهِ بِمَعْنَى الضَّرْبِ الْأَوَّلِ لِلْبَوْلِ فِيهِ بَلْ يُوقَفُ أَمْرُهُ حَتَّى يَصِيرَ مُكَلَّفًا فَيُخْتَبَرَ بِمَيْلِهِ .
