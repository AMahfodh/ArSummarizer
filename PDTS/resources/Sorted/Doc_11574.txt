( قَوْلُهُ : وَلَوْ نَذَرَ لِبَعْضِ وَرَثَتِهِ إلَخْ ) سَيَأْتِي مَا يَتَعَلَّقُ بِهِ قُبَيْلَ التَّنْبِيهِ ( قَوْلُهُ : مِنْ غَيْرِ مُشَارِكٍ ) أَيْ : مِنْ بِقِيَّةِ الْوَرَثَةِ ( قَوْلُهُ : أَخْذًا مِمَّا مَرَّ إلَخْ ) وَقَدْ يُقَالُ لَا حَاجَةَ لِلْأَخْذِ مِنْهُ ؛ لِأَنَّ مَا مَرَّ فِي النَّذْرِ الْغَيْرِ الْمُؤَقَّتِ أَصْلًا وَمَا هُنَا مُؤَقَّتٌ فَيَنْبَغِي أَنْ لَا يَلْزَمَ قَبْلَ مَجِيءِ الْوَقْتِ بِالِاتِّفَاقِ ( قَوْلُهُ : وَقَدْ يُنَازِعُ ) بِكَسْرِ الزَّايِ ( قَوْلُهُ : فِي ذَلِكَ كُلِّهِ ) أَيْ : مِنْ عَدَمِ لُزُومِ التَّعْجِيلِ وَعَدَمِ صِحَّةِ الدَّعْوَى وَالْبُطْلَانِ بِالْمَوْتِ قَبْلَ الْغَايَةِ ( قَوْلُهُ : فَقِيَاسُهُ هُنَا صِحَّتُهُ إلَخْ ) قَدْ يُقَالُ إنَّمَا يَكُونُ ذَلِكَ قِيَاسُهُ لَوْ كَانَ الْمَنْذُورُ ذَلِكَ الشَّيْءَ وَلَيْسَ كَذَلِكَ وَإِنَّمَا الْمَنْذُورُ التَّصَدُّقُ بِهِ فَمَا لَمْ يُوجَدْ التَّصَدُّقُ بِهِ لَا يَمْلِكُهُ الْمَنْذُورُ لَهُ فَلْيُتَأَمَّلْ .
