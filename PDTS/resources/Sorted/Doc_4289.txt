وَيَأْتِي عَنْ عَمِيرَةَ مِثْلُهُ ( قَوْلُهُ فَلَوْ أَحْدَثَ ) أَيْ بَعْدَ اللُّبْسِ وَ ( قَوْلُهُ فِيهِ ) أَيْ فِي الْخُفِّ ( قَوْلُهُ قَبْلَ الْحَدَثِ ) مُتَعَلِّقٌ بِمَا بَعْدَهُ ( قَوْلُهُ وَاغْتُفِرَ لَهُ ) أَيْ لِمُجَدِّدِ الْوُضُوءِ ( هَذَا ) أَيْ الْمَسْحُ ( قَوْلُهُ لِأَنَّ وُضُوءَهُ إلَخْ ) عِبَارَةُ الْمُغْنِي فَإِنَّهُ وَإِنْ جَازَ لَيْسَ مَحْسُوبًا مِنْ الْمُدَّةِ ؛ لِأَنَّ جَوَازَ الصَّلَاةِ وَنَحْوِهَا لَيْسَ مُسْتَنِدًا إلَيْهِ ا ه .
