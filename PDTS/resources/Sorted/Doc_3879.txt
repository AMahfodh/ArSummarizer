( فَرْعٌ : لَوْ اسْتَعَارَ الْحَلَالُ مِنْ الْمُحْرِمِ صَيْدًا ) لَمْ يُرْسِلْهُ فِي إحْرَامِهِ ( فَتَلِفَ ) فِي يَدِهِ ( لَمْ يَضْمَنْهُ ) لَهُ ؛ لِأَنَّهُ غَيْرُ مَالِكٍ ( وَعَلَى الْمُحْرِمِ الْجَزَاءُ ) لِلَّهِ تَعَالَى ؛ لِأَنَّهُ مُتَعَدٍّ بِالْإِعَارَةِ إذْ يَلْزَمُهُ إرْسَالُهُ ( فَإِنْ اسْتَعَارَهُ الْمُحْرِمُ مِنْهُ ) أَيْ مِنْ الْحَلَالِ فَتَلِفَ فِي يَدِهِ ( ضَمِنَ الْجَزَاءَ ) لِلَّهِ تَعَالَى ( وَالْقِيمَةَ ) لِلْحَلَالِ .
