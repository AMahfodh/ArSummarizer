( فَلَا نَقْضَ بِ ) تَنَاوُلِ ( بَقِيَّةِ أَجْزَائِهَا ) - أَيْ : الْإِبِلِ - ( كَسَنَامٍ وَكَبِدٍ ) وَقَلْبٍ ( وَكِرْشٍ ) وَطِحَالٍ وَمُصْرَانٍ ؛ لِأَنَّ النَّصَّ لَمْ يَتَنَاوَلْهَا ، ( وَلَا ) نَقْضَ ( بِشُرْبِ لَبَنٍ وَ ) شُرْبِ ( مَرَقِ لَحْمٍ ) لِأَنَّ الْأَخْبَارَ الصَّحِيحَةَ إنَّمَا وَرَدَتْ فِي اللَّحْمِ .( 1 / 322 )
