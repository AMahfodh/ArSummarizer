( ثُمَّ يَتَوَضَّأُ وُضُوءًا كَامِلًا وَيُرَوِّي ) بِتَشْدِيدِ الْوَاوِ ( رَأْسَهُ ) أَيْ أُصُولَ شَعْرِهِ ( ثَلَاثًا ) يَحْثِي الْمَاءَ عَلَيْهِ ثَلَاثَ حَثَيَاتٍ ( ثُمَّ ) يَغْسِلُ ( بَقِيَّةَ جَسَدِهِ ) بِإِفَاضَةِ الْمَاءِ عَلَيْهِ ( ثَلَاثًا ) لِحَدِيثِ عَائِشَةَ قَالَتْ : { كَانَ النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ إذَا اغْتَسَلَ مِنْ الْجَنَابَةِ غَسَلَ يَدَيْهِ ثَلَاثًا وَتَوَضَّأَ وُضُوءَهُ لِلصَّلَاةِ ، ثُمَّ يُخَلِّلُ شَعْرَهُ بِيَدَيْهِ ، حَتَّى إذَا ظَنَّ أَنَّهُ قَدْ رَوَى بَشَرَتَهُ أَفَاضَ الْمَاءَ عَلَيْهِ ثَلَاثَ مَرَّاتٍ ، ثُمَّ غَسَّلَ سَائِرَ جَسَدِهِ } مُتَّفَقٌ عَلَيْهِ .
