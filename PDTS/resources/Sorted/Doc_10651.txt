( وَلَا يَحْلِفُ ) مُدَّعًى عَلَيْهِ لَا حَقَّ لَهُ عَلَيْهِ ( فِي شَيْءٍ مُخْتَلَفٍ فِيهِ لَا يَعْتَقِدُهُ ) مُدَّعًى عَلَيْهِ حَقًّا ( نَصًّا وَحَمَلَهُ ) ؛ أَيْ : النَّصَّ ( الْمُوَفَّقُ عَلَى الْوَرَعِ ) دُونَ التَّحْرِيمِ ؛ لِأَنَّ الْمُدَّعِيَ لَا يَعْتَقِدُ أَنَّهُ فِي ذِمَّتِهِ ( وَيَتَّجِهُ صِحَّةُ حَمْلِهِ ) ؛ أَيْ : النَّصِّ ( عَلَى مَا إذَا حَكَمَ بِهِ ) ؛ أَيْ : الشَّيْءِ الْمُخْتَلَفِ فِيهِ ( مَنْ ) ؛ أَيْ : حَاكِمٌ ( يَرَاهُ ، أَوْ عَلَى مَا إذَا قَلَّدَهُ ) ؛ أَيْ : قَلَّدَ مُدَّعًى عَلَيْهِ ( فِيهِ ) ؛ أَيْ : الْمُخْتَلَفِ فِيهِ ( حَالَ فِعْلِهِ ) أَمَّا إذَا كَانَ لَمْ يَكُنْ كَذَلِكَ فَلَا يَصِحُّ الْحَمْلُ ، وَهُوَ مُتَّجِهٌ .
