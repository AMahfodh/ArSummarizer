( وَ ) إنْ كَانَ عَدَمُ نَبَاتِ الزَّرْعِ فِي الْعَامِ ( بِتَفْرِيطِهِ ) - أَيْ : الْمُسْتَأْجِرُ - ( كَتَأْخِيرِ زَرْعٍ لِمُدَّةٍ لَا يَكْمُلُ فِيهَا ) عَادَةً ؛ فَحُكْمُهُ حُكْمُ زَرْعِ الْغَاصِبِ ، ( لِلْمَالِكِ ) بَعْدَ انْقِضَاءِ الْمُدَّةِ ( إبْقَاؤُهُ بِأَجْرِ مِثْلِهِ ) لِمَا زَادَ عَلَى الْمُدَّةِ ؛ لِأَنَّهُ أَبْقَى زَرْعَهُ بِأَرْضِ غَيْرِهِ بِعِدْوَانِهِ ، ( وَ ) لَهُ ( تَمْلِيكُهُ بِقِيمَتِهِ ) - وَهِيَ مِثْلُ بَذْرِهِ وَعِوَضِ لَوَاحِقِهِ - وَمَحَلُّ ذَلِكَ ، ( مَا لَمْ يَخْتَرْ مُكْتَرٍ إزَالَتَهُ ) - أَيْ : الزَّرْعُ - ( حَالًا ) وَتَفْرِيغَ الْأَرْضِ ، فَإِنْ اخْتَارَهُ فَلَهُ ذَلِكَ ؛ لِأَنَّهُ يُزِيلُ الضَّرَرَ ، وَيُسَلِّمُ الْأَرْضَ عَلَى الْوَجْهِ الَّذِي اقْتَضَاهُ الْعَقْدُ ، وَلَا يَلْزَمُ الْمُسْتَأْجِرَ قَلْعُ زَرْعِهِ لَوْ طَلَبَهُ الْمَالِكُ فِي هَذِهِ الْحَالَةِ ؛ لِأَنَّ لَهُ حَدًّا يَنْتَهِي إلَيْهِ ، بِخِلَافِ الْغَرْسِ .
