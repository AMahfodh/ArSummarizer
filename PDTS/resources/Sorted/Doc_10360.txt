( قَوْلُهُ : وَسَوَاءٌ ادَّعَى غَلَطًا إلَخْ ) رَاجِعٌ لِقَوْلِ الْمَتْنِ مُضَمَّنٌ كَمَا يُعْلَمُ مِنْ عِبَارَةِ التُّحْفَةِ ( قَوْلُهُ : وَخَرَجَ بِطَلَبِ الْمَالِكِ ابْتِدَاءً أَوْ جَوَابًا إلَخْ ) عِبَارَةُ التُّحْفَةِ : وَخَرَجَ بِطَلَبِ الْمَالِكِ قَوْلُهُ ابْتِدَاءً أَوْ جَوَابًا لِسُؤَالِ غَيْرِ الْمَالِكِ وَلَوْ بِحَضْرَتِهِ أَوْ لِقَوْلِ الْمَالِكِ لِي عِنْدَك وَدِيعَةٌ لَا وَدِيعَةَ لِأَحَدٍ عِنْدِي إلَخْ ( قَوْلُهُ : وَالظَّاهِرُ كَمَا قَالَهُ الزَّرْكَشِيُّ الِاكْتِفَاءُ فِي جَوَابِهِ ) أَيْ مَنْ قَامَتْ عَلَيْهِ الْبَيِّنَةُ بِأَصْلِ الْإِيدَاعِ كَمَا هُوَ ظَاهِرُ السِّيَاقِ فَلْيُرَاجَعْ ( قَوْلُهُ : أَوْ يَدْفَعُهُ لِلْإِمَامِ إلَخْ ) مُقَابِلُ قَوْلِهِ فَيَصْرِفُهُ فِي مَصَارِفِهَا مَنْ هُوَ تَحْتَ يَدِهِ ( قَوْلُهُ : سَقَطَتْ دَعْوَى الْآخَرِ ) كَانَ الْأَوْضَحُ الْإِضْمَارَ( 19 / 495 )
