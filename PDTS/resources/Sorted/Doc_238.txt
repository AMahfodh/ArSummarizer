( فَإِنْ قَلَعَ ) الْبِنَاءَ أَوْ الْغِرَاسَ ( بَقِيَ وَقْفًا ) كَمَا كَانَ إنْ نَفَعَ ( فَلَوْ لَمْ يَنْفَعْ فَهَلْ يَصِيرُ ) مِلْكًا ( لِلْوَاقِفِ أَوْ لِلْمَوْقُوفِ عَلَيْهِ ؟ وَجْهَانِ ) قَالَ الْإِسْنَوِيُّ وَالصَّحِيحُ غَيْرُهُمَا وَهُوَ شِرَاءُ عَقَارٍ ، أَوْ جُزْءٍ مِنْ عَقَارٍ وَهُوَ قِيَاسُ النَّظَائِرِ( 12 / 427 )
