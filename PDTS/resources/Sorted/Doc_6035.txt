( وَ ) حَنِثَ الْحَالِفُ الْمُخْبَرُ بِفَتْحِ الْبَاءِ ( بِقَوْلِهِ مَا ظَنَنْته ) أَيْ ذَلِكَ الشَّخْصَ ( قَالَهُ ) أَيْ ذَلِكَ الْخَبَرَ ( لِغَيْرِي ) أَوْ لِأَحَدٍ بِدُونِ غَيْرِي ( لِمُخْبِرٍ ) بِالْكَسْرِ مُتَعَلِّقٌ بِقَوْلِهِ أَيْ بِقَوْلِهِ لِمَنْ أَخْبَرَ بِخَبَرٍ نَاقِلًا لَهُ عَنْ شَخْصٍ كَانَ قَدْ أَسَرَّ بِهِ الْحَالِفُ وَحَلَّفَهُ لَيَكْتُمَنَّهُ وَلَا يُبْدِيهِ لِأَحَدٍ كَمَا أَشَارَ لَهُ بِقَوْلِهِ ( فِي ) حَلِفِهِ ( لَيُسِرَّنَّهُ ) ، وَلَا يُخْبِرُ بِهِ أَحَدًا فَنَزَلَ قَوْلُهُ : مَا ظَنَنْته .
