ثَانِيهُمَا وَاجِبٌ مُقَيَّدٌ كَمَا قَالَ ( وَمُقَيَّدٌ بِهِ ) أَيْ بِوَقْتٍ مَحْدُودٍ ( يَفُوتُ ) الْوَاجِبُ ( بِهِ ) أَيْ بِفَوَاتِ الْوَقْتِ ( وَهُوَ ) أَيْ الْوَقْتُ الْمُقَيَّدُ بِهِ الْوَاجِبُ ( بِالِاسْتِقْرَاءِ ) أَقْسَامٌ ( أَرْبَعَةٌ ) : الْقِسْمُ ( الْأَوَّلُ أَنْ يَفْضُلَ الْوَقْتُ عَنْ الْأَدَاءِ وَيُسَمَّى عِنْدَ الْحَنَفِيَّةِ ظَرْفًا اصْطِلَاحًا ) مُوَافِقًا لِلُّغَةِ لِأَنَّهُ لُغَةُ مَا يَحِلُّ بِهِ الشَّيْءُ وَالْأَدَاءُ يَحِلُّ فِيهِ نَعَمْ تَخْصِيصُهُ بِهِ مُجَرَّدُ اصْطِلَاحٍ ( وَمُوَسَّعًا عِنْدَ الشَّافِعِيَّةِ وَبِهِ ) أَيْ بِالْمُوَسَّعِ ( سَمَّاهُ فِي( 3 / 241 )
