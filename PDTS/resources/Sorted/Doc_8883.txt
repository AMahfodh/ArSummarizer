( وَلَوْ شَكَّ ) فِي غُرُوبِ الشَّمْسِ ( بَعْدَهُ ) أَيْ : بَعْدَ الْأَكْلِ وَنَحْوِهِ ( وَدَامَ ) شَكُّهُ فَلَا قَضَاءَ عَلَيْهِ ؛ لِأَنَّهُ لَمْ يُوجَدْ يَقِينٌ أَزَالَ ذَلِكَ الظَّنَّ الَّذِي بُنِيَ عَلَيْهِ ، فَأَشْبَهَ مَا لَوْ صَلَّى بِالِاجْتِهَادِ ثُمَّ شَكَّ فِي الْإِصَابَةِ بَعْدَ صَلَاتِهِ ( أَوْ أَكَلَ يَظُنُّ بَقَاءَ النَّهَارِ قَضَى ) مَا لَمْ يَتَحَقَّقْ أَنَّهُ كَانَ بَعْدَ الْغُرُوبِ ؛ لِأَنَّ اللَّهَ تَعَالَى أَمَرَ بِإِتْمَامِ الصَّوْمِ إلَى اللَّيْلِ وَلَمْ يُتِمَّهُ ، ( وَإِنْ بَانَ ) أَنَّ أَكْلَهُ وَنَحْوَهُ كَانَ ( لَيْلًا لَمْ يَقْضِ ) ؛ لِأَنَّهُ أَتَمَّ صَوْمَهُ .( 6 / 62 )
