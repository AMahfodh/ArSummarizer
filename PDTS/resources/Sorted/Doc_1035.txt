قَالَ الْأَذْرَعِيُّ : وَكَفَانَا مُؤْنَةً ( قَوْلُهُ : فَإِنْ امْتَنَعَ ) أَيْ لَمْ يَكْتَسِبْ وَإِنْ لَمْ يَسْبِقْ أَمْرٌ لَهُ بِالِاكْتِسَابِ ( قَوْلُهُ : لِزَمَانَتِهِ ) هِيَ كُلُّ دَاءٍ يُزْمِنُ الْإِنْسَانَ فَيَمْنَعُهُ عَنْ الْكَسْبِ كَالْعَمَى وَشَلَلِ الْيَدَيْنِ انْتَهَى شَيْخُنَا زِيَادِيٌّ ( قَوْلُهُ فَإِنْ تَعَذَّرَ إلَخْ ) أَيْ بِأَنْ لَا يَتَيَسَّرَ( 14 / 219 )
