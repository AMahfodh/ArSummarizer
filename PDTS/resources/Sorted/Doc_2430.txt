( وَهُوَ ) - أَيْ : التَّيَمُّمَ - ( عَزِيمَةٌ ) كَمَسْحِ الْجَبِيرَةِ ، فَلَا يَجُوزُ تَرْكُهُ ، ( وَجَوَازُهُ ) - أَيْ : التَّيَمُّمَ - ( مَعَ ) جَوَازِ ( أَكْلِ مَيْتَةٍ لِمُضْطَرٍّ ) لَيْسَ خَاصًّا بِسَفَرٍ ، ( وَ ) جَوَازُ ( صَلَاةٍ ) نَافِلَةٍ ( عَلَى رَاحِلَةٍ لَيْسَ خَاصًّا بِسَفَرٍ ) مُبَاحٌ ؛ لِأَنَّهُ عَزِيمَةٌ .
