( قَوْلُهُ وَقَوْلُ الْأَطِبَّاءِ إلَخْ ) جَوَابٌ عَمَّا يَرُدُّ عَلَى قَوْلِهِ مَعَ إزَالَتِهِ لِضَعْفِ الْبَصَرِ ( قَوْلُهُ أَيْ عِنْدَ الْمُدَاوَمَةِ إلَخْ ) خَبَرُ وَقَوْلُ الْأَطِبَّاءِ ( قَوْلُهُ وَصَرِيحُهُمَا إلَخْ ) أَيْ الْخَبَرَيْنِ كُرْدِيٌّ ( قَوْلُهُ وَالْأَذْرَعِيِّ إلَخْ ) أَيْ : قَوْلُ الْأَذْرَعِيِّ ( قَوْلُهُ وَإِنَّمَا ذَكَرَهُ إلَخْ ) أَيْ :( 13 / 411 )
