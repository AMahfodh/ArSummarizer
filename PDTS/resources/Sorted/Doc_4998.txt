( قَوْلُهُ نَزْعُهُ ) أَيْ : مَا ذَكَرَ مِمَّا قَبَضَهُ مِنْ الدَّيْنِ وَمَا أَخَذَهُ فِي التَّعْلِيقِ ( قَوْلُهُ بَعْدَ إمْكَانِهِ ) أَيْ : النَّزْعِ ( ضَمِنَهُ ) أَيْ : الْوَلِيُّ ( قَوْلُهُ وَكَذَا لَوْ خَالَعَهَا إلَخْ ) أَيْ : فَيَلْزَمُ الْوَلِيَّ نَزْعُ الْعَيْنِ فَإِنْ تَلِفَتْ فِي يَدِهِ بَعْدَ إمْكَانِهِ ضَمِنَهَا ( قَوْلُهُ عَلَى عَيْنٍ ) وَأَمَّا الْمُخَالَعَةُ عَلَى الدَّيْنِ فَتَدْخُلُ فِي قَوْلِهِ السَّابِقِ نَعَمْ قَضِيَّتُهُ إلَخْ ا ه سم .
