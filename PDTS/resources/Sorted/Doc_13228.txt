وَاعْلَمْ أَنَّ تَفْوِيضَ الْمَرْأَةِ نَوْعَانِ : تَفْوِيضُ مَهْرٍ كَقَوْلِهَا : زَوِّجْنِي بِمَا شِئْتَ أَوْ شَاءَهُ الْخَاطِبُ أَوْ فُلَانٌ ، وَقَدْ مَرَّ فِي قَوْلِهِ : أَوْ بِمَا شِئْتَ ، وَتَفْوِيضُ بُضْعٍ وَهُوَ مَا ذَكَرَهُ بِقَوْلِهِ ( وَأَنْ يُزَوِّجْ ) سَيِّدٌ ( أَمَةً ) لَهُ ( مِنْ غَيْرِ مَا صَدَاقٍ ) بِأَنْ نَفَاهُ ( أَوْ بِالْمَهْرِ مَا تَكَلَّمَا ) بِأَنْ سَكَتَ عَنْهُ ( أَوْ قَالَتْ ) الْحُرَّةُ ( الرَّشِيدُ ) مُرَخَّمُ الرَّشِيدَةُ لِلْوَزْنِ بِكْرًا كَانَتْ أَوْ ثَيِّبًا لِوَلِيِّهَا ( زَوِّجْنِي بِلَا مَهْرٍ ) أَوْ عَلَى أَنْ لَا مَهْرَ لِي ( فَيَنْفِي ) وَلِيُّهَا فِي الْعَقْدِ ( مَهْرَهَا أَوْ أَهْمَلَا ) ذِكْرَهُ بِأَنْ سَكَتَ عَنْهُ ( أَوْ أُنْكِحَتْ ) أَيْ : أَنْكَحَهَا ( بِدُونِ مَهْرِ الْمِثْلِ أَوْ غَيْرِ نَقْدِ ذَلِكَ الْمَحَلِّ فَمَهْرُ مِثْلٍ بِدُخُولٍ وَجَبَا ) ؛ لِأَنَّ الْوَطْءَ لَا يُبَاحُ بِالْإِبَاحَةِ لِمَا فِيهِ مِنْ حَقِّ اللَّهِ تَعَالَى وَلِأَنَّ شَرْطَ الْمَالِ فِي الزِّنَا لَا يُثْبِتُهُ لِعَدَمِ تَعَلُّقِهِ بِهِ شَرْعًا فَلَا يَنْتَفِي فِي الْوَطْءِ الْمُحْتَرَمِ بِنَفْيِهِ لِتَعَلُّقِهِ بِهِ شَرْعًا .
