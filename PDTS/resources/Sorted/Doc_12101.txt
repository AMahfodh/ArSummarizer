؛ لِأَنَّ هَذِهِ لَا قِيمَةَ لَهَا ، فَلَا يَنْقَسِمُ عَلَيْهَا الْبَدَلُ ، أَشْبَهَ مَا لَوْ كَانَ الثَّمَنُ كُلُّهُ كَذَلِكَ ( وَلَا ) الْبَيْعُ ( بِمَا يَنْقَطِعُ بِهِ السِّعْرُ ) أَيْ يَقِفُ عَلَيْهِ لِلْجَهَالَةِ ( وَلَا كَمَا يَبِيعُ النَّاسُ ) لِمَا تَقَدَّمَ ( وَلَا بِدِينَارٍ ) مُطْلَقٍ ( أَوْ دِرْهَمٍ مُطْلَقٍ ) أَوْ قِرْشٍ مُطْلَقٍ ( وَثُمَّ ) بِالْبَلَدِ ( نُقُودٌ ) مِنْ الْمُسَمَّى الْمُطْلَقِ ( مُتَسَاوِيَةٌ رَوَاجًا ) لِتَرَدُّدِ الْمُطْلَقِ بَيْنَهَا وَرَدُّهُ إلَى أَحَدِهِمَا مَعَ التَّسَاوِي تَرْجِيحٌ بِلَا مُرَجِّحٍ فَهُوَ مَجْهُولٌ وَإِذَا لَمْ يَكُنْ ) بِالْبَلَدِ ( إلَّا ) دِينَارٌ أَوْ دِرْهَمٌ أَوْ قِرْشٌ ( وَاحِدٌ ) صَحَّ وَصُرِفَ إلَيْهِ لِتَعَيُّنِهِ ( أَوْ غَلَبَ أَحَدُهُمَا ) أَيْ النُّقُودِ رَوَاجًا ( صَحَّ ) الْعَقْدُ ( وَصُرِفَ ) الْمُطْلَقُ مِنْ دِينَارٍ أَوْ دِرْهَمٍ أَوْ قِرْشٍ ( إلَيْهِ ) عَمَلًا بِالظَّاهِرِ( 4 / 385 )
