( وَ ) لَا يُجْزِئُ ( مَعِيبٌ ) مِمَّا تَقَدَّمَ لِقَوْلِهِ تَعَالَى : { وَلَا تَيَمَّمُوا الْخَبِيثَ مِنْهُ تُنْفِقُونَ } ( كَمُسَوِّسٍ ) لِأَنَّ السُّوسَ أَكَلَ جَوْفَهُ ( وَمَبْلُولٍ ) لِأَنَّ الْبَلَلَ يَنْفُخُهُ ( وَقَدِيمٍ تَغَيَّرَ طَعْمُهُ ) لِعَيْبِهِ بِتَغَيُّرِ طَعْمِهِ ، فَإِنْ لَمْ يَتَغَيَّرْ طَعْمُهُ وَلَا رِيحُهُ أَجْزَأَ ؛ لِعَدَمِ عَيْبِهِ ، وَالْجَدِيدُ أَفْضَلُ ( وَنَحْوُهُ ) أَيْ : مَا تَقَدَّمَ مِنْ أَمْثِلَةِ الْمَعِيبِ .
