( وَمَنْ مَلَك قَرِيبَهُ ) بِسَبَبٍ مَا ( مَعَ ) رَجُلٍ ( آخَرَ عَتَقَ حَظُّهُ بِلَا ضَمَانٍ عَلِمَ ) الشَّرِيكُ ( بِقَرَابَتِهِ أَوْ لَا ) - عَلَى الظَّاهِرِ ؛ لِأَنَّ الْحُكْمَ يُدَارُ عَلَى السَّبَبِ ( وَلِشَرِيكِهِ أَنْ يُعْتِقَ أَوْ يَسْتَسْعَى ) أَمَّا لَوْ مَلَك مُسْتَوْلَدَتَهُ بِالنِّكَاحِ مَعَ آخَرَ فَيَضْمَنُ حَظَّ شَرِيكِهِ لِكَوْنِهِ ضَمَانُ تَمَلُّكٍ ( وَإِنْ اشْتَرَى نِصْفَهُ أَجْنَبِيٌّ ثُمَّ الْقَرِيبُ بَاقِيَهُ فَلَهُ أَنْ يُضَمِّنَ الْمُشْتَرِي ) مُوسِرًا ( أَوْ يَسْتَسْعَى ) الْعَبْدَ ، هَذِهِ سَاقِطَةٌ مِنْ نُسَخِ الشَّارِحِ .
