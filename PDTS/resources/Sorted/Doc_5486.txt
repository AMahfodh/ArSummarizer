ا ه شَرْحُ رَوْضٍ ( قَوْلُهُ : وَبَنَاتِهِ ) عَطْفٌ عَلَى أَزْوَاجِهِ ( قَوْلُهُ : لَا تُغَالُوا بِصُدُقِ النِّسَاءِ ) أَيْ بِأَنْ تُشَدِّدُوا عَلَى الْأَزْوَاجِ بِطَلَبِ الزِّيَادَةِ عَلَى مُهُورِ أَمْثَالِهِنَّ ( قَوْلُهُ : فَإِنَّهَا لَوْ كَانَتْ ) أَيْ هَذِهِ الْخَصْلَةُ ( قَوْلُهُ وَجَبَ تَسْمِيَتُهُ ) أَيْ فَلَوْ خَالَفَ أَثِمَ وَصَحَّ الْعَقْدُ بِمَهْرِ الْمِثْلِ ( قَوْلُهُ : وَجَبَتْ تَسْمِيَتُهُ ) أَيْ فَلَوْ لَمْ يُسَمِّ أَثِمَ وَصَحَّ كَالَّتِي قَبْلَهَا .( 21 / 212 )
