ثُمَّ شَرَعَ يُبَيِّنُ أَنَّ الْيَمِينَ تُغَلَّظُ بِالْهَيْئَةِ وَالْمَكَانِ ، أَمَّا الْهَيْئَةُ فَأَشَارَ إلَيْهَا بِقَوْلِهِ : ( وَيَحْلِفُ قَائِمًا ) ظَاهِرُهُ أَنَّ الْقِيَامَ شَرْطٌ وَهُوَ ظَاهِرُ رِوَايَةِ ابْنِ الْقَاسِمِ ، فَلَوْ حَلَفَ جَالِسًا لَمْ يُجْزِهِ عَلَى الْمَشْهُورِ ( وَ ) أَمَّا الْمَكَانُ فَإِنْ كَانَ بِالْمَدِينَةِ الشَّرِيفَةِ يَحْلِفُ ( عِنْدَ مِنْبَرِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فِي رُبْعِ دِينَارٍ فَأَكْثَرَ ) ؛ لِأَنَّ ذَلِكَ أَرْدَعُ لِلْحَالِفِ وَأَرْجَى أَنْ يَرْجِعَ لِلْحَقِّ ( وَ ) إنْ كَانَ ( فِي غَيْرِ الْمَدِينَةِ ) الْمُشَرَّفَةِ ( يَحْلِفُ فِي ذَلِكَ ) أَيْ فِي رُبْعِ دِينَارٍ فَأَكْثَرَ ( فِي الْجَامِعِ ) الَّذِي تُصَلَّى فِيهِ الْجُمُعَةُ ( وَ ) يَكُونُ ذَلِكَ ( بِمَوْضِعٍ يُعَظِّمُ مِنْهُ ) بِكَسْرِ الظَّاءِ الْمُعْجَمَةِ الْمُشَالَةِ وَهُوَ الْمِحْرَابُ .
