( وَ ) إنْ بَاعَ الرَّاهِنُ الرَّهْنَ ( بَعْدَهُ ) أَيْ قَبَضَهُ الْمُرْتَهِنُ بِلَا إذْنِهِ ( فَلَهُ ) أَيْ الْمُرْتَهِنِ ( رَدُّهُ ) أَيْ بَيْعِ الرَّهْنِ ( إنْ بِيعَ ) الرَّهْنُ ( ب ) ثَمَنٍ ( أَقَلَّ ) مِنْ الدَّيْنِ الْمَرْهُونِ فِيهِ عَيْنًا كَانَ أَوْ عَرْضًا مِنْ بَيْعٍ كَانَ أَوْ قَرْضٍ لِضَرَرِهِ بِهِ ( أَوْ ) بَيْعٍ بِقَدْرِهِ أَوْ أَكْثَرَ وَكَانَ ( دَيْنُهُ ) أَيْ الْمُرْتَهِنِ ( عَرْضًا ) مِنْ بَيْعٍ إذْ لَا يَلْزَمُ الْمُرْتَهِنَ قَبُولُهُ قَبْلَ أَجَلِهِ لِأَنَّهُ حَقٌّ لَهُ أَيْضًا ، فَإِنْ بَاعَهُ بِقَدْرِ الدَّيْنِ الْعَيْنِ مُطْلَقًا أَوْ الْعَرْضِ مِنْ قَرْضٍ فَلَيْسَ لِلْمُرْتَهِنِ رَدُّهُ وَيَتَعَجَّلُ دَيْنُهُ إنْ شَاءَ( 12 / 14 )
