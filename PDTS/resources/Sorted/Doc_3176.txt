( وَ ) يَصِحُّ بَيْعُ ( نَحْلٍ مُنْفَرِدًا عَنْ كُوَّارَتِهِ ) ؛ لِأَنَّهُ حَيَوَانٌ طَاهِرٌ يَخْرُجُ مِنْ بُطُونِهِ شَرَابٌ فِيهِ مَنَافِعُ لِلنَّاسِ فَهُوَ كَبَهِيمَةِ الْأَنْعَامِ ، وَكَذَا يَصِحُّ بَيْعُهُ خَارِجًا عَنْ كُوَّارَتِهِ مَعَهَا ( بِشَرْطِ كَوْنِهِ مَقْدُورًا عَلَيْهِ ) وَإِلَّا لَمْ يَصِحَّ بَيْعُهُ لِلْغَرَرِ ( وَفِيهَا ) أَيْ : وَيَصِحُّ بَيْعُ نَحْلٍ فِي كُوَّارَتِهِ ( مَعَهَا ) إذَا شُوهِدَ دَاخِلًا إلَيْهَا .
