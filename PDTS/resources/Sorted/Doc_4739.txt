فَفِي الْعِتْقِ أَوْلَى ( وَلَوْ لَمْ يَمْلِكْهُ ) أَيْ : الْحَمْلَ رَبُّ الْأَمَةِ كَمَا لَوْ اشْتَرَى أَمَةً مِنْ وَرَثَةِ مَيِّتٍ مُوصًى بِحَمْلِهَا لِغَيْرِهِ فَأَعْتَقَهَا فَيَسْرِي الْعِتْقُ إلَى الْحَمْلِ ( إنْ كَانَ ) مُعْتِقُهَا ( مُوسِرًا ) بِقِيمَةِ الْحَمْلِ يَوْمَ عِتْقِهِ كَفِطْرَةٍ ( وَيَضْمَنُ ) مُعْتِقُهَا ( قِيمَتَهُ ) أَيْ : الْحَمْلِ ( لِمَالِكِهِ ) الْمُوصَى لَهُ بِهِ يَوْمَ وِلَادَتِهِ حَيًّا ، فَإِنْ اسْتَثْنَى الْحَمْلَ مُعْتِقُ أُمِّهِ لَمْ يُعْتَقْ .
