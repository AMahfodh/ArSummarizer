( وَلَوْ ) عُقِدَتْ الْجِزْيَةُ لِمَنْ زَعَمَ التَّمَسُّكَ بِكِتَابٍ ، ثُمَّ ( أَسْلَمَ اثْنَانِ ) مِنْ أَهْلِ ذَلِكَ الدِّينِ ( وَجَادَ الْحَالُ ) مِنْ زِيَادَتِهِ أَيْ : وَحَسُنَ حَالُهُمَا بِحَيْثُ تُقْبَلُ شَهَادَتُهُمَا ( وَشَهِدَا بِكُفْرِهِ ) أَيْ : بِكَذِبِهِ فِيمَا زَعَمَهُ ( يُغْتَالُ ) وَلَا يَبْلُغُ الْمَأْمَنَ لِتَدْلِيسِهِ وَالْأَمَانُ الْفَاسِدُ إنَّمَا يَمْنَعُ الِاغْتِيَالَ عِنْدَ ظَنِّ الْكَافِرِ صِحَّتَهُ وَهُوَ مُنْتَفٍ هُنَا ( لَا إنْ تَوَثَّنَ ) كِتَابِيٌّ عُقِدَ لَهُ الْجِزْيَةُ فَلَا يُغْتَالُ بَلْ يُلْحَقُ بِمَأْمَنِهِ وَإِنْ لَمْ يُقَرَّ عَلَى ذَلِكَ ؛ لِأَنَّهُ قَدْ ثَبَتَ لَهُ عَلَقَةُ كَوْنِهِ كِتَابِيًّا ، فَلَا تَزُولُ بِمَا حَدَثَ بَعْدَهُ وَمَنْ يُخَافُ شَرُّهُ كَجَاسُوسٍ لَا يُقَرُّ بِالْجِزْيَةِ كَمَا يُؤْخَذُ مِمَّا مَرَّ فِي الْأَمَانِ( 18 / 474 )
