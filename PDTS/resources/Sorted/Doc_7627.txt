( وَالنَّفَقَةُ ) عَلَى الْحَامِلِ ( لِلْحَمْلِ ) نَفْسِهِ ، لَا لَهَا مِنْ أَجْلِهِ ؛ لِأَنَّهَا لَا تَجِبُ بِوُجُودِهِ وَتَسْقُطُ عِنْدَ انْقِضَائِهِ ؛ قَالَ الْبُهُوتِيُّ قُلْت : فَلَوْ مَاتَ بِبَطْنِهَا انْقَطَعَتْ ؛ لِأَنَّهَا لَا تَجِبُ لِمَيِّتٍ ( فَتَجِبُ ) النَّفَقَةُ ( لِنَاشِزٍ حَامِلٍ ) ؛ لِأَنَّ النَّفَقَةَ لِلْحَمْلِ فَلَا تَسْقُطُ بِنُشُوزِ أُمِّهِ ( وَ ) تَجِبُ ( لِحَامِلٍ مِنْ وَطْءِ شُبْهَةٍ أَوْ نِكَاحٍ فَاسِدٍ ) لِلُحُوقِ نَسَبِهِ فِيهِمَا ( وَ ) لِحَامِلٍ فِي ( مِلْكِ يَمِينٍ وَلَوْ أَعْتَقَهَا ) ؛ لِأَنَّ النَّفَقَةَ لِلْحَمْلِ وَهُوَ وَلَدُهُ .
