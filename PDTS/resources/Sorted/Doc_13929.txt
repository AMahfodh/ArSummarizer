قَالَ الشَّافِعِيُّ : وَاسْتِكْثَارُ الْقِيمَةِ فِي الْأُضْحِيَّةِ أَوْلَى مِنْ الْعَدَدِ بِخِلَافِ الْعِتْقِ فَلَوْ تَعَارَضَ أَسْوَدُ سَمِينٌ وَأَبْيَضُ هَزِيلٌ فَالظَّاهِرُ تَقْدِيمُ - الْأَسْوَدِ وَالْأَكْمَلِ ( الذَّكَرْ ) ؛ لِأَنَّ لَحْمَهُ أَطْيَبُ مِنْ لَحْمِ الْأُنْثَى إلَّا أَنْ تَكُونَ الْأُنْثَى لَمْ تَلِدْ فَهِيَ أَفْضَلُ وَعَلَيْهَا حَمَلُوا قَوْلَ الشَّافِعِيِّ الْأُنْثَى أَحَبُّ إلَيَّ فَإِنْ كَانَتْ الْأُنْثَى حَامِلًا لَمْ تَجُزْ ؛ لِأَنَّ الْحَمْلَ يُهْزِلُهَا كَذَا نَقَلَهُ النَّوَوِيُّ فِي مَجْمُوعِهِ عَنْ الْأَصْحَابِ ، وَقَالَ ابْنُ الرِّفْعَةِ الْمَشْهُورُ أَنَّهَا تُجْزِئُ ( وَ ) الْأَكْمَلُ ( تَرْكُ ذِي تَضْحِيَةٍ ) يَعْنِي مُرِيدَهَا ( تَقْلِيمَهُ ) ظُفْرَهُ ( وَحَلْقَهُ ) يَعْنِي إزَالَتَهُ شَعْرَهُ ( فِي الْعَشَرَةِ الْمَعْلُومَةِ ) وَهِيَ عَشَرُ ذِي الْحِجَّةِ لِخَبَرِ مُسْلِمٍ { إذَا رَ أَيْتُمْ هِلَالَ ذِي الْحِجَّةِ وَأَرَادَ أَحَدُكُمْ أَنْ يُضَحِّيَ فَلْيُمْسِكْ عَنْ شَعْرِهِ وَأَظْفَارِهِ } وَفِي رِوَايَةٍ { فَلَا يَأْخُذَنَّ مِنْ شَعْرِهِ وَأَظْفَارِهِ شَيْئًا حَتَّى يُضَحِّيَ } وَتُكْرَهُ مُخَالَفَةُ ذَلِكَ وَمَنَعَ مِنْ تَحْرِيمِهِ قَوْلُ عَائِشَةَ فِي خَبَرِ الصَّحِيحَيْنِ { كُنْت أَفْتِلُ قَلَائِدَ هَدْيِ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ثُمَّ يُقَلِّدُهَا هُوَ بِيَدِهِ ، ثُمَّ يَبْعَثُ بِهَا فَلَا يَحْرُمُ عَلَيْهِ شَيْءٌ أَحَلَّهُ اللَّهُ تَعَالَى حَتَّى يُنْحَرَ الْهَدْيُ }( 19 / 148 )
