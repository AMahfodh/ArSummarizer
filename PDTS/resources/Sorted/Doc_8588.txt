( وَرَدَّ ) مُشْتَرٍ ( بَدَلَهُ ) أَيْ الْمَعِيبِ التَّالِفِ عِنْدَهُ ( وَاسْتَرْجَعَ الثَّمَنَ ) إنْ كَانَ أَقْبَضَهُ لِبَائِعٍ لِتَعَذُّرِ أَخْذِ الْأَرْشِ ، لِإِفْضَائِهِ لِلرِّبَا ، ( وَكَسْبُ مَبِيعٍ مَعِيبٍ ) مِنْ عَقْدٍ إلَى رَدٍّ ( لِمُشْتَرٍ ) لِحَدِيثِ { الْخَرَاجُ بِالضَّمَانِ } وَلَوْ هَلَكَ الْمَبِيعُ لَكَانَ مِنْ ضَمَانِهِ ( وَلَا يَرُدُّ ) مُشْتَرٍ رَدَّ مَبِيعًا ( لِعَيْبِهِ نَمَاءً مُنْفَصِلًا ) مِنْهُ كَثَمَرَةٍ وَوَلَدِ بَهِيمَةٍ ( إلَّا لِعُذْرٍ كَوَلَدِ أَمَةٍ ) فَيَرُدُّ مَعَهَا لِتَحْرِيمِ التَّفْرِيقِ ( وَلَهُ ) أَيْ الْمُشْتَرِي ( قِيمَتُهُ ) أَيْ الْوَلَدِ عَلَى بَائِعٍ ؛ لِأَنَّهُ نَمَاءُ مِلْكِهِ .
