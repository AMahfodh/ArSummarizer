( كِتَابُ الْحَجِّ وَالْعُمْرَةِ ) ( قَوْلُهُ اتَّفَقَ الْحُفَّاظُ عَلَى ضَعْفِهِ ) ؛ لِأَنَّ فِي رِجَالِهِ ابْنَ أَرْطَاةَ وَابْنَ لَهِيعَةَ وَهُمَا ضَعِيفَانِ ( قَوْلُهُ لِعَدَمِ اسْتِطَاعَتِهِ ) أَوْ سَأَلَ عَنْ عُمْرَةٍ ثَانِيَةٍ ( قَوْلُهُ بِأَنَّ الْغُسْلَ أَصْلٌ فَأَغْنَى عَنْ بَدَلِهِ إلَخْ ) وَجْهُهُ أَنَّ الْغُسْلَ فِي حَقِّ الْمُحْدِثِ هُوَ الْأَصْلُ وَإِنَّمَا حُطَّ عَنْهُ إلَى الْأَعْضَاءِ الْأَرْبَعَةِ تَخْفِيفًا .( 5 / 464 )
