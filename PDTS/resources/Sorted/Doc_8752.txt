( قَوْلُهُ : فَكَمَا مَرَّ فِي الْوُضُوءِ ) أَيْ مِنْ أَنَّهُ إذَا قَصَدَ التَّبَرُّكَ صَحَّ وَإِلَّا فَلَا ( قَوْلُهُ : فَلَا يَجْزِيهِ ) كَانَ الْأَوْلَى فِي التَّفْرِيعِ أَنْ يَقُولَ وَلَا يَصِحُّ لِأَنَّ رَمَضَانَ لَا يَقْبَلُ غَيْرَهُ ( قَوْلُهُ : الْمَبْنِيَّةُ عَلَيْهِ ) أَيْ عَلَى غَلَبَةِ الظَّنِّ ( قَوْلُهُ : وَهُوَ مِمَّنْ يَحِلُّ لَهُ صَوْمُهُ ) أَيْ بِأَنْ وَافَقَ عَادَةً لَهُ ( قَوْلُهُ وَلَا أَثَرَ لِتَرَدُّدِ ) هَذَا تَقَدَّمَ فِي قَوْلِهِ كَالتَّرَدُّدِ فِي الْقَلْبِ لَكِنَّهُ سَبَقَ هُنَاكَ لِعَدَمِ ضَرَرِ التَّرَدُّدِ مَعَ الظَّنِّ الْمُسْتَنِدِ لِخَبَرِ ثِقَةٍ وَهُنَا لِبَيَانِ الْحُكْمِ قَصْدًا .( 9 / 337 )
