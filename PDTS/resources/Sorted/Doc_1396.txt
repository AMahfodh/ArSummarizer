( وَإِذَا أَعْتَقْنَا بَعْضَهُمْ ) أَيْ : الْأَرِقَّاءِ ( بِقُرْعَةٍ فَظَهَرَ مَالٌ ) آخَرُ لِلْمَيِّتِ لَمْ يُعْلَمْ وَقْتَ الْقُرْعَةِ ( وَخَرَجَ كُلُّهُمْ مِنْ الثُّلُثِ عَتَقُوا ) أَيْ : بَانَ عِتْقُهُمْ وَأَنَّهُمْ أَحْرَارٌ تَجْرِي عَلَيْهِمْ أَحْكَامُ الْأَحْرَارِ مِنْ حِينِ إعْتَاقِهِ ( وَ ) مِنْ ثَمَّ كَانَ ( لَهُمْ كَسْبُهُمْ ) وَنَحْوُهُ كَأَرْشِ جِنَايَةٍ وَمَهْرِ أَمَةٍ .
