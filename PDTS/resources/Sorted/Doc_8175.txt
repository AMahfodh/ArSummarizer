( وَلَا يَدْعُو ) بَعْدَ الرَّابِعَةِ لِظَاهِرِ الْخَبَرِ ( وَيُسَلِّمُ ) تَسْلِيمَةً ( وَاحِدَةً عَنْ يَمِينِهِ ) نَصًّا لِأَنَّهُ أَشْبَهُ بِالْحَالِ وَأَكْثَرُ مَا رُوِيَ فِي التَّسْلِيمِ ( وَيَجُوزُ ) أَنْ يُسَلِّمَهَا ( تِلْقَاءَ وَجْهِهِ ) نَصًّا ( وَ ) يَجُوزُ أَنْ يُسَلِّمَ ( ثَانِيَةً ) وَيُجْزِئُ وَإِنْ لَمْ يَقُلْ : وَرَحْمَةُ اللَّهِ ، لِمَا رَوَى الْخَلَّالُ وَحَرْبٌ عَنْ عَلِيٍّ رَضِيَ اللَّهُ عَنْهُ أَنَّهُ صَلَّى عَلَى زَيْدِ بْنِ الْمُلَقِّفِ فَسَلَّمَ وَاحِدَةً عَنْ يَمِينِهِ : السَّلَامُ عَلَيْكُمْ لَكِنَّ ذِكْرَ الرَّحْمَةِ أَلْيَقُ بِالْحَالِ ، فَكَانَ أَوْلَى .( 2 / 437 )
