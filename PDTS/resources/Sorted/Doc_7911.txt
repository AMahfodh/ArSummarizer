( قَوْلُهُ : وَذَلِكَ ظَاهِرٌ لَا شَكَّ فِيهِ ) أَيْ لَكِنْ لَوْ سَمِعُوا النِّدَاءَ مِنْ قَرْيَةٍ أُخْرَى وَجَبَ عَلَيْهِمْ السَّعْيُ إلَيْهَا ( قَوْلُهُ يُقِيمُ عِنْدَ كُلٍّ يَوْمًا مَثَلًا ) وَكَذَا مَنْ لَهُ مَسْكَنَانِ وَكَثُرَتْ إقَامَتُهُ فِي أَحَدِهِمَا وَزَوْجَتُهُ فِي الْآخَرِ أَوْ لَا زَوْجَةَ لَهُ فِي وَاحِدٍ مِنْهُمَا فَتَعْبِيرُهُ بِالزَّوْجَتَيْنِ مُجَرَّدُ تَصْوِيرٍ ( قَوْلُهُ : فَإِنْ اسْتَوَيَا فِيهَا ) أَيْ الْإِقَامَةِ ( قَوْلُهُ : فِيهِ ) أَيْ الْمَالِ ( قَوْلُهُ : بِأَنَّهُ يَصْدُقُ عَلَيْهِ أَنَّهُ مُتَوَطِّنٌ ) أَيْ فَتَنْعَقِدُ بِهِ الْجُمُعَةُ فِيهِمَا .( 7 / 46 )
