( وَلَا تَوَاطُؤَ ) أَيْ تَوَافُقَ عَلَى قَتْلِهِ بِأَنْ ضَرَبَهُ كُلٌّ مِنْهُمْ بِحَجَرٍ صَغِيرٍ حَتَّى مَاتَ وَلَمْ يَكُونُوا اتَّفَقُوا عَلَى ذَلِكَ ( فَلَا ) قِصَاصَ ؛ لِأَنَّهُ لَمْ يَحْصُلْ مَا يُوجِبُهُ مِنْ وَاحِدٍ مِنْهُمْ فَإِنْ تَوَاطَئُوا عَلَيْهِ قُتِلُوا بِهِ لِئَلَّا يُؤَدِّيَ إلَى التَّسَارُعِ إلَى الْقَتْلِ بِهِ وَتَفُوتَ حِكْمَةُ الرَّدْعِ وَالزَّجْرِ عَنْ الْقَتْلِ ( وَلَا يَجِبُ ) عَلَيْهِمْ ( مَعَ عَفْوٍ ) عَنْ قَوَدٍ ( أَكْثَرُ مِنْ دِيَةٍ ) ؛ لِأَنَّ الْقَتِيلَ وَاحِدٌ فَلَا يَلْزَمُهُمْ أَكْثَرُ مِنْ دِيَتِهِ كَمَا لَوْ قَتَلُوا خَطَأً( 10 / 302 )
