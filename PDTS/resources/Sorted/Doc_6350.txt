الْبَائِنِ ( وَلَا يَتَوَقَّفُ ذَلِكَ عَلَى قَضَاءِ الْقَاضِي وَلَا عَلَى لِعَانِهَا ) بَلْ يَحْصُلُ بِمُجَرَّدِ لِعَانِ الزَّوْجِ ( وَإِنَّمَا هُوَ ) أَيْ لِعَانُهَا ( لِدَرْءِ الْحَدِّ عَنْهَا ) لِلْآيَةِ ( فَإِنْ أَثْبَتَ ) أَيْ أَقَامَ بَيِّنَةً ( بِزِنَاهَا ) أَوْ بِإِقْرَارِهَا بِهِ ( لَمْ تُلَاعِنْ ) لِدَرْءِ الْحَدِّ ؛ لِأَنَّ اللِّعَانَ حُجَّةٌ ضَعِيفَةٌ لَا تُقَاوِمُ الْبَيِّنَةَ ( وَسَقَطَ ) بِذَلِكَ ( الْحَدُّ ) لِلْقَذْفِ ( عَنْهُ وَوَجَبَ ) بِهِ حَدُّ الزِّنَا ( عَلَيْهَا ) عَمَلًا بِالْبَيِّنَةِ .
