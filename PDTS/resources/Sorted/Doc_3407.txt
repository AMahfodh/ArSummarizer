( وَ ) ، إنْ خَلَطَ مَغْصُوبًا ( بِدُونِهِ أَوْ ) خَلَطَهُ ( بِخَيْرٍ مِنْهُ ) مِنْ جِنْسِهِ ( أَوْ ) خَلَطَهُ ( بِغَيْرِ جِنْسِهِ عَلَى وَجْهٍ لَا يَتَمَيَّزُ ) كَزَيْتٍ بِشَيْرَجٍ وَدَقِيقِ حِنْطَةٍ بِدَقِيقِ شَعِيرٍ وَنَحْوِهِ ( فَ ) الْمَالِكَانِ ( شَرِيكَانِ ) فِي الْمُخْتَلَطِ ( بِقَدْرِ قِيمَتَيْهِمَا كَاخْتِلَاطِهِمَا مِنْ غَيْرِ غَصْبٍ ) نَصًّا لِيَصِلَ كُلٌّ مِنْهُمَا إلَى بَدَلِ عَيْنِ مَالِهِ .
