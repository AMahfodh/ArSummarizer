( وَ ) يُبَاحُ أَنْ ( يَجُزَّ صُوفَهَا ) أَيْ : الْمُعَيَّنَةَ هَدْيًا أَوْ أُضْحِيَّةً ( وَنَحْوَهُ ) كَوَبَرِهَا ( لِمَصْلَحَةٍ ) لِانْتِفَاعِهَا بِهِ ( وَيَتَصَدَّقُ ) بِهِ نَدْبًا ( أَوْ يَنْتَفِعُ بِهِ كَجِلْدٍ ) لِلِانْتِفَاعِ بِهِ دَوَامًا ، فَإِنْ كَانَ بَقَاؤُهُ أَنْفَعَ لَهَا لِيَقِيَهَا حَرًّا أَوْ بَرْدًا ، حَرُمَ جَزُّهُ كَأَخْذِ بَعْضِ أَعْضَائِهَا .
