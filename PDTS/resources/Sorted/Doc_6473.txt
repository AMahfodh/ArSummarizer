( وَلَا ) يَصِحُّ أَيْضًا وَقْفُ أُمِّ وَلَدٍ ؛ لِأَنَّهُ لَا يَصِحُّ بَيْعُهَا وَلَا يَصِحُّ أَيْضًا الْوَقْفُ عَلَيْهَا ، وَيَأْتِي ( فَإِنْ وَقَفَ عَلَى غَيْرِهَا ) كَعَلَى زَيْدٍ ( عَلَى أَنْ يُنْفِقَ عَلَيْهَا ) أَيْ : عَلَى أُمِّ وَلَدِهِ مِنْهُ مُدَّةَ حَيَاتِهِ أَوْ وَقَفَ عَلَى زَيْدٍ مَثَلًا عَلَى أَنْ يَكُونَ ( الرِّيعُ لَهَا ) أَيْ : لِأُمِّ وَلَدِهِ ( مُدَّةَ حَيَاتِهِ صَحَّ ) الْوَقْفُ ؛ لِأَنَّ اسْتِثْنَاءَ الْمَنْفَعَةِ لِأُمِّ وَلَدِهِ كَاسْتِثْنَائِهَا لِنَفْسِهِ .( 14 / 346 )
