( فَإِنْ نَوَى ) بِقَوْلِهِ لَا أَفْعَلُ كَذَا أَوْ عَلَى زَوْجَتِهِ وَنَحْوِهَا : لَا تَفْعَلُ كَذَا فَعَلَ ( الْجَمِيعَ أَوْ ) فَعَلَ ( الْبَعْضَ فَيَمِينُهُ عَلَى مَا نَوَى ) لِأَنَّ النِّيَّةَ مُخَصِّصَةٌ ، وَكَذَا لَوْ اقْتَضَى سَبَبُ الْيَمِينِ أَحَدَ الْأَمْرَيْنِ ( وَإِنْ دَلَّتْ قَرِينَةٌ تَقْتَضِي أَحَدَ الْأَمْرَيْنِ ) الْجَمِيعَ أَوْ الْبَعْضَ ( تَعَلَّقَ ) الْحِنْثُ ( بِهِ ) أَيْ بِمَا دَلَّتْ الْقَرِينَةُ عَلَيْهِ ( كَمَنْ حَلَفَ لَا شَرِبْتُ هَذَا النَّهْرَ أَوْ لَا أَكَلْتُ الْخُبْزَ ) أَوْ اللَّحْمَ ( أَوْ لَا شَرِبْتُ الْمَاءَ وَمَا أَشْبَهَهُ ) كَلَا لَبِسْتُ الْغَزْلَ وَنَحْوَهُ ( مِمَّا عَلَّقَ عَلَى اسْمٍ أَوْ جِنْسٍ أَوْ عَلَى اسْمِ جَمْعٍ كَالْمُسْلِمِينَ وَالْمُشْرِكِينَ وَالْفُقَرَاءِ وَالْمَسَاكِينِ حَنِثَ بِالْبَعْضِ ) لِأَنَّ فِعْلَ الْجَمِيعِ مُمْتَنِعٌ( 18 / 320 )
