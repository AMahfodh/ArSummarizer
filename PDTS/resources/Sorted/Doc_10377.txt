( لَا ) نَاذِرَ ( الْحَجِّ ) الْمُطْلَقِ أَوْ الْحَالِفِ بِهِ فَحَنِثَ فِي غَيْرِ أَشْهُرِهِ فَلَا يُعَجِّلُهُ قَبْلَهَا ( وَ ) لَا نَاذِرَ ( الْمَشْيِ ) الْمُطْلَقِ أَيْ مِنْ غَيْرِ تَقْيِيدٍ بِعَامٍ وَلَا حَجٍّ وَلَا عُمْرَةٍ فَلَا يُؤْمَرُ بِالتَّعْجِيلِ ، وَإِذَا لَمْ يُؤْمَرْ بِهِ فِي الصُّورَتَيْنِ ( فَلِأَشْهُرِهِ ) أَيْ فَيَلْزَمُهُ التَّعْجِيلُ فِيهِمَا عِنْدَ أَشْهُرِ الْحَجِّ ( إنْ ) كَانَ إذَا خَرَجَ فِي أَشْهُرِهِ ( وَصَلَ ) لِمَكَّةَ وَأَدْرَكَ الْحَجَّ لَكِنْ فِي الْحَجِّ يُحْرِمُ مِنْ مَكَانِهِ ، وَيَخْرُجُ وَفِي الْمَشْيِ الْمُطْلَقِ مِنْ الْمِيقَاتِ ( وَإِلَّا ) يَصِلُ فِيهَا كَإِفْرِيقِيٍّ ( فَمِنْ حَيْثُ ) أَيْ فَيُحْرِمُ مِنْ الزَّمَانِ الَّذِي إذَا خَرَجَ فِيهِ ( عَلَى الْأَظْهَرِ ) حَقُّهُ عَلَى الْأَرْجَحِ .
