( وَإِذَا كَانَ فِي الْبُسْتَانِ شَجَرٌ مِنْ أَجْنَاسٍ ، كَتِينٍ وَزَيْتُونٍ وَكَرْمٍ فَشَرَطَ ) رَبُّ الْبُسْتَانِ ( لِلْعَامِلِ مِنْ كُلِّ جِنْسٍ ) مِنْ الشَّجَرِ ( قَدْرًا ) مَعْلُومًا ( كَنِصْفِ ثَمَرِ التِّينِ وَثُلُثِ ) ثَمَرِ ( الزَّيْتُونِ وَرُبْعِ ) ثَمَرِ ( الْكَرْمِ ) صَحَّ ( أَوْ كَانَ فِيهِ ) أَيْ الْبُسْتَانِ ( أَنْوَاعٌ مِنْ جِنْسٍ فَشَرَطَ مِنْ كُلِّ نَوْعٍ قَدْرًا ) مَعْلُومًا كَنِصْفِ الْبَرْنِيِّ ، وَثُلُثِ الصَّيْحَانِيِّ ، وَرُبْعِ الْإِبْرَاهِيمِيِّ .
