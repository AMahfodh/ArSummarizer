( وَإِنْ ) ( اسْتَشْعَرَ ) الْإِمَامُ أَيْ ظَنَّ ( خِيَانَتَهُمْ ) قَبْلَ الْمُدَّةِ بِظُهُورِ أَمَارَتِهَا ( نَبَذَهُ ) وُجُوبًا ، وَإِنَّمَا سَقَطَ الْعَهْدُ الْمُتَيَقَّنُ بِالظَّنِّ الَّذِي ظَهَرَتْ عَلَامَاتُهُ لِلضَّرُورَةِ ( وَأَنْذَرَهُمْ ) وُجُوبًا بِأَنَّهُ لَا عَهْدَ لَهُمْ فَإِنْ تَحَقَّقَ خِيَانَتُهُمْ نَبَذَهُ بِلَا إنْذَارٍ ( وَوَجَبَ الْوَفَاءُ ) بِمَا عَاهَدُونَا عَلَيْهِ ( وَإِنْ ) كَانَ عَهْدُنَا لَهُمْ ( بِرَدِّ رَهَائِنَ ) كُفَّارٍ عِنْدَنَا ( وَلَوْ أَسْلَمُوا ) حَيْثُ وَقَعَ اشْتِرَاطُ رَدِّهِمْ ، وَإِنْ لَمْ يَشْتَرِطُوا فِي الرَّدِّ إنْ( 7 / 291 )
