مُغْنِي وَيَأْتِي فِي الشَّارِحِ مَا يُوَافِقُهُ ( قَوْلُهُ : وَأَثِمَ بِالْإِقَامَةِ ) مِنْ عَطْفٍ لَازِمٍ ( قَوْلُهُ : عَلَى نَفْسِهَا ) أَيْ : أَوْ بُضْعِهَا ( قَوْلُهُ : فَمَعْذُورٌ ) أَيْ : إلَى أَنْ يُطِيقَهَا فَإِنْ فُتِحَ الْبَلَدُ قَبْلَ أَنْ يُهَاجِرَ سَقَطَ عَنْهُ الْهِجْرَةُ أَسْنَى وَمُغْنِي ( قَوْلُهُ : وَلِلْخَبَرِ الصَّحِيحِ إلَخْ ) فِي الِاسْتِدْلَالِ بِهِ تَوَقُّفٌ عِبَارَةُ الْأَسْنَى وَالْمُغْنِي وَخَبَرُ أَبِي دَاوُد وَغَيْرِهِ { أَنَا بَرِيءٌ مِنْ كُلِّ مُسْلِمٍ يُقِيمُ بَيْنَ أَظْهُرِ الْمُشْرِكِينَ } .
