( وَلَوْ تَرَكَ جَدَّ مَوْلَاهُ وَأَخَا ) مَوْلَاهُ ( فَالْجَدُّ أَوْلَى ) وَيَكُونُ الْوَلَاءُ كُلُّهُ لِلْجَدِّ عِنْدَ الْإِمَامِ ؛ لِأَنَّهُ أَقْرَبُ لِلْمَيِّتِ فِي الْعُصُوبَةِ مِنْ الْأَخِ عَلَى مَذْهَبِهِ ( وَعِنْدَهُمَا يَسْتَوِيَانِ ) فَيَكُونُ الْوَلَاءُ بَيْنَهُمَا نِصْفَيْنِ ( وَالْعَصَبَةُ إنَّمَا يَأْخُذُ مَا فَضَلَ عَنْ ذَوِي الْفُرُوضِ ) كَمَا مَرَّ .( 9 / 380 )
