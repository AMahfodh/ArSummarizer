( وَإِنْ كَفَلَ الْكَفِيلَ ) شَخْصٌ ( آخَرُ وَ ) كَفَلَ ( الْآخَرَ آخَرُ ) وَهَكَذَا ( بَرِئَ كُلٌّ ) مِنْ الْكُفَلَاءِ ( بِبَرَاءَةِ مَنْ قَبْلَهُ ) فَيَبْرَأُ الثَّانِي بِبَرَاءَةِ الْأَوَّلِ وَالثَّالِثُ بِبَرَاءَةِ الثَّانِي وَهَكَذَا ؛ لِأَنَّهُ فَرْعُهُ ( وَلَا عَكْسَ ) فَلَا يَبْرَأُ وَاحِدٌ بِبَرَاءَةِ مَنْ بَعْدَهُ ؛ لِأَنَّهُ أَصْلُهُ ( كَضَمَانٍ ) وَمَتَى سَلَّمَ أَحَدُهُمْ الْمَكْفُولَ بَرِئَ الْجَمِيعُ ؛ لِأَنَّهُ أَدَّى مَا عَلَيْهِمْ ، كَمَا لَوْ سَلَّمَ مَكْفُولٌ بِهِ نَفْسَهُ( 5 / 301 )
