( وَ ) مَنْ قَالَ ( لِزَوْجَتَيْهِ أَوْ أَمَتَيْهِ إحْدَاكُمَا طَالِقٌ ) غَدًا ( أَوْ حُرَّةٌ غَدًا فَمَاتَتْ إحْدَاهُمَا ) أَيْ الزَّوْجَتَيْنِ أَوْ الْأَمَتَيْنِ قَبْلَهُ ( أَوْ زَالَ مِلْكُهُ عَنْهُمَا ) بِأَنْ بَانَتْ مِنْهُ إحْدَى الزَّوْجَتَيْنِ أَوْ بَاعَ أَوْ وَهَبَ وَنَحْوُهُ إحْدَى الْأَمَتَيْنِ ( قَبْلَهُ ) أَيْ الْغَدِ ( وَقَعَ ) الطَّلَاقُ أَوْ الْعِتْقُ ( بِالْبَاقِيَةِ ) إذَا دَخَلَ الْغَدُ ؛ لِأَنَّ الْمَيِّتَةَ وَمَنْ زَالَ مِلْكُهُ عَنْهَا قَبْلَ وَقْتِ الْوُقُوعِ لَيْسَتْ مَحَلًّا لِلطَّلَاقِ وَلَا لِلْعِتْقِ أَشْبَهَ مَا لَوْ قَالَ لِزَوْجَتِهِ وَأَجْنَبِيَّةٍ : إحْدَاكُمَا طَالِقٌ أَوْ لِأَمَةٍ وَأَجْنَبِيَّةٍ : إحْدَاكُمَا حُرَّةٌ .( 9 / 459 )
