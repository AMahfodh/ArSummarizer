( وَ ) مَنَعَ مِنْ الرَّدِّ ( تَبَرِّي غَيْرِهِمَا ) أَيْ الْحَاكِمِ وَالْوَارِثِ ( فِي ) بَيْعِ ( ه ) أَيْ الرَّقِيقِ ( مِمَّا ) أَيْ عَيْبٍ ( لَمْ يَعْلَمْ ) ه الْبَائِعُ الْمُتَبَرِّي مِنْهُ فَلَا يَرُدُّ بِهِ إنْ ظَهَرَ بَعْدَ الشِّرَاءِ ( إنْ طَالَتْ إقَامَتُهُ ) أَيْ الرَّقِيقِ عِنْدَ بَائِعِهِ حُدَّتْ بِسِتَّةِ أَشْهُرٍ وَلَمْ يَطَّلِعْ عَلَى عَيْبِهِ ، وَغَلَبَ عَلَى ظَنِّهِ أَنَّهُ لَوْ كَانَ بِهِ عَيْبٌ لَظَهَرَ لَهُ وَجَوَّزَ أَنَّ بِهِ عَيْبًا أَخْفَاهُ لِأَنَّ الْإِنْسَانَ مَجْبُولٌ عَلَى إخْفَاءِ عُيُوبِهِ وَإِظْهَارِ بَرَاءَتِهِ مِنْهَا ، فَإِنْ كَانَ عَلِمَهُ أَوْ لَمْ تَطُلْ إقَامَتُهُ عِنْدَهُ فَلَا يَنْفَعُهُ تَبَرِّيهِ مِنْ عُيُوبِهِ ، وَمَتَى ظَهَرَ فِيهِ عَيْبٌ قَدِيمٌ فَلِمُشْتَرِيهِ رَدُّهُ عَلَى بَائِعِهِ .
