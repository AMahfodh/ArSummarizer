الشَّرْطُ ( الْخَامِسُ أَنْ يَكُونَ ) مَا يَتَمَلَّكُهُ الْأَبُ ( عَيْنًا مَوْجُودَةً ) فَلَا يَتَمَلَّكَ دَيْنَ ابْنِهِ لِأَنَّهُ لَا يَمْلِكُ التَّصَرُّفَ فِيهِ قَبْلَ قَبْضِهِ ( وَيَحْصُلُ تَمَلُّكُهُ ) أَيْ : الْأَبِ لِمَالِ وَلَدِهِ ( بِقَبْضِ ) مَا يَتَمَلَّكُهُ ( مَعَ قَوْلٍ ) تَمَلَّكْتُهُ أَوْ نَحْوِهِ ( أَوْ نِيَّةٍ ) قَالَ فِي الْفُرُوعِ : وَيَتَوَجَّهُ أَوْ قَرِينَةٍ لِأَنَّ الْقَبْضَ أَعَمُّ مِنْ أَنْ يَكُونَ لِلتَّمَلُّكِ أَوْ غَيْرِهِ فَاعْتُبِرَ الْقَوْلُ أَوْ النِّيَّةُ لِيَتَعَيَّنَ وَجْهُ الْقَبْضِ .
