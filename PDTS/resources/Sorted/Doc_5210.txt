( وَ ) السُّنَّةُ ( ذَبْحُ بَقَرٍ وَغَنَمٍ ) وقَوْله تَعَالَى { إنَّ اللَّهَ يَأْمُرُكُمْ أَنْ تَذْبَحُوا بَقَرَةً } وَلِحَدِيثِ أَنَسٍ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ ضَحَّى بِكَبْشَيْنِ ذَبَحَهُمَا بِيَدِهِ } ( وَيَجُوزُ عَكْسُهُ ) أَيْ : ذَبْحُ الْإِبِلِ وَنَحْرُ الْبَقَرِ وَالْغَنَمِ ؛ لِأَنَّهُ لَمْ يَتَجَاوَزْ مَحَلَّ الذَّكَاةِ وَلِعُمُومِ قَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { مَا أَنْهَرَ الدَّمَ وَذُكِرَ اسْمُ اللَّهِ عَلَيْهِ فَكُلْ } ( وَيَأْتِي ) ذَلِكَ .
