( وَإِلَّا ) لَمْ يَبْلُغْ الزَّرْعُ حَدَّ الِانْتِفَاعِ بِهِ بِأَنْ اُسْتُحِقَّتْ الْأَرْضُ بَعْدَ بُلُوغِهِ طَوْرًا يُنْتَفَعُ بِهِ فِيهِ إذَا قُلِعَ وَلَوْ لِرَعْيِ الْبَهَائِمِ ( فَلَهُ ) أَيْ الْمُسْتَحِقِّ ( قَلْعُهُ ) أَيْ أَمَرَ زَارِعَهُ بِهِ ( إنْ لَمْ يَفُتْ ) بِفَتْحِ التَّحْتِيَّةِ وَضَمِّ الْفَاءِ ، أَيْ يَمْضِ ( وَقْتُ مَا ) أَيْ الزَّرْعِ الَّذِي ( تُرَادُ ) بِضَمِّ الْفَوْقِيَّةِ الْأَرْضُ ( لَهُ ) سَوَاءٌ كَانَ مِنْ جِنْسِ زَرْعِ الْمُتَعَدِّي أَمْ لَا كَمَا لَوْ زُرِعَتْ سِمْسِمًا وَأَرَادَ الْمُسْتَحِقُّ زَرْعَهَا مِقْثَأَةً أَوْ بَقْلًا .
