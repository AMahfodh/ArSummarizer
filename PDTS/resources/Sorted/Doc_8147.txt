الْحَطّ اُنْظُرْ مَنْ صَحَّحَهُ ، وَأَشَارَ بِ وَلَوْ لِقَوْلِ ابْنِ الْمَوَّازِ يُبَاعُ الْمُعَيَّنُ الْمَعِيبُ وَيُشْتَرَى بِثَمَنِهِ سَلِيمٌ ، وَاتَّفَقَا عَلَى لُزُومِ سَلِيمٍ إذَا لَمْ يُعَيِّنْ كَعَلَيَّ هَدْيٌ أَعْوَرُ ( وَلَهُ ) أَيْ النَّاذِرِ ( فِيهِ ) أَيْ الْهَدْيُ سَلِيمًا أَوْ مَعِيبًا ( إذَا بِيعَ ) لِتَعَذُّرِ وُصُولِهِ ( الْإِبْدَالُ ) بِالنَّوْعِ ( الْأَفْضَلِ ) كَإِبْدَالِ كَبْشٍ بِبَقَرَةٍ أَوْ بَدَنَةٍ أَوْ بَقَرَةٍ بِبَدَنَةٍ فَلَا يُنَافِي وُجُوبَ إبْدَالِ الْمَعِيبِ بِسَلِيمٍ ( وَإِنْ كَانَ ) الْمَنْذُورُ هَدْيُهُ مُعَيَّنًا مِنْ جِنْسِ مَا لَا يُهْدَى ( كَثَوْبٍ ) وَعَبْدٍ( 5 / 436 )
