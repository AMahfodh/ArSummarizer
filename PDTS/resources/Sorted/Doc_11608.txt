( وَإِنْ ) ( بَاعَهَا ) الْمُلْتَقِطُ ( بَعْدَهَا ) أَيْ بَعْدَ السَّنَةِ الَّتِي عَرَّفَهَا بِهَا ( فَمَا لِرَبِّهَا إلَّا الثَّمَنُ ) الَّذِي بِيعَتْ بِهِ وَالْبَيْعُ مَاضٍ يَرْجِعُ بِهِ عَلَى الْمُلْتَقِطِ وَلَوْ عَدِيمًا لَا عَلَى الْمُشْتَرِي وَلَوْ مَلِيئًا ( بِخِلَافِ مَا لَوْ ) ( وَجَدَهَا ) رَبُّهَا ( بِيَدِ الْمِسْكِينِ ) الْمُتَصَدَّقِ بِهَا عَلَيْهِ ( أَوْ ) بِيَدِ ( مُبْتَاعٍ مِنْهُ ) أَيْ مِنْ الْمِسْكِينِ ( فَلَهُ ) أَيْ لِرَبِّهَا ( أَخْذُهَا ) مِنْ الْمِسْكِينِ أَوْ الْمُشْتَرِي مِنْهُ وَرَجَعَ الْمُشْتَرِي بِالثَّمَنِ عَلَى الْمِسْكِينِ إنْ وَجَدَ عِنْدَهُ وَإِلَّا فَعَلَى الْمُلْتَقِطِ الْمُتَصَدِّقِ بِهَا عَلَيْهِ ، قَوْلُهُ فَلَهُ أَخْذُهَا أَيْ أَوْ تَضْمِينُ الْمُلْتَقِطِ الْقِيمَةَ إنْ تَصَدَّقَ بِهَا عَنْ نَفْسِهِ مُطْلَقًا ، أَوْ عَنْ رَبِّهَا وَتَعَيَّبَتْ فَإِنْ بَقِيَتْ بِحَالِهَا تَعَيَّنَ أَخْذُهَا ، وَإِنْ فَاتَتْ تَعَيَّنَتْ الْقِيمَةُ عَلَى الْمُلْتَقِطِ .
