يَعْتَمِدْهُ ) وِفَاقًا لِلنِّهَايَةِ وَسَمِّ والبجيرمي وَشَيْخِنَا ( قَوْلُهُ وَالْحَدَثُ مِنْ هَذَا ) يُتَأَمَّلُ سم أَيْ إذْ الْحَدَثُ قَدْ يَكُونُ مِنْ غَيْرِ فِعْلِهِ كَمَا يَأْتِي ( قَوْلُهُ الْأَخْذَ بِهَا ) أَيْ بِالطَّهَارَةِ ( قَوْلُهُ وَحِكَايَتِهِمْ إلَخْ ) عَطْفٌ عَلَى قَطْعِهِمْ ( قَوْلُهُ غَلَبَتْ نَجَاسَتُهُ ) يَعْنِي غَلَبَ عَلَى الظَّنِّ تَنَجُّسُهُ بَعْدَ تَيَقُّنِ طَهَارَتِهِ ( قَوْلُهُ : بِأَنَّ الْأَسْبَابَ إلَخْ ) مُتَعَلِّقٌ بِفَرَّقَ ( قَوْلُهُ : فَكَانَ التَّمَسُّكُ ) أَيْ فِيمَا إذَا غَلَبَ عَلَى ظَنِّهِ الْحَدَثُ بَعْدَ تَيَقُّنِ الطَّهَارَةِ ( قَوْلُهُ : لِمَا ذَكَرْته ) أَيْ مِنْ الْفَرْقِ بَيْنَ الْحَدَثِ وَالنَّجَاسَةِ ( قَوْلُهُ : وَجَبَ عَلَيْهِ إلَخْ ) تَقَدَّمَ عَنْ الْبَصْرِيِّ تَرْجِيحُهُ وَعَنْ الرَّمْلِيِّ وَسَمِّ وَشَيْخِنَا خِلَافُهُ .
