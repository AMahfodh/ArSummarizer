( وَلَوْ ) كَانَ الْحَائِلُ ( كَمُؤَخَّرَةِ رَحْلٍ ) - بِضَمِّ الْمِيمِ وَسُكُونِ الْهَمْزَةِ وَمِنْهُمْ مَنْ يُثَقِّلُ الْخَاءَ ، وَهِيَ : الْخَشَبَةُ الَّتِي يَسْتَنِدُ إلَيْهَا الرَّاكِبُ - ( وَ ) يَكْفِي ( اسْتِتَارٌ بِدَابَّةٍ وَجَبَلٍ ) وَجِدَارٍ وَشَجَرَةٍ ، ( وَ ) يَكْفِي ( أَرِخَاءُ ذَيْلٍ ) ، لِحُصُولِ التَّسَتُّرِ بِهِ .
