( وَهُوَ ) أَيْ : الصَّيْدُ ( ضَرْبَانِ ) ( أَحَدُهُمَا لَهُ مِثْلٌ ) أَيْ : شَبِيهٌ ( مِنْ النَّعَمِ خِلْقَةً لَا قِيمَةً فَيَجِبُ فِيهِ مِثْلُهُ ) نَصَّ عَلَيْهِ لِلْآيَةِ ( وَهُوَ ) أَيْ : الَّذِي لَهُ مِثْلٌ ( نَوْعَانِ أَحَدُهُمَا مَا قَضَتْ فِيهِ الصَّحَابَةُ ) أَيْ : وَلَوْ الْبَعْضُ لَا كُلُّهُمْ ( فَفِيهِ مَا قَضَتْ بِهِ ) الصَّحَابَةُ وَتَقَدَّمَ تَعْرِيفُ الصَّحَابِيِّ فِي الْخُطْبَةِ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { أَصْحَابِي كَالنُّجُومِ بِأَيِّهِمْ أَقْتَدَيْتُمْ اهْتَدَيْتُمْ } وَلِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { عَلَيْكُمْ بِسُنَّتِي وَسُنَّةِ الْخُلَفَاءِ الرَّاشِدِينَ الْمَهْدِيِّينَ عَضُّوا عَلَيْهَا بِالنَّوَاجِذِ } رَوَاهُ أَحْمَدُ وَالتِّرْمِذِيُّ وَحَسَّنَهُ ؛ وَلِأَنَّهُمْ أَقْرَبُ إلَى الصَّوَابِ وَأَعْرَفُ بِمَوَاقِعِ الْخِطَابِ كَانَ حُكْمُهُمْ حُجَّةً عَلَى غَيْرِهِمْ كَالْعَالِمِ مَعَ الْعَامِّيِّ ( فَفِي النَّعَامَةِ بَدَنَةً ) حَكَمَ بِهِ عُمَرُ وَعُثْمَانُ وَعَلِيٌّ وَزَيْدٌ وَأَكْثَرُ الْعُلَمَاءِ ؛ لِأَنَّهَا تُشْبِهُ الْبَعِيرَ فِي خِلْقَتِهِ فَكَانَ مِثْلًا لَهَا فَيَدْخُلُ فِي عُمُومِ النَّصِّ وَجَعَلَهَا الْخِرَقِيُّ مِنْ أَقْسَامِ الطَّيْرِ ؛ لِأَنَّ لَهَا جَنَاحَيْنِ فَيُعَايَى بِهَا فَيُقَالُ : طَائِرٌ يَجِبُ فِيهِ بَدَنَةٌ .( 7 / 98 )
