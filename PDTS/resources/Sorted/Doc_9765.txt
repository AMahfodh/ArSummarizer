( وَيَتَّجِهُ ) أَنَّهُ ( يَجِبُ ) عَلَى دَاخِلِ حَمَّامٍ ( اقْتِصَادٌ فِي ) اسْتِعْمَالِ ( الْمَاءِ عَلَى قَدْرِ حَاجَتِهِ ) حَيْثُ كَانَ مَمْلُوكًا أَوْ مُسَبَّلًا ، ( فَإِنَّهُ ) - أَيْ : قَدْرَ الْحَاجَةِ - ( الْمَأْذُونُ فِيهِ ) شَرْعًا وَعُرْفًا ( بِقَرِينَةِ الْحَالِ ، لَا سِيَّمَا ) الْمَاءَ ( الْحَارَّ لِمَا فِيهِ مِنْ مُؤْنَةِ التَّعَبِ ) بِتَحْصِيلِ الْوَقُودِ وَأُجْرَةِ الْعَمَلَةِ ( وَ ) يَتَّجِهُ : أَنَّ ( مِثْلَهُ كُلُّ مَاءٍ سُبِّلَ لِنَحْوِ وُضُوءٍ ) كَغُسْلٍ مِنْ جَنَابَةٍ أَوْ حَيْضٍ أَوْ نِفَاسٍ أَوْ إزَالَةِ نَجَاسَةٍ ، فَلَا يُزَادُ فِي ذَلِكَ عَلَى قَدْرِ الْحَاجَةِ ، وَهُوَ مُتَّجِهٌ .( 1 / 440 )
