( وَيَجْهَرُ فِيهِمَا بِالْقِرَاءَةِ ) اعْتِبَارًا بِصَلَاةِ الْعِيدِ ( ثُمَّ يَخْطُبُ ) لِمَا رُوِيَ { أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ خَطَبَ } ثُمَّ هِيَ كَخُطْبَةِ الْعِيدِ عِنْدَ مُحَمَّدٍ ، وَعِنْدَ أَبِي يُوسُفَ خُطْبَةٌ وَاحِدَةٌ ( وَلَا خُطْبَةَ عِنْدَ أَبِي حَنِيفَةَ ) ؛ لِأَنَّهَا تَبَعٌ لِلْجَمَاعَةِ وَلَا جَمَاعَةَ عِنْدَهُ ( وَيَسْتَقْبِلُ الْقِبْلَةَ بِالدُّعَاءِ ) لِمَا رُوِيَ { أَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ اسْتَقْبَلَ الْقِبْلَةَ وَحَوَّلَ رِدَاءَهُ } ( وَيَقْلِبُ رِدَاءَهُ ) لِمَا رَوَيْنَا .
