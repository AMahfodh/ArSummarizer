( وَإِنْ فَقَأَ أَعْوَرُ مِنْ سَالِمٍ مُمَاثِلَتَهُ ) أَيْ مُمَاثِلَةَ الْجَانِي السَّالِمَةِ ( فَلَهُ ) أَيْ لِسَالِمِ الْعَيْنَيْنِ عَلَيْهِ ( الْقِصَاصُ ) مِنْ الْأَعْوَرِ الْجَانِي بِأَنْ يَفْقَأَ عَيْنَهُ السَّالِمَةَ فَيُصَيِّرَهُ أَعْمَى ( أَوْ ) يَتْرُكَ الْقِصَاصَ وَيَأْخُذَ مِنْ الْجَانِي ( دِيَةَ مَا تَرَكَهُ ) : وَهِيَ عَيْنُ الْجَانِي ، وَدِيَتُهَا أَلْفُ دِينَارٍ عَلَى أَهْلِ الذَّهَبِ .
