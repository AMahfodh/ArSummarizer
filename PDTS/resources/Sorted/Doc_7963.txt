وَاسْتَثْنَى مِنْ قَوْلِهِ : وَإِلَّا فَلَا يَصِحُّ قَوْلَهُ : ( إلَّا أَنْ يُجِيزَ ) : الْمُجْبِرُ ( عَقْدَ مَنْ فَوَّضَ ) الْمُجْبِرُ ( لَهُ أُمُورَهُ ) مِنْ الْأَوْلِيَاءِ كَابْنٍ وَأَخٍ وَجَدٍّ وَغَيْرِهِمْ وَثَبَتَ التَّفْوِيضُ لَهُ ، ( بِبَيِّنَةٍ ) لَا بِمُجَرَّدِ دَعْوَى وَلَا بِإِقْرَارٍ مِنْ الْمُجْبِرِ بَعْدَ الْعَقْدِ ، ( فَيُمْضِيَ ) ذَلِكَ الْعَقْدَ وَلَا يُفْسَخُ ( إنْ لَمْ يَبْعُدْ ) بِأَنْ قَرُبَ مَا بَيْنَ الْعَقْدِ مِنْ الْمُفَوَّضِ لَهُ وَالْإِجَازَةُ مِنْ الْمُجْبِرِ ( عَلَى الْأَوْجَهِ ) مِنْ التَّأْوِيلَيْنِ ، لِأَنَّ عَقْدَ الْمُفَوَّضِ مَعَ وُجُودِ الْمُجْبِرِ خِلَافُ الْأَصْلِ .
