أَقْسَمَ كُلٌّ مِنْهُمَا عَلَى مَنْ عَيَّنَهُ الْآخَرُ ، وَأَخَذَ رُبْعَ الدِّيَةِ ( وَهَلْ يَحْلِفُ كُلٌّ ) مِنْهُمَا فِي الْمَرَّةِ الثَّانِيَةِ ( خَمْسِينَ يَمِينًا أَوْ نِصْفَهَا ) فِيهِ ( خِلَافٌ ) يَأْتِي فِي نَظَائِرِهِ ( أَوْ ) قَالَ كُلٌّ مِنْهُمَا بَعْدَمَا ذَكَرَ ( الْمَجْهُولَ غَيْرَ مَنْ عَيَّنَهُ ) صَاحِبِي ( رَدَّ كُلٌّ ) مِنْهُمَا ( مَا أَخَذَهُ ) لِتَكَاذُبِهِمَا ( وَإِنْ قَالَ ذَلِكَ أَحَدُهُمَا رَدَّ صَاحِبُهُ وَحْدَهُ ) مَا أَخَذَهُ ؛ لِأَنَّ قَائِلَ ذَلِكَ كَذَّبَهُ بِخِلَافِ قَائِلِهِ وَلِصَاحِبِهِ أَنْ يُحَلِّفَ مَنْ عَيَّنَهُ ، وَقَوْلُهُ ( وَلِكُلٍّ ) مِنْهُمَا ( تَحْلِيفُ مَنْ عَيَّنَهُ ) مُتَعَلِّقٌ بِاَلَّتِي قَبْلَ هَذِهِ وَكَلَامُهُ يَقْتَضِي خِلَافَهُ .
