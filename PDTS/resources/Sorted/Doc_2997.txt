( فَصْلٌ ) فِي مُبْطِلَاتِ التَّيَمُّمِ ( يَبْطُلُ التَّيَمُّمُ بِخُرُوجِ الْوَقْتِ ) لِقَوْلِ عَلِيٍّ التَّيَمُّمُ لِكُلِّ صَلَاةٍ وَلِأَنَّهُ طَهَارَةُ ضَرُورَةٍ فَتَقَيَّدَ بِالْوَقْتِ ، كَطَهَارَةِ الْمُسْتَحَاضَةِ ( حَتَّى ) التَّيَمُّمِ ( مِنْ جُنُبٍ لِقُرْآنٍ وَلُبْثٍ فِي مَسْجِدٍ وَ ) حَتَّى التَّيَمُّمِ مِنْ ( حَيْضٍ لِوَطْءٍ وَ ) حَتَّى التَّيَمُّمِ ( لِطَوَافٍ ، وَ ) حَتَّى التَّيَمُّمِ مِنْ ( نَجَاسَةٍ ) بِبَدَنٍ .
