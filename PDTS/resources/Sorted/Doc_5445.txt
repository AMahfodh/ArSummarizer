( أَوْ ) قَالُوا كُلُّهُمْ عَمْدًا وَ ( نَكَلُوا ) عَنْ الْقَسَامَةِ فَيَبْطُلُ الدَّمُ ( بِخِلَافِهِ ذِي ) أَيْ صَاحِبِ أَيْ مُدَّعِي قَتْلَ ( الْخَطَأِ ) وَقَالَ غَيْرُهُ مِنْ الْأَوْلِيَاءِ لَا نَعْلَمُ ( فَلَهُ ) أَيْ ذِي الْخَطَأِ ( الْحَلِفُ ) لِجَمِيعِ أَيْمَانِ الْقَسَامَةِ ( وَأَخْذُ نَصِيبِهِ ) مِنْ الدِّيَةِ لِأَنَّهُ مَالٌ أَمْكَنَ تَوْزِيعُهُ ، بِخِلَافِ الْعَمْدِ وَكَذَا إنْ اتَّفَقُوا عَلَى الْخَطَأِ وَنَكَلَ بَعْضُهُمْ فَلِمَنْ حَلَفَ تَكْمِيلُ أَيْمَانِ الْقَسَامَةِ وَأَخْذُ نَصِيبِهِ مِنْهَا .
