( وَ ) إنْ وَلَدْت ثَلَاثَةً ( مُتَعَاقِبِينَ ) وَاحِدًا بَعْدَ وَاحِدٍ ( طَلُقَتْ بِأَوَّلٍ ) طَلْقَةً ( وَبِثَانٍ ) طَلْقَةً ( وَبَانَتْ بِثَالِثٍ ) وَلَمْ تَطْلُقْ بِهِ ( لِانْقِضَاءِ عِدَّتِهَا بِهِ ) أَيْ : بِوَصْفِهِ ، وَإِنَّمَا لَمْ تَنْقَضِ عِدَّتُهَا بِالثَّانِي ، لِأَنَّهُ لَيْسَ تَمَامَ حَمْلِهَا ، وَالْعِدَّةُ إنَّمَا تَتِمُّ بِوَضْعِ جَمِيعِ الْحَمْلِ ( وَإِنْ وَلَدَتْ اثْنَيْنِ ) مُتَعَاقِبَيْنِ ( وَ ) كَانَ ( زَادَ : لِلسُّنَّةِ ) بِأَنْ قَالَ كُلَّمَا وَلَدْتِ فَأَنْتِ طَالِقٌ لِلسُّنَّةِ (( 16 / 167 )
