، وَالْأَوْجَهُ أَنْ يُقَالَ لَيْسَ ذَلِكَ مِنْ الْقَاعِدَةِ الْمَذْكُورَةِ لِأَنَّ صُورَتَهُ أَنْ يَأْذَنَ الْوَاهِبُ لِلْمُتَّهِبِ فِي الْقَبْضِ مِمَّنْ ذُكِرَ وَيُوَكِّلَهُ الْمُتَّهِبُ فِي الْقَبْضِ لَا أَنْ يَأْذَنَ الْوَاهِبُ لِمَنْ ذُكِرَ فِي الْإِقْبَاضِ وَيُوَكِّلَهُ الْمُتَّهِبُ فِي الْقَبْضِ ( وَلَوْ وَهَبَ مَرْهُونًا وَكَلْبًا ) وَلَوْ مُعَلَّمًا ( وَخَمْرًا ) وَلَوْ ( مُحْتَرَمَةً وَجِلْدَ مَيْتَةٍ قَبْلَ دَبْغِهِ لَمْ يَصِحَّ ) كَالْبَيْعِ ( وَهِبَةُ الدَّيْنِ ) لِلْمَدِينِ ( إبْرَاءٌ ) لَهُ مِنْهُ ( لَا تَحْتَاجُ قَبُولًا ) نَظَرًا لِلْمَعْنَى وَتَرْكُهُ لَهُ كِنَايَةُ إبْرَاءٍ وَقِيلَ صَرِيحُهُ ( وَهِبَتُهُ لِلْأَجْنَبِيِّ بَاطِلَةٌ ) لِعَجْزِهِ عَنْ تَسْلِيمِهِ ؛ لِأَنَّ مَا يُقْبَضُ مِنْ الْمَدِينِ عَيْنٌ لَا دَيْنٌ ( وَتَمْلِيكُ الْمِسْكِينِ الدَّيْنَ ) اللَّازِمَ لِمَنْ عَلَيْهِ زَكَاةٌ ( وَلَوْ ) كَانَ الدَّيْنُ ( عَلَى غَيْرِهِ ) أَيْ غَيْرِ الْمِسْكِينِ ( عَنْ الزَّكَاةِ لَا يَصِحُّ ) لِأَنَّ ذَلِكَ فِيمَا عَلَيْهِ إبْدَالٌ وَهُوَ لَا يَجُوزُ فِيهَا وَفِيمَا عَلَى غَيْرِهِ تَمْلِيكٌ وَهُوَ لَا يَجُوزُ أَيْضًا كَمَا مَرَّ .
