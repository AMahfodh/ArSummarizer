لِأَنَّا نَقُولُ : الْعِلَّةُ فِي الصِّحَّةِ لُزُومُهُمْ جِهَةً وَاحِدَةً وَعَقْلُهُمْ يَقْتَضِي ذَلِكَ وَإِنْ كَانُوا مِلْكًا أَوْ اعْتَقَدُوا الْوُجُوبَ ( قَوْلُهُ : جَازَ ذَلِكَ ) مُعْتَمَدٌ ( قَوْلُهُ : وَمِثْلُهَا الْوَثْبَةُ الْفَاحِشَةُ وَهُوَ مُحْتَمَلٌ ) مُعْتَمَدٌ ( قَوْلُهُ : وَلِنُدْرَةِ هَذِهِ الصَّلَاةِ ) قَالَ حَجّ : وَالْفَرْقُ بِهَذَا أَوْلَى مِنْ الْفَرْقِ بِأَنَّ الْجُلُوسَ يَمْحُو صُورَتَهَا لِأَنَّهُ مُنْتَقَضٌ بِامْتِنَاعِ فِعْلِهَا عَلَى السَّائِرَةِ عَلَى الْمُعْتَمَدِ مَعَ بَقَاءِ الْقِيَامِ ( قَوْلُهُ : حَتَّى لَوْ فُرِضَ إتْمَامُهُ ) أَيْ الْقِيَامُ ( قَوْلُهُ : فَكَذَلِكَ كَمَا اقْتَضَاهُ كَلَامُهُمْ ) أَيْ لَا يَصِحُّ حَيْثُ( 3 / 475 )
