( وَتَصِحُّ ) مُسَاقَاةٌ ( بِلَفْظِهَا ) كَسَاقَيْتُكَ عَلَى هَذَا الْبُسْتَانِ ؛ لِأَنَّهُ لَفْظُهَا الْمَوْضُوعُ لَهَا ، ( وَ ) تَصِحُّ ( بِلَفْظِ مُعَامَلَةٍ وَمُفَالَحَةٍ ، وَ ) بِلَفْظِ ( اعْمَلْ بِبُسْتَانِي هَذَا ) حَتَّى تَكْمُلَ ثَمَرَتُهُ عَلَى النِّصْفِ مَثَلًا ( وَنَحْوِهِ ) مِمَّا يُؤَدِّي إلَى ذَلِكَ الْمَعْنَى ؛ لِأَنَّهُ الْعَقْدُ ، فَإِذَا دَلَّ عَلَيْهِ أَيُّ لَفْظٍ كَانَ ؛ صَحَّ كَالْبَيْعِ .( 10 / 197 )
