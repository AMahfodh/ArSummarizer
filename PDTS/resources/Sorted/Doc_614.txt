( وَ ) لِرَاهِنٍ ( انْتِفَاعٌ ) بِرَهْنٍ مُطْلَقًا ( بِإِذْنِ مُرْتَهِنٍ وَ ) لَهُ ( وَطْءُ ) مَرْهُونَةٍ ( بِشَرْطِ ) وَطْئِهَا ( أَوْ إذْنِ ) مُرْتَهِنٍ ؛ لِأَنَّ الْمَنْعَ لَحِقَهُ وَقَدْ أَسْقَطَهُ بِإِذْنِهِ فِيهِ أَوْ الرِّضَا بِهِ ، فَإِنْ لَمْ يَكُنْ إذْنٌ وَلَا شَرْطٌ حَرُمَ ذَلِكَ( 5 / 206 )
