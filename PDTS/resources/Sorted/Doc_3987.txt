( قَوْلَانِ ) ( وَعَبْدُ الْحَرْبِيِّ يُسْلِمُ ) دُونَ سَيِّدِهِ ( حُرٌّ ) وَكَذَا إنْ لَمْ يُسْلِمْ ( إنْ فَرَّ ) إلَيْنَا ( أَوْ ) أَسْلَمَ ، وَ ( بَقِيَ حَتَّى غَنِمَ ) قَبْلَ إسْلَامِ سَيِّدِهِ فَحُرٌّ أَيْضًا ( لَا إنْ ) ( خَرَجَ ) فَارًّا إلَيْنَا ( بَعْدَ إسْلَامِ سَيِّدِهِ أَوْ بِمُجَرَّدِ إسْلَامِهِ ) أَيْ السَّيِّدِ أَيْ خَرَجَ مُصَاحِبًا لِإِسْلَامِ سَيِّدِهِ فَهُوَ رَقِيقٌ لَهُ .
