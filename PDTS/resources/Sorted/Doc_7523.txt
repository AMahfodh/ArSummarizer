الْقِسْمُ ( الثَّانِي ) مِنْ أَقْسَامِ الْخِيَارِ : خِيَارُ الشَّرْطِ ب ( أَنْ يَشْتَرِطَاهُ ) أَيْ يَشْتَرِطَ الْعَاقِدَانِ الْخِيَارَ ( فِي ) صُلْبِ ( الْعَقْدِ ، أَوْ ) يَشْتَرِطَاهُ بَعْدَهُ ( زَمَنَ الْخِيَارَيْنِ ) أَيْ خِيَارِ الْمَجْلِسِ وَخِيَارِ الشَّرْطِ ؛ لِأَنَّهُ بِمَنْزِلَةِ حَالِ الْعَقْدِ ( إلَى أَمَدٍ مَعْلُومٍ فَيَصِحُّ ) وَلَوْ فَوْقَ ثَلَاثَةِ أَيَّامٍ لِحَدِيثِ { الْمُسْلِمُونَ عِنْدَ شُرُوطِهِمْ } وَلِأَنَّهُ حَقٌّ يَعْتَمِدُ الشَّرْطَ فَرَجَعَ فِي تَقْدِيرِهِ إلَى مُشْتَرِطِهِ كَالْأَجَلِ قَالَ فِي شَرْحِهِ : وَلَمْ يَثْبُتْ مَا رُوِيَ عَنْ عُمَرَ أَيْ مِنْ تَقْدِيرِهِ بِثَلَاثٍ .
