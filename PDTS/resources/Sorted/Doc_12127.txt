( قَوْلُهُ : ) ( فَيُؤَجَّلُ ) أَيْ الْمَفْقُودُ الْحُرُّ أَرْبَعَ سِنِينَ سَوَاءٌ كَانَتْ الزَّوْجَةُ مَدْخُولًا بِهَا أَمْ لَا دَعَتْهُ قَبْلَ غَيْبَتِهِ لِلدُّخُولِ أَوْ لَا وَالْحَقُّ أَنَّ تَأْجِيلَ الْحُرِّ بِأَرْبَعِ سِنِينَ وَالْعَبْدِ بِنِصْفِهَا تَعَبُّدِيٌّ أَجْمَعَ الصَّحَابَةُ عَلَيْهِ ( قَوْلُهُ وَإِلَّا طَلَّقَ عَلَيْهِ ) أَيِّ مِنْ حِينِ الْعَجْزِ عَنْ خَبَرِهِ مِنْ غَيْرِ تَأْجِيلٍ بَعْدَ ذَلِكَ ( قَوْلُهُ : مِنْ حِينِ الْعَجْزِ عَنْ خَبَرِهِ ) مُتَعَلِّقٌ بِقَوْلِهِ فَيُؤَجِّلُ إلَخْ ( قَوْلُهُ بِالْبَحْثِ عَنْهُ ) أَيْ بَعْدَ الْبَحْثِ عَنْهُ مِنْ هُنَا نَقَلَ الْمَشَذَّالِيُّ عَنْ السُّيُورِيِّ أَنَّ الْمَفْقُودَ الْيَوْمَ يُنْتَظَرُ بِهِ مُدَّةُ التَّعْمِيرِ لِعَدَمِ مَنْ يَبْحَثُ عَنْهُ الْآنَ وَأَقَرَّهُ تِلْمِيذُهُ عَبْدُ الْحَمِيدِ كَمَا فِي الْبَدْرِ الْقَرَافِيِّ ( قَوْلُهُ : بِأَنْ يُرْسِلَ إلَخْ ) هَذَا تَصْوِيرٌ لِلْبَحْثِ عَنْهُ وَأُجْرَةُ الرَّسُولِ عَلَيْهَا ؛ لِأَنَّهَا الطَّالِبَةُ هَذَا إنْ كَانَ لَهَا مَالٌ وَإِلَّا فَمِنْ بَيْتِ الْمَالِ( 10 / 123 )
