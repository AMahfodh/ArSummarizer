قَالَهُ فِي الْمُبْدِعِ ( إنْ لَمْ يَكُنْ عُذْرٌ ) مِنْ مَرَضٍ وَسَفَرٍ وَنَحْوِهِمَا ( فَإِنْ كَانَ ) ثَمَّ عُذْرٌ ( لَمْ يُكْرَهْ ) أَنْ يَقْرَأَ ( بِأَقْصَرَ مِنْهُ ) أَيْ : مِمَّا ذَكَرَ ، وَقِرَاءَةُ السُّورَةِ وَإِنْ قَصُرَتْ أَفْضَلُ مِنْ بَعْضِهَا ، وَلَا يُعْتَدُّ بِالسُّورَةِ قَبْلَ الْفَاتِحَةِ .( 2 / 496 )
