وَكَتَبَ سم أَيْضًا : قَوْلَهُ : وَإِنْ بَانَ إمَامًا : أَيْ إنْ طَالَ زَمَنُ التَّرَدُّدِ أَوْ مَضَى رُكْنٌ كَمَا هُوَ ظَاهِرٌ ( قَوْلُهُ : فَيَنْبَغِي أَنْ يَصِحَّ ) أَيْ فَلَوْ تَغَيَّرَ اجْتِهَادُهُ فَهَلْ يَجِبُ الِاسْتِئْنَافُ أَوْ نِيَّةُ الْمُفَارَقَةِ ؟ فِيهِ نَظَرٌ ، وَلَا يَبْعُدُ الثَّانِي ( قَوْلُهُ : أَنَّهُ إمَامٌ ) أَيْ بِصَاحِبِهِ ( قَوْلُهُ : وَقَدْ مَرَّ ) أَيْ وَهُوَ أَنَّهُ إذَا طَالَ التَّرَدُّدُ أَوْ مَضَى رُكْنٌ ضُرَّ وَإِلَّا فَلَا ( قَوْلُهُ : وَهَذَا ) أَيْ طَرِيقُ الْمَرَاوِزَةِ .
