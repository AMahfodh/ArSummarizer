( وَلَا يَلْزَمُ ) الْمُجْتَهِدَ ( إعْلَامُ الْمُقَلِّدِ ) بِكَسْرِ اللَّامِ ( بِتَغَيُّرِهِ ) أَيْ تَغَيُّرِ اجْتِهَادِهِ لِأَنَّهُ لَا يَلْزَمُ الْمُقَلِّدَ أَنْ يُفَارِقَ بِتَغَيُّرِ اجْتِهَادِ مَنْ قَلَّدَهُ لِمَا فِيهِ مِنْ الْحَرَجِ وَالْمَشَقَّةِ ( وَإِنْ بَانَ خَطَؤُهُ ) أَيْ الْحَاكِمِ ( فِي إتْلَافٍ ) كَقَطْعٍ وَقَتْلٍ ( لِمُخَالَفَةِ دَلِيلٍ قَاطِعٍ أَوْ ) بَانَ ( خَطَأُ مُفْتٍ لَيْسَ أَهْلًا ) لَلْفُتْيَا ( ضَمِنَا ) أَيْ الْحَاكِمُ وَالْمُفْتِي لِأَنَّهُ إتْلَافٌ حَصَلَ بِفِعْلِهِمَا أَشْبَهَ مَا لَوْ بَاشَرَاهُ وَعَلِمَ مِنْهُ أَنَّهُ لَوْ أَخْطَأَ فِيمَا لَيْسَ بِقَاطِعٍ مِمَّا يَقْبَلُ الِاجْتِهَادَ لَا ضَمَانَ .( 22 / 334 )
