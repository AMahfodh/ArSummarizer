وَشَبَّهَ فِي التَّرْكِ فَقَالَ ( كَ ) ضَالَّةِ ( إبِلٍ ) فَيَجِبُ تَرْكُهَا بِمَحَلِّ وُجُودِهَا وَيَحْرُمُ الْتِقَاطُهَا ( فَإِنْ أُخِذَتْ ) بِضَمٍّ فَكَسْرٍ أَيْ اُلْتُقِطَتْ ضَالَّةُ الْإِبِلِ ( عُرِّفَتْ ) بِضَمٍّ فَكَسْرٍ مُثَقَّلًا ضَالَّةُ الْإِبِلِ سَنَةً ( ثُمَّ ) إنْ لَمْ يُوجَدْ مُسْتَحِقُّهَا ( تُرِكَتْ ) بِضَمٍّ فَكَسْرٍ ضَالَّةُ الْإِبِلِ ( بِمَحَلِّهَا ) الَّذِي وُجِدَتْ بِهِ .
