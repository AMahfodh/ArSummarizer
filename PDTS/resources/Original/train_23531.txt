( قَوْلُهُ : بِالنِّسْبَةِ لِأَهْلِ الذِّمَّةِ ) أَيْ فِيمَا لَوْ أَعَانَ أَهْلُ الذِّمَّةِ الْبُغَاةَ وَادَّعَوْا أَنَّهُمْ أَكْرَهُوهُمْ عَلَى إعَانَتِهِمْ فَلَا يُكَلَّفُونَ بَيِّنَةً عَلَى ذَلِكَ ( قَوْلُهُ : بِالنِّسْبَةِ لِغَيْرِهِمْ ) مِنْ الْمُعَاهَدِينَ وَالْمُؤَمَّنِينَ ( قَوْلُهُ : ضَمِنُوهُ ) أَيْ بِغَيْرِ الْقِصَاصِ( 25 / 412 )
