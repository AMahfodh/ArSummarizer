( وَمَنْ صَحَّ أَمَانُهُ ) مِمَّنْ تَقَدَّمَ ( قَبْلَ إخْبَارِهِ بِهِ ) ( إذَا كَانَ عَدْلًا كَمُرْضِعَةٍ عَلَى فِعْلِهَا ) ، وَقَاسَمَ وَنَحْوَهُ ، ( وَلَا يَنْقُضُهُ ) ، أَيْ : أَمَانُ مُسْلِمٍ ( إمَامٌ ) حَيْثُ صَحَّ لِوُقُوعِهِ لَازِمًا ( إلَّا إذَا خَافَ خِيَانَةً ) مِمَّنْ أُعْطِيَهُ فَيَنْقُضُهُ ، لِفَوَاتِ شَرْطِهِ( 7 / 151 )
