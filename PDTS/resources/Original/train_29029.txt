( وَ ) كَ ( مُضِيِّ ) بِضَمِّ الْمِيمِ وَكَسْرِ الضَّادِ الْمُعْجَمَةِ وَشَدِّ الْيَاءِ أَيْ فَرَاغِ ( يَوْمِ ) أَيْ زَمَنٍ يَوْمًا كَانَ أَوْ أَقَلَّ أَوْ أَكْثَرَ وَلَمْ تَخْتَرْ فِيهِ شَيْئًا فَقَدْ سَقَطَ مَا جَعَلَهُ لَهَا سَوَاءٌ عَلِمَتْ بِمُضِيِّهِ أَمْ لَا بِأَنْ أُغْمِيَ عَلَيْهَا أَوْ جُنَّتْ حَتَّى فَاتَ ( وَ ) كَ ( رَدِّهَا ) أَيْ الزَّوْجَةِ مِنْ إضَافَةِ( 8 / 216 )
