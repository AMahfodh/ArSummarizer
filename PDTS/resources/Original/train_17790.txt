يُطْعِمَ ) الْمُكَفِّرُ ( بَعْضًا ) مِنْ الْعَشَرَةِ ( وَيَكْسُوَ بَعْضًا ) مِنْهُمْ لِأَنَّ اللَّهَ تَعَالَى خَيَّرَ مَنْ وَجَبَتْ عَلَيْهِ الْكَفَّارَةُ بَيْنَ الْإِطْعَامِ وَالْكُسْوَةِ فَكَانَ مَرْجِعُهُمَا إلَى اخْتِيَاره فِي الْعَشَرَةِ وَفِي بَعْضِهِمْ بِخِلَافِ مَا لَمْ يُخَيِّرْهُ فِيهِ ( فَإِنْ أَطْعَمَ الْمِسْكِينَ بَعْضَ الطَّعَامِ وَكَسَاهُ بَعْضَ الْكُسْوَةِ ) لَمْ يُجْزِئْهُ لِأَنَّهُ لَمْ يُطْعِمْهُ وَلَمْ يَكْسُهُ ( أَوْ أَعْتَقَ نِصْفَ عَبْدٍ وَأَطْعَمَ خَمْسَةً أَوْ كَسَاهُمْ ) لَمْ يُجْزِئْهُ لِأَنَّهُ لَمْ يُحَرِّرْ رَقَبَةً وَلَمْ يُطْعِم أَوْ يَكْسُو عَشْرَة ( أَوْ أَطْعَمَ ) بَعْض الْعَشَرَة ( وَصَامَ ) دُون الثَّلَاثَة .
