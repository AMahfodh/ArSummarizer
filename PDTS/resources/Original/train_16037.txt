( فَصْلٌ : وَأَمَّا أَحْكَامُهُ ) أَيْ الِالْتِقَاطِ ( فَعَلَى الْمُلْتَقِطِ ) مِنْهَا ( حِفْظُ اللَّقِيطِ وَرِعَايَتُهُ ) أَيْ تَرْبِيَتُهُ ؛ لِأَنَّ ذَلِكَ مَقْصُودُ الِالْتِقَاطِ ( لَا نَفَقَتُهُ وَحَضَانَتُهُ ) الْمُفَصَّلَةُ فِي الْإِجَارَةِ ؛ لِأَنَّ فِيهِمَا مَشَقَّةً وَمُؤْنَةً كَثِيرَةً فَالْمُرَادُ بِقَوْلِهِمْ هُنَا وَحَضَانَتُهُ عَلَى الْمُلْتَقِطِ حِفْظُهُ وَتَرْبِيَتُهُ لَا الْحَضَانَةُ الْمَذْكُورَةُ ( فَإِنْ عَجَزَ ) عَنْ حِفْظِهِ وَرِعَايَتِهِ لِأَمْرٍ عَرَضَ ( فَالْقَاضِي ) أَيْ فَيُسَلِّمُهُ لَهُ ( وَلَهُ تَسْلِيمُهُ إلَيْهِ ) لِتَبَرُّمٍ ، أَوْ غَيْرِهِ ( وَلَوْ قَدَرَ ) عَلَى ذَلِكَ أَيْضًا فَتَقْيِيدُ الْأَصْلِ ذَلِكَ بِالتَّبَرُّمِ جَرَى عَلَى الْغَالِبِ ( وَيَحْرُمُ ) عَلَيْهِ ( نَبْذُهُ وَرَدُّهُ إلَى مَا كَانَ ) بِالِاتِّفَاقِ .( 13 / 143 )
