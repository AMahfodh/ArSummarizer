( وَإِنْ اشْتَرَى ) مُظَاهِرٌ ( زَوْجَتَهُ ) الَّتِي ظَاهَرَ مِنْهَا وَهِيَ أَمَةٌ فَظِهَارُهُ بِحَالِهِ وَلَهُ عِتْقُهَا عَنْهُ فَإِنْ عَادَ وَتَزَوَّجَهَا فَلَا كَفَّارَةَ وَإِنْ أَعْتَقَهَا عَنْ غَيْرِهِ ثُمَّ تَزَوَّجَهَا فَعَلَيْهِ الْكَفَّارَةُ بِالْوَطْءِ ( أَوْ بَانَتْ زَوْجَةٌ مُظَاهَرٌ مِنْهَا ) حُرَّةً كَانَتْ أَوْ أَمَةً ( قَبْلَ الْوَطْءِ ثُمَّ أَعَادَهَا مُطْلَقًا ) ارْتَدَّ أَوْ لَا ( فَظِهَارُهُ بِحَالِهِ ) نَصًّا لِعُمُومِ الْآيَةِ وَالْخَبَرِ ؛ وَلِأَنَّ التَّحْرِيمَ إنَّمَا يَزُولُ بِالتَّكْفِيرِ .( 10 / 44 )
