( وَإِنْ قَالَ ) شَخْصٌ لِآخَرَ ( اقْضِنِي دَيْنِي عَلَيْك أَلْفًا ) فَقَالَ : نَعَمْ ( أَوْ قَالَ اشْتَرِ ثَوْبِي هَذَا ) ، فَقَالَ : نَعَمْ ( أَوْ قَالَ لَهُ أَعْطِنِي ثَوْبِي هَذَا ) فَقَالَ : نَعَمْ ، أَوْ قَالَ لَهُ : سَلِّمْ ( إلَيَّ أَلْفًا مِنْ الَّذِي عَلَيْك ) فَقَالَ : نَعَمْ ، أَوْ قَالَ لَهُ ( هَلْ لِي عَلَيْك ) أَلْفٌ ؟ فَقَالَ : نَعَمْ ( أَوْ قَالَ لَهُ أَلِي عَلَيْك أَلْفٌ ، فَقَالَ : نَعَمْ ) فَقَدْ أَقَرَّ لِأَنَّهَا صَرِيحَةٌ فِيهِ ( أَوْ ) قَالَ ( أَمْهِلْنِي يَوْمًا أَوْ ) أَمْهِلْنِي ( حَتَّى أَفْتَحَ الصُّنْدُوقَ ) فَقَدْ أَقَرَّ ؛ لِأَنَّ طَلَبَ الْمُهْلَةِ يَقْتَضِي أَنَّ الْحَقَّ عَلَيْهِ ( أَوْ ) قَالَ لَهُ ( عَلَيَّ أَلْفٌ إنْ شَاءَ اللَّهُ ) فَقَدْ أَقَرَّ لَهُ بِهِ نَصًّا ؛ لِأَنَّهُ وَصَلَ إقْرَارَهُ بِمَا يَرْفَعُهُ كُلَّهُ ، وَيَصْرِفُهُ إلَى غَيْرِ الْإِقْرَارِ ؛ فَلَزِمَهُ مَا أَقَرَّ بِهِ ، وَبَطَلَ مَا وَصَلَهُ بِهِ ، كَقَوْلِهِ لَهُ عَلَيَّ أَلْفٌ إلَّا أَلْفًا ، وَكَقَوْلِهِ عَلَيَّ أَلْفٌ فِي مَشِيئَةِ اللَّهِ ، أَوْ قَالَ عَلَيَّ أَلْفٌ ( لَا يَلْزَمُنِي إلَّا أَنْ يَشَاءَ اللَّهُ ) فَقَدْ أَقَرَّ لَهُ بِالْأَلْفِ ، لِمَا تَقَدَّمَ ، أَوْ قَالَ لَهُ عَلَيَّ أَلْفٌ ( فِي عِلْمِي أَوْ قَالَ : فِي عِلْمِ اللَّهِ أَوْ قَالَ : فِيمَا أَعْلَمُ لَا إنْ قَالَ : فِيمَا أَظُنُّ ؛ فَقَدْ أَقَرَّ ) لَهُ بِالْأَلْفِ ؛ لِأَنَّهُ مُثْبِتٌ لِإِقْرَارِهِ بِالْعِلْمِ بِهِ ؛ إذْ مَا فِي عِلْمُهُ لَا يَحْتَمِلُ غَيْرَ الْوُجُوبِ ، بِخِلَافِ الظَّنِّ .( 20 / 260 )
