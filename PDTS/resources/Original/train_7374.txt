( وَ ) بِدُخُولِ ( بَيْتِ شَعْرٍ وَ ) بَيْتِ ( أُدُمٍ وَخَيْمَةٍ ) لِقَوْلِهِ تَعَالَى : { وَجَعَلَ لَكُمْ مِنْ جُلُودِ الْأَنْعَامِ بُيُوتًا } الْآيَةَ وَالْخَيْمَةُ فِي مَعْنَى بَيْتِ الشَّعْرِ وَ ( لَا ) يَحْنَثُ ( بِ ) دُخُولِ ( صِفَةِ دَارٍ وَدِهْلِيزِ ) هَا لِأَنَّهُ لَا يُسَمَّى بَيْتًا لِأَنَّهُ لَيْسَ مَحَلَّ الْبَيْتُوتَةِ .
