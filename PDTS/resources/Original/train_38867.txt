وَ ( لَا ) يَجُوزُ الْإِلْزَامُ بِشَرْطِ ( تَأْجِيلِ ) قَرْضٍ ( أَوْ ) شَرْطِ ( نَقْصٍ فِي وَفَاءٍ ) ؛ لِأَنَّهُ يُنَافِي مُقْتَضَى الْعَقْدِ ( أَوْ ) شَرْطٍ ( جَرِّ نَفْعًا ) فَيَحْرُمُ ( كَ ) شَرْطِهِ ( أَنْ يُسْكِنَهُ ) أَيْ الْمُقْتَرِضُ ( دَارِهِ أَوْ يَقْضِيَهُ خَيْرًا مِنْهُ ) أَيْ : مِمَّا أَقْرَضَهُ ( أَوْ ) أَنْ يَقْضِيَهُ ( بِبَلَدٍ آخَرَ ) وَلِحَمْلِهِ مُؤْنَةً ؛ لِأَنَّهُ عَقْدُ إرْفَاقٍ وَقُرْبَةٍ ، فَشَرْطُ النَّفْعِ فِيهِ يُخْرِجُهُ عَنْ مَوْضُوعِهِ فَإِنْ لَمْ يَكُنْ لِحَمْلِهِ مُؤْنَةٌ فَقَالَ فِي الْمُغْنِي : الصَّحِيحُ جَوَازُهُ ؛ لِأَنَّهُ مَصْلَحَةٌ لَهُمَا مِنْ غَيْرِ ضَرَرٍ وَكَذَا لَوْ أَرَادَ إرْسَالَ نَفَقَةٍ إلَى أَهْلِهِ فَأَقْرَضَهَا لِيُوَفِّيَهَا الْمُقْتَرِضُ لَهُمْ جَازَ وَلَا يَفْسُدُ الْقَرْضُ بِفَسَادِ الشَّرْطِ( 5 / 162 )
