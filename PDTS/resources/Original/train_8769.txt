( وَأَمَّا قَوْلُهُمْ ) أَيْ اسْتِدْلَالُ الْأَكْثَرِينَ بِأَنَّهُ لَوْ جَازَ التَّفْصِيلُ ( يَلْزَمُ تَخْطِئَةُ كُلِّ فَرِيقٍ ) مِنْ الْمُطْلِقِينَ لِكَوْنِهِمْ لَمْ يُفَصِّلُوا ( فَيَلْزَمُ تَخْطِئَتُهُمْ ) أَيْ الْأُمَّةِ كُلِّهَا وَتَخْطِئَتُهَا غَيْرُ جَائِزٍ لِلنَّصِّ عَلَى أَنَّهَا لَا تَجْتَمِعُ عَلَى ضَلَالَةٍ فَالتَّفْصِيلُ غَيْرُ جَائِزٍ ( فَدُفِعَ بِأَنَّ الْمُنْتَفِي ) فِي النَّصِّ ( تَخْطِئَةُ الْكُلِّ فِيمَا اتَّفَقُوا عَلَيْهِ( 5 / 119 )
