بَعْدَ النُّزُولِ ( عَبْدٌ ) هُوَ ( يَعْدِلُ مِائَةً فَدُفِعَ ) بِصِيغَةِ الْمَجْهُولِ ( بِهِ ) أَيْ دُفِعَ الْعَبْدُ الْجَانِي مَقَامَ الْعَبْدِ الْمَقْتُولِ بِسَبَبِ قَتْلِهِ ( افْتَكَّهُ الرَّاهِنُ بِكُلِّ الدَّيْنِ ) وَهُوَ الْأَلْفُ عِنْدَ الشَّيْخَيْنِ لِأَنَّ التَّغَيُّرَ لَمْ يَظْهَرْ فِي نَفْسِ الْعَبْدِ إذَا الْعَبْدُ الثَّانِي قَامَ مَقَامَ الْأَوَّلِ مِنْ حَيْثُ أَنَّهُ دَمٌ وَلَحْمٌ فَكَأَنَّهُ تَرَاجَعَ سِعْرُهُ إلَى مِائَةٍ فَلَوْ كَانَ الْأَوَّلُ قَائِمًا وَتَرَاجَعَ سِعْرُهُ لَمْ يَكُنْ لَهُ خِيَارٌ فَكَذَلِكَ هُنَا ( وَعِنْدَ مُحَمَّدٍ ) هُوَ بِالْخِيَارِ ( إنْ شَاءَ دَفَعَهُ ) أَيْ الْعَبْدَ الْمَدْفُوعَ ( إلَى الْمُرْتَهِنِ ) بِدَيْنِهِ وَلَا شَيْءَ عَلَيْهِ غَيْرُهُ .
