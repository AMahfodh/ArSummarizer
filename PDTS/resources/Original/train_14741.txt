( إذَا شَرَطَ ) الْكَفِيلُ ( الْبَرَاءَةَ مِنْهُ ) أَيْ مِنْ الْمَالِ عِنْدَ تَعَذُّرِ إحْضَارِهِ عَلَيْهِ لِحَدِيثِ { الْمُسْلِمُونَ عَلَى شُرُوطِهِمْ } وَلِأَنَّهُ إنَّمَا الْتَزَمَ إحْضَارَهُ عَلَى هَذَا الْوَجْهِ فَلَا يَلْزَمُهُ غَيْرُ مَا الْتَزَمَهُ ( وَإِنْ ثَبَتَ ) بِبَيِّنَةٍ أَوْ إقْرَارِ مَكْفُولٍ لَهُ ( مَوْتُهُ ) أَيْ الْمَكْفُولِ الْغَائِبِ أَوْ نَحْوِهِ ( قَبْلَ غُرْمِهِ ) أَيْ الْكَفِيلِ الْمَالَ لِانْقِطَاعِ خَبَرِهِ ( اسْتَرَدَّهُ ) أَيْ مَا غَرِمَهُ كَفِيلٌ لِتَبَيُّنِ بَرَاءَةِ الْكَفِيلِ بِمَوْتِ الْمَكْفُولِ فَلَا يَسْتَحِقُّ الْأَخْذَ مِنْهُ وَإِنْ قَدَرَ عَلَى مَكْفُولٍ بَعْدَ أَدَائِهِ عَنْهُ مَا لَزِمَهُ فَظَاهِرُ كَلَامِهِمْ : أَنَّهُ فِي رُجُوعِهِ عَلَيْهِ كَضَامِنٍ وَأَنْ لَا يُسَلِّمَهُ إلَى الْمَكْفُولِ لَهُ ثُمَّ يَسْتَرِدَّ مَا أَدَّاهُ ، بِخِلَافِ مَغْصُوبٍ تَعَذَّرَ إحْضَارُهُ مَعَ بَقَائِهِ ، لِامْتِنَاعِ بَيْعِهِ قَالَهُ( 5 / 297 )
