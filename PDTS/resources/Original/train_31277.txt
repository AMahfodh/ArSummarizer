( وَشَرْطُهُ ) أَيْ الْمُنْقَطِعِ ( كَوْنُهُ ) أَيْ الْمُسْتَثْنَى ( مِمَّا يُقَارِنُهُ ) أَيْ الْمُسْتَثْنَى مِنْهُ ( كَثِيرًا ) لِمُلَابَسَتِهِ إيَّاهُ وَكَوْنُهُ مِنْ تَوَابِعِهِ حَتَّى يَسْتَحْضِرَ بِذِكْرِهِ أَوْ بِذِكْرِ مَا يُنْسَبُ إلَيْهِ ( كَجَاءُوا ) أَيْ الْقَوْمُ مَثَلًا ( إلَّا حِمَارًا ) ؛ لِأَنَّهُ لَيْسَ مِنْهُمْ بَلْ مِنْ تَوَابِعِهِمْ بِحَيْثُ يُسْتَحْضَرُ بِذِكْرِهِمْ فِي( 2 / 190 )
