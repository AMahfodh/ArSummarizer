( قَوْلُهُ : إلَّا أَنْ تُجْعَلَ الْإِضَافَةُ بَيَانِيَّةً ) فِيهِ نَظَرٌ فَإِنَّ شَرْطَهَا أَنْ يَكُونَ بَيْنَ الْمُضَافِ وَالْمُضَافِ إلَيْهِ عُمُومٌ وَخُصُوصٌ وَجْهِيٌّ وَمَا هُنَا لَيْسَ كَذَلِكَ ( قَوْلُهُ تَحْدِيدَهُ ) أَيْ : بَيَانَ آخِرِ الظَّاهِرِ مِنْ جِهَةِ الْجَوْفِ وَيُحْتَمَلُ أَنَّ الْمَعْنَى بَيَانُ حَدِّ الظَّاهِرِ وَتَعْرِيفُهُ ( قَوْلُهُ وَذِكْرَ الْخِلَافِ إلَخْ ) عَطْفٌ عَلَى قَوْلِهِ تَحْدِيدَهُ ( قَوْلُهُ أَهُوَ الْمُعْجَمَةُ ) أَيْ : مَخْرَجُهَا ( قَوْلُهُ وَهُوَ الْمُعْتَمَدُ ) وِفَاقًا لِلنِّهَايَةِ وَالْمُغْنِي ( قَوْلُهُ فَيَدْخُلُ ) أَيْ : فِي الظَّاهِرِ ( قَوْلُهُ كُلُّ مَا قَبْلَهُ ) أَيْ : قَبْلَ مَخْرَجِ الْمُهْمَلَةِ ( قَوْلُهُ : إنْ أَمْكَنَهُ ) إلَى قَوْلِهِ بِخِلَافِ جَوْفِ إلَخْ فِي النِّهَايَةِ وَكَذَا فِي الْمُغْنِي( 13 / 308 )
