وَلَمَّا كَانَتْ دِمَاءُ الْحَجِّ أَوْ الْعُمْرَةِ ثَلَاثَةً : الْفِدْيَةَ ، وَجَزَاءَ الصَّيْدِ ، وَالْهَدْيَ وَقَدَّمَ الْكَلَامَ عَلَى الْأَوَّلَيْنِ أَشَارَ لِلثَّالِثِ بِقَوْلِهِ : ( وَغَيْرُ الْفِدْيَةِ وَ ) غَيْرُ ( جَزَاءِ الصَّيْدِ : هَدْيٌ ) مُرَتَّبٌ ( وَهُوَ ) : أَيْ الْهَدْيُ ( مَا وَجَبَ لِتَمَتُّعٍ ) قَالَ تَعَالَى : { فَمَنْ تَمَتَّعَ بِالْعُمْرَةِ إلَى الْحَجِّ فَمَا اسْتَيْسَرَ مِنْ الْهَدْيِ } .
