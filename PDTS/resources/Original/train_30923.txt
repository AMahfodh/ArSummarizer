( وَلَوْ أَخْبَرَهُ بِحَقِّ أَبِيهِ ثِقَةٌ ) أَيْ عَدْلٌ ضَابِطٌ ( فَسَكَنَ إلَيْهِ جَازَ أَنْ يَحْلِفَ عَلَيْهِ ) إذَا أَقَامَ بِهِ شَاهِدًا ( وَلَمْ يَجُزْ أَنْ يَشْهَدَ بِهِ ) وَالْفَرْقُ بَيْنَ الْيَمِينِ وَالشَّهَادَةِ مِنْ وَجْهَيْنِ أَحَدِهِمَا أَنَّ الشَّهَادَةَ حَقٌّ لِغَيْرِهِ فَيُحْتَمَلُ أَنَّ مَنْ لَهُ الشَّهَادَةُ قَدْ زَوَّرَ عَلَى خَطِّهِ الثَّانِي أَمَّا مَا يَكْتُبُهُ الْإِنْسَانُ مِنْ حُقُوقٍ بِكُتُبِهِ فَيَنْسَى بَعْضَهُ بِخِلَافِ الشَّهَادَةِ ( وَالْأَوْلَى الْوَرَعُ عَنْ ) الْحَلِفِ عَلَى ( ذَلِكَ ) احْتِيَاطًا .( 23 / 97 )
