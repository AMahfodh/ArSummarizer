( وَإِنْ قَالَ ) لِزَوْجَتِهِ ( إنْ أَمَرْتُكِ فَخَالَفْتِينِي فَأَنْتِ طَالِقٌ فَنَهَاهَا ) عَنْ شَيْءٍ ( وَخَالَفَتْهُ ) فِيهِ ( لَمْ يَحْنَثْ ) وَلَوْ لَمْ تَعْرِفْ حَقِيقَةَ الْأَمْرِ وَالنَّهْيِ لِأَنَّهَا خَالَفَتْ نَهْيَهُ لَا أَمْرَهُ ( إلَّا أَنْ يَنْوِي مُطْلَقَ الْمُخَالَفَةِ ) فَيَحْنَثُ بِمُخَالَفَةِ النَّهْيِ لِأَنَّهَا مُخَالَفَةٌ ( و ) لَوْ قَالَ ( إنْ نَهَيْتُكِ فَخَالَفْتِينِي فَأَنْتِ طَالِقٌ فَأَمَرَهَا ) بِشَيْءٍ ( وَخَالَفَتْهُ لَمْ يَحْنَثْ فِي قِيَاسِ الَّتِي قَبْلَهَا إلَّا أَنْ يَنْوِي مُطْلَقَ الْمُخَالَفَةِ ) لِمَا تَقَدَّمَ .( 18 / 274 )
