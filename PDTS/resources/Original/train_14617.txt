( وَلَا يَطْهُرُ جِلْدُ مَا كَانَ نَجِسًا فِي حَيَاتِهِ ) كَالْكَلْبِ ( بِذَكَاتِهِ كَ ) مَا لَا يَطْهُرُ ( لَحْمُهُ ) بِهَا لِأَنَّهُ لَيْسَ مَحِلًّا لِلذَّكَاةِ فَهُوَ مَيْتَةٌ ( فَلَا يَجُوزُ ذَبْحُهُ لِذَلِكَ ) أَيْ لِجِلْدِهِ أَوْ لَحْمِهِ ، لِأَنَّهُ عَبَثٌ وَإِضَاعَةٌ لِمَا قَدْ يُنْتَفَعُ بِهِ ( وَلَا ) يَجُوزُ ذَبْحُهُ أَيْضًا ( لِغَيْرِهِ ) كَلِإِرَاحَتِهِ ( وَلَوْ ) كَانَ ( فِي النَّزْعِ ) وَكَذَا الْآدَمِيُّ بَلْ أَوْلَى وَلَوْ وَصَلَ إلَى حَالَةٍ لَا يَعِيشُ فِيهَا عَادَةً أَوْ كَانَ بَقَاؤُهُ أَشَدَّ تَأْلِيمًا لَهُ وَقَدْ عَمَّتْ بِذَلِكَ الْبَلْوَى .( 1 / 133 )
