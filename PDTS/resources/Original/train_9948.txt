وَإِذَا نَكَلَ بَعْضُ الْأَوْلِيَاءِ وَسَقَطَ الدَّمُ ( فَتُرَدُّ ) بِضَمِّ التَّاءِ وَفَتْحِ الرَّاءِ الْقَسَامَةُ ( عَلَى الْمُدَّعَى عَلَيْهِمْ ) الْقَتْلُ ( فَيَحْلِفُ كُلٌّ ) مِنْهُمْ ( خَمْسِينَ يَمِينًا ) إنْ تَعَدَّدُوا لِأَنَّ كُلَّ وَاحِدٍ مِنْهُمْ مُتَّهَمٌ بِهِ ، وَإِنْ كَانَ وَاحِدًا حَلَفَهَا وَحْدَهُ ( وَمَنْ نَكَلَ ) عَنْهَا مِنْ الْمُدَّعَى عَلَيْهِمْ ( حُبِسَ ) بِضَمٍّ فَكَسْرٍ ( حَتَّى يَحْلِفَ ) خَمْسِينَ يَمِينًا أَوْ يَمُوتَ لِأَنَّ كُلَّ مَنْ طُلِبَ مِنْهُ أَمْرٌ سُجِنَ بِسَبَبِهِ فَلَا يَخْرُجُ إلَّا بَعْدَ حُصُولِ ذَلِكَ الْمَطْلُوبِ .
