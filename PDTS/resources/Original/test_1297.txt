( وَالْمُعْتَبَرُ فِيهِ الْقُرْبُ وَالْجُزْئِيَّةُ ) فَلَوْ لَهُ بِنْتٌ وَابْنُ ابْنٍ أَوْ بِنْتُ بِنْتٍ وَأَخٌ النَّفَقَةُ عَلَى الْبِنْتِ أَوْ بِنْتِهَا ؛ لِأَنَّهُ ( لَا ) يُعْتَبَرُ ( الْإِرْثُ ) إلَّا إذَا اسْتَوَيَا كَجَدٍّ وَابْنِ ابْنٍ فَكَإِرْثِهِمَا إلَّا لِمُرَجِّحٍ - كَوَالِدٍ وَوَلَدٍ ( فَعَلَى وَلَدِهِ لِتَرَجُّحِهِ ، ب { أَنْتَ وَمَالُك لِأَبِيك } ) وَفِي الْخَانِيَّةِ : لَهُ أُمٌّ وَأَبُو أَبٍ فَكَإِرْثِهِمَا .
