وَلَمَّا فَرَغَ مِنْ تَعْدَادِ مَنْ يَرِثُ شَرَعَ يُبَيِّنُ مِقْدَارَ مَا يَرِثُ كُلُّ وَاحِدٍ مِنْهُمْ فَقَالَ : ( فَمِيرَاثُ الزَّوْجِ مِنْ الزَّوْجَةِ إنْ لَمْ تَتْرُكْ وَلَدًا وَلَا وَلَدَ ابْنٍ النِّصْفُ فَإِنْ تَرَكَتْ وَلَدًا ) ذَكَرًا كَانَ أَوْ أُنْثَى ( أَوْ وَلَدَ ابْنٍ ) كَذَلِكَ سَوَاءٌ كَانَ الْوَلَدُ ( مِنْهُ ) أَيْ مِنْ الزَّوْجِ ( أَوْ مِنْ غَيْرِهِ ) بِنِكَاحٍ أَوْ زِنًا أَوْ لِعَانٍ مِنْ حُرٍّ أَوْ عَبْدٍ مُسْلِمٍ أَوْ كَافِرٍ ، وَيُشْتَرَطُ فِي الْوَلَدِ أَوْ وَلَدِهِ أَنْ يَكُونَ حُرًّا مُسْلِمًا غَيْرَ قَاتِلٍ ( فَلَهُ ) أَيْ الزَّوْجِ ( الرُّبْعُ ) ، وَدَلِيلُ الْفَرِيضَتَيْنِ قَوْله تَعَالَى : { لَكُمْ نِصْفُ مَا تَرَكَ أَزْوَاجُكُمْ } .
