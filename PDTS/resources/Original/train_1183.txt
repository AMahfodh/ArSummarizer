( وَلَا تَصِحُّ الْخُطْبَةُ بِغَيْرِ الْعَرَبِيَّةِ مَعَ الْقُدْرَةِ ) عَلَيْهَا بِالْعَرَبِيَّةِ ( كَقِرَاءَةٍ ) فَإِنَّهَا لَا تُجْزِئُ بِغَيْرِ الْعَرَبِيَّةِ وَتَقَدَّمَ ( وَتَصِحُّ ) الْخُطْبَةُ بِغَيْرِ الْعَرَبِيَّةِ ( مَعَ الْعَجْزِ ) عَنْهَا بِالْعَرَبِيَّةِ لِأَنَّ الْمَقْصُودَ بِهَا الْوَعْظُ وَالتَّذْكِيرُ وَحَمْدُ اللَّهِ وَالصَّلَاةُ عَلَى رَسُولِ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ بِخِلَافِ لَفْظِ الْقُرْآنِ فَإِنَّهُ دَلِيلُ النُّبُوَّةِ وَعَلَامَةُ الرِّسَالَةِ وَلَا يَحْصُلُ بِالْعَجَمِيَّةِ ( غَيْرَ الْقِرَاءَةِ ) فَلَا تُجْزِئُ( 4 / 145 )
