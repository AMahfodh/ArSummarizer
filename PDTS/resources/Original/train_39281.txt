( وَإِنْ قَطَعَهُ ) أَيْ : الصَّوْمَ ( بِلَا عُذْرٍ اسْتَأْنَفَهُ ) لِئَلَّا يَفُوتَ التَّتَابُعُ ( وَ ) إنْ قَطَعَهُ لِعُذْرٍ يُخَيَّرُ ( بَيْنَهُ ) أَيْ : الِاسْتِئْنَافِ ( بِلَا كَفَّارَةٍ ) لِفِعْلِهِ الْمَنْذُورَ عَلَى وَجْهِهِ ( وَبَيْنَ الْبِنَاءِ ) عَلَى مَا مَضَى ( وَيُتِمُّ ثَلَاثِينَ ) يَوْمًا ( وَيُكَفِّرُ ) كَمَا لَوْ حَلَفَ عَلَيْهِ لِأَنَّهُ لَمْ يَأْتِ( 12 / 40 )
