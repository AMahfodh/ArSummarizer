( وَإِنْ جَنَتْ ) الْأَمَةُ الْمُوصَى بِنَفْعِهَا ( سَلَّمَهَا وَارِثٌ ) إلَى وَلِيِّ الْجِنَايَةِ ( أَوْ فَدَاهَا مَسْلُوبَةَ ) الْمَنْفَعَةِ بِالْأَقَلِّ مِنْ أَرْشِ الْجِنَايَةِ أَوْ قِيمَتِهَا ؛ لِأَنَّهُ يَمْلِكُهَا كَذَلِكَ كَأُمِّ الْوَلَدِ ( وَعَلَيْهِ ) ؛ أَيْ : الْوَارِثِ ( إنْ قَتَلَهَا قِيمَةُ الْمَنْفَعَةِ لِلْوَصِيِّ ) ؛ أَيْ : لِلْمُوصَى لَهُ بِمَنْفَعَتِهَا .
