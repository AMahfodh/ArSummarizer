وَ ( لَا ) يَخْرُجُ مِنْهُ ( بِجُنُونٍ وَإِغْمَاءٍ وَسُكْرٍ وَمَوْتٍ ) لِخَبَرِ الْمُحْرِمِ الَّذِي وَقَصَتْهُ رَاحِلَتهُ ( وَلَا يَنْعَقِدُ ) الْإِحْرَامُ ( مَعَ وُجُودِ أَحَدِهَا ) أَيْ : الْجُنُونِ أَوْ الْإِغْمَاءِ أَوْ السُّكْرِ لِعَدَمِ أَهْلِيَّتِهِ لِلنِّيَّةِ ( وَتَقَدَّمَ بَعْضُ ذَلِكَ ) مُوَضَّحًا .( 6 / 392 )
