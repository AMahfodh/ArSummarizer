( وَكَذَا إنْ لَمْ يَرَ ) الْمَأْمُومُ ( أَحَدَهُمَا ) أَيْ : الْإِمَامِ أَوْ مَنْ وَرَاءَهُ ( إنْ سَمِعَ التَّكْبِيرَ ) ؛ لِأَنَّهُمْ فِي مَوْضِعِ الْجَمَاعَةِ وَيُمْكِنُهُمْ الِاقْتِدَاءُ بِهِ بِسَمَاعِ التَّكْبِيرِ أَشْبَهَ الْمُشَاهَدَةَ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَسْمَعْ التَّكْبِيرَ وَلَمْ يَرَهُ وَلَا بَعْضُ مَنْ وَرَاءَهُ ( فَلَا ) تَصِحُّ صَلَاةُ الْمَأْمُومِ ، لِعَدَمِ تَمَكُّنِهِ مِنْ الِاقْتِدَاءِ بِإِمَامِهِ .( 3 / 496 )
