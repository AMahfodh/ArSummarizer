( وَ ) غَيْرُ ( صُلْحٍ ) بِهَا ( عَنْ دَمٍ عَمْدٍ ) فِي نَفْسٍ أَوْ طَرَفٍ ( بِكَوْنِهَا ) أَيْ الدَّرَاهِمُ وَالدَّنَانِيرُ الْمُعَيَّنَةُ ( مَغْصُوبَةً ) كَالْمَبِيعِ يَظْهَرُ مُسْتَحَقًّا ( أَوْ ) بِكَوْنِهَا ( مَعِيبَةً ) عَيْبًا ( مِنْ غَيْرِ جِنْسِهَا ) كَكَوْنِ الدَّرَاهِمِ نُحَاسًا أَوْ رَصَاصًا ؛ لِأَنَّهُ بَاعَهُ غَيْرَ مَا سُمِّيَ لَهُ ( وَ ) يَبْطُلُ غَيْرُ مَا تَقَدَّمَ اسْتِثْنَاؤُهُ ( فِي بَعْضٍ هُوَ كَذَلِكَ ) أَيْ مَغْصُوبٌ أَوْ مَعِيبٌ مِنْ غَيْرِ جِنْسِهَا ( فَقَطْ ) وَيَصِحُّ فِي الْبَاقِي بِنَاءً عَلَى تَفْرِيقِ الصَّفْقَةِ( 5 / 73 )
