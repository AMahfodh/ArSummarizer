( أَوْ ) أَيْ وَلَا تُقْبَلُ شَهَادَةُ الشَّفِيعِ ( بِبَيْعِ الشِّقْصِ الَّذِي تَجِبُ فِيهِ الشُّفْعَةُ ) لِلتُّهْمَةِ ( وَإِنْ أَسْقَطَ ) الشَّفِيعُ ( شُفْعَتَهُ قَبْلَ الْحُكْمِ بِشَهَادَتِهِ ) بِعَفْوِ شَرِيكِهِ أَوْ بَيْعِ الشِّقْصِ ( قُبِلَتْ ) شَهَادَتُهُ لِانْتِفَاءِ التُّهْمَةِ وَ ( لَا ) تُقْبَلُ شَهَادَتُهُ إنْ عَفَا عَنْ شُفْعَتِهِ ( بَعْدَ الرَّدِّ ) لِشَهَادَتِهِ لِأَنَّهُ مُتَّهَمٌ لِكَوْنِهِ إنَّمَا عَفَا لِتُقْبَلَ شَهَادَتُهُ .( 23 / 60 )
