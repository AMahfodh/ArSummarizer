( فَإِنْ أَتَتْ بِهِ ) أُمُّهُ ( لِأَكْثَرَ مِنْ ذَلِكَ ) أَيْ مِنْ سِتَّةِ أَشْهُرٍ ( وَكَانَ لَهَا زَوْجٌ ) يَطَؤُهَا ( أَوْ ) لَهَا ( سَيِّدٌ يَطَؤُهَا لَمْ يَرِثْ ) لِاحْتِمَالِ تَجَدُّدِهِ بَعْدَ الْمَوْتِ ( إلَّا أَنْ تُقِرَّ الْوَرَثَةُ أَنَّهُ كَانَ مَوْجُودًا حَالَ الْمَوْتِ ) فَيَلْزَمَهُمْ دَفْعُ مِيرَاثِهِ إلَيْهِ مُؤَاخَذَةً لَهُمْ بِإِقْرَارِهِمْ .
