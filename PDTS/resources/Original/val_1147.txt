( وَإِنْ تَلِفَ أَجِيرٌ ) مُكَلَّفٌ ( لِحَفْرِهَا بِهَا ) فَهَدَرٌ ؛ لِأَنَّهُ لَا فِعْلَ لِلْمُسْتَأْجِرِ فِي قَتْلِهِ بِمُبَاشَرَةٍ وَلَا سَبَبٍ ( أَوْ دَعَا مَنْ يَحْفِرُ لَهُ بِدَارِهِ ) أَوْ أَرْضِهِ حَفِيرَةً ( أَوْ ) مَنْ يَحْفِرُ لَهُ ( بِمَعْدِنٍ ) يَسْتَخْرِجُهُ لَهُ ( فَمَاتَ بِهَدْمِ ) ذَلِكَ عَلَيْهِ بِلَا فِعْلِ أَحَدٍ ( فَهَدَرٌ ) نَصًّا لِمَا تَقَدَّمَ .( 10 / 395 )
