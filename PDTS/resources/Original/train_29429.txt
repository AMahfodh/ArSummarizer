لِتَقَدُّمِهَا ( فَإِنْ نَكَحَهَا ) أَيْ شَخْصٌ امْرَأَةً نِكَاحًا ( فَاسِدًا وَوَطِئَهَا غَيْرُهُ بِشُبْهَةٍ ) قَبْلَ وَطْئِهِ أَوْ بَعْدَهُ ( ثُمَّ فُرِّقَ بَيْنَهُمَا قُدِّمَتْ عِدَّةُ الْوَاطِئِ ) بِالشُّبْهَةِ ( لِتَوَقُّفِ تِلْكَ ) أَيْ عِدَّةِ النِّكَاحِ ( عَلَى التَّفْرِيقِ ) بِخِلَافِ عِدَّةِ الشُّبْهَةِ فَإِنَّهَا مِنْ وَقْتِ الْوَطْءِ وَلَيْسَ لِلْفَاسِدِ قُوَّةُ الصَّحِيحِ حَتَّى يُرَجَّحَ بِهَا فَهُمَا كَوَاطِئَيْنِ وَطِئَاهَا بِشُبْهَةٍ ( وَإِنْ نَكَحَتْ فَاسِدًا بَعْدَ ) مُضِيِّ ( قُرْأَيْنِ وَوَطِئَتْ وَلَمْ يُفَرَّقْ بَيْنَهُمَا إلَى ) مُضِيِّ ( سِنِّ الْيَأْسِ أَتَمَّتْ ) لِعِدَّةِ ( الْأُولَى بِشَهْرٍ ) بَدَلًا عَنْ الْقُرْءِ الْبَاقِي ( وَاعْتَدَّتْ لِلشُّبْهَةِ ) الْأُولَى قَوْلُ أَصْلِهِ ثُمَّ اعْتَدَّتْ لِلْفَاسِدِ ( بِثَلَاثَةِ أَشْهُرٍ ، وَإِنْ كَانَ ) ثَمَّ ( حَمْلٌ فَعِدَّةُ صَاحِبِهِ مُقَدَّمَةٌ مُطْلَقًا ) أَيْ سَوَاءٌ أَكَانَ الْحَمْلُ مُتَقَدِّمًا أَمْ مُتَأَخِّرًا ؛ لِأَنَّ عِدَّتَهُ لَا تَقْبَلُ التَّأْخِيرَ ( فَإِنْ كَانَتَا ) أَيْ عِدَّةُ الْحَمْلِ وَعِدَّةُ غَيْرِهِ ( مِنْ وَطْءِ شُبْهَةٍ ) .
