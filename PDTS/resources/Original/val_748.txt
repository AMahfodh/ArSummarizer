( وَ ) إنْ حَلَفَ ( لَا يَأْكُلُ وَلَا يَشْرَبُ فَمَصَّ قَصَبَ السُّكَّرِ أَوْ ) مَصَّ ( الرُّمَّانَ وَنَحْوَهُ لَمْ يَحْنَثْ ) لِأَنَّهُ فِي الْعُرْفِ لَا يُسَمَّى أَكْلًا وَلَا شُرْبًا ( وَكَذَا ) لَوْ حَلَفَ ( لَا يَأْكُلُ سُكَّرًا فَتَرَكَهُ فِي فِيهِ حَتَّى ذَابَ وَابْتَلَعَهُ ) لَمْ يَحْنَثْ لِأَنَّهُ لَيْسَ أَكْلًا حَقِيقَةً كَمَا تَقَدَّمَ عَنْ الرُّمَّانِ .( 21 / 433 )
