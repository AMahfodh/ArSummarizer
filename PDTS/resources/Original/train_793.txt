( وَإِنْ ) ادَّعَى شَخْصٌ أَنَّهُ مَوْلًى أَوْ قَرِيبٌ لِمَيِّتٍ لَا وَارِثَ لَهُ مَعْرُوفٌ وَ ( شَهِدَ ) شَاهِدٌ ( وَاحِدٌ بِالْوَلَاءِ ) أَوْ النَّسَبِ لِمُدَّعِيهِ ( أَوْ ) شَهِدَ شَاهِدَانِ ( اثْنَانِ أَنَّهُمَا لَمْ يَزَالَا يَسْمَعَانِ ) مِنْ الثِّقَاتِ وَغَيْرِهِمْ ( أَنَّهُ ) أَيْ الْمُدَّعِيَ ( مَوْلَاهُ ) أَيْ الْمَيِّتُ لِأَنَّهُ كَانَ أَعْتَقَهُ أَوْ انْجَرَّ لَهُ وَلَاؤُهُ بِوِلَادَةٍ أَوْ إعْتَاقٍ ( أَوْ ) أَنَّهُ ( ابْنُ عَمِّهِ ) أَيْ الْمَيِّتِ مَثَلًا ( لَمْ ) الْأَوْلَى فَلَا ( يَثْبُتْ ) وَلَاؤُهُ وَلَا نَسَبُهُ بِشَهَادَةِ الْوَاحِدِ بَتًّا ، وَلَا بِشَهَادَةِ الِاثْنَيْنِ سَمَاعًا ( لَكِنَّهُ ) أَيْ الْمُدَّعِيَ الْوَلَاءَ أَوْ النَّسَبَ ( يَحْلِفُ ) يَمِينًا أَنَّهُ مَوْلَاهُ أَوْ ابْنُ عَمِّهِ مَثَلًا ( وَيَأْخُذُ الْمَالَ ) الَّذِي تَرَكَهُ الْمَيِّتُ ( بَعْدَ الِاسْتِينَاءِ ) أَيْ التَّأْخِيرِ بِاجْتِهَادِ الْحَاكِمِ عَسَى أَنْ يَأْتِيَ غَيْرُهُ بِأَثْبَتَ مِنْهُ ، وَتَقَدَّمَ الْكَلَامُ عَلَى هَذِهِ الْمَسْأَلَةِ آخِرَ بَابِ الْعِتْقِ مُسْتَوْفًى .( 20 / 459 )
