( قَوْلُهُ : وَأُلْحِقَ بِهِ ) أَيْ بِقَتْلِ السَّيِّدِ أَمَتَهُ الْمُزَوَّجَةَ ( قَوْلُهُ : كَذَلِكَ ) خَبَرُ وَتَفْوِيتُهُ إلَخْ وَالْمُشَارُ إلَيْهِ التَّفْوِيتُ بِالْقَتْلِ ( قَوْلُهُ : كَإِرْضَاعِ السَّيِّدَةِ إلَخْ ) مِثَالُ تَفْوِيتِ السَّيِّدِ بِغَيْرِ الْقَتْلِ ( قَوْلُهُ : مُطْلَقًا ) أَيْ خَافَ الْعَنَتَ أَوْ لَا ا ه سم ( قَوْلُهُ : وَكَقَتْلِ سَيِّدٍ إلَخْ ) عَطْفٌ عَلَى كَإِرْضَاعِ السَّيِّدَةِ إلَخْ ( قَوْلُهُ : وَكَقَتْلِ سَيِّدٍ إلَخْ ) وَفِي الْأَنْوَارِ لَوْ قَتَلَ السَّيِّدُ زَوْجَ الْأَمَةِ أَوْ قَتَلَتْهُ الْأَمَةُ سَقَطَ مَهْرُهَا وَلَوْ قَتَلَتْ الْحُرَّةُ زَوْجَهَا قَبْلَ الدُّخُولِ - فَفِي بَعْضِ شُرُوحِ الْمُخْتَصَرِ أَنَّهُ لَا مَهْرَ لَهَا وَاعْتَمَدَهُ الشِّهَابُ الرَّمْلِيُّ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : أَيْ وَقَتْلِ الْأَمَةِ ) عَطْفٌ عَلَى قَتْلِ سَيِّدٍ إلَخْ ( قَوْلُهُ : كَمَا هُوَ إلَخْ ) أَيْ قَوْلُهُ : أَيْ أَوْ قَتْلِ الْأَمَةِ إلَخْ .( 31 / 124 )
