( وَ ) كُرِهَ ( تَغْسِيلُ جُنُبٍ ) الْمَيِّتَ فَهِيَ مِنْ إضَافَةِ الْمَصْدَرِ إلَى فَاعِلِهِ لِكَرَاهَتِهِ الْمَلَائِكَةُ وَشَبَّهَ فِي الْكَرَاهَةِ فَقَالَ ( كَ ) تَغْسِيلِ ( سِقْطٍ ) نَزَلَ مَيِّتًا أَوْ حَيًّا حَيَاةً ضَعِيفَةً فَيُكْرَهُ وَلَوْ أُسْقِطَ بَعْدَ تِسْعَةِ أَشْهُرٍ وَيُنْدَبُ غَسْلُ دَمِهِ وَيَجِبُ لَفُّهُ بِخِرْقَةٍ وَمُوَارَاتُهُ وَيُنْدَبُ كَوْنُهَا بِالْمَقْبَرَةِ .( 3 / 186 )
