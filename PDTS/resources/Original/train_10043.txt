وَوَارِثِ الْآخَرِ فِي أَمْتِعَةِ دَارٍ وَإِنْ صَلَحَتْ لِأَحَدِهِمَا فَقَطْ وَلَا بَيِّنَةَ وَلَا اخْتِصَاصَ بِيَدٍ فَلِكُلٍّ تَحْلِيفُ الْآخَرِ فَإِنْ حَلَفَا جُعِلَتْ بَيْنَهُمَا وَإِنْ حَلَفَ أَحَدُهُمَا قَضَى لَهُ شَيْخُنَا الرَّمْلِيُّ قَوْلُهُ : ( رَجُلٌ وَامْرَأَتَانِ ) أَوْ أَرْبَعُ نِسْوَةٍ فِيمَا يُقْبَلْنَ فِيهِ قَوْلُهُ : ( شَاهِدٌ وَيَمِينٌ ) أَيْ وَلَيْسَ مَعَهُ يَدٌ وَالْأَرْجَحُ قَوْلُهُ : ( رَجَحَ الشَّاهِدَانِ ) وَكَذَا الرَّجُلُ وَالْمَرْأَتَانِ وَكَذَا الْأَرْبَعُ نِسْوَةٍ فِيمَا يُقْبَلْنَ فِيهِ قَوْلُهُ : بِمِلْكٍ ) أَيْ مِنْ غَيْرِ إسْنَادٍ لِسَبَبِ قَوْلِهِ : ( وَالْعَيْنُ فِي يَدِ غَيْرِهِمَا ) أَخَذَهُ الشَّارِحُ مِنْ كَلَامِ الْمُصَنِّفِ سَابِقًا وَلَاحِقًا وَمِثْلُهُ وَلَوْ كَانَتْ بِيَدِهِمَا أَوْ لَا بِيَدِ أَحَدٍ .
