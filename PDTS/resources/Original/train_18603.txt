( وَلَوْ نَوَى ) الْمُسَافِرُ الْمُسْتَقِلُّ وَلَوْ مُحَارِبًا ( إقَامَةَ أَرْبَعَةِ أَيَّامٍ ) تَامَّةٍ بِلَيَالِيِهَا أَوْ نَوَى الْإِقَامَةَ وَأَطْلَقَ ( بِمَوْضِعٍ ) عَيَّنَهُ صَالِحٌ لِلْإِقَامَةِ ، وَكَذَا غَيْرُ صَالِحٍ كَمَفَازَةٍ عَلَى الْأَصَحِّ ( انْقَطَعَ سَفَرُهُ بِوُصُولِهِ ) أَيْ بِوُصُولِ ذَلِكَ الْمَوْضِعِ ، سَوَاءٌ أَكَانَ مَقْصُودُهُ أَمْ فِي طَرِيقِهِ أَوْ نَوَى بِمَوْضِعٍ وَصَلَ إلَيْهِ إقَامَةَ أَرْبَعَةِ أَيَّامٍ انْقَطَعَ سَفَرُهُ بِالنِّيَّةِ مَعَ مُكْثِهِ إنْ كَانَ مُسْتَقِلًّا .( 3 / 354 )
