ابْنُ الْقَاسِمِ وَإِنْ وَقَعَ الْعَقْدُ فُسِخَ ظَاهِرُهُ وَلَوْ لَمْ يَتَوَلَّهُ الْمُقِرُّ بِأَنَّ رُشْدَ الْوَلَدِ وَعَقْدَهُ لِنَفْسِهِ وَهُوَ أَحَدُ قَوْلَيْنِ وَعَلَى الْآخَرِ مَشَى ابْنُ الْحَاجِبِ وَظَاهِرُهُ وَلَوْ صَدَّقَتْهُ فِي اعْتِذَارِهِ قَرِينَةٌ وَيَنْبَغِي الْعَمَلُ عَلَيْهَا بِخِلَافِ قَوْلِ ) أَيْ إقْرَارِ ( أُمِّ أَحَدِهِمَا ) أَيْ الزَّوْجَيْنِ بِالرَّضَاعِ الْمُوجِبِ لِلْحُرْمَةِ بَيْنَهُمَا قَبْلَهُ ( فَالتَّنَزُّهُ ) أَيْ تَرْكُ الْعَقْدِ ( مُسْتَحَبٌّ ) وَلَوْ اسْتَمَرَّتْ( 9 / 208 )
