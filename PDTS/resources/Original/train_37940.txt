( وَمَنْ ادَّعَى دَيْنًا عَلَى مَيِّتٍ وَهُوَ جَمِيعُ تَرِكَتِهِ فَصَدَّقَهُ الْوَرَثَةُ ثُمَّ ) ادَّعَى ( آخَرُ مِثْلَ ذَلِكَ فَصَدَّقُوهُ فِي مَجْلِسٍ ) وَاحِدٍ ( فَ ) التَّرِكَةُ ( بَيْنَهُمَا ) لِأَنَّ حَالَةَ الْمَجْلِسِ كُلَّهَا كَحَالَةٍ وَاحِدَةٍ بِدَلِيلِ الْقَبْضِ فِيمَا يُعْتَبَرُ فِيهِ وَالْخِيَارُ وَلُحُوقُ الزِّيَادَةِ بِالْعَقْدِ ( وَإِلَّا ) يَكُنْ تَصْدِيقُ الْوَرَثَةِ لِلْمُدَّعِي ثَانِيًا فِي مَجْلِسٍ وَاحِدٍ ( فَ ) التَّرِكَةُ كُلُّهَا ( لِلْأَوَّلِ ) لِأَنَّهُمْ لَا يُقْبَلُ إقْرَارُهُمْ لِلثَّانِي لِأَنَّهُمْ يُقِرُّونَ بِحَقٍّ عَلَى غَيْرِهِمْ لِأَنَّهُمْ يُقِرُّونَ بِمَا يَقْتَضِي مُشَارَكَةَ الْأَوَّلِ فِي التَّرِكَةِ وَيَنْقُصُ حَقُّهُ مِنْهَا ( وَإِنْ أَقَرُّوا ) أَيْ الْوَرَثَةُ ( بِهَا ) أَيْ التَّرِكَةِ وَلَا دَيْنَ ( لِزَيْدٍ ثُمَّ ) أَقَرُّوا بِهَا ( لِعَمْرٍو فَهِيَ لِزَيْدٍ ) سَوَاءٌ أَقَرُّوا فِي مَجْلِسٍ أَوْ أَكْثَرَ لِثُبُوتِ الْمِلْكِ لِزَيْدٍ بِالْإِقْرَارِ لَهُ بِهَا فَإِقْرَارُهُمْ لِعَمْرٍو إقْرَارٌ بِمِلْكِ الْغَيْرِ ( وَيَغْرَمُونَهَا ) أَيْ يَغْرَمُ الْوَرَثَةُ التَّرِكَةَ أَيْ بَدَلَهَا ( لِعَمْرٍو ) لِأَنَّهُمْ فَوَّتُوهَا عَلَيْهِ بِالْإِقْرَارِ بِهَا لِزَيْدٍ .
