وَشَبَّهَ فِي كَوْنِ الْقَوْلِ لِلْوَكِيلِ فَقَالَ ( كَقَوْلِهِ ) أَيْ الْوَكِيلِ ( أَمَرْت ) ني ( بِبَيْعِهِ ) أَيْ الْمُوَكِّلُ عَلَى بَيْعِهِ ( بِعَشَرَةٍ ) مِنْ الدَّرَاهِمِ مَثَلًا ( وَ ) قَدْ ( أَشْبَهَتْ ) الْعَشَرَةُ أَنْ تَكُونَ ثَمَنَهُ ( وَقُلْت ) يَا مُوَكِّلُ أَمَرْتُك بِبَيْعِهِ ( بِأَكْثَرَ ) مِنْ الْعَشَرَةِ كَاثْنَيْ عَشَرَ ( وَ ) قَدْ ( فَاتَ الْمَبِيعُ ) فَوَاتًا مُصَوَّرًا ( بِزَوَالِ عَيْنِهِ ) فَالْقَوْلُ لِلْوَكِيلِ بِيَمِينِهِ ، فَإِنْ حَلَفَ بَرِئَ لِأَنَّهُ مُدَّعًى عَلَيْهِ الضَّمَانُ ، وَإِنْ نَكَلَ حَلَفَ الْآمِرُ وَغَرِمَ الْوَكِيلُ اثْنَيْنِ ، وَإِنْ نَكَلَ أَيْضًا فَلَا شَيْءَ لَهُ .
