الْمَقْبُوضُ عَنْهَا فِي حُكْمِ الْمَقْبُوضِ بِالْمُعَامَلَةِ الْفَاسِدَةِ فِيمَا يَظْهَرُ كَمَا قَالَهُ الزَّرْكَشِيُّ ( أَوْ عَكْسُهُ ) أَيْ قَصَدَ الْأَصْلَ ( بَرِئَ ) فَلَا شَيْءَ عَلَيْهِ ( أَوْ قَصَدَهُمَا ، وَزَّعَ ) عَلَيْهِمَا ( وَسَقَطَ بَاقِي الزِّيَادَةِ ، وَلَوْ لَمْ يَقْصِدْ ) شَيْئًا ( عَيَّنَهُ لِمَا شَاءَ مِنْهُمَا ، وَلَوْ سَلَّمَ ) الْمَدْيُونَ ( إلَى وَكِيلِ غَرِيمِهِ ، وَأَطْلَقَ عَيَّنَهُ لِمَا شَاءَ مِنْهُمَا ، وَلَوْ أَمَرَ ) هُوَ ( هَذَا الْوَكِيلَ بِالتَّسْلِيمِ ) أَيْ بِتَسْلِيمِهِ مَا قَبَضَهُ مِنْهُ ( إلَى أَحَدِهِمَا ، وَعَيَّنَهُ صَارَ وَكِيلًا فِي الْأَدَاءِ لَهُ لَا لِمَنْ وَكَّلَهُ فِي الْقَبْضِ ) مِنْهُ لِانْعِزَالِهِ عَنْ وَكَالَتِهِ بِالْقَبْضِ ( فَلِلْمَدْيُونِ تَعْيِينُهُ لِلْآخَرِ مَا لَمْ يُقَبِّضْهُ ) الْمُعَيَّنَ ( الْأَوَّلَ ) ، وَيَصِيرُ أَيْضًا وَكِيلًا لَهُ فَقَطْ فِيمَا لَوْ أَمَرَهُ بِالتَّسْلِيمِ إلَيْهِمَا ، وَلَهُ تَعْيِينُ أَحَدِهِمَا مَا لَمْ يُسَلِّمْ كَمَا صَرَّحَ بِذَلِكَ الْأَصْلُ ( وَإِنْ تَلِفَ ) الْمَقْبُوضُ ( قَبْلَهُ ) أَيْ قَبْلَ التَّسْلِيمِ ( تَلِفَ مَعَ وَكِيلِهِ فِي الْأَدَاءِ ) فَيَكُونُ مِنْ ضَمَانِ الْمَدِينِ ، وَالدَّيْنُ بَاقٍ عَلَيْهِ ثُمَّ إنْ لَمْ يُقَصِّرْ الْوَكِيلُ فَلَا ضَمَانَ عَلَيْهِ ، وَإِلَّا فَعَلَيْهِ الضَّمَانُ لِلْمَدِينِ ( وَإِنْ ) أَبْرَأَ الدَّائِنُ الْمَدِينَ مِنْ أَحَدِ دَيْنَيْهِ عَلَيْهِ ، وَبِأَحَدِهِمَا رَهْنٌ مَثَلًا ثُمَّ ( اخْتَلَفَا فَقَالَ ) الْمَدِينُ ( أَبْرَأْتنِي مِنْ دَيْنِ الرَّهْنِ فَقَالَ ) الدَّائِنُ ( بَلْ مِنْ الْحَالِيِّ ) عَنْهُ ( صُدِّقَ الْمُبْرِئُ بِيَمِينِهِ ) لِأَنَّهُ أَعْلَمُ بِقَصْدِهِ
