( وَلَهُ ) أَيْ لِلْحَاكِمِ ( أَنْ يَجْمَعَ لَهُ ) أَيْ لِشَاهِدِ الزُّورِ ( مِنْ عُقُوبَاتٍ إنْ لَمْ يَرْتَدِعْ إلَّا بِهِ ) قَالَهُ ابْنُ عَقِيلٍ وَغَيْرِهِ ( وَلَا يُعَزَّرُ حَتَّى يَتَحَقَّقَ أَنَّهُ شَاهِدُ زُورٍ وَ ) أَنَّهُ ( تَعَمَّدَ ذَلِكَ إمَّا بِإِقْرَارِهِ ) بِذَلِكَ ( أَوْ يَشْهَدَ بِمَا يَقْطَعُ بِكَذِبِهِ ) فِيهِ ( مِثْلُ أَنْ يَشْهَدَ عَلَى رَجُلٍ يَفْعَلُ فِي الشَّامِ وَيَعْلَمُ أَنَّ الْمَشْهُودَ عَلَيْهِ فِي ذَلِكَ الْوَقْتِ فِي الْعِرَاقِ أَوْ يَشْهَدَ بِقَتْلِ رَجُلٍ وَهُوَ حَيٌّ أَوْ ) يَشْهَدَ ( أَنَّ هَذِهِ الْبَهِيمَةَ فِي يَدِ هَذَا مُنْذُ ثَلَاثَةِ أَعْوَامٍ وَسِنُّهَا أَقَلُّ مِنْ ذَلِكَ أَوْ شَهِدَ عَلَى رَجُلٍ أَنَّهُ قُتِلَ فِي ) وَقْتِ كَذَا ( وَقَدْ مَاتَ قَبْلَ ذَلِكَ وَأَشْبَاهُ هَذَا بِمَا يُعْلَمُ بِهِ كَذِبُهُ وَيُعْلَمُ تَعَمُّدُهُ لِذَلِكَ ) وَإِلَّا لَمْ يُعَزَّرْ لِأَنَّهُ يُدْرَأُ بِالشُّبْهَةِ .
