( وَإِنْ ) ( مَاتَ ) الْمُكَلَّفُ ( وَسَطَ ) يَعْنِي أَثْنَاءَ ( الْوَقْتِ ) الِاخْتِيَارِيِّ ( بِلَا أَدَاءٍ ) لَهَا فِيهِ ( لَمْ يَعْصِ ) لِعَدَمِ تَفْرِيطِهِ ( إلَّا أَنْ يَظُنَّ الْمَوْتَ ) وَلَمْ يُؤَدِّ حَتَّى مَاتَ فَإِنَّهُ يَكُونُ عَاصِيًا وَكَذَا إذَا تَخَلَّفَ ظَنُّهُ فَلَمْ يَمُتْ لِأَنَّ الْمُوَسَّعَ صَارَ فِي حَقِّهِ مُضَيَّقًا وَهَذَا إذَا أَمْكَنَهُ الطَّهَارَةُ وَإِلَّا سَقَطَتْ كَمَا تَقَدَّمَ
