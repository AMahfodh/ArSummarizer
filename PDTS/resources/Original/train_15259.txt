( وَإِنْ أَقَرُّوا بِهَا لَهُمَا ) أَيْ أَقَرَّ الْوَرَثَةُ بِالتَّرِكَةِ لِزَيْدٍ وَعَمْرٍو ( مَعًا ) أَيْ بِلَفْظٍ وَاحِدٍ ( فَ ) التَّرِكَةُ ( بَيْنَهُمَا ) سَوِيَّةً لِعَدَمِ الْمُرَجِّحِ ( وَ ) إنْ أَقَرَّ الْوَرَثَةُ بِالتَّرِكَةِ ( لِأَحَدِهِمَا ) دُونَ الْآخَرَ ( فَهِيَ لَهُ ) لِثُبُوتِ الْمِلْكِ بِإِقْرَارِهِمْ ( وَيَحْلِفُونَ لِلْآخَرِ ) إنْ ادَّعَاهَا وَلَا بَيِّنَةَ لِإِنْكَارِهِمْ( 13 / 31 )
