( قَوْلُهُ : بِحَيْضٍ ) أَيْ بِوُجُودِ حَيْضٍ فَالْمُوجِبُ لِلْغُسْلِ وُجُودُ الْحَيْضِ لَا انْقِطَاعُهُ وَإِنَّمَا هُوَ شَرْطٌ فِي صِحَّتِهِ كَمَا قَالَهُ الشَّارِحُ ( قَوْلُهُ : تَنَفُّسُ الرَّحِمِ ) أَيْ طَرْحُ الرَّحِمِ لِلْوَلَدِ ( قَوْلُهُ : بِدَمٍ ) أَيْ مُتَلَبِّسٍ بِدَمٍ مَعَ الْوَلَدِ أَوْ قَبْلَهُ أَوْ بَعْدَهُ فَلَوْ خَرَجَ الْوَلَدُ جَافًّا فَلَا يَجِبُ عَلَيْهَا غُسْلٌ بَلْ يُنْدَبُ فَقَطْ وَعَلَى هَذَا الْقَوْلِ اقْتَصَرَ اللَّخْمِيُّ وَعَلَيْهِ فَهَلْ يُنْتَقَضُ الْوُضُوءُ بِتَنَفُّسِ الرَّحِمِ بِدُونِ دَمٍ أَمْ لَا قَوْلَانِ ( قَوْلُهُ : وَاسْتُحْسِنَ ) أَيْ عِنْدَ ابْنِ عَبْدِ السَّلَامِ وَالْمُؤَلِّفِ مِنْ رِوَايَتَيْنِ عَنْ مَالِكٍ ( قَوْلُهُ : وَبِغَيْرِهِ ) عَطْفٌ عَلَى مَحْذُوفٍ كَمَا أَشَارَ لَهُ الشَّارِحُ فِي خِيَاطَتِهِ( 1 / 453 )
