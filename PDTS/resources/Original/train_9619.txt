( وَلَوْ تَوَضَّأَ ) مُكَلَّفٌ ( وَصَلَّى الظُّهْرَ ثُمَّ أَحْدَثَ ثُمَّ تَوَضَّأَ وَصَلَّى الْعَصْرَ ، ثُمَّ ذَكَرَ أَنَّهُ تَرَكَ فَرْضًا ) أَوْ شَرْطًا ( مَنْ إحْدَى الطَّهَارَتَيْنِ ، وَلَمْ يَعْلَمْ عَيْنَهَا لَزِمَهُ إعَادَةُ الْوُضُوءِ ) لِاحْتِمَالِ أَنْ يَكُونَ الْمَتْرُوكُ مِنْ الْوُضُوءِ الثَّانِيَ ( وَ ) أَعَادَ ( الصَّلَاتَيْنِ ) لِيَخْرُجَ مِنْ الْعُهْدَةِ بِيَقِينٍ .( 2 / 229 )
