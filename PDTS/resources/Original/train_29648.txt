( وَ ) يَصِحُّ الظِّهَارُ ( مُطْلَقًا ) كَأَنْتِ عَلَيَّ كَظَهْرِ أُمِّي ( وَ ) يَصِحُّ ( مُؤَقَّتًا كَأَنْتِ عَلَيَّ كَظَهْرِ أُمِّي شَهْرَ رَمَضَانَ إنْ وَطِئَ فِيهِ ) أَيْ رَمَضَانَ ( كَفَّرَ وَإِلَّا ) يَطَأْ فِيهِ ( زَالَ ) حُكْمُ الظِّهَارِ بِمُضِيِّهِ لِحَدِيثِ صَخْرِ بْنِ سَلَمَةَ وَفِيهِ ( { ظَاهَرْتُ مِنْ امْرَأَتِي حَتَّى يَنْسَلِخَ شَهْرُ رَمَضَانَ وَأَخْبَرَ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَنَّهُ أَصَابَهَا فِيهِ فَأَمَرَهُ بِالْكَفَّارَةِ وَلَمْ يُنْكِرْ تَقَيُّدَهُ } ) بِخِلَافِ الطَّلَاقِ فَإِنَّهُ يُزِيلُ الْمِلْكَ وَهَذَا يُوقِعُ تَحْرِيمًا يَرْفَعُهُ التَّكْفِيرُ أَشْبَهَ الْإِيلَاءَ .( 10 / 40 )
