مِنْ أَوَّلِ الْعَشْرِ أَوْ بِالتَّحَرِّي عَلَى الْوَجْهَيْنِ وَهِيَ حَيْضٌ مَشْكُوكٌ فِيهِ ( وَحُكْمُ الْحَيْضِ الْمَشْكُوكِ فِيهِ حُكْمُ الْمُتَيَقَّنِ فِي تَرْكِ الْعِبَادَاتِ ) وَتَحْرِيمِ الْوَطْءِ وَوُجُوبِ الْغُسْلِ ( كَمَا تَقَدَّمَ وَإِنْ شَاءَتْ أَسْقَطَتْ الزَّائِدَ مِنْ أَيَّامِهَا ) عَنْ نِصْفِ الْوَقْتِ ( مِنْ آخِرِ الْمُدَّةِ ) وَأَسْقَطَتْ ( مِثْلَهُ مِنْ أَوَّلِهَا فَمَا بَقِيَ ) أَيْ : صَارَ بِمَعْنَى : اجْتَمَعَ ، كَمَا فِي بَعْضِ النُّسَخِ ( فَهُوَ حَيْضٌ بِيَقِينٍ وَالشَّكُّ فِيمَا بَقِيَ مِنْ الْوَقْتِ الْمُعَيَّنِ ) كَمَا تَقَدَّمَ تَمْثِيلُهُ .( 2 / 79 )
