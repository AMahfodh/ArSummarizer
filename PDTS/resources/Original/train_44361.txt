( وَإِنْ فَاتَتْهُ الصَّلَاةُ مَعَ الْإِمَامِ بِهَا ) أَيْ : بِمُزْدَلِفَةَ ( أَوْ بِعَرَفَةَ جَمَعَ وَحْدَهُ ) لِفِعْلِ ابْنِ عُمَرَ ( ثُمَّ يَبِيتُ بِهَا حَتَّى يُصْبِحَ وَيُصَلِّيَ الْفَجْرَ ) لِقَوْلِ جَابِرٍ ثُمَّ { اضْطَجَعَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ حَتَّى طَلَعَ الْفَجْرُ فَصَلَّى الْفَجْرَ حِينَ تَبَيَّنَ لَهُ الصُّبْحُ بِأَذَانٍ وَإِقَامَةٍ } .( 7 / 247 )
