( أَوْ ) أَيْ وَإِلَّا أَنْ ( لَا يَقْبَلَ الْوَلَدُ ) الْمَحْضُونُ ( غَيْرَ أُمِّهِ ) وَنَحْوِهَا مِمَّنْ لَهَا الْحَضَانَةُ فَلَا يُسْقِطُهَا دُخُولُ زَوْجٍ بِهَا ( أَوْ ) أَيْ وَإِلَّا إنْ ( لَمْ تُرْضِعْهُ ) أَيْ الْمَحْضُونَ ( الْمُرْضِعَةُ عِنْدَ ) بَدَلِ ( أُمِّهِ ) الَّذِي انْتَقَلَتْ لَهُ الْحَضَانَةُ بِدُخُولِ زَوْجٍ بِأُمِّهِ فَلَا تَسْقُطُ حَضَانَةُ أُمِّهِ( 9 / 308 )
