( وَلَوْ ) ( أَخْبَرَهُ ) الشَّفِيعُ ( بِالْبَيْعِ بِأَلْفٍ ) أَوْ جِنْسٍ أَوْ نَوْعٍ أَوْ وَصْفٍ أَوْ أَنَّ الْمَبِيعَ قَدْرُهُ كَذَا ، أَوْ أَنَّ الْبَيْعَ مِنْ فُلَانٍ ، أَوْ أَنَّ الْبَائِعَ اثْنَانِ أَوْ وَاحِدٌ ( فَتَرَكَ ) الشُّفْعَةَ ( فَبَانَ ) بِأَقَلَّ كَأَنْ بَانَ ( بِخَمْسِمِائَةٍ ) أَوْ بِغَيْرِ الْجِنْسِ أَوْ النَّوْعِ أَوْ الْوَصْفِ أَوْ الْقَدْرِ الَّذِي أَخْبَرَ بِهِ ، أَوْ أَنَّ الْمَبِيعَ مِنْ غَيْرِ فُلَانٍ ، أَوْ أَنَّ الْبَائِعَ أَكْثَرُ أَوْ أَقَلُّ مِمَّا أَخْبَرَ بِهِ ( بَقِيَ حَقُّهُ ) لِأَنَّهُ إنَّمَا تَرَكَهُ لِغَرَضٍ بَانَ خِلَافُهُ وَلَمْ يَتْرُكْهُ رَغْبَةً عَنْهُ ( أَوْ بَانَ بِأَكْثَرَ ) مِنْ أَلْفٍ ( بَطَلَ ) حَقُّهُ لِأَنَّهُ إذَا لَمْ يَرْغَبْ فِيهِ بِالْأَقَلِّ فَبِالْأَكْثَرِ أَوْلَى ، وَكَذَا لَوْ أُخْبِرَ بِمُؤَجَّلٍ فَعَفَا عَنْهُ فَبَانَ حَالًّا لِأَنَّ عَفْوَهُ يَدُلُّ عَلَى عَدَمِ رَغْبَتِهِ لِمَا مَرَّ أَنَّ لَهُ التَّأْخِيرَ إلَى الْحُلُولِ .
