( وَإِنْ ) ( وَهَبَتْ ) ضَرَّةٌ ( نَوْبَتَهَا مِنْ ضَرَّةٍ ) كَانَ ( لَهُ ) الْمَنْعُ أَيْ مَنْعُهَا مِنْ ذَلِكَ إذْ قَدْ يَكُونُ لَهُ غَرَضٌ فِي الْوَاهِبَةِ ( لَا لَهَا ) أَيْ لَيْسَ الْمَنْعُ لِلْمَوْهُوبَةِ أَيْ رَدُّ الْهِبَةِ إذَا رَضِيَ الزَّوْجُ ( وَتَخْتَصُّ ) الْمَوْهُوبَةُ بِمَا وُهِبَ لَهَا حَيْثُ رَضِيَ الزَّوْجُ وَلَيْسَ لَهُ جَعْلُهَا لِغَيْرِهَا ( بِخِلَافِ ) هِبَتِهَا نَوْبَتَهَا ( مِنْهُ ) أَيْ مِنْ الزَّوْجِ أَيْ لَهُ فَلَا يَخْتَصُّ بِهَا بِحَيْثُ يَجْعَلُهَا لِمَنْ شَاءَ بَلْ تُقَدَّرُ الْوَاهِبَةُ كَالْعَدَمِ فَإِذَا كُنَّ أَرْبَعًا فَالْقَسْمُ عَلَى ثَلَاثٍ فَإِذَا كَانَتْ هِيَ التَّالِيَةَ لِمَنْ بَاتَ عِنْدَهَا بَاتَ عِنْدَ مَنْ يَلِيهَا وَهَكَذَا ( وَلَهَا ) أَيْ لِلْوَاهِبَةِ ( الرُّجُوعُ ) فِيمَا وَهَبَتْهُ لِزَوْجِهَا أَوْ ضَرَّتِهَا لِمَا يُدْرِكُهَا مِنْ الْغَيْرَةِ فَلَا قُدْرَةَ لَهَا عَلَى الْوَفَاءِ ( وَإِنْ ) ( سَافَرَ ) الزَّوْجُ أَيْ أَرَادَ السَّفَرَ ( اخْتَارَ ) مَنْ شَاءَ مِنْهُنَّ لِلسَّفَرِ مَعَهُ ( إلَّا فِي ) سَفَرِ ( الْحَجِّ وَالْغَزْوِ فَيُقْرِعُ ) لِأَنَّ الْمُشَاحَّةَ تَعْظُمُ فِي سَفَرِ الْقُرُبَاتِ ( وَتُؤُوِّلَتْ بِالِاخْتِيَارِ مُطْلَقًا ) وَلَوْ فِي حَجٍّ وَغَزْوٍ وَهُوَ اخْتِيَارُ ابْنِ الْقَاسِمِ .
