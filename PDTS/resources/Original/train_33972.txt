أَتَتْ ( بِآخَرَ وَبَيْنَهُمَا سِتَّةُ أَشْهُرٍ فَصَاعِدًا لَحِقَهُ إنْ أَقَرَّ بِوَطْءٍ جَدِيدٍ ) بِخِلَافِ مَا إذَا لَمْ يُقِرَّ ؛ لِأَنَّ هَذَا الْفِرَاشَ يَبْطُلُ بِالِاسْتِبْرَاءِ فَبِالْوِلَادَةِ أَوْلَى ( أَوْ ) أَتَتْ بِالْآخَرِ ( لِأَقَلَّ مِنْ ) سِتَّةِ أَشْهُرٍ ( لَحِقَهُ مُطْلَقًا ) أَيْ سَوَاءٌ أَقَرَّ بِوَطْءٍ جَدِيدٍ أَمْ لَا ؛ لِأَنَّ الْوَلَدَيْنِ حِينَئِذٍ حَمْلٌ وَاحِدٌ ( وَلَوْ أَقَرَّ ) السَّيِّدُ ( بِوَطْءِ الْأَمَةِ فِيمَا دُونَ الْفَرْجِ أَوْ فِي الدُّبُرِ ) ، وَأَتَتْ بِوَلَدٍ ( لَمْ يَلْحَقْهُ ) ؛ لِأَنَّ سَبْقَ الْمَاءِ إلَى الْفَرْجِ بِالْوَطْءِ فِيمَا عَدَاهُ بَعِيدٌ ، وَكَمَا فِي التَّحْصِينِ وَالتَّحْلِيلِ وَنَحْوِهِمَا ( وَلَوْ قَالَ كُنْت ) حَالَةَ الْوَطْءِ ( أَعْزِلُ ) عَنْهَا ( لَحِقَهُ ) ؛ لِأَنَّ الْمَاءَ قَدْ يَسْبِقُهُ إلَى الرَّحِمِ ، وَهُوَ لَا يُحِسُّ بِهِ ؛ وَلِأَنَّ أَحْكَامَ الْوَطْءِ لَا يُشْتَرَطُ فِيهَا الْإِنْزَالُ .
