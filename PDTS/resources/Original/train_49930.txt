( وَإِنْ كَذَّبَهُ ) السَّيِّدُ فِي ذِكْرِهِ أَنَّهُ كَانَ مُوسِرًا أَوْ لَمْ يَخْشَ الْعَنَتَ ( فَلَهُ ) أَيْ السَّيِّدِ ( نِصْفُهُ ) أَيْ الْمَهْرِ لِأَنَّ إقْرَارَهُ غَيْرُ مَقْبُولٍ عَلَى السَّيِّدِ فِي إسْقَاطِهِ ( وَإِنْ كَانَ ) إقْرَارُهُ بِذَلِكَ ( بَعْدَ الدُّخُولِ فَعَلَيْهِ الْمُسَمَّى جَمِيعُهُ ) بِمَا اسْتَحَلَّ مِنْ فَرْجِهَا فَإِنْ كَانَ مَهْرُ الْمِثْلِ أَكْثَرَ مِنْ الْمُسَمَّى لَزِمَهُ لِإِقْرَارِهِ بِهِ وَإِنْ كَانَ الْمُسَمَّى أَكْثَرَ وَجَبَ لِلسَّيِّدِ ( وَإِذَا تَزَوَّجَ الْأَمَةَ وَفِيهِ الشَّرْطَانِ ) بِأَنْ كَانَ عَادِمَ الطَّوْلِ خَائِفَ الْعَنَتِ ( ثُمَّ أَيْسَرَ أَوْ نَكَحَ حُرَّةً أَوْ زَالَ خَوْفُ الْعَنَتِ أَوْ نَحْوِهِ ) .
