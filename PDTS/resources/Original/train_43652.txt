الْقِسْمُ ( السَّادِسُ : خِيَارٌ فِي الْبَيْعِ بِتَخَيُّرِ الثَّمَنِ ) إذَا أَخْبَرَ بَائِعٌ بِخِلَافِ الْوَاقِعِ ( وَيَثْبُتُ ) الْخِيَارُ فِي الْبَيْعِ بِتَخْيِيرِ الثَّمَنِ عَلَى قَوْلٍ ( فِي صُوَرٍ ) أَرْبَعٍ مِنْ صُوَرِ الْبَيْعِ وَاخْتُصَّتْ بِهَذِهِ الْأَسْمَاءِ كَاخْتِصَاصِ السَّلَمِ بِاسْمِهِ ( فِي تَوْلِيَةٍ ك ) قَوْلِهِ ( وَلَّيْتُكَهُ ) أَيْ الْمَبِيعَ ( أَوْ بِعْتُكَهُ بِرَأْسِ مَالِهِ ، أَوْ ) بِعْتُكَهُ ( بِمَا اشْتَرَيْتُهُ بِهِ ، أَوْ ) بِعْتُكَهُ ( بِرَقْمِهِ ) أَيْ بِثَمَنِهِ الْمَكْتُوبِ عَلَيْهِ ( وَ ) هُمَا ( يَعْلَمَانِهِ ) أَيْ الثَّمَنَ وَالرَّقْمَ ( وَ ) فِي ( شَرِكَةٍ وَهِيَ بَيْعُ بَعْضِهِ ) أَيْ الْمَبِيعِ ( بِقِسْطِهِ ) أَيْ الْمَبِيعِ مِنْ الثَّمَنِ ( كَ ) قَوْلِهِ ( أَشْرَكْتُك فِي ثُلُثِهِ أَوْ ) أَشْرَكْتُكَ ( فِي رُبْعِهِ وَنَحْوِهِمَا ) كَثُلُثِهِ أَوْ ثُمْنِهِ ( وَأَشْرَكْتُ ) فَقَطْ ف ( يَنْصَرِفُ إلَى نِصْفِهِ ) ؛ لِأَنَّهَا تَقْتَضِي التَّسْوِيَةَ .
