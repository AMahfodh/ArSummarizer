( فَلَوْ جَاوَزَتْ دِيَةُ ) مَنْ عَتَقَ بَعْدَ أَنْ جُرِحَ ثُمَّ مَاتَ ( أَرْشَ جِنَايَةٍ ) أَيْ قِيمَتِهِ رَقِيقًا ( فَالزَّائِدُ ) عَلَى قِيمَتِهِ ( لِوَرَثَتِهِ ) أَيْ الْعَبْدِ لِحُصُولِهِ بِحُرِّيَّتِهِ وَلَا حَقَّ لِلسَّيِّدِ فِيمَا حَصَلَ بِهَا إلَّا أَنَّ السَّيِّدَ يَرِثُهُ بِالْوَلَاءِ إنْ لَمْ يَكُنْ مُسْتَغْرِقٌ مِنْ نَسَبٍ وَنِكَاحٍ ( وَلَوْ وَجَبَ بِهَذِهِ الْجِنَايَةِ قَوَدٌ ) بِأَنْ كَانَتْ عَمْدًا مِنْ مُكَافِئٍ لَهُ ( فَطَلَبُهُ ) أَيْ الْقَوَدُ (( 10 / 321 )
