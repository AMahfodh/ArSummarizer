( وَ ) سُنَّ لَهُ ( الدُّعَاءُ لِلْمُسْلِمِينَ ) لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { كَانَ إذَا خَطَبَ يَوْمَ الْجُمُعَةِ دَعَا وَأَشَارَ بِإِصْبَعِهِ وَأَمَّنَ النَّاسُ } رَوَاهُ حَرْبٌ فِي مَسَائِلِهِ ( وَيُبَاحُ ) دُعَاؤُهُ ( لِمُعَيَّنٍ ) لِمَا رُوِيَ أَنَّ أَبَا مُوسَى كَانَ يَدْعُو فِي خُطْبَتِهِ لِعُمَرَ ( وَ ) يُبَاحُ ( أَنْ يَخْطُبَ مِنْ صَحِيفَةٍ ) كَقِرَاءَةٍ فِي الصَّلَاةِ مِنْ مُصْحَفٍ .( 2 / 307 )
