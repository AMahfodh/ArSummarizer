لَمْ يُبَرْهِنْ أَنَّهُ أَبَقَ عِنْدَهُ ) أَيْ عِنْدَ الْبَائِعِ الْأَوَّلِ الْمُقِرِّ ؛ لِأَنَّ الْمَوْجُودَ مِنْ الْبَائِعِ الثَّانِي السُّكُوتُ عِنْدَ إقْرَارِ الْبَائِعِ الْأَوَّلِ وَإِقْرَارُهُ لَيْسَ بِحُجَّةٍ عَلَى الْمُشْتَرِي الْأَوَّلِ وَهُوَ الْبَائِعُ الثَّانِي ( مُشْتَرٍ ) لِعَبْدٍ أَوْ أَمَةٍ ( قَالَ أَعْتَقَ الْبَائِعُ ) الْعَبْدَ ( أَوْ دَبَّرَهُ أَوْ أَوْلَدَ ) الْأَمَةَ ( أَوْ هُوَ حُرُّ الْأَصْلِ وَأَنْكَرَ الْبَائِعُ وَحَلَفَ ) لِعَجْزِ الْمُدَّعِي عَنْ الْإِثْبَاتِ ( قَضَى عَلَيْهِ ) أَيْ عَلَى الْمُشْتَرِي ( بِالْعِتْقِ وَالتَّدْبِيرِ وَالِاسْتِيلَادِ ) لِإِقْرَارِهِ بِمَا ذَكَرَ ( وَرَجَعَ بِالْعَيْبِ إنْ عَلِمَ بِهِ ) لِأَنَّ الْمُبْطِلَ لِلرُّجُوعِ إزَالَتُهُ عَنْ مِلْكِهِ إلَى غَيْرِهِ بِإِنْشَائِهِ أَوْ إقْرَارِهِ وَلَمْ يُوجَدْ حَتَّى لَوْ قَالَ بَاعَهُ وَهُوَ مِلْكُ فُلَانٍ وَصَدَّقَهُ فُلَانٌ وَأَخَذَهُ لَا يَرْجِعُ بِالنُّقْصَانِ ؛ لِأَنَّهُ أَخْرَجَهُ عَنْ مِلْكِهِ فِي الظَّاهِرِ بِإِقْرَارِهِ كَأَنَّهُ وَهَبَهُ كَذَا فِي الْجَامِعِ الْكَبِيرِ .
