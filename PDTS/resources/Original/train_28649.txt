( وَإِنْ نَزَعَهَا ) أَيْ الْجَبِيرَةَ أَوْ الْعِصَابَةَ أَوْ الْمَرَارَةَ أَوْ الْقِرْطَاسَ أَوْ الْعِمَامَةَ بَعْدَ مَسْحِهَا لِدَوَاءٍ ) مَثَلًا أَوْ ( سَقَطَتْ ) بِنَفْسِهَا رَدَّهَا وَمَسَحَهَا إنْ لَمْ يَكُنْ بِصَلَاةٍ ( وَإِنْ ) كَانَ مُتَلَبِّسًا ( بِصَلَاةِ ) فَرْضٍ أَوْ نَفْلٍ ( قَطَعَ ) هَا لِبُطْلَانِهَا وَكَذَا مَأْمُومَةٌ فَلَا يَسْتَخْلِفُ وَإِنْ كَانَ مَأْمُومًا فِي جُمُعَةٍ وَهُوَ أَحَدُ الِاثْنَيْ عَشَرَ بَطَلَتْ عَلَى جَمِيعِهِمْ ( وَرَدَّهَا ) أَيْ الْجَبِيرَةَ مَثَلًا ( وَمَسَحَ ) هَا إنْ قُرْبَ أَوْ بَعْدَ نَسْيٍ فَيَجْرِي فِيهِ حُكْمُ الْمُوَالَاةِ السَّابِقُ .( 1 / 349 )
