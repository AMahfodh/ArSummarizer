( أَوْ ) حَلَفَ ( لَا يَسْتَخْدِمُ رَجُلًا ) مَثَلًا حُرًّا أَوْ عَبْدًا ( فَخَدَمَهُ ) الْمَحْلُوفُ عَلَيْهِ ( وَهُوَ ) أَيْ الْحَالِفُ ( سَاكِتٌ حَنِثَ ) لِأَنَّ إقْرَارَهُ عَلَى خِدْمَتِهِ اسْتِخْدَامٌ لَهُ وَلِهَذَا يُقَالُ فُلَانٌ اسْتَخْدَمَ عَبْدَهُ إذَا خَدَمَهُ وَلَوْ بِلَا أَمْرِهِ .( 12 / 20 )
