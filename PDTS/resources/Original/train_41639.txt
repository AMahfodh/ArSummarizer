( وَمَنْ صَحَّ عَفْوُهُ مَجَّانًا فَإِنْ أَوْجَبَ الْجُرْحُ مَالًا عَيْنًا ) كَالْجَائِفَةِ وَجِنَايَةِ الْخَطَإِ ( فَكَوَصِيَّةٍ ) يُعْتَبَرُ مِنْ الثُّلُثِ لِأَنَّهُ تَبَرُّعٌ بِمَالٍ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يُوجِبْ الْمَالُ عَيْنًا كَالْعَمْدِ الْمَحْضِ ( فَمِنْ رَأْسِ الْمَالِ ) لِأَنَّ الْمَالَ لَمْ يَتَعَيَّنْ .( 20 / 144 )
