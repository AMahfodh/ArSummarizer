( قَوْلُهُ : إنْ أَرَادَهَا ) أَيْ الثَّانِيَةَ أَيْ وَطْأَهَا ( قَوْلُهُ : أَوْ بَعْدَ وَطْئِهَا ) أَيْ الثَّانِيَةِ عَطْفٌ عَلَى قَوْلِهِ قَبْلَ وَطْءِ إلَخْ ( قَوْلُهُ : وَعُلِمَ إلَخْ ) أَيْ فَلَا يَرِدُ ذَلِكَ عَلَى الْمَتْنِ ( قَوْلُهُ : مِمَّا مَرَّ ) أَيْ عَنْ قَرِيبٍ بِقَوْلِ الْمَتْنِ وَمَنْ وَطِئَ امْرَأَةً بِمِلْكٍ حُرِّمَ عَلَيْهِ أُمَّهَاتُهَا وَبَنَاتُهَا ( قَوْلُهُ : لَوْ مَلَكَ أُمًّا وَبِنْتَهَا ) أَيْ مَعَ أَنَّهُمَا مِمَّا حُرِّمَ جَمْعُهُمَا بِنِكَاحٍ ا ه سم .( 30 / 289 )
