أَشَارَ بِقَوْلِهِ ( فَإِنْ ضَمَّنَ ) الْمَالِكُ ( الْمُشْتَرِيَ ) الْعَيْنَ وَالْمَنْفَعَةَ ( أَوْ ) ضَمَّنَ ( الْمُسْتَعِيرَ ) الْعَيْنَ وَالْمَنْفَعَةَ ( رَجَعَا ) أَيْ الْمُشْتَرِي وَالْمُسْتَعِيرُ عَلَى الْغَاصِبِ ( بِقِيمَةِ الْمَنْفَعَةِ ) إذْ هِيَ غَيْرُ مَضْمُونَةٍ عَلَيْهِمَا ( دُونَ الْعَيْنِ ) فَإِنَّهَا تَسْتَقِرُّ عَلَيْهِمَا لِدُخُولِهِمَا فِي الْعَقْدِ عَلَى ضَمَانِهَا .( 13 / 128 )
