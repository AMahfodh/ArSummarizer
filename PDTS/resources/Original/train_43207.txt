اسْمِهِ إلَخْ ) عَطْفٌ عَلَى حُرْمَةِ كِتَابَةِ الْقُرْآنِ وَاعْتَمَدَهُ النِّهَايَةُ بِلَا عَزْوٍ إلَى الْأَذْرَعِيِّ وَنَقَلَ شَيْخُنَا عَنْ شَرْحِ الْبَهْجَةِ اعْتِمَادُهُ مَعَ الْعَزْوِ إلَى الزَّرْكَشِيّ وَأَقَرَّهُ ( قَوْلُهُ لِمُجَرَّدِ التَّعْرِيفِ بِهِ إلَخْ ) أَيْ لِيُزَارَ نِهَايَةٌ ( قَوْلُهُ النَّهْيَ ) أَيْ عَنْ الْكِتَابَةِ ( قَوْلُهُ فَهُوَ ) أَيْ كَتْبُ الِاسْمِ عَلَى الْقُبُورِ ( قَوْلُهُ وَيُرَدُّ ) أَيْ قَوْلُ الْحَاكِمِ فَإِنَّ أَئِمَّةَ الْمُسْلِمِينَ إلَخْ ( قَوْلُهُ أَكْثَرُ مِنْ الْكِتَابَةِ إلَخْ ) فِيهِ نَظَرٌ ظَاهِرٌ ( قَوْلُهُ فَكَذَا هِيَ ) أَيْ فَلَا يَكُونُ اتِّفَاقُهُمْ عَلَى الْكِتَابَةِ حُجَّةً لِنَدْبِهَا ( قَوْلُهُ هُوَ إجْمَاعٌ ) أَيْ عَمَلُ كِتَابَةِ الِاسْمِ لِمُجَرَّدِ التَّعْرِيفِ بِهِ ( قَوْلُهُ حَتَّى عَنْ الْعُلَمَاءِ الَّذِينَ يَرَوْنَ مَنْعَهُ ) لَعَلَّ الْمُنَاسِبَ إمَّا لَا يَرَوْنَ إلَخْ بِزِيَادَةِ لَا أَوْ إسْقَاطِ لَفْظَةِ حَتَّى( 11 / 393 )
