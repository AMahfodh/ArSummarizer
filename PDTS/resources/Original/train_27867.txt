( قَوْلُهُ بَعْدَ الْفُرُوضِ ) الْمُرَادُ بِالْجَمْعِ هُنَا الْجِنْسُ الصَّادِقُ لِلْقَلِيلِ وَالْكَثِيرِ ( قَوْلُهُ ثَمَانِيَةَ عَشَرَ ) مَعَ قَوْلِهِ وَسِتَّةً وَثَلَاثِينَ بَدَلٌ مِنْ أَصْلَيْنِ آخَرَيْنِ أَوْ مَفْعُولٌ لِأَعِنِي الْمُقَدَّرَةِ ( قَوْلُهُ هَذَا ) أَيْ طَرِيقُ الْمُتَأَخِّرِينَ ( قَوْلُهُ وَاخْتَارَهُ إلَخْ ) وَيُؤَيِّدُهُ مُقْتَضَى الْقَوَاعِدِ الْحِسَابِيَّةِ فِيمَا إذَا( 27 / 267 )
