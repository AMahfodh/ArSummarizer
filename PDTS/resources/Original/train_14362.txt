قُلْتُ لَوْ كَانَتْ الْأَرْضُ ) الْمُؤَجَّرَةُ لِغَرْسٍ أَوْ بِنَاءٍ ( وَقْفًا ) وَانْقَضَتْ مُدَّةُ الْإِجَارَةِ ( لَمْ يُتَمَلَّكْ ) غِرَاسٌ وَلَا بِنَاءٌ لِجِهَةِ وَقْفِ الْأَرْضِ ( إلَّا بِشَرْطِ وَاقِفٍ ) بِأَنْ كَانَ شَرَطَهُ فِي وَقْفِهِ ( أَوْ ) إلَّا ( بِرِضَا مُسْتَحِقٍّ ) لِرَيْعِ وَقْفٍ إنْ لَمْ يَكُنْ شُرِطَ ؛ لِأَنَّ فِي دَفْعِ قِيمَتِهِ مِنْ رَيْعِ الْوَقْفِ تَفْوِيتًا عَلَى الْمُسْتَحِقِّ .
