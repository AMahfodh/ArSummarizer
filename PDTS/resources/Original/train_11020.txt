( قَوْلُهُ كَمَا يَأْتِي ) أَيْ فِي قَوْلِهِ وَبَعْدَهُ يَقْلَعُ ا ه سم ( قَوْلُهُ وَحُكْمُ هَذَيْنِ ) أَيْ الْبِنَاءِ وَالْغِرَاسِ ا ه نِهَايَةٌ ( قَوْلُهُ كَاَلَّذِي قَبْلَهُ ) أَيْ قَوْلِهِ وَكُلُّ انْتِفَاعٍ إلَخْ ( قَوْلُهُ مِمَّا مَرَّ ) أَيْ مِنْ قَوْلِ الْمَتْنِ ( وَلَا رَهْنُهُ ) إلَى قَوْلِهِ وَلَوْ وَطِئَ ا ه كُرْدِيٌّ أَيْ لِأَنَّ هَذَيْنِ مِنْ جُمْلَةِ مَا يُنْقِصُ الْمَرْهُونَ كَنَحْوِ التَّزْوِيجِ وَأَمَّا جَوَازُ الِانْتِفَاعِ بِنَحْوِ الرُّكُوبِ فَعُلِمَ مِنْ مَفْهُومِ الْقَوْلِ الْمَذْكُورِ ( قَوْلُهُ( 19 / 384 )
