) أَمَةً مُوصًى بِهَا ( وَلَمْ تَحْمِلْ ) مِنْ وَطْئِهِ ( أَوْ لَبِسَ ) ثَوْبًا مُوصًى بِهِ ( أَوْ سَكَنَ مُوصًى بِهِ ) مِنْ دَارٍ أَوْ بُسْتَانٍ أَوْ بَيْت شَعْرٍ وَنَحْوِهِ ، فَلَيْسَ رُجُوعًا لِأَنَّهُ لَا يُزِيلُ الْمِلْكَ وَلَا الِاسْمَ وَلَمْ يَمْنَعْ التَّسْلِيمَ كَغَسْلِ ثَوْبٍ مُوصًى بِهِ ، أَوْ كَنْسِ دَارٍ مُوصًى بِهَا ، أَوْ عَلَّمَ رَقِيقًا مُوصًى بِهِ صَنْعَةً ( أَوْ وَصَّى بِثُلُثِ مَالِهِ فَتَلِفَ ) مَالُهُ الَّذِي كَانَ يَمْلِكُهُ حِينَ الْوَصِيَّةِ بِإِتْلَافِهِ أَوْ غَيْرِهِ ثُمَّ مَلَكَ مَالًا ( أَوْ بَاعَهُ ثُمَّ مَلَكَ مَالًا ) غَيْرَهُ فَلَيْسَ رُجُوعًا ، لِأَنَّ الْوَصِيَّةَ بِجُزْءٍ مُشَاعٍ مِمَّا يَمْلِكُهُ حِينَ الْمَوْتِ فَلَا يُؤَثِّرُ ذَلِكَ فِيهَا ( أَوْ ) كَانَتْ الْوَصِيَّةُ ( بِقَفِيزٍ مِنْ صُبْرَةٍ فَخَلَطَهَا ) أَيْ الصُّبْرَةَ ( وَلَوْ بِخَيْرٍ مِنْهَا ) مِمَّا لَا تَتَمَيَّزُ مِنْهُ فَلَيْسَ رُجُوعًا ، لِأَنَّ الْقَفِيزَ كَانَ مُشَاعًا وَبَقِيَ عَلَى إشَاعَتِهِ ( وَزِيَادَةُ مُوصٍ فِي دَارٍ ) بَعْدَ وَصِيَّةٍ بِهَا ( لِلْوَرَثَةِ ) لِأَنَّهَا لَمْ تَدْخُلْ فِي الْوَصِيَّةِ لِعَدَمِ وُجُودِهَا حِينِهَا ( لَا الْمُنْهَدِمِ ) مِنْ دَارٍ إذَا أَعَادَهُ مُوصٍ ، فَلَيْسَ لِلْوَرَثَةِ بَلْ لِمُوصًى لَهُ بِهَا لِدُخُولِهِ فِي الْوَصِيَّةِ بِوُجُودِهِ حِينِهَا( 7 / 321 )
