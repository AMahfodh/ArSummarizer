( وَ ) شَرْطُهُ : ( عَدَمُ الْوَطْءِ ) لَهَا ( مُطْلَقًا ) فِي الْحَمْلِ وَالْوَلَدِ وَالرُّؤْيَةِ ، ( فَإِنْ وَطِئَ ) الْمَرْأَةَ الْمُلَاعَنَةَ ( بَعْدَ عِلْمِهِ بِحَمْلٍ ) مِنْ غَيْرِهِ ( أَوْ وَضْعٍ أَوْ رُؤْيَةٍ ) لَهَا تَزْنِي ، ( أَوْ أَخَّرَ ) لِعَانَهَا وَلَوْ يَوْمًا ( بِلَا عُذْرٍ بَعْدَ عِلْمِهِ بِالْأَوَّلَيْنِ ) أَيْ الْحَمْلِ أَوْ الْوَضْعِ ، ( امْتَنَعَ ) لِعَانُهُ لَهَا وَلَا يُمَكَّنُ مِنْهُ ، فَالْمَانِعُ فِي الرُّؤْيَةِ الْوَطْءُ فَقَطْ لَا( 6 / 14 )
