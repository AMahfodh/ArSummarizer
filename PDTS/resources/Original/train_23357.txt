( وَيَتَنَجَّسُ ظَاهِرُهَا ) أَيْ : الْبَيْضَةِ الَّتِي صَلُبَ قِشْرُهَا ( بِرُطُوبَةِ ) مَا لَاصَقَهَا مِنْ جَوْفِ الْمَيْتَةِ ( وَمَا أُبِينَ مِنْ ) حَيَوَانٍ ( حَيٍّ : كَظُفُرٍ وَقَرْنٍ وَيَدٍ ) وَأَلْيَة وَحَافِرٍ وَجِلْدٍ ( فَ ) هُوَ ( كَمَيْتَةٍ طَهَارَةً وَنَجَاسَةً ) ، لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { وَمَا يُقْطَعُ مِنْ الْبَهِيمَةِ وَهِيَ حَيَّةٌ فَهُوَ مَيْتَةٌ } وَرَوَاهُ التِّرْمِذِيُّ ، وَقَالَ : حَسَنٌ غَرِيبٌ .
