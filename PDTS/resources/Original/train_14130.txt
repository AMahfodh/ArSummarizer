( فَرْعٌ ) لَوْ ( دَعَا ) شَخْصٌ ( لِلْمُهَنَّأِ بِالْوَلَدِ فَقَالَ ) فِي جَوَابِهِ ( آمِينَ وَنَحْوَهُ مِمَّا يَتَضَمَّنُ الْإِقْرَارَ ) بِهِ كَنَعَمْ أَوْ اسْتَجَابَ اللَّهُ دُعَاءَك ( لَمْ يَنْفِ ) أَيْ لَيْسَ لَهُ نَفْيُهُ لِرِضَاهُ بِهِ نَعَمْ إنْ عُرِفَ لَهُ وَلَدٌ آخَرُ وَادَّعَى حَمْلَ التَّهْنِئَةِ وَالتَّأْمِينِ أَوْ نَحْوِهِ عَلَيْهِ فَلَهُ نَفْيُهُ إلَّا إنْ كَانَ وَأَشَارَ إلَيْهِ فَقَالَ مَتَّعَك اللَّهُ بِهَذَا الْوَلَدِ فَقَالَ آمِينَ أَوْ نَحْوَهُ فَلَيْسَ لَهُ نَفْيُهُ لِتَضَمُّنِ ذَلِكَ الْإِقْرَارَ بِهِ ( أَوْ ) أَجَابَ بِمَا ( لَمْ يَتَضَمَّنْ إقْرَارُهُ كَقَوْلِهِ جَزَاك اللَّهُ خَيْرًا وَرَزَقَك مِثْلَهُ ) وَأَسْمَعَك خَيْرًا ( لَمْ يُؤَثِّرْ ) فِي جَوَازِ نَفْيِهِ لِاحْتِمَالِ أَنَّهُ قَصَدَ مُكَافَأَةَ الدُّعَاءِ بِالدُّعَاءِ وَصُورَةُ ذَلِكَ أَنْ يُهَنَّأَ بِهِ فِي وَقْتِ الْعُذْرِ أَوْ يُهَنِّئَهُ مَنْ لَا يَسْقُطُ حَقُّهُ بِإِخْبَارِهِ قَالَهُ ابْنُ الرِّفْعَةِ وَغَيْرُهُ
