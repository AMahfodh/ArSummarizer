( وَإِنْ اشْتَرَكَ جَمَاعَةٌ فِي ) سَرِقَةٍ ( نِصَابٍ قُطِعُوا ) كُلُّهُمْ ؛ لِوُجُودِ سَبَبِ الْقَطْعِ مِنْهُمْ كَالْقَتْلِ ( حَتَّى مَنْ لَمْ يُخْرِجْ ) مِنْهُمْ ( نِصَابًا ) كَامِلًا نَصًّا ؛ لِاشْتِرَاكِهِمْ فِي هَتْكِ الْحِرْزِ وَإِخْرَاجِ النِّصَابِ كَمَا لَوْ كَانَ ثَقِيلًا فَحَمَلُوهُ ( وَلَوْ لَمْ يُقْطَعْ بَعْضُهُمْ لِنَحْوِ شُبْهَةٍ ) كَكَوْنِهِ شَرِيكًا لِأَبِي رَبِّ الْمَالِ أَوْ عَبْدًا لَهُ أَوْ غَيْرَ مُكَلَّفٍ ( قُطِعَ الْبَاقِي ) إنْ أَخَذَ نِصَابًا ، وَقِيلَ أَوْ أَقَلَّ .
