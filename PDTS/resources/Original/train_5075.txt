( وَ ) مَنْ حَلَفَ ( لَا يَطَأُ امْرَأَتَهُ أَوْ أَمَتَهُ حَنِثَ بِجِمَاعِهَا ) أَيْ : الْمَحْلُوفِ عَلَيْهَا لِانْصِرَافِ اللَّفْظِ إلَيْهِ عُرْفًا وَكَذَلِكَ لَوْ حَلَفَ عَلَى تَرْكِ وَطْءِ زَوْجَتِهِ كَانَ مُولِيًا ( وَ ) مَنْ حَلَفَ ( لَا يَتَسَرَّى حَنِثَ بِوَطْءِ أَمَتِهِ ) مُطْلَقًا لِأَنَّ التَّسَرِّيَ مَأْخُوذٌ مِنْ السِّرِّ وَهُوَ الْوَطْءُ قَالَ تَعَالَى { وَلَكِنْ لَا تُوَاعِدُوهُنَّ سِرًّا } وَقَالَ الشَّاعِرُ : أَلَا زَعَمَتْ بَسْبَاسَةُ الْقَوْمِ أَنَّنِي كَبِرْتُ وَأَنْ لَا يُحْسِنَ السِّرَّ أَمْثَالِي وَلَا يُعْتَبَرُ الْإِنْزَالُ كَسَائِرِ أَحْكَامِ الْوَطْءِ .
