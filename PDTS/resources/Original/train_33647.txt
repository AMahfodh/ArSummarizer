( وَ ) لَا قَطْعَ فِي سَرِقَةِ ( أُضْحِيَّةٍ بَعْدَ ذَبْحِهَا ) ؛ لِأَنَّهَا وَجَبَتْ بِذَبْحِهَا وَخَرَجَتْ لِلَّهِ لَا قَبْلَهُ فَيُقْطَعُ وَلَوْ نَذَرَهَا ؛ لِأَنَّهَا لَا تَتَعَيَّنُ بِالنَّذْرِ وَالْفِدْيَةُ كَالْأُضْحِيَّةِ فِي الْوَجْهَيْنِ ( بِخِلَافِ ) سَرِقَةِ ( لَحْمِهَا ) أَوْ جِلْدِهَا ( مِنْ فَقِيرٍ ) تَصَدَّقَ بِهِ عَلَيْهِ أَوْ مُهْدَى لَهُ فَيُقْطَعُ لِجَوَازِ بَيْعِهِ ( تَامِّ الْمِلْكِ ) لِلْمَسْرُوقِ مِنْهُ ( لَا شُبْهَةَ لَهُ ) أَيْ لِلسَّارِقِ ( فِيهِ ) قَوِيَّةٌ فَيُقْطَعُ .
