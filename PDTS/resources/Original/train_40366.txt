وَاسْتَثْنَى مِنْ ذَلِكَ اسْتِثْنَاءً مُنْقَطِعًا - لِأَنَّ غَالِبَهُ لَا يَدْخُلُ فِي أَحْكَامِ الصَّبِيِّ - قَوْلُهُ : ( إلَّا طَلَاقَهُ ) فَإِنَّهُ يَلْزَمُهُ بِخِلَافِ الصَّبِيِّ فَلَا يَلْزَمُهُ وَلِلْوَلِيِّ رَدُّهُ وَلَهُ هُوَ إنْ رَشَدَ كَمَا تَقَدَّمَ ( وَ ) إلَّا ( اسْتِلْحَاقَ نَسَبٍ ) بِأَنْ يَقُولَ : هَذَا وَلَدِي ( وَنَفْيُهُ ) : أَيْ النَّسَبَ بِلِعَانٍ فَلَازِمٌ لَهُ لَيْسَ لِوَلِيِّهِ رَدُّهُ ( وَ ) إلَّا ( عَتَقَ مُسْتَوْلَدَتُهُ ) فَلَازِمٌ لَهُ وَيَتْبَعُهَا مَالُهَا وَلَوْ كَثُرَ عَلَى الْأَرْجَحِ وَهَذِهِ الثَّلَاثَةُ لَا تُتَصَوَّرُ فِي الصَّبِيِّ .
