( وَإِنْ خَلَّفَ ابْنَيْنِ مُكَلَّفَيْنِ فَأَقَرَّ أَحَدُهُمَا بِأَخٍ صَغِيرٍ ) أَوْ مَجْنُونٍ ( ثُمَّ مَاتَ الْمُنْكِرُ وَالْمُقِرُّ وَحْدَهُ وَارِثٌ ) لِلْمُنْكِرِ ( ثَبَتَ نَسَبُ الْمُقَرِّ بِهِ مِنْهُمَا ) لِانْحِصَارِ الْإِرْثِ فِيهِ ( فَلَوْ مَاتَ الْمُقِرُّ بَعْدَ ذَلِكَ عَنْ بَنِي عَمٍّ وَعَنْ الْأَخِ الْمُقَرِّ بِهِ وَرِثَهُ ) الْأَخُ الْمُقِرُّ بِهِ ( دُونَهُمْ ) أَيْ دُونَ بَنِي الْعَمِّ لِأَنَّ الْأَخَ يَحْجُبُهُمْ وَقَدْ ثَبَتَ نَسَبُهُ بِإِقْرَارِ الْمَيِّتِ .( 23 / 277 )
