( قَوْلُهُ قَرْضًا لَهُ ) الْأَوْلَى عَلَيْهِ ( قَوْلُهُ فَقِيَاسُ هَذَا ) أَيْ مَا مَرَّ ( قَوْلُهُ ذَلِكَ ) أَيْ قَوْلُ الشَّخْصِ ضَحِّ عَنِّي ( قَوْلُهُ لِأَنَّهُ ) أَيْ الْأَقَلَّ ( قَوْلُهُ وَلِإِذْنِهِ إلَخْ ) عَطْفٌ عَلَى لِاقْتِرَاضِهِ إلَخْ ( قَوْلُهُ بِالنِّيَّةِ مِنْهُ ) حَالٌ مِنْ ذَبْحِهَا وَالضَّمِيرُ لِلْمُوَكِّلِ ( قَوْلُهُ وَيَأْتِي ) أَيْ آنِفًا ( قَوْلُهُ إذَا لَمْ يُعَيَّنْ ) أَيْ الْمَيِّتُ ( قَوْلُهُ هُنَا ) أَيْ فِي ضَحِّ عَنِّي ( قَوْلُهُ لِوُصُولِ إلَخْ ) هَذَا رَاجِعٌ لِلْمَعْطُوفِ عَلَيْهِ فَقَطْ ( قَوْلُهُ إلَيْهِ ) أَيْ الْمَيِّتِ وَقَوْلُهُ وَلِأَنَّ الشَّارِعَ إلَخْ رَاجِعٌ لِلْمَعْطُوفِ فَقَطْ ( قَوْلُهُ جَعَلَ لَهُ ) أَيْ( 41 / 166 )
