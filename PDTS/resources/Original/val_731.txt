( قَوْلُهُ : إنْ ذَكَرَ التَّصَدُّقَ بِهِ ) أَيْ بِمَا يَنْحَرُهُ قَوْلُهُ : لَزِمَهُ لِمَسَاكِينِهِ ) أَيْ الْمُقِيمِينَ أَوْ الْمُسْتَوْطِنِينَ : أَيْ وَلَا يَجُوزُ لَهُ الْأَكْلُ مِنْهُ وَلَا لِمَنْ تَلْزَمُهُ نَفَقَتُهُمْ قِيَاسًا عَلَى الْكَفَّارَةِ ( قَوْلُهُ : وَقِيَاسُ مَا مَرَّ ) أَيْ فِي قَسْمِ الصَّدَقَاتِ وَفِي قَوْلِهِ هُنَا وَيَجِبُ تَعْمِيمُ الْمَحْصُورِينَ .( 28 / 49 )
