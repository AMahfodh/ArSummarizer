( وَ ) يُكْرَهُ أَيْضًا ( أَكْلُ حَبٍّ ) مِنْ نَحْوِ بُرٍّ ( دُبِّسَ بِحُمُرٍ أَهْلِيَّةٍ وَبِغَالٍ ) نَصَّ عَلَيْهِ وَقَالَ لَا يَنْبَغِي أَنْ يُدَبِّسُوهُ بِهَا وَقَالَ حَرْبٌ أَكْرَهُهُ كَرَاهَةً شَدِيدَةً ( وَيَنْبَغِي أَنْ يُغْسَلَ ) نَقَلَ أَبُو طَالِبٍ لَا يُبَاعُ وَلَا يُشْتَرَى وَلَا يُؤْكَلُ حَتَّى يُغْسَلَ .( 21 / 200 )
