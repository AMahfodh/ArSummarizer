( وَلَوْ ) ( أَسْلَمَ أَمَةً فِي كُرٍّ ) مِنْ بُرٍّ مَثَلًا أَيْ جَعَلَ أَمَةً رَأْسَ الْمَالِ فِي اشْتِرَاءِ كُرٍّ بِعَقْدِ السَّلَمِ ( وَقُبِضَتْ ) الْأَمَةُ أَيْ قَبَضَهَا الْمُسْلَمُ إلَيْهِ ( ثُمَّ تَقَايَلَا ) عَقْدَ السَّلَمِ ( فَمَاتَتْ ) أَيْ ، ثُمَّ مَاتَتْ الْأَمَةُ فِي يَدِ الْمُسْلَمِ إلَيْهِ ( قَبْلَ رَدِّهَا ) أَيْ الْأَمَةِ إلَى رَبِّ السَّلَمِ ( بَقِيَ التَّقَايُلُ ) عَلَى حَالِهِ وَلَمْ يَبْطُلْ بِهَلَاكِهَا ( وَتَجِبُ ) عَلَى الْمُسْلَمِ إلَيْهِ ( قِيمَتُهَا ) أَيْ الْأَمَةِ ( يَوْمَ قَبَضَهَا ) أَيْ الْأَمَةَ .
