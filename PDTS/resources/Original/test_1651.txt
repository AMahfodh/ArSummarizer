( وَمَنْ قَالَ لِمُحْصَنَةٍ زَنَيْت وَأَنْتِ صَغِيرَةٌ ، فَإِنْ فَسَّرَهُ بِدُونِ تِسْعِ سِنِينَ عُزِّرَ ) أَوْ قَالَهُ ؛ أَيْ : زَنَيْت وَأَنْتَ صَغِيرٌ لِمُحْصَنٍ ( ذَكَرَ ، وَفَسَّرَهُ بِدُونِ عَشْرِ ) سِنِينَ ؛ عُزِّرَ لِمَا تَقَدَّمَ ( وَإِلَّا ) يُفَسِّرْهُ بِدُونِ ذَلِكَ ( حُدَّ ) لِأَنَّهُ لَا يُشْتَرَطُ بُلُوغُ مَقْذُوفٍ .( 18 / 159 )
