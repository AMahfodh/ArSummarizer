( وَ ) إنْ نَذَرَ ( حَجًّا لَزِمَهُ ) مَعَ قُدْرَتِهِ كَبَقِيَّةِ الْعِبَادَاتِ ( فَإِنْ لَمْ يُطِقْهُ وَلَا شَيْئًا مِنْهُ ؛ حَجَّ عَنْهُ ) كَمَنْ عَجَزَ عَنْ حَجَّةِ الْإِسْلَامِ ، وَإِلَّا بِأَنْ أَطَاقَ بَعْضَ مَا نَذَرَهُ كَأَنْ نَذَرَ حَجَّاتٍ وَقَدَرَ عَلَى بَعْضِهَا ( أَتَى بِمَا يُطِيقُهُ مِنْ الْحَجَّاتِ الْمُتَعَدِّدَةِ ، وَكَفَّرَ لِلْبَاقِي ) الَّذِي لَمْ يُطِقْهُ ( وَمَعَ عَجْزِهِ عَنْ زَادٍ وَرَاحِلَةٍ حَالَ نَذْرِهِ لَا يَلْزَمُهُ شَيْءٌ ) كَحَجَّةِ الْإِسْلَامِ ( ثُمَّ إنْ وَجَدَهُمَا ) ؛ أَيْ : الزَّادَ وَالرَّاحِلَةَ ( لَزِمَهُ ) بِالنَّذْرِ السَّابِقِ فَيَنْعَقِدُ النَّذْرُ بِالْعَجْزِ كَمَا تَقَدَّمَ .( 19 / 209 )
