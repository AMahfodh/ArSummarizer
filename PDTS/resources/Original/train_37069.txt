( وَ ) إنْ قَاعَدَ خَصْمَهُ كَثَلَاثٍ وَأَرَادَ السَّفَرَ وَالتَّوْكِيلَ ( حَلَفَ فِي كَسَفَرٍ ) وَاعْتِكَافٍ وَمَرَضٍ خَفِيفٍ أَنَّهُ مَا قَصَدَهُ لِلتَّوْكِيلِ ( وَلَيْسَ لَهُ ) أَيْ الْمُوَكِّلِ ( حِينَئِذٍ ) أَيْ حِينَ قَاعَدَ وَكِيلُهُ خَصْمَهُ ثَلَاثًا ( عَزْلُهُ ) أَيْ الْوَكِيلِ عَنْ وَكَالَتِهِ فِي الْخُصُومَةِ لِذَلِكَ( 13 / 335 )
