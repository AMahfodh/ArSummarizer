( فَصْلٌ : لَا يَضْمَنُ ) الْمَالِكُ ( التَّصَرُّفَ الْمُعْتَادَ فِي مِلْكِهِ ) أَيْ مَا يَتَوَلَّدُ مِنْهُ إذْ لِكُلِّ أَحَدٍ أَنْ يَتَصَرَّفَ فِي مِلْكِهِ بِالْمَعْرُوفِ ، وَلَا يَتَقَيَّدُ بِسَلَامَةِ الْعَاقِبَةِ لِئَلَّا يُؤَدِّيَ إلَى حَرَجٍ عَظِيمٍ ، وَيَنْجَرَّ إلَى بُطْلَانِ فَائِدَةِ الْمِلْكِ بِخِلَافِ إشْرَاعِ الْجَنَاحِ إذْ لَا ضَرُورَةَ إلَيْهِ ، وَلَا يَرْغَبُ فِيهِ كُلُّ أَحَدٍ ( كَمَنْ نَصَبَ فِيهِ ) أَيْ فِي مِلْكِهِ ( سِكِّينًا ) أَوْ شَبَكَةً ( فَأَهْلَكَتْ ) شَيْئًا ( أَوْ ) وَضَعَ ( جَرَّةً عَلَى طَرَفِ سَطْحٍ ) لَهُ ( فَسَقَطَتْ بِرِيحٍ أَوْ هَدْمٍ ) لِمَحَلِّهَا ( بِبَلِّهَا ) فَأَهْلَكَتْ شَيْئًا ( أَوْ أَوْقَفَ دَابَّةً فِي مِلْكِهِ فَرَفَسَتْ رَجُلًا ) فَأَهْلَكَتْهُ ( وَلَوْ كَانَ خَارِجَ مِلْكِهِ أَوْ نَجَّسَتْ ثَوْبَهُ ) مَثَلًا ( أَوْ كَسَرَ حَطَبًا فِيهِ ) أَيْ فِي مِلْكِهِ ( فَتَطَايَرَ ) مِنْهُ شَيْءٌ فَأَهْلَكَ شَيْئًا ( أَوْ حَفَرَ ) فِيهِ ( بِئْرًا أَوْ بَالُوعَةً فَتَنَدَّى جِدَارُ جَارِهِ ) فَانْهَدَمَ ( أَوْ غَارَتْ ) بِذَلِكَ ( بِئْرُهُ ) أَيْ مَاؤُهَا ( أَوْ تَغَيَّرَتْ ) فَإِنَّهُ لَا يَضْمَنُ ؛ لِأَنَّ الْمُلَّاكَ لَا يَسْتَغْنُونَ عَنْ مِثْلِ ذَلِكَ بِخِلَافِ إشْرَاعِ الْجَنَاحِ كَمَا مَرَّ فَقَوْلُهُ ( لَمْ يَضْمَنْ ) لَا حَاجَةَ إلَيْهِ ، وَكَأَنَّهُ تَوَهَّمَ أَنَّهُ جَوَابُ شَرْطٍ تَقَدَّمَ ( فَإِنْ وَسَّعَ حَفَرَهَا أَوْ قَرَّبَهَا مِنْ الْجِدَارِ ) أَيْ جِدَارِ جَارِهِ ( خِلَافَ الْعَادَةِ أَوْ وَضَعَ فِي أَصْلِ جِدَارِ غَيْرِهِ سِرْجِينًا أَوْ لَمْ يَطْوِ بِئْرَهُ ، وَمِثْلُهَا ) أَيْ مِثْلُ أَرْضِهَا ( يَنْهَارُ ) إذَا لَمْ تُطْوَ ( ضَمِنَ ) مَا هَلَكَ بِذَلِكَ لِتَقْصِيرِهِ .
