( لَا ) يُلَاعِنُ الزَّوْجُ زَوْجَتَهُ إنْ ( كَفَرَا ) أَيْ الزَّوْجَانِ مَعًا إلَّا أَنْ يَتَرَافَعَا إلَيْهِ رَاضِيَيْنِ بِحُكْمِنَا وَقَدْ تَقَدَّمَ هَذَا عَنْ أَبِي عِمْرَانَ وَيُلَاعِنُ الزَّوْجُ ( إنْ قَذَفَهَا ) أَيْ الزَّوْجُ زَوْجَتَهُ ( ب ) رُؤْيَةِ ( زِنًا ) فِي قُبُلٍ أَوْ دُبُرٍ ادَّعَى طَوْعَهَا فِيهِ وَرَفَعَتْهُ لِأَنَّهُ مِنْ حَقِّهَا وَإِلَّا فَلَا لِعَانَ ابْنُ عَرَفَةَ مُوجِبُهُ فِيهَا بِأَحَدِ أَمْرَيْنِ مُجْمَعٍ عَلَيْهِمَا أَنْ يَدَّعِيَ رُؤْيَةَ زِنَاهَا كَالْمِرْوَدِ فِي الْمُكْحُلَةِ ثُمَّ لَمْ يَطَأْهَا بَعْدَ ذَلِكَ أَوْ( 8 / 466 )
