وَتَقَدَّمَ عَكْسُهُ فِي فَصْلِ الْخُلْعِ قِسْمَانِ وَأَنَّهُ الْأَوْجَهُ تَغْلِيبًا لِجَانِبِ التَّعْلِيقِ لِكَوْنِهَا لَا مِلْكَ لَهَا وَلَا يَدَ وَأَنَّهُ يَجِبُ مَهْرُ الْمِثْلِ ( إلَّا إنْ عَيَّنَهُ ) كَأَنْ قَالَ لَهَا إنْ أَعْطَيْتنِي هَذَا الْعَبْدَ فَأَنْت طَالِقٌ فَأَعْطَتْهُ لَهُ فَتَطْلُقُ لِتَعَيُّنِهِ بِالْإِشَارَةِ وَيَلْزَمُهَا لَهُ مَهْرُ الْمِثْلِ كَمَا لَوْ عَلَّقَ بِعَبْدٍ مَغْصُوبٍ ( وَإِنْ وَصَفَهُ كَمَا فِي السَّلَمِ فَأَعْطَتْهُ بِالصِّفَةِ طَلَقَتْ وَمَلَكَهُ ) الزَّوْجُ كَمَا فِي السَّلَمِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ تُعْطِهِ بِالصِّفَةِ ( فَلَا ) تَطْلُقُ وَلَا يَمْلِكُهُ لِعَدَمِ وُجُودِ الْمُعَلَّقِ عَلَيْهِ ( فَإِذَا خَرَجَ ) الْمَقْبُوضُ بِالصِّفَةِ ( مَعِيبًا وَرَدَّهُ ) بِالْعَيْبِ ( رَجَعَ ) عَلَيْهَا ( بِمَهْرِ الْمِثْلِ ) لَا بِعَبْدٍ بِتِلْكَ الصِّفَةِ سَلِيمًا لِأَنَّهُ بِالصِّفَةِ كَالْمُعَيَّنِ فِي الْعَقْدِ .
