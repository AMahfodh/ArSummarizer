غَرَضٌ صَحِيحٌ ، ( أَوْ نَقْصِ قِيمَتِهِ عُرْفًا ) ؛ أَيْ : فِي عُرْفِ التُّجَّارِ - وَإِنْ لَمْ تَنْقُصْ عَيْنُهُ - ( كَمَرَضٍ ) بِحَيَوَانٍ عَلَى جَمِيعِ حَالَاتِهِ ، ( وَبَخَرٍ ) فِي فَمٍ أَوْ تَحْتَ إبِطٍ أَوْ فَرْجٍ ، ( وَحَوَلٍ ) فِي عَيْنٍ ، ( وَحَوَصٍ - بِحَاءٍ مُهْمَلَةٍ - هُوَ ضِيقُ الْعَيْنِ ، وَبِالْخَاءِ الْمُعْجَمَةِ ضِيقُهَا ) ؛ أَيْ : الْعَيْنِ ( مَعَ عَوَرِهَا ، وَسَبَلٍ وَهُوَ زِيَادَةُ أَجْفَانِ ) الْعَيْنِ ، ( وَلَخَصٍ ) : هُوَ ( غِلَظُ جَفْنٍ أَسْفَلَ ) مِنْ الْعَيْنِ ، ( وَقِيلَ ) إنَّ اللَّخَصَ ( مَيْلُ أَحَدِ الْحَدَقَتَيْنِ لِلْأُخْرَى فِي نَظَرِهَا ) ، فَيَكُونُ فِي مَعْنَى الْحَوَلِ ؛ وَفِي الْقَامُوسِ : لَخِصَتْ عَيْنُهُ كَفَرِحَ ، وَرِمَ مَا حَوْلَهَا ، وَاللَّخَصُ مُحَرَّكَةٌ أَيْضًا كَوْنُ الْجَفْنِ الْأَعْلَى لَحْمًا ، ( وَمَيَلٍ : هُوَ كَوْنُ إحْدَى الْخَدَّيْنِ مَائِلًا إلَى الْآخَرِ ، وَصَدَرٍ ) هُوَ ( مَيَلُ عُنُقٍ ، وَزَوَرٍ : مَيَلُ مَنْكِبٍ ) ، وَوَكَعٍ : وَهُوَ إقْبَالُ الْإِبْهَامِ عَلَى السَّبَّابَةِ مِنْ الرِّجْلِ حَتَّى يُرَى أَصْلُهَا خَارِجًا ؛ كَالْعُقْدَةِ ، ( وَظُفْرٍ ) قَالَ فِي الْقَامُوسِ : وَالظُّفْرُ جُلَيْدَةٌ تُغَشِّي الْعَيْنَ كَالظَّفَرَةِ مُحَرَّكَةً ، وَقَدْ ظَفِرَتْ الْعَيْنُ كَفَرِحَ فَهِيَ ظَفِرَةٌ ( وَكَثْرَةِ كَذِبٍ ) ، فَإِنَّهُ أَقْبَحُ الْعُيُوبِ ، ( وَإِهْمَالِ أَدَبٍ بِمَوْضِعِهِ ) ؛ أَيْ : الْأَدَبِ ، ( وَلَعَلَّهُ ) ؛ أَيْ : إهْمَالَ الْأَدَبِ ، يَكُونُ عَيْبًا ( فِي غَيْرِ ) رَقِيقِ ( جَلْبٍ ، وَ ) فِي غَيْرِ ( صَغِيرٍ ) ، أَمَّا فِيهِمَا ؛ فَلَيْسَ بِعَيْبٍ ، ( وَضِرْسٍ ، وَكَلْفٍ ) : هُوَ تَبْقِيعُ الْوَجْهِ فِي السَّوَادِ ، ( وَطَرَشٍ ، وَقَرَعٍ ) وَإِنْ لَمْ يَكُنْ لَهُ رِيحٌ مُنْكَرَةٌ ، ( وَخُنُوثَةٍ ) ؛ أَيْ : تَكَسُّرٍ وَتَثَنٍّ ، ( وَتَخَنُّثٍ ) إذَا خَنَّثَ غَيْرَهُ ، يُقَال : خَنَّثَهُ تَخْنِيثًا :( 8 / 2 )
