( وَمَا وُجِدَ مَعَهُ ) أَيْ : اللَّقِيطِ ( مِنْ فِرَاشٍ تَحْتَهُ وَ ) مِنْ ( ثِيَابٍ ) عَلَيْهِ أَوْ فَوْقَهُ ( أَوْ مَالٍ فِي جَيْبِهِ أَوْ تَحْتَ فِرَاشِهِ أَوْ ) وُجِدَ ( مَدْفُونًا تَحْتَهُ ) دَفْنًا ( طَرِيًّا ) بِأَنْ تَجَدَّدَ حَفْرُهُ ( أَوْ ) وُجِدَ ( مَطْرُوحًا قَرِيبًا مِنْهُ أَوْ ) وُجِدَ مَعَهُ ( حَيَوَانٌ مَشْدُودٌ بِثِيَابِهِ ) أَوْ وُجِدَ اللَّقِيطُ مَشْدُودًا عَلَى دَابَّةٍ أَوْ فِي سَرِيرٍ أَوْ صُنْدُوقٍ ( فَ ) هُوَ ( لَهُ ) ؛ لِأَنَّ الطِّفْلَ يَمْلِكُ مِلْكًا صَحِيحًا فَلَهُ يَدٌ صَحِيحَةٌ كَالْبَالِغِ .
