( وَإِنْ مَاتَ ) سَيِّدُهَا ( وَلَمْ تُؤَدِّ ) أَيْ قَبْلَ أَنْ تُؤَدِّيَ جَمِيعَ مَا كُوتِبَتْ عَلَيْهِ ( أَوْ عَجَزَتْ ) عَنْ أَدَاءِ مَا كُوتِبَتْ عَلَيْهِ أَوْ أُعِيدَتْ لِلرِّقِّ ( عَتَقَتْ بِمَوْتِهِ ) لِأَنَّهَا أُمُّ وَلَدِهِ كَمَا لَوْ لَمْ تَكُنْ كُوتِبَتْ ( وَسَقَطَ مَا بَقِيَ عَلَيْهَا مِنْ كِتَابَتِهَا ) لِفَوَاتِ مَحِلِّ الْكِتَابَةِ بِالْعِتْقِ .( 16 / 306 )
