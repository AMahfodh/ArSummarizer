( أَوْ ) سَرَقَ مِنْ ( حَمَّامٍ ) بِشَدِّ الْمِيمِ نِصَابًا مِنْ آلَاتِهِ أَوْ مِنْ ثِيَابِ الدَّاخِلِينَ فَيُقْطَعُ إنْ ) كَانَ ( دَخَلَ ) ه لِلسَّرِقَةِ لَا لِيَتَحَمَّمَ ( أَوْ نَقَبَ ) حَائِطَهُ وَدَخَلَ مِنْ النَّقْبِ وَسَرَقَ ( أَوْ تَسَوَّرَ ) بِفَتَحَاتٍ مُثَقَّلًا ، أَيْ تَخَطَّى سُورَهُ وَسَرَقَ مِنْهُ ، سَوَاءٌ كَانَ لِلْحَمَّامِ حَارِسٌ أَمْ لَا ، وَسَوَاءٌ خَرَجَ بِالْمَسْرُوقِ أَمْ لَا ( أَوْ ) دَخَلَ الْحَمَّامَ مِنْ بَابِهِ لِيَتَحَمَّمَ وَهُوَ ( بِحَارِسٍ ) لِثِيَابِ الدَّاخِلِينَ ( لَمْ يَأْذَنْ ) الْحَارِسُ لِلسَّارِقِ ( فِي تَقْلِيبِ ) ثِيَابِ الدَّاخِلِينَ لِمَعْرِفَةِ ثِيَابِهِ فَسَرَقَ ثِيَابَ غَيْرِهِ فَيُقْطَعُ .
