( وَ ) إنْ حَلَفَ ( لَا فَارَقْتَنِي ) حَتَّى أَسْتَوْفِيَ حَقِّي مِنْكَ وَنَحْوَهُ ( فَفَارَقَهُ الْغَرِيمُ أَوْ الْحَالِفُ طَوْعًا حَنِثَ ) لِأَنَّ مَعْنَى الْيَمِينِ لَا حَصَلَ مِنَّا فُرْقَةٌ وَقَدْ حَصَلَتْ وَ ( لَا ) يَحْنَثُ إنْ فَارَقَهُ ( كَرْهًا ) سَوَاءٌ كَانَ الْمُكْرَهُ الْحَالِفَ أَوْ الْغَرِيمُ لِمَا سَبَقَ .( 21 / 458 )
