( وَ ) يَقُولُ ( النَّصْرَانِيُّ وَاَللَّهِ الَّذِي أَنْزَلَ الْإِنْجِيلَ عَلَى عِيسَى وَجَعَلَهُ يُحْيِي الْمَوْتَى وَيُبْرِئُ الْأَكْمَهَ وَالْأَبْرَصَ ) لِأَنَّهُ لَفْظٌ تَتَأَكَّدُ بِهِ يَمِينُهُ أَشْبَهَ الْيَهُودِيَّ ( وَ ) يَقُولُ ( الْمَجُوسِيُّ : وَاَللَّهِ الَّذِي خَلَقَنِي وَصَوَّرَنِي وَرَزَقَنِي ) لِأَنَّهُ يُعَظِّمُ خَالِقَهُ وَرَازِقَهُ أَشْبَهَ كَلِمَةَ التَّوْحِيدِ عِنْدَ الْمُسْلِمِ ( وَالْوَثَنِيُّ وَالصَّابِئُ وَمَنْ يَعْبُدُ غَيْرَ اللَّهِ يَحْلِفُ بِاَللَّهِ وَحْدَهُ ) لِأَنَّهُ لَا يَجُوزُ الْحَلِفُ بِغَيْرِ اللَّهِ لِمَا تَقَدَّمَ وَلِأَنَّهُ إنْ لَمْ يَعْتَقِدْ هَذِهِ يَمِينًا ازْدَادَ إثْمًا وَرُبَّمَا عُجِّلَتْ عُقُوبَتُهُ فَيَسْقُطُ بِذَلِكَ وَيَرْتَدُّ بِهِ .( 23 / 184 )
