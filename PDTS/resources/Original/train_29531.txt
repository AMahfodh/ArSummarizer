( وَإِنْ حَلَفَ ) الزَّوْجُ بِطَلَاقِ زَوْجَتِهِ أَوْ عِتْقِ أَمَتِهِ ( عَلَى فِعْلِ غَيْرِهِ ) أَيْ الزَّوْجِ الشَّامِلِ لِلزَّوْجَةِ وَغَيْرِهَا ( فَفِي ) حَلِفِهِ بِالطَّلَاقِ عَلَى فِعْلِ غَيْرِهِ بِصِيغَةِ ( الْبِرِّ ) بِأَنْ قَالَ إنْ فَعَلَ فُلَانٌ كَذَا فَزَوْجَتُهُ طَالِقٌ فَ ( كَ ) حَلِفِهِ عَلَى فِعْلِ ( نَفْسِهِ ) فِي جَمِيعِ مَا تَقَدَّمَ فَيُنْتَظَرُ وَلَا يُمْنَعُ مِنْ بَيْعِ أَمَتِهِ الَّتِي حَلَفَ بِعِتْقِهَا وَلَا مِنْ وَطْءِ زَوْجَتِهِ الَّتِي حَلَفَ بِطَلَاقِهَا .
