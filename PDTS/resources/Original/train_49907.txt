قَالَ فِي الشَّرْحِ لِأَنَّ الظَّاهِرَ أَنَّهَا خَطُّهُ ( وَمَنْ تَحَقَّقَ الْحَاكِمُ مِنْهُ أَنَّهُ لَا يُفَرِّقُ بَيْنَ أَنْ يَذْكُرَ الشَّهَادَةَ أَوْ يَعْتَمِدَ عَلَى مَعْرِفَةِ الْخَطِّ يَتَجَوَّزُ بِذَلِكَ لَمْ يَجُزْ ) لِلْحَاكِمِ ( قَبُولُ شَهَادَتِهِ ) كَالْمُغَفَّلِ لِاحْتِمَالِ أَنْ يَكُونَ اعْتَمَدَ عَلَى الْخَطِّ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَتَحَقَّقْ الْحَاكِمُ ذَلِكَ مِنْهُ ( حَرُمَ أَنْ يَسْأَلَهُ عَنْهُ ) أَهَلْ ذَكَرَ( 22 / 321 )
