( وَلَوْ قَطَعَهَا ) أَيْ الْمَسَافَةَ ( فِي سَاعَةٍ وَاحِدَةٍ ) ؛ لِأَنَّهُ صَدَقَ عَلَيْهِ أَنَّهُ سَافَرَ أَرْبَعَةَ بُرُدٍ ( وَمَتَى صَارَ الْأَسِيرُ بِبَلَدِهِمْ ) أَيْ الْكُفَّارِ ( أَتَمَّ ) الصَّلَاةَ ( نَصًّا ) ؛ لِأَنَّهُ صَارَ مُقِيمًا ( وَامْرَأَةٌ وَعَبْدٌ وَجُنْدِيٌّ : تَبَعٌ لِزَوْجٍ وَسَيِّدٍ وَأَمِيرٍ ) لَفٌّ وَنَشْرٌ مُرَتَّبٌ ( فِي نِيَّتِهِ ) أَيْ الزَّوْجِ أَوْ السَّيِّدِ أَوْ الْأَمِيرِ الْمَسَافَةُ وَالْإِقَامَةُ .
