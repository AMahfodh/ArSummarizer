( وَإِنْ قُذِفَ بِالْبِنَاءِ لِلْمَفْعُولِ مَيِّتٌ مُحْصَنٌ أَوْ لَا ) أَوْ أَيْ غَيْرُ مُحْصَنٍ ( وَلَوْ ) كَانَ الْمَيِّتُ الْمَقْذُوفُ ( مِنْ غَيْرِ أُمَّهَاتِ الْوَارِثِ حُدَّ قَاذِفٌ بِطَلَبِ وَارِثٍ مُحْصَنٍ خَاصَّةً ) لِمَا فِيهِ مِنْ التَّعْيِيرِ ( وَإِنْ كَانَ الْوَارِثُ غَيْرَ مُحْصَنٍ ) بِأَنْ كَانَ عَبْدًا أَوْ كَافِرًا وَنَحْوَهُ ( فَلَا حَدَّ ) كَمَا لَوْ قَذَفَهُ ابْتِدَاءً ( وَثَبَتَ حَدُّ قَذْفِ الْمَيِّتِ وَالْقَذْفُ الْمَوْرُوثُ لِجَمِيعِ الْوَرَثَةِ حَتَّى الزَّوْجَيْنِ ) لِأَنَّهُ حَقٌّ وُرِثَ عَنْ الْمَيِّتِ فَاشْتَرَكَ فِيهِ جَمِيعُ الْوَرَثَةِ كَسَائِرِ الْحُقُوقِ ( وَإِنْ عَفَا بَعْضُهُمْ ) أَيْ الْوَرَثَةُ ( حُدَّ ) الْقَاذِفُ ( لِلْبَاقِي ) مِنْ الْوَرَثَةِ حَدًّا ( كَامِلًا ) لِلُحُوقِ الْعَارِ بِكُلِّ وَاحِدٍ مِنْهُمْ عَلَى انْفِرَادِهِ .( 20 / 466 )
