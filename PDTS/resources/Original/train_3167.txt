( وَ ) مَنْ حَلَفَ ( لَا يَلْبَسُ حُلِيًّا فَلَبِسَ حِلْيَةَ ذَهَبٍ أَوْ فِضَّةٍ أَوْ ) لَبِسَ ( خَاتَمًا وَلَوْ فِي غَيْرِ الْخِنْصَرِ أَوْ دَرَاهِمَ أَوْ دَنَانِيرَ فِي مُرْسَلَةٍ وَنَحْوِهَا أَوْ ) لَبِسَ ( لُؤْلُؤًا وَجَوَاهِرَ فِي مِخْنَقَةٍ أَوْ ) لَبِسَ ذَلِكَ ( مُنْفَرِدًا أَوْ ) لَبِسَ ( مِنْطَقَةً ) وَتُسَمِّيهَا الْعَامَّة حِيَاصَةً ( مُحَلَّاةً حَنِثَ ) قَالَ تَعَالَى { يُحَلَّوْنَ فِيهَا مِنْ أَسَاوِرَ مِنْ ذَهَبٍ وَلُؤْلُؤًا } .
