( وَلَوْ أَذِنَ السَّيِّدُ فِي ضَرْبِ عَبْدِهِ ) ضَرْبًا مُحَرَّمًا ( أَوْ ) أَذِنَ ( الْوَالِدُ فِي ضَرْبِ وَلَدِهِ ) ضَرْبًا مُحَرَّمًا ( فَضَرَبَهُ الْمَأْذُونُ لَهُ ضَمِنَهُ ) إنْ تَلِفَ لِأَنَّ الْمُحَرَّمَاتِ لَا تُسْتَبَاحُ بِالْإِذْنِ وَأَمَّا الضَّرْبُ الْمُبَاحُ لِلتَّأْدِيبِ فَقَدْ تَقَدَّمَ أَوَّلَ الْفَصْلِ .( 20 / 231 )
