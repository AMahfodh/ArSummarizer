( وَ ) تَجِبُ ( فِي الْعَبْدِ الْمَرْهُونِ ، وَ ) الْعَبْدُ ( الْمُوصَى بِهِ عَلَى مَالِكِهِ وَقْتَ الْوُجُوبِ ) أَيْ عِنْدَ غُرُوبِ الشَّمْسِ مِنْ آخِرِ رَمَضَانَ ( وَكَذَا ) الْعَبْدُ ( الْمَبِيعُ فِي مُدَّةِ الْخِيَارِ ) تَجِبُ فِطْرَتُهُ عَلَى مَنْ حَكَمَ لَهُ بِالْمِلْكِ وَهُوَ الْمُشْتَرِي عَلَى الْمَذْهَبِ ( فَإِنْ لَمْ يَكُنْ لِلرَّاهِنِ شَيْءٌ غَيْرُ الْعَبْدِ ) الْمَرْهُونِ ( بِيعَ مِنْهُ بِقَدْرِ الْفِطْرَةِ ) كَأَرْشِ جِنَايَتِهِ ( إذَا فَضَلَ عِنْدَهُ ) أَيْ عِنْدَ الْمُسْلِمِ الَّذِي تَلْزَمُهُ مُؤْنَةُ نَفْسِهِ ( عَنْ قُوتِهِ وَقُوتِ عِيَالِهِ يَوْمَ الْعِيدِ وَلَيْلَتِهِ صَاعٌ ) لِأَنَّ ذَلِكَ أَهَمُّ فَيَجِبُ تَقْدِيمُهُ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { ابْدَأْ بِنَفْسِك ثُمَّ بِمَنْ تَعُولُ } فَظَاهِرُهُ أَنَّهُ لَا يَعْتَبِرُ لِوُجُوبِهَا مِلْكَ نِصَابٍ وَقَالَهُ الْأَكْثَرُ .( 5 / 247 )
