وَكَذَلِكَ لَوْ كَانَ لِرَجُلٍ عَلَى رَجُلٍ دَيْنٌ فَبَاعَهُ مِنْ ثَالِثٍ بِدَيْنٍ ( وَتَأْخِيرُ رَأْسِ الْمَالِ ) أَيْ مَالِ السَّلَمِ ( بِشَرْطٍ إلَى مَحِلِّ السَّلَمِ ) أَيْ أَجَلِهِ ( أَوْ ) إلَى ( مَا بَعُدَ مِنْ الْعُقْدَةِ ) أَيْ عَنْ عُقْدَةِ السَّلَمِ بِأَكْثَرَ مِنْ ثَلَاثَةِ أَيَّامٍ ( مِنْ ذَلِكَ ) أَيْ مِنْ الدَّيْنِ بِالدَّيْنِ ؛ لِأَنَّ فِيهِ( 6 / 81 )
