النَّارِ ) أَوْ أَخَذْت بِصَلَى الْفَرَسِ ، وَهُوَ مَا اتَّصَلَ بِخَاصِرَتِهِ إلَى فَخِذَيْهِ ( أَوْ قَالَ ) : إنْ فَعَلَ ذَلِكَ ( فَهُوَ كَافِرٌ ، وَنَوَى الْمُسْتَتِرَ ) الْمُتَغَطِّيَ أَوْ السَّاتِرَ الْمُغَطَّى ، وَمِنْهُ قِيلَ لِلزَّارِعِ كَافِرٌ ، فَلَهُ نِيَّتُهُ ؛ لِأَنَّ لَفْظَهُ يَحْتَمِلُهُ ( أَوْ ) قَالَتْ لَهُ زَوْجَتُهُ : قُلْ ( كُلُّ زَوْجَةٍ أَطَؤُهَا غَيْرَك فَطَالِقٌ ) فَقَالَ : ( وَنَوَى أَطَؤُهَا بِرِجْلِي ) فَلَهُ نِيَّتُهُ ( وَ ) إنْ قَالَ لِزَوْجَتِهِ : ( إنْ خَرَجْت بِلَا إذْنِي فَطَالِقٌ ، وَنَوَى إنْ خَرَجَتْ عُرْيَانَةً أَوْ رَاكِبَةً وَنَحْوَهُ ) كَحَامِلَةِ الشَّيْءِ أَوْ مَحْمُولَةٍ عَلَى شَيْءٍ ، لَمْ تَطْلُقْ ، لِعَدَمِ وُجُودِ الصِّفَةِ .( 16 / 227 )
