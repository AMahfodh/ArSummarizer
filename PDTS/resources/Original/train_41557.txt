( قَوْلُهُ : وَالتَّوَسُّعُ عَلَى الْعِيَالِ ) كَذَا فِي أَصْلِهِ رَحِمَهُ اللَّهُ تَعَالَى وَفِي نُسْخَةِ السَّيِّدِ عُمَرَ الْبَصْرِيِّ وَنُسَخٍ صَحِيحَةٍ أُخْرَى التَّوْسِيعُ مُصْطَفَى الْحَمَوِيِّ ( قَوْلُهُ : وَإِيثَارِ شَهْوَتِهِمْ إلَخْ ) كَقَوْلِهِ وَالتَّوَسُّعُ عُطِفَ عَلَى إكْرَامِ ضَيْفٍ وَقَوْلُهُ مِنْ غَيْرِ تَكَلُّفٍ رَاجِعٌ لِكُلٍّ مِنْ الثَّلَاثِ ( قَوْلُهُ : وَيُؤَيِّدُهُ ) أَيْ نَدْبَ الْحَفَاءِ ( قَوْلُهُ : لِنَحْوِ دُخُولِ مَكَّةَ ) أَيْ كَدُخُولِ الْمَدِينَةِ ( قَوْلُهُ : بِهَذِهِ الشُّرُوطِ ) وَهِيَ قَصْدُ التَّوَاضُعِ وَأَمْنُ الْمُؤْذِي وَأَمْنُ التَّنَجُّسِ .
