( وَإِنْ قَالَ ) لَهُ ( أَنْكِحْ فُلَانَةَ بِأَلْفٍ وَمَهْرُ مِثْلِهَا أَقَلُّ ) مِنْهُ ( بَطَلَ الْإِذْنُ ) أَيْ فَلَا يَصِحُّ النِّكَاحُ قَالَ الزَّرْكَشِيُّ تَبَعًا لِلْأَذْرَعِيِّ وَالْقِيَاسُ صِحَّتُهُ بِمَهْرِ الْمِثْلِ كَمَا لَوْ قَبِلَ لَهُ الْوَلِيُّ بِزِيَادَةٍ عَلَيْهِ ( وَإِلَّا ) بِأَنْ كَانَ مِثْلَهُ أَوْ أَكْثَرَ مِنْهُ ( فَيَصِحُّ ) الْإِذْنُ وَحِينَئِذٍ ( فَإِنْ نَكَحَ بِأَكْثَرَ ) مِنْ أَلْفٍ ( وَمَهْرُ مِثْلِهَا أَكْثَرُ ) مِنْهُ أَيْضًا ( بَطَلَ ) النِّكَاحُ ( أَوْ بِالْأَلْفِ ) أَيْ أَوْ نَكَحَ بِالْأَلْفِ ( فَيَصِحُّ بِهِ أَوْ ) نَكَحَ ( بِأَكْثَرَ ) مِنْهُ وَمَهْرُ مِثْلِهَا أَلْفٌ ( صَحَّ بِالْأَلْفِ ) وَسَقَطَتْ الزِّيَادَةُ وَهَذِهِ فُهِمَتْ مِنْ قَوْلِهِ وَمَهْرُ مِثْلِهَا أَكْثَرُ ( أَوْ ) نَكَحَ ( بِمَا دُونَهُ صَحَّ ) النِّكَاحُ ( بِهِ ) وَالتَّصْرِيحُ بِهَذِهِ فِيمَا إذَا كَانَ مَهْرُ الْمِثْلِ أَكْثَرَ مِنْ الْأَلْفِ مِنْ زِيَادَتِهِ .
