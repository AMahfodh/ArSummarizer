وَلَا ( تَصِحُّ ) أَيْضًا ( بِزِيَادَةٍ عَلَى الثَّمَنِ ) الْمَعْقُودِ بِهِ ( أَوْ ) بِ ( نَقْصٍ مِنْهُ أَوْ بِغَيْرِ جِنْسِهِ ) لِأَنَّ مُقْتَضَى الْإِقَالَةِ رَدُّ الْأَمْرِ إلَى مَا كَانَ عَلَيْهِ ( وَالْمِلْكُ بَاقٍ لِلْمُشْتَرِي ) لِأَنَّهُ شَرَطَ التَّفَاضُلَ فِيمَا يُعْتَبَرُ فِيهِ التَّمَاثُلُ فَبَطَلَ كَبَيْعِ دِرْهَمٍ بِدِرْهَمَيْنِ .( 9 / 223 )
