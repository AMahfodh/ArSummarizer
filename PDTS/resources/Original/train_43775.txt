( وَإِنْ أُعِينَ ) الْمُكَاتَبُ ( بِشَيْءٍ ) : بِأَنْ دَفَعَ لَهُ جَمَاعَةٌ أَوْ وَاحِدٌ دَرَاهِمَ أَوْ غَيْرِهَا يَسْتَعِينُ بِهَا ( فَإِنْ لَمْ يَقْصِدْ الصَّدَقَةَ عَلَيْهِ ) : بِأَنْ قَصَدَ فَكَّ الرَّقَبَةِ أَوْ لَا قَصَدَ ( رَجَعَ عَلَيْهِ ) : أَيْ عَلَى الْعَبْدِ ( بِالْفَضْلَةِ إنْ عَتَقَ ) فَيَأْخُذُ الْمُعَيَّنُ مِنْ الْعَبْدِ الزَّائِدِ ( وَ ) رَجَعَ ( عَلَى السَّيِّدِ بِمَا قَبَضَهُ ) مِنْ مَالِهِمْ ( إنْ عَجَزَ ، وَإِلَّا ) بِأَنْ قَصَدَ بِمَا دَفَعَ لَهُ الصَّدَقَةُ ( فَلَا ) رُجُوعَ عَلَى الْعَبْدِ بِالْفَضْلَةِ وَلَا عَلَى السَّيِّدِ بِمَا قَبَضَ .
