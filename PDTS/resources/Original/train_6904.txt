( و ) إنْ بَاعَ الْكَافِرُ عَبْدَهُ الْكَافِرَ بِخِيَارٍ لِمُسْلِمٍ أَوْ كَافِرٍ فَأَسْلَمَ الْعَبْدُ زَمَنَ الْخِيَارِ ، فَإِنْ حَصَلَ إسْلَامُهُ ( فِي ) زَمَنِ ( خِيَارِ مُشْتَرٍ ) بِالتَّنْوِينِ ( مُسْلِمٍ ) نَعْتُهُ ( يُمْهَلُ ) الْمُشْتَرِي الْمُسْلِمُ ذُو الْخِيَارِ ( لِانْقِضَائِهِ ) أَيْ لِانْقِضَاءِ زَمَنِ خِيَارِهِ لِسَبْقِ حَقِّهِ عَلَى حَقِّ الْعَبْدِ ، فَإِنْ رَدَّهُ لِبَائِعِهِ جُبِرَ عَلَى إخْرَاجِهِ بِمَا تَقَدَّمَ ( و ) إنْ أَسْلَمَ فِي خِيَارِ الْكَافِرِ بَائِعًا أَوْ مُشْتَرِيًا فَلَا يُمْهَلُ بَلْ ( يُسْتَعْجَلُ الْكَافِرُ ) صَاحِبُ ( الْخِيَارِ مِنْهُمَا ) بِالْإِمْضَاءِ أَوْ الرَّدِّ لِئَلَّا يَدُومَ مِلْكُ الْكَافِرِ عَلَى الْمُسْلِمِ وَشَبَّهَ فِي الِاسْتِعْجَالِ قَوْلُهُ ( كَبَيْعِهِ ) أَيْ كَمَا يَسْتَعْجِلُ السُّلْطَانُ بِبَيْعِ الْعَبْدِ ( إنْ أَسْلَمَ ) فِي غَيْبَةِ سَيِّدِهِ الْكَافِرِ ( وَبَعُدَتْ غَيْبَةُ سَيِّدِهِ ) بِأَنْ يَكُونَ عَلَى عَشَرَةِ أَيَّامٍ فَأَكْثَرَ أَوْ يَوْمَيْنِ عَلَى الْخَوْفِ ، فَإِنْ قَرُبَتْ لَمْ يَبِعْ بَلْ يُكْتَبُ لَهُ فَإِنْ أَجَابَ وَإِلَّا بِيعَ عَلَيْهِ .
