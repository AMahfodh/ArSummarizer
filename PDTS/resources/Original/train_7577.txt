( وَإِنْ سَقَطَ ) الْوَلَدُ ( مَيِّتًا ) بِغَيْرِ جِنَايَةٍ ( لَمْ يَضْمَنْهُ ) كَوَلَدِ الْمَغْصُوبَةِ ( وَعَلَيْهِ ) أَيْ عَلَى الْمُشْتَرِي ( ضَمَانُ نَقْصِ الْوِلَادَةِ ) لِحُصُولِهِ بِيَدِهِ الْعَادِيَةِ ( وَإِنْ مَلَكَهَا الْوَاطِئُ ) لَهَا فِي الْعَقْدِ الْفَاسِدِ بَعْدَ أَنْ حَمَلَتْ مِنْهُ فِيهِ ( لَمْ تَصِرْ أُمَّ وَلَدٍ ) لَهُ بِذَلِكَ الْحَمْلِ ؛ لِأَنَّهُ لَمْ يَكُنْ مَالِكًا لَهَا إذْ ذَاكَ ( وَيَأْتِي ) ذَلِكَ ( فِي أَوَاخِرِ الْخِيَارِ فِي الْبَيْعِ وَ ) يَأْتِي فِي ( الْغَصْبِ ) أَيْضًا مُفَصَّلًا .( 9 / 47 )
