( وَإِلَّا ) يُوجَدْ مَنْ يُرْضِعُهُ ( فَ ) لَا يُقَادُ مِنْهَا ( حَتَّى تَفْطِمَهُ لِحَوْلَيْنِ ) كَمَا تَقَدَّمَ وَلِأَنَّهُ إذَا أُخِّرَ الِاسْتِيفَاءُ لِحِفْظِهِ وَهُوَ حَمْلٌ فَلَأَنْ يُؤَخَّرَ لِحِفْظِهِ بَعْدَ وَضْعِهِ أَوْلَى ( وَكَذَا حَدٌّ بِرَجْمٍ ) لِمَا تَقَدَّمَ ( وَتُقَادُ ) حَامِلٌ ( فِي طُرُقٍ ) بِمُجَرَّدِ وَضْعٍ ( وَتُحَدُّ ) حَامِلٌ ( بِجَلْدٍ ) لِقَذْفٍ أَوْ شُرْبٍ أَوْ غَيْرِهِمَا ( بِمُجَرَّدِ وَضْعِ ) حَمْلٍ فِي الْمُغْنِي وَسَقْيِ اللِّبَإِ .
