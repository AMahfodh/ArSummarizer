إذَا تَقَرَّرَ هَذَا ( فَيُجْبِرُ أَبٌ لَا جَدٌّ ثَيِّبًا دُونَ ذَلِكَ ) أَيْ : دُونَ تِسْعِ سِنِينَ ؛ لِأَنَّهُ لَا إذْنَ لَهَا مُعْتَبَرٌ ، ( وَ ) يُجْبِرُ أَبٌ ( بِكْرًا أَوْ لَوْ ) كَانَتْ ( مُكَلَّفَةً ) لِحَدِيثِ ابْنِ عَبَّاسٍ مَرْفُوعًا : { الْأَيِّمُ أَحَقُّ بِنَفْسِهَا مِنْ وَلِيِّهَا ، وَالْبِكْرُ تُسْتَأْمَرُ ، وَإِذْنُهَا صُمَاتُهَا } رَوَاهُ( 14 / 442 )
