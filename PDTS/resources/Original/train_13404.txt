( وَ ) يَحْرُمُ ( حَكْرُ ) بِفَتْحِ الْحَاءِ أَيْ احْتِكَارُ ( قُوتٍ ) كَتَمْرٍ وَزَبِيبٍ وَشَعِيرٍ لِلتَّضْيِيقِ وَلِخَبَرِ مُسْلِمٍ : { لَا يَحْتَكِرُ إلَّا خَاطِئٌ } أَيْ آثِمٌ وَذَلِكَ بِأَنْ ( اشْتَرَاهُ فِي ) وَقْتِ ( الْغَلَا ) وَحَبَسَهُ ( لِبَيْعِهِ الضَّعْفَا ) بِفَتْحِ الضَّادِ وَإِسْكَانِ الْعَيْنِ جَمْعُ ضَعِيفٍ بِأَكْثَرَ مِمَّا اشْتَرَاهُ بِهِ ( إذَا السِّعْرُ غَلَا ) فَإِنْ لَمْ يَشْتَرِهِ كَغَلَّةِ ضَيْعَتِهِ أَوْ اشْتَرَاهُ وَقْتَ الرُّخْصِ لِيَبِيعَهُ وَقْتَ الْغَلَاءِ أَوْ اشْتَرَى وَقْتَ الْغَلَاءِ لِنَفْسِهِ وَعِيَالِهِ أَوْ لِيَبِيعَهُ بِمِثْلِ مَا اشْتَرَاهُ بِهِ أَوْ أَقَلَّ لَمْ يَحْرُمْ وَالْأَوْلَى أَنْ يَبِيعَ مِنْ ذَلِكَ مَا فَضَلَ عَنْ كِفَايَتِهِ وَكِفَايَةِ عِيَالِهِ أَيْ : سَنَةً كَمَا يُؤْخَذُ مِنْ كَلَامِهِمْ فِي السِّيَرِ وَفِي كَرَاهَةِ إمْسَاكِهِ وَجْهَانِ فِي الرَّوْضَةِ وَأَصْلِهَا وَتَفْسِيرُ احْتِكَارِ الْقُوتِ بِمَا ذُكِرَ مِنْ زِيَادَةِ النَّظْمِ ، وَذِكْرُهُ الضَّعْفَى جَرَى عَلَى الْغَالِبِ فَإِنَّهُ لَا فَرْقَ بَيْنَهُمْ وَبَيْنَ غَيْرِهِمْ وَخَرَجَ بِالْقُوتِ غَيْرُهُ فَلَا يَحْرُمُ احْتِكَارُهُ وَغَلَا فِي كَلَامِهِ بِالْمُهْمَلَةِ وَيَجُوزُ قِرَاءَتُهُ بِالْمُعْجَمَةِ ؛ لِأَنَّهُ فِعْلٌ وَمَا ذَكَرَ فِي التَّصْرِيحِ مَصْدَرٌ .
