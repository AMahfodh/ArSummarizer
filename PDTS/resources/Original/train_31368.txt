( قَوْلُهُ : وَلَوْ أَوْصَتْ ) أَيْ وَمَاتَتْ ( قَوْلُهُ : فَادَّعَى ) أَيْ شَخْصٌ آخَرُ ( قَوْلُهُ : إنَّهُ ابْنُ عَمِّهَا ) أَيْ لِيَرِثَ مِنْهَا ( قَوْلُهُ : وَهُنَا لَوْ صَدَّقَهُ أَحَدُهُمَا ) أَيْ الْوَصِيُّ أَوْ الزَّوْجُ ( قَوْلُهُ : لِأَنَّ النَّسَبَ ) إنَّمَا يَثْبُتُ بِقَوْلِهِ : أَيْ الْمُدَّعِي لِلنَّسَبِ لِأَنَّهُ الْوَارِثُ فِي زَعْمِهِ وَإِقْرَارِهِ عَلَى نَفْسِهِ بِالنَّسَبِ لَا أَثَرَ لَهُ( 29 / 54 )
