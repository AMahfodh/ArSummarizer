( وَلَا يُصَلَّى عَلَى قَبْرٍ إلَّا أَنْ يُدْفَنَ بِغَيْرِهَا ) الرِّسَالَةُ : وَمَنْ دُفِنَ وَلَمْ يُصَلَّ عَلَيْهِ وُورِيَ فَإِنَّهُ يُصَلَّى عَلَى قَبْرِهِ اُنْظُرْ قَبْلَ هَذَا عِنْدَ قَوْلِهِ : فَإِنْ دُفِنَ فَعَلَى الْقَبْرِ ( وَلَا غَائِبٍ ) اُنْظُرْ عِنْدَ قَوْلِهِ : وَلَا دُونَ الْجُلِّ ( وَلَا تُكَرَّرُ ) / 97 قَدْ تَقَدَّمَ / 97 .( 2 / 403 )
