وَذَكَرَ الْخَامِسَ بِقَوْلِهِ ( وَإِمَّا قَتْلٌ بِالسَّبَبِ ) أَيْ بِكَوْنِهِ سَبَبًا لِلْقَتْلِ ( كَإِتْلَافِهِ بِحَفْرِ الْبِئْرِ أَوْ وَضْعِ الْحَجَرِ فِي غَيْرِ مِلْكِهِ ) قَيْدٌ لِلْحَفْرِ وَالْوَضْعِ ( أَوْ ) وَضْعِ ( خَشَبَةٍ عَلَى قَارِعَةِ الطَّرِيقِ وَنَحْوِهِ ) مِمَّا هُوَ سَبَبٌ لِلْإِتْلَافِ ( إلَّا أَنْ يَمْشِيَ ) الْهَالِكُ ( عَلَيْهِ ) أَيْ عَلَى الْبِئْرِ وَنَحْوِهِ ( بَعْدَ عِلْمِهِ بِالْحَفْرِ وَنَحْوِهِ ) فَحِينَئِذٍ لَا يَلْزَمُ شَيْءٌ عَلَى الْحَافِرِ وَنَحْوِهِ ( وَحُكْمُهُ الدِّيَةُ عَلَى الْعَاقِلَةِ ) لِأَنَّ الْفَاعِلَ سَبَبُ التَّلَفِ وَهُوَ مُتَعَدٍّ فِيهِ فَكَأَنَّهُ مُوقِعٌ فِي الْبِئْرِ وَدَافِعٌ عَلَيْهِ الْحَجَرَ فَوَجَبَ الدِّيَةُ وَهِيَ عَلَى الْعَاقِلَةِ ( بِلَا كَفَّارَةٍ وَلَا إثْمِ الْقَتْلِ ) لِأَنَّ الْقَتْلَ مِنْهُ مَعْدُومٌ حَقِيقَةً وَأُلْحِقَ بِهِ الْخَطَأُ فِي حَقِّ الضَّمَانِ فَبَقِيَ فِي حَقِّ غَيْرِهِ عَلَى الْأَصْلِ وَإِنَّمَا قَالَ وَلَا إثْمَ الْقَتْلِ لِأَنَّهُ يَأْثَمُ بِالْحَفْرِ فِي غَيْرِ مِلْكِهِ ( وَلَا إرْثَ إلَّا هُنَا ) لِأَنَّ الْحِرْمَانَ بِسَبَبِ الْقَتْلِ وَلَا قَتْلَ هُنَا
