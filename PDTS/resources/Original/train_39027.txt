فَإِنْ أَبَى رَجَعَ مِيرَاثًا ( وَإِنْ عَيَّنَ ) الْمُوصِي ( غَيْرَ وَارِثٍ ) فَإِنْ سَمَّى لَهُ شَيْئًا لَمْ يُزَدْ عَلَيْهِ ( وَ ) إنْ ( لَمْ يُسَمِّ ) لَهُ شَيْئًا مُعَيَّنًا ( زِيدَ إنْ لَمْ يَرْضَ بِأُجْرَةِ مِثْلِهِ ثُلُثُهَا ) بِالرَّفْعِ نَائِبُ فَاعِلِ زِيدَ إنْ كَانَ الثُّلُثُ يَحْمِلُ ذَلِكَ ( ثُمَّ ) إنْ لَمْ يَرْضَ بِزِيَادَةِ الثُّلُثِ ( تُرُبِّصَ ) قَلِيلًا لَعَلَّهُ أَنْ يَرْضَى ( ثُمَّ ) إنْ لَمْ يَرْضَ أَيْضًا ( أُوجِرَ لِلصَّرُورَةِ ) بِالصَّادِ الْمُهْمَلَةِ وَهُوَ مَنْ لَمْ يَحُجَّ مِنْ الْأَحْرَارِ الْمُكَلَّفِينَ وَيُطْلَقُ عَلَى مَنْ لَمْ يَتَزَوَّجْ أَيْضًا ؛ لِأَنَّهُمَا صَرَّا دَرَاهِمَهُمَا فَلَمْ يُنْفِقَاهَا ( فَقَطْ ) دُونَ مَا لَيْسَ بِصَرُورَةٍ فَتَبْطُلُ الْوَصِيَّةُ لِلْمُعَيَّنِ وَيَرْجِعُ الْمَالُ كُلُّهُ مِيرَاثًا وَقَوْلُهُ ( غَيْرُ عَبْدٍ وَصَبِيٍّ ، وَإِنْ ) كَانَ غَيْرُهُمَا ( امْرَأَةً ) شَرْطٌ فِي كُلِّ أَجِيرٍ حَاجٍّ عَنْ الصَّرُورَةِ وَلَا يَخْتَصُّ بِالصَّرُورِةِ قَبْلَهُ ( وَلَمْ يَضْمَنْ وَصِيٌّ دَفَعَ لَهُمَا ) الْمَالَ لِيَحُجَّا بِهِ عَنْ الصَّرُورَةِ حَالَ كَوْنِ الْوَصِيِّ ( مُجْتَهِدًا ) بِأَنْ ظَنَّ الْعَبْدَ حُرًّا وَالصَّبِيَّ بَالِغًا وَحَجَّا أَوْ لَمْ يَحُجَّا وَتَلِفَ الْمَالُ ، وَيَضْمَنَانِ إنْ غَرَّا وَلَوْ حَجَّا بِالْفِعْلِ ، وَيَكُونُ جِنَايَةً فِي رَقَبَةِ الْعَبْدِ وَمَالِ الصَّبِيِّ فَلَوْ وُجِدَ الْمَالُ لَنُزِعَ مِنْهُمَا .
