( وَكَذَا لَوْ أَمَرَ ) الْمُسْلَمُ إلَيْهِ ( رَبَّ سَلَمِهِ بِقَبْضِهِ ) أَيْ بِقَبْضِ الْكُرِّ مِنْهُ ( لَهُ ) أَيْ لِأَجْلِ مُسْلَمٍ إلَيْهِ ( ثُمَّ ) بِقَبْضِهِ ثَانِيًا ( لِنَفْسِهِ ) أَيْ لِنَفْسِ رَبِّ السَّلَمِ ( فَاكْتَالَهُ ) أَيْ رَبُّ السَّلَمِ ( لِأَجْلِ الْمُسْلَمِ إلَيْهِ ، ثُمَّ ) اكْتَالَهُ ( لِنَفْسِهِ صَحَّ ) لِاجْتِمَاعِ الْكَيْلَيْنِ .
