( مَعَ ) بِفَتْحِ الْعَيْنِ عَلَى الْفَصِيحِ أَيْ سَأَلْتنِي أَنْ أَكْتُبَ لَك جُمْلَةً مُخْتَصَرَةً مُصَاحِبَةً لِكِتَابَةِ ( مَا ) أَيْ الَّذِي ( سَهَّلَ ) أَيْ يَسَّرَ ( سَبِيلَ ) أَيْ طَرِيقَ ( مَا أَشْكَلَ ) أَيْ صَعُبَ وَعَسِرَ فَهْمُهُ ( مِنْ ذَلِكَ ) الْمَذْكُورِ فِي الْجُمْلَةِ الْمُخْتَصَرَةِ أَوْ مِنْ مَذْهَبِ مَالِكٍ وَطَرِيقَتِهِ .
