فَلَا تُسَنُّ لَهُ إعَادَتُهُ ؛ لِأَنَّ الْقَصْدَ التَّخْفِيفُ أَوْ النَّشَاطُ ، وَظَاهِرُ كَلَامِ الشَّيْخِ تَقِيِّ الدِّينِ ، يَتَوَضَّأُ لِمَبِيتِهِ عَلَى إحْدَى الطَّهَارَتَيْنِ ( وَيُكْرَهُ ) لِلْجُنُبِ وَنَحْوِهِ ( تَرْكُهُ ) أَيْ : الْوُضُوءِ ( لِنَوْمٍ فَقَطْ ) لِظَاهِرِ الْحَدِيثِ ، وَلَا يُكْرَهُ تَرْكُهُ لِأَكْلٍ وَشُرْبٍ وَمُعَاوَدَةِ وَطْءٍ ( وَلَا يُكْرَهُ أَنْ يَأْخُذَ الْجُنُبُ وَنَحْوُهُ ) كَالْحَائِضِ وَالنُّفَسَاءَ شَيْئًا ( مِنْ شَعْرِهِ وَأَظْفَارِهِ ) وَتَقَدَّمَ ( وَلَا أَنْ يَخْتَضِبَ قَبْلَ الْغُسْلِ نَصًّا ) .( 1 / 436 )
