( فَإِنْ شَقَّ اسْتِلَامُهُ ) وَتَقْبِيلُهُ لَمْ يُزَاحِمْ وَاسْتَلَمَهُ بِيَدِهِ ( وَقَبَّلَ يَدَهُ ) لِحَدِيثِ ابْنِ عَبَّاسٍ أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ اسْتَلَمَهُ وَقَبَّلَ يَدَهُ وَرُوِيَ عَنْ ابْنِ عُمَرَ وَجَابِرٍ وَأَبِي هُرَيْرَةَ وَأَبِي سَعِيدٍ وَابْنِ عَبَّاسٍ ( فَإِنْ شَقَّ ) اسْتِلَامُهُ بِيَدِهِ ( اسْتَلَمَهُ بِشَيْءٍ وَقَبَّلَهُ ) رُوِيَ عَنْ ابْنِ عَبَّاسٍ مَوْقُوفًا ( فَإِنْ شَقَّ ) اسْتِلَامُهُ بِشَيْءٍ ( أَشَارَ إلَيْهِ بِيَدِهِ أَوْ بِشَيْءٍ وَاسْتَقْبَلَهُ بِوَجْهِهِ وَلَا يُقَبِّلُ الْمُشَارَ بِهِ ) لِعَدَمِ وُرُودِهِ ( وَلَا يُزَاحِم ) لِاسْتِلَامِ الْحَجَرِ أَوْ تَقْبِيلِهِ أَوْ السُّجُودِ عَلَيْهِ ( فَيُؤْذِي أَحَدًا ) مِنْ الطَّائِفِينَ .( 7 / 165 )
