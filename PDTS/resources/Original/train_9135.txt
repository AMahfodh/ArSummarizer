( وَإِنْ غَرَبَتْ الشَّمْسُ وَ ) بَقِيَ ( عَلَيْهِمَا شَيْءٌ ) مِنْ وَظِيفَةِ الْيَوْمِ لَمْ يَرْمِيَا لَيْلًا لِلْعَادَةِ ( وَ ) إنْ ( شَرَطَا رَمْيَهُ ) أَيْ الْبَاقِي عَلَيْهِمَا ( لَيْلًا لَزِمَ وَالْقَمَرُ قَدْ يَكْفِي ) ضَوْءُهُ ( وَإِلَّا ) بِأَنْ لَمْ يَكُنْ قَمَرٌ ( فَشَمْعَةٌ ) مَثَلًا يَكْفِي ضَوْءُهَا إنْ وَجَدَاهَا ( أَوْ ) يَرْمِيَانِ ( مِنْ الْغَدِ ) إنْ لَمْ يَجِدَاهَا وَذِكْرُ لُزُومِ الرَّمْيِ لَيْلًا وَالرَّمْيِ مِنْ الْغَدِ مِنْ زِيَادَتِهِ( 21 / 302 )
