( زَادَ الشَّيْخُ أَوْ غَضَبٍ أَوْ لَعْنَةٍ أَوْ نَفْيِ إيمَانٍ وَالْكَذِبُ صَغِيرَةٌ ) فَلَا تُرَدُّ الشَّهَادَةُ بِهِ إذَا لَمْ يُدْمَنْ عَلَيْهِ ( إلَّا فِي شَهَادَةِ زُورٍ أَوْ كَذِبٍ عَلَى نَبِيٍّ أَوْ رَمْيِ فِتَنٍ وَنَحْوِهِ ) كَكَذِبٍ عَلَى أَحَدِ الرَّعِيَّةِ عِنْدَ حَاكِمٍ ظَالِمٍ ( فَكَبِيرَةٌ ) قَالَ أَحْمَدُ فِي رِوَايَةِ عَبْدِ اللَّهِ : وَيُعْرَفُ الْكَذَّابُ بِخَلْفِ الْمَوَاعِيدِ ( وَيَجِبُ أَنْ يُخَلَّصَ بِهِ ) أَيْ الْكَذِبِ ( مُسْلِمٌ مِنْ قَتْلٍ ) قَالَ ابْنُ الْجَوْزِيِّ لَوْ كَانَ الْمَقْصُودُ وَاجِبًا ( وَيُبَاحُ ) الْكَذِبُ ( لِإِصْلَاحٍ ) بَيْنَ مُتَخَاصِمَيْنِ .
