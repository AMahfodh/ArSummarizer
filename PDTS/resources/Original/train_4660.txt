أَيْ عَلَى مُقَابِلِ الْأَظْهَرِ ( قَوْلُهُ : مِنْ تَرْجِيحِهِ ) أَيْ بَلَدِ الْمَوْقُوفِ ( قَوْلُهُ : عَلَى مُقَابِلِ الْأَظْهَرِ ) أَيْ الْمَارِّ بِقَوْلِ الْمَتْنِ وَأَنَّ مَصْرِفَهُ أَقْرَبُ النَّاسِ إلَخْ ( قَوْلُهُ : الْقَائِلِ ) أَيْ لِلْقَابِلِ ( قَوْلُهُ : وَمِنْ ثَمَّ ) أَيْ مِنْ أَجْلِ أَنَّ الْمُرَادَ فُقَرَاءُ وَمَسَاكِينُ بَلَدِ الْمَوْقُوفِ قَوْلُهُ : مَنْعُهُ ) أَيْ مَنْعِ رِيعَ الْوَقْفِ ( قَوْلُهُ : ، أَمَّا الْإِمَامُ ) إلَى الْمَتْنِ فِي النِّهَايَةِ وَالْمُغْنِي ( قَوْلُهُ : إذَا وَقَفَ ) أَيْ مِنْ أَمْوَالِ بَيْتِ الْمَالِ ، أَمَّا وَقْفُهُ مِنْ مَالِ نَفْسِهِ فَيَنْبَغِي أَنَّهُ كَغَيْرِهِ فِي الصَّرْفِ لِأَقَارِبِهِ ع ش وَرَشِيدِيٌّ وَمُغْنِي( 25 / 401 )
