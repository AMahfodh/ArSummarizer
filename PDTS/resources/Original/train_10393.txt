( فَصْلُ وَإِذَا اسْتَوَى وَلِيَّانِ فَأَكْثَرُ ) لِامْرَأَةٍ ( فِي الدَّرَجَةِ ) كَإِخْوَةٍ لَهَا كُلُّهُمْ لِأَبَوَيْنِ أَوْ لِأَبٍ أَوْ أَعْمَامٍ كَذَلِكَ أَوْ بَنِي إخْوَةٍ كَذَلِكَ ( فَإِنْ أَذِنَتْ لِوَاحِدٍ مِنْهُمْ ) بِعَيْنِهِ ( تَعَيَّنَ وَلَمْ يَصِحَّ نِكَاحُ غَيْرِهِ ) مِمَّنْ لَمْ تَأْذَنْ لِعَدَمِ الْإِذْنِ ( وَإِنْ أَذِنَتْ لَهُمْ ) أَيْ لِكُلِّ وَاحِدٍ مِنْهُمْ أَنْ يُزَوِّجَهَا ( صَحَّ التَّزْوِيجُ مِنْ كُلِّ وَاحِدٍ مِنْهُمْ ) لِأَنَّ سَبَبَ الْوِلَايَةِ مَوْجُودٌ فِي كُلِّ وَاحِدٍ مِنْهُمْ .
