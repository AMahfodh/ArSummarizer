( وَرُخِّصَ لَهُ إتْلَافُ مَالِ مُسْلِمٍ ) أَوْ ذِمِّيٍّ اخْتِيَارٌ ( بِقَتْلٍ أَوْ قَطْعٍ ) وَيُؤْجَرُ لَوْ صَبَرَ ابْنُ مَالِكٍ ( وَضَمَّنَ رَبُّ الْمَالِ الْمُكْرِهَ ) بِالْكَسْرِ لِأَنَّ الْمُكْرَهَ - بِالْفَتْحِ - كَالْآلَةِ ( لَا ) يُرَخَّصُ ( قَتْلُهُ ) أَوْ سَبُّهُ أَوْ قَطْعُ عُضْوِهِ وَمَا لَا يُسْتَبَاحُ بِحَالٍ اخْتِيَارٌ .
