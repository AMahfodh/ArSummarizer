( وَإِنْ كَانَ فِيهَا ) أَيْ الْأَرْضِ الَّتِي انْقَضَتْ إجَارَتُهَا ( زَرْعٌ بَقَاؤُهُ بِتَفْرِيطِ مُسْتَأْجِرٍ مِثْلُ أَنْ يَزْرَعَ ) الْمُسْتَأْجِرُ ( زَرْعًا لَمْ تَجْرِ الْعَادَةُ بِكَمَالِهِ قَبْلَ انْقِضَاءِ الْمُدَّةِ فَحُكْمُهُ حُكْمُ زَرْعِ الْغَاصِبِ ) لِأَنَّ بَقَاءَهُ فِيهَا بِعُدْوَانِهِ ( لِلْمَالِكِ ) لِلْأَرْضِ ( أَخْذُهُ ) أَيْ الزَّرْعِ ( بِالْقِيمَةِ ) هَكَذَا فِي الْمُقْنِعِ وَالْمُغْنِي وَالتَّنْقِيحِ وَالْمُنْتَهَى .
