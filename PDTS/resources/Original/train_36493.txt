وَالْإِكْرَاهُ الَّذِي لَا حِنْثَ مَعَهُ يَكُونُ ( بِخَوْفِ مُؤْلِمٍ ) وَيَكْفِي غَلَبَةُ الظَّنِّ وَلَا يُشْتَرَطُ تَيَقُّنُهُ وَبَيَّنَ الْمُؤْلِمَ بِقَوْلِهِ ( مِنْ قَتْلٍ أَوْ ضَرْبٍ ) وَإِنْ قَلَّ ( أَوْ سِجْنٍ ) ظُلْمًا ( أَوْ قَيْدٍ ) وَلَوْ لَمْ يَطُلْ ( أَوْ صَفْعٍ ) بِكَفٍّ فِي قَفًا ( لِذِي مُرُوءَةٍ ) بِفَتْحِ الْمِيمِ فِي الْأَفْصَح وَضَمِّهَا ( بِمَلَإٍ ) أَيْ جَمَاعَةٍ مِنْ النَّاسِ لَا فِي خَلْوَةٍ وَلَا غَيْرِ ذِي مُرُوءَةٍ أَيْ إنْ قَلَّ فَإِنْ كَثُرَ فَإِكْرَاهٌ مُطْلَقًا ( أَوْ قَتْلِ وَلَدِهِ ) وَإِنْ سَفَلَ وَكَذَا بِعُقُوبَتِهِ إنْ كَانَ بَارًّا .
