وَلَمَّا قَدَّمَ أَنَّهُ يَدْخُلُ الْبَذْرُ وَالثَّمَرُ الْغَيْرُ الْمُؤَبَّرِ فِي الْعَقْدِ عَلَى أَصْلِهِمَا شَرَعَ فِي الْكَلَامِ عَلَى بَيْعِهِمَا مُنْفَرِدَيْنِ فَقَالَ ( وَصَحَّ بَيْعُ ثَمَرٍ ) بِمُثَلَّثَةٍ مِنْ بَلَحٍ وَرُمَّانٍ وَتِينٍ وَعِنَبٍ وَغَيْرِ ذَلِكَ ( وَنَحْوِهِ ) كَقَمْحٍ وَشَعِيرٍ وَفُولٍ وَخَسٍّ وَكُرَّاتٍ وَجَزَرٍ وَفُجْلٍ ( بَدَا ) أَيْ ظَهَرَ ( صَلَاحُهُ ) بِيُبْسِ حَبٍّ وَبِانْتِفَاعٍ بِكَخَسٍّ وَعُصْفُرٍ ( إنْ لَمْ يَسْتَتِرْ ) بِأَكْمَامِهِ فَإِنْ اسْتَتَرَ بِهَا كَقَلْبِ جَوْزٍ وَلَوْزٍ فِي قِشْرِهِ وَكَقَمْحٍ فِي سُنْبُلِهِ وَبِزْرِ كَتَّانٍ فِي جَوْزِهِ لَمْ يَصِحَّ جُزَافًا لِأَنَّهُ غَيْرُ مَرْئِيٍّ وَيَصِحُّ كَيْلًا كَمَا سَبَقَ فِي قَوْلِهِ وَحِنْطَةٍ فِي سُنْبُلٍ وَتِبْنٍ إنْ بِكَيْلٍ وَأَمَّا شِرَاءُ مَا ذُكِرَ مَعَ قِشْرِهِ فَيَجُوزُ جُزَافًا وَلَوْ كَانَ بَاقِيًا فِي شَجَرِهِ لَمْ يُقْطَعْ إذَا بَدَا صَلَاحُهُ مَا لَمْ يَسْتَتِرْ فِي وَرَقِهِ فِيمَا لَهُ وَرَقٌ وَإِلَّا مُنِعَ بَيْعُهُ جُزَافًا أَيْضًا ( وَ ) صَحَّ بَيْعُ مَا ذُكِرَ ( قَبْلَهُ ) أَيْ قَبْلَ بُدُوِّ صَلَاحِهِ فِي ثَلَاثِ مَسَائِلَ وَهِيَ بَيْعُهُ ( مَعَ أَصْلِهِ ) كَبَلَحٍ صَغِيرٍ مَعَ نَخْلِهِ وَزَرْعٍ مَعَ أَرْضِهِ ( أَوْ ) بَيْعُ أَصْلِهِ مِنْ نَخْلٍ أَوْ أَرْضٍ ثُمَّ بَعْدَ ذَلِكَ بِقُرْبٍ أَوْ بُعْدٍ ( أُلْحِقَ ) الزَّرْعُ أَوْ الثَّمَرُ ( بِهِ ) أَيْ بِأَصْلِهِ الْمَبِيعِ قَبْلَهُ ( أَوْ ) بِيعَ مَا ذُكِرَ مُنْفَرِدًا قَبْلَ بُدُوِّ صَلَاحِهِ ( عَلَى ) شَرْطِ ( قَطْعِهِ ) فِي الْحَالِ أَوْ قَرِيبًا مِنْهُ بِحَيْثُ لَا يَزِيدُ وَلَا يَنْتَقِلُ عَنْ طَوْرِهِ إلَى طَوْرٍ آخَرَ فَيَجُوزُ بِشُرُوطٍ ثَلَاثَةٍ ( إنْ نَفَعَ ) أَيْ شَرْطٌ ( قَطَعَهُ ) فِي الْحَالِ أَوْ قَرِيبًا مِنْهُ بِحَيْثُ لَا يَزِيدُ وَلَا يَنْتَقِلُ عَنْ طَوْرِهِ إلَى طَوْرٍ آخَرَ فَيَجُوزُ بِشُرُوطٍ ثَلَاثَةٍ ( إنْ نَفَعَ ) أَيْ بَلَغَ حَدَّ الِانْتِفَاعِ بِهِ كَالْحِصْرِمِ وَإِلَّا فَهُوَ إضَاعَةُ مَالٍ كَالْكُمَّثْرَى قَبْلَ ظُهُورِ الْحَلَاوَةِ فِيهَا فَإِنَّهَا غَيْرُ مُنْتَفَعٍ بِهَا إذْ هِيَ مُرَّةٌ فِي هَذِهِ الْحَالَةِ ( وَاضْطُرَّ لَهُ ) الْمُرَادُ بِالِاضْطِرَارِ الْحَاجَةُ( 12 / 251 )
