( وَحُنِّثَ ) بِضَمٍّ فَكَسْر مُثْقَلًا الزَّوْجُ أَيْ قُضِيَ عَلَيْهِ بِالْحِنْثِ ( إنْ حَلَفَ ) أَنْ لَا يُدْخِلَ لَهَا أَبَوَاهَا وَوَلَدَهَا مِنْ غَيْرِهِ وَشَبَّهَ فِي التَّحْنِيثِ فَقَالَ ( كَحَلِفِهِ ) أَيْ الزَّوْجِ عَلَى ( أَنْ لَا تَزُورَ ) زَوْجَتُهُ ( وَالِدَيْهَا ) فَتَخْرُجُ لِزِيَارَتِهِمَا ( إنْ كَانَتْ ) الزَّوْجَةُ ( مَأْمُونَةً ) عَلَى نَفْسِهَا إنْ كَانَتْ مُتَجَالَّةً بَلْ ( وَلَوْ ) كَانَتْ ( شَابَّةً ) فَإِنْ لَمْ تَكُنْ مَأْمُونَةً فَلَا تَخْرُجُ وَلَوْ مُتَجَالَّةً أَوْ مَعَ أَمِينَةٍ .
