أَوْ هِبَةٍ أَوْ غَيْرِهِمَا سَقَطَ ؛ لِأَنَّ السَّيِّدَ لَا يَثْبُتُ لَهُ الدَّيْنُ بِذِمَّةِ عَبْدِهِ ( أَوْ ) مَلَكَ رَبُّ دَيْنٍ ( مَنْ تَعَلَّقَ ) دَيْنُهُ ( بِرَقَبَتِهِ بِلَا عِوَضٍ ) بِأَنْ وَرِثَهُ أَوْ وُهِبَ لَهُ ( سَقَطَ ) الدَّيْنُ ؛ لِأَنَّهُ لَا بَدَلَ لِلرَّقَبَةِ يَتَحَوَّلُ الدَّيْنُ إلَيْهِ( 5 / 450 )
