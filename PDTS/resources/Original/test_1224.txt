( وَلَوْ كَانَتْ الْعَارِيَّةُ عَبْدًا فَأَعْتَقَهُ الْمُعِيرُ جَازَ ) لِقِيَامِ مِلْكِ الرَّقَبَةِ ( ثُمَّ الْمُرْتَهِنُ بِالْخِيَارِ إنْ شَاءَ رَجَعَ بِالدَّيْنِ عَلَى الرَّاهِنِ ) ؛ لِأَنَّهُ لَمْ يَسْتَوْفِهِ ( وَإِنْ شَاءَ ضَمِنَ الْمُعِيرُ قِيمَتَهُ ) ؛ لِأَنَّ الْحَقَّ قَدْ تَعَلَّقَ بِرَقَبَتِهِ بِرِضَاهُ وَقَدْ أَتْلَفَهُ بِالْإِعْتَاقِ ( وَتَكُونُ رَهْنًا عِنْدَهُ إلَى أَنْ يَقْبِضَ دَيْنَهُ فَيَرُدَّهَا إلَى الْمُعِيرِ ) ؛ لِأَنَّ اسْتِرْدَادَ الْقِيمَةِ كَاسْتِرْدَادِ الْعَيْنِ
