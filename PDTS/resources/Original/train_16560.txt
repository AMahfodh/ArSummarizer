( فَإِنْ خَالَعَهَا ) الْوَكِيلُ ( بِمَهْرِهَا فَمَا زَادَ صَحَّ ) الْخُلْعُ وَلَزِمَ الْمُسَمَّى لِأَنَّهُ زَادَ خَيْرًا ( وَإِنْ نَقَصَ ) الْوَكِيلُ ( مِنْ الْمَهْرِ ) مَعَ الْإِطْلَاقِ ( رَجَعَ ) الْمُوَكِّلُ ( عَلَى الْوَكِيلِ بِالنَّقْصِ وَصَحَّ الْخُلْعُ ) لِأَنَّهُ عَقْدُ مُعَاوَضَةٍ أَشْبَهَ الْبَيْعَ .( 18 / 4 )
