( وَ ) هِيَ تَثْبُتُ ( بِالْعَقْدِ ) أَيْ عَقْدِ النِّكَاحِ ( عِنْدَهُ ) أَيْ عِنْدَ أَبِي حَنِيفَةَ ( فِي وَطْءٍ مُحَرَّمٍ نِكَاحُهَا ) وَإِنْ كَانَ حُرْمَتُهُ مُتَّفَقًا عَلَيْهَا وَهُوَ عَالِمٌ بِهِ حَيْثُ لَا حَدَّ عَلَيْهِ عِنْدَهُ وَلَكِنْ يُوجَعُ عُقُوبَةً إنْ عَلِمَ بِذَلِكَ وَعِنْدَ غَيْرِهِ إنْ عَلِمَ يُحَدُّ وَإِلَّا فَلَا وَسَيَأْتِي بَيَانُهُ
