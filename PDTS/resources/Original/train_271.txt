( فَإِنْ أَبَى ) الْمَنْزُولُ بِهِ ضِيَافَةَ الْمُسْلِمِ ( فَلِلضَّيْفِ طَلَبُهُ بِهِ ) أَيْ بِنَحْوِ ضِيَافَتِهِ ( عِنْد حَاكِمٍ ) لِوُجُوبِهِ عَلَيْهِ كَالزَّوْجَةِ ( فَإِنْ تَعَذَّرَ ) عَلَى الضَّيْفِ أَنْ يُحَاكِمَهُ ( جَازَ لَهُ الْأَخْذُ مِنْ مَالِهِ بِقَدْرِ ضِيَافَتِهِ ) الْوَاجِبَةِ ( بِغَيْرِ إذْنِهِ ) لِمَا تَقَدَّمَ .( 21 / 227 )
