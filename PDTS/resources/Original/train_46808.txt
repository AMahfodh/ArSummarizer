( وَإِقْرَارُ الْأَبَوَيْنِ ) لِلزَّوْجَيْنِ الصَّغِيرَيْنِ وَنَحْوِهِمَا مِمَّنْ يُزَوِّجُ بِلَا إذْنِهِ بِالرَّضَاعِ الْمُوجِبِ لِلْحُرْمَةِ بَيْنَهُمَا وَخَبَرُ إقْرَارِ ( مَقْبُولٌ ) إنْ أَقَرَّا بِهِ ( قَبْلَ ) عَقْدِ ( النِّكَاحِ ) فَيُمْنَعُ وَإِنْ وَقَعَ فَيُفْسَخُ ( لَا ) يُقْبَلُ إقْرَارُهُمَا بِهِ ( بَعْدَهُ ) أَيْ النِّكَاحِ فَلَا يُفْسَخُ كَإِقْرَارٍ أَبَوَيْ الْكَبِيرَيْنِ وَلَوْ قَبْلَ الْعَقْدِ وَهُمَا كَالْأَجْنَبِيَّيْنِ فَيَجْرِي فِيهِمَا مَا يَأْتِي فِيهِمَا وَشَمِلَ قَوْلُهُ الْأَبَوَيْنِ أَبَا أَحَدِهِمَا وَأُمَّ الْآخَرِ أَيْضًا لَا أُمَّيْهِمَا .
