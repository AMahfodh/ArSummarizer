( { وَوَجِلَتْ } ) بِكَسْرِ الْجِيمِ بِمَعْنَى الْخَوْفِ ( { مِنْهَا } ) تَعْلِيلِيَّةٌ أَيْضًا ( { الْقُلُوبُ فَقَالَ رَجُلٌ } ) مِنْ الْحَاضِرِينَ لِشِدَّةِ اهْتِمَامِهِ خِلَافَ عَادَتِهِ ( { يَا رَسُولَ اللَّهِ كَأَنَّ هَذِهِ } ) الْمَوْعِظَةَ ( { مَوْعِظَةُ مُوَدِّعٍ } ) أَيْ كَمَوْعِظَةِ مُوَدِّعٍ أَوْ هِيَ مَوْعِظَةُ مُوَدِّعٍ لِأَهْلِهِ وَعِيَالِهِ حِينَ إرَادَةِ السَّفَرِ بِنَصَائِحَ يُحْتَاجُ إلَيْهَا غَايَةَ الِاحْتِيَاجِ فَرْطًا لِحُبِّهِ وَحِرْصِهِ عَلَيْهِ أَنْ لَا يَضِلَّ بَعْدَهُ كَمَا فِي حَدِيثِ الْجَامِعِ الصَّغِيرِ صَلِّ صَلَاةَ مُوَدِّعٍ .
