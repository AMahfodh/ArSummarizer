( وَلَا يُجْزِئُ ) فِي الطَّهَارَةِ ( دَلْكُ خُفٍّ ) ، أَوْ نَعْلٍ ( تَنَجَّسَ بِأَرْضٍ ) أَوْ نَحْوِهَا كَالثَّوْبِ وَأَمَّا خَبَرُ أَبِي دَاوُد { إذَا أَصَابَ خُفَّ أَحَدِكُمْ أَذًى فَلْيَدْلُكْهُ فِي الْأَرْضِ } فَمَحْمُولٌ عَلَى الْمُسْتَقْذَرِ الظَّاهِرِ وَقَوْلُهُ بِأَرْضٍ مُتَعَلِّقٌ بِدَلْكِ خُفٍّ .( 3 / 30 )
