( وَ ) أَنْ ( يَقُولَ ) مَعَ كُلِّ حَصَاةٍ ( اللَّهُمَّ اجْعَلْهُ حَجًّا مَبْرُورًا ، وَذَنْبًا مَغْفُورًا ، وَسَعْيًا مَشْكُورًا ) لِمَا رَوَى حَنْبَلٌ عَنْ زَيْدِ بْنِ أَسْلَمَ قَالَ : رَأَيْت { سَالِمَ بْنَ عَبْدِ اللَّهِ اسْتَبْطَنَ الْوَادِيَ وَرَمَى الْجَمْرَةَ بِسَبْعِ حَصَيَاتٍ يُكَبِّرُ مَعَ كُلِّ حَصَاةٍ : اللَّهُ أَكْبَرُ اللَّهُ أَكْبَرُ ثُمَّ قَالَ : اللَّهُمَّ اجْعَلْهُ - فَذَكَرَهُ - فَسَأَلْته عَمَّا صَنَعَ فَقَالَ : حَدَّثَنِي أَبِي أَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ رَمَى الْجَمْرَةَ مِنْ هَذَا الْمَكَانِ وَيَقُولُ كُلَّمَا رَمَى مِثْلَ ذَلِكَ } ( وَ ) نُدِبَ أَنْ ( يَسْتَبْطِنَ الْوَادِيَ ، وَ ) أَنْ ( يَسْتَقْبِلَ الْقِبْلَةَ ، وَ ) أَنْ ( يَرْمِيَ عَلَى جَانِبِهِ الْأَيْمَنِ ) لِحَدِيثِ عَبْدِ اللَّهِ بْنِ يَزِيدَ { لَمَّا أَتَى عَبْدُ اللَّهِ جَمْرَةَ الْعَقَبَةِ اسْتَبْطَنَ الْوَادِيَ( 4 / 43 )
