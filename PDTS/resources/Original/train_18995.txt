( قَوْلُهُ : لَا تَسْتَأْنِفُ ) لِأَنَّهُ لَمْ يَتَبَيَّنْ بِالْحَيْضِ أَنَّهَا كَانَتْ قَبْلُ مِنْ ذَوَاتِ الْأَقْرَاءِ بِخِلَافِ الْآيِسَةِ ط ( قَوْلُهُ : إلَّا إذَا حَاضَتْ ) اسْتِثْنَاءٌ مُنْقَطِعٌ ط ( قَوْلُهُ : فِي أَثْنَائِهَا ) أَيْ قَبْلَ تَمَامِهَا وَلَوْ بِسَاعَةٍ ط ( قَوْلُهُ : ثُمَّ أَيِسَتْ ) أَيْ بَلَغَتْ سِنَّ الْإِيَاسِ عِنْدَ الْحَيْضَتَيْنِ وَانْقَطَعَ دَمُهَا فَتْحٌ .( 12 / 375 )
