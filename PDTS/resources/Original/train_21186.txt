قَوْلُهُ ( بِخِلَافِ ) شَهَادَةِ ( أَخٍ لِأَخٍ ) فَتَجُوزُ ( إنْ بَرَزَ ) فِي الْعَدَالَةِ بِأَنْ فَاقَ أَقْرَانَهُ فِيهَا وَلَمْ يَكُنْ فِي عِيَالِهِ ( وَلَوْ بِتَعْدِيلٍ ) أَيْ بِأَنْ يُعَدِّلَهُ كَمَا هُوَ ظَاهِرُهَا وَهُوَ الْمَشْهُورُ ( وَتُؤُوِّلَتْ أَيْضًا بِخِلَافِهِ ) أَيْ بِأَنْ لَا يُعَدِّلَهُ
