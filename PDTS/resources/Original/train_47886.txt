( الْخَامِسُ ) مِنْ الْمُحَرَّمَاتِ ( اصْطِيَادُ كُلِّ ) صَيْدٍ ( مَأْكُولٍ بَرِّيٍّ ) وَحْشِيٍّ كَبَقَرِ وَحْشٍ وَدَجَاجَةٍ وَحَمَامَةٍ ( قُلْت : وَكَذَا الْمُتَوَلَّدُ مِنْهُ ) أَيْ الْمَأْكُولِ الْبَرِّيِّ الْوَحْشِيِّ ( وَمِنْ غَيْرِهِ ) كَمُتَوَلَّدٍ بَيْنَ حِمَارٍ وَحْشِيٍّ وَحِمَارٍ أَهْلِيٍّ أَوْ بَيْنَ شَاةٍ وَظَبْيٍ ( وَاَللَّهُ أَعْلَمُ ) أَمَّا الْأَوَّلُ فَلِقَوْلِهِ تَعَالَى : { وَحُرِّمَ عَلَيْكُمْ صَيْدُ الْبَرِّ } أَيْ أَخْذُهُ { مَا دُمْتُمْ حُرُمًا } .
