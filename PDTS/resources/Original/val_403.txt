فِي الْإِعْتَاقِ لِلْوَلَدِ ( وَيُشْتَرَطُ فِي ) صُورَةِ ( الِاسْتِدْعَاءِ ) لِوُقُوعِ الْعِتْقِ عَنْ الْمُسْتَدْعِي وَلُزُومِ الْعِوَضِ لَهُ ( الْجَوَابُ ) لَهُ ( فَوْرًا وَإِلَّا ) بِأَنْ لَمْ يَكُنْ جَوَابٌ أَوْ طَالَ الْفَصْلُ ( وَقَعَ ) الْعِتْقُ ( عَنْ الْمَالِكِ وَلَا شَيْءَ لَهُ ) عَلَى الْمُسْتَدْعِي( 17 / 251 )
