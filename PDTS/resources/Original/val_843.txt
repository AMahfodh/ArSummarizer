( وَمِثْلُهُ ) ؛ أَيْ : مِثْلُ التَّوْكِيلِ فِيمَا ذُكِرَ ( تَوَكَّلَ ) عَنْ غَيْرِهِ ، فَلَا يَصِحُّ أَنْ يَتَوَكَّلَ فِي شَيْءٍ إلَّا مِمَّنْ يَصِحُّ مِنْهُ لِنَفْسِهِ ، ( فَلَا يَصِحُّ أَنْ يُوجِبَ ) عَنْ غَيْرِهِ ( نِكَاحًا مَنْ لَا يَصِحُّ ) مِنْهُ إيجَابُهُ ( لِمُوَلِّيَتِهِ ) لِنَحْوِ فِسْقٍ ؛ لِأَنَّهُ إذَا لَمْ يَجُزْ أَنْ يَتَوَلَّاهُ أَصَالَةً ، فَلَمْ يَجُزْ بِالنِّيَابَةِ ؛ كَالْمَرْأَةِ ، ( وَلَا يَقْبَلُهُ ) ؛ أَيْ : النِّكَاحَ ( مَنْ لَا يَصِحُّ مِنْهُ ) قَبُولُهُ ( لِنَفْسِهِ ) ؛ كَالْكَافِرِ يَتَوَكَّلُ فِي قَبُولِ نِكَاحِ مُسْلِمَةٍ لِمُسْلِمٍ .( 9 / 326 )
