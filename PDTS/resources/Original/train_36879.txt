( وَلَا حَدَّ لِأَكْثَرِهِ ) أَيْ السِّنِّ لِجَوَازِ أَنْ لَا تَحِيضَ أَصْلًا كَمَا مَرَّ ( وَأَقَلُّ ) زَمَنِ ( الْحَمْلِ ) ( سِتَّةُ أَشْهُرٍ ) وَلَحْظَتَانِ لَحْظَةٌ لِلْوَطْءِ وَلَحْظَةٌ لِلْوَضْعِ مِنْ إمْكَانِ اجْتِمَاعِهِمَا بَعْدَ عَقْدِ النِّكَاحِ ( وَأَكْثَرُهُ ) أَيْ زَمَنِ الْحَمْلِ ( أَرْبَعُ سِنِينَ ) وَغَالِبُهُ تِسْعَةُ أَشْهُرٍ( 3 / 247 )
