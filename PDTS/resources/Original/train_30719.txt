( وَالشَّجَرَةُ أَمَانَةٌ فِي يَدِ الْمُشْتَرِي ) لِتَعَذُّرِ تَسَلُّمِ الثَّمَرَةِ بِدُونِهَا ( بِخِلَافِ مَا لَوْ اشْتَرَى سَمْنًا ) أَوْ نَحْوَهُ ( فَقَبَضَهُ فِي ظَرْفٍ ) لِلْبَائِعِ ( فَإِنَّهُ مَضْمُونٌ عَلَيْهِ ) لِتَمَكُّنِهِ مِنْ التَّسَلُّمِ فِي غَيْرِهِ ( وَأَمَّا ) بَيْعُهُ ( قَبْلَ بُدُوِّ الصَّلَاحِ فَلَا يَجُوزُ ) لِمَا مَرَّ ( إلَّا فِي مُنْتَفِعٍ بِهِ ) كَحِصْرِمٍ وَبَلَحٍ وَلَوْزٍ ( بِشَرْطِ الْقَطْعِ ) بِقَيْدٍ زَادَهُ بِقَوْلِهِ ( مُنَجَّزًا ) فَيَجُوزُ بِالْإِجْمَاعِ الْمُخَصِّصِ لِلْخَبَرِ السَّابِقِ فَدَخَلَ فِي الْمُسْتَثْنَى مِنْهُ مَا لَا يُنْتَفَعُ بِهِ كَ كُمَّثْرَى( 8 / 402 )
