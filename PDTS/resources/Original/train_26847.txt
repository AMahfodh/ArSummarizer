( قَوْلُهُ مِنْهُمَا ) أَيْ مِنْ رَجُلٍ أَوْ امْرَأَةٍ ا ه مُغْنِي ( قَوْلُهُ وَفَعَلَهُ ) أَيْ الرَّضِيعُ مِنْهُمَا ( قَوْلُهُ لَغْوٌ ) أَيْ لِأَنَّهُ كَانَ صَغِيرًا مُغْنِي وَنِهَايَةٌ ( قَوْلُهُ نَعَمْ الْيَمِينُ الْمَرْدُودَةُ إلَخْ ) أَيْ وَأَمَّا مَا فِي الْمَتْنِ فَفِي الْيَمِينِ الْأَصْلِيَّةِ مُغْنِي وَنِهَايَةٌ ( قَوْلُ الْمَتْنِ وَمُدَّعِيهِ إلَخْ ) أَيْ الْإِرْضَاعِ مِنْ رَجُلٍ أَوْ امْرَأَةٍ مُغْنِي وَمُحَلَّيْ وَشَرْحُ الْمَنْهَجِ وَقَدْ يُشْكِلُ ذَلِكَ فِي الرَّجُلِ لِأَنَّهُ إذَا ادَّعَى الرَّضَاعَ انْفَسَخَ نِكَاحُهُ مُؤَاخَذَةً لَهُ( 35 / 340 )
