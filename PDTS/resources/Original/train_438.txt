( وَيَتَّجِهُ وَكَذَا ) أَيْ : كَاَلَّتِي قَبْلَهَا ( قَوْلُهُ ) أَيْ : قَوْلُ رَجُلٍ ( لِآخَرَ : إنَّك لَحَسُودٌ ) أَيْ : كَثِيرُ الْحَسَدِ ( فَقَالَ الْآخَرُ ) مُجِيبًا لَهُ ( أَحَسَدَنَا ) أَيْ : أَكْثَرُنَا حَسَدًا ( امْرَأَتُهُ طَالِقٌ ) فَقَالَ : نَعَمْ ؛ لَمْ تَطْلُقْ امْرَأَةٌ وَاحِدَةٌ مِنْهُمَا ، لِبَقَاءِ يَقِينِ نِكَاحِهِمَا ، وَوُقُوعُ الطَّلَاقِ مَشْكُوكٌ فِيهِ ، وَهُوَ مُتَّجِهٌ .
