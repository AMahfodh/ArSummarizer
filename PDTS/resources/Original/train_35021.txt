( قَوْلُهُ : مَعَ قُرْبِهِ ) أَيْ وَمُسَاوَاتِهِ لَهَا فَمَعَ بُعْدِهِ وَقُرْبِهَا عَنْهُ أَوْلَى ( قَوْلُهُ لَا يُعَصِّبُ مَنْ هِيَ دُونَهُ ) أَيْ لِقُرْبِهِ وَبُعْدِهَا فَتَسْقُطُ بِهِ ( قَوْلُهُ : وَلَا مَنْ فَوْقَهُ إلَخْ ) أَيْ لِعَدَمِ حِرْمَانِهَا فَانْتَفَى الْمَعْنَى السَّابِقُ ( قَوْلُهُ : أَمَّا إذَا لَمْ يُنْفَ إلَخْ ) بَيَانٌ لِمَفْهُومِ الْمَتْنِ وَإِنْ أَفَادَهُ قَوْلُهُ : وَلِهَذَا الْمَعْنَى إلَخْ ؛ لِأَنَّ الْمَقْصُودَ بِهِ بَيَانُ الْحِكْمَةِ( 13 / 104 )
