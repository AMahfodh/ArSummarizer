( قَوْلُهُ ؛ لِأَنَّهَا ) أَيْ عِمَارَةَ نَحْوِ الْمَسْجِدِ ( قَوْلُهُ لَا لِمَسْجِدٍ سَيُبْنَى ) أَيْ بِالنِّسْبَةِ لِلْمَصَالِحِ كَمَا هُوَ ظَاهِرٌ ا ه رَشِيدِيٌّ ( قَوْلُهُ عَلَى قِيَاسِ إلَخْ ) رَاجِعٌ عَلَى الِاسْتِثْنَاءِ فَقَطْ ، وَإِلَّا فَقَدْ مَرَّ الْمُسْتَثْنَى مِنْهُ بِنَفْسِهِ ( قَوْلُهُ مَرَّ آنِفًا ) أَيْ فِي شَرْحِ أَنْ يُتَصَوَّرَ لَهُ الْمِلْكُ ( قَوْلُهُ وَيَصْرِفُهُ النَّاظِرُ إلَخْ ) أَيْ فَلَيْسَ لِلْمُوصِي الصَّرْفُ بِنَفْسِهِ بَلْ يَدْفَعُهُ لِلنَّاظِرِ أَوْ لِمَنْ أَقَامَهُ مَقَامَهُ وَمِثْلُهَا النَّذْرُ لِلْأَضْرِحَةِ الْمَشْهُورَةِ كَضَرِيحِ إمَامِنَا الشَّافِعِيِّ رَضِيَ اللَّهُ( 27 / 352 )
