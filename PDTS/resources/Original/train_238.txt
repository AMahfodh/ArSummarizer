( وَكَانَتْ الْمُرْضِعَاتُ بَنَاتِهِ ) أَيْ : رَجُلٍ وَاحِدٍ ( أَوْ بَنَاتِ زَوْجَتِهِ ) أَوْ أَرْضَعَتْ طِفْلًا أَوْ طِفْلَةَ زَوْجَةٍ لِأَبِيهِنَّ ، ( أَوْ لَمْ تَكُنْ زَوْجَتَهُ كُلُّ وَاحِدَةٍ مِنْهُنَّ رَضْعَةً فَلَا أُمُومَةَ ) لِوَاحِدَةٍ مِنْ الْمُرْضِعَاتِ ؛ لِأَنَّهَا لَمْ تُرْضِعْ خَمْسًا ( وَلَا يَصِيرُ ) أَبُو الْمُرْضِعَاتِ ( جَدًّا ) لِلطِّفْلِ أَوْ الطِّفْلَةِ ، لِعَدَمِ ثُبُوتِ الْأُمُومَةِ ( وَلَا ) تَصِيرُ ( زَوْجَتُهُ ) أُمَّ الْمُرْضِعَاتِ ( جَدَّةً ) لِلطِّفْلِ أَوْ الطِّفْلَةِ ( وَلَا ) تَصِيرُ ( إخْوَةُ الْمُرْضِعَاتِ أَخْوَالًا ) لِلطِّفْلِ أَوْ الطِّفْلَةِ ( وَلَا ) تَصِيرُ ( أَخَوَاتُهُنَّ ) أَيْ الْمُرْضِعَاتِ ( خَالَاتٍ ) لِلطِّفْلِ أَوْ الطِّفْلَةِ ؛ لِأَنَّ تِلْكَ فُرُوعُ الْأُمُومَةِ ، وَلَمْ تَثْبُتْ .( 17 / 14 )
