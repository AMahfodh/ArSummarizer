( وَ ) جَازَ دَفْعُهَا لِصَحِيحٍ ( قَادِرٍ عَلَى الْكَسْبِ ) وَلَوْ تَرَكَهُ اخْتِيَارًا ( وَمَالِكِ نِصَابٍ ) أَوْ أَكْثَرَ حَيْثُ لَا يَكْفِيهِ لِعَامِهِ ( وَ ) جَازَ ( دَفْعُ أَكْثَرَ مِنْهُ ) أَيْ مِنْ النِّصَابِ ( وَ ) دَفْعُ ( كِفَايَةِ سَنَةٍ ) فَالْمَدَارُ عَلَى كِفَايَةِ سَنَةٍ وَلَوْ أَكْثَرَ مِنْ نِصَابٍ فَلَا يُعْطَى أَكْثَرَ مِنْ كِفَايَةِ سَنَةٍ وَلَوْ أَقَلَّ مِنْ نِصَابٍ
