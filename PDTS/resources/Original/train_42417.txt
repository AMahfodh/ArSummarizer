( وَافْرِضْ مُشَبَّهًا بِهِ فِي الِاسْتِوَا ) قُرْبًا إلَى الْوَارِثِ ( بِأَنَّهُ ) بِزِيَادَةِ الْبَاءِ أَيْ أَنَّهُ ( الْوَارِثَ لِلَّذِي تَوَا ) بِالْمُثَنَّاةِ أَيْ هَلَكَ ( وَاقْسِمْ نَصِيبًا لِمُشَبَّهٍ بِهِ قَدَّرْت ) أَيْ قَدَّرْته ( وَارِثًا ) لِلْمَيِّتِ ( عَلَى الْمُشَبَّهِ كَإِرْثِهِ مِنْهُ ) أَيْ كَإِرْثِ الْمُشَبَّهِ مِنْ الْمُشَبَّهِ بِهِ فَلَوْ خَلَّفَ أَوْلَادَ أَخَوَاتٍ مُتَفَرِّقَاتٍ فُرِضَ الْمُشَبَّهُ بِهِ وَارِثًا فَلِلْأُخْتِ لِلْأَبَوَيْنِ النِّصْفُ وَلِكُلٍّ مِنْ الْآخَرَيْنِ السُّدُسُ وَيُرَدُّ عَلَيْهِنَّ الْبَاقِي فَتَكُونُ الْمَسْأَلَةُ مِنْ خَمْسَةٍ ثَلَاثَةٌ لِأَوْلَادِ الْأُخْتِ لِلْأَبَوَيْنِ { لِلذَّكَرِ مِثْلُ حَظِّ الْأُنْثَيَيْنِ } وَوَاحِدٌ لِأَوْلَادِ الْأُخْتِ لِلْأَبِ كَذَلِكَ وَوَاحِدٌ لِأَوْلَادِ الْأُخْتِ لِلْأُمِّ بِالسَّوِيَّةِ بَيْنَ الذَّكَرِ وَالْأُنْثَى ، وَمِثْلُهُمْ أَوْلَادُ الْأَخِ لِلْأُمِّ وَهَذَا مَا عَلَيْهِ الْجُمْهُورُ وَقَالَ الْإِمَامُ : قِيَاسُ الْمُنَزِّلِينَ تَفْضِيلُ الذَّكَرِ وَهُوَ قَضِيَّةُ قَوْلِ النَّظْمِ كَأَصْلِهِ كَإِرْثِهِ مِنْهُ وَصَرَّحَ مِنْ زِيَادَتِهِ بِقَوْلِهِ ( وَإِنْ بَعْضٌ ) مِنْ الْمُشَبَّهِ بِهِ ( حَجَبْ بَعْضًا ) مِنْهُ ( فَهَذَا فِي مُشَبَّهٍ وَجَبْ ) لِتَنْزِيلِهِ مَنْزِلَتَهُ فَلَوْ خَلَّفَ ثَلَاثَ بَنَاتِ أُخُوَّةٍ مُتَفَرِّقِينَ فَلِبِنْتِ الْأَخِ لِلْأُمِّ السُّدُسُ وَلِبِنْتِ الْأَخِ لِلْأَبَوَيْنِ الْبَاقِي وَتُحْجَبُ بِهَا بِنْتُ الْآخَرِ كَمَا يُحْجَبُ بِأَبِيهَا أَبُوهَا ، وَقَدْ بَسَطْت الْكَلَامَ عَلَى الرَّدِّ وَتَوْرِيثِ ذِي الرَّحِمِ فِي نِهَايَةِ الْهِدَايَةِ .
