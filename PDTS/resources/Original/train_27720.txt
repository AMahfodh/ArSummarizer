وَ ( لَا ) يَجُوزُ ( صُعُودُ شَجَرِهِ ) أَيْ الثَّمَرِ ( وَلَا ضَرْبُهُ أَوْ رَمْيُهُ بِشَيْءٍ ) نَصًّا وَلَوْ كَانَ الْبُسْتَانُ غَيْرَ مَحُوطٍ وَلَا حَارِسَ لِحَدِيثِ الْأَثْرَمِ { وَكُلْ مَا وَقَعَ أَشْبَعَكَ اللَّهُ وَأَرْوَاكَ } رَوَاهُ التِّرْمِذِيُّ وَقَالَ حَسَنٌ صَحِيحٌ وَلِأَنَّ الضَّرْبَ وَالرَّمْيَ يُفْسِدُ الثَّمَرَ ( وَلَا يَحْمِلُ ) مِنْ الثَّمَرِ مُطْلَقًا كَغَيْرِهِ لِقَوْلِ عُمَرَ وَلَا تَتَّخِذْ خُبْنَةً( 11 / 373 )
