( فَرْعٌ ) لَوْ ( عَلَّقَ الظِّهَارَ بِدُخُولِهَا الدَّارَ فَدَخَلَتْ وَهُوَ مَجْنُونٌ أَوْ نَاسٍ فَمُظَاهِرٌ ) مِنْهَا كَنَظِيرِهِ فِي الطَّلَاقِ الْمُعَلَّقِ بِدُخُولِهَا وَإِنَّمَا يُؤَثِّرُ النِّسْيَانُ وَالْجُنُونُ فِي فِعْلِ الْمَحْلُوفِ عَلَى فِعْلِهِ ( وَلَا عَوْدَ ) مِنْهُ ( حَتَّى يُفِيقَ ) مِنْ جُنُونِهِ ( أَوْ يَتَذَكَّرَ ) أَيْ يَتَذَكَّرَ بَعْدَ نِسْيَانِهِ ( ثُمَّ يُمْسِكَ ) الْمُظَاهِرُ مِنْهَا زَمَنًا يُمْكِنُ فِيهِ الطَّلَاقُ وَلَمْ يُطَلِّقْ وَوَقَعَ فِي الْأَصْلِ هُنَا مَا يُخَالِفُ ذَلِكَ وَسَبَبُهُ سُقُوطُ لَفْظَةِ لَا مِنْهُ .( 17 / 208 )
