( وَلَا يَقْتُلُهُ إلَّا الْإِمَامُ أَوْ نَائِبُهُ حُرًّا كَانَ الْمُرْتَدُّ أَوْ عَبْدًا ) لِأَنَّهُ قُتِلَ لِحَقِّ اللَّهِ تَعَالَى فَكَانَ إلَى الْإِمَامِ أَوْ نَائِبِهِ كَقَتْلِ الْحُرِّ وَلَا يُعَارِضُهُ قَوْلُهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ : { أَقِيمُوا الْحُدُودَ عَلَى مَا مَلَكَتْ أَيْمَانُكُمْ } لِأَنَّ قَتْلَ الْمُرْتَدِّ لِكُفْرِهِ لَا حَدًّا ( وَلَا يَجُوزُ أَخْذُ فِدَاءٍ عَنْهُ ) أَيْ عَنْ الْمُرْتَدِّ بَلْ يُقْتَلُ بَعْدَ الِاسْتِتَابَةِ لِمَا تَقَدَّمَ مِنْ قَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ : { مَنْ بَدَّلَ دِينَهُ فَاقْتُلُوهُ } ( وَإِنْ قَتَلَهُ ) أَيْ الْمُرْتَدَّ ( غَيْرُهُ ) أَيْ غَيْرُ الْإِمَامِ وَنَائِبِهِ ( بِلَا إذْنِهِ أَسَاءَ وَعُزِّرَ ) لِافْتِيَاتِهِ عَلَى الْإِمَامِ أَوْ نَائِبِهِ ( وَلَمْ يَضْمَنْ ) الْقَاتِلُ الْمُرْتَدَّ لِأَنَّهُ مَحَلٌّ غَيْرُ مَعْصُومٍ ( سَوَاءٌ قَتَلَهُ قَبْلَ الِاسْتِتَابَةِ أَوْ بَعْدَهَا ) لِأَنَّهُ مُهْدَرُ الدَّمِ فِي الْجُمْلَةِ وَرِدَّتُهُ مُبِيحَةٌ لِدَمِهِ وَهِيَ مَوْجُودَةٌ قَبْلَ الِاسْتِتَابَةِ كَمَا هِيَ مَوْجُودَةٌ بَعْدَهَا ( إلَّا أَنْ يَلْحَقَ ) الْمُرْتَدُّ ( بِدَارِ حَرْبٍ فَلِكُلِّ ) أَحَدٍ ( قَتْلُهُ ) بِلَا اسْتِتَابَةٍ ( وَأَخْذُ مَا مَعَهُ مِنْ مَالٍ ) لِأَنَّهُ صَارَ حَرْبِيًّا وَمَا تَرَكَهُ بِدَارِنَا مَعْصُومٌ نَصَّ عَلَيْهِ .( 21 / 122 )
