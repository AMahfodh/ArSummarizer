ثُمَّ رَأَيْته أَشَارَ بِهَذَا الْفَرْقِ بِالنِّسْبَةِ لِلْمُحَارِبِ بِقَوْلِهِ وَلَا يُرَدُّ إلَخْ فَكَانَ الْوَجْهُ أَنْ يُجْرِيَهُ هُنَا أَيْضًا ا ه سم وَلَك أَنْ تَخْتَارَ الْأَوَّلَ وَتَمْنَعَ قَوْلَهُ فَهَذَا نَظِيرُ إلَخْ بِأَنَّ مِنْ تَتِمَّةِ الْفَرْقِ إمْكَانُ التَّدَارُكِ هُنَا لَاثَمَّ ( قَوْلُهُ بَيْنَ هَذَيْنِ ) أَيْ الْمُرْتَدِّ وَالْجَانِي الْمُتَعَلِّقِ بِرَقَبَتِهِ قَوْدٌ ( قَوْلُهُ ثَمَّ ) أَيْ فِي مُسْرَعِ الْفَسَادِ ( قَوْلُهُ لَا هُنَا ) أَيْ فِي الْمُرْتَدِّ وَالْجَانِي ( قَوْلُهُ بِأَنَّ الْمَانِعَ إلَخْ ) مُتَعَلِّقٌ بِقَوْلِهِ وَيُفَرَّقُ ( قَوْلُهُ عَلَى مَا يَأْتِي ) أَيْ( 19 / 285 )
