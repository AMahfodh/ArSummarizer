وَخَرَجَ بِدُيُونِ الْمُعَامَلَةِ أَرْشُ الْجِنَايَةِ وَنَحْوِهَا مِنْ الْإِتْلَافَاتِ وَتَقَدَّمَ ( فَيُقَدِّمُهَا ) أَيْ دُيُونَ الْمُعَامَلَةِ عَلَى دَيْنِ كِتَابَتِهِ إنْ كَانَ ( مَحْجُورًا عَلَيْهِ ) بِأَنْ ضَاقَتْ دُيُونُهُ عَنْهَا وَسَأَلَ غُرَمَاؤُهُ الْحَاكِمَ الْحَجْرَ عَلَيْهِ فَحَجَرَ عَلَيْهِ ( بَعْدَ تَعَلُّقِهَا بِرَقَبَتِهِ ) أَيْ الْمُكَاتَبِ ( فَلِهَذَا إنْ لَمْ يَكُنْ بِيَدِهِ ) أَيْ الْمُكَاتَبِ ( مَالٌ فَلَيْسَ لِغَرِيمِهِ تَعْجِيزُهُ ) بِعَوْدِهِ إلَى الرِّقِّ ( بِخِلَافِ أَرْشِ ) جِنَايَةٍ لِتَعَلُّقِهِ بِرَقَبَتِهِ ( وَ ) بِخِلَافِ ( دَيْنِ كِتَابَةٍ ) لِأَنَّهُ بَدَلُ رَقَبَتِهِ ( وَيَشْتَرِطُ رَبُّ دَيْنٍ ) مُعَامَلَةً ( وَ ) رَبُّ ( أَرْشٍ ) جِنَايَةً فِي تَرِكَةِ مُكَاتَبٍ ( بَعْدَ مَوْتِهِ ) فَيَتَحَاصَّانِ لِفَوَاتِ الرَّقَبَةِ ( وَلِ ) لْمُكَاتَبِ ( غَيْرِ الْمَحْجُورِ عَلَيْهِ تَقْدِيمُ أَيِّ دَيْنٍ شَاءَ ) مِنْ دَيْنِ كِتَابَةٍ وَمُعَامَلَةٍ وَأَرْشِ جِنَايَةٍ كَالْحُرِّ .( 8 / 268 )
