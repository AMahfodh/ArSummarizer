( وَيَحْرُمُ نَظَرُ خَصِيٍّ وَمَجْبُوبٍ ) وَمَمْسُوحٍ ( إلَى ) امْرَأَةٍ ( أَجْنَبِيَّةٍ نَصًّا ) قَالَ الْأَثْرَمُ اسْتَعْظَمَ الْإِمَامُ أَحْمَدُ إدْخَالَ الْخُصْيَانِ عَلَى النِّسَاءِ لِأَنَّ الْعُضْوَ وَإِنْ تَعَطَّلَ أَوْ عُدِمَ فَشَهْوَةُ الرِّجَالِ لَا تُزَالُ مِنْ قُلُوبِهِمْ وَلَا يُؤْمَنُ التَّمَتُّعُ بِالْقُبْلَةِ وَغَيْرِهَا فَهُوَ ( كَفَحْلٍ ) وَلِذَلِكَ لَا تُبَاحُ خَلْوَةُ الْفَحْلِ بِالرَّتْقَاءِ مِنْ النِّسَاءِ .( 16 / 416 )
