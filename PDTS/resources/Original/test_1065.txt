( الْعَاشِرَةُ ) : قَوْلُهَا { فَإِذَا عِقْدٌ مِنْ جَزْعِ ظَفَارِ وَقَدْ انْقَطَعَ : } ( الْعِقْدُ ) : بِكَسْرِ الْعَيْنِ وَإِسْكَانِ الْقَافِ كُلُّ مَا يُعْقَدُ وَيُعَلَّقُ فِي الْعُنُقِ وَهُوَ نَحْوُ الْقِلَادَةِ وَ ( الْجَزْعُ ) : بِفَتْحِ الْجِيمِ وَإِسْكَانِ الزَّايِ وَآخِرُهُ عَيْنٌ مُهْمَلَةٌ خَرَزُ يَمَانٍ ( وَظَفَارِ ) : بِفَتْحِ الظَّاءِ الْمُعْجَمَةِ وَكَسْرِ الرَّاءِ قَرْيَةٌ بِالْيَمَنِ وَهِيَ مَبْنِيَّةٌ عَلَى الْكَسْرِ تَقُولُ هَذِهِ ظَفَارِ وَدَخَلَتْ ظَفَارِ وَإِلَى ظَفَارِ بِكَسْرِ الرَّاءِ بِلَا تَنْوِينٍ فِي الْأَحْوَالِ كُلِّهَا وَقَالَ أَبُو الْعَبَّاسِ الْقُرْطُبِيُّ هَكَذَا فِي صَحِيحِ الرِّوَايَةِ وَمَنْ قَيَّدَهُ جَزْعَ أَظْفَارٍ بِأَلْفٍ فَقَدْ أَخْطَأَ وَبِالْوَجْهِ الصَّحِيحِ رَوَيْتُهُ .
