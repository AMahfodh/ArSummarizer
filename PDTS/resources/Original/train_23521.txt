( وَإِنْ قَطَعَ إحْدَى يَدَيْهِ فَ ) عَلَيْهِ ( رُبْعُ الدِّيَةِ وَرُبْعُ قِيمَتِهِ وَيَكُونُ ) الْجَمِيعُ عَلَى الْجَانِي لِأَنَّ الْعَاقِلَةَ لَا تَحْمِلُ مَا دُونَ الثُّلُثِ وَلَا الْقِيمَةِ ( وَإِنْ قَطَعَ ) الْجَانِي ( خُصْيَتَيْهِ أَوْ ) قَطَعَ ( أَنْفَهُ أَوْ قَطَعَ ) أُذُنَيْهِ لَزِمَتْهُ قِيمَتُهُ لِلسَّيِّدِ لِأَنَّ الْقِيمَةَ بَدَلٌ عَنْ الدِّيَةِ فِي الْأَعْضَاءِ الْمَمْلُوكَةِ لِلسَّيِّدِ ( وَلَمْ يَزُلْ مِلْكُ السَّيِّد عَنْهُ ) لِأَنَّهُ لَمْ يُوجَدْ سَبَبٌ يَقْتَضِي الزَّوَالَ فَوَجَبَ بَقَاؤُهُ عَلَى مِلْكِهِ عَمَلًا بِاسْتِصْحَابِ الْحَالِ لِأَنَّ قَطْعَ بَعْضِ أَعْضَائِهِ بِمَنْزِلَةِ تَلَفِ بَعْضِ مَالِهِ ( وَإِنْ قَطَعَ ) الْجَانِي ( ذَكَرَهُ ) أَيْ الْقِنُّ ( ثُمَّ خَصَّاهُ لَزِمَتْهُ قِيمَتُهُ لِقَطْعِ الذَّكَرِ ) لِأَنَّ الْوَاجِبِ فِي غَيْرِ ذَلِكَ مِنْ الْحُرِّ دِيَةٌ كَامِلَةٌ .
