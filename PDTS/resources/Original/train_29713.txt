، وَلَا نِيَّةَ ) لَهُ تُخَالِفُ ظَاهِرَ أَلْفَاظِهِ ( لَمْ يَحْنَثْ ، وَلَوْ لَمْ يَعْرِفْ حَقِيقَتَهُمَا ) أَيْ : الْأَمْرِ وَالنَّهْيِ ؛ لِأَنَّهَا خَالَفَتْ نَهْيَهُ لَا أَمْرَهُ ، ( وَ ) إنْ قَالَ لَهَا ، ( إنْ نَهَيْتُك فَخَالَفْتنِي ) فَأَنْتِ طَالِقٌ ( فَأَمَرَهَا ) بِشَيْءٍ ( وَخَالَفَتْهُ ؛ لَمْ يَحْنَثْ فِي قِيَاسِ ) الْمَسْأَلَةِ ( الَّتِي قَبْلَهَا ) وَلَوْ لَمْ يَعْرِفْ حَقِيقَةَ الْأَمْرِ وَالنَّهْيِ ؛ لِأَنَّهَا خَالَفَتْ أَمْرَهُ ، لَا نَهْيَهُ ( إلَّا بِنِيَّةِ مُطْلَقِ الْمُخَالَفَةِ ) فَيَحْنَثُ بِمُخَالَفَةِ الْأَمْرِ ؛ لِأَنَّهَا مُخَالَفَةٌ .
