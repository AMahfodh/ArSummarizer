( وَنُقِلَ عَنْهُ ) ؛ أَيْ : الْإِمَامُ أَحْمَدُ أَنَّهُ قَالَ : ( لَا يُعْجِبُنِي ) ؛ أَيْ : أَنْ يَحْلِفَ عَلَى مُخْتَلَفٍ فِيهِ لَا يَعْتَقِدُهُ ، فَلَوْ بَاعَ شَافِعِيٌّ حَنْبَلِيًّا لَحْمًا مَتْرُوكَ التَّسْمِيَةِ بِدِينَارٍ مَثَلًا ، ثُمَّ ادَّعَى عَلَيْهِ بِهِ فَأَجَابَ الْحَنْبَلِيُّ بِأَنْ لَا حَقَّ لَهُ عَلَيَّ ، فَالْتَمَسَ الْمُدَّعِي يَمِينَهُ عَلَى حَسَبِ جَوَابِهِ ، فَمُقْتَضَى الْإِمَامِ لَا يَحْلِفُ ؛ لِأَنَّهُ يَقْطَعُ بِهَذِهِ مَا يَعْتَقِدُهُ الْمُدَّعِي مَا لَا عِنْدَهُ ( وَتَوَقَّفَ ) الْإِمَامُ أَحْمَدُ ( فِيهَا ) أَيْ : فِي الْيَمِينِ ( فِيمَنْ عَامَلَ بِحِيلَةٍ رِبَوِيَّةٍ كَعِينَةٍ ) إذَا أَنْكَرَ الْآخِذُ الزِّيَادَةَ وَأَرَادَ الْحَلِفَ عَلَيْهَا هَلْ يَحْلِفُ أَنَّ مَا عَلَيْهِ إلَّا رَأْسُ مَالِهِ ؟ نَقَلَهُ حَرْبٌ قَالَ الْقَاضِي ؛ لِأَنَّ يَمِينَهُ هُنَا عَلَى الْقَطْعِ ، وَمَسَائِلُ الِاجْتِهَادِ ظَنِّيَّةٌ ( فَلَوْ أُبْرِئَ ) مُدَّعًى عَلَيْهِ ( مِنْهَا ) ؛ أَيْ : الْيَمِينِ ؛ بِأَنْ قَالَ لَهُ مُدَّعٍ : أَبْرَأْتُك مِنْ الْيَمِينِ بَرِئَ الْمُدَّعَى عَلَيْهِ ( مِنْهَا ) ؛ أَيْ : الْيَمِينِ ( فِي هَذِهِ الدَّعْوَى فَقَطْ ) فَلَيْسَ لَهُ تَحْلِيفُهُ فِيهَا لِإِسْقَاطِهِ ( فَلَوْ جَدَّدَهَا ) ؛ أَيْ : اسْتَأْنَفَ الدَّعْوَى عَلَيْهِ ، فَأَنْكَرَ ( وَطَلَبَ ) الْمُدَّعِي ( الْيَمِينَ كَانَ لَهُ ذَلِكَ ) لِعَدَمِ مَا يُسْقِطُهُ ، فَإِذَا حَلَفَ لَمْ يَحْلِفْ مَرَّةً أُخْرَى ( وَلَوْ أَمْسَكَ( 19 / 378 )
