( وَلَوْ خَرَجَ عَبْدٌ ) حَرْبِيٌّ ( إلَيْنَا بِأَمَانٍ أَوْ نَزَلَ ) عَبْدٌ ( مِنْ حِصْنٍ ) إلَيْنَا بِأَمَانٍ ( فَهُوَ حُرٌّ ) نَصًّا لِلْخَبَرِ ( وَلَوْ جَاءَنَا ) عَبْدٌ ( مُسْلِمًا ) وَأُسِرَ سَيِّدُهُ الْحَرْبِيُّ ( أَوْ ) أُسِرَ ( غَيْرُهُ ) مِنْ الْحَرْبِيِّينَ ( فَهُوَ ) أَيْ الْعَبْدُ ( حُرٌّ ) لِمَا تَقَدَّمَ .
