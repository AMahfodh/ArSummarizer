) يَكْفِي أَيْضًا ( رَهْنُهَا ) لِأَنَّ مَنْعَهُ مِنْ وَطْئِهَا لِحَقِّ الْمُرْتَهِنِ لَا لِتَحْرِيمِهَا وَلِذَلِكَ يَجُوزُ لَهُ وَطْؤُهَا بِإِذْنِ الْمُرْتَهِنِ وَلِأَنَّهُ يَقْدِرُ عَلَى فَكّهَا مَتَى شَاءَ ( وَلَا ) يَكْفِي أَيْضًا ( بَيْعُهَا بِشَرْطِ خِيَارٍ ) لَهُ لِأَنَّهُ يَقْدِرُ عَلَى اسْتِرْجَاعِهَا مَتَى شَاءَ يَفْسَخُ الْبَيْعَ ( وَمِثْلِهِ ) أَيْ مِثْلِ الْبَيْعِ بِشَرْطِ خِيَارٍ لَهُ فِي عَدَمِ الِاكْتِفَاءِ بِهِ ( هِبَتُهَا ) أَيْ الْمَوْطُوءَةِ ( لِمَنْ يَمْلِكُ اسْتِرْجَاعَهَا مِنْهُ كَهِبَتِهَا لِوَلَدِهِ ) .
