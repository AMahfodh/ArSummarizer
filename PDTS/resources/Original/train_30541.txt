( وَ ) عُيِّنَتْ ( دَابَّةٌ ) لِرُكُوبٍ أَوْ حَمْلٍ وَلَا يَكْفِي الْوَصْفُ فِيهَا ( إلَّا ) الدَّابَّةَ ( الْمَضْمُونَةَ ) فِي الذِّمَّةِ لِيُتَوَصَّلَ بِهَا إلَى مَحَلٍّ كَمَكَّةَ وَذَلِكَ بِأَنْ لَمْ تُقْصَدْ عَيْنُهَا : ( فَنَوْعٌ ) أَيْ : فَالْوَاجِبُ تَعْيِينُ نَوْعِهَا كَإِبِلٍ أَوْ بِغَالٍ ( وَصِنْفٌ ) كَعِرَابٍ أَوْ بُخْتٍ ( وَذُكُورَةٌ وَأُنُوثَةٌ ) .
