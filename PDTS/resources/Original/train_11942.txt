( وَفِي ) قَوْلِ الْمُدَبَّرِ ( كَسَبْتُ الْمَالَ ) الَّذِي بِيَدِي ( بَعْدَ ) مَوْتِ ( سَيِّدِي ) فَهُوَ لِي وَقَالَ الْوَارِثُ : قَبْلَهُ فَهُوَ تَرِكَةٌ حَلَفَ الْمُدَبِّرُ ؛ لِأَنَّ الْيَدَ لَهُ ، وَإِنْ أَقَامَا بَيِّنَتَيْنِ بِمَا قَالَاهُ قُدِّمَتْ بَيِّنَتُهُ لِذَلِكَ ( لَا فِي ) قَوْلِ الْمُدَبَّرَةِ ( وَلَدْتُ ) هَذَا الْوَلَدَ بَعْدَ مَوْتِ سَيِّدِي فَهُوَ حُرٌّ ، وَقَالَ الْوَارِثُ قَبْلَ التَّدْبِيرِ فَهُوَ قِنٌّ فَلَا تَحْلِفُ الْمُدَبَّرَةُ بَلْ الْوَارِثُ ؛ لِأَنَّ الْأَصْلَ رِقُّ وَلَدِهَا ، فَقَوْلُهُ : فِي كَسَبْت مُتَعَلِّقٌ بِقَوْلِهِ ( حَلِّفْ الْمُدَبَّرَا ) وَذَكَرَ مِنْ زِيَادَتِهِ الْفَرْقَ بَيْنَ الصُّورَتَيْنِ بِقَوْلِهِ : ( إذْ مَا عَلَى الْحُرِّ يَدٌ فَتَظْهَرَا ) أَيْ : إذْ الْحُرُّ لَا يَدْخُلُ تَحْتَ الْيَدِ حَتَّى تَظْهَرَ عَلَيْهِ بِخِلَافِ الْمَالِ لَكِنْ لَا حَاجَةَ فِي الْفَرْقِ إلَى قَوْلِهِ : فَتَظْهَرَا .
