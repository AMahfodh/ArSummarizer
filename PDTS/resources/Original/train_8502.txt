( وَإِنْ عُدِمَتْ النِّيَّةُ مِنْهُمَا أَوْ نَوَى كُلٌّ مِنْهُمَا عَنْ الْآخَرِ لَمْ يَصِحَّ ) الطَّوَافُ ( لِوَاحِدٍ مِنْهُمَا ) لِخُلُوِّ طَوَافِ كُلٍّ مِنْهُمَا عَنْ نِيَّةٍ مِنْهُ ( وَإِنْ حَمَلَهُ بِعَرَفَاتٍ ) لِعُذْرٍ أَوْ لَا ( أَجْزَأَ ) الْوُقُوفُ ( عَنْهُمَا ) ؛ لِأَنَّ الْمَقْصُودَ الْحُصُولُ بِعَرَفَةَ وَهُوَ مَوْجُودٌ .( 7 / 180 )
