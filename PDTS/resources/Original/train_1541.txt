( وَتُؤُوِّلَتْ ) بِضَمِّ الْمُثَنَّاةِ وَالْهَمْزِ وَكَسْرِ الْوَاوِ مُثَقَّلَةً ، أَيْ حُمِلَتْ الْمُدَوَّنَةُ ( عَلَى أَنَّهُ ) أَيْ الشَّأْنَ ( إنَّمَا يَلْزَمُ الطَّلَاقُ ) فِيمَنْ تَزَوَّجَهَا مِنْ غَيْرِهَا ( إذَا تَزَوَّجَ مِنْ غَيْرِهَا ) أَيْ الْمَدِينَةِ ( قَبْلَ ) تَزَوُّجِهِ مِنْ ( هَا ) أَيْ الْمَدِينَةِ .
