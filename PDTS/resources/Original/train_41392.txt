( وَ ) لَوْ تَزَوَّجَ حُرٌّ أَمَةً بِشَرْطِهِ ، ثُمَّ تَزَوَّجَ عَلَيْهَا بِحُرَّةٍ وَلَمْ تَعْلَمْ بِهَا ( خُيِّرَتْ الْحُرَّةُ مَعَ ) الزَّوْجِ ( الْحُرِّ ) لَا الْعَبْدِ ( فِي نَفْسِهَا ) بَيْنَ أَنْ تُقِيمَ مَعَ الْأَمَةِ أَوْ تُفَارِقَ ( بِطَلْقَةٍ ) وَاحِدَةٍ ( بَائِنَةٍ ) صِفَةٌ كَاشِفَةٌ ، إذْ هُوَ كَطَلَاقِ الْحَاكِمِ فَإِنْ أُوقِعَتْ أَكْثَرَ لَمْ يَلْزَمْ إلَّا وَاحِدَةٌ ( كَتَزْوِيجِ أَمَةٍ عَلَيْهَا ) عَكْسُ مَا قَبْلَهُ ( أَوْ ) تَزْوِيجِ أَمَةٍ ( ثَانِيَةٍ ) عَلَى الَّتِي رَضِيَتْ بِهَا الْحُرَّةُ ( أَوْ عَلِمَهَا ) أَيْ الْحُرَّةَ ( بِوَاحِدَةٍ فَأَلْفَتْ أَكْثَرَ ) فَتُخَيَّرُ فِي نَفْسِهَا فِي الصُّوَرِ الثَّلَاثِ بِطَلْقَةٍ .
