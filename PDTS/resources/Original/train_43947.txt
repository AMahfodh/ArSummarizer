( وَتُجْزِئُ عُمْرَةُ الْقَارِنِ ) عَنْ عُمْرَةِ الْإِسْلَامِ ( وَ ) تُجْزِئُ ( عُمْرَةٌ ) مِنْ ( التَّنْعِيمِ ) عَنْ عُمْرَةِ الْإِسْلَامِ لِحَدِيثِ { عَائِشَةَ حِينَ قَرَنَتْ الْحَجَّ وَالْعُمْرَةَ فَقَالَ لَهَا النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ حِينَ حَلَّتْ مِنْهُمَا قَدْ حَلَلْتِ مِنْ حَجِّكِ وَعُمْرَتِكِ } وَإِنَّمَا أَعْمَرَهَا مِنْ التَّنْعِيمِ قَصْدًا لِتَطْيِيبِ خَاطِرِهَا وَإِجَابَةِ مَسْأَلَتِهَا لَا ؛ لِأَنَّهَا كَانَتْ وَاجِبَةً عَلَيْهَا .( 7 / 336 )
