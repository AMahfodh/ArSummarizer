( وَ ) لَزِمَ ( تَعْمِيمُ وَجْهِهِ ) بِالْمَسْحِ وَلَوْ بِيَدٍ وَاحِدَةٍ أَوْ أُصْبُعَ وَيَدْخُلُ فِيهِ اللِّحْيَةُ وَلَوْ طَالَتْ وَتُرَاعَى الْوَتْرَةُ وَمَا غَارَ مِنْ الْعَيْنِ وَلَا يَتَتَبَّعُ الْغُضُونَ ( وَ ) لَزِمَ تَعْمِيمُ ( كَفَّيْهِ ) الْأَوْلَى يَدَيْهِ ( لِكُوعَيْهِ ) مَعَ تَخْلِيلِ أَصَابِعِهِ عَلَى الرَّاجِحِ لَكِنْ بِبَطْنِ أُصْبُعٍ أَوْ أَكْثَرَ لَا بِجَنْبِهِ إذْ لَمْ يَمَسَّهُ صَعِيدٌ ( وَ ) يَلْزَمُ ( نَزْعُ خَاتَمِهِ ) وَلَوْ مَأْذُونًا فِيهِ أَوْ وَاسِعًا وَإِلَّا كَانَ حَائِلًا
