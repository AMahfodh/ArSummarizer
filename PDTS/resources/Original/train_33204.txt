( وَلَوْ وُطِئَتْ مَنْكُوحَةٌ بِشُبْهَةٍ أَوْ وَطِئَ اثْنَانِ ) امْرَأَةً ( بِشُبْهَةٍ فَوَلَدَتْ ) بَعْدَ وَطْئِهَا وَلَدًا ( فَاللَّبَنُ ) النَّازِلُ بِهِ ( لِمَنْ لَحِقَهُ الْوَلَدُ ) مِنْهُمَا ( بِقَائِفٍ ) لِإِمْكَانِهِ مِنْهُمَا ( أَوْ غَيْرِهِ ) كَانْحِصَارِ الْإِمْكَانِ فِيهِ وَكَانْتِسَابِ الْوَلَدِ أَوْ فَرْعِهِ بَعْدَ مَوْتِهِ إلَيْهِ بَعْدَ كَمَالِهِ لِفَقْدِ الْقَائِفِ أَوْ غَيْرِهِ وَيَجِبُ ذَلِكَ فَيُجْبَرُ عَلَيْهِ حِفْظًا لِلنَّسَبِ مِنْ الضَّيَاعِ وَلَوْ انْتَسَبَ بَعْضُ فُرُوعِهِ لِوَاحِدٍ وَبَعْضُهُمْ لِآخَرَ دَامَ الْإِشْكَالُ فَإِنْ مَاتُوا أَوْ لَمْ يَكُنْ لَهُ وَلَدٌ انْتَسَبَ الرَّضِيعُ إنْ شَاءَ وَقِيلَ ذَلِكَ لَا يَحِلُّ لَهُ بِنْتُ أَحَدِهِمَا وَنَحْوُهَا
