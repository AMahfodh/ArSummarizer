( أَوْ ) لِكَوْنِهِ ( حِصَّةً ) مَعَ شَرِيكٍ فَيُبَاعُ لِيَشْتَرِيَ لَهُ كَامِلًا لِلسَّلَامَةِ مِنْ ضَرَرِ الشَّرِكَةِ ( أَوْ قَلَّتْ غَلَّتُهُ ) : وَأَوْلَى إذَا لَمْ يَكُنْ لَهُ غَلَّةٌ فَيُبَاعُ لِيَسْتَبْدِلَ لَهُ مَا فِيهِ غَلَّةٌ كَثِيرَةٌ ( أَوْ ) كَانَ ( بَيْنَ ذِمِّيِّينَ أَوْ جِيرَانِ سُوءٍ أَوْ ) كَانَ ( فِي مَحَلِّ خَوْفٍ ) فَيُبَاعُ لِيَشْتَرِيَ لَهُ غَيْرَهُ فِي مَكَان غَيْرِ مَا ذَكَرَ ( أَوْ ) كَانَ شَرِكَةً فَيُبَاعُ ( لِإِرَادَةِ شَرِيكِهِ بَيْعًا ) لِنَصِيبِهِ ( وَلَا مَالَ لَهُ ) : أَيْ لِلْيَتِيمِ يَشْتَرِي بِهِ مَنَابَ الشَّرِيكِ ، فَيُبَاعُ حِصَّةُ الْيَتِيمِ مَعَ( 7 / 369 )
