( فَصْلٌ وَشُرُوطُهُ ) أَيْ : الْوَقْفِ ( أَرْبَعَةٌ ) أَحَدُهَا ( مُصَادَفَتُهُ عَيْنًا يَصِحُّ بَيْعُهَا وَيُنْتَفَعُ بِهَا ) انْتِفَاعًا ( عُرْفًا كَإِجَارَةٍ ) بِأَنْ يَكُونَ النَّفْعُ مُبَاحًا بِلَا ضَرُورَةٍ ، مَقْصُودًا مُتَقَوِّمًا يُسْتَوْفَى ( مَعَ بَقَائِهَا ) أَيْ : الْعَيْنِ ؛ لِأَنَّهُ يُرَادُ لِلدَّوَامِ لِيَكُونَ صَدَقَةً جَارِيَةً .
