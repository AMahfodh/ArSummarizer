قَالَ الْمُصَنِّفُ ( وَلِأَنَّهُ إهْلَاكٌ مَعْنًى ) هُوَ مِنْ قَوْلِ عَلِيٍّ رَضِيَ اللَّهُ عَنْهُ : قَتَلْته إذًا ( وَالْحَدُّ زَاجِرٌ ) لَا مُهْلِكٌ ( وَلِأَنَّهُ نَادِرُ الْوُجُودِ ) أَيْ يَنْدُرُ أَنْ يَسْرِقَ الْإِنْسَانُ بَعْدَ قَطْعِ يَدِهِ وَرِجْلِهِ ( وَالْحَدُّ ) لَا يُشْرَعُ إلَّا ( فِيمَا يَغْلِبُ ) عَلَى مَا مَرَّ غَيْرَ مَرَّةٍ ( بِخِلَافِ الْقِصَاصِ ) يَعْنِي لَوْ قَطَعَ رَجُلٌ يَدَيَّ رَجُلٍ قُطِعَتْ يَدَاهُ أَوْ أَرْبَعَتَهُ قُطِعَتْ أَرْبَعَتُهُ ( لِأَنَّهُ حَقٌّ لِلْعَبْدِ( 12 / 288 )
