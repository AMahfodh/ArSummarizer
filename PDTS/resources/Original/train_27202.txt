بَاعَ بِمَا ذُكِرَ لِأَجَلٍ فَهُوَ الْمُتَقَدِّمُ فِي قَوْلِهِ ، وَإِنْ أَمَرَ بِبَيْعِ سِلْعَةٍ إلَخْ ( أَوْ أَنْكَرَ ) الْوَكِيلُ ( الْقَبْضَ ) لِمَا وَكَّلَ عَلَى قَبْضِهِ ( فَقَامَتْ ) عَلَيْهِ ( الْبَيِّنَةُ ) بِهِ ( فَشَهِدَتْ ) لَهُ ( بَيِّنَةٌ بِالتَّلَفِ ) لِلْمَقْبُوضِ أَوْ بِالرَّدِّ إنْ ادَّعَاهُ فَيَضْمَنُ وَلَا تَنْفَعُهُ بَيِّنَتُهُ بِذَلِكَ ؛ لِأَنَّهُ أَكْذَبَهَا بِإِنْكَارِهِ الْقَبْضَ ( كَالْمِدْيَانِ ) يُنْكِرُ مَا عَلَيْهِ مِنْ الدَّيْنِ فَتَقُومُ الْبَيِّنَةُ عَلَيْهِ بِهِ فَيَدَّعِي الدَّفْعَ وَيُقِيمُ بَيِّنَةً بِهِ فَيَغْرَمُ وَلَا تُسْمَعُ دَعْوَاهُ ؛ لِأَنَّهُ أَكْذَبَهَا كَمَا سَيَأْتِي فِي الْقَضَاءِ فِي قَوْلِهِ : وَإِنْ أَنْكَرَ مَطْلُوبُ الْمُعَامَلَةِ فَالْبَيِّنَةُ ثَمَّ لَا تُسْمَعُ بَيِّنَتُهُ بِالْقَضَاءِ بِخِلَافِ لَا حَقَّ لَك عَلَيَّ
