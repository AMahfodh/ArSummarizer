( وَ ) يَقْرَأُ الْمَأْمُومُ نَدْبًا ( مَعَ الْفَاتِحَةِ سُورَةً فِي أُولَتَيْ ظُهْرٍ وَعَصْرٍ ) لِمَا تَقَدَّمَ عَنْ جَابِرٍ وَعَلِيٍّ ( فَإِنْ سَمِعَ ) الْمَأْمُومُ ( قِرَاءَةَ الْإِمَامِ كُرِهَتْ لَهُ الْقِرَاءَةُ ) لِلْفَاتِحَةِ وَالسُّورَةِ لِمَا تَقَدَّمَ وَفِيهِ تَكْرَارٌ إلَّا أَنْ يُحْمَلَ هَذَا الْأَخِيرُ عَلَى السِّرِّيَّةِ ، وَمَا تَقَدَّمَ عَلَى الْجَهْرِيَّةِ ( فَلَوْ سَمِعَ ) الْمَأْمُومُ ( هَمْهَمَتَهُ وَلَمْ يَفْهَمْ مَا يَقُولُ ) الْإِمَامُ ( لَمْ يَقْرَأْ ) ؛ لِأَنَّهُ سَامِعٌ لِقِرَاءَةِ إمَامِهِ .( 3 / 389 )
