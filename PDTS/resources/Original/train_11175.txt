( وَإِنْ كَانَ النَّظَرُ لِمَوْقُوفٍ عَلَيْهِ ، إمَّا بِجَعْلِهِ ) أَيْ : الْوَاقِفِ النَّظَرَ ( لَهُ ) أَيْ : الْمَوْقُوفِ عَلَيْهِ ( أَوْ لِكَوْنِهِ ) أَيْ : الْمَوْقُوفِ عَلَيْهِ ( أَحَقَّ ) بِالنَّظَرِ ( لِعَدَمِ ) تَعْيِينِ ( غَيْرِهِ ؛ فَهُوَ ) أَيْ : الْمَوْقُوفُ عَلَيْهِ ( مَعَ رُشْدٍ أَحَقُّ ) بِالنَّظَرِ ( مُطْلَقًا ) أَيْ : عَدْلًا كَانَ ، أَوْ فَاسِقًا ، رَجُلًا أَوْ امْرَأَةً ، لِأَنَّهُ يَمْلِكُ الْوَقْفَ ، فَهُوَ يَنْظُرُ لِنَفْسِهِ ( وَإِلَّا ) بِأَنْ كَانَ الْمَوْقُوفُ عَلَيْهِ غَيْرَ رَشِيدٍ ، وَلَمْ يُشْرَطْ النَّظَرُ لِغَيْرِهِ ( فَوَلِيُّهُ ) يَقُومُ بِالنَّظَرِ مَقَامَهُ ؛ لِأَنَّهُ يَمْلِكُهُ ، فَهُوَ كَمِلْكِهِ الطَّلْقِ .( 12 / 378 )
