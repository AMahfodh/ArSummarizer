( إذَا ضَرَّ ) النَّاسَ ( إجْدَابُ أَرْضٍ ) ، يُقَالُ : أَجْدَبَ الْقَوْمُ ، إذَا أَمْحَلُوا ( أَوْ ) ضَرَّهُمْ ( قَحْطُ مَطَرٍ ) ، أَيْ : احْتِبَاسُهُ ( عَنْ أَرْضٍ مَسْكُونَةٍ أَوْ مَسْلُوكَةٍ ) لِعَدَمِ الضَّرَرِ فِي غَيْرِهِمَا ( وَلَوْ ) ضَرَّ ( غَيْرَ أَرْضِهِمْ ) ، لِحُصُولِ الضَّرَرِ بِهِ ( أَوْ ) ضَرَّهُمْ ( غَوْرُ مَاءِ عُيُونٍ ) فِي الْأَرْضِ ، ( أَوْ ) ضَرَّهُمْ غَوْرُ مَاءِ ( أَنْهَارٍ ) جَمْعُ : نَهْرٍ ، بِفَتْحِ الْهَاءِ وَسُكُونِهَا : مَجْرَى الْمَاءِ .
