( وَإِنْ أَجَّرَ سَيِّدٌ رَقِيقَهُ ؛ أَوْ ) أَجَّرَ ( وَلِيَّ يَتِيمًا ) مَحْجُورًا لَهُ ( أَوْ ) أَجَّرَ ( مَالِهِ ) أَيْ : مَالِ مَحْجُورِهِ كَدَارِهِ أَوْ رَقِيقِهِ أَوْ بَهَائِمِهِ ( ثُمَّ عَتَقَ ) الرَّقِيقُ ( الْمَأْجُورُ أَوْ بَلَغَ ) الْيَتِيمُ ( وَرَشَدَ أَوْ مَاتَ ) السَّيِّدُ أَوْ الْوَلِيُّ ( الْمُؤَجِّرُ ) قَبْلَ مُضِيِّ مُدَّةِ الْإِجَارَةِ ( أَوْ عُزِلَ ) الْوَلِيُّ بِأَنْ أَقَامَ الْحَاكِمُ غَيْرَهُ ( لَمْ تَنْفَسِخْ ) الْإِجَارَةُ .
