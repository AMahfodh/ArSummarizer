( وَلَوْ مَنَعَهُ ) أَيْ مَنَعَ الْمَوْهُوبُ لَهُ الْمَوْهُوبَ عَنْ الْوَاهِبِ بَعْدَ أَنْ يَرْجِعَ قَبْلَ الْقَضَاءِ أَوْ بَعْدَهُ ( فَهَلَكَ ) الْمَوْهُوبُ فِي يَدِ الْمَوْهُوبِ لَهُ ( لَا يَضْمَنُ ) ؛ لِأَنَّ يَدَهُ غَيْرُ مَضْمُونَةٍ إلَّا إذَا طَلَبَهُ بَعْدَ الْقَضَاءِ فَمَنَعَهُ مَعَ الْقُدْرَةِ عَلَى التَّسْلِيمِ فَحِينَئِذٍ يَكُونُ يَدُهُ يَدَ ضَمَانٍ لِمَنْعِهِ طَلَبَهُ فَظَهَرَ الْفَرْقُ بَيْنَ الْمَنْعِ بَعْدَ الرُّجُوعِ وَبَيْنَ الْمَنْعِ بَعْدَ الطَّلَبِ ( وَهُوَ ) أَيْ الرُّجُوعُ ( مَعَ أَحَدِهِمَا ) أَيْ مَعَ التَّرَاضِي أَوْ قَضَاءِ الْقَاضِي ( فَسْخٌ ) لِعَقْدِ الْهِبَةِ ( مِنْ الْأَصْلِ ) ، أَوْ إعَادَةٌ لِلْمِلْكِ الْقَدِيمِ ( لَا هِبَةٌ مِنْ الْمَوْهُوبِ لَهُ ) وَعِنْدَ زُفَرَ الرُّجُوعُ بِالتَّرَاضِي عَقْدٌ جَدِيدٌ فَيُجْعَلُ بِمَنْزِلَةِ الْهِبَةِ الْمُبْتَدَأَةِ .
