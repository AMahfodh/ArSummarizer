( وَإِنْ زَرَعَ ) الْغَاصِبُ ( الْأَرْضَ ) الْمَغْصُوبَةَ ( فَلَيْسَ لِرَبِّهَا ) أَيْ : الْأَرْضِ إذَا رُدَّتْ ( بَعْدَ حَصْدِ ) الزَّرْعِ ( إلَّا الْأُجْرَةُ ) أَيْ : أُجْرَةُ الْمِثْلِ مِنْ وَضْعِ يَدِهِ عَلَى الْأَرْضِ إلَى رَدِّهَا ، وَلَيْسَ لَهُ تَمَلُّكُ الزَّرْعِ بَعْدَ حَصَادِهِ ؛ لِأَنَّهُ انْفَصَلَ عَنْ مِلْكِهِ كَمَا لَوْ غَرَسَ فِيهَا غَرْسًا ثُمَّ قَلَعَهُ .( 6 / 344 )
