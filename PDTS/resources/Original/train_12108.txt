( وَلَوْ أَقَرَّ فِي صِحَّتِهِ بِدَيْنٍ ) لِإِنْسَانٍ ( وَفِي مَرَضِهِ ) بِدَيْنٍ ( لِآخَرَ لَمْ يُقَدَّمْ الْأَوَّلُ ) بَلْ يَتَسَاوَيَانِ كَمَا لَوْ ثَبَتَا بِالْبَيِّنَةِ ( وَلَوْ أَقَرَّ فِي صِحَّتِهِ أَوْ مَرَضِهِ ) بِدَيْنٍ لِإِنْسَانٍ أَوْ ثَبَتَ بِبَيِّنَةٍ ( وَأَقَرَّ وَارِثُهُ بَعْدَ مَوْتِهِ ) بِدَيْنٍ ( لِآخَرَ لَمْ يُقَدَّمْ الْأَوَّلُ فِي الْأَصَحِّ ) ؛ لِأَنَّ إقْرَارَ الْوَارِثِ كَإِقْرَارِ الْمُوَرِّثِ ؛ لِأَنَّهُ خَلِيفَتُهُ فَكَأَنَّهُ أَقَرَّ بِدَيْنَيْنِ .
