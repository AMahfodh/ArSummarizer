وَكَذَا قَوْلُهُ : ( أَوْ يَكُونُ خَالِقٌ ) بِالرَّفْعِ عَلَى أَنْ يَكُونَ تَامَّةً ( لِشَيْءٍ إلَّا هُوَ ) رَدٌّ عَلَى الْمُعْتَزِلَةِ أَيْضًا دَلِيلُهُ قَوْله تَعَالَى { لَا إلَهَ إلَّا هُوَ خَالِقُ كُلِّ شَيْءٍ } } وَفِيهِ عُمُومٌ أُرِيدَ بِهِ الْخُصُوصُ ، إذْ يَخْرُجُ مِنْهُ ذَاتُهُ وَصِفَاتُهُ وَأَسْمَاؤُهُ سُبْحَانَهُ وَتَعَالَى ( رَبُّ الْعِبَادِ ) أَيْ خَالِقُهُمْ وَسَيِّدُهُمْ ( وَرَبُّ أَعْمَالِهِمْ وَالْمُقَدِّرُ لِحَرَكَاتِهِمْ ) وَسَكَنَاتِهِمْ ( وَآجَالِهِمْ ) جَمْعُ أَجَلٍ وَهُوَ مُدَّةُ الشَّيْءِ وَوَقْتُهُ ، وَهَذَا رَدٌّ عَلَى الْقَدَرِيَّةِ الْقَائِلِينَ بِأَنَّ الْقَاتِلَ قَدْ قَطَعَ عَلَى الْمَقْتُولِ أَجَلَهُ وَمَا قَالُوهُ بَاطِلٌ بَلْ هُوَ مَيِّتٌ بِأَجَلِهِ .
