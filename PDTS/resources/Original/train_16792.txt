وَإِنَّمَا يَصِحُّ إنْ أُرِيدَ الْوُجُوبُ وَإِنْ لَمْ يَأْمُرْهُ بِهِ السَّيِّدُ وَإِلَّا فَالْقِنُّ يَلْزَمُهُ الِاكْتِسَابُ لِلسَّيِّدِ حَيْثُ أَمْكَنَهُ وَطَلَبَهُ مِنْهُ ( وَالْأَصَحُّ وُجُوبُ إجَارَةٍ ) نَحْوِ ( أُمِّ وَلَدِهِ وَ ) نَحْوِ ( الْأَرْضِ ) الْمُوصَى لَهُ بِمَنْفَعَتِهَا أَوْ ( الْمَوْقُوفَةِ عَلَيْهِ ) حَيْثُ لَمْ يُخَالِفْ شَرْطَ الْوَاقِفِ مَرَّةً( 20 / 198 )
