وَلَوْ ( قَالَ إنْ ابْتَلَعْت شَيْئًا فَأَنْت طَالِقٌ طَلُقَتْ بِابْتِلَاعِ رِيقِهَا إلَّا إنْ أَرَادَ شَيْئًا غَيْرَهُ ) فَلَا تَطْلُقُ بِذَلِكَ عَمَلًا بِإِرَادَتِهِ ( أَوْ ) إنْ ابْتَلَعَتْ ( الرِّيقَ طَلُقَتْ بِكُلِّ رِيقٍ ) أَيْ رِيقِهَا أَوْ رِيقِ غَيْرِهَا ( فَإِنْ أَرَادَ غَيْرَ رِيقِهَا ) أَيْ رِيقِ غَيْرِهَا ( دُيِّنَ ) وَلَمْ يُقْبَلْ ظَاهِرٌ أَوْ أَرَادَ رِيقَهَا قُبِلَ صَرَّحَ بِهِ الْأَصْلُ ( وَإِنْ عَلَّقَ ) الطَّلَاقَ ( بِضَرْبِهَا فَضَرَبَ غَيْرَهَا ) وَلَمْ يَعْلَمْ قَصْدَهُ ( فَأَصَابَهَا ) ضَرْبُهُ ( طَلُقَتْ وَلَا يُصَدَّقُ ) فِي ( أَنَّهُ قَصَدَ غَيْرَهَا ) لِأَنَّ الضَّرْبَ يَقِينٌ وَهُوَ قَادِرٌ عَلَى إظْهَارِ قَصْدِهِ قَبْلَ الضَّرْبِ نَعَمْ إنْ دَلَّتْ قَرِينَةٌ ظَاهِرَةٌ عَلَى تَصْدِيقِهِ كَأَنْ رَجَمَ ابْنَهُ أَوْ عَبْدَهُ بِحَجَرٍ وَهِيَ غَائِبَةٍ فَبَرَزَتْ مِنْ بَابِ الْبَيْتِ مَثَلًا فَأَصَابَهَا صُدِّقَ قَالَهُ الْأَذْرَعِيُّ أَمَّا إذَا عَلِمْنَا أَنَّهُ قَصَدَ ضَرْبَ غَيْرِهَا فَلَا تَطْلُقُ كَالْمُكْرَهِ .
