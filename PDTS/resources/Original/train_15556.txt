وَذَكَرَهُ الْمُصَنِّفُ هُنَا اقْتِدَاءً بِالْمُزَنِيِّ وَغَيْرِهِ ، فَقَالَ : ( الْفَيْءُ : مَالٌ ) أَوْ نَحْوُهُ كَكَلْبٍ يُنْتَفَعُ بِهِ ( حَصَلَ ) لَنَا ( مِنْ كُفَّارٍ ) مِمَّا هُوَ لَهُمْ ( بَلَا قِتَالٍ وَ ) لَا ( إيجَافِ ) أَيْ إسْرَاعِ ( خَيْلٍ وَ ) لَا سَيْرِ ( رِكَابٍ ) أَيْ إبِلٍ وَنَحْوِهَا كَبِغَالٍ وَحَمِيرٍ وَسُفُنٍ وَرَجَّالَةٍ ، وَخَرَجَ بِزِيَادَةِ لَنَا مَا حَصَّلَهُ أَهْلُ الذِّمَّةِ مِنْ أَهْلِ الْحَرْبِ فَإِنَّهُ لَا يُنْزَعُ مِنْهُمْ وَبِزِيَادَةِ مِمَّا هُوَ لَهُمْ مَا أَخَذُوهُ مِنْ مُسْلِمٍ أَوْ ذِمِّيٍّ أَوْ نَحْوِهِ بِغَيْرِ حَقٍّ فَإِنَّا لَمْ نَمْلِكْهُ بَلْ يُرَدُّ عَلَى مَالِكِهِ إنْ عُرِفَ وَإِلَّا فَيُحْفَظُ .
