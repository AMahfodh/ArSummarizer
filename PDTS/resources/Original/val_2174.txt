( وَلَوْ قَالَ ) الْأَصْلُ ( أَشْهَدْته ) أَيْ الْفَرْعَ ( وَغَلِطْت ضَمِنَ عِنْدَ مُحَمَّدٍ ) ؛ لِأَنَّ الْفُرُوعَ نَقَلُوا شَهَادَةَ الْأَصْلِ فَكَأَنَّ الْأَصْلَ حَضَرَ وَشَهِدَ عِنْدَ مَجْلِسِ الْقَاضِي ثُمَّ رَجَعَ ( لَا ) يَضْمَنُ ( عِنْدَهُمَا ) ؛ لِأَنَّ الْحُكْمَ لَمْ يَقَعْ بِشَهَادَةِ الْأَصْلِ بَلْ بِشَهَادَةِ الْفَرْعِ وَقَوْلُهُ غَلِطْت( 6 / 209 )
