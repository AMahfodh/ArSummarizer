( فَرْعٌ لَوْ قِيلَ ) لَهُ ( أَلَكَ زَوْجَةٌ فَقَالَ لَا لَمْ تَطْلُقْ وَلَوْ نَوَى ) لِأَنَّهُ كَذِبٌ مَحْضٌ وَهَذَا مَا نَقَلَهُ الْأَصْلُ عَنْ نَصِّ الْإِمْلَاءِ وَقَطَعَ كَثِيرٌ مِنْ الْأَصْحَابِ ثُمَّ ذَكَرَ تَفَقُّهًا مَا حَاصِلُهُ أَنَّهُ كِنَايَةٌ عَلَى الْأَصَحِّ وَبِهِ صَرَّحَ النَّوَوِيُّ فِي تَصْحِيحِهِ وَأَنَّ لَهَا تَحْلِيفَهُ أَنَّهُ لَمْ يُرِدْ طَلَاقَهَا وَعَلَيْهِ جَرَى الْأَصْفُونِيُّ وَشَيْخُنَا أَبُو عَبْدِ اللَّهِ الْحِجَازِيُّ فِي اخْتِيَارِهِمَا كَلَامَ الرَّوْضَةِ ( وَلَوْ قِيلَ ) لَهُ ( أَطَلَّقْت ) زَوْجَتَك ( ثَلَاثًا ) أَوْ سَكَتَ عَنْ ذِكْرِ الثَّلَاثِ كَمَا فُهِمَ بِالْأُولَى ( فَقَالَ قَدْ كَانَ بَعْضُ ذَلِكَ فَلَيْسَ بِإِقْرَارٍ بِالطَّلَاقِ ) لِاحْتِمَالِ جَرَيَانِ تَعْلِيقٍ أَوْ وَعْدٍ أَوْ مُخَاصَمَةٍ تَئُولُ إلَيْهِ فَلَوْ فَسَّرَ بِشَيْءٍ مِنْ ذَلِكَ قُبِلَ ( وَقَوْلُهُ ) لَهَا ( مَا أَنْت لِي بِشَيْءٍ لَغْوٌ ) فَلَا يَقَعُ بِهِ طَلَاقٌ ( وَإِنْ نَوَى أَوْ ) قَالَ ( امْرَأَتِي طَلَّقَهَا زَوْجُهَا ) وَالْحَالُ إنَّهَا ( لَمْ تَتَزَوَّجْ غَيْرَهُ ) وَفِي نُسْخَةٍ قَبْلَهُ ( طَلُقَتْ ) وَهَذِهِ وَاَلَّتِي قَبْلَهَا مِنْ زِيَادَتِهِ .
