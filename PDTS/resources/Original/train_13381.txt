تَكَفَّلَ إلَخْ ( قَوْلُهُ : أَيْضًا ) أَيْ كَرُجُوعِهِ لِمَسْأَلَةِ الِاسْتِخْدَامِ ( قَوْلُهُ : وَلَمْ يَتَعَلَّقْ إلَخْ ) عَطْفٌ عَلَى قَوْلِهِ تَكَفَّلَ الْمَهْرَ وَقَوْلُهُ : بِهِ أَيْ الْعَبْدِ رِضَاهُ أَيْ الْغَيْرِ ا ه سم ( قَوْلُهُ : كَرَهْنٍ ) أَيْ أَوْ اسْتِئْجَارٍ أَوْ كِتَابَةٍ أَوْ جِنَايَةٍ ا ه حَلَبِيٌّ ( قَوْلُ الْمَتْنِ وَيُفَوِّتَ ) بِالنَّصْبِ مِنْ التَّفْوِيتِ ( قَوْلُهُ : لِلْعَبْدِ اسْتِصْحَابُ زَوْجَتِهِ إلَخْ ) فَإِنْ امْتَنَعَتْ مِنْ السَّفَرِ مَعَهُ وَلَوْ بِمَنْعِ السَّيِّدِ لَهَا إذَا كَانَتْ رَقِيقَةً سَقَطَتْ نَفَقَتُهَا مُغْنِي وَرَوْضٌ مَعَ شَرْحِهِ .
