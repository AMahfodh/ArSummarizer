( وَإِنْ دَفَعَ عَبْدٌ مَالًا لِمَنْ يَشْتَرِيهِ ) أَيْ الْعَبْدَ مِنْ مَالِكِهِ ( بِهِ ) أَيْ الْمَالِ ( فَإِنْ ) كَانَ ( قَالَ ) الْعَبْدُ لِلْمَدْفُوعِ لَهُ الْمَالُ ( اشْتَرِنِي لِنَفْسِك ) أَوْ لِتُعْتِقَنِي وَاشْتَرَاهُ بِهِ لِنَفْسِهِ أَوْ لِعِتْقِهِ ( فَلَا شَيْءَ عَلَيْهِ ) أَيْ الْمُشْتَرِي لِلْبَائِعِ ( إنْ ) كَانَ ( اسْتَثْنَى ) أَيْ اشْتَرَطَ الْمُشْتَرِي ( مَالَهُ ) أَيْ الْعَبْدِ حِينَ شِرَائِهِ لِنَفْسِهِ وَلِلْعَبْدِ لِأَنَّهُ قَدْ اشْتَرَى الْعَبْدَ وَمَالَهُ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَسْتَثْنِ الْمُشْتَرِي مَالَ الْعَبْدِ حِينَ شِرَائِهِ لِنَفْسِهِ صَحَّ الشِّرَاءُ فِي الْعَبْدِ وَحْدَهُ وَ ( غَرِمَهُ ) أَيْ الْمُشْتَرِي ثَمَنَ الْعَبْدِ لِبَائِعِهِ وَأَمَّا الثَّمَنُ الْأَوَّلُ فَهُوَ لِلْبَائِعِ بِطَرِيقِ الْأَصَالَةِ لِأَنَّ مَالَ الْعَبْدِ لَا يَتْبَعُهُ فِي الْبَيْعِ الْمُطْلَقِ .( 20 / 280 )
