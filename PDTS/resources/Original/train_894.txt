( وَوَجَدَ مُحَرَّمًا ) كَمَيْتَةٍ وَلَحْمِ خِنْزِيرٍ ( لَزِمَهُ أَكْلُهُ وَقِيلَ يَجُوزُ ) لَهُ وَالْأَكْلُ وَتَرْكُهُ ( فَإِنْ تَوَقَّعَ حَلَالًا قَرِيبًا ) أَيْ عَلَى قُرْبٍ ( لَمْ يَجُزْ غَيْرُ سَدِّ الرَّمَقِ ) وَفِي سَدِّهِ الْوُجُوبُ وَقِيلَ الْجَوَازُ أَخْذًا مِمَّا تَقَدَّمَ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَتَوَقَّعْهُ ( فَفِي قَوْلٍ يُشْبِعُ ) جَوَازًا ( وَالْأَظْهَرُ سَدُّ الرَّمَقِ ) فَقَطْ لِانْدِفَاعِ الضَّرُورَةِ بِهِ فَيَجِبُ فِي الْأَصَحِّ .
