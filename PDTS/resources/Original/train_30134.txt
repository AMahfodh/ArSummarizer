( فَرْعٌ ) لَوْ ( لُقِّنَ ) الزَّوْجُ ( الطَّلَاقَ ) أَيْ كَلِمَتُهُ ( بِلُغَةٍ لَا يَعْرِفُهَا ) فَقَالَهَا ( جَاهِلًا مَعْنَاهَا فَقَصَدَ بِهِ ) الْأَوْلَى بِهَا ( الطَّلَاقَ لَمْ يَقَعْ ) كَمَا لَوْ قَصَدَهُ بِلَفْظٍ لَا مَعْنَى لَهُ وَكَمَا لَوْ لُقِّنَ كَلِمَةَ الْكُفْرِ فَقَالَهَا لَا يُحْكَمُ بِكُفْرِهِ وَقَوْلُ الْمِنْهَاجِ وَلَوْ لَفَظَ عَجَمِيٌّ بِهِ بِالْعَرَبِيَّةِ وَلَمْ يَعْرِفْ مَعْنَاهُ لَمْ يَقَعْ يَقْتَضِي أَنَّهُ لَا فَرْقَ بَيْنَ أَنْ يُلَقَّنَ وَأَنْ لَا يُلَقَّنَ وَهُوَ ظَاهِرٌ ( وَكَذَا ) لَا يَقَعُ ( لَوْ قَصَدَ ) بِهَا ( مَعْنَاهَا بِالْعَرَبِيَّةِ ) لِأَنَّهُ إذَا لَمْ يَعْرِفْ مَعْنَاهَا لَمْ يَصِحَّ قَصْدُهُ ( وَيُؤَاخَذُ ) بِمَعْنَاهَا فِيمَا ذُكِرَ ( ظَاهِرًا مُخَالِطُ أَهْلِهَا ) وَيُدَيَّنُ .
