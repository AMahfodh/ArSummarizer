( وَإِنْ أَتَتْ بِوَلَدٍ لَا يَلْحَقُهُ ) أَيْ الزَّوْجَ ( نَسَبُهُ كَامْرَأَةِ صَغِيرٍ لَا يُولَدُ لِمِثْلِهِ وَ ) امْرَأَةِ ( خَصِيٍّ مَجْبُوبٍ ) أَوْ خَصِيٍّ غَيْرِ مَجْبُوبٍ كَمَا سَبَقَ ( وَمُطَلَّقَةٌ عَقِبَ عَقْدٍ ) بِأَنْ طَلَّقَهَا بِالْمَجْلِسِ وَكَذَا لَوْ مَاتَ ( وَمَنْ وُلِدَ لِدُونِ سِتَّةِ أَشْهُرٍ مُنْذُ عَقَدَ عَلَيْهَا وَعَاشَ أَوْ بَعْدَ أَرْبَعِ سِنِينَ مُنْذُ مَاتَ أَوْ ) مُنْذُ ( بَانَتْ مِنْهُ أَوْ ) مُنْذُ ( انْقِضَاءِ عِدَّتِهَا إنْ كَانَتْ رَجْعِيَّةً لَمْ تَنْقَضِ عِدَّتُهَا بِهِ ) لِأَنَّهُ حَمْلٌ لَيْسَ مِنْهُ يَقِينًا فَلَمْ تَعْتَدَّ بِوَضْعِهِ كَمَا لَوْ ظَهَرَ بَعْدَ مَوْتِهِ ( وَتَعْتَدُّ بَعَدَهُ عِدَّةَ وَفَاةٍ ) إنْ كَانَتْ مُتَوَفًّى عَنْهَا ( أَوْ عِدَّةَ فِرَاقٍ ) إنْ كَانَ فَارَقَهَا فِي الْحَيَاةِ ( حَيْثُ وَجَبَتْ ) عِدَّةُ الْفِرَاقِ عَلَى مَا تَقَدَّمَ تَفْصِيلُهُ .( 19 / 140 )
