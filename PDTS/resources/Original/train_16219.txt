( وَإِنْ وَصَّى لَهُ ) أَيْ : لِزَيْدٍ ( بِخَاتَمٍ وَ ) وَصَّى ( لِآخَرَ بِفَصِّهِ صَحَّ ) ذَلِكَ لِأَنَّ فِيهِ نَفْعًا مُبَاحًا ( وَلَيْسَ لِوَاحِدٍ مِنْهُمَا الِانْتِفَاعُ بِهِ ) أَيْ : بِالْخَاتَمِ ( إلَّا بِإِذْنِ الْآخَرِ ) كَالْمُشْتَرَكِ ( وَأَيُّهُمَا طَلَبَ قَلْعَ الْفَصِّ مِنْ الْخَاتَمِ أُجِيبَ إلَيْهِ ، وَأُجْبِرَ الْآخَرُ عَلَيْهِ ) لِتَمْيِيزِ حَقِّهِ .( 15 / 271 )
