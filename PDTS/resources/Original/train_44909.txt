لَهُمَا أَنَّهُ تَوَكَّلَ ) أَيْ قَبِلَ الْوَكَالَةَ ( بِالْقَبْضِ دُونَ إسْقَاطِ الْخِيَارِ فَلَا يَمْلِكُ مَا لَمْ يَتَوَكَّلْ بِهِ ) وَإِذَا لَمْ يَمْلِكْهُ لَا يَثْبُتُ عَنْ فِعْلِهِ ( وَصَارَ كَخِيَارِ الْعَيْبِ وَالشَّرْطِ ) بِأَنْ اشْتَرَى مَعِيبًا لَمْ يَرَ عَيْبَهُ ثُمَّ وَكَّلَ بِقَبْضِهِ فَقَبَضَهُ لَا يَسْقُطُ خِيَارُ الشَّرْطِ ( و ) صَارَ أَيْضًا ( كَالْإِسْقَاطِ قَصْدًا ) بِأَنْ قَبَضَهُ الْوَكِيلُ بِالْقَبْضِ مَسْتُورًا ثُمَّ رَآهُ فَأَسْقَطَ الْخِيَارَ قَصْدًا لَا يَسْقُطُ ( وَلَهُ أَنَّ الْقَبْضَ عَلَى نَوْعَيْنِ ) قَبْضٌ ( تَامٌّ ) وَهُوَ أَنْ يَقْبِضَهُ وَهُوَ يَرَاهُ .
