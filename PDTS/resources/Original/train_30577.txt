وَأَمَّا الضِّدَّانِ فَهُمَا الْأَمْرَانِ الْوُجُودِيَّانِ الْمُمْتَنِعُ اجْتِمَاعُهُمَا فِي مَحَلٍّ وَاحِدٍ وَالْخِلَافَانِ أَعَمُّ ، وَكَانَ الْأَوْلَى حَذْفَ قَوْلِهِ أَوْ ضِدَّ ؛ لِأَنَّ النَّقِيضَ أَخَصُّ مِنْهُ وَلَا يَلْزَمُ مِنْ ذِكْرِ الْأَخَصِّ ذِكْرُ الْأَعَمِّ فَإِنَّ السَّوَادَ الَّذِي هُوَ ضِدُّ الْبَيَاضِ يَشْمَلُهُ قَوْلُنَا لَا بَيَاضَ ( قَوْلُهُ : عَمَّا قَصَدَ ) أَيْ قَصَدَهُ الْمُتَنَاظِرَانِ بِالْمُنَاظَرَةِ ( قَوْلُهُ إلَى غَيْرِهِ ) مُتَعَلِّقٌ بِ خُرُوجٌ وَضَمِيرُ غَيْرِهِ رَاجِعٌ إلَى مَا قَصَدَ إلَخْ وَذَلِكَ الْغَيْرُ هُوَ مَعْرِفَةُ صِحَّةِ نَظَرِ الْمُعْتَرِضِ فِي دَلِيلِهِ ( قَوْلُهُ : بِأَنْ قَصَدَ ) أَيْ قَصَدَ الْمُعْتَرِضُ ( قَوْلُهُ : لَا إثْبَاتُ مُقْتَضَاهَا ) أَيْ الْمُعَارَضَةِ وَإِنْ كَانَ حَاصِلًا لَكِنَّهُ غَيْرُ مَقْصُودٍ ( قَوْلُهُ : إلَى مَا تَقَدَّمَ ) أَيْ الِانْقِلَابِ ( قَوْلُهُ : فَعِنْدِي إلَخْ ) أَشَارَ بِالْفَاءِ إلَى أَنَّ الْعِلَّةَ الرُّكْنِيَّةَ أَيْ فَبِسَبَبِ رُكْنِيَّتِهِ يُسَنُّ تَثْلِيثُهُ ( قَوْلُهُ : فَلَا يُسَنُّ ) نَقِيضُ يُسَنُّ ( وَقَوْلُهُ فَيُسْتَحَبُّ ) ضِدُّ يَجِبُ ( وَقَوْلُهُ فَيُوجِبُ التَّعْزِيرَ ) خِلَافُ لَا يُوجِبُ الْكَفَّارَةَ وَلَا مُنَافَاةَ بَيْنَهُمَا ( قَوْلُهُ : كَمَسْحِ الْخُفِّ ) بِجَامِعِ أَنَّ كُلًّا مَسْحٌ فِي الْوُضُوءِ (( 5 / 23 )
