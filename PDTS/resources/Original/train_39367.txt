( فَلَوْ مَاتَ ) الرَّقِيقُ الَّذِي أَعْتَقَ سَيِّدُهُ جُزْءًا مِنْهُ فِي مَرَضِهِ ( قَبْلَ ) مَوْتِ ( سَيِّدِهِ ) ثُمَّ مَاتَ سَيِّدُهُ ( عَتَقَ بِقَدْرِ ثُلُثِهِ ) ؛ أَيْ : ثُلُثِ مَالِ السَّيِّدِ عِنْدَ الْمَوْتِ ، بِخِلَافِ الْمُدَبَّرِ وَالْمُوصَى بِعِتْقِهِ ؛ فَإِنَّهُ يَمُوتُ قِنًّا .( 14 / 226 )
