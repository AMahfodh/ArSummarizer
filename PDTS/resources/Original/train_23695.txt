( وَالصَّغِيرَةُ ) لَوْ حَاضَتْ بَعْدَ تَمَامِ الْأَشْهُرِ ( لَا ) تَسْتَأْنِفُ ( إلَّا إذَا حَاضَتْ فِي أَثْنَائِهَا ) فَتَسْتَأْنِفُ بِالْحَيْضِ ( كَمَا تَسْتَأْنِفُ ) الْعِدَّةَ ( بِالشُّهُورِ مَنْ حَاضَتْ حَيْضَةً ) ، أَوْ ثِنْتَيْنِ ( ثُمَّ أَيِسَتْ ) تَحَرُّزًا عَنْ الْجَمْعِ بَيْنَ الْأَصْلِ وَالْبَدَلِ .
