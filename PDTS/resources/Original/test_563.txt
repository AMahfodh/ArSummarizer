( وَلَا ) يَصِحُّ الْبَيْعُ إنْ قَالَ : ( كُلَّمَا أَخَذْتَ قَفِيزًا ) مِنْ هَذِهِ الْحِنْطَةِ وَنَحْوِهَا ( فَعَلَيْكَ دِرْهَمٌ ، أَوْ ) قَالَ : كُلَّمَا ( أَوْقَدْت مِنْ ) هَذَا ( الدُّهْنِ رِطْلًا فَعَلَيْكَ دِرْهَمٌ ) ؛ لِجَهَالَةِ الْمَأْخُوذِ وَالْمَوْقُودِ ابْتِدَاءً ، ( خِلَافًا لِلشَّيْخِ ) تَقِيِّ الدِّينِ فَإِنَّهُ قَالَ بِالصِّحَّةِ فِيهِمَا ، ( وَصَحَّ ) قَوْلُ شَخْصٍ لِآخَرَ : ( كُلَّمَا أَعْتَقْتَ عَبْدًا ) مِنْ عَبِيدِك ( فَعَلَيَّ ثَمَنُهُ ) ، فَإِذَا أَعْتَقَ أَحَدًا مِنْهُمْ ، صَحَّ الْعِتْقُ وَرَجَعَ عَلَى الْقَائِلِ بِثَمَنِ مِثْلِهِ .( 7 / 345 )
