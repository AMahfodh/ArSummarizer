( وَكَذَا أَيَّامُ التَّشْرِيقِ ) يَحْرُمُ صَوْمُهَا وَلَا يَصِحُّ فَرْضًا وَلَا نَفْلًا ؛ لِمَا رَوَى مُسْلِمٌ عَنْ نُبَيْشَةَ الْهُذَلِيِّ مَرْفُوعًا { : أَيَّامُ التَّشْرِيقِ أَيَّامُ أَكْلٍ وَشُرْبٍ وَذِكْرٍ لِلَّهِ } وَلِأَحْمَدَ النَّهْيُ عَنْ صَوْمِهَا مِنْ حَدِيثِ أَبِي هُرَيْرَةَ وَسَعْدٍ بِإِسْنَادَيْنِ ضَعِيفَيْنِ ( إلَّا عَنْ دَمِ مُتْعَةٍ وَقِرَانٍ وَيَأْتِي ) فِي بَابِ الْفِدْيَةِ لِقَوْلِ ابْنِ عُمَرَ وَعَائِشَةَ { لَمْ يُرَخَّصْ فِي أَيَّامِ التَّشْرِيقِ أَنْ يُصَمْنَ إلَّا لِمَنْ لَمْ يَجِدْ الْهَدْيَ } رَوَاهُ الْبُخَارِيُّ .( 6 / 147 )
