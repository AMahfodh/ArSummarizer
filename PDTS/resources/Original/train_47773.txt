( وَلَوْ أَجَّرَ نَفْسَهُ فِيهِمَا ) أَيْ الْمَهْرِ وَالنَّفَقَةِ ( جَازَ ) بِنَاءً عَلَى جَوَازِ بَيْعِ الْمُسْتَأْجَرِ ( وَيُصْرَفُ كَسْبُهُ كُلَّ يَوْمٍ لِلنَّفَقَةِ ) لِأَنَّهَا ضَرُورِيَّةٌ ( ثُمَّ ) إنْ فَضَلَ مِنْهُ شَيْءٌ صَرَفَهُ ( لِلْمَهْرِ ثُمَّ ) إنْ فَضَلَ مِنْهُ شَيْءٌ صَرَفَهُ ( لِلسَّيِّدِ وَلَا يَدَّخِرُ ) مِنْهُ شَيْئًا ( لِلنَّفَقَةِ فَإِذَا لَمْ يَكُنْ كَسُوبًا تَعَلَّقَ ) كُلٌّ مِنْ الْمَهْرِ وَالنَّفَقَةِ ( بِذِمَّتِهِ ) كَالْقَرْضِ لِلُزُومِهِ بِرِضَى مُسْتَحِقِّهِ ( لَا غَيْرِهَا ) مِنْ رَقَبَتِهِ إذْ لَا جِنَايَةَ وَلَا مِنْ ذِمَّةِ سَيِّدِهِ لِأَنَّهُ الْمَالِكُ( 15 / 266 )
