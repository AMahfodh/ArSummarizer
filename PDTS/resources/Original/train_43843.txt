( فَرْعٌ الثَّمَنُ مَا اسْتَقَرَّ عَلَيْهِ الْعَقْدُ فَيَلْحَقُهُ الزِّيَادَةُ وَالنُّقْصَانُ قَبْلَ لُزُومِهِ ) أَيْ الْعَقْدِ ( فَإِنْ حَطَّ ) مِنْهُ بَعْضَهُ ( بَعْدَ لُزُومِهِ وَبَاعَ بِلَفْظِ ) مَا ( اشْتَرَيْت لَمْ يَلْزَمْهُ الْحَطُّ وَلَوْ ) بَاعَ ( بِلَفْظِ ) مَا ( قَامَ عَلَيَّ ) أَوْ رَأْسِ الْمَالِ ( أَخْبَرَ بِالْبَاقِي فَإِنْ انْحَطَّ الْكُلُّ لَمْ يَنْعَقِدْ بَيْعُهُ مُرَابَحَةً بِلَفْظِ قَامَ عَلَيَّ ) أَوْ رَأْسِ الْمَالِ قَالَ الْمُتَوَلِّي لِأَنَّهُ لَمْ يَقُمْ عَلَيْهِ بِشَيْءٍ وَلَا لَهُ فِيهِ رَأْسُ الْمَالِ ( بَلْ بِاشْتَرَيْتُ وَالْحَطُّ ) لِلْكُلِّ أَوْ لِلْبَعْضِ ( بَعْدَ جَرَيَانِ الْمُرَابَحَةِ لَمْ يَلْحَقْ ) مَنْ اشْتَرَى بِخِلَافِ نَظِيرِهِ فِي التَّوْلِيَةِ وَالْإِشْرَاكِ قَالَ الْقَاضِي لِأَنَّ ابْتِنَاءَهُمَا عَلَى الْعَقْدِ الْأَوَّلِ أَقْوَى مِنْ ابْتِدَاءِ الْمُرَابَحَةِ عَلَيْهِ بِدَلِيلِ أَنَّهُمَا لَا يَقْبَلَانِ الزِّيَادَةَ بِخِلَافِ الْمُرَابَحَةِ
