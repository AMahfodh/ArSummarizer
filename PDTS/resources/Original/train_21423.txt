( إلَّا الطَّلَاقَ ) لِزَوْجَةِ مُوَكِّلِهِ ( وَإِنْكَاحَ ) أَيْ تَزْوِيجَ ( بِكْرِهِ ) أَيْ مُوَكِّلِهِ بِكَسْرٍ فَسُكُونٍ ( وَبَيْعَ دَارِ سُكْنَاهُ ) أَيْ مُوَكِّلِهِ ( وَ ) بَيْعَ ( عَبْدِ ) خَدَمْتِ ( ه ) أَيْ مُوَكِّلِهِ فَلَا يَدْخُلُ وَاحِدٌ مِنْ هَذِهِ الْأَرْبَعَةِ فِي وَكَالَةِ التَّفْوِيضِ الْعَامَّةِ الْجَامِعَةِ .
