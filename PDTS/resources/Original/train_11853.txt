( وَيُحَرَّمُ ) عَلَيْك بِالْمُصَاهَرَةِ ( زَوْجَةُ مَنْ وَلَدْتَ ) وَإِنْ سَفَلَ مِنْ نَسَبٍ أَوْ رَضَاعٍ ( أَوْ وَلَدَك ) وَإِنْ عَلَا ( مِنْ نَسَبٍ أَوْ رَضَاعٍ ) لِقَوْلِهِ تَعَالَى { وَحَلَائِلُ أَبْنَائِكُمْ الَّذِينَ مِنْ أَصْلَابِكُمْ } وَمَنْطُوقُ خَبَرِ يَحْرُمُ مِنْ الرَّضَاعِ السَّابِقِ يُعَيِّنُ حَمْلَ مِنْ أَصْلَابِكُمْ عَلَى أَنَّهُ لِإِخْرَاجِ زَوْجَةِ الْمُتَبَنَّى دُونَ ابْنِ الرَّضَاعِ وَلِقَوْلِهِ تَعَالَى { وَلَا تَنْكِحُوا مَا نَكَحَ آبَاؤُكُمْ مِنْ النِّسَاءِ } .
