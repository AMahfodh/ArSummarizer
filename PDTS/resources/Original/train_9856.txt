( وَصَارِفُ فِضَّةٍ بِدِينَارٍ ) إنْ ( أَعْطَى ) فِضَّةً ( أَكْثَرَ ) مِمَّا بِالدِّينَارِ ( لِيَأْخُذَ ) رَبُّ الدِّينَارِ ( قَدْرَ حَقِّهِ مِنْهُ ) أَيْ مِمَّا أُعْطِيَهُ أَكْثَرَ ( فَفَعَلَ ) أَيْ أَخَذَ صَاحِبُ الدِّينَارِ قَدْرَ حَقِّهِ ( جَازَ ) هَذَا الْفِعْلُ مِنْهُمَا ( وَلَوْ ) كَانَ أَخْذُهُ بِقَدْرِ حَقِّهِ ( بَعْدَ تَفَرُّقٍ ) لِوُجُودِ التَّقَابُضِ قَبْلَ التَّفَرُّقِ وَإِنَّمَا تَأَخَّرَ التَّمْيِيزُ ( وَالزَّائِدُ ) عَنْ حَقِّهِ ( أَمَانَةٌ ) بِيَدِهِ لِوَضْعِ يَدِهِ عَلَيْهِ بِإِذْنِ رَبِّهِ( 5 / 54 )
