( وَلَا تَجِبُ ) الضِّيَافَةُ ( مِنْ غَيْرِ شَرْطٍ ) لِأَنَّهَا مَالٌ فَلَا يَلْزَمُهُمْ بِغَيْرِ رِضَاهُمْ كَالْجِزْيَةِ ( فَلَا يُكَلَّفُونَ الضِّيَافَةُ ) ( مَعَ عَدَمِ الشَّرْطِ ) وَلَا ( يُكَلَّفُونَ الذَّبِيحَةَ ) وَإِنْ شُرِطَتْ عَلَيْهِمْ الضِّيَافَةُ ( وَلَا ) يُكَلَّفُونَ ( أَنْ يُضَيِّفُونَا بِأَرْفَعَ مِنْ طَعَامِهِمْ ) لِمَا تَقَدَّمَ مِنْ قَوْلِ عُمَرَ أَطْعِمُوهُمْ مِمَّا تَأْكُلُونَ .( 8 / 303 )
