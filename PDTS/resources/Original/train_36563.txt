( وَإِنْ أَجَّرَ ) الْمُفْلِسُ ( دَارًا ) بِعَيْنِهَا ( أَوْ بَعِيرًا بِعَيْنِهِ ، أَوْ ) أَجَّرَ ( شَيْئًا غَيْرَهُمَا بِعَيْنِهِ ، ثُمَّ أَفْلَسَ لَمْ تَنْفَسِخْ الْإِجَارَةُ ) بِالْحَجْرِ عَلَيْهِ ( بِالْفَلَسِ ) لِلُزُومِهَا ( وَكَانَ الْمُسْتَأْجِرُ أَحَقَّ بِالْعَيْنِ الَّتِي اسْتَأْجَرَهَا مِنْ الْغُرَمَاءِ حَتَّى يَسْتَوْفِيَ حَقَّهُ فَإِنْ هَلَكَ الْبَعِيرُ ) الْمُؤَجَّرُ ( أَوْ انْهَدَمَتْ الدَّارُ ) الْمُؤَجَّرَةُ ( قَبْلَ انْقِضَاءِ الْمُدَّةِ انْفَسَخَتْ الْإِجَارَةُ ) لِفَوَاتِ الْمَعْقُودِ عَلَيْهِ .
