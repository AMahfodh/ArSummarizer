( وَلَوْ لَمْ يَكُنْ مُعْتَادًا فَدَخَلَ فِي ذَلِكَ الْجُلُودُ وَاللُّبُودُ وَالْخَشَبُ وَالزُّجَاجُ وَالْحَدِيدُ وَنَحْوُهَا ) ؛ لِأَنَّهُ خُفٌّ سَاتِرٌ يُمْكِنُ الْمَشْيُ فِيهِ أَشْبَهَ الْجُلُودَ ( وَ ) يُشْتَرَطُ أَيْضًا ( طَهَارَةُ عَيْنِهِ ) لِأَنَّ نَجِسَ الْعَيْنِ مَنْهِيٌّ عَنْهُ ( فَلَا يَصِحُّ ) الْمَسْحُ ( عَلَى نَجِسٍ وَلَوْ فِي ضَرُورَةٍ ) لِمَا تَقَدَّمَ فِي الْحَرِيرِ ( فَيَتَيَمَّمُ مَعَهَا ) أَيْ : الضَّرُورَةِ ( لِلرِّجْلَيْنِ ) أَيْ : لَا بُدَّ عَنْ غَسْلِهِمَا .
