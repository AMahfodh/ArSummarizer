( قَوْلُهُ : وَكَوْنُ أَكْثَرِ مَنْ وَلِيَ إلَخْ ) جَوَابُ سُؤَالٍ ظَاهِرِ الْبَيَانِ ( قَوْلُهُ : فَلَا يَرِدُ ) أَيْ عَلَى اشْتِرَاطِ الِاجْتِهَادِ ( قَوْلُ الْمَتْنِ شُجَاعًا ) بِتَثْلِيثِ الْمُعْجَمَةِ وَالشَّجَاعَةُ قُوَّةُ الْقَلْبِ عِنْدَ الْبَأْسِ مُغْنِي وَعِ ش ( قَوْلُهُ : يَسُوسُ ) عَلَى وَزْنِ يَصُونُ أَيْ يَحْكُمُ بِهِ ا ه كُرْدِيٌّ ( قَوْلُهُ : أَنْ يَعْرِفَ أَقْدَارَ النَّاسِ ) أَيْ بِأَنْ يَعْرِفَ مَنْ يَسْتَحِقُّ الرِّعَايَةَ وَمَنْ لَا يَسْتَحِقُّهَا وَيُعَامِلُهُمْ بِذَلِكَ إذَا وَرَدَ عَلَيْهِ ا ه ع ش ( قَوْلُهُ : يُفْهِمُ ) بِبِنَاءِ الْفَاعِلِ وَيَجُوزُ كَوْنُهُ لِلْمَفْعُولِ ( قَوْلُهُ : وَإِنْ فَقَدَ الذَّوْقَ إلَخْ ) عِبَارَةُ الْمُغْنِي وَفُهِمَ مِنْ اقْتِصَارِهِ عَلَى مَا ذُكِرَ أَنَّهُ لَا يُؤَثِّرُ فَقْدُ شَمٍّ وَذَوْقٍ ، وَهُوَ كَذَلِكَ كَمَا جَزَمَ بِهِ فِي زَوَائِدِ الرَّوْضَةِ وَلَا يُشْتَرَطُ كَوْنُهُ مَعْصُومًا ؛ لِأَنَّ الْعِصْمَةَ لِلْأَنْبِيَاءِ وَلَا يَضُرُّ قَطْعُ ذَكَرٍ وَأُنْثَيَيْنِ ا ه .
