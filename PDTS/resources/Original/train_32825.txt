( قَوْلُهُ : وَفِي فَرْقِهِ نَظَرٌ ) فَإِنَّهُ حَمَلَ نُقُولَ الْأَصْحَابِ الْمُصَرِّحَةَ بِمَنْعِ بَيْعِ الدَّارِ الْمَوْقُوفَةِ وَإِنْ تَعَطَّلَتْ وَخَرِبَتْ عَلَى غَيْرِ الْمَوْقُوفَةِ عَلَى الْمَسْجِدِ ( قَوْلُهُ : وَكَلَامُ الْأَصْلِ يَقْتَضِي جَوَازَ بَيْعِهَا مُطْلَقًا ) أَشَارَ إلَى تَصْحِيحِهِ قَالَ شَيْخُنَا الَّذِي أَفْتَى بِهِ الْوَالِدُ رَحِمَهُ اللَّهُ تَعَالَى مَنْعُ بَيْعِهَا مُطْلَقًا ( قَوْلُهُ : وَقَالَ السُّبْكِيُّ ) أَيْ وَالْأَذْرَعِيُّ وَغَيْرُهُمَا ( قَوْلُهُ : لِأَنَّ الظَّاهِرَ إلَخْ ) أَشَارَ إلَى تَصْحِيحِهِ ( قَوْلُهُ قَالَ الْمَاوَرْدِيُّ وَتُصْرَفُ غَلَّةُ وَقْفِهِ حِينَئِذٍ إلَى الْفُقَرَاءِ إلَخْ ) الَّذِي تَحَرَّرَ لِي أَنَّهُ إنْ تُوُقِّعَ عَوْدُهُ حُفِظَ لَهُ وَهُوَ مَا قَالَهُ الْإِمَامُ وَإِلَّا فَإِنْ أَمْكَنَ( 13 / 19 )
