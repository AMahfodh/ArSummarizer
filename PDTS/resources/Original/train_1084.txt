( وَ ) صَحَّ ( إعْتَاقُهُ ) حَالَ كَوْنِ ( الْمَأْذُون مَدْيُونًا ) وَلَوْ بِمُحِيطٍ ( وَضَمِنَ الْمَوْلَى لِلْغُرَمَاءِ الْأَقَلَّ مِنْ دَيْنِهِ وَقِيمَتِهِ ) وَإِنْ شَاءُوا اتَّبَعُوا الْعَبْدَ بِكُلِّ دُيُونِهِمْ وَبِاتِّبَاعِ أَحَدِهِمَا لَا يَبْرَأُ الْآخَرُ فَهُمَا كَكَفِيلٍ مَعَ مَكْفُولٍ عَنْهُ ( وَطُولِبَ بِمَا بَقِيَ ) مِنْ دَيْنِهِمْ إذَا لَمْ تَفِ بِهِ قِيمَتُهُ ( بَعْدَ عِتْقِهِ ) لِتَقَرُّرِهِ فِي ذِمَّتِهِ وَصَحَّ تَدْبِيرُهُ وَلَا يَنْحَجِرُ وَيُخَيَّرُ الْغُرَمَاءُ كَعِتْقِهِ إلَّا أَنَّ مَنْ اخْتَارَ أَحَدَ الشَّيْئَيْنِ لَيْسَ لَهُ الرُّجُوعُ شَرْحُ تَكْمِلَةٍ .
