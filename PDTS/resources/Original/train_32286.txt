الشَّرْطُ ( الثَّالِثُ أَنْ تُكَذِّبَهُ ) الزَّوْجَةُ فِي قَذْفِهَا ( وَيَسْتَمِرُّ ) تَكْذِيبُهَا ( إلَى اسْتِيفَاءِ اللِّعَانِ ) لِأَنَّهَا إذَا لَمْ تُكَذِّبْهُ لَا تُلَاعِنُهُ وَالْمُلَاعَنَةُ إنَّمَا تَنْتَظِمُ مِنْهُمَا ( فَإِنْ صَدَّقَتْهُ ) فِيمَا قَذَفَهَا بِهِ ( وَلَوْ مَرَّةً أَوْ عَفَتْ ) عَنْ الطَّلَبِ بِحَدِّ الْقَذْفِ ( أَوْ سَكَتَتْ ) فَلَمْ تُقِرَّ وَلَمْ تُنْكِرْ لَحِقَهُ النَّسَبُ وَلَا لِعَانَ ( أَوْ ثَبَتَ زِنَاهَا بِ ) شَهَادَةِ ( أَرْبَعَةٍ سِوَاهُ ) أَيْ الزَّوْجِ ( أَوْ قَذَفَ مَجْنُونَةً بِزِنَا قَبْلَهُ ) أَيْ جُنُونِهَا لَحِقَهُ النَّسَبُ وَلَا لِعَانَ ( أَوْ ) قَذَفَ ( مُحْصَنَةً فَجُنَّتْ ) قَبْلَ لِعَانٍ ( أَوْ ) قَذَفَ ( خَرْسَاءَ أَوْ نَاطِقَةً فَخَرِسَتْ ) قَبْلَ لِعَانٍ ( وَلَمْ تُفْهَمْ إشَارَتُهَا أَوْ قَذَفَ صَمَّاءَ لَحِقَهُ النَّسَبُ ) إنْ كَانَ بَيْنَهُمَا وَلَدٌ نَصًّا ( وَلَا لِعَانَ ) لِمَا سَبَقَ مِنْ أَنَّهُ شُرِعَ لِدَرْءِ الْحَدِّ عَنْ الْقَاذِفِ فَإِذَا لَمْ يَجِبْ حَدٌّ فَلَا فَائِدَةَ فِيهِ لَهُ وَنَفْيُ الْوَلَدِ تَابِعٌ لِإِسْقَاطِ الْحَدِّ لَا مَقْصُودٌ لِنَفْسِهِ .( 10 / 86 )
