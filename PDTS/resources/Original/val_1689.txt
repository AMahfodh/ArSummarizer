( وَإِنْ لَمْ يُرَدْ بِهِ الدَّوَامُ ) فِي الْأَرْضِ ( بَلْ ) أُرِيدَ بِهِ ( النَّقْلُ ) مِنْهَا ( إلَى مَوْضِعٍ آخَرَ وَيُسَمَّى الشَّتْلَ ، أَوْ كَانَ أَصْلُهُ لَا يَبْقَى فِي الْأَرْضِ ) كَبَذْرِ الْبُرِّ وَنَحْوِهِ ( فَكَزَرْعٍ ) فَهُوَ لِلْبَائِعِ وَنَحْوِهِ ( فَإِنْ لَمْ يَعْلَمْ الْمُشْتَرِي بَذْرَ الزَّرْعِ وَنَحْوِهِ ) كَالشَّتْلِ ( فَلَهُ فَسْخُ الْبَيْعِ وَإِمْضَاؤُهُ ) مَجَّانًا لِأَنَّ فِيهِ تَفْوِيتًا لِمَنْفَعَةِ الْأَرْضِ عَلَيْهِ مُدَّةً ( فَإِنْ تَرَكَهُ ) أَيْ الزَّرْعَ أَوْ الْبَذْرَ لَهُ أَوْ الشَّتْلَ ( الْبَائِعُ لِلْمُشْتَرِي ) فَلَا خِيَارَ لَهُ لِأَنَّهُ زَادَهُ خَيْرًا فَلَزِمَهُ قَبُولُهُ لِأَنَّ فِيهِ تَصْحِيحًا لِلْعَقْدِ ( أَوْ قَالَ الْبَائِعُ أَنَا أُحَوِّلهُ وَأَمْكَنَ ذَلِكَ ) أَيْ تَحْوِيلُهُ ( فِي زَمَنٍ يَسِيرٍ لَا يَضُرُّ بِمَنَافِعِ الْأَرْضِ فَلَا خِيَارَ لِلْمُشْتَرِي ) لِأَنَّهُ أَزَالَ الْعَيْبَ بِالنَّقْلِ عَلَى وَجْهٍ لَا يَضُرُّ بِمَنَافِعِ الْأَرْضِ .( 9 / 325 )
