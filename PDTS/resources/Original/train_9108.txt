( وَلَوْ لَمْ يُقَدِّرْهُ حَاكِمٌ ) وَأَمَّا الْحَاكِمُ وَأَمِينُهُ فَلَا يَأْكُلَانِ شَيْئًا لِأَنَّهُمَا يَسْتَغْنِيَانِ بِمَالِهِمَا فِي بَيْتِ الْمَالِ كَمَا يَأْتِي ( وَلَا يَلْزَمُهُ ) أَيْ الْوَلِيَّ ( عِوَضُهُ ) أَيْ مَا أَكَلَهُ ( إذَا أَيْسَرَ ) لِأَنَّ ذَلِكَ جُعِلَ عِوَضًا لَهُ عَنْ عَمَلِهِ فَلَمْ يَلْزَمْهُ عِوَضُهُ ، كَالْأَجِيرِ وَالْمُضَارِبِ وَلِأَنَّهُ تَعَالَى أَمَرَ بِالْأَكْلِ وَلَمْ يَذْكُرْ عِوَضًا ( وَإِنْ كَانَ ) الْوَلِيُّ ( غَنِيًّا لَمْ يَجُزْ لَهُ ذَلِكَ ) أَيْ الْأَكْلُ مِنْ مَالِ الْمُوَلَّى عَلَيْهِ وقَوْله تَعَالَى { وَمَنْ كَانَ غَنِيًّا فَلْيَسْتَعْفِفْ } ( إذَا لَمْ يَكُنْ أَبًا ) لِمَا يَأْتِي : أَنَّ الْأَبَ لَهُ أَنْ يَتَمَلَّكَ مِنْ مَالِ وَلَدِهِ مَا شَاءَ .( 11 / 109 )
