( الدَّيْنُ الْمُشْتَرَكُ ) بِسَبَبٍ مُتَّحِدٍ كَثَمَنِ مَبِيعٍ بِيعَ صَفْقَةً وَاحِدَةً ، أَوْ دَيْنِ مَوْرُوثٍ أَوْ قِيمَةِ مُسْتَهْلَكٍ مُشْتَرَكٍ ( إذَا قَبَضَ أَحَدُهُمَا شَيْئًا مِنْهُ شَارَكَهُ الْآخَرُ فِيهِ ) إنْ شَاءَ أَوْ أَتْبَعَ الْغَرِيمَ كَمَا يَأْتِي وَحِينَئِذٍ ( فَلَوْ صَالَحَ أَحَدُهُمَا عَنْ نَصِيبِهِ عَلَى ثَوْبٍ ) أَيْ خِلَافِ جِنْسِ الدَّيْنِ ( أَخَذَ الشَّرِيكُ الْآخَرُ نِصْفَهُ إلَّا أَنْ يَضْمَنَ ) لَهُ ( رُبْعَ ) أَصْلِ ( الدَّيْنِ ) فَلَا حَقَّ لَهُ فِي الثَّوْبِ ( وَلَوْ لَمْ يُصَالِحْ بَلْ اشْتَرَى بِنِصْفِهِ شَيْئًا ضَمَّنَهُ ) شَرِيكُهُ ( الرُّبْعَ ) لِقَبْضِهِ النِّصْفَ بِالْمُقَاصَّةِ ( أَوْ أَتْبَعَ غَرِيمَهُ ) فِي جَمِيعِ مَا مَرَّ لِبَقَاءِ حَقِّهِ فِي ذِمَّتِهِ .
