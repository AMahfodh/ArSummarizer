( وَلَا ) يَعْتِقُ وَلَدُ مُكَاتَبَةٍ ( إنْ مَاتَتْ ) قَبْلَ أَدَاءِ مَالِ كِتَابَةٍ أَوْ إبْرَاءٍ مِنْهُ لِبُطْلَانِ الْمُكَاتَبَةِ بِمَوْتِهَا وَكَغَيْرِ الْمُكَاتَبَةِ ( وَوَلَدُ بِنْتِهَا ) أَيْ الْمُكَاتَبَةِ ( كَوَلَدِهَا ) فَيَعْتِقُ إذَا عَتَقَتْ بِأَدَاءٍ أَوْ إبْرَاءٍ تَبَعًا لِأُمِّهِ وَ ( لَا ) يَتْبَعُ الْمُكَاتَبَةَ ( وَلَدُ ابْنِهَا ) أَيْ الْمُكَاتَبَةِ ذَكَرًا كَانَ أَوْ أُنْثَى مِنْ غَيْرِ أَمَتِهِ لِأَنَّ وَلَدَهُ تَابِعٌ لِأُمِّهِ دُونَ أَبِيهِ( 8 / 252 )
