وَإِنْ خَلَّفَ ثَلَاثَةَ بَنِينَ وَوَصَّى بِمِثْلِ نَصِيبِ أَحَدِهِمْ ( إلَّا رُبْعَ الْبَاقِي بَعْدَ النَّصِيبِ فَزِدْ عَلَى عَدَدِ الْبَنِينَ سَهْمًا وَرُبْعًا ) لِيَكُونَ لِلْبَاقِي بَعْدَ النَّصِيبِ مِنْ الْمَبْلَغِ الْحَاصِلِ بَعْدَ الضَّرْبِ رُبْعٌ صَحِيحٌ ( وَاضْرِبْهُ ) أَيْ الْحَاصِلَ مِنْ عَدَدِ الْبَنِينَ وَالْمُزَادَ عَلَيْهِ وَهُوَ أَرْبَعَةٌ وَرُبْعٌ ( فِي الْمَخْرَجِ ) أَيْ مَخْرَجِ الْكَسْرِ الْمُسْتَثْنَى وَهُوَ أَرْبَعَةٌ ( يَكُنْ ) خَارِجُ الضَّرْبِ ( سَبْعَةَ عَشَرَ ) لِلْمُوصَى ( لَهُ ) مِنْهَا ( سَهْمَانِ ) لِأَنَّ النَّصِيبَ خَمْسَةٌ .
