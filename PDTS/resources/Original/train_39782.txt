( وَيَجُوزُ عَقْدُ الْإِجَارَةِ عَلَى رَعْيِ مَاشِيَةٍ مُعَيَّنَةٍ ) بِأَنْ يَقُولَ : اسْتَأْجَرْتُكَ لِتَرْعَى هَذِهِ الْمَاشِيَةَ ( وَعَلَى ) رَعْيِ ( جِنْسٍ ) مَوْصُوفٍ ( فِي الذِّمَّةِ ) بِأَنْ يَقُولَ : اسْتَأْجَرْتُكَ لِرَعْيِ إبِلٍ ، أَوْ بَقَرٍ أَوْ غَنَمٍ ، وَيَصِفُهَا ( يَرْعَاهَا ) مُدَّةً مَعْلُومَةً .( 12 / 333 )
