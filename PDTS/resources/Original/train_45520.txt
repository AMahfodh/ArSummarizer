شِرَاءُ بَعْضِهِ ، وَإِنَّمَا عَتَقَ بِالْفَاسِدِ ؛ لِأَنَّ الْحَقَائِقَ الشَّرْعِيَّةَ تُطْلَقُ عَلَى فَاسِدِهَا كَصَحِيحِهَا ( كَأَنْ ) ( اشْتَرَى ) الْعَبْدُ ( نَفْسَهُ ) مِنْ سَيِّدِهِ شِرَاءً ( فَاسِدًا ) ، فَإِنَّهُ يَعْتِقُ لِتَشَوُّفِ الشَّارِعِ لِلْحُرِّيَّةِ وَيَأْخُذُ السَّيِّدُ مِنْ الْعَبْدِ مَا اشْتَرَى بِهِ نَفْسَهُ وَكَأَنَّهُ انْتَزَعَهُ مِنْهُ وَأَعْتَقَهُ ( وَ ) عَتَقَ عَلَى السَّيِّدِ ( الشِّقْصَ ) الَّذِي يَمْلِكُهُ مِنْ عَبْدٍ وَكَمَّلَ عَلَيْهِ ( فِيهِ ) إنْ كَانَ مَلِيًّا ( وَ ) عَتَقَ عَلَيْهِ ( الْمُدَبَّرُ وَأُمُّ الْوَلَدِ ) أَيْ يُنَجِّزُ عَلَيْهِ عِتْقَهُمَا وَكَذَا مُكَاتَبُهُ كَمَا فِي الْمُدَوَّنَةِ ( وَ ) عَتَقَ عَلَيْهِ ( وَلَدُ عَبْدِهِ ) الْكَائِنِ ( مِنْ أَمَتِهِ ) أَيْ أَمَةِ الْعَبْدِ ( وَإِنْ ) حَدَثَ الْوَلَدُ ( بَعْدَ يَمِينِهِ ) وَقَبْلَ حِنْثِهِ فَحُكْمُهُ كَمَنْ وُجِدَ قَبْلَ يَمِينِهِ لَكِنْ هَذَا فِي صِيغَةِ الْحِنْثِ كَقَوْلِهِ عَلَيَّ عِتْقُ عَبِيدِي لَأَدْخُلَنَّ الدَّارَ أَوْ إنْ لَمْ أَدْخُلْهَا فَعَبِيدِي أَحْرَارٌ .
