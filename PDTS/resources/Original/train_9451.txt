( وَإِنْ جَاوَزَهُ ) أَيْ : جَاوَزَ الدَّمُ أَقَلَّ الْحَيْضِ ، بِأَنْ زَادَ عَلَى يَوْمٍ بِلَيْلَتِهِ ( وَلَمْ يَعْبُرْ ) أَيْ يُجَاوِزْ ( الْأَكْثَرَ ) أَيْ : أَكْثَرَ الْحَيْضِ ، وَهُوَ خَمْسَةَ عَشَرَ يَوْمًا ، بِأَنْ انْقَطَعَ لِخَمْسَةَ عَشَرَ فَمَا دُونَهَا ( لَمْ تَجْلِسْ الْمُجَاوِزَ ) لِأَنَّهُ مَشْكُوكٌ فِيهِ ( بَلْ تَغْتَسِلُ عَقِبَ أَقَلِّهِ ) أَيْ : الْحَيْضِ ، لِأَنَّهُ آخِرُ حَيْضِهَا حُكْمًا ، أَشْبَهَ آخِرَهُ حِسًّا ( وَتَصُومُ وَتُصَلِّي فِيمَا جَاوَزَهُ ) لِأَنَّ الْمَانِعَ مِنْهُمَا هُوَ الْحَيْضُ وَقَدْ حُكِمَ بِانْقِطَاعِهِ ( وَيَحْرُمُ وَطْؤُهَا فِيهِ ) أَيْ : فِي الدَّمِ ، أَيْ : زَمَنِهِ الْمُجَاوِزِ لِأَقَلِّ الْحَيْضِ ( قَبْلَ تَكْرَارِهِ نَصًّا ) لِأَنَّ الظَّاهِرَ أَنَّهُ حَيْضٌ وَإِنَّمَا أَمَرْنَاهَا بِالْعِبَادَةِ احْتِيَاطًا لِبَرَاءَةِ ذِمَّتِهَا فَتَعَيَّنَ تَرْكُ وَطْئِهَا احْتِيَاطًا .
