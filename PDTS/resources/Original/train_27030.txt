( وَإِنْ كَانَ الِابْنُ وَطِئَهَا ) وَلَوْ لَمْ يَسْتَوْلِدْهَا ( لَمْ تَنْتَقِلْ ) الْجَارِيَةُ ( لِمِلْكِ أَبٍ ) بِالْإِحْبَالِ ( فَلَا تَصِيرُ أُمَّ وَلَدٍ لَهُ ) - أَيْ : الْأَبِ - إنْ حَمَلَتْ مِنْهُ نَصًّا ؛ لِأَنَّهَا بِالْوَطْءِ صَارَتْ مُلْحَقَةً بِالزَّوْجَةِ ؛ فَلَا يَصِحُّ أَنْ يَتَمَلَّكَهَا بِالْقَوْلِ كَمَا تَقَدَّمَ ، فَلَا يَمْلِكُهَا بِالْإِحْبَالِ ( وَلَا حَدَّ ) عَلَى الْأَبِ ، لِلشُّبْهَةِ ( وَوَلَدُهُ حُرٌّ ) لَا( 13 / 47 )
