سَيِّدُ عُمَرَ ( قَوْلُهُ : وَفِيهِ نَظَرٌ ) أَيْ فِيمَا قَالَهُ بَعْضُهُمْ ( قَوْلُهُ : أَنَّهُ لَا فَرْقَ ) أَيْ بَيْنَ الْعَارِفِ وَغَيْرِهِ ( قَوْلُهُ : مِنْ تَفْصِيلِ إلَخْ ) صِلَةُ قَوْلِهِ لَا بُدَّ سم وَكُرْدِيٌّ ( قَوْلُهُ : عَلَيْهِ ) أَيْ الْإِكْرَاهِ ( قَوْلُهُ : مِنْ الْبَيِّنَةِ ) أَيْ عَلَى الْإِكْرَاهِ .
