( وَلَا تَجِبُ ) عَلَى الْأَبِ ( فِطْرَةُ وَلَدٍ ) لَهُ ( مَلَكَ قُوتَ يَوْمِ الْعِيدِ وَلَيْلَتَهُ فَقَطْ ) أَوْ قَدَرَ عَلَى كَسْبِهِ ( وَلَوْ صَغِيرًا ) لِسُقُوطِ نَفَقَتِهِ عَنْهُ بِذَلِكَ ( وَتَسْقُطُ عَنْ الْوَلَدِ أَيْضًا لِإِعْسَارِهِ وَتَجِبُ لِرَجْعِيَّةٍ وَكَذَا بَائِنٌ حَامِلٌ ) وَلَوْ أَمَةً كَنَفَقَتِهِمَا بِخِلَافِ الْبَائِنِ غَيْرِ الْحَامِلِ لِسُقُوطِ نَفَقَتِهَا( 5 / 153 )
