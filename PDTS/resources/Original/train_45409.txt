( وَكَذَا ) لَوْ قَالَ لِأَجْنَبِيَّةٍ : ( أَنْتِ عَلَيَّ حَرَامٌ ، وَنَوَى أَبَدًا ) فَمُظَاهِرٌ ، لِأَنَّهُ ظِهَارٌ فِي الزَّوْجَةِ ، فَكَذَا الْأَجْنَبِيَّةُ ، فَلَا يَطَؤُهَا إذَا تَزَوَّجَهَا حَتَّى يُكَفِّرَ وَلَا يَكُونُ قَوْلُهُ لِأَجْنَبِيَّةٍ أَنْتِ عَلَيَّ حَرَامٌ ظِهَارًا ( إنْ أَطْلَقَ ) فَلَمْ يَنْوِ أَبَدًا ( أَوْ نَوَى ) أَنَّهَا حَرَامٌ عَلَيْهِ ( إذَنْ ) لِأَنَّهُ صَادِقٌ فِي حُرْمَتِهَا عَلَيْهِ قَبْلَ التَّزْوِيجِ ( وَيُقْبَلُ ) مِنْهُ دَعْوَى ذَلِكَ ( حُكْمًا ) لِأَنَّهُ الظَّاهِرُ .( 16 / 320 )
