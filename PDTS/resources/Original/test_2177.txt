( وَ ) فِيهِ ( الدِّيَةُ عَلَى عَاقِلَتِهِ ) لِقَوْلِهِ تَعَالَى : { وَدِيَةٌ مُسَلَّمَةٌ إلَى أَهْلِهِ } وَحَدِيثِ أَبِي هُرَيْرَةَ { اقْتَتَلَتْ امْرَأَتَانِ مِنْ هُذَيْلٍ فَرَمَتْ إحْدَاهُمَا الْأُخْرَى بِحَجَرٍ فَقَتَلَتْهَا وَمَا فِي بَطْنِهَا فَقَضَى النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَنَّ دِيَةَ جَنِينِهَا عَبْدٌ أَوْ وَلِيدَةٌ وَقَضَى بِدِيَةِ الْمَرْأَةِ عَلَى عَاقِلَتِهَا } مُتَّفَقٌ عَلَيْهِ فَإِنْ صَاحَ بِمُكَلَّفٍ لَمْ يَغْتَفِلْهُ فَلَا شَيْءَ عَلَيْهِ مَاتَ أَوْ ذَهَبَ عَقْلُهُ .( 10 / 298 )
