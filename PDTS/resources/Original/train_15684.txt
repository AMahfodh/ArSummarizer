وَإِنْ قَالَ أَنْتِ طَالِقٌ ( أَشَدَّ الطَّلَاقِ أَوْ أَغْلَظَهُ أَوْ أَكْبَرَهُ بِالْبَاءِ الْمُوَحَّدَةِ أَوْ أَطْوَلَهُ أَوْ أَعْرَضَهُ أَوْ مِلْءَ الدُّنْيَا أَوْ مِلْءَ الْبَيْتِ وَنَحْوَهُ ) كَالْمَسْجِدِ ( أَوْ ) أَنْتِ طَالِقٌ ( مِثْل الْجَبَلِ أَوْ مِثْلَ عِظَمِ الْجَبَلِ فَوَاحِدَةٌ رَجْعِيَّةٌ مَا لَمْ يَنْوِ أَكْثَرَ ) لِأَنَّ هَذَا الْوَصْفَ لَا يَقْتَضِي عَدَدًا وَالطَّلْقَةُ الْوَاحِدَةُ تُوصَفُ بِأَنَّهَا تَمْلَأُ الدُّنْيَا ذِكْرُهَا وَأَنَّهَا أَشَدُّ الطَّلَاقِ وَأَعْرَضُهُ ، فَإِنْ نَوَى ثَلَاثًا وَقَعَتْ لِأَنَّ اللَّفْظَ صَالِحٌ لَأَنْ يُرَادَ بِهِ ذَلِكَ ( وَكَذَا ) لَوْ قَالَ أَنْتِ طَالِقٌ ( أَقْصَاهُ ) فَتَقَعُ وَاحِدَةً ( صَحَّحَهُ فِي الْإِنْصَافِ ، وَصُحِّحَ فِي التَّنْقِيحِ وَتَصْحِيحُ الْفُرُوعِ أَنَّهَا ثَلَاثٌ وَإِنْ نَوَى وَاحِدَةً ) وَتَبِعَهُمَا فِي الْمُنْتَهَى .( 18 / 128 )
