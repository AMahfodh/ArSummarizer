( وَ ) إنْ قَالَ ( أَوْقَعْتُ بَيْنَكُنَّ طَلْقَةً فَطَلْقَةً أَوْ ) قَالَ أَوْقَعْتُ ( طَلْقَةً ثُمَّ طَلْقَةً ثُمَّ أَوْقَعْتُ بَيْنَكُنَّ طَلْقَةً وَأَوْقَعْتُ بَيْنكُنَّ طَلْقَةً أَوْ أَوْقَعْتُ بَيْنَكُنَّ طَلْقَةً طَلُقْنَ ) الْكُلُّ ( ثَلَاثًا إلَّا الَّتِي لَمْ يَدْخُلْ بِهَا فَإِنَّهَا تَبِينُ بِالْأُولَى ) فَلَا يَلْحَقُهَا مَا بَعْدَهَا .( 18 / 139 )
