( وَتُكْرَهُ ) زِيَارَةُ قُبُورٍ ( لِنِسَاءٍ ) لِحَدِيثِ أُمِّ عَطِيَّةَ { نُهِينَا عَنْ زِيَارَةِ الْقُبُورِ وَلَمْ يُعْزَمْ عَلَيْنَا } مُتَّفَقٌ عَلَيْهِ ( وَإِنْ عَلِمْنَ ) أَيْ النِّسَاءُ ( أَنَّهُ يَقَعُ مِنْهُنَّ مُحَرَّمٌ ) بِزِيَارَتِهِنَّ ( حَرُمَتْ ) زِيَارَتُهُنَّ لَهَا لِأَنَّهَا وَسِيلَةٌ لِلْمُحَرَّمِ ( إلَّا ) زِيَارَةَ النِّسَاءِ ( لِقَبْرِ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ وَقَبْرِ صَاحِبَيْهِ ) أَبِي بَكْرٍ وَعُمَرَ ( فَتُسَنُّ ) كَالرِّجَالِ ، لِعُمُومِ مَنْ حَجَّ فَزَارَنِي وَنَحْوَهُ .( 3 / 11 )
