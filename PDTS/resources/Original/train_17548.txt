وَأَمَّا قَوْله تَعَالَى { آيَتُكَ أَلَّا تُكَلِّمَ النَّاسَ ثَلَاثَةَ أَيَّامٍ إلَّا رَمْزًا } فَهُوَ اسْتِثْنَاءٌ مُنْقَطِعٌ وَقَوْلُ أَبِي الْخَطَّابِ هُوَ مُقْتَضَى مَا تَقَدَّمَ فِي الطَّلَاقِ أَنَّهُ لَا يَحْنَثُ بِهَا ( وَإِنْ نَادَاهُ ) الْحَالِفُ ( بِحَيْثُ ) إنَّ الْمَحْلُوفَ عَلَيْهِ ( يَسْمَعُ فَلَمْ يَسْمَعْ لِتَشَاغُلِهِ أَوْ غَفْلَةٍ ) حَنِثَ لِأَنَّهُ كَلَّمَهُ ( أَوْ سَلَّمَ ) الْحَالِفُ ( عَلَيْهِ ) أَيْ عَلَى مَنْ حَلَفَ لَا يُكَلِّمهُ ( حَنِثَ ) لِأَنَّ السَّلَامَ كَلَامٌ تَبْطُلُ بِهِ الصَّلَاةُ فَحَنِثَ بِهِ كَغَيْرِهِ .
