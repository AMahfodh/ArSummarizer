( وَإِنْ خَالَعَهَا عَلَى عَبْدٍ مُطْلَقٍ ) أَيْ غَيْرِ مُعَيَّنٍ وَلَا مَوْصُوفٍ ( فَلَهُ أَقَلُّ مَا يُسَمَّى عَبْدًا ) كَالْوَصِيَّةِ ( وَإِنْ قَالَ : إنْ أَعْطَيْتِنِي عَبْدًا فَأَنْتِ طَالِقٌ طَلُقَتْ بِأَيِّ عَبْدٍ أَعْطَتْهُ ) لِأَنَّ الشَّرْطَ عَطِيَّةُ عَبْدٍ وَقَدْ وُجِدَ وَقَوْلُهُ ( يَصِحُّ تَمْلِيكُهُ )( 17 / 475 )
