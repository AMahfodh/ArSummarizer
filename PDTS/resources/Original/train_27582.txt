189 - ( 11 ) - حَدِيثُ : رُوِيَ عَنْ { عُمَرَ أَنَّهُ قَالَ : يَا رَسُولَ اللَّهِ أَيَرْقُدُ أَحَدُنَا وَهُوَ جُنُبٌ ؟ قَالَ : نَعَمْ ، إذَا تَوَضَّأَ أَحَدُكُمْ فَلْيَرْقُدْ } قَالَ : وَيُرْوَى أَنَّهُ قَالَ : { اغْسِلْ فَرْجَك وَتَوَضَّأْ } مُتَّفَقٌ عَلَيْهِ مِنْ حَدِيثِ عَبْدِ اللَّهِ بْنِ عُمَرَ ، وَالْأَوَّلُ لَفْظُ الْبُخَارِيِّ ، وَفِي رِوَايَةٍ لِمُسْلِمٍ : { نَعَمْ لِيَتَوَضَّأْ ، ثُمَّ لِيَنَمْ حَتَّى يَغْتَسِلَ إذَا شَاءَ } وَلِابْنِ خُزَيْمَةَ : { أَيَنَامُ( 1 / 261 )
