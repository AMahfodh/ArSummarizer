( فَرْعٌ ) : لَوْ أَنْكَرَ - السَّيِّدُ الْعِتْقَ وَصَدَّقَ الزَّوْجُ صُدِّقَ السَّيِّدُ وَهَلْ تَفْسَخُ قَالَ صَاحِبُ الْكَافِي قَالَ شَيْخُنَا سَمِعْت شَيْخِي أَبَا عَلِيٍّ سُئِلَ عَنْ ذَلِكَ فَقَالَ يَحْتَمِلُ وَجْهَيْنِ وَالْأَصَحُّ ثُبُوتُ الْخِيَارِ لِأَنَّهَا حُرَّةٌ فِي زَعْمِهَا وَالْحَقُّ لَا يَعْدُوهُمَا قَالَ صَاحِبُ الْكَافِي فَعَلَى هَذَا لَوْ فَسَخَتْ قَبْلَ الدُّخُولِ لَمْ يَسْقُطْ الصَّدَاقُ لِأَنَّهُ حَقُّ السَّيِّدِ وَلَوْ عَتَقَ الْعَبْدُ وَأَيْسَرَ فَلَيْسَ لَهُ نِكَاحُهَا لِأَنَّ أَوْلَادَهَا أَرِقَّاءُ بِرّ ( قَوْلُهُ : لِتَسْتَرِيحَ مِنْ تَعَبِ الْفَسْخِ ) أَيْ بِظُهُورِ رَغْبَتِهَا عَنْهُ ( قَوْلُهُ : فَتُصَدَّقُ بِيَمِينِهَا ) لَمْ يَقُلْ إنْ أَمْكَنَ جَهْلُهَا كَمَا فِي الْفَوْرِ ( قَوْلُهُ : عَدَمَ قَبُولِ دَعْوَى الْجَهْلِ بِالرَّدِّ بِالْعَيْبِ ) إذَا لَمْ يَكُنْ الْمُدَّعِي قَرِيبَ عَهْدٍ بِالْإِسْلَامِ وَلَمْ يَنْشَأْ بِبَادِيَةٍ بَعِيدَةٍ شَرْحُ رَوْضٍ ( قَوْلُهُ : وَتُصَدَّقُ أَيْضًا ) وَإِنْ لَمْ تَكُنْ قَرِيبَةَ عَهْدٍ بِالْإِسْلَامِ ( قَوْلُهُ فِي الْمَتْنِ وَبَعْدَهُ بِعِتْقٍ بَعْدَهُ إلَخْ ) سَكَتَ عَمَّا لَوْ قَالَ فَسَخْت مَعَ الْوَطْءِ وَيُتَّجَهُ مَهْرُ الْمِثْلِ ( قَوْلُهُ : وَبَعْدَهُ بِعِتْقٍ بَعْدَهُ إلَخْ ) وَظَاهِرٌ أَنَّهُ لَا يُتَصَوَّرُ فَسْخُهَا مَعَ الْوَطْءِ بِعِتْقٍ بَعْدَهُ وَفِي تَصَوُّرِ فَسْخِهَا مَعَ الْوَطْءِ بِعِتْقٍ مَعَهُ نَظَرٌ ( قَوْلُهُ : وَمَا وَجَبَ مِنْهُمَا لِلسَّيِّدِ ) قَالَ فِي الرَّوْضِ إلَّا إذَا كَانَتْ مُفَوِّضَةً وَوَطِئَهَا أَيْ الزَّوْجُ أَوْ فَرَضَ لَهَا بَعْدَ الْعِتْقِ أَيْ فَالْمَهْرُ لَهَا انْتَهَى( 31 / 52 )
