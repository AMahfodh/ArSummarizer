( وَ ) مَنَعَ حَدَثٌ ( حَمْلَهُ ) أَيْ الْمُصْحَفِ بِيَدِهِ بَلْ ( وَإِنْ بِعِلَاقَةٍ أَوْ وِسَادَةٍ ) مُثَلَّثُ الْوَاوِ فِي كُلِّ حَالٍ ( إلَّا ) حَمْلَهُ ( بِأَمْتِعَةٍ ) أَيْ مَعَهَا ( قُصِدَتْ ) بِضَمٍّ فَكَسْرٍ أَيْ الْأَمْتِعَةُ وَحْدَهَا بِالْحَمْلِ فَيَجُوزُ إنْ حُمِلَتْ عَلَى مُؤْمِنٍ بَلْ ( وَإِنْ ) حُمِلَتْ ( عَلَى ) شَخْصٍ ( كَافِرٍ ) فَإِنْ قَصَدَ الْمُصْحَفَ وَحْدَهُ بِالْحَمْلِ أَوْ قُصِدَا مَعًا بِهِ فَلَا يَجُوزُ وَمِنْ مَسِّهِ وَحَمْلِهِ كُتُبَهُ فَلَا يَجُوزُ لِلْمُحْدِثِ عَلَى الرَّاجِحِ .( 1 / 242 )
