ثُمَّ انْتَقَلَ يَتَكَلَّمُ عَلَى مَسَائِلَ مَمْنُوعَةٍ فَقَالَ : ( وَلَا يَجُوزُ فِي الْبُيُوعِ التَّدْلِيسُ ) وَهُوَ أَنْ يَعْلَمَ أَنَّ بِسِلْعَتِهِ عَيْبًا فَيَكْتُمَهُ عَنْ الْمُشْتَرِي ( وَلَا ) يَجُوزُ ( الْغِشُّ ) وَهُوَ أَنْ يَخْلِطَ الشَّيْءَ بِغَيْرِ جِنْسِهِ كَخَلْطِ الْعَسَلِ بِالْمَاءِ ( وَلَا ) تَجُوزُ ( الْخِلَابَةُ ) بِكَسْرِ الْخَاءِ الْمُعْجَمَةِ وَتَخْفِيفِ اللَّامِ وَهِيَ الْخَدِيعَةُ بِالْكَذِبِ فِي الثَّمَنِ أَوْ يَرْقُمُ عَلَيْهَا أَكْثَرَ مِمَّا اشْتَرَاهَا بِهِ ، وَلَا يُصَرِّحُ بِذَلِكَ ( وَلَا ) يَجُوزُ ( الْخَدِيعَةُ ) وَهِيَ أَنْ يَخْدَعَهُ بِالْكَلَامِ حَتَّى يُوقِعَهُ مِثْلُ أَنْ يَقُولَ : تَعَالَ اشْتَرِ مِنِّي وَأَنَا أُرَخِّصُ لَك .
