سم ( قَوْلُهُ : وَلَيْسَتْ الْقَبْلِيَّةَ وَالْبَعْدِيَّةُ إلَخْ ) وَكَذَا سُنَّةُ الظُّهْرِ وَالْعَصْرِ بِالْأَوْلَى خِلَافًا لِمَا مَرَّ مِنْ بَحْثِ سم ( قَوْلُهُ : وَلَوْ مِنْ غَيْرِ سُنَّتِهَا ) إلَى الْمَتْنِ فِي النِّهَايَةِ ، وَالْمُغْنِي ( قَوْلُهُ : وَلَوْ فَرْضًا ) أَيْ كَالْعِشَاءِ ( قَوْلُهُ : لِمَنْ وَثِقَ ) إلَى قَوْلِهِ وَلَوْ أَوْتَرَ فِي النِّهَايَةِ إلَّا قَوْلَهُ الَّتِي لِلْأَمْرِ وَقَوْلَهُ عَلَى أَنَّ إلَى وَخَرَجَ وَقَوْلُهُ أَوْ عَكَسَ وَقَوْلُهُ وَلَا غَيْرَهُ إلَى قَوْلِ الْمَتْنِ ( وَيُسَنُّ جَعْلُهُ إلَخْ ) أَيْ وَلَوْ نَامَ قَبْلَهُ مُغْنِي( 7 / 274 )
