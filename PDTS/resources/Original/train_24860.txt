( وَمَنْ ضَرَبَ صُلْبَ غَيْرِهِ فَانْقَطَعَ مَاؤُهُ تَجِبُ الدِّيَةُ ) لِتَفْوِيتِ جِنْسِ الْمَنْفَعَةِ ( وَكَذَا لَوْ أَحْدَبَهُ ) لِأَنَّهُ فَوَّتَ جَمَالًا عَلَى الْكَمَالِ وَهُوَ اسْتِوَاءُ الْقَامَةِ ( فَلَوْ زَالَتْ الْحُدُوبَةُ لَا شَيْءَ عَلَيْهِ ) لِزَوَالِهَا لَا عَنْ أَثَرٍ .( 23 / 378 )
