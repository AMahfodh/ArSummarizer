[ فَرْعٌ ] فِي اجْتِمَاعِ جِنَايَاتٍ مِمَّا مَرَّ عَلَى شَخْصٍ وَاحِدٍ ، وَيَجْتَمِعُ فِي الْإِنْسَانِ سَبْعٌ وَعِشْرُونَ دِيَةً بَلْ أَكْثَرُ كَمَا يُعْلَمُ مِمَّا مَرَّ إذَا ( أَزَالَ ) جَانٍ ( أَطْرَافًا ) كَأُذُنَيْنِ وَيَدَيْنِ وَرِجْلَيْنِ ( وَلَطَائِفَ ) كَعَقْلٍ وَسَمْعٍ وَشَمٍّ ( تَقْتَضِي دِيَاتٍ فَمَاتَ سِرَايَةً ) مِنْ جَمِيعِهَا كَمَا بِأَصْلِهِ وَأَوْمَأَ إلَيْهِ بِالْفَاءِ فَلَا اعْتِرَاضَ عَلَيْهِ ( فَدِيَةٌ ) وَاحِدَةٌ تَلْزَمُهُ لِكَوْنِ الْجِنَايَةِ صَارَتْ نَفْسًا ، وَخَرَجَ بِجَمِيعِهَا انْدِمَالُ بَعْضِهَا فَلَا يَدْخُلُ وَاجِبُهُ فِي دِيَةِ النَّفْسِ ( وَكَذَا لَوْ حَزَّهُ الْجَانِي قَبْلَ انْدِمَالِهِ ) لَا يَجِبُ سِوَى دِيَةٍ وَاحِدَةٍ إنْ اتَّحَدَ الْحَزُّ وَالْفِعْلُ الْأَوَّلُ عَمْدًا أَوْ غَيْرَهُ ( فِي الْأَصَحِّ ) لِوُجُوبِ دِيَةِ النَّفْسِ قَبْلَ اسْتِقْرَارِ دِيَاتِ غَيْرِهَا فَتَدْخُلُ فِيهَا كَالسِّرَايَةِ ؛ إذْ لَا تَسْتَقِرُّ إلَّا بِانْدِمَالِهَا ، وَمِنْ ثَمَّ لَوْ حَزَّهُ بَعْدَ الِانْدِمَالِ وَجَبَتْ دِيَاتُ غَيْرِهَا قَطْعًا ( فَإِنْ حَزَّهُ ) الْجَانِي قَبْلَ الِانْدِمَالِ ( عَمْدًا ، وَالْجِنَايَاتُ ) بِإِزَالَةِ مَا ذُكِرَ ( خَطَأٌ ) أَوْ شِبْهُ عَمْدٍ ( أَوْ عَكْسُهُ ) بِأَنْ حَزَّهُ خَطَأً أَوْ شِبْهَ عَمْدٍ وَالْجِنَايَةُ عَمْدٌ ، أَوْ حَزَّهُ خَطَأً وَالْجِنَايَةُ شِبْهُ عَمْدٍ أَوْ عَكْسُهُ ( فَلَا تَدَاخُلَ فِي الْأَصَحِّ ) الْمَبْنِيِّ مَعَ مُقَابِلِهِ عَلَى الْأَصَحِّ السَّابِقِ مِنْ الدُّخُولِ عِنْدَ اتِّفَاقِ الْحَزِّ بَلْ يَجِبُ كُلٌّ مِنْ وَاجِبِ النَّفْسِ وَالْأَطْرَافِ لِاخْتِلَافِهِمَا حِينَئِذٍ بِاخْتِلَافِ حُكْمِهِمَا ( وَلَوْ ) ( حَزَّ ) رَقَبَتَهُ قَبْلَ الِانْدِمَالِ ( غَيْرُهُ ) أَيْ غَيْرُ الْجَانِي تِلْكَ الْجِنَايَاتِ أَوْ مَاتَ بِسُقُوطِهِ مِنْ نَحْوِ سَطْحٍ كَمَا أَفْتَى بِهِ الْبُلْقِينِيُّ ، وَفُرِّقَ بَيْنَهُ وَبَيْنَ مَا مَرَّ مِنْ اعْتِبَارِ التَّبَرُّعِ فِي الْمَرَضِ الْمَخُوفِ مِنْ الثُّلُثِ لَوْ مَاتَ بِهَا بِأَنَّ التَّبَرُّعَ صَدَرَ عِنْدَ الْخَوْفِ مِنْ الْمَوْتِ فَاسْتَمَرَّ حُكْمُهُ ( تَعَدَّدَتْ ) الْجِنَايَاتُ فَلَا تَدَاخُلَ إذْ فِعْلُ شَخْصٍ لَا يُبْنَى عَلَى فِعْلِ( 25 / 116 )
