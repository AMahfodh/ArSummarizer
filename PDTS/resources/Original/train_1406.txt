( قَوْلُهُ : وَيُؤَدَّى النُّسُكَانِ ) أَيْ الْحَجُّ وَالْعُمْرَةُ ( قَوْلُهُ : أَنَّهُ لَوْ أَتَى بِنُسُكٍ ) أَيْ مِنْ حَجٍّ أَوْ عُمْرَةٍ ( قَوْلُهُ : أَوْ عُمْرَةٍ ) أَيْ وَبِعَدَمِهَا عَلَى الْإِتْيَانِ بِمَا أَحْرَمَ بِهِ ( قَوْلُهُ : أَنْ يَأْتِيَ بِالْحَجِّ وَحْدَهُ فِي سَنَةٍ ) أَيْ ثُمَّ بِالْعُمْرَةِ فِي أُخْرَى ( قَوْلُهُ : فَسَيَأْتِي بَيَانُهُ ) قَدْ تَقَدَّمَ بَيَانُهُ فِي قَوْلِهِ أَحَدُهَا الْإِفْرَادُ ، فَلَعَلَّ الْمُرَادَ بِهِ أَنْ يَأْتِيَ مَا يَتَعَلَّقُ بِهِ مِنْ التَّفْصِيلِ وَبَيَانِ أَفْضَلِيَّتِهِ( 10 / 474 )
