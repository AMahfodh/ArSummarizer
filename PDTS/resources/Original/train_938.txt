وَتَصْوِيرُهُ ) أَيْ الْعَكْسِ بِضَرْبِهِ أَيْ بِقَصْدِ ضَرْبِهِ ( قَوْلُهُ : لِحَدِّهِ ) أَيْ لِضَرْبِهِ بِحَدِّ السَّيْفِ ( قَوْلُهُ : بِأَنَّ الْمُرَادَ بِالْفِعْلِ الْجِنْسُ ) أَيْ لَا خُصُوصُ الْفِعْلِ الْوَاقِعِ مِنْهُ حَتَّى يُسْتَشْكَلَ بِأَنَّ الضَّرْبَ بِخُصُوصِ الْحَدِّ لَمْ يَقْصِدْهُ ع ش ( قَوْلُهُ : وَبِمَا إلَخْ ) عُطِفَ عَلَى قَوْلِهِ بِضَرْبِهِ إلَخْ ( قَوْلُهُ : وَهُوَ غَيْرُ الْفِعْلِ إلَخْ ) يَعْنِي أَنَّ الْكَلَامَ الَّذِي صَدَرَ مِنْ الْمُهَدِّدِ غَيْرُ الْفِعْلِ الْمُهْلِكِ الَّذِي يَقَعُ مِنْ الْجَانِي كَالضَّرْبِ بِسَيْفٍ فَلَيْسَ الْمُرَادُ أَنَّ الْمُهَدِّدَ صَدَرَ مِنْهُ فِعْلٌ تَعَلَّقَ بِالْمَجْنِيِّ عَلَيْهِ غَيْرُ الْكَلَامِ بَلْ الْمُرَادُ أَنَّ هَذِهِ صُورَةٌ قَصَدَ فِيهَا الشَّخْصَ وَلَمْ يُقْصَدْ فِيهَا فِعْلٌ أَصْلًا وَمِنْ ثَمَّ رُدَّ بِأَنَّ مِثْلَ هَذَا الْكَلَامِ قَدْ يَقْتُلُ فَالْفِعْلُ وَالشَّخْصُ فِيهَا مَقْصُودَانِ ع ش ( قَوْلُهُ بِأَنَّ مِثْلَ هَذَا الْكَلَامِ إلَخْ ) الْمُنَاسِبُ فِي الرَّدِّ أَنْ يَقُولَ بِأَنَّ الْمُرَادَ بِالْفِعْلِ مَا يَشْمَلُ الْكَلَامَ وَمِثْلُ هَذَا الْكَلَامِ إلَخْ رَشِيدِيٌّ .
