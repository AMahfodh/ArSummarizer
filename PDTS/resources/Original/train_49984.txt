( وَلَا جِزْيَةَ عَلَى امْرَأَةٍ وَخُنْثَى ) لِأَنَّ آيَتَهَا السَّابِقَةَ لِلذُّكُورِ ( وَمَنْ فِيهِ رِقٌّ ) وَقِيلَ : تَجِبُ بِقِسْطِ حُرِّيَّتِهِ ( وَصَبِيٍّ وَمَجْنُونٍ ) لِعَدَمِ تَكْلِيفِهِمَا ( فَإِنْ تَقَطَّعَ جُنُونُهُ قَلِيلًا كَسَاعَةٍ مِنْ شَهْرٍ لَزِمَتْهُ أَوْ كَثِيرًا كَيَوْمٍ وَيَوْمٍ ) أَوْ يَوْمَيْنِ ( فَالْأَصَحُّ بِلَفْقِ الْإِفَاقَةِ فَإِذَا بَلَغَتْ سَنَةً وَجَبَتْ ) وَالثَّانِي لَا تَجِبُ وَالثَّالِثُ تَجِبُ كَالْعَاقِلِ وَالرَّابِعُ يُحْكَمُ بِمُوجِبِ الْأَغْلَبِ فَإِنْ اسْتَوَى الزَّمَانُ وَجَبَتْ
