بِنَائِهِ صَحَّتْ ) الصَّلَاةُ ( فِيهِ ) أَيْ : فِي الْمَسْجِدَ لِأَنَّهُ لَمْ يَتْبَعْ مَا حَدَثَ بَعْدَهُ ( وَالْمَنْعُ ) مِنْ الصَّلَاةِ ( فِي هَذِهِ الْمَوَاضِعِ تَعَبُّدٌ ) لَيْسَ مُعَلَّلًا بِوَهْمِ النَّجَاسَةِ وَلَا غَيْرِهِ لِنَهْيِ الشَّارِعِ عَنْهَا وَلَمْ يُعْقَلْ مَعْنَاهُ ( وَلَا تَصِحُّ ) صَلَاةٌ ( فِي بُقْعَةِ غَصْبٍ مِنْ أَرْضٍ أَوْ حَيَوَانٍ بِأَنْ يَغْصِبَهُ ) أَيْ : مَا ذُكِرَ مِنْ الْأَرْضِ وَالْحَيَوَانِ ( وَيُصَلِّيَ عَلَيْهِ ) الْغَاصِبُ ( أَوْ غَيْرُهُ ) لِأَنَّهَا عِبَادَةٌ أُتِيَ بِهَا عَلَى الْوَجْهِ الْمَنْهِيِّ عَنْهُ فَلَمْ تَصِحَّ كَصَلَاةِ الْحَائِضِ ، قَالَ فِي الْمُبْدِعِ : وَيُلْحَقُ بِهِ .( 2 / 372 )
