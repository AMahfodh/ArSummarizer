( وَإِذَا كَانَ لِرَجُلٍ خَمْسُ أُمَّهَاتِ أَوْلَادٍ لَهُنَّ لَبَنٌ مِنْهُ فَأَرْضَعْنَ امْرَأَةً لَهُ صُغْرَى كُلُّ وَاحِدَةٍ مِنْهُنَّ رَضْعَةً صَارَ ) سَيِّدُهُنَّ ( أَبًا لَهَا ) لِأَنَّهَا ارْتَضَعَتْ مِنْ لَبَنِهِ خَمْسَ رَضْعَاتٍ ، كَمَا لَوْ أَرْضَعَتْهَا وَاحِدَةٌ مِنْهُنَّ ( وَحُرِّمَتْ عَلَيْهِ ) عَلَى التَّأْبِيدِ لِأَنَّهَا بِنْتُهُ ، وَ ( لَا ) تُحَرَّمُ عَلَيْهِ ( أُمَّهَاتُ الْأَوْلَادِ لِعَدَمِ ثُبُوتِ الْأُمُومَةِ ) فَلَا يَثْبُتُ تَحْرِيمُهُنَّ ( وَإِنْ أَرْضَعْنَ ) أَيْ أُمَّهَاتُ أَوْلَادِهِ الْخَمْسُ بِلَبَنِهِ ( طِفْلًا كَذَلِكَ ) أَيْ كُلُّ وَاحِدَةٍ مِنْهُنَّ أَرْضَعَتْهُ رَضْعَةً ( صَارَ الْمَوْلَى ) صَاحِبُ اللَّبَنِ ( أَبًا لَهُ ) لِأَنَّهُ ارْتَضَعَ مِنْ لَبَنِهِ خَمْسَ رَضْعَاتٍ ( وَحُرِّمَتْ عَلَيْهِ ) أَيْ الطِّفْلِ ( الْمُرْضِعَاتُ ، لِأَنَّهُ رَبِيبُهُنَّ وَهُنَّ مَوْطُوآتُ أَبِيهِ ) فَيَتَنَاوَلَهُنَّ قَوْله تَعَالَى { وَلَا تَنْكِحُوا مَا نَكَحَ آبَاؤُكُمْ مِنْ النِّسَاءِ } .( 19 / 272 )
