( وَمَنْ أَوْصَى بِعِتْقِ عَبْدٍ بِعَيْنِهِ ، أَوْ ) وَصَّى ( بِوَقْفِهِ ؛ لَزِمَ ) الْعَمَلُ بِوَصِيَّتِهِ وُجُوبًا ( وَلَمْ يَقَعْ ) الْعِتْقُ أَوْ الْوَقْفُ ( حَتَّى يُنَجِّزَهُ وَارِثُهُ ) لِأَنَّ الْوَصِيَّةَ بِذَلِكَ أَمْرٌ بِفِعْلِهِ ، فَلَمْ يَقَعْ إلَّا بِفِعْلِ الْمَأْمُورِ كَالتَّوْكِيلِ فِي ذَلِكَ ، لَكِنْ هُنَا يَلْزَمُ تَنْفِيذُ الْوَصِيَّةِ ( فَإِنْ أَبَى ) وَارِثٌ تَنْجِيزَهُ ( فَحَاكِمٌ ) يُنَجِّزُهُ ، وَيَكُونُ حُرًّا أَوْ وَقْفًا مِنْ حِينِ عَتَقَ أَوْ وُقِفَ ، وَوَلَاؤُهُ لِمُوصٍ ( وَكَسْبُهُ ) ؛ أَيْ : الْمُوصَى بِعِتْقِهِ أَوْ وَقْفِهِ ( بَيْنَ مَوْتِ ) [ مُوصٍ ( وَتَنْجِيزِ ) ] مَا وَصَّى بِهِ مِنْ عِتْقٍ أَوْ وَقْفٍ ( إرْثٌ ) لِبَقَائِهِ فِي الْمِلْكِ إلَى التَّنْجِيزِ .( 13 / 224 )
