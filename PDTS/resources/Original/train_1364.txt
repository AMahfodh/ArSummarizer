( وَقُدِّرَتْ ) نَفَقَةُ الزَّوْجَةِ عَلَى الزَّوْجِ ( بِحَالِهِ ) أَيْ بِحَسَبِ حَالِهِ الَّتِي هُوَ عَلَيْهَا ( مِنْ يَوْمٍ ) أَيْ فِي يَوْمٍ فَتَكُونُ مُيَاوَمَةً كَأَرْبَابِ الصَّنَائِعِ وَالْأُجَرَاءِ ( أَوْ جُمُعَةٍ ) كَبَعْضِ أَرْبَابِ الصَّنَائِعِ ( أَوْ شَهْرٍ ) كَأَرْبَابِ الْمَدَارِسِ وَالْمَسَاجِدِ وَبَعْضِ الْجُنْدِ وَخَدَمِهِمْ ( أَوْ سَنَةٍ ) كَأَرْبَابِ الرِّزْقِ وَالْبَسَاتِينِ .
