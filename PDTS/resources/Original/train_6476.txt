عَلَى أَنَّهُ قَدْ يُقَالُ الَّذِي يَسْتَدْبِرُهُ هُنَا فِيمَا لَوْ كَانَتْ الْقِبْلَةُ خَلْفَهُ وَالْتَفَتَ إلَيْهَا هُوَ مَقْصِدُهُ وَلَيْسَ هُوَ قِبْلَةً بَلْ بَدَلَهَا ، وَاَلَّذِي اسْتَدْبَرَهُ فِي النُّزُولِ فِي شِدَّةِ الْخَوْفِ هُوَ الْقِبْلَةُ وَفَرْقُ مَا بَيْنَهُمَا ( قَوْلُهُ : لِكَوْنِهِ وَصْلَةً ) أَيْ طَرِيقًا ( قَوْلُهُ وَلَوْ قَهْرًا ) أَيْ بِأَنْ أُكْرِهَ ( قَوْلُهُ : وَإِنْ عَزَمَ عَلَى الْعَوْدِ ) أَيْ بَعْدَ الِانْحِرَافِ فَلَا يُخَالِفُ مَا مَرَّ ( قَوْلُهُ لَمْ يَضُرَّ ) أَيْ وَلَا سُجُودَ عَلَيْهِ وَإِنْ خَرَجَ عَنْ جِهَةِ مَقْصِدِهِ ( قَوْلُهُ : فَلْيَنْحَرِفْ إلَيْهَا ) أَيْ إلَى الْجِهَةِ الَّتِي قَصَدَ الرُّجُوعَ إلَيْهَا ( قَوْلُهُ : يُحْتَمَلُ تَخْرِيجُهُ إلَخْ ) أَيْ فَيَمْتَنِعُ عَلَيْهِ ذَلِكَ( 3 / 458 )
