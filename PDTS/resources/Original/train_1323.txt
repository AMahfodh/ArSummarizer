( وَكُرِهَ ) لِزَوْجٍ وَسَيِّدٍ حَالَةَ إحْرَامِهِ ( حَمْلُهَا لِلْمَحْمِلِ ) مُحْرِمَةً أَمْ لَا وَأَمَّا مَحْرَمُهَا فَلَا يُكْرَهُ وَأَمَّا الْأَجْنَبِيُّ فَظَاهِرٌ أَنَّهُ يُمْنَعُ ( وَلِذَلِكَ ) أَيْ وَلِأَجْلِ كَرَاهَةِ الْحَمْلِ الْمَذْكُورِ ( اُتُّخِذَتْ السَّلَالِمُ ) لِرُقِيِّ النِّسَاءِ عَلَيْهَا لِلْمَحْمِلِ ( وَ ) يُكْرَهُ لَهُ ( رُؤْيَةُ ذِرَاعَيْهَا ) لِغَيْرِ لَذَّةٍ وَإِلَّا حَرُمَ ( لَا ) يُكْرَهُ لَهُ رُؤْيَةُ ( شَعْرِهَا ) لِخِفَّتِهِ وَفِيهِ نَظَرٌ ( وَ ) لَا يُكْرَهُ ( الْفَتْوَى فِي أُمُورِهِنَّ ) وَلَوْ فِي حَيْضٍ وَنِفَاسٍ وَلَمَّا أَنْهَى الْكَلَامَ عَلَى مُحَرَّمَاتِ الْإِحْرَامِ شَرَعَ فِي مُحَرَّمَاتِهِ مَعَ الْحَرَمِ فَقَالَ .
