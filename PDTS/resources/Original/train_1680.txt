فِيهِمَا ) أَيْ فِي مَسْأَلَةِ زَوْجٍ وَأَبَوَيْنِ وَزَوْجَةٍ وَأَبَوَيْنِ ( وَابْنُ عَبَّاسٍ ثُلُثُ الْكُلِّ ) فِيهِمَا كَمَا رَوَاهُ الْجَارُودِيُّ عَنْهُ وَعَنْ عَلِيٍّ أَيْضًا ( فَأَحْدَثَ ابْنُ سِيرِينَ وَغَيْرُهُ ) وَهُوَ جَابِرُ بْنُ زَيْدٍ أَبُو الشَّعْثَاءِ كَمَا ذَكَرَ الْجَصَّاصُ ( أَنَّ ) لِلْأُمِّ ( فِي مَسْأَلَةِ الزَّوْجِ ) وَأَبَوَيْنِ ( كَابْنِ عَبَّاسٍ وَالزَّوْجَةِ ) أَيْ وَلِلْأُمِّ فِي مَسْأَلَتِهَا مَعَ الْأَبَوَيْنِ ( كَالصَّحَابَةِ ، وَعَكَسَ تَابِعِيٌّ آخَرَ ) وَهُوَ الْقَاضِي شُرَيْحٌ كَمَا نَقَلَهُ صَاحِبُ الْكَافِي فَقَالَ لَهَا فِي مَسْأَلَةِ الزَّوْجِ كَالصَّحَابَةِ وَفِي مَسْأَلَةِ الزَّوْجَةِ كَابْنِ عَبَّاسٍ ( وَلَمْ يُنْكَرْ ) إحْدَاثُ كُلٍّ مِنْ هَذَيْنِ الْقَوْلَيْنِ ( وَإِلَّا ) لَوْ أُنْكِرَ ( نُقِلَ ) وَلَمْ يُنْقَلْ وَالْوُقُوعُ دَلِيلُ الْجَوَازِ .
