( قَوْلُهُ : فَالْمُتَّجِهُ كَمَا قَالَ الزَّرْكَشِيُّ إلَخْ ) أَشَارَ إلَى تَصْحِيحِهِ ( قَوْلُهُ : فَإِنْ مَاتَ قَبْلَ الْمُوصِي ) أَوْ مَعَهُ ( قَوْلُهُ : قَبِلَ وَارِثُهُ أَوْ سَيِّدُهُ ) وَلَيْسَ لَنَا عَقْدٌ لَا يَبْطُلُ بِمَوْتِ الْقَابِلِ إلَّا الْوَصِيَّةُ ( قَوْلُهُ : يَجِبُ عَلَى الْوَلِيِّ الْقَبُولُ لَهُ ) أَشَارَ إلَى تَصْحِيحِهِ .( 13 / 404 )
