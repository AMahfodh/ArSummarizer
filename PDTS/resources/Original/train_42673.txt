( قَوْلُهُ فِي الْأَشْهَرِ ) وَحُكِيَ كَسْرُ الْمِيمِ آلَةٌ يُرْمَى بِهَا الْحِجَارَةُ ا ه مُغْنِي ( قَوْلُ الْمَتْنِ الْبَاقِي ) وَهُوَ تِسْعَةُ أَعْشَارِهَا عَلَى كُلٍّ مِنْهُمْ عُشْرُهَا ا ه مُغْنِي ( قَوْلُهُ وَغَلَبَتْ إصَابَتُهُ ) وَإِنْ لَمْ تَغْلِبْ فَشِبْهُ عَمْدٍ كَمَا هُوَ ظَاهِرٌ ا ه سم ( قَوْلُ الْمَتْنِ أَوْ غَيْرَهُمْ ) لَيْسَ مِنْ مَسْأَلَةِ الْعَوْدِ بَلْ هُوَ فِيمَا لَوْ رَمَوْا غَيْرَهُمْ كَمَا لَا يَخْفَى ا ه رَشِيدِيٌّ ( قَوْلُهُ بِعَيْنِهِ ) وَلَوْ قَصَدُوا غَيْرَ مُعَيَّنٍ كَأَحَدِ الْجَمَاعَةِ أَنَّ شِبْهَ عَمْدٍ ا ه مُغْنِي ( قَوْلُهُ فَإِنْ عُفِيَ عَنْهُ ) أَيْ عَلَى مَالٍ ( قَوْلُهُ فَإِنْ لَمْ يَغْلِبْ ) بِأَنْ غَلَبَ عَدَمُهَا أَوْ اسْتَوَى الْأَمْرَانِ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ دُونَ وَاضِعِهِ ) أَيْ الْحَجَرِ ( قَوْلُهُ إذْ لَا دَخْلَ لَهُمْ إلَخْ ) الْجَمْعُ هُنَا وَفِيمَا يَأْتِي نَظَرًا لِجَانِبِ الْمَعْنَى وَإِلَّا فَالظَّاهِرُ التَّثْنِيَةُ( 37 / 428 )
