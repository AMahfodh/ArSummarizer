( أَوْ ) سَرَقَهُ ( أَجْنَبِيٌّ ) غَيْرُ كَافِرٍ مِنْ الْغَنِيمَةِ ( قَبْلَ إفْرَازِ الْخُمْسِ أَوْ مِنْ الْخُمْسِ ) بَعْدَ إفْرَازِهِ وَقَبْلَ إخْرَاجِ خُمُسِهِ أَوْ مِنْ ( خُمُسِهِ ) أَيْ خُمُسِ الْخُمُسِ بَعْدَ إفْرَازِهِ ( لَمْ يُقْطَعْ ) ؛ لِأَنَّ فِيهِ مَالًا لِبَيْتِ الْمَالِ وَلَهُ فِيهِ حَقٌّ ( أَوْ مِنْ أَرْبَعَةِ أَخْمَاسِ الْغَنِيمَةِ قُطِعَ ) ؛ لِأَنَّهُ لَا حَقَّ لَهُ فِيهَا ( وَكَذَا ) لَوْ سَرَقَ ( مِنْ أَرْبَعَةِ أَخْمَاسِ الْخُمُسِ إنْ لَمْ يَكُنْ مِنْ أَهْلِهَا ) أَيْ أَهْلِ اسْتِحْقَاقِهَا وَإِلَّا فَلَا يُقْطَعْ .( 21 / 41 )
