( قَوْلُهُ : وَفَارَقَ ) أَيْ عَدَمُ وُجُوبِ الْقِصَاصِ فِي ذَهَابِ نَحْوِ أُصْبُعٍ بِالسِّرَايَةِ ( قَوْلُهُ : مَا تَقَرَّرَ إلَخْ ) أَيْ مِنْ وُجُوبِ الْقِصَاصِ ( قَوْلُهُ : بِأَنَّهَا ) أَيْ الْمَعَانِيَ ( قَوْلُهُ : عَلَيْهِ ) أَيْ مَحَلِّ الْمَعَانِي أَوْ مُجَاوِرِهِ ( قَوْلُهُ : وَالْإِجْرَامَ ) عُطِفَ عَلَى الْهَاءِ فِي قَوْلِهِ بِأَنَّهَا ( قَوْلُهُ : وَلَمْ تَعُدْ ) أَيْ الْجِنَايَةُ عَلَى غَيْرِ الْإِجْرَامِ ( قَوْلُهُ : أُصْبُعُهُ ) أَيْ الْجَانِي ( قَوْلُهُ : وَتَدْخُلُ فِيهَا ) أَيْ فِي الْأَرْبَعَةِ أَخْمَاسٍ .( 36 / 462 )
