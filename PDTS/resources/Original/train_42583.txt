( وَأَنْ يَبْدَأَ ) الْحَامِلُ ( فَيَضَعُ مُقَدَّمَهَا ) أَيْ مُقَدَّمَ الْجِنَازَةِ ( عَلَى يَمِينِهِ ثُمَّ ) يَضَعُ ( مُؤَخَّرَهَا ) عَلَى يَمِينِهِ ( ثُمَّ ) يَضَعُ ( مُقَدَّمَهَا عَلَى يَسَارِهِ ثُمَّ مُؤَخَّرَهَا ) عَلَى يَسَارِهِ فَيَتِمُّ الْحَمْلُ مِنْ الْجَوَانِبِ الْأَرْبَعِ وَيَنْبَغِي أَنْ يَحْمِلَهَا مِنْ كُلِّ جَانِبٍ عَشْرَ خُطُوَاتٍ لِقَوْلِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ { مَنْ حَمَلَ جِنَازَةً أَرْبَعِينَ خُطْوَةً كَفَّرَتْ عَنْهُ أَرْبَعِينَ كَبِيرَةً } .
