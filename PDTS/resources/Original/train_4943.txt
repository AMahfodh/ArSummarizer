( وَإِنْ قَالَ ) الْمُقِرُّ ( وَصَّى لَهُ بِأَلْفٍ مِنْ ثَمَنِهِ بِيعَ ) الْعَبْدُ ( وَصُرِفَ لَهُ مِنْ ثَمَنِهِ أَلْفٌ ) عَمَلًا بِمُقْتَضَى الْوَصِيَّةِ ( وَإِنْ أَرَادَ أَنْ يُعْطِيَهُ ) الْمُقِرُّ ( أَلْفًا مِنْ مَالِهِ مِنْ غَيْرِ ثَمَنِ الْعَبْدِ لَمْ يَلْزَمْهُ قَوْلُهُ لِأَنَّ الْمُوصَى لَهُ يَتَعَيَّنُ حَقُّهُ فِي ثَمَنِهِ ) فَلَا يَلْزَمُهُ أَنْ يَعْتَاضَ عَنْهُ كَالْوَصِيَّةِ بِالْعَبْدِ نَفْسِهِ ( وَإِنْ فُسِّرَ ذَلِكَ ) أَيْ لَهُ فِي هَذَا الْعَبْدِ أَلْفٌ ( بِأَلْفٍ مِنْ جِنَايَةٍ جَنَاهَا الْعَبْدُ فَتَعَلَّقَتْ بِرَقَبَتِهِ قُبِلَ ذَلِكَ ) مِنْهُ لِأَنَّهُ مُحْتَمَلٌ ( وَلَهُ بَيْعُ الْعَبْدِ وَدَفْعُ الْأَلْفِ مِنْ ثَمَنِهِ ) وَلَهُ دَفْعُ الْأَلْفِ مِنْ مَالِهِ وَلَهُ تَسْلِيمُ الْعَبْدِ فِي ذَلِكَ كَمَا تَقَدَّمَ فِي الْعَبْدِ الْجَانِي ( وَإِنْ قَالَ ) الْمُقِرُّ ( أَرَدْتُ ) بِقَوْلِي لَهُ فِي هَذَا الْعَبْدِ أَلْفٌ ( أَنَّهُ رَهْنٌ عِنْدَهُ ) أَيْ الْمُقَرِّ لَهُ ( بِأَلْفٍ قُبِلَ ) مِنْهُ ذَلِكَ لِأَنَّهُ مُحْتَمَلٌ لِتَعَلُّقِ الدَّيْنِ بِالرَّهْنِ .( 23 / 337 )
