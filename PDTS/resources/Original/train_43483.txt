( قَوْلُهُ بِخِلَافِهِ ) أَيْ الدَّيْنِ ( فِي الْأَوَّلِ ) أَيْ فِي الْإِبْرَاءِ ( قَوْلُهُ قَبْلَ الْحَوَالَةِ ) مَقُولُ الْقَوْلِ ( قَوْلُهُ مِنْهُ ) أَيْ الْمُحَالِ عَلَيْهِ ( قَوْلُهُ بِأَنَّهُ ) أَيْ الْإِبْرَاءَ ( قَوْلُهُ لَوْ أَقَامَ ) أَيْ الْمُحْتَالُ ( قَوْلُهُ وَلَيْسَ هَذَا ) أَيْ إقَامَةُ كُلٍّ مِنْ الْمُحْتَالِ وَالْمُحَالِ عَلَيْهِ الْبَيِّنَةَ ( قَوْلُهُ بِهِ ) أَيْ بِالْإِبْرَاءِ الْمُطْلَقِ ( قَوْلُهُ فَاسِدَانِ ) الْأَوْلَى التَّأْنِيثُ .( 21 / 173 )
