وَ ( لَا ) يُقْبَلُ فِيهِ قَوْلُ ( مَسْتُورٍ وَلَا مُمَيِّزٍ ) لِعَدَمِ الثِّقَةِ بِقَوْلِهِ ( فِي الْغَيْمِ وَالصَّحْوِ ) مُتَعَلِّق بِ يُقْبَلُ وَالْمِصْرِ وَخَارِجِهِ ( وَلَوْ كَانَ ) الرَّائِي ( فِي جَمْعٍ كَثِيرٍ ) وَلَمْ يَرَهُ مِنْهُمْ غَيْرُهُ لِمَا سَبَقَ ( وَهُوَ خَبَرٌ ) لَا شَهَادَةٌ ( فَيُصَامُ بِقَوْلِهِ ) : رَأَيْتُ الْهِلَالَ وَلَوْ لَمْ يَقُلْ : أَشْهَدُ أَوْ شَهِدْتُ أَنِّي رَأَيْتُهُ .( 5 / 455 )
