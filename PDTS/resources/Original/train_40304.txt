( وَإِنْ كَانَ الْمُدَّعِي ) بِهِ ( جَارِيَةً سُلِّمَتْ إلَى أَمِينٍ يُوصِلُهَا ) لِلْحَاكِمِ الْكَاتِبِ احْتِيَاطًا لِلْفُرُوجِ فَإِذَا شُهِدَتْ الْبَيِّنَةُ عَلَى عَيْنهَا سُلِّمَتْ لِلْمُدَّعِي ( وَإِنْ لَمْ يَثْبُت لَهُ ) أَيْ لِلْمُدَّعِي بِمَا ذَكَرَهُ ( مَا ادَّعَاهُ ) كَمَا تَقَدَّمَ ( لَزِمَهُ رَدُّهُ وَمُؤْنَتُهُ ) أَيْ الرَّدِّ وَنَفَقَةِ الْحَيَوَانِ أَوْ الْعَبْدِ أَوْ الْجَارِيَةِ ( مُنْذُ تَسَلَّمَهُ ) الْمُدَّعِي ( فَهُوَ ) أَيْ الْمُدَّعِي ( فِيهِ ) أَيْ فِيمَا قَبَضَهُ لِتَشْهَدَ الْبَيِّنَةُ عَلَى عَيْنِهِ إذَا لَمْ تَثْبُتْ لَهُ ( كَغَاصِبٍ فِي ضَمَانِهِ ) إنْ تَلِفَ ( وَضَمَانِ نَقْصِهِ ) إنْ نَقَصَ ( وَ ) ضَمَانِ ( مَنْفَعَتِهِ ) وَهُوَ مَعْنَى قَوْلِهِ .( 22 / 351 )
