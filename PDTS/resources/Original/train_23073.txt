وَ ( لَا ) يَكْفُرُ وَلَا يُقْتَلُ ( مَنْ سَحَرَ بِأَدْوِيَةٍ وَتَدْخِينٍ وَسَقْيٍ شَيْئًا يَضُرُّ ) لِأَنَّ الْأَصْلَ الْعِصْمَةُ وَلَمْ يَثْبُتْ مَا يُزِيلُهَا ( وَيُعَزَّرُ ) سَاحِرٌ بِذَلِكَ ( بَلِيغًا ) لِيَنْكَفَّ هُوَ وَمِثْلُهُ عَنْهُ ( وَلَا ) يَكْفُرُ ( مَنْ يُعَزِّمُ عَلَى الْجِنِّ وَيَزْعُمُ أَنَّهُ يَجْمَعُهَا وَتُطِيعُهُ ) وَذَكَرَهُ أَبُو الْخَطَّابِ فِي السَّحَرَةِ الَّذِينَ يُقْتَلُونَ ( وَلَا ) يَكْفُرُ ( كَاهِنٌ ) أَيْ : مَنْ لَهُ رِدْءٌ مِنْ الْجِنِّ يَأْتِيهِ بِالْأَخْبَارِ ( وَلَا ) يَكْفُرُ ( عَرَّافٌ ) أَيْ مِنْ يَحْدُسُ أَوْ يَتَخَرَّصُ ( وَلَا ) يَكْفُرُ ( مُنَجِّمٌ ) أَيْ : نَاظِرٌ فِي النُّجُومِ يَسْتَدِلُّ بِهَا عَلَى الْحَوَادِثِ فَإِنْ أَوْهَمَ قَوْمًا بِطَرِيقَتِهِ أَنَّهُ يَعْلَمُ الْغَيْبَ فَلِلْإِمَامِ قَتْلُهُ لِسَعْيِهِ بِالْفَسَادِ .
