لَا يَحِلُّ لَهُ ) أَيْ لِبَقَاءِ عِدَّةِ الْأَوَّلِ ( قَوْلُهُ : وَلَا تَعْرِيضَ لِلرَّجْعِيَّةِ ) أَيْ وَلَوْ بِإِذْنِ الزَّوْجِ ( قَوْلُهُ : وَالْإِسْلَامِ ) أَمَّا فِي الرَّجْعَةِ فَظَاهِرٌ وَأَمَّا فِي الْإِسْلَامِ فَهُوَ بِمَعْنَى أَنَّهُ يَتَبَيَّنُ بِإِسْلَامِهَا أَنَّهَا لَمْ تَخْرُجْ عَنْ الزَّوْجِيَّةِ ( قَوْلُهُ : بِغَيْرِ جِمَاعٍ ) أَيْ أَمَّا بِهِ فَيَحْرُمُ كَأَنْ يَقُولَ عِنْدِي جِمَاعٌ يَرْضَى ( قَوْلُهُ : فَلَعَلَّ الْمُصَنِّفَ يَرْتَضِيهِ ) أَيْ جَرَيَانَ الْخِلَافِ ( قَوْلُهُ : وَالْعِدَّةُ عَنْ شُبْهَةٍ ) هَذَا عُلِمَ مِنْ قَوْلِهِ قَبْلُ وَمَا أَوْرَدَ عَلَى مَفْهُومِهِ مِنْ الْمُعْتَدَّةِ عَنْ وَطْءِ شُبْهَةٍ إلَخْ ، وَلَعَلَّ حِكْمَةَ ذِكْرِهِ هُنَا التَّنْبِيهُ عَلَى حِكَايَةِ الْخِلَافِ فِيهِ ( قَوْلُهُ : مَا يَقْطَعُ بِهِ ) أَيْ بِسَبَبِهِ ( قَوْلُهُ : وَهُوَ بِالْجِمَاعِ ) أَيْ التَّعْرِيضُ بِالْجِمَاعِ( 20 / 239 )
