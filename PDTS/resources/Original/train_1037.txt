قَالَ الْبَنْدَنِيجِيُّ : إنَّهَا نَوْعٌ مِنْهُ وَهِيَ الْأُنْثَى مِنْ الْحَرَابِيّ وَالذَّكَرُ حَرْبِيٌّ وَ ( حَوْصَلِ ) هُوَ طَائِرٌ أَبْيَضُ أَكْبَرُ مِنْ الْكُرْكِيِّ ذُو حَوْصَلَةٍ عَظِيمَةٍ يُتَّخَذُ مِنْهَا فَرْوٌ وَيُقَالُ لَهُ حَوَاصِلُ بِصِفَةِ الْجَمْعِ وَ ( زَاغٍ ) هُوَ غُرَابُ الزَّرْعِ أَسْوَدُ صَغِيرٌ وَقَدْ يَكُونُ مُحْمَرَّ الْمِنْقَارِ وَالرِّجْلَيْنِ ( وَيَرْبُوعٍ ) هُوَ دُوَيْبَّةٌ تُشْبِهُ الْفَأْرَ لَكِنَّهُ قَصِيرُ الْيَدَيْنِ طَوِيلُ الرِّجْلَيْنِ أَبْيَضُ الْبَطْنِ أَغْبَرُ الظَّهْرِ بِطَرَفِ ذَنَبِهِ شَعَرَاتٌ ( وَوَبْرٍ ) بِإِسْكَانِ الْمُوَحَّدَةِ دُوَيْبَّةٌ أَصْغَرُ مِنْ الْهِرِّ كَحْلَاءُ الْعَيْنِ لَا ذَنَبَ لَهَا وَ ( دُلْدُلِ ) بِإِسْكَانِ اللَّامِ بَيْنَ الْمُهْمَلَتَيْنِ الْمَضْمُومَتَيْنِ دَابَّةٌ قَدْرُ السَّخْلَةِ ذَاتُ شَوْكٍ طِوَالٍ يُشْبِهُ السِّهَامَ وَفِي الصِّحَاحِ أَنَّهُ عَظِيمُ الْقَنَافِذِ ( وَبِنْتِ عِرْسٍ ) بِكَسْرِ الْعَيْنِ وَيُعَبَّرُ عَنْهَا بِابْنِ عِرْسٍ كَمَا عَبَّرَ بِهِ الْحَاوِي دُوَيْبَّةٌ رَقِيقَةٌ تُعَادِي الْفَأْرَ تَدْخُلُ جُحْرَهُ وَتُخْرِجُهُ وَ ( قُنْفُذٍ )( 19 / 178 )
