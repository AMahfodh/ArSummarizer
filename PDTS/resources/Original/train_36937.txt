( وَتُعْتَبَرُ نِيَّةُ وَاهِبٍ وَ ) هُوَ الزَّوْجُ وَنِيَّةُ ( مَوْهُوبٍ ) لَهُ عِنْدَ قَبُولِهِ ؛ لِأَنَّهُ كِنَايَةٌ ، فَاعْتُبِرَتْ النِّيَّةُ فِيهِ كَسَائِرِ الْكِنَايَاتِ ( وَتَقَعُ ) بِقَوْلِهِ وَهَبْتُك لِنَفْسِك أَوْ أَهْلِك إذَا قَبِلَ وَنَوَى أَحَدُهُمَا أَكْثَرَ مِنْ طَلْقَةٍ ، وَالْآخَرُ طَلْقَةً ، أَوْ نَوَى أَحَدُهُمَا طَلْقَتَيْنِ وَالْآخَرُ طَلْقَةً ( أَقَلُّهُمَا ) أَيْ : الْعَدَدَيْنِ ؛ لِاتِّفَاقِهِمَا عَلَيْهِ دُونَ مَا زَادَ ( وَإِنْ نَوَى ) زَوْجٌ ( بِ ) قَوْلِهِ ( وَهَبْتُك ) لِنَفْسِك أَوْ أَهْلِك أَوْ لِزَيْدٍ مَثَلًا الطَّلَاقَ فِي الْحَالِ ؛ وَقَعَ ( أَوْ ) نَوَى بِقَوْلِهِ ( أَمْرُك بِيَدِك ) الطَّلَاقُ فِي الْحَالِ وَقَعَ ؟ ( أَوْ ) نَوَى بِقَوْلِهِ ( اخْتَارِي نَفْسَك الطَّلَاقَ فِي الْحَالِ وَقَعَ ) مُؤَاخَذَةً لَهُ بِإِقْرَارِهِ .( 16 / 68 )
