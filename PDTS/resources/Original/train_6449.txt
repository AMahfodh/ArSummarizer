( وَبَرِئَ صَاحِبُهُ مِنْ ضَمَانِهِ ) لِبَرَاءَةِ الْأَصْلِ ، فَيَبْرَأُ الْفَرْعُ ( وَبَقِيَ عَلَيْهِ ) أَيْ : عَلَى صَاحِبِهِ ( خَمْسُمِائَةٍ ) وَهِيَ مَا كَانَ عَلَيْهِ أَصَالَةً ؛ لِأَنَّهُ لَمْ يُوجَدْ مَا يُسْقِطُهَا عَنْهُ ( وَإِنْ قَضَاهُ ) أَيْ : رَبُّ الْحَقِّ أَحَدَهُمَا أَيْ : أَحَدَ الرَّجُلَيْنِ ( خَمْسَمِائَةٍ أَوْ أَبْرَأَهُ ) أَيْ : أَحَدَهُمَا ( الْغَرِيمَ مِنْهَا ) أَيْ : مِنْ خَمْسِمِائَةٍ مِائَةٍ .
