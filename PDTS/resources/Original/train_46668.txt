( وَإِنْ سَقَطَ طَائِرٌ غَيْرِهِ فِي دَارِهِ لَمْ يَلْزَمْهُ ) أَيْ : رَبَّ الدَّارِ ( حِفْظُهُ وَلَا إعْلَامُ صَاحِبِهِ ) ؛ لِأَنَّهُ لَمْ يَزَلْ مُمْتَنِعًا ( إلَّا أَنْ يَكُونَ ) الطَّيْرُ ( غَيْرَ مُمْتَنِعٍ ) كَالْمَقْصُوصِ جَنَاحُهُ ( فَكَالثَّوْبِ ) إنْ لَمْ يَعْرِفْ صَاحِبَهُ فَلُقَطَةٌ وَإِنَّ عَرَفَهُ أَعْلَمَهُ فَوْرًا وَإِلَّا ضَمِنَ .( 13 / 217 )
