( وَلَوْ ) كَانَ الْمَخُوفُ مِنْهُ ( فُسَّاقًا ) يَفْسُقُونَ بِطَالِبِ الْمَاءِ ( غَيْرَ جَبَانٍ ) يَخَافُ بِلَا سَبَبٍ يُخَافُ مِنْهُ ( أَوْ ) لَمْ يَخَفْ بِقَصْدِهِ عَلَى ( مَالِهِ ) كَشُرُودِ دَابَّتِهِ ، أَوْ عَلَى أَهْلِهِ مِنْ لِصٍّ أَوْ سَبُعٍ أَوْ نَحْوِهِ ( لَزِمَهُ قَصْدُهُ ) أَيْ الْمَاءِ لِتَمَكُّنِهِ مِنْهُ بِلَا ضَرَرٍ ( وَإِلَّا ) بِأَنْ خَافَ شَيْئًا مِمَّا تَقَدَّمَ ( تَيَمَّمَ ) وَسَقَطَ طَلَبُهُ ، لِعَدَمِ تَمَكُّنِهِ مِنْ اسْتِعْمَالِهِ فِي الْوَقْتِ بِلَا ضَرَرٍ فَأَشْبَهَ عَادِمَهُ إعَادَةً وَلَيْسَ لَهُ تَأْخِيرُ الصَّلَاةِ إلَى( 1 / 230 )
