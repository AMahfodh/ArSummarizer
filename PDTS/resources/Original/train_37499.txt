( وَإِنْ ذَبَحَ كِتَابِيٌّ مَا يَحْرُمُ عَلَيْهِ يَقِينًا كَذِي الظُّفْرِ وَهِيَ الْإِبِلُ وَالنَّعَامُ وَالْبَطُّ وَمَا لَيْسَ بِمَشْقُوقِ الْأَصَابِعِ ) لَمْ يَحْرُمْ عَلَيْنَا لِأَنَّ قَصْدَهُ لِحِلِّهِ غَيْرُ مُعْتَبَرٍ ( أَوْ ) ذَبَحَ كِتَابِيٌّ ( مَا زَعَمَ أَنَّهُ يَحْرُمُ عَلَيْهِ وَلَمْ يَثْبُتْ عِنْدنَا تَحْرِيمُهُ عَلَيْهِ كَحَالِ الرِّئَةِ وَنَحْوِهَا لَمْ يَحْرُمْ عَلَيْنَا ) لِأَنَّهُ مِنْ أَهْلِ الذَّكَاةِ وَذَبَحَ مَا يَحِلَّ لَنَا أَشْبَهَ الْمُسْلِمَ ( وَمَعْنَاهُ ) أَيْ حَالِ الرِّئَةِ ( أَنَّ الْيَهُودَ إذَا وَجَدُوا الرِّئَةَ لَاصِقَةً بِالْأَضْلَاعِ امْتَنَعُوا مِنْ أَكْلِهَا زَاعِمِينَ تَحْرِيمَهَا وَيُسَمُّونَهَا اللَّازِقَةَ وَإِنْ وَجَدُوهَا غَيْرَ لَاصِقَةٍ أَكَلُوهَا وَإِنْ ذَبَحَ ) الْكِتَابِيُّ ( حَيَوَانًا غَيْرَهُ ) أَيْ غَيْرَ مَا يَحْرُمُ عَلَيْهِ ( مِمَّا يَحِلُّ لَهُ لَمْ تَحْرُمْ عَلَيْنَا الشُّحُومُ الْمُحَرَّمَةُ عَلَيْهِمْ وَهِيَ شَحْمُ الثَّرْبِ ) بِفَتْحِ الثَّاءِ الْمُثَلَّثَةِ وَسُكُونِ الرَّاءِ ( شَحْمٌ رَقِيقٌ يَغْشَى الْكَرِشَ وَالْأَمْعَاءِ وَشَحْمِ الْكُلْيَتَيْنِ ) وَاحِدُهَا كُلْوَةٌ وَكُلْيَةٌ بِضَمِّ الْكَافِ فِيهِمَا وَالْجَمْعُ كُلُيَاتٌ وَكُلًى ( وَلَنَا ) مَعْشَرَ الْمُسْلِمِينَ .
