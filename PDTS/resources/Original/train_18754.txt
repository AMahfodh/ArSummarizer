( وَوَلَدُهُ ) أَيْ السَّيِّدِ ( مِنْهَا ) أَيْ مِنْ جَارِيَة مُكَاتَبِهِ أَوْ مُكَاتَبَتِهِ ( حُرٌّ يَلْحَقُهُ نَسَبُهُ ) لِشُبْهَةِ الْمِلْكِ ( وَتَصِيرُ أُمَّ وَلَدٍ لَهُ ) لِمَا تَقَدَّمَ ( وَعَلَيْهِ قِيمَتُهَا لِسَيِّدِهَا ) لِأَنَّهُ فَوَّتَهَا عَلَيْهِ إذْ الِاسْتِيلَادُ كَالْإِتْلَافِ ( وَلَا يَجِبُ عَلَيْهِ قِيمَةُ الْوَلَدِ ) مِنْ أَمَةِ مُكَاتَبِهِ أَوْ مُكَاتَبَتِهِ لِأَنَّ وَلَدَ السَّيِّدِ كَجُزْءٍ مِنْهُ فَلَا يَجِبُ عَلَيْهِ أَنْ يَدْفَعَ قِيمَتَهُ لِرَقِيقِهِ وَلِأَنَّهُ انْعَقَدَ حُرًّا .( 16 / 312 )
