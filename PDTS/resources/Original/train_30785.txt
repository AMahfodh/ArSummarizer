( وَلَا ) يَرْجِعُ ( إلَى تَنْبِيهِ فَاسِقَيْنِ ) لِعَدَمِ قَبُولِ خَبَرِهِمَا ( وَلَا إذَا نَبَّهَهُ وَاحِدٌ ) نَصَّ عَلَيْهِ لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ لَمْ يَرْجِع إلَى قَوْلِ ذِي الْيَدَيْنِ وَحْدَهُ ( إلَّا أَنْ يَتَيَقَّنَ صَوَابُهُ ) فَيَعْمَلُ بِيَقِينِهِ لَا بِتَنْبِيهِهِ .( 3 / 159 )
