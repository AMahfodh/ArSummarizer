( أَوْ ) تَزَوَّجَتْ امْرَأَةُ شَخْصٍ غَائِبٍ ( بِشَهَادَةِ ) رَجُلَيْنِ ( غَيْرِ عَدْلَيْنِ ) عَلَى مَوْتِهِ ( فَيُفْسَخُ ) لِعَدَمِ عَدَالَةِ الشُّهُودِ فَتَزَوَّجَتْ ثَالِثًا بِشَهَادَةِ عَدْلَيْنِ وَدَخَلَ بِهَا ( ثُمَّ يَظْهَرُ أَنَّهُ ) أَيْ نِكَاحُ الْمُتَزَوِّجِ بِشَهَادَةِ غَيْرِ الْعَدْلَيْنِ ( كَانَ عَلَى الصِّحَّةِ ) لِكَوْنِ الْعُدُولِ أَرَّخُوا مَوْتَهُ بِتَارِيخٍ مُتَقَدِّمٍ تَنْقَضِي فِيهِ عِدَّتُهَا قَبْلَ نِكَاحِهِ فَتُرَدُّ إلَيْهِ وَلَا يُفِيتُهَا دُخُولُ الثَّالِثِ فَقَوْلُهُ : ( فَلَا تَفُوتُ ) وَاحِدَةٌ مِنْ السَّبْعِ ( بِدُخُولٍ ) جَوَابُ إمَّا .
