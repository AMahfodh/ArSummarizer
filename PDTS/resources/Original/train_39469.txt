( وَلَوْ أَنَّ بَابَهُ فِي آخِرِ دَرْبٍ غَيْرِ نَافِذٍ مَلَكَ نَقْلَهُ ) أَيْ الْبَابِ ( إلَى أَوَّلِهِ ) أَيْ الدَّرْبِ لِأَنَّهُ تَرَكَ بَعْضَ حَقِّهِ لِأَنَّ لَهُ الِاسْتِطْرَاقَ إلَى آخِرِهِ ( إنْ لَمْ يَحْصُلْ مِنْهُ ضَرَرٌ كَفَتْحِهِ مُقَابِلَ بَابِ غَيْرِهِ وَنَحْوِهِ ) كَفَتْحِهِ عَالِيًا يَصْعَدُ إلَيْهِ بِسُلَّمٍ يُشْرِفُ مِنْهُ عَلَى دَارِ غَيْرِهِ ( وَ ) إنْ كَانَ بَابُهُ فِي أَوَّلِ الدَّرْبِ أَوْ وَسَطِهِ ( لَمْ يَمْلِكْ نَقْلَهُ إلَى دَاخِلٍ مِنْهُ ) تِلْقَاءَ صَدْرِ الزُّقَاقِ لِأَنَّهُ يُقَدِّمُ بَابَهُ إلَى مَوْضِعِ الِاسْتِطْرَاقِ لَهُ فِيهِ ( إنْ لَمْ يَأْذَنْ ) لَهُ ( مَنْ فَوْقَهُ ) أَيْ مَنْ هُوَ دَاخِلٌ عَنْهُ فَإِنْ أَذِنَ جَازَ .
