( وَ ) حَرُمَ ( عَقْرُ دَابَّةٍ شَقَّ نَقْلُهَا ) إلَى دَارِنَا ( فَتُذْبَحُ وَتُحْرَقُ ) بَعْدَهُ إذْ لَا يُعَذِّبُ بِالنَّارِ إلَّا رَبُّهَا ( كَمَا تُحْرَقُ أَسْلِحَةٌ وَأَمْتِعَةٌ تَعَذَّرَ نَقْلُهَا وَمَا لَا يُحْرَقُ مِنْهَا ) كَحَدِيدٍ ( يُدْفَنُ بِمَوْضِعٍ خَفِيٍّ ) وَتُكْسَرُ أَوَانَيْهِمْ وَتُرَاقُ أَدْهَانُهُمْ مُغَايَظَةً لَهُمْ
