وَالسَّلَمُ شَرْعًا ( عَقْدٌ عَلَى ) مَا يَصِحُّ بَيْعُهُ ( مَوْصُوفٌ ) بِمَا يَضْبِطُهُ ( بِذِمَّةٍ ) ، وَهِيَ وَصْفٌ يَصِيرُ بِهِ الْمُكَلَّفُ أَهْلًا لِلْإِلْزَامِ وَالِالْتِزَامِ ( مُؤَجَّلٌ ) أَيْ : الْمَوْصُوفُ ( بِثَمَنٍ ) - مُتَعَلِّقٌ بِعَقْدٍ - ( مَقْبُوضٍ ) ذَلِكَ الثَّمَنُ ( بِمَجْلِسِ عَقْدٍ ) .
