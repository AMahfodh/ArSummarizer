( فَإِذَا ظَاهَرَ ) سَيِّدٌ مِنْ أَمَتِهِ ( أَوْ ) مِنْ ( أُمّ وَلَدِهِ أَوْ قَالَ لَهَا ) أَيْ لِأَمَتِهِ أَوْ لِأُمِّ وَلَدِهِ ( أَنْتِ عَلَيَّ حَرَامٌ فَعَلَيْهِ كَفَّارَةُ يَمِينٍ ) كَتَحْرِيمِ سَائِرِ مَالَهُ وَقَالَ نَافِعٌ { حَرَّمَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ جَارِيَتَهُ فَأَمَرَهُ اللَّهُ أَنْ يُكَفِّرَ يَمِينَهُ } .( 19 / 23 )
