( وَلَا بَأْسَ بِلُبْسِ الْفِرَاءِ ) بِكَسْرِ الْفَاءِ مَمْدُودًا جَمْعُ فَرْوٍ بِغَيْرِهَا قَالَهُ الْجَوْهَرِيُّ وَأَثْبَتَهَا ابْنُ فَارِسٍ وَيَدُلُّ لَهُ الْحَدِيثُ الْآتِي ( إذَا كَانَتْ ) الْفِرَاءُ ( مِنْ جِلْدٍ مَأْكُولٍ مُذَكًّى مُبَاحٍ وَتَصِحُّ الصَّلَاةُ فِيهَا ) كَسَائِرِ الطَّاهِرَاتِ وَتَقَدَّمَ فِي الْآنِيَةِ : يَحْرُمُ لُبْسُ جُلُودِ السِّبَاعِ وَإِنَّهُ يُبَاحُ دَبْغُ جِلْدٍ نَجِسٍ بِمَوْتٍ وَاسْتِعْمَالُهُ بَعْدَهُ فِي يَابِسٍ ( وَلَا تَصِحُّ ) الصَّلَاةُ ( فِي غَيْرِ ذَلِكَ ) أَيْ : غَيْرِ جِلْدٍ مُذَكًّى ( كَجِلْدِ ثَعْلَبٍ وَسَمُّورٍ وَفَنَكٍ وَقَاقَمٍ وَسِنَّوْرٍ ، وَسِنْجَابٍ وَنَحْوِهِ ) كَذِئْبٍ وَنَمِرٍ ( وَلَوْ ذُكِّيَ ) أَوْ دُبِغَ لِأَنَّهُ لَا يَطْهُرُ بِذَلِكَ كَلَحْمِهِ .( 2 / 348 )
