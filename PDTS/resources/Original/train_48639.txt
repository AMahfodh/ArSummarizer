( وَمَنْ كَفَلَهُ اثْنَانِ ) مَعًا أَوَّلًا ( فَسَلَّمَهُ أَحَدُهُمَا لَمْ يَبْرَأْ الْآخَرُ ) لِانْحِلَالِ إحْدَى الْوَثِيقَتَيْنِ بِلَا اسْتِيفَاءٍ فَلَا تَنْحَلُّ الْأُخْرَى ، كَمَا لَوْ أَبْرَأَ أَحَدُهُمَا أَوْ انْفَكَّ أَحَدُ الرَّهْنَيْنِ بِلَا قَضَاءٍ ( وَإِنْ سَلَّمَ ) مَكْفُولٌ ( نَفْسَهُ بَرِئَا ) أَيْ الْكَفِيلَانِ لِأَدَاءِ الْأَصِيلِ مَا عَلَيْهِمَا ( وَإِنْ كَفَلَ كُلَّ وَاحِدٍ مِنْهُمَا ) أَيْ الْكَفِيلِينَ ( شَخْصٌ آخَرُ فَأَحْضَرَ ) هَذَا الْآخَرَ ( الْمَكْفُولَ بِهِ ) أَيْ مَكْفُولَ مَكْفُولِهِ ( بَرِئَ ) مَنْ أَحْضَرَهُ ( وَهُوَ مَنْ تَكَفَّلَ بِهِ ) مِنْ الْكَفِيلِينَ لِأَدَائِهِ مَا عَلَيْهِمَا كَمَا لَوْ سَلَّمَهُ مَنْ تَكَفَّلَ بِهِ ( فَقَطْ ) أَيْ دُونَ الْكَفِيلِ الثَّانِي وَكَفِيلِهِ لِمَا تَقَدَّمَ وَإِنْ تَكَفَّلَ ثَلَاثَةٌ بِوَاحِدٍ وَكُلٌّ مِنْهُمْ كَفِيلٌ بِصَاحِبَيْهِ صَحَّ وَمَتَى سَلَّمَهُ أَحَدُهُمْ بَرِئَ هُوَ وَصَاحِبَاهُ مِنْ كَفَالَتِهِمَا بِهِ خَاصَّةً ؛ لِأَنَّهُ أَصْلٌ لَهُمَا وَهُمَا فَرْعَانِ لَهُ وَيَبْقَى عَلَى كُلِّ وَاحِدٍ مِنْهُمَا الْكَفَالَةُ بِالدَّيْنِ ؛ لِأَنَّهُمَا أَصْلَانِ فِيهَا( 5 / 299 )
