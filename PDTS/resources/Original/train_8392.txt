( وَإِنْ نَقَصَتْ ) قِيمَةُ الْمَغْصُوبِ ( لِمَرَضٍ ثَمَّ عَادَتْ ) الْقِيمَةُ ( بِبُرْئِهِ ) رَدَّهُ وَلَا شَيْءَ عَلَيْهِ ( أَوْ ابْيَضَّتْ عَيْنُهُ ) أَيْ الْمَغْصُوبِ مِنْ عَبْدٍ أَوْ أَمَةٍ ( ثَمَّ زَالَ بَيَاضُهَا وَنَحْوُهُ ) بِأَنْ نَسِيَ صَنْعَةً فَنَقَصَتْ قِيمَتُهُ ثُمَّ تَعَلَّمَهَا ( رَدَّهُ ) الْغَاصِبُ ( وَلَمْ يَلْزَمْهُ شَيْءٌ ) لِأَنَّ الْقِيمَةَ لَمْ تَنْقُصْ فَلَمْ يَلْزَمْهُ شَيْءٌ .
