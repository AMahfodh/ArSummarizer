( وَمِثْلُهُ ) ب أَيْ التَّوْكِيلِ فِيمَا تَقَدَّمَ ( تَوَكُّلٌ ) فَلَا يَصِحَّ أَنْ يَتَوَكَّل فِي شَيْءٍ إلَّا مَنْ يَصِحُّ مِنْهُ لِنَفْسِهِ ( فَلَا يَصِحُّ أَنْ يُوجِبَ نِكَاحًا ) عَنْ غَيْرِهِ ( مَنْ لَا يَصِحُّ مِنْهُ ) إيجَادٌ ( لِمَوْلِيَّتِهِ ) لِنَحْوِ فِسْقٍ ؛ لِأَنَّهُ إذَا لَمْ يَجُزْ أَنْ يَتَوَلَّاهُ أَصَالَةً لَمْ يَجُزْ بِالنِّيَابَةِ كَالْمَرْأَةِ ( وَلَا ) يَصِحُّ أَنْ ( يَقْبَلَهُ ) أَيْ النِّكَاحَ لِغَيْرِهِ ( مَنْ لَا يَصِحُّ مِنْهُ ) قَبُولُهُ ( لِنَفْسِهِ ) كَكَافِرٍ يَتَوَكَّلُ فِي قَبُولِ نِكَاحِ مُسْلِمَةٍ لِمُسْلِمٍ ( سِوَى ) قَبُولِ ( نِكَاحِ أُخْتِهِ وَنَحْوِهَا ) كَعَمَّتِهِ وَخَالَتِهِ وَحَمَاتِهِ ( لِأَجْنَبِيٍّ ) تَحِلُّ لَهُ .
