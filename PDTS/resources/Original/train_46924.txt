( وَلَا يَلْزَمُ ) الرَّهْنُ ( إلَّا بِقَبْضِهِ ) أَيْ الْمَرْهُونِ كَائِنًا ( مِمَّنْ يَصِحُّ عَقْدُهُ ) أَيْ مَنْ يَصِحُّ مِنْهُ عَقْدُ الرَّهْنِ يَصِحُّ مِنْهُ الْقَبْضُ ( وَتَجْرِي فِيهِ النِّيَابَةُ ) كَالْعَقْدِ ( لَكِنْ لَا يَسْتَنِيبُ رَاهِنًا ) لِأَنَّ الْوَاحِدَ لَا يَتَوَلَّى طَرَفَيْ الْقَبْضِ وَلَا ( عَبْدَهُ ) لِأَنَّ يَدَهُ كَيَدِهِ وَيُصَدَّقُ بِالْمَأْذُونِ لَهُ وَالْمُدَبَّرِ وَمِثْلُهُ أُمُّ الْوَلَدِ .
