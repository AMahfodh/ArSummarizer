( قَوْلُهُ كَقَوْلِهِ غَصَبْتهَا إلَخْ ) أَوْ سَرَقْتهَا ( قَوْلُهُ كَمَا لَوْ أَقَرَّ بِدَيْنِ إتْلَافٍ أَوْ بِدَيْنِ مُعَامَلَةٍ إلَخْ ) مَتَى قَبِلَ إقْرَارَهُ فَطَلَبَ الْغُرَمَاءُ تَحْلِيفَهُ عَلَى ذَلِكَ لَمْ يَحْلِفْ عَلَى الْأَصَحِّ ؛ إذْ لَا يُقْبَلُ رُجُوعُهُ عَنْهُ ( قَوْلُهُ كَذَا نَقَلَهُ الْأَذْرَعِيُّ وَأَقَرَّهُ ) الْأَصَحُّ أَنَّهُمْ لَيْسَ لَهُمْ تَحْلِيفُهُ كَمَا أَنَّهُ لَيْسَ لَهُ تَحْلِيفُهُمْ فِيمَا إذَا أَقَرَّ لَهُ الْمُفْلِسُ بِعَيْنٍ ، وَقُلْنَا بِأَنَّهُ لَا يُقْبَلُ إقْرَارُهُ بِهَا ( قَوْلُهُ قَالَ الْبُلْقِينِيُّ ) أَيْ وَغَيْرُهُ ( قَوْلُهُ أَوْ أَوْصَى لَهُ بِهِ ) أَيْ أَوْ وَرِثَهُ ( قَوْلُهُ فِيمَا لَوْ أَصَدَقَتْ الْمَحْجُورَةُ أَبَاهَا ) أَوْ أَوْصَى لَهَا بِهِ أَوْ وَرِثَتْهُ ( قَوْلُهُ فَيُحْتَمَلُ حَمْلُهُ عَلَى مَا إذَا اُشْتُهِرَ إلَخْ ) الثَّانِي هُوَ الظَّاهِرُ وَهُوَ مُقْتَضَى كَلَامِهِمْ .( 9 / 345 )
