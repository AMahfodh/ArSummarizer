( وَإِنْ أَكْرَاهَا ) ذُو الشُّبْهَةِ لِغَيْرِهِ ( سِنِينَ ) : الْمُرَادُ مَا فَوْقَ الْوَاحِدِ فَاسْتَحَقَّهَا مَالِكُهَا بَعْدَ الْإِجَارَةِ ( فَلِلْمَالِكِ الْفَسْخُ ) أَيْ فَسْخُ الْإِجَارَةِ ( بَعْدَ الْحَرْثِ ) فَأَوْلَى قَبْلَهُ وَلَهُ الْإِمْضَاءُ ( وَقِيلَ ) لَهُ ، إنْ اخْتَارَ الْفَسْخَ بَعْدَ الْحَرْثِ وَقَبْلَ الزَّرْعِ ( ادْفَعْ ) لِلْمُكْتَرِي ( أُجْرَتَهُ ) : أَيْ أُجْرَةَ الْحَرْثِ ( إنْ لَمْ يَزْرَعْ ، فَإِنْ أَبَى ) مِنْ دَفْعِ الْأُجْرَةِ ( قِيلَ لِلْمُكْتَرِي ) الَّذِي حَرَثَهَا : ( ادْفَعْ ) لِلْمُسْتَحِقِّ ( كِرَاءَ سَنَةٍ ) وَازْرَعْهَا ( وَإِلَّا ) تَدْفَعْ لَهُ كِرَاءَ سَنَةٍ ( أَسْلِمْهَا ) لَهُ ( بِلَا شَيْءٍ ) تَأْخُذُهُ مِنْهُ ، وَمِثْلُ ذَلِكَ فِيمَا لَوْ أَكْرَاهَا ذُو الشُّبْهَةِ سَنَةً فَقَطْ أَوْ اُسْتُحِقَّتْ بَعْدَ حَرْثِ ذِي الشُّبْهَةِ مِنْهُ .
