ابْنُ عَرَفَةَ وَالْأَظْهَرُ الْفَرْقُ بَيْنَ مَنْ يُخْشَى مِنْ خُرُوجِهَا مَفْسَدَةٌ وَمَنْ لَا وَيُشْتَرَطُ غَيْبَةُ الْأَصْلِ ( بِمَكَانٍ ) بَعِيدٍ ( لَا يَلْزَمُ ) الْأَصْلَ ( الْأَدَاءُ ) لِلشَّهَادَةِ عِنْدَ الْقَاضِي الَّذِي الْخُصُومَةُ عِنْدَهُ ( مِنْهُ وَ ) لَكِنْ ( لَا يَكْفِي ) فِي صِحَّةِ نَقْلِ الشَّهَادَةِ( 18 / 257 )
