وَقَوْلُهُ : ( عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ { لَا رَضَاعَ بَعْدَ حَوْلَيْنِ } وَلِأَبِي حَنِيفَةَ رَحِمَهُ اللَّهُ هَذِهِ الْآيَةُ ) يَعْنِي : قَوْله تَعَالَى { وَحَمْلُهُ وَفِصَالُهُ ثَلَاثُونَ شَهْرًا } ( وَوَجْهُهُ مَا ذَكَرَهُ أَنَّ اللَّهَ تَعَالَى ذَكَرَ شَيْئَيْنِ ) يَعْنِي الْحَمْلَ وَالْفِصَالَ ، ( وَضَرَبَ لَهُمَا مُدَّةً ) وَهُوَ قَوْله تَعَالَى { ثَلَاثُونَ شَهْرًا } وَكُلُّ مَا كَانَ كَذَلِكَ كَانَتْ الْمُدَّةُ لِكُلِّ وَاحِدَةٍ مِنْهُمَا بِكَمَالِهَا كَمَا فِي الْأَجَلِ الْمَضْرُوبِ لِلدَّيْنَيْنِ مِثْلُ أَنْ يَقُولَ : لِفُلَانٍ عَلَيَّ أَلْفُ دِرْهَمٍ وَخَمْسَةُ أَقْفِزَةِ حِنْطَةٍ إلَى شَهْرَيْنِ ، يَكُونُ الشَّهْرَانِ أَجَلًا لِكُلِّ وَاحِدٍ مِنْ الدَّيْنَيْنِ بِكَمَالِهِ إلَّا أَنَّهُ قَامَ الْمُنْقَصُ فِي أَحَدِهِمَا : يَعْنِي الْحَمْلَ وَهُوَ حَدِيثُ عَائِشَةَ : { الْوَلَدُ لَا يَبْقَى فِي بَطْنِ أُمِّهِ أَكْثَرَ مِنْ سَنَتَيْنِ وَلَوْ بِفَلْكَةِ مِغْزَلٍ } .
