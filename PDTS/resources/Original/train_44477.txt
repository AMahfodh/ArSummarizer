قَوْلُهُ : ( فَهُوَ مُخَالِفٌ ) أَيْ لِأَنَّهُ مُخَالِفٌ إلَخْ قَوْلُهُ : ( لَا يَقْرَبُ امْرَأَتَهُ ) بِفَتْحِ الرَّاءِ قَالَ تَعَالَى : { وَلَا تَقْرَبُوهُنَّ } قَوْلُهُ : ( لِأَنَّ الْيَمِينَ سَاقِطَةٌ عَنْهُ ) أَيْ مَرْفُوعَةٌ عَنْهُ أَيْ لِمُضِيِّ الزَّمَنِ الْمَحْلُوفِ عَلَيْهِ قَوْلُهُ : ( بِضَرْبِ الْمُدَّةِ بِنَفْسِهَا ) الْمُرَادُ بِضَرْبِهَا بِنَفْسِهَا حُسْبَانُهَا مِنْ غَيْرِ تَوَقُّفٍ عَلَى طَلَبٍ وَلَا ضَرْبِ الْقَاضِي .
