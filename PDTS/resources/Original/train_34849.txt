( وَمَنْ ادَّعَى شَيْئًا أَنَّهُ لَهُ ) أَيْ يَمْلِكُهُ ( الْآنَ لَمْ تُسْمَعْ بَيِّنَتُهُ ) إنْ شَهِدَتْ ( أَنَّهُ كَانَ لَهُ أَمْسِ أَوْ ) أَنَّهُ كَانَ ( فِي يَدِهِ ) أَمْسِ لِعَدَمِ التَّطَابُقِ ، ( حَتَّى تُبَيِّنَ ) الْبَيِّنَةُ ( سَبَبَ يَدِ الثَّانِي نَحْوِ غَاصِبَةٍ ) أَوْ مُسْتَعِيرَةٍ ، ( بِخِلَافِ مَا لَوْ شَهِدَتْ ) الْبَيِّنَةُ ( أَنَّهُ كَانَ مَلَكَهُ بِالْأَمْسِ اشْتَرَاهُ مِنْ رَبِّ الْيَدِ فَإِنَّهُ يُقْبَلُ ) .
