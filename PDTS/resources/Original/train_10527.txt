( وَلَوْ قَالَ عَبْدٌ ) لِزَوْجَتِهِ ( إذَا مَاتَ سَيِّدِي فَأَنْت طَالِقٌ طَلْقَتَيْنِ وَقَالَ سَيِّدُهُ ) لَهُ ( إذَا مِتُّ فَأَنْت حُرٌّ فَعَتَقَ بِهِ ) أَيْ بِمَوْتِ السَّيِّدِ بِأَنْ خَرَجَ مِنْ ثُلُثِ مَالِهِ ( فَالْأَصَحُّ أَنَّهَا لَا تَحْرُمُ ) عَلَيْهِ ( بَلْ لَهُ الرَّجْعَةُ ) فِي الْعِدَّةِ ( وَتَجْدِيدٌ ) بَعْدَ انْقِضَائِهَا ( قَبْلَ زَوْجٍ ) وَالثَّانِي تَحْرُمُ فَلَا تَحِلُّ لَهُ إلَّا بَعْدَ زَوْجٍ .
