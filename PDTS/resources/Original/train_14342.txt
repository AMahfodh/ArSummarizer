( قَوْلُهُ : أَنَّهُمْ إلَخْ ) أَيْ أَهْلَ الْخِبْرَةِ ( قَوْلُهُ : أَنَّهَا تُقْطَعُ إلَخْ ) أَيْ الشَّلَّاءُ بِالصَّحِيحَةِ جَوَابُ إذَا قَالُوا إلَخْ ( قَوْلُهُ : لِأَنَّ الْعِلَّةَ إلَخْ ) أَيْ عِلَّةَ عَدَمِ الْقَطْعِ وَالْجَارُ وَالْمَجْرُورُ مُتَعَلِّقٌ بِعَدَمِ الْإِفْهَامِ وَتَعْلِيلٌ لَهُ ( قَوْلُهُ : الْمَعْلُومُ إلَخْ ) نَعْتُ فَوَاتُ النَّفْسِ وَقَوْلُهُ عُلِمَتْ إلَخْ خَبَرُ ؛ لِأَنَّ إلَخْ ( قَوْلُهُ : فَدَفَعَتْ ) أَيْ تِلْكَ الْعِلَّةُ الْمَعْلُومَةُ مِنْ كَلَامِهِ ( قَوْلُهُ : ذَلِكَ الْإِيهَامَ ) لَعَلَّ وَجْهَ الْإِيهَامِ أَنَّ تَقْدِيمَ الِاسْتِثْنَاءِ عَلَى الْقَنَاعَةِ قَدْ يُتَوَهَّمُ مِنْهُ أَنَّهُ مَخْصُوصٌ بِمَا إذَا لَمْ تُوجَدْ فَلَوْ أَخَّرَهُ عَنْهَا لَكَانَ كَلَامُهُ نَصًّا فِي عُمُومِهِ وَعَدَمِ الِاخْتِصَاصِ بِذَلِكَ .( 36 / 490 )
