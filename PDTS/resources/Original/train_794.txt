( وَإِنْ غَصَبَ فَحْلَ غَيْرِهِ فَأَنْزَاهُ عَلَى شَاتِهِ فَالْوَلَدُ لَهُ ) أَيْ لِلْغَاصِبِ تَبَعًا لِلْأُمِّ وَلَا يَلْزَمُهُ أُجْرَةُ الْفَحْلِ لِأَنَّهُ لَا تَصِحُّ إجَارَتُهُ لِذَلِكَ ( لَكِنْ إنْ نَقَصَ ) الْفَحْلُ بِالْإِنْزَاءِ أَوْ غَيْرِهِ ( لَزِمَهُ ) أَيْ الْغَاصِبَ ( أَرْشُ نَقْصِهِ ) لِتَعَدِّيهِ .( 13 / 103 )
