( وَإِنْ ) قَالَ سَيِّدُ عَبْدَيْنِ فَأَكْثَرَ : إنْ ( مِتَّ فِي الْمُحَرَّمِ فَسَالِمٌ حُرٌّ وَ ) إنْ مِتَّ ( فِي صَفَرٍ فَغَانِمٌ حُرٌّ ) ثُمَّ مَاتَ ( وَأَقَامَ كُلٌّ ) مِنْ سَالِمٍ وَغَانِمٍ ( بَيِّنَةً بِمُوجَبِ عِتْقِهِ تَسَاقَطَتَا ) ؛ لِأَنَّ كُلًّا مِنْهُمَا تَنْفِي مَا تُثْبِتُهُ الْأُخْرَى ( وَرَقَّا ) لِجَوَازِ مَوْتِهِ فِي غَيْرِ الْمُحَرَّمِ وَصَفَرٍ لِمَا سَبَقَ ( كَمَا لَوْ لَمْ تَقُمْ بَيِّنَةٌ ) لِوَاحِدٍ مِنْهُمَا ( وَجُهِلَ وَقْتُهُ ) أَيْ : وَقْتُ مَوْتِهِ ، فَيَرِقَّانِ لِمَا سَبَقَ ، ( وَإِنْ عُلِمَ مَوْتُهُ فِي أَحَدِهِمَا ) أَيْ الشَّهْرَيْنِ وَجُهِلَ أَهُوَ الْمُحَرَّمُ أَوْ صَفَرٌ ( أُقْرِعَ ) بَيْنَ الْعَبْدَيْنِ ، فَمَنْ خَرَجَتْ لَهُ الْقُرْعَةُ عَتَقَ وَرَقَّ الْآخَرُ .
