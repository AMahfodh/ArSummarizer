( وَيَتَّجِهُ أَنْ يُزَادَ ) عَلَى أَقْسَامِ الْخِيَارِ قِسْمًا تَاسِعًا ، وَإِلَيْهِ الْإِشَارَةُ بِقَوْلِهِ : الْقِسْمُ ( التَّاسِعُ ) مِنْ أَقْسَامِ الْخِيَارِ ( خِيَارٌ يَثْبُتُ ) لِلْمُشْتَرِي ( لِفَقْدِ شَرْطٍ صَحِيحٍ أَوْ ) فَقْدِ شَرْطٍ ( فَاسِدٍ ) سَوَاءٌ كَانَ يُبْطِلُ الْعَقْدَ أَوْ لَا يُبْطِلُهُ ، فَإِنْ ادَّعَى أَحَدُهُمَا اشْتِرَاطَهُ ، وَأَنْكَرَهُ الْآخَرُ ، فَقَوْلُ مُنْكِرِهِ ( عَلَى مَا مَرَّ ) تَفْصِيلُهُ فِي خِيَارِ الشَّرْطِ .( 8 / 69 )
