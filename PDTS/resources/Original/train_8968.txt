( قَوْلُهُ وَلَوْ قَالَ زَوْجَانِ ) خَرَجَ بِهِ إقْرَارُ أَبِي الزَّوْجِ أَوْ أُمِّ أَحَدِهِمَا بِذَلِكَ فَلَا عِبْرَةَ بِهِ ( قَوْلُهُ : رَضَاعٌ مُحَرَّمٌ ) وَكَذَا مَعَ إسْقَاطِ مُحَرَّمٍ عَلَى مَا قَالَ حَجّ أَنَّهُ الَّذِي يُتَّجَهُ مِنْ خِلَافٍ لِلْمُتَأَخِّرِينَ : أَيْ لِأَنَّ الرَّضَاعَ إذَا أُطْلِقَ انْصَرَفَ لِلْمُحَرَّمِ ( قَوْلُهُ : وَإِنْ قَضَتْ الْعَادَةُ بِجَهْلِهِمَا ) وَمِنْهُ مَا لَوْ قَرُبَ عَهْدُ الْمُقِرِّ بِالْإِسْلَامِ لِلْعِلَّةِ الْمَذْكُورَةِ ( قَوْلُهُ : عَالِمَةً مُخْتَارَةً ) أَيْ وَكَانَتْ بَالِغَةً وَإِنْ لَمْ تَكُنْ رَشِيدَةً( 23 / 496 )
