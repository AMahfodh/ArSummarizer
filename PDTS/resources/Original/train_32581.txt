( وَتَلَفُهَا ) أَيْ : الْأُجْرَةِ ( بِلَا تَفْرِيطٍ بِيَدِ أَحَدِهِمَا ) عَلَيْهِمَا ؛ لِأَنَّ كُلًّا وَكِيلُ الْآخَرِ فِي قَبْضِهَا وَالطَّلَبِ بِهَا ( وَإِقْرَارُهُ ) أَيْ : إقْرَارُ أَحَدِهِمَا ( بِمَا فِي يَدِهِ ) يُقْبَلُ ( عَلَيْهِمَا ) ؛ لِأَنَّ الْيَدَ لَهُ فَقُبِلَ إقْرَارُهُ بِمَا فِيهَا بِخِلَافِ مَا فِي يَدِ شَرِيكِهِ أَوْ دَيْنٍ عَلَيْهِ ؛ لِأَنَّهُ لَا يَدَ لَهُ عَلَيْهِ .( 6 / 111 )
