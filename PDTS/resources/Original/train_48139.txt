( قُلْتُ ) إذَا عُرِفَ مَا مَرَّ ( فَلَا حَضَانَةَ لِمَحْرَمِ أُنْثَى دَلَتْ ) بِمَعْنَى أَدْلَتْ ( بِذَكَرٍ إنْ يُحْرَمْ إرْثًا ) كَبِنْتِ ابْنِ الْبِنْتِ وَأُمِّ أَبِي الْأُمِّ ( وَلَا لِلذَّكَرِ الَّذِي هُوَ لَمْ يَرِثْ ) سَوَاءٌ فِيهِ الْمَحْرَمُ وَغَيْرُهُ كَمَا قَالَ ( الْمَحْرَمُ ) كَالْخَالِ وَالْعَمِّ لِلْأُمِّ وَابْنِ الْأُخْتِ ( وَالْغَيْرُ ) أَيْ وَغَيْرُ الْمَحْرَمِ كَابْنَيْ الْخَالَةِ وَالْخَالِ وَابْنِ الْعَمِّ لِلْأُمِّ فِي ذَلِكَ ( سَوَا ) هَذَا كُلُّهُ فِي غَيْرِ الْمُمَيِّزِ .
