( وَلَا ) تَصِحُّ ( إمَامَةُ مُحْدِثٍ ) يَعْلَمُ ذَلِكَ ( وَلَا ) إمَامَةُ ( نَجِسٍ يَعْلَمُ ذَلِكَ ) ؛ لِأَنَّهُ أَخَلَّ بِشَرْطِ الصَّلَاةِ مَعَ الْقُدْرَةِ أَشْبَهَ الْمُتَلَاعِبَ لِكَوْنِهِ لَا صَلَاةَ لَهُ فِي نَفْسِهِ فَيُعِيدُ مَنْ صَلَّى خَلْفَهُ ( وَلَوْ جَهِلَهُ ) أَيْ الْحَدَثَ أَوْ النَّجَسَ ( مَأْمُومٌ فَقَطْ ) أَيْ وَحْدَهُ وَعَلِمَهُ الْإِمَامُ ، فَيَكُونُ كُلُّهُمْ وَلَا فَرْقَ بَيْنَ الْحَدَثِ الْأَكْبَرِ وَالْأَصْغَرِ وَلَا بَيْنَ نَجَاسَةِ الثَّوْبِ وَالْبَدَنِ وَالْبُقْعَةِ فَإِنْ جَهِلَهُ أَيْ الْحَدَثَ أَوْ النَّجَسَ ( هُوَ ) أَيْ الْإِمَامُ .
