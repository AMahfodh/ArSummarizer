وَلَا يَكْفِي فِيهِ ادَّعَى أَنَّ لِي عَلَى فُلَانِ بْنِ فُلَانٍ كَذَا مِنْ غَيْرِ رَبْطٍ بِالْحَاضِرِ ( فَإِنْ أَحْضَرَ رَجُلًا ) عِنْدَ الْقَاضِي ( وَقَالَ هَذَا أَقَرَّ لِفُلَانِ بْنِ فُلَانٍ بِكَذَا وَأَنَا هُوَ وَقَالَ الْخَصْمُ ) نَعَمْ ( أَقْرَرْت وَلَكِنْ لِرَجُلٍ آخَرَ شَارَكَك فِي الِاسْمِ وَالنَّسَبِ أَثْبَتَ ) الْمُقِرُّ أَيْ أَقَامَ بَيِّنَةً ( بِالْآخَرِ ) أَيْ بِوُجُودِ الْآخَرِ الْمُشَارِكِ لِلْمُدَّعِي فِي الِاسْمِ وَالنَّسَبِ ( ثُمَّ يَسْأَلُ ) الْآخَرَ ( فَإِنْ صَدَّقَهُ سَلَّمَ إلَيْهِ ) مَا أَقَرَّ لَهُ بِهِ ( وَحَلَفَ لِلْأَوَّلِ ) أَنَّهُ لَا شَيْءَ لَهُ عَلَيْهِ وَإِنْ كَذَّبَهُ سَلَّمَ ذَلِكَ لِلْمُدَّعِي ( وَإِنْ قَالَ ) أَقْرَرْت لِأَحَدِهِمَا وَ ( لَا أَعْرِفُهُ مِنْهُمَا سَأَلَ الْآخَرَ فَإِنْ قَالَ لَا شَيْءَ لِي عِنْدَهُ أَعْطَى ) ذَلِكَ ( الْأَوَّلَ ) كَمَا لَوْ كَانَتْ عِنْدَهُ وَدِيعَةٌ فَقَالَ هِيَ لِأَحَدِكُمَا وَلَا أَدْرِي أَنَّهَا لِأَيِّكُمَا فَقَالَ أَحَدُهُمَا لَيْسَتْ لِي فَإِنَّهَا تَكُونُ لِلْآخَرِ ( وَإِنْ ادَّعَاهُ كُلٌّ ) مِنْهُمَا ( فَكَمَا فِي الْوَدِيعَةِ إذَا قَالَ كُلٌّ ) مِنْ اثْنَيْنِ ( هِيَ لِي )
