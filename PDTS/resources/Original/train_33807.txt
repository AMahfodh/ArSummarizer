( وَ ) يُسَنُّ ( أَنْ يَجْلِسَ بَيْنَ الْخُطْبَتَيْنِ جَلْسَةً خَفِيفَةً جِدًّا ) لِمَا رَوَى ابْنُ عُمَرَ قَالَ { كَانَ النَّبِيُّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يَخْطُبُ خُطْبَتَيْنِ وَهُوَ قَائِمٌ يَفْصِلُ بَيْنَهُمَا بِجُلُوسٍ } مُتَّفَقٌ عَلَيْهِ قَالَ جَمَاعَةٌ مِنْهُمْ صَاحِبُ التَّلْخِيصِ ( بِقَدْرِ سُورَةِ الْإِخْلَاصِ فَإِنْ أَبَى ) أَنْ يَجْلِسَ بَيْنَهُمَا ( أَوْ خَطَبَ جَالِسًا ) لِعُذْرٍ أَوْ غَيْرِهِ ( فَصَلَ بِسَكْتَةٍ ) وَلَا يَجِبُ الْجُلُوسُ لِأَنَّ جَمَاعَةً مِنْ الصَّحَابَةِ مِنْهُمْ عَلِيٌّ ، سَرَدُوا الْخُطْبَتَيْنِ مِنْ غَيْرِ جُلُوسٍ وَلِأَنَّهُ لَيْسَ ، فِي الْجِلْسَةِ ذِكْرٌ مَشْرُوعٌ .( 4 / 151 )
