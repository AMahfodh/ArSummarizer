( الثَّانِيَةُ وَالثَّلَاثُونَ ) : قَوْلُهَا { قَالَتْ أَيْ هَنْتَاه : } أَمَّا { أَيْ : } بِفَتْحِ الْهَمْزَةِ وَإِسْكَانِ الْيَاءِ فَحَرْفُ نِدَاءٍ لِلْبَعِيدِ أَوْ لَمُنَزَّلٍ مَنْزِلَتَهُ وَهِيَ هُنَا لِلْمُنَزَّلِ مَنْزِلَتَهُ وَكَأَنَّهَا عَدَّتْ أُمَّ الْمُؤْمِنِينَ بَعِيدَةً عَنْهَا لِغَفْلَتِهَا عَنْ هَذَا الْأَمْرِ وَأَمَّا { هَنْتَاه : } فَهُوَ بِفَتْحِ الْهَاءِ وَإِسْكَانِ النُّونِ وَفَتْحِهَا ؛ وَالْإِسْكَانُ أَشْهَرُ .
