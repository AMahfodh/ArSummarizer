( وَإِنْ قَتَلَهُ ) أَيْ قَتَلَ الْمُسْلِمُ الْكَافِرَ ( أَوْ أَثْخَنَهُ ) بِالْجِرَاحِ ( فَلَهُ ) أَيْ الْمُسْلِمِ ( سَلَبُهُ ) بِفَتْحِ السِّينِ وَاللَّامِ وَيَأْتِي ( وَكَذَا مَنْ غَرَّرَ بِنَفْسِهِ ) فَقَتَلَ كَافِرًا ( وَلَوْ ) كَانَ الْمُسْلِمُ الْقَاتِلُ ( عَبْدًا بِإِذْنِ سَيِّدِهِ أَوْ امْرَأَةً أَوْ كَافِرًا أَوْ صَبِيًّا بِإِذْنِ ) إمَامٍ أَوْ نَائِبِهِ .
