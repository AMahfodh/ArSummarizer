وَالْعُرُوقِ لَيْسَ بِنَجَسٍ ( وَالْبَوْلِ وَلَوْ مِنْ صَغِيرِ لَمْ يَأْكُلْ ) لِإِطْلَاقِ قَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { اسْتَنْزِهُوا عَنْ الْبَوْلِ } الْحَدِيثَ ( وَكُلِّ مَا يَخْرُجُ مِنْ بَدَنِ الْآدَمِيِّ ) مَعْطُوفٌ عَلَى قَوْلِهِ كَالدَّمِ ( مُوجِبًا لِلتَّطْهِيرِ ) احْتَرَزَ بِهِ عَنْ الْعِرْقِ وَالْبُزَاقِ وَنَحْوِهِمَا .( 1 / 148 )
