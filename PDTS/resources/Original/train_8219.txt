( وَجَازَ الرَّمْيُ بِكُلِّ مَا كَانَ مِنْ جِنْسِ الْأَرْضِ كَالْحَجَرِ وَالْمَدَرِ ) وَالطِّينِ وَالْمَغْرَةِ ( وَ ) كُلِّ مَا ( يَجُوزُ التَّيَمُّمُ بِهِ وَلَوْ كَفًّا مِنْ تُرَابٍ ) فَيَقُومُ مَقَامَ حَصَاةٍ وَاحِدَةٍ ( لَا ) يَجُوزُ ( بِخَشَبٍ وَعَنْبَرٍ وَلُؤْلُؤٍ ) كِبَارٍ ( وَجَوَاهِرَ ) لِأَنَّهُ إعْزَازٌ لَا إهَانَةٌ وَقِيلَ يَجُوزُ ( وَذَهَبٍ وَفِضَّةٍ ) لِأَنَّهُ يُسَمَّى نِثَارًا لَا رَمْيًا ( وَبَعْرٍ ) لِأَنَّهُ لَيْسَ مِنْ جِنْسِ الْأَرْضِ وَمَا فِي فُرُوقِ الْأَشْبَاهِ مِنْ جَوَازِهِ بِالْبَعْرِ خِلَافُ الْمَذْهَبِ -
