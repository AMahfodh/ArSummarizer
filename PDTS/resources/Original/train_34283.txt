وَجَازَ لَهُ أَنْ يَتَزَوَّجَ الْأَمَةَ حَيْثُ خَافَ الْعَنَتَ لِأَنَّهُ لَمْ يَسْتَطِعْ طَوْلًا لِنِكَاحِ حُرَّةٍ بِلَا ضَرَرٍ عَلَيْهِ ( وَالْقَوْلُ قَوْلُهُ فِي خَشْيَةِ الْعَنَتِ وَ ) فِي ( عَدَمِ الطَّوْلِ ) لِأَنَّهُ أَدْرَى بِحَالِ نَفْسِهِ ( حَتَّى لَوْ كَانَ فِي يَدِهِ مَالٌ فَادَّعَى أَنَّهُ وَدِيعَةٌ أَوْ ) أَنَّهُ ( مُضَارَبَةٌ قُبِلَ قَوْلُهُ ) لِأَنَّهُ مُمْكِنٌ قُلْت بِلَا يَمِينٍ لِعَدَمِ الْخَصْمِ ( وَنِكَاحُ مَنْ بَعْضُهَا حُرٌّ ) مَعَ وُجُودِ الشَّرْطَيْنِ ( أَوْلَى مِنْ ) نِكَاحِ ( أَمَةٍ ) لِأَنَّ اسْتِرْقَاقَ بَعْضِ الْوَلَدِ أَخَفُّ مِنْ اسْتِرْقَاقِ كُلِّهِ .
