( وَإِنْ أَعْتَقَ ) بَعْضُ الْغَانِمِينَ ( قِنًّا ) مِنْ الْغَنِيمَةِ ( أَوْ كَانَ ) فِي الْغَنِيمَةِ قِنٌّ ( يُعْتَقُ عَلَيْهِ ) كَأَبِيهِ وَعَمِّهِ وَخَالِهِ ( عَتَقَ قَدْرُ حَقِّهِ ) لِمُصَادَفَتِهِ مِلْكَهُ ( وَالْبَاقِي ) مِنْهُ ( كَعِتْقِهِ شِقْصًا ) مِنْ مُشْتَرَكٍ عَلَى مَا يَأْتِي تَفْصِيلُهُ .
