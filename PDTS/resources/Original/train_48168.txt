( اشْتَرَى عَبْدًا فَقَالَ لِمَنْ سَاوَمَهُ إيَّاهُ : اشْتَرِهِ فَلَا عَيْبَ بِهِ فَلَمْ يَتَّفِقْ بَيْنَهُمَا الْبَيْعُ فَوَجَدَ ) مُشْتَرِيهِ ( بِهِ عَيْبًا ) فَلَهُ ( رَدُّهُ عَلَى بَائِعِهِ ) بِشَرْطِهِ ( وَلَأَنْ يَمْنَعَهُ ) مِنْ الرَّدِّ عَلَيْهِ ( إقْرَارُهُ السَّابِقُ ) بِعَدَمِ الْعَيْبِ ، ؛ لِأَنَّهُ مَجَازٌ عَنْ التَّرْوِيجِ ( وَلَوْ عَيَّنَهُ ) أَيْ الْعَيْبَ فَقَالَ لَا عَوَرَ بِهِ أَوْ لَا شَلَلَ ( لَا ) يَرُدُّهُ لِإِحَاطَةِ الْعِلْمِ بِهِ إلَّا أَنْ لَا يَحْدُثَ مِثْلُهُ كَلَا أُصْبُعَ بِهِ زَائِدَةٌ ثُمَّ وَجَدَهَا فَلَهُ رَدُّهُ لِلتَّيَقُّنِ بِكَذِبِهِ .
