فَصَلِّ ( إذَا بَنَى مَسْجِدًا لَا يَزُولُ مِلْكُهُ ) أَيْ مِلْكُ الْمَالِكِ الْمَجَازِيِّ ( عَنْهُ ) أَيْ عَنْ الْمَسْجِدِ وَإِنَّمَا قَالَ بَنَى لِأَنَّهُ لَوْ كَانَ سَاحَةً زَالَ مِلْكُهُ بِمُجَرَّدِ الْأَمْرِ بِالصَّلَاةِ فِيهَا ذَكَرَ الْأَبَدَ أَوْ لَا كَمَا فِي الْمُحِيطِ ( حَتَّى يَفْرِزَهُ ) أَيْ يُمَيِّزَهُ ( عَنْ مِلْكِهِ ) مِنْ كُلِّ الْوُجُوهِ ( بِطَرِيقِهِ ) أَيْ مَعَ طَرِيقِ الْمَسْجِدِ بِأَنْ يَجْعَلَ لَهُ سَبِيلًا عَامًّا يَدْخُلُ فِيهِ الْمُسْلِمُونَ مِنْهُ لِأَنَّهُ لَا يَخْلُصُ لِلَّهِ تَعَالَى إلَّا بِهِ ( وَيَأْذَنَ ) أَيْ لِكُلِّ النَّاسِ ( بِالصَّلَاةِ ) أَيْ بِكُلِّ الصَّلَاةِ ( فِيهِ ) أَيْ فِي الْمَسْجِدِ عِنْدَ الطَّرَفَيْنِ لِأَنَّهُ تَسْلِيمٌ وَهُوَ شَرْطٌ عِنْدَهُمَا فَلَوْ أَذِنَ لِقَوْمٍ أَوْ لِلنَّاسِ شَهْرًا أَوْ سَنَةً مَثَلًا لَا يَزُولُ مِلْكُهُ كَمَا فِي الْقُهُسْتَانِيِّ ( وَيُصَلِّي فِيهِ ) وَلَوْ بِلَا أَذَانٍ وَإِقَامَةٍ ( وَاحِدٌ ) .
