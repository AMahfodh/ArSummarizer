( وَ ) عَلَى هَذَا ( يَلْزَمُهُمْ إذَنْ تَسْوِيَةُ الْأَرْضِ وَ ) يَلْزَمُهُمْ ( أَرْشٌ بِنَقْصِهَا الْحَاصِلِ بِهِ ) لِأَنَّ ذَلِكَ نَقْصٌ حَصَلَ لِتَخْلِيصِ مِلْكِ الْمُفْلِسِ فَكَانَ عَلَيْهِ ( وَيَضْرِبُ بِهِ ) أَيْ بِأَرْشِ نَقْصِ الْأَرْضِ ( الْبَائِعُ مَعَ الْغُرَمَاءِ ) كَسَائِرِ دُيُونِ الْمُفْلِسِ ( وَلَهُ ) أَيْ لِبَائِعِ الْأَرْضِ ( الرُّجُوعُ فِيهَا ) أَيْ أَرْضِهِ ( وَلَوْ قَبْلَ الْقَلْعِ ) أَيْ قَلْعِ الْغِرَاسِ وَالْبِنَاءِ ( وَدَفْعُ قِيمَةِ الْغِرَاسِ وَالْبِنَاءِ أَوْ قَلْعُهُ ) وَضَمَانُ نَقْصِهِ وَتَقَدَّمَ قَرِيبًا .
