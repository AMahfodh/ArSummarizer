طَارِقَ فِي ذَلِكَ الْوَقْتِ فَالْمُتَّجَهُ وُجُوبُ الْقَوَدِ كَالْمَحْبُوسِ مَرْدُودٌ مُخَالِفٌ لِكَلَامِهِمْ ( وَإِلَّا ) بِأَنْ لَمْ تَمْضِ تِلْكَ الْمُدَّةُ وَمَاتَ وَهُوَ بِالْجُوعِ مَثَلًا لَا بِنَحْوِ هَدْمٍ ( فَإِنْ لَمْ يَكُنْ بِهِ جُوعٌ وَعَطَشٌ ) أَيْ أَوْ عَطَشٌ ( سَابِقٌ ) عَلَى حَبْسِهِ ( فَشِبْهُ عَمْدٍ ) وَعُلِمَ مِنْ كَلَامِهِ السَّابِقِ أَنَّهُ لَا بُدَّ مِنْ مُضِيِّ مُدَّةٍ يُمْكِنُ عَادَةً إحَالَةُ الْهَلَاكِ عَلَيْهَا فَإِيهَامُ عُمُومٍ ، وَإِلَّا هُنَا غَيْرُ مُرَادٍ ( وَإِنْ كَانَ ) بِهِ ( بَعْضُ جُوعٍ وَعَطَشٍ ) الْوَاوُ بِمَعْنَى أَوْ كَمَا مَرَّ سَابِقًا ( وَعَلِمَ الْحَابِسُ الْحَالَ فَعَمْدٌ ) لِشُمُولِ حَدِّهِ السَّابِقِ لَهُ ؛ إذْ الْفَرْضُ أَنَّ مَجْمُوعَ الْمُدَّتَيْنِ بَلَغَ الْمُدَّةَ الْقَاتِلَةَ وَأَنَّهُ مَاتَ بِذَلِكَ كَمَا عُلِمَ مِنْ كَلَامِهِ ( وَإِلَّا ) بِأَنْ لَمْ يَعْلَمْ الْحَالَ ( فَلَا ) يَكُونُ عَمْدًا ( فِي الْأَظْهَرِ ) لِانْتِفَاءِ قَصْدِ إهْلَاكِهِ وَلَمْ يَأْتِ بِمُهْلِكٍ بَلْ شَبَهِهِ فَيَجِبُ نِصْفُ دِيَتِهِ لِحُصُولِ الْهَلَاكِ بِالْأَمْرَيْنِ .
