( وَ ) يَجُوزُ أَنْ ( يَقْصُرَ مَنْ ) نَوَى بَلَدًا بِعَيْنِهِ يَبْلُغُ الْمَسَافَةَ ، وَ ( عَلِمَهَا ) ابْتِدَاءً ، ( ثُمَّ نَوَى ) فِي سَفَرِهِ ( إنْ وَجَدَ عَزِيمَةً ) فِي طَرِيقِهِ ( رَجَعَ ) ؛ لِأَنَّ سَبَبَ الرُّخْصَةِ انْعَقَدَ ، فَلَا يَتَغَيَّرُ بِالنِّيَّةِ الْمُتَعَلِّقَةِ قَبْلَ وُجُودِ الشَّرْطِ .( 4 / 54 )
