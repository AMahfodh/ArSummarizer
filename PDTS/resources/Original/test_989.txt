( الْعَبْدُ الْمُفْسِدُ لِلْحَجِّ يَلْزَمُهُ الْقَضَاءُ ) ؛ لِأَنَّهُ مُكَلَّفٌ ( فَإِنْ أُعْتِقَ ) بَعْدَ الْإِفْسَادِ ( ثُمَّ نَذَرَ حَجًّا قَدَّمَ حِجَّةَ الْإِسْلَامِ ) لِأَصَالَتِهَا وَلِأَهَمِّيَّتِهَا الْمَفْهُومَةِ مِنْ خَبَرِ أَبِي دَاوُد بِإِسْنَادٍ صَحِيحٍ { أَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ لِرَجُلٍ لَبَّى عَنْ شُبْرُمَةَ أَخٍ أَوْ قَرِيبٍ لَهُ : حُجَّ عَنْ نَفْسِك ، ثُمَّ عَنْ شُبْرُمَةَ } ( ثُمَّ الْقَضَاءُ ) لِوُجُوبِهِ بِأَصْلِ الشَّرْعِ ، وَلَا يُجْزِئُ الْقَضَاءُ عَنْ حِجَّةِ الْإِسْلَامِ لِكَوْنِهِ تَدَارُكًا لِغَيْرِهَا ( ثُمَّ النَّذْرُ ) ؛ لِأَنَّهُ أَهَمُّ مِنْ النَّفْلِ ( فَإِنْ أَحْرَمَ بِغَيْرِهَا ) أَيْ بِغَيْرِ حِجَّةِ الْإِسْلَامِ ( مَنْ هِيَ عَلَيْهِ أَوْ عَلَى مَنْ اسْتَنَابَهُ فِيهَا انْصَرَفَ إلَيْهَا ) ؛ لِأَنَّ غَيْرَهَا لَا يَتَقَدَّمُ عَلَيْهَا ، وَهَذَا يُغْنِي عَنْهُ قَوْلُهُ ( وَإِنْ قَدَّمَ مُؤَخَّرًا لَغَا وَوَقَعَ الْمُقَدَّمُ ) وَلِمَنْ حَجَّ حِجَّةَ الْإِسْلَامِ ، وَلَمْ يَعْتَمِرْ أَنْ يُقَدِّمَ حِجَّةَ التَّطَوُّعِ عَلَى الْعُمْرَةِ ، وَلِمَنْ اعْتَمَرَ عُمْرَةَ الْإِسْلَامِ ، وَلَمْ يَحُجَّ أَنْ يُقَدِّمَ عُمْرَةَ التَّطَوُّعِ عَلَى الْحَجِّ ( وَإِنْ نَذَرَ مَنْ لَمْ يَحُجَّ أَنْ يَحُجَّ هَذِهِ السَّنَةِ فَحَجَّ خَرَجَ عَنْ فَرْضِهِ وَنَذْرِهِ ) إذْ لَيْسَ فِيهِ إلَّا تَعْجِيلُ مَا كَانَ لَهُ تَأْخِيرُهُ فَيَقَعُ أَصْلُ الْفِعْلِ عَنْ فَرْضِهِ وَتَعْجِيلُهُ عَنْ نَذْرِهِ .
