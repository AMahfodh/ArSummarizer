( فَصْلٌ ) ( وَسُنَّ ) لِمُتَخَلٍّ ( إذَا فَرَغَ ) مِنْ حَاجَتِهِ ( مَسْحُ ذَكَرٍ ) بِيَدٍ يُسْرَى ( مِنْ حَلْقَةِ دُبُرٍ ) بِسُكُونِ اللَّامِ - فَيَضَعُ أُصْبُعَ يَدِهِ الْيُسْرَى الْوُسْطَى تَحْتَ الذَّكَرِ ، وَالْإِبْهَامَ فَوْقَهُ ، وَيَمُرُّ بِهِمَا ( إلَى رَأْسِهِ ) - أَيْ : الذَّكَرِ - ( ثَلَاثًا لِيَجْذِبَ بَقَايَا ) الْبَلَلِ .( 1 / 151 )
