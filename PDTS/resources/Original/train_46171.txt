( فِي قَتْلِهِ ) أَيْ الصَّيْدِ ( بِهِ ) أَيْ بِسَهْمِهِ لَمْ يَحِلَّ ( أَوْ أَكَلَ مِنْهُ سَبُعٌ يَصْلُحْ أَنْ يَكُونَ ) أَكْلُهُ مِنْهُ ( قَتَلَهُ لَمْ يَحِلَّ ) لِلْخَبَرِ السَّابِقِ وَكَمَا لَوْ وَجَدَ مَعَ كَلْبِهِ كَلْبًا سِوَاهُ ( وَإِنْ كَانَ الْأَثَرُ مِمَّا لَا يَقْتُلُ مِثْلُهُ ) أَيْ مِثْلُ ذَلِكَ الصَّيْدِ ( مِثْلُ أَكْلِ حَيَوَانٍ ضَعِيفٍ كَسِنَّوْرٍ وَثَعْلَبٍ مِنْ حَيَوَانٍ قَوِيٍّ أَوْ تَهَشَّمَ مِنْ وَقْعَتِهِ فَمُبَاحٌ ) لِأَنَّهُ مَعْلُومٌ أَنَّ هَذَا لَمْ يَقْتُلْهُ ( وَلَوْ أَرْسَلَ عَلَيْهِ ) أَيْ الصَّيْدِ ( كَلْبَهُ فَعَقَرَهُ فَغَابَ ) ثُمَّ وَجَدَهُ مَيِّتًا ( أَوْ غَابَ ) الصَّيْدُ ( قَبْلَ عَقْرِهِ ثُمَّ وُجِدَ مَيِّتًا وَالْكَلْبُ وَحْده أَوْ ) وَجَدَ ( الصَّيْدَ بِفَمِهِ أَوْ ) وَهُوَ ( يَعْبَثُ بِهِ أَوْ عَلَيْهِ حَلَّ ) الصَّيْدُ لِأَنَّ وُجُودَهُ بِهَذِهِ الْحَالَةِ وَعَدَمَ أَثَرِ ذَلِكَ فِيهِ يَغْلِبُ عَلَى الظَّنِّ أَنَّ الْمَوْتَ حَصَلَ بِجَارِحِهِ فَحَلَّ كَمَا لَوْ لَمْ يَغِبْ عَنْهُ قَالَ فِي( 21 / 298 )
