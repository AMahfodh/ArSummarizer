( وَيُقَدَّرُ ) السَّقْيُ ( بِالزَّمَانِ ) كَيَوْمٍ ، وَأُسْبُوعٍ ( أَوْ بِعَدَدِ الْغُرُوبِ أَوْ بِمِلْءِ بِرْكَةٍ ) وَ ( لَا ) يَصِحُّ تَقْدِيرُهُ ( بِسَقْيِ أَرْضٍ ) لِأَنَّهُ لَا يَنْضَبِطُ ( وَإِنْ قَدَّرَهُ ) أَيْ السَّقْيَ ( بِشِرْبِ مَاشِيَةٍ جَازَ ، لِأَنَّ شِرْبَهَا يَتَقَارَبُ فِي الْغَالِبِ كَ ) مَا يَجُوزُ تَقْدِيرُهُ ( بِبَلِّ تُرَابٍ مَعْرُوفٍ ) لَهُمَا لِأَنَّهُ مَعْلُومٌ بِالْعُرْفِ .( 12 / 94 )
