( وَالنِّصْفُ ) أَيْ : وَنِصْفُ شَاةٍ ( فِيمَا ) تَمَّ ( بَعْدَهُ ) مِنْ الْأَحْوَالِ لِحُصُولِ الْخُلْطَةِ بَعْدَ ذَلِكَ كُلَّ الْحَوْلِ ( وَعَمْرُو عَلَيْهِ نِصْفُ الشَّاةِ يَسْتَمِرُّ عِنْدَ تَمَامِ كُلِّ حَوْلٍ هُوَ لَهْ وَذَاكَ كُلَّ ) أَيْ : فِي كُلِّ ( صَفَرٍ أَيْ أَوَّلَهُ ) لِعَدَمِ انْفِرَادِهِ أَصْلًا
