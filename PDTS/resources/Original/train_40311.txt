( فَإِنْ كَانَ ) الرَّهْنُ ( جُعِلَ لِلْمُرْتَهِنِ بَيْعُهُ ) فِي الْعَقْدِ ( أَوْ أُذِنَ لَهُ فِيهِ بَعْدَ الْعَقْدِ ) بَاعَهُ الْمُرْتَهِنُ ؛ لِأَنَّهُ وَكِيلُ رَبِّهِ ( أَوْ اتَّفَقَا ) أَيْ : الرَّاهِنُ وَالْمُرْتَهِنُ ( عَلَى أَنَّ الرَّاهِنَ ) يَبِيعُهُ بَاعَهُ ( أَوْ ) اتَّفَقَا عَلَى أَنَّ ( غَيْرَهُ يَبِيعُهُ بَاعَهُ ) ؛ لِأَنَّهُ وَكِيلُ مَالِكِهِ وَمَأْذُونٌ لَهُ مِنْ قِبَلِ الْمُرْتَهِنِ .
