وَعَطَفَ عَلَى قَوْلِهِ حَضَرَ فَقَالَ ( أَوْ غَابَ ) الْمُقَرُّ لَهُ بِمَا ادَّعَاهُ الْمُدَّعِي غَيْبَةً بَعِيدَةً لَا يُعْذَرُ لَهُ فِيهَا ( لَزِمَهُ ) أَيْ الْمُقِرَّ ( يَمِينٌ ) أَنَّهُ مَا أَقَرَّ إلَّا بِحَقٍّ ( أَوْ بَيِّنَةٌ ) عَلَى أَنَّ الْمُقَرَّ بِهِ لِفُلَانٍ الْغَائِبِ أَوْدَعَهُ أَوْ رَهَنَهُ عِنْدَهُ ( وَ ) إنْ حَلَفَ أَوْ أَقَامَ بَيِّنَةً عَلَى ذَلِكَ ( انْتَقَلَتْ الْحُكُومَةُ لَهُ ) أَيْ الْغَائِبِ فَيُنْتَظَرُ قُدُومُهُ ( وَإِنْ نَكَلَ ) الْمُقِرُّ وَلَمْ يَأْتِ بِبَيِّنَةٍ عَلَى ذَلِكَ ( أَخَذَهُ ) أَيْ الْمُدَّعِي الْمُدَّعَى بِهِ ( بِلَا يَمِينٍ وَإِنْ جَاءَ الْمُقَرُّ لَهُ فَصَدَّقَ الْمُقِرَّ ) فِي إقْرَارِهِ أَنَّ الْمُدَّعَى بِهِ لَهُ ( أَخَذَهُ ) أَيْ الْمُقَرُّ لَهُ الْمُدَّعَى بِهِ مِنْ الْمُدَّعِي بِيَمِينٍ لِقَوْلِهِ وَانْتَقَلَتْ( 18 / 411 )
