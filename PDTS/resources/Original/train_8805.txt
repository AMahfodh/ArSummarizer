( فَرْعٌ لِلْعَبْدِ تَأْجِيرُ ) الْمَعْرُوفِ إجَارَةً أَوْ إيجَارَ ( نَفْسِهِ بِإِذْنِ السَّيِّدِ وَكَذَا بَيْعُهَا وَرَهْنُهَا ) إذْ لَا مَانِعَ ( وَلَوْ تَوَكَّلَ لِغَيْرِهِ فِيمَا لَزِمَ ذِمَّتَهُ عُهْدَةٌ ) كَبَيْعٍ وَشِرَاءٍ ( بِلَا إذْنٍ ) مِنْ سَيِّدِهِ ( لَمْ يَصِحَّ ) لِتَعَلُّقِ الْعُهْدَةِ بِالْوَكِيلِ بِخِلَافِهِ فِيمَا يَلْزَمُ ذَلِكَ كَطَلَاقٍ وَقَبُولِ نِكَاحٍ فَإِنَّهُ يَصِحُّ وَالتَّصْرِيحُ بِقَوْلِهِ فِيمَا يَلْزَمُ ذِمَّتَهُ عُهْدَةٌ مِنْ زِيَادَتِهِ وَهُوَ سَاقِطٌ مِنْ بَعْضِ النُّسَخِ( 8 / 459 )
