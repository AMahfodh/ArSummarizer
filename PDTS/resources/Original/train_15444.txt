( وَ ) كُرِهَ ( نِدَاءٌ بِهِ ) أَيْ الْمَيِّتِ بِأَنْ يُقَالَ بِصَوْتٍ مُرْتَفِعٍ فُلَانٌ مَاتَ فَاسْعَوْا لِجِنَازَتِهِ ( بِمَسْجِدٍ ) لِكَرَاهَةِ رَفْعَ الصَّوْتِ فِيهِ وَلَوْ بِالْعِلْمِ زِيَادَةً عَنْ الْحَاجَةِ ( أَوْ بَابِهِ ) أَيْ الْمَسْجِدِ ؛ لِأَنَّهُ ذَرِيعَةٌ لِرَفْعِهِ فِيهِ وَلِأَنَّهُ مِنْ فِعْلِ الْجَاهِلِيَّةِ ( لَا ) يُكْرَهُ الْإِعْلَامُ ( بِحِلَقٍ ) بِكَسْرِ الْحَاءِ وَفَتْحِ اللَّامِ جَمْعُ حَلْقَةٍ بِفَتْحٍ فَسُكُونٍ ( بِصَوْتٍ خَفِيٍّ ) بَلْ يُنْدَبُ لِأَنَّهُ وَسِيلَةٌ لِتَشْيِيعِهِ .( 3 / 203 )
