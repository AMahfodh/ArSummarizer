وَهُوَ ثَمَانِيَةٌ وَثَلَاثَةٌ أَوْ سِتَّةٌ وَأَرْبَعَةٌ وَ ( اقْسِمْهَا ) أَيْ السِّتَّمِائَةِ ( عَلَى سِتَّةٍ ؛ لِأَنَّهَا أَحَدُ ضِلْعَيْ الْقِيرَاطِ يَخْرُجْ ) بِالْقِسْمَةِ ( مِائَةٌ اقْسِمْهَا عَلَى الضِّلْعِ الْآخَرِ وَهُوَ أَرْبَعَةٌ يَخْرُجْ خَمْسَةٌ وَعِشْرُونَ وَهِيَ سَهْمُ الْقِيرَاطِ ، وَإِنْ شِئْتَ قَسَمْتَ وَفْقَ السِّهَامِ ) أَيْ سِهَامِ الْمَسْأَلَةِ يَعْنِي نَفْسَ الْمَسْأَلَةِ ( عَلَى وَفْقِ الْقِيرَاطِ ) يَحْصُلُ الْمَطْلُوبُ ( فَتَأْخُذُ سُدُسَ السِّتِّمِائَةِ ، وَهُوَ مِائَةٌ فَقَسِّمْهُ عَلَى سُدُسِ الْأَرْبَعَةِ وَعِشْرِينَ وَهُوَ أَرْبَعَةٌ فَيَخْرُجْ خَمْسَةٌ وَعِشْرُونَ ) وَهُوَ الْمَطْلُوبُ ( وَإِنْ شِئْتَ أَخَذْتَ ثُمُنَ السِّتِّمِائَةِ وَخُمُسَهُ وَسَبْعِينَ( 15 / 447 )
