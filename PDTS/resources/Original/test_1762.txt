( وَلَوْ صَامَتْ لِكَفَّارَةٍ ) بِلَا إذْنِهِ فَلَا نَفَقَةَ لَهَا ( أَوْ ) صَامَتْ لِنَذْرٍ ( أَوْ قَضَاءِ رَمَضَانَ وَوَقْتُهُ مُتَّسِعٌ فِيهِمَا ) أَيْ فِي النَّذْرِ وَقَضَاءِ رَمَضَانَ ( بِلَا إذْنِهِ ) فَلَا نَفَقَةَ لَهَا ، لِأَنَّهَا مَنَعَتْ نَفْسَهَا عَنْهُ بِسَبَبٍ لَا مِنْ جِهَتِهِ ( أَوْ سَافَرَتْ لِتَغْرِيبٍ ) بِأَنْ زَنَتْ فَغُرِّبَتْ ( أَوْ حُبِسَتْ وَلَوْ ظُلْمًا فَلَا نَفَقَةَ لَهَا ) زَمَنَ تَغْرِيبِهَا ، أَوْ حَبْسِهَا لِفَوَاتِ التَّمْكِينِ الْمُقَابِلِ لِلنَّفَقَةِ .( 19 / 373 )
