( وَإِنْ أَمَرَهُ ) رَبُّ الْوَدِيعَةِ ( أَنْ يَجْعَلَهَا فِي صُنْدُوقٍ وَقَالَ ) رَبُّ الْوَدِيعَةِ لِلْمُسْتَوْدَعِ ( لَا تَقْفِلْ عَلَيْهَا ) الصُّنْدُوقَ ( وَلَا تَنَمْ فَوْقَهَا فَخَالَفَهُ ) وَقَفَلَ عَلَيْهَا أَوْ نَامَ عَلَيْهَا ، فَلَا ضَمَانَ عَلَيْهِ ؛ لِأَنَّهُ مُحْسِنٌ ( أَوْ قَالَ ) اجْعَلْهَا فِي صُنْدُوقٍ وَ ( لَا تَقْفِلْ عَلَيْهَا إلَّا قُفْلًا وَاحِدًا فَجَعَلَ عَلَيْهَا قُفْلَيْنِ فَلَا ضَمَانَ عَلَيْهِ ) لِمَا تَقَدَّمَ .( 13 / 394 )
