( أَوْ ) حَلَفَ ( لَا يَتَصَدَّقُ فَأَطْعَمَ عِيَالَهُ ) لِأَنَّهُ لَا يُسَمَّى صَدَقَةً عُرْفًا وَإِطْلَاقُ اسْمِ الصَّدَقَةِ عَلَيْهِ فِي الْخَبَرِ بِاعْتِبَارِ تَرَتُّبِ الثَّوَابِ عَلَيْهِ كَالصَّدَقَةِ ( وَإِنْ نَذَرَ أَنْ يَهَبُ لَهُ ) أَيْ : فُلَانٍ شَيْئًا ( بَرَّ بِالْإِيجَابِ ) لِلْهِبَةِ سَوَاءٌ قَبِلَ الْمَوْهُوبُ لَهُ أَوْ لَا ( كَيَمِينِهِ ) أَيْ : كَمَا لَوْ حَلَفَ لَيَهَبَنَّ لَهُ فَأَوْجَبَ لَهُ الْهِبَةَ فَإِنَّهُ يَبَرُّ مُطْلَقًا لِمَا تَقَدَّمَ .( 11 / 484 )
