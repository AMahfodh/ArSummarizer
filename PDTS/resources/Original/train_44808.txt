( الْمَانِعُ الرَّابِعُ : إبْهَامُ وَقْتِ الْمَوْتِ فَإِنْ مَاتَا ) أَيْ مُتَوَارِثَانِ ( بِغَرَقٍ أَوْ هَدْمٍ ) أَوْ نَحْوِهِ ( وَلَمْ يُعْلَمْ السَّابِقُ ) مِنْهُمَا مَوْتًا ( أَوْ عُلِمَ السَّبْقُ وَجُهِلَ ) السَّابِقُ مِنْهُمَا ( أَوْ مَاتَا مَعًا لَمْ يَتَوَارَثَا ) بَلْ مَالُ كُلٍّ مِنْهُمَا لِبَاقِي وَرَثَتِهِ لِمَا رَوَى الْحَاكِمُ بِسَنَدٍ صَحِيحٍ { أَنَّ أُمَّ كُلْثُومٍ بِنْتَ عَلِيٍّ رَضِيَ اللَّهُ عَنْهُمَا تُوُفِّيَتْ هِيَ وَابْنُهَا زَيْدُ بْنُ عُمَرَ بْنِ الْخَطَّابِ فِي يَوْمٍ فَلَمْ يُدْرَ أَيُّهُمَا مَاتَ قَبْلُ فَلَمْ تَرِثْهُ وَلَمْ يَرِثْهَا } وَلِأَنَّ مِنْ شُرُوطِ الْإِرْثِ تَحَقُّقَ حَيَاةِ الْوَارِثِ بَعْدَ مَوْتِ الْمُوَرِّثِ وَهُوَ هُنَا مُنْتَفٍ وَلِأَنَّا إنْ وَرَّثْنَا أَحَدَهُمَا فَقَطْ فَهُوَ تَحَكُّمٌ وَإِنْ وَرَّثْنَا كُلًّا مِنْ الْآخَرِ تَيَقَّنَّا الْخَطَأَ ( فَلَوْ عُلِمَ ) السَّابِقُ ( وَنَسِيَ وَقَفَ ) الْمِيرَاثُ ( إلَى الْبَيَانِ أَوْ الصُّلْحِ ) لِأَنَّ التَّذَكُّرَ غَيْرُ مَيْئُوسٍ مِنْهُ ، أَمَّا إذَا عُلِمَ السَّابِقُ وَلَمْ يُنْسَ فَحُكْمُهُ بَيِّنٌ .( 13 / 270 )
