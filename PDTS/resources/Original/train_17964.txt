( فَصْلٌ مَتَى كَانَ مُفْسِدُ النِّكَاحِ جَمَاعَةً وُزِّعَ الْمَهْرُ عَلَى مُرْضِعَاتِهِنَّ الْمُحَرَّمَةِ ) لِأَنَّهُ إتْلَافٌ اشْتَرَكْنَ فِيهِ فَكَانَ عَلَى كُلِّ وَاحِدَةٍ بِقَدَرِ مَا أَتْلَفَتْ وَ ( لَا ) يُوَزَّعُ عَلَى عَدَدِ ( رُءُوسِهِنَّ ) كَمَا لَوْ أَتْلَفْنَ مَالًا وَتَفَاوَتْنَ فِيهِ ( فَلَوْ سَقَى خَمْسٌ زَوْجَةً صَغِيرَةً مِنْ لَبَن أُمِّ الزَّوْجِ خَمْسَ مَرَّاتٍ انْفَسَخَ نِكَاحُهَا ) لِأَنَّهَا صَارَتْ أُخْتَهُ مِنْ الرَّضَاعِ ( وَلَزِمَهُنَّ نِصْفُ مَهْرِهَا بَيْنَهُنَّ ) بِحَسَبِ مَا سَقَيْنَهَا لِتَسَبُّبِهِنَّ فِي اسْتِقْرَارِهِ عَلَيْهِ ( فَإِنْ سَقَتْهَا وَاحِدَةٌ شَرْبَتَيْنِ وَ ) سَقَتْهَا ( أُخْرَى ثَلَاثًا فَعَلَى الْأُولَى خُمْسُ الْمَهْرِ وَعَلَى الثَّانِيَةِ خُمْسُ ) الْمَهْرِ ( وَعُشْرٌ ) ( وَإِنْ سَقَتْهَا وَاحِدَةٌ شَرْبَتَيْنِ وَسَقَاهَا ثَلَاثٌ ) أُخَرُ ( ثَلَاثَ شَرَبَاتٍ فَعَلَى الْأُولَى الْخُمْسُ وَعَلَى كُلِّ وَاحِدَةٍ مِنْ الثَّلَاثِ عُشْرٌ ) .( 19 / 283 )
