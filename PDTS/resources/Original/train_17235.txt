( وَلَا يَعُبُّ الْمَاءَ عَبًّا ) لِلْخَبَرِ ( وَأَنْ يَأْخُذَ إنَاءَ الْمَاءِ بِيَمِينِهِ ) مَعَ الْقُدْرَةِ ( وَيُسَمِّي ) وَتَقَدَّمَ ( وَيَنْظُر فِيهِ ) خَشْيَة أَنْ يَكُونَ فِيهِ مَا يَكْرَهُ أَوْ يُؤْذِيهِ ( ثُمَّ يَشْرَبُ مِنْهُ مَصًّا مُقَطِّعًا ثَلَاثًا ) لِقَوْلِهِ - صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ - { مُصُّوا الْمَاءَ مَصًّا فَإِنَّ الْكُبَادَ مِنْ الْعَبِّ } وَالْكُبَادُ بِضَمِّ الْكَافِ وَبِالْبَاءِ الْمُوَحَّدَةِ قِيلَ وَجَعُ الْكَبِدِ وَيُعَبُّ اللَّبَنُ لِأَنَّهُ طَعَامٌ ، ( وَيَتَنَفَّسُ ) كُلّ مَرَّةٍ ( خَارِجَ الْإِنَاءِ وَيُكْرَهُ أَنْ يَتَنَفَّسَ فِيهِ ) وَتَقَدَّمَ .
