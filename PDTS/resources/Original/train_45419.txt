( وَلَوْ ) حَلَفَ ( لَا يَدْخُلُ بَيْتًا فَدَخَلَ مَسْجِدًا أَوْ الْكَعْبَةَ أَوْ بَيْتَ رَحَا أَوْ ) دَخَلَ ( حَمَّامًا أَوْ بَيْتَ شَعْرٍ أَوْ ) بَيْتَ ( أُدْمٍ ) أَيْ جِلْدٍ ( أَوْ ) دَخَلَ ( خَيْمَةً حَنِثَ حَضَرِيًّا كَانَ الْحَالِفُ أَوْ بَدَوِيًّا ) لِأَنَّهَا بُيُوتٌ حَقِيقَةً لِقَوْلِهِ تَعَالَى : { فِي بُيُوتٍ أَذِنَ اللَّهُ أَنْ تُرْفَعَ } وَقَوْلُهُ : { إنَّ أَوَّلَ بَيْتٍ وُضِعَ لِلنَّاسِ } الْآيَة - وَقَوْلُهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ : { بِئْسَ الْبَيْتُ الْحَمَّامُ } رَوَاهُ أَبُو دَاوُد وَغَيْرُهُ وَفِيهِ ضَعْفٌ وَإِذَا كَانَ فِي الْحَقِيقَةِ بَيْتًا وَفِي عُرْفِ الشَّارِعِ حَنِثَ بِدُخُولِهِ وَأَمَّا بَيْتُ الشَّعْرِ وَالْأَدَمِ فَلِأَنَّ اسْمَ الْبَيْتِ يَقَعُ عَلَيْهِ لِقَوْلِهِ تَعَالَى : { وَاَللَّهُ جَعَلَ لَكُمْ مِنْ بُيُوتِكُمْ سَكَنًا } الْآيَة - وَالْخَيْمَةُ كَذَلِكَ وَ ( لَا ) يَحْنَثُ ( إنْ دَخَلَ دِهْلِيزَ الدَّارِ أَوْ صُفَّتَهَا الَّتِي تَكُونُ وَرَاءَ الْبَابِ ) لِأَنَّ ذَلِكَ لَا يُسَمَّى بَيْتًا .( 21 / 426 )
