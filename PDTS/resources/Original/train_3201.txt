كُلَّ أَحَدٍ لَا يُحْسِنُهُ فَيَلْحَقُ الْحَيَوَانَ بِهِ تَعْذِيبٌ ط وَأَشَارَ الْمُصَنِّفُ إلَى أَنَّ الْإِشْعَارَ خَاصٌّ بِالْإِبِلِ ( قَوْلُهُ بِوَضْعِ الْجَلِّ ) أَيْ عَلَى ظَهْرِهَا وَهُوَ بِالضَّمِّ وَالْفَتْحِ مَا تَلْبَسُهُ الْفَرَسُ لِتُصَانَ بِهِ قَامُوسٌ ( قَوْلُهُ لَا لِمُتْعَةٍ وَقِرَانٍ ) وَكَذَا لَوْ لَهُمَا قَبْلَ أَشْهُرِ الْحَجِّ رَحْمَتِيٌّ ( قَوْلُهُ كَمَا مَرَّ ) أَيْ لُحُوقًا كَاللُّحُوقِ الَّذِي مَرَّ وَهُوَ كَوْنُهُ قَبْلَ الْمِيقَاتِ وَهَذَا مُحْتَرَزُ قَوْلِهِ وَلَحِقَهَا ط ( قَوْلُهُ أَوْ قَلَّدَ شَاةً ) مُحْتَرَزُ قَوْلِهِ بَدَنَةً ط ( قَوْلُهُ لِعَدَمِ اخْتِصَاصِهِ بِالنُّسُكِ ) لِأَنَّ الْإِشْعَارَ قَدْ يَكُونُ لِلْمُدَاوَاةِ وَالْجَلَّ لِدَفْعِ الْحَرِّ وَالْبَرْدِ وَالْأَذَى وَلِأَنَّهُ إذَا لَمْ يَكُنْ بَيْنَ يَدَيْهِ هَدْيٌ يَسُوقُهُ عِنْدَ التَّوَجُّهِ لَمْ يُوجَدْ إلَّا مُجَرَّدُ النِّيَّةِ وَبِهِ لَا يَصِيرُ مُحْرِمًا وَتَقْلِيدُ الشَّاةِ لَيْسَ بِمُتَعَارَفٍ وَلَا سُنَّةٍ رَحْمَتِيٌّ( 8 / 209 )
