وَالْعِلْمُ بِهِ ؛ لِأَنَّهُ لَا بُدَّ مِنْ الرُّجُوعِ بِرَأْسِ الْمَالِ وَلَا يُمْكِنُ مَعَ جَهْلِهِ ( لِيَعْمَلَ ) مُتَعَلِّقٌ ب يُحْضِرَ ( فِيهِ ) أَيْ : الْمَالِ جَمِيعَهُ ( كُلٌّ ) مِمَّنْ لَهُ فِيهِ شَيْءٌ ( عَلَى أَنَّ لَهُ ) أَيْ : كُلِّ مَنْ لَهُ فِي الْمَالِ شَيْءٌ ( مِنْ الرِّبْحِ بِنِسْبَةِ مَالِهِ ) بِأَنْ شَرَطُوا لِرَبِّ النِّصْفِ نِصْفَ الرِّبْحِ وَلِرَبِّ الثُّلُثِ ثُلُثَ الرِّبْحِ وَلِرَبِّ السُّدُسِ سُدُسَ الرِّبْحِ مَثَلًا ( أَوْ ) عَلَى أَنَّ لِكُلٍّ مِنْهُمْ ( جُزْءًا مُشَاعًا مَعْلُومًا ) وَلَوْ أَكْثَرَ مِنْ نِسْبَةِ مَالِهِ ، كَأَنْ جُعِلَ لِرَبِّ السُّدُسِ نِصْفُ الرِّبْحِ لِقُوَّةِ حَذِقِهِ ( أَوْ يُقَالُ ) : عَلَى أَنَّ الرِّبْحَ ( بَيْنَنَا فَيَسْتَوُونَ فِيهِ ) لِإِضَافَتِهِ إلَيْهِمْ إضَافَةً وَاحِدَةً بِلَا تَرْجِيحٍ ( أَوْ ) لِيَعْمَلَ فِيهِ ( الْبَعْضُ ) مِنْ أَرْبَابِ الْأَمْوَالِ ( عَلَى أَنْ يَكُونَ لَهُ ) أَيْ : الْعَامِلِ مِنْهُمْ ( أَثَرٌ مِنْ رِبْحِ مَالِهِ ) كَأَنْ تَعَاقَدُوا عَلَى أَنْ يَعْمَلَ رَبُّ السُّدُسِ وَلَهُ ثُلُثُ الرِّبْحِ أَوْ نِصْفُهُ وَنَحْوُهُ ( وَتَكُونُ ) الشَّرِكَةُ إذَا تَعَاقَدُوا عَلَى أَنْ يَعْمَلَ بَعْضُهُمْ كَذَلِكَ ( عِنَانًا ) مِنْ حَيْثُ إحْضَارُ كُلٍّ مِنْهُمْ لِمَا لَهُ ( وَمُضَارَبَةً ) ؛ لِأَنَّ مَا يَأْخُذُهُ الْعَامِلُ زَائِدًا عَنْ رِبْحِ مَالِهِ فِي نَظِيرِ عَمَلِهِ فِي مَالِ غَيْرِهِ .( 6 / 32 )
