( وَلَا شَيْءَ عَلَى مَنْ لَمْ يَجْتَمِعْ لَهُ نِصَابٌ فِي وَاحِدٍ مِنْهَا ) ، أَيْ : الْمَحَالِّ الْمُتَبَاعِدَةِ ( غَيْرَ خَلِيطٍ ) لِأَهْلِهَا فِي نِصَابٍ ( فَإِذَا كَانَ لَهُ ) أَيْ : الشَّخْصِ مِنْ أَهْلِ الزَّكَاةِ ( سِتُّونَ شَاةً ) بِثَلَاثِ مَحَالَّ مُتَبَاعِدَةٍ ( فِي كُلِّ مَحِلٍّ عِشْرُونَ ) مِنْهَا ( خُلْطَةً بِعِشْرِينَ لِآخَرَ لَزِمَ رَبَّ السِّتِّينَ شَاةٌ وَنِصْفُ ) شَاةٍ ( وَ ) لَزِمَ ( كُلَّ خَلِيطٍ نِصْفُ شَاةٍ ) وَإِنْ لَمْ يَكُنْ خُلْطَةً مَعَ أَهْلِهَا فِي نِصَابٍ ، فَلَا شَيْءَ عَلَيْهِ( 5 / 92 )
