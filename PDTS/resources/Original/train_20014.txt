وَالْأُخْتِ لِأَبٍ ( أَخٌ لِأَبٍ ) اسْتَوَى لِلْجَدِّ الْمُقَاسَمَةُ وَالثُّلُثُ ؛ لِأَنَّ الْإِخْوَةَ مِثْلَاهُ ( فَلِلْجَدِّ ثُلُثٌ ) فَرْضًا أَوْ مُقَاسَمَةً ( وَلِلْأُخْتِ لِأَبَوَيْنِ نِصْفٌ يَبْقَى لَهُمَا ) أَيْ : لِلْأُخْتِ وَالْأَخِ لِأَبٍ ( سُدُسٌ عَلَى ) عَدَدِ رُءُوسِهِمْ ( ثَلَاثَةٌ ) لَا يَصِحُّ أَيْ : لَا يَنْقَسِمُ وَيُبَايِنُ ؛ فَاضْرِبْ الثَّلَاثَةَ فِي أَصْلِ الْمَسْأَلَةِ سِتَّةٍ ( فَتَصِحُّ مِنْ ثَمَانِيَةَ عَشَرَ ) لِلْجَدِّ سِتَّةٌ وَلِلْأُخْتِ لِأَبَوَيْنِ تِسْعَةٌ وَلِلْأَخِ لِأَبٍ سَهْمَانِ وَلِأُخْتِهِ سَهْمٌ ، وَكَذَا لَوْ كَانَ بَدَلَ الْأَخِ أُخْتَانِ لِأَبٍ .
