قَالَ السُّبْكِيُّ : وَلَا شَكَّ أَنَّ الرَّحِمَ هِيَ قَرَابَةٌ فَقَدْ قَالَ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فِي الْقِبْطِ : { إنَّ لَهُمْ ذِمَّةً وَرَحِمًا } لِأَنَّ أُمَّ إسْمَاعِيلَ مِنْهُمْ وَقَدْ افْتَخَرَ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ بِخَالِهِ سَعْدٍ { فَقَالَ : سَعْدٌ خَالِي فَلْيُرِنِي امْرُؤٌ خَالَهُ } حَسَّنَهُ التِّرْمِذِيُّ وَصَحَّحَهُ الْحَاكِمُ ( وَالْعِبْرَةُ ) فِيمَا ذُكِرَ ( بِأَقْرَبِ جَدٍّ يُنْسَبُ إلَيْهِ زَيْدٌ ، وَتُعَدُّ أَوْلَادُهُ ) أَيْ ذَلِكَ الْجَدِّ ( قَبِيلَةً ) فَيَرْتَقِي فِي بَنِي الْأَعْمَامِ إلَيْهِ وَلَا يُعْتَبَرُ مَنْ فِي دَرَجَتِهِ أَوْ مَنْ فَوْقَهُ فَالْوَصِيَّةُ لِأَقَارِبِ حَسَنِيٍّ لِأَوْلَادِ الْحَسَنِ دُونَ أَوْلَادِ مَنْ فَوْقَهُ وَأَوْلَادِ الْحُسَيْنِ( 11 / 186 )
