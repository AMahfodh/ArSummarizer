وَعَرْضَهُ ( وَصِفَتَهُ ) أَيْ غِلَظَهُ وَرِقَّتَهُ ( وَوَزْنَهُ إنْ بِيعَ بِهِ ) أَيْ بِالْوَزْنِ ( فَصَحَّ فِي السَّمَكِ الْمَلِيحِ ) أَيْ الْقَدِيدِ بِالْمِلْحِ يُقَالُ سَمَكٌ مَلِيحٌ وَمَمْلُوحٌ وَلَا يُقَالُ مَالِحٌ إلَّا فِي لُغَةٍ رَدِيئَةٍ ( وَالطَّرِيُّ حَيْثُ يُوجَدُ ) غَيْرُ مُقَيَّدٍ بِوَقْتٍ دُونَ وَقْتٍ حَتَّى لَوْ كَانَ فِي بَلَدٍ لَا يَنْقَطِعُ يَجُوزُ مُطْلَقًا ( وَزْنًا ضَرْبًا ) أَيْ نَوْعًا ( مَعْلُومَيْنِ ) قَيْدٌ لِلْمَلِيحِ وَالطَّرِيِّ .
