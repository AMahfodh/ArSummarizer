وَالْعَاشِرُ الدَّرْءُ إذَا لَمْ يَكُنْ بَيْنَ يَدَيْهِ سُتْرَةٌ أَوْ مَرَّ بَيْنَهُ وَبَيْنَ السُّتْرَةِ لِقَوْلِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ { ادْرَءُوا مَا اسْتَطَعْتُمْ } ( وَيَدْرَأُ ) أَيْ يَدْفَعُ ( بِالْإِشَارَةِ كَمَا فَعَلَ النَّبِيُّ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ بِوَلَدَيْ أُمِّ سَلَمَةَ ) { حَيْثُ كَانَ يُصَلِّي فِي بَيْتِهَا فَقَامَ وَلَدُهَا عُمَرُ لِيَمُرَّ بَيْنَ يَدَيْهِ( 2 / 153 )
