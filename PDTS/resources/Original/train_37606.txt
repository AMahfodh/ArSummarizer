( قَوْلُهُ مِمَّا بَاشَرَهُ الْقَطْعُ ) أَيْ بِخِلَافِ الْبَاطِنِ الَّذِي كَانَ مُنْفَتِحًا قَبْلَ الْقَطْعِ فَلَا يَجِبُ غَسْلُهُ وَإِنْ ظَهَرَ بَعْدَ قَطْعِ مَا كَانَ يَسْتُرُهُ شَيْخُنَا وَكُرْدِيٌّ ( قَوْلُهُ جُدِعَ ) بِالدَّالِ الْمُهْمَلَةِ ع ش ( قَوْلُهُ وَذَلِكَ ) أَيْ وُجُوبُ التَّعْمِيمِ ( قَوْلُهُ وَمَرَّ ) أَيْ فِي شَرْحِ قَوْلِ الْمُصَنِّفِ وَالْمُتَغَيِّرُ بِمُسْتَغْنًى عَنْهُ كُرْدِيٌّ .( 3 / 193 )
