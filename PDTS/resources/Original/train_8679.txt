( قَوْلُهُ : أَوْ جَارِيًا مُطْلَقًا ) أَيْ كَثِيرًا أَوْ قَلِيلًا ( قَوْلُهُ : كَالْمِهْرَاسِ ) هُوَ الْحَوْضُ الصَّغِيرُ وَلَا بُدَّ مِنْ حَذْفٍ فِي الْعِبَارَةِ أَيْ ، فَإِنْ كَانَ ظَرْفُ الْمَاءِ ( قَوْلُهُ : وَنَحْوِهِ ) أَيْ كَحُفْرَةٍ فِيهَا مَاءٌ قَلِيلٌ قَوْلُهُ : وَلَمْ يُمْكِنْ الْإِفْرَاغُ مِنْهُ ) ، فَإِنْ كَانَ يُمْكِنُ الْإِفْرَاغُ مِنْهُ فَلَا تَحْصُلُ السُّنَّةُ إلَّا إذَا غَسَلَهُمَا خَارِجَهُ ( قَوْلُهُ : فَإِنَّهُ يُدْخِلُ يَدَيْهِ ) أَيْ وَيَغْسِلُهُمَا فِيهِ كَمَا صَرَّحَ بِهِ بَعْضُ الشُّرَّاحِ ( قَوْلُهُ : فَإِنْ أَمْكَنَهُ أَنْ يَتَوَصَّلَ ) أَيْ بِأَنْ يَتَحَيَّلَ بِفِيهِ أَوْ بِثَوْبٍ ( قَوْلُهُ : فَإِنَّهُ يُدْخِلُهُمَا( 2 / 134 )
