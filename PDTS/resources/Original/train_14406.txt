( وَ ) عَلَى الْإِمَامِ أَنْ ( يَخْتَارَ لِذَلِكَ ) أَيْ : نَصْبِ الْقُضَاةِ ( أَفْضَلَ مَنْ يَجِدُ عِلْمًا وَوَرَعًا ) لِأَنَّ الْإِمَامَ يَنْظُرُ لِلْمُسْلِمِينَ فَوَجَبَ عَلَيْهِ تَحَرِّي الْأَصْلَحِ لَهُمْ ( وَيَأْمُرُهُ ) أَيْ : الْإِمَامُ إذَا وَلَّاهُ ( بِالتَّقْوَى ) لِأَنَّهَا رَأْسُ الْأَمْرِ وَمِلَاكُهُ .
