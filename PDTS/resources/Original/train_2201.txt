( قَوْلُهُ إجْمَاعًا ) إلَى الْمَتْنِ فِي النِّهَايَةِ إلَّا قَوْلَهُ بِخِلَافِهِ إلَى وَلَوْ شَكَّ وَكَذَا فِي الْمُغْنِي إلَّا قَوْلَهُ وَقِيَاسُهُ إلَخْ ( قَوْلُهُ وَكَذَا الصُّوفُ ) أَيْ لِلضَّأْنِ ( وَالْوَبَرُ ) أَيْ لِلْإِبِلِ ( وَالرِّيشُ ) أَيْ لِلطَّيْرِ ( قَوْلُهُ سَوَاءٌ أَنُتِفَ إلَخْ ) وَيُكْرَهُ نَتْفُ شَعْرِ الْحَيَوَانِ حَيْثُ كَانَ تَأَلُّمُهُ بِهِ يَسِيرًا وَإِلَّا حَرُمَ كُرْدِيٌّ ( قَوْلُهُ أَوْ تَنَاثَرَ ) أَيْ بِنَفْسِهِ ( قَوْلُهُ وَخَرَجَ بِشَعْرِ الْمَأْكُولِ عُضْوٌ إلَخْ ) وَكَذَا خَرَجَ بِذَلِكَ الْقَرْنُ وَالظِّلْفُ وَالظُّفْرُ الْمُبَانَةُ فَهِيَ نَجِسَةٌ شَرْحُ بَافَضْلٍ وَكُرْدِيٌّ .
