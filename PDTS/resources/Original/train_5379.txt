( أَوْ ) كَانَ الزَّوْجُ ( مَقْطُوعَ الذَّكَرِ وَالْأُنْثَيَيْنِ أَوْ ) مَقْطُوعَ ( الْأُنْثَيَيْنِ فَقَطْ ) أَيْ مَعَ بَقَاءِ الذَّكَرِ ( لَمْ يَلْحَقْهُ نَسَبُهُ ) لِأَنَّ الْوَلَدَ لَا يُوجَدُ إلَّا مِنْ مَنِيٍّ ، وَمَنْ قُطِعَ خُصْيَتَاهُ لَا مَنِيَّ لَهُ ، لِأَنَّهُ لَا يُنْزِلُ إلَّا مَاءً رَقِيقًا لَا يُخْلَقُ مِنْهُ الْوَلَدُ وَلَا وَجَدَ ذَلِكَ ، وَلَا اعْتِبَارَ بِإِيلَاجٍ لَا يُخْلَقُ مِنْهُ الْوَلَدُ ، كَمَا لَوْ أَوْلَجَ الصَّغِيرُ ( وَيَلْحَقُ ) الْوَلَدُ ( مَقْطُوعَ الذَّكَرِ فَقَطْ ) لِأَنَّهُ يُمْكِنُ أَنْ يُسَاحِقَ فَيُنْزِلَ مَا يُخْلَقُ مِنْهُ الْوَلَدُ ، وَلِهَذَا أَلْحَقْنَا وَلَدَ الْأَمَةِ بِسَيِّدِهَا إذَا اعْتَرَفَ بِوَطْئِهَا دُونَ الْفَرْجِ ( وَ ) يَلْحَقُ ( الْعِنِّينُ ) لِإِمْكَانِ إنْزَالِهِ مَا يُخْلَقُ مِنْهُ الْوَلَدُ .( 19 / 116 )
