( وَإِنْ بَيَّنَ صِفَتَهُ أَيْضًا ) كَمَا بَيَّنَ جِنْسَهُ ( وَجَبَ هُوَ ) أَيْ الْمُسَمَّى ( لَا قِيمَتُهُ ) فَيُجْبَرُ عَلَى تَسْلِيمِهِ ؛ لِأَنَّ مَوْصُوفَهُ يَجِبُ فِي الذِّمَّةِ ثُبُوتًا صَحِيحًا حَالًّا ، أَوْ مُؤَجَّلًا ( وَقِيلَ الثَّوْبُ مِثْلُهُ ) أَيْ مِثْلُ الْمَكِيلِ ( إنْ بُولِغَ فِي وَصْفِهِ ) وَهُوَ قَوْلُ زُفَرَ كَمَا بَيَّنَّاهُ آنِفًا .( 3 / 139 )
