( أَوْ قَالَتْ ) الْمُطَلَّقَةُ عِنْدَ قَصْدِهِ ارْتِجَاعَهَا أَنَا ( حِضْت ثَالِثَةً ) فَلَا رَجْعَةَ لَك عَلَيَّ ( فَأَقَامَ ) الزَّوْجُ ( بَيِّنَةً ) شَهِدَتْ ( عَلَى قَوْلِهَا قَبْلَهُ ) أَيْ قَبْلَ هَذَا الْقَوْلِ ( بِمَا يُكَذِّبُهَا ) بِأَنْ شَهِدَتْ بِأَنَّهَا قَالَتْ : لَمْ أَحِضْ أَصْلًا أَوْ لَمْ أَحِضْ ثَالِثَةً وَلَيْسَ بَيْنَ قَوْلَيْهَا مَا يُمْكِنُ أَنْ تَحِيضَ فِيهِ فَتَصِحُّ رَجْعَتُهُ فَإِنْ لَمْ يُقِمْهَا لَمْ تَصِحَّ ، وَلَوْ رَجَعَتْ لِتَصْدِيقِهِ
