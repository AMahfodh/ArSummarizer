( قَوْلُهُ وَحِكْمَتُهُ ) عَطْفُ تَفْسِيرٍ ( قَوْلُهُ مَعَ تَكْبِيرَةِ الْإِحْرَامِ رُكُوعٌ ) الْمُنَاسِبُ أَنْ يَقُولَ حَرَكَةُ رُكْنٍ ( قَوْلُهُ كَالرَّمَلِ فِي الْحَجِّ ) فَإِنَّ حِكْمَتَهُ قَوْلُ الْمُشْرِكِينَ إنَّ الْمُؤْمِنِينَ قَدْ وَهَنَتْهُمْ حُمَّى يَثْرِبَ فَأُمِرُوا بِالرَّمَلِ تَكْذِيبًا لَهُمْ قَوْلُهُ لِلْإِشَارَةِ إلَى أَنَّ الْمُصَلِّيَ رَفَضَ الدُّنْيَا ) هَذَا يَكُونُ عَلَى صُورَةِ النَّابِذِ .( 3 / 384 )
