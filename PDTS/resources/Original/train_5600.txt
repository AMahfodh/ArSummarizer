( وَمَنْ أُبِيعَتْ ) بِالْبِنَاءِ لِلْمَجْهُولِ مِنْ الْإِمَاءِ ( وَلَمْ تُسْتَبْرَأْ ) قَبْلَ بَيْعٍ ( فَأَعْتَقَهَا مُشْتَرٍ ) أَوْ أَرَادَ تَزَوُّجَهَا ( قَبْلَ وَطْءٍ وَ ) قَبْلَ ( اسْتِبْرَاءٍ اسْتَبْرَأَتْ ) نَفْسَهَا ( أَوْ تَمَّمَتْ مَا وُجِدَ عِنْد مُشْتَرٍ ) مِنْ اسْتِبْرَاءٍ إنْ عَتَقَتْ فِي أَثْنَائِهِ لِتَعْلَمَ بَرَاءَةَ رَحِمِهَا .( 16 / 494 )
