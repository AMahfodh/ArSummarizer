( قَوْلُهُ : بِإِقْرَارِهِ بِهَا ) كَأَنْ شَهِدُوا عَلَيْهِ بِأَنَّهُ أَقَرَّ بِأَنَّهُ سَجَدَ لِصَنَمٍ ا ه رَشِيدِيٌّ ( قَوْلُهُ : وَبَحَثَ ابْنُ الرِّفْعَةِ إلَخْ ) اعْتَمَدَهُ الْمُغْنِي وَالرَّشِيدِيُّ ( قَوْلُهُ : وَيُرَدُّ ) أَيْ بَحْثُهُ ( قَوْلُهُ : وَمِنْهُ ) أَيْ الرُّجُوعِ ( قَوْلُهُ : ثَمَّ ) أَيْ فِي الْإِقْرَارِ بِالزِّنَا وَ ( قَوْلُهُ : لَا هُنَا ) أَيْ فِي الْإِقْرَارِ بِالرِّدَّةِ ( قَوْلُهُ : بِالْإِسْلَامِ ) أَيْ بِالنُّطْقِ بِالشَّهَادَتَيْنِ ( قَوْلُهُ : فَلَوْ لَمْ يُنْكِرْ ) ، وَإِنَّمَا عِبَارَةُ الْمُغْنِي فَلَوْ( 38 / 281 )
