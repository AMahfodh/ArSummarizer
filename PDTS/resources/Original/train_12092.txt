هَذِهِ يُرَاعَى الِاسْتِصْحَابُ فَتُحْسَبُ حَيَّةً وَتَتَعَلَّقُ بِهَا الزَّكَاةُ بِخِلَافِ الْمُزْلَقِ ( إلَّا أَنْ تَتَحَقَّقَ الْحَيَاةُ ) ابْنُ عَرَفَةَ : يُصَلَّى عَلَى مَنْ وُلِدَ إنْ عُلِمَتْ حَيَاتُهُ ، وَإِنْ جُهِلَتْ فَكَالسِّقْطِ لَا يُصَلَّى عَلَيْهِ ( وَغُسِلَ دَمُهُ وَلُفَّ بِخِرْقَةٍ ) / 97 تَقَدَّمَتْ رِوَايَةُ عَلِيٍّ بِهَذَا وَقَوْلُ ابْنِ حَبِيبٍ عِنْدَ قَوْلِهِ : وَتَلَازُمًا / 97 ( وُورِيَ ) اُنْظُرْ عِنْدَ قَوْلِهِ : وَدَفْنُهُ بِدَارٍ .( 2 / 402 )
