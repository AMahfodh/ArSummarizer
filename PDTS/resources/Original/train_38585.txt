( قَوْلُهُ وَلِكَوْنِهَا ) إلَى قَوْلِهِ هَذَا إنْ لَمْ تُنْذَرْ فِي الْمُغْنِي إلَّا قَوْلَهُ وَآثَرَ إلَى فَالْأَفْضَلُ وَقَوْلَهُ أَيْ إلَى لِلْقَابِلَةِ ( قَوْلُهُ وَلِكَوْنِهَا إلَخْ ) مُتَعَلِّقٌ بِأَشْبَهَتْ ( قَوْلُهُ وَتُجْزِئُ ) إلَى قَوْلِهِ هَذَا إنْ لَمْ تُنْذَرْ فِي النِّهَايَةِ ( قَوْلُهُ وَآثَرَ ) أَيْ الْمُصَنِّفُ ( قَوْلُهُ نَظِيرُ مَا مَرَّ ) هُوَ بِرَفْعِ نَظِيرُ خَبَرًا عَنْ الْأَفْضَلِ ا ه رَشِيدِيٌّ ( قَوْلُهُ مِنْ سَبْعِ شِيَاهٍ إلَخْ ) هَلْ هُوَ مَخْصُوصٌ بِالذَّكَرِ أَمْ لَا وَظَاهِرُ الْإِطْلَاقِ الثَّانِي ( قَوْلُهُ ثُمَّ الْإِبِلُ ثُمَّ الْبَقَرُ ) وَلَوْ ذَبَحَ بَقَرَةً أَوْ بَدَنَةً عَنْ سَبْعَةٍ أَوْ لَا جَازَ وَكَذَا لَوْ اشْتَرَكَ فِيهَا جَمَاعَةٌ سَوَاءٌ أَرَادَ كُلُّهُمْ الْعَقِيقَةَ أَوْ بَعْضُهُمْ ذَلِكَ وَبَعْضُهُمْ اللَّحْمَ نِهَايَةٌ وَمُغْنِي .( 41 / 183 )
