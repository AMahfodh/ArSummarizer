كَقَوْلِهِ أَعْطُوا ثُلُثِي قَرَابَتِي فُلَانًا فَلَمْ يُعْلَمْ لَهُ قَرِيبٌ بِهَذَا الِاسْمِ ( فَتَصَدَّقَ هُوَ ) أَيْ الْوَصِيُّ بِهِ ( أَوْ ) تَصَدَّقَ ( حَاكِمٌ بِهِ ) أَيْ الثُّلُثِ ( ثُمَّ ثَبَتَ ) الْمُوصَى لَهُ ( لَمْ يَضْمَنْ ) مُوصًى إلَيْهِ وَلَا حَاكِمٌ شَيْئًا أَيْ لِأَنَّهُ مَعْذُورٌ بِعَدَمِ عِلْمِهِ بِهِ .
