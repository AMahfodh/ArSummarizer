( وَقَدْ يَتَرَدَّدُ بَيْنَ الثَّلَاثَةِ ) أَعْنِي ( الرِّيَاءَ وَالْإِخْلَاصَ وَالْحَيَاءَ كَرَجُلٍ يَطْلُبُ مِنْهُ صَدِيقُهُ ) مَثَلًا ( قَرْضًا ) مَثَلًا أَيْضًا ( وَلَا يَسْخُو ) أَيْ لَا تَسْمَحُ نَفْسُهُ ( بِإِقْرَاضِهِ ) أَيْ إعْطَاءِ الْقَرْضِ ( إلَّا أَنَّهُ يَسْتَحِيْ مِنْ رَدِّهِ وَيَعْلَمُ أَنَّهُ ) أَيْ الصِّدِّيقَ ( لَوْ أَرْسَلَهُ ) أَيْ الْمَطْلُوبُ مِنْهُ ( عَلَى لِسَانِ غَيْرِهِ لَا يَسْتَحِيْ مِنْهُ وَلَا يُقْرِضُ رِيَاءً ) لِلنَّاسِ ( وَلَا يَطْلُبُ الثَّوَابَ ) فِي الْقَرْضِ ( فَلَهُ عِنْدَ ذَلِكَ ) الْمَطْلَبِ أَحْوَالٌ سِتٌّ ثَلَاثٌ فِي الْمَنْعِ وَثَلَاثٌ فِي الْإِعْطَاءِ ( أَنْ يُشَافِهَ ) أَيْ يَتَكَلَّمَ فِي حُضُورِهِ ( بِالرَّدِّ الصَّرِيحِ ) نَحْوُ أَنْ يَقُولَ لَا أُقْرِضُك ( فَيُنْسَبَ ) عِنْدَ النَّاسِ ( إلَى قِلَّةِ الْحَيَاءِ ) بِالْمُشَافَهَةِ بِالرَّدِّ الصَّرِيحِ ( أَوْ يَتَعَلَّلَ بِكَذِبٍ ) بِأَنْ يَقُولَ لَيْسَ عِنْدِي مَالٌ ( أَوْ تَعْرِيضٍ ) كَمَنْ يَجِدُ مَا يَطْلُبُهُ وَيَقُولُ لَيْسَ فِي يَدِي شَيْءٌ وَيَقْصِدُ حَقِيقَةَ الْيَدِ لَا الْمِلْكَ أَوْ لَيْسَ عِنْدِي مَالٌ وَيَقْصِدُ مِنْ النَّوْعِ الْمَخْصُوصِ .
