( وَ ) لَوْ حَلَفَ ( لَا أَشْرَبُ إلَّا أَنْ يَشَاءَ زَيْدٌ فَإِنْ شَاءَ فَلَهُ الشُّرْبُ ) وَلَا حِنْثَ لِعَدَمِ شَرْطِهِ ( وَإِنْ لَمْ يَشَأْ ) زَيْدٌ ( لَمْ يَشْرَبْ ) الْحَالِفُ وَيَحْنَثُ بِهِ لِوُجُودِ شَرْطِهِ وَهُوَ الشُّرْبُ بِغَيْرِ إذْنِ زَيْدٍ ( فَإِنْ خَفِيَتْ مَشِيئَتُهُ لِغَيْبَةٍ أَوْ مَوْتٍ أَوْ جُنُونٍ لَمْ يَشْرَبْ ) لِأَنَّ الْأَصْلَ عَدَمُهَا ( وَإِنْ شَرِبَ حَنِثَ ) لِوُجُودِ الْمُعَلَّقِ عَلَيْهِ .( 21 / 345 )
