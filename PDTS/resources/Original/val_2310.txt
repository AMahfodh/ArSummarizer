وَتُعْرَفُ الْقُوَّةُ بِأَحَدِ ثَلَاثَةِ أُمُورٍ ذَكَرَهَا مَعَ زِيَادَةِ أَمْثِلَتِهَا وَإِنْ لَمْ يُنَبِّهْ إلَّا عَلَى الْأَخِيرِ كَمَا سَيَأْتِي فَقَالَ ( أَمَّا بِأَنْ تَحْجُبَ ) إحْدَاهُمَا الْأُخْرَى ( مَثِّلْ ) أَنْتَ لَهُ ( بِابْنَةِ أُخْتٍ لِأُمٍّ ) بِأَنْ ( وُطِئَتْ ) أَيْ الْأُمُّ أَيْ وَطِئَهَا ابْنُهَا فَوَلَدَتْ بِنْتًا فَهِيَ بِنْتُهُ وَأُخْتُهُ لِأُمِّهِ فَتَرِثُهُ بِالْبُنُوَّةِ لَا بِأُخُوَّةِ الْأُمِّ لِأَنَّهَا( 13 / 184 )
