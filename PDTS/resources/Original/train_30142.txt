( وَلَا يُحَدُّ حَالَ سُكْرِهِ ) بَلْ يُؤَخَّرُ إلَى أَنْ يُفِيقَ لِيَرْتَدِعَ ( وَسَوْطُ الْحُدُودِ ) فِي الشُّرْبِ وَالزِّنَى وَالْقَذْفِ ( بَيْنَ قَضِيبٍ وَعَصًا وَرَطْبٍ وَيَابِسٍ ) لِلِاتِّبَاعِ ( وَيُفَرِّقُهُ ) أَيْ السَّوْطَ مِنْ حَيْثُ الْعَدَدِ ( عَلَى الْأَعْضَاءِ ) وَلَا يَجْمَعُ فِي عُضْوٍ وَاحِدٍ ( إلَّا الْمُقَاتِلُ ) كَثُغْرَةِ النَّحْرِ وَالْفَرْجِ وَنَحْوِهِمَا ( وَالْوَجْهُ قِيلَ وَالرَّأْسُ ) لِشَرَفِهِ كَالْوَجْهِ وَالْأَصَحُّ لَا وَالْفَرْقُ أَنَّهُ مُغَطًّى غَالِبًا فَلَا يُخَافُ تَشْوِيهُهُ بِالضَّرْبِ بِخِلَافِ الْوَجْهِ ( وَلَا تُشَدُّ يَدُهُ ) بَلْ تُتْرَكُ يَدَاهُ مُطْلَقَتَيْنِ حَتَّى يَتَّقِيَ بِهِمَا ( وَلَا تُجَرَّدُ ثِيَابُهُ ) بَلْ يُتْرَكُ عَلَيْهِ قَمِيصٌ أَوْ قَمِيصَانِ دُونَ جُبَّةٍ مَحْشُوَّةٍ أَوْ فَرْوَةٍ ( وَيُوَالِي الضَّرْبَ ) عَلَيْهِ ( بِحَيْثُ يَحْصُلُ زَجْرٌ وَتَنْكِيلٌ ) فَلَا يَجُوزُ أَنْ يَضْرِبَ فِي كُلِّ يَوْمٍ سَوْطًا أَوْ سَوْطَيْنِ .
