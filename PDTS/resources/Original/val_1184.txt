( فَإِنْ أَدَّى قَدْرَ زَكَاةِ أَحَدِهِمَا ) أَيْ : الْحَاضِرِ وَالْغَائِبِ وَلَمْ يُعَيِّنْهُ ( جَعَلَهَا ) أَيْ : الزَّكَاةَ ( لِأَيِّهِمَا شَاءَ ، كَتَعَيُّنِهِ ابْتِدَاءً ) حِينَ إخْرَاجٍ ( وَإِنْ لَمْ يُعَيِّنْ ) وَاحِدًا مِنْهُمَا ( أَجْزَأَ ) مُخْرَجٌ ( عَنْ أَحَدِهِمَا ) فَيُخْرِجُ عَنْ الْآخَرِ .( 3 / 207 )
