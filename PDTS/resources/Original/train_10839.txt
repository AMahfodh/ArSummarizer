سم ( قَوْلُهُ : وَمِنْهُ ) أَيْ : الْحُلْقُومِ ( قَوْلُهُ : النَّاتِئُ ) أَيْ : الْمُرْتَفِعُ ( قَوْلُهُ : الْمُتَّصِلُ ) أَيْ : كَالْمُتَّصِلِ فَهُوَ كِنَايَةٌ عَنْ الْقُرْبِ ، وَإِلَّا فَلَا اتِّصَالَ حَقِيقَةً كَمَا هُوَ مُشَاهَدٌ ( قَوْلُهُ : بِالْفَمِ ) أَيْ : آخِرِهِ ( قَوْلُهُ : وَيُسَمَّى الْحَرْقَدَةَ ) وَهِيَ بِفَتْحِ الْحَاءِ ، وَالْقَافِ عُقْدَةُ الْحُنْجُورِ ا ه .
