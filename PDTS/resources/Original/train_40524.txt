( فَرْعٌ لَوْ قَالَ بِعْتُك هَذِهِ الصُّبْرَةَ أَوْ الْقَطِيعَ أَوْ الْأَرْضَ ) أَوْ الثَّوْبَ كَمَا صَرَّحَ بِهِ الْأَصْلُ ( كُلَّ صَاعٍ أَوْ شَاةٍ أَوْ ذِرَاعٍ بِدِرْهَمٍ ) مَثَلًا ( صَحَّ ) وَلَا يَضُرُّ الْجَهْلُ بِجُمْلَةِ الثَّمَنِ لِلْعِلْمِ بِهِ تَفْصِيلًا وَفَارَقَ عَدَمَ الصِّحَّةِ فِيمَا لَوْ بَاعَ ثَوْبًا بِمَا رُقِمَ عَلَيْهِ مِنْ الدَّرَاهِمِ الْمَجْهُولَةِ الْقَدْرِ بِأَنَّ الْغَرَرَ مُنْتَفٍ فِي الْحَالِ لِأَنَّ مَا يُقَابِلُ كُلَّ صَاعٍ مَعْلُومِ الْقَدْرِ حِينَئِذٍ بِخِلَافِهِ فِي تِلْكَ ( وَلَا يَصِحُّ بَيْعُ عَشْرِ شِيَاهٍ مِنْ هَذِهِ الْمِائَةِ ) وَإِنْ عَلِمَ عَدَدَ الْجُمْلَةِ ( بِخِلَافِ مِثْلِهِ مِنْ الصُّبْرَةِ وَالْأَرْضِ ) وَالثَّوْبِ لِاخْتِلَافِ قِيمَةِ الشِّيَاهِ فَلَا يَدْرِي كَمْ قِيمَةِ الْعَشَرَةِ مِنْ الْجُمْلَةِ وَلِأَنَّهُ لَا يُمْكِنُ فِيهَا إلَّا الْإِشَاعَةُ بِخِلَافِ نَظِيرِهِ مِنْ الْأَشْيَاءِ الْمَذْكُورَةِ وَبِخِلَافِ مَا لَوْ بَاعَهُ ذِرَاعًا مِنْ ثَوْبٍ أَوْ أَرْضٍ وَذَرْعُهُ مَعْلُومٌ لَهُمَا ( وَلَوْ قَالَ بِعْتُك مِنْ هَذِهِ الصُّبْرَةِ كُلَّ صَاعٍ بِدِرْهَمٍ ) مَثَلًا ( أَوْ كُلَّ صَاعٍ مِنْ هَذِهِ الصُّبْرَةِ بِدِرْهَمٍ ) مَثَلًا ( لَمْ يَصِحَّ ) لِأَنَّهُ لَمْ يَبِعْ الْجُمْلَةَ بَلْ بَعْضَهَا الْمُحْتَمِلَ لِلْقَلِيلِ وَالْكَثِيرِ فَلَا يَعْلَمُ قَدْرَ الْمَبِيعِ تَحْقِيقًا وَلَا تَخْمِينًا ( أَوْ ) قَالَ ( بِعْتُكهَا بِعَشَرَةِ دَرَاهِمَ ) مَثَلًا ( كُلَّ صَاعٍ بِدِرْهَمٍ ) أَوْ قَالَ مِثْلَهُ فِي الْأَرْضِ وَالثَّوْبِ ( صَحَّ إنْ وَافَقَ ) عَدَدُ الصِّيعَانِ أَوْ الْأَذْرُعِ الثَّمَنَ بِأَنْ خَرَجَ عَشْرٌ لِتُوَافِقَ جُمْلَةَ الثَّمَنِ وَتَفْصِيلَهُ ( لَا أَنْ زَادَ أَوْ نَقَصَ ) فَلَا يَصِحُّ لِتَعَذُّرِ الْجَمْعِ بَيْنِ الْجُمْلَةِ وَالتَّفْصِيلِ .
