( وَلَوْ ) ( أَصْدَقَ عَبْدَيْنِ ) مَثَلًا ( فَتَلِفَ عَبْدٌ ) بِآفَةٍ أَوْ إتْلَافِ الزَّوْجِ ( قَبْلَ قَبْضِهِ ) ( انْفَسَخَ ) عَقْدُ الصَّدَاقِ ( فِيهِ لَا فِي الْبَاقِي عَلَى الْمَذْهَبِ ) تَفْرِيقًا لِلصَّفْقَةِ فِي الدَّوَامِ ( وَلَهَا الْخِيَارُ فِيهِ ) لِتَلَفِ بَعْضِ الْمَعْقُودِ عَلَيْهِ ( فَإِنْ فَسَخَتْ فَمَهْرُ مِثْلٍ ) عَلَى الْأَوَّلِ ( وَإِلَّا ) بِأَنْ أَجَازَتْ ( ف ) لَهَا ( حِصَّةٌ ) أَيْ قِسْطُ قِيمَةِ ( التَّالِفِ مِنْهُ ) أَيْ مَهْرِ الْمِثْلِ ، فَلَوْ كَانَتْ قِيمَتُهُ ثُلُثَ مَجْمُوعِ قِيمَتِهِمَا فَلَهَا ثُلُثُ مَهْرِ الْمِثْلِ ، وَإِنْ أَتْلَفَتْهُ فَقَابِضَةٌ لِقِسْطِهِ مِنْ الصَّدَاقِ أَوْ أَجْنَبِيٍّ تَخَيَّرَتْ كَمَا مَرَّ .
