الْأَهْوَاءُ عَمَّتْ فَأَعْمَتْ ( لَا ) يُكْرَهُ الْكَلَامُ ( بَعْدَ ) صَلَاةِ ( فَجْرٍ وَقَبْلَ ) صَلَاةِ ( صُبْحٍ وَ ) كُرِهَ ( ضِجْعَةٌ ) بِكَسْرِ الضَّادِ الْمُعْجَمَةِ أَيْ الِاضْطِجَاعُ عَلَى شِقِّهِ الْأَيْمَنِ مُسْتَقْبِلًا وَاضِعًا كَفَّهُ الْيُمْنَى تَحْتَ خَدِّهِ ( بَيْنَ ) صَلَاةِ ( صُبْحٍ وَرَكْعَتَيْ الْفَجْرِ ) إذَا فَعَلَهَا اسْتِنَانًا لَا اسْتِرَاحَةً مِنْ طُولِ قِيَامِ اللَّيْلِ( 2 / 259 )
