( وَنُفِّذَتْ الْوَصِيَّةُ بِهِ ) أَيْ بِالْحَجِّ ( مِنْ الثُّلُثِ ) صَرُورَةً ، أَوْ غَيْرَهُ سَمَّى مَالًا ، أَوْ ثُلُثًا ، أَوْ أَطْلَقَ ( وَحُجَّ عَنْهُ ) أَيْ عَنْ الْمُوصِي ( حِجَجٌ إنْ ) سَمَّى الثُّلُثَ ( وَسِعَ ) الثُّلُثُ حِجَجًا ( وَقَالَ ) الْمُوصِي ( يُحَجُّ بِهِ ) أَيْ بِالثُّلُثِ ( لَا ) إنْ قَالَ يُحَجُّ عَنِّي ( مِنْهُ ) فَحَجَّةٌ وَاحِدَةٌ لِأَنَّ مِنْ( 5 / 331 )
