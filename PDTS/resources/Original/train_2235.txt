( وَمَتَى رَجَعَ مُقِرٌّ ) أَيْ بِزِنًا عَنْ إقْرَارٍ لَمْ يَقُمْ ( أَوْ ) رَجَعَ مُقِرٌّ ( بِسَرِقَةٍ أَوْ ) بِشُرْبِ خَمْرٍ عَنْ إقْرَارِهِ ( قَبْلَهُ ) أَيْ قَبْلَ أَنْ يُقَامَ عَلَيْهِ الْحَدُّ ( وَلَوْ بَعْدَ الشَّهَادَةِ عَلَى إقْرَارِهِ ) بِالزِّنَا أَوْ السَّرِقَةِ أَوْ الشُّرْبِ ( لَمْ يَقُمْ ) عَلَيْهِ ( وَإِنْ رَجَعَ فِي أَثْنَائِهِ ) أَيْ الْحَدِّ ( أَوْ هَرَبَ تُرِكَ ) لِأَنَّ { مَاعِزًا هَرَبَ فَذُكِرَ ذَلِكَ لِلنَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فَقَالَ هَلَّا تَرَكْتُمُوهُ يَتُوبُ فَيَتُوبَ اللَّهُ عَلَيْهِ } .
