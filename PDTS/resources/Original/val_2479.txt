( وَلَا ) يَحِلُّ لِلْفَرْعِ أَنْ ( يَبْدَأَ أَصْلَهُ الْمُشْرِكَ بِقَتْلٍ ) كَمَا لَا يَبْدَأُ قَرِيبَهُ الْبَاغِيَ ( وَيَمْتَنِعُ الْفَرْعُ ) عَنْ قَتْلِهِ بَلْ يَشْغَلُهُ ( لِ ) لِأَجْلِ أَنْ ( يَقْتُلَهُ غَيْرُهُ ) فَإِنْ فُقِدَ قَتَلَهُ ( وَلَوْ قَتَلَهُ فَهَدَرٌ ) لِعَدَمِ الْعَاصِمِ ( وَلَوْ قَصَدَ الْأَصْلُ قَتْلَهُ وَلَمْ يُمْكِنْ دَفْعُهُ إلَّا بِقَتْلِهِ قَتَلَهُ ) لِجَوَازِ الدَّفْعِ مُطْلَقًا
