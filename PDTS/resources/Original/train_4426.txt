أَيْ الْمَأْخُوذَةُ مِنْ كُفَّارٍ ( ثَلَاثُ ) أَصْنَافٍ ( إحْدَاهَا ) الْمَأْخُوذَةُ ( عَنْوَةً ) أَيْ قَهْرًا وَغَلَبَةً ( وَهِيَ مَا أُجْلُوا ) أَيْ أَهْلُهَا الْحَرْبِيُّونَ ( عَنْهَا بِالسَّيْفِ وَيُخَيَّرُ إمَامٌ بَيْنَ قَسْمِهَا ) بَيْنَ الْغَانِمِينَ ( كَمَنْقُولٍ وَ ) بَيْنَ ( وَقْفِهَا لِلْمُسْلِمِينَ بِلَفْظٍ يَحْصُلُ بِهِ ) الْوَقْفُ ( وَيَضْرِبُ عَلَيْهَا خَرَاجًا ) مُسْتَمِرًّا ( يُؤْخَذُ مِمَّنْ هِيَ بِيَدِهِ مِنْ مُسْلِمٍ وَذِمِّيٍّ ) هُوَ أُجْرَتُهَا كُلَّ عَامٍ .
