( وَإِنْ قَالَ ) مَنْ بِيَدِهِ نَحْوِ عَبْدٍ ( لَهُ ) أَيْ لِفُلَانٍ ( فِيهِ ) أَلْفٌ ( أَوْ ) قَالَ لَهُ عَلَيَّ ( مِنْهُ أَلْفٌ ) صَحَّ إقْرَارُهُ و ( قِيلَ لَهُ فَسِّرْ ) سَبَبَهُ لِأَنَّهُ لَا يُعْلَمُ إلَّا مِنْ جِهَتِهِ ( وَيُقْبَلُ ) تَفْسِيرُهُ ( بِجِنَايَةِ ) الْعَبْدِ عَلَى الْمُقَرِّ لَهُ .
