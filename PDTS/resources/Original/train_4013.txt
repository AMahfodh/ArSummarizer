تَدَافُعٍ ( فَدَمُ الرَّابِعِ هَدْرٌ ) لِأَنَّهُ لَمْ يَسْقُطْ عَلَيْهِ أَحَدٌ وَإِنَّمَا مَاتَ لِسُقُوطِهِ ( وَعَلَيْهِ ) أَيْ عَلَى عَاقِلَةِ الرَّابِعِ ( دِيَةُ الثَّالِثِ ) لِأَنَّهُ مَاتَ بِسُقُوطِهِ عَلَيْهِ ( وَدِيَةُ الثَّانِي عَلَيْهِ ) أَيْ عَلَى عَاقِلَتِهِ ( وَعَلَى ) عَاقِلَةِ ( الثَّالِثِ نِصْفَيْنِ ) لِأَنَّهُ مَاتَ بِسُقُوطِهِمَا عَلَيْهِ ( وَدِيَةُ الْأَوَّلِ عَلَى عَاقِلَةِ الثَّلَاثَةِ أَثْلَاثًا ) لِأَنَّهُ مَاتَ بِسُقُوطِهِمْ عَلَيْهِ .( 20 / 221 )
