( وَ ) يُفْرَضُ لَهَا ( زِينَةٌ تَسْتَضِرُّ ) أَيْ تَتَضَرَّرُ الزَّوْجَةُ ( بِتَرْكِهَا ) أَيْ الزِّينَةِ ( كَكُحْلٍ وَدُهْنٍ مُعْتَادَيْنِ ) لَهَا ( وَحِنَّاءٍ ) مُعْتَادَةٍ لَهَا ، بِالْمَدِّ مُنْصَرِفًا ؛ لِأَنَّ أَلِفَهُ أَصْلِيَّةٌ تت لِرَأْسِهَا لَا لِخَضْبِ يَدَيْهَا وَرِجْلَيْهَا وَلَوْ طِيبًا وَلَا جَرَى بِهِ عُرْفٌ .
