( فَصْلٌ ) لَوْ ( اشْتَرَى عَبْدًا فَغَرَبَتْ الشَّمْسُ ) لَيْلَةَ الْفِطْرِ ( وَهُمَا فِي خِيَارِ الْمَجْلِسِ أَوْ الشَّرْطِ فَفِطْرَتُهُ عَلَى مَنْ لَهُ الْمِلْكُ ) بِأَنْ يَكُونَ الْخِيَارُ لِأَحَدِهِمَا ( وَإِنْ لَمْ يَتِمَّ ) لَهُ الْمِلْكُ ( وَإِنْ قُلْنَا بِالْوَقْفِ ) لِلْمِلْكِ بِأَنْ كَانَ الْخِيَارُ لَهُمَا ( فَعَلَى مَنْ يَئُولَ إلَيْهِ الْمِلْكُ ) فِطْرَتُهُ ( وَمَنْ مَاتَ قَبْلَ الْغُرُوبِ ) عَنْ رَقِيقٍ ( فَفِطْرَةُ رَقِيقِهِ عَلَى الْوَرَثَةِ ) كُلٌّ بِقِسْطِهِ لِأَنَّهُ مِلْكُهُمْ وَقْتَ الْوُجُوبِ ( وَلَوْ ) كَانَ عَلَيْهِ دَيْنٌ وَ ( اسْتَغْرَقَ الدَّيْنُ التَّرِكَةَ ) فَإِنَّ عَلَيْهِمْ فِطْرَتَهُ وَإِنْ بِيعَ فِي الدَّيْنِ بِنَاءً عَلَى أَنَّ الدَّيْنَ لَا يَمْنَعُ الْإِرْثَ قَالَ الرَّافِعِيُّ وَلَا يُؤَثِّرُ فِي وُجُوبِهَا كَوْنُ الْمِلْكِ غَيْرَ مُسْتَقِرٍّ لِأَنَّهَا تَجِبُ مَعَ انْتِفَاءِ الْمِلْكِ فَمَعَ ضَعْفِهِ أَوْلَى ( وَإِنْ مَاتَ بَعْدَهُ ) أَيْ بَعْدَ الْغُرُوبِ عَنْ أَرِقَّاءَ ( فَالْفِطْرَةُ عَنْهُ وَعَنْهُمْ فِي التَّرِكَةِ مُقَدَّمَةٌ عَلَى الدَّيْنِ ) فَعَلَى الْمِيرَاثِ وَالْوَصَايَا بِالْأَوْلَى وَذَلِكَ لِمَا مَرَّ فِي فَرْعٍ فِي الشَّرْطِ الْخَامِسِ لِزَكَاةِ الْمَوَاشِي .( 5 / 184 )
