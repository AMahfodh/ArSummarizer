وَقَدْ يُقَالُ : لَمَّا لَمْ يَكُنْ الْمَقْصُودُ مِنْهُ إلَّا ذَلِكَ وَلَمْ يَصْلُحْ لَهُ عُدَّ غَيْرَ مُنْتَفَعٍ بِهِ بِالْمَرَّةِ قَوْلُهُ : أَوْ مِثْلُ ) أَيْ أَوَّلَهُ عَلَى مِثْلِ مَا عَلَيَّ لِزَيْدٍ ( قَوْلُهُ : فَلَا يُقْبَلُ بِأَقَلَّ مِنْ ذَلِكَ عَدَدًا ) أَيْ وَيُقْبَلُ بِغَيْرِ جِنْسِهِ وَنَوْعِهِ ( قَوْلُهُ : فِي النَّوْعَيْنِ ) أَيْ الْمُبْهَمِ وَغَيْرِهِ ( قَوْلُهُ : وَمُرَكَّبَةً ) أَيْ مُكَرَّرَةً مَرَّةً فَأَكْثَرَ ( قَوْلُهُ : وَإِنْ زَادَ ) أَيْ وَإِنْ كَانَ الْمَجْلِسُ مُخْتَلِفًا ( قَوْلُهُ : وَالْأَوْجَهُ أَنَّ مِثْلَ الْوَاوِ هُنَا مَا يَأْتِي ) لَكِنَّ مَحَلَّهُ فِي الْفَاءِ حَيْثُ أَرَادَ الْعَطْفَ وَإِلَّا فَلَا تَعَدُّدَ لِمَجِيئِهَا كَثِيرًا لِلتَّفْرِيعِ وَتَزْيِينِ اللَّفْظِ وَمُقْتَرِنَةٍ بِجَزَاءِ حَذْفٍ إلَى آخِرِ مَا يَأْتِي فِي الْفَصْلِ الْآتِي بَعْدَ قَوْلِ الْمُصَنِّفِ فَإِنْ قَالَ : وَدِرْهَمٌ لَزِمَهُ دِرْهَمَانِ ( قَوْلُهُ : وَيَلْزَمُهُ ) أَيْ السُّبْكِيُّ ( قَوْلُهُ : وَإِنَّمَا الْمُقْتَضِي إلَخْ ) هَذَا عَلَى خِلَافِ مَا صَحَّحَهُ فِي بَلْ بُعْدٌ مِنْ لُزُومِ التَّعَدُّدِ ( قَوْلُهُ : لِمَا يَأْتِي ) أَيْ فِي الْفَصْلِ الْآتِي( 16 / 41 )
