( وَ ) شُرِطَ أَيْضًا ( سَعَةُ ) بِفَتْحِ السِّينِ وَكَسْرِهَا أَيْ كِبَرُ ( الْحَائِطِ ) بِحَيْثُ يَغْلِبُ اسْتِيفَاءُ الْقَدْرِ الْمُشْتَرَى مِنْ ثَمَرِهِ لِكَثْرَةِ شَجَرِهِ ( وَ ) شُرِطَ أَيْضًا بَيَانُ ( كَيْفِيَّةِ قَبْضِهِ ) أَيْ الثَّمَرِ الْمُشْتَرَى أَمُتَوَالِيًا أَمْ مُتَفَرِّقًا ، وَقَدْرُ مَا يُؤْخَذُ مِنْهُ كُلَّ يَوْمٍ ، فَإِنْ سُمِّيَ بَيْعًا فَلَا يُشْتَرَطُ ذَلِكَ ، وَيُحْمَلُ عَلَى الْحُلُولِ لِاقْتِضَاءِ الْبَيْعِ الْمُنَاجَزَةَ وَلَفْظِ السَّلَمِ التَّأْجِيلَ ( وَ ) شُرِطَ أَيْضًا فِيهِمَا إسْلَامُهُ لِمَالِكِهِ ) أَيْ الْحَائِطِ ( وَ ) شُرِطَ فِيهِمَا ( شُرُوعُهُ ) أَيْ الْمُسْلِمِ فِي أَخْذِ الثَّمَرَةِ مِنْ يَوْمِ الْعَقْدِ ، بَلْ ( وَإِنْ ) تَأَخَّرَ الشُّرُوعُ فِيهِ ( لِنِصْفِ شَهْرٍ ) لَا أَكْثَرَ( 11 / 328 )
