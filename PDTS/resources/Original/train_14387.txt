( وَ ) النِّصَابُ ( فِي مَزْرُوعٍ ) ثَمَرٍ وَزَرْعٍ ( تَحْدِيدٌ ) كَالْمَاشِيَةِ فَلَوْ نَقْصَ يَسِيرًا لَمْ تَجِبْ ( وَقِيلَ ) النِّصَابُ فِي ثَمَرٍ وَزَرْعٍ ( تَقْرِيبٌ ) كَالْأَثْمَانِ ( فَلَا يُؤَثِّرُ ) نَقْصٌ ( نَحْوَ رَطْلَيْنِ ) بِنَحْوِ الْبَغْدَادِيِّ ( وَمُدَّيْنِ وَيُؤَثِّرَانِ ) أَيْ : نَقْصُهُمَا ( عَلَى ) الْقَوْلِ ( الْأَوَّلِ ) وَعَلَيْهِ الْمُعَوَّلُ ( وَعَلَيْهِمَا ) أَيْ : الْقَوْلَيْنِ ( لَا اعْتِبَارَ بِنَقْصٍ بِتَدَاخُلٍ فِي الْمَكَايِيلِ كَالْأُوقِيَّةِ ) فَلَا يَمْنَعُ نَقْصُهَا الْوُجُوبَ .( 5 / 23 )
