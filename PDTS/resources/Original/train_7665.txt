( يَجِبُ ) عَلَى مُسْلَمٍ إلَيْهِ ( رَدُّ مَا أَخَذَ ) مِنْ رَأْسِ مَالِهِ إنْ بَقِيَ لِرُجُوعِهِ لِمُشْتَرٍ ( وَإِلَّا ) يَكُنْ بَاقِيًا ( فَ ) عَلَيْهِ ( مِثْلُهُ ) إنْ كَانَ مِثْلِيًّا ( ثُمَّ قِيمَتُهُ ) إنْ كَانَ مُتَقَوِّمًا ، أَوْ تَعَذَّرَ الْمِثْلُ ؛ لِأَنَّ مَا تَعَذَّرَ رَدُّهُ رُجِعَ بِبَدَلِهِ( 5 / 137 )
