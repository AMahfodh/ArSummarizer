( وَ ) يَلْزَمُهُ أَنْ ( يَغْرَمَهُ ) أَيْ بَدَلَهُ ( لِلْمُقَرِّ لَهُ ) لِأَنَّهُ فَوَّتَهُ عَلَيْهِ بِتَصَرُّفِهِ فِيهِ ( وَإِنْ قَالَ لَمْ يَكُنْ ) التَّصَرُّفُ مَا بِعْتُهُ أَوْ وَهَبْتُهُ وَنَحْوِهِ ( مِلْكِي ثُمَّ مَلَكْتُهُ بَعْدَ ) الْبَيْعِ أَوْ الْهِبَةِ وَنَحْوِهِمَا ( قُبِلَ ) مِنْهُ ذَلِكَ ( بِبَيِّنَةٍ ) تَشْهَدُ بِهِ ( مَا لَمْ يُكَذِّبْهَا ) أَيْ الْبَيِّنَةَ ( بِأَنْ كَانَ أَقَرَّ أَنَّهُ ) أَيْ الْمَبِيعَ أَوْ الْمَوْهُوبَ وَنَحْوَهُ ( مِلْكُهُ أَوْ قَالَ قَبَضْت ثَمَنَ مِلْكِي وَنَحْوِهِ ) كَأَنْ قَالَ بِعْتُك أَوْ وَهَبْتُك مِلْكِي هَذَا فَإِنْ وُجِدَ ذَلِكَ لَمْ تُسْمَعْ بَيِّنَتُهُ لِأَنَّهَا تَشْهَدُ بِخِلَافِ مَا أَقَرَّ بِهِ وَعُلِمَ مِنْهُ أَنَّهُ إذَا لَمْ يَكُنْ لَهُ بَيِّنَةٌ لَمْ يُقْبَلْ قَوْلُهُ مُطْلَقًا لِأَنَّ الْأَصْلَ أَنَّهُ إنَّمَا تَصَرَّفَ فِيمَا لَهُ التَّصَرُّفُ فِيهِ قَالَ الشَّيْخُ تَقِيُّ الدِّينِ فِيمَا إذَا ادَّعَى بَائِعٌ بَعْدَ الْبَيْعِ وَقْفًا عَلَيْهِ أَنَّهُ بِمَنْزِلَةِ أَنْ يَدَّعِيَ أَنَّهُ مَلَكَهُ الْآنَ( 13 / 22 )
