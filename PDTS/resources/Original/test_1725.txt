الشَّرْطُ ( الثَّالِثُ كَوْنُ مَسْرُوقٍ مَالًا ) لِأَنَّ غَيْرَ الْمَالِ لَيْسَ لَهُ حُرْمَةُ الْمَالِ وَلَا يُسَاوِيهِ فَلَا يُلْحَقُ بِهِ وَالْأَخْبَارُ مُقَيِّدَةٌ لِلْآيَةِ ( مُحْتَرَمًا ) لِأَنَّ غَيْرَ الْمُحْتَرَمِ كَمَالِ الْحَرْبِيِّ تَجُوزُ سَرِقَتُهُ ( وَلَوْ ) كَانَ الْمَسْرُوقُ ( مِنْ غَلَّةِ وَقْفٍ وَلَيْسَ ) السَّارِقُ ( مِنْ مُسْتَحِقِّيهِ ) أَيْ الْوَقْفِ لِأَنَّهُ مَالٌ مُحْتَرَمٌ لِغَيْرِهِ وَلَا شُبْهَةَ لَهُ فِيهِ أَشْبَهَ غَيْرَ مَالِ الْوَقْفِ وَ ( لَا ) يُقْطَعُ إنْ سَرَقَ ( مِنْ سَارِقٍ أَوْ غَاصِبٍ مَا سَرَقَهُ ) السَّارِقُ ( أَوْ غَصَبَهُ ) الْغَاصِبُ لِأَنَّهُ لَمْ يَسْرِقْهُ مِنْ مَالِكِهِ وَلَا نَائِبِهِ ( وَثَمِينٌ ) مُبْتَدَأٌ ( كَجَوْهَرٍ وَمَا يُسْرِعُ فَسَادُهُ كَفَاكِهَةٍ ) كَغَيْرِهِ لِعُمُومِ الْآيَةِ وَلِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { فِي التَّمْرِ مَنْ سَرَقَ مِنْهُ شَيْئًا فَبَلَغَ ثَمَنَ الْمِجَنِّ فَفِيهِ الْقَطْعُ } رَوَاهُ أَحْمَدُ وَأَبُو دَاوُد وَالنَّسَائِيُّ مِنْ حَدِيثِ عَمْرِو بْنِ شُعَيْبٍ عَنْ أَبِيهِ عَنْ جَدِّهِ .
