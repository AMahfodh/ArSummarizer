( وَمَا لَحِقَ بِهِ ) أَيْ : بِمَا ذُكِرَ مِنْ الْمَحْظُورَاتِ ( حِين فَعَلَهُ ) أَيْ : الْمَحْظُورَ ( وَلَهُ الذَّبْحُ قَبْلَهُ ) إذَا أَرَادَ فِعْلَهُ ( لِعُذْرٍ ) كَكَفَّارَةِ الْيَمِينِ وَنَحْوِهَا وَتَقَدَّمَ أَوَّلَ الْبَابِ ( وَكَذَلِكَ مَا وَجَبَ لِتَرْكِ وَاجِبٍ ) أَيْ : يَكُونُ وَقْتُهُ مِنْ تَرْكِ ذَلِكَ الْوَاجِبِ ( وَلَوْ أَمْسَكَ صَيْدًا أَوْ جَرَحَهُ ثُمَّ أَخْرَجَ جَزَاءَهُ ثُمَّ تَلِفَ الْمَجْرُوحُ أَوْ الْمُمْسَكُ أَوْ قَدَّمَ مَنْ أُبِيحَ لَهُ الْحَلْقُ فَدِيَتُهُ قَبْلَ الْحَلْقِ ثُمَّ حَلَقَ أَجْزَأَهُ ) وَلَا يَخْلُو عَنْ نَوْعِ تَكْرَارِ مَعَ مَا قَبْلَهُ .( 7 / 90 )
