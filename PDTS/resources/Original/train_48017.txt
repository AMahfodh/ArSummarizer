( لَا عِدَّةَ عَلَى مَسْبِيَّةٍ افْتَرَقَتْ بِتَبَايُنِ الدَّارَيْنِ ) لِأَنَّ الْعِدَّةَ حَيْثُ وَجَبَتْ إنَّمَا وَجَبَتْ حَقًّا لِلْعَبْدِ وَالْحَرْبِيُّ مُلْحَقٌ بِالْجَمَادِ وَالْبَهَائِمِ حَتَّى صَارَ مَحَلًّا لِلتَّمَلُّكِ فَلَا حُرْمَةَ لِفِرَاشِهِ ( إلَّا الْحَامِلَ ) لِأَنَّ فِي بَطْنِهَا وَلَدًا ثَابِتَ النَّسَبِ ( وَلَا ) عَلَى ( ذِمِّيَّةٍ طَلَّقَهَا ذِمِّيٌّ إذَا اعْتَقَدُوا عَدَمَهَا ) لِأَنَّ وُجُوبَ الْعِدَّةِ لَا يَجُوزُ أَنْ يَكُونَ لِحَقِّ الشَّرْعِ لِأَنَّهَا غَيْرُ مُخَاطَبَةٍ بِحُقُوقِ الشَّرْعِ وَلَا لِحَقِّ الزَّوْجِ لِأَنَّهُ خِلَافُ مُعْتَقَدِهِ وَقَدْ أُمِرْنَا أَنْ نَتْرُكَهُمْ وَمَا يَدِينُونَ ( وَلَا ) عَلَى ( حَرْبِيَّةٍ خَرَجَتْ إلَيْنَا مُسْلِمَةً ، أَوْ ذِمِّيَّةٍ أَوْ مُسْتَأْمَنَةٍ ، ثُمَّ أَسْلَمَتْ ، أَوْ صَارَتْ ذِمِّيَّةً ) لِقَوْلِهِ تَعَالَى { وَلَا جُنَاحَ عَلَيْكُمْ أَنْ تَنْكِحُوهُنَّ } مُطْلَقًا بِلَا قَيْدٍ وَلِمَا عَرَفْت أَنَّ الْحَرْبِيَّ مُلْحَقٌ بِالْجَمَادِ وَالْبَهَائِمِ فَلَا حُرْمَةَ لِفِرَاشِهِ ( إلَّا الْحَامِلَ ) لِمَا عَرَفْت أَنَّ فِي بَطْنِهَا وَلَدًا ثَابِتَ النَّسَبِ .
