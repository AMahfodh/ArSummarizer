كَمَا لَوْ وَصَّى لِثَلَاثَةِ أَنْفُسٍ ( وَلَا يَسْتَحِقُّ ) زَيْدٌ ( مَعَهُمْ ) أَيْ الْفُقَرَاءِ وَالْمَسَاكِينِ ( بِالْفَقْرِ وَالْمَسْكَنَةِ ) لِاقْتِضَاءِ الْعَطْفِ الْمُغَايَرَةَ ( وَلَوْ وَصَّى بِشَيْءٍ لِزَيْدٍ وَبِشَيْءٍ ) آخَرَ ( لِلْفُقَرَاءِ ) وَزَيْدٌ مِنْهُمْ لَمْ يُشَارِكْهُمْ ( أَوْ ) وَصَّى لِزَيْدٍ بِشَيْءٍ وَبِشَيْءٍ ( لِجِيرَانِهِ وَزَيْدٌ مِنْهُمْ لَمْ يُشَارِكْهُمْ ) لِمَا تَقَدَّمَ .
