( وَإِنْ كَانَتْ الشَّجَرَةُ مَغْرُوسَةً قَبْلَ بِنَائِهِ ) أَيْ : الْمَسْجِدِ ( مَثَلًا وَوَقْفُهَا مَعَهُ فَإِنْ عَيَّنَ ) الْوَاقِفُ ( مَصْرِفَهَا عُمِلَ بِهِ ) كَسَائِرِ الشُّرُوطِ ( وَإِلَّا ) يُعَيِّنْ مَصْرِفَهَا ( فَكَوَقْفٍ مُنْقَطِعٍ ) تُصْرَفُ ثَمَرَتُهَا لِوَرَثَةِ الْوَاقِفِ نَسَبًا وَقْفًا فَإِنْ انْقَرَضُوا فَلِمَسَاكِينَ .( 14 / 497 )
