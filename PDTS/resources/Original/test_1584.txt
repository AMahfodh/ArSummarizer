( وَلَوْ تَوَكَّلَ الْحَالِف فِيمَا حَلَفَ أَنْ لَا يَفْعَلَهُ وَكَانَ ) الْمَحْلُوفُ عَلَيْهِ ( عَقْدًا أَضَافَهُ إلَى الْمُوَكِّلِ ) بِأَنْ قَالَ بِعْتُ عَنْ مُوَكِّلِي أَوْ اشْتَرَيْتُ لَهُ ( وَأَطْلَقَ ) فَلَمْ يُضِفْهُ إلَى الْمُوَكِّلِ ( لَمْ يَحْنَثْ ) الْحَالِفُ لِأَنَّ حُقُوقَ الْعَقْدِ مُتَعَلِّقَةٌ بِالْمُوَكِّلِ كَمَا تَقَدَّمَ لَكِنْ تَقَدَّمَ فِي النِّكَاحِ لَا يَصِحُّ إذَا لَمْ يُضِفْهُ لِمُوَكِّلِهِ .( 21 / 422 )
