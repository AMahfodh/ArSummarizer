( وَ ) الذَّكَاةُ أَيْضًا خَالِصُ ( جَرْحِ مَا لَمْ يُقْدَرْ ) عَلَيْهِ عِنْدَ الرَّمْيِ أَوْ نَحْوِهِ فِي أَيِّ مَحَلٍّ كَانَ مِنْ جَائِزِ الْمُنَاكَحَةِ أَوْ أَمَةٍ كِتَابِيَّةٍ ( كَإِبِلٍ يَشْرُدُ ) بِضَمِّ الرَّاءِ ( أَوْ ) يَتَرَدَّى ( فِي حُفْرَهْ ) وَعَجَزَ عَنْ ذَبْحِهِ { لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ فِي بَعِيرٍ نَدَّ فَضَرَبَهُ رَجُلٌ بِسَهْمٍ فَحَبَسَهُ اللَّهُ أَنَّ لِهَذِهِ الْبَهَائِمِ أَوَابِدَ كَأَوَابِدِ الْوَحْشِ فَمَا غَلَبَكُمْ مِنْهَا فَاصْنَعُوا بِهِ هَكَذَا } رَوَاهُ الشَّيْخَانِ .
