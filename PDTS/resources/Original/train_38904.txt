( وَقُدِّمَ ) فِي الْإِرْثِ بِهِ ( عَاصِبُ النَّسَبِ ) عَلَى عَاصِبِ الْوَلَاءِ وَهُوَ الْمُعْتِقُ بِالْكَسْرِ وَعَصَبَتُهُ ( ثُمَّ ) إذَا لَمْ يَكُنْ عَاصِبُ نَسَبٍ قُدِّمَ ( الْمُعْتِقُ ) لَهُ مُبَاشَرَةً عَلَى عَصَبَتِهِ ( ثُمَّ ) إذَا لَمْ يُوجَدْ الْمُعْتِقُ مُبَاشَرَةً وَرِثَهُ ( عَصَبَتُهُ ) أَيْ عَصَبَةُ الْمُعْتِقِ بِالْكَسْرِ .
