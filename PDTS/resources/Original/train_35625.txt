( وَإِلَّا ) يَكُنْ الْعِتْقُ مُرَتَّبًا بِأَنْ أَعْتَقَاهُ مَعًا أَوْ مُرَتَّبًا وَجَهِلَ الْأَوَّلُ قُوِّمَ نَصِيبُ الثَّالِثِ عَلَيْهِمَا وَإِذَا قُوِّمَ عَلَيْهِمَا ( فَعَلَى ) قَدْرِ ( حِصَصِهِمَا إنْ أَيْسَرَا ) مَعًا ( وَإِلَّا فَعَلَى الْمُوسِرِ ) مِنْهُمَا يُقَوَّمُ الْجَمِيعُ ( وَ ) لَوْ أَعْتَقَ فِي حَالَ مَرَضِهِ شِقْصًا لَهُ فِي عَبْدٍ أَوْ أَعْتَقَ بَعْضَ عَبْدٍ يَمْلِكُ جَمِيعَهُ ( عَجَّلَ ) عِتْقَ( 19 / 115 )
