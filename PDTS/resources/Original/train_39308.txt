قَالَهُ بْن ( قَوْلُهُ : فِي الْأُولَى ) أَيْ مَا إذَا نَزَعَ الْخُفَّيْنِ بَعْدَ الْمَسْحِ عَلَيْهِمَا ( قَوْلُهُ : وَكَذَا الثَّالِثَةُ ) أَيْ وَهِيَ مَا إذَا نَزَعَ أَحَدَ الْخُفَّيْنِ الْمُنْفَرِدَيْنِ بَعْدَ مَسْحِهِمَا ( قَوْلُهُ : بَلْ يَنْزِعُ إلَخْ ) الْأَوْلَى التَّفْرِيعُ بِالْفَاءِ عَلَى قَوْلِهِ وَكَذَا الثَّالِثَةُ ( قَوْلُهُ : لِئَلَّا يَجْمَعَ إلَخْ ) عِلَّةٌ لِمَحْذُوفٍ أَيْ وَلَا يَغْسِلُ الرِّجْلَ الَّتِي نَزَعَ الْخُفَّ مِنْهَا وَمَسَحَ الْأُخْرَى لِئَلَّا إلَخْ ( قَوْلُهُ : وَمَسَحَ الْأَسْفَلَيْنِ ) عَطْفٌ عَلَى قَوْلِهِ غَسَلَ الرِّجْلَيْنِ فِي الْأُولَى وَقَوْلُهُ فِي الثَّانِيَةِ أَيْ وَهِيَ مَا إذَا نَزَعَ الْأَعْلَيَيْنِ بَعْدَ مَسْحِهِمَا ( قَوْلُهُ : فِي الرَّابِعَةِ ) أَيْ وَهِيَ مَا إذَا نَزَعَ أَحَدَ الْأَعْلَيَيْنِ بَعْدَ مَسْحِهَا ( قَوْلُهُ : فَيَبْنِي بِنِيَّةٍ ) أَيْ فَإِذَا لَمْ يُبَادِرْ لِلْأَسْفَلِ بَنَى بِنِيَّةٍ إنْ نَسِيَ مُطْلَقًا أَيْ طَالَ أَوْ لَمْ يُطِلْ أَيْ أَنَّهُ يَبْنِي عَلَى مَا قَبْلَ الرِّجْلَيْنِ وَيَغْسِلُهُمَا بِنِيَّةٍ مُطْلَقًا ( قَوْلُهُ : وَإِنْ عَجَزَ ) أَيْ وَيَبْنِي عَلَى مَا قَبْلَ الرِّجْلَيْنِ إنْ عَجَزَ مَا لَمْ يُطِلْ وَكَذَا إنْ كَانَ عَامِدًا عَلَى مَا مَرَّ( 2 / 26 )
