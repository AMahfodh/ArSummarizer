( وَإِنْ اسْتَعَانُوا ) أَيْ الْبُغَاةُ ( بِأَهْلِ الْحَرْبِ وَأَمَّنُوهُمْ لَمْ يَصِحَّ أَمَانُهُمْ ) كَمَا لَوْ عَقَدُوا لَهُمْ ذِمَّةً لِأَنَّ الْأَمَانَ مِنْ شَرْطِ صِحَّتِهِ إلْزَامُ كَفِّهِمْ عَنْ الْمُسْلِمِينَ ( وَأُبِيحَ ) لِأَهْلِ الْعَدْلِ ( قَتْلُهُمْ ) مُقْبِلِينَ وَمُدْبِرِينَ وَأَخْذُ أَمْوَالِهِمْ ( وَحُكْمُ أَسِيرِهِمْ حُكْمُ أَسِيرِ سَائِرِ أَهْلِ الْحَرْبِ ) يُخَيَّرُ فِيهِ الْإِمَامُ بَيْنَ الْقَتْلِ وَالرِّقِّ وَالْمَنِّ وَالْفِدَاءِ إلَّا أَنَّهُمْ فِي أَمَانٍ بِالنِّسْبَةِ إلَى مَنْ أَمَّنَهُمْ مِنْ الْبُغَاةِ .( 21 / 91 )
