( فَصْلٌ : وَتَصِحُّ ) الْوَكَالَةُ ( فِي بَيْعِ مَالِهِ ) ؛ أَيْ : الْمُوَكِّلِ ( كُلِّهِ ) ؛ لِأَنَّهُ يَعْرِفُ مَالَهُ فَلَا غَرَرَ ( أَوْ ) بَيْعِ ( مَا شَاءَ ) الْوَكِيلُ ( مِنْهُ ) ؛ أَيْ : مِنْ مَالِ الْمُوَكِّلِ ؛ لِأَنَّهُ إذَا جَازَ التَّوْكِيلُ فِي الْجَمِيعِ فَفِي بَعْضِهِ أَوْلَى .( 9 / 366 )
