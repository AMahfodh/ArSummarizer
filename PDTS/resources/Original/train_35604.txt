( قَوْلُهُ : أَوْ لَمْ يَقْصِدْ شَيْئًا ) يَنْبَغِي أَوْ قَصَدَ وَاحِدًا لَا بِعَيْنِهِ بِأَنْ قَصَدَ أَحَدَ الْأَمْرَيْنِ مِنْ التَّفْهِيمِ وَالْقِرَاءَةِ ( قَوْلُهُ : إلَّا بِالْقَصْدِ ) أَيْ مَعَ وُجُودِ الصَّارِفِ كَمَا هُنَا ( قَوْلُهُ : فَأَثَّرَتْ ) أَيْ الْقَرِينَةُ ( قَوْلُهُ : نَفْيَ كُلٍّ مِنْ الْمُقْسِمِ ) ، وَهُوَ قَوْلُهُ : بِقَصْدِ التَّفْهِيمِ وَقَوْلُهُ وَقَيْدِ الْمُقْسِمِ ، وَهُوَ قَوْلُهُ : إنْ قَصَدَ مَعَهُ قِرَاءَةً ( قَوْلُهُ : وَإِنْ بَحَثَ فِي الْمَجْمُوعِ إلَخْ ) ضَعِيفٌ ( قَوْلُهُ : وَسَوَاءٌ ) أَيْ فِي التَّفْصِيلِ الْمَارِّ ( قَوْلُهُ : خِلَافًا لِجَمْعٍ مُتَقَدِّمِينَ ) أَيْ فَإِنَّهُمْ يَخُصُّونَ التَّفْصِيلَ بِمَا يَصْلُحُ لِلْمُخَاطَبَةِ كَمَا ذَكَرَهُ سم عَلَى الْعُبَابِ .
