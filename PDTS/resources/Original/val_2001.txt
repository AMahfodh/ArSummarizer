( فَيَصِحُّ ) لِلْوَكِيلِ إذَا ( تَوَلَّى طَرَفَيْ الْعَقْدِ فِيهِمَا كَأَبِ الصَّغِيرِ ) وَنَحْوِهِ ، إذَا بَاعَ مِنْ مَالِهِ لِوَلَدِهِ ، أَوْ اشْتَرَى مِنْهُ لَهُ ( وَ ) كَ ( تَوْكِيلِهِ ) أَيْ جَائِزِ التَّصَرُّفِ ( فِي بَيْعِهِ ، وَ ) تَوْكِيلِ ( آخَرَ ) لِذَلِكَ الْوَكِيلِ ( فِي شِرَائِهِ ) فَيَتَوَلَّى طَرَفَيْ عَقْدِهِ ( وَمِثْلُهُ ) أَيْ عَقْدِ الْبَيْعِ ( نِكَاحٌ ) بِأَنْ يُوَكِّلَ الْوَلِيُّ الزَّوْجَ ، أَوْ عَكْسَهُ أَوْ يُوَكِّلَا وَاحِدًا ، أَوْ يُزَوِّجَ عَبْدَهُ الصَّغِيرَ بِأَمَتِهِ وَنَحْوَهُ ، فَيَتَوَلَّى طَرَفَيْ الْعَقْدِ .
