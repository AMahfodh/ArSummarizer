( وَ ) يَتَّجِهُ ( أَنَّهُ ) - أَيْ : الْأَجِيرُ ( يُحَاصِصُ الْغُرَمَاءَ ) بِحُدُوثِ زِيَادَةٍ فِي ثَمَنِ الْمَعْمُولِ بِغَيْرِ عَمَلِهِ ؛ لِأَنَّهُمْ شُرَكَاؤُهُ ( وَلَا يَخْتَصُّ ) الْأَجِيرُ ( بِهِ ) ؛ أَيْ : الزَّائِدُ عَنْ الثَّمَنِ الَّذِي بِيعَ بِهِ ، ( إلَّا إنْ زَادَ ) ثَمَنُهُ ( بِعَمَلِهِ ) فَيَخْتَصُّ بِالزَّائِدِ حَيْثُ كَانَ أَنْقَصَ مِنْ أُجْرَتِهِ ؛ ( كَمُضَارِبٍ ) يَأْخُذُ مَا يَخُصُّهُ مِنْ رِبْحٍ ظَهَرَ قَبْلَ أَنْ يُفْلِسَ رَبُّ الْمَالِ ، وَلَا يُحَاصِصُ الْغُرَمَاءُ فِي نَصِيبِهِ .
