( وَلَوْ كَانَ عَبْدٌ مُشْتَرَكٌ بَيْنَ مُوسِرِينَ فَقَالَ أَحَدُهُمَا عَنْ ) طَائِرٍ ( إنْ كَانَ غُرَابًا فَنَصِيبِي ) مِنْ الْعَبْدِ ( حُرٌّ وَقَالَ ) الشَّرِيكُ ( الْآخَرُ إنْ لَمْ يَكُنْ غُرَابًا فَنَصِيبِي حُرٌّ عَتَقَ ) الْعَبْدُ ( عَلَى أَحَدِهِمَا ) لِأَنَّ أَحَدَهُمَا حَانِثٌ قَطْعًا ( فَيُمَيَّزُ بِالْقُرْعَةِ ) وَيُغَرَّمُ قِيمَةُ نَصِيبِ شَرِيكِهِ ( وَالْوَلَاءُ لَهُ ) لِأَنَّهُ مُعْتِقٌ .( 18 / 401 )
