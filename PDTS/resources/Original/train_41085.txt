( وَ ) يَكُونُ ( إمَامُهُمْ فِي وَسَطِهِمْ ، أَيْ : بَيْنَهُمْ ) وَإِنْ لَمْ يَتَسَاوَوْا مِنْ عَنْ يَمِينِهِ وَشِمَالِهِ ( وُجُوبًا ) لِأَنَّهُ أَسْتَرُ مِنْ أَنْ يَتَقَدَّمَ عَلَيْهِمْ ( فَإِنْ تَقَدَّمَهُمْ ) الْإِمَامُ ( بَطَلَتْ ) قَالَ فِي الْمُبْدِعِ : فِي الْأَصَحِّ ( إلَّا فِي ظُلْمَةٍ ) فَيَجُوزُ أَنْ يَتَقَدَّمَ عَلَيْهِمْ لِلْأَمْنِ مِنْ رُؤْيَتِهِمْ عَوْرَتَهُ وَكَذَا لَوْ كَانُوا عُمْيَانًا ( وَلَا إعَادَةَ عَلَيْهِمْ ) .( 2 / 281 )
