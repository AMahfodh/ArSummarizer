ع ش ( قَوْلُهُ : أَنَّ الْمَنْقُولَ فِيهِ ) أَيْ فِي الْفَرْعِ الْأَخِيرِ ( مَا ذُكِرَ ) أَيْ الصِّحَّةُ ( قَوْلُهُ : بَعْدَهُ ) أَيْ بَعْدَ بُدُوِّ الصَّلَاحِ ( قَوْلُهُ : وَلَمْ يَرَهَا ) أَيْ وَالْحَالُ أَنَّهُ لَمْ يَرَ الثَّمَرَةَ بَعْدَ بُدُوِّ الصَّلَاحِ ( قَوْلُهُ : لَمْ يَصِحَّ ) مُعْتَمَدٌ .
