( وَوَقَعَ ) النَّسْخُ بِأَثْقَلَ ( بِتَعْيِينِ الصَّوْمِ ) أَيْ صَوْمِ رَمَضَانَ لِلْمُكَلَّفِ الْقَادِرِ عَلَيْهِ غَيْرَ مُسَافِرٍ ( بَعْدَ التَّخْيِيرِ ) لِلْمُكَلَّفِ الْقَادِرِ عَلَيْهِ مُطْلَقًا ( بَيْنَهُ ) أَيْ الصَّوْمِ ( وَبَيْنَ الْفِدْيَةِ ) عَنْ كُلِّ صَوْمِ يَوْمٍ بِإِطْعَامِ مِسْكِينٍ نِصْفَ صَاعِ بُرٍّ أَوْ صَاعِ تَمْرٍ أَوْ شَعِيرٍ عِنْدَ أَصْحَابِنَا وَمُدَّ طَعَامٍ بُرًّا كَانَ أَوْ غَيْرَهُ مِنْ أَقْوَاتِ الْبَلَدِ عِنْدَ الشَّافِعِيَّةِ وَمَدَّ بُرٍّ أَوْ مُدَّيْ تَمْرٍ أَوْ شَعِيرٍ عِنْدَ أَحْمَدَ فَإِنَّ( 4 / 485 )
