( وَيُعَزِّرُ الْمَوْلَى عَبْدَهُ وَالزَّوْجُ زَوْجَتَهُ ) وَلَوْ صَغِيرَةً لِمَا سَيَجِيءُ ( عَلَى تَرْكِهَا الزِّينَةَ ) الشَّرْعِيَّةَ مَعَ قُدْرَتِهَا عَلَيْهَا ( وَ ) تَرْكِهَا ( غُسْلَ الْجَنَابَةِ ، وَ ) عَلَى ( الْخُرُوجِ مِنْ الْمَنْزِلِ ) لَوْ بِغَيْرِ حَقٍّ ( وَتَرْكِ الْإِجَابَةِ إلَى الْفِرَاشِ ) لَوْ طَاهِرَةً مِنْ نَحْوِ حَيْضٍ .
