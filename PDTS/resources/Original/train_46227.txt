( وَإِنْ عَلِمَ ) الْمُشْتَرِي بِذَلِكَ ( بَعْدَ الْبَيْعِ فَلَهُ الرَّدُّ ) وَأَخْذُ الثَّمَنِ كَامِلًا ( أَوْ ) الْإِمْسَاكُ مَعَ ( الْأَرْشِ ) لِأَنَّهُ عَيْبٌ فَمَلَكَ بِهِ الْخِيَارَ كَبَقِيَّةِ الْعُيُوبِ ( وَإِنْ لَمْ يَعْلَمْ ) الْمُشْتَرِي بِالْعُقُوبَةِ ( حَتَّى قُتِلَ ) الْمَبِيعُ ( تَعَيَّنَ لَهُ ) أَيْ الْمُشْتَرِي ( الْأَرْشُ عَلَى الْبَائِعِ ) لِتَعَذُّرِ الرَّدِّ ، وَالْأَرْشُ قِسْطُ مَا بَيْنَ قِيمَتِهِ مَعَ كَوْنِهِ جَانِيًا وَغَيْرَ جَانٍ فَلَوْ قَوَّمَ غَيْرَ جَانٍ بِمِائَةٍ وَجَانِيًا بِخَمْسِينَ فَمَا بَيْنَهُمَا النِّصْف فَالْأَرْشُ إذَنْ نِصْفُ الثَّمَنِ .( 9 / 142 )
