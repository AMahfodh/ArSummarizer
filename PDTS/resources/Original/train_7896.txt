( وَإِنْ قَالَ لَهُ عَلَيَّ بَعْضُ الْعَشَرَةِ قُبِلَ تَفْسِيرُهُ بِمَا شَاءَ مِنْهَا ) أَيْ مِنْ الْعَشَرَة لِأَنَّ الْبَعْضَ يَصْدُقُ بِكُلِّ جُزْءٍ مِنْهَا ( وَإِنْ قَالَ لَهُ ) عَلَيَّ ( شَطْرُهَا ) أَيْ الْعَشَرَةِ ( فَهُوَ نِصْفُهَا ) فَيَلْزَمُهُ خَمْسَةٌ لِأَنَّهَا نِصْفُ الْعَشَرَةِ .( 23 / 370 )
