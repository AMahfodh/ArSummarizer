( وَقِسْ عَلَيْهَا ) أَيْ الْمَسَائِلُ الْمَذْكُورَةُ ( مَا لَوْ ) مَاتَ الْأَبُ حُرًّا وَ ( كَانَ أَحَدُهُمَا رَقِيقًا وَالْآخَرُ حُرًّا ) بِاتِّفَاقِهِمَا عَلَى حُرِّيَّتِهِ وَاخْتَلَفَا هَلْ عَتَقَ الْأَوَّلُ قَبْلَ مَوْتِ الْأَبِ أَوْ بَعْدَهُ ( وَإِنْ قَالَ كُلٌّ مِنْ أَبَوَيْنِ كَافِرَيْنِ وَابْنَيْنِ مُسْلِمَيْنِ مَاتَ ) مُوَرِّثُنَا ( عَلَى دِينِنَا صُدِّقَ الْأَبَوَانِ ) لِأَنَّ وَلَدَهُمَا مَحْكُومٌ بِكُفْرِهِ ابْتِدَاءً( 23 / 266 )
