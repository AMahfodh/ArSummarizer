مَسْأَلَةِ الْحُضُورِ خِلَافًا لِمَا فَهِمَهُ الْإِسْنَوِيُّ وَكَصُوَرِ الْحَيَوَانِ فِي ذَلِكَ فُرُشُ الْحَرِيرِ كَمَا يُومِئُ إلَيْهِ كَلَامُ الْأَصْلِ ( وَلَا بَأْسَ بِصُوَرٍ مَبْسُوطَةٍ ) كَأَنْ كَانَتْ عَلَى بُسُطٍ ( تُدَاسُ أَوْ ) مَخَادِرَ ( يُتَّكَأُ عَلَيْهَا أَوْ ) بِصُوَرٍ ( مُمْتَهَنَةٍ بِالِاسْتِعْمَالِ ) لِمَحَلِّهَا ( كَطَبَقٍ وَقَصْعَةٍ وَكَذَا إنْ ) كَانَتْ مُرْتَفِعَةً لَكِنْ ( قُطِعَ رَأْسُهَا ) لِأَنَّ مَا يُدَاسُ وَيُطْرَحُ مُهَانٌ مُبْتَذَلٌ وَمَقْطُوعُ الرَّأْسِ لَا يُشْبِهُ حَيَوَانًا فِيهِ رُوحٌ بِخِلَافِ الْمَنْصُوبِ فَإِنَّهُ مُرْتَفِعٌ يُشْبِهُ الْأَصْنَامَ
