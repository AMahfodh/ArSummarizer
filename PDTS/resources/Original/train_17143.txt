الْمُحَرَّمَاتُ فِي النِّكَاحِ ضَرْبَانِ : أَيْ صِنْفَانِ ( ضَرْبٌ ) يَحْرُمُ ( عَلَى الْأَبَدِ ) أَيْ الْمُحَرَّمَاتُ عَلَى الْأَبَدِ ( هُنَّ أَقْسَامٌ ) خَمْسَةٌ ( قِسْمٌ ) يَحْرُمْنَ ( بِالنَّسَبِ وَهُنَّ سَبْعٌ الْأُمُّ وَالْجَدَّةُ لِأَبٍ ) وَإِنْ عَلَتْ ( أَوْ ) الْجَدَّةُ ( لِأُمٍّ وَإِنْ عَلَتْ ) لِقَوْلِهِ تَعَالَى : { حُرِّمَتْ عَلَيْكُمْ أُمَّهَاتُكُمْ } وَأُمَّهَاتُك كُلُّ مَنْ انْتَسَبْتَ إلَيْهَا بِوِلَادَةٍ سَوَاءٌ وَقَعَ عَلَيْهَا اسْمُ الْأُمِّ حَقِيقَةً وَهِيَ الَّتِي وَلَدَتْكَ ، أَوْ مَجَازًا وَهِيَ وَاَلَّتِي وَلَدَتْ مَنْ وَلَدَتْكَ وَإِنْ عَلَتْ ؛ وَمِنْهُ جَدَّاتُك أُمُّ أَبِيكَ ، وَأُمُّ أُمِّكَ ، وَجَدَّةُ أُمِّكِ وَجَدَّاتُ أَجْدَادِكِ ، وَجَدَّاتُ جَدَّاتِكِ .
