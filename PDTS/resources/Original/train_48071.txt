( أَوْ كَبِيرَةً لَا تَحْمِلَانِ ) أَيْ الصَّغِيرَةُ الْمُطِيقَةُ وَالْكَبِيرَةُ ( عَادَةً ) كَبِنْتِ سِتِّينَ سَنَةً ( أَوْ ) كَانَتْ ( وَخْشًا ) بِفَتْحِ الْوَاوِ وَسُكُونِ الْخَاءِ الْمُعْجَمَةِ أَيْ غَيْرَ جَمِيلَةٍ شَأْنُهَا تُقْتَنَى لِلْخِدْمَةِ لَا لِلْوَطْءِ ( أَوْ ) كَانَتْ ( بِكْرًا ) بِكَسْرِ الْمُوَحَّدَةِ أَيْ عَذْرَاءَ لِإِمْكَانِ وَطْئِهَا دُونَ الْبَكَارَةِ وَحَمْلِهَا مَعَ بَقَائِهَا ( أَوْ رَجَعَتْ ) الْأَمَةُ لِمَالِكِهَا ( مِنْ غَصْبٍ أَوْ سَبْيٍ ) مِنْ بَالِغٍ غَابَ عَلَيْهَا غَيْبَةً يُمْكِنُهُ وَطْؤُهَا فِيهَا فَلَا يَجِبُ اسْتِبْرَاؤُهَا .
