وَأَشَارَ لِلْمَرْتَبَةِ الثَّانِيَةِ بِقَوْلِهِ : ( وَلِمَا لَيْسَ بِمَالٍ وَلَا آيِلٍ لَهُ ) : أَيْ لِلْمَالِ ( كَعِتْقٍ ) وَطَلَاقٍ وَنَسَبٍ ( وَوَلَاءِ رَجْعَةٍ ) ادَّعَتْهَا هِيَ أَوْ وَلِيُّهَا عَلَى زَوْجِهَا الْمُنْكِرِ لَهَا وَنِكَاحٍ ( وَرِدَّةٍ وَإِحْصَانٍ وَكِتَابَةٍ ) وَتَدْبِيرٍ ( وَتَوْكِيلٍ بِغَيْرِ مَالٍ ) : أَيْ شَيْءٍ غَيْرِ مَالٍ كَتَوْكِيلٍ عَلَى نِكَاحٍ أَوْ طَلَاقٍ وَكَشُرْبِ خَمْرٍ وَقَذْفٍ وَقَتْلٍ : ( عَدْلَانِ ) .
