( وَمِنْ أَخْرَجَ بَعْضَ ثَوْبٍ قِيمَتُهُ ) ؛ أَيْ بَعْضِ الثَّوْبِ ( نِصَابٌ ؛ قُطِعَ بِهِ ) أَيْ بِالْبَعْضِ الَّذِي أَخْرَجَهُ ( إنْ قَطَعَهُ ) مِنْ الثَّوْبِ ؛ لِتَحَقُّقِ إخْرَاجِهِ مِنْ الْحِرْزِ ( وَإِلَّا ) يُقْطَعُ مَا أَخْرَجَهُ ( فَلَا قَطَعَ ) عَلَيْهِ ، كَمَا لَوْ أَخْرَجَ بَعْضَ خَشَبَةٍ وَبَاقِيهَا دَاخِلُ الْحِرْزِ ، وَلَمْ يَقْطَعْهَا لِلتَّبَعِيَّةِ .( 18 / 262 )
