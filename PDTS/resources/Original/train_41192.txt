( قَوْلُهُ أَنَّ لَهُ إمْسَاكَهَا إلَخْ ) أَيْ : وَمُقْتَضَى الْمَتْنِ أَنَّهُ لَيْسَ لَهُ ذَلِكَ إلَّا بِقَضَاءِ جَمِيعِ الدَّيْنِ وَالْمَوْرِدِ شَيْخُ الْإِسْلَامِ ( قَوْلُهُ عَلَيْهِ ) أَيْ : عَلَى الْمَتْنِ ( قَوْلُهُ لَهُ ذَلِكَ ) أَيْ كَانَ لَهُ إلَخْ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ نَعَمْ إلَخْ ) اسْتِدْرَاكٌ عَلَى الْمَتْنِ ( قَوْلُهُ لَوْ أَوْصَى ) إلَى قَوْلِهِ وَكَذَا فِي النِّهَايَةِ وَالْمُغْنِي إلَّا قَوْلُهُ أَوْ أَوْصَى بِبَيْعِ عَيْنٍ مِنْ مَالِهِ لِفُلَانٍ ( قَوْلُهُ إلَيْهِ ) أَيْ الدَّائِنِ ع ش ( قَوْلُهُ عِوَضًا عَنْ دَيْنِهِ ) ثُمَّ إنْ كَانَتْ( 20 / 79 )
