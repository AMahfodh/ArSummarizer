( وَلَا تَسْقُطُ ) الشُّفْعَةُ بِرُجُوعِ الشِّقْصِ إلَى الشَّرِيكِ بِسَبَبِ ( فَسْخِ ) الْبَيْعِ ( لِتَحَالُفٍ ) عَلَى قَدْرِ الثَّمَنِ بِسَبَبِ اخْتِلَافِهِمَا فِيهِ ؛ لِسَبْقِ اسْتِحْقَاقِ الشُّفْعَةِ الْفَسْخَ ، ( وَيُؤْخَذُ ) الشِّقْصُ ( بِهِ ) - أَيْ : بِثَمَنٍ - ( حَلَفَ عَلَيْهِ بَائِعٌ ) ؛ لِأَنَّ الْبَائِعَ مُقِرٌّ بِالْبَيْعِ بِالثَّمَنِ الَّذِي حَلَفَ عَلَيْهِ وَمُقِرٌّ لِلشَّفِيعِ بِاسْتِحْقَاقِ الشُّفْعَةِ بِذَلِكَ ، فَإِذَا بَطَلَ حَقُّ الْمُشْتَرِي بِإِنْكَارِهِ لَمْ يَبْطُلْ حَقُّ الشَّفِيعِ بِذَلِكَ ، فَلَهُ أَنْ يَطْلُبَ فَسْخَهُمَا وَيَأْخُذَ ؛ لِأَنَّ حَقَّهُ أَسْبَقُ .
