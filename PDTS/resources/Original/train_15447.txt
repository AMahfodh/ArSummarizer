( فَإِنْ ) صَلَّى بِاجْتِهَادٍ ثُمَّ ( تَيَقَّنَ ) أَنَّ ( صَلَاتَهُ ) وَقَعَتْ ( قَبْلَ الْوَقْتِ ) أَوْ بَعْضُهَا وَلَوْ تَكْبِيرَةَ التَّحَرُّمِ أَوْ أَخْبَرَهُ بِهِ مَقْبُولُ الرِّوَايَةِ عَنْ مُشَاهَدَةٍ وَعِلْمٍ بِذَلِكَ فِي وَقْتِهَا أَوْ قَبْلَ دُخُولِهِ أَعَادَهَا قَطْعًا أَوْ عَلِمَ بِهِ بَعْدَهُ ( قَضَى ) الصَّلَاةَ الْمَذْكُورَةَ ( فِي الْأَظْهَرِ ) لِفَوَاتِ شَرْطِهَا وَهُوَ الْوَقْتُ ، وَمُقَابِلُ الْأَظْهَرِ قَضَاءٌ اعْتِبَارًا بِمَا فِي ظَنِّهِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَتَيَقَّنْ وُقُوعَهَا قَبْلَ الْوَقْتِ بِأَنْ لَمْ يَبِنْ الْحَالُ أَوْ بَانَ وُقُوعُهَا فِيهِ أَوْ بَعْدَهُ ( فَلَا ) قَضَاءَ عَلَيْهِ وَالْوَاقِعَةُ بَعْدَهُ قَضَاءٌ لَكِنْ لَا إثْمَ فِيهَا ( وَيُبَادِرُ بِالْفَائِتِ ) اسْتِحْبَابًا مُسَارَعَةً لِبَرَاءَةِ ذِمَّتِهِ إنْ فَاتَ بِعُذْرٍ كَنَوْمٍ وَنِسْيَانٍ ، وَوُجُوبًا إنْ فَاتَ بِغَيْرِ عُذْرٍ تَعْجِيلًا لِبَرَاءَةِ الذِّمَّةِ لِخَبَرِ { مَنْ نَامَ عَنْ صَلَاةٍ أَوْ نَسِيَهَا فَلْيُصَلِّهَا إذَا ذَكَرَهَا } .
