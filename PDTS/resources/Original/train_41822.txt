( وَمَا ) ( وَجَدَهُ ابْنٌ بَعْدَ مَوْتِ أَبٍ ) فِي تِرْكَتِهِ ( مِنْ عَيْنِ مَالِهِ ) - أَيْ : الِابْنِ - ( الَّذِي أَقْرَضَهُ ) لِأَبِيهِ ( أَوْ بَاعَهُ ) لَهُ ( أَوْ غَصَبَهُ ) الْأَبُ مِنْهُ ( فَلَهُ ) - أَيْ : الِابْنِ - ( أَخْذُهُ ) - أَيْ : مَا وَجَدَ مِنْ عَيْنِ مَالِهِ - دُونَ بَقِيَّةِ وَرَثَةِ الْأَبِ ( إنْ لَمْ يَكُنْ الْأَبُ دَفَعَ ) لِوَلَدِهِ ( ثَمَنَهُ ) لِتَعَذُّرِ الْعِوَضِ .
