( وَيَرْجِعُ ) زَوْجٌ ( بِهِ ) أَيْ : بِنَظِيرِ مُسَمًّى غَرِمَهُ لَا إنْ أُبْرِئَ مِنْهُ ( عَلَى مُغْرٍ لَهُ مِنْ زَوْجَةٍ عَاقِلَةٍ ) وَشَرَطَ أَبُو عَبْدِ اللَّهِ مُحَمَّدٌ فَخْرُ الدِّينِ بْنُ تَيْمِيَّةَ الْحَرَّانِيِّ مَعَ ذَلِكَ كَوْنَ الزَّوْجَةِ بَالِغَةً وَقْتَ عَقْدٍ ( لِيُوجَدَ ) مِنْهَا ( تَغْرِيرٌ مُحَرَّمٌ ) إذْ الصَّغِيرَةُ لَا تُنْسَبُ أَفْعَالُهَا إلَى التَّحْرِيمِ ( وَوَلِيٍّ وَوَكِيلٍ )( 15 / 175 )
