ط ( قَوْلُهُ : سَكَنَ أَحَدُهُمَا إلَخْ ) تَقَدَّمَتْ مَسَائِلُ الِانْتِفَاعِ بِالْمُشْتَرَكِ فِي غَيْبَةِ شَرِيكِهِ أَوَّلَ الْبَابِ عِنْدَ قَوْلِهِ إلَّا فِي الْخَلْطِ وَالِاخْتِلَاطِ ، وَقَدَّمْنَا الْكَلَامَ عَلَيْهَا ( قَوْلُهُ : طَاحُونٌ مُشْتَرَكَةٌ ) الْمُرَادُ بِهَا كُلُّ مَا لَا يُقْسَمُ ط ( قَوْلُهُ : عَمِّرْهَا ) بِصِيغَةِ الْأَمْرِ : أَيْ قَالَ لِلْآخَرِ عَمِّرْهَا مَعِي فَافْهَمْ ( قَوْلُهُ : لَمْ يَرْجِعْ ) ؛ لِأَنَّهُ شَرِيكُهُ يُجْبَرُ عَلَى أَنْ يَفْعَلَ مَعَهُ كَمَا يُعْلَمُ مِنْ الضَّابِطِ الْآتِي ( قَوْلُهُ : فَلَيْسَ بِمُتَطَوِّعٍ ) مُخَالِفٌ لِمَا قَبْلَهُ وَلِلضَّابِطِ ( قَوْلُهُ : فَهُوَ مُتَطَوِّعٌ ) ؛ لِأَنَّهُ يُجْبَرُ عَلَى الْإِنْفَاقِ وَعَلَى أَدَاءِ الْخَرَاجِ ط .
