( وَمِنْ الْفَرَائِضِ ) فَرْضِ عَيْنٍ ( الْأَمْرُ بِالْمَعْرُوفِ ) وَهُوَ مَا أَمَرَ اللَّهُ وَرَسُولُهُ بِهِ ( وَالنَّهْيُ عَنْ الْمُنْكَرِ ) وَهُوَ مَا نَهَى اللَّهُ وَرَسُولُهُ عَنْهُ ( عَلَى كُلِّ مَنْ بُسِطَتْ يَدُهُ ) أَيْ حُكْمُهُ ( فِي الْأَرْضِ ) كَالسُّلْطَانِ وَمَنْ دُونَهُ مِنْ الْحُكَّامِ وَقَوْلُهُ : ( وَعَلَى كُلِّ مَنْ تَصِلُ يَدُهُ إلَى ذَلِكَ ) تَكْرَارٌ ( فَإِنْ لَمْ ) يَكُنْ مِمَّنْ ( يَقْدِرُ ) عَلَى ذَلِكَ التَّغْيِيرِ بِيَدِهِ ( فَبِلِسَانِهِ فَإِنْ لَمْ يَقْدِرْ ) بِلِسَانِهِ ( فَبِقَلْبِهِ ) .
