( فَصْلٌ تُشْتَرَطُ تَسْمِيَةُ الْأُصُولِ وَتَعْرِيفُهُمْ ) مِنْ الْفَرْعِ إذْ لَا بُدَّ مِنْ مَعْرِفَةِ عَدَالَتِهِمْ وَلَا تُعْرَفُ عَدَالَتُهُمْ مَا لَمْ يُعْرَفُوا وَلِيَتَمَكَّنَ الْخَصْمُ مِنْ جَرْحِهِمْ إذَا عَرَفَهُمْ ( فَلَا يَكْفِي ) قَوْلُ الْفَرْعِ ( أَشْهَدَنِي عَدْلٌ ) أَوْ نَحْوُهُ لِأَنَّ الْحَاكِمَ قَدْ يَعْرِفُ جَرْحَهُ لَوْ سَمَّاهُ وَلِأَنَّهُ يَسُدُّ بَابَ الْجَرْحِ عَلَى الْخَصْمِ ( وَلِفَرْعٍ تَزْكِيَةُ أَصْلٍ ) لَهُ لِأَنَّهُ غَيْرُ مُتَّهَمٍ فِيهَا ( لَا ) تَزْكِيَةُ ( أَحَدِ الشَّاهِدَيْنِ الْآخَرَ ) لِأَنَّهَا مِنْ تَتِمَّةِ شَهَادَتِهِ هُنَا وَالْمُزَكِّي قَائِمٌ بِأَحَدِ شَطْرَيْ الشَّهَادَةِ فَلَا يَصِحُّ قِيَامُهُ بِالثَّانِي وَبِمَا قَالَهُ عُلِمَ أَنَّهُ لَا يُشْتَرَطُ فِي شَهَادَةِ الْفَرْعِ تَزْكِيَةُ الْأَصْلِ بَلْ لَهُ إطْلَاقُهَا ثُمَّ الْقَاضِي يَبْحَثُ عَنْ عَدَالَتِهِ وَأَنَّهُ لَا يَلْزَمُهُ أَنْ يَتَعَرَّضَ فِي شَهَادَتِهِ لِصِدْقِ أَصْلِهِ لِأَنَّهُ لَا يَعْرِفُهُ بِخِلَافِ مَا إذَا حَلَفَ الْمُدَّعِي مَعَ شَاهِدِهِ حَيْثُ يَتَعَرَّضُ لِصِدْقِهِ لِأَنَّهُ يَعْرِفُهُ وَبِذَلِكَ صَرَّحَ الْأَصْلُ ( فَرْعٌ ) لَوْ اجْتَمَعَ أَصْلٌ وَفَرْعَا أَصْلٍ آخَرَ قُدِّمَ عَلَيْهِمَا فِي الشَّهَادَةِ كَمَا لَوْ كَانَ مَعَهُ مَاءٌ لَا يَكْفِيه يَسْتَعْمِلُهُ ثُمَّ يَتَيَمَّمُ قَالَهُ صَاحِبُ الِاسْتِقْصَاءِ
