( لَوْ حَجَّ ذَا ) أَيْ : شَخْصٌ ( عَنْ فَرْضِ مَنْ فِي قَبْرِهِ ) أَيْ : عَنْ فَرْضِ مَيِّتٍ ( أَوْ فَرْضِ مَعْضُوبٍ ) وَهُوَ الْعَاجِزُ عَنْ مُبَاشَرَتِهِ لِزَمَانَةٍ ، أَوْ غَيْرِهَا مِمَّا لَا يُرْجَى زَوَالُهُ ( وَذَا ) أَيْ : وَآخَرُ ( عَنْ نَذْرِهِ ، أَوْ الْقَضَا ) لَهُ ( فِي سَنَةٍ ) وَاحِدَةٍ ( لَمْ يُمْنَعْ ) ؛ لِأَنَّ الْمَرْعِيَّ أَنَّ غَيْرَ فَرِيضَةِ الْإِسْلَامِ لَا يَتَقَدَّمُ عَلَيْهَا لَا أَنَّهَا تَتَقَدَّمُ وَعَلَيْهِ لَوْ حَجَّ وَاحِدٌ عَنْ فَرْضِ مَيِّتٍ وَآخَرُ عَنْ قَضَائِهِ وَآخَرُ عَنْ نَذْرِهِ وَآخَرُ عَنْ نَفْلِهِ فِي سَنَةٍ جَاوَزَ الْمَعْضُوبُ بِالضَّادِ الْمُعْجَمَةِ سُمِّيَ بِهِ مَنْ ذُكِرَ لِقَطْعِ الزَّمَانَةِ حَرَكَتَهُ وَأَصْلُ الْعَضْبِ الْقَطْعُ وَقِيلَ بِالْمُهْمَلَةِ كَأَنَّهُ ضَرَبَ عَلَى عَصَبِهِ فَبَطَلَ عَمَلُ أَعْضَائِهِ
