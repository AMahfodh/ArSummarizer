فَصْلٌ الْخَامِسُ الطِّيبُ إجْمَاعًا ؛ لِأَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { أَمَرَ يَعْلَى بْنَ أُمَيَّةَ بِغَسْلِ الطِّيبِ } { وَقَالَ فِي الْمُحْرِمِ الَّذِي وَقَصَتْهُ نَاقَتُهُ لَا تُحَنِّطُوهُ } مُتَّفَقٌ عَلَيْهِمَا وَلِمُسْلِمٍ لَا تَمَسُّوهُ بِطِيبٍ ( فَيَحْرُمُ عَلَيْهِ ) أَيْ : الْمُحْرِمِ ( بَعْدَ إحْرَامِهِ تَطْيِيبُ بَدَنِهِ وَثِيَابِهِ ) أَيْ : شَيْءٍ مِنْ بَدَنِهِ نَصَّ عَلَيْهِ أَوْ شَيْءٍ مِنْ ثَوْبِهِ لِحَدِيثِ ابْنِ عُمَرَ ؛ وَلِأَنَّهُ يُعَدُّ مُتَطَيِّبًا بِكُلِّ وَاحِدٍ مِنْهُمَا ( وَلَوْ ) كَانَ التَّطَيُّبُ لَهُ ( مِنْ غَيْرِهِ بِإِذْنِهِ ) وَكَذَا لَوْ سَكَتَ وَلَمْ يَنْهَهُ كَمَا تَقَدَّمَ ، وَسَبَقَ حُكْمُ مَا لَوْ تَطَيَّبَ قَبْلَ إحْرَامِهِ ثُمَّ اسْتَدَامَهُ .( 6 / 467 )
