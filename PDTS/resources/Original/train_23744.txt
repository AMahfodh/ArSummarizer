( قَوْلُهُ أَوْ سِنُّك ) أَيْ الْمُتَّصِلُ بِهَا فِي الْجَمِيعِ أَخْذًا مِنْ قَوْلِهِ نَعَمْ لَوْ انْفَصَلَ إلَخْ ( قَوْلُهُ : يَجِبُ قَطْعُهَا ) يُؤْخَذُ مِنْهُ أَنَّهُ لَوْ حَلَّتْهَا الْحَيَاةُ وَقَعَ الطَّلَاقُ لِامْتِنَاعِ قَطْعِهَا حِينَئِذٍ ( قَوْلُهُ : وَصَوَّبَهُ ) أَيْ التَّسْوِيَةُ ( قَوْلُهُ : وَهُوَ الْأَوْجَهُ ) أَيْ التَّسْوِيَةُ بَيْنَ الشَّحْمِ وَالسِّمَنِ خِلَافًا لحج ( قَوْلُهُ : وَهَذَا وَاضِحٌ ) أَيْ هَذَا التَّوْجِيهُ عَلَى الْقَوْلِ بِعَدَمِ الْحِنْثِ ( قَوْلُهُ : مَا لَوْ أَرَادَ ) أَيْ فَلَا تَطْلُقُ ( قَوْلُهُ : كَالدَّمِ ) أَيْ قِيَاسًا عَلَى الْوُقُوعِ بِالْإِضَافَةِ إلَى الدَّمِ ( قَوْلُهُ : أَوْ لِحْيَتُك طَالِقٌ ) أَيْ فَإِنَّهُ لَا يَقَعُ وَمَحَلُّهُ حَيْثُ لَمْ يَكُنْ لَهَا لِحْيَةٌ وَإِنْ قَلَّتْ ( قَوْلُهُ : هَلْ تَطْلُقُ إلَى الْمَنْكِبِ ) وَالرَّاجِحُ أَنَّهَا تَطْلُقُ إلَى الْمَنْكِبِ فَمَتَى بَقِيَ مِنْ مُسَمَّى الْيَدِ جُزْءٌ وَقَعَ الطَّلَاقُ بِإِضَافَتِهِ لَهُ وَإِنْ قَلَّ( 22 / 158 )
