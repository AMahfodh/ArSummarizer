( وَمَنْ أَقَرَّ لِمُكَلَّفٍ بِمَالٍ فِي يَدِهِ وَلَوْ بِرِقِّ نَفْسِهِ ) مَعَ جَهْلِ نَسَبِهِ ( أَوْ كَانَ الْمُقَرُّ بِهِ قِنًّا فَكَذَّبَهُ الْمُقَرُّ لَهُ ) فِي إقْرَارِهِ ( بَطَلَ ) إقْرَارُهُ بِتَكْذِيبِهِ ( وَيَقِرُّ ) الْمُقَرُّ بِهِ ( بِيَدِ الْمُقِرِّ ) لِأَنَّهُ مَالٌ بِيَدِهِ لَا يَدَّعِيهِ غَيْرُهُ أَشْبَهَ اللُّقَطَةَ وَكَذَا يَبْقَى مَنْ أَقَرَّ بِرِقِّ نَفْسِهِ وَكَذَّبَهُ مُقَرٌّ لَهُ بِيَدِ نَفْسِهِ ( وَلَا يُقْبَلُ عَوْدُ مُقَرٍّ لَهُ إلَى دَعْوَاهُ ) أَيْ الْمُقَرِّ بِهِ بِأَنْ رَجَعَ فَصَدَّقَهُ الْمُقِرُّ لِأَنَّهُ مُكَذِّبٌ لِنَفْسِهِ ( وَإِنْ عَادَ الْمُقِرُّ فَادَّعَاهُ ) أَيْ الْمُقَرَّ بِهِ ( لِنَفْسِهِ أَوْ ) ادَّعَاهُ ( لِثَالِثٍ قُبِلَ ) مِنْهُ ذَلِكَ لِأَنَّهُ فِي يَدِهِ .( 12 / 477 )
