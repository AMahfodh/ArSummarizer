( وَإِنْ كَانَ لِأَحَدِهِمَا ) أَيْ أَحَدِ الْمُدَّعَيْنِ لِلْعَيْنِ ( بَيِّنَةٌ بِالْعَيْنِ ) الْمُدَّعَى بِهَا بِيَدِهَا أَوْ لَيْسَتْ بِيَدِ أَحَدٍ ( حُكِمَ لَهُ بِهَا ) لِرُجْحَانٍ بِالْبَيِّنَةِ ( وَإِنْ كَانَ لِكُلِّ وَاحِدٍ مِنْهُمَا بَيِّنَةٌ لَمْ يُقَدِّمْ أَسْبَقَهَا تَارِيخًا بَلْ ) هُمَا ( سَوَاءٌ ) خِلَافًا لِلْقَاضِي .
