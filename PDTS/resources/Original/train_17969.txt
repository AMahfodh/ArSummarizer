( وَيَنْقَطِعُ ) الْخِيَارُ ( بِالتَّخَايُرِ بِأَنْ يَخْتَارَ لُزُومَهُ ) أَيْ الْعَقْدَ بِهَذَا اللَّفْظِ أَوْ نَحْوِهِ كَأَمْضَيْنَاهُ أَوْ أَلْزَمْنَاهُ أَوْ أَجَزْنَاهُ ( فَلَوْ اخْتَارَ أَحَدُهُمَا ) لُزُومَهُ ( سَقَطَ حَقُّهُ ) مِنْ الْخِيَارِ ( وَبَقِيَ الْحَقُّ ) فِيهِ ( لِلْآخَرِ ) وَلَوْ قَالَ أَحَدُهُمَا لِلْآخَرِ : اخْتَرْ ، سَقَطَ خِيَارُهُ لِتَضَمُّنِهِ الرِّضَا بِاللُّزُومِ .
