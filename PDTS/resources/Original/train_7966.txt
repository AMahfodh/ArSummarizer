( وَمَنْ أَعْتَقَ عَبْدَهُ ) أَوْ أَمَتَهُ ( عَنْ مَيِّتٍ أَوْ ) أَعْتَقَهُ عَنْ ( حَيٍّ بِلَا أَمَرَهُ : فَوَلَاؤُهُ لِلْمُعْتِقِ ) لِحَدِيثِ { الْوَلَاءُ لِمَنْ أَعْتَقَ } وَكَمَا لَوْ لَمْ يَقْصِدْ غَيْرَهُ ( إلَّا إذَا أَعْتَقَ وَارِثٌ عَنْ مَيِّتٍ ) يَرِثُهُ ( فِي وَاجِبٍ عَلَيْهِ ) أَيْ الْمَيِّتِ ( كَكَفَّارَةِ ظِهَارٍ وَ ) كَفَّارَةِ وَطْءِ نَهَارِ ( رَمَضَانَ وَ ) كَفَّارَةِ ( قَتْلٍ ) وَيَمِينٍ .
