( أَوْصَى ) لِوَاحِدٍ ( بِثُلْثٍ ) مِنْ مَالِهِ ( وَ ) لِآخَرَ ( بِرُبُعِ مَا فَضَلْ ) وَلِآخَرَ ( بِنَصِيبِ ابْنٍ أَبُو ابْنَيْنِ ) هُوَ فَاعِلُ أَوْصَى ( جَعَلْ ) أَيْ الْحَاسِبُ عَلَى الطَّرِيقِ الْأَوَّلِ ( مَسْأَلَةً لِثُلُثٍ ) أَيْ مَسْأَلَةَ الثُّلُثِ الَّذِي هُوَ الْوَصِيَّةُ الْأُولَى ( مِنْ مَخْرَجِهْ ) وَهُوَ ثَلَاثَةٌ لِلْأَوَّلِ ثُلُثُهَا ( وَمَا تَبَقَّى ) وَهُوَ اثْنَانِ لَا يَنْقَسِمُ عَلَى أَرْبَعَةٍ مَخْرَجُ جُزْءِ الْبَاقِي لَكِنَّهُ ( ذُو وَفَاقٍ مُتَّجِهْ ) بِزِيَادَةِ مُتَّجَهٍ تَكْمِلَةً أَيْ مُوَافِقٌ ( لِمَخْرَجِ الرُّبُعِ بِنِصْفٍ فَاضْرِبْ ) وَفْقَ الْأَرْبَعَةِ ( اثْنَيْنِ فِي ثَلَاثَةٍ ) تَبْلُغْ سِتَّةً لِلْأَوَّلِ اثْنَانِ وَلِلثَّانِي وَاحِدٌ ، وَالْبَاقِي بَيْنَ الِابْنَيْنِ ، وَالثَّالِثِ أَثْلَاثًا ( أَوْ احْسِبْ ) عَلَى الطَّرِيقِ الثَّانِي ( مَسْأَلَةَ الْإِرْثِ مِنْ اثْنَيْنِ ) ثُمَّ ( زِدْ ) عَلَيْهِمَا ( فَرْدًا ) أَيْ وَاحِدًا لِلْمُوصَى لَهُ بِالنَّصِيبِ تَبْلُغُ ثَلَاثَةً ( فَثُلْثَهُ فَنِصْفَهُ ) أَيْ ثُمَّ زِدْ عَلَى الْمَبْلَغِ ثُلُثَهُ تَبْلُغْ أَرْبَعَةً ثُمَّ عَلَى الْمَبْلَغِ الثَّانِي نِصْفَهُ تَبْلُغْ سِتَّةً .
