( وَإِنْ قَتَلَ ) الْمُرْتَدُّ ( مَنْ يُكَافِئُهُ عَمْدًا فَعَلَيْهِ الْقِصَاصُ ) كَالْمُسْلِمِ وَأَوْلَى ( وَالْوَلِيُّ مُخَيَّرٌ بَيْنَ الْقَتْلِ وَالْعَفْوِ عَنْهُ ) كَالْمُسْلِمِ ( فَإِنْ اخْتَارَ ) الْوَلِيُّ ( الْقِصَاصَ قُدِّمَ ) الْقِصَاصُ ( عَلَى قَتْلِ الرِّدَّةِ تَقَدَّمَتْ الرِّدَّةُ أَوْ تَأَخَّرَتْ ) لِأَنَّهُ حَقٌّ آدَمِيٍّ جَزَمَ بِهِ فِي الشَّرْحِ وَغَيْرِهِ وَتَقَدَّمَ مَا فِيهِ الْقِصَاصُ ( وَإِنْ عَفَا ) الْوَلِيُّ ( عَلَى مَالٍ وَجَبَتْ الدِّيَةُ فِي مَالِهِ ) أَيْ الْمُرْتَدِّ كَسَائِرِ الْحُقُوقِ عَلَيْهِ ( وَإِنْ كَانَ ) الْقَتْلُ ( خَطَأً وَجَبَتْ ) الدِّيَةُ ( أَيْضًا فِي مَالِهِ ) وَكَذَا شِبْهُ الْعَمْدِ لِأَنَّهُ لَا عَاقِلَةَ لَهُ .
