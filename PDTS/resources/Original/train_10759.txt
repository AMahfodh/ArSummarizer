( وَإِنَّ أَلْحَقَتْهُ ) الْقَافَةُ ( بِهِمَا ) أَيْ الْمُدَّعِيَيْنِ ( لَحِقَ ) نَسَبُهُ ( بِهِمَا ) لِمَا رَوَى سَعِيدٌ عَنْ عُمَرَ فِي امْرَأَةٍ وَطِئَهَا رَجُلَانِ فِي طُهْرٍ فَقَالَ الْقَائِفُ : قَدْ اشْتَرَكَا فِيهِ جَمِيعًا فَجَعَلَهُ بَيْنَهُمَا وَبِإِسْنَادِهِ عَنْ الشَّعْبِيِّ قَالَ وَعَلِيٌّ يَقُولُ هُوَ ابْنُهُمَا وَهُمَا أَبَوَاهُ يَرِثُهُمَا وَيَرِثَانِهِ وَرَوَاهُ الزُّبَيْرُ بْنُ بِكَارٍ عَنْ عُمَرَ ( فَيَرِثُ ) الْمُلْحَقُ بِأَبَوَيْنِ ( كُلَّ وَاحِدٍ مِنْهُمَا إرْثَ وَلَدٍ كَامِلٍ ، وَيَرِثَانِهِ إرْثَ أَبٍ وَاحِدٍ ) لِمَا تَقَدَّمَ .( 14 / 313 )
