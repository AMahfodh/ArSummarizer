( وَ ) مَنْ وَصَّى لِشَخْصٍ ( بِعَبْدٍ ) مُعَيَّنٍ ( قِيمَتُهُ مِائَةٌ ، و ) وَصَّى ( لِآخَرَ بِثُلُثِ مَالِهِ وَمَالُهُ غَيْرُهُ ) أَيْ الْعَبْدِ ( مِائَتَانِ ، فَأَجَازَ الْوَرَثَةُ ) الْوَصِيَّتَيْنِ ( فَلِمُوصًى لَهُ بِالثُّلُثِ ثُلُثُ الْمِائَتَيْنِ ) لِأَنَّهُ لَا مُزَاحِمَ لَهُ فِيهِمَا وَهُوَ سِتَّةٌ وَسِتُّونَ وَثُلُثَانِ ( وَ ) لَهُ ( رُبْعُ الْعَبْدِ ) لِدُخُولِهِ فِي الْمَالِ الْمُوصَى لَهُ بِثُلُثِهِ مَعَ الْوَصِيَّةِ بِجَمِيعِهِ لِلْآخَرِ .
