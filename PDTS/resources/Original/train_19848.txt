( وَ ) إنْ حَلَفَ ( لَا يَأْكُلُ فَاكِهَةً حَنِثَ بِأَكْلِ تَمْرٍ وَرُمَّانٍ وَبِطِّيخٍ ) ؛ لِأَنَّهُ يَنْضَجُ وَيَحْلُو وَيُتَفَكَّهُ بِهِ فَيَدْخُلُ فِي مُسَمَّى الْفَاكِهَةِ ، وَسَوَاءٌ الْأَصْفَرُ أَوْ غَيْرُهُ ( وَ ) بِأَكْلِ ( كُلِّ ثَمَرِ شَجَرٍ غَيْرِ بَرِّيٍّ ) كَبَلَحٍ وَعِنَبٍ وَتُفَّاحٍ وَكُمَّثْرَى وَخَوْخٍ وَمِشْمِشٍ وَسَفَرْجَلٍ وَتُوتٍ وَتِينٍ وَمَوْزٍ وَأُتْرُجٍّ وَجُمَّيْزٍ ، وَعَطْفُ النَّخْلِ عَلَى الْفَاكِهَةِ فِي قَوْله تَعَالَى : { فِيهِمَا فَاكِهَةٌ وَنَخْلٌ وَرُمَّانٌ } لِلتَّشْرِيفِ لَا لِلْمُغَايَرَةِ كَقَوْلِهِ تَعَالَى : { مَنْ كَانَ عَدُوًّا لِلَّهِ وَمَلَائِكَتِهِ وَرُسُلِهِ وَجِبْرِيلَ وَمِيكَالَ } .
