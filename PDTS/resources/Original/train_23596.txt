وَأَجَابَ الْأَوَّلُ بِأَنَّ الْمُقِرَّ لَا يَتَحَتَّمُ حَدُّهُ بِخِلَافِ الْمُعَايِنِ ، فَلِذَلِكَ غُلِّظَتْ بَيِّنَتُهُ ( وَ ) يُشْتَرَطُ ( لِمَالٍ ) عَيْنٍ أَوْ دَيْنٍ ( وَعَقْدٍ مَالِيٍّ ) وَفَسْخِهِ ( كَبَيْعٍ وَإِقَالَةٍ وَحَوَالَةٍ وَضَمَانٍ ) وَصُلْحٍ وَرَهْنٍ وَشُفْعَةٍ وَمُسَابَقَةٍ وَحُصُولِ السَّبَقِ ( وَحَقٍّ مَالِيٍّ كَخِيَارٍ ) لِمَجْلِسٍ أَوْ شَرْطٍ ( وَأَجَلٍ ) وَجِنَايَةٍ تُوجِبُ مَالًا ( رَجُلَانِ أَوْ رَجُلٌ وَامْرَأَتَانِ ) لِعُمُومِ قَوْله تَعَالَى : { فَاسْتَشْهِدُوا } أَيْ فِيمَا يَقَعُ لَكُمْ : { شَهِيدَيْنِ مِنْ رِجَالِكُمْ ، فَإِنْ لَمْ يَكُونَا رَجُلَيْنِ فَرَجُلٌ وَامْرَأَتَانِ } فَكَانَ عُمُومُ الْأَشْخَاصِ فِيهِ مُسْتَلْزِمًا لِعُمُومِ الْأَحْوَالِ الْمُخْرِجِ مِنْهُ بِدَلِيلِ مَا يُشْتَرَطُ فِيهِ الْأَرْبَعَةُ ، وَمَا لَا يُكْتَفَى فِيهِ بِالرَّجُلِ وَالْمَرْأَتَيْنِ ، وَالْمَعْنَى فِي تَسْهِيلِ ذَلِكَ كَثْرَةُ جِهَاتِ الْمُدَايَنَاتِ وَعُمُومِ الْبَلْوَى بِهَا ، وَفُهِمَ مِنْ التَّخْيِيرِ قَبُولُ الْمَرْأَتَيْنِ مَعَ وُجُودِ الرَّجُلَيْنِ .
