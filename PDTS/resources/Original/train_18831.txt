وَشَبَّهَ فِي الْقَوْلَيْنِ فَقَالَ ( كَتَمْكِينِ ) شَخْصٍ ( بَائِعٍ ) بِالتَّنْوِينِ ( مُتْلِفٍ ) بِالتَّنْوِينِ بِضَمِّ الْمِيمِ وَكَسْرِ اللَّامِ نَعْتُ بَائِعٍ وَتَنَازَعَ بَائِعٌ وَمُتْلِفٌ ( مَا ) أَيْ شَيْئًا ( قِيمَتُهُ ) أَيْ الشَّيْءِ الْمَبِيعِ ( أَقَلُّ ) مِنْ ثَمَنِهِ الَّذِي بَاعَهُ الْبَائِعُ بِهِ لِأَجَلٍ كَبَيْعِهِ شَيْئًا بِعَشَرَةٍ لِشَهْرٍ ثُمَّ أَتْلَفَهُ الْبَائِعُ عَمْدًا قَبْلَ تَمَامِهِ فَقُوِّمَ عَلَيْهِ بِخَمْسَةٍ وَغَرِمَهَا حَالَّةً فَهَلْ يُمَكَّنُ الْبَائِعُ ( مِنْ ) أَخْذِ ( الزِّيَادَةِ ) الَّتِي زَادَهَا الثَّمَنُ عَلَى الْقِيمَةِ ( عِنْدَ ) حُلُولِ ( الْأَجَلِ ) فَيَأْخُذُ الْعَشَرَةَ الَّتِي بَاعَهُ بِهَا وَعَدَمُ تَمْكِينِهِ مِنْهَا فَيَأْخُذُ الْخَمْسَةَ الَّتِي غَرِمَهَا فَقَطْ( 10 / 267 )
