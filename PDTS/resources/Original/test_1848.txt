الْحَقُّ قَدْ يُرَادُ بِهِ الثَّابِتُ الْمُقَابِلُ لِلْبَاطِلِ وَلَمَّا كَانَتْ لِجَبْرِ أَلَمِ الْفِرَاقِ وَغَيْرُ الْمَدْخُولِ بِهَا لَمْ تَتَأَنَّسْ بِالزَّوْجِ حَتَّى تَتَأَلَّمَ قَالَ : ( وَ ) الْمُطَلَّقَةُ ( الَّتِي لَمْ يَدْخُلْ بِهَا ) الزَّوْجُ ( وَ ) الْحَالُ أَنَّهُ ( قَدْ فَرَضَ لَهَا ) صَدَاقًا ( فَلَا مُتْعَةَ لَهَا ) لِأَنَّ الْغَالِبَ عَدَمُ تَأَلُّمِهَا بِفِرَاقِهِ ، أَوْ لِأَنَّهَا أَخَذَتْ نِصْفَ صَدَاقِهَا مَعَ بَقَاءِ سِلْعَتِهَا ، وَمَفْهُومُ فَرَضَ لَهَا أَنَّ الْمَنْكُوحَةَ تَفْوِيضًا إذَا طَلُقَتْ قَبْلَ الْفَرْضِ لَهَا الْمُتْعَةُ وَهُوَ كَذَلِكَ .
