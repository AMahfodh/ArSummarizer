ا ه أَيْ بِالْإِفْضَاءِ ( قَوْلُهُ الْغَائِطُ ) فَاعِلُ لَمْ يَسْتَمْسِكْ ( قَوْلُهُ فَعَلَى الْأَوَّلِ ) أَيْ الْأَصَحِّ ( قَوْلُهُ فِي هَذَا ) أَيْ رَفْعُ مَا بَيْنَ مَدْخَلِ ذَكَرٍ وَمَخْرَجِ بَوْلٍ ( قَوْلُهُ وَعَلَى الثَّانِي ) أَيْ الضَّعِيفِ ( قَوْلُهُ بِالْعَكْسِ ) أَيْ فِي هَذَا دِيَةٌ وَفِي الْأَوَّلِ حُكُومَةٌ ( قَوْلُهُ بَلْ عَلَيْهِ ) أَيْ عَلَى الثَّانِي ( قَوْلُهُ فِي الْأَوَّلِ ) أَيْ رَفْعِ مَا بَيْنَ مَدْخَلِ ذَكَرٍ وَدُبُرٍ ( قَوْلُهُ فَإِنْ لَمْ يَسْتَمْسِكْ الْبَوْلُ إلَخْ ) أَيْ فِي الثَّانِي مُغْنِي وَرَوْضٌ ( قَوْلُهُ فَإِنْ أَزَالَهُمَا )( 37 / 287 )
