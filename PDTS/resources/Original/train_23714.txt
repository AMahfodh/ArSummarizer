( وَ ) يَصِحُّ أَيْضًا مَعَ ( هَزْلٍ ) هُوَ عَدَمُ قَصْدِ حَقِيقَةٍ وَلَا مَجَازٍ ( وَإِنْ عَلَّقَ ) الْعِتْقَ ( بِشَرْطٍ ) كَدُخُولِ دَارٍ ( صَحَّ ) وَعَتَقَ إنْ دَخَلَ ( وَالتَّعْلِيقُ بِأَمْرٍ كَائِنٍ تَنْجِيزٌ ، فَلَوْ قَالَ لِعَبْدِهِ ) وَهُوَ فِي مِلْكِهِ ( إنْ مَلَكْتُك فَأَنْتَ حُرٌّ عَتَقَ لِلْحَالِ ، بِخِلَافِ قَوْلِهِ لِمُكَاتَبِهِ إنْ أَنْتَ عَبْدِي فَأَنْتَ حُرٌّ ) لَا يَعْتِقُ لِقُصُورِ الْإِضَافَةِ ظَهِيرِيَّةٌ .
