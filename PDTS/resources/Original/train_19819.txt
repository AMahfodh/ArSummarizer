( وَ ) إنْ حَلَفَ الْمَطْلُوبُ ( لَا أَخَذْتَ حَقَّكَ مِنِّي فَأُكْرِهَ ) الْحَالِفُ ( عَلَى دَفْعِهِ ) لِغَرِيمِهِ فَأَخَذَهُ حَنِثَ ( أَوْ أَخَذَهُ ) أَيْ الْحَقَّ ( حَاكِمٌ فَدَفَعَهُ إلَى غَرِيمِهِ فَأَخَذَهُ ) الْغَرِيمُ ( حَنِثَ ) الْحَالِفُ لِأَنَّ غَرِيمَهُ أَخَذَهُ بِاخْتِيَارِهِ فَقَدْ وُجِدَ الْمَحْلُوفُ عَلَيْهِ لَا بِفِعْلِهِ اخْتِيَارًا ( كَ ) مَا لَوْ حَلَفَ مَنْ عَلَيْهِ الْحَقُّ عَلَى رَبِّهِ ( لَا تَأْخُذُ حَقَّكَ عَلِيَّ ) فَأُكْرِهَ الْحَالِفُ عَلَى الدَّفْعِ لَهُ أَوْ أَخَذَهُ حَاكِمٌ فَدَفَعَهُ إلَى غَرِيمِهِ حَنِثَ الْحَالِفُ لِمَا سَبَقَ وَ ( لَا ) يَحْنَثُ الْحَالِفُ ( إنْ أُكْرِهَ قَابِضُهُ ) عَلَى قَبْضِهِ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ : { وَمَا اُسْتُكْرِهُوا عَلَيْهِ } ( وَلَا إنْ وَضَعَهُ الْحَالِفُ بَيْنَ يَدَيْهِ ) أَيْ الْغَرِيمِ ( أَوْ فِي حِجْرِهِ فَلَمْ يَأْخُذهُ الْغَرِيمُ ) فَلَا حِنْثَ عَلَى الْحَالِفِ لِأَنَّ ذَلِكَ لَيْسَ بِأَخْذٍ ( لِأَنَّهُ لَا يَضْمَنُ مِثْلَ هَذَا مَالٌ وَلَا صَيْدٌ ) فِي إحْرَامٍ أَوْ حَرَمٍ ( وَيَحْنَثُ ) الْحَالِفُ ( لَوْ كَانَتْ يَمِينُهُ لَا أُعْطِيكَ لِأَنَّهُ إعْطَاءٌ إذْ هُوَ ) أَيْ الْإِعْطَاءُ ( تَمْكِينٌ وَتَسْلِيمٌ بِحَقٍّ فَهُوَ كَتَسْلِيمِ ثَمَنٍ وَمُثَمَّنٍ وَأُجْرَةٍ وَزَكَاةٍ ) فَإِنْ أَخَذَهُ حَاكِمٌ وَأَعْطَاهُ لِلْغَرِيمِ لَمْ يَحْنَثْ الْحَالِفُ لَا يُعْطِي لِأَنَّهُ لَيْسَ بِإِعْطَاءٍ .( 21 / 456 )
