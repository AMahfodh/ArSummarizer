( وَيَحْرُمُ ) وَلَا يَنْعَقِدُ ( تَزْوِيجُ أَمَةٍ مَوْطُوءَةٍ ) أَيْ وَطِئَهَا مَالِكُهَا ( وَمُسْتَوْلَدَةٌ قَبْلَ ) مُضِيِّ ( الِاسْتِبْرَاءِ ) بِمَا يَأْتِي ( لِئَلَّا يَخْتَلِطَ الْمَاءَانِ ) وَإِنَّمَا حَلَّ بَيْعُهَا قَبْلَهُ مُطْلَقًا لِأَنَّ الْقَصْدَ مِنْ الشِّرَاءِ مِلْكُ الْعَيْنِ وَالْوَطْءُ قَدْ يَقَعُ وَقَدْ لَا بِخِلَافِ النِّكَاحِ لَا يُقْصَدُ بِهِ إلَّا الْوَطْءُ أَمَّا مَنْ لَمْ يَطَأْهَا مَالِكُهَا فَإِنْ لَمْ تُوطَأْ زَوَّجَهَا مَنْ شَاءَ وَإِنْ وَطِئَهَا غَيْرُهُ زَوَّجَهَا لِلْوَاطِئِ وَكَذَا لِغَيْرِهِ إنْ كَانَ الْمَاءُ غَيْرُ مُحْتَرَمٍ أَوْ مَضَتْ مُدَّةُ الِاسْتِبْرَاءِ مِنْهُ
