( وَالْكَلْبُ وَالْخِنْزِيرُ نَجِسَانِ ) وَكَذَا مَا تَوَلَّدَ مِنْهُمَا وَسُؤْرُ ذَلِكَ وَعَرَقُهُ ، وَكُلُّ مَا خَرَجَ مِنْهُ لَا يَخْتَلِفُ الْمَذْهَبُ فِيهِ ، قَالَهُ فِي الشَّرْحِ ( يَطْهُرُ مُتَنَجِّسٌ بِهِمَا وَ ) مُتَنَجِّسٌ ( بِمُتَوَلِّدٍ مِنْهُمَا أَوْ مِنْ أَحَدِهِمَا أَوْ بِشَيْءٍ مِنْ أَجْزَائِهِمَا ) أَوْ أَجْزَاءِ مَا تَوَلَّدَ مِنْهُمَا ، أَوْ مِنْ أَحَدِهِمَا ( غَيْر أَرْضٍ وَنَحْوِهَا ) كَصَخْرٍ وَحِيطَانٍ ( بِسَبْعِ غَسَلَاتِ مُنَقِّيَةٍ ، إحْدَاهُنَّ بِتُرَابٍ طَهُورٍ وُجُوبًا ) لِحَدِيثِ أَبِي هُرَيْرَةَ مَرْفُوعًا قَالَ { إذَا وَلَغَ الْكَلْبُ فِي إنَاءِ أَحَدِكُمْ فَلْيَغْسِلْهُ سَبْعًا } مُتَّفَقٌ عَلَيْهِ وَلِمُسْلِمٍ { فَلْيُرِقْهُ ثُمَّ لِيَغْسِلْهُ سَبْعَ مَرَّاتٍ } وَلَهُ أَيْضًا { طَهُورُ إنَاءِ أَحَدِكُمْ إذَا وَلَغَ الْكَلْبُ فِيهِ أَنْ يَغْسِلَهُ سَبْعَ مَرَّاتٍ أُولَاهُنَّ بِالتُّرَابِ } .
