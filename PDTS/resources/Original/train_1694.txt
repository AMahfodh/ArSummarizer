( وَإِنْ لَمْ يَدَّعِهِ ) أَيْ الزَّوْجُ الْوَطْءَ فِيهَا بِأَنْ أَقَرَّ بِعَدَمِهِ أَوْ سَكَتَ ( طَلَّقَهَا ) أَيْ الزَّوْجُ الزَّوْجَةَ إنْ شَاءَتْهُ الزَّوْجَةُ ( وَإِلَّا ) أَيْ وَإِنْ امْتَنَعَ مِنْ طَلَاقِهَا ( فَهَلْ يُطَلِّقُ ) بِضَمٍّ فَفَتْحٍ فَكَسْرٍ مُثَقَّلًا ( الْحَاكِمُ ) الزَّوْجَةَ ( أَوْ يَأْمُرُهَا ) أَيْ الْحَاكِمُ الزَّوْجَةَ ( بِهِ ) أَيْ طَلَاقِهَا نَفْسَهَا بِأَنْ تَقُولَ أَنْتِ طَالِقٌ أَوْ طَلَّقْتُك أَوْ طَلَّقْت نَفْسِي مِنْك أَوْ أَنَا طَالِقٌ مِنْك وَهُوَ بَائِنٌ لِكَوْنِهِ قَبْلَ الْوَطْءِ ( ثُمَّ يَحْكُمُ ) الْحَاكِمُ بِوُقُوعِ الطَّلَاقِ لِيَرْتَفِعَ الْخِلَافُ فِيهِ عَلَى أَنَّ أَمْرَ الْحَاكِمِ بِطَلَاقِهَا نَفْسَهَا لَيْسَ حُكْمًا أَفَادَهُ عب .
