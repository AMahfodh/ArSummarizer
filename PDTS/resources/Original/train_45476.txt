( وَيَنْقَطِعُ ) تَتَابُعٌ ( بِصَوْمِ غَيْرِ رَمَضَانَ ) لِأَنَّهُ فَرَّقَهُ بِشَيْءٍ يُمْكِنُ تَحَرُّزُهُ مِنْهُ أَشْبَهَ مَا لَوْ أَفْطَرَ بِلَا عُذْرٍ ( وَيَقَعُ ) صَوْمُهُ ( عَمَّا نَوَاهُ ) لِأَنَّهُ زَمَانٌ لَمْ يَتَعَيَّنْ لِلْكَفَّارَةِ ( وَ ) يَنْقَطِعُ تَتَابُعٌ ( بِفِطْرٍ ) فِي أَثْنَاءِ الشَّهْرَيْنِ ( بِلَا عُذْرٍ ) لِقَطْعِهِ إيَّاهُ أَوْ بِفِطْرٍ ( لِجَهْلٍ ) لِأَنَّ مِثْلَ ذَلِكَ لَا يَخْفَى ( أَوْ نَسِيَ وُجُوبَ التَّتَابُعِ ، أَوْ ظَنَّ أَنَّهُ أَتَمَّ الشَّهْرَيْنِ ، فَبَانَ بِخِلَافِهِ ) انْقَطَعَ التَّتَابُعُ ، كَمَا لَوْ ظَنَّ أَنَّ الْوَاجِبَ شَهْرٌ وَاحِدٌ ، فَأَفْطَرَ .
