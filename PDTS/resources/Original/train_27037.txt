وَمِنْ قَبِيلِ قَوْلِ الْمُعَايِنِ مَا ذَكَرَهُ بِقَوْلِهِ ( وَلَوْ قِيلَ لِلْأَعْمَى ) وَهُوَ فِي صَلَاتِهِ ( صَلَاتُك إلَى الشَّمْسِ وَهُوَ يَعْلَمُ أَنَّ قِبْلَتَهُ غَيْرُهَا اسْتَأْنَفَ ) لِبُطْلَانِ تَقْلِيدِ الْأَوَّلِ بِذَلِكَ ( وَإِنْ أَبْصَرَ ) وَهُوَ ( فِي أَثْنَائِهَا وَعَلِمَ أَنَّهُ عَلَى الْإِصَابَةِ ) لِلْقِبْلَةِ بِمِحْرَابٍ أَوْ نَجْمٍ أَوْ خَبَرِ ثِقَةٍ أَوْ غَيْرِهَا ( أَتَمَّهَا أَوْ عَلَى الْخَطَأِ أَوْ تَرَدَّدَ بَطَلَتْ ) لِانْتِفَاءِ ظَنِّ الْإِصَابَةِ ( وَإِنْ ظَنَّ الصَّوَابَ غَيْرَهَا ) أَيْ غَيْرَ جِهَتِهِ ( انْحَرَفَ ) إلَى مَا ظَنَّهُ كَمَا لَوْ تَغَيَّرَ اجْتِهَادُ الْبَصِيرِ فِيهَا وَقَوْلُهُ وَإِنْ أَبْصَرَ إلَخْ مِنْ زِيَادَتِهِ وَصَرَّحَ بِهِ النَّوَوِيُّ فِي مَجْمُوعِهِ
