( لَا يُسَاوِي مِقْدَارَ أُجْرَتِهِ ) ، بَلْ وَكَذَلِكَ إذَا كَانَ قَدْرَ أُجْرَتِهِ ( قَوْلُهُ : وَفِي عِبَارَةٍ وَبُدِئَ ) ظَاهِرُ الْعِبَارَةِ بُدِئَ الْعَامِلُ مَعَ أَنَّهُ لَا يُنَاسِبُ قَوْلَهُ ؛ لِأَنَّ سَدَّ الْخَلَّةِ أَفْضَلُ ؛ لِأَنَّ هَذَا لَا يُنَاسِبُ إلَّا تَقْدِيمَ الْفَقِيرِ وَالْمِسْكِينِ عَلَى الْعِتْقِ ( قَوْلُهُ : وَتُقَدَّمُ الْمُؤَلَّفَةُ ) أَيْ : : عَلَى الْفُقَرَاءِ ( قَوْلُهُ : عَلَى الْفَقِيرِ ) أَرَادَ بِهِ مَا يَشْمَلُ الْمِسْكِينَ ، وَالْخَلَّةُ بِفَتْحِ الْخَاءِ الْفَقْرُ وَالْحَاجَةُ ( قَوْلُهُ وَأَخَذَ الْفَقِيرُ بِوَصْفَيْهِ ) لَكِنْ لَا يَأْخُذُ إلَّا بِإِعْطَاءِ الْإِمَامِ وَكَذَا لَا يَأْخُذُ بِوَصْفِ الْغُرْمِ إذَا كَانَ مِدْيَانًا إلَّا بِإِعْطَاءِ إمَامٍ ؛ لِأَنَّهُ يَقْسِمُهَا فَلَا يُحْكَمُ لِنَفْسِهِ وَقَوْلُهُ : وَكَذَا كُلُّ مَنْ جَمَعَ بَيْنَ وَصْفَيْنِ كَأَنْ يَكُونَ فَقِيرًا وَمِدْيَانًا .( 6 / 355 )
