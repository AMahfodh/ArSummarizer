( وَبَرِئَ ) الزَّوْجُ ( مِنْ جَمِيعِهِ ) نِصْفِهِ بِالْخُلْعِ وَنِصْفَهُ بِجَعْلِهِ عِوَضًا لَهُ فِيهِ ( وَإِنْ خَالَعَهَا ) قَبْلَ الدُّخُولِ ( بِمِثْلِ جَمِيعِ الصَّدَاقِ فِي ذِمَّتِهَا أَوْ ) خَالَعَهَا ( بِصَدَاقِهَا كُلِّهِ صَحَّ ) الْخُلْعُ لِصُدُورِهِ مِنْ أَهْلِهِ فِي مَحَلِّهِ ( وَيَرْجِعُ عَلَيْهَا بِنِصْفِهِ ) وَسَقَطَ عَنْهُ الصَّدَاقُ لِمَا تَقَدَّمَ .( 17 / 289 )
