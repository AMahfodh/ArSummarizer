وَلَمَّا فَرَغَ مِنْ الْكَلَامِ عَلَى الْأَذَانِ انْتَقَلَ يَتَكَلَّمُ عَلَى حُكْمِ الْإِقَامَةِ لِلصَّلَاةِ فَقَالَ : ( وَالْإِقَامَةُ ) لِلصَّلَاةِ ( سُنَّةُ عَيْنٍ لِذَكَرٍ بَالِغٍ فَذٍّ ) : أَيْ مُنْفَرِدٍ ( أَوْ مَعَ نِسَاءٍ ) : يُصَلِّي بِهِنَّ أَيْ أَوْ مَعَ صِبْيَانٍ ، ( وَ ) سُنَّةُ ( كِفَايَةٍ لِجَمَاعَةِ الذُّكُورِ الْبَالِغِينَ ) : مَتَى أَقَامَهَا وَاحِدٌ مِنْهُمْ كَفَى ، وَيُنْدَبُ أَنْ يَكُونَ الْمُؤَذِّنُ ( وَنُدِبَتْ ) الْإِقَامَةُ ( لِمَرْأَةٍ ) وَصَبِيٍّ سِرًّا فِيهِمَا .
