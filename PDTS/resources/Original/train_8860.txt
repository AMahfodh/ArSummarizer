( قَوْلُهُ : لِعُمُومِ الْحَاجَةِ ) إلَى قَوْلِهِ بِشُرُوطِهِ فِي النِّهَايَةِ ، وَالْمُغْنِي إلَّا قَوْلَهُ وَغَيْرُهُ ( قَوْلُهُ : مُطْلَقًا ) أَيْ : مَعَ الْقُدْرَةِ وَبِدُونِهَا ( قَوْلُهُ : وَغَيْرُهُ ) لَعَلَّهُ كَجَمْعِ أَنْوَاعٍ مِنْهُ بِتَيَمُّمٍ وَاحِدٍ ( قَوْلُهُ : نَعَمْ يُشْتَرَطُ أَنْ يَكُونَ مَقْصِدُهُ إلَخْ ) قَدْ يُفِيدُ أَنَّهُ لَوْ خَرَجَ إلَى بَعْضِ بَسَاتِينِ الْبَلَدِ أَوْ غِيطَانِهَا الْبَعِيدَةِ لَا يَجُوزُ لَهُ( 5 / 179 )
