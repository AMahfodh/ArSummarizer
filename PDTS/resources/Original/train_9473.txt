( قَوْلُهُ : لِأَنَّهُ فِي وَجْهٍ لَا يَدْخُلُ فِي مِلْكِ الْمُفْلِسِ ) أَشَارَ إلَى تَصْحِيحِهِ ( قَوْلُهُ : وَيُحْتَمَلُ خِلَافُهُ ) أَشَارَ إلَى تَصْحِيحِهِ ( قَوْلُهُ : لِأَنَّهُ مُقَصِّرٌ حَيْثُ أَخَّرَ إلَخْ ) قَدْ يُؤْخَذُ مِنْ تَعْلِيلِهِ أَنَّهُ فِي الْعَالَمِ بِالْمُزَاحَمَةِ ، وَلَيْسَ كَذَلِكَ ( قَوْلُهُ : فَلَوْ قَالَ وَارِثُ الْمُشْتَرِي : أَنَا أُعْطِيك مِنْ مَالِي لَزِمَهُ الْقَبُولُ ) شَمِلَ مَا إذَا لَمْ يَكُنْ لِلْمُشْتَرِي تَرِكَةٌ ( قَوْلُهُ وَلِأَنَّهُ يَبْغِي بِذَلِكَ بَقَاءَ مِلْكِهِ ) ؛ لِأَنَّ التَّرِكَةَ مِلْكُهُ فَأَشْبَهَ فَكَّ الْمَرْهُونِ وَفِدَاءَ الْجَانِي .( 9 / 399 )
