قُلْت : مَا قَالَهُ الْمُحَقِّقُونَ مُخَالِفٌ لِعَامَّةِ الْمُتُونِ كَمَا مَرَّ فَتَدَبَّرْ ثُمَّ رَأَيْت بَعْضَهُمْ نَقَلَ أَنَّ الْعَلَّامَةَ قَاسِمًا تَعَقَّبَهُ قَوْلُهُ كَذَبْحِ شَاةٍ ) تَمْثِيلٌ لِقَوْلِهِ فَإِنْ غَصَبَ وَغَيَّرَ ، أَوْ تَنْظِيرٌ لِقَوْلِهِ ضَمِنَهُ وَمَلَكَهُ أَيْ كَمَا يَضْمَنُهُ فِي ذَبْحِ شَاةٍ إلَخْ ( قَوْلُهُ بِالتَّنْوِينِ بَدَلِ الْإِضَافَةِ ) فِيهِ أَنَّهُمْ قَسَّمُوا تَنْوِينَ الْعِوَضِ إلَى مَا يَكُونُ عِوَضًا عَنْ جُمْلَةٍ أَوْ عَنْ حَرْفٍ أَوْ عَنْ كَلِمَةٍ كَقَوْلِهِ تَعَالَى { - فَضَّلْنَا بَعْضَهُمْ عَلَى بَعْضٍ - } { وَكُلٌّ فِي فَلَكٍ - } { أَيًّا مَا تَدْعُوا } - وَالْإِضَافَةُ أَمْرٌ مَعْنَوِيٌّ ، فَالْأَنْسَبُ( 25 / 293 )
