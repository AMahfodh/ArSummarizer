وَعُلِمَ مِمَّا سَبَقَ : أَنَّ ذَوِي أَرْحَامِ الْمُعْتِقِ لَا يَرِثُونَ عَتِيقَهُ وَإِنْ عُدِمَتْ عَصَبَتُهُ وَ ( قَالَ ابْنُ أَبِي مُوسَى : فَإِنْ مَاتَ الْعَبْدُ ) الْعَتِيقُ ( وَلَمْ يَتْرُكْ عَصَبَةً ) مِنْ النَّسَبِ ( وَلَا ذَا سَهْمٍ ) أَيْ فَرْضٍ ( وَلَا كَانَ لِمُعْتِقِهِ عَصَبَةٌ ) مِنْ النَّسَبِ وَلَا مِنْ الْوَلَاءِ ( وَرِثَهُ الرِّجَالُ مِنْ ذَوِي أَرْحَامِ مُعْتِقِهِ دُونَ نِسَائِهِمْ وَعِنْدَ عَدَمِهِمْ ) أَيْ عَدَمِ الرِّجَالِ مِنْ ذَوِي أَرْحَامِ مُعْتِقٍ يَكُونُ مِيرَاثُهُ ( لِبَيْتِ الْمَالِ ) .( 16 / 75 )
