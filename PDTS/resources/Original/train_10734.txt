وَإِذَا أَمْكَنَ إفْضَاءُ الْعَقْدِ إلَى مَقْصُودِهِ فَلَا نُبْطِلُهُ بِاحْتِمَالِ عَدَمِ الْإِفْضَاءِ إلَيْهِ وَإِذَا عَجَزَ قُسِّمَ مَا كَسَبَ بَيْنَهُمَا عَلَى قَدْرِ الْمِلْكَيْنِ فَلَمْ يَكُنْ أَحَدُهُمَا يَنْتَفِعُ إلَّا بِمَا يُقَابِلُ مِلْكَهُ وَعَادَ الْأَمْرُ بَعْدَ زَوَالِ الْكِتَابَةِ إلَى حُكْمِ الرِّقِّ كَمَا لَوْ لَمْ يَزُلْ ( وَلَمْ يُؤَدِّ ) أَيْ وَيَجُوزُ لِلْمُكَاتَبِ أَنْ يُؤَدِّيَ ( إلَيْهِمَا ) أَيْ إلَى سَيِّدَيْهِ ( إلَّا عَلَى قَدْرِ مِلْكَيْهِمَا ) مِنْهُ فَلَا يَجُوزُ أَنْ يُؤَدِّيَ إلَى أَحَدِهِمَا أَكْثَرَ مِنْ الْآخَرِ وَلَا يُقَدِّمُ أَحَدَهُمَا عَلَى الْآخَرِ لِأَنَّهُمَا سَوَاءٌ فِيهِ فَيَتَسَاوَيَانِ فِي كَسْبِهِ وَحَقِّهِمَا مُتَعَلِّقٌ بِمَا فِي يَدِهِ تَعْلِيقًا وَاحِدًا فَلَمْ يَكُنْ لَهُ أَنْ يَخُصَّ أَحَدَهُمَا بِشَيْءٍ دُونَ الْآخَرِ ( فَإِنْ قَبَّضَ أَحَدَهُمَا ) أَيْ الشَّرِيكَيْنِ ( دُونَ الْآخَرِ بِغَيْرِ( 16 / 352 )
