( وَلَا ) يَجُوزُ الْإِلْزَامُ بِشَرْطِ ( تَأْجِيلِ ) قَرْضٍ ، ( أَوْ ) شَرْطِ ( نَقْصٍ فِي وَفَاءٍ ) ؛ لِأَنَّهُ يُنَافِي مُقْتَضَى الْعَقْدِ ( أَوْ ) شَرْطِ ( جَرِّ نَفْعٍ ) ؛ فَيَحْرُمُ ؛ كَشَرْطِ ( أَنْ يُسْكِنَهُ ) الْمُقْتَرِضُ ( دَارِهِ ، أَوْ يَقْضِيَهُ خَيْرًا مِنْهُ أَوْ أَكْثَرَ ) مِمَّا أَقْرَضَهُ ( أَوْ ) أَنْ يَقْضِيَهُ ( بِبَلَدٍ آخَرَ ) ، وَلِحَمْلِهِ مُؤْنَةٌ ؛ لِأَنَّهُ عَقْدُ إرْفَاقٍ وَقُرْبَةٍ ، فَشَرْطُ النَّفْعِ فِيهِ يُخْرِجُهُ عَنْ مَوْضُوعِهِ ، فَإِنْ لَمْ يَكُنْ لِحَمْلِهِ مُؤْنَةٌ ، فَقَالَ فِي الْمُغْنِي : الصَّحِيحُ جَوَازُهُ ؛ لِأَنَّهُ مَصْلَحَةٌ لَهُمَا مِنْ غَيْرِ ضَرَرٍ .
