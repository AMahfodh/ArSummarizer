لِسَعِيدٍ أَنَّ صَاحِبَ جَيْشِ الشَّامِ كَتَبَ لِعُمَرَ إنَّا أَصَبْنَا أَرْضًا كَثِيرَةَ الطَّعَامِ وَالْغَلَّةِ وَكَرِهْت أَنْ أَتَقَدَّمَ فِي شَيْء مِنْ ذَلِكَ فَكَتَبَ إلَيْهِ : دَعْ النَّاس يَعْلِفُونَ وَيَأْكُلُونَ فَمَنْ بَاعَ مِنْهُمْ شَيْئًا بِذَهَبٍ أَوْ فِضَّةٍ فَفِيهِ خُمْسُ اللَّهِ وَسِهَامُ الْمُسْلِمِينَ وَ ( لَا ) يَجُوزُ أَنْ يَعْلِفَ مِنْهُ دَابَّةً ( لِصَيْدٍ ) كَجَارِحٍ وَفَهْدٍ لِعَدَمِ الْحَاجَةِ إلَيْهَا ( وَيَرُدُّ فَاضِلًا ) مِنْ طَعَامٍ وَعَلَفٍ ( وَلَوْ ) كَانَ ( يَسِيرًا ) لِاسْتِغْنَائِهِ عَنْهُ ( وَ ) يَرُدُّ ( ثَمَنَ مَا بَاعَ ) مِنْ طَعَامٍ وَعَلَفٍ لِلْخَبَرِ( 4 / 200 )
