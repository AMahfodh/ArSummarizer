( وَإِذَا نَوَى خَارِجَ رَمَضَانَ ) صَوْمًا ( وَقَضَاءً وَنَفْلًا ) فَنَفْلٌ ( أَوْ ) نَوَى قَضَاءً ، و ( نَذْرًا ، أَوْ ) نَوَى قَضَاءً ، و ( كَفَّارَةَ نَحْوِ ظِهَارٍ فَ ) هُوَ ( نَفْلٌ ) إلْغَاءً لِلْقَضَاءِ وَالنَّذْرِ وَالْكَفَّارَةِ لِعَدَمِ الْجَزْمِ بِنِيَّتِهَا فَتَبْقَى نِيَّةُ الصَّوْمِ ، وَرَدَّهُ صَاحِبُ الْإِقْنَاعِ بِأَنَّ مَنْ عَلَيْهِ قَضَاءُ رَمَضَانَ ، لَا يَصِحُّ تَطَوُّعُهُ قَبْلَهُ .( 3 / 336 )
