( وَيَصِحُّ ) الْعَقْدُ لِأَنَّ الْمَبِيعَ مَعْلُومٌ بِالْمُشَاهَدَةِ ( وَلِمُشْتَرٍ ) اشْتَرَى صُبْرَةً جِزَافًا مَعَ عِلْمِ الْبَائِعِ وَحْدَهُ مِقْدَارَهَا ( الرَّدُّ ) ؛ لِأَنَّ كَتْمَ الْبَائِعِ قَدْرَهَا غِشٌّ وَغَرَرٌ ( وَكَذَا ) بَيْعُ الصُّبْرَةِ جِزَافًا وَنَحْوِهَا مَعَ ( عِلْمِ الْمُشْتَرِي وَحْدَهُ ) مِقْدَارَهَا يَحْرُمُ ذَلِكَ عَلَى الْمُشْتَرِي لِمَا تَقَدَّمَ فِي الْبَائِعِ وَيَصِحُّ الْعَقْدُ ( وَلِبَائِعٍ ) وَحْدَهُ ( الْفَسْخُ ) لِمَا تَقَدَّمَ فِي عَكْسِهِ .
