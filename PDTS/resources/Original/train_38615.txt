( وَمَتَى حَصَدَ غَاصِبُ أَرْضٍ زَرْعَهُ ) مِنْ أَرْضٍ مَغْصُوبَةٍ بِأَنْ لَمْ يَتَمَلَّكْهُ رَبُّهَا قَبْلَ حَصَادِهِ ( زَكَّاهُ ) غَاصِبٌ لِاسْتِقْرَارِ مِلْكِهِ عَلَيْهِ ( وَيُزَكِّيهِ ) أَيْ الزَّرْعَ ( رَبُّهَا ) أَيْ : الْأَرْضِ الْمَغْصُوبَةِ ( إنْ تَمَلَّكَهُ ) أَيْ : الزَّرْعَ ( قَبْلَ ) حَصْدِهِ ، وَلَوْ بَعْدَ اشْتِدَادٍ لِأَنَّهُ يَتَمَلَّكُهُ بِمِثْلِ بَذْرِهِ ، وَعِوَضِ لَوَاحِقِهِ ، فَقَدْ اسْتَنَدَ مِلْكُهُ إلَى أَوَّلِ زَرْعِهِ ، فَكَأَنَّهُ أَخَذَهُ إذَنْ .( 3 / 105 )
