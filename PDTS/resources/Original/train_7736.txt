( وَإِنْ غَصَبَ عَصِيرًا فَتَخَمَّرَ ) عِنْدَهُ ( فَعَلَيْهِ ) أَيْ الْغَاصِبِ ( مِثْلُهُ ) أَمَّا ضَمَانُهُ فَلِأَنَّهُ صَارَ فِي حُكْمِ التَّالِفِ لِذَهَابِ مَالِيَّتِهِ بِتَخَمُّرِهِ ، وَأَمَّا كَوْنُهُ بِالْمِثْلِ فَلِأَنَّهُ مِثْلِيٌّ ( وَإِنْ انْقَلَبَ ) الْخَمْرُ ( خَلًّا رَدَّهُ ) الْغَاصِبُ ( وَ ) رَدَّ ( مَا نَقَصَ مِنْ قِيمَةِ الْعَصِيرِ أَوْ ) نَقَصَ ( مِنْهُ ب ) سَبَبِ ( غَلَيَانِهِ ) ؛ لِأَنَّهُ نَقْصٌ حَصَلَ بِيَدِهِ .( 13 / 159 )
