( أَوْ ) قَالَ أَنْتِ طَالِقٌ ( نِصْفُ ) طَلْقَتَيْنِ ( أَوْ ) قَالَ ( ثُلُثَ ) طَلْقَتَيْنِ ( أَوْ ) قَالَ ( سُدُسَ طَلْقَتَيْنِ أَوْ ) قَالَ ( رُبُعَ ) طَلْقَتَيْنِ ( أَوْ ) قَالَ ( ثُمُنَ طَلْقَتَيْنِ ، وَنَحْوَهُ ) كَخُمْسِ أَوْ سُبْعِ أَوْ تُسْعِ أَوْ عُشْرِ طَلْقَتَيْنِ ( فَوَاحِدَةٌ ) تَطْلُقُ ؛ لِأَنَّ نِصْفَ الطَّلْقَتَيْنِ طَلْقَةٌ ، وَثُلُثَهُمَا ثُلُثَا طَلْقَةٍ ، وَسُدُسَهُمَا ثُلُثُ طَلْقَةٍ ، وَرُبْعَهُمَا نِصْفُ طَلْقَةٍ ، وَثُمْنَهُمَا رُبْعُ طَلْقَةٍ ، وَخَمْسَهُمَا خُمْسَا طَلْقَةٍ ، وَقِسْ عَلَيْهِ ثُمَّ تُكْمَلُ ( أَوْ ) أَنْتِ طَالِقٌ ( نِصْفَيْ طَلْقَتَيْنِ ) فَثِنْتَانِ ؛ لِأَنَّ نِصْفَيْ الشَّيْءِ جَمِيعُهُ فَهُوَ كَأَنْتِ طَالِقٌ طَلْقَتَيْنِ ( أَوْ ) أَنْتِ طَالِقٌ ( ثَلَاثَةَ أَنْصَافِ طَلْقَةٍ فَثِنْتَانِ ؛ لِأَنَّ ثَلَاثَةَ الْأَنْصَافِ طَلْقَةٌ ، وَنِصْفٌ فَيُكْمَلُ أَوْ ) أَنْتِ طَالِقٌ ( أَرْبَعَةُ أَثْلَاثِ ) طَلْقَةٍ فَثِنْتَانِ ( أَوْ خَمْسَةِ أَرْبَاعِ ) طَلْقَةٍ فَثِنْتَانِ ( وَنَحْوِهِ ) كَثَمَانِيَةِ أَسْبَاعِ طَلْقَةٍ ( فَثِنْتَانِ ) ؛ لِأَنَّ ذَلِكَ طَلْقَةٌ ، وَجُزْءٌ فَيُكْمَلُ ؛ لِأَنَّهُ لَا يَتَبَعَّضُ .( 9 / 292 )
