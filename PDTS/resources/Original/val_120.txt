( قَوْلُهُ بِأَنَّهُ لَا اسْتِحَالَةَ عِنْدَ غَيْرِ الْمُنَجِّمِينَ ) أَيْ وَقَوْلُ الْمُنَجِّمِينَ لَا عِبْرَةَ بِهِ { وَاَللَّهُ عَلَى كُلِّ شَيْءٍ قَدِيرٌ } نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ : عَنْ الْوَاقِدِيِّ ) صَرِيحُ صَنِيعِ النِّهَايَةِ وَالْمُغْنِي أَنَّهُ رَاجِعٌ لِلْمَعْطُوفِ فَقَطْ ( قَوْلُهُ : يَوْمَ عَاشُورَاءَ ) أَيْ مِنْ الْمُحَرَّمِ ع ش .
