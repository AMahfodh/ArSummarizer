( وَ ) كَ ( نَصْبِ شَبَكَةٍ وَشَرَكِ فَخٍّ وَمِنْجَلٍ لِذَلِكَ ) أَيْ لِلصَّيْدِ ( وَحَبْسِ جَارِحٍ لَهُ ) أَيْ لِلصَّيْدِ ( أَوْ بِإِلْجَائِهِ بِمَضِيقٍ لَا يُفْلِتُ مِنْهُ ) فَيَمْلِكُهُ بِذَلِكَ لِأَنَّهُ بِمَنْزِلَةِ إثْبَاتِهِ ( وَإِنْ صَنَعَ بِرْكَةً لِيَصِيدَ بِهَا سَمَكًا فَمَا حَصَلَ فِيهَا مَلَكَهُ ) كَمَا لَوْ حَصَلَ بِشَبَكَتِهِ .
