( أَوْ ) لَمْ ( يَتَكَرَّرْ ) ، وَإِلَيْهِ الْإِشَارَةُ بِقَوْلِهِ : ( فَلَوْ رَأَتْ يَوْمًا وَلَيْلَةً ) دَمًا ( أَسْوَدَ ، ثُمَّ ) رَأَتْ دَمًا ( أَحْمَرَ وَجَاوَزَ أَكْثَرُ الْحَيْضِ ) ، أَيْ : جَاوَزَ خَمْسَةَ عَشَرَ يَوْمًا ، ( فَحَيْضُهَا زَمَنُ ) الدَّمِ ( الْأَسْوَدِ ) فَتَجْلِسُهُ ، وَمَا عَدَاهُ اسْتِحَاضَةٌ ، لِأَنَّهُ لَا يَصْلُحُ حَيْضًا ، ( أَوْ رَأَتْ فِي الشَّهْرِ الْأَوَّلِ خَمْسَةَ عَشَرَ يَوْمًا ) دَمًا ( أَسْوَدَ وَفِي ) الشَّهْرِ ( الثَّانِي أَرْبَعَةَ عَشَرَ وَفِي ) الشَّهْرِ ( الثَّالِثِ ثَلَاثَةَ عَشَرَ فَحَيْضُهَا زَمَنُ الْأَسْوَدِ فَقَطْ ) لِصَلَاحِيَّتِهِ لِذَلِكَ .( 2 / 137 )
