( بِخِلَافِ ) مَوْتِ ( الزَّوْجَةِ وَالصَّبِيِّ ) بِضَرْبِهِمَا لِلتَّأْدِيبِ فَإِنَّهُ يَضْمَنُهُمَا ( لِإِمْكَانِ تَأْدِيبِهِمَا بِغَيْرِهِ ) وَمَسْأَلَةُ الصَّبِيِّ مِنْ زِيَادَتِهِ هُنَا وَقَدْ قَدَّمَهَا كَأَصْلِهِ قَبْلَ فَرْعٍ وَإِنْ خَتَنَ الْأَجِيرُ حُرًّا ( وَالْقَرَارُ ) لِلضَّمَانِ ( عَلَى ) الْمُسْتَعْمِلِ ( الثَّانِي إنْ عَلِمَ ) الْحَالَ ( وَإِلَّا فَعَلَى الْأَوَّلِ ) إنْ كَانَتْ يَدُ الثَّانِي يَدَ أَمَانَةٍ كَالْمُسْتَأْجِرِ فَإِنْ كَانَتْ يَدَ ضَمَانٍ كَالْمُسْتَعِيرِ فَالْقَرَارُ عَلَيْهِ كَمَا أَوْضَحُوهُ فِي الْغَصْبِ نَبَّهَ عَلَيْهِ فِي الْمُهِمَّاتِ ( وَإِنْ أَرْكَبَهَا مِثْلَهُ فَتَعَدَّى ) بِمُجَاوَزَةِ الضَّرْبِ أَوْ بِغَيْرِهِ ( اُخْتُصَّ ) هُوَ ( بِالضَّمَانِ لِتَعَدِّيهِ ) فَلَا ضَمَانَ عَلَى الْمُرْكِبِ لِعَدَمِ تَعَدِّيهِ ( وَإِنْ اكْتَرَى ) دَابَّةً ( لِمِائَةِ رِطْلٍ حِنْطَةً فَحَمَلَ وَزْنَهَا شَعِيرًا أَوْ عَكْسَهُ ضَمِنَ لِأَنَّ الشَّعِيرَ أَخَفُّ ) فَمَا أَخَذَهُ مِنْ ظَهْرِ الدَّابَّةِ( 12 / 228 )
