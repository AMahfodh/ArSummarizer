وَشَبَّهَ فِي التَّرَبُّعِ فَقَالَ ( كَالْمُتَنَفِّلِ ) مِنْ جُلُوسٍ لِيُمَيِّزَ بَيْنَ الْجُلُوسِ الْبَدَلِ مِنْ الْقِيَامِ وَالْجُلُوسِ الْأَصْلِيِّ ( وَغَيَّرَ ) بِفَتَحَاتٍ مُثَقَّلًا أَيْ الْمُتَرَبِّعُ ( جِلْسَتَهُ ) بِكَسْرِ الْجِيمِ أَيْ هَيْئَةَ جُلُوسِهِ نَدْبًا حَالَ سُجُودِهِ وَ ( بَيْنَ سَجْدَتَيْهِ ) وَحَالَ تَشَهُّدِهِ بِإِفْضَاءِ الْيُسْرَى لِلْأَرْضِ وَالْيُمْنَى عَلَيْهَا وَإِذَا فَرَغَ مِنْ التَّشَهُّدِ تَرَبَّعَ وَهَكَذَا .( 2 / 105 )
