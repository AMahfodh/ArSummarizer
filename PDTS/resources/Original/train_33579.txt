كَوْنِهِ ( مُدَّعِيًا ) أَنَّ لَهُ زَوْجَةً ( غَائِبَةً ) اسْمُهَا عَمْرَةُ وَأَنَّهُ قَصَدَهَا بِقَوْلِهِ عَمْرَةُ طَالِقٌ فَلَمْ يُصَدَّقْ ( وَطَلَّقَ ) الْحَاكِمُ ( عَلَيْهِ ) الْحَاضِرَةَ فَاعْتَدَّتْ وَتَزَوَّجَتْ غَيْرَهُ وَدَخَلَ بِهَا ( ثُمَّ أَثْبَتَهُ ) أَيْ الزَّوْجُ نِكَاحَ عَمْرَةَ الْغَائِبَةَ فَتُرَدُّ الْحَاضِرَةُ إلَيْهِ وَلَا تَفُوتُ بِدُخُولِ الثَّانِي .( 9 / 77 )
