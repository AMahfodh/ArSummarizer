( قَالَ ) الزَّوْجُ ( خَالِعَتك فَقَبِلَتْ ) الْمَرْأَةُ وَلَمْ يَذْكُرَا مَالًا ( طَلُقَتْ ) لِوُجُودِ الْإِيجَابِ وَالْقَبُولِ ( وَ بَرِئَ عَنْ ) الْمَهْرِ ( الْمُؤَجَّلِ لَوْ ) كَانَ ( عَلَيْهِ وَإِلَّا ) يَكُنْ عَلَيْهِ مِنْ الْمُؤَجَّلِ شَيْءٌ ( رَدَّتْ ) عَلَيْهِ ( مَا سَاقَ إلَيْهَا مِنْ الْمَهْرِ الْمُعَجَّلِ ) لِمَا مَرَّ أَنَّهُ مُعَاوَضَةٌ فَتُعْتَبَرُ بِقَدْرِ الْإِمْكَانِ .
