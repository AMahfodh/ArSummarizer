( قَوْلُهُ : وَصَلَّى عَلَيْهِ غَيْرُ فَاضِلٍ ) أَيْ يُكْرَهُ لِلْفَاضِلِ أَنْ يُصَلِّيَ عَلَيْهِ زَجْرًا لِأَمْثَالِهِ ( قَوْلُهُ وَلَا يُطْمَسُ قَبْرُهُ ) أَيْ يُكْرَهُ فِيمَا يَظْهَرُ ( قَوْلُهُ : فَعَلَى الْمَذْهَبِ ) أَيْ فَيَنْبَنِي عَلَى الْمَذْهَبِ أَيْ الْمُعْتَمَدِ مِنْ كَوْنِهِ مُسْلِمًا غَيْرَ كَافِرٍ ( قَوْلُهُ : لَا فَائِتَةٍ ) أَيْ فَائِتَةٍ لَمْ يُطْلَبْ بِهَا فِي الْوَقْتِ وَإِلَّا أَدَّى إلَى أَنْ لَا يُقْتَلَ أَحَدٌ لِأَنَّهُ يُؤَخِّرُ إلَى أَنْ يَبْقَى مِقْدَارُ رَكْعَةٍ ثُمَّ يَتَطَهَّرُ فَيَفُوتُ الْوَقْتُ فَنَقُولُ لَا قَتْلَ( 3 / 103 )
