وَقَوْلُهُ وَلَاءً مِثَالٌ لَا شَرْطٌ ( أَوْ ) نَسِيَ ( مُتَّفِقَتَيْنِ ) لَا يَعْلَمُ عَيْنَهُمَا مِنْ صَلَوَاتِ يَوْمَيْنِ ( صَلَّى الْخَمْسَ مَرَّتَيْنِ بِتَيَمُّمَيْنِ ) وَفِي الْوَجْهِ السَّابِقِ بِعَشْرِ تَيَمُّمَاتٍ ( وَلَا يَتَيَمَّمُ لِفَرْضٍ قَبْلَ ) دُخُولِ ( وَقْتِ فِعْلِهِ ) لِأَنَّ التَّيَمُّمَ طَهَارَةُ ضَرُورَةٍ وَلَا ضَرُورَةَ قَبْلَ الْوَقْتِ وَيَدْخُلُ فِي وَقْتِ الْفِعْلِ مَا تُجْمَعُ فِيهِ( 1 / 480 )
