( وَلِلْمُكَاتَبِ ) وَلَوْ بِلَا إذْنٍ ( شِرَاءُ مَنْ يَعْتِقُ عَلَى سَيِّدِهِ ) وَقَبُولُ هِبَتِهِ وَالْوَصِيَّةُ لَهُ بِهِ تَوْسِيعًا لَهُ فِي طُرُقِ الِاكْتِسَابِ ( وَلَا يَعْتِقُ ) عَلَى سَيِّدِهِ ( إلَّا إنْ رَقَّ ) الْمُكَاتَبُ ( وَهُوَ ) أَيْ وَمِنْ يَعْتِقُ عَلَى سَيِّدِهِ ( مِلْكُهُ ) فَيَعْتِقُ عَلَيْهِ لِدُخُولِهِ فِي مِلْكِهِ .
