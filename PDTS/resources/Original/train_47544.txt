( فَصْلٌ وَكِنَايَتُهُ ) أَيْ الْقَذْفِ ( وَالتَّعْرِيضِ ) بِهِ ( نَحْوَ زَنَتْ يَدَاكَ وَرِجْلَاكَ أَوْ ) زَنَتْ ( يَدُكَ أَوْ رِجْلُكَ ) لِأَنَّ زِنَا هَذِهِ الْأَعْضَاء لَا يُوجِبُ الْحَدَّ لِقَوْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { الْعَيْنَانِ تَزْنِيَانِ وَزِنَاهُمَا النَّظَرُ } الْحَدِيثَ ( أَوْ ) زَنَى ( بَدَنُكَ ) لِأَنَّ زِنَاهُ يُحْتَمَلُ أَنْ يَكُونَ بِزِنَا شَيْءٍ مِنْ أَعْضَائِهِ عَلَى الْمَعْنَى السَّابِقِ غَيْرِ الْفَرْجِ ( وَنَحْوُ قَوْلِهِ لِامْرَأَةٍ رَجُلٌ قَدْ فَضَحْتِهِ ) لِأَنَّهُ يُحْتَمَلُ أَنْ يَكُونَ بِشَكْوَاكِ ( وَغَطَّيْتِ ) رَأْسَهُ ( أَوْ نَكَّسْتِ رَأْسَهُ ) لِأَنَّهُ يُحْتَمَلُ أَنْ يَكُونَ حَيَاءً مِنْ النَّاسِ ( وَجَعَلْتِ لَهُ قُرُونًا وَعَلَّقْتِ عَلَيْهِ أَوْلَادًا مِنْ غَيْرِهِ ) .
