( وَإِنْ ) ( حَلَفَ ) زَوْجٌ ( عَلَى فِعْلِ غَيْرِهِ ) ( فَفِي ) صِيغَةِ ( الْبِرِّ ) الْمُطْلَقِ حُكْمُهُ ( كَنَفْسِهِ ) فَلَا فَرْقَ بَيْنَ إنْ دَخَلْت أَنَا الدَّارَ فَأَنْتِ طَالِقٌ وَبَيْنَ إنْ دَخَلْت أَنْتِ أَوْ فُلَانٌ الدَّارَ فَأَنْتِ طَالِقٌ فَيَنْتَظِرُ إذَا أَثَبَتَ وَلَا يُمْنَعُ مِنْ وَطْءٍ وَلَا بَيْعٍ أَمَّا الْبِرُّ الْمُوَقَّتُ كَإِنْ لَمْ يَدْخُلْ فُلَانٌ الدَّارَ قَبْلَ شَهْرٍ فَأَنْتِ طَالِقٌ أَوْ حُرَّةٌ فَيُمْنَعُ فِي الرَّقِيقِ مِنْ الْبَيْعِ وَلَا يُمْنَعُ فِيهِ وَلَا فِي الزَّوْجَةِ مِنْ الْوَطْءِ
