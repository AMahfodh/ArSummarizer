( قَوْلُهُ وَبِخَلْطِهَا ) ظَاهِرُهُ أَنَّهُ بِمُجَرَّدِ الْخَلْطِ يَضْمَنُ وَإِنْ لَمْ يَحْصُلْ تَلَفٌ وَهُوَ كَذَلِكَ كَمَا قَالَهُ اللَّخْمِيُّ ( قَوْلُهُ إلَّا كَقَمْحٍ بِمِثْلِهِ ) أَيْ جِنْسًا وَصِفَةً فَلَوْ خَلَطَ سَمْرَاءَ بِمَحْمُولَةٍ فَإِنَّهُ يَضْمَنُ ( قَوْلُهُ أَوْ الرِّفْقِ ) أَيْ بِأَنْ كَانَ أَرْفَقَ بِهِ مِنْ شَغْلِ مَخْزَنَيْنِ بِذَلِكَ وَكِرَائِهِمَا ( قَوْلُهُ عَلَى الْمُعْتَمَدِ ) أَيْ خِلَافًا لِمَنْ يَقُولُ عَلَى كُلِّ وَاحِدٍ نِصْفُهُ .( 18 / 251 )
