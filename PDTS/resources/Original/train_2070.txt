( وَمَنْ بَيْنَهُمَا نَحْوُ عَبِيدٍ أَوْ بَهَائِمَ وَثِيَابٍ مِنْ جِنْسٍ ) ؛ أَيْ : نَوْعٍ وَاحِدٍ كَأَنْ تَكُونَ الْعَبِيدُ كُلُّهُمْ نُوبَةً أَوْ حَبَشًا وَنَحْوَهُ ، وَالْبَهَائِمُ كُلُّهَا إبِلًا أَوْ بَقَرًا وَنَحْوَهُ ، وَالثِّيَابُ كُلُّهَا مِنْ كَتَّانٍ وَنَحْوِهِ ، وَالْأَوَانِي كُلُّهَا مِنْ نُحَاسٍ أَوْ زُجَاجٍ وَنَحْوِهِ ( فَطَلَبَ أَحَدُهُمَا ) ؛ أَيْ : الشَّرِيكَيْنِ فِيهَا ( قَسْمَهَا أَعْيَانًا بِالْقِيمَةِ بِأَنْ تُعَدَّلَ بِهَا ) وَأَبَى شَرِيكُهُ ( أُجْبِرَ مُمْتَنِعٌ إنْ تَسَاوَتْ الْقِيَمُ ) لِحَدِيثِ عِمْرَانَ بْنِ حُصَيْنٍ : { أَنَّ رَجُلًا أَعْتَقَ فِي مَرَضِهِ سِتَّةَ أَعْبُدٍ ، وَأَنَّ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ جَزَّأَهُمْ ثَلَاثَةَ أَجْزَاءٍ فَأَعْتَقَ اثْنَيْنِ وَأَرَقَّ أَرْبَعَةً } وَهَذِهِ قِسْمَةٌ لَهُمْ وَلِأَنَّهَا أَعْيَانٌ أَمْكَنَ قِسْمَتُهَا بِلَا ضَرُورَةٍ وَلَا رَدِّ عِوَضٍ ؛ أَشْبَهَتْ الْأَرْضَ ( وَإِلَّا ) تَكُنْ مُتَسَاوِيَةً ( فَلَا ، كَمَا لَوْ اخْتَلَفَ الْجِنْسُ ) بِأَنْ كَانَ بَعْضُ الثِّيَابِ قُطْنًا وَبَعْضُهَا كَتَّانًا وَنَحْوَهُ .( 19 / 432 )
