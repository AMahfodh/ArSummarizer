( وَ ) فَسَدَ بَيْعُ ( سَمَكٍ لَمْ يُصَدْ ) لِأَنَّهُ بَيْعُ مَا لَا يَمْلِكُهُ ( أَوْ صِيدَ وَأُلْقِيَ فِيمَا ) أَيْ حَظِيرَةٍ ( لَا يُؤْخَذُ مِنْهُ إلَّا بِحِيلَةٍ ) لِأَنَّهُ غَيْرُ مَقْدُورِ التَّسْلِيمِ ( وَإِنْ أُخِذَ بِدُونِهَا صَحَّ ) لِأَنَّهُ مَقْدُورُ التَّسْلِيمِ ( إلَّا إذَا دَخَلَ ) فِي الْحَظِيرَةِ بِنَفْسِهِ ( وَلَمْ يَسُدَّ مَدْخَلَهُ ) لِعَدَمِ الْمِلْكِ .
