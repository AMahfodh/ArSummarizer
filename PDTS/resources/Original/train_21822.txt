فَصْلٌ ( وَمَنْ أُرِيدَتْ ) ؛ أَيْ : قُصِدَتْ ( نَفْسُهُ ) لِيُقْتَلَ ( أَوْ ) يُفْعَلَ بِهَا الْفَاحِشَةُ ، أَوْ أُرِيدَتْ ( حُرْمَتُهُ ) كَأُمِّهِ أَوْ أُخْتِهِ أَوْ زَوْجَتِهِ وَنَحْوِهِنَّ لِزِنًا أَوْ قَتْلٍ ، أَوْ أُرِيدَ أَخْذُ مَالِهِ ( وَلَوْ قَلَّ ) مَا أُرِيدَ مِنْ مَالِهِ ( أَوْ لَمْ يَكْفِ ) مَنْ أُرِيدَتْ نَفْسُهُ أَوْ حُرْمَتُهُ أَوْ مَالُهُ ( الْمَرِيدَ ) لِذَلِكَ ( فَلَهُ دَفْعُهُ ) عَنْ نَفْسِهِ وَحُرْمَتِهِ وَمَالِهِ ( إنْ لَمْ يَخَفْ ) الدَّافِعُ ( مُبَادَرَتَهُ لَهُ بِالْقَتْلِ بِأَسْهَلِ مَا ) ؛ أَيْ : شَيْءٍ ( يَظُنُّ انْدِفَاعَهُ بِهِ ) ؛ لِأَنَّهُ لَوْ مُنِعَ مِنْ ذَلِكَ لَأَدَّى إلَى تَلَفِهِ وَأَذًى فِي نَفْسِهِ وَحُرْمَتِهِ وَمَالِهِ ، وَلِأَنَّهُ لَوْ لَمْ يَجُزْ ذَلِكَ لَتَسَلَّطَ النَّاسُ بَعْضُهُمْ عَلَى بَعْضٍ [ ، وَأَدَّى إلَى الْهَرَجِ وَالْمَرَجِ ، وَلِحَدِيثِ أَبِي هُرَيْرَةَ ] .
