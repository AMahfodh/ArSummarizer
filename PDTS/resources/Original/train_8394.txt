بُدَّ مِنْ اجْتِمَاعِ الْإِذْنَيْنِ ، وَكَذَا لَا بُدَّ مِنْ سَبْقِ إذْنِهَا لِلْخُنْثَى إذْ لَا يَصِحُّ إذْنُهُ لِمَنْ يَلِيهِ بِتَقْدِيرِ ذُكُورَتِهِ إلَّا إذَا أَذِنَتْ لَهُ الْعَتِيقَةُ فِي التَّزْوِيجِ لِيَصِحَّ تَوْكِيلُهُ قَوْلُهُ أَمَّا الْعَبْدُ ) أَيْ الْمَوْقُوفُ ( قَوْلُهُ : وَنَحْوُهُ ) أَيْ الْمَسْجِدِ ( قَوْلُهُ : مِنْ تَعَلُّقِ الْمَهْرِ وَالنَّفَقَةِ ) ظَاهِرُهُ وَإِنْ خَافَ الْعَنَتَ وَهُوَ ظَاهِرٌ لِلْعِلَّةِ الْمَذْكُورَةِ ( قَوْلُهُ أَنْ يَكْتُبَ بِتَزْوِيجِهَا ) أَيْ وَالْحَالَةُ أَنَّهَا خَارِجَةٌ عَنْ مَحَلِّ وِلَايَتِهِ ( قَوْلُهُ : لَا تَتَعَلَّقُ بِالْخَاطِبِ ) أَيْ الزَّوْجِ .( 20 / 342 )
