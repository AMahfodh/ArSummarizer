وَيَظْهَرُ تَعَيُّنُ الصَّيْدِ لِبِنَاءِ حَقِّ اللَّهِ عَلَى الْمُسَامَحَةِ ( قُلْتُ قَدْ طُعِنْ ) أَيْ : اُعْتُرِضَ ( عَلَى الَّذِي يَظُنُّ ) أَخْذًا مِنْ تَعْبِيرِ الْحَاوِي وَغَيْرِهِ عَمَّا ذُكِرَ ( بِالْأَوْلَى هُنَا رُجْحَانُهُ ) مَعَ جَوَازِ الْآخَرِ وَهَذَا الظَّنُّ مَرْدُودٌ ( فَإِنَّهُ ) قَدْ ( تَعَيَّنَا ) ذَلِكَ بِقِيَامِ الدَّلِيلِ عَلَى تَعَيُّنِهِ فَصِيغَةُ أَوْلَى هُنَا لِلتَّعْيِينِ لَا لِجَوَازِ كُلٍّ مِنْ الْأَمْرَيْنِ مَعَ رُجْحَانِ الْآخَرِ وَإِنْ كَانَ هُوَ الْأَصْلُ فِيهَا ( وَمَيْتَةٌ مَعَ لَحْمِ صَيْدٍ ) لِلْحَرَمِ أَوْ لِغَيْرِهِ وَقَدْ ذَبَحَهُ مُحْرِمٌ مُطْلَقًا أَوْ حَلَالٌ لِمُحْرِمٍ ( تَسْتَوِي ) مَعَهُ فَيَتَخَيَّرُ الْمُضْطَرُّ بَيْنَهُمَا ؛ لِأَنَّ كُلًّا مِنْهُمَا مَيْتَةٌ وَلَا مُرَجِّحَ وَزَادَ قَوْلَهُ ( وَمَيْتَتَانِ ) إحْدَاهُمَا طَاهِرَةُ الْأَصْلِ فِي الْحَيَاةِ دُونَ الْأُخْرَى كَحِمَارٍ وَكَلْبٍ ( طَاهِرِ الْأَصْلِ ) مِنْهُمَا ( الْقَوِيّ ) أَيْ : الرَّاجِحُ فَيَتَعَيَّنُ كَمَا رَجَّحَهُ فِي الرَّوْضَةِ بَحْثًا ، وَقَضِيَّةُ كَلَامِ النَّاظِمِ أَنَّهُ لَوْ كَانَ إحْدَى الْمَيْتَتَيْنِ مِنْ جِنْسِ( 19 / 209 )
