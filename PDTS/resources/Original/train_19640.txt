( وَإِنْ مَاتَ بَعْضٌ ) مِنْ وَرَثَةِ الْمَيِّتِ الْأَوَّلِ ( قَبْلَ الْقِسْمَةِ ) لِتَرِكَتِهِ ( وَوَرِثَهُ ) أَيْ الْمَيِّتَ الثَّانِيَ ( الْبَاقُونَ ) مِنْ وَرَثَةِ الْأَوَّلِ بِالْوَجْهِ الَّذِي وَرِثُوا الْأَوَّلَ بِهِ ( كَثَلَاثَةِ بَنِينَ ) لِرَجُلٍ أَوْ امْرَأَةٍ مَاتَ أَوْ مَاتَتْ ( ثُمَّ مَاتَ أَحَدُهُمْ ) أَيْ الْبَنِينَ قَبْلَ قَسْمِ تَرِكَةِ الْأَوَّلِ ، وَلَا وَارِثَ لِلثَّانِي غَيْرَ أَخَوَيْهِ فَالْمَيِّتُ الثَّانِي كَأَنَّهُ لَمْ يَكُنْ ، وَتُقْسَمُ تَرِكَةُ الْأَوَّلِ بَيْنَ الِابْنَيْنِ الْبَاقِيَيْنِ ، وَكَذَا لَوْ كَانُوا أَرْبَعَةً أَوْ أَكْثَرَ وَمَاتَ بَعْضُهُمْ قَبْلَهَا وَبَقِيَ اثْنَانِ أَوْ أَكْثَرُ ( أَوْ ) وَرِثَ الْمَيِّتَ الثَّانِيَ ( بَعْضٌ ) مِنْ الْبَاقِينَ مِنْ وَرَثَةِ الْمَيِّتِ الْأَوَّلِ وَبَعْضٌ مِنْهُمْ لَا يَرِثُهُ ( كَزَوْجٍ مَعَهُمْ ) أَيْ الْبَنِينَ بِأَنْ مَاتَتْ زَوْجَتُهُ عَنْهُ وَعَنْ بَنِيهَا الثَّلَاثَةِ ثُمَّ مَاتَ أَحَدُهُمْ قَبْلَ قَسْمِ تَرِكَتِهَا ( وَلَيْسَ ) الزَّوْجُ الَّذِي مَعَهُمْ ( أَبَاهُمْ ) وَكَذَا مَوْتُ الزَّوْجِ عَنْ زَوْجَتِهِ وَبَنِيهِ مِنْ غَيْرِهَا ثُمَّ مَوْتُ أَحَدِهِمْ قَبْلَهَا .
