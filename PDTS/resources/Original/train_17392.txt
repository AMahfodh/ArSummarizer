، ( وَ ) يَمْنَعُهُ ( هِبَةً أُقْبِضَتْ ) فَإِنْ وَهَبَتْهُ ، وَلَمْ تَقْبِضْهُ حَتَّى طَلَّقَ وَنَحْوَهُ ؛ رَجَعَ بِنِصْفِهِ ، ( وَ ) وَيَمْنَعُهُ ( عِتْقٌ ) بِأَنْ كَانَ رَقِيقًا فَأَعْتَقَتْهُ ؛ لِزَوَالِ مِلْكِهَا عَنْهُ بِهَذِهِ الْأُمُورِ ( وَ ) يَمْنَعُهُ ( رَهْنٌ ) أُقْبِضَ ؛ لِأَنَّهُ يُرَادُ لِلْبَيْعِ الْمُزِيلِ لِلْمِلْكِ ، وَلِهَذَا لَا يَجُوزُ رَهْنُ مَا لَا يَجُوزُ بَيْعُهُ ( وَ ) يَمْنَعُهُ ( كِتَابَةٌ ) ؛ لِأَنَّهَا تُرَادُ لِلْعِتْقِ الْمُزِيلِ لِلْمِلْكِ ، وَهِيَ عَقْدٌ لَازِمٌ فَجَرَتْ مَجْرَى الرَّهْنِ وَ ( لَا ) يَمْنَعُهُ ( عَقْدٌ جَائِزٌ ) كَشَرِكَةٍ وَمُضَارَبَةٍ وَإِيدَاعٍ وَإِعَارَةٍ .
