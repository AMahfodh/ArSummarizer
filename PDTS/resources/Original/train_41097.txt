( وَلَوْ ادَّعَيَا ) أَيْ اثْنَانِ ( زَوْجِيَّةَ امْرَأَةٍ ) فَأَنْكَرَتْهُمَا أَوْ أَحَدَهُمَا دُونَ الْآخَرِ ، ( وَأَقَامَ كُلٌّ ) مِنْهُمَا ( بَيِّنَةً ) بِدَعْوَاهُ ، ( وَلَوْ كَانَتْ ) الْمَرْأَةُ ( بِيَدِ أَحَدِهِمَا ) أَيْ الْمُدَّعِيَيْنِ ( سَقَطَتَا ) أَيْ الْبَيِّنَتَانِ لِتَعَارُضِهِمَا ، وَالْيَدُ لَا تَثْبُتُ عَلَى الْحُرِّ وَإِنْ أَقَرَّتْ لِأَحَدِهِمَا لَمْ تُقْبَلْ ؛ لِأَنَّهَا مُتَّهَمَةٌ ، وَإِنْ كَانَ لِأَحَدِهِمَا بَيِّنَةٌ وَحْدَهُ حُكِمَ لَهُ بِهَا ، وَإِنْ ادَّعَاهَا وَاحِدٌ فَصَدَّقَتْهُ قُبِلَ إقْرَارُهَا ؛ لِأَنَّهَا غَيْرُ مُتَّهَمَةٍ إذَنْ .( 12 / 323 )
