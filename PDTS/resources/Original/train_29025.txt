( وَإِنْ أَقَامَ ) أَيْ أَشْهَدَ ( الرَّجُلُ بَيِّنَةً عَلَى شِرَاءِ مَا ) أَيْ مَتَاعِ الْبَيْتِ الَّذِي هُوَ مُعْتَادٌ ( لَهَا ) أَيْ الْمَرْأَةِ كَحُلِيِّ النِّسَاءِ ( حَلَفَ ) الرَّجُلُ أَنَّهُ اشْتَرَاهُ لِنَفْسِهِ ، وَأَنَّهَا لَمْ تُعْطِهِ ثَمَنَهُ إنْ كَانَ اشْتَرَاهُ مِنْ غَيْرِهَا ( وَقُضِيَ ) بِضَمٍّ فَكَسْرٍ ( لَهُ ) أَيْ الرَّجُلِ ( بِهِ ) أَيْ الْحُلِيِّ مَثَلًا .
