وَمَنْ أَمْكَنَ أَنْ يَتَزَوَّجَ لِنَفْسِهِ لَمْ تَثْبُتْ وِلَايَةُ تَزْوِيجِهِ لِغَيْرِهِ كَالْعَاقِلِ وَمَنْ زَالَ عَقْلُهُ بِغَيْرِ سَامٍّ أَوْ مَرَضٍ يُرْجَى زَوَالُهُ فَكَالْعَاقِلِ ( وَيَصِحُّ قَبُولُ ) صَبِيٍّ ( مُمَيِّزٍ لِنِكَاحِهِ بِإِذْنِ وَلِيِّهِ ) كَتَوَلِّيهِ الْبَيْعَ وَالشِّرَاءَ لِنَفْسِهِ بِإِذْنِ وَلِيِّهِ ( وَلِكُلِّ وَلِيٍّ ) مِنْ أَبٍ وَوَصِيِّهِ وَبَقِيَّةِ الْعَصَبَاتِ وَالْحَاكِمِ( 8 / 342 )
