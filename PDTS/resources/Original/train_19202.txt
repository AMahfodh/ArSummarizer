( لَا ) يُرَدُّ الْمَالُ الْمُخَالَعُ بِهِ ( إنْ لَمْ يَقُلْ ) الزَّوْجُ ( ثَلَاثًا ) بِأَنْ أَطْلَقَ أَوْ قَيَّدَ بِوَاحِدَةٍ وَلَمْ يُطَلِّقْهَا قَبْلَ ذَلِكَ اثْنَتَيْنِ ( وَلَزِمَهُ ) أَيْ الزَّوْجَ الَّذِي قَالَ إنْ خَالَعْتكِ فَأَنْتِ طَالِقٌ ( طَلْقَتَانِ ) وَاحِدَةٌ بِالْخُلْعِ وَوَاحِدَةٌ بِالتَّعْلِيقِ ، فَإِنْ قَيَّدَ بِاثْنَتَيْنِ لَزِمَهُ ثَلَاثٌ وَاحِدَةٌ بِالْخُلْعِ وَاثْنَتَانِ بِالتَّعْلِيقِ قَالَهُ اللَّخْمِيُّ ، وَأَنْكَرَهُ ابْنُ رُشْدٍ .( 7 / 410 )
