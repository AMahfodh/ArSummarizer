الْيَوْمَ وَبَقِيَّتُهَا غَدًا ؛ فَوَاحِدَةٌ ) لِأَنَّهُ يَقَعُ بِالْبَعْضِ طَلْقَةٌ ؛ فَلَا يَبْقَى لَهَا بَقِيَّةٌ تَقَعُ غَدًا ، كَقَوْلِهِ أَنْتِ طَالِقٌ بَعْضَ طَلْقَةٍ الْيَوْمَ وَبَقِيَّةَ الطَّلْقَةِ غَدًا ( وَ ) إنْ قَالَ ( أَنْتِ طَالِقٌ مِنْ الْيَوْمِ إلَى سَنَةٍ ؛ طَلُقَتْ فِي الْحَالِ مَا لَمْ يَرُدَّ ) عَقْدَ الصِّفَةِ مِنْ الْيَوْمِ ، فَإِنْ قَالَ أَرَدْت ( أَنَّ عَقْدَ الصِّفَةِ مِنْ الْيَوْمِ ، وَأَنَّ وُقُوعَهُ بَعْدَ سَنَةٍ فَلَا يَقَعُ ) الطَّلَاقُ ( إلَّا بَعْدَهَا ) أَيْ السَّنَةِ عَمَلًا بِنِيَّتِهِ ، وَاللَّفْظُ يَحْتَمِلُهُ ، وَإِنْ قَالَ أَرَدْتُ تَكْرِيرَ طَلَاقِهَا مِنْ حِينِ تَلَفَّظْت إلَى سَنَةٍ ؛ طَلُقَتْ فِي الْحَالِ ثَلَاثًا إنْ كَانَتْ مَدْخُولًا بِهَا ، وَإِلَّا بَانَتْ بِالْأُولَى ، وَلَمْ يَلْحَقْهَا مَا بَعْدَهَا ( وَ ) أَنْتِ ( طَالِقٌ إلَى شَهْرٍ أَوْ ) أَنْتِ طَالِقٌ إلَى ( حَوْلٍ أَوْ ) أَنْتِ طَالِقٌ إلَى ( الشَّهْرِ أَوْ الْحَوْلِ وَنَحْوِهِ ) كَأَنْتِ طَالِقٌ إلَى أُسْبُوعٍ أَوْ الْأُسْبُوعِ ( يَقَعُ ) الطَّلَاقُ ( بِمُضِيِّهِ ) أَيْ : الشَّهْرِ أَوْ الْحَوْلِ وَنَحْوِهِ ؛ رُوِيَ نَحْوُهُ عَنْ ابْنِ عَبَّاسٍ وَأَبِي ذَرٍّ ، وَلِأَنَّهُ يَحْتَمِلُ أَنْ يَكُونَ تَوْقِيتًا لِإِيقَاعِهِ كَقَوْلِهِ أَنَا خَارِجٌ إلَى سَنَةٍ - أَيْ : بَعْدَهَا - فَإِذَا احْتَمَلَ الْأَمْرَيْنِ لَمْ يَقَعْ الطَّلَاقُ بِالشَّكِّ ؛ وَقَدْ تَرَجَّحَ هَذَا الِاحْتِمَالُ بِأَنَّهُ جَعَلَ لِلطَّلَاقِ غَايَةً ، وَلَا غَايَةَ لِآخِرِهِ ، بَلْ لِأَوَّلِهِ ( إلَّا أَنْ يَنْوِيَ وُقُوعَهُ إذَنْ ) أَيْ حِينَ التَّكَلُّمِ بِهِ ( فَيَقَعُ ) الطَّلَاقُ فِي الْحَالِ ( كَ ) قَوْلِهِ أَنْتِ طَالِقٌ ( بَعْدَ مَكَّةَ أَوْ إلَيْهَا ) أَيْ : مَكَّةَ ( وَلَمْ يَنْوِ بُلُوغَهَا ) فَيَقَعُ فِي الْحَالِ ( وَ ) إنْ قَالَ لَهَا ( أَنْتِ طَالِقٌ فِي أَوَّلِ الشَّهْرِ أَوْ فِي غُرَّتِهِ أَوْ ) قَالَ أَنْتِ طَالِقٌ غُرَّتَهُ ( أَوْ ) رَأْسِهِ أَوْ اسْتِقْبَالِهِ أَوْ مَجِيئِهِ إلَى هِلَالِهِ فَتَطْلُقُ ( بِ ) مُجَرَّدِ ( دُخُولِهِ ) أَيْ : بِغُرُوبِ آخِرِ يَوْمٍ مِنْ الَّذِي قَبْلَهُ ، وَلَا يُقْبَلُ قَوْلُهُ أَرَدْت آخِرَهُ أَوْ وَسَطَهُ وَنَحْوَهُ ظَاهِرًا وَلَا بَاطِنًا( 16 / 121 )
