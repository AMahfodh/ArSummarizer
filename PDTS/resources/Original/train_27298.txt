( وَيَبِيعُ ) مُرْتَهِنٌ ( فَضْلَ لَبَنٍ ) مَرْهُونٍ ( بِإِذْنِ ) رَاهِنٍ ؛ لِأَنَّهُ مِلْكُهُ ( وَإِلَّا ) يَأْذَنْ لِامْتِنَاعِهِ أَوْ غَيْبَتِهِ ( فَحَاكِمٌ ) لِقِيَامِهِ مَقَامَهُ ( وَيَرْجِعُ ) مُرْتَهِنٌ ( بِفَضْلِ نَفَقَتِهِ ) عَنْ رُكُوبٍ وَحَلْبٍ وَاسْتِرْضَاعٍ ( عَلَى رَاهِنٍ ) بِنِيَّةِ رُجُوعٍ وَظَاهِرُهُ وَإِنْ لَمْ يَرْجِعْ فِي غَيْرِهَا( 5 / 242 )
