( وَمَنْ غَصَبَ أَرْضًا وَغِرَاسًا مَنْقُولًا مِنْ ) مَالِكٍ ( وَاحِدٍ فَغَرَسَهُ ) أَيْ : الْغِرَاسَ الْمَغْصُوبَ ( فِيهَا ) أَيْ : الْأَرْضِ الْمَغْصُوبَةِ ( لَمْ يَمْلِكْ ) الْغَاصِبُ ( قَلْعَهُ ) ؛ لِأَنَّ مَالِكَهُمَا وَاحِدٌ وَلَا يَتَصَرَّفُ غَيْرُهُ فِي مِلْكِهِ بِلَا إذْنِهِ ( وَعَلَيْهِ ) أَيْ : الْغَاصِبِ ( إنْ فَعَلَ ) أَيْ : قَلَعَ الْغِرَاسَ بِغَيْرِ إذْنِ مَالِكٍ : تَسْوِيَتُهَا وَأَرْشُ نَقْصِهَا وَأَرْشُ نَقْصِ غِرَاسٍ لِتَعَدِّيهِ بِهِ ( أَوْ طَلَبَهُ ) أَيْ : الْقَلْعَ ( رَبُّهُمَا ) أَيْ : رَبُّ الْأَرْضِ وَالْغِرَاسِ ( لِغَرَضٍ صَحِيحٍ ) بِأَنْ كَانَ لَا يَنْتُجُ مِثْلُهُ فِي تِلْكَ الْأَرْضِ مَثَلًا ( تَسْوِيَتُهَا ) أَيْ : الْأَرْضِ ( وَ ) أَرْشُ ( نَقْصِهَا وَ ) أَرْشُ ( نَقْصِ غِرَاسٍ ) لِتَعَدِّيهِ بِهِ .
