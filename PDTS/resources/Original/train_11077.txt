وَدَوَاخِلِهِ وَخَوَارِجِهِ وَشُعُورِهِ قَالَهُ أَحْمَدُ وَكَرِهَ أَنْ يَأْخُذَ الْمَاءَ ثُمَّ يَصُبُّهُ ثُمَّ يَغْسِلُ وَجْهَهُ وَقَالَ هَذَا مَسْحٌ وَلَيْسَ بِغَسْلٍ ( وَالْخَفِيفُ ) مِنْ شُعُورِ الْوَجْهِ كُلِّهَا وَهُوَ الَّذِي يَصِفُ الْبَشَرَةَ ( يَجِبُ غَسْلُهُ وَ ) غَسْلُ ( مَا تَحْتَهُ ) ؛ لِأَنَّ الَّذِي لَا يَسْتُرُهُ شَعْرُهُ يُشْبِهُ مَا لَا شَعْرَ عَلَيْهِ وَيَجِبُ غَسْلُ الشَّعْرِ تَبَعًا لِلْمَحَلِّ فَإِنْ كَانَ فِي شَعْرِهِ كَثِيفٌ وَخَفِيفٌ فَلِكُلٍّ حُكْمُهُ ( وَتَخْلِيلُ اللِّحْيَةِ عِنْدَ غَسْلِهَا ) لِحَدِيثِ عُثْمَانَ السَّابِقِ ( وَإِنْ شَاءَ إذَا مَسَحَ رَأْسَهُ نَصًّا ) .( 1 / 284 )
