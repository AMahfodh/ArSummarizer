( وَإِنْ مَاتَ ) مَنْ أَسْلَمَ ( قَبْلَ اخْتِيَارٍ ) وَتَحْتَهُ أَكْثَرُ مِنْ أَرْبَعٍ ( فَعَلَى الْجَمِيعِ ) مِمَّنْ أَسْلَمَ مِنْ نِسَائِهِ ( أَطْوَلُ الْأَمْرَيْنِ مِنْ عِدَّةِ وَفَاةٍ أَوْ ) عِدَّةِ ( حَيَاةٍ ) ؛ لِأَنَّ كُلَّ وَاحِدَةٍ مِنْهُنَّ يُحْتَمَلُ أَنْ تَكُونَ مُخْتَارَةً أَوْ مُفَارَقَةً وَعِدَّةُ الْمُخْتَارَةِ عِدَّةُ الْوَفَاةِ ، وَعِدَّةُ الْمُفَارَقَةِ ثَلَاثَةُ قُرُوءٍ ، فَأَوْجَبْنَا أَطْوَلَهُمَا احْتِيَاطًا ، وَتَعْتَدُّ حَامِلٌ بِوَضْعِهِ ؛ لِأَنَّهُ لَا تَخْتَلِفُ عِدَّتُهَا ، وَصَغِيرَةٌ وَآيِسَةٌ بِعِدَّةِ وَفَاةٍ ؛ لِأَنَّهُ أَطْوَلُ مِنْ ثَلَاثَةِ أَشْهُرٍ ( وَيَرِثُ مِنْهُ ) أَيْ : الْمَيِّتِ ( أَرْبَعٌ ) مِمَّنْ أَسْلَمَ عَلَيْهِنَّ وَأَسْلَمْنَ ( بِقُرْعَةٍ ) كَمَا لَوْ مَاتَ عَنْ نِسْوَةٍ ؛ نِكَاحُ بَعْضِهِنَّ فَاسِدٌ وَجُهِلَ .( 15 / 207 )
