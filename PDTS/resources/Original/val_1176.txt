( وَلَا حَدَّ لِأَكْثَرِهِ ) أَيْ الطُّهْرِ بِالْإِجْمَاعِ فَقَدْ لَا تَحِيضُ الْمَرْأَةُ فِي عُمْرِهَا إلَّا مَرَّةً وَقَدْ لَا تَحِيضُ أَصْلًا ( وَلَوْ اسْتَمَرَّتْ عَادَةٌ ) لِامْرَأَةٍ أَوْ أَكْثَرَ ( تُخَالِفُ الْأَقَلَّ ) مِنْ الْحَيْضِ وَالطُّهْرِ ( وَالْأَكْثَرَ ) أَيْ أَوْ الْأَكْثَرَ مِنْ الْحَيْضِ ( لَمْ تُعْتَبَرْ ) تِلْكَ الْعَادَةُ لِأَنَّ بَحْثَ الْأَوَّلِينَ أَتَمُّ وَإِحَالَةَ مَا وَقَعَ عَلَى عِلَّةٍ أَقْرَبَ مِنْ خَرْقِ مَا مَضَتْ عَلَيْهِ الْعُصُورُ( 2 / 81 )
