ثَمَّ وَخَرَجَ بِمَا يَنْفُذُ فِيهِ قَضَاؤُنَا غَيْرُهُ كَأَنْ حَكَمُوا بِمَا يُخَالِفُ النَّصَّ أَوْ الْإِجْمَاعَ أَوْ الْقِيَاسَ الْجَلِيَّ فَلَا نَنْفُذُهُ ( ، وَلَوْ كَتَبُوا بِحُكْمٍ ) مِنْهُمْ إلَى حَاكِمِنَا ( جَازَ تَنْفِيذُهُ ) ؛ لِأَنَّهُ حُكْمٌ أُمْضِيَ وَالْحَاكِمُ بِهِ مِنْ أَهْلِهِ بَلْ لَوْ كَانَ الْحُكْمُ لِوَاحِدٍ مِنَّا عَلَى وَاحِدٍ مِنْهُمْ فَالْمُتَّجَهُ وُجُوبُ تَنْفِيذِهِ قَالَهُ الْأَذْرَعِيُّ ( وَكَذَا ) لَوْ كَتَبُوا ( بِسَمَاعِ بَيِّنَةٍ ) يَجُوزُ لَنَا الْحُكْمُ بِهَا لِتَعَلُّقِ الْحُكْمِ بِرَعَايَانَا ( وَيُسْتَحَبُّ ) لَنَا ( أَنْ لَا نُنَفِّذَ حُكْمَهُمْ ) اسْتِخْفَافًا بِهِمْ ( وَلْيُعْتَدَّ بِمَا اسْتَوْفَوْهُ ) بِالْبَلَدِ الَّذِي اسْتَوْلَوْا عَلَيْهِ ( مِنْ حُدُودٍ ) ، وَتَعَازِيرَ ( وَخَرَاجٍ ) وَزَكَاةٍ (( 19 / 374 )
