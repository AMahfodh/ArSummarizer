( وَإِنْ قَالَ : وَقَفْتُهُ ) أَيْ : الْعَبْدَ أَوْ الدَّارَ أَوْ الْبُسْتَانَ ، وَنَحْوَهُ ( سَنَةً ) لَمْ يَصِحَّ ( أَوْ ) قَالَ وَقَفْتُهُ ( إلَى سَنَةٍ ) لَمْ يَصِحَّ ( أَوْ ) قَالَ وَقَفْتُهُ ( إلَى يُومِ يَقْدُمُ الْحَاجِّ ، وَنَحْوَهُ ) أَيْ : نَحْوُ مَا ذُكِرَ مِمَّا فِيهِ تَأْقِيتُ الْوَقْفِ ( لَمْ يَصِحَّ ) الْوَقْفُ ؛ لِأَنَّ مُقْتَضَاهُ التَّأْبِيدُ وَالتَّأْقِيتُ يُنَافِيهِ ( وَهُوَ ) أَيْ : الْوَقْفُ الْمَذْكُورُ ( الْوَقْفُ الْمُؤَقَّتُ ، وَإِنْ قَالَ ) وَقَفْتُ دَارِي مَثَلًا ( عَلَى أَوْلَادِي سَنَةً أَوْ مُدَّةَ حَيَاتِي ثُمَّ عَلَى الْفُقَرَاءِ صَحَّ ) الْوَقْفُ لِاتِّصَالِهِ ابْتِدَاءً ، وَانْتِهَاءً ، وَكَذَا لَوْ وَقَفَهُ عَلَى وَلَدِهِ سَنَةً ثُمَّ عَلَى زَيْدٍ سَنَةً ثُمَّ عَلَى عَمْرٍو سَنَة ثُمَّ عَلَى الْمَسَاكِينِ .
