( فَمَنْ قَرَعَ حَلَفَ ) لِصَاحِبِهِ ( وَأَخَذَهَا ) لِتَرَجُّحِهِ بِالْقُرْعَةِ ( فَإِنْ كَانَ الْمُدَّعَى بِهِ عَبْدًا مُكَلَّفًا فَأَقَرَّ ) الْعَبْدُ بِالرِّقِّ ( لِأَحَدِهِمَا فَهُوَ ) أَيْ الْعَبْدُ ( لَهُ ) أَيْ لِلْمُقَرِّ لَهُ كَمَا لَوْ كَانَ الْمُدَّعَى وَاحِدًا وَأَقَرَّ لَهُ ( وَإِنْ صَدَّقَهُمَا ) الْعَبْدُ ( فَهُوَ لَهُمَا ) عَمَلًا بِإِقْرَارِهِ أَنَّهُ لَهُمَا .
