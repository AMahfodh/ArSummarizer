( وَإِنْ رَهَنَهَا ) أَيْ : الثَّمَرَةَ ( بِدَيْنٍ حَالٍ ، أَوْ ) رَهَنَهَا بِدَيْنٍ مُؤَجَّلٍ ، و ( شَرَطَ قَطْعَهَا عِنْدَ خَوْفِ اخْتِلَاطِهَا ) بِأُخْرَى ( جَازَ ) ؛ لِأَنَّهُ لَا غَرَرَ فِيهِ ( فَإِنْ لَمْ يَقْطَعْهَا ) أَيْ : الثَّمَرَةَ ( حَتَّى اخْتَلَطَتْ ) بِغَيْرِهَا ( لَمْ يَبْطُلْ الرَّهْنُ ) ؛ لِأَنَّهُ وَقَعَ صَحِيحًا ( فَإِنْ سَمَحَ الرَّاهِنُ بِبَيْعِ الْجَمِيعِ ) مِنْ الثَّمَرَةِ الْمَرْهُونَةِ وَمَا اخْتَلَطَتْ بِهِ ( عَلَى أَنَّهُ رَهْنٌ ) جَازَ ؛ لِأَنَّهُ كَزِيَادَةِ الرَّهْنِ ( أَوْ اتَّفَقَا ) أَيْ : الرَّاهِنُ وَالْمُرْتَهِنُ ( عَلَى ) بَيْعِ ( قَدْرٍ مِنْهُ جَازَ ) ؛ لِأَنَّ الْحَقَّ لَا يَعْدُوهُمَا ( وَإِنْ اخْتَلَفَا أَوْ تَشَاحَّا ف ) يُقَدَّمُ ( قَوْلُ الرَّاهِنِ مَعَ يَمِينِهِ ) ؛ لِأَنَّهُ مُنْكِرٌ .( 10 / 86 )
