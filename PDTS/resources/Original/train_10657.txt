( وَ ) مَنْ أَتْلَفَ ( مَا فِيهِ ) ؛ أَيْ : الْإِنْسَانِ مِنْهُ ( شَيْئَانِ ؛ فَفِيهِمَا الدِّيَةُ وَفِي أَحَدِهِمَا نِصْفُهَا ) نَصَّ عَلَيْهِ ( كَعَيْنَيْنِ وَلَوْ مَعَ حَوَلٍ أَوْ عَمَشٍ ) وَسَوَاءٌ الصَّغِيرَتَانِ وَالْكَبِيرَتَانِ ؛ لِعُمُومِ حَدِيثِ عَمْرِو بْنِ حَزْمٍ ( وَ ) مَعَ ( بَيَاضٍ ) بِالْعَيْنَيْنِ أَوْ إحْدَاهُمَا ( بِنَقْصِ الْبَصَرِ تَنْقُصُ الدِّيَةُ بِقَدْرِهِ ) ؛ أَيْ : نَقْصِ الْبَصَرِ ( وَكَأُذُنَيْنِ ) قَضَى بِهِ عُمَرُ وَعَلِيٌّ ( وَشَفَتَيْنِ ) إذَا اسْتَوْعَبَتَا ، وَفِي الْبَعْضِ بِقِسْطِهِ مِنْ دِيَتِهِمَا تُقَدَّرُ بِالْأَجْزَاءِ ( وَكَلَحْيَيْنِ ) وَهُمَا الْعَظْمَانِ اللَّذَانِ فِيهِمَا الْأَسْنَانُ ؛ لِأَنَّ فِيهِمَا نَفْعًا وَجَمَالًا وَلَيْسَ فِي الْبَدَنِ مِثْلُهُمَا ( وَكَثُنْدُوَتَيْ رَجُلٍ ) بِالثَّاءِ الْمُثَلَّثَةِ وَهُمَا بِمَنْزِلَةِ ثَدْيَيْ الْمَرْأَةِ ، فَإِنْ ضَمَمْت الْأَوَّلَ هُمِزَتْ ، وَإِنْ فَتَحْته لَمْ تَهْمِزْ ، فَالْوَاحِدَةُ مَعَ الْهَمْزَةِ فَعْلَلَةٌ ، وَمَعَ الْفَتْحِ فَعُلْوَةٌ ( وَكَأُنْثَيَيْهِ ) ؛ أَيْ : الرَّجُلِ فَفِيهِمَا الدِّيَةُ ، وَفِي إحْدَاهُمَا نِصْفُهُمَا ( وَكَثَدْيِ أُنْثَى وَإِسْكَتَيْهَا ) بِكَسْرِ الْهَمْزَةِ وَفَتْحِهَا ( وَهُمَا شَفَرَاهَا ) ؛ أَيْ : حَافَّتَا فَرْجِهَا ؛ فَفِيهِمَا الدِّيَةُ ؛ لِأَنَّ فِيهِمَا نَفْعًا وَجَمَالًا ، وَلَيْسَ فِي الْبَدَنِ غَيْرُهُمَا مِنْ جِنْسِهِمَا ، وَإِنْ جَنَى عَلَيْهِمَا فَأَشَلَّهُمَا ؛ فَالدِّيَةُ كَمَا لَوْ أَشَلَّ الشَّفَتَيْنِ ، وَسَوَاءٌ الرَّتْقَاءُ ، وَغَيْرُهُمَا .
