وَلَا ضَرَرَ فِيهِ عَلَى السَّفِيهِ ( وَإِلَّا ) بِأَنْ أَطْلَقَ أَوْ أَضَافَ الْمَالَ إلَيْهِ ( وَقَعَ رَجْعِيًّا ) كَاخْتِلَاعِ السَّفِيهَةِ نَفْسَهَا ( وَلَهُمَا ) أَيْ لِلزَّوْجَيْنِ مَعًا ( تَوْكِيلُ ذِمِّيٍّ ) وَإِنْ كَانَا مُسْلِمَيْنِ لِأَنَّهُ قَدْ يُخَالِعُ الْمُسْلِمَةَ وَيُطَلِّقُهَا بِدَلِيلِ أَنَّهَا لَوْ أَسْلَمَتْ وَتَخَلَّفَ فَخَالَعَهَا فِي الْعِدَّةِ ثُمَّ أَسْلَمَ حُكِمَ بِصِحَّةِ الْخَلْعِ وَلَا يَتَقَيَّدُ ذَلِكَ بِالذِّمِّيِّ بَلْ الْحَرْبِيُّ كَذَلِكَ كَمَا صَرَّحَ بِهِ الرُّويَانِيُّ وَعَبَّرَ الْمَاوَرْدِيُّ وَغَيْرُهُ بِالْكَافِرِ ( وَلَوْ وَكَّلَا رَجُلًا فِي تَوَلِّي الطَّرَفَيْنِ ) لِلْخُلْعِ ( لَمْ يَتَوَلَّهُمَا ) كَمَا فِي سَائِرِ الْعُقُودِ ( وَلَهُ أَنْ يَتَوَلَّى طَرَفًا ) مِنْهُمَا مَعَ الْآخَرِ أَوْ وَكِيلِهِ
