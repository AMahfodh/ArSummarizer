( فَإِذَا ظَهَرْنَا ) أَيْ غَلَبْنَا ( عَلَيْهِمْ ) بِعَوْنِهِ تَعَالَى ( فَمَنْ وَجَدَ ) مِنَّا ( مِلْكَهُ ) فِي يَدِ الْغَانِمِينَ بَعْدَ الِاسْتِيلَاءِ ( أَخَذَهُ مُطْلَقًا ) أَيْ سَوَاءٌ كَانَ مِثْلِيًّا أَوْ قِيَمِيًّا ( قَبْلَ الْقِسْمَةِ ) أَيْ قِسْمَةِ الْإِمَامِ الْغَنَائِمَ ( مَجَّانًا ) أَيْ أَخَذَهُ بِلَا شَيْءٍ ( وَبَعْدَهَا ) أَيْ لَوْ وَجَدَ مِلْكَهُ بَعْدَ قِسْمَةِ الْإِمَامِ الْغَنَائِمَ ( إنْ كَانَ ) مَا وَجَدَهُ ( مِثْلِيًّا ) الْمِثْلِيُّ يَدْخُلُ تَحْتَ الْكَيْلِ وَالْوَزْنِ وَالْعَدَدِ كَمَا سَيَجِيءُ إنْ شَاءَ اللَّهُ تَعَالَى ( لَا يَأْخُذُهُ ) لِأَنَّهُ لَا فَائِدَةَ فِي أَخْذِهِ لِوُجُودِ مِثْلِهِ .
