( قَوْلُهُ : فَإِنْ لَمْ يُفَوِّتْ عَلَيْهِ شَيْئًا ) أَيْ فَإِنْ لَمْ يُفَوِّتْ عَلَى الْأَوَّلِ شَيْئًا مِمَّا اسْتَأْجَرَهُ عَلَيْهِ ( قَوْلُهُ : فَإِنَّهُ يَسْقُطُ مِنْ كِرَائِهِ ) أَيْ لِلْأَوَّلِ وَقَوْلُهُ : بِقَدْرِ قِيمَةِ مَا عَمِلَ أَيْ لِلثَّانِي ( قَوْلُهُ : بِرَاعٍ آخَرَ لِرَعْيِهَا ) أَيْ لِيَرْعَاهَا مَعَ رَاعِي الْأُمَّهَاتِ لَا مُنْفَرِدًا لِمَا فِيهِ مِنْ تَعْذِيبِ الْحَيَوَانِ ( قَوْلُهُ : فَإِنَّهُ يُعْمَلُ بِهِ ) أَيْ ؛ لِأَنَّ الْعُرْفَ يُقَيِّدُ مَا أَطْلَقَاهُ وَيُفَسِّرُ مَا أَجْمَلَاهُ وَيَكُونُ شَاهِدًا لِمَنْ ادَّعَاهُ .( 15 / 475 )
