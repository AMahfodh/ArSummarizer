لِحَدِيثِ مُسْلِمٍ { لَا يَنْكِحُ الْمُحْرِمُ وَلَا يُنْكِحُ وَلَا - يَخْطُبُ } ( وَتَقَدَّمَ فِي مَحْظُورَاتِ الْإِحْرَامِ ) بِأَوْسَعَ مِنْ هَذَا ( وَلَا يَحِلُّ لِمُسْلِمَةٍ نِكَاحَ كَافِرٍ بِحَالٍ ) حَتَّى يُسْلِمَ لِقَوْلِهِ تَعَالَى { وَلَا تُنْكِحُوا الْمُشْرِكِينَ حَتَّى يُؤْمِنُوا } وَقَوْلِهِ { فَإِنْ عَلِمْتُمُوهُنَّ مُؤْمِنَاتٍ فَلَا تَرْجِعُوهُنَّ إلَى الْكُفَّارِ لَا هُنَّ حِلٌّ لَهُمْ وَلَا هُمْ يَحِلُّونَ لَهُنَّ } .
