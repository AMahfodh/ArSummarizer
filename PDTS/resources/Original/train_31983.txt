وَإِنَّمَا قُدِّمَ الْأَقْرَأُ جَوْدَةً عَلَى الْأَكْثَرِ قُرْآنًا لِأَنَّهُ أَعْظَمَ أَجْرًا ، لِحَدِيثِ { مَنْ قَرَأَ الْقُرْآنَ فَأَعْرَبَهُ فَلَهُ بِكُلِّ حَرْفٍ عَشْرُ حَسَنَاتٍ وَمَنْ قَرَأَهُ وَلَحَنَ فِيهِ فَلَهُ بِكُلِّ حَرْفٍ حَسَنَةٌ } رَوَاهُ التِّرْمِذِيُّ وَقَالَ : حَسَنٌ صَحِيح وَقَالَ أَبُو بَكْرٍ وَعُمَرُ : رَضِيَ اللَّهُ عَنْهُمَا إعْرَابُ الْقُرْآنِ أَحَبُّ إلَيْنَا مِنْ حِفْظِ بَعْضِ حُرُوفِهِ ( ثُمَّ ) مَعَ الِاسْتِوَاءِ فِي الْجَوْدَةِ يُقَدَّمُ ( الْأَكْثَرُ قُرْآنًا الْأَفْقَهُ ) لِجَمْعِهِ الْفَضِيلَتَيْنِ ( ثُمَّ ) يَلِيهِ ( الْأَكْثَرُ قُرْآنًا الْفَقِيهُ ثُمَّ ) يَلِيهِ ( قَارِئٌ ) أَيْ : حَافِظٌ لِمَا يَجِبُ فِي الصَّلَاةِ ( أَفْقَهُ ثُمَّ ) يَلِيهِ ( قَارِئٌ فَقِيهٌ ثُمَّ قَارِئٌ عَالَمٌ فِقْهَ صَلَاتِهِ مِنْ شُرُوطِهَا وَأَرْكَانِهَا ) وَوَاجِبَاتِهَا وَمُبْطِلَاتِهَا وَنَحْوِهَا ( ثُمَّ قَارِئٌ لَا يَعْلَمُهُ ) أَيْ : فِقْهَ صَلَاتِهِ ، بَلْ يَأْتِي بِهَا عَادَةً .
