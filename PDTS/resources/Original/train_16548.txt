( وَإِنْ كَانَ ) الْحَيَوَانُ ( مَأْكُولًا ) وَخَاطَ جُرْحَهُ بِالْخَيْطِ الْمَغْصُوبِ وَهُوَ مِلْكٌ ( لِلْغَاصِبِ ذُبِحَ ) الْحَيَوَانُ وَلَوْ نَقَصَتْ بِهِ قِيمَتُهُ أَكْثَرَ مِنْ ثَمَنِ الْخَيْطِ أَوْ لَمْ يَكُنْ مُعَدًّا لِأَكْلٍ ، كَالْخَيْلِ ( وَلَزِمَهُ ) أَيْ الْغَاصِبَ ( رَدُّهُ ) أَيْ الْخَيْطِ لِرَبِّهِ لِأَنَّهُ مُتَمَكِّنٌ مِنْ رَدِّهِ بِذَبْحِ الْحَيَوَانِ وَالِانْتِفَاعِ بِلَحْمِهِ وَلَا أَثَرَ لِتَضَرُّرِهِ بِذَلِكَ لِتَعَدِّيهِ ( وَإِنْ كَانَ ) الْحَيَوَانُ الَّذِي خِيطَ جُرْحُهُ مُحْتَرَمًا ( غَيْرَ مَأْكُولٍ رَدَّ ) الْغَاصِبُ ( قِيمَةَ الْخَيْطِ ) لِأَنَّ حُرْمَةَ الْحَيَوَانِ آكَدُ كَمَا سَبَقَ .
