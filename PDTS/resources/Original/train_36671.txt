( وَ ) مِنْ ( عَمَلٍ ) وَهُوَ جَمِيعُ أَعْمَالِ الطَّاعَةِ ( قَبْلَ بُلُوغِهِمْ لِ ) كَيْ ( يَأْتِيَ عَلَيْهِمْ الْبُلُوغُ وَقَدْ تَمَكَّنَ ) أَيْ ثَبَتَ وَرَسَخَ ( ذَلِكَ ) أَيْ الَّذِي فَرَضَهُ اللَّهُ عَلَى الْعِبَادِ ( مِنْ ) بِمَعْنَى فِي ( قُلُوبِهِمْ وَسَكَنَتْ ) أَيْ مَالَتْ ( إلَيْهِ أَنْفُسُهُمْ وَأَنِسَتْ ) أَيْ اسْتَأْنَسَتْ ( بِمَا ) أَيْ بِاَلَّذِي ( يَعْلَمُونَ بِهِ مِنْ ذَلِكَ ) الَّذِي فَرَضَ( 1 / 108 )
