( وَالْأَفْضَلُ أَنْ يَجْمَعَهُمْ وَيُصَلِّيَ عَلَى الْمُسْلِمِينَ أَوْ غَيْرِ الشُّهَدَاءِ مِنْهُمْ ) أَوْ غَيْرِ السَّقْطِ وَيَقُولُ فِي الْأُولَى اللَّهُمَّ اغْفِرْ لِلْمُسْلِمِينَ مِنْهُمْ ( وَإِنْ أَفْرَدَ كُلًّا ) مِنْهُمْ بِصَلَاةٍ ( وَنَوَاهُ إنْ كَانَ مُسْلِمًا ) أَوْ غَيْرَ شَهِيدٍ أَوْ غَيْرَ سَقْطٍ ( جَازَ ) وَيُغْتَفَرُ التَّرَدُّدُ فِي النِّيَّةِ لِلضَّرُورَةِ كَمَنْ نَسِيَ صَلَاةً مِنْ الْخَمْسِ ( وَيَقُولُ ) فِي الْأَوْلَى (( 4 / 272 )
