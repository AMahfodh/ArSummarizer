( وَسُبُعَانِ ) يُعْتَقَانِ ( بِإِعْتَاقِ الْمُتَوَفَّى ) قَالَ فِي الْمُبْدِعِ : وَفِي التَّشْبِيهِ نَظَرٌ مِنْ حَيْثُ إنَّ الْكَسْبَ يَزِيدُ بِهِ مِلْكُ السَّيِّدِ وَذَلِكَ يَقْتَضِي الزِّيَادَةَ فِي الْعِتْقِ ، وَالْمَهْرُ يُنْقِصُهُ وَذَلِكَ يَقْتَضِي نُقْصَانَ الْعِتْقِ ، وَنَقَلَهُ الْحَارِثِيُّ عَنْ بَعْضِ مُتَأَخِّرِي الْأَصْحَابِ وَقَالَ هُوَ كَمَا قَالَ ( وَلَوْ وَهَبَهَا ) الْمَرِيضُ ( لِمَرِيضٍ آخَرَ لَا مَالَ لَهُ فَوَهَبَهَا الثَّانِي لِلْأَوَّلِ ) وَمَاتَا ( صَحَّتْ هِبَةُ الْأَوَّلِ فِي شَيْءٍ وَعَادَ إلَيْهِ بِالْهِبَةِ الثَّانِيَةِ ثُلُثُهُ بَقِيَ لِوَرَثَةِ الْآخَرِ ثُلُثَا شَيْءٍ وَلِلْأَوَّلِ ) أَيْ : وَرَثَتِهِ ( شَيْئَانِ ) فَاضْرِبْهَا فِي ثَلَاثَةٍ لِيَزُولَ الْكَسْرُ تَكُنْ ثَمَانِيَةَ أَشْيَاءَ تَعْدِلُ الْأَمَةَ الْمَوْهُوبَةَ ( فَلَهُمْ ) أَيْ : لِوَرَثَةِ الْأَوَّلِ ( ثَلَاثَةُ أَرْبَاعِهَا ) سِتَّةٌ .
