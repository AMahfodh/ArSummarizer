( فَصْلٌ ) فِي فُرُوعٍ تَتَعَلَّقُ بِالسَّرِقَةِ ( قَوْلُهُ فِي فُرُوعٍ ) إلَى قَوْلِهِ قَالَ شَيْخُنَا فِي النِّهَايَةِ إلَّا قَوْلَهُ أَوْ الْمُسْتَحِقُّ لِمَنْفَعَتِهِ وَقَوْلُهُ وَإِلَّا إلَى لَمْ يُقْطَعْ ( قَوْلُهُ بِذِكْرِ ضِدِّهَا ) أَيْ السَّرِقَةِ وَكَذَا ضَمِيرُ مَنْعِهَا ( قَوْلُهُ لِقَطْعِهِ ) مُتَعَلِّقٌ بِمَنْعِهَا وَقَوْلُهُ وَعَدَمِهِ أَيْ عَدَمِ الْمَنْعِ ( قَوْلُهُ وَالْحِرْزِ ) عُطِفَ عَلَى السَّارِقِ ( قَوْلُهُ وَالْأَحْوَالِ ) كَمَا لَوْ أَخْرَجَ مِنْ بَيْتِ دَارٍ إلَى صَحْنِهَا حَيْثُ يُفَرَّقُ فِيهِ بَيْنَ كَوْنِ الْبَابَيْنِ مَفْتُوحِينَ أَوْ مُغْلَقَيْنِ أَوْ غَيْرِ ذَلِكَ عَلَى مَا يَأْتِي ا ه ع ش ( قَوْلُ الْمَتْنِ يُقْطَعُ مُؤَجِّرُ الْحِرْزِ ) أَيْ : إجَارَةً صَحِيحَةً بِخِلَافِ مَا لَوْ كَانَتْ فَاسِدَةً فَلَا قَطْعَ مُغْنِي و ع ش ( قَوْلُهُ بِسَرِقَتِهِ ) إلَى قَوْلِهِ أَيْ بِخِلَافٍ فِي الْمُغْنِي إلَّا قَوْلَهُ فِيمَا نُهِيَ عَنْهُ ( قَوْلُهُ لِلْمُسْتَأْجِرِ ) مُتَعَلِّقٌ بِانْتِقَالٍ إلَخْ .
