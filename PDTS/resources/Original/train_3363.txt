( وَيَصِحُّ أَنْ يَسْتَأْجِرَ ) الْأَرْمَدُ ( كَحَّالًا لِيُكَحِّلَ عَيْنَيْهِ ) لِأَنَّهُ عَمَلٌ جَائِزٌ يُمْكِنُ تَسْلِيمُهُ ( وَيُقَدَّرُ ذَلِكَ بِالْمُدَّةِ ) دُونَ الْبُرْءِ لِأَنَّهُ غَيْرُ مَعْلُومٍ ( وَيَحْتَاجُ إلَى بَيَانِ عَدَدِ مَا يُكَحِّلُهُ كُلَّ يَوْمٍ ) فَيَقُولُ ( مَرَّةً أَوْ مَرَّتَيْنِ ، فَإِنْ كَحَّلَهُ فِي الْمُدَّةِ فَلَمْ يَبْرَأْ اسْتَحَقَّ الْأُجْرَةَ ) لِأَنَّهُ وَفَّى بِالْعَمَلِ .( 12 / 128 )
