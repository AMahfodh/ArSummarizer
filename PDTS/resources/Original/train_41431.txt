( وَيَحْرُمُ ) أَنْ يُيَمِّمَ ( بِدُونِهِ ) أَيْ دُونَ الْحَائِلِ ( لِغَيْرِ مَحْرَمٍ ) لِمَا فِيهِ مِنْ الْمَسِّ ( وَرَجُلٌ أَوْلَى بِتَيَمُّمِ خُنْثَى مُشْكِلٍ ) مِنْ امْرَأَةٍ ، إذَا مَاتَ الْخُنْثَى بَيْنَ رِجَالٍ وَنِسَاءٍ لِأَنَّ الصِّنْفَيْنِ قَدْ اشْتَرَكَا فِي الْمَحْذُورِ وَامْتَازَ الرَّجُلُ بِفَضِيلَةِ الذُّكُورِيَّةِ لَكِنْ إذَا مَاتَتْ الْمَرْأَةُ مَعَ الرِّجَالِ وَفِيهِمْ صَبِيٌّ لَا شَهْوَةَ لَهُ عَلَّمُوهُ الْغُسْلَ وَبَاشَرَهُ نَصَّ عَلَيْهِ كَذَا الرَّجُلُ يَمُوتُ مَعَ نِسْوَةٍ فِيهِنَّ صَغِيرَةٌ تُطِيقُ الْغُسْلَ ذَكَرَهُ فِي شَرْحِ الْهِدَايَةِ قُلْتُ : وَكَذَا الْخُنْثَى يَمُوتُ مَعَ رِجَالٍ أَوْ نِسْوَةٍ فِيهِنَّ صَغِيرٌ أَوْ صَغِيرَةٌ تُطِيقُهُ ( وَإِنْ كَانَتْ لَهُ ) أَيْ لِلْخُنْثَى الْمُشْكِلِ ( أَمَةٌ غَسَّلَتْهُ ) لِأَنَّهُ إنْ كَانَ أُنْثَى فَلَا كَلَامَ وَإِنْ كَانَ ذَكَرًا فَلِأَمَتِهِ أَنْ تُغَسِّلَهُ .( 4 / 297 )
