( قَوْلُهُ وَلَعَلَّهُ ) أَيْ الْمُصَنِّفَ ( أَغْفَلَهُ ) أَيْ تَرَكَ التَّنْبِيهَ عَلَى ذَلِكَ الْبِنَاءِ ( هُنَا ) أَيْ فِي الْمِنْهَاجِ وَ ( قَوْلُهُ لِلْعِلْمِ بِهِ ) أَيْ بِالْبِنَاءِ الْمَذْكُورِ ( مِمَّا قَدَّمَهُ ) أَيْ هُنَا وَفِي الرَّوْضَةِ وَ ( قَوْلُهُ أَنَّ الْعِبْرَةَ إلَخْ ) بَيَانٌ لِمَا قَدَّمَهُ ( قَوْلُهُ الْمُعْتَمَدَ إلَخْ ) أَيْ مِنْ أَنَّ الْعِبْرَةَ بِبَلَدِ الْمُؤَدَّى عَنْهُ فَيُخْرِجُ إلَخْ ( قَوْلُهُ وَهُوَ فَاسِدٌ مَعْنًى( 12 / 474 )
