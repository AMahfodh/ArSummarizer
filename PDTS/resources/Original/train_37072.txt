( وَمَنْ مَلَكَ وَضْعَ خَشَبٍ عَلَى حَائِطٍ ، فَزَالَ الْخَشَبُ ) عَنْ الْحَائِطِ ، ( أَوْ ) زَالَ ( الْحَائِطُ ، ثُمَّ أُعِيدَ ؛ فَلِرَبِّ الْخَشَبِ إعَادَتُهُ ) ؛ أَيْ : الْخَشَبِ ( بِشَرْطِهِ ) بِأَنْ لَا يُمْكِنَ تَسْقِيفٌ إلَّا بِهِ بِلَا ضَرَرٍ ؛ لِأَنَّ السَّبَبَ الْمُجَوِّزَ لِوَضْعِهِ مُسْتَمِرٌّ ، فَاسْتَمَرَّ اسْتِحْقَاقُ ذَلِكَ ، ( وَإِنْ خِيفَ سُقُوطُ الْحَائِطِ بِاسْتِمْرَارِهِ ) ؛ أَيْ : الْخَشَبِ ( عَلَيْهِ ) ؛ أَيْ : الْحَائِطِ بَعْدَ وَضْعِهِ ؛ ( لَزِمَهُ إزَالَتُهُ ) ؛ لِأَنَّهُ يَضُرُّ بِالْمَالِكِ ، وَ ( لَا ) تَلْزَمُهُ الْإِزَالَةُ ( إنْ اسْتَغْنَى رَبُّ الْخَشَبِ عَنْ إبْقَائِهِ ) عَنْ الْحَائِطِ .
