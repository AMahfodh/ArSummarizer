( وَإِنْ فُتِحَتْ ) قَلْعَةٌ جُوعِلَ مِنْهَا بِجَارِيَةٍ مِنْهُمْ ( صُلْحًا وَلَمْ يَشْتَرِطُوهَا ) أَيْ يَشْتَرِطْ الْمُسْلِمُونَ الْجَارِيَةَ عَلَى أَهْلِ الْقَلْعَةِ ( وَأَبَوْهَا ) أَيْ أَبَى أَهْلُ الْقَلْعَةِ الْجَارِيَةَ ( وَأَبَى ) مَجْعُولٌ لَهُ ( أَخْذُ الْقِيمَةِ ) عَنْهَا ( فُسِخَ ) الصُّلْحُ لِتَعَذُّرِ إمْضَائِهِ ، لِسَبْقِ حَقِّ صَاحِبِ الْجُعْلِ وَتَعَذُّرِ الْجَمْعِ بَيْنَهُ وَبَيْنَ الصُّلْحِ .
