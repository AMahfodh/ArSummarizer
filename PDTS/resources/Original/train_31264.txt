( وَ ) لَوْ حَلَفَ ( لَا يَدْخُلُ دَارًا فَأَدْخَلَهَا بَعْضُ جَسَدِهِ أَوْ دَخَلَ طَاقَ الْبَابِ ) مِنْهَا لَمْ يَحْنَثْ لِأَنَّهُ لَمْ يَدْخُلْهَا ( أَوْ ) حَلَفَ ( لَا يَشْرَبُ مَاءَ هَذَا الْإِنَاءِ فَشَرِبَ بَعْضَهُ ) لَمْ يَحْنَثْ لِأَنَّهُ لَمْ يَشْرَبْهُ ( أَوْ ) حَلَفَ ( لَا يَبِيعُ عَبْدَهُ وَلَا يَهَبَهُ فَبَاعَ ) بَعْضَهُ ( أَوْ وَهَبَ بَعْضَهُ لَمْ يَحْنَثْ ) وَكَذَا لَوْ بَاعَ الْبَعْضَ وَوَهَبَ الْبَعْضَ لِأَنَّهُ لَمْ يَبِعْهُ وَلَمْ يَهَبْهُ .
