( وَ ) رَجَعَ عَلَيْهَا ( فِي ) الْمَسْأَلَةِ ( الثَّانِيَةِ ) وَهِيَ مَا إذَا ارْتَدَّتْ بَعْدَ أَنْ أَسْقَطَتْ عَنْهُ صَدَاقَهَا ( بِبَدَلِ جَمِيعِهِ ) ؛ لِأَنَّ عَوْدَ نِصْفِ الصَّدَاقِ أَوْ كُلِّهِ إلَى الزَّوْجِ بِالطَّلَاقِ أَوْ الرِّدَّةِ ، وَهُمَا غَيْرُ الْجِهَةِ الْمُسْتَحَقِّ بِهَا الصَّدَاقُ أَوَّلًا ؛ فَأَشْبَهَ مَا لَوْ أَبْرَأَ إنْسَانٌ آخَرَ مِنْ دَيْنٍ ، ثُمَّ ثَبَتَ لَهُ عَلَيْهِ مِثْلُهُ مِنْ وَجْهٍ آخَرَ ( كَعَوْدِهِ ) أَيْ : الصَّدَاقِ ( إلَيْهِ ) أَيْ : الزَّوْجِ مِنْ زَوْجَتِهِ ( بِبَيْعٍ ) ثُمَّ يُطَلِّقُهَا أَوْ تَرْتَدُّ قَبْلَ الدُّخُولِ ؛ فَإِنَّهُ يَرْجِعُ عَلَيْهَا بِبَدَلِ النِّصْفِ فِي صُورَةِ تَنَصُّفِهِ ، أَوْ بِكُلِّهِ فِي صُورَةِ إسْقَاطِهِ ( أَوْ هِبَتِهَا الْعَيْنَ ) أَيْ : وَكَمَا لَوْ أَصْدَقَهَا عَيْنًا ، فَوَهَبَتْهَا ( لِأَجْنَبِيٍّ ثُمَّ وَهَبَهَا ) الْأَجْنَبِيُّ ( لَهُ ) أَيْ لِلزَّوْجِ ثُمَّ طَلَّقَهَا أَوْ ارْتَدَّتْ ، فَلَهُ الرُّجُوعُ بِبَدَلِ نِصْفِهَا أَوْ كُلِّهَا .( 15 / 267 )
