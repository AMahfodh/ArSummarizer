ثُمَّ ذَكَرَ قَسِيمَ قَوْلِهِ : وَإِنْ صَدَّقَاهُ بِقَوْلِهِ : ( وَإِنْ صَدَّقَهُ أَحَدُهُمَا ) أَيْ الِابْنَيْنِ ( فَنَصِيبُهُ ) وَحْدَهُ ( مُكَاتَبٌ ) مُؤَاخَذَةً لَهُ بِإِقْرَارِهِ ، وَاغْتُفِرَ التَّبْعِيضُ لِلضَّرُورَةِ ( وَنَصِيبُ الْمُكَذِّبِ قِنٌّ ) إذَا حَلَفَ عَلَى نَفْيِ الْعِلْمِ بِكِتَابَةِ أَبِيهِ اسْتِصْحَابًا لِأَصْلِ الرِّقِّ ، وَيَكُونُ نِصْفُ الْكَسْبِ لَهُ ، وَنِصْفُهُ لِلْمُكَاتَبِ يَصْرِفُهُ إلَى جِهَةِ النُّجُومِ ( فَإِنْ أَعْتَقَهُ الْمُصَدِّقُ ) أَيْ أَعْتَقَ نَصِيبَهُ ( فَالْمَذْهَبُ أَنَّهُ ) يَسْرِي الْعِتْقُ عَلَيْهِ إلَى نَصِيبِ الْمُكَذِّبِ ، وَ ( يُقَوَّمُ عَلَيْهِ إنْ كَانَ مُوسِرًا ) ؛ لِأَنَّ مُنْكِرَ الْكِتَابَةِ يَقُولُ : إنَّهُ رَقِيقٌ .
