( وَمَنْ ) ( غَابَ ) عَنْهَا ( زَوْجُهَا ) غَيْبَةً ظَاهِرُهَا السَّلَامَةُ كَتَاجِرٍ وَأَسِيرٍ عِنْدَ مَنْ لَيْسَتْ عَادَتُهُ الْقَتْلَ ، وَلَمْ يُعْلَمْ خَبَرُهُ أَهُوَ حَيٌّ أَوْ مَيِّتٌ ( وَتَضَرَّرَتْ بِتَرْكِ النِّكَاحِ ) مَعَ وُجُودِ النَّفَقَةِ عَلَيْهَا ( لَمْ تَفْسَخْ ) نِكَاحَهَا ( لِذَلِكَ ) أَيْ لِتَضَرُّرِهَا بِتَرْكِ الْوَطْءِ ؛ لِأَنَّهُ يُمْكِنُ أَنْ يَكُونَ لَهُ عُذْرٌ .( 15 / 391 )
