( وَإِنْ ) شُرِطَ ( مِنْ كِلَا الْجَانِبَيْنِ يَحْرُمُ ) بِأَنْ يَقُولَ إنْ سَبَقَ فَرَسُك أَعْطَيْتُك كَذَا وَإِنْ سَبَقَ فَرَسِي فَأَعْطِنِي كَذَا لِأَنَّهُ يَصِيرُ قِمَارًا وَالْقِمَارُ حَرَامٌ ( إلَّا أَنْ يَكُونَ بَيْنَهُمَا ) فَرَسٌ ( مُحَلِّلٌ كُفْءٌ لَهُمَا ) أَيْ لِفَرَسَيْهِمَا يُتَوَهَّمُ أَنَّهُ يَسْبِقُهُمَا ( إنْ سَبَقَهُمَا أَخَذَ ) الْجُعْلَ ( مِنْهُمَا وَإِنْ سَبَقَاهُ لَا يُعْطِيهِمَا ) شَيْئًا أَوْ بِالْعَكْسِ يَعْنِي شُرِطَ أَيُّهُمَا لَوْ سَبَقَاهُ يُعْطِيهِمَا وَلَوْ سَبَقَهُمَا لَا يَأْخُذُ شَيْئًا مِنْهُمَا كَمَا فِي التَّسْهِيلِ ( وَفِيمَا بَيْنَهُمَا أَيُّهُمَا سَبَقَ أَخَذَ ) الْمَالَ الْمَشْرُوطَ ( مِنْ الْآخَرِ ) لِأَنَّ بِالْمُحَلِّلِ خَرَجَ مِنْ أَنْ يَكُونَ قِمَارًا فَيَجُوزُ وَإِنْ لَمْ يَكُنْ الْفَرَسُ الْمُحَلِّلُ مِثْلَهُمَا لَمْ يَجُزْ لِأَنَّهُ لَا فَائِدَةَ فِي إدْخَالِهِ بَيْنَهُمَا فَلَمْ يَخْرُجْ حِينَئِذٍ مِنْ أَنْ يَكُونَ قِمَارًا .( 8 / 200 )
