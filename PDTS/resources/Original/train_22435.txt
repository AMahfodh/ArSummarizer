( وَإِنْ قَالَ الْمَعْزُولُ ) لِلْأَمِينِ ( أَعْطَيْتُك الْمَالَ ) أَيَّامَ قَضَائِي لِتَحْفَظَهُ ( لِفُلَانٍ فَقَالَ الْأَمِينُ بَلْ ) أَعْطَيْتنِيهِ لِأَحْفَظَهُ ( لِفُلَانٍ فَالْقَوْلُ قَوْلُ الْمَعْزُولِ ) لَكِنْ هَلْ يَغْرَمُ الْأَمِينُ لِمَنْ عَيَّنَهُ هُوَ قَدْرَ ذَلِكَ ، فِيهِ وَجْهَانِ فِي تَعْلِيقِ الْقَاضِي أَوْجَهُهُمَا الْمَنْعُ ( أَوْ قَالَ ) لَهُ الْأَمِينُ ( لَمْ تُعْطِنِي ) شَيْئًا ( بَلْ هُوَ لِفُلَانٍ فَالْقَوْلُ قَوْلُ الْأَمِينِ ) ؛ لِأَنَّ الْأَصْلَ عَدَمُ الْإِعْطَاءِ
