( قَوْلُهُ : مَعَ مَا فِي الْخَبَرِ إلَخْ ) أَيْ كَمَا بَيَّنَهُ فِي شَرْحِ الرَّوْضِ سم ( قَوْلُهُ : الْمُوجِبُ إلَخْ ) أَيْ الْعَمَلُ ( قَوْلُهُ : وَثَبَتَ إلَخْ ) عَطْفٌ عَلَى قُلْنَا إلَخْ قَوْلُ الْمَتْنِ ( أَوْ تَرْكُهُ سُنَّةً إلَخْ ) أَيْ فَلَهُ مُفَارَقَتُهُ لِيَأْتِيَ بِتِلْكَ السُّنَّةِ وَمَحَلُّ جَوَازِ الْقَطْعِ فِي غَيْرِ الْجُمُعَةِ أَمَّا فِي( 8 / 370 )
