( وَزَكَّى الزِّنَةَ ) لِذَهَبٍ أَوْ فِضَّةٍ الْمُرَصَّعَ بَعْدِ نَزْعِ الْجَوْهَرِ مِنْهُ ( إنْ نُزِعَ ) بِضَمٍّ فَكَسْرٍ أَيْ : أَمْكَنَ نَزْعُ الْجَوْهَرِ مِنْهُ ( بِلَا ضَرَرٍ ) أَيْ فَسَادٍ وَغُرْمِ أُجْرَةٍ وَحُكْمُ الْجَوْهَرِ حُكْمُ سَائِرِ الْعُرُوضِ ( وَإِلَّا ) أَيْ : وَإِنْ لَمْ يُمْكِنْ نَزْعُهُ أَوْ كَانَ فِيهِ ضَرَرٌ ( تَحَرَّى ) زِنَةَ الذَّهَبِ أَوْ الْفِضَّةِ وَزَكَّاهُ ( وَضُمَّ ) بِضَمٍّ فَفَتْحٍ مُثَقَّلًا وَنَائِبُهُ ( الرِّبْحُ ) أَيْ : الزَّائِدُ عَلَى ثَمَنِ مَا اشْتَرَاهُ وَبَاعَهُ لِلتِّجَارَةِ ذَهَبًا أَوْ فِضَّةً وَصِلَةُ ضُمَّ ( لِأَصْلِهِ ) أَيْ : الرِّبْحِ فِي الْحَوْلِ فَيُزَكَّى مَعَ أَصْلِهِ عِنْدَ تَمَامِهِ مِنْ يَوْمِ مِلْكِهِ أَوْ زَكَاتِهِ .
