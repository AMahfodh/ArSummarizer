( وَمَنْ لَزِمَتْهُ حُدُودٌ وَقَتْلٌ ) مِثْلُ أَنْ يَزْنِيَ وَيَشْرَبَ الْخَمْرَ وَيَسْرِقَ وَيَقْتُلَ مُسْلِمًا ( فَالْقَتْلُ يُجْزِئُ عَنْ ذَلِكَ ) كُلِّهِ وَلَا يُحَدُّ ( إلَّا فِي ) اجْتِمَاعِ ( الْقَذْفِ ) مَعَ الْقَتْلِ ( فَلْيُحَدَّ ) لِلْقَذْفِ ( قَبْلَ أَنْ يُقْتَلَ ) لِنَفْيِ الْمَعَرَّةِ .
