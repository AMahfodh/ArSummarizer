( قَوْلُهُ وَفِيهَا ) أَيْ الْأُمِّ عَطْفٌ عَلَى فِيهِ ( قَوْلُهُ وَغَيْرُهُمَا ) أَيْ كَالْمُدَبَّرَةِ ا ه مُغْنِي ( قَوْلُهُ إنْ كَانَتْ هِيَ ) أَيْ الْأُمُّ ( قَوْلُهُ لَمْ يَجِبْ فِيهِ ) أَيْ فِيمَا إذَا كَانَتْ هِيَ الْجَانِيَةُ إلَخْ ( قَوْلُهُ لَهُ ) أَيْ السَّيِّدِ قَوْلُهُ عَلَيْهِ ) أَيْ الْجَنِينِ ( قَوْلُهُ وَقْتَ الِاسْتِقْرَارِ ) أَيْ اسْتِقْرَارِ الْجِنَايَةِ ( قَوْلُهُ وَالْأَصَحُّ كَمَا إلَخْ ) أَيْ خِلَافًا لِمَا يَقْتَضِيه كَلَامُ الْمُصَنِّفِ مِنْ اعْتِبَارِ يَوْمِ الْجِنَايَةِ مُطْلَقًا سَوَاءٌ كَانَتْ الْقِيمَةُ فِيهِ أَكْثَرَ مِنْ يَوْمِ الْإِجْهَاضِ أَمْ أَقَلَّ وَبِهِ صَرَّحَ الْقَاضِي حُسَيْنُ وَغَيْرُهُ ا ه مُغْنِي ( قَوْلُهُ بِأَنْ يَعْتِقَهَا ) تَصْوِيرُ لِكَوْنِهَا( 38 / 26 )
