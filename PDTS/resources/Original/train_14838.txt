لِأَنَّهُ أَحَقُّ مِنْ الْأَجْنَبِيِّ ( وَيُعْزَلُ لَوْ خَانَ كَالْوَصِيِّ ) رِعَايَةً لِمَصْلَحَةِ الْوَقْفِ ( وَإِنْ ) ( شَرَطَ ) الْوَاقِفُ ( أَنْ لَا يُعْزَلَ ) لِأَنَّهُ شَرْطٌ مُخَالِفٌ لِمُقْتَضَى الشَّرْعِ ( وَلَّاهُ ) أَيْ الْوَاقِفُ الْمُتَوَلِّيَ ( وَأَخْرَجَهُ ) ( صَحَّ ) وَإِنْ لَمْ يَكُنْ لَهُ جَرِيمَةٌ ( وَإِنْ شَرَطَ أَنْ لَا يَخْرُجَ ) لِأَنَّهُ فِي مَعْنَى التَّوْكِيلِ وَلَا عِبْرَةَ بِالشَّرْطِ ( طَالِبُ التَّوْلِيَةِ لَا يُوَلَّى ) كَمَا لَا يُوَلَّى طَالِبُ الْقَضَاءِ .
