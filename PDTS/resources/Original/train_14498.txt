( وَيَجُوزُ الصُّلْحُ ) عَلَى تَرْكِ الْجِهَادِ ( مَعَهُمْ بِمَالٍ ) مِنْهُمْ أَوْ مِنَّا ( لَوْ خُيِّرَا ) - لِقَوْلِهِ تَعَالَى { وَإِنْ جَنَحُوا لِلسَّلْمِ فَاجْنَحْ لَهَا } - ( وَنَنْبِذُ ) أَيْ نُعْلِمُهُمْ بِنَقْضِ الصُّلْحِ تَحَرُّزًا عَنْ الْغَدْرِ الْمُحَرَّمِ ( لَوْ خُيِّرَا ) لِفِعْلِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ بِأَهْلِ مَكَّةَ ( وَنُقَاتِلُهُمْ بِلَا نَبْذٍ مَعَ خِيَانَةِ مَلِكِهِمْ ) وَلَوْ بِقِتَالِ ذِي مَنَعَةٍ بِإِذْنِهِ وَلَوْ بِدُونِهِ انْتَقَضَ حَقُّهُمْ فَقَطْ
