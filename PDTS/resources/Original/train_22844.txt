( وَإِنْ أَخْرَجَ ) إنْسَانٌ ( جَنَاحًا ) وَهُوَ الرَّوْشَنُ ( أَوْ مِيزَابًا وَنَحْوَهُ ) كَسَابَاطٍ وَحَجَرٍ بَرَزَ بِهِ فِي الْبُنْيَانِ ( إلَى طَرِيقٍ نَافِذٍ ) مُطْلَقًا إلَّا بِإِذْنِ إمَامٍ أَوْ نَائِبِهِ فِي جَنَاحٍ أَوْ سَابَاطٍ أَوْ مِيزَابٍ بِلَا ضَرَرٍ ( أَوْ ) أَخْرَجَ مَا ذُكِرَ فِي دَرْبٍ ( غَيْرِ نَافِذٍ بِغَيْرِ إذْنِ أَهْلِهِ فَسَقَطَ عَلَى شَيْءٍ فَأَتْلَفَهُ ضَمِنَ وَلَوْ ) كَانَ سُقُوطُهُ ( بَعْدَ بَيْعِهِ وَقَدْ طُولِبَ بِنَقْضِهِ لِحُصُولِهِ ) أَيْ : التَّلَفِ ( بِفِعْلِهِ ) أَيْ : بِسَبَبِ فِعْلِهِ الَّذِي تَعَدَّى بِهِ وَمَفْهُومُهُ : أَنَّهُ إذَا سَقَطَ بَعْدَ الْبَيْعِ وَلَمْ يَكُنْ طُولِبَ بِنَقْضِهِ لَا يَضْمَنُ ( مَا لَمْ يَأْذَنْ فِيهِ ) أَيْ : الْجَنَاحِ وَالْمِيزَابِ وَالسَّابَاطِ ( إلَى الطَّرِيقِ النَّافِذِ فَقَطْ إمَامٌ أَوْ نَائِبُهُ وَلَمْ يَكُنْ مِنْهُ ضَرَرٌ عَلَى الْمَارَّةِ بِإِخْرَاجِهِ فَلَا ضَمَانَ ) ؛ لِأَنَّ النَّافِذَ حَقٌّ لِلْمُسْلِمِينَ وَالْإِمَامُ وَكِيلُهُمْ فَإِذْنُهُ كَإِذْنِهِمْ أَشْبَهَ مَا لَوْ أَذِنَ أَهْلُ غَيْرِ النَّافِذِ لَهُ فِي ذَلِكَ .( 13 / 230 )
