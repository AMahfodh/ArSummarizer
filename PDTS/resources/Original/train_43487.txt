( وَلَا شُفْعَةَ ) ؛ لِأَنَّهُ لَا يُمْكِنُ الْأَخْذُ بِغَيْرِ ثَمَنٍ ، وَلَا يُمْكِنُ أَنْ يَدْفَعَ إلَيْهِ مَا لَا يَدَّعِيهِ إلَّا أَنْ يَفْعَلَ ذَلِكَ تَحَيُّلًا عَلَى إسْقَاطِهَا فَلَا يَسْقُطُ ( فَإِنْ اتَّهَمَهُ ) الشَّفِيعُ ( أَنَّهُ ) أَيْ : الْمُشْتَرِي ( فَعَلَهُ حِيلَةً ) لِإِسْقَاطِ الشُّفْعَةِ ( حَلَّفَهُ ) أَنَّهُ لَمْ يَفْعَلْهُ حِيلَةً ( وَإِنْ وَقَعَ ) ذَلِكَ ( حِيلَةً دَفَعَ ) الشَّفِيعُ ( إلَيْهِ ) أَيْ : الْمُشْتَرِي مِثْلَ ( مَا أَعْطَاهُ ) لِلْبَائِعِ إنْ عَلِمَ ( أَوْ قِيمَةَ الشِّقْصِ ) إنْ تَعَذَّرَتْ مَعْرِفَةُ الثَّمَنِ .
