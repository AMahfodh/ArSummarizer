( وَمَنْ أَقْبَضَهُ ) أَيْ : الصَّدَاقَ الَّذِي تَزَوَّجَ عَلَيْهِ ( ثُمَّ طَلَّقَ ) الزَّوْجَةَ ( قَبْلَ دُخُولٍ ) بِهَا ( مَلَكَ نِصْفَهُ ) الصَّدَاقِ ( قَهْرًا ) كَمِيرَاثٍ وَلَوْ صَيْدًا وَهُوَ مُحَرَّمٌ فَمَا يَحْدُثُ مِنْ نَمَائِهِ بَعْدَ طَلَاقِهِ فَهُوَ بَيْنَهُمَا ، لِقَوْلِهِ تَعَالَى { وَإِنْ طَلَّقْتُمُوهُنَّ مِنْ قَبْلِ أَنْ تَمَسُّوهُنَّ وَقَدْ فَرَضْتُمْ لَهُنَّ فَرِيضَةً فَنِصْفُ مَا فَرَضْتُمْ } أَيْ : لَكُمْ أَوَّلُهُنَّ فَاقْتَضَى أَنَّ النِّصْفَ لَهَا ، وَالنِّصْفَ لَهُ بِمُجَرَّدِ الطَّلَاقِ ( إنْ بَقِيَ ) فِي مِلْكِهَا ( بِصِفَتِهِ ) حِينَ عَقَدَ بِأَنْ لَمْ يَزِدْ وَلَمْ يَنْقُصْ .
