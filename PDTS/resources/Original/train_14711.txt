( وَ ) مَنْ قَالَ لِزَوْجَتِهِ ( أَنْتِ طَالِقٌ ، وَنَوَى ثَلَاثًا فَثَلَاثٌ ) تَقَعُ بِهَا ( كَنِيَّتِهِ الثَّلَاثَ بِ ) قَوْلِهِ ( أَنْتِ طَالِقٌ طَلَاقًا ) لِأَنَّ الْمَصْدَرَ يَقَعُ عَلَى الْقَلِيلِ وَالْكَثِيرِ ، كَمَا لَوْ فَقَدْ نَوَى بِلَفْظِهِ مَا يَحْتَمِلُهُ ، وَإِنْ أَطْلَقَ فَوَاحِدٌ ، لِأَنَّهَا الْيَقِينَ نَوَى وَاحِدَةً ( وَ ) قَوْلِهِ لَهَا ( أَنْتِ طَالِقٌ وَاحِدَةً ؛ أَوْ ) طَالِقٌ ( وَاحِدَةً بِالنِّيَّةِ أَوْ ) طَالِقٌ ( وَاحِدَةً بَتَّةً ) وَلَا عِوَضَ ( فَ ) وَاحِدَةٌ ( رَجْعِيَّةٌ فِي مَدْخُولٍ بِهَا ، وَلَوْ نَوَى أَكْثَرَ ) مِنْ وَاحِدَةٍ لِوَصْفِهَا بِوَاحِدَةٍ وَالْأَصْلُ فِيهَا أَنْ تَكُونَ رَجْعِيَّةً ؛ فَلَا تَخْرُجُ بِوَصْفِهَا بِذَلِكَ عَنْ أَصْلِهَا ، وَإِنَّمَا كَانَتْ بَائِنًا بِالْعِوَضِ لِضَرُورَةِ الِافْتِدَاءِ .
