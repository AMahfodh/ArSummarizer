( وَإِلَّا ) أَيْ : وَإِنْ لَمْ يَخْرُجْ شَيْءٌ ( فَلَا ) تَتَوَضَّأُ لِكُلِّ وَقْتِ صَلَاةٍ ( وَتُصَلِّي ) الْمُسْتَحَاضَةُ بِوُضُوئِهَا ( مَا شَاءَتْ ) مَا دَامَ الْوَقْتُ ( حَتَّى جَمْعًا بَيْنَ فَرْضَيْنِ ) لِبَقَاءِ وُضُوئِهَا إلَى آخِرِ الْوَقْتِ ، وَكَالْمُتَيَمِّمِ وَأَوْلَى ( وَلَهَا ) أَيْ الْمُسْتَحَاضَةِ ( الطَّوَافُ ) فَرْضًا وَنَفْلًا ( وَلَوْ لَمْ تَطُلْ اسْتِحَاضَتُهَا ) كَالصَّلَاةِ وَأَوْلَى ( وَتُصَلِّي عَقِبَ طُهْرِهَا نَدْبًا ) خُرُوجًا مِنْ الْخِلَافِ ( فَإِنْ أَخَّرَتْ ) الصَّلَاةَ عَنْ طُهْرِهَا ( وَلَوْ ) كَانَ التَّأْخِيرُ ( لِغَيْرِ حَاجَةٍ لَمْ يَضُرَّ ) مَا دَامَ الْوَقْتُ لِأَنَّهَا مُتَطَهِّرَةٌ كَالْمُتَيَمِّمِ ( وَإِنْ كَانَ لَهَا ) أَيْ : الْمُسْتَحَاضَةِ ( عَادَةٌ بِانْقِطَاعِهِ ) أَيْ : الدَّمِ ( زَمَنًا يَتَّسِعُ لِلْوُضُوءِ وَالصَّلَاةِ ، تَعَيَّنَ فِعْلُهُمَا فِيهِ ) لِأَنَّهُ قَدْ أَمْكَنَ الْإِتْيَانُ بِالْعِبَادَةِ عَلَى وَجْهٍ لَا عُذْرَ مَعَهُ وَلَا ضَرُورَةَ ، فَتَعَيَّنَ فِعْلُهُمَا عَلَى هَذَا الْوَجْهِ كَمَنْ لَا عُذْرَ لَهُ فَإِنْ تَوَضَّأَتْ زَمَنَ انْقِطَاعِهِ ثُمَّ عَادَ بَطَلَ .( 2 / 89 )
