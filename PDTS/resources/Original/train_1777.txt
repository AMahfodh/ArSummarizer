( وَإِنْ لَمْ يَفِ ) الْمُولِي بِوَطْءِ مَنْ آلَى مِنْهَا ( وَأَعْفَتْهُ الْمَرْأَةُ سَقَطَ حَقّهَا ) لِأَنَّ الْحَقّ لَهَا وَقَدْ أَسْقَطَتْهُ ( كَعَفْوِهَا ) عَنْ الْعِنِّينِ ( بَعْدَ ) مُضِيِّ ( مُدَّةِ الْعُنَّةِ ) وَهِيَ السَّنَةُ ( وَإِنْ لَمْ تَعْفُهُ أُمِرَ بِالطَّلَاقِ ) إنْ طَلَبَتْهُ لِقَوْلِهِ تَعَالَى : { فَإِمْسَاكٌ بِمَعْرُوفٍ أَوْ تَسْرِيحٌ بِإِحْسَانٍ } فَإِذَا امْتَنَعَ مِنْ أَدَاءِ الْوَاجِبِ فَقَدْ امْتَنَعَ مِنْ الْإِمْسَاكِ بِالْمَعْرُوفِ فَيُؤْمَر بِالتَّسْرِيحِ بِالْإِحْسَانِ .( 19 / 8 )
