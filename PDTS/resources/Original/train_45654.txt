( وَإِنْ قَطَعَ ) الْجَانِي ( بَعْضَ أُذُنِهِ فَلَهُ أَنْ يَقْتَصَّ مِنْ أُذُنِ الْجَانِي بِقَدْرِ مَا قَطَعَ مِنْ أُذُنِهِ وَيُقَدَّرُ ذَلِكَ بِالْأَجْزَاءِ ) كَالنِّصْفِ وَالثُّلُثِ وَالرُّبْعِ وَ ( لَا ) يُؤْخَذُ ( بِالْمِسَاحَةِ ) لِأَنَّهُ قَدْ يُفْضِي إلَى أَخْذِ جَمِيعِ أُذُنِ الْجَانِي لِصِغَرِهِ بِبَعْضِ أُذُنِ الْمَجْنِيِّ عَلَيْهِ لِكِبَرِهِ وَكَذَا أَنْفٌ وَلِسَانٌ وَشَفَةٌ .( 20 / 154 )
