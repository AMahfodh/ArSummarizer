الْمَازِرِيُّ إنْ كَانَ مَعْنَاهُ إذَا سَقَطَتْ بِاخْتِيَارِهِ فَهُوَ جَائِزٌ إذَا وَقَعَ مُؤَجَّلًا إلَّا أَنْ يَكُونَ ثَمَنُهُ مَجْهُولًا ، أَوْ مُضَافًا إلَيْهِ شَيْءٌ يُفْسِدُهُ مِثْلُ أَنْ يَقُولَ : مَتَى نَزَلَتْ حَصَاةٌ وَلَوْ بَعْدَ عَامٍ وَجَبَ الْبَيْعُ ( أَوْ ) هُوَ بَيْعُ شَيْءٍ غَيْرِ مُعَيَّنٍ مِنْ أَشْيَاءَ مُخْتَلِفَةٍ ( عَلَى ) شَرْطِ أَنَّ الْمَبِيعَ ( مَا ) أَيْ الشَّيْءُ الَّذِي ( تَقَعُ ) الْحَصَاةُ ( عَلَيْهِ ) مِنْ تِلْكَ الْأَشْيَاءِ ( بِلَا قَصْدٍ ) مِمَّنْ هِيَ مَعَهُ وَمَفْهُومُهُ أَنَّهُ إنْ كَانَ بِقَصْدِهِ جَازَ إنْ كَانَ الْمُشْتَرِي أَوْ الْبَائِعُ وَشَرَطَ الْخِيَارَ لِلْمُشْتَرِي ، فَإِنْ اتَّفَقَتْ الْأَشْيَاءُ جَازَ كَانَ وُقُوعُهَا بِقَصْدٍ أَوْ لَا ( أَوْ ) هُوَ بَيْعُ شَيْءٍ مُعَيَّنٍ بِدَرَاهِمَ أَوْ دَنَانِيرَ عَدَدُهَا ( بِعَدَدِ مَا يَقَعُ ) مِنْ أَجْزَاءِ الْحَصَاةِ الْمَرْمِيَّةِ عَلَى الْأَرْضِ بِقُوَّةٍ بِأَنْ يَقُولَ الْبَائِعُ لِلْمُشْتَرِي : ارْمِ بِهَا ، فَمَا خَرَجَ فَلِي( 10 / 128 )
