( وَ ) كَفَى ( حَبْسُ شَاةِ لَحْمٍ ، وَنَظَرُ ) جَمِيعِ جَسَدِ شَاةِ قُنْيَةٌ ) لِلدَّرِّ وَالنَّسْلِ مَعَ ضَرْعِهَا ظَهِيرِيَّةٌ وَضَرْعِ بَقَرَةٍ حَلُوبٍ وَنَاقَةٍ ؛ لِأَنَّهُ الْمَقْصُودُ جَوْهَرَةٌ ( وَ ) كَفَى ذَوْقُ مَطْعُومٍ وَشَمُّ مَشْمُومٍ ( لَا خَارِجُ دَارٍ وَصَحْنُهَا ) عَلَى الْمُفْتَى بِهِ كَمَا مَرَّ ( أَوْ رُؤْيَةُ دُهْنٍ فِي زُجَاجٍ ) لِوُجُودِ الْحَائِلِ
