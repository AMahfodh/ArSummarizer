( وَ ) جَازَ ( خُرُوجُ ) مَرْأَةٍ ( مُتَجَالَّةٍ ) لَا إرْبَ لِلرِّجَالِ فِيهَا لِجِنَازَةِ كُلِّ أَحَدٍ ( أَوْ ) مَرْأَةٍ شَابَّةٍ ( إنْ لَمْ يُخْشَ مِنْهَا ) أَيْ الشَّابَّةِ ( الْفِتْنَةُ ) لِلرِّجَالِ بِتَعَلُّقِ نُفُوسِهِمْ بِهَا ( فِي ) جِنَازَةِ مَنْ عَظُمَتْ مُصِيبَتُهُ عَلَيْهَا ( كَأَبٍ ) لَهَا وَأُمٍّ ( وَزَوْجٍ ) لَهَا ( وَابْنٍ ) وَبِنْتٍ لَهَا ( وَأَخٍ ) وَأُخْتٍ مُطْلَقًا ، وَكُرِهَ خُرُوجُهَا لِغَيْرِ مَنْ ذُكِرَ وَحَرُمَ عَلَى مَخْشِيَّةِ الْفِتْنَةِ لِكُلِّ أَحَدٍ وَمِثْلُ الشَّابَّةِ غَيْرُ الْمَخْشِيَّةِ الْمُتَجَالَّةُ الَّتِي فِيهَا إرْبٌ لِلرِّجَالِ( 3 / 158 )
