وَإِنْ ظَنَّ أَنْ يُدْرِكَ الْجُمُعَةَ سَعَى إلَيْهَا وَإِلَّا انْتَظَرَ حَتَّى يَتَيَقَّنَ فَوْتَهَا ( وَتَصِحُّ ) الظُّهْرُ ( مِنْ مَعْذُورٍ ) قَبْلَ تَجْمِيعِ إمَامٍ لِأَنَّهَا فَرْضُهُ ، وَقَدْ أَدَّاهُ ( وَلَوْ زَالَ عُذْرُهُ قَبْلَهُ ) أَيْ قَبْلَ تَجْمِيعِ الْإِمَامِ كَمَعْضُوبٍ حُجَّ عَنْهُ ثُمَّ عُوفِيَ ( إلَّا الصَّبِيَّ ) إذَا بَلَغَ وَلَوْ كَانَ بُلُوغُهُ ( بَعْدَهُ ) أَيْ بَعْدَ تَجْمِيعِ الْإِمَامِ ، وَكَانَ قَدْ صَلَّى الظُّهْرَ أَوَّلًا أَعَادَهَا بَلْ وَلَوْ بَلَغَ قَبْلَ الْمَغْرِبِ أَعَادَ الظُّهْرَ وَالْعَصْرَ كَمَا تَقَدَّمَ ، لِأَنَّ الْأُولَى كَانَتْ نَفْلًا وَقَدْ صَارَتْ فَرْضًا .( 2 / 294 )
