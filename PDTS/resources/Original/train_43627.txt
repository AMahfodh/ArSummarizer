( وَلَهَا ) أَيْ : الزَّوْجَةِ ( هِبَةُ نَوْبَتِهَا ) مِنْ الْقَسْمِ ( بِلَا مَالٍ لِزَوْجٍ يَجْعَلُهُ لِمَنْ شَاءَ ) مِنْ ضَرَّاتِهَا ؛ لِأَنَّ الْحَقَّ لَا يَخْرُجُ عَنْ الْوَاهِبَةِ ، وَالزَّوْجِ ( وَ ) لِلزَّوْجَةِ هِبَةُ نَوْبَتَهَا بِلَا مَالٍ ( لِضَرَّةٍ ) مُعَيَّنَةٍ ( بِإِذْنِهِ ) أَيْ : الزَّوْجِ ( وَلَوْ أَبَتْ ) ذَلِكَ ( مَوْهُوبٌ لَهَا ) لِثُبُوتِ حَقِّ الزَّوْجِ فِي الِاسْتِمْتَاعِ بِهَا كُلَّ وَقْتٍ وَإِنَّمَا مَنَعَتْهُ الْمُزَاحِمَةُ فِي حَقِّ صَاحِبَتِهَا فَإِذَا أَزَالَتْ الْمُزَاحِمَةُ بِهِبَتِهَا ثَبَتَ حَقُّهُ فِي الِاسْتِمْتَاعِ بِهَا وَإِنْ كَرِهَتْ كَمَا لَوْ كَانَتْ مُنْفَرِدَةً { ، وَوَهَبَتْ سَوْدَةُ يَوْمَهَا لِعَائِشَةَ فَكَانَ رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يَقْسِمُ لِعَائِشَةَ يَوْمَهَا ، وَيَوْمَ سَوْدَةَ } مُتَّفَقٌ عَلَيْهِ .
