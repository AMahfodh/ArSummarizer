( قَوْلُهُ بِخِلَافِ تِلْكَ النُّقَطِ ) أَيْ فَلَيْسَ لَهَا تِلْكَ الْقُوَّةُ وَعَلَى فَرْضِ وُجُودِهَا فِيهِ تَطْهُرُ مَحَلُّهَا كُرْدِيٌّ ( قَوْلُهُ لِأَنَّهَا عَمَّتُهُ ) أَيْ عَمَّتْ النَّجَاسَةُ الْمَحَلَّ قَوْلُ الْمَتْنِ ( لَا الْعَصْرُ إلَخْ ) لَكِنَّهُ يُسْتَحَبُّ فِيمَا يَكُنْ عَصْرُهُ خُرُوجًا مِنْ خِلَافِ مَنْ أَوْجَبَهُ نِهَايَةٌ وَمُغْنِي ( قَوْلُهُ وَلَوْ فِيمَا لَهُ خَمْلٌ إلَخْ ) كَذَا فِي النِّهَايَةِ ( قَوْلُهُ فِيهِ ) أَيْ فِي الْمَحَلِّ ( قَوْلُهُ وَمَحَلُّ الْخِلَافِ ) ذَكَرَهُ ع ش عَنْهُ وَأَقَرَّهُ قَوْلُ الْمَتْنِ ( وَالْأَظْهَرُ طَهَارَةُ غُسَالَةٍ تَنْفَصِلُ إلَخْ ) وَلَيْسَتْ بِطَهُورٍ لِاسْتِعْمَالِهَا فِي( 3 / 391 )
