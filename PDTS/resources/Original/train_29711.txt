( وَيَصِحُّ ) السَّلَمُ ( فِي عَرْضٍ بِعَرْضٍ ) إنْ لَمْ يَجْرِ بَيْنَهُمَا رِبَا النَّسِيئَةِ ( فَلَوْ جَاءَهُ ) أَيْ جَاءَ الْمُسْلِمُ الْمُسْلَمَ إلَيْهِ ( بِعَيْنِ مَا أَخَذَ مِنْهُ عِنْدَ مَحَلِّهِ ) بِكَسْرِ الْحَاءِ أَيْ حُلُولِهِ ( لَزِمَهُ ) أَيْ الْمُسْلِمَ ( قَبُولُهُ إنْ اتَّحِدَا صِفَةً ) لِأَنَّهُ آتَاهُ بِالْمُسْلَمِ فِيهِ عَلَى صِفَتِهِ فَلَزِمَهُ قَبُولُهُ كَمَا لَوْ أَتَاهُ بِغَيْرِهِ وَالْمُثَمَّنُ إنَّمَا هُوَ فِي الذِّمَّةِ وَهَذَا عِوَضٌ عَنْهُ .
