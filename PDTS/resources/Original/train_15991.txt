( وَ ) الدَّمُ ( الْمُمَيَّزُ ) فِي زَمَنِ الِاسْتِحَاضَةِ بِتَغَيُّرِ رَائِحَةٍ أَوْ لَوْنٍ أَوْ رِقَّةٍ أَوْ ثِخَنٍ أَوْ بِتَأَلُّمِهَا لَا بِكَثْرَةٍ أَوْ قِلَّةٍ لِتَبَعِيَّتِهِمَا لِلْمِزَاجِ ( بَعْدَ طُهْرٍ تَمَّ ) خَمْسَةَ عَشَرَ يَوْمًا ( حَيْض ) فَإِنْ لَمْ تُمَيِّزْ فَهِيَ مُسْتَحَاضَةٌ وَلَوْ مَكَثَتْ طُولَ عُمْرِهَا وَكَذَا لَوْ مَيَّزَتْ قَبْلَ تَمَامِ الطُّهْرِ فَهِيَ مُسْتَحَاضَةٌ ( وَلَا تَسْتَظْهِرُ ) الْمُمَيِّزَةُ بَلْ تَقْتَصِرُ عَلَى عَادَتِهَا ( عَلَى الْأَصَحِّ ) مَا لَمْ يَسْتَمِرَّ مَا مَيَّزَتْهُ بِصِفَةِ الْحَيْضِ الْمُمَيَّزِ فَإِنْ اسْتَمَرَّ بِصِفَتِهِ اسْتَظْهَرَتْ عَلَى الْمُعْتَمَدِ
