( وَ ) خُصَّ بِأَنَّهُ ( يُزَوِّجُ مِنْ نَفْسِهِ ) وَيَتَوَلَّى الطَّرَفَيْنِ وَلَوْ لَمْ تَرْضَ الْمَرْأَةُ وَوَلِيُّهَا وَلَكِنْ إذَا كَرِهَتْ بَعْدَ ذَلِكَ الْإِقَامَةَ حَرُمَ عَلَيْهِ إمْسَاكُهَا فِي عِصْمَتِهِ ( وَ ) يُزَوِّجُ ( مَنْ شَاءَ ) مِنْ الرِّجَالِ أَوْ النِّسَاءِ بِغَيْرِ إذْنٍ ( وَ ) يُزَوِّجُ مِنْ نَفْسِهِ وَمَنْ شَاءَ ( بِلَفْظِ الْهِبَةِ ) بِلَا ذِكْرِ صَدَاقٍ ( وَ ) خُصَّ بِجَوَازِ أَنْ يُزَوِّجَ نَفْسَهُ ( بِزَائِدٍ عَلَى أَرْبَعٍ ) مِنْ النِّسْوَةِ وَكَذَا سَائِرُ الْأَنْبِيَاءِ .
