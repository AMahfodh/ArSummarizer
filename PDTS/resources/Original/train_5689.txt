( وَيُبَاحُ ) ( قَوْلُ ) أَيْ إنْشَاءُ ( شِعْرٍ وَإِنْشَادُهُ ) وَاسْتِمَاعُهُ { لِأَنَّهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ كَانَ لَهُ شُعَرَاءُ يُصْغِي إلَيْهِمْ كَحَسَّانٍ وَعَبْدِ اللَّهِ بْنِ رَوَاحَةَ وَكَعْبِ بْنِ مَالِكٍ رَضِيَ اللَّهُ عَنْهُمْ وَاسْتَنْشَدَ مِنْ شِعْرِ أُمَيَّةَ بْنِ أَبِي الصَّلْتِ مِائَةَ بَيْتٍ } : أَيْ لِأَنَّ أَكْثَرَ شِعْرِهِ حِكَمٌ وَأَمْثَالٌ وَتَذْكِيرٌ بِالْبَعْثِ ، وَلِهَذَا قَالَ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { كَادَ أَنْ يُسْلِمَ } وَرَوَى الْبُخَارِيُّ { إنَّ( 28 / 289 )
