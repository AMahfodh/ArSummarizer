( وَ ) كَ ( بَيْضٍ ) لِدَجَاجٍ أَوْ حَمَامٍ أَوْ إوَزٍّ غُصِبَ وَحُضِنَ حَتَّى ( أَفْرَخَ ) أَيْ صَارَ فِرَاخًا فَلَيْسَ لِرَبِّهِ إلَّا مِثْلُهُ وَالْفِرَاخُ لِلْغَاصِبِ ( إلَّا ) فِرَاخَ ( مَا ) أَيْ الطَّيْرَ الَّذِي ( بَاضَ ) فَهِيَ لِرَبِّهِ ( إنْ حَضَنَ ) الطَّيْرُ بَيْضَهُ كَدَجَاجِ وَحَمَامٍ وَإِوَزٍّ ، وَظَاهِرُهُ وَلَوْ كَانَ الذَّكَرُ لِلْغَاصِبِ وَهُوَ كَذَلِكَ .
