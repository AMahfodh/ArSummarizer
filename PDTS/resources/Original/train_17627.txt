( وَلَمْ يُصَلَّ خَلْفَهُ ) لِفِسْقِهِ ( وَإِنْ لَمْ يُدَاوِمْ ) عَلَيْهِ ( فَقَالَ الْمُوَفَّقُ ) وَالشَّارِحُ ( هُوَ مِنْ الصَّغَائِرِ وَلَا بَأْسَ بِالصَّلَاةِ خَلْفَهُ ) ؛ لِأَنَّ الْفِسْقَ لَا يَحْصُلُ بِالصَّغِيرَةِ بَلْ بِالْمُدَاوَمَةِ عَلَيْهَا كَمَا تَقَدَّمَ وَيَأْتِي قَالَ تَعَالَى { إنْ تَجْتَنِبُوا كَبَائِرَ مَا تُنْهَوْنَ عَنْهُ نُكَفِّرْ عَنْكُمْ سَيِّئَاتِكُمْ } وَقَالَ الشَّيْخُ تَقِيُّ الدِّينِ : لَوْ فَعَلَ الْإِمَامُ مَا هُوَ مُحَرَّمٌ عِنْدَ الْمَأْمُومِ دُونَهُ مِمَّا يَسُوغُ فِيهِ الِاجْتِهَادُ : صَحَّتْ صَلَاتُهُ خَلْفَهُ وَهُوَ الْمَشْهُورُ عَنْ أَحْمَدَ .( 3 / 442 )
