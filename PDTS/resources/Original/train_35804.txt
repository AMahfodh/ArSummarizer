( وَلَوْ بَانَ بَعْدَ الْحُكْمِ كُفْرُ الشُّهُودِ أَوْ فِسْقُهُمْ لَزِمَهُ ) أَيْ الْحَاكِمَ ( نَقْضُهُ ) أَيْ الْحُكْمِ لِفَقْدِ شَرْطِ صِحَّتِهِ ( وَيَرْجِعُ بِالْمَالِ ) الْمَحْكُومِ بِهِ إنْ بَقِيَ ( أَوْ بَدَلَهُ ) إنْ تَلِفَ عَلَى الْمَحْكُومِ لَهُ لِأَنَّهُ أَخْذَ بِغَيْرِ حَقٍّ ( أَوْ ) يَرْجِعُ بِ ( بَدَلِ قَوَدٍ مُسْتَوْفًى عَلَى الْمَحْكُومِ لَهُ ) إنْ لَمْ يَكُنْ بَعْدَ قَتْلِهِ عَلَى مَا سَبَقَ تَفْصِيلُهُ فِي الْعَمْدِ .( 22 / 335 )
