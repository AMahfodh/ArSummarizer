( وَإِنْ أَرْضَعَتْ ) امْرَأَةٌ ( بِلَبَنِ اثْنَيْنِ وَطِئَاهَا بِشُبْهَةٍ طِفْلًا وَثَبَتَتْ أُبُوَّتُهُمَا ) أَيْ الْوَاطِئَيْنِ ( أَوْ ) ثَبَتَتْ ( أُبُوَّةُ أَحَدِهِمَا لِمَوْلُودٍ ) بِأَنْ أَلْحَقَتْهُ الْقَافَةُ بِهِمَا أَوْ بِأَحَدِهِمَا بِعَيْنِهِ ( فَالْمُرْتَضِعُ ابْنُهُمَا ) إنْ ثَبَتَتْ أُبُوَّتُهُمَا ( أَوْ ابْنُ أَحَدِهِمَا ) إنْ ثَبَتَتْ أُبُوَّتُهُ فَقَطْ ؛ لِأَنَّ حُكْمَ الرَّضِيعِ تَابِعٌ لِحُكْمِ الْمَوْلُودِ ( وَإِلَّا ) ثَبَتَتْ أُبُوَّتُهُمَا وَلَا أُبُوَّةُ أَحَدِهِمَا لِمَوْلُودٍ ( بِأَنْ مَاتَ مَوْلُودٌ قَبْلَهُ ) أَيْ قَبْلَ الْإِلْحَاقِ بِهِمَا أَوْ بِأَحَدِهِمَا ( أَوْ فَقَدَتْ قَافَةٌ أَوْ نَفَتْهُ ) الْقَافَةُ ( عَنْهُمَا ) أَيْ الْوَاطِئَيْنِ ( أَوْ أَشْكَلَ أَمْرُهُ ) عَلَى الْقَافَةِ ( ثَبَتَتْ حُرْمَةُ الرَّضَاعِ ) مِنْ جِهَةِ الْمُرْتَضِعِ ( فِي حَقِّهِمَا ) أَيْ الْوَاطِئَيْنِ تَغْلِيبًا لِلْحَظْرِ .
