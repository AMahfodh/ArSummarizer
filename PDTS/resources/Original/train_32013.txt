( فَإِنْ كَانَ فِيهَا ) أَيْ الْقِسْمَةِ ( تَقْوِيمٌ لَمْ يَجُزْ ) أَنْ يَقْسِمَ بَيْنَهُمَا ( أَقَلُّ مِنْ قَاسِمَيْنِ لِأَنَّهَا شَهَادَةٌ بِالْقُرْعَةِ ) فَلَمْ يُقْبَلْ فِيهَا أَقُلُّ مِنْ اثْنَيْنِ كَسَائِرِ الشَّهَادَاتِ ( وَإِلَّا ) أَيْ وَإِنْ لَمْ يَكُنْ فِيهَا تَقْوِيمٌ ( أَجْزَأَ وَاحِدٌ ) لِأَنَّهُ يَنْفُذُ مَا يَجْتَهِدُ فِيهِ أَشْبَهَ الْقَائِفَ وَالْحَاكِمَ ( وَإِذَا سَأَلُوا ) أَيْ الشُّرَكَاءُ ( الْحَاكِمَ قِسْمَةَ عَقَارٍ لَمْ يَثْبُتْ عِنْدَهُ أَنَّهُ لَهُمْ لَمْ يَجِبْ عَلَيْهِ قِسْمَةٌ ) بَيْنَهُمْ لِعَدَمِ ثُبُوتِ الْمِلْكِ لَهُمْ فِيهِ ( بَلْ يَجُوزُ ) لَهُ قِسْمَةٌ بِإِقْرَارِهِمْ وَتَرَاضِيهِمْ لِأَنَّ الْيَدَ دَلِيلُ الْمِلْكِ وَإِنْ لَمْ يَثْبُتْ بِهَا الْمِلْكُ وَلَا مُنَازِعَ لَهُمْ فِي الظَّاهِرِ .
