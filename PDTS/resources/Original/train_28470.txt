( لَكِنْ لِلْإِمَامِ وَنَائِبِهِ فَقَطْ ) دُونَ غَيْرِهِمَا ( أَخْذُ ذَلِكَ ) أَيْ : مَا ذُكِرَ مِنْ الضَّوَالِّ ( لِيَحْفَظَهُ لِرَبِّهِ ) ؛ لِأَنَّ لَهُمَا نَظَرًا فِي حِفْظِ مَالِ الْغَائِبِ ، وَفِي أَخَذِهَا عَلَى وَجْهِ الْحِفْظِ مَصْلَحَةً لِرَبِّهَا لِصَوْنِهَا ، وَ ( لَا ) يَجُوزُ لَهُمَا كَغَيْرِهِمَا أَخَذُهَا ( عَلَى سَبِيلِ الِالْتِقَاطِ ) لِمَا تَقَدَّمَ .( 14 / 128 )
