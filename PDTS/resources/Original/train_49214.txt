التَّعْلِيقَ إنَّمَا يَكُونُ فِي الْمُسْتَقْبَلِ ( قَوْلُهُ : لِأَنَّ الْمُعَلَّقَ إلَخْ ) بَدَلٌ مِنْ قَوْلِهِمْ : أَوْ مَقُولٌ لَهُ ( قَوْلُهُ : بِالتَّعْلِيقِ بِهِ ) أَيْ بِالْمُحَالِ ( قَوْلُهُ : عَدَمَ الْوُقُوعِ ) أَيْ فِيهِ ( قَوْلُهُ : لَا يَقْصِدُ أَهْلُ الْعُرْفِ بِهِ إلَخْ ) قَدْ يُمْنَعُ ا ه سم( 33 / 281 )
