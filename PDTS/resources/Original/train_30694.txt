( قَوْلُهُ : أَنَّ بَعْدَهَا ) أَيْ الْمُدَّةِ وَقَوْلُهُ : فِي ذَيْنِكَ أَيْ شَرْطِ الْخِيَارِ وَالنِّكَاحِ فِي الْعِدَّةِ ا ه ع ش ( قَوْلُهُ : وَقَبْلَهَا ) أَيْ الْمُدَّةِ ( قَوْلُهُ : الْحُكْمُ وَاحِدٌ إلَخْ ) وَهُوَ عَدَمُ التَّقْدِيرِ ( قَوْلُ الْمَتْنِ : عِدَّةَ شُبْهَةٍ ) أَيْ بَعْدَ الْعَقْدِ ا ه مُغْنِي ( قَوْلُهُ : فَهَذَا أَوْلَى ) أَيْ لِأَنَّهُ يُحْتَمَلُ فِي أَنْكِحَةِ الْكُفَّارِ مَا لَا يُحْتَمَلُ فِي أَنْكِحَةِ الْمُسْلِمِينَ مُغْنِي وَنِهَايَةٌ ( قَوْلُهُ : دُونَ نَظَائِرِهِ ) أَيْ كَطُرُوِّ الْمَحْرَمِيَّةِ بِنَحْوِ( 30 / 402 )
