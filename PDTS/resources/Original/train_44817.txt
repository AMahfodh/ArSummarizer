( فَصْلٌ لَوْ قَبِلَ الرَّقِيقُ ) وَلَوْ سَفِيهًا ( هِبَةً أَوْ وَصِيَّةً بِلَا إذْنٍ صَحَّ ) وَإِنْ نَهَاهُ سَيِّدُهُ عَنْ الْقَبُولَةِ لِأَنَّهُ اكْتِسَابٌ لَا يَعْقُبُ عِوَضًا كَالِاحْتِطَابِ نَعَمْ إنْ كَانَ الْمَوْهُوبُ أَوْ الْمُوصَى بِهِ بَعْضًا لِلسَّيِّدِ تَجِبُ عَلَيْهِ نَفَقَتُهُ حَالَ الْقَبُولِ لِنَحْوِ زَمَانَةٍ أَوْ صِغَرٍ لَمْ يَصِحَّ الْقَبُولُ وَنَظِيرُهُ قَبُولُ الْوَلِيِّ لِمُوَلِّيهِ ذَلِكَ ( وَدَخَلَ مِلْكَ السَّيِّدِ قَهْرًا ) كَعِوَضِ الْخُلْعِ بِغَيْرِ إذْنِهِ وَيُسَلِّمُ الْمَالَ لَهُ لَا لِلرَّقِيقِ ( وَلَوْ اشْتَرَى أَوْ اقْتَرَضَ ) بِلَا إذْنٍ ( لَمْ يَصِحَّ ) لِمَا عُلِمَ مِنْ أَوَّلِ الْبَابِ ( وَلِلْمَالِكِ اسْتِرْدَادُهُ ) أَيْ مَا ذُكِرَ مِنْ الْمَبِيعِ أَوْ الْمُقْرَضِ سَوَاءٌ أَكَانَ بِيَدِ الرَّقِيقِ أَمْ بِيَدِ سَيِّدِهِ لِأَنَّهُ لَمْ يَخْرُجْ عَنْ مِلْكِهِ ( وَإِنْ أَتْلَفَهُ ) الرَّقِيقُ مُطْلَقًا أَوْ تَلِفَ فِي يَدِهِ ( تَعَلَّقَ ) الضَّمَانُ ( بِذِمَّتِهِ ) فَيُطَالَبُ بِهِ بَعْدَ عِتْقِهِ لِأَنَّهُ وَجَبَ بِرِضَا مُسْتَحِقِّهِ وَلَمْ يَأْذَنْ فِيهِ السَّيِّدُ وَضَابِطُ تَعَلُّقِ الْحُقُوقِ بِالرَّقِيقِ أَنَّهَا إنْ ثَبَتَتْ بِغَيْرِ اخْتِيَارِ أَرْبَابِهَا كَإِتْلَافٍ وَتَلَفٍ بِغَصْبٍ تَعَلَّقَتْ بِرَقَبَتِهِ أَوْ بِاخْتِيَارِهِمْ كَمَا فِي الْمُعَامَلَاتِ فَإِنْ كَانَ بِغَيْرِ إذْنِ سَيِّدِهِ تَعَلَّقَتْ بِذِمَّتِهِ يُتْبَعُ بِهَا بَعْدَ عِتْقِهِ أَوْ بِإِذْنِهِ تَعَلَّقَتْ بِذِمَّتِهِ وَكَسْبِهِ وَمَالِ تِجَارَتِهِ كَمَا مَرَّ قَالَ ابْنُ الرِّفْعَةِ وَحَيْثُ قُلْنَا تَتَعَلَّقُ بِكَسْبِهِ لَزِمَهُ أَنْ يَكْتَسِبَ لِلْفَاضِلِ قَالَ الزَّرْكَشِيُّ وَفِيهِ نَظَرٌ لِمَا سَيَأْتِي فِي الْمُفْلِسِ ( وَإِنْ قَبَضَهُ ) مِنْهُ ( السَّيِّدُ وَتَلِفَ ) وَلَوْ فِي غَيْرِهِ ( فَلَهُ ) أَيْ لِلْمَالِكِ ( مُطَالَبَةُ السَّيِّدِ ) لِوَضْعِ يَدِهِ ( وَكَذَا الْعَبْدُ إنْ عَتَقَ ) لِذَلِكَ ( وَإِنْ أَدَّى ) الرَّقِيقُ ( الثَّمَنَ مِنْ مَالِ السَّيِّدِ فَلَهُ اسْتِرْدَادُهُ ) لِأَنَّهُ لَمْ يَخْرُجْ عَنْ مِلْكِهِ ( وَلَا ضَمَانَ عَلَى سَيِّدِهِ ) وَإِنْ ( رَأَى الْمَبِيعَ ) مَثَلًا مَعَ عَبْدِهِ ( فَلَمْ يَأْخُذْهُ مِنْ الْعَبْدِ( 8 / 457 )
