( وَإِنْ بَاعَ ) الْمُشْتَرِي ( بَعْضَهُ ) أَيْ بَعْضَ الْمَبِيعِ غَيْرَ عَالِمٍ بِعَيْبِهِ ( فَلَهُ أَرْشُ الْبَاقِي ) الَّذِي لَمْ يَبِعْهُ ( لَا رَدُّهُ ) عَلَى الْبَائِعِ لِتَضَرُّرِهِ بِتَفْرِيقِ الْمَبِيعِ ( وَلَهُ ) أَيْ لِلْمُشْتَرِي أَيْضًا ( أَرْشُ ) الْبَعْضِ ( الْمَبِيعِ ) كَمَا لَوْ كَانَ بَاعَهُ كُلَّهُ وَإِنْ بَاعَ بَعْضَهُ عَالِمًا بِعَيْبِهِ فَكَمَا لَوْ بَاعَهُ كُلَّهُ عَلَى الْخِلَافِ السَّابِقِ .( 9 / 125 )
