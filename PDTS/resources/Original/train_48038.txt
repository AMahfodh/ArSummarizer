( وَإِنْ امْتَنَعَ مَنْ ) أَيْ زَوْجٌ أَوْ مُبِينٌ ( لَزِمَتْهُ سُكْنَى ) زَوْجَتِهِ أَوْ مُبَانَتِهِ الْحَامِلِ ( أُجْبِرَ ) أَيْ أَجْبَرَهُ الْحَاكِمُ بِطَلَبِ مَنْ وَجَبَتْ لَهَا كَسَائِرِ الْحُقُوقِ عَلَيْهِ ( وَإِنْ غَابَ ) مَنْ لَزِمَتْهُ السُّكْنَى ( اكْتَرَى عَنْهُ الْحَاكِمُ مِنْ مَالِهِ ) مَسْكَنًا لَهَا لِقِيَامِهِ مَقَامَهُ فِي أَدَاءِ مَا وَجَبَ عَلَيْهِ ( أَوْ اقْتَرَضَ ) الْحَاكِمُ ( عَلَيْهِ ) إنْ لَمْ يَجِدْ لَهُ مَالًا أُجْرَةَ الْمَسْكَنِ ( أَوْ فَرَضَ ) الْحَاكِمُ ( أُجْرَتَهُ ) أَيْ الْمَسْكَنِ لِتُؤْخَذَ مِنْهُ إذَا حَضَرَ .
