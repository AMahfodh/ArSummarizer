( فَإِنْ قَالَ ) لِمَنْ يَسْتَحْلِفُهُ ( قُلْ ) إنْ فَعَلْتُ كَذَا أَوْ إنْ لَمْ أَفْعَلْهُ أَوْ إنْ كُنْتُ فَعَلْتُهُ ( وَإِلَّا فَقَعِيدَةُ بَيْتِي الَّتِي يَجُوزُ عَلَيْهَا أَمْرِي طَالِقٌ وَهِيَ حَرَامٌ فَقَالَ وَنَوَى بِالْقَعِيدَةِ الْغِرَارَةَ ) فَلَهُ نِيَّتُهُ ( وَقَالَ فِي الْمُسْتَوْعِبِ نَسِيجُهُ ) أَيْ مَنْسُوجَةٌ ( تُنْسَجُ كَهَيْئَةِ الْعَيْبَةِ فَلَهُ نِيَّتُهُ ) لِأَنَّ اللَّفْظَ صَالِحٌ لِذَلِكَ .( 18 / 360 )
