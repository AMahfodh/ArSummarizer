( قَوْلُهُ ذَبَحَ شَاةً إلَخْ ) بَيَانٌ لِعَلَامَاتٍ أُخَرَ ( قَوْلُهُ وَلَمْ تَتَحَرَّكْ إلَخْ ) أَيْ بَعْدَ الذَّبْحِ بِحَرَكَةٍ اضْطِرَابِيَّةٍ كَحَرَكَةِ الْمَذْبُوحِ وَإِلَّا فَضَمُّ الْعَيْنِ وَقَبْضُ الرِّجْلِ حَرَكَةٌ ( قَوْلُهُ وَهَذَا كُلُّهُ إلَخْ ) أَعَادَهُ لِلدُّخُولِ عَلَى الْمَتْنِ ( قَوْلُهُ بِكُلِّ حَالٍ ) سَوَاءٌ وُجِدَتْ تِلْكَ الْعَلَامَاتِ أَوْ لَا( 26 / 198 )
