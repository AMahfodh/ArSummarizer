الْفَتْوَى بِأَنْ يَقُولَ إنِّي أَطْلُبُ مِنْك اسْتِحْلَالَ جَمِيعِ حُقُوقِك الْمُتَعَلِّقَةِ بِدَارِ الْآخِرَةِ إذْ لَوْ صَرَّحَ لَزِمَ الْوُقُوعُ فِي مَفْسَدَةٍ عَظِيمَةٍ ( { وَإِنَّ صَاحِبَ الْغِيبَةِ لَا يُغْفَرُ لَهُ } ) مِنْ جِهَتِهِ تَعَالَى ( { حَتَّى يَغْفِرَ } ) أَيْ يَتُوبَ وَيُحَلِّلَ ( { صَاحِبُهُ } ) يَشْكُلُ بِقَوْلِهِ تَعَالَى وَيَغْفِرُ مَا دُونَ ذَلِكَ لِمَنْ يَشَاءُ فَتَأَمَّلْ تَصِلْ .
