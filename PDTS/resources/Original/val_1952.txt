( أَوْ ) حَلَفَ ( لَا يَأْكُلُ السَّكَرَ حَنِثَ بِبَلْعِهِ بِمَضْغٍ وَغَيْرِهِ ) قَوْلُهُ وَغَيْرُهُ يُخَالِفُ مَا مَرَّ فِي الطَّلَاقِ كَمَا مَرَّ التَّنْبِيهُ عَلَيْهِ ( فَلَوْ وَضَعَهُ بِفِيهِ وَذَابَ وَابْتَلَعَهُ لَمْ يَحْنَثْ ) ؛ لِأَنَّهُ لَمْ يَأْكُلْهُ ( وَلَا يَحْنَثُ بِمَا اُتُّخِذَ مِنْهُ إلَّا إنْ نَوَى وَكَذَا الْحُكْمُ فِي التَّمْرِ وَالْعَسَلِ ) وَنَحْوِهِمَا ( فَإِنْ حَلَفَ لَا يَأْكُلُ الْعِنَبَ وَالرُّمَّانَ فَامْتَصَّهُمَا وَرَمَى الثِّقْلَ ) بِضَمِّ الْمُثَلَّثَةِ ( لَمْ يَحْنَثْ كَأَكْلِهِ ) أَوْ شُرْبِهِ ( عَصِيرَهُمَا ) ؛ لِأَنَّ ذَلِكَ لَا يُسَمَّى أَكْلًا لَهُمَا وَمِثْلُهُمَا كُلُّ مَا يُمَصُّ
