أَنْ يَأْتِيَ مَكَانَ أُمِّهِ وَهِيَ فِي حِرْزِ مَالِكِهَا حَتَّى يَسْتَتْبِعَ الْأُمَّ سَخْلَهَا بِأَنْ يَبْعَثُهُ عَلَيْهَا حَتَّى تَتْبَعُهُ قُطِعَ ) لِتَسَبُّبِهِ فِي أَخْذِ ذَلِكَ وَ ( لَا ) يُقْطَعُ ( إنْ تَبِعَهَا ) السَّخْلُ ( مِنْ غَيْرِ اسْتِتْبَاعٍ ) لِأَنَّهُ لَيْسَ مِنْ فِعْلِهِ ( وَإِنْ تَطَيَّبَ فِي الْحِرْزِ بِمَا لَوْ اجْتَمَعَ بَعْدَ تَطْيِيبِهِ وَ ) بَعْدَ ( خُرُوجِهِ مِنْ الْحِرْزِ لَبَلَغَ نِصَابًا ) قُطِعَ لِأَنَّهُ هَتَكَ الْحِرْزَ وَأَخْرَجَ مِنْهُ نِصَابًا أَشْبَهَ مَا لَوْ كَانَ غَيْرَ طَيِّبٍ ( أَوْ هَتَكَ الْحِرْزَ وَأَخَذَ الْمَالَ وَقْتًا آخَرَ ) وَقَرَّبَ مَا بَيْنَهُمَا قُطِعَ لِأَنَّهَا سَرِقَةٌ وَاحِدَةٌ ( أَوْ ) هَتْكَ الْحِرْزَ وَ ( أَخَذَ بَعْضَهُ ) أَيْ الْمَالَ ( ثُمَّ أَخَذَ بَقِيَّتَهُ وَقَرَّبَ مَا بَيْنَهُمَا ) قُطِعَ لِأَنَّهَا سَرِقَةٌ وَاحِدَةٌ وَلِأَنَّهُ إذَا بَنَى فِعْلَ أَحَدِ الشَّرِيكَيْنِ عَلَى فِعْل شَرِيكِهِ إذَا سَرَقَا نِصَابًا فَبِنَاءُ فِعْلِ الْوَاحِدِ بَعْضُهُ عَلَى بَعْضٍ أَوْلَى ( أَوْ فَتَحَ أَسْفَلَ كُوَّارَةٍ فَخَرَجَ الْعَسَلُ شَيْئًا فَشَيْئًا ) حَتَّى بَلَغَتْ قِيمَةُ مَا أَخْرَجَهُ نِصَابًا قُطِعَ لِأَنَّهُ لَمْ يُهْمِلْ الْأَخْذُ أَشْبَهَ مَا لَوْ وَجَدَهُ مَجْمُوعًا فَأَخْرَجَهُ ( أَوْ أَخْرَجَهُ ) أَيْ النِّصَابَ الْمَسْرُوقَ ( إلَى سَاحَةِ دَارٍ أَوْ ) سَاحَةِ ( خَانٍ مِنْ بَيْتٍ مُغْلَقٍ مِنْ الدَّارِ أَوْ الْخَانِ ) سَوَاء ( فَتَحَهُ ) أَيْ الْبَيْتَ ( أَوْ نَقَّبَهُ ) وَلَوْ أَنَّ بَابَ الدَّارِ أَوْ الْخَانِ مُغْلَقٌ قُطِعَ لِأَنَّهُ هَتَكَ الْحِرْزَ وَأَخْرَجَ مِنْهُ نِصَابًا كَمَا لَوْ لَمْ يَكُنْ عَلَى الدَّارِ أَوْ الْخَانِ بَابُ آخَرُ ( أَوْ احْتَلَبَ لَبَنًا مِنْ مَاشِيَةٍ فِي الْحِرْزِ وَأَخْرَجَهُ ) مِنْ الْحِرْزِ ( قُطِعَ ) لِسَرِقَتِهِ نِصَابًا كَغَيْرِ اللَّبَنِ ( فَإِنْ شَرِبَ اللَّبَنَ فِي الْحِرْزِ أَوْ شَرِبَ مِنْهُ فَانْتَقَصَ النِّصَابُ ) لَمْ يُقْطَعْ لِأَنَّهُ لَمْ يُخْرِجْ نِصَابًا مِنْ الْحِرْزِ ( أَوْ تَرَكَ الْمَتَاعَ فِي مَاءٍ رَاكِدٍ فَانْفَتَحَ مِنْ غَيْر فِعْله فَخَرَجَ بِهِ ) لَمْ يُقْطَعْ لِأَنَّ خُرُوجَهُ بِغَيْرِ فِعْلِهِ ( أَوْ أَخْرَجَ النِّصَابَ فِي( 21 / 8 )
