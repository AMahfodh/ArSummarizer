( بِخِلَافِ فِطْرِهِ ) فِي نَذْرِ السَّنَةِ الْمُعَيَّنَةِ بِتَسْمِيَتِهَا أَوْ إشَارَةٍ إلَيْهَا ( لِسَفَرٍ ) أَوْ إكْرَاهٍ أَوْ نِسْيَانٍ فَيُوجِبُ عَلَيْهِ قَضَاءَ مَا أَفْطَرَهُ لَهَا ( وَ ) وَجَبَ ( صَبِيحَةُ ) أَيْ : صَوْمُ يَوْمِ لَيْلَةِ ( الْقُدُومِ ) بِضَمِّ الْقَافِ أَيْ : قُدُومِ شَخْصِ مِنْ سَفَرِهِ مَثَلًا ، وَهَذَا بَيَانٌ لِلْحُكْمِ وَبَيَّنَ الْمَسْأَلَةَ بِقَوْلِهِ ( فِي ) نَذْرِ صَوْمِ ( يَوْمِ قُدُومِهِ ) أَيْ : زَيْدٌ الْمُسَافِرُ مَثَلًا ( إنْ قَدِمَ )( 4 / 92 )
