( فَلَوْ ) جَنَى رَقِيقٌ عَلَى الرَّقِيقِ الْمَرْهُونِ وَ ( وَجَبَ قِصَاصٌ اقْتَصَّ الرَّاهِنُ ) مِنْهُ أَوْ عُفِيَ مَجَّانًا ( وَفَاتَ الرَّهْنُ ) لِفَوَاتِ مَحَلِّهِ مِنْ غَيْرِ بَدَلٍ هَذَا إنْ كَانَتْ الْجِنَايَةُ فِي النَّفْسِ ، فَإِنْ كَانَتْ فِي طَرَفٍ أَوْ نَحْوِهِ فَالرَّاهِنُ بَاقٍ بِحَالِهِ ، وَلَوْ أَعْرَضَ الرَّاهِنُ عَنْ الْقِصَاصِ وَالْعَفْوِ بِأَنْ سَكَتَ عَنْهُمَا لَمْ يُجْبَرْ عَلَى أَحَدِهِمَا ( فَإِنْ وَجَبَ الْمَالُ بِعَفْوِهِ ) عَنْ الْقِصَاصِ عَلَيْهِ ( أَوْ بِجِنَايَةٍ خَطَأٍ ) أَوْ شَبَهِ عَمْدٍ أَوْ عَمْدٍ يُوجِبُ مَالًا لِانْتِفَاءِ الْمُكَافَأَةِ مَثَلًا صَارَ الْمَالُ مَرْهُونًا وَإِنْ لَمْ يَقْبِضْ كَمَا مَرَّ وَ ( لَمْ يَصِحَّ عَفْوُهُ ) أَيْ الرَّاهِنُ عَنْهُ لِتَعَلُّقِ حَقِّ الْمُرْتَهِنِ بِهِ ( وَلَا ) يَصِحُّ ( إبْرَاءُ الْمُرْتَهِنِ الْجَانِي ) لِأَنَّهُ غَيْرُ مَالِكٍ وَلَا يَسْقُطُ بِإِبْرَائِهِ حَقُّهُ مِنْ الْوَثِيقَةِ إلَّا إنْ أَسْقَطَهُ مِنْهَا .
