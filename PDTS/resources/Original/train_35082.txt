وَتَقَدَّمَ فِي الْحَجِّ ( أَوْ ) مِنْ ( أُجْرَةٍ ) إنْ صَحَّتْ الْإِجَارَةُ لِلْحَجِّ ( وَالْبَقِيَّةُ ) أَيْ بَقِيَّةُ أَلْفٍ بَعْدَ نَفَقَةِ مِثْلِهِ أَوْ أُجْرَةٍ ( لِلْوَرَثَةِ ) لِبُطْلَانِ مَحَلِّ الْوَصِيَّةِ بِامْتِنَاعِ الْمُعَيَّنِ لِلْحَجِّ كَمَا لَوْ وَصَّى بِهِ لِإِنْسَانٍ فَرَدَّ الْوَصِيَّةَ ( فِي ) حَجِّ ( فَرْضٍ وَنَفْلٍ ، فَإِنْ لَمْ يَمْتَنِعْ ) الْمُعَيَّنُ مِنْ الْحَجِّ ( أَعْطَى الْأَلْفَ ) لِأَنَّهُ مُوصًى لَهُ بِالزِّيَادَةِ بِشَرْطِ حَجِّهِ .
