( قَوْلُهُ لِشَكٍّ ) أَيْ مُطْلَقِ تَرَدُّدٍ مَعَ وُجُوبِ الْإِخْرَاجِ حِينَئِذٍ فَلَوْ بَقِيَ عَلَى شَكِّهِ لَمْ يُكَرِّرْ وَكَذَا إنْ تَحَقَّقَ بَعْدَ الْإِخْرَاجِ مَوْتَهُ قَبْلَ الْإِخْرَاجِ لَمْ يَجِبْ التَّكْرَارُ ( قَوْلُهُ لَا يَجِبُ إلَّا بَعْدَ تَحَقُّقِ مَوْتِ الصَّيْدِ ) فِيهِ نَظَرٌ لِمَا عَلِمْت مِنْ قَوْلِ الْمُصَنِّفِ وَتَعْرِيضُهُ لِلتَّلَفِ وَجَرْحُهُ وَلَمْ تَتَحَقَّ سَلَامَتُهُ ( قَوْلُهُ لِأَنَّهُ أُخْرِجَ قَبْلَ الْوُجُوبِ ) أَيْ بِحَسَبِ نَفْسِ الْأَمْرِ لَا بِحَسَبِ الظَّاهِرِ لِمَا تَقَدَّمَ أَنَّهُ يَجِبُ عِنْدَ الشَّكِّ أَيْ أَنَّهُ كَشَفَ الْغَيْبَ أَنَّهُ أُخْرِجَ قَبْلَ الْوُجُوبِ ( قَوْلُهُ تَحَقُّقِ مَوْتِهِ ) أَيْ حُصُولُ مَوْتِهِ بَعْدَ الْإِخْرَاجِ وَلَا بُدَّ مِنْ هَذَا التَّقْدِيرِ وَإِلَّا لَمْ يَنْفَعْ هَذَا بِشَيْءٍ ( قَوْلُهُ قَدْ يَكُونُ بِمَوْتٍ مُتَقَدِّمٍ ) أَيْ مَعَ أَنَّهُ إذَا تَحَقَّقَ مَوْتُهُ قَبْلَ الْإِخْرَاجِ لَا يَتَكَرَّرُ قَوْلُهُ إلَّا أَنْ تَكُونَ ضَرْبَةُ غَيْرِهِ هِيَ الَّتِي عَاقَتْهُ ) أَيْ بِأَنْ يَكُونَ ضَرْبُهُ أَوَّلًا عَاقَهُ عَنْ كَوْنِهِ يَنْجُو بِنَفْسِهِ ، ثُمَّ ضَرَبَهُ إنْسَانٌ بَعْدَ ذَلِكَ ضَرْبَةً مَاتَ بِهَا فَكُلُّ وَاحِدٍ مِنْهُمَا عَلَيْهِ جَزَاءٌ بِمَثَابَةِ الْمُشْتَرِكَيْنِ .( 8 / 199 )
