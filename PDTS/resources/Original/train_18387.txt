[ فَرْعٌ ] لَا يَجِبُ عَلَى الْبَائِعِ تَسْلِيمُ الْمَبِيعِ وَلَا عَلَى الْمُشْتَرِي تَسْلِيمُ الثَّمَنِ فِي زَمَنِ الْخِيَارِ ، فَلَوْ تَبَرَّعَ أَحَدُهُمَا بِالتَّسْلِيمِ لَمْ يَبْطُلْ خِيَارُهُ ، وَلَا يُجْبَرُ الْآخَرُ عَلَى تَسْلِيمِ مَا عِنْدَهُ وَلَهُ اسْتِرْدَادُ الْمَدْفُوعِ إلَيْهِ ا ه ( قَوْلُهُ وَأَعَارَهَا لِلْمُشْتَرِي قَبْلَ الْقَبْضِ ) أَيْ فَلَيْسَ لَهُ اسْتِرْدَادُهَا وَيَكُونُ تَسْلِيمُهُ عَنْ الْإِعَارَةِ إقْبَاضًا لِأَنَّهُ سَلَّطَهُ عَلَى الْعَيْنِ كَمَا يَأْتِي فِي الْفَرْقِ بَيْنَ الْإِعَارَةِ وَالْإِيدَاعِ ( قَوْلُهُ : وَلَوْ أَوْدَعَهُ ) أَيْ الْبَائِعُ لَهُ : أَيْ لِلْمُشْتَرِي ( قَوْلُهُ : كَتَلَفِهِ فِي يَدِ الْبَائِعِ ) أَيْ فَيَنْفَسِخُ الْبَيْعُ وَيَسْقُطُ الثَّمَنُ عَنْ الْمُشْتَرَى ( قَوْلُهُ وَلَهُ ) أَيْ الْبَائِعِ اسْتِرْدَادُهُ إلَخْ ( قَوْلُهُ : زُيُوفًا ) وَمِنْهُ مَا لَوْ بَانَ فِي الدَّرَاهِمِ وَلَوْ لِبَعْضٍ مِنْهَا وَإِنْ قَلَّ قَصٌّ فَإِنَّهُ يَرُدُّ وَيَأْخُذُ جَيِّدًا فَلَهُ اسْتِرْدَادُ الْمَبِيعَ لِأَجْلِ ذَلِكَ ( قَوْلُهُ : أَنَّ الِاعْتِبَارَ بِالْعَاقِدِ ) مُعْتَمَدٌ ( قَوْلُهُ : وَلِكُلٍّ مِنْهُمَا ) أَيْ وَالْحَالُ أَنَّ لِكُلٍّ إلَخْ ( قَوْلُهُ : بِنَاءً عَلَى أَنَّ الصَّفْقَةَ إلَخْ ) مُعْتَمَدٌ .( 12 / 389 )
