( فَإِنْ وَطِئَ ) الْمُشْتَرَاةَ أَوْ تَلَذَّذَ بِهَا بِدُونِ وَطْءٍ وَقَفَ عَنْهُمَا وَحَرُمَ إحْدَاهُمَا ، فَإِنْ أَبْقَى الثَّانِيَةَ اسْتَبْرَأَهَا ( أَوْ عَقَدَ ) النِّكَاحَ عَلَى الْأُخْتِ مَثَلًا ( بَعْدَ تَلَذُّذِهِ ) بِمُقَدِّمَةِ جِمَاعٍ فَمَا زَادَ عَلَيْهَا ( بِأُخْتِهَا ) أَيْ الْمَعْقُودِ عَلَيْهَا النِّكَاحُ وَهَذِهِ لِلتَّعَدِّيَةِ ( بِ ) سَبَبِ ( مِلْكٍ ) لِلْأُخْتِ السَّابِقَةِ ( فَ ) حُكْمُهُ فِي الصُّورَتَيْنِ ( كَ ) حُكْمِ ( الْأَوَّلِ ) أَيْ وَاطِئِ مُحَرَّمَتَيْ الْجَمْعِ بِمِلْكٍ مِنْ إيقَافِهِ عَنْهُمَا( 6 / 400 )
