( وَإِنْ عَتَقَتْ ) إحْدَاهُنَّ ( ثُمَّ أَسْلَمَتْ ثُمَّ أَسْلَمْنَ ) أَيْ الْبَوَاقِي تَعَيَّنَتْ الْأُولَى إنْ كَانَتْ تُعِفُّهُ ؛ لِأَنَّ تَحْتَهُ حُرَّةً عِنْدَ اجْتِمَاعِهِمَا عَلَى الْإِسْلَامِ ( أَوْ عَتَقَتْ ) وَاحِدَةٌ مِنْ الْإِمَاءِ ( ثُمَّ أَسْلَمْنَ ) أَيْ الْبَوَاقِيَ ( ثُمَّ أَسْلَمَتْ ) الْعَتِيقَةُ تَعَيَّنَتْ إنْ كَانَتْ تُعِفُّهُ لِمَا تَقَدَّمَ ( أَوْ عَتَقَتْ بَيْنَ إسْلَامِهِ وَإِسْلَامِهَا ) كَأَنْ أَسْلَمَ ثُمَّ عَتَقَتْ ثُمَّ أَسْلَمَتْ ثُمَّ أَسْلَمَ( 8 / 476 )
