( وَ ) ذَكَرَ ( صِفَتَهُ ) كَسَلَمٍ ( كَانَ أَوْلَى ) لِصَيْرُورَتِهِ مَعْلُومًا بِالْوَصْفِ ( وَالْحَصَادُ وَنَحْوُهُ ) كَالْجُذَاذِ وَاللَّقَاطِ فَمَا قُلْنَا أَنَّهُ لِلْبَائِعِ وَنَحْوِهِ ( عَلَى الْبَائِعِ ) وَنَحْوِهِ لِأَنَّ ذَلِكَ مِنْ مُؤْنَةِ نَقْلِ مِلْكِهِ فَهُوَ كَنَقْلِ الطَّعَامِ الْمَبِيعِ .( 9 / 327 )
