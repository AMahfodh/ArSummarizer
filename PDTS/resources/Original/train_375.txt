( وَ النَّذْرُ الْمُنْعَقِدُ أَقْسَامُهُ ) سِتَّةٌ ( أَحَدُهَا ) النَّذْرُ ( الْمُطْلَقُ كَعَلَيَّ نَذْرٌ أَوْ لِلَّهِ عَلِيَّ نَذْرٌ ) سَوَاءٌ ( أَطْلَقَ أَوْ قَالَ إنْ فَعَلْتُ كَذَا ) وَفَعَلَهُ ( وَلَمْ يَنْوِ ) بِنَذْرِهِ ( شَيْئًا ) مُعَيَّنًا ( فَيَلْزَمهُ كَفَّارَةُ يَمِينٍ ) لِحَدِيثِ عُقْبَةَ بْنِ عَامِرٍ مَرْفُوعًا : { كَفَّارَةُ النَّذْرِ إذَا لَمْ يُسَمَّ كَفَّارَةَ يَمِينٍ } رَوَاهُ ابْنُ مَاجَهْ وَالتِّرْمِذِيُّ وَقَالَ حَسَنٌ صَحِيحٌ غَرِيبٌ .
