( وَمَنْ خَلَّفَتْ ابْنًا وَعَصَبَةً ) مِنْ إخْوَةٍ وَأَعْمَامٍ ( وَلَهَا عَتِيقٌ فَوَلَاؤُهُ ) أَيْ : الْعَتِيقِ ( وَإِرْثُهُ لِابْنِهَا إنْ لَمْ يَحْجُبْهُ ) أَيْ : ابْنَهَا ( نَسِيبٌ ) لِلْعَتِيقِ ؛ لِأَنَّهُ أَقْرَبُ عَصَبَتِهَا ( وَعَقْلُهُ ) أَيْ : الْعَتِيقِ ( عَلَيْهِ ) أَيْ : الِابْنِ ( وَعَلَى عَصَبَتِهَا ) لِحَدِيثِ أَحْمَدَ عَنْ زِيَادِ بْنِ أَبِي مَرْيَمَ وَتَقَدَّمَ ( فَإِنْ بَادَ ) أَيْ : انْقَرَضَ ( بَنُوهَا ) وَإِنْ سَفُلُوا فَوَلَاءُ عَتِيقِهَا ( لِعَصَبَتِهَا دُونَ عَصَبَتِهِمْ ) أَيْ : بَنِيهَا ؛ لِأَنَّ الْوَلَاءَ لَا يُورَثُ وَعَنْ إبْرَاهِيمَ قَالَ : ( اخْتَصَمَ عَلِيٌّ وَالزُّبَيْرُ فِي مَوْلَى صَفِيَّةَ فَقَالَ عَلِيٌّ : مَوْلَى عَمَّتِي وَأَنَا أَعْقِلُ عَنْهُ .
