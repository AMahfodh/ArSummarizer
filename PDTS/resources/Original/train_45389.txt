وَ ( هِيَ ) أَيْ : الْوَدِيعَةُ ( بَعْدَهُ ) أَيْ : بَعْدَ عَزْلِهِ نَفْسَهُ ( أَمَانَةٌ حُكْمُهَا ) مَا دَامَتْ ( فِي يَدِهِ حُكْمُ الثَّوْبِ الَّذِي أَطَارَتْهُ الرِّيحُ إلَى دَارِهِ ) ؛ لِأَنَّهُ لَمْ يَتَعَدَّ بِوَضْعِ يَدِهِ عَلَيْهَا وَإِذْنُ رَبِّهَا لَهُ فِي حِفْظِهَا بَطَلَ بِعَزْلِهِ نَفْسَهُ ( يَجِبُ ) عَلَيْهِ ( رَدُّهُ ) إلَى رَبِّهِ فَوْرًا مَعَ التَّمَكُّنِ ، لِعَدَمِ إذْنِ رَبِّهِ فِي بَقَائِهِ بِيَدِهِ ( فَإِنْ تَلِفَ ) الْمَالُ الْمُودَعُ عِنْدَ الْوَدِيعِ بَعْدَ عَزْلِهِ نَفْسِهِ ، أَوْ الثَّوْبُ الَّذِي أَطَارَتْهُ الرِّيحُ إلَى دَارِهِ ( قَبْلَ التَّمَكُّنِ مِنْ رَدِّهِ فَهَدَرٌ ) لَا ضَمَانَ فِيهِ ، وَفُهِمَ مِنْهُ أَنَّهُ إنْ تَلِفَ بَعْدَ تَمَكُّنِهِ مِنْ رَدِّهِ أَنَّهُ يَضْمَنُهُ ؛ لِأَنَّهُ مُتَعَدٍّ بِإِمْسَاكِهِ فَوْقَ مَا يَتَمَكَّنُ فِيهِ مِنْ الرَّدِّ .( 13 / 376 )
