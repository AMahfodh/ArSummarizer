ثُمَّ شَرَعَ فِي الْكَلَامِ عَلَى مَا يَنُوبُ فِي الصُّغْرَى عَنْ بَعْضٍ مَخْصُوصٍ وَهُوَ مَسْحُ الْخُفِّ فَقَالَ دَرْسٌ [ فَصْلٌ ] ( رُخِّصَ ) جَوَازًا بِمَعْنَى خِلَافُ الْأَفْضَلِ إذْ الْأَفْضَلُ الْغُسْلُ ( لِرَجُلٍ وَامْرَأَةٍ ) غَيْرِ مُسْتَحَاضَةٍ بَلْ ( وَإِنْ ) كَانَتْ ( مُسْتَحَاضَةً ) لَازَمَهَا الدَّمُ نِصْفَ الزَّمَنِ فَأَكْثَرَ ( بِحَضَرٍ أَوْ سَفَرٍ ) الْبَاءُ ظَرْفِيَّةٌ مُتَعَلِّقَةٌ بِمَسَحَ ( مَسْحِ جَوْرَبٍ ) نَائِبُ فَاعِلٍ رُخِّصَ بِتَضْمِينِهِ أُبِيحَ أَوْ أُجِيزَ وَإِلَّا فَرُخِّصَ إنَّمَا يَتَعَدَّى لِلْمُرَخَّصِ فِيهِ بِفِي وَلِلْمُرَخَّصِ لَهُ بِاللَّامِ نَحْو رُخِّصَ لِرَجُلٍ فِي مَسْحِ جَوْرَبٍ وَهُوَ مَا كَانَ عَلَى شَكْلِ الْخُفِّ مِنْ نَحْوِ قُطْنٍ ( جِلْدٍ ظَاهِرُهُ ) وَهُوَ مَا يَلِي السَّمَاءَ ( وَبَاطِنُهُ ) وَهُوَ مَا يَلِي الْأَرْضَ وَلَيْسَ الْمُرَادُ بِالظَّاهِرِ مَا فَوْقَ الْقَدَمِ وَبِالْبَاطِنِ مَا تَحْتَ الْقَدَمِ الْمُبَاشِرِ لِلرِّجْلِ مِنْ دَاخِلِهِ إذْ هَذَا لَا يَجُوزُ الْمَسْحُ عَلَيْهِ كَمَا يَأْتِي فِي قَوْلِهِ بِلَا حَائِلٍ
