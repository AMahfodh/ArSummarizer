( وَ ) كُرِهَ ( تَكْرَارُهُ ) أَيْ مَسْحِ الْخُفِّ أَوْ الْجَوْرَبِ لِأَنَّهُ غُلُوٌّ فِي الدِّينِ وَلَوْ جَفَّتْ يَدُهُ أَثْنَاءَ مَسْحِهِ فَلَا يُجَدِّدُ الْبَلَلَ لِتَكْمِيلِ مَسْحِ الرِّجْلِ الَّتِي جَفَّتْ يَدُهُ حَالَ مَسْحِهَا وَيُجَدِّدُهُ لِلْأُخْرَى ( وَ ) كُرِهَ ( تَتَبُّعُ غُضُونِهِ ) بِضَمِّ الْغَيْنِ وَالضَّادِ الْمُعْجَمَتَيْنِ أَيْ تَكَامِيشَ الْخُفِّ لِأَنَّهُ غُلُوٌّ فِي الدِّينِ وَشَأْنُ الْمَسْحِ التَّخْفِيفُ .( 1 / 296 )
