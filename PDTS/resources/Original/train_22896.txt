( وَ ) إنْ قَالَ مَنْ بِيَدِهِ نَحْوُ عَبْدٍ ( لَهُ ) ؛ أَيْ : لِفُلَانٍ ( عَلَيَّ فِيهِ ) أَلْفٌ ، أَوْ قَالَ لَهُ عَلَيَّ ( مِنْهُ أَلْفٌ ) صَحَّ إقْرَارُهُ ( وَقِيلَ لَهُ فَسِّرْ ) سَبَبَهُ ؛ لِأَنَّهُ لَا يَعْلَمُ إلَّا مِنْ جِهَتِهِ ( وَيُقْبَلُ تَفْسِيرُهُ بِجِنَايَةِ ) الْعَبْدِ عَلَى الْمُقِرِّ لَهُ ؛ وَيُقْبَلُ تَفْسِيرُهُ ( بِقَوْلِهِ نَقَدَهُ ) ؛ أَيْ : الْأَلْفَ ( فِي ثَمَنِهِ ) ؛ أَيْ : الْعَبْدَ وَنَحْوَهُ ( أَوْ ) ؛ أَيْ : وَيُقْبَلُ تَفْسِيرُهُ بِقَوْلِهِ نَقْدُهُ ( اشْتَرَى ) الْمُقَرُّ لَهُ ( رُبْعَهُ ) ؛ أَيْ : الْعَبْدِ وَنَحْوِهِ ( بِهِ ) ؛ أَيْ : الْأَلْفَ أَوْ بِقَوْلِهِ ( لَهُ فِيهِ شِرْكٌ ) أَوْ بِقَوْلِهِ : إنَّ مُورِثِي أَوْصَى لَهُ بِأَلْفٍ مِنْ ثَمَنِهِ ، ( وَلَا ) يُقْبَلُ تَفْسِيرُهُ ( بِأَنَّهُ رَهَنَهُ عِنْدَهُ بِهِ ) ؛ أَيْ : الْأَلْفَ ؛ لِأَنَّ حَقَّهُ فِي الذِّمَّةِ .
