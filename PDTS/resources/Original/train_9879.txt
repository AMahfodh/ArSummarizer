( وَلَوْ قَصَدَهَا ) : أَيْ السَّجْدَةَ بَعْدَ قِرَاءَةِ مَحَلِّهَا وَانْخَفَضَ بِنِيَّتِهَا ( فَرَكَعَ سَاهِيًا ) عَنْهَا ( اُعْتُدَّ بِهِ ) : أَيْ بِرُكُوعِهِ ( عِنْدَ مَالِكٍ ) - بِنَاءً عَلَى أَنَّ الْحَرَكَةَ لِلرُّكْنِ لَا تُشْتَرَطُ - ( لَا ) عِنْدَ ( ابْنِ الْقَاسِمِ ) فَلَا يُعْتَدُّ بِهِ عِنْدَهُ ، وَإِذَا لَمْ يُعْتَدَّ بِهِ ( فَيَخِرُّ ) إذَا تَذَكَّرَ ( سَاجِدًا وَلَوْ بَعْدَ رَفْعِهِ ) مِنْ رُكُوعِهِ ثُمَّ يَأْتِي بِالرُّكُوعِ .
