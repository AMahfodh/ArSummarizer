( وَلَوْ بَاعَهُ ) أَيْ الْعَبْدَ ( الْوَصِيُّ لِأَجْلِهِمْ ) أَيْ لِأَجْلِ الْغُرَمَاءِ ( بِأَمْرِ الْقَاضِي ) لَهُ بِالْبَيْعِ وَقَبَضَ ثَمَنَهُ ( ثُمَّ اُسْتُحِقَّ ) الْعَبْدُ ( أَوْ مَاتَ قَبْلَ قَبْضِهِ ) أَيْ قَبْضِ الْمُشْتَرِي مِنْ الْوَصِيِّ ( وَضَاعَ الْمَالُ ) أَيْ ثَمَنُ الْعَبْدِ ( رَجَعَ الْمُشْتَرِي ) بِالثَّمَنِ ( عَلَى الْوَصِيِّ ) لِأَنَّهُ عَاقِدٌ نِيَابَةً عَنْ الْمَيِّتِ فَتَرْجِعُ الْحُقُوقُ إلَيْهِ كَمَا إذَا وَكَّلَهُ حَالَ حَيَاتِهِ ، وَكَذَا الْوَصِيُّ الَّذِي نَصَّبَهُ الْقَاضِي لِأَنَّهُ نَصَّبَهُ لِيَكُونَ قَائِمًا مَقَامَ الْمَيِّتِ ( وَهُوَ ) أَيْ الْوَصِيُّ يَرْجِعُ ( عَلَى الْغُرَمَاءِ ) لِأَنَّهُ عَامِلٌ لَهُمْ وَمَنْ عَمِلَ عَمَلًا لِغَيْرِهِ وَلَحِقَهُ بِسَبَبِهِ ضَمَانٌ يَرْجِعُ بِهِ مَنْ يَقَعُ لَهُ الْعَمَلُ .
