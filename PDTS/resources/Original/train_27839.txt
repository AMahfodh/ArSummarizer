( وَقُرَيْشٌ قِيلَ بَنُو النَّضْرِ بْنُ كِنَانَةَ ) قَدَّمَهُ فِي الشَّرْحِ وَالْمُبْدِعِ وَالْإِقْنَاعِ وَغَيْرِهِمَا وَجَزَمَ بِهِ الْمُوَفَّقُ فِي التَّبْيِينِ ( وَقِيلَ بَنُو فِهْرِ بْنِ مَالِكِ بْنِ النَّضْرِ ) بْنِ كِنَانَةَ ( ثُمَّ بِأَوْلَادِ الْأَنْصَارِ ) وَهُمْ الْأَوْسُ وَالْخَزْرَجُ قُدِّمُوا عَلَى غَيْرِهِمْ لِسَابِقَتِهِمْ فِي الْإِسْلَامِ وَإِذَا اسْتَوَى اثْنَانِ ) فِيمَا سَبَقَ ( فَأَسْبَقُ بِإِسْلَامٍ فَأَسَنُّ فَأَقْدَمُ هِجْرَةً وَسَابِقَةً وَيُفَضِّلُ ) بَيْنَهُمْ أَيْ أَهْلِ الْعَطَاءِ ( بِسَابِقَةٍ ) فِي إسْلَامٍ ( وَنَحْوِهَا ) كَسَبْقٍ بِهِجْرَةٍ لِأَنَّ عُمَرَ قَسَمَ بَيْنَهُمْ عَلَى السَّوَابِقِ وَقَالَ لَا أَجْعَلُ مَنْ قَاتَلَ عَلَى الْإِسْلَامِ كَمَنْ قُوتِلَ عَلَيْهِ وَفَضَّلَ عُمَرُ وَعُثْمَانُ وَلَمْ يُفَضِّلْ أَبُو بَكْرٍ وَعَلِيٌّ( 4 / 255 )
