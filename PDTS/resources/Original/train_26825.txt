وَرَوَاهُ النَّسَائِيُّ وَزَادَ { عَلَيْكُمْ بِرُخْصَةِ اللَّهِ الَّتِي رَخَّصَ لَكُمْ فَاقْبَلُوهَا } وَإِنْ صَامَ أَجْزَأَهُ نَصًّا لِحَدِيثِ { هِيَ رُخْصَةٌ مِنْ اللَّهِ فَمَنْ أَخَذَ بِهَا فَحَسَنٌ وَمَنْ أَحَبَّ أَنْ يَصُومَ فَلَا جُنَاحَ عَلَيْهِ } رَوَاهُ مُسْلِمٌ وَالنَّسَائِيُّ .( 3 / 324 )
