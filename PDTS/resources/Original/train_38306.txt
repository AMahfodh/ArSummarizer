( وَلَا يَصِحُّ بَيْعُ عِنَبٍ ) أَوْ زَبِيبٍ وَنَحْوِهِ ( أَوْ عَصِيرٍ لِمُتَّخِذِهِ خَمْرًا ) وَلَوْ ذِمِّيًّا ( وَلَا ) بَيْعُ ( سِلَاحٍ وَنَحْوِهِ ) كَتُرْسٍ وَدَرْعٍ ( فِي فِتْنَةٍ ، أَوْ لِأَهْلِ حَرْبٍ ، أَوْ قُطَّاعِ طَرِيقٍ مِمَّنْ عَلِمَ ذَلِكَ ) مِمَّنْ يَشْتَرِيهِ ( وَلَوْ بِقَرَائِنَ ، وَلَا ) بَيْعُ ( مَأْكُولٍ وَمَشْرُوبٍ وَمَشْمُومٍ وَقَدَحٍ لِمَنْ يَشْرَبُ عَلَيْهِ ) أَيْ الْمَأْكُولِ أَوْ الْمَشْرُوبِ أَوْ الْمَشْمُومِ مُسْكِرًا ( أَوْ ) يَشْرَبُ ( بِهِ ) أَيْ الْقَدَحِ ( مُسْكِرًا ، وَ ) لَا بَيْعُ ( جَوْزٍ وَبَيْضٍ وَنَحْوِهِمَا ) كَبُنْدُقٍ ( لِقِمَارٍ .
