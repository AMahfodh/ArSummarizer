قَالَ الْمَاوَرْدِيُّ : يُرِيدُ بِالضَّمَانِ وَاَللَّهُ أَعْلَمُ أَنَّهُ يَتَحَمَّلُ سَهْوَ الْمَأْمُومِ ، وَلِأَنَّ مُعَاوِيَةَ شَمَّتَ الْعَاطِسَ خَلْفَ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ وَلَمْ يَسْجُدْ وَلَا أَمَرَهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ بِالسُّجُودِ ( فَلَوْ ) ( ظَنَّ سَلَامَهُ ) أَيْ الْإِمَامِ ( فَسَلَّمَ ) الْمَأْمُومُ ( فَبَانَ خِلَافُهُ ) أَيْ خِلَافُ مَا ظَنَّهُ ( سَلَّمَ مَعَهُ ) أَيْ بَعْدَهُ كَمَا عُلِمَ مِمَّا مَرَّ أَنَّهُ الْأَوْلَى إذْ سَلَامُهُ قَبْلَ سَلَامِ إمَامِهِ مُمْتَنِعٌ ( وَلَا سُجُودَ ) لِسَهْوِهِ حَالَ الْقُدْوَةِ فَيَتَحَمَّلُهُ الْإِمَامُ ( وَلَوْ ذَكَرَ ) الْمَأْمُومُ ( فِي تَشَهُّدِهِ ) أَوْ قَبْلَهُ أَوْ بَعْدَهُ ( تَرْكَ رُكْنٍ غَيْرِ ) سَجْدَةٍ مِنْ الْأَخِيرَةِ كَمَا مَرَّ فِي التَّرْتِيبِ وَغَيْرِ ( النِّيَّةِ وَالتَّكْبِيرَةِ ) لِلتَّحَرُّمِ أَوْ شَكَّ فِيهِ امْتَنَعَ عَلَيْهِ تَدَارُكُهُ مَعَ بَقَاءِ الْقُدْوَةِ لِمَا فِيهِ مِنْ تَرْكِ الْمُتَابَعَةِ الْوَاجِبَةِ وَ ( قَامَ بَعْدَ سَلَامِ إمَامِهِ إلَى رَكْعَتِهِ ) الْفَائِتَةِ بِفَوَاتِ الرُّكْنِ كَمَا عُلِمَ مِمَّا مَرَّ ( وَلَا يَسْجُدُ ) فِي التَّذَكُّرِ لِوُقُوعِ السَّهْوِ حَالَ الْقُدْوَةِ ، بِخِلَافِ مَا لَوْ شَكَّ فِي فِعْلِهِ بَعْدَ انْقِضَاءِ الْقُدْوَةِ فَيَتَدَارَكُ ذَلِكَ وَيَسْجُدُ لِلسَّهْوِ كَمَا فِي التَّحْقِيقِ ؛ لِأَنَّهُ فَعَلَ زَائِدًا عَلَى تَقْدِيرٍ وَلَا يَتَحَمَّلُهُ الْإِمَامُ كَمَا مَرَّ .
