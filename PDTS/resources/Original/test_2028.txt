( وَإِنْ لَمْ يَرْضَ ) رَبُّ الْمَالِ ( بِأَخْذِهِ ) أَيْ الْمَالِ ( مِنْ ذَلِكَ ) الْعَرْضِ ( وَطَلَبَ الْبَيْعَ ، أَوْ طَلَبَهُ ) أَيْ الْبَيْعَ ابْتِدَاءً مِنْ غَيْرِ فَسْخِ الْمُضَارَبَةِ فَلَهُ ذَلِكَ ( وَيَلْزَمُ الْمُضَارِبَ بَيْعُهُ وَلَوْ لَمْ يَكُنْ فِي الْمَالِ رِبْحٌ ) وَقَبْضُ ثَمَنِهِ ؛ لِأَنَّ عَلَيْهِ رَدَّ الْمَالِ نَاضًّا كَمَا أَخَذَهُ .( 11 / 365 )
