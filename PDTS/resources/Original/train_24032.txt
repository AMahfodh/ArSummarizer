( فَإِنْ أَتَى بِهِ ) أَيْ بِتَكْبِيرِ الْإِحْرَامِ كُلِّهِ غَيْرَ قَائِمٍ ، بِأَنْ قَالَ وَهُوَ قَاعِدٌ أَوْ رَاكِعٌ وَنَحْوُهُ : اللَّهُ أَكْبَرُ ( أَوْ ابْتَدَأَهُ ) أَيْ التَّكْبِيرَ غَيْرَ قَائِمٍ ، كَأَنْ ابْتَدَأَهُ قَاعِدًا وَأَتَمَّهُ قَائِمًا ( أَوْ أَتَمَّهُ غَيْرَ قَائِمٍ ) ، بِأَنْ ابْتَدَأَهُ قَائِمًا وَأَتَمَّهُ رَاكِعًا مَثَلًا ( صَحَّتْ ) صَلَاتُهُ ( نَفْلًا ) ؛ لِأَنَّ تَرْكَ الْقِيَامِ يُفْسِدُ الْفَرْضَ فَقَطْ دُونَ النَّفْلِ فَتَنْقَلِبُ بِهِ صَلَاتُهُ نَفْلًا ( إنْ اتَّسَعَ الْوَقْتُ ) لِإِتْمَامِ النَّفْلِ وَالْفَرْضِ كُلِّهِ قَبْلَ خُرُوجِهِ ، وَإِلَّا اسْتَأْنَفَ الْفَرْضَ قَائِمًا ( وَتَنْعَقِدُ ) الصَّلَاةُ ( إنْ مَدَّ اللَّامَ ) أَيْ( 1 / 407 )
