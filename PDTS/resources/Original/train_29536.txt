وَعَلَيْهِ فَاللَّائِقُ أَنْ يَقُولَ : بَارَكَ اللَّهُ لِمُوَكِّلِك فِيمَا أَعْطَى ، وَجَعَلَهُ لَهُ طَهُورًا وَبَارَكَ لَهُ فِيمَا أَبْقَى ( قَوْلُهُ وَيُكْرَهُ أَنْ يُصَلَّى عَلَيْهِ ) أَيْ بِأَنْ يَقُولَ اللَّهُمَّ صَلِّ عَلَيْك ( قَوْلُهُ : مَنْزِلَةَ مَا يَقَعُ خِطَابًا ) أَيْ فَلَا كَرَاهَةَ فِيهِ عَلَى غَيْرِ الْأَنْبِيَاءِ وَالْمَلَائِكَةِ ( قَوْلُهُ كَإِقْرَاءِ دَرْسٍ ) أَيْ وَكَقِرَاءَةِ شَيْءٍ مِنْ الْقُرْآنِ أَوْ تَسْبِيحٍ أَوْ ذِكْرٍ أَوْ غَيْرِهِمَا مِنْ سَائِرِ الْقُرَبِ ( قَوْلُهُ أَنْ يَقُولَ { رَبَّنَا تَقَبَّلْ مِنَّا } إلَخْ ) وَكَذَا يَنْبَغِي لِلطَّالِبِ بَعْدَ حُضُورِهِ أَنْ يَقُولَ ذَلِكَ ؛ لِأَنَّ تَعَبَهُ فِي التَّحْصِيلِ عِبَادَةٌ .( 8 / 493 )
