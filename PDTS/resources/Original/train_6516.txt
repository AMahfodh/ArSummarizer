قَوْلُ الْمَتْنِ ( وَأَوْدَعَهُ ) أَيْ رَأْسَ مَالِ السَّلَمِ فَالْهَاءُ مَفْعُولٌ ثَانٍ قَدَّمَهُ لِاتِّصَالِهِ بِالْعَامِلِ عَلَى الْمُسْلِمِ الَّذِي هُوَ الْمَفْعُولُ الْأَوَّلُ لِأَنَّهُ فَاعِلٌ فِي الْمَعْنَى قَوْلُ الْمَتْنِ ( جَازَ ) أَيْ كُلٌّ مِنْ عَقْدِ السَّلَمِ وَالْإِيدَاعُ ( وَقَوْلُهُ لِأَنَّ تَصَرُّفَ إلَخْ ) تَعْلِيلٌ لِلْجَوَازِ بِالنِّسْبَةِ لِلْإِيدَاعِ وَالرَّدِّ إلَيْهِ قَرْضًا أَوْ عَنْ دَيْنٍ ( قَوْلُهُ لَا يَسْتَدْعِي إلَخْ ) أَيْ لَا يَتَوَقَّفُ عَلَى لُزُومِ الْمِلْكِ بَلْ يَصِحُّ قَبْلَ لُزُومِهِ بِخِلَافِهِ مَعَ الْأَجْنَبِيِّ ا ه بُجَيْرِمِيٌّ ( قَوْلُهُ وَلَوْ أَعْتَقَهُ ) أَيْ عَلَى رَأْسِ الْمَالِ ( وَقَوْلُهُ فَإِنْ قَبَضَهُ ) أَيْ رَأْسَ الْمَالِ وَهُوَ الْعَبْدُ ا ه ع ش ( قَوْلُهُ بَانَتْ صِحَّتُهُ إلَخْ ) وَالْفَرْقُ بَيْنَ هَذَا وَبَيْنَ مَا تَقَدَّمَ فِي الْبَيْعِ حَيْثُ جَعَلَ الْإِعْتَاقَ قَبْضًا ثَمَّ لَا هُنَا أَنَّهُ لَمَّا كَانَ الْمُعْتَبَرُ هُنَا الْقَبْضَ الْحَقِيقِيَّ لَمْ يَكْتَفِ بِالْإِعْتَاقِ لِأَنَّهُ لَيْسَ قَبْضًا حَقِيقِيًّا بِخِلَافِهِ ثَمَّ فَإِنَّهُ يَكْفِي فِيهِ الْقَبْضُ الْحُكْمِيُّ ا ه ع ش .
