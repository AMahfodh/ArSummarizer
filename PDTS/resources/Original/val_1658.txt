( قَوْلُهُ : : لَا حُلُولُ النُّجُومِ ) أَيْ فَلَا يُشْتَرَطُ ( قَوْلُهُ وَيُسْتَرَدُّ مِنْهُ ) أَيْ الزَّكَاةُ الَّتِي أَخَذَهَا مِنْ غَيْرِ سَيِّدِهِ وَكَانَ الْأَوْلَى تَأْخِيرَهُ عَمَّا بَعْدَهُ ( قَوْلُهُ : نَعَمْ مَا أَتْلَفَهُ قَبْلَ الْعِتْقِ ) اسْتِدْرَاكٌ عَلَى قَوْلِهِ وَتُسْتَرَدُّ إلَخْ( 20 / 82 )
