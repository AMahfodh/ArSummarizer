وَفِي الْفُرُوعِ وَالْمُبْدِعِ : وَالْخَمْرِ ( يَقَعُ مَا نَوَاهُ مِنْ الطَّلَاقِ ) لِأَنَّهُ يَصْلُحُ أَنْ يَكُونَ كِنَايَةً فِيهِ ( وَالظِّهَارُ ) إذَا نَوَاهُ أَنْ يَقْصِدَ تَحْرِيمَهَا عَلَيْهِ مَعَ بَقَاءِ نِكَاحِهَا لِأَنَّهُ يُشْبِهُهُ ( وَالْيَمِينُ ) إنْ أَرَادَ بِذَلِكَ تَرْكَ وَطْئِهَا وَأَقَامَ ذَلِكَ مَقَامَ وَاَللَّهِ لَا وَطِئْتُكِ لَا تَحْرِيمِهَا وَلَا طَلَاقِهَا وَفَائِدَتُهُ تَرَتُّبُ الْحِنْثِ وَالْبِرِّ ثُمَّ تَرَتُّبُ الْكَفَّارَةِ بِالْحِنْثِ قَالَ فِي الْمُبْدِعِ : وَفِي ذَلِكَ نَظَرٌ مِنْ حَيْثُ إنَّ قَوْلَهُ كَالْمَيْتَةِ لَيْسَ بِصَرِيحٍ فِي الْيَمِينِ لِأَنَّهُ لَوْ كَانَ صَرِيحًا لَمَا انْصَرَفَ إلَى غَيْرِهَا بِالنِّيَّةِ وَإِذَا لَمْ يَكُنْ صَرِيحًا لَمْ يَلْزَمهُ الْكَفَّارَةُ ، لِأَنَّ الْيَمِينَ بِالْكِنَايَةِ لَا يَنْعَقِدُ لِأَنَّ الْكَفَّارَةَ إنَّمَا تَجِبُ لِهَتْكِ الْقَسَمِ ( فَإِنْ نَوَى ) بِذَلِكَ ( الطَّلَاقَ وَلَمْ يَنْوِ عَدَدًا وَقَعَ وَاحِدَةً ) لِأَنَّهَا الْيَقِينُ ( وَإِنْ لَمْ يَنْوِ ) بِذَلِكَ ( شَيْئًا فَهُوَ ظِهَارٌ ) لِأَنَّ مَعْنَاهُ أَنْتِ حَرَامٌ عَلَيَّ كَالْمَيْتَةِ وَالدَّمِ .( 18 / 96 )
