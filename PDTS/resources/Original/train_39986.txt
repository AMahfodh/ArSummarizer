( فَرْعٌ وَيُجْبِرُ ) السَّيِّدُ الْأَمَةَ بِأَيِّ صِفَةٍ كَانَتْ ( لَا مُكَاتَبَةٍ وَمُبَعَّضَةٍ عَلَى النِّكَاحِ ) لِمَا مَرَّ قَبْلَ الْفَرْعِ بِخِلَافِ الْمُكَاتَبَةِ وَالْمُبَعَّضَةِ ؛ لِأَنَّهُمَا فِي حَقِّهِ كَالْأَجْنَبِيَّاتِ وَتَقَدَّمَ فِي أَوَاخِرِ الطَّرَفِ السَّابِعِ مَا يُقَيِّدُ ذَلِكَ ( وَلَا يَلْزَمُهُ إجَابَتُهَا ) إلَيْهِ إذَا طَلَبَتْهُ ( وَلَوْ كَانَتْ مُحَرَّمَةً ) عَلَيْهِ بِنَسَبٍ - أَوْ رَضَاعٍ أَوْ غَيْرِهِمَا ( أَوْ مُكَاتَبَةً أَوْ مُبَعَّضَةً ) لِمَا مَرَّ قَبْلَ الْفَرْعِ( 14 / 488 )
