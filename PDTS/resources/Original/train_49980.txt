( فَصْلٌ ) فِي تَدَاخُلِ الْعِدَّتَيْنِ إذَا ( لَزِمَهَا ) فَ ( عِدَّتَا شَخْصٍ ) وَاحِدٍ ( مِنْ جِنْسٍ ) وَاحِدٍ ( بِأَنْ ) بِمَعْنَى كَانَ ( طَلَّقَ ، ثُمَّ وَطِئَ ) رَجْعِيَّةً أَوْ بَائِنًا ( فِي عِدَّةِ ) غَيْرِ حَمْلٍ مِنْ ( أَقْرَاءٍ أَوْ أَشْهُرٍ ) وَلَمْ تَحْبَلْ مِنْ وَطْئِهِ ( جَاهِلًا ) بِأَنَّهَا الْمُطَلَّقَةُ أَوْ بِتَحْرِيمِ وَطْءِ الْمُعْتَدَّةِ وَعُذِرَ لِنَحْوِ بُعْدِهِ عَنْ الْعُلَمَاءِ ( أَوْ عَالِمًا ) بِذَلِكَ ( فِي رَجْعِيَّةٍ ) لَا بَائِنٍ ؛ لِأَنَّهُ زَانٍ ( تَدَاخَلَتَا ) أَيْ عِدَّتَا الطَّلَاقِ وَالْوَطْءِ ( فَتَبْتَدِئُ عِدَّةً ) بِأَقْرَاءٍ أَوْ أَشْهُرٍ ( مِنْ ) فَرَاغِ ( الْوَطْءِ وَيَدْخُلُ فِيهَا بَقِيَّةُ عِدَّةِ الطَّلَاقِ ) وَهَذِهِ الْبَقِيَّةُ وَاقِعَةٌ عَنْ الْجِهَتَيْنِ ؛ فَلَهُ الرَّجْعَةُ فِي الرَّجْعِيِّ فِيهَا إجْمَاعًا عَلَى مَا حَكَاهُ الْعَبَّادِيُّ دُونَ مَا بَعْدَهَا
