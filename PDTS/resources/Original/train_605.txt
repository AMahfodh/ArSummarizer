وَشَبَّهَ فِي الْحُكْمِ بِإِسْلَامِ الْوَلَدِ بِإِسْلَامِ أَبِيهِ فَقَالَ ( كَأَنْ ) بِفَتْحِ الْهَمْزِ وَسُكُونِ النُّونِ حَرْفٌ مَصْدَرِيٌّ صِلَتُهُ ( مَيَّزَ ) بِفَتَحَاتٍ مُثَقَّلًا الْوَلَدُ الَّذِي أَسْلَمَ أَبُوهُ فَيُحْكَمُ بِإِسْلَامِهِ تَبَعًا لِأَبِيهِ ، وَاسْتَثْنَى الْمُرَاهِقَ مِنْهُ فَقَالَ ( إلَّا ) الْمُمَيِّزَ ( الْمُرَاهِقَ ) بِضَمِّ الْمِيمِ( 19 / 401 )
