( وَ ) يَجُوزُ اصْطِيَادُ ( مَا لَا يُؤْكَلُ ) لَحْمُهُ ( لِجِلْدِهِ وَشَعْرِهِ ) لِإِطْلَاقِ قَوْله تَعَالَى { وَإِذَا حَلَلْتُمْ فَاصْطَادُوا } وَلَا يَخْتَصُّ بِمَأْكُولِ اللَّحْمِ قَالَ قَائِلُهُمْ صَيْدُ الْمُلُوكِ أَرَانِبُ وَثَعَالِبُ وَإِذَا رَكِبْتُ فَصَيْدِي الْأَبْطَالُ وَلِأَنَّ صَيْدَهُ سَبَبُ الِانْتِفَاعِ بِجِلْدِهِ أَوْ شَعْرِهِ أَوْ رِيشِهِ أَوْ لِاسْتِدْفَاعِ شَرِّهِ وَكُلُّ ذَلِكَ مَشْرُوحٌ كَمَا فِي الْهِدَايَةِ .( 8 / 279 )
