( وَمِنْ مَكَارِمِ ) أَيْ مَحَاسِنِ ( الْأَخْلَاقِ أَنْ تَعْفُوَ ) أَيْ تَصْفَحَ ( عَنْ ) زَلَّةِ ( مَنْ ظَلَمَك ) وَتَعَدَّى عَلَيْك بِشَتْمٍ أَوْ ضَرْبٍ أَوْ أَخْذِ مَالٍ لِقَوْلِهِ تَعَالَى : { وَالْكَاظِمِينَ الْغَيْظَ وَالْعَافِينَ عَنْ النَّاسِ } وَقَالَ أَيْضًا : { فَمَنْ عَفَا وَأَصْلَحَ فَأَجْرُهُ عَلَى اللَّهِ } وَغَيْرُ ذَلِكَ مِنْ الْآيَاتِ الصَّرِيحَةِ فِي مَدْحِ مَنْ يَصْفَحُ عَنْ زَلَّةِ غَيْرِهِ .
