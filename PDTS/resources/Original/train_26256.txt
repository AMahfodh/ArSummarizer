( وَلَا تَثْبُتُ أَحْكَامُهَا ) أَيْ : الْهِبَةِ فِيمَا وَقَعَتْ فِيهِ الْإِجَازَةُ ( فَلَوْ كَانَ الْمُجِيزُ أَبًا لِلْمُجَازِ لَهُ لَمْ يَكُنْ لَهُ الرُّجُوعُ ) فِيمَا أَجَازَهُ لِابْنِهِ ، لِأَنَّ الْأَبَ إنَّمَا يَمْلِكُ الرُّجُوعَ فِيمَا وَهَبَهُ لِابْنِهِ ، وَالْإِجَازَةُ تَنْفِيذٌ لِمَا وَهَبَهُ غَيْرُهُ لِابْنِهِ ( وَلَا يَحْنَثُ بِهَا ) أَيْ بِالْإِجَازَةِ ( مَنْ حَلَفَ لَا يَهَبُ ) شَيْئًا ، فَأَجَازَ الْوَصِيَّةَ بِهِ ، لِأَنَّ الْإِجَازَةَ لَيْسَتْ بِهِبَةٍ ( وَلَا يُعْتَبَرُ ) لِصِحَّةِ الْإِجَازَةِ ( أَنْ يَكُونَ الْمُجَازُ مَعْلُومًا ) لِأَنَّهُ لَيْسَ هِبَةً ( وَلَوْ كَانَ الْمُجَازُ عِتْقًا كَانَ الْوَلَاءُ لِلْمُوصِي تَخْتَصُّ بِهِ ) أَيْ : بِالْإِرْثِ بِهِ ( عَصَبَتُهُ ) دُونَ بَاقِي وَرَثَتِهِ لِأَنَّ الْإِجَازَةَ تَنْفِيذٌ لِفِعْلِ الْمَيِّتِ .( 15 / 149 )
