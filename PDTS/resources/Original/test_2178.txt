قَالَ أَحْمَدُ : قَالَ ابْنُ عُمَرَ وَابْنُ عَبَّاسٍ وَغَيْرُهُمَا ( وَلَدُهَا بِمَنْزِلَتِهَا ) ( إلَّا أَنَّهُ ) أَيْ وَلَدَهَا ( لَا يَعْتِقُ بِإِعْتَاقِهَا ) لِأَنَّهَا عَتَقَتْ بِغَيْرِ السَّبَبِ الَّذِي تَبِعَهَا فِيهِ وَيَبْقَى عِتْقُهُ مَوْقُوفًا عَلَى مَوْتِ سَيِّدِهِ وَكَذَا لَوْ عَتَقَ وَلَدُهَا لَمْ تَعْتِقْ بَلْ يَمُوتُ سَيِّدُهَا ( أَوْ ) أَيْ وَلَا يَعْتِقُ وَلَدُهَا ( بِمَوْتِهَا قَبْلَ سَيِّدِهَا ) وَيَبْقَى عِتْقُهُ مَوْقُوفًا عَلَى مَوْتِ سَيِّدِهَا لِبَقَاءِ التَّبَعِيَّةِ ، بِخِلَافِ الْمُكَاتَبَةِ إذَا مَاتَتْ بَطَلَتْ التَّبَعِيَّةُ لِأَنَّ سَبَبَ الْعِتْقِ فِي الْكِتَابَةِ الْأَدَاءُ ، وَقَدْ تَعَذَّرَ بِمَوْتِهَا ، وَالسَّبَبُ فِي أُمِّ الْوَلَدِ مَوْتُ السَّيِّدِ وَلَا يَتَعَذَّرُ بِمَوْتِهَا ( وَإِنْ مَاتَ سَيِّدُهَا وَهِيَ حَامِلٌ ) مِنْهُ ( فَنَفَقَتُهَا لِمُدَّةِ حَمْلِهَا مِنْ مَالِ حَمْلِهَا ) أَيْ نَصِيبِهِ الَّذِي وُقِفَ لَهُ لِمِلْكِهِ لَهُ ( وَإِلَّا ) بِأَنْ لَمْ يَكُنْ لِلْحَمْلِ مَالٌ بِأَنْ لَمْ يُخَلِّفْ السَّيِّدُ مَا يَرِثُ مِنْهُ الْحَمْلُ ( فَ ) نَفَقَةُ الْحَمْلِ ( عَلَى وَارِثِهِ ) لِقَوْلِهِ تَعَالَى : { وَعَلَى الْوَارِثِ مِثْلُ ذَلِكَ }( 8 / 303 )
