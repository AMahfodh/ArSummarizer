وَلَمَّا فَرَغَ مِنْ الْكَلَامِ عَلَى مَا يُوجِبُ الِاسْتِبْرَاءَ شَرَعَ فِي مَفَاهِيمِ قُيُودِهِ وَإِنْ لَمْ يَكُنْ عَلَى التَّرْتِيبِ فَقَالَ ( وَلَا اسْتِبْرَاءَ ) عَلَى صَغِيرَةٍ ( إنْ لَمْ تُطِقْ الْوَطْءَ ) كَبِنْتِ ثَمَانٍ فَأَقَلَّ وَهَذَا مَفْهُومُ أَطَاقَتْ الْوَطْءَ ( أَوْ ) أَطَاقَتْهُ لَكِنْ ( حَاضَتْ تَحْتَ يَدِهِ ) أَيْ عِنْدَهُ وَلَمْ تَحْصُلْ إسَاءَةُ ظَنٍّ ( كَمُودَعَةٍ ) وَمَرْهُونَةٍ وَأَمَةِ زَوْجَتِهِ وَوَلَدِهِ الصَّغِيرِ أَوْ نَحْوِ ذَلِكَ ( وَمَبِيعَةٍ بِالْخِيَارِ ، وَلَمْ تَخْرُجْ ) لِلتَّصَرُّفِ فِي حَوَائِجِهَا ( وَلَمْ يَلِجْ ) أَيْ يَدْخُلْ ( عَلَيْهَا سَيِّدُهَا ) ثُمَّ اشْتَرَاهَا أَوْ مَلَكَهَا بِوَجْهٍ أَوْ بَتَّ الْبَيْعَ مَنْ لَهُ الْخِيَارُ فَلَا اسْتِبْرَاءَ ؛ لِأَنَّهُ عَلِمَ بَرَاءَةَ رَحِمِهَا بِالْحَيْضِ ، وَلَمْ يَحْصُلْ إسَاءَةُ ظَنٍّ ، وَهَذَا مَفْهُومُ قَوْلِهِ : إنْ لَمْ تُوقَنْ الْبَرَاءَةُ .
