( وَقَوْلُهُ : لِقُدْرَتِهِ عَلَى تَحْلِيلِهِ ) أَيْ بِأَنْ يَأْمُرَهُ بِفِعْلِ مَا يَحْرُمُ عَلَى الْمُحْرِمِ ا ه ع ش ( قَوْلُهُ : لَا مَشَقَّةَ فِيهِ ) أَيْ التَّحْلِيلِ ( قَوْلُهُ : وَهَذَا لَيْسَ مِنْهُ ) أَيْ وَالْمَهَابَةُ لَيْسَتْ مِنْ السَّبَبِ الْقَوِيِّ ( قَوْلُهُ : بِخِلَافِهِ فِي نَحْوِ التَّمَتُّعِ إلَخْ ) يَعْنِي بِخِلَافِ مَهَابَةِ إبْطَالِ صَوْمِ الْمَرْأَةِ فَإِنَّهَا يُنْظَرُ إلَيْهَا فِي حُرْمَةِ صَوْمِهَا نَفْلًا وَالزَّوْجُ حَاضِرٌ فَإِنَّ الصَّوْمَ لَا يُؤَدِّي إلَى تَفْوِيتِ مَالٍ عَلَى الْغَيْرِ ( قَوْلُهُ : وَلَوْ كَانَ حُدُوثُ الْعَيْبِ بِفِعْلِهِ إلَخْ ) أَيْ الْمُشْتَرِي وَهَذَا تَقْيِيدٌ لِكَلَامِ الْمَتْنِ عِبَارَةُ الْمُغْنِي وَيُسْتَثْنَى مِنْ طَرْدِهِ مَسَائِلُ مِنْهَا مَا إذَا حَدَثَ الْعَيْبُ قَبْلَ الْقَبْضِ بِفِعْلِ الْمُشْتَرِي كَمَا سَيَأْتِي إلَخْ ا ه ( قَوْلُهُ : أَوْ كَانَتْ الْغِبْطَةُ ) أَيْ أَوْ لَمْ يَحْدُثْ كَذَلِكَ كَأَنْ حَدَثَ بِآفَةٍ سَمَاوِيَّةٍ أَوْ بِفِعْلِ الْبَائِعِ قَبْلَ الْقَبْضِ وَلَكِنْ( 17 / 323 )
