( وَلَمْ يُوقَفْ لَهُ ) أَيْ لِلدَّيْنِ الْمُؤَجَّلِ ( شَيْءٌ ) مِنْ الْمَالِ ( وَلَا يَرْجِعُ ) رَبُّ الدَّيْنِ الْمُؤَجَّلِ ( عَلَى الْغُرَمَاءِ إذَا حَلَّ ) دَيْنُهُ بِشَيْءٍ لِأَنَّهُ لَمْ يَسْتَحِقَّ مُشَارَكَتُهُمْ حَالَ الْقِسْمَةِ فَلَمْ يَسْتَحِقَّ الرُّجُوعَ عَلَيْهِمْ بَعْدُ ( لَكِنْ إنْ حَلَّ ) دَيْنُهُ ( قَبْلَ الْقِسْمَةِ شَارَكَهُمْ ) لِمُسَاوَاتِهِ لَهُمْ .
