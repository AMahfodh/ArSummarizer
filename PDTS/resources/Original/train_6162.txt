( فَرْعٌ ) لَوْ ( ابْتَلَعَ رِيقَهُ الصِّرْفَ ) بِكَسْرِ الصَّادِ أَيْ الْخَالِصَ ( لَمْ يُفْطِرْ ) لِعُسْرِ التَّحَرُّزِ عَنْهُ ( وَلَوْ بَعْدَ جَمْعِهِ ) وَلَوْ بِنَحْوِ مُصْطَكَى فَإِنَّهُ لَا يُفْطِرُ بِهِ لِأَنَّهُ لَمْ يَخْرُجْ مِنْ مَعْدِنِهِ وَابْتِلَاعُهُ مُتَفَرِّقًا جَائِزٌ ( وَيُفْطِرُ بِهِ إنْ تَنَجَّسَ ) كَمَنْ دَمِيَتْ لِثَتُهُ أَوْ أَكَلَ شَيْئًا نَجِسًا وَلَمْ يَغْسِلْ فَمَهُ حَتَّى أَصْبَحَ وَإِنْ ابْيَضَّ رِيقُهُ وَكَذَا لَوْ اخْتَلَطَ بِطَاهِرٍ آخَرَ كَمَا أَفْهَمَهُ قَوْلُهُ الصِّرْفُ كَمَنْ فَتَلَ خَيْطًا مَصْبُوغًا تَغَيَّرَ بِهِ رِيقُهُ ( أَوْ زَايَلَ ) رِيقُهُ ( فَمَهُ ) أَيْ خَرَجَ مِنْهُ وَلَوْ إلَى ظَاهِرِ الشَّفَةِ ( وَلَوْ فِي خَيْطِ ) الْخَيَّاطِ أَوْ امْرَأَةٍ فِي غَزْلِهَا لِإِمْكَانِهِ التَّحَرُّزَ عَنْ ذَلِكَ وَلِمُفَارِقَةِ الرِّيقِ مَعْدِنَهُ فِي الْأَخِيرَةِ ( لَا ) إنْ زَايَلَ رِيقُهُ فَمَهُ ( فِي لِسَانِهِ ) فَلَا يُفْطِرُ بِبَلْعِهِ إذْ اللِّسَانُ كَيْفَمَا تَقَلَّبَ مَعْدُودٌ مِنْ دَاخِلِ الْفَمِ فَلَمْ يُفَارِقْ مَا عَلَيْهِ مَعْدِنَهُ
