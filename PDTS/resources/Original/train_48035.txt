( وَلَوْ عَجَزَ ) عَنْ السُّجُودِ ( إلَّا أَنْ يَسْجُدَ بِمُقَدَّمِ رَأْسِهِ ، أَوْ صُدْغِهِ وَكَانَ بِذَلِكَ أَقْرَبَ إلَى الْأَرْضِ وَجَبَ ) ؛ لِأَنَّ الْمَيْسُورَ لَا يَسْقُطُ بِالْمَعْسُورِ ( فَإِنْ عَجَزَ ) عَنْ ذَلِكَ أَيْضًا ( أَوْمَأَ بِرَأْسِهِ وَالسُّجُودُ أَخْفَضُ ) مِنْ الرُّكُوعِ ( فَإِنْ عَجَزَ ) عَنْ إيمَائِهِ بِرَأْسِهِ ( فَبِطَرْفِهِ ) أَيْ بَصَرِهِ ( فَإِنْ عَجَزَ ) عَنْ الْإِيمَاءِ بِطَرْفِهِ إلَى أَفْعَالِ الصَّلَاةِ ( أَجْرَاهَا عَلَى قَلْبِهِ بِسُنَنِهَا ) وَلَا إعَادَةَ عَلَيْهِ ( وَلَا تَسْقُطُ عَنْهُ ) الصَّلَاةُ ( وَعَقْلُهُ ثَابِتٌ ) لِوُجُودِ مَنَاطِ التَّكْلِيفِ .( 2 / 359 )
