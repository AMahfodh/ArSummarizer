( وَإِنْ قَالَ ) لِمُحْصَنَةٍ زَنَيْت ( وَأَنْتِ كَافِرَةٌ ، أَوْ ) وَأَنْتِ ( أَمَةٌ ، أَوْ ) وَأَنْتِ ( مَجْنُونَةٌ ، وَلَمْ يُثْبِتْ كَوْنَهَا كَذَلِكَ ) ؛ أَيْ : كَافِرَةً أَوْ أَمَةً أَوْ مَجْنُونَةً ( حُدَّ ) لِأَنَّ الْأَصْلَ عَدَمُ ذَلِكَ ( كَمَا لَوْ قَذَفَ مَجْهُولَةَ النَّسَبِ ، وَادَّعَى رِقَّهَا فَأَنْكَرَتْ ) فَيُحَدُّ ؛ لِأَنَّ الْأَصْلَ الْحُرِّيَّةُ ( وَإِنْ ثَبَتَ كَوْنُهَا كَذَلِكَ ) ؛ أَيْ : كَانَتْ كَافِرَةً أَوْ مَجْنُونَةً أَوْ أَمَةً ( لَمْ يُحَدَّ ) لِإِضَافَةِ الزِّنَا إلَى حَالٍ لَمْ تَكُنْ فِيهَا مُحْصَنَةً .
