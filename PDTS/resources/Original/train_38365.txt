( وَمَنْ ) ادَّعَى أَنْ الْمَيِّتَ أَوْصَى لَهُ بِثُلُثِ مَالِهِ وَ ( شَهِدَ لَهُ بَيِّنَةٌ بِالثُّلُثِ ) الَّذِي ادَّعَاهُ ( فَأَقَرَّ وَارِثٌ ) مُكَلَّفٌ ( ذَكَرٌ ) لَا أُنْثَى وَلَا خُنْثَى ( عَدْلٌ ) لَا فَاسِقٌ إذْ إقْرَارُ الْفَاسِقِ غَيْرُ مُعْتَدٍّ بِهِ أَنْ مُوَرِّثَهُ أَوْصَى ( بِهِ ) أَيْ الثُّلُثِ الْمُدَّعَى بِهِ ( لِآخَرَ ) وَرَدَّ الْوَارِثُ الْوَصِيَّتَيْنِ ( فَ ) الثُّلُثُ ( بَيْنَهُمَا ) سَوِيَّةٌ إنْ حَلَفَ الْمُقَرُّ لَهُ يَمِينًا مَعَ شَهَادَةِ الْوَارِثِ ، لِأَنَّ الْمَالَ يَثْبُتُ بِشَاهِدٍ وَيَمِينٍ ( وَإِلَّا ) يَكُنْ الْوَارِثُ الْمُقِرُّ عَدْلًا ، أَوْ كَانَ الْمُقِرُّ امْرَأَةً أَوْ خُنْثَى ( فَ ) الثُّلُثُ ( لِذِي الْبَيِّنَةِ ) لِثُبُوتِ وَصِيَّتِهِ دُونَ الْمُقِرِّ ( وَإِنْ ) فَعَلَ مُوصٍ مَا يَقْتَضِي عُدُولَهُ عَنْ الْوَصِيَّةِ ؛ بِأَنْ ( بَاعَ مَا أَوْصَى بِهِ أَوْ وَهَبَهُ ) فَرُجُوعٌ ؛ لِأَنَّهُ إزَالَةُ مِلْكٍ ، وَهُوَ يُنَافِي الْوَصِيَّةَ .
