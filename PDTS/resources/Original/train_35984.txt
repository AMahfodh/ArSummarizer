( مَقْسُومٍ ) أَيْ تَعَلَّقَ بِهِ الْقِسْمَةُ وَلَمْ يَبْقَ مَشَاعًا ( وَمَشَاعٌ لَا يُقْسَمُ ) أَيْ لَيْسَ مِنْ شَأْنِهِ أَنْ يُقْسَمَ بِمَعْنَى أَنَّهُ لَا يَبْقَى مُنْتَفَعًا بِهِ بَعْدَ الْقِسْمَةِ أَصْلًا كَعَبْدٍ وَاحِدٍ وَدَابَّةٍ وَاحِدَةٍ إذْ لَا يَبْقَى مُنْتَفَعًا بِهِ بَعْدَ الْقِسْمَةِ مِنْ جِنْسِ الِانْتِفَاعِ الَّذِي كَانَ قَبْلَ الْقِسْمَةِ كَالْبَيْتِ الصَّغِيرِ وَالْحَمَّامِ الصَّغِيرِ وَالثَّوْبِ الصَّغِيرِ ( لَا ) أَيْ لَا تَتِمُّ بِالْقَبْضِ ( فِيمَا ) أَيْ مَشَاعٍ ( يُقْسَمُ ) أَيْ مِنْ شَأْنِهِ الْقِسْمَةُ كَالْأَرْضِ وَالثَّوْبِ الْمَذْرُوعِ وَنَحْوِ ذَلِكَ( 7 / 3 )
