فَتَعْبِيرُهُ بِاشْتُهِرَتْ عَدَالَتُهُ أَوْلَى مِنْ قَوْلِ الشَّارِحِ وَإِنْ وَثِقَ بِهِ لِأَنَّهُ كَيْفَ يَثِقُ بِهِ مَعَ التُّهْمَةِ ، وَالْأَقْرَبُ مَا اسْتَوْجَهَهُ سم ( قَوْلُهُ : أَيْ وَلَا يَجِبُ عَلَيْهِ ) مُتَّصِلٌ بِقَوْلِهِ لَا ظَاهِرُ الْعَدَالَةِ ( قَوْلُهُ : أَصْلًا ) أَيْ لَا مَرَّةً وَلَا غَيْرَهَا ( قَوْلُهُ : وَالْمُصَنِّفُ إلَخْ ) أَيْ الْمَذْكُورُ فِي قَوْلِهِ فَلَا يُكَلَّفُ الْإِشْهَادَ فِي كُلِّ مَرَّةٍ إلَخْ ( قَوْلُهُ : الْإِشْهَادُ ) أَيْ عَلَى أَنَّ الطَّرِيقَ مُشْتَرَكَةٌ بَيْنَهُمَا ( قَوْلُهُ : وَبَيْنَ إجَابَةٍ ) لَعَلَّهُ عَدَمُ إجَابَةٍ .( 13 / 499 )
