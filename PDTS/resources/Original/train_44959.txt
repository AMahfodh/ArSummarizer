( وَيَتَوَكَّأُ ) أَيْ يَعْتَمِدُ ( الْإِمَامُ ) فِي قِيَامِهِ لَخُطْبَتِهِ اسْتِحْبَابًا بِيَدِهِ الْيُمْنَى ( عَلَى قَوْسٍ أَوْ عَصًا ) ابْنُ الْعَرَبِيِّ : وَلَا يُقَالُ عُصَاةٌ وَهِيَ أَوَّلُ لَحْنٍ سُمِعَ بِالْبَصْرَةِ أَوْ سَيْفٍ وَنَحْوِهِ لِفِعْلِهِ صَلَّى اللَّهُ عَلَيْهِ وَآلِهِ وَسَلَّمَ وَالْخُلَفَاءِ ذَلِكَ ( وَيَجْلِسُ فِي أَوَّلِهَا ) أَيْ الْخُطْبَةِ ( وَفِي وَسَطِهَا ) وَاخْتُلِفَ فِي هَذَا( 3 / 145 )
