( وَ ) يُوسَعُ ( لَهُ ) أَيْ الْحَالِفِ ( يَوْمٌ وَلَيْلَةٌ ) مِنْ الشَّهْرِ التَّالِي لِلشَّهْرِ الَّذِي حَلَفَ فِيهِ ( فِي ) حَلِفِهِ لَيَقْضِيَنَّهُ حَقَّهُ فِي ( رَأْسِ ) أَيْ أَوَّلِ ( الشَّهْرِ ) الْفُلَانِيِّ كَرَجَبٍ فَلَهُ لَيْلَةٌ وَيَوْمٌ مِنْ الشَّهْرِ الَّذِي أَضَافَ الرَّأْسَ إلَيْهِ كَرَجَبٍ ، وَالْأَوْلَى لَيْلَةٌ وَيَوْمٌ لِسَبْقِ اللَّيْلَةِ الْيَوْمَ لِسَبْقِ لَيْلَةِ الْهِلَالِ ، وَلِإِبْهَامِ تَقْدِيمِ الْيَوْمِ أَنَّ لَهُ لَيْلَةً بَعْدَهُ وَلَيْسَ كَذَلِكَ ( أَوْ عِنْدَ رَأْسِهِ ) أَيْ أَوَّلِ الشَّهْرِ ( أَوْ إذَا اسْتَهَلَّ ) الشَّهْرُ الْفُلَانِيُّ .
