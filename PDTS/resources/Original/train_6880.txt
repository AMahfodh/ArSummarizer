( وَمَنْ اعْتَمَرَ بِلَا سَوْقِ ) هَدْيٍ ( ثُمَّ ) بَعْدَ عُمْرَتِهِ ( عَادَ إلَى بَلَدِهِ ) وَحَلَقَ ( فَقَدْ أَلَمَّ ) إلْمَامًا صَحِيحًا فَبَطَلَ تَمَتُّعُهُ ( وَمَعَ سَوْقِهِ تَمَتُّعٌ ) كَالْقَارِنِ ( وَإِنْ طَافَ لَهَا أَقَلَّ مِنْ أَرْبَعَةٍ قَبْلَ أَشْهُرِ الْحَجِّ وَأَتَمَّهَا فِيهَا وَحَجَّ فَقَدْ تَمَتَّعَ ، وَلَوْ طَافَ أَرْبَعَةً قَبْلَهَا لَا ) اعْتِبَارًا لِلْأَكْثَرِ ( كُوفِيٌّ ) أَيْ آفَاقِيٌّ ( حَلَّ مِنْ عُمْرَتِهِ فِيهَا ) أَيْ الْأَشْهُرِ ( وَسَكَنَ بِمَكَّةَ ) أَيْ دَاخِلَ الْمَوَاقِيتِ ( أَوْ بَصْرَةَ ) أَيْ غَيْرِ بَلَدِهِ ( وَحَجَّ ) مِنْ عَامِهِ ( مُتَمَتِّعٌ ) لِبَقَاءِ سَفَرِهِ ( وَلَوْ أَفْسَدَهَا وَرَجَعَ مِنْ الْبَصْرَةِ ) إلَى مَكَّةَ ( وَقَضَاهَا وَحَجَّ لَا ) يَكُونُ مُتَمَتِّعًا لِأَنَّهُ كَالْمَكِّيِّ ( إلَّا إذَا أَلَمَّ بِأَهْلِهِ ثُمَّ ) رَجَعَ وَ ( أَتَى بِهِمَا ) لِأَنَّهُ سَفَرٌ آخَرُ وَلَا يَضُرُّ كَوْنُ الْعُمْرَةِ قَضَاءً عَمَّا أَفْسَدَهُ ( وَأَيُّ ) النُّسُكَيْنِ ( أَفْسَدَهُ ) الْمُتَمَتِّعُ ( أَتَمَّهُ بِلَا دَمٍ ) لِلتَّمَتُّعِ بَلْ لِلْفَسَادِ
