( فَتَضْرِبُ مَسْأَلَةَ الْإِقْرَارِ فِي ) مَسْأَلَةِ ( الْإِنْكَارِ تَكُنْ اثْنَيْ عَشَرَ ، لِلْمُنْكِرِ سَهْمٌ مِنْ ) مَسْأَلَةِ ( الْإِنْكَارِ ) تَضْرِبُهُ ( فِي ) مَسْأَلَةِ ( الْإِقْرَارِ ) وَذَلِكَ ( أَرْبَعَةٌ وَلِلْمُقِرِّ سَهْمٌ مِنْ ) مَسْأَلَةِ ( الْإِقْرَارِ ) تَضْرِبُ ( فِي ) مَسْأَلَةِ ( الْإِنْكَارِ ثَلَاثَةً .
