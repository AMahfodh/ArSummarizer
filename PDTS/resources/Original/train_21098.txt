( وَ ) يَجُوزُ ( لِعَبْدٍ ) غَيْرِ مُكَاتَبٍ ( بِلَا شِرْكٍ ) لِسَيِّدَتِهِ فِيهِ ( وَمُكَاتَبٍ ) أَيْ مُعْتَقٍ عَلَى مَالٍ مُؤَجَّلٍ بِلَا شِرْكٍ أَيْضًا ( وَغْدَيْنِ ) بِفَتْحِ الْوَاوِ وَسُكُونِ الْغَيْنِ الْمُعْجَمَةِ أَيْ قَبِيحَيْ الْمَنْظَرِ ( نَظَرُ شَعْرِ السَّيِّدَةِ ) الْمَالِكَةِ لَهُمَا وَبَقِيَّةُ أَطْرَافِهَا الَّتِي يَنْظُرُهَا مَحْرَمُهَا مِنْهَا وَالْخَلْوَةُ بِهَا .
