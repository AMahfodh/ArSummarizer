( وَتَصِحُّ فَتْوَى الْعَبْدِ وَالْمَرْأَةِ وَالْأُمِّيِّ وَالْأَخْرَسِ الْمَفْهُومِ الْإِشَارَةِ أَوْ الْكِتَابَةِ ) كَخَبَرِهِمْ ( وَتَصِحُّ ) الْفُتْيَا ( مَعَ أَحَدِ الشَّفْعِ وَدَفْعِ الضَّرَرِ مِنْ الْعَدُوِّ وَأَنْ يُفْتَى أَبَاهُ وَأُمَّهُ وَشَرِيكَهُ وَ ) سَائِرَ ( مَنْ لَا تُقْبَلُ شَهَادَتُهُ لَهُ ) كَزَوْجَتِهِ وَمُكَاتِبَتِهِ لِأَنَّ الْقَصْدَ بَيَانُ الْحُكْمِ الشَّرْعِيِّ وَهُوَ لَا يَخْتَلِفُ وَلَيْسَ مِنْهُ إلْزَامٌ بِخِلَافِ الْحَاكِمِ .( 22 / 66 )
