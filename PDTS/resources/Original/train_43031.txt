( قَوْلَهُ لَمْ يُقْبَلْ ) أَيْ بِالنِّسْبَةِ لِسُقُوطِ الْحَقِّ وَلَهُ تَحْلِيفُ الْمُقَرِّ لَهُ أَنَّ كُلًّا مِنْهُمَا صَحِيحٌ كَمَا يَأْتِي ( قَوْلُهُ : لِجَوَازِ إرَادَةِ الْخُرُوجِ ) أَيْ أَوْ الْمِلْكِ ( قَوْلُهُ بِوَجْهٍ يَكُونُ ) أَيْ خَرَجَتْ إلَخْ ( قَوْلُهُ : وَمَحَلُّ مَا مَرَّ ) أَيْ فِي قَوْلِهِ لَا يَكُونُ مُقِرًّا بِالْإِقْبَاضِ ( قَوْلُهُ : فَهُوَ إقْرَارٌ بِالْقَبْضِ ) وَفِيهِ أَنَّ مُجَرَّدَ الْيَدِ لَا يَسْتَلْزِمُ كَوْنَ الْقَبْضِ عَنْ الْهِبَةِ بَلْ يَجُوزُ كَوْنُهُ فِي يَدِهِ عَارِيَّةً أَوْ غَصْبًا وَلَمْ يَأْذَنْ لَهُ بَعْدَ الْهِبَةِ فِي الْقَبْضِ عَنْهَا ( قَوْلُهُ : وَحُكِمَ بِهِ ) أَيْ الْفَسَادِ .( 16 / 99 )
