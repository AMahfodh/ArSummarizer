( فَإِنْ عَتَقَ ) زَوْجُ عَتِيقَةٍ ( قَبْلَ فَسْخٍ ) بَطَلَ خِيَارُهَا لِزَوَالِ عِلَّتِهِ وَهِيَ الرِّقُّ ( أَوْ مَكَّنَتْهُ ) أَيْ الرَّقِيقَ الْعَتِيقَةُ ( مِنْ وَطْئِهَا أَوْ ) مِنْ ( مُبَاشَرَتِهَا وَنَحْوِهِ ) كَقُبْلَتِهَا ( وَلَوْ جَاهِلَةً عِتْقَهَا أَوْ ) جَاهِلَةً ( مِلْكَ الْفَسْخِ بَطَلَ خِيَارُهَا ) لِحَدِيثِ الْحَسَنِ عَنْ عَمْرِو بْنِ أُمَيَّةَ قَالَ : سَمِعْتُ رِجَالًا يُحَدِّثُونَ عَنْ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَنَّهُ قَالَ : { إذَا عَتَقَتْ الْأَمَةُ فَهِيَ بِالْخِيَارِ مَا لَمْ يَطَأْهَا إنْ شَاءَتْ فَارَقَتْ فَإِنْ وَطِئَهَا فَلَا خِيَارَ لَهَا } رَوَاهُ أَحْمَدُ .
