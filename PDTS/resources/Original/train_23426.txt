( وَإِنْ أَقَرَّ لِمَسْجِدٍ أَوْ مَقْبَرَةٍ أَوْ طَرِيقٍ وَنَحْوِهِ ) كَقَنْطَرَةِ وَسِقَايَةٍ ( صَحَّ الْإِقْرَارُ وَلَوْ لَمْ يَذْكُرْ سَبَبًا ) كَغَلَّةِ وَقْفٍ أَوْ وَصِيَّةٍ لِأَنَّهُ إقْرَارٌ مِنْ مُكَلَّفٍ مُخْتَارٍ فَلَزِمَهُ كَمَا لَوْ عَيَّنَ السَّبَبَ ( وَيَكُونُ لِمَصَالِحِهَا ) أَيْ الْمَذْكُورَاتِ .( 23 / 258 )
