( فَإِنْ قُلْتَ ) وَهَلْ يُقَالُ ذَلِكَ فِي الِاسْتِدْلَالِ عَلَى الْكِتَابَةِ ( قُلْتُ ) الْبَحْثُ يُمْكِنُ فِي ذَلِكَ كُلِّهِ ( فَإِنْ قُلْتَ ) وَقَعَ فِي كُتُبِ الْفُقَهَاءِ لَفْظُ الِاقْتِضَاءِ عُرْفًا وَلَفْظُ الْقَضَاءِ وَقَدْ وَقَعَ أَصْلُهُ فِي الْحَدِيثِ { إنَّ خِيَارَكُمْ أَحْسَنُكُمْ قَضَاءً } وَكَذَلِكَ قَوْلُهُ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ { رَحِمَ اللَّهُ عَبْدًا سَمْحًا إذَا قَضَى سَمْحًا إذَا اقْتَضَى } وَالْقَضَاءُ مِنْ فِعْلِ الْمَدِينِ وَالِاقْتِضَاءُ مِنْ فِعْلِ صَاحِبِ الدَّيْنِ فَمَا سِرُّ كَوْنِ الشَّيْخِ رَضِيَ اللَّهُ عَنْهُ تَعَرَّضَ لِحَقِيقَةِ أَحَدِهِمَا فَقَطْ مَعَ أَنَّ ذَلِكَ كَثِيرٌ تَرَدُّدُهُ فِيهِمَا فِي كِتَابِ الصَّرْفِ وَغَيْرِهِ ( قُلْتُ ) لَمْ يَظْهَرْ سِرُّ تَخْصِيصِهِ وَلَعَلَّهُ لِغَلَبَةِ كَثْرَةِ اسْتِعْمَالِ الِاقْتِضَاءِ وَانْظُرْهُ مَعَ مَا وَقَعَ لَهُمْ فِي( 2 / 17 )
