( قَوْلُهُ : لِأَنَّ لَهُ ) أَيْ لِلدَّائِنِ بِأَنْوَاعِهِ ( قَوْلُهُ : أَوْ لِحَرْبِيٍّ ) مُحْتَرَزُ قَوْلِهِ لِمُسْلِمٍ إلَخْ ( قَوْلُهُ : وَلَهُ دَيْنٌ عَلَى ) أَيْ فَإِنَّهُ يَسْقُطُ ( قَوْلُهُ : وَأُلْحِقَ بِهِ ) أَيْ فِي السُّقُوطِ ( قَوْلُهُ : وَإِنْ كَانَ غَيْرَ مُلْتَزِمٍ ) أَيْ الْمُعَاهَدُ وَالْمُؤَمَّنُ ( قَوْلُهُ : بِخِلَافِهِ عَلَى ذِمِّيٍّ ) أَيْ فَلَا يَسْقُطُ بَلْ إلَخْ ( قَوْلُهُ : لِوُضُوحِ الْفَرْقِ ) وَهُوَ أَنَّ مَا فِي الذِّمِّيِّ لَيْسَ مُتَعَيِّنًا فِي شَيْءٍ يُطَالِبُ بِهِ السَّيِّدُ وَهُوَ مُعَرَّضٌ لِلسُّقُوطِ بِخِلَافِ الْوَدِيعَةِ ( قَوْلُهُ : وَلَهُ عَلَيْهِ دَيْنٌ سَقَطَ ) أَيْ وَهُوَ الرَّاجِحُ وَإِنْ حُكِمَ بِزَوَالِ مِلْكِهِ بِالرِّدَّةِ ، أَوْ مَحْمُولٌ عَلَى مَا إذَا اتَّصَلَتْ رِدَّتُهُ بِالْمَوْتِ .
