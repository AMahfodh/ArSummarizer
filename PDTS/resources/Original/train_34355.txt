( وَإِنْ عُجِّلَتْ ) بِضَمِّ الْعَيْنِ وَكَسْرِ الْجِيمِ مُثَقَّلًا الْعَشَرَةُ لِلْمَأْمُورِ ( أَخَذَهَا ) أَيْ الْآمِرُ الْعَشَرَةَ مِنْ الْمَأْمُورِ وَلَا يَتْرُكُهَا عِنْدَهُ إلَى الْأَجَلِ وَلَا يَفْسُدُ الْعَقْدُ بِتَعْجِيلِهَا لِأَنَّهُ سَلَفٌ مُسْتَقِلٌّ بَعْدَ بَيْعٍ صَحِيحٍ ( وَلَهُ ) أَيْ الْمَأْمُورِ ( جُعْلُ مِثْلِهِ ) وَلَوْ زَادَ عَلَى الدِّرْهَمَيْنِ لِأَنَّ الْمُسْلِفَ هُنَا( 10 / 300 )
