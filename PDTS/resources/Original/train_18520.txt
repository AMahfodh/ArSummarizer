( وَإِنْ ادَّعَتْ انْقِضَاءَهَا ) أَيْ الْعِدَّةَ ( بِوَضْعِ حَمْلٍ ) تَمَامٍ لَيْسَ سَقْطًا ( لَمْ يُقْبَلْ قَوْلُهَا فِي أَقَلَّ مِنْ سِتَّةِ أَشْهُرٍ مِنْ حِينِ إمْكَانِ الْوَطْءِ بَعْدَ الْعَقْدِ ) لِأَنَّ ذَلِكَ أَقَلُّ مُدَّةِ الْحَمْلِ كَمَا تَقَدَّمَ ( وَإِنْ ادَّعَتْ أَنَّهَا أَسْقَطَتْهُ ) أَيْ سَقَطَتْ مَا تَنْقَضِي بِهِ الْعِدَّةُ ( لَمْ يُقْبَلْ ) قَوْلُهَا ( فِي أَقَلِّ مِنْ ثَمَانِينَ يَوْمًا ) مِنْ حِينِ إمْكَانِ الْوَطْءِ بَعْدَ الْعَقْدِ ، لِأَنَّ الْعِدَّةَ لَا تَنْقَضِي إلَّا بِمَا يَبِينُ فِيهِ خَلْقُ إنْسَانٍ وَأَقَلُّ مُدَّةٍ يَتَبَيَّنُ فِيهَا خَلْقُ إنْسَانٍ أَحَدٌ وَثَمَانُونَ يَوْمًا كَمَا تَقَدَّمَ .( 18 / 432 )
