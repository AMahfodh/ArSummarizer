( وَلَوْ نَقَصَتْ قِيمَتُهُ ) أَيْ وَلَدِ اللَّبُونِ ( عَنْهَا ) أَيْ : عَنْ قِيمَةِ بِنْتِ الْمَخَاضِ لِعُمُومِ قَوْلِهِ فِي حَدِيثِ أَنَسٍ { فَإِنْ لَمْ يَكُنْ فِيهَا ابْنَةُ مَخَاضٍ فَفِيهَا ابْنُ لَبُونٍ ذَكَرٌ } رَوَاهُ أَبُو دَاوُد ( أَوْ حَقُّ مَا تَمَّ لَهُ ثَلَاثُ سِنِينَ ) سُمِّيَ بِذَلِكَ لِأَنَّهُ اسْتَحَقَّ أَنْ يُحْمَلَ عَلَيْهِ وَيُرْكَبَ وَيُقَالُ لِلْأُنْثَى : حِقَّةٌ لِذَلِكَ وَلِاسْتِحْقَاقِهَا طَرْقَ الْفَحْلِ لَهَا ( أَوْ جَذَعٍ ) بِالذَّالِ الْمُعْجَمَةِ : ( مَا تَمَّ لَهُ أَرْبَعُ سِنِينَ ) سُمِّيَ بِذَلِكَ لِأَنَّهُ يَجْذَعُ إذَا سَقَطَتْ سِنَّةُ ذَكَرِهِ فِي الْمُغْنِي وَغَيْرِهِ .
