الْحَادِيَةَ عَشْرَةَ ذَكَرَهَا بِقَوْلِهِ ( أَوْ ) ائْتَمَّ ( بِمَنْ يَشُكُّ فِيهِ ) أَيْ فِي كَوْنِهِ مُسَافِرًا ( أَوْ ) ائْتَمَّ ( بِمَنْ يَغْلِبُ عَلَى ظَنِّهِ أَنَّهُ مُقِيمٌ ، وَلَوْ بَانَ ) الْإِمَامُ بَعْدُ ( مُسَافِرًا ) لَزِمَ الْمَأْمُومُ أَنْ يُتِمَّ لِعَدَمِ الْجَزْمِ بِكَوْنِهِ مُسَافِرًا عِنْدَ الْإِحْرَامِ .( 4 / 57 )
