( وَإِنْ ) فَسَدَتْ وَالْبَذْرُ ( لِلْعَامِلِ ) لَا يَطِيبُ لَهُ الْخَارِجُ فَحِينَئِذٍ ( تَصَدَّقَ بِمَا فَضَلَ عَنْ قَدْرِ بَذْرِهِ ، وَ ) قَدْرِ ( أُجْرَةِ الْأَرْضِ ) ؛ لِأَنَّهُ حَصَلَ مِنْ بَذْرِهِ لَكِنْ فِي أَرْضٍ مَمْلُوكٍ لِلْغَيْرِ بِعَقْدٍ فَاسِدٍ فَأَوْجَبَ خَبُثَا فَمَا كَانَ عِوَضَ مَالِهِ طَابَ لَهُ وَتَصَدَّقَ بِالْفَضْلِ كَمَا فِي الِاخْتِيَارِ ( وَإِذَا أَبَى رَبُّ الْبَذْرِ عَنْ الْمُضِيِّ وَقَدْ كَرَبَ الْعَامِلُ الْأَرْضَ ) أَيْ قَلَّبَهَا لِلْحَرْثِ ( فَلَا شَيْءَ لَهُ ) أَيْ لِلْعَامِلِ فِي عَمَلِ الْكِرَابِ ( حُكْمًا ) أَيْ قَضَاءً ؛ لِأَنَّ الْمَنَافِعَ إنَّمَا تَتَقَوَّمُ بِالْعَقْدِ وَهُوَ إنَّمَا يَتَقَوَّمُ بِالْخَارِجِ فَإِذَا انْعَدَمَ الْخَارِجُ لَا يَجِبُ شَيْءٌ ( وَيُسْتَرْضَى ) أَيْ الْآبِي فِي عَمَلِهِ ( دِيَانَةً ) عَلَى وَجْهٍ يُمْكِنُ إذْ الْغُرُورُ فِي الْكِرَابِ مِنْ جَانِبِ الْآبِي .( 8 / 43 )
