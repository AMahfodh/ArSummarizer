( وَإِنْ هَشَمَهُ بِمُثْقَلٍ وَلَمْ يُوضِحْهُ ) فَحُكُومَةٌ ( أَوْ طَعَنَهُ فِي خَدِّهِ فَوَصَلَ ) الطَّعْنُ ( إلَى فَمِهِ ) فَحُكُومَةٌ ( أَوْ نَفَذَ ) جَانٍ بِخَرْزَةٍ ( أَنْفًا أَوْ ذَكَرًا ) فَحُكُومَةٌ ( أَوْ ) نَفَذَ ( جَفْنًا إلَى بَيْضَةِ الْعَيْنِ ) فَحُكُومَةٌ ( أَوْ أَدْخَلَ ) غَيْرُ زَوْجٍ ( أُصْبُعَهُ فَرْجَ بِكْرٍ ) فَحُكُومَةٌ ( أَوْ أَدْخَلَ أُصْبُعَهُ دَاخِلَ عَظْمٍ فَخَذَفَ ) عَلَيْهِ ( حُكُومَةٌ ) ؛ لِأَنَّهُ لَا تَقْدِيرَ فِي ذَلِكَ .( 17 / 491 )
