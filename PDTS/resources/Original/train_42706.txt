( وَإِنْ رَضِيَ ) الْمُشْتَرِي ( بِالتَّصْرِيَةِ فَأَمْسَكَهَا ) أَيْ الْمُصَرَّاةَ ( ثُمَّ وَجَدَ بِهَا عَيْبًا رَدَّهَا بِهِ ) لِأَنَّ رِضَاهُ بِعَيْبٍ لَا يَمْنَعُ الرَّدَّ بِعَيْبٍ آخَرَ ( وَلَزِمَهُ ) أَيْ الْمُشْتَرِي ( صَاعُ التَّمْرِ عِوَضَ اللَّبَنِ ) الَّذِي حَلَبَهُ مِنْهَا لِمَا تَقَدَّمَ .( 9 / 98 )
