( وَلَوْ ) كَانَ الْأَسِيرُ أَوْ الْمَقْهُورُ ( مَنْ حَرَّرَهْ ذُو ذِمَّةٍ ) أَيْ : ذِمِّيٌّ فَإِنَّهُ يَرِقُّ بِخِلَافِ مَنْ حَرَّرَهُ مُسْلِمٌ ؛ لِأَنَّ الذِّمِّيَّ لَوْ الْتَحَقَ بِدَارِهِمْ جَازَ إرْقَاقُهُ فَعَتِيقُهُ أَوْلَى ( أَوْ حَمَلَتْ مِنَّا الْمَرَهْ ) الْمَسْبِيَّةُ فَإِنَّهَا تَرِقُّ بِالْأَسْرِ ؛ لِأَنَّهَا حَرْبِيَّةٌ كَغَيْرِهَا وَلَا يَمْنَعُ إسْلَامُ وَلَدِهَا بِتَبَعِيَّةِ الْأَبِ رِقَّهَا( 18 / 378 )
