( أَوْ ) حَلَفَ ( لَا وَطِئْتُك فِي هَذِهِ السَّنَةِ إلَّا مَرَّتَيْنِ ) فَلَا يَلْزَمُهُ إيلَاءٌ ؛ لِأَنَّهُ يَتْرُكُ وَطْأَهَا أَرْبَعَةَ أَشْهُرٍ ثُمَّ يَطَأُ ثُمَّ يَتْرُكُ الْوَطْءَ أَرْبَعَةَ أَشْهُرٍ ثُمَّ يَطَأُ فَلَمْ يَبْقَ مِنْ السَّنَةِ إلَّا أَرْبَعَةُ أَشْهُرٍ ، وَهِيَ دُونَ أَجَلِ الْإِيلَاءِ ( أَوْ ) حَلَفَ لَا وَطِئَ فِي هَذِهِ السَّنَةِ إلَّا ( مَرَّةً ) فَلَا يَلْزَمُهُ إيلَاءٌ ( حَتَّى يَطَأَ وَتَبْقَى الْمُدَّةُ ) لِلْإِيلَاءِ لِلْحُرِّ أَوْ الْعَبْدِ فَيَدْخُلُ عَلَيْهِ الْإِيلَاءُ
