( وَصِفَاتِ ذَاتِهِ ) وَهِيَ ( مَا دَلَّ عَلَيْهَا فِعْلُهُ ) لِتَوَقُّفِهِ عَلَيْهَا ( مِنْ قُدْرَةٍ ) وَهِيَ صِفَةٌ تُؤَثِّرُ فِي الشَّيْءِ عِنْدَ تَعَلُّقِهَا بِهِ ( وَعِلْمٍ ) وَهُوَ صِفَةٌ يَنْكَشِفُ بِهَا الشَّيْءُ عِنْدَ تَعَلُّقِهَا بِهِ ( وَحَيَاةٍ ) وَهِيَ صِفَةٌ تَقْتَضِي صِحَّةَ الْعِلْمِ لِمَوْصُوفِهَا ( وَإِرَادَةٍ ) وَهِيَ صِفَةٌ تُخَصِّصُ أَحَدَ طَرَفَيْ الشَّيْءِ مِنْ الْفِعْلِ وَالتَّرْكِ بِالْوُقُوعِ ( أَوْ ) دَلَّ عَلَيْهَا ( التَّنْزِيهُ ) لَهُ تَعَالَى ( عَنْ النَّقْصِ مِنْ سَمْعٍ وَبَصَرٍ ) وَهُمَا صِفَتَانِ يَزِيدُ الِانْكِشَافُ بِهِمَا عَلَى الِانْكِشَافِ بِالْعِلْمِ .
