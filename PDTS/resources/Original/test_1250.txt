( وَإِنْ خَلَّفَ ) الْمَيِّتَ ( تَرِكَةً وَجَبَ ) الْفِعْلُ كَقَضَاءِ الدَّيْنِ ( فَيَفْعَلُهُ الْوَلِيُّ بِنَفْسِهِ اسْتِحْبَابًا ) ؛ لِأَنَّهُ أَحْوَطُ لِبَرَاءَةِ الْمَيِّتِ ( فَإِنْ لَمْ يَفْعَلْ ) الْوَلِيُّ بِنَفْسِهِ ( وَجَبَ أَنْ يَدْفَعَ مِنْ تَرِكَتِهِ إلَى مَنْ يَصُومُ عَنْهُ عَنْ كُلِّ يَوْمٍ طَعَامَ مِسْكِينٍ ) ؛ لِأَنَّ ذَلِكَ فِدْيَةُ الصَّوْمِ لِمَا تَقَدَّمَ .
