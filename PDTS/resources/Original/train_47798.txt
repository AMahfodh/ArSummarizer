( وَيَتَأَكَّدُ ) التَّسَوُّكُ ( عِنْدَ ) كُلِّ ( صَلَاةٍ ) لِحَدِيثِ أَبِي هُرَيْرَةَ مَرْفُوعًا { لَوْلَا أَنْ أَشُقَّ عَلَى أُمَّتِي لَأَمَرْتُهُمْ بِالسِّوَاكِ عِنْدَ كُلِّ صَلَاةٍ } رَوَاهُ الْجَمَاعَةُ يَعْنِي أَمْرَ إيجَابٍ لِحَدِيثِ أَحْمَدَ { لَوْلَا أَنْ أَشُقَّ عَلَى أُمَّتِي لَفَرَضْتُ عَلَيْهِمْ السِّوَاكَ } .
