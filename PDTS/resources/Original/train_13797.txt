أَوْ عِنْدَ ) رَأْسِ الشَّهْرِ ( أَوْ مَعَ رَأْسِ الشَّهْرِ فَمَحِلُّهُ ) أَيْ : الْقَضَاءِ الَّذِي يَبَرُّ بِهِ ( عِنْدَ غُرُوبِ الشَّمْسِ مِنْ آخِرِ الشَّهْرِ ) فَيَبَرُّ بِقَضَائِهِ فِيهِ ( وَيَحْنَثُ ) بِقَضَائِهِ ( بَعْدَهُ ) أَيْ : غُرُوبِ الشَّمْسِ مِنْ آخِرِ الشَّهْرِ لِفَوَاتِ مَا حَلَفَ عَلَيْهِ ( وَلَا يَضُرُّ فَرَاغُ تَأَخُّرِ كَيْلِهِ وَوَزْنِهِ وَعَدِّهِ وَذَرْعِهِ ) لِكَثْرَتِهِ حِينَ شَرَعَ مِنْ الْغُرُوبِ ( وَ ) لَا يَضُرُّ تَأَخُّرُ فَرَاغِ أَكْلِهِ ( إذَا حَلَفَ لَيَأْكُلَنَّهُ عِنْدَ رَأْسِ الْهِلَالِ وَنَحْوِهِ وَشَرَعَ فِيهِ إذَا تَأَخَّرَ لِكَثْرَتِهِ ) لِأَنَّهُ غَيْرُ مَقْصُودٍ عَمَلًا بِالْعَادَةِ .
