ابْنُ عَرَفَةَ : سَدُّ الْحِظَارِ هُوَ تَحْصِينُ الْجِدَارِ / 97 وَقَدْ تَقَدَّمَ نَصُّ الْمُدَوَّنَةِ / 97 ( وَكَنْسُ عَيْنٍ ) / 97 تَقَدَّمَ نَصُّ الْمُدَوَّنَةِ : وَخَمُّ الْعَيْنِ / 97 ( وَسَدُّ حَظِيرَةٍ ) / 97 تَقَدَّمَ أَنَّ سَدَّ الْحِظَارِ هُوَ تَحْصِينُ الْجُدُرِ وَتَزْرِيبِهَا / 97 ( وَإِصْلَاحُ ضَفِيرِهِ ) / 97 تَقَدَّمَ نَصُّ الْمُدَوَّنَةِ : وَالْيَسِيرُ مِنْ إصْلَاحِ الضَّفِيرَةِ / 97 ( أَوْ مَا قَلَّ ) / 97 تَقَدَّمَ نَصُّ الْمُدَوَّنَةِ : يُشْتَرَطُ عَلَى الْعَامِلِ مَا تَقِلُّ مُؤْنَتُهُ / 97 .
