( وَلَا ) يَصِحُّ ( اعْتِيَاضٌ عَنْهُ ) أَيْ الْمُسْلَمِ فِيهِ ( وَلَا ) يَصِحُّ ( بَيْعُهُ أَوْ ) بَيْعُ ( رَأْسِ مَالِهِ ) الْمَوْجُودِ ( بَعْدَ فَسْخِ ) عَقْدٍ ( وَقَبْلَ قَبْضِ ) رَأْسِ مَالِهِ ( وَلَوْ ) كَانَ الْبَيْعُ ( لِمَنْ هُوَ عَلَيْهِ وَلَا حَوَالَةَ بِهِ وَلَا ) حَوَالَةَ ( عَلَيْهِ ) لِحَدِيثِ نَهْيِهِ عَلَيْهِ الصَّلَاةُ وَالسَّلَامُ { عَنْ بَيْعِ الطَّعَامِ قَبْلَ قَبْضِهِ } عَنْ رِبْحِ مَا لَمْ يُضَمْنَ وَحَدِيثِ { مَنْ أَسْلَمَ فِي شَيْءٍ فَلَا يَصْرِفْهُ إلَى غَيْرِهِ } وَلِأَنَّهُ لَمْ يَدْخُلْ فِي ضَمَانِهِ أَشْبَهَ الْمَكِيلَ قَبْلَ قَبْضِهِ وَأَيْضًا فَرَأْسُ مَالِ السَّلَمِ بَعْدَ فَسْخِهِ وَقَبْلَ قَبْضِهِ مَضْمُونٌ عَلَى الْمُسْلَمِ إلَيْهِ بِعَقْدِ السَّلَمِ أَشْبَهَ الْمُسْلَمَ فِيهِ( 5 / 129 )
