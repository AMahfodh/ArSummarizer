وَقِيلَ : مِثْلُهُ فِي الْعِبَادَةِ فَقَطْ وَقِيلَ : لَا مُطْلَقًا بَلْ يَكُونُ كَمَجْهُولِ الصِّفَةِ وَسَيَأْتِي ( وَتُعْلَمُ ) صِفَةُ فِعْلِهِ ( بِنَصٍّ ) عَلَيْهَا كَقَوْلِهِ : هَذَا وَاجِبٌ مَثَلًا ( وَتَسْوِيَةٌ بِمَعْلُومِ الْجِهَةِ ) كَقَوْلِهِ هَذَا الْفِعْلِ مُسَاوٍ لِكَذَا فِي حُكْمِهِ الْمَعْلُومِ ( وَوُقُوعِهِ بَيَانًا أَوْ امْتِثَالًا لَدَالٍّ عَلَى( 4 / 145 )
