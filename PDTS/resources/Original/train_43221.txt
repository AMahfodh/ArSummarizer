( فَإِنْ قَالَ ) الزَّوْجُ ( قَدْ عَلِمْتُ أَنِّي عِنِّينٌ قَبْلَ أَنْ أَنْكِحَهَا فَإِنْ أَقَرَّتْ ) بِذَلِكَ ( أَوْ ثَبَتَ ) عِلْمُهَا بِهِ ( بِبَيِّنَةٍ فَلَا يُؤَجَّلُ وَهِيَ امْرَأَتُهُ ) وَلَا فَسْخَ لَهَا لِدُخُولِهَا عَلَى بَصِيرَةٍ ( وَإِنْ عَلِمَتْ أَنَّهُ عِنِّينٌ بَعْدَ الدُّخُولِ فَسَكَتَتْ عَنْ الْمُطَالَبَةِ ثُمَّ طَالَبَتْ بَعْدُ فَلَهَا ذَلِكَ ) لِأَنَّهُ عَلَى التَّرَاخِي ( وَيُؤَجَّلُ سَنَةً مِنْ يَوْمِ تَرَافُعِهِ ) لَا مِنْ الْعَقْدِ وَلَا مِنْ الدُّخُولِ .
