( وَنَحْوِهَا ) كَزَرْعٍ قَائِمٍ قِيَاسًا عَلَى الثَّمَرَةِ ( وَلَا يَحْمِلُ ) مِنْ الثَّمَرَةِ إذَا مَرَّ بِهَا وَلَوْ بِلَا حَائِطٍ وَلَا نَاطُورٍ لِقَوْلِ عُمَرَ : يَأْكُلُ وَلَا يَتَّخِذُ خُبْنَةً وَهِيَ بِضَمِّ الْخَاءِ الْمُعْجَمَةِ مَا يَحْمِلُهُ فِي حِضْنِهِ ( وَلَا يَأْكُلُ مِنْ ) ثَمَرٍ ( مَجْمُوعٍ ) وَ ( مَجْنِيٍّ ) لِإِحْرَازِهِ ( وَلَا ) يَأْكُلُ مِنْ ثَمَرِ ( مَا وَرَاءَ حَائِطٍ ) أَوْ عَلَيْهِ نَاطُورٌ لِأَنَّ إحْرَازَهُ بِذَلِكَ يَدُلُّ عَلَى شُحِّ صَاحِبِهِ .
