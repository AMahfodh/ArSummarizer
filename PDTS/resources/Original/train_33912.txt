( وَهَلْ كَذَلِكَ ) أَيْ الْحَلِفُ عَلَى فِعْلِ نَفْسِهِ الْحَلِفُ عَلَى فِعْلِ غَيْرِهِ ( فِي ) صِيغَةِ ( الْحِنْثِ ) الْمُطْلَقِ فِي مَنْعِهِ مِنْ وَطْءِ زَوْجَتِهِ وَضَرْبِ أَجَلِ الْإِيلَاءِ إنْ رَفَعَتْهُ ( أَوْ لَا ) يَكُونُ كَحَلِفِهِ عَلَى فِعْلِ نَفْسِهِ فَلَا ( يُضْرَبُ ) بِضَمِّ التَّحْتِيَّةِ وَفَتْحِ الرَّاءِ ( لَهُ ) أَيْ الْحَالِفِ ( أَجَلُ الْإِيلَاءِ وَ ) لَكِنْ ( يُتَلَوَّمُ ) بِضَمٍّ فَفَتْحٍ مُثَقَّلًا أَيْ يُسْتَأْنَى ( لَهُ ) بِقَدْرِ مَا يَرَاهُ الْحَاكِمُ أَنَّهُ أَرَادَهُ بِيَمِينِهِ ثُمَّ يَحْنَثُ فِي الْجَوَابِ (( 8 / 167 )
