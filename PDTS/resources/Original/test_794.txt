قَالَهُ فِي الْقَامُوسِ ( أَوْ ) عَلَى ( بُيُوتِ نَارٍ ) تَعْبُدُهَا الْمَجُوسُ ( أَوْ ) عَلَى ( بِيَعٍ ) جَمْعُ بِيعَةٍ بِكَسْرِ الْبَاءِ الْمُوَحَّدَةِ مُتَعَبَّدِ النَّصَارَى ( وَنَحْوِهَا ) كَصَوَامِعِ الرُّهْبَانِ ( وَلَوْ ) كَانَ الْوَقْفُ عَلَيْهَا ( مِنْ ذِمِّيٍّ ) لِأَنَّهُ مَعْصِيَةٌ وَإِعَانَةٌ لَهُمْ عَلَى إظْهَارِ الْكُفْرِ ، بِخِلَافِ الْوَقْفِ عَلَى ذِمِّيٍّ مُعَيَّنٍ لِأَنَّهُ لَا يَتَعَيَّنُ كَوْنُ الْوَاقِفِ عَلَيْهِ لِأَجْلِ دِينِهِ لِاحْتِمَالِ كَوْنِهِ لِفَقْرِهِ أَوْ قَرَابَتِهِ وَنَحْوِهِمَا .
