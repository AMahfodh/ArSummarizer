( وَإِنْ نَذَرَ اعْتِكَافَ يَوْمٍ يَقْدَمُ فُلَانٌ فَقَدِمَ فِي بَعْضِ النَّهَارِ لَزِمَهُ اعْتِكَافُ الْبَاقِي مِنْهُ وَلَمْ يَلْزَمْهُ قَضَاءُ مَا فَاتَ ) مِنْ الْيَوْمِ قَبْلَ قُدُومِهِ ؛ لِأَنَّهُ فَاتَ قَبْلَ شَرْطِ الْوُجُوبِ فَلَمْ يَجِبْ ( كَ نَذْرِ اعْتِكَافِ زَمَنٍ مَاضٍ ) لِعَدَمِ انْعِقَادِهِ ، ( وَإِنْ قَدِمَ لَيْلًا لَمْ يَلْزَمْهُ شَيْءٌ ) ؛ لِأَنَّهُ إنَّمَا نَذَرَ يَوْمَ يَقْدَمُ لَا لَيْلَةَ يَقْدَمُ وَيَرُدُّ عَلَيْهِ مَا ذَكَرُوهُ فِي : أَنْتِ طَالِقٌ يَوْمَ يَقْدَمُ فُلَانٌ فَقَدِمَ لَيْلًا يَحْنَثُ مَا لَمْ يَنْوِ النَّهَارَ ، ( فَإِنْ كَانَ لِلنَّاذِرِ عُذْرٌ يَمْنَعُهُ الِاعْتِكَافَ عِنْدَ قُدُومِ فُلَانٍ مِنْ حَبْسٍ أَوْ مَرَضٍ قَضَى وَكَفَّرَ ) كَفَّارَةَ يَمِينٍ لِفَوَاتِ الْمَحَلِّ ( وَيَقْضِي بَقِيَّةَ الْيَوْمِ ) الَّذِي قَدِمَ فِيهِ فُلَانٌ ( فَقَطْ ) دُونَ مَا مَضَى مِنْهُ ؛ لِأَنَّ الْقَضَاءَ تَابِعٌ لِلْأَدَاءِ .( 6 / 194 )
