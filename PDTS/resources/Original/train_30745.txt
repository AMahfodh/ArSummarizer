وَيَجُوزُ تَجَدُّدُ الْمِلْكِ لَهُ بَعْدَ مَوْتِهِ كَمَنْ نَصَبَ شَبَكَةً وَنَحْوَهَا فَسَقَطَ فِيهَا صَيْدٌ بَعْدَ مَوْتِهِ ، فَتَحْدُثُ الدِّيَةُ عَلَى مِلْكِ الْمَيِّتِ ( وَتُحْسَبُ ) الدِّيَةُ ( عَلَى الْوَرَثَةِ ) أَيْ وَرَثَةِ الْمَقْتُولِ ( إنْ ) كَانَ ( وَصَّى بِمُعَيَّنٍ بِقَدْرِ نِصْفِهَا ) كَعَبْدٍ قِيمَتُهُ خَمْسِمِائَةِ دِينَارٍ فَيُعْطَى لِمُوصًى لَهُ( 7 / 372 )
