وَقَالَ إِسْحَاقُ : وَهُوَ السُّنَّةُ الْمُجْمَعُ عَلَيْهَا ( فَإِنْ رَأَتْهُ ) أَيْ : الدَّمَ ( قَبْلَهُ ) أَيْ : قَبْلَ خُرُوجِ بَعْضِ الْوَلَدِ ( بِثَلَاثَةِ أَيَّامٍ فَأَقَلَّ بِأَمَارَةٍ ) كَتَوَجُّعٍ ( فَ ) هُوَ ( نِفَاسٌ ) كَالْخَارِجِ مَعَ الْوِلَادَةِ ( وَلَا يُحْسَبُ ) مَا قَبْلَ الْوِلَادَةِ ( مِنْ مُدَّتِهِ ) أَيْ : النِّفَاسِ .( 2 / 98 )
