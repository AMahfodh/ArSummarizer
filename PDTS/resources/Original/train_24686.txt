( فَإِذَا أَتْلَفَ ) الرَّاهِنُ ( الرَّهْنَ فَحُكْمُهُ حُكْمُ مَا إذَا أَعْتَقَهُ غَنِيًّا ) كَمَا مَرَّ ( وَ ) الرَّهْنُ ( إنْ أَتْلَفَهُ أَجْنَبِيٌّ ) أَيْ غَيْرُ الرَّاهِنِ ( فَالْمُرْتَهِنُ يَضْمَنُهُ ) أَيْ الْمُتْلِفُ ( قِيمَتَهُ يَوْمَ هَلَكَ وَتَكُونُ ) الْقِيمَةُ ( رَهْنًا عِنْدَهُ ) كَمَا مَرَّ .
