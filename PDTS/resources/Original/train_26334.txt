قَوْلُهُ فَإِذَا تَرَكَهُ ) أَيْ الْقِيَامَ الْمَذْكُورَ فَيَشْمَلُ تَرْكَ بَعْضِهِ وَمَرَّ عَنْ النِّهَايَةِ وَالْمُغْنِي مَا يُوَافِقُهُ ( قَوْلُهُ وَبِقَوْلِي زِيَادَةً إلَخْ ) أَيْ الْمُفِيدِ أَنَّ الْقِيَامَ بَعْضٌ مُسْتَقِلٌّ ( قَوْلُهُ قِيَامُهُ ) أَيْ الْقُنُوتُ وَ ( قَوْلُهُ لِتَرْكِهِ ) أَيْ الْقِيَامِ ( قَوْلُهُ فَعَلَ ) أَيْ نَدْبًا وَ ( وَقَوْلُهُ وَإِلَّا فَلَا ) أَيْ فَلَا يُنْدَبُ وَيَبْطُلُ إنْ تَخَلَّفَ بِرُكْنَيْنِ سم ( قَوْلُهُ لِأَنَّهُ بِتَرْكِهِ إلَخْ ) قَضِيَّتُهُ أَنَّهُ لَوْ أَتَى بِهِ إمَامُهُ الْحَنَفِيُّ لَمْ يَسْجُدْ وَهُوَ أَيْضًا قَضِيَّةُ قَوْلِ الْمُغْنِي وَالنِّهَايَةِ وَلَوْ تَرَكَ الْقُنُوتَ تَبَعًا لِلْإِمَامِ الْحَنَفِيِّ سَجَدَ لِلسَّهْوِ ؛ لِأَنَّ الْعِبْرَةَ بِعَقِيدَةِ الْمَأْمُومِ عَلَى الْأَصَحِّ خِلَافًا لِلْقَفَّالِ فِي عَدَمِ السُّجُودِ فَإِنَّهُ بَنَاهُ عَلَى طَرِيقَتِهِ الْمَرْجُوحَةِ مِنْ أَنَّ الْعِبْرَةَ بِعَقِيدَةِ الْإِمَامِ ا ه .
