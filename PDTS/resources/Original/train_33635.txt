( بَابُ قَطْعِ الطَّرِيقِ ) لَمَّا فَرَغَ مِنْ بَيَانِ السَّرِقَةِ الصُّغْرَى شَرَعَ فِي بَيَانِ السَّرِقَةِ الْكُبْرَى فَقَالَ ( مَنْ قَصَدَهُ ) أَيْ قَطْعَ الطَّرِيقِ سَوَاءً كَانَ جَمَاعَةً مُمْتَنِعِينَ عَنْ طَاعَةِ الْإِمَامِ فَقَصَدُوهُ أَوْ وَاحِدًا يَقْدِرُ عَلَى الِامْتِنَاعِ فَقَصْدُهُ وَهُوَ مُبْتَدَأٌ خَبَرُهُ قَوْلُهُ الْآتِي حُبِسَ ( مَعْصُومًا ) أَيْ حَالَ كَوْنِ الْقَاصِدِ مَعْصُومَ الدَّمِ بِأَنْ كَانَ مُسْلِمًا أَوْ ذِمِّيًّا فَإِنَّهُ إنْ كَانَ مُسْتَأْمَنًا فَفِي إقَامَةِ الْحَدِّ عَلَيْهِ خِلَافٌ ( عَلَى مَعْصُومٍ ) مُتَعَلِّقٌ بِالضَّمِيرِ الْبَارِزِ فِي قَصْدِهِ أَيْ قَصْدِ الْقَطْعِ عَلَى مُسْلِمٍ أَوْ ذِمِّيٍّ حَتَّى لَوْ قَطَعَهُ عَلَى مُسْتَأْمَنٍ لَا يَجِبُ عَلَيْهِ الْحَدُّ ( فَأَخَذَ ) أَيْ أَمْسَكَ ( قَبْلَ أَخْذِ شَيْءٍ ) مِنْ الْمَارَّةِ ( وَ ) قَبْلَ ( قَتْلٍ ) لِوَاحِدٍ مِنْهُمْ أَوْ أَكْثَرَ ( حُبِسَ ) بَعْدَ التَّعْزِيرِ لِمُبَاشَرَتِهِ مُنْكَرًا ( حَتَّى يَتُوبَ ) لَا بِمُجَرَّدِ الْقَوْلِ بَلْ بِأَنْ يَظْهَرَ فِيهِ سِيمَا الصُّلَحَاءِ ( وَإِنْ أَخَذَ ) أَيْ الْقَاصِدُ ( مَالًا ) وَنَصِيبُ كُلٍّ مِنْهُ نِصَابٌ ( قُطِعَ يَدُهُ وَرِجْلُهُ مِنْ خِلَافٍ ) إنْ كَانَ صَحِيحَ الْأَطْرَافِ كَذَا فِي تُحْفَةِ الْفُقَهَاءِ ( وَإِنْ قَتَلَ بِلَا أَخْذٍ قُتِلَ حَدًّا ) لَا قِصَاصًا ( فَلَا يَعْفُو عَنْهُ وَلِيٌّ ) تَفْرِيعٌ عَلَى كَوْنِهِ حَدًّا وَلَوْ كَانَ قِصَاصًا لَعَفَا وَلِيُّ الْقِصَاصِ ( وَإِنْ قَتَلَ وَأَخَذَ قُطِعَ ثُمَّ قُتِلَ أَوْ صُلِبَ ) عُطِفَ عَلَى قُتِلَ ( أَوْ قُتِلَ ) عُطِفَ عَلَى قُطِعَ أَيْ قُتِلَ ابْتِدَاءً بِلَا قَطْعٍ ثُمَّ قُتِلَ أَوْ صُلِبَ ( أَوْ صُلِبَ حَيًّا وَيُبْعَجُ ) أَيْ يُشَقُّ بَطْنَهُ بِرُمْحٍ ( حَتَّى يَمُوتَ ) وَالْأَصْلُ فِيهِ قَوْله تَعَالَى { إنَّمَا جَزَاءُ الَّذِينَ يُحَارِبُونَ اللَّهَ وَرَسُولَهُ } الْآيَةَ .
