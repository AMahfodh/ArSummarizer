( وَيَأْكُلُ ) مِنْ لَحْمِ أُضْحِيَّتِهِ ( وَيُؤْكِلُ غَيْرَهُ ) مِنْ الْأَغْنِيَاءِ وَالْفُقَرَاءِ ( وَيَهَبُ لِمَنْ يَشَاءُ وَلَا يُعْطِي أَجْرَ الْجَزَّارِ مِنْهَا ) لِلنَّهْيِ عَنْهُ ( وَنُدِبَ التَّصَدُّقُ بِثُلُثِهَا ) لِأَنَّ الْجِهَاتِ ثَلَاثٌ الْأَكْلُ وَالِادِّخَارُ وَالْإِطْعَامُ ( وَ ) نُدِبَ ( تَرْكُهُ ) أَيْ تَرْكُ التَّصَدُّقِ ( لِذِي عِيَالٍ ) تَوْسِعَةً عَلَيْهِمْ
