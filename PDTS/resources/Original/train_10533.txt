( وَذُو بَيْتٍ مِنْ دَارٍ ) فِيهَا بُيُوتٌ كَثِيرَةٌ ( كَذِي بُيُوتٍ ) مِنْهَا ( فِي حَقِّ سَاحَتِهَا فَهِيَ بَيْنَهُمَا نِصْفَيْنِ ) كَالطَّرِيقِ ( بِخِلَافِ الشُّرْبِ ) إذَا تَنَازَعَا فِيهِ ( فَإِنَّهُ يُقَدَّرُ بِالْأَرْضِ ) : بِقَدْرِ سَقْيِهَا ( بَرْهَنَا ) أَيْ الْخَارِجَانِ ( عَلَى يَدٍ ) لِكُلٍّ مِنْهُمَا ( فِي أَرْضٍ قُضِيَ بِيَدِهِمَا ) فَتُنَصَّفُ ( وَلَوْ بَرْهَنَ عَلَيْهِ ) أَيْ عَلَى الْيَدِ ( أَحَدُهُمَا أَوْ كَانَ تَصَرَّفَ فِيهَا ) بِأَنَّ لَبَّنَ أَوْ بَنَى ( قُضِيَ بِيَدِهِ ) لِوُجُودِ تَصَرُّفِهِ .
