( وَ ) الَّذِي يُرِيدُ قَضَاءَ حَاجَتِهِ ( فِي غَيْرِ الْبُنْيَانِ ) أَنْ ( يُقَدِّمَ يُسْرَاهُ ) أَيْ يُسْرَى رِجْلَيْهِ ( إلَى مَوْضِعِ جُلُوسِهِ وَ ) يُقَدِّمَ ( يُمْنَاهُ عِنْدَ مُنْصَرِفِهِ ) مِنْهُ ( مَعَ ) إتْيَانِهِ بِ ( مَا تَقَدَّمَ ) عِنْدَ دُخُولِهِ الْخَلَاءَ ، ؛ لِأَنَّ مَوْضِعَ قَضَاءِ حَاجَتِهِ فِي الصَّحْرَاءِ فِي مَعْنَى الْمَوْضِعِ الْمُعَدِّ لِذَلِكَ فِي الْبُنْيَانِ ( وَمِثْلُهُ ) أَيْ مِثْلُ الْخَلَاءِ فِي تَقْدِيمِ الْيُسْرَى دُخُولًا وَالْيُمْنَى خُرُوجًا ( حَمَّام وَمُغْتَسَلٌ وَنَحْوُهُمَا ) مَنْ أَمَاكِنِ الْأَذَى كَالْمَزْبَلَةِ وَالْمَجْزَرَةِ ، وَكَذَا خَلْعُ نَعْلٍ وَنَحْوِهِ ( عَكْسُ مَسْجِدٍ وَمَنْزِلٍ وَنَعْلٍ ) أَيْ انْتِعَالٍ ( وَنَحْوِهِ ) كَخُفٍّ وَسُرْمُوزَةٍ ( وَقَمِيصٍ وَنَحْوِهِ ) كَقَبَاءٍ ، فَيُدْخِلُ يَدَهُ الْيُمْنَى قَبْلَ الْيُسْرَى فِي اللُّبْسِ ، وَيُقَدِّمُ الْيُسْرَى فِي الْخَلْعِ .
