( وَكَذَا ) يُقْبَلُ تَفْسِيرُهُ ( بِالْمُسْتَوْلَدَةِ فِي الْأَصَحِّ ) ؛ لِأَنَّهَا يُنْتَفَعُ بِهَا وَتُسْتَأْجَرُ وَإِنْ كَانَتْ لَا تُبَاعُ ، وَالثَّانِي يَنْظُرُ إلَى امْتِنَاعِ بَيْعِهَا ( لَا بِكَلْبٍ وَجِلْدِ مَيْتَةٍ ) ؛ لِأَنَّهُ لَا يَصْدُقُ عَلَيْهِمَا اسْمُ الْمَالِ ، وَفِي الرَّوْضَةِ كَأَصْلِهَا وَالْمُحَرَّرِ إذَا قَالَ لَهُ : عَلَيَّ مَالٌ إلَى آخِرِهِ ، وَمِنْهُ الْمَقْبُولُ بِالْمُسْتَوْلَدَةِ وَالْمُنَاسِبُ فِيهَا أَنْ يَقُولَ لَهُ : عِنْدِي مَالٌ ( وَقَوْلُهُ ) لَهُ ( كَذَا ) عَلَيَّ ( كَقَوْلِهِ ) لَهُ ( شَيْءٌ ) عَلَيَّ فَيُقْبَلُ تَفْسِيرُهُ بِمَا تَقَدَّمَ فِيهِ ( وَقَوْلُهُ شَيْءٌ شَيْءٌ أَوْ كَذَا كَذَا كَمَا لَوْ لَمْ يُكَرَّرْ ) ؛ لِأَنَّ الثَّانِيَ تَأْكِيدٌ .
