( قَوْلُهُ : لَمْ يَكُنْ لَهُمَا ) أَيْ لِلِابْنَيْنِ أَوْ الْبِنْتَيْنِ إذَا وَلَدَتْهُمَا ( قَوْلُهُ : وَوَجْهُ قَوْلِ الْمُصَنِّفِ ) يَعْنِي فِي الرَّوْضَةِ ( قَوْلُهُ : رَدًّا عَلَى الرَّافِعِيِّ ) أَيْ فِي قَوْلِهِ وَلَيْسَ الْفَرْقُ بِوَاضِحٍ وَالْقِيَاسُ التَّسْوِيَةُ ( قَوْلُهُ : إنَّهُ وَاضِحٌ ) مَقُولُ قَوْلِ الْمُصَنِّفِ ( قَوْلُهُ : أَنَّ الْمَدَارَ ) خَبَرُ قَوْلِهِ وَوَجْهُ ( قَوْلُهُ : وَهُوَ مِنْ كُلِّ ) أَيْ وَالْمُتَبَادِرُ مِنْ كُلِّ إلَخْ( 19 / 279 )
