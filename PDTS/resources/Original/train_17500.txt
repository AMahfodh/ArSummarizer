( وَمَنْ تَزَوَّجَ ) نِسَاءً ( أَوْ خَالَعَ نِسَاءً ) ، وَكَانَ تَزَوُّجُهُ لَهُنَّ ( بِمَهْرٍ ) وَاحِدٍ ( أَوْ ) كَانَ خُلْعُهُ لَهُنَّ ( عَلَى عِوَضٍ وَاحِدٍ ) ، وَلَمْ يَقُلْ بَيْنَهُنَّ بِالسَّوِيَّةِ ( صَحَّ ) فِيهِمَا ؛ لِأَنَّهُ عَقْدُ مُعَاوَضَةٍ عُلِمَ الْعِوَضُ فِيهِ إجْمَالًا فَلَمْ تُؤَثِّرْ جَهَالَةُ تَفْصِيلِهِ فَصَحَّ ، كَمَا لَوْ اشْتَرَى ثَلَاثَةَ أَعْبُدٍ بِثَمَنٍ وَاحِدٍ ( ، وَقَسْمُ ) الْمَهْرِ فِي التَّزْوِيجِ ، وَالْعِوَضِ فِي الْخُلْعِ ( بَيْنَهُنَّ ) أَيْ : الزَّوْجَاتِ أَوْ الْمُخْتَلِعَاتِ ( عَلَى قَدْرِ مُهُورِ مِثْلِهِنَّ ) ؛ لِأَنَّ الصَّفْقَةَ اشْتَمَلَتْ عَلَى أَشْيَاءَ مُخْتَلِفَةِ الْقِيمَةِ فَوَجَبَ تَقْسِيمُ الْعِوَضِ عَلَيْهَا بِالْقِيمَةِ ، كَمَا لَوْ اشْتَرَى شِقْصًا وَسَيْفًا( 8 / 487 )
