( وَشَرْطُ ) وُجُوبِ ( هَا ) أَيْ : الْفِدْيَةِ ( فِي اللُّبْسِ ) لِمُحِيطٍ مَمْنُوعٍ لُبْسُهُ بِالْإِحْرَامِ ( انْتِفَاعٌ ) بِالْمَلْبُوسِ ( مِنْ ) دَفْعِ ( حَرٍّ أَوْ بَرْدٍ ) أَيْ شَأْنًا ، وَإِنْ لَمْ يَنْتَفِعْ بِالْفِعْلِ فَمَنْ لَبِسَ ثَوْبًا شَفَّافًا لَا يَقِي حَرًّا وَلَا بَرْدًا وَتَرَاخَى زَمَنًا طَوِيلًا فَعَلَيْهِ الْفِدْيَةُ .
