( تَتِمَّةٌ ) قَالَ الشَّيْخُ تَقِيُّ الدِّينِ : تَعْلِيقُ النَّذْرِ بِالْمِلْكِ نَحْوَ : إنْ رَزَقَنِي اللَّهُ مَالًا فَلِلَّهِ عَلَيَّ أَنْ أَتَصَدَّقَ بِهِ أَوْ بِشَيْءٍ مِنْهُ يَصِحُّ اتِّفَاقًا وَقَدْ دَلَّ عَلَيْهِ قَوْله تَعَالَى : { وَمِنْهُمْ مِنْ عَاهَدَ اللَّهَ لَئِنْ آتَانَا مِنْ فَضْلِهِ } الْآيَةَ ( وَيَجُوزُ فِعْلُهُ ) أَيْ النَّذْرِ ( قَبْلَهُ ) أَيْ قَبْلَ وُجُودِ شَرْطِهِ كَإِخْرَاجِ الْكَفَّارَةِ بَعْدَ الْيَمِينِ وَقَبْلَ الْحِنْثِ ( وَقَالَ الشَّيْخُ فِيمَنْ قَالَ : إنْ قَدِمَ فُلَانٌ أَصُومُ كَذَا : هَذَا نَذْرٌ يَجِبُ الْوَفَاءُ بِهِ مَعَ الْقُدْرَةِ لَا أَعْلَمُ فِيهِ نِزَاعًا وَمَنْ قَالَ : لَيْسَ بِنَذْرٍ فَقَدْ أَخْطَأَ .
