( وَإِنْ قَالَ لِعَبْدِهِ إنْ أَقْرَرْت بِك لِزَيْدٍ فَأَنْتَ حُرٌّ سَاعَةً قَبْلَ إقْرَارِي فَأَقَرَّ بِهِ لِزَيْدٍ صَحَّ الْإِقْرَارُ ) لِخُلُوِّهِ عَنْ الْمُعَارِضِ ( دُونَ الْعِتْقِ ) لِأَنَّ عِتْقَ مِلْكِ الْغَيْرِ لَا يَصِحُّ ( وَإِنْ قَالَ ) إنْ أَقْرَرْتُ بِك لِزَيْدٍ فَ ( أَنْتَ حُرٌّ سَاعَةَ إقْرَارِي ) وَأَقَرَّ بِهِ لِزَيْدٍ ( لَمْ يَصِحَّا ) أَيْ الْإِقْرَارُ وَوَلَاءُ الْعِتْقِ لِلتَّنَافِي ( ذَكَرَهُ فِي الرِّعَايَةِ ) .( 23 / 378 )
