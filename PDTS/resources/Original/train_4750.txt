الشَّرْطُ الثَّالِثُ أَنْ يَفْضُلَ مَا يُنْفِقُهُ عَلَيْهِمْ عَنْ حَاجَتِهِ وَإِلَيْهِ الْإِشَارَةُ بِقَوْلِهِ ( إذَا فَضَلَ عَنْ قُوتِ نَفْسِهِ ) أَيْ الْمُنْفِقِ ( وَ ) قُوتِ ( زَوْجَتِهِ وَرَقِيقِهِ يَوْمَهُ وَلَيْلَتَهُ وَكِسْوَةٍ وَسُكْنَى ) لَهُمْ ( مِنْ حَاصِلٍ ) بِيَدِهِ ( أَوْ مُتَحَصِّلٍ ) مِنْ صِنَاعَةٍ أَوْ تِجَارَةٍ أَوْ أُجْرَةِ عَقَارٍ أَوْ رِيعِ وَقْفٍ وَنَحْوِهِ فَإِنْ لَمْ يَفْضُلْ عِنْدَهُ عَمَّنْ ذَكَرَ شَيْئًا فَلَا شَيْءَ عَلَيْهِ لِحَدِيثِ جَابِرٍ مَرْفُوعًا { إذَا كَانَ أَحَدُكُمْ فَقِيرًا فَلْيَبْدَأْ بِنَفْسِهِ فَإِنْ كَانَ فَضْلٌ فَعَلَى عِيَالِهِ فَإِنْ كَانَ فَضْلٌ فَعَلَى قَرَابَتِهِ } .
