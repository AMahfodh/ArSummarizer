( وَ ) لَوْ قَالَ لِامْرَأَتَيْهِ ( إنْ حَلَفْتُ بِطَلَاقِكُمَا فَأَنْتُمَا طَالِقَتَانِ وَأَعَادَهُ وَقَعَ بِكُلٍّ ) مِنْهُمَا ( طَلْقَةٌ ) لِمَا سَبَقَ ( وَإِنْ لَمْ يَدْخُلْ بِإِحْدَاهُمَا ) أَيْ الْمَرْأَتَيْنِ ( فَأَعَادَهُ بَعْدُ ) أَنْ وَقَعَ بِكُلٍّ مِنْهُمَا طَلْقَةٌ ( فَلَا طَلَاقَ ) ؛ لِأَنَّ الْحَلِفَ بِطَلَاقِ الْبَائِنِ غَيْرُ مُعْتَدٍّ بِهِ ( وَلَوْ نَكَحَ الْبَائِنَ ثُمَّ حَلَفَ بِطَلَاقِهَا طَلُقَتَا أَيْضًا طَلْقَةً طَلْقَةً ) لِانْعِقَادِ الْيَمِينِ الثَّانِيَة فِي حَقِّهِمَا جَمِيعًا وَاكْتِفَاءً بِوُجُودِ آخِرِ الصِّفَةِ فِي النِّكَاحِ لِيَقَعَ الطَّلَاقُ عَقِبَهُ وَاسْتُشْكِلَ كَمَا أَوْضَحْتُهُ فِي الْحَاشِيَةِ .
