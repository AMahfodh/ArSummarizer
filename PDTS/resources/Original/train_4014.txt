( وَإِنْ نَذَرَ ) مُكَلَّفٌ ( صَوْمًا ) وَأَطْلَقَ ( أَوْ ) نَذَرَ ( صَوْمَ بَعْضِ يَوْمٍ ) كَنِصْفِهِ ( لَزِمَهُ ) صَوْمُ ( يَوْمٍ ) تَامٍّ ( بِنِيَّةٍ مِنْ اللَّيْلِ ) لِأَنَّهُ أَقَلُّ الصَّوْمِ ( وَنَذَرَ صَوْمَ لَيْلَةٍ لَا يَنْعَقِدُ وَلَا كَفَّارَةَ ) لِأَنَّهَا لَيْسَتْ مَحَلًّا لِلصَّوْمِ كَنَذْرٍ مُسْتَحِيلٍ ( وَكَذَا نَذْرُ صَوْمِ يَوْمٍ أَتَى فِيهِ بِمُنَافٍ ) لِلصَّوْمِ نَحْوِ أَكْلٍ وَشُرْبٍ أَوْ جِمَاعٍ( 12 / 50 )
