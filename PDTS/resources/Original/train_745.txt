سَيِّدُ عُمَرَ وَقَدْ يُجَابُ بِأَنَّ نَحْوَ الْقَرَابَةِ شَامِلٌ لِوَكِيلَيْ الْمُجْبَرِ الْمَشْرُوطِ فِيهِمَا الْعَدْلُ وَالْأَمَانَةُ كَمَا أَنَّهُ شَامِلٌ لِنَحْوِ الْقَاضِي ( قَوْلُهُ : ثَمَّ ) أَيْ فِي وَلِيِّ النِّكَاحِ ( قَوْلُهُ : لِلْأَوْلِيَاءِ ) الْمُرَادُ بِهِ مَا يَشْمَلُ الْوُكَلَاءَ ( قَوْلُهُ : فِيهِ ) أَيْ الْعَقْدِ( 22 / 194 )
