وَظَاهِرُ كَلَامِ الْمَجْدِ وَالشَّيْخِ تَقِيِّ الدِّينِ لَا يَكْفِي ( وَ ) يَكْفِي ( حَائِلٌ ) بَيْنَهُ وَبَيْنَ الْقِبْلَةِ ( وَلَوْ ) كَانَ الْحَائِلُ ( كَمُؤَخَّرَةِ رَحْلٍ ) بِضَمِّ الْمِيمِ وَسُكُونِ الْهَمْزَةِ ، وَمِنْهُمْ مَنْ يُثَقِّلُ الْخَاءَ ، وَهِيَ الْخَشَبَةُ الَّتِي يَسْتَنِدُ إلَيْهَا الرَّاكِبُ ( وَيَكْفِي الِاسْتِتَارُ بِدَابَّةٍ ) لِفِعْلِ ابْنِ عُمَرَ وَتَقَدَّمَ ( وَ ) بِ ( جِدَارٍ وَجَبَلٍ وَنَحْوِهِ ) كَشَجَرَةٍ ( وَ ) يَكْفِي ( إرْخَاءُ ذَيْلِهِ ) لِحُصُولِ التَّسَتُّرُ بِهِ .
