( وَإِنْ سَلَّمَهَا ) أَيْ سَلَّمَ الْمُؤَجِّرُ الْعَيْنَ الْمُؤَجَّرَةَ ( إلَيْهِ ) أَيْ الْمُسْتَأْجِرِ ( فِي أَثْنَائِهَا ) أَيْ الْمُدَّةِ ( انْفَسَخَتْ ) الْإِجَارَةُ ( فِيمَا مَضَى ) مِنْ مُدَّةِ الْإِجَارَةِ ( وَتَجِبُ أُجْرَةُ الْبَاقِي بِالْحِصَّةِ ) أَيْ بِالْقِسْطِ مِنْ الْمُسَمَّى .( 12 / 231 )
