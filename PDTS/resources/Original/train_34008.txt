( قَوْلُهُ : وَتَقَدَّمَ الْجَمْعُ ) أَيْ فِي قَوْلِهِ وَمُطَالَبَتِهِ لِيُؤَدِّيَ مِمَّا فِي يَدِ الرَّقِيقِ إلَخْ قَوْلُهُ : كَمَا مَرَّ ) أَيْ عَلَى مَا مَرَّ وَمِنْهُ أَنَّهُ لَا بُدَّ مِنْ إعْتَاقِهِ جَمِيعَهُ ( قَوْلُهُ : وَقُلْنَا بِالْأَصَحِّ ) ضَعِيفٌ ( قَوْلُهُ : فَلَا خِيَارَ ) هَذَا هُوَ الْمُعْتَمَدُ ( قَوْلُهُ : وَفِيهَا ) أَيْ الْجَوَاهِرِ ( قَوْلُهُ : وَعَلَيْهِ دُيُونٌ ) أَيْ بِسَبَبِ التِّجَارَةِ ( قَوْلُهُ : وَمَاتَ ) أَيْ الْعَبْدُ ( قَوْلُهُ : بَلْ الْوَجْهُ ) هَذَا هُوَ الْمُعْتَمَدُ ( قَوْلُهُ : بِسَائِرِ أَنْوَاعِهِ ) دَخَلَ فِيهِ الْمُدَبَّرُ وَالْمُكَاتَبُ وَأُمُّ الْوَلَدِ ، وَيُشْكِلُ عَلَى ذَلِكَ مَا ذَكَرَهُ بَعْضُهُمْ مِنْ أَنَّ الْقِنَّ هُوَ الرَّقِيقُ الَّذِي لَمْ يَتَعَلَّقْ بِهِ سَبَبُ الْعِتْقِ عَلَى مَا مَرَّ عَنْ تَهْذِيبِ الْأَسْمَاءِ ، وَالْجَوَابُ أَنَّ الشَّارِحَ اسْتَعْمَلَ الْقِنَّ فِي مُطْلَقِ الرَّقِيقِ تَجَوُّزًا وَإِنْ لَمْ يُوَافِقْ اللُّغَةَ وَلَا كَلَامَ الْفُقَهَاءِ عَلَى مَا مَرَّ أَوَّلَ الْبَابِ ( قَوْلُهُ : لِلِاخْتِصَاصِ ) مُتَعَلِّقٌ بِإِضَافَةٍ .( 13 / 162 )
