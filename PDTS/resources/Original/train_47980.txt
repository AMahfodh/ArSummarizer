( أَوْ ) كَوْنِهِ ( إجَارَةً ) لِلْعَامِلِ ( فَاسِدَةً ) فَالْأَرْضُ وَالشَّجَرُ لِرَبِّ الْأَرْضِ وَلَا شَيْءَ مِنْهُمَا لِلْعَامِلِ حَالَ كَوْنِهَا ( كَذَلِكَ ) الْمَذْكُورُ فِي أَنَّهُ لَيْسَ لِلْعَامِلِ إلَّا قِيمَةُ غَرْسِهِ وَعَمَلِهِ ( قَوْلَانِ ) مُبْتَدَأٌ خَبَرُهُ فِي كَوْنِهِ كِرَاءً فَاسِدًا أَوْ إجَارَةً كَذَلِكَ ،( 16 / 37 )
