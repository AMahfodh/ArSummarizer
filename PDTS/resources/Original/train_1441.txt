وَشَبَّهَ فِي وُجُوبِ الْحَفْنَةِ أَيْضًا فَقَالَ ( كَحَلْقِ ) شَخْصٍ ( مُحْرِمٍ ) بِحَجٍّ أَوْ عُمْرَةٍ ( لَ ) شَخْصٍ ( مِثْلِهِ ) فِي كَوْنِهِ مُحْرِمًا بِحَجٍّ أَوْ عُمْرَةٍ بِإِذْنِهِ ( مَوْضِعَ الْحِجَامَةِ ) فَيَلْزَمُ الْحَالِقَ حَفْنَةٌ مِنْ طَعَامٍ ( إلَّا أَنْ يَتَحَقَّقَ ) الْحَالِقُ ( نَفْيَ الْقَمْلِ ) عَنْ مَوْضِعِ الْحَلْقِ فَلَا حَفْنَةَ عَلَى الْحَالِقِ وَعَلَى الْمَحْلُوقِ شَعْرُهُ فِي الْحَالَيْنِ الْفِدْيَةُ .
