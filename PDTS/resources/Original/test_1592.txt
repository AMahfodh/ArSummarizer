( وَلَوْ فَصَّلَ الثَّمَنَ ) بِأَنْ قَالَ بِعْتُك هَذَا الْعِدْلَ عَلَى أَنَّهُ عَشَرَةُ أَثْوَابٍ كُلُّ ثَوْبٍ بِدِرْهَمٍ ( فَكَذَا ) يَفْسُدُ الْبَيْعُ ( فِي الْأَكْثَرِ ) أَيْ فِيمَا إذَا كَانَ أَحَدَ عَشَرَ مَثَلًا لِأَنَّ الْعَقْدَ يَتَنَاوَلُ الْعَشَرَةَ فَعَلَى الْمُشْتَرِي رَدُّ الثَّوْبِ الزَّائِدِ وَهُوَ مَجْهُولٌ لِاحْتِمَالِ كَوْنِهِ جَيِّدًا أَوْ رَدِيًّا وَلِجَهَالَتِهِ يَصِيرُ الْمَبِيعُ أَيْضًا مَجْهُولًا فَيَفْسُدُ ( وَيَصِحُّ ) الْبَيْعُ ( فِي الْأَقَلِّ بِحِصَّتِهِ ) يَعْنِي إذَا كَانَ تِسْعَةً مَثَلًا لِأَنَّ حِصَّةَ الْمَعْدُومِ مَعْلُومَةٌ وَهُوَ دِرْهَمٌ لِكُلِّ ثَوْبٍ فَتَكُونُ حِصَّةُ الْبَاقِي مَعْلُومَةً أَيْضًا ( وَيُخَيَّرُ الْمُشْتَرِي ) إنْ شَاءَ أَخَذَ الْمَوْجُودَ بِحِصَّتِهِ مِنْ الثَّمَنِ وَإِنْ شَاءَ تَرَكَ لِتَفَرُّقِ الصَّفْقَةِ عَلَيْهِ .( 5 / 119 )
