( وَإِنْ ) قَالَ لَا أَعْلَمُ لِي بَيِّنَةً فَ ( قَالَتْ بَيِّنَةٌ نَحْنُ نَشْهَدُ لَك فَقَالَ هَذِهِ بَيِّنَتِي سُمِعَتْ ) وَهِيَ أَوْلَى مِنْ الَّتِي قَبْلَهَا لِأَنَّهُ لَا تُهْمَةَ فِيهِ ( لَكِنْ لَوْ شَهِدَتْ ) الْبَيِّنَةُ ( لَهُ بِغَيْرِهِ ) أَيْ غَيْرِ مَا ادَّعَاهُ ( فَهُوَ مُكَذِّبٌ لَهَا ) فَلَا تُسْمَعُ وَاخْتَارَ فِي الْمُسْتَوْعِبِ تُقْبَلُ فَيَدَّعِيهِ ثُمَّ يُقِيمُهَا وَفِيهِ وَفِي الرِّعَايَةِ إنْ قَالَ اسْتَحَقَّهُ وَمَا شَهِدُوا بِهِ وَإِنَّمَا ادَّعَيْتُ بِأَحَدِهِمَا لِأَدَّعِيَ الْآخَرَ وَقْتًا آخَرَ ثُمَّ شَهِدُوا بِهِ قُبِلَتْ .( 22 / 232 )
