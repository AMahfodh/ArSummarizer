رَوَاهُ الْجَمَاعَةُ إلَّا التِّرْمِذِيَّ ( وَتَبَعِيَّةُ نَسَبٍ لِأَبٍ ) إجْمَاعًا لِقَوْلِهِ تَعَالَى : { اُدْعُوهُمْ لِآبَائِهِمْ } ( مَا لَمْ يَنْتَفِ كَابْنِ مُلَاعَنَةٍ ) وَإِلَّا وَلَدَ الزِّنَا فَوَلَدُ الْقُرَشِيِّ قُرَشِيٌّ وَلَوْ مِنْ غَيْرِ قُرَشِيَّةٍ ، وَوَلَدُ قُرَشِيَّةٍ مِنْ غَيْرِ قُرَشِيٍّ لَيْسَ قُرَشِيًّا ( وَتَبَعِيَّةُ مِلْكٍ أَوْ حُرِّيَّةٍ لِأُمٍّ ) فَوَلَدُ حُرَّةٍ حُرٌّ وَإِنْ كَانَ مِنْ رَقِيقٍ ، وَوَلَدُ أَمَةٍ وَلَوْ مِنْ حُرٍّ قِنٌّ لِمَالِكِ أُمِّهِ ( إلَّا مَعَ شَرْطِ ) زَوْجِ أَمَةٍ ( حُرِّيَّةُ أَوْلَادِهَا فَهُمْ أَحْرَارٌ ) لِحَدِيثِ { الْمُسْلِمُونَ عِنْدَ شُرُوطِهِمْ } ( أَوْ ) إلَّا مَعَ ( غُرُورٍ ) بِأَنْ تَزَوَّجَ بِامْرَأَةٍ شَرَطَهَا أَوْ ظَنَّهَا حُرَّةً فَتَبِينُ أَمَةً فَوَلَدُهَا حُرٌّ وَلَوْ كَانَ أَبُوهُ رَقِيقًا( 10 / 106 )
