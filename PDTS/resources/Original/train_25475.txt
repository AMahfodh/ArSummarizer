( وَإِذَا كَانَ الْبَذْرُ وَالْأَرْضُ لِأَحَدِهِمَا وَالْعَمَلُ وَالْبَقَرُ لِلْآخَرِ أَوْ ) كَانَتْ ( الْأَرْضُ لِأَحَدِهِمَا وَالْبَقِيَّةُ ) مِنْ الْعَمَلِ وَالْبَذْرِ وَالْبَقَرِ ( لِلْآخَرِ أَوْ ) كَانَ الْعَمَلُ ( لِأَحَدِهِمَا وَالْبَقِيَّةُ ) مِنْ الْأَرْضِ وَالْبَذْرِ ( لِلْآخَرِ صَحَّتْ ) الْمُزَارَعَةُ فِي الْكُلِّ .
