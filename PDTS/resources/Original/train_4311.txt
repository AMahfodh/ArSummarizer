( حَلَّ أَكْلُهُ ) لِأَهْلِيَّةِ الصَّائِدَيْنِ ( ثُمَّ إنْ كَانَ الْكَلْبَانِ مُتَعَلِّقَيْنِ بِهِ فَهُوَ ) أَيْ الصَّيْدُ ( بَيْنَهُمَا ) أَيْ بَيْنَ صَاحِبَيْ الْجَارِحَيْنِ لِأَنَّ الظَّاهِرَ أَنَّ جَارِحَيْهِمَا قَتَلَاهُ ( وَإِنْ كَانَ أَحَدُهُمَا ) أَيْ أَحَدُ الْكَلْبَيْنِ ( مُتَعَلِّقًا بِهِ ) وَحْدَهُ ( فَهُوَ ) أَيْ الصَّيْدُ ( لِصَاحِبِهِ ) أَيْ صَاحِبِ الْجَارِحِ الْمُتَعَلِّقِ بِهِ لِأَنَّ الظَّاهِرَ أَنَّهُ هُوَ الَّذِي قَتَلَهُ ( وَعَلَى مَنْ( 21 / 290 )
