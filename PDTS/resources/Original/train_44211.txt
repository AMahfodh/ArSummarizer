( وَإِنْ بَنَى ) الْمُقِرُّ ( فَوْقَ الْبَيْتِ غُرْفَةً ) بِنَاءً عَلَى السَّطْحِ ( أُجْبِرَ ) بِالْبِنَاءِ لِلْمَفْعُولِ أَيْ : الْمُقِرُّ ( عَلَى نَقْضِهَا ) ؛ لِأَنَّهُ وَضَعَهَا بِغَيْرِ حَقٍّ ( وَ ) أُجْبِرَ أَيْضًا عَلَى ( أَدَاءِ أُجْرَةِ السَّطْحِ مُدَّةَ مُقَامِهِ فِي يَدِهِ ) ؛ لِأَنَّهُ بِيَدِهِ بِعَقْدٍ فَاسِدٍ ( وَلَهُ ) أَيْ : الْمُقِرِّ ( أَخْذُ آلَتِهِ ) الَّتِي بَنَى بِهَا الْغَرْفَةَ( 10 / 407 )
