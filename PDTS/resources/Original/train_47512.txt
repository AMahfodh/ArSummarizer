وَقَدْ عَرَفْت وَجْهَ التَّخْصِيصِ قَرِيبًا ( { وَسِبَاعُ الْبَرِّ } ) بِالْفَتْحِ ضِدَّ الْبَحْرِ ( { وَأَنْعَامُهُ } ) جَمْعُ نَعَمٍ بِالتَّحْرِيكِ ، وَقَدْ يُسَكَّنُ عَيْنُهُ ، وَهِيَ : الْإِبِلُ ، وَالْبَقَرُ ، وَالْغَنَمُ أَوْ خَاصٌّ بِالْإِبِلِ وَيُجْمَعُ عَلَى أَنَاعِيمَ كَمَا نُقِلَ عَنْ الْقَامُوسِ ( { لِأَنَّ الْعِلْمَ } ) الْمَقْرُونَ بِالْعَمَلِ ، وَالْإِخْلَاصِ ( { حَيَاةُ الْقُلُوبِ مِنْ } ) مَوْتِ ( { الْجَهْلِ ، وَمَصَابِيحُ الْأَبْصَارِ } ) يَعْنِي نُورَ الْأَبْصَارِ وَضِيَاءَهَا ( { مِنْ الظُّلَمِ } ) لِأَنَّ كُلَّ خَفِيٍّ يَنْكَشِفُ بِالْعِلْمِ .
