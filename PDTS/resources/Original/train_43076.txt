وَاَلَّذِي قُلْنَاهُ هُوَ قَوْلُ أَحْمَدَ كَمَا اقْتَضَاهُ كَلَامُ الْمُسْتَوْعِبِ وَغَيْرِهِ لِلْحَنَابِلَةِ وَقَوْلُ الْمُصَنِّفِ مِنْ زِيَادَتِهِ ( لَا تَحْدِيدًا ) تَأْكِيدٌ ( إلَّا لِغَيْبَةِ مُعَزٍّ أَوْ مُعَزًّى ) فَتَبْقَى التَّعْزِيَةُ لَهُ إلَى قُدُومِهِ قَالَ الْمُحِبُّ الطَّبَرِيُّ وَالظَّاهِرُ امْتِدَادُهَا بَعْدَهُ ثَلَاثَةَ أَيَّامٍ وَيَلْحَقُ بِالْغَيْبَةِ الْمَرَضُ وَعَدَمُ الْعِلْمِ كَمَا قَالَهُ الْمُصَنِّفُ فِي شَرْحِ الْإِرْشَادِ ( وَالتَّعْزِيَةُ ) لُغَةً التَّصْبِيرُ لِمَنْ أُصِيبَ بِالتَّعْزِيَةِ عَلَيْهِ وَشَرْعًا ( هِيَ الْحَمْلُ عَلَى الصَّبْرِ بِالْوَعْدِ بِالْأَجْرِ وَالتَّحْذِيرِ ) عَنْ الْوِزْرِ بِالْجَزَعِ ( وَ ) أَنْ ( يَدْعُوَ ) عِبَارَةُ أَصْلِهِ وَالدُّعَاءُ ( لِلْمَيِّتِ وَالْمُصَابِ فَفِي تَعْزِيَةِ كَافِرٍ بِمُسْلِمٍ وَعَكْسِهِ يَخُصُّ الْمُسْلِمَ بِالدُّعَاءِ الْأُخْرَوِيَّ ) فَيَقُولُ فِي تَعْزِيَةِ الْمُسْلِمِ بِالْمُسْلِمِ عَظَّمَ اللَّهُ أَجْرَك وَأَحْسَنَ عَزَاءَك بِالْمَدِّ أَيْ جَعَلَ صَبْرَك( 4 / 366 )
