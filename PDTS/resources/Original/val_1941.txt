( وَإِنْ وَافَقَ ) صَوْمُهُ شَهْرًا ( قَبْلَهُ ) أَيْ : قَبْلَ رَمَضَانَ كَشَعْبَانَ ( لَمْ يُجْزِهِ ) نَصَّ عَلَيْهِ ؛ لِأَنَّهُ أَتَى بِالْعِبَادَةِ قَبْلَ وَقْتِهَا فَلَمْ يُجْزِهِ كَالصَّلَاةِ فَلَوْ وَافَقَ بَعْضُهُ رَمَضَانَ فَمَا وَافَقَهُ أَوْ بَعْدَهُ أَجْزَأَهُ دُونَ مَا قَبْلَهُ ، ( وَإِنْ تَحَرَّى وَشَكَّ : هَلْ وَقَعَ ) الشَّهْرُ الَّذِي صَامَهُ ( قَبْلَهُ ) أَيْ : قَبْلَ رَمَضَانَ ( أَوْ بَعْدَهُ ؟ أَجْزَاهُ ) لِتَأْدِيَةِ فَرْضِهِ بِالِاجْتِهَادِ ، وَلَا يَضُرُّ التَّرَدُّدُ فِي النِّيَّةِ لِمَكَانِ الضَّرُورَةِ .( 5 / 472 )
