( وَ ) إذَا وَقَعَ بَيْعٌ مِنْ رَبِّهَا أَوْ إبْدَالٌ ( فُسِخَ ) إنْ كَانَ الْمَبِيعُ قَائِمًا لَمْ يَفُتْ ، ( فَإِنْ فَاتَ ) الْمَبِيعُ بِأَكْلٍ وَنَحْوِهِ ( وَجَبَ التَّصَدُّقُ بِالْعِوَضِ ) إنْ كَانَ قَائِمًا ( مُطْلَقًا ) سَوَاءٌ كَانَ الْبَائِعُ هُوَ الْمُضَحِّي أَوْ غَيْرُهُ بِإِذْنِهِ أَوْ لَا ، ( فَإِنْ فَاتَ ) الْعِوَضُ أَيْضًا بِصَرْفٍ فِي لَوَازِمِهِ أَوْ غَيْرِهَا أَوْ بِضَيَاعِهِ أَوْ تَلَفِهِ ( فَبِمِثْلِهِ ) يَتَصَدَّقُ وُجُوبًا .
