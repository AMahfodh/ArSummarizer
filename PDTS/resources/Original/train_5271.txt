( فَرْعٌ يَدُ الْعَامِلِ ) عَلَى مَا يَقَعُ فِي يَدِهِ إلَى أَنْ يَرُدَّهُ ( يَدُ أَمَانَةٍ فَإِنْ خَلَّاهُ بِتَفْرِيطٍ ) كَأَنْ خَلَّاهُ بِمَضْيَعَةٍ ( ضَمِنَ ) لِتَقْصِيرِهِ وَإِنْ خَلَّاهُ بِلَا تَفْرِيطٍ كَأَنْ خَلَّاهُ عِنْدَ الْحَاكِمِ لَمْ يَضْمَنْ ( وَإِنْ أَنْفَقَ ) عَلَيْهِ مُدَّةَ الرَّدِّ ( فَمُتَبَرِّعٌ ) إلَّا أَنْ يَأْذَنَ لَهُ الْحَاكِمُ أَوْ يَشْهَدَ عِنْدَ فَقْدِهِ لِيَرْجِعَ ( وَمَنْ وَجَدَ مَرِيضًا ) عَاجِزًا عَنْ السَّيْرِ ( بِبَادِيَةٍ ) أَوْ نَحْوِهَا ( لَزِمَهُ الْمُقَامُ مَعَهُ لَا إنْ خَافَ عَلَى نَفْسِهِ ) أَوْ نَحْوِهَا فَلَا يَلْزَمُهُ ذَلِكَ ( وَإِذَا أَقَامَ ) مَعَهُ ( فَلَا أُجْرَةَ لَهُ وَلَوْ مَاتَ ) الْمَرِيضُ ( أَوْ غُشِيَ عَلَيْهِ لَزِمَهُ إنْ كَانَ أَمِينًا حَمْلُ مَالِهِ إلَى وَرَثَتِهِ ) وَإِنْ لَمْ يَكُنْ أَمِينًا لَمْ يَلْزَمْهُ الْحَمْلُ وَإِنْ جَازَ لَهُ ( وَلَا يَضْمَنُهُ ) فِي الْحَالَيْنِ وَلَيْسَ حُكْمُ الْمَغْشِيِّ عَلَيْهِ حُكْمَ الْمَيِّتِ كَمَا قَالَهُ الْأَذْرَعِيُّ إذْ لَا عِبْرَةَ بِوَرَثَتِهِ فِي حَيَاتِهِ بَلْ حُكْمُهُ حُكْمُ الْمَرِيضِ كَمَا أَفَادَهُ كَلَامُ الرَّوْضَةِ .
