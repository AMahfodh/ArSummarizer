( وَلَهُ ) أَيْ لِلْمُدَّعِي ( تَحْلِيفُ الْمُدَّعَى عَلَيْهِ أَنَّهُ لَا يَلْزَمُهُ تَسْلِيمُهَا إلَيْهِ فَإِنْ حَلَفَ ) أَقَرَّتْ الْعَيْنُ بِيَدِهِ لِأَنَّ الْمُدَّعِي انْدَفَعَتْ دَعْوَاهُ بِالْيَمِينِ ( وَإِنْ نَكَلَ ) الْمُدَّعَى عَلَيْهِ عَنْ الْيَمِينِ ( غَرِمَ بَدَلَهَا ) لِلْمُدَّعِي أَيْ مِثْلَهَا إنْ كَانَتْ مِثْلِيَّةً وَقِيمَتَهَا إنْ كَانَتْ مُتَقَوِّمَةً ( فَإِنْ كَانَ الْمُدَّعِي لِلْعَيْنِ اثْنَيْنِ ) وَنَكَلَ عَنْ الْيَمِينِ لَهُمَا ( فَبَدَلَانِ ) لَهُمَا لِكُلِّ وَاحِدٍ مِنْهُمَا بَدَلٌ .( 22 / 259 )
